타일 세트 편집기 기본 설정

타일 세트 편집기 기본 설정 에는 자동 타일링에 대한 다음 옵션이 있습니다.

자동 타일 배경색: 이 옵션은 자동 타일 생성 섹션의 타일 세트 편집기 내에서 배경색을 설정합니다. 기본 색상은 (rgba) #808080FF입니다.

타일 세트 편집기 환경 설정에는 대화 확인 하위 섹션도 있습니다.

여기에서 사용 가능한 옵션은 타일 세트의 셀 크기를 변경할 때 팝업되는 메시지를 제어하며 이는 룸 타일맵 레이어가 렌더링되는 방식에 영향을 미칩니다(결과적으로 레이어가 잘림). 기본적으로 작업을 계속할지 취소할지 묻는 메시지가 표시되며 이 옵션을 사용하면 룸 타일맵 레이어에 대한 영향에 관계없이 타일 세트 셀 변경을 계속 진행하는 "확인"을 선택하여 이 메시지를 표시하지 않을 수 있습니다. 기본값은 "메시지 표시"입니다.
