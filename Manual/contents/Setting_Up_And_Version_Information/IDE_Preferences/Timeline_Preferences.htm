타임라인 기본 설정

타임라인 기본 설정 은 타임라인 편집기 로 작업할 때 특정 속성을 정의하는 데 사용됩니다. 사용 가능한 옵션은 다음과 같습니다.

  • GameMaker Language로의 GML Visual 변환에 대한 자동 응답: 마우스 오른쪽 버튼을 사용할 때 GML Visual 작업을 스크립트(코드)로 변환하는 타임라인 순간에 메뉴 옵션을 선택하면 단방향 변환이므로 계속할지 묻는 메시지가 표시됩니다. 이 옵션을 "확인"으로 변경하면 메시지가 표시되지 않고 대화 상자에서 "확인"을 클릭한 것처럼 변환이 진행됩니다. 기본 설정은 "메시지 표시"입니다.
  • 영구적인 작업 경고에 대한 자동 응답: 타임라인 내의 특정 작업은 취소할 수 없으며, 이 경우 계속할 것인지 묻는 경고 메시지가 표시됩니다. 이 옵션을 "확인"으로 변경하면 메시지가 표시되지 않고 대화 상자에서 "확인"을 클릭한 것처럼 작업이 계속 진행됩니다. 기본 설정은 "메시지 표시"입니다.
  • 생성 시 편집을 위한 새로운 순간 열기: 일반적으로 타임라인에 새로운 순간을 추가하면 GameMaker 가 자동으로 편집기를 열어 코드 또는 GML Visual을 추가할 수 있습니다. 단, 이 옵션의 체크를 해제하면 에디터를 열지 않고 모멘트가 추가됩니다. (더블클릭으로 열 수 있습니다. 순간). 기본적으로 이 옵션은 켜져 있습니다.
