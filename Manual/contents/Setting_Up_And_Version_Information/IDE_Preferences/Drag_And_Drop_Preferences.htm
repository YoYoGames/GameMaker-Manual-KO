드래그 앤 드롭™ 기본 설정

Drag and Drop™ 기본 설정은 Drag and Drop™ 스크립트 편집기의 모양과 느낌을 정의하는 데 사용됩니다. 사용 가능한 다양한 옵션이 각각의 기능에 대한 설명과 함께 아래에 나열되어 있습니다.

  • DnD™ 도구 상자 기본 위치: 기본적으로 DnD™ 도구 상자(다른 모든 작업 라이브러리가 있는 위치)는 편집기의 오른쪽에 배치되지만 이 옵션을 사용하여 기본값을 왼쪽으로 설정할 수 있습니다.
  • DnD™ 노드 개요 가시성 모드: 노드 "개요"는 작업 작업 공간의 특정 작업 옆에 표시되는 짧은 설명 텍스트입니다(예: 이 Destroy Instance 작업).

    항상 활성화하거나, 작업이 축소된 경우에만 활성화(기본 설정)하거나, 모두 비활성화하도록 선택할 수 있습니다.
  • GameMaker Language로의 DnD™ 변환에 대한 자동 응답: 드래그 앤 드롭 이벤트 또는 스크립트를 GML 코드로 변환하면 기본적으로 수행하려는 작업에 대해 경고하고 계속할 것인지 묻는 메시지가 표시됩니다. 이 옵션을 "확인"으로 설정하면 메시지를 표시하지 않고 변경 사항이 승인됩니다.
  • DnD™ 노드 삭제에 대한 자동 응답: 끌어서 놓기 이벤트 노드를 삭제하면 기본적으로 수행하려는 작업에 대해 경고하고 계속할 것인지 묻는 메시지가 표시됩니다. 이 옵션을 "확인"으로 설정하면 메시지를 표시하지 않고 삭제가 승인됩니다.
  • DnD™ 도구 상자 작업 축소판의 최대 크기: 작업 도구 상자 하단에는 끌어서 놓기 아이콘의 크기를 변경할 수 있는 슬라이더가 있습니다. 이 설정은 최대 허용 크기(기본값 64px)를 정의하는 데 사용됩니다.
  • DnD™ 도구 상자 작업 축소판의 최소 크기: 작업 도구 상자 하단에는 끌어서 놓기 아이콘의 크기를 변경하기 위한 슬라이더가 있습니다. 이 설정은 허용되는 최소 크기(기본값 16px)를 정의하는 데 사용됩니다.
  • DnD™ 도구 상자 작업 축소판의 기본 크기: 이 설정은 도구 상자의 모든 작업 아이콘에 대한 기본 크기를 정의하는 데 사용됩니다(기본값 32px).
  • 표시할 최근에 사용한 작업의 최대 수: 작업 도구 상자의 맨 위에는 가장 최근에 사용한 작업을 보관하는 라이브러리가 있습니다. 이 옵션을 사용하여 표시할 최근 작업의 최대 수를 변경할 수 있습니다(기본값 5).
  • DnD™ 액션을 연결하는 체인 의 두께: DnD 코드 블록(기본값 3px)에서 다양한 액션을 연결하는 체인 그래픽의 두께를 정의합니다.
  • 노드 드래그를 시작하기 위한 마우스 다운 시간(ms): 노드를 클릭하고 드래그하면 작업 코드 내에서 노드의 위치가 변경됩니다. 그러나 잠시 동안 마우스를 누른 상태에서 이동하면 작업 공간 내에서 노드 위치가 이동하지만 코드 순서 내에서는 이동하지 않습니다. 이 설정은 노드를 이동(재배치하지 않음)하기 전에 마우스 버튼을 누른 상태에서 기다려야 하는 시간(밀리초)을 설정합니다. 기본값은 350입니다.
  • DnD™ 노드 개요 패널 숨기기: 여기에서 Action Code Editor의 개요 패널을 숨기도록 선택할 수 있습니다. 기본적으로 이것은 비활성화되어 있고 활성화하면 편집기에서 작업 체인의 일반 텍스트 목록이 제거됩니다.

DnD™ 댓글을 위한 하위 메뉴도 있습니다.

주석 기본 설정은 Drag and Drop™ 동작 스크립트에 추가된 주석에 대한 다양한 색상 및 서식 옵션을 제어합니다. 이러한 옵션은 다음과 같습니다.

  • 색상: 창에 쓰여지는 텍스트의 기본 색상입니다(기본값 ARGB #FF53E270).
  • 글꼴 이름: 사용할 글꼴의 이름(사용 가능한 글꼴은 시스템에 설치된 글꼴에 따라 다름).
  • 스타일: 굵게 또는 기울임꼴 과 같이 사용할 글꼴 스타일(모든 창에서 동일한 스타일을 허용하는 것은 아니며 사용된 글꼴에 따라 다름).
  • Size: 주어진 출력 창에 대해 글꼴을 설정할 크기입니다.
  • 앤티 앨리어싱: 여기에서 출력 창 글꼴 렌더링에 대한 앤티 앨리어싱 모드를 설정할 수 있습니다. 기본값 사용 을 선택하여 IDE가 결정하도록 선택하거나 이러한 옵션 중 하나를 선택하여 강제로 켜 거나 끌 수 있습니다.
