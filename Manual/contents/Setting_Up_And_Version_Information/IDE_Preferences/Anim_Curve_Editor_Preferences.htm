애니메이션 곡선 편집기 기본 설정

애니메이션 곡선 기본 설정 은 애니메이션 곡선 편집기 로 작업할 때의 특정 속성을 정의하는 데 사용됩니다. 사용 가능한 옵션은 다음과 같습니다.

  • 곡선 삭제에 대한 자동 응답: 이 옵션은 애니메이션 곡선 편집기에서 곡선을 삭제할 때 표시되는 팝업 메시지를 제어하는 데 사용할 수 있습니다. 여기서 기본 동작은 경고 메시지를 표시하는 것이지만 "확인"으로 설정할 수 있으며 대화 상자는 표시되지 않습니다.
