플러그인 기본 설정

플러그인 기본 설정 은 GameMaker IDE에 추가된 추가 플러그인을 제어하는 데 사용됩니다. 기본적으로 여기에는 GameMaker 프로젝트를 Git 과 통합하기 위한 소스 제어 플러그인 전용 섹션이 하나 있습니다.

소스 제어(Git)

이 섹션에서는 Git SCM 플러그인이 GameMaker 와 함께 작동하도록 하는 데 필요한 기본 설정을 다룹니다.

  • Git 실행 파일 경로: GameMaker가 모든 소스 제어 명령에 사용하는 Git 실행 파일의 위치입니다. 위의 이미지에서 볼 수 있듯이 이것은 cmd Git 설치 폴더.

이 아래에서 소스 제어를 위한 병합 도구 및 차이 도구 를 다음 설정( 선택 사항 )으로 설정할 수 있습니다.

  • 병합(도구 위치): 여기에서 Git 병합 도구의 전체 파일 경로를 지정합니다.
  • 병합(도구 옵션): 이 필드에서 병합 도구를 사용할 때마다 실행할 추가 명령을 추가할 수 있습니다.
  • Diff(도구 위치): 여기에서 Git diff 도구의 전체 파일 경로를 지정합니다.
  • Diff(도구 옵션): 이 필드에서 diff 도구를 사용할 때마다 실행할 추가 명령을 추가할 수 있습니다.
