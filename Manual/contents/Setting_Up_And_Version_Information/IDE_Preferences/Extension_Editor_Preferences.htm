확장 편집기 기본 설정

Extension Editor 기본 설정 은 아래에 설명된 정확한 세부 정보와 함께 사용할 때 표시되는 메시지와 관련된 몇 가지 옵션을 다룹니다.

  • 파일 삭제에 대한 자동 응답: 기본 설정은 삭제할 것인지 묻는 메시지를 표시하는 것이지만 "확인"으로 설정하면 확인을 클릭한 것처럼 이 메시지 없이 모든 삭제가 완료될 수 있습니다. 이미.
  • 프록시 파일 삭제에 대한 자동 응답: 위와 같이 프록시 파일을 삭제할 때 표시되는 메시지를 제어합니다.
