리소스 기본 설정

여기에 있는 기본 설정에는 프로젝트에서 특정 리소스 를 그리는 방법에 대한 옵션이 포함되어 있습니다.

  • 새 프로젝트의 초기 기본 그리기 색상: 글꼴을 사용하거나 색상을 지정하지 않고 GameMaker 기능 또는 GML Visual 작업을 사용하여 모양을 그리는 경우 여기에 설정된 색상이 사용됩니다. 기본 색상은 흰색( ARGB: $FFFFFFFF )입니다.
  • 새 프로젝트의 초기 기본 창 색상: 이 옵션은 HTML5 대상에 대한 HTML 페이지 배경색을 설정하고 모든 대상(HTML5 포함)의 방 외부에 표시되는 색상을 설정하는 데 사용됩니다. 기본 색상은 검정색(ARGB: $000000FF )입니다.
