배경 기본 설정

배경 환경 설정은 작업 공간 배경과 표시할 내용 및 표시 방법을 다룹니다. 옵션은 다음과 같습니다.

  • 배경 이미지: 이것은 다른 작업 공간의 배경으로 사용될 이미지 파일입니다. 기본 이미지가 제공되며 기본적으로 활성화되어 있지만 여기에서 자신의 이미지를 선택할 수 있습니다. 선택할 수 있는 이미지 유형은 다음과 같습니다. *.png, *.gif, *.jpg, *.jpeg, *.tif, *.tiff, *.ico, *.bmp, *.2bp 또는 *.mdi, 그러나 초기 이미지 형식에 관계없이 모든 파일은 다음으로 사용하기 위해 저장됩니다. *.png 파일, 그래서 애니메이션 *.gif 예를 들어 파일은 단일 초기 프레임으로만 렌더링됩니다.
  • Image Scaling: 이것은 다른 작업 공간 내에서 배경 이미지의 크기를 조정하는 방법을 제어합니다. 선택 사항은 다음과 같습니다.
    • 늘이기 - 보이는 독에 따라 크기가 변경되어 작업 공간에 맞게 이미지가 늘어납니다.
    • 타일 - 작업 공간을 덮도록 이미지가 타일링됩니다.
    • 최적 맞춤 - 1:1 맞춤이 아닌 경우 가장자리 주위에 막대가 추가되어 이미지가 작업 공간에 비례하여 크기가 조정됩니다.
    • 바탕 화면 늘이기 - 배경이 전체 작업 공간을 덮고 도크를 열거나 닫을 때 크기가 변경되지 않고 이동합니다.
    • 화면 늘이기 - 배경이 창 크기의 영역을 덮고 독이 열리거나 닫힐 때 이미지를 더 많거나 적게 표시하도록 배경이 잘리거나 확장됩니다.
  • 작업 공간에 배경 이미지 표시: 이 옵션은 선택한 배경 이미지를 표시할지 여부를 제어합니다. 기본적으로 이것은 켜짐으로 선택되어 있습니다.
