대화 상자 기본 설정 확인

대화 확인 기본 설정은 GameMaker IDE에서 특정 작업을 수행할 때 표시되는 일부 일반 확인 팝업을 처리합니다. 옵션은 다음과 같습니다.

  • 키보드 단축키를 통한 작업 공간 닫기에 대한 자동 응답: 위에서 언급한 키보드 단축키를 사용하여 열려 있는 탭을 닫을 수 있지만, 때로는 필요한 빈 탭을 실수로 닫을 수 있습니다. 이를 방지하려면 이 옵션을 "메시지 표시"(기본 옵션)로 설정하여 빈 탭을 정말로 닫을 것인지 묻습니다. 다른 설정은 바로 가기를 사용하는 경우 아무 작업도 수행하지 않는 "아니요"이고 메시지 없이 탭을 닫으려면 "확인"입니다.
  • 실행 중인 빌드 다시 시작에 대한 자동 응답: 빌드를 다시 시작하고 다른 이전 빌드가 아직 실행 중인 경우 현재 빌드를 먼저 중지할지 묻는 메시지가 표시됩니다. 이 메시지를 선택 취소하면 해당 메시지가 표시 되지 않고 GameMaker 가 응답으로 "확인"을 클릭했다고 가정합니다. 이 옵션은 기본적으로 "메시지 표시"로 설정되어 있습니다.
  • YYC 디버그 시도 시 경고에 대한 자동 응답: YYC 대상은 네이티브 코드로 컴파일되므로 내장 디버거를 사용하여 디버그할 수 없습니다. YYC 대상에서 디버거를 사용하려고 하면 이를 알리는 경고가 표시되며 프로젝트가 대신 VM을 사용하여 컴파일된다는 사실이 표시됩니다. 이 옵션은 기본적으로 "메시지 표시"로 설정되어 있지만 "확인"으로 설정하면 이 경고가 표시되지 않습니다.
  • 프로젝트 레이아웃 재설정에 대한 자동 응답: 레이아웃 메뉴 에서 프로젝트 레이아웃 재설정을 선택하면 기본적으로 열려 있는 리소스 창을 닫는 방법에 대한 경고 메시지가 표시됩니다. 그러나 "확인"을 선택하여 이 메시지를 표시하지 않고 레이아웃을 자동으로 재설정하거나 "아니요"를 선택하여 아무 작업도 수행하지 않도록 할 수 있습니다. 기본값은 "메시지 표시"입니다.
