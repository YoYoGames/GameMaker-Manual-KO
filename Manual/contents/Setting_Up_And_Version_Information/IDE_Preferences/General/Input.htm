입력 기본 설정

입력 기본 설정 은 IDE 내에서 마우스 및 키보드 입력의 특정 측면을 제어합니다. 사용 가능한 옵션은 다음과 같습니다.

  • 느린 두 번 클릭 시간(ms): IDE의 일부 영역에서는 리소스 이름을 변경할 때와 같이 올바르게 작동하기 위해 느린 두 번 클릭이 필요합니다(빠른 두 번 클릭하면 리소스 편집기가 열리고 느린 두 번 클릭하면 이름을 바꿀 수 있습니다. ). 이 옵션을 사용하면 감지 속도를 설정할 수 있으며 기본값은 500ms입니다.
  • Scroll lock time(ms): 이 옵션은 GameMaker 가 겹치는 스크롤 창 사이에서 기다려야 하는 시간을 처리합니다. 기본적으로 "A" 창에서 스크롤할 때 커서가 "B" 창의 다른 스크롤 가능한 영역 위에 있을 수 있으므로 이 기본 설정은 "A" 스크롤을 중지한 후 다음으로 스크롤할 수 있기 전에 기다려야 하는 시간을 정의합니다. B"이므로 실수로 다른 패널을 스크롤하기 시작하지 않습니다. 기본값은 500ms입니다.
  • 패닝할 때 화면 주위에 커서 감싸기: 활성화되면 작업 공간을 패닝할 때 화면 경계 주위에 커서를 래핑합니다. 기본값은 활성화되어 있습니다.
  • 스크롤 속도: 이것은 다른 작업 공간 내에서 스크롤 속도를 제어합니다. 기본값은 1입니다.
  • 바탕 화면 최대화 해제까지의 탭 드래그 거리(px): IDE 내 기본 위치에서 작업 영역 탭을 드래그할 때 해당 시점에 활성화된 작업 영역이 하나만 있으면 GameMaker 창이 최대화 해제되고 마우스와 함께 이동합니다. (둘 이상의 작업 공간이 활성화되어 있는 경우 작업 공간을 유지하기 위해 IDE의 새 인스턴스가 생성됩니다.) 이 설정은 작업 공간이 탭을 인식하기 전에 탭을 끌어야 하는 거리를 제어하며 기본값은 50px입니다.
  • 창 내용과 상호 작용하기 위한 최소 확대/축소 수준(%): 이 옵션은 마우스가 더 이상 작업 공간 창과 상호 작용할 수 없는 확대/축소 지점을 제어합니다(예: 노드 추가, 창 끌기 등...). 기본값은 40입니다.
  • 마우스 휠 줌에는 Ctrl 키가 필요합니다. 이 옵션을 선택하면 모든 편집기(및 기본 작업 영역)에서 / + 마우스 휠, 선택을 취소하면 마우스 휠 MMB 아이콘으로만 확대/축소가 활성화됩니다.
  • Ctrl + MMB를 사용하여 수동 대신 상속된 이벤트 열기: 체크 시 / + 마우스 가운데 버튼 개체에서 수동이 아닌 개체 상속 이벤트(있는 경우)를 엽니다.
  • 라인 점프를 방지하기 위해 텍스트 선택에 바이어스 추가: 이 옵션을 선택하면 텍스트 창에서 단일 라인을 선택하는 동안 생각한 라인 높이가 채워져 수직 부정확성이 덜 발생합니다. 이것은 기본적으로 활성화되어 있습니다.
  • 모든 장치에서 랩톱 모드 활성화: 이 옵션을 선택하면 IDE에 랩톱 모드 버튼이 표시되어 GameMaker 를 실행하는 장치가 랩톱인지 여부에 관계없이 필요에 따라 켜거나 끌 수 있습니다. 기본적으로 선택 해제되어 있습니다.
  • 강제 랩탑 모드: 이 설정은 랩탑 모드의 사용 여부를 변경하며 기본적으로 활성화되어 있습니다. 랩톱에서 IDE를 시작할 때 랩톱 모드가 자동으로 활성화되지 않도록 이 기능을 비활성화할 수 있습니다.
