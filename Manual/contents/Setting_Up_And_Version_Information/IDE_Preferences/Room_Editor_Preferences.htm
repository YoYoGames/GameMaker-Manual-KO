룸 편집기 기본 설정

룸 편집기 기본 설정 은 룸 편집기 의 모양과 동작을 정의하는 데 사용됩니다. 일반 설정은 다음과 같습니다.

  • 체인 페이드 아웃 시작 거리(px): 체인 링크가 페이드 아웃되는 부모 창으로부터의 거리를 설정합니다. 기본값은 10px입니다.
  • 체인 페이드 아웃 정지 거리(px): 체인 링크가 페이드 아웃을 종료할 부모 창으로부터의 거리를 설정합니다. 기본값은 100px입니다.
  • 체인 최소 알파: 체인 알파에 대한 페이드 아웃 폴오프 지점 이후의 최종 알파 값을 설정합니다. 기본값은 0.3입니다.
  • 기본 레이어 깊이 간격: 룸 편집기에서 여러 레이어를 생성하면 레이어 목록 내 위치에 따라 자동으로 깊이가 할당됩니다. 이 깊이는 여기에 주어진 양만큼 증가합니다. 기본값은 100이므로 각 레이어는 이전 레이어 + 깊이 100이 됩니다.
  • 기본 방 너비: 새 방의 기본값으로 선택될 초기 너비를 설정합니다. 기본값은 1024픽셀입니다.
  • 기본 방 높이: 새 방의 기본값으로 선택될 초기 높이를 설정합니다. 기본값은 768px입니다.
  • 기본 룸 레이어 추가: 새로운 룸 리소스를 생성할 때 GameMaker 는 배경 레이어와 인스턴스 레이어라는 몇 가지 "기본" 레이어를 생성합니다. 새 방에 미리 만들어진 레이어가 없도록 여기에서 해당 동작을 끌 수 있습니다. 기본적으로 켜져 있습니다.
  • 방 보기 표시: 방 안에 보기가 활성화되어 있는 경우 보기 경계가 방 안에 있는 윤곽선으로 표시되며 편집기 자체의 전용 버튼을 사용하여 이 영역의 가시성을 토글할 수 있습니다. 그러나 이 옵션을 선택 취소하여 방을 열 때 보기 경계가 보이지 않도록 선택할 수 있습니다(버튼을 사용하여 다시 표시할 수 있음). 기본값은 켜짐으로, 방을 열 때 뷰 경계가 표시됩니다.
  • Inspector 선택 색상: Inspector Multi-Select 목록에서 선택한 항목을 강조 표시하는 데 사용되는 색상입니다.
  • 기타 색상 옵션: 룸 편집기 인터페이스 전체에서 사용되는 다양한 색상과 관련이 있으며 견본을 두 번 클릭하면 해당 설정의 색상을 편집할 수 있습니다.
  • 마스터 볼륨 설정: 룸 편집기에서 미리보기 재생을 위한 마스터 볼륨 설정으로 0과 1 사이의 값입니다. 기본값은 1입니다.

아래 나열된 하위 범주에서 추가 옵션을 편집할 수 있습니다.

확인 대화 상자

확인 대화 상자 기본 설정은 특정 작업을 수행할 때 회의실 편집기에 표시되는 메시지를 변경하는 데 사용됩니다. 변경할 수 있는 메시지는 다음과 같습니다.

  • 레이어 깊이 업데이트에 대한 자동 응답: 레이어 깊이를 변경하면 다른 모든 레이어 깊이도 자동으로 업데이트된다는 경고 메시지가 표시됩니다. 이 옵션을 사용하면 메시지를 표시하거나 "확인"을 클릭한 것처럼 레이어를 업데이트하거나 "아니요"를 클릭한 것처럼 아무 것도 업데이트하지 않도록 선택할 수 있습니다.
  • 레이어 잠김 경고에 대한 자동 대응: 방에 있는 레이어를 삭제하려고 시도했는데 잠긴 경우 기본적으로 메시지가 표시됩니다. 그러나 이 옵션을 "확인"으로 설정하면 메시지가 표시되지 않으며 레이어를 삭제할 수 없습니다.
  • 항목 잠김 경고에 대한 자동 응답: 항목을 삭제하려고 하면 잠긴 상태가 되며 기본적으로 메시지가 표시됩니다. 그러나 이 옵션을 "확인"으로 설정하면 메시지가 표시되지 않으며 항목을 삭제할 수 없습니다.
  • 항목 소유자 레이어 잠김 경고에 대한 자동 응답: 레이어에서 항목을 삭제하려고 할 때 레이어가 잠긴 경우(항목이 아닌 경우에도), 기본적으로 다음을 수행할 수 있다는 메시지가 표시됩니다. 하지마. 그러나 이 옵션을 "확인"으로 설정하면 메시지가 표시되지 않으며 항목을 삭제할 수 없습니다.
  • 시퀀스/스프라이트를 부적절한 레이어로 드래그할 때 새 자산 레이어 생성: 자산 레이어가 아닌 레이어에 스프라이트(또는 시퀀스)를 추가하려고 하면 기본적으로 추가할 수 있다는 메시지가 표시됩니다. 완료하지 않고 드래그 중인 항목에 대한 새 자산 레이어를 생성할 것인지 묻습니다. 그러나 이 옵션을 "확인"으로 설정하면 메시지가 표시되지 않고 자동으로 스프라이트(및 시퀀스)에 대한 새 레이어가 생성됩니다.
  • 상위 회의실 항목 삭제에 대한 자동 응답: 여기에서 상위 회의실의 일부인 항목(모든 하위 회의실에 있는 항목)을 삭제할 때 발생하는 작업을 선택할 수 있습니다. 기본값은 경고 메시지를 표시하고 수행할 작업을 결정할 수 있도록 하는 것이지만 "모두 삭제"로 설정하여 메시지를 표시하지 않고 상위 방과 모든 하위 항목에서 항목 삭제를 계속하거나 다음을 수행할 수 있습니다. 삭제를 중단하는 메시지를 표시하지 않으려면 "모두 유지"를 선택하십시오.
  • 상위 룸 레이어 삭제에 대한 자동 응답: 여기에서 상위 룸의 일부인(따라서 모든 하위 룸에 있는) 레이어를 삭제할 때 발생하는 작업을 선택할 수 있습니다. 기본값은 경고 메시지를 표시하고 수행할 작업을 결정할 수 있도록 하는 것이지만 "모두 삭제"로 설정하여 메시지를 표시하지 않고 상위 방과 모든 하위 방에서 레이어 삭제를 계속 진행하거나 다음을 수행할 수 있습니다. 삭제를 중단하는 메시지를 표시하지 않으려면 "모두 유지"를 선택하십시오.
  • 잘못된 자산 드래그 오류에 대한 자동 응답: 해당 자산 유형에 대한 것으로 지정되지 않은 레이어에 자산을 추가하려고 하면(예: 인스턴스 레이어에 대한 경로 추가) 기본적으로 이를 알리는 메시지가 표시됩니다. 당신은 그것을 할 수 없습니다. 그러나 이 옵션을 "확인"으로 설정하면 메시지가 표시되지 않으며 자산을 추가할 수 없습니다. 이 옵션은 인스턴스, 스프라이트 또는 시퀀스에는 적용되지 않습니다.
  • 개체를 부적절한 레이어로 드래그할 때 새 인스턴스 레이어 생성: 인스턴스 레이어가 아닌 레이어에 개체 인스턴스를 추가하려고 하면 기본적으로 수행할 수 없다는 메시지가 표시되고 묻습니다. 드래그 중인 항목에 대한 새 인스턴스 레이어를 생성하려는 경우. 그러나 이 옵션을 "확인"으로 설정하면 메시지가 표시되지 않고 인스턴스에 대한 새 레이어가 자동으로 생성됩니다.
  • 잘못된 레이어 선택 드래그 오류에 대한 자동 응답: 레이어 편집기에서 여러 레이어를 선택하거나 레이어를 전혀 선택하지 않은 경우 리소스를 룸 편집기로 드래그하면 어떻게 되는지 선택할 수 있습니다. 기본값은 경고 메시지를 표시하는 것이지만 이 옵션을 "확인"으로 설정하여 리소스를 추가하지 않고 메시지를 표시하지 않고 계속할 수 있습니다.
  • 잠겼거나 보이지 않는 레이어 끌기 오류에 대한 자동 응답: 여기에서 잠겼거나 보이지 않는 것으로 플래그가 지정된 레이어로 리소스를 끌 때 발생하는 작업을 선택할 수 있습니다. 기본값은 경고 메시지를 표시하는 것이지만 이 옵션을 "확인"으로 설정하여 리소스를 추가하지 않고 메시지를 표시하지 않고 계속할 수 있습니다.

그리드 설정

그리드 설정 기본 설정은 그리드 오버레이가 작동하는 방식을 변경하는 데 사용되며 다음 옵션을 사용할 수 있습니다.

  • 격자 수평 간격(px): 방 격자의 격자 셀에 대한 수평 공간을 픽셀 단위로 설정합니다. 기본값은 32px입니다.
  • 격자 수직 간격(px): 방 격자의 격자 셀에 대한 수직 공간을 픽셀 단위로 설정합니다. 기본값은 32px입니다.
  • 격자 표시: 이 옵션은 방을 처음 열 때 방 격자를 켜거나 끕니다. 기본적으로 켜져 있지만 꺼져 있더라도 룸 편집기 자체에서 적절한 버튼을 사용하여 다시 활성화할 수 있습니다.
  • 격자에 맞추기: 이 옵션을 사용하면 방이 처음 열릴 때 기본적으로 방에 자산이 방 격자에 스냅되도록 할지 여부를 설정할 수 있습니다. 기본적으로 켜져 있지만 꺼져 있더라도 룸 편집기 자체에서 적절한 버튼을 사용하여 다시 활성화할 수 있습니다.
  • 격자선 색상: 이 옵션을 사용하면 방 격자선의 색상(알파 구성요소 포함)을 설정할 수 있습니다. 기본값은 ( ARGB ) #30FFFFFF입니다.


스마트 가이드

이 섹션에서는 룸 편집기에 대한 스마트 가이드 기본 설정 에 대해 자세히 설명합니다. 사용 가능한 옵션은 다음과 같습니다.

  • 스마트 가이드 활성화: 룸 에디터에서 스마트 가이드 사용 여부를 설정합니다. 기본값은 활성화되어 있습니다.
  • 눈금자 활성화: 이 옵션은 룸 편집기에서 측면 눈금자를 활성화/비활성화합니다. 기본값은 활성화되어 있습니다.
  • Snap Ruler To Edges: 이 옵션을 활성화하면 눈금자에서 룸 편집기로 끌어온 안내선을 룸에 배치된 모든 자산으로 스냅합니다. 기본값은 활성화되어 있습니다.
  • 스냅 거리: 이 설정은 가이드가 자산 측면에 "스냅"되기 전에 자산에서 가이드가 떨어져야 하는 거리를 룸 편집기에 알려줍니다. 기본값은 6px입니다.
