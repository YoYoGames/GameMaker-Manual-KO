자산 브라우저 기본 설정

자산 브라우저 기본 설정 은 자산 브라우저 및 해당 요소가 작동하는 방식에 대한 특정 속성을 정의하는 데 사용됩니다. 주요 옵션은 다음과 같습니다.

  • 빠른 액세스 표시: 활성화되면 이 옵션은 자산 브라우저 상단에 "빠른 액세스" 섹션을 표시합니다. 이 옵션은 기본적으로 활성화되어 있습니다.
  • 새 프로젝트에 기본 폴더 생성: 이 옵션을 활성화하면 모든 새 프로젝트가 이미 생성된 기본 자산 폴더로 시작되고 비활성화되면 모든 프로젝트에 필요한 기본 공간이 있는 공간 폴더만 생성됩니다. 이 옵션은 기본적으로 활성화되어 있습니다.
  • 경고 없이 동시에 열 수 있는 최대 자산 수: 다음을 사용하여 한 번에 여러 자산을 열 수 있습니다. / + 그들을 선택하고 마우스 오른쪽 버튼을 클릭하려면 모두 열기를 선택합니다. 이 옵션에 지정된 수보다 더 많이 선택하면 한 번에 너무 많은 자산을 열면 저사양 컴퓨터에서 성능 문제가 발생할 수 있으므로 경고가 표시됩니다. 기본값은 10입니다.
  • 최대 최근 편집 항목: 이 설정은 자산 브라우저의 "최근" 섹션에 나열된 자산 수를 제어합니다. 기본값은 10입니다.
  • 색상 팔레트 크기: 여기에서 자산 그룹의 자산 브라우저 팔레트에서 사용할 수 있는 색상 수를 설정할 수 있습니다. 기본값은 16이며 최소값은 16이고 최대값은 100입니다.
  • 빠른 액세스에서 자산 색상 적용: 이 옵션을 선택하면 주어진 자산 유형에 대해 설정된 색상을 사용하여 브라우저의 빠른 액세스 섹션에서 자산에 색상을 지정합니다. 이 옵션을 선택 취소하면 자산 색상이 사용되지 않습니다. 이것은 기본적으로 활성화되어 있습니다.
  • 그룹 색상이 하위 색상보다 우선합니다. 선택하면 최상위 그룹 색상을 사용하여 그룹 내의 모든 자산에 색상을 지정하고 개별 자산에 대해 설정된 색상을 무시합니다. 기본적으로 비활성화되어 있습니다.
  • 자산 대화 스타일 생성: 자산 브라우저에서 "자산 추가" 버튼을 클릭하면 프로젝트에 추가할 수 있는 다양한 자산을 보여주는 창이 표시됩니다. 이 창은 목록 보기 또는 그리드 보기로 표시될 수 있으며 이 설정은 기본적으로 표시되는 것을 결정합니다. 기본값은 그리드입니다.
  • 기본 정렬 순서: 브라우저에서 자산의 기본 정렬 순서를 설정하는 데 사용할 수 있습니다. 알파벳 순서는 A에서 Z 또는 Z에서 A입니다. 기본값은 A에서 Z입니다.
  • 기본 정렬 폴더 우선: 이 옵션이 활성화되면 자산 브라우저는 폴더를 먼저 정렬한 다음 "느슨한" 자산을 두 번째로 정렬하고, 비활성화하면 모든 자산과 폴더를 함께 정렬합니다. 이것은 기본적으로 활성화되어 있습니다.
  • Sprite 두 번 클릭하면 이미지 편집기가 열립니다. 기본적으로 두 번 클릭하면 모든 자산에서 편집을 위해 해당 자산이 열리지만 종종 스프라이트를 처리할 때 새 자산을 생성할 때 초기 설정 후에 열어서 스프라이트 속성이 아닌 이미지를 편집합니다. 이 옵션을 선택하면 스프라이트 자산 자체 대신 스프라이트 자산에 대한 이미지 편집기가 열립니다. 마우스 오른쪽 버튼을 클릭하여 스프라이트 편집기를 열 수 있습니다. 편집 을 선택합니다. 이 옵션은 기본적으로 꺼져 있습니다.
  • 개체 더블 클릭은 모든 개체 이벤트를 엽니다. 기본적으로 더블 클릭 모든 자산에서 편집을 위해 해당 자산이 열리지만 종종 개체를 처리할 때 새 자산을 생성할 때 초기 설정 후에 열어 개체 속성이 아닌 이벤트만 편집합니다. 이 옵션을 선택하면 새 작업 공간에서 다른 개체 이벤트가 열리고 각 이벤트에는 새 작업 공간 내에서 고유한 탭이 제공됩니다. 마우스 오른쪽 버튼을 클릭하여 개체 편집기를 열 수 있습니다. 편집을 선택합니다. 이 옵션은 기본적으로 꺼져 있습니다.
  • 방 순서 그룹 해제는 빈 그룹을 제거합니다. 활성화되면 그룹 폴더에 속한 모든 방의 그룹을 해제하면 그룹 폴더도 제거됩니다. 비활성화되면 동일한 작업이 방의 그룹을 해제하지만 자산 브라우저에 빈 그룹 폴더를 남깁니다. 이것은 기본적으로 활성화되어 있습니다.
  • 기본 확대/축소(%): 새 프로젝트를 시작할 때 자산 브라우저의 기본 확대/축소 비율을 설정합니다. 기본값은 100%입니다.
  • 최소 줌(%): 자산 브라우저에 허용되는 최소 줌 값을 설정합니다. 기본값은 50%입니다.
  • 최대 줌(%): 자산 브라우저에 허용되는 최대 줌 값을 설정합니다. 기본값은 250%입니다.
  • 트리 라인 표시: 여기에서 자산 브라우저에서 자산 측면에 표시되는 트리 라인을 활성화/비활성화할 수 있습니다. 트리 라인을 표시하려면 기본적으로 선택되어 있습니다.
  • 필터 변경 시 트리 확장: 이 옵션은 필터 옵션을 변경할 때 자산 브라우저가 축소된 트리 노드를 확장하도록 합니다. 옵션을 비활성화하면 이를 방지할 수 있습니다. 기본값은 활성화되어 있습니다.

확인 대화 상자

여기에서 자산 브라우저에서 자산으로 작업할 때 확인 대화 상자의 동작을 선택할 수 있습니다. 사용 가능한 옵션은 다음과 같습니다.

  • 자산 삭제에 대한 자동 응답: 자산 브라우저에서 자산을 삭제하면 일반적으로 계속 진행할 것인지 묻는 경고가 표시됩니다. 이 옵션을 "확인"으로 설정하면 계속 진행하기 위해 "확인" 버튼을 클릭한 것처럼 메시지가 표시되지 않습니다. 기본값은 "메시지 표시"입니다.
  • 방 순서 그룹 삭제에 대한 자동 응답: 자산 브라우저에서 방이 포함된 그룹 폴더를 삭제할 때 일반적으로 확실하고 계속할 것인지 묻는 경고가 표시됩니다(삭제될 그룹의 모든 방이 나열됨). 이 옵션을 "확인"으로 설정하면 확인 버튼을 클릭하여 그룹의 모든 방과 그룹 폴더 자체를 삭제한 것처럼 메시지가 표시되지 않습니다. 기본값은 "메시지 표시"입니다.
  • 여러 자산 열기에 대한 자동 응답: 위에 일반 자산 브라우저 환경 설정에서 언급했듯이 자산 브라우저에서 한 번에 여러 자산을 열 때 숫자가 지정된 수를 초과하면 경고를 받을 수 있습니다. 이 옵션을 "확인"으로 설정하면 계속 진행하기 위해 "확인" 버튼을 클릭한 것처럼 경고가 표시되지 않습니다. 기본값은 "메시지 표시"입니다.
  • GameMaker Language로의 GML Visual 변환에 대한 자동 응답: GML Visual 노드를 GML로 변환하는 옵션을 선택하면 취소할 수 없으므로 변경 사항을 확인하라는 메시지가 표시됩니다. 이 옵션을 "확인"으로 설정하면 계속 진행하기 위해 "확인" 버튼을 클릭한 것처럼 경고가 표시되지 않습니다. 기본값은 "메시지 표시"입니다.
  • 노드 끌기를 통한 방 부모 설정에 대한 자동 응답: 방 관리자 창 "상속" 섹션에서 방을 다른 방으로 끌어서 끌어오고 있는 방의 자식으로 설정할 수 있습니다. 이 작업을 수행할 때 확신하고 계속할 것인지 묻는 경고가 표시됩니다. 이 옵션을 "확인"으로 설정하면 계속 진행하기 위해 확인 버튼을 클릭한 것처럼 메시지가 표시되지 않습니다. 기본값은 "메시지 표시"입니다.
  • 중복 그룹 이름 오류에 대한 자동 응답: 자산을 구성하기 위해 그룹 폴더를 생성하고 브라우저의 다른 폴더와 동일한 이름을 지정하면 계속할 것인지 묻는 메시지가 표시되며 동일한 이름의 폴더가 두 개 있습니다.. 이 옵션을 "확인"으로 설정하면 계속 진행하기 위해 "확인" 버튼을 클릭한 것처럼 경고가 표시되지 않습니다. 기본값은 "메시지 표시"입니다.
  • 노드 끌기를 통한 방 상위 설정에 대한 자동 응답: 방 관리자에서 방을 다른 방 위로 드래그한 다음 놓으면 방을 현재 방의 "자식"으로 만들 것인지 묻는 메시지가 표시됩니다. 떨어졌다. 이 옵션을 "확인"으로 설정하면 계속 진행하기 위해 "확인" 버튼을 클릭한 것처럼 경고가 표시되지 않습니다. 기본값은 "메시지 표시"입니다.
