개체 편집기 기본 설정

개체 편집기 기본 설정 은 개체 편집기 창의 특정 속성을 정의하는 데 사용됩니다. 기본 섹션에는 다음 옵션이 있습니다.

  • 클릭하여 열기 이벤트 방법: 이 옵션은 개체 이벤트를 열기 위해 두 번 클릭 또는 한 번 클릭을 수행해야 하는지 여부를 제어합니다. 기본적으로 더블 클릭으로 설정되어 있습니다.
  • Inherited Events alpha: 이 옵션을 사용하면 개체의 이벤트 목록에서 "상속된 이벤트가 그려질" 알파를 변경할 수 있습니다. 기본적으로 이것은 0.5로 설정됩니다(값은 0과 1 사이여야 함).
  • 이벤트 레이블 형식: 여기에서 개체 편집기 이벤트 목록의 이벤트 레이블에 적용할 수 있는 다양한 텍스트를 활성화하거나 비활성화할 수 있습니다. 기본적으로 이벤트 설명만 활성화됩니다.
  • 기본 이벤트 콘텐츠 추가: 기본적으로 이 옵션은 활성화되어 있으며 모든 새 이벤트에 대해 코드 편집기에 코드를 추가할 수 있음을 알려주는 간단한 주석을 코드 편집기에 추가하기만 하면 됩니다.
  • 기본 이벤트 내용: 여기에서 새 이벤트에 대한 코드 창에 기본적으로 추가되는 주석을 편집할 수 있습니다.

개체 편집기에는 다음 하위 범주도 있습니다.

확인 대화 상자

이 섹션에서는 다음 옵션을 사용할 수 있는 개체 편집기를 사용하는 동안 표시될 수 있는 대화 상자를 다룹니다.

  • Gamemaker Language로의 GML Visual 변환에 대한 자동 응답: 마우스 오른쪽 버튼을 사용할 수 있습니다. GML Visual 작업 공간 내에서 해당 이벤트에 대한 모든 GML Visual을 GML 코드로 변환하는 옵션을 선택합니다. 여기서 기본 작업은 계속할지 묻는 경고 메시지를 표시하는 것이지만 "확인"으로 설정할 수 있으며 변환 전에 대화 상자가 표시되지 않습니다.
  • 물리 정점 제한 초과에 대한 자동 응답: 이 옵션은 물리 모양 편집기에서 허용된 정점 수를 초과할 때 표시되는 팝업 메시지를 제어하는 데 사용할 수 있습니다. 여기서 기본 동작은 경고 메시지를 표시하는 것이지만 "확인"으로 설정할 수 있으며 대화 상자는 표시되지 않습니다.
  • 이벤트 삭제 시 자동 선택: 이 옵션은 개체 편집기 이벤트 목록에서 이벤트를 삭제할 때 표시되는 팝업 메시지를 제어하는 데 사용할 수 있습니다. 여기서 기본 동작은 경고 메시지를 표시하는 것이지만 "확인"으로 설정할 수 있으며 대화 상자는 표시되지 않습니다.
