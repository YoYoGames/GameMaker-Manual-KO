이미지 편집기 기본 설정

이미지 편집기 기본 설정 은 이미지 편집기 창의 모양과 느낌을 정의하는 데 사용됩니다. 다음은 일반 옵션입니다.

  • 기본적으로 스프라이트 컨트롤 오버레이 확장: 이 옵션을 선택하면 편집을 위해 스프라이트를 열 때마다 스프라이트 컨트롤과 프레임 정보가 이미지 편집기에 표시됩니다(접기 버튼을 사용하여 다시 닫을 수 있습니다., 그러나 나중에 다시 스프라이트를 편집하면 다시 열립니다). 체크하지 않으면 스프라이트를 열 때 스프라이트 컨트롤과 프레임 정보가 숨겨집니다. 기본값은 이 옵션을 선택하지 않는 것입니다.
  • 기본 루프 모드: 이미지 편집기의 루프 버튼은 두 가지 다른 모드에서 작동하도록 설정할 수 있습니다. 애니메이션이 끝날 때 첫 번째 프레임 또는 핑퐁에서 애니메이션이 이미지를 통해 끝까지 올라가고 다시 시작 부분으로 다시 내려갑니다. 여기서 기본값은 "루프"입니다.
  • 얇은 선택 윤곽선 그리기: 이 옵션을 선택하면 이미지의 크기에 관계없이 선택 영역의 윤곽선 너비가 절대 1픽셀이 됩니다. 영상. 기본값은 이 옵션을 선택하는 것입니다.
  • 선택 윤곽 기본 색상: 단면 윤곽은 두 가지 색상을 번갈아 가며 나타나는 점선입니다. 이 옵션을 사용하면 사용할 첫 번째 색상(알파 포함)을 설정할 수 있으며 기본값은 ARGB #FFFFFD0입니다.
  • 선택 윤곽 보조 색상: 단면 윤곽은 두 가지 색상을 번갈아 가며 나타나는 점선입니다. 이 옵션을 사용하면 사용할 두 번째 색상(알파 포함)을 설정할 수 있으며 기본값은 ARGB #0000D0입니다.
  • 편집기 배경색: 스프라이트 프레임을 편집하는 동안 이미지가 축소 또는 축소되거나 창의 전체 도면에 맞지 않을 수 있습니다. 이 경우 여기에서 설정할 수 있는 배경색이 표시됩니다. 기본값은 ARGB #282828FF입니다.

이미지 편집기 기본 설정에는 다음과 같은 하위 범주도 있습니다.

배경

여기에서 스프라이트 이미지를 편집할 배경에 대한 옵션을 설정할 수 있습니다. 기본적으로 이것은 회색 사각형 격자이지만 다음 설정을 사용하여 변경할 수 있습니다.

  • 캔버스 스타일: 여기에서 " 블록 패턴 " 또는 " 단색 "이 될 수 있는 배경의 "스타일"을 설정합니다. 단일 색상은 단순히 배경을 원하는 색상으로 설정하는 반면(아래에 설명된 대로 색상 1 설정 사용) 블록 패턴은 기본 설정입니다(아래에 표시된 나머지 대화 옵션을 사용하여 편집할 수도 있음). 기본값은 " 블록 패턴 "입니다.
  • 캔버스 블록 크기: 배경으로 블록 패턴 을 선택한 경우 여기에서 개별 블록의 크기를 설정할 수 있습니다. 기본값은 16px이며 최소값은 2px, 최대값은 256px입니다.
  • 캔버스 기본 색상: 배경에 단일 색상 을 선택한 경우 사용되는 색상은 이 색상입니다. 색상 견본을 클릭하여 색상 편집기를 열고 변경할 수 있습니다. 블록 패턴 옵션을 사용할 때 대체 블록의 첫 번째 색상을 설정합니다. 기본 색상은 ( ARGB ) #FFC3C3C3입니다.
  • 캔버스 보조 색상(블록 패턴 스타일): 배경으로 블록 패턴 을 선택한 경우 색상 1(위)과 대체하는 데 사용되는 색상이 이 색상입니다. 색상 견본을 클릭하여 색상 편집기를 열고 변경할 수 있습니다. 기본 색상은 (ARGB) #FF808080입니다.

확인 대화 상자

여기에서 현재 스프라이트에 대해 다른 작업을 수행할 때 확인 대화 상자의 동작을 선택할 수 있습니다. 그들은:

  • 프레임 삭제에 대한 자동 응답: 기본적으로 프레임을 삭제할 때 정말로 삭제할 것인지 묻는 메시지가 표시됩니다. 이 옵션을 "확인"으로 설정하면 이 메시지를 제거하고 추가 메시지를 표시하지 않고 프레임 삭제를 허용할 수 있습니다. 기본값은 "메시지 표시"입니다.
  • 스트립 가져오기 중 이미지 교체에 대한 자동 응답: 이미지 드롭다운 메뉴에서 스트립 가져오기를 선택하면 계속할지 여부에 대한 옵션과 함께 이미 스프라이트 리소스에 있는 이미지를 덮어쓸 것이라는 경고 메시지가 표시됩니다. 여기에서 이 메시지를 표시할지 여부를 설정하거나 IDE에서 "예" 또는 "아니요" 옵션을 선택하도록 할 수 있습니다. 기본값은 "메시지 표시"입니다.
  • 도구 변경 시 변경 사항 커밋에 대한 자동 응답: 프레임을 편집한 후 그리기 도구를 변경하면 기본적으로 변경 사항을 커밋할지 묻는 메시지가 표시됩니다. 이 옵션을 "확인"으로 설정하면 이 메시지를 제거하고 도구를 변경할 때 추가 메시지가 표시되지 않고 프레임 편집이 커밋되도록 허용할 수 있습니다. "아니오"를 선택하면 도구를 변경할 때 프레임 편집이 자동으로 취소됩니다. 기본값은 "메시지 표시"입니다.
  • 잠긴 레이어 경고에 대한 자동 응답: 잠긴 레이어를 편집하려고 하면 기본적으로 이 작업을 수행할 수 없다는 메시지가 표시됩니다. 이 옵션을 "확인"으로 설정하면 메시지가 표시되지 않고 에디션이 자동으로 실패합니다.
  • 그룹 선택 경고에 대한 자동 응답: 레이어 편집기에서 여러 레이어를 선택한 다음 그리기 또는 편집을 시도하면 기본적으로 불가능하다는 메시지가 표시됩니다. 여기에서 "확인"을 선택하여 이 메시지를 제거하고 단일 레이어만 선택될 때까지 이미지를 함께 편집하는 데 자동으로 실패할 수 있습니다. 기본값은 "메시지 표시"입니다.



그리드 설정

격자 설정 기본 설정은 다음 옵션을 사용하여 이미지 격자가 표시되는 방식을 설정하는 데 사용됩니다.

  • 격자 가로 간격(px): 이 값은 이미지 격자의 가로줄 간격을 설정합니다. 기본값은 1px입니다.
  • 격자 세로 간격(px): 이 값은 이미지 격자의 세로줄 간격을 설정합니다. 기본값은 1px입니다.
  • 그리드 표시: 이 옵션을 선택하면 이미지 그리드가 켜진 상태로 이미지 편집기가 열립니다. 기본값은 꺼져 있습니다.
  • Snap To Grid: 이 옵션을 선택하면 이미지 편집기가 자동으로 특정 도구를 이미지 격자에 맞춥니다. 기본값은 꺼져 있습니다.
  • 격자선 색상: 여기에서 이미지 격자의 색상(알파 값 포함)을 설정할 수 있습니다. 기본값은 ( ARGB ) #FF000000입니다.

도구

여기에서 일부 그리기 도구가 작동하고 표시되는 방식에 대한 ddifferent 옵션을 설정할 수 있습니다.

호 도구 를 사용하는 경우 그런 다음 여기에서 노드 속성을 그리는 데 사용되는 다양한 색상을 설정할 수 있습니다.

  • 앵커 포인트 색상: 스플라인/호의 다른 앵커 포인트에 대한 색상을 설정합니다. 기본값은 ( ARGB ) #FFFF0000입니다.
  • 제어점 색상: 기준점의 각 제어점 "핸들"에 대한 색상을 설정합니다. 기본값은 (ARGB) #FFFF0000입니다.
  • 접선 색상: 기준점과 기준점을 연결하는 선의 색상을 설정합니다. 기본값은 (ARGB) #FF00FFFF입니다.
  • 선택한 포인트 색상: 스플라인/호에서 선택한 앵커 포인트에 사용할 색상을 설정합니다. 기본값은 (ARGB) #FF00FF00입니다.
  • 포인트 크기: 사용할 다양한 기준점과 제어점의 크기를 설정합니다. 기본값은 10px입니다.
