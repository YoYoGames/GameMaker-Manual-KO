스플라인 편집기 기본 설정

스플라인 편집기 기본 설정 은 IDE 의 다른 특정 측면(예: 물리 모양 편집기)에 영향을 주기는 하지만 주로 경로 편집기 의 부드러운 경로가 표시되는 방식을 정의하는 데 사용됩니다. 다음 옵션이 있습니다.

  • 삽입 경로 포인트 정밀도(px): 패스에 포인트를 삽입하기 위한 정밀도 값(픽셀 단위)을 설정합니다. 즉, 새 포인트를 추가하는 대신 포인트를 삽입하려면 마우스가 이 범위 내에 있어야 합니다. 기본값은 20px입니다.
  • 앵커 포인트 너비: 앵커 포인트의 너비를 픽셀 단위로 설정합니다. 기본값은 8px입니다.
  • 앵커 포인트 높이: 앵커 포인트의 높이를 픽셀 단위로 설정합니다. 기본값은 8px입니다.
  • 경로 색상: 편집할 때 경로를 표시하는 데 사용할 색상입니다. 기본값은 ( ARGB ) #FFFFFFFF입니다.
  • 강조 표시된 경로 선분 색상: 선택한 경로 선분을 강조 표시하는 데 사용할 색상입니다. 기본값은 (ARGB) #FFFFFFFF입니다.
  • 잘못된 경로 색상: 유효하지 않은 경로를 강조 표시하는 데 사용할 색상입니다. 기본값은 (ARGB) #FF0000FF입니다.
  • 앵커 포인트 색상: 패스 앵커 포인트를 그리는 데 사용할 색상입니다. 기본값은 (ARGB) #FFC0C0C0입니다.
  • 잘못된 기준점 색상: 패스 기준점을 그리는 데 사용할 색상입니다. 기본값은 (ARGB) #FF800000입니다.
  • 선택한 기준점 색상: 강조 표시된(선택된) 패스 기준점을 그리는 데 사용할 색상입니다. 기본값은 (ARGB) #FF00A4FF입니다.
  • 잘못된 앵커 포인트 색상 선택: 강조 표시되었지만 유효하지 않은 앵커 포인트를 그리는 데 사용할 색상입니다. 기본값은 (ARGB) #FFA400FF입니다.
  • 마우스 오버 앵커 포인트 색상: 활성화된 앵커 포인트를 그리는 데 사용할 색상입니다. 기본값은 (ARGB) #FFFFFFFF입니다.
  • 마우스 오버 유효하지 않은 앵커 포인트 색상: 활성이지만 유효하지 않은 앵커 포인트를 그리는 데 사용할 색상입니다. 기본값은 (ARGB) #FF0000FF입니다.
  • 마스크 오버레이 색상: 활성화된 경우 마스크의 오버레이 색상을 설정합니다(위 참조). 기본값은 (ARGB) #FF404040입니다.
