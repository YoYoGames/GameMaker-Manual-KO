시작 화면 기본 설정

시작 화면 기본 설정을 사용하면 시작 페이지가 작동하는 방식을 변경할 수 있습니다. 주요 옵션은 다음과 같습니다.

  • 둘러보기 표시: IDE를 시작할 때 소개 둘러보기를 표시할지 여부를 선택할 수 있습니다. 둘러보기를 처음 건너뛰면 자동으로 비활성화됩니다.
  • 설치 마법사 표시: IDE가 시작될 때 설치 마법사를 표시할지 여부를 선택할 수 있습니다. 설치 마법사는 기본적으로 새 프로젝트를 만드는 데 필요한 단계를 안내합니다.
  • 최근 프로젝트 목록 스타일: 시작 페이지의 최근 프로젝트 목록 표시를 "목록"과 "그리드" 사이에서 변경할 수 있습니다.
