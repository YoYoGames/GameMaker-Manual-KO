일반 기본 설정

일반 기본 설정 은 GameMaker IDE 의 일반 동작을 정의하는 데 사용됩니다. 가장 일반적인 것들은 메인 페이지에 나열되고 다른 많은 것들은 이 페이지 아래에서 더 자세히 설명되는 여러 하위 범주로 나뉩니다.

  • IDE 언어: 이 옵션을 사용하면 제공된 옵션 목록에서 IDE의 언어를 선택할 수 있습니다. 기본값은 영어 입니다.
  • IDE 글꼴: 이 옵션을 사용하면 IDE 전체에서 사용되는 글꼴을 변경할 수 있습니다. 기본값은 Open Sans 입니다.
  • GameMaker Editor 의 최대 프레임 속도: 이 옵션을 사용하면 GameMaker IDE가 업데이트할 최대 프레임 속도를 설정할 수 있으므로 모니터의 업데이트 속도와 일치하도록 프레임 속도를 설정할 수 있습니다. 기본값은 60입니다.
  • IDE 스킨: 여기에서 IDE에서 사용할 수 있는 다양한 스킨을 선택할 수 있습니다. 기본적으로 GameMaker 에는 밝은 스킨과 어두운 스킨의 두 가지가 있으며 초기 설정은 어두운 스킨을 사용합니다.
  • 기본 오디오 장치: 여기에서 GameMaker IDE에서 사용할 기본 오디오 장치를 선택할 수 있습니다. 새로 고침 버튼을 클릭하여 목록을 새로 고칠 수 있습니다..
  • 종료 확인에 대한 자동 응답: 이 옵션을 사용하면 GameMaker 를 종료할 때 확인 메시지를 표시할지 여부를 설정할 수 있습니다. 기본적으로 "메시지 표시"로 설정되어 있지만 확인을 클릭한 것처럼 메시지를 건너뛰도록 설정할 수 있습니다.
  • 시스템 글꼴 변경 대화 상자에 대한 자동 응답: 이 옵션을 사용하면 시스템 글꼴이 추가, 제거 또는 편집되었을 때 확인 메시지를 표시할지 여부를 설정할 수 있습니다. 기본적으로 "메시지 표시"로 설정되어 있지만 확인을 클릭한 것처럼 메시지를 건너뛰도록 설정할 수 있습니다.
  • 알림 활성화: 이 옵션을 선택하면 IDE에서 새 런타임 등에 대한 알림을 표시할 수 있습니다. 기본적으로 켜져 있습니다.
  • 알림 시간 초과(ms): 알림이 표시되는 시간(밀리초)을 설정합니다. 기본값은 6000입니다.
  • 종료 시 로그아웃: 이 옵션을 선택하면 GameMaker 가 닫힐 때 IDE가 자동으로 사용자 세션을 로그아웃합니다. 이것은 주로 교육 사용자를 대상으로 합니다. 선택하면 프로그램이 종료될 때마다 사용자가 로그아웃되어 다른 사용자가 사용할 수 있도록 세션 자리를 확보할 수 있습니다. 기본적으로 꺼져 있습니다.
  • 시작할 때 시작 화면 표시: 기본적으로 켜져 있으며 프로그램을 시작할 때 GameMaker 에 간단한 시작 화면을 표시하도록 지시합니다. 선택을 취소하면 시작 화면이 표시되지 않습니다.
  • 단축키를 통해 빈 탭 닫기 허용: 키보드 단축키를 사용하여 열려 있는 탭을 닫을 수 있습니다. / + W, 하지만 이 옵션을 비활성화하면 해당 바로 가기가 더 이상 작동하지 않습니다. 이것은 기본적으로 활성화되어 있습니다.
  • 파일을 IDE로 드래그하여 편집할 새 리소스 열기: 특정 리소스(예: 스프라이트 또는 사운드)는 GameMaker IDE의 기본 작업 공간으로 드래그할 수 있으며 자산 브라우저에서 감지되어 새 리소스로 추가됩니다. 이 경우 기본적으로 해당 리소스 편집기가 편집을 위해 열리지만 이 옵션을 선택 취소하면 편집기가 열리지 않으며 수동으로 열어야 합니다.
  • 생성 시 편집을 위해 새 리소스 열기: 이 옵션을 선택하면 자산 브라우저에서 새 리소스를 생성할 때마다 해당 리소스 편집기가 자동으로 열립니다. 이것은 기본적으로 선택되어 있으며 선택을 취소하면 새로 생성된 모든 리소스를 수동으로 열어야 함을 의미합니다.
  • 변경된 파일 자동 다시 로드: 이 옵션을 선택하면 IDE가 자동으로 파일을 스캔하고 확인하여 GameMaker 외부에서 변경된 파일이 있는지 확인하고 변경된 파일이 있으면 다시 로드합니다.
  • IDE 전환 애니메이션 비활성화: GameMaker IDE에는 창 열기 등과 같은 여러 애니메이션 전환이 있습니다. 그러나 모든 사람의 취향에 맞는 것은 아니므로 이전 또는 전력이 부족한 컴퓨터에서 비활성화하면 성능 향상을 경험할 수 있습니다. 비활성화됩니다. 여기에서 비활성화할 수 있으며 기본적으로 켜져 있습니다.
  • 피드백 설문조사 수신 동의: 이 옵션을 선택하면 GameMaker 에 피드백 설문조사에 참여하기를 원한다는 의미입니다. 즉, 작업하는 동안 때때로 작업 중인 영역에 대한 메시지가 표시되고 평가 및 피드백을 요청하는 메시지가 표시됩니다. 이 옵션은 기본적으로 켜져 있습니다.
  • DPI 재정의 활성화: 이 옵션을 사용하면 기본 DPI 설정을 자신의 것으로 재정의할 수 있습니다. 먼저 옵션을 활성화한 다음 원시 DPi 재정 의 값(기본값은 96)을 설정하거나 기본 DPi의 백분율 (기본값은 100%)을 설정해야 하며, 일단 변경되면 IDE를 다시 시작하라는 메시지가 표시됩니다. 현재 열려 있는 프로젝트에서 수행 중인 작업을 저장해야 할 수 있습니다. DPi의 최소 백분율은 50%, 최대 백분율은 500%이며 정보 아이콘 위로 마우스를 가져가면 사용 중인 디스플레이에 권장되는 최대 설정을 볼 수 있습니다.

    모니터의 권장 DPi 설정을 초과하면 DPi를 재설정할 수 있는 알림이 표시됩니다.

위의 옵션은 일반 기본 설정에서 사용할 수 있는 기본 옵션이지만 아래 나열된 하위 범주에서 사용할 수 있는 추가 옵션이 많이 있습니다.
