GML 코드 기본 설정

GML 코드 기본 설정 은 코딩을 위한 일부 언어 기능이 GameMaker 의 IDE에 표시되는 방식을 정의하는 데 사용됩니다. GML 코드 에 대한 옵션은 다음과 같습니다.

  • 상태 표시줄에 인수 유형 표시: 이 옵션은 사용자 정의 스크립트를 사용하고 다른 인수 등을 식별하기 위해 JSDoc 표기법을 사용한 경우와 관련됩니다(자세한 내용은 여기 참조). 이 옵션을 선택하고 스크립트에서 인수 유형을 지정하면 편집기 맨 아래에 있는 상태 표시줄에 표시됩니다. {}. 기본적으로 꺼져 있습니다.
  • 상태 표시줄에 인수 설명 표시: 이 옵션을 선택하면 JSDoc 표기법을 사용하여 사용자 정의 스크립트에 제공한 모든 설명이 편집기 창 하단의 상태 표시줄에 표시됩니다. 기본적으로 꺼져 있습니다.
  • 시스템 기능에 대한 미국 철자 표시: 기본적으로 GameMaker 는 기능에 GB 영어 철자를 사용합니다. draw_text_colour(), 그러나 이것은 익숙하지 않을 수 있으므로 이 옵션을 사용하여 미국 철자로 전환할 수 있으므로 기능이 draw_text_color(), 예를 들어. 기본적으로 꺼져 있습니다.
  • 표시할 최대 구문 오류 수: 이 값은 한 번에 표시될 구문 오류의 최대 수를 설정합니다. IDE가 지연되는 경우 이 값을 줄일 수 있습니다. 기본적으로 50으로 설정되어 있습니다.
  • 변수 참조 경고 활성화: 활성화되면 변수가 어딘가에 정의되었지만 실제로 코드에서 사용되지 않는 경우와 같이 변수에 대한 잠재적인 문제에 플래그를 지정합니다. 비활성화되면 이러한 경고가 더 이상 표시되지 않습니다. 기본값은 활성화된 것입니다.
  • 영역으로 문 접기 활성화: 활성화하면 코드의 대괄호가 영역처럼 접힐 수 있습니다. 이것은 기본적으로 활성화되어 있습니다.
