라이선스 정보

GameMaker 를 다운로드하여 설치한 후 처음 사용할 때 라이선스 세부 정보를 입력하라는 메시지가 표시됩니다.

요요 계정

귀하의 라이선스는 YoYo 계정 (YYA)에 연결되어 있으며 GameMaker 를 사용하려면 계정을 만들어야 합니다. 이 계정은 소유하고 있는 YoYo Games 제품에 대한 모든 라이선스 세부 정보를 보유하고 있으며 GameMaker Marketplace 에도 사용됩니다. 다음 페이지에서 YYA에 대한 추가 정보를 찾을 수 있습니다.

오페라 계정

" Opera 단일 로그인 " 버튼을 사용하여 Opera 계정을 사용하여 로그인할 수 있지만, 여전히 YoYo 계정이 있어야 하고 Opera 계정이 YoYo 계정과 먼저 연결되어 있어야 합니다( 이 페이지에서 수행할 수 있음 ).

프록시 지원

GameMaker 설치에 프록시 지원이 필요한 경우 프록시 를 클릭하여 간단히 추가할 수 있습니다. 그러면 다음 창이 나타납니다.

이제 프록시 세부 정보를 추가할 수 있습니다. 위에 표시된 필드 중 일부는 선택 사항이며 필요하지 않은 필드가 있으면 필드를 비워 두십시오. 그러나 계속하기 전에 Windows 인터넷 옵션 에서 프록시를 설정했고 사용자 이름/비밀번호가 필요하지 않은 경우 "있는 그대로" 설정해야 하며 아마도 설정할 필요가 없을 것입니다. GameMaker 의 프록시.

프록시에 사용자 이름 및/또는 비밀번호가 필요한 경우 위 창에 표시된 필드에 추가해야 합니다. 이러한 경우 사용자 이름/비밀번호/프록시 세부 정보가 항상 필요하며 프록시 IP에도 포트가 추가되어야 합니다. 예: xx.xx.xx.xx:port.

관련 정보를 추가했으면 확인을 클릭하고 평소처럼 GameMaker 에 로그인합니다.
