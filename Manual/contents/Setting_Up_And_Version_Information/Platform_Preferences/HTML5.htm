HTML5 환경 설정

HTML5 기본 설정에는 다음과 같은 옵션이 있습니다.

  • 기본 패키징 유형: 최종 HTML5 실행 패키지를 빌드할 때 폴더에 필요한 모든 파일을 생성하는 느슨한 파일로 패키지 또는 압축 파일을 생성하는 Zip으로 패키지 중에서 선택할 수 있습니다. ZIP 게임과 추가 파일이 들어 있는 파일입니다. 기본적으로 컴파일할 때 이 두 가지 옵션 중 어느 것을 선택할지 선택할 수 있지만 항상 둘 중 하나를 사용하도록 이 기본 설정을 지정할 수 있습니다. 이 옵션이 Show Dialogue 로 설정되어 있을 때 컴파일 시 팝업되는 실제 대화 창에서 "Remember Packaging Option"을 선택하면 이 기본 설정이 선택한 옵션으로 설정됩니다.
  • 기본 웹 서버 포트: HTML5 웹 서버의 기본 포트를 변경할 수 있습니다. 기본값은 51264입니다.
  • 기본 웹 서버 주소: HTML5 웹 서버의 기본 웹 서버 주소를 변경할 수 있습니다. 기본값은 127.0.0.1입니다.
  • 기본 웹 서버 허용 IP: 여기에서 웹 서버에 대해 허용된 모든 IP를 추가할 수 있으며 대상 관리자를 통해 추가된 모든 장치에 대해 기본값으로 설정됩니다.

중요 특정 대상 장치(예: 브라우저)를 사용할 때 이러한 기본 설정 대신 해당 특정 장치에 대해 설정된 IP 및 포트 값이 사용됩니다. 장치 관리자 를 확인하여 세부 정보와 함께 장치 목록을 확인하십시오.
