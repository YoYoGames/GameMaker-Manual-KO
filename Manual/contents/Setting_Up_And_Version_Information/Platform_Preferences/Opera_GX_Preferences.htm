오페라 GX 환경 설정

Opera GX 기본 설정에는 다음과 같은 옵션이 있습니다.

  • Emscripten SDK 위치: Opera GX용으로 만들어진 YYC 빌드에 필요한 Emscripten SDK의 위치입니다. SDK 설정에 대한 정보는 Opera GX 설정을 참조하십시오.
  • 기본 웹 서버 포트: Opera GX 웹 서버의 기본 포트를 변경할 수 있습니다. 기본값은 51264입니다.
  • 기본 웹 서버 주소: 여기에서 Opera GX 웹 서버의 기본 웹 서버 주소를 변경할 수 있습니다. 기본값은 127.0.0.1입니다.
  • 기본 웹 서버 허용 IP: 여기에서 웹 서버에 대해 허용된 모든 IP를 추가할 수 있으며 대상 관리자를 통해 추가된 모든 장치에 대해 기본값으로 설정됩니다.
