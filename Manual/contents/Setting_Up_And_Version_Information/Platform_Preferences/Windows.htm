Windows 기본 설정

Windows 기본 설정에는 다음과 같은 옵션이 있습니다.

  • Visual Studio 위치: 이 옵션은 GameMaker 가 빌드 PC의 Microsoft Visual Studio 설치를 가리키도록 하기 위한 것입니다.
  • 기본 패키징 유형: 최종 Windows 실행 패키지를 빌드할 때 설치 프로그램( NSIS 사용) 또는 압축된 패키지 생성 중에서 선택할 수 있습니다. ZIP 게임과 그 안에 있는 추가 파일이 있는 파일. 기본적으로 컴파일할 때 이 두 가지 옵션 중에서 선택하는 대화 상자가 표시되지만 항상 둘 중 하나를 사용하도록 이 기본 설정을 지정할 수 있습니다. 이 옵션이 Show Dialog 로 설정된 경우 컴파일 시 팝업되는 실제 대화 창에서 "Remember Packaging Option"을 활성화하면 이 기본 설정이 선택한 옵션으로 설정됩니다.
