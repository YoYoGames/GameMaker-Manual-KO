Windows UWP 기본 설정

Windows UWP 기본 설정에는 다음과 같은 옵션이 있습니다.

  • Visual Studio 위치: 이 옵션은 GameMaker 가 빌드 PC의 Microsoft Visual Studio 설치를 가리키도록 하기 위한 것입니다.
  • Win UWP 도구 위치: 이 옵션은 GameMaker 가 빌드 PC의 UWP 도구 설치 경로를 가리키도록 하기 위한 것입니다.
  • 기본 패키지 아키텍처: 기본적으로 UWP 실행 파일을 컴파일할 때 x86, x64 및 ARM 아키텍처 중에서 선택할 수 있는 대화 상자 창이 표시됩니다. 이 옵션을 세 가지 중 하나로 설정할 수 있으며 실행 파일을 컴파일할 때마다 기본적으로 선택됩니다.
  • 스토어 패키지 생성에 대한 자동 응답: 컴파일 시 표시되는 대화창에서 스토어 패키지 생성 여부를 선택할 수 있습니다. 이 옵션을 Yes 또는 No 로 설정할 수 있으며 실행 파일을 컴파일하도록 선택할 때마다 기본적으로 사용됩니다.
