안드로이드 환경설정

Android 기본 설정은 아래 나열된 다양한 섹션으로 나뉩니다.

기본 설정 상단에서 JDK 메모리 힙 의 최대 크기를 설정할 수 있습니다(기본값은 2GB). 다음으로 GameMaker 가 최종 실행 패키지를 빌드하고 게임을 테스트하는 데 필요한 다양한 도구를 찾을 위치를 알 수 있도록 다양한 경로 를 설정해야 합니다. 이것은 Android SDK, Android NDK 및 Java JDK 에 대한 세 섹션으로 나뉩니다. 이러한 경로는 자동으로 채워져야 하지만 문제가 있는 경우 오른쪽에 있는 버튼을 클릭하여 파일 탐색기를 열고 각 요소의 위치를 수동으로 선택할 수 있습니다. 여기 에서 각 SDK 요소에 필요한 버전에 대한 세부 정보를 찾을 수 있습니다. 경로를 적용할 때 경로의 유효성을 검사해야 하며 메시지에 " 찾음 "이 표시되거나 오류가 있음을 알려줍니다(이 경우 모든 정보를 수정하여 올바른지 확인해야 함).

그 아래에는 다음과 같은 다양한 포장 옵션이 있습니다.

  • 항상 APK 전체 설치 수행: 이 플래그를 지정하면 GameMaker 가 기기에서 게임의 이전 실행자는 물론 모든 관련 파일(예: *.ini 파일) 새 버전의 게임을 설치하기 전에 테스트 장치에서 다운로드하십시오.
  • 패키지에 설치: 이 옵션에 플래그가 지정되면 최종 실행 파일을 컴파일합니다. *.apk 패키지는 현재 선택한 대상 장치에도 자동으로 설치합니다.

마침내 우리는 KeyStore 에 왔습니다. 이것은 모든 Android 앱을 "서명"하는 데 사용되는 파일이므로 모든 세부 정보를 올바르게 입력하고 이 파일은 향후 모든 Android 앱을 만들고 업데이트하는 데 필요합니다. 즉, 설정 후 사용된 설정의 스크린샷을 찍고 최종 파일을 백업하는 것이 좋습니다. 이 파일을 분실하면 스토어에 업로드된 기존 게임을 업데이트할 수 없습니다.

새 키 저장소 파일을 만들려면 다음 세부 정보가 필요합니다.

  • 파일 이름: 키 저장소 파일의 이름입니다.

참고: 이것은 귀하의 이름이나 회사 이름이 아니라 생성할 파일 이름입니다! 이전 프로젝트의 키 저장소 파일이 이미 있는 경우 여기에서 파일 브라우저 버튼을 클릭하고 GameMaker 를 이전에 생성된 키 저장소 파일로 안내할 수 있습니다(또는 가져오기 버튼 클릭). 이 경우에도 필요한 나머지 정보를 입력해야 하지만 (파일을 처음 생성했을 때와 동일하게 ) 키 저장소 생성 버튼을 클릭하면 안 됩니다. 그러면 새 키 저장소 파일이 생성되어 이전 파일을 덮어씁니다.

  • 일반 이름: 일반적으로 귀하의 이름입니다.
  • 비밀번호: 최소 6자 이상이어야 하는 키 저장소 파일의 보안 비밀번호입니다.
  • Alias: 키스토어 파일에 사용할 수 있는 또 다른 이름으로 위에서 입력한 이름과 달라야 합니다.
  • 별칭 암호: 키 저장소 파일의 별칭 보안 암호는 6자 이상이어야 합니다.

    참고 Android Studio 버전이 4.2 이상인 경우 위의 "비밀번호" 및 "별칭 비밀번호" 필드에 동일한 비밀번호를 사용해야 합니다. 다른 암호와 별칭 암호를 사용하면 빌드가 실패합니다.
  • 조직 단위: 귀하가 속한 회사 내의 부서입니다.
  • 조직: 회사의 이름입니다.
  • 위치: 귀하가 기반을 둔 마을 또는 도시의 이름.
  • 국가: 귀하가 거주하는 국가의 표준 2자리 코드 입니다.

이 정보를 입력하고 만족하면 GameMaker 용 키 저장소 생성 이라고 표시된 버튼을 눌러 필요한 키 저장소 파일을 생성하십시오. 시간이 좀 걸릴 수 있지만 완료되면 일반적으로 다시 변경할 필요가 없습니다. 가져오기 버튼을 클릭하여 이전에 생성된 키 저장소 파일을 가져올 수 있고(위의 파일 이름 섹션 참조) 키 해시 생성 버튼을 클릭하여 키 저장소 SHA1 해시를 검색할 수도 있습니다. 이를 누르면 일부 웹 사이트에서 올바른 기능을 위해 필요한 키 저장소 파일에 직접 연결된 두 개의 특수 보안 해시 키가 생성됩니다(예: Sha1 해시의 경우 Facebook 또는 Google Play). 필요한 경우 버튼을 누른 다음 생성된 해시 키를 복사하여 붙여넣기만 하면 됩니다.
