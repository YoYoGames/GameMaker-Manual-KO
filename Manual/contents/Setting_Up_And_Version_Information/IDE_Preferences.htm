IDE 기본 설정

환경 설정 창 은 IDE를 사용하여 환경을 설정 및 사용자 정의할 수 있을 뿐만 아니라 특정 플랫폼 SDK 등을 설정할 수 있는 GameMaker 의 필수 부분입니다. 이 섹션에서는 환경 설정의 IDE 특정 영역을 다룹니다. 창 - 개별 대상 플랫폼의 경우 대상 플랫폼 기본 설정 섹션을 참조하십시오. 이 창에서 기본 설정을 변경한 후 적용하려면 적용 버튼을 클릭해야 합니다.

GameMaker 를 자신의 취향에 맞게 사용자 정의하기 위해 여기에서 옵션을 변경한 후 설정한 기본 설정을 백업하려면 Windows에서 이 폴더로 이동해야 합니다.

%AppData%/GameMaker/<your_account_name>/

또는 Mac에서 이 폴더:

~/.config/GameMaker

그런 다음 다음 두 파일을 안전한 장소에 복사합니다.

  • local_settings.json
  • cloud_settings.json

그러나 변경한 기본 설정을 재설정하려면 창 상단의 기본값 복원 버튼을 클릭해야 합니다. 이렇게 하면 현재 편집 중인 섹션의 기본값만 복원됩니다.

IDE 왼쪽 상단의 파일 메뉴에서 GameMaker 환경 설정을 열 수 있습니다.
