IDE 입력 및 탐색

GameMaker IDE 는 마우스와 키보드 입력을 허용하며, 둘 중 하나를 사용하여 많은 작업을 수행할 수 있습니다. 일반적으로 마우스 왼쪽 버튼을 클릭할 수 있습니다. 무엇이든 선택하려면 사용 / + 여러 항목을 선택하려면 항목을 다른 도크나 작업 공간으로 끌어다 놓거나 상황별 메뉴를 엽니다. macOS 시스템에서 GameMaker 를 실행 중이고 단일 버튼 마우스를 사용하는 경우 다음을 사용하십시오. + 마우스 오른쪽 버튼을 클릭하려면.

또한 다양한 작업 공간 요소 및 리소스 편집기를 탐색하는 데 사용할 수 있는 수많은 키보드 단축키가 있으며 가장 중요한 몇 가지를 빠르게 살펴보겠습니다( 여기 에서 전체 목록을 찾을 수 있음).

  • / + "Z": 이전 작업을 취소하고 대부분의 편집기에서 작동합니다. 여러 수준의 실행 취소도 있으므로 이를 여러 번 눌러 변경 사항을 "롤백"할 수 있습니다.
  • / + "Y": 대부분의 편집기에서 실행 취소되고 작동하는 이전 작업을 다시 실행합니다. 실행 취소와 마찬가지로 여러 수준의 다시 실행도 있습니다.
  • / + "S": GameMaker 가 프로젝트를 저장하도록 합니다.
  • / + "T": 이렇게 하면 게임의 자산을 빠르게 검색하고 이동할 수 있을 뿐만 아니라 기본 설정 및 게임 옵션 을 검색할 수 있는 이동 창이 열립니다. 모든 항목의 목록이 표시되고 검색어를 입력하면 이 목록이 적절하게 선택됩니다. 따라서 모든 방에 "rm_" 접두사가 붙는 경우 이를 입력하면 해당 자산만 표시하도록 목록이 선택됩니다. 그 이름으로 시작합니다.

  • / +: 이렇게 하면 다른 작업 공간에 열려 있는 항목 사이를 빠르게 탐색하는 데 사용할 수 있는 작업 공간 개요 창이 열립니다. 하는 동안 창은 열린 상태로 유지됩니다. 또는 를 누르고 한 번은 창의 다음 항목으로 이동합니다. 그런 다음 바로 가기를 놓으면 선택한 항목으로 이동합니다.

    참고 이 창에 대한 바인딩을 다시 매핑하여 둘 이상의 수정자(예: + + B ) 또는 단일 키(예: B 만), 그런 다음 창을 수동으로 닫거나 창에서 항목을 선택할 때까지 단축키를 누르고 있는지 여부에 관계없이 창은 계속 열려 있습니다.

  • : 매뉴얼을 엽니다. GML Visual 또는 GML 코드를 사용할 때 작업이나 기능(또는 키워드 등)을 선택한 다음 키를 누를 수도 있습니다. 또는 마우스 가운데 버튼을 클릭하십시오 관련 페이지에서 설명서를 엽니다.
  • : IDE에서 도킹된 모든 창을 축소하거나 확장하는 데 사용할 수 있습니다.

작업 공간을 탐색하는 다른 방법에는 키보드 단축키 사용이 포함됩니다. / + + <Arrow Keys> 열려 있는 창 사이를 누른 방향으로 이동하려면 마우스 가운데 버튼을 길게 누릅니다. 그런 다음 마우스를 끌어 작업 공간 주위를 이동할 수도 있습니다.

다양한 작업 공간과 창을 탐색할 때 마우스 오른쪽 버튼을 사용할 수 있습니다. 텍스트 필드에서 기본적으로 다음 필드가 있는 상황에 맞는 메뉴를 엽니다.

어떤 상황에서는 포커스가 있는 편집기나 창에 따라 이러한 옵션이 확장될 수 있습니다.

이러한 입력 방법 외에도 펜 장치에 대한 지원이 제한적이며 랩톱을 사용하여 프로젝트 작업을 하는 사람들을 위한 특수 모드도 있습니다. 이 두 가지 모두 아래에 설명되어 있습니다.

노트북 모드

랩탑에서 GameMaker 를 사용하는 경우 랩탑 모드용 IDE 상단에 추가 옵션이 있습니다.

기본적으로 켜져 있지만 원하는 경우 이 버튼을 토글하여 끌 수 있습니다. 랩톱 모드는 IDE의 일부 도구와 결합하여 터치패드를 사용할 때 훨씬 더 나은 경험을 제공하여 3가지 주요 마우스 상호 작용( 이동, 확대/축소 및 스크롤 )을 단순화합니다. 이를 위해 두 개의 수정자 키를 사용합니다. 왼쪽 / 그리고 왼쪽. 노트북 모드가 켜져 있을 때 왼쪽 / 그리고 왼쪽 룸 편집기 패닝 과 같은 마우스 상호 작용을 위해 "예약"되어 있습니다. / 줌, 왼쪽 스크롤 및 팬입니다. 랩톱 모드에서 스크롤하는 것은 패닝처럼 작동하므로 스크롤 막대를 사용할 필요가 없을 정도로 직관적이고 정확해야 합니다. 이 버튼을 누른 상태에서 마우스를 움직이기만 하면 됩니다. 클릭이나 드래그가 아니라 이동만 하면 됩니다. 일반적으로 사용이 필요한 일반 단축키가 있는 경우 / 또는 키를 사용하는 경우 대신 키보드의 오른쪽 키를 사용합니다.



터치 스크린 지원

여기서 마지막으로 언급할 사항은 GameMaker 용 IDE도 터치 스크린을 최소한으로 지원한다는 것입니다. 모든 운영 체제에서 터치 스크린을 사용하여 기본 작업 공간에서 항목을 클릭하고 끌 수 있으며 두 번째 탭에서 오른쪽 클릭을 수행하는 2개의 동시 포인터를 지원합니다. Windows 8 이상에서 GameMaker IDE는 펜 장치도 지원합니다.
