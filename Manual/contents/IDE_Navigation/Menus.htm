메뉴

GameMaker IDE의 상단은 메뉴 막대 로 채워져 있습니다. 여기에서 IDE 탐색, 작업 수행 또는 게임 제작에 도움이 되는 도구가 있는 추가 창을 여는 데 사용할 수 있는 여러 드롭다운 메뉴를 찾을 수 있습니다. 여기에 사용 가능한 기본 메뉴가 나열되어 있지만 일부 자산 편집기는 컨텍스트에 따라 추가 메뉴를 추가합니다. 이러한 추가 메뉴는 해당 메뉴를 만든 편집기와 관련된 페이지에 설명되어 있습니다.

사용 가능한 기본 메뉴는 다음과 같습니다.

  • 파일 메뉴
  • 편집 메뉴
  • 빌드 메뉴
  • 윈도우 메뉴
  • 도구 메뉴
  • 마켓플레이스 메뉴
  • 레이아웃 메뉴
  • 도움말 메뉴
