파일 메뉴

파일 메뉴에서 프로젝트를 변경, 저장 또는 가져오고 GameMaker IDE 기본 설정을 지정할 수 있습니다. 사용 가능한 옵션은 다음과 같습니다.

  • 새 프로젝트 - GameMaker 에서 처음부터 새 프로젝트를 생성하기 위한 것입니다. "저장" 대화 상자가 열리고 새 프로젝트를 저장할 위치를 선택하라는 메시지가 표시됩니다. 프로젝트에 이름과 저장할 위치를 지정하면 새 작업 공간으로 이동하여 작업 을 시작할 수 있습니다. 프로젝트 파일은 *.yyp 파일 확장자.
  • 프로젝트 열기 - 이전에 만든 파일을 선택할 수 있는 파일 탐색기가 열립니다. *.yyz 로드할 프로젝트 파일입니다. 가져올 때 프로젝트가 2.3.0보다 낮은 버전을 사용하여 생성된 경우 다음 경고 메시지가 표시됩니다. 계속 진행하기로 선택하면 파일 탐색기가 열리고 프로젝트에 업데이트된 이름과 위치를 지정하여 저장할 수 있으므로 원래 프로젝트를 덮어쓰지 않습니다. 프로젝트가 새로운 2.3.0+ 형식으로 변환되면 IDE에서 열리고 변환이 성공했다는 메시지가 표시됩니다. 변환 프로세스에서 생성된 호환성 보고서를 계속 확인하고 프로젝트를 실행하거나 컴파일하기 전에 수정해야 할 사항이 있는지 확인해야 합니다. 2.3.0 이전 버전으로 만든 프로젝트를 열려고 시도하는 동안 반복되는 오류가 발생하면 YoYo Games에 문의해야 합니다( 도움말 메뉴 에서 "버그 보고" 옵션 사용). *.yyp 변환하려는 파일.
  • 프로젝트 가져오기 - 이전에 파일로 내보낸 프로젝트를 가져올 수 있습니다. *.yyz 아카이브(아래 "프로젝트 내보내기" 참조) 또는 - 레거시 GameMaker: Studio 1.4로 만든 프로젝트가 있는 경우 - 이 옵션을 선택하여 *.gmx 아니면 그 *.gmz 해당 제품 라인의 파일입니다. 프로젝트를 열 때와 마찬가지로 *.yyz 2.3.0 업데이트 이전의 GameMaker 버전으로 생성된 경우 해당 업데이트로 프로젝트 파일 형식이 변경되므로 변환해야 합니다.

참고: 이전 GameMaker 프로젝트를 GameMaker 로 가져오려면 1.4 버전을 사용하여 생성해야 하며 가져오기 시 직접 작동하지 않을 수 있습니다. GameMaker 와 GameMaker: Studio 1.4 사이에는 많은 것이 다르므로 이전 프로젝트가 작동하려면 약간의 조정이 필요할 수 있습니다. GameMaker 기술 자료 의 GM:S 1.4 게임을 GameMaker 로 이식 기사와 사용되지 않는 기능 및 호환성 스크립트 를 다루는 매뉴얼 섹션을 참조하십시오.

중요한! 가져오기 *.yyz 글꼴 리소스가 포함된 프로젝트에서는 글꼴을 제거/다시 추가하거나 글꼴 편집기에서 글꼴을 변경한 다음 다시 설정하여 글꼴을 "재생성"해야 할 수 있습니다. 기존 글꼴 정보가 *.yyz 내보낼 때, 그리고 나중에 가져올 때나 다른 컴퓨터에서 프로젝트에 필요한 글꼴이 이미 설치되어 있지 않을 수 있습니다(또는 같은 이름의 다른 글꼴이 있을 수 있음). 가져온 후 글꼴이 렌더링되는 방식에 문제가 있는 경우에만 이 작업을 수행해야 합니다.

  • 최근 프로젝트 - 이 옵션을 사용하여 최근에 사용한 프로젝트를 볼 수 있습니다. 그것을 선택하면 시작 페이지 가 다시 열리고(닫았어야 함) 거기에 있는 목록에서 최근에 사용한 프로젝트를 선택할 수 있습니다.
  • 프로젝트 저장 - 프로젝트를 생성할 때 정의한 것과 동일한 위치(또는 프로젝트를 연 위치)에 게임을 저장합니다.
  • 다른 이름으로 프로젝트 저장 - 위와 동일하게 프로젝트의 위치와 새 이름만 지정합니다.
  • 프로젝트 내보내기 - 프로젝트를 파일이나 템플릿으로 내보낼 수 있습니다.
    • YYZ - 프로젝트를 단일 파일로 내보냅니다. *.yyz 파일을 저장하고 사람들에게 사본을 보내거나 백업을 저장하는 데 매우 유용합니다.
    • 템플릿으로 - 프로젝트를 템플릿으로 내보내 나중에 시작 페이지 에서 다른 프로젝트의 기반으로 사용할 수 있습니다. 이 옵션을 선택하면 다음 프롬프트가 표시됩니다.
      먼저 "템플릿 정보 편집"을 클릭하여 템플릿 정보 가 올바른지 확인한 다음 템플릿으로 저장하려면 이 프롬프트를 두 번째로 열 때 "내보내기"를 선택해야 합니다. 그러면 시작 화면의 템플릿 메뉴에서 볼 수 있습니다.
  • 기본 설정 - 이것을 선택하면 GameMaker의 모양과 느낌을 설정할 수 있는 기본 설정 창 으로 이동합니다.
  • 새 IDE - 이 옵션을 선택하면 IDE의 새 인스턴스가 열려 사용할 수 있습니다(시작 페이지에서 열립니다).
  • 내 계정 - 이 옵션은 기본 웹 브라우저를 열고 YoYo 계정 페이지로 이동합니다.
  • 로그아웃 - 이것을 선택하면 GameMaker 에서 로그아웃됩니다. 나중에 제품을 사용하려면 동일한 라이선스(또는 다른 라이선스)로 다시 로그인해야 합니다.

중요한! GameMaker 용 무료 라이선스를 사용한 다음 다른 라이선스 중 하나를 구입한 경우 IDE에서 로그아웃한 다음 GameMaker 를 다시 시작하고 다시 로그인 하여 새 라이선스가 올바르게 적용되었는지 확인해야 할 수 있습니다. 자동으로 발생해야 함).

  • 종료 - GameMaker 를 바탕 화면으로 종료합니다.
