도구 메뉴

이 메뉴를 사용하면 GameMaker 내에서 사용할 수 있는 다양한 독립 실행형 도구를 다음 옵션과 함께 열 수 있습니다.

  • 텍스처 그룹 - 텍스처 그룹 창을 봅니다.
  • 오디오 그룹 - 오디오 그룹 창을 봅니다.
  • 사운드 믹서 - 사운드 믹서 창을 봅니다.
  • 프로젝트 이미지 생성 - 프로젝트 이미지 생성 창을 엽니다.
  • 로컬 패키지 만들기 - 나중에 사용할 수 있도록 로컬 자산 패키지를 만들고 저장할 수 있는 자산 패키지 도구 창을 엽니다.
  • 로컬 패키지 가져오기 - 파일 탐색기 창을 열어 로컬 자산 패키지 를 선택하고 현재 프로젝트에 추가하거나 새 프로젝트를 생성하여 가져올 수 있습니다.
