레이아웃 메뉴

이 메뉴는 다양한 IDE 레이아웃을 저장하고 불러오는 것과 관련이 있습니다. 자산 브라우저 또는 출력 창과 같은 특정 요소를 기본값과 다르게 배열하여 원하는 대로 IDE를 설정할 수 있습니다. 이를 통해 워크플로에 맞게 GameMaker 를 설정할 수 있으며 이 메뉴의 옵션을 사용하여 현재 프로젝트 레이아웃을 다음과 같이 저장할 수 있습니다. *.xml 그런 다음 레이아웃을 동일하게 설정하기 위해 모든 프로젝트에 로드할 수 있는 파일입니다. 이 메뉴의 옵션은 다음과 같습니다.

  • 레이아웃 로드 - 이전에 저장한 레이아웃 *.xml 파일을 로드할 수 있는 파일 탐색기가 열립니다.
  • 레이아웃 저장 - 현재 IDE 레이아웃의 이름을 지정하고 *.xml 파일로 저장할 수 있는 파일 탐색기가 열립니다.
  • 레이아웃 재설정 - 현재 IDE 레이아웃을 기본 설정으로 재설정합니다.
