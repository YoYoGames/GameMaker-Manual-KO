편집 메뉴

편집 메뉴는 GameMaker 인터페이스의 일반 편집 도구를 사용할 수 있는 곳입니다. 사용 가능한 옵션은 다음과 같습니다.

  • 실행 취소 - 작업을 실행 취소합니다.
  • 다시 실행 - 이전 작업을 다시 실행합니다.
  • 검색 및 바꾸기 - 주어진 입력에 대한 전체 코드(방 생성 코드, 스크립트 및 이벤트 포함)를 검색할 수 있는 검색 창이 열립니다. 자세한 내용은 아래를 참조하십시오.

편집 메뉴에서 검색 및 바꾸기를 선택하면 다음 창이 열립니다.

이것을 사용하여 방 코드, 이벤트 코드, 스크립트 및 셰이더를 포함하여 게임의 모든 스크립팅된 자산을 검색할 수 있습니다. 검색 시 상단 상자에 새 검색어를 입력하거나 오른쪽에 있는 화살표 아이콘을 클릭하여 이전에 검색한 용어 목록을 열 수 있습니다. 또한 초기 검색 용어를 대체하는 데 사용할 대체 용어를 제공할 수 있는 옵션이 있으며 이 옵션도 오른쪽에 있는 화살표 버튼에서 이전에 사용한 대체 문자열을 사용하는 옵션이 있습니다.

검색은 일반적이거나 검색어 입력 아래의 옵션을 사용하여 세분화할 수 있습니다.

  • 검색을 대소문자 를 구분하도록 선택할 수 있습니다. 이 경우 검색 문자열의 대소문자와 정확히 일치하는 단어만 포함됩니다. 예: 이 옵션 없이 "My_Var"를 검색하면 "my_var", "mY_VAR", 또는 "My_Var"이지만 이 옵션을 선택하면 "My_Var"만 검색 결과에 포함됩니다.
  • 전체 단어만 포함하는 검색 결과만 포함하도록 선택할 수 있습니다. 이 경우 검색어를 포함하지만 정확히 일치하지 않는 모든 결과는 무시됩니다. 즉, 이 항목이 선택되지 않은 "게임" 검색이 표시됩니다 "game_end()" 함수 또는 "game_state"와 같은 변수와 같은 것을 확인하고 검색을 수행하는 동안 "게임" 자체인 결과만 표시됩니다.
  • 검색에서 주석을 무시 하도록 선택할 수 있습니다. 이 옵션을 선택하면 검색어가 포함된 코드의 모든 주석이 무시되고 출력 결과에 표시되지 않습니다.
  • 자산 브라우저에서 다양한 자산 유형을 기반으로 검색 결과에 필터 를 적용할 수 있습니다. 기본적으로 이것은 "Everything"으로 설정되어 있지만, 이것을 클릭하면 검색에 포함할 자산을 선택하거나 선택 취소할 수 있는 다음 메뉴가 열립니다:

검색(및 선택적으로 바꾸기) 용어를 지정하고 검색 옵션을 설정하면 하단에 있는 다른 버튼을 사용하여 검색 수행 방법 및 수행할 작업을 선택할 수 있습니다. 이러한 버튼은 다음과 같습니다.

  • 모두 찾기: 모든 검색어를 찾아 출력 창에 표시합니다.
  • 이전 찾기: 검색어의 이전 인스턴스를 찾습니다. 이렇게 하면 작업 공간에서 참조가 포함된 자산이 열리고(아직 열려 있지 않은 경우) 해당 자산으로 이동합니다.
  • 다음 찾기: 검색어의 다음 인스턴스를 찾습니다. 이렇게 하면 작업 공간에서 참조가 포함된 자산이 열리고(아직 열려 있지 않은 경우) 해당 자산으로 이동합니다.
  • 모두 바꾸기: 옵션 세트로 주어진 용어를 검색하고 자산에서 지정한 새 바꾸기 용어로 바꿉니다.
  • 바꾸기: 선택된 바꾸기 용어로 강조 표시된 현재 편집기에서 현재 선택된 용어를 바꿉니다.
  • 검색 중지: 현재 검색이 진행 중인 경우 중지합니다.

이 창이 열려 있는 동안 Esc 키를 누르면 선택한 필드가 지워지지만 해당 필드가 이미 비어 있으면 창이 닫힙니다.

검색 및 바꾸기 창을 사용한 모든 결과는 IDE 하단의 검색 출력 창 에 표시됩니다. 각 결과는 두 번 클릭하여 현재 작업 공간에서 용어가 있는 자산을 열고 편집할 준비가 된 자산으로 이동합니다.
