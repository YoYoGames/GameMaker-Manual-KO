책갈피

작업 공간을 탐색하는 또 다른 방법은 책갈피 를 사용하는 것입니다. 스크립트 또는 개체 이벤트에서 IDE 창 또는 코드/GML Visual 섹션에 최대 9개의 "책갈피"를 추가할 수 있으며 올바른 키 조합을 사용하여 이들 사이를 빠르게 건너뛸 수 있습니다.

책갈피를 추가하려면 태그를 지정하려는 창이나 코드 줄을 선택한 다음 + + NumberKey[1 - 9] Windows 또는 + +NumberKey[1 - 9] Mac에서는 다음과 같이 IDE에서 항목에 레이블을 지정하여 책갈피를 설정합니다.

위의 이미지에서 IDE의 다른 부분에 1, 2, 4로 번호가 매겨진 책갈피를 할당했으며 책갈피를 통해 다시 바로 돌아갈 수 있으므로 이제 이 창에서 벗어나 탐색하거나 아예 닫을 수도 있습니다. 나중에 언제든지. 이것은 를 눌러 수행됩니다. / + NumberKey[1 - 9], 북마크된 항목이 닫혀 있으면 해당 항목이 열리고, 그렇지 않으면 작업 공간이 선택한 북마크로 이동합니다.

중요 책갈피는 프로젝트와 함께 또는 IDE를 닫을 때 저장되지 않습니다. 생성된 세션에만 유효합니다.

책갈피를 제거하려면 책갈피가 지정된 창이나 코드 줄을 선택한 다음 반복하기만 하면 됩니다. + / + NumberKey[1 - 9] 당신이 그것을 만드는 데 사용합니다.
