최근 창

최근 창 목록은 Windows 메뉴에서 열 수 있으며 이전에 연 창의 짧은 목록(기본적으로 최대 10개 항목이 있음)을 표시합니다. 이 목록으로 시작하려면 비어 있지만 IDE에서 다른 창(예: 스프라이트, 스크립트 또는 개체 등)을 열면 목록이 채워집니다. 이전에 열린 창으로 돌아가야 하는 경우에는 마우스 왼쪽 버튼을 클릭하기만 하면 됩니다. 목록에 있는 항목을 클릭하면 작업 공간이 창으로 이동합니다. 마우스 오른쪽 버튼을 클릭하면 항목을 선택하면 다음 메뉴가 열립니다. 여기에서 항목을 삭제하도록 선택할 수 있습니다. + 또는 / + 여러 개 선택) 또는 목록의 맨 위로 이동할 수 있습니다. RMB 메뉴에는 목록을 완전히 지울 수 있는 추가 옵션도 있습니다. 최근 Windows 목록을 IDE 측면에 고정할 수 있으며 특정 기본 설정 을 조정하여 동작 방식을 변경할 수도 있습니다.
