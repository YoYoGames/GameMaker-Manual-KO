인스턴스 타임라인 설정

이 작업은 자산 브라우저에서 인스턴스로 타임라인을 할당합니다. 할당할 타임라인을 제공한 다음 마지막 순간에 도달했을 때 타임라인 루프를 갖도록 선택할 수 있습니다(마지막 순간이 실행된 후 다음 프레임은 타임라인의 첫 번째 순간으로 다시 루프백됨). 타임라인을 즉시 시작하거나 시작하지 마십시오. 타임라인을 즉시 시작하지 않으면 타임라인 상태 설정 작업을 사용하여 나중에 다른 이벤트나 스크립트에서 시작할 수 있습니다.

예시:

위의 작업 블록 코드는 작업을 호출하는 인스턴스에 타임라인을 할당하고 초기 순간을 10으로 설정한 다음 타임라인 속도를 2로 설정합니다.
