타임라인 상태 설정

이 작업을 사용하여 타임라인을 시작, 중지, 일시 중지 또는 재개할 수 있습니다. 먼저 인스턴스 타임라인 설정 작업을 사용하여 인스턴스에 대해 타임라인이 설정되어 있어야 하며, 그런 다음 언제든지 이 작업을 사용하여 다음 옵션에서 실행 상태를 변경할 수 있습니다.

옵션 설명
시작 인스턴스에 할당된 타임라인을 시작합니다(타임라인이 일시 중지된 경우 시작 위치가 다시 순간 0으로 설정됨).
이력서 일시 중지된 타임라인을 다시 시작합니다(타임라인이 중지되고 일시 중지되지 않은 경우 아무 작업도 수행하지 않음).
일시중지됨 인스턴스에 할당된 타임라인을 일시 중지합니다(일시 중지된 동안 타임라인의 순간은 게임 프레임과 함께 진행되지 않음).
중지 인스턴스에 할당된 타임라인을 중지합니다.

State 타임라인을 넣을 상태(위 표 참조)

위의 작업 블록 코드는 전역 변수에 저장된 값을 확인하고 true 타임라인이 일시 중지되고 false 그러면 타임라인이 다시 시작됩니다.
