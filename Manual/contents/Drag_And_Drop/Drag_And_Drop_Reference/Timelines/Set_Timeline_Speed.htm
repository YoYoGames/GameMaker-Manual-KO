타임라인 속도 설정

이 작업은 타임라인이 포함된 순간을 이동하는 속도를 설정하는 데 사용할 수 있습니다. 일반적으로 각 게임 프레임에서 타임라인의 위치가 1만큼 증가하기를 원하지만 이 작업을 다른 값으로 설정하여 이 양을 변경할 수 있습니다. 선택한 값은 승수 로 처리되므로 예를 들어 값이 2이면 타임라인이 두 배 속도로 실행되고 값이 0.5이면 절반의 속도로 실행됩니다. 이것이 순간에 의미하는 바는 값이 1보다 크면 동일한 게임 프레임 내에서 여러 순간이 발생할 수 있고(어떤 순간도 건너뛰지 않음) 속도가 1보다 작으면 순간이 실행된 후, 다음 순간이 실행될 때까지 아무 일도 일어나지 않는 프레임이 있을 것입니다. 예를 들어 타임라인 속도를 0.1로 설정하면 한 게임 프레임에서 순간을 얻은 다음 모멘트가 없는 9개의 게임 프레임을 얻고 10번째 게임 프레임에서 다음 순간이 실행됩니다.

인스턴스 타임라인 설정 또는 타임라인 상태 설정 작업을 사용하여 시작하도록 타임라인을 설정했지만 속도를 설정 하지 않은 경우 타임라인은 기본적으로 게임 프레임당 1분으로 설정됩니다.
