물체가 있는 경우

이 작업은 호출 인스턴스의 충돌 마스크를 기반으로 위치에 배치된 개체의 인스턴스가 있는지 확인하고 확인하는 데 사용됩니다. 인스턴스에 대한 절대 위치 또는 상대적 위치가 될 수 있는 위치를 지정하면 작업이 반환됩니다. true 충돌이 발생하거나 false 없는 경우. 또한 "Not" 플래그를 확인하여 지정된 위치에 충돌이 없는지 확인할 수 있으며 그러면 작업이 반환됩니다. true 충돌이 발견되지 않고 false 그렇지 않으면. 충돌은 해당 위치에 있는 인스턴스의 충돌 마스크와 겹치는 호출 인스턴스의 충돌 마스크를 기반으로 계산됩니다(위치에 "배치"되는 것처럼).

중요한! 충돌은 유효한 충돌 마스크가 있는 인스턴스에 대해서만 등록됩니다. 예: 인스턴스에 할당된 스프라이트가 있습니다. sprite_index, 또는 할당된 스프라이트 mask_index. 충돌의 인스턴스 중 하나 에 충돌 마스크가 없으면 해당 인스턴스가 그 당시에 무엇을 그리는지에 관계없이 충돌이 감지되지 않습니다.

"에 작업을 추가하려면if " 블록은 아래 이미지와 같이 작업 옆 에 놓아야 합니다.
