Random 라이브러리에는 난수 생성 및 확률 연산을 처리하기 위한 다음과 같은 작업이 있습니다 .

GameMaker 는 IDE에서 게임을 실행할 때마다 난수를 생성할 때 동일한 시드를 사용합니다. 이것은 디버깅을 용이하게 하기 위한 것이며 임의의 값을 생성할 때 동일한 초기 결과를 얻을 수 있지만 게임을 실행 파일로 컴파일한 경우에는 발생 하지 않음 을 의미합니다(게임이 실행될 때마다 새 시드가 생성됨). 테스트할 때 이 동작을 원하지 않는 경우 임의의 값을 생성하기 전에 게임 시작 시 Randomise 작업을 한 번 호출해야 합니다.
