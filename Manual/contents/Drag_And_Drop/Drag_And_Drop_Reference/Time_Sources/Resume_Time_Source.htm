재개 시간 소스

이 작업은 주어진 시간 소스 와 그 자식을 다시 시작합니다. Pause Time Source 를 사용하여 이전에 일시 중지되었어야 합니다.

동작 구문:

인수:

논쟁 설명
Time Source 일시 중지할 시간 소스 ID

예시:


이 코드 블록은 Key Press 이벤트에서 이상적으로 실행됩니다.

시간 소스 의 상태가 "활성"인지 확인하고, 이 경우 해당 시간 소스 를 일시 중지합니다.

그러나 상태가 "일시 중지됨"이면 시간 소스 를 다시 시작합니다.
