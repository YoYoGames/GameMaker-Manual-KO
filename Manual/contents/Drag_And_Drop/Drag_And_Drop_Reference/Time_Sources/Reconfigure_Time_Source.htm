시간 소스 재구성

이 작업을 통해 "시간 소스" 필드에 지정된 기존 시간 소스 에 대한 설정을 변경할 수 있습니다.

여기에 제공된 설정은 상위 항목을 제외하고 Create Time Source 와 동일합니다.

시간 소스 가 재설정되고 비활성화되므로 다시 시작해야 합니다.

논쟁 설명
Time Source 재구성할 시간 소스 ID

Units 기간을 나타내는 단위 (초 또는 프레임)
