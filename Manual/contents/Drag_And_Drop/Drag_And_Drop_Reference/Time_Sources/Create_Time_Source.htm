시간 소스 생성

이 작업은 새 시간 소스 를 만들고 이를 time_source_create()>Target 변수에 저장합니다. 개요를 보려면 시간 소스 를 읽으십시오.

이것은 기능. 인수에 대한 자세한 정보는 해당 페이지를 읽으십시오.

생성된 시간 원본을 활성화하려면 시작 시간 원본 을 사용해야 합니다.

더 이상 필요하지 않을 때 Destroy Time Source 를 사용하여 시간 소스를 파괴해야 합니다.

동작 구문:

인수:

논쟁 설명
Target 새 시간 소스 가 저장될 변수
Parent 상위 시간 소스: 기본 제공 시간 소스 또는 사용자 지정 시간 소스
Period 시간 소스 의 기간 길이, 만료되는 데 걸리는 시간
Units 기간을 나타내는 단위 (초 또는 프레임)
Callback 시간 소스 가 만료될 때 호출할 method 또는 스크립트 함수
Arguments OPTIONAL 콜백 함수에 전달할 인수를 포함하는 배열
Repetitions OPTIONAL 시간 소스 가 총 몇 번 실행되어야 하는지 또는 무한 반복의 경우 -1
Expiry Type OPTIONAL 시간 소스 의 만료 유형

예시:

이 코드 블록은 다음을 수행합니다.

  • "Temp" 옵션이 활성화된 새 기능을 생성합니다. 이것은 로컬 메소드 변수를 생성합니다.
    • 이 메서드는 하나의 인수 msg 를 취하여 출력 로그에 인쇄합니다.
  • 그런 다음 기간 길이가 1초인 새 시간 소스 를 만듭니다.
    • callback_method 변수를 "콜백"으로 지정하고 함수에 대한 하나의 인수가 있는 배열을 지정합니다.
    • 이 시간 소스 는 "반복"에 -1이 지정되어 있으므로 무기한 반복되도록 설정됩니다.
  • 그러면 시간 소스 가 시작됩니다.

이 시간 소스 는 "1초가 지났습니다!" 를 인쇄하여 1초마다 콜백을 실행합니다. 출력 로그에.
