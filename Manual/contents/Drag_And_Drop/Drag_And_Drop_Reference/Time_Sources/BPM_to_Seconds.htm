BPM에서 초

이 작업은 분당 비트 수 값을 사용하고 각 비트의 길이를 초 단위로 반환합니다.

이것은 시간 소스에 대한 BPM 값을 사용하기 위해 시간 소스를 생성 할 때 사용할 수 있습니다. 이러한 시간 소스 는 초 를 단위 로 사용하는 것이 중요합니다.
