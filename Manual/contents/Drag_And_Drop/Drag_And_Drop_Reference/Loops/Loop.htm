Loop 액션은 Break 액션이 호출될 때까지 무한 반복됩니다. 루프 내에 배치된 작업에 관계없이 Break가 항상 호출되도록 이 작업에 주의해야 합니다. 그렇지 않으면 게임에서 대상 OS에서 "응답하지 않음" 오류가 발생하고 작동이 중지됩니다. 이 작업을 작업 코드 작업 공간에 추가한 다음 "if"에 작업을 추가하는 것처럼 루프 내에서 수행할 작업을 추가합니다.
