이 작업은 루프 작업(예: Repeat 또는 While ) 내에서만 사용할 수 있으며 이 작업을 호출하면 해당 순간에 루프가 중단됩니다. 일반적으로 루프는 고정된 횟수로 실행되거나 특정 조건이 충족될 때까지 실행되지만 Break 작업을 사용하여 언제든지 루프를 조기에 종료할 수 있습니다(Break 후 루프 내의 작업은 실행되지 않음).
