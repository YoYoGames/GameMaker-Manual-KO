반복하다

이 작업은 추가된 모든 작업을 지정된 횟수만큼 반복합니다. "if"에 작업을 추가하는 것처럼 작업을 반복할 횟수를 지정한 다음 반복 루프에 작업을 추가합니다.:

측면에 추가된 모든 작업은 반복에 포함되며 이벤트 또는 스크립트가 아래에 추가된 작업으로 계속되기 전에 수행됩니다(아래 예 참조). Break 작업을 호출하면 루프를 조기에 종료할 수 있습니다.

동작 구문:

인수:

논쟁 설명
Times 반복 횟수(정수 값)

예시:

위의 액션 블록 코드는 목록의 항목 수를 가져와 임시 로컬 변수에 값을 저장합니다. 그런 다음 이 변수를 사용하여 반복 루프가 실행되는 횟수를 설정하고 반복 루프에서 목록의 항목을 다른 임시 변수로 읽은 다음 참조하는 인스턴스를 삭제하는 데 사용합니다. 루프가 끝나면 데이터 구조가 해제됩니다.
