을위한

For 액션은 값이 주어진 조건에 도달할 때만 종료되는 루프를 수행하는 데 사용됩니다. 초기 값을 변수로 제공하고(임시 로컬 변수로 플래그가 지정될 수 있음 - 일반적으로 원하는 것입니다), 변수를 검사할 조건을 제공합니다(예: 특정 값보다 작은 경우). 그런 다음 마지막으로 루프를 반복할 때마다 변수를 증가시킬 양입니다. 따라서 일반적인 흐름은 다음과 같습니다.

  • 변수를 값으로 초기화합니다.
  • 이 변수를 확인하는 조건을 설정합니다.
  • 루프를 반복할 때마다 변수를 얼마나 증가시켜야 하는지 설정합니다.
  • 그런 다음 루프에 작업을 추가하면 조건이 반환될 때까지 각 반복마다 실행됩니다. false 또는 중단 조치가 발생했습니다.

루프 내에서 수행할 작업을 추가할 때 "if"에 작업을 추가하는 것처럼 추가됩니다.

측면에 추가된 모든 작업은 For에 포함되며 이벤트 또는 스크립트가 아래에 추가된 작업으로 계속되기 전에 수행됩니다. Break 액션을 호출하면 루프를 조기에 종료할 수 있으며 For 루프를 사용할 때 사용하는 조건이 적절하다면 증가할 필요 가 없고 감소시킬 수도 있습니다. 예를 들어, 초기 변수를 10으로 설정하고 0보다 작은지 확인하도록 조건을 설정한 다음 증가 값을 -1로 설정하면 루프가 10번 반복되어 10에서 0까지 역으로 계산됩니다.
