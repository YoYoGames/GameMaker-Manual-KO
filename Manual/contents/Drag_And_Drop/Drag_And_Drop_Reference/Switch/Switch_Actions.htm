스위치 동작 라이브러리

Switches 작업 라이브러리에는 게임 작업 코드에서 switch 문을 만드는 데 필요한 작업이 포함되어 있습니다. 여러 상황에서 인스턴스가 특정 값에 따라 작업을 완료하도록 하고 여러 개의 연속 "if" 문을 사용하여 이 작업을 수행할 수 있지만 가능한 선택이 2개 또는 3개 이상이면 지저분해질 수 있습니다. 이러한 경우 "전환" 작업을 사용하는 것이 좋습니다. 스위치 작업은 다음과 같이 작동합니다.

  • 값을 제공합니다(일반적으로 일종의 표현식 또는 변수에서).
  • 값은 다양한 "case" 문에 할당된 값과 비교됩니다.
  • 값이 케이스 값과 같으면 케이스의 내용이 수행되고 그렇지 않으면 다음 케이스가 평가됩니다.
  • 케이스가 수행 되거나 모든 케이스가 평가되고 실패하면 코드는 스위치 끝에서 계속됩니다.
  • 모든 케이스가 실패하고 기본 케이스가 있는 경우 스위치 끝에서 코드가 계속되기 전에 수행됩니다.
