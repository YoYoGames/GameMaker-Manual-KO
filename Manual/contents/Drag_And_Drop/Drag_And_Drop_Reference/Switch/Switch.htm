스위치

이 작업은 새 Switch 문을 만든 다음 평가를 위해 추가 사례를 추가할 수 있습니다. 스위치의 케이스와 비교하는 데 사용할 값을 제공한 다음 "if"에 액션을 추가하는 것처럼 다른 케이스 액션을 추가합니다. 그 아래:

먼저 포함할 케이스를 하나 이상 추가하지 않고는 다른 유형의 작업을 스위치에 추가할 수 없습니다.

인수:

논쟁 설명
Value 평가를 위해 스위치에서 사용할 값

예시:

위의 액션 블록 코드는 변수에 저장된 값을 가져온 다음 가능한 다른 경우와 비교합니다. 케이스 중 하나가 값과 같으면 해당 케이스 아래의 조치가 수행되고, 그렇지 않으면 다음 케이스가 평가됩니다. 스위치가 실행된 후 알람은 30으로 설정됩니다.
