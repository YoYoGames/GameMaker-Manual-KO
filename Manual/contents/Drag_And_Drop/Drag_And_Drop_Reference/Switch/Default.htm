기본

이 작업은 Switch 내에서 새 기본 케이스를 생성합니다. 기본 케이스는 입력 값이 다른 케이스 상수와 일치하지 않는 경우 수행되는 케이스입니다. 기본 사례에 추가된 작업은 "if"에 작업을 추가하는 것처럼 작업 옆에 삭제해야 합니다.

예시:

위의 액션 블록 코드는 전역 변수에 저장된 값을 가져온 다음 가능한 다른 경우와 비교합니다. 케이스 중 하나가 값과 같으면 해당 케이스 아래의 조치가 수행되고, 그렇지 않으면 다음 케이스가 평가됩니다. 입력 값과 일치하는 케이스가 없으면 대신 기본 케이스가 실행됩니다.
