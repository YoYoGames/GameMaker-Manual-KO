사례

이 작업은 Switch 내에서 새 케이스를 생성합니다. Case에 대한 값을 지정하면 Switch의 입력 값과 비교하여 Case에 추가된 조치를 수행합니다. 해당 사례 에서만 수행되도록 사례에 추가된 작업은 "if"에 작업을 추가하는 것처럼 작업 측면으로 삭제해야 합니다.

케이스 값은 컴파일 시 단일 값으로 평가되는 한 모든 데이터 유형(실제, 문자열 등...)이 될 수 있는 상수여야 하므로 변수에 의존하는 표현식을 사용할 수 없습니다.. 또한 Case는 Switch 외부에서 사용할 수 없으며 그렇게 하려고 하면 오류가 발생합니다.

Constant Switch 입력값에 대해 평가할 상수값

예시:

위의 액션 블록 코드는 변수에 저장된 값을 가져온 다음 가능한 다른 경우와 비교합니다. 케이스 중 하나가 값과 같으면 해당 케이스 아래의 조치가 수행되고, 그렇지 않으면 다음 케이스가 평가됩니다. 스위치가 실행된 후 알람은 30으로 설정됩니다.
