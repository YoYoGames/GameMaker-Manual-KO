문자열을 숫자로

이 작업은 문자열을 숫자로 변환할 수 있습니다. 변경할 문자열을 제공한 다음 실수를 다시 반환할 대상 변수를 제공합니다(이 변수는 임시 지역 변수로 플래그가 지정될 수 있음). 문자열에 숫자와 소수점 이외의 문자가 있는 경우 변환 되지 않지만 문자열이 숫자로 시작 하면 변환되고 나머지 문자열은 제외됩니다. 이 작업에 소수점을 전달할 때 반환된 문자열은 소수점 이하 두 자리까지만 반올림된다는 점에 유의하는 것이 중요합니다. 다음은 입력 값과 반환될 것으로 예상되는 표입니다.

끈 반환된 값
"123" 123
"123 Hello World" 123
"Hello World" 0(문자열을 변환할 수 없는 경우 기본적으로 0이 반환됨)
"Hello World 123" 0
"123.45678" 123.46

동작 구문:

인수:

논쟁 설명
Value 변환할 문자열
Target 반환된 값을 저장할 대상 변수

예시:

위의 액션 블록 코드는 전역 변수를 초기화한 다음 버퍼가 저장된 파일을 로드합니다. 버퍼에서 두 개의 문자열을 읽고 그 중 하나가 실수로 변환되어 다른 임시 지역 변수에 저장되기 전에 임시 지역 변수에 저장됩니다. 그런 다음 전역 변수에 할당되고 버퍼가 삭제됩니다.
