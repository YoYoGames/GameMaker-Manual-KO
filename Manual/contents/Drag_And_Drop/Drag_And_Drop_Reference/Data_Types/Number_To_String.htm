숫자를 문자열로

이 작업은 실수를 문자열로 변환합니다. 숫자가 정수 값이 아니고 소수 자릿수가 있는 경우 가장 가까운 두 자리로 반올림됩니다. 예를 들어 숫자 123.45678은 "123.46" 문자열을 반환합니다.

동작 구문:

인수:

논쟁 설명
Value 변환할 실수
Target 반환된 값을 저장할 대상 변수

예시:

위의 액션 블록 코드는 10진수 값을 정수로 변환하여 반환된 값을 임시 로컬 변수에 저장합니다. 이 값은 문자열로 변환되어 화면에 그려집니다.
