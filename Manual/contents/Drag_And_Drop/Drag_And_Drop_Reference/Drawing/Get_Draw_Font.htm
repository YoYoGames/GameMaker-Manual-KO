그리기 글꼴 가져오기

이 작업으로 현재 그리기에 할당된 글꼴의 ID 값을 얻을 수 있습니다. 반환되는 값은 글꼴이 할당되지 않은 경우 -1이거나 자산 브라우저의 글꼴 상수를 나타내는 값 >= 0입니다. 반환 값은 이전에 생성되었거나 새로운 임시 변수일 수 있는 지정하는 대상 변수 에 저장됩니다("Temp" 확인란을 선택한 경우).
