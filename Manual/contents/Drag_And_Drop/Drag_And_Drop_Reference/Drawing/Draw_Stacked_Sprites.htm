누적 스프라이트 그리기

이 작업은 방 내의 주어진 위치에 주어진 수의 스프라이트를 차례로 그립니다. 그릴 스프라이트와 스택 순서는 행 (가로, 왼쪽에서 오른쪽으로) 또는 열 (세로, 위에서 아래로)일 수 있으며 그릴 스프라이트 수와 위치도 제공합니다. 위치는 방 내의 절대 위치이거나 그림을 그리는 인스턴스의 위치에 상대적일 수 있으며 이미지 사이의 간격은 스프라이트의 너비 또는 높이를 기준으로 합니다. 이것은 단순히 정적 이미지(주어진 스프라이트의 초기 단일 이미지(프레임 0))를 그리고 인스턴스 변수 (예: image_xscale 또는 image_blend ).

참고: 이 작업은 다양한 그리기 이벤트 에서만 사용되며 다른 곳에서 사용되는 경우 아무 것도 그리지 않습니다.

동작 구문:

인수:

논쟁 설명
Sprite 그릴 스프라이트
Stack Order 그리는 순서(행 또는 열)
Number 그릴 스프라이트의 수
x 그릴 x 위치
y 그릴 y 위치

예시:

위의 액션 블록 코드는 "obj_Player" 개체의 인스턴스 수를 가져온 다음 이를 사용하여 화면에 마커 스프라이트 수를 그립니다.
