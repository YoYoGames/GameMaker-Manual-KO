그리기 작업 라이브러리

그리기 작업 라이브러리는 특정 그리기 속성을 설정하고 스프라이트, 텍스트 또는 모양을 그리는 데 필요한 작업을 찾을 수 있는 곳입니다. 이러한 작업의 대부분은 개체 의 다양한 그리기 이벤트 에서만 사용하기 위한 것이며 그리기 이벤트 외부에서 사용할 경우 작동하지 않을 수 있습니다. 이에 대한 예외는 모든 이벤트에 추가할 수 있고 이후의 모든 인스턴스에 대한 모든 도면에 영향을 줄 수 있는 설정 작업입니다.

개체의 기본 그리기 이벤트 에 작업을 추가하는 경우 자체 그리기 와 같은 작업을 사용하여 GameMaker 에 명시적으로 그리도록 지시하지 않는 한 인스턴스에 할당된 스프라이트를 그리지 않는다는 점에 유의하는 것이 중요합니다. 기본적으로 GameMaker 는 그리기 이벤트에 다른 것이 없는 경우에만 인스턴스에 할당된 모든 스프라이트를 기본적으로 그립니다.
