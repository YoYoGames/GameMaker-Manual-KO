그리기 색상 설정

이 작업을 통해 모든 모양(사각형/타원/선) 또는 이후에 그려지는 텍스트에 사용할 색상을 설정할 수 있습니다. 모든 그리기 작업 전에 호출할 필요는 없으며 게임 시작 시 한 번만 설정할 수 있습니다. 예를 들어 모든 텍스트가 노란색이 될 경우 게임의 첫 번째 방에 있는 이벤트 만들기에서 이 작업을 호출하면 모든 텍스트(및 모양)가 그때부터 노란색으로 그려집니다. 그러나 여러 색상을 사용해야 하는 경우 다른 색상이 필요한 각 텍스트 또는 모양 앞에 이를 호출해야 합니다.

작업 하단에 알파 사용 확인란이 있습니다. 이렇게 하면 색상의 알파 구성 요소를 사용하여 그리기 알파도 설정할 수 있습니다( Set Draw Alpha 를 사용하는 것과 거의 동일합니다. 알파 구성 요소를 포함하지 않고 단색만 사용하려는 경우 비활성화할 수 있습니다( 기본적으로 활성화되어 있습니다).

예시:

위의 작업 블록 코드는 그려지는 일부 텍스트의 글꼴, 그리기 색상 및 정렬을 설정합니다.
