그리기 알파 설정

이 작업을 사용하면 모든 모양(사각형/타원/선) 또는 이후에 그려지는 텍스트에 사용할 알파를 설정할 수 있습니다. 모든 그리기 작업 전에 호출할 필요는 없으며 게임 시작 시 한 번만 설정할 수 있습니다. 알파는 "투명한" 사물을 그리는 방법을 제어하는 설정이며 0에서 1까지의 척도를 기반으로 합니다. 여기서 0은 완전히 투명하고 1은 완전히 불투명합니다.
