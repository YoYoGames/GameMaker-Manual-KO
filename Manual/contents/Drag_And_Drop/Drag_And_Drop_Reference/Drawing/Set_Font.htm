글꼴 설정

이 작업으로 모든 후속 그리기 텍스트 작업에 사용할 글꼴을 설정할 수 있습니다. 대화 상자를 클릭하면 다음 창이 열리고 사용하려는 글꼴을 선택할 수 있습니다.

이것은 이후의 모든 그리기 텍스트 호출에 사용할 글꼴을 설정하므로 게임 시작 시 한 번만 호출할 수 있습니다. 그리기 이벤트뿐만 아니라 모든 그리기 텍스트 작업은 설정된 글꼴을 사용합니다. 그러나 게임에 여러 글꼴이 필요한 경우 필요한 글꼴로 텍스트를 그리기 전에 이 작업을 호출해야 합니다.

동작 구문:

예시:

위의 작업 블록 코드는 그려지는 일부 텍스트의 글꼴, 그리기 색상 및 정렬을 설정합니다.
