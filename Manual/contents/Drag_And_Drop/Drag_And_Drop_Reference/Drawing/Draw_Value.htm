값 그리기

이 작업으로 문자열 캡션 및/또는 변수 또는 값을 화면에 그릴 수 있습니다. 캡션 인수는 값 앞에 추가하려는 문자열이며 필요한 경우 공백으로 둘 수 있지만 텍스트를 포함하려면 따옴표로 묶어야 하며 값은 변수, 고정 값 또는 다른 값일 수 있습니다. string - 이 경우 따옴표로 묶어야 하지만 공백으로 둘 수 없습니다. 그런 다음 문자열을 그려야 하는 위치를 절대 위치로 지정하거나 텍스트를 그리는 인스턴스에 상대적인 위치로 지정합니다. 그릴 때 텍스트가 어떻게 보이는지는 사용된 글꼴, 사용된 색상 및 정렬에 따라 다릅니다. 이 모든 것은 아래 예와 같이 그리기 전에 변경할 수 있습니다.

참고: 이 작업은 다양한 그리기 이벤트 에서만 사용되며 다른 곳에서 사용되는 경우 아무 것도 그리지 않습니다.

x 그릴 x 위치
y 그릴 y 위치
