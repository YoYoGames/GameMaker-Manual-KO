인스턴스 라이브 그리기

이 작업은 인스턴스가 현재 가지고 있는 각 생명을 나타내는 스프라이트를 그립니다. 그릴 숫자는 로컬 인스턴스 변수의 현재 값에서 가져옵니다. lives, 필요한 작업을 사용하는 경우 인스턴스 내에서 자동으로 생성되는 특수 변수입니다. 행 (가로, 왼쪽에서 오른쪽으로) 또는 열 (세로, 위에서 아래로)과 위치가 될 수 있는 스택 순서와 생명에 대해 그릴 스프라이트를 제공합니다. 위치는 방 내의 절대 위치이거나 그림을 그리는 인스턴스의 위치에 상대적일 수 있으며 이미지 사이의 간격은 스프라이트의 너비 또는 높이를 기준으로 합니다. 이것은 단순히 정적 이미지(주어진 스프라이트의 초기 단일 이미지(프레임 0))를 그리고 인스턴스 변수(예: image_xscale 또는 image_blend ). 캡션 텍스트는 lives 값보다 먼저 그려지며 텍스트인 경우 ""로 감싸야 하며 둘 다 현재 그리기 글꼴, 색상 및 정렬을 사용하여 형식이 지정됩니다.

참고: 이 작업은 다양한 그리기 이벤트 에서만 사용되며 다른 곳에서 사용되는 경우 아무 것도 그리지 않습니다.
