스프라이트 그리기

이 작업은 방 어딘가에 스프라이트를 그립니다. 스프라이트, 위치(방 내의 절대 위치 또는 그리기를 수행하는 인스턴스에 대한 상대 위치) 및 스프라이트의 애니메이션 프레임(여러 프레임이 있는 경우)을 지정합니다(이미지 프레임은 0부터 위쪽으로 번호가 지정됩니다. ). 애니메이션된 스프라이트가 있고 개체에 할당된 것처럼 프레임을 그리려면 프레임 인덱스 또는 기본 제공 인스턴스 변수에 -1을 사용합니다. image_index, 그러나 인스턴스에 할당된 스프라이트가 없으면 애니메이션이 적용되지 않으며 할당된 스프라이트에 그려지는 프레임 수가 다른 경우 스프라이트는 추가 프레임이나 누락된 프레임으로 그려집니다.

중요: 이 작업은 이미지 블렌드 또는 배율을 변경한 경우에도 변형된 스프라이트를 그리지 않습니다. 이에 대해서는 Draw Sprite Transformed 작업을 참조하십시오.

참고: 이 작업은 다양한 그리기 이벤트 에서만 사용되며 다른 곳에서 사용되는 경우 아무 것도 그리지 않습니다.

동작 구문:

인수:

논쟁 설명
Sprite 그릴 스프라이트
x 그릴 x 위치
y 그릴 y 위치
Frame Index 그릴 프레임 인덱스

예시:

위의 액션 블록 코드는 인스턴스와 동일한 상대 위치에 "그림자" 스프라이트를 그린 다음 그 위에 인스턴스 스프라이트를 그립니다.
