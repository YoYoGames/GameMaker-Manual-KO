텍스트 정렬 설정

이 작업은 모든 후속 텍스트 그리기 작업에 대한 글꼴 정렬을 설정합니다. 아래 이미지와 같이 수평 정렬을 왼쪽, 중앙 또는 오른쪽으로 설정하고 수직 정렬을 상단, 중간 또는 하단으로 설정할 수 있으며 텍스트는 그리기 작업의 x/y 위치를 기준으로 정렬됩니다.:

텍스트가 왼쪽에 수평으로 정렬됩니다.
텍스트가 중앙에 수평으로 정렬됩니다.
텍스트가 오른쪽에 수평으로 정렬됨
텍스트가 상단에 수직으로 정렬됩니다.
텍스트가 가운데에 수직으로 정렬됩니다.
텍스트가 아래쪽에 수직으로 정렬됩니다.

이것은 모든 후속 그리기 텍스트 호출에 사용할 정렬을 설정하므로 예를 들어 게임 시작 시 한 번만 호출할 수 있습니다. 그리기 이벤트뿐만 아니라 모든 이벤트에서 모든 그리기 텍스트 작업은 다음을 사용합니다. 설정 정렬. 그러나 서로 다른 방식으로 정렬하기 위해 여러 개의 텍스트 그리기 작업이 필요한 경우 다르게 정렬하려는 모든 그리기 텍스트 작업 전에 이 작업을 호출해야 합니다.
