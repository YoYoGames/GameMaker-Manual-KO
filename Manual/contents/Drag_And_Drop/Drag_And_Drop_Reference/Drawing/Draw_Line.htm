선 그리기

이 작업은 방 내의 주어진 위치에 선을 그립니다. 초기 x/y 위치를 지정한 다음 최종 x/y 위치를 지정하면 현재 그리기 색상을 사용하여 두 위치 사이에 선이 그려집니다. 위치는 룸 내의 절대 위치이거나 작업을 호출하는 인스턴스에 상대적인 위치일 수 있습니다.

참고: 이 작업은 다양한 그리기 이벤트 에서만 사용되며 다른 곳에서 사용되는 경우 아무 것도 그리지 않습니다.

동작 구문:

인수:

논쟁 설명
x 시작 x 위치
y 시작 y 위치
x2 마무리 x 위치
y2 마무리 Y 위치

예시:

위의 액션 블록 코드는 방의 상단을 가로질러 분홍색 선을 그립니다.
