게임패드 버튼 임계값 설정

이 작업은 게임패드 아날로그 버튼의 "임계값"을 설정하는 데 사용할 수 있습니다. 설정할 게임패드 인덱스를 지정한 다음 0~1 사이의 값을 설정하고 아날로그 버튼 입력량 이 주어진 값보다 작으면 게임패드 버튼은 0으로 간주됩니다. 게임패드에 연결된 모든 아날로그 버튼에 영향을 줍니다. 이 값은 주어진 게임패드에 대한 모든 down, 눌림 및 해제 검사에 사용되지만 Get Gamepad Trigger 작업에서는 무시 됩니다.
