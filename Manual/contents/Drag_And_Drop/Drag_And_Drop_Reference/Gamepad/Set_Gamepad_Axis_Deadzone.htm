게임패드 축 데드존 설정

이 작업은 게임패드 스틱 축의 "데드존"을 설정하는 데 사용할 수 있습니다. 설정할 게임패드 인덱스를 지정한 다음 0에서 1 사이의 값을 설정하고 입력량이 지정된 값보다 작으면 게임패드 스틱 축은 0으로 간주됩니다. 연결된 스틱의 모든 축.

이것은 Get Gamepad Axis 액션의 출력을 수정하지 않는다는 점에 유의하십시오. 항상 -1과 0, 0과 1의 값을 반환합니다. 예를 들어 데드존을 0.2로 설정하면 스틱을 오른쪽으로 밀면 시작만 시작됩니다. 원시 축 값이 0.2를 초과하면 0 - 1의 값을 반환합니다. 따라서 원시 값이 0.2이면 반환 값은 0이 되고, 원시 값이 0.5이면 반환 값은 0.375가 됩니다. 값이 0.9이면 반환 값은 0.875가 됩니다.
