게임패드 연결하기

이 작업을 사용하여 지정된 게임패드 "슬롯"에 게임패드가 연결되어 있는지 확인할 수 있습니다. 게임패드 인덱스(게임패드가 차지하는 "슬롯" 수)를 제공하고 반환된 값은 향후 사용을 위해 제공하는 대상 변수에 저장됩니다(이는 사용자가 생성한 임시 로컬 변수로 플래그 지정 가능 이 이벤트 또는 스크립트).

인수:

논쟁 설명
Gamepad 확인할 게임패드 인덱스입니다.
Target 반환된 값을 저장할 대상 변수입니다.

예시:

위의 작업 블록 코드는 다음을 실행합니다. for 루프를 사용하여 게임패드가 사용 가능한 12개의 "슬롯" 중 하나에 연결되어 있는지 확인합니다. 게임패드가 발견되면 전역 변수가 현재 슬롯 번호로 설정되고 루프가 끊어집니다.
