게임패드 버튼을 놓으면

이 작업은 주어진 게임패드 인덱스에 대해 게임패드 버튼을 놓았는지 확인합니다. 그것은 돌아올 것이다 true 버튼을 놓은 단일 게임 틱에서 false 다른 모든 시간에. 활성화하면 not 수정자, 이 작업은 취소되고 버튼이 해제 되지 않았는지 확인하고 반환 false 출시된 게임 틱에 true 다른 모든 시간에.

확인 중인 버튼이 아날로그 버튼이면 버튼 값이 Set Gamepad Button Threshold 작업을 사용하여 설정된 임계값 아래로 내려갈 때까지 릴리스가 감지되지 않습니다.

"if" 블록에 작업을 추가하려면 아래 이미지와 같이 작업 옆에 놓아야 합니다.

"if"가 다음과 같이 평가되면 이러한 작업이 실행됩니다. true, 다른 곳에 드롭된 모든 작업은 "if" 블록 다음에 수행됩니다.

논쟁 설명
Gamepad 게임패드 인덱스.
Button 확인할 버튼의 이름입니다.
Not 검사를 무효화합니다(true는 false가 되고 그 반대도 마찬가지입니다).

예시:

위의 동작 블록 코드는 매 단계마다 게임패드 0의 버튼 상태를 폴링하고 버튼을 누르고 있으면 초기 버튼 누름을 확인합니다. 체크인 경우 true 초기 다운 누름 시 소리가 나다가 버튼을 누르고 있는 동안 인스턴스 속도는 10으로 설정됩니다. 버튼을 누르고 있지 않은 경우 버튼을 놓을 때 확인이 수행되어 속도가 으로 재설정됩니다. 다시 0.
