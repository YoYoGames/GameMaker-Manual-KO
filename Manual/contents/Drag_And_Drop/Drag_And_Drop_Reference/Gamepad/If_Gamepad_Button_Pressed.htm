게임패드 버튼을 누른 경우

이 작업으로 게임패드를 폴링하여 현재 누르고 있는 버튼이 있는지 확인할 수 있습니다. 액션이 돌아올 것이다 true 게임패드 버튼을 누르고 있는 모든 단계 또는 false 그렇지 않으면 not 수정자는 버튼이 눌려 있지 않은지 확인할 수 있습니다. 예: 작업이 반환됩니다. true 버튼을 누르지 않은 상태에서 false 있다면. 단일 버튼 누름만 확인해야 하는 경우 게임 패드 버튼을 누른 상태 에서 작업을 사용합니다.

확인 중인 버튼이 아날로그 버튼이면 버튼 값이 Set Gamepad Button Threshold 작업을 사용하여 설정된 임계값 이상으로 올라갈 때까지 누르기가 감지되지 않습니다.

"if" 블록에 작업을 추가하려면 아래 이미지와 같이 작업 옆에 놓아야 합니다.

"if"가 다음과 같이 평가되면 이러한 작업이 이제 실행됩니다. true, 다른 곳에 드롭된 모든 작업은 "if" 블록 다음에 수행됩니다.

논쟁 설명
Gamepad 게임패드 인덱스.
Button 확인할 버튼의 이름입니다.
Not 검사를 무효화합니다(true는 false가 되고 그 반대도 마찬가지입니다).

예시:

위의 동작 블록 코드는 매 단계마다 게임패드 0의 버튼 상태를 폴링하고 버튼을 누르고 있으면 초기 버튼 누름을 확인합니다. 체크인 경우 true 초기 다운 누름 시 소리가 나다가 버튼을 누르고 있는 동안 인스턴스 속도는 10으로 설정됩니다. 버튼을 누르고 있지 않은 경우 버튼을 놓을 때 확인이 수행되어 속도가 으로 재설정됩니다. 다시 0.
