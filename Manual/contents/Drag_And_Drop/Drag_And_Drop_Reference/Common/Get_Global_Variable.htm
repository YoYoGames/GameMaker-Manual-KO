전역 변수 가져오기

이 작업은 전역 변수의 값을 검색하고 다른 인스턴스나 임시(로컬) 변수에 할당하는 데 사용됩니다. 전역 값을 할당하는 변수는 이전에 생성된 변수일 수 있습니다. 또는 "Temp" 확인란을 선택하고 새 변수 이름을 지정하여 액션이 반환된 값을 저장할 새 임시(로컬) 변수를 생성하도록 할 수 있습니다. 값.

더하기 아이콘을 클릭하여 동일한 작업에서 추가 변수를 추가할 수 있습니다. 작업 옆에 다른 전역 변수 이름과 임시 지역 변수 이름을 지정합니다. 변수에 대한 고급 정보는 변수 및 변수 범위 섹션을 참조하십시오.

논쟁 설명
Name 값을 가져올 전역 변수의 이름입니다.
Target 반환된 값을 보유할 대상 변수 - "Temp"가 선택되지 않은 경우 이전에 생성되어야 합니다(아래 참조).
Temp 반환된 값을 보관할 임시(로컬) 변수를 만듭니다(기본적으로 꺼져 있음).

예시:

위의 작업 블록 코드는 두 개의 전역 변수 값을 검색하고 이를 임시(로컬) 변수에 할당한 다음 "if " 비교. 결과가 true, 세 개의 전역 변수가 새 값으로 설정됩니다.
