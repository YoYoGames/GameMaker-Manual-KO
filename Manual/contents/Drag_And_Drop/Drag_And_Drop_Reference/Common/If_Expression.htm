If 표현식

이 작업은 표현식이 다음 중 하나인지 확인하는 데 사용됩니다. true 또는 false, 여기에서 논리 표현식이 하나 또는 다른 값으로 평가되는 한 확인할 수 있습니다. 예를 들어 다음 표현식 중 하나를 사용할 수 있습니다.

x < (room_width / 2)
a == b
++val >= 100

"Not" 인수에 플래그를 지정하면 위의 표현은 부정 표현식이 됩니다(예: "== "된다"not= ". 또한 표현식 은 숫자 값으로 제한되지 않으며 문자열, 포인터, 리소스 ID 또는 함수가 반환하거나 사용할 수 있는 기타 항목을 확인할 수 있지만(자세한 내용은 데이터 유형 섹션 참조) 두 가지를 비교할 수는 없습니다. 오류 없이 다른 데이터 유형 - 예를 들어 표현식에 문자열과 정수를 사용하려고 하면 오류가 발생합니다. 표현식에 대한 자세한 내용은 여기 를 참조하십시오.

중요 참고 사항: 부동 소수점 정밀도 문제 로 인해 두 값이 정확히 같은지 확인하는 것이 반환될 수 있습니다. false, 하나는 정확히 1일 수 있고 다른 하나는 1.00000000000001일 수 있기 때문입니다. 이것은 "보다 큼(>)" 또는 "보다 작음(<)" 식을 확인하거나 사용하기 전에 10진수에서 정수로 작업을 사용하여 피할 수 있습니다.

"에 작업을 추가하려면if " 블록은 아래 이미지와 같이 작업 옆에 놓아야 합니다.

"if"가 다음과 같이 평가되면 이러한 작업이 이제 실행됩니다. true, 다른 곳에 드롭된 모든 작업은 "if" 블록 다음에 수행됩니다. 더하기 아이콘을 클릭하여 동일한 작업에서 표현식을 "스택"할 수 있습니다. 동작 옆에, 확인하기 위해 다른 표현을 제공합니다. 이것은 주어진 모든 표현식이 반환되는지 확인합니다. true 한 번만 확인하고 추가된 작업 코드가 수행되는 경우에만 수행합니다(아래 제공된 예 참조).
