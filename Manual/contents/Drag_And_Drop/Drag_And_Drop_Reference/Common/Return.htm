반품

이 액션은 액션 블록 함수 를 생성할 때 사용하며 스크립트를 호출하는 인스턴스에 값을 반환하는 데 사용됩니다. 반환 값은 문자열, 숫자 값, 포인터 또는 리소스 ID와 같이 변수가 보유할 수 있는 모든 것이거나 변수 자체일 수 있습니다. 함수에서 반환 작업을 호출하면 함수가 종료되고(주어진 값을 반환함) 반환 이후에 더 이상 작업이 호출되지 않습니다. 여전히 실행됩니다.
