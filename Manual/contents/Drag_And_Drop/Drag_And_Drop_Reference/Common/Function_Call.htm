함수 호출

이 작업은 GML 함수 또는 사용자 정의 함수 를 호출하는 데 사용할 수 있습니다. 호출할 함수를 제공한 다음 필요한 추가 인수, 즉 사용할 수 있도록 함수에 입력하려는 모든 값(문자열, 정수, 리소스 인덱스 등...)을 제공합니다. 더하기 아이콘을 클릭하여 추가 인수 입력을 추가할 수 있습니다. 액션 옆에, 그리고 당신의 인수 개수가 사용 중인 함수의 개수와 일치하는지 확인해야 합니다(예: 함수에 3개의 인수가 필요한 경우 3개의 인수를 제공해야 함). 일부 함수는 나중에 작업 차단 목록에서 사용하기 위해 값을 반환할 수도 있으므로 이 반환 값을 저장해야 하는 대상 변수 를 지정할 수도 있습니다. " Temp " 체크박스). 필요하지 않은 경우 대상 및 인수 필드를 모두 공백으로 둘 수 있습니다.
