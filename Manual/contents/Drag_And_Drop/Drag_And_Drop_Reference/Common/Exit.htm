출구

이 작업은 호출된 컨텍스트에 따라 두 가지 작업 중 하나를 수행합니다. 이벤트 내에서 호출되면 이벤트를 종료하고 다음 이벤트로 이동하여 뒤에 배치될 수 있는 작업 블록 코드를 건너뜁니다. 그러나 Action Block Function 내에서 사용되는 경우 함수만 종료하고 함수를 호출한 이벤트로 돌아가서 후속 작업을 실행합니다. 함수 중간에 이것을 호출하면 값이 반환 되지 않습니다 ( return 작업 사용).

예시:

위의 액션 블록 코드는 변수를 확인하고 100보다 크거나 같으면 이벤트를 종료하고, 그렇지 않으면 변수를 1씩 증가시키고 인스턴스 스프라이트에 임의의 크기를 부여합니다.
