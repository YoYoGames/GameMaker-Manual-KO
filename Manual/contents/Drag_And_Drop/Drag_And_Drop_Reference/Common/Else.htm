또 다른

"else" 작업은 If Variable (또는 다른 "if" 작업)과 함께만 사용되며 "if" 평가가 반환되는 경우 수행해야 하는 작업 집합을 정의하는 방법입니다. false. 예를 들어, 플레이어 건강 변수가 0보다 작거나 같은지 확인하는 "if" 작업이 있을 수 있으며, 이것이 true로 평가되면 인스턴스를 파괴합니다. 그러나 플레이어 건강이 0보다 작지 않은 경우 다른 코드 작업 세트를 수행하려는 경우 "else"를 추가한 다음 if가 다음으로 평가되는 동안 수행할 작업을 지정할 수 있습니다. false.

"else" 블록에 작업을 추가하려면 아래 이미지와 같이 작업 옆에 놓아야 합니다.

다른 곳에 드롭된 모든 작업은 "else" 블록과 해당 내용이 실행된 후에 수행됩니다.
