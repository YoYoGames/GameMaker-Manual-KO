스크립트 실행

이 작업은 스크립트 함수 또는 사용자 정의 함수 를 호출하는 데 사용할 수 있습니다. 일반적으로 이 작업은 레거시 지원에만 사용되며 현재 또는 새 프로젝트에서는 실제로 사용해서는 안 됩니다. 작업을 사용하려면 호출할 스크립트 자산 또는 호출할 사용자 정의 함수 변수 이름을 제공한 다음 필요한 인수, 즉 모든 값(문자열, 정수, 리소스 인덱스 등...)을 제공합니다. 스크립트나 함수에 입력하여 사용할 수 있도록 합니다. 더하기 아이콘을 클릭하여 필요에 따라 추가 인수를 추가할 수 있습니다. 행동 옆에.

일부 스크립트 및 사용자 정의 함수는 나중에 작업 차단 목록에서 사용하기 위해 값을 반환할 수도 있으므로 이 반환 값을 저장해야 하는 대상 변수 를 지정할 수 있습니다. 이것은 이전에 생성되었거나 새로운 임시 로컬일 수 있습니다(" Temp " 체크박스를 선택한 경우). 필요하지 않은 경우 대상 및 인수 필드를 모두 공백으로 둘 수 있습니다.

일반적으로 함수 호출 을 사용하여 사용자 정의 함수를 호출하는 것이 더 나으며 일반적으로 이 작업을 사용하거나 코드에서 직접 스크립트 자산을 호출할 필요가 없습니다.
