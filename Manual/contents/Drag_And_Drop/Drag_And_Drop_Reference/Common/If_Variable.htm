변수인 경우

이 작업은 특정 표현식을 사용하여 주어진 값에 대해 주어진 변수의 상태를 확인하는 데 사용됩니다. "if" 문은 항상 다음 중 하나를 반환합니다. true 또는 false 확인을 위한 표현식은 다음 중 하나일 수 있습니다.

  • 같음 - 변수와 값이 모두 정확히 같습니다.
  • 보다 작음 - 변수가 값보다 작음
  • 보다 큼 - 변수가 값보다 큼
  • 작거나 같음 - 변수가 값보다 작 거나 같음
  • 크거나 같음 - 변수가 값보다 크 거나 같음

"Not" 인수에 플래그를 지정하면 위의 표현이 부정됩니다. 예를 들어 "같음"은 " 같지 않음 "이 됩니다. 또한 변수나 값 인수는 숫자 값으로 제한되지 않으며 문자열, 포인터, 리소스 ID 또는 함수가 반환하거나 사용할 수 있는 기타 모든 것이 될 수 있습니다(자세한 내용은 데이터 유형 섹션 참조). 오류가 발생하지 않고는 서로 다른 두 데이터 유형을 비교할 수 없습니다. 예를 들어 변수에 문자열이 포함되어 있고 숫자와 같은지 확인하려고 하면 구문 오류가 발생합니다.

중요 참고 사항: 부동 소수점 정밀도 문제 로 인해 두 값이 정확히 같은지 확인하는 것이 반환될 수 있습니다. false, 하나는 정확히 1일 수 있고 다른 하나는 1.00000000000001일 수 있기 때문입니다. "보다 큼" 또는 "보다 작음" 식을 확인하거나 사용하기 전에 10진수에서 정수로 작업을 사용하여 이를 방지할 수 있습니다.

"if" 블록에 작업을 추가하려면 아래 이미지와 같이 작업 옆에 놓아야 합니다.

"if"가 다음과 같이 평가되면 이러한 작업이 이제 실행됩니다. true, 다른 곳에 드롭된 모든 작업은 "if" 블록 다음에 수행됩니다.
