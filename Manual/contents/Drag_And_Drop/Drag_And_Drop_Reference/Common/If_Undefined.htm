정의되지 않은 경우

이 작업은 변수 또는 표현식이 "정의되지 않음" 값을 반환하는지 확인하는 데 사용되며, 그렇다면 이 작업은 다음을 반환합니다. true 그리고 그렇지 않은 경우 반환됩니다 false. "Undefined"는 여러 가지 이유로 반환될 수 있으며 정의되지 않은 값은 프로젝트에서 충돌이나 심각한 오류를 일으킬 수 있으므로 이를 확인하고 발생하기 전에 잡을 수 있습니다. "Not" 인수에 플래그를 지정하면 값이 "undefined"가 아닌지 확인하는 것입니다.

"if" 블록에 작업을 추가하려면 아래 이미지와 같이 작업 옆에 놓아야 합니다.

"if"가 다음과 같이 평가되면 이러한 작업이 실행됩니다. true, 다른 곳에 드롭된 모든 작업은 "if" 블록 다음에 수행됩니다. 더하기 아이콘을 클릭하여 동일한 작업에서 정의되지 않은 검사를 "스택"할 수 있습니다. 액션 옆에 체크할 또 다른 변수를 제공합니다. 이것은 주어진 모든 변수가 한 번의 검사에서 정의되지 않았는지 확인하고 정의되지 않은 경우에만 추가된 작업 코드를 수행합니다.
