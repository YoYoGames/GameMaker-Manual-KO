전역 변수 설정

이 작업은 전역 변수에 값을 할당하거나(프로세스에서 생성) 기존 전역 변수의 값을 변경하는 데 사용됩니다. 전역 변수는 특정 인스턴스가 아니라 전체 프로젝트에 "속하는" 변수이며 모든 인스턴스, 스크립트 또는 코드에서 액세스하고 변경할 수 있습니다. 전역 변수에 대한 고급 정보는 여기를 참조 하십시오.

전역 변수의 이름을 지정한 다음 필요한 경우 원래 값에 상대적 으로 추가할 수 있는 값을 제공합니다. 새 변수 이름을 지정하면(예: 전체 게임에서 이전에 할당된 적이 없는 변수) 새 전역 변수를 생성하고 지정한 값을 할당하는 반면 기존 전역 변수의 이름을 지정하면, 단순히 값을 새 값으로 변경합니다. 상대 를 선택하면 기존 전역 변수 값에 값이 추가되고(빼기 위해 음수 사용) 표현식 s 및 이전에 생성한 변수를 값으로 사용할 수도 있습니다.

"값"에 대해 이야기할 때 변수는 문자열, 포인터, 리소스 ID 또는 함수가 반환하거나 사용할 수 있는 모든 것이 될 수 있으므로 숫자 값만을 의미하는 것이 아닙니다. 문자열의 경우 상대 를 확인하면 이전에 보유했던 변수와 값이 연결되므로 변수가 "Hello"이고 상대 확인을 사용하여 변수에 " World"를 할당하면 최종 변수 값은 "Hello World"가 됩니다. ". 그러나 예를 들어 정수와 문자열을 추가하려고 하면 컴파일러 오류가 발생하므로 변수 값이 모두 동일한 데이터 유형 인지 확인하기 위해 주의를 기울여야 합니다.

더하기 아이콘을 클릭하여 동일한 작업에 추가 변수를 추가할 수도 있습니다. 액션 옆에 다른 전역 변수 이름과 값을 제공합니다. 변수에 대한 고급 정보는 변수 및 변수 범위 섹션을 참조하십시오.
