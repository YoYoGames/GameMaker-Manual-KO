버퍼 읽기

이 작업은 이전에 생성된 버퍼에서 데이터를 읽는 데 사용할 수 있습니다. Create Buffer 작업에서 반환된 고유한 버퍼 ID 값을 제공하고 반환된 값을 저장할 변수를 제공합니다. 변수에 임시 지역 변수로 플래그를 지정하면 해당 변수가 자동으로 생성되고 이벤트 또는 작업 스크립트가 끝날 때 다시 삭제됩니다. 모든 데이터는 문자열 로 반환되므로 실수를 저장한 경우 데이터 유형 작업 중 하나를 사용하여 문자열에서 값으로 변환해야 합니다. 버퍼는 순차적으로 읽고 기록되므로 버퍼에서 읽으면 "검색" 위치가 다음 항목으로 이동합니다. 즉, 다양한 데이터 조각을 읽으려면 이 작업을 필요한 횟수만큼 호출하고 저장하기만 하면 됩니다. 각 반환된 데이터 조각을 변수로 보냅니다.

동작 구문:

인수:

논쟁 설명
Buffer 버퍼 인덱스(변수에 저장됨)
Target 반환된 데이터를 저장할 대상 변수

예시:

위의 작업 블록 코드는 버퍼 읽기/쓰기 위치를 버퍼의 시작으로 설정한 다음 두 개의 데이터를 다시 읽어서 임시 지역 변수에 (문자열로) 저장합니다. 이러한 변수는 데이터를 화면에 그리는 데 사용됩니다.
