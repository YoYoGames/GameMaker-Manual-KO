버퍼의 끝인 경우

이 작업은 읽기/쓰기 위치가 버퍼 끝에 도달했는지 확인하는 데 사용할 수 있습니다. 버퍼 생성 작업에서 반환된 고유한 버퍼 ID 값을 제공하면 작업이 반환됩니다. true 읽기/쓰기 위치가 버퍼에서 사용 가능한 모든 데이터의 끝에 도달한 경우 또는 false 그렇지 않은 경우. 읽기/쓰기 위치가 버퍼의 끝에 있지 않은 경우 조치를 확인하기 위해 "not"에 플래그를 지정할 수 있습니다.

"if" 블록에 작업을 추가하려면 아래 이미지와 같이 작업 옆에 놓아야 합니다.
