버퍼 크기 가져오기

이 작업은 버퍼 크기(바이트)를 검색하는 데 사용할 수 있습니다. Create Buffer 작업에서 반환된 고유한 버퍼 ID 값을 제공한 다음 크기 값을 반환할 대상 변수를 제공합니다. 대상 변수는 임시 지역 변수로 플래그를 지정할 수 있습니다. 이 경우 현재 작업 스크립트 또는 이벤트가 끝날 때 자동으로 생성되고 메모리에서 다시 제거됩니다.

동작 구문:

인수:

논쟁 설명
Buffer 버퍼 인덱스(변수에 저장됨)
Target 반환된 값을 저장할 대상 변수

예시:

위의 액션 블록 코드는 버퍼의 크기를 가져온 다음 해당 값을 사용하여 현재 저장된 데이터의 끝으로 읽기/쓰기 위치를 설정합니다.
