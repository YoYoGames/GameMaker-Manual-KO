버퍼 위치 가져오기

이 작업은 버퍼에 읽기/쓰기를 위한 현재 위치를 검색하는 데 사용할 수 있습니다. 버퍼를 생성할 때 읽기/쓰기 위치는 0바이트로 설정되며 버퍼에 데이터를 추가하면 이 위치가 순차적으로 이동합니다. 즉, 이 작업을 사용하여 버퍼에 들어갈 바이트 수를 가져올 수 있습니다. 언제든지 로 설정됩니다. Create Buffer 작업에서 반환된 고유한 버퍼 ID 값을 제공한 다음 위치 값을 반환할 대상 변수를 제공합니다. 대상 변수는 임시 지역 변수로 플래그를 지정할 수 있습니다. 이 경우 현재 작업 스크립트 또는 이벤트가 끝날 때 자동으로 생성되고 메모리에서 다시 제거됩니다.

Position 버퍼 내 읽기/쓰기 위치(바이트)

예시:

위의 액션 블록 코드는 주어진 버퍼에 대한 읽기/쓰기 위치를 가져오고 그것이 0이 아니면(즉, 버퍼의 시작에서) 버퍼의 시작으로 설정됩니다.
