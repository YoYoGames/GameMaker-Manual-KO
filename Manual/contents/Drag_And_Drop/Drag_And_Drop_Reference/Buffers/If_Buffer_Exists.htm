버퍼가 있는 경우

이 작업은 주어진 버퍼 ID가 유효하고 버퍼가 실제로 메모리에 존재하는지 확인하는 데 사용할 수 있습니다. 버퍼 ID가 유효하고 존재하는 경우 작업이 반환됩니다. true 그렇지 않으면 반환됩니다 false. 버퍼가 존재하지 않는 경우 조치를 확인하기 위해 "not"에 플래그를 지정할 수 있습니다.

"if" 블록에 작업을 추가하려면 아래 이미지와 같이 작업 옆에 놓아야 합니다.
