버퍼 찾기

이 작업은 버퍼 내의 위치로 이동(검색)하는 데 사용할 수 있습니다. 버퍼 생성 작업에서 반환된 고유 버퍼 ID 값을 제공한 다음 이동할 오프셋 값(바이트)을 제공합니다. 오프셋 값은 버퍼의 시작 부분부터 적용되므로 이 함수를 호출하면 읽기/쓰기 위치가 버퍼의 시작 부분으로 이동한 다음 주어진 바이트 수를 이동하고 새로운 읽기/쓰기 위치를 거기에 배치합니다. 그러나 오프셋에 상대적인 플래그를 지정하면 읽기/쓰기 위치가 현재 위치에서 앞으로(또는 오프셋이 음수 값인 경우 뒤로) 이동합니다.

동작 구문:

인수:

논쟁 설명
Buffer 버퍼 인덱스(변수에 저장됨)
offset 현재 탐색 위치에 대한 오프셋(바이트)

예시:

위의 액션 블록 코드는 버퍼의 크기를 가져온 다음 해당 값을 사용하여 현재 저장된 데이터의 끝으로 읽기/쓰기 위치를 설정합니다.
