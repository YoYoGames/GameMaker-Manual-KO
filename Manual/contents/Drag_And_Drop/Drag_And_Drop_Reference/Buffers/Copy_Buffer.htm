버퍼 복사

이 작업은 한 버퍼에 저장된 모든 데이터를 다른 버퍼로 복사하는 데 사용할 수 있습니다. 두 개의 버퍼를 사용하고 한 버퍼에서 다른 버퍼로 복사할 때 두 버퍼는 모두 이전에 버퍼 생성 작업을 사용하여 생성 되어야 하며 시작/끝 인수로 반환된 고유 버퍼 ID를 제공해야 합니다.

참고: 동일한 버퍼에 복사할 수 없습니다.

동작 구문:

인수:

논쟁 설명
Buffer 복사 할 버퍼 인덱스(변수에 저장됨)
Copy To 복사 할 버퍼 인덱스(변수에 저장됨)

예시:

위의 작업 블록 코드는 새 버퍼를 만들고 해당 ID를 임시 변수에 저장합니다. 그런 다음 주어진 전역 변수에 인덱스된 버퍼가 존재하는지 확인하고 존재하면 임시 버퍼에 복사하고 읽기/쓰기 위치를 다시 시작으로 재설정하고 존재하지 않으면 존재합니다. 만들어진.
