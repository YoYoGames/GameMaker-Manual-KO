쓰기 버퍼

이 작업은 버퍼에 데이터를 쓰는 데 사용할 수 있습니다. 작성할 고유 버퍼 ID( Create Buffer 에 의해 반환됨)를 제공한 다음 작성할 값(실제 또는 문자열일 수 있음)을 제공하지만 모든 값은 문자열로 저장 되므로 읽을 때 버퍼( 버퍼 읽기 사용)를 사용하여 데이터 유형 작업 중 하나를 사용하여 문자열에서 다시 원래 값으로 변환해야 할 수도 있습니다. 버퍼는 순차적으로 읽고 쓰므로 버퍼에 쓰는 것은 "검색" 위치를 하나씩 이동합니다. 즉, 다양한 데이터 조각을 쓰려면 이 작업을 필요한 횟수만큼 호출하기만 하면 됩니다.

Value 버퍼에 추가할 값

예시:

위의 액션 블록 코드는 전역 변수를 생성한 다음 버퍼를 생성하여 변수에 고유한 버퍼 ID를 할당합니다. 이 버퍼에는 두 개의 데이터가 기록되어 있습니다.
