버퍼 삭제

이 작업은 메모리에서 버퍼를 삭제하고 버퍼를 생성하는 데 사용된 리소스를 해제하고 현재 포함할 수 있는 데이터를 제거하는 데 사용할 수 있습니다. Create Buffer 작업에서 반환된 고유한 버퍼 ID 값을 제공합니다. 더하기 아이콘을 클릭하여 제거할 버퍼를 여러 개 선택할 수 있습니다. 작업 옆에 다른 버퍼 ID를 추가합니다.

참고: 메모리 누수를 방지하기 위해 더 이상 필요하지 않을 때 동적으로 생성된 리소스를 항상 메모리에서 제거하는 것이 중요합니다.
