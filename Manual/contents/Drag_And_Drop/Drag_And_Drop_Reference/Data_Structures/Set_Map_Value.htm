맵 값 설정

이 작업으로 맵 데이터 구조에 키/값 쌍을 추가할 수 있습니다. 맵 인덱스를 보유하는 변수( Create Map 에서 반환됨)를 제공한 다음 "key"(값에 대한 맵 내 식별자)를 제공한 다음 해당 키와 연결할 "value"를 제공합니다.. 두 키는 정수 값(예: 0, 200, 36712 등...) 또는 문자열(예: "Name", "Shield" 등...)일 수 있지만 값은 모든 유효한 데이터 유형일 수 있습니다.

동작 구문:

인수:

논쟁 설명
Map 설정할 맵의 인덱스(변수에 저장됨)
Key 설정할 키(정수 또는 문자열)
Value 주어진 키로 저장할 값

예시:

위의 작업 블록 코드는 새 맵 데이터 구조를 만든 다음 그 안에 세 개의 키/값 항목을 설정합니다.
