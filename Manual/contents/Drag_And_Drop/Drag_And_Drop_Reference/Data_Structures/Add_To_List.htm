목록에 추가

이 작업은 목록에 새 값(모든 데이터 유형)을 추가하는 데 사용할 수 있으며 이 값은 끝에 추가됩니다. 목록 인덱스( Create List 작업에서 반환됨)와 저장할 값을 저장하는 변수를 제공합니다.

동작 구문:

인수:

논쟁 설명
List 추가할 목록의 인덱스(변수에 저장됨)
Value 목록에 추가할 값

예시:

위의 액션 블록 코드는 전역 범위 변수를 생성한 다음 인덱스 값을 전역 변수에 할당하여 새로운 목록 데이터 구조를 생성합니다. 그런 다음 "obj_Enemy_Parent" 개체의 모든 인스턴스가 고유한 인스턴스 ID 값을 목록에 추가하도록 범위가 변경됩니다.
