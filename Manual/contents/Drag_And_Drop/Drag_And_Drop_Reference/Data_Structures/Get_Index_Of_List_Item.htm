목록 항목의 인덱스 가져오기

이 작업으로 주어진 목록 내 항목의 인덱스 위치를 복구할 수 있습니다. Create List 작업에서 반환된 목록 인덱스를 저장하는 변수와 목록 내에서 인덱스 위치를 찾고자 하는 값은 물론 항목에 대해 반환된 인덱스 위치를 저장하기 위한 대상 변수( 스크립트 또는 이벤트가 끝날 때까지 사용할 임시 지역 변수로 플래그를 지정할 수 있습니다. 찾고 있는 값이 목록 내에 존재하지 않으면 반환할 목록 인덱스 위치가 없으므로 반환 값은 -1이 됩니다.

동작 구문:

인수:

논쟁 설명
List 제거할 목록의 인덱스(변수에 저장됨)
Value 목록을 확인할 값
Target 반환된 인덱스를 저장할 대상 변수

예시:

위의 액션 블록 코드는 인스턴스 위치에서 충돌을 확인하고 충돌이 발견되면 인스턴스의 고유 ID 값을 임시 변수에 저장한 다음 목록 데이터 구조 내에 존재하는지 확인합니다. 존재하는 경우 항목이 목록에서 제거되고 인스턴스가 파괴됩니다.
