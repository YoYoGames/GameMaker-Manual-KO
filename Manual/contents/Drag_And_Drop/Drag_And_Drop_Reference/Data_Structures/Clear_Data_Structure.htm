명확한 데이터 구조

이 작업은 주어진 데이터 구조에서 모든 값을 지웁니다. 먼저 지울 데이터 구조의 유형을 선택한 다음 데이터 구조의 인덱스를 지정해야 합니다(인덱스는 원래 데이터 구조를 생성할 때 반환되어 변수에 저장된 값입니다). 지워진 데이터 구조는 그리드를 제외하고 "빈"(즉, 항목이 없음)으로 간주됩니다. 그리드를 사용하면 이 작업은 단순히 모든 그리드 셀을 0으로 지웁니다.

참고: 이렇게 하면 메모리에서 데이터 구조가 해제되지 않고 모든 정보만 지워집니다. 데이터 구조에 할당된 메모리를 해제하려면 Free Data Structure 작업을 사용해야 합니다.

동작 구문:

인수:

논쟁 설명
Structure 지우려는 데이터 구조 유형(목록, 맵, 그리드 또는 스택)
Variable 지울 데이터 구조의 인덱스를 보유하는 변수

예시:

위의 액션 블록 코드는 전역 변수에 참조된 데이터 구조가 비어 있지 않은지 확인하고 비어 있으면 데이터 구조를 지웁니다.
