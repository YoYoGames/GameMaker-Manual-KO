목록 항목 가져오기

이 작업으로 주어진 목록 내 항목의 인덱스 위치를 복구할 수 있습니다. 목록 인덱스를 저장하는 변수( 목록 만들기 작업에서 반환됨)와 항목 값을 가져올 목록 내의 인덱스 위치, 반환된 항목 값(다음으로 플래그 지정 가능)을 저장할 대상 변수를 제공합니다. 스크립트 또는 이벤트가 끝날 때까지 사용할 임시 로컬 변수).

참고: 지정된 목록 크기를 벗어나는 위치를 지정하면(예: 10 값 목록에서 위치 11) 작업이 반환됩니다. undefined. 정의되지 않은 경우 작업을 사용하여 이를 확인할 수 있습니다.

동작 구문:

인수:

논쟁 설명
List 제거할 목록의 인덱스(변수에 저장됨)
Index 항목을 가져올 목록 내의 인덱스 위치
Target 반환된 값을 저장할 대상 변수

예시:

위의 액션 블록 코드는 전역 변수에서 참조하는 목록의 크기를 가져온 다음 for 루프를 사용하여 반복하고 목록에 저장된 ID로 인스턴스를 파괴합니다. 루프가 완료되면 목록이 메모리에서 해제됩니다.
