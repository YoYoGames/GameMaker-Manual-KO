목록에 삽입

이 작업으로 주어진 목록 내 항목의 인덱스 위치를 복구할 수 있습니다. Create List 작업에서 반환된 목록 인덱스를 저장하는 변수와 목록 내에서 인덱스 위치를 찾고자 하는 값은 물론 항목에 대해 반환된 인덱스 위치를 저장하기 위한 대상 변수( 스크립트 또는 이벤트가 끝날 때까지 사용할 임시 지역 변수로 플래그를 지정할 수 있습니다. 목록에 값을 삽입하면 삽입을 위해 지정된 인덱스 위의 모든 인덱스가 위로 이동하므로 목록에 4개의 항목(인덱스 위치 0~3)이 있는 목록이 있는 경우 인덱스 위치 2에 새 항목을 배치합니다., 이전 항목 2는 항목 3이 되고 이전 항목 3은 항목 4가 됩니다. 찾고 있는 값이 목록 내에 없으면 반환할 목록 인덱스 위치가 없으므로 반환합니다. 값은 -1이 됩니다.

동작 구문:

인수:

논쟁 설명
List 삽입할 목록의 인덱스(변수에 저장됨)
Index 값을 삽입할 목록 내 인덱스 위치
Value 해당 위치에 저장할 값

예시:

위의 액션 블록 코드는 반복 루프를 사용하여 목록의 크기를 얻은 다음 목록 내에서 임의의 위치를 생성합니다. 그런 다음 임의의 값을 생성하고 이 값을 이전에 정의된 임의의 위치에 있는 목록에 삽입합니다.
