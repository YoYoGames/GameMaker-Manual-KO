목록에서 제거하다

이 작업은 주어진 목록에서 인덱스 항목을 제거하는 데 사용할 수 있습니다. 목록 인덱스( Create List 작업에서 반환됨)와 제거할 목록 내의 인덱스 위치를 저장하는 변수를 제공합니다. 여기서 인덱스 위치는 1과 (목록 길이 -1) 사이입니다. 이 작업은 아무 것도 반환하지 않으므로 인덱스 위치에 값이 필요한 경우 인덱스를 제거하기 전에 Get List Item At 를 사용해야 합니다.

동작 구문:

인수:

논쟁 설명
List 제거할 목록의 인덱스(변수에 저장됨)
Index 제거할 목록 내의 인덱스

예시:

위의 액션 블록 코드는 인스턴스 위치에서 충돌을 확인하고 충돌이 발견되면 인스턴스의 고유 ID 값을 임시 변수에 저장한 다음 목록 데이터 구조 내에 존재하는지 확인합니다. 존재하는 경우 항목이 목록에서 제거되고 인스턴스가 파괴됩니다.
