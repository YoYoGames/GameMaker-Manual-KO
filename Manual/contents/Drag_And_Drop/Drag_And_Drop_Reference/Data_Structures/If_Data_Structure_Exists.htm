데이터 구조가 존재하는 경우

이 작업으로 변수를 검사하여 데이터 구조를 색인화하는지 여부를 확인할 수 있습니다. 먼저 확인할 데이터 구조 유형(목록, 맵, 그리드 또는 스택)을 선택한 다음 데이터 구조 인덱스를 보유하는 변수를 제공해야 합니다( Create List, Create Map 작업에서 반환됨)., 그리드 생성 및 스택 생성 ). 액션이 돌아올 것이다 true 데이터 구조가 존재하는 경우 false 그렇지 않은 경우. 변수에 데이터 구조 인덱스가 없는지 확인하기 위해 조치에서 "not"에 플래그를 지정할 수 있습니다.

"if" 블록에 작업을 추가하려면 아래 이미지와 같이 작업 옆에 놓아야 합니다.

"if"가 다음과 같이 평가되면 이러한 작업이 실행됩니다. true, 다른 곳에 드롭된 모든 작업은 "if" 블록 다음에 수행됩니다.
