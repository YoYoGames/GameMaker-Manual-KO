그리드 값 설정

이 작업으로 그리드 데이터 구조 내의 특정 셀을 값으로 설정할 수 있습니다. Create Grid 작업에서 반환된 그리드 인덱스를 보유하는 변수를 제공한 다음 설정할 셀의 x 및 y 위치(열 및 행)를 제공하고 마지막으로 해당 셀의 값을 제공합니다. 값은 모든 유효한 데이터 유형일 수 있지만 x 및 y 위치는 그리드 경계 내에서 정수여야 합니다.

동작 구문:

인수:

논쟁 설명
Grid 설정할 그리드의 인덱스(변수에 저장)
x 설정할 x축을 따른 셀 위치(열)
y 설정할 y축의 셀 위치(행)
Value 그리드 셀을 설정할 값

예시:

위의 액션 블록 코드는 새로운 그리드 데이터 구조를 생성하고 해당 인덱스 값을 전역 변수에 보관합니다. 그런 다음 방의 모든 인스턴스를 반복하고 해당 위치에 해당하는 셀에 고유 ID 값을 추가합니다.
