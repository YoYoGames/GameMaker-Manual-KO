지도 값 가져오기

이 작업을 통해 맵 데이터 구조에서 지정된 키와 연결된 값을 검색할 수 있습니다. 맵 인덱스를 보유하는 변수( Create Map 에서 반환됨)를 제공한 다음 "key"(값에 대한 맵 내 식별자)를 제공하고 반환된 값( 임시 지역 변수로 표시될 수 있음).

참고: 지도에 없는 키를 지정하면 작업이 반환됩니다. undefined. 정의되지 않은 경우 작업을 사용하여 이를 확인할 수 있습니다.

동작 구문:

인수:

논쟁 설명
Map 값을 가져올 맵의 인덱스(변수에 저장됨)
Key 가져올 키(실제 또는 문자열)
Target 반환된 맵 값을 담을 대상 변수

예시:

위의 작업 블록 코드는 인스턴스 위치에서 충돌을 확인하고 충돌이 발견되면 "손상" 키에서 값을 얻기 위해 맵을 조회한 다음 이 값을 사용하여 호출하는 인스턴스 변수 "hp"를 변경합니다..
