스택에 푸시

이 작업으로 스택 데이터 구조에 값을 푸시할 수 있습니다. 스택 인덱스를 보유하는 변수( Create Stack 에서 반환됨)를 제공한 다음 맨 위에 푸시할 값을 제공합니다. 스택 데이터 구조를 사용하면 값이 항상 스택의 "상단"에 추가되며 끝이나 중간에 추가되지 않습니다(이 기능이 필요한 경우 목록 데이터 구조 사용).

동작 구문:

인수:

논쟁 설명
Stack 사용할 스택의 인덱스(변수에 저장됨)
Value 스택에 푸시할 값

예시:

위의 액션 블록 코드는 인스턴스 변수와 새로운 스택 데이터 구조를 생성합니다. 스택의 인덱스는 새 변수에 저장되고 10개의 인스턴스를 생성하고 고유한 ID 값을 스택에 푸시하는 루프가 수행됩니다.
