그리드 값 가져오기

이 작업으로 그리드 데이터 구조 내의 특정 셀을 값으로 설정할 수 있습니다. Create Grid 작업에서 반환된 그리드 인덱스를 보유하는 변수를 제공한 다음 설정할 셀의 x 및 y 위치(열 및 행)를 제공하고 마지막으로 해당 셀의 값을 제공합니다. 또한 반환된 셀 값을 저장하려면 대상 변수(임시 지역 변수로 플래그 지정 가능)를 설정해야 하며 값은 모든 유효한 데이터 유형이 될 수 있습니다. 셀의 x 및 y 위치는 그리드 경계 내의 정수여야 합니다.

인수:

논쟁 설명
Grid 가져올 그리드의 인덱스(변수에 저장됨)
x 가져올 x 축을 따른 셀 위치(열)
y (행)을 얻을 y 축을 따른 셀 위치
Target 반환된 값을 저장하는 데 사용할 대상 변수

예시:

위의 액션 블록 코드는 방 내의 인스턴스 위치에 해당하는 전역 변수에 저장된 그리드의 값을 가져옵니다. 그런 다음 이 값을 확인하고 인스턴스 ID 값이면 해당 인스턴스가 소멸되고 그리드 셀이 키워드로 설정됩니다. noone.
