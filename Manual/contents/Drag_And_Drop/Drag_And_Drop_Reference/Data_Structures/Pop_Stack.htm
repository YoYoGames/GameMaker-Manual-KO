팝 스택

이 작업으로 스택 데이터 구조의 맨 위에서 값을 팝할 수 있습니다. 스택 인덱스를 보유하는 변수( Create Stack 에서 반환됨)를 제공한 다음 반환된 변수(임시 로컬 변수로 플래그 지정 가능)를 보유해야 하는 대상 변수를 제공합니다.

동작 구문:

인수:

논쟁 설명
Stack 사용할 스택의 인덱스(변수에 저장됨)
Target 반환된 값을 저장할 대상 변수

예시:

위의 액션 블록 코드는 주어진 스택이 비어 있는지 확인하고 비어 있지 않으면 최상위 값을 팝하고 임시 변수에 저장합니다. 그런 다음 이것은 알람을 설정하는 데 사용되며 스택이 비어 있으면 10개의 임의 값이 스택에 푸시됩니다.
