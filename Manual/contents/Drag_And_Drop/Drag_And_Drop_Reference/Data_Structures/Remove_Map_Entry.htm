지도 항목 제거

이 작업으로 맵 데이터 구조에서 키(및 해당 값)를 제거할 수 있습니다. 맵 인덱스를 보유하는 변수( 맵 생성 작업에서 반환됨)를 제공한 다음 "키"(맵 내의 값에 대한 식별자)를 제공하면 작업이 맵에서 해당 키를 제거합니다. 이 작업은 아무 것도 반환하지 않으며 지정된 키에 대한 값이 필요한 경우 제거하기 전에 Get Map Value 작업을 사용해야 합니다.

동작 구문:

인수:

논쟁 설명
Map 값을 제거할 맵의 인덱스(변수에 저장됨)
Key 사용할 키(실수 또는 문자열)

예시:

위의 액션 블록 코드는 맵 데이터 구조에서 특정 키의 값을 검색한 다음 값을 확인하여 0보다 작거나 같은지 확인합니다. 그렇다면 키가 맵에서 제거됩니다.
