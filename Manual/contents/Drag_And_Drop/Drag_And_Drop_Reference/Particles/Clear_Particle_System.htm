투명 입자 시스템

이 작업으로 주어진 파티클 시스템을 기본 상태로 지우고 모든 이미터를 제거하고(자동으로 처리되기 때문에 Destroy Particle Emitter 액션을 호출할 필요도 없음) 화면에서 모든 파티클을 지울 수 있습니다. Create Particle System 작업으로 시스템을 만들 때 반환된 대로 지울 시스템의 고유 ID 값을 제공합니다. 이 작업을 사용할 때는 파티클 시스템과 관련된 모든 에미터를 파괴하므로 주의해야 합니다. 즉, 에미터를 사용하는 방의 인스턴스는 해당 에미터가 더 이상 존재하지 않기 때문에 오류가 발생할 수 있으므로 작업이 다음과 같은지 확인하십시오. 해당 이미터에서 새 파티클을 생성하지 않을 때만 호출됩니다(파티클 이미터에 대한 자세한 내용은 파티클 이미터 생성 참조).

참고: 이 작업은 방에서 보이는 입자만 지우지 만 입자 속성을 지우거나 메모리에서 제거하지 않습니다. 이를 위해서는 Destroy Particle Type 작업을 사용해야 합니다.
