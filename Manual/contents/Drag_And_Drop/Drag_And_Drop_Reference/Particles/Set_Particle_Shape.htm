입자 모양 설정

이 작업은 파티클이 그릴 때 사용해야 하는 내장 스프라이트 모양을 설정하는 데 사용할 수 있습니다. Create Particle Type 작업으로 유형을 생성할 때 반환된 모양과 사용할 모양을 설정할 유형의 고유 ID 값을 제공합니다. 입자 모양은 GameMaker 가 특히 입자로 사용하도록 생성된 단순한 흰색 스프라이트이며 다음 형식으로 구성됩니다.

참고: 이 작업을 사용하는 경우 스프라이트 또는 모양만 가질 수 있지만 둘 다 가질 수 없으므로 입자 스프라이트 설정 작업을 사용하면 안 됩니다.
