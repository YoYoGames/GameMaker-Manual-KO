입자 시스템 일시 중지

이 작업은 지정된 입자 시스템을 일시 중지합니다. Create Particle System 작업으로 시스템을 만들 때 반환된 대로 시스템이 일시 중지할 고유 ID 값을 제공합니다. 일시 중지된 입자 시스템은 생성된 입자를 계속 그리지만 더 이상 색상, 위치 또는 알파를 업데이트하지 않으며 시스템을 재개할 때까지 수명을 완료하고 사라지지 않습니다. "상태" 옵션을 사용하여 시스템을 일시 중지하거나 재개하도록 선택할 수 있습니다.

이것은 스트리밍 파티클인 에미터를 일시 중지하지 않으며 에미터 버스트 파티클을 중지하지 않습니다. 에미터는 호출될 때 여전히 파티클을 생성하지만 시스템이 일시 중지되면 파티클은 아무 작업도 하지 않고 생성된 위치에 정확히 그려집니다. 일시 중지 상태를 재설정하거나 입자 시스템 업데이트 를 호출합니다.
