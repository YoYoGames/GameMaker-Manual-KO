파티클 스프라이트 설정

이 작업은 파티클이 그릴 때 사용할 수 있는 스프라이트를 설정하는 데 사용할 수 있습니다. ( Create Particle Type 작업으로 유형을 생성할 때 반환된 대로) 스프라이트를 설정할 유형과 사용할 스프라이트 리소스에 대한 고유 ID 값을 제공합니다. 스프라이트에 하위 이미지가 있는 경우 입자는 이미지 편집기에서 스프라이트에 대해 설정된 속도로 프레임을 통해 애니메이션됩니다. 또한 입자의 색상 값을 설정하면( Set Particle Color 작업을 사용하여) 이 색상이 사용된 스프라이트와 혼합됩니다. 마치 인스턴스에 이미지 혼합을 설정하면 색상이 인스턴스 스프라이트와 혼합되는 것과 같습니다(사용 블렌딩이 없는 경우 흰색).

참고: 이 작업을 사용하는 경우 스프라이트 또는 모양만 가질 수 있고 둘 다 가질 수 없으므로 입자 모양 설정 작업을 사용하면 안 됩니다.
