파티클 시스템 업데이트

이 작업은 주어진 파티클 시스템이 위치, 색상, 알파 등을 업데이트하도록 강제합니다... 업데이트할 시스템의 고유 ID 값을 제공합니다( Create Particle System 작업으로 시스템을 생성할 때 반환됨). 액션을 호출할 때 주어진 시스템은 하나의 게임 프레임 을 업데이트합니다. 즉, 예를 들어 반복 루프 내에서 이 액션을 호출하고 단일 게임 프레임에서 파티클 시스템을 여러 번 업데이트할 수 있습니다(레벨을 원할 때 유용 눈이나 비와 같이 맨 처음에 입자가 있어야 하는 효과를 내기 위해). 또한 입자 시스템이 일시 중지된 경우 이 작업을 사용하여 시스템 의 입자를 수동으로 업데이트할 수 있습니다.
