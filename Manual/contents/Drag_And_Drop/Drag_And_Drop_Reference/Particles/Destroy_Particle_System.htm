파티클 시스템 파괴

이 작업은 주어진 입자 시스템을 "파괴"합니다(즉, 입자 시스템에서 사용하는 메모리를 해제합니다). Create Particle System 작업으로 시스템을 생성할 때 반환된 파괴할 시스템의 고유 ID 값을 제공합니다. 이 작업은 게임에 더 이상 파티클 시스템이 필요하지 않거나 파티클 시스템을 다시 생성하려는 경우(예: Restart Game 을 호출하기 직전) 호출해야 합니다. 파티클 시스템이 Persistent 로 플래그가 지정 되지 않은 경우 이 작업을 호출할 필요가 없습니다. 파티클 시스템(및 관련 이미터)이 룸 끝에서 자동으로 정리되기 때문입니다.

중요한! 이 함수는 또한 생성되고 파괴되는 시스템과 연관되었을 수 있는 모든 이미터를 파괴하고 제거합니다(자세한 내용은 파티클 이미터 생성 참조).
