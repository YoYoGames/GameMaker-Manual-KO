버스트 입자

이 작업을 통해 이미터에 의존하지 않고 언제든지 파티클을 버스트할 수 있습니다. 시스템이 내부에 입자를 생성할 고유 ID 값( Create Particle System 작업으로 시스템을 생성할 때 반환됨)과 생성할 입자 유형의 고유 ID( action Create Particle Type ) 뿐만 아니라 생성할 x/y 위치와 생성할 숫자. 위치는 룸 내의 절대 위치가 될 수도 있고 작업을 호출하는 인스턴스에 대한 상대 위치로 플래그를 지정할 수도 있습니다.

게임 프레임마다 이 작업을 호출하여 이미터의 스트림 효과를 복제하거나 이벤트(예: Destroy Event)에서 한 번 호출하여 단순히 입자를 버스트할 수 있습니다.

동작 구문:

인수:

논쟁 설명
System 사용할 파티클 시스템의 ID 값
Type 버스트할 입자 유형의 ID 값
X 입자를 버스트할 x 위치
Y 그들은 입자를 파열시키는 위치에 있습니다.
Count 버스트할 입자 수

예시:

위의 액션 블록 코드는 마지막 게임 프레임이 업데이트된 이후 인스턴스가 위치를 변경했는지 확인하고 변경했다면 새 위치에서 일부 파티클을 버스트합니다.
