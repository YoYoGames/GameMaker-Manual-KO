효과를 낸다

이 작업은 게임룸에서 그려지는 레이어 위나 아래에 내장 입자 효과를 만듭니다. 드롭다운 목록(아래 이미지 참조)에서 생성할 효과 유형을 선택한 다음 위치(인스턴스 위치에 상대적일 수 있음)를 설정하고 크기(작은, 중간 또는 큰)를 선택합니다. 효과를 위해. 또한 사용할 색상을 설정할 수 있습니다(알파 값은 무시됨). 효과를 방의 모든 레이어 아래에 만들지 아니면 방의 모든 레이어 위에 만들지 결정할 수 있습니다. 비 또는 눈 효과를 사용하기로 선택한 경우 효과가 전체 화면에서 보이도록 설계되었으므로 위치 및 크기 값은 고려되지 않습니다.

인수:

논쟁 설명
Type 생성할 입자 효과 유형(위 이미지 참조)
Where 파티클을 룸 레이어 위 또는 아래에 생성해야 하는지 여부
Size 효과의 크기(소, 중 또는 큼)
X 효과를 만들기 위한 공간 내 x 위치(픽셀 단위)
Y 효과를 만들기 위한 방 내의 y 위치(픽셀 단위)
Colour 효과의 색상(알파는 무시됨)

위의 동작 블록 코드는 마우스 클릭을 확인하고 감지되면 마우스 위치에 작은 효과를 생성합니다.
