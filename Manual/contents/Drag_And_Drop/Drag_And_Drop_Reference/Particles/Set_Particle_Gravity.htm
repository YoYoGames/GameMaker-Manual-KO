입자 중력 설정

이 작업은 주어진 유형의 모든 입자에 대한 중력 효과를 시뮬레이션합니다. 먼저 입자 유형 ID 값( 입자 유형 만들기 작업을 사용하여 입자를 만들 때 사용한 값과 동일해야 함)을 지정한 다음 중력과 방향을 설정할 수 있습니다. 힘은 누적 값이며 각 게임 프레임의 속도 벡터에 추가됩니다. 따라서 1의 중력은 각 게임 프레임의 속도 값에 1을 추가하는 반면 방향 값은 단순히 중력이 "당겨야" 하는 방향입니다. 입자. 힘 과 방향 모두에 대해 음수 값을 사용할 수 있으며 GameMaker 의 해당 방향은 오른쪽으로 0°, 위쪽으로 90°, 왼쪽으로 180°, 아래쪽으로 270°로 계산됩니다.
