입자 알파 설정

이 작업은 입자를 생성할 때의 알파와 수명 동안의 알파를 설정합니다. 먼저 입자 유형 ID 값( 입자 유형 만들기 작업을 사용하여 입자를 만들 때 반환된 값과 같아야 함)을 지정한 다음 사용할 알파 값을 설정할 수 있습니다.

사용할 세 가지 알파 값을 선택할 수 있으며 파티클은 파티클 수명 동안 두 값 사이에서 혼합됩니다. 예를 들어 입자가 알파 값으로 0.5, 1, 0이고 수명이 60 게임 프레임인 경우 생성될 때(프레임 0) 0.5 알파가 되고 다음의 전체 알파를 향해 페이드됩니다. 1(프레임 30), 알파 0(프레임 60)으로 보이지 않도록 페이드온하기 전에.
