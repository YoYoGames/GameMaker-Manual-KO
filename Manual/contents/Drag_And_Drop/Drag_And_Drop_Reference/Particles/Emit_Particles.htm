입자 방출

이 작업은 이미터에서 입자를 만드는 데 사용됩니다. 이미터가 속한 입자 시스템 ID( Create Particle System 작업에서 반환됨)와 실제 이미터 ID( Create Particle Emitter 작업에서 반환됨), 마지막으로 생성할 입자 유형( 작업 입자 유형 만들기 ). 그런 다음 이미터에서 입자를 "버스트" 또는 "스트림"으로 선택하고 생성하려는 입자 수를 지정할 수 있습니다.

파티클 스트리밍을 선택하면 단계 이벤트나 그리기 이벤트와 같은 게임 프레임 이벤트에 이 작업을 배치할 필요가 없습니다. 대신 Create Event(예: 생성 이벤트)에서 한 번만 호출하면 이미터가 매 시간마다 파티클을 스트리밍합니다. 이 작업을 다시 호출하여 중지할 때까지 게임 프레임을 유지합니다(카운트 값을 0으로 설정). 파티클 버스트를 선택하면 선택한 이벤트에서 이것을 호출할 수 있으며 파티클은 한 번 생성됩니다.
