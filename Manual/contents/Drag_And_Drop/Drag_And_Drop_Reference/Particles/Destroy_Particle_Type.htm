입자 유형 파괴

이 작업은 Create Particle Type 작업에 의해 생성된 지정된 입자 유형을 "파괴"합니다(즉: 입자 유형에서 사용하는 메모리 확보). 이 작업은 게임에서 더 이상 파티클 유형이 필요하지 않거나 파티클 유형을 다시 생성하려는 경우(예: Game Restart 를 호출하기 직전) 호출해야 합니다.

인수:

논쟁 설명
Type 파괴할 유형의 ID 값

위의 액션 블록 코드는 "Space" 키가 눌렸는지 확인하고 감지되면 파티클 시스템과 주어진 전역 변수에 정의된 다른 파티클 유형을 제거한 다음 게임을 다시 시작합니다.
