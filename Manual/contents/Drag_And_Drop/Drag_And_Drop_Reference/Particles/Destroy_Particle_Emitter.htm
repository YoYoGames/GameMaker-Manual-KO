파티클 이미터 파괴

이 작업은 주어진 파티클 이미터를 "파괴"합니다. 즉, 파티클 이미터가 사용하는 메모리를 해제합니다. 이미터가 속한 입자 시스템 ID( Create Particle System 작업에서 반환됨)와 실제 이미터 ID( Create Particle Emitter 작업에서 반환됨)를 제공합니다. 이 작업은 게임에서 파티클 이미터가 더 이상 필요하지 않거나 파티클 이미터를 다시 생성하려는 경우(예: Game Restart 를 호출하기 직전) 호출해야 합니다. 이미터가 할당된 파티클 시스템에 영구 플래그가 지정 되지 않은 경우 이 액션을 호출할 필요가 없습니다. 이미터는 파티클 시스템과 함께 룸 끝에서 자동으로 정리되기 때문입니다.
