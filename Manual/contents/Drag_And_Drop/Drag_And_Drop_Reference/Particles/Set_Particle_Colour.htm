입자 색상 설정

이 작업은 스프라이트가 생성되고 수명이 다할 때 사용할 색상을 설정합니다. 먼저 입자 유형 ID 값(입자 유형 생성 작업을 사용하여 입자를 생성할 때 반환된 값과 동일해야 함)을 지정한 다음 사용할 색상을 설정할 수 있습니다.

사용할 세 가지 색상을 선택할 수 있으며 스프라이트는 파티클의 수명 동안 두 색상을 혼합합니다. 예를 들어 입자가 빨간색, 녹색 및 노란색 색상이고 수명이 60 게임 프레임인 경우 생성될 때(프레임 0) 빨간색으로 착색되고 수명의 절반이 지나면 녹색으로 흐려집니다( 프레임 30), 수명이 다하면 노란색으로 변하기 전에(프레임 60). 입자 모양 대신 사용자 정의 스프라이트를 사용하는 경우 흰색 이외의 색상이 사용된 스프라이트와 혼합됩니다.

참고: 색상의 알파 구성요소는 사용 되지 않고 색상 선택기의 RGB 구성요소만 사용됩니다. 알파를 설정하려면 Set Particle Alpha 액션을 사용하십시오.
