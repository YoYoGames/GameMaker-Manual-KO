입자 속도 설정

이 동작은 정적이거나 시간이 지남에 따라 증가/감소하도록 설정할 수 있는 입자의 이동 속도를 설정하는 데 사용됩니다. 크기를 설정할 유형의 고유 ID 값( Create Particle Type 작업으로 유형을 생성할 때 반환됨)을 제공한 다음 초기 시작 속도 범위를 제공한 다음 증가량 및 마지막으로 "흔들기"를 제공합니다. 파티클이 생성되는 속도는 게임 프레임당 픽셀로 계산되므로 최소 설정이 2이고 최대 설정이 5이면 게임 프레임당 2~5픽셀 사이에서 움직이는 파티클이 방출됩니다.

입자 속도를 높이는 옵션을 사용하면 입자 수명의 각 게임 프레임에 기본 속도 값에 추가될 값을 입력할 수 있습니다. 따라서 이 값을 예를 들어 0.1로 설정하면 각 게임 프레임의 속도에 0.1이 추가됩니다. 속도가 0 아래로 떨어지지는 않지만 여기에서도 음수 값을 사용하여 입자 속도를 늦출 수 있습니다. 허용되는 최소 속도 감소 는 -0.01이고 최소 증가 는 0.01입니다.

마지막으로 "흔들기" 옵션을 설정할 수 있습니다. 이것은 0과 지정한 값 사이에서 무작위로 생성된 다음 입자 수명의 각 단계에서 속도에 더하거나 빼서 그려지는 이미지의 속도 변동을 일으키는 값입니다. 분명히 큰 값은 작은 값보다 더 뚜렷하며 이 값은 최대 범위가 -20에서 20 사이인 음수일 수도 있습니다.

인수:

논쟁 설명
Type 크기를 설정할 입자 유형의 고유 ID 값
Min Speed 입자 유형의 최소 속도(기본값 0.1)
Max Speed 입자 유형의 최대 속도(기본값 1)
Increase 매 게임 프레임마다 파티클 유형 속도에 추가할 값(음수일 수 있음, 최소 +/-0.01)
Wiggle 단계당 입자 속도에서 무작위로 더하거나 뺄 최대값(+/-20)입니다.
