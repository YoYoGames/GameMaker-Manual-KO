입자 수명 설정

이 작업은 주어진 입자 유형의 수명을 설정합니다. 먼저 입자 유형 ID 값( 입자 유형 만들기 작업을 사용하여 입자를 만들 때 반환된 값)을 지정한 다음 게임 프레임에서 입자의 최소 및 최대 수명을 설정할 수 있습니다. 예를 들어 게임에 초당 프레임 수가 60인 경우 여기에서 두 수명 값을 모두 60으로 설정하면 입자가 화면에서 1초 동안 지속되어 사라지게 됩니다. 두 개의 다른 값을 사용하면 생성된 각 입자의 수명이 지정된 최소값과 최대값 내에서 무작위로 설정됩니다. 모든 값은 양의 정수여야 합니다.

중요한! 매우 긴 수명을 설정한 다음 수백 개의 입자를 방출하면 덜 강력한 시스템과 모바일 장치에서 문제가 발생할 수 있습니다. 시각적 효과를 최대화하지만 성능 비용을 최소화하기 위해 입자 수명을 설정할 때 주의하고 수명이 매우 긴 입자만 아껴서 사용하십시오.
