입자 방향 설정

이 작업은 입자 방향(입자 이미지가 그려지는 각도)을 설정하는 데 사용됩니다. 이 각도는 정적이거나 시간이 지남에 따라 증가/감소하도록 설정할 수 있습니다. 크기를 설정할 유형의 고유 ID 값( Create Particle Type 작업으로 유형을 만들 때 반환됨)을 제공한 다음 초기 시작 방향 범위를 지정한 다음 증가량을 지정하고 마지막으로 "흔들기"를 지정합니다. 파티클이 생성되는 방향은 0이 오른쪽, 90이 위쪽, 180이 왼쪽, 270이 아래쪽이 되도록 계산되므로 최소 설정이 0이고 최대 설정이 360이면 파티클이 방출되며 각 위치는 1만큼 회전합니다. 약 360°의 임의 양(입자 방향에는 영향을 미치지 않음 ).

입자 방향을 증가시키는 옵션을 사용하면 입자 수명의 각 게임 프레임에 기본 방향 각도에 추가될 값을 입력할 수 있습니다. 따라서 이 값을 예를 들어 0.1로 설정하면 각 게임 프레임의 이미지 각도에 0.1이 추가됩니다. 여기서도 음수 값을 사용하여 입자가 반대 방향으로 회전하도록 할 수 있으므로 양수 값은 입자를 시계 반대 방향으로 회전하고 음수는 시계 방향으로 회전합니다. 허용되는 최소 방향 감소 는 -0.01이고 최소 증가 는 0.01입니다.

입자 방향에 대해 "흔들기" 옵션을 설정할 수도 있습니다. 이 값은 0과 사용자가 지정한 값 사이에서 무작위로 생성된 다음 입자 수명의 각 단계에서 입자 이미지 각도에 더하거나 빼서 그려진 이미지가 회전에서 변동하도록 하는 값입니다. 분명히 큰 값은 작은 값보다 더 뚜렷하며 이 값은 최대 범위가 -20에서 20 사이인 음수일 수도 있습니다.

마지막으로 방향을 입자 방향에 상대적 인지 여부를 선택할 수 있습니다. 즉, 입자에 방향(및 속도)이 설정되어 있는 동안 입자 이미지는 해당 벡터를 기준으로 방향이 지정됩니다. 입자 속도를 줄이도록 설정하고 입자 수명 동안 0에 도달하면 속도가 없으면 방향이 기본값인 0°로 설정되므로 상대 방향으로 인해 입자 이미지가 "점프"됩니다. 다른 방향 각도로.

인수:

논쟁 설명
Type 방향을 설정할 입자 유형의 고유 ID 값
Min Angle 입자 유형의 최소 방향(-360~+360, 기본값 0)
Max Angle 입자 유형의 최대 방향(-360에서 +360, 기본값 360)
Increase 매 게임 프레임마다 파티클 유형 방향에 추가할 값(음수일 수 있음, 최소 +/-0.01)
Wiggle 단계당 입자 방향에서 임의로 더하거나 뺄 최대값(+/-20)입니다.
Relative To Direction 입자의 모션 방향을 기준으로 입자 이미지 각도를 설정할지(활성화됨) 또는 설정하지 않을지(비활성화됨) 여부입니다.
