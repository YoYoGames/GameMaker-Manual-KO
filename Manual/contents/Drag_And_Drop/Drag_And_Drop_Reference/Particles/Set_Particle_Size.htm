입자 크기 설정

이 작업은 입자 크기를 설정하는 데 사용되며 정적이거나 시간이 지남에 따라 증가/감소하도록 설정할 수 있습니다. 크기를 설정할 유형의 고유 ID 값( Create Particle Type 작업으로 유형을 생성할 때 반환됨)을 제공한 다음 초기 시작 크기 범위를 제공한 다음 증가량 및 마지막으로 "흔들기"를 제공합니다. 입자가 생성되는 크기는 스칼라 값입니다. 여기서 1은 기본 입자 이미지와 동일한 크기이고 0.5는 기본 이미지 크기의 절반이고 값 2는 두 배입니다. 최소 및 최대 크기가 동일하지 않은 경우 생성된 입자의 초기 크기는 둘 사이의 임의 값이 됩니다.

입자 크기를 늘리는 옵션을 사용하면 입자 수명의 각 게임 프레임에 기본 스칼라 값에 추가될 값을 입력할 수 있습니다. 따라서 이 값을 예를 들어 0.1로 설정하면 각 게임 프레임의 스케일 값에 0.1이 추가됩니다. 여기에서도 음수 값을 사용하여 입자 이미지를 축소할 수 있습니다. 허용되는 최소 스케일 감소 는 -0.01이고 최소 증가 는 0.01입니다.

마지막으로 "흔들기" 옵션을 설정할 수 있습니다. 이것은 0과 사용자가 지정한 값 사이에서 무작위로 생성된 다음 입자 수명의 각 단계에서 크기에 더하거나 빼서 그려지는 이미지의 크기가 변동하는 값입니다. 분명히 큰 값은 작은 값보다 더 뚜렷하며 이 값은 최대 범위가 -20에서 20 사이인 음수일 수도 있습니다.

논쟁 설명
Type 크기를 설정할 입자 유형의 고유 ID 값
Min Size 입자 유형의 최소 스케일(기본값 0.1)
Max Size 입자 유형의 최대 배율(기본값 1)
Increase 매 게임 프레임마다 입자 유형 크기에 추가할 값(음수일 수 있음, 최소 +/-0.01)
Wiggle 단계당 입자 크기에서 임의로 더하거나 뺄 최대값(+/-20)입니다.
