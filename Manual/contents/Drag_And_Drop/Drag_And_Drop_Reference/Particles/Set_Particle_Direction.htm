입자 방향 설정

이 동작은 입자 방향을 설정하는 데 사용되며, 정적으로 설정되거나 시간이 지남에 따라 증가/감소하도록 설정할 수 있습니다. 크기를 설정할 유형의 고유 ID 값( Create Particle Type 작업으로 유형을 생성할 때 반환됨)을 제공한 다음 초기 시작 방향 범위를 제공한 다음 증가량 및 마지막으로 "흔들기"를 제공합니다. 파티클이 생성되는 방향은 0이 오른쪽, 90이 위쪽, 180이 왼쪽, 270이 아래쪽으로 계산되므로 최소 설정이 -45이고 최대 설정이 45이면 오른쪽으로만 이동하는 파티클이 방출됩니다. 예시.

입자 방향을 증가시키는 옵션을 사용하면 입자 수명의 각 게임 프레임에 기본 방향 값에 추가될 값을 입력할 수 있습니다. 따라서 이 값을 예를 들어 0.1로 설정하면 각 게임 프레임의 방향에 0.1이 추가됩니다. 여기서도 음수 값을 사용하여 입자가 반대 방향으로 아래로 이동하도록 할 수 있으므로 양수 값은 입자를 시계 반대 방향으로 나선형으로 만들고 음수는 시계 방향으로 나선형으로 만듭니다. 허용되는 최소 방향 감소 는 -0.01이고 최소 증가 는 0.01입니다.

마지막으로 "흔들기" 옵션을 설정할 수 있습니다. 이것은 0과 사용자가 지정한 값 사이에서 무작위로 생성된 다음 입자 수명의 각 단계에서 방향에 더하거나 빼서 그려지는 이미지의 방향이 변동하는 값입니다. 분명히 큰 값은 작은 값보다 더 뚜렷하며 이 값은 최대 범위가 -20에서 20 사이인 음수일 수도 있습니다.

논쟁 설명
Type 방향을 설정할 입자 유형의 고유 ID 값
Min Dir 입자 유형의 최소 방향(-360~+360, 기본값 0)
Max Dir 입자 유형의 최대 방향(-360에서 +360, 기본값 360)
Increase 매 게임 프레임마다 파티클 유형 방향에 추가할 값(음수일 수 있음, 최소 +/-0.01)
Wiggle 단계당 입자 방향에서 임의로 더하거나 뺄 최대값(+/-20)입니다.
