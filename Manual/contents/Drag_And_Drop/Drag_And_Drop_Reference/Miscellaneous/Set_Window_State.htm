창 상태 설정

이 작업으로 게임 창의 상태를 전체 화면 또는 창으로 설정할 수 있습니다. 여기에는 세 가지 옵션이 있습니다.

  • 토글 - 이 옵션을 선택하면 액션을 호출할 때마다 전체 화면과 창 상태 사이에서 창을 토글합니다.
  • 전체 화면 - 창 상태에서 창을 전체 화면으로 만듭니다.
  • 창 - 전체 화면인 경우 창을 창 상태로 되돌립니다.

전체 화면 상태에서 게임이 어떻게 보이는지는 게임 옵션 - 대상 플랫폼에 대한 그래픽 설정에 따라 달라집니다. 게임에 다른 종횡비와 해상도를 조정하는 메커니즘이 없는 경우 늘어나게 될 수 있기 때문입니다(게임의 경우 옵션이 전체 크기 로 설정되어 있거나 검은색 막대를 추가할 수 있습니다(게임 옵션이 가로 세로 비율 유지 로 설정된 경우). 일반적으로 다른 해상도를 처리할 장소가 없다면 컴파일하려는 대상 플랫폼에 대해 Keep Aspect Ratio 를 사용해야 합니다.
