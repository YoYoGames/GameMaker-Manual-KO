이 작업을 사용하여 "lives"에 대해 생성된 인스턴스 변수의 값을 검색할 수 있습니다. 반환된 "lives" 값(임시 로컬 변수로 플래그 지정 가능)을 저장하기 위해 대상 변수를 제공하지만 변수가 이전에 설정되지 않은 경우( Set Lives 사용) 변수가 자동으로 생성되고 작업은 0을 반환합니다(먼저 설정하지 않고 액세스하려고 하면 오류가 발생하는 일반 인스턴스 변수와 달리).
