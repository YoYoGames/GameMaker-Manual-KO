점수 설정

이 작업으로 점수에 대한 인스턴스 변수를 생성하거나 설정할 수 있습니다. 점수 변수를 설정하거나 점수 변수에 추가할 값을 제공합니다( 상대 확인란을 선택하면 값이 현재 "점수" 값에 추가됨). 일반 인스턴스 변수와 달리 "점수"에 대한 변수 이름은 내부적으로 저장되므로 점수 값을 검색하려면 Get Score 작업을 사용해야 합니다.
