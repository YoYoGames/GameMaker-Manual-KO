GML 시각적 참조

매뉴얼의 이 섹션은 GML Visual 을 사용하여 게임을 프로그래밍할 때 사용할 수 있는 모든 다양한 작업을 나열합니다. 아래에서 GML Visual Toolbox 개체에서 사용할 수 있는 각기 다른 작업 라이브러리 를 찾을 수 있으며 각 라이브러리 섹션 내에서 사용 예와 함께 개별 작업이 설명됩니다.

  • 일반적인 작업
  • 인스턴스 작업
  • 시퀀스 작업
  • 마우스 및 키보드 동작
  • 게임패드 작업
  • 이동 동작
  • 충돌 동작
  • 그리기 작업
  • 타일 작업
  • 오디오 작업
  • 루프 작업
  • 스위치 동작
  • 데이터 구조 작업
  • 버퍼 작업
  • 파일 작업
  • 데이터 유형 작업
  • 임의 작업
  • 카메라 액션
  • 룸 액션
  • 경로 작업
  • 타임라인 작업
  • 게임 액션
  • 인스턴스 변수 작업
  • 기타 작업
  • 입자 동작
  • 시간 소스 작업
  • 멀티플레이어 액션 롤백

위의 작업 라이브러리 외에도 즐겨찾기 라이브러리도 있습니다. 이 라이브러리는 작업을 드래그하여 가장 편리하게 사용하는 모든 항목을 보관할 수 있는 도구 상자의 특수 섹션이며, 최근에 사용한 라이브러리도 있습니다. 마지막으로 사용한 5가지 작업을 유지합니다.
