롤백 게임 시작

이 작업은 모든 플레이어가 참여하기 전에 현재 멀티플레이어 게임을 강제로 시작합니다. 먼저 롤백 게임 만들기 를 사용하여 멀티플레이어 게임을 만들어야 합니다.

시스템은 일반적으로 자동으로 게임을 시작하기 전에 모든 플레이어가 참여할 때까지 기다립니다. 이 작업은 모든 플레이어가 참여하기 전에 게임을 일찍 시작하는 데 사용할 수 있습니다.

모든 플레이어가 참가한 후에도 수동 시작 사용 을 호출하여 롤백이 이 호출을 기다리도록 할 수 있습니다.

이 작업을 실행하면 롤백 시작 이벤트가 트리거됩니다.
