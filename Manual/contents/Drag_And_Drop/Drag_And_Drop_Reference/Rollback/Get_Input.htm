입력 받기(롤백)

이 작업은 플레이어의 입력 값이 포함된 구조체를 반환합니다. 입력이 반환될 플레이어의 ID를 지정하는 선택적 인수가 필요합니다.

멀티 플레이에서 입력을 확인하려면 If Key Down(롤백), If Key Pressed(롤백) 및 If Key Up(롤백) 을 사용하는 것이 좋습니다.

플레이어 정의(롤백) 에 의해 생성된 플레이어 인스턴스에서 사용되는 경우 플레이어 ID를 지정할 필요가 없으므로 player_id 만 지정할 수 있습니다.

다른 인스턴스에서 사용하거나 수동으로 생성된 플레이어 인스턴스에서 사용하는 경우 플레이어 ID를 지정해야 합니다.

반환되는 입력과 변경 방법에 대한 정보는 입력 정의 를 참조하십시오.
