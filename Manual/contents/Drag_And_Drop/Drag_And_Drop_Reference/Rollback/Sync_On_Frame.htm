프레임에 동기화(롤백)

이 작업은 모든 플레이어가 현재 프레임에서 동기화되었는지 확인합니다. 즉, 상태가 예측이 아니라 플레이어로부터 받은 실제 데이터를 기반으로 합니다. 이 경우 rollback_confirmed_frame>true 를 반환합니다.

플레이어가 현재 프레임에서 동기화되지 않은 경우 false 를 반환하고 모든 플레이어를 동기화하려고 시도합니다.

이 작업으로 플레이어를 동기화하려고 하면 정지될 수 있습니다. 이 작업이 Step 이벤트에서 계속 호출되지 않도록 해야 합니다.

대신 토글로 사용하십시오. 페이지 하단의 예를 참조하십시오.

변수 모든 플레이어가 동기화되면 자동으로 업데이트됩니다.
