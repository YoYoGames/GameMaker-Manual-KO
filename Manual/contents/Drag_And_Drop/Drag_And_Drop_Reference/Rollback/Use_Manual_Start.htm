수동 시작 사용(롤백)

기본적으로 멀티플레이어 게임은 모든 플레이어가 참가하면 자동으로 시작됩니다.

이 작업을 호출하면 자동 시작 동작이 비활성화되므로 게임은 사용자가 수동으로 시작할 때 까지 기다립니다.

사용되는 경우 이 작업은 조인 / 만들기 작업보다 먼저 실행되어야 합니다.
