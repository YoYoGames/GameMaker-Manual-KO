롤백 정보 가져오기

이 함수는 플레이어에 대한 정보가 포함된 구조체를 반환합니다. 플레이어 인스턴스를 수동으로 관리할 때만 사용해야 합니다. 플레이어 정의 를 사용하는 경우 각 플레이어 인스턴스는 자동으로 이 정보를 변수 로 받습니다.

반환된 구조체의 내용에 대한 정보는 rollback_get_info 를 참조하십시오.
