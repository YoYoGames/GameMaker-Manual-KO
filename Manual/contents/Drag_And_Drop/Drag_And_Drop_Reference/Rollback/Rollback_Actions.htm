멀티플레이어 액션 롤백

롤백 동작은 GameMaker의 롤백 시스템을 사용하여 멀티플레이어 게임을 만드는 데 사용됩니다.

소개는 다음을 참조하십시오. 롤백 멀티플레이어

롤백 시스템과 함께 사용되는 변수 목록은 롤백 변수 를 참조하십시오.

  • 게임 나가기
  • 입력 정의
  • 입력 프레임 지연 정의
  • 모의 입력 정의
  • 플레이어 정의
  • 디스플레이 이벤트
  • 정보를 얻다
  • 입력 받기
  • 키 다운(롤백)인 경우
  • 키를 누른 경우(롤백)
  • 키업인 경우(롤백)

  • 프레임에 동기화
