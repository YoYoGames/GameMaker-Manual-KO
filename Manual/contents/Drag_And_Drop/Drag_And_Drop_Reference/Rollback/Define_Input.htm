입력 정의(롤백)

이 기능은 모든 플레이어에 대한 사용자 정의 입력을 정의하는 데 사용됩니다. 입력/값 쌍을 취합니다. 여기서 입력은 입력의 이름이고 값은 할당된 입력 상수입니다.

사용자 지정 입력을 할당하는 방법을 알아보려면 사용자 지정 컨트롤 을 참조하십시오.

이 함수를 사용하는 경우 조인 / 생성 작업 전에 실행해야 합니다.

동작 구문:

인수:

논쟁 설명
Input 정의할 입력의 이름
Value 할당할 입력 상수( 사용자 정의 컨트롤 참조)

예시:

위의 작업은 모든 플레이어에 대해 fire, left, right, up 및 down 입력을 정의합니다.
