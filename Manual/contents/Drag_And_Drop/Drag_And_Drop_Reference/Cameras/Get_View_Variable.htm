보기 변수 가져오기

이 작업을 통해 주어진 뷰 포트 와 관련된 특정 내장 변수의 값을 얻을 수 있습니다. 검색할 뷰 포트 변수를 선택한 다음 대상에 뷰 포트를 제공하고(0에서 7까지) 마지막으로 반환된 값을 보유할 대상 변수(이는 임시 로컬 변수로 플래그 지정될 수 있음)를 제공합니다. 뷰 포트는 주어진 카메라 뷰(룸 편집기에서 설정한 대로)를 그리는 데 사용되는 화면 영역이며 이러한 방식으로 얻을 수 있는 다양한 변수는 아래 표에 요약되어 있습니다.

변하기 쉬운 설명
Camera 반환된 값은 포트에 할당된 카메라의 고유 ID 값입니다(자세한 내용은 여기 참조).
Visibility 다음 값을 반환합니다. true 또는 false 어디 true 뷰 포트가 보이는 것입니다(즉, 화면에 그려지는 것), 그리고 false 아무것도 그려지지 않는다는 뜻입니다.
View Port X Coordinate 이것은 게임 창 내 뷰 포트의 X 위치입니다.
View Port Y Coordinate 이것은 게임 창 내 뷰 포트의 Y 위치입니다.
View Port Width 이것은 뷰 포트의 너비(픽셀 단위)입니다.
View Port Height 이것은 뷰 포트의 높이(픽셀 단위)입니다.
View Port Surface ID 이것은 뷰 포트에 할당된 표면에 대한 고유 ID 값을 반환하거나 할당된 표면이 없는 경우 -1을 반환합니다.

예시:

위의 액션 블록 코드는 뷰 포트 [0]에 할당된 현재 카메라 ID를 가져온 다음 전역 변수에 저장된 것과 동일한지 확인합니다. 그렇지 않은 경우 카메라가 새 카메라로 설정됩니다.
