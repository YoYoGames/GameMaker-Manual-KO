보기 변수 설정

이 작업을 통해 뷰 포트 가 방의 카메라 뷰를 표시하는 방법과 관련된 특정 내장 변수를 설정할 수 있습니다. 편집할 뷰 포트 변수를 선택한 다음 대상(0에서 7까지)에 뷰 포트를 지정하고 마지막으로 변수에 대한 새 값을 지정합니다. 뷰 포트는 주어진 카메라 뷰(룸 편집기에서 설정한 대로)를 그리는 데 사용되는 화면 영역이며 이러한 방식으로 설정할 수 있는 다양한 변수는 아래 표에 요약되어 있습니다.

변하기 쉬운 설명
Camera 다른 Camera View를 보여주도록 뷰포트를 설정하기 위한 것입니다. 변경할 카메라의 고유 ID 값을 제공해야 합니다(자세한 내용은 여기 참조).
Visibility 이것은 뷰 포트에 대한 가시성 토글이며 다음 값을 기대합니다. true 또는 false 어디에 설정 true 뷰 포트를 보이게 하고(즉, 화면에 그려짐) false 화면에 아무 것도 그리지 않습니다.
View Port X Coordinate 이것은 게임 창 내 뷰 포트의 X 위치입니다. 예를 들어 분할 화면 효과를 생성하기 위해 둘 이상의 뷰 포트가 활성화된 경우에만 실제로 필요하지만 하나의 포트만 활성화된 경우에 사용하면 예기치 않은 결과를 얻게 됩니다.
View Port Y Coordinate 이것은 게임 창 내 뷰 포트의 Y 위치입니다. 예를 들어 분할 화면 효과를 생성하기 위해 둘 이상의 뷰 포트가 활성화된 경우에만 실제로 필요하지만 하나의 포트만 활성화된 경우에 사용하면 예기치 않은 결과를 얻게 됩니다.
View Port Width 이것은 뷰 포트의 너비(픽셀 단위)입니다. 보기 포트의 너비(또는 둘 이상이 활성화된 경우 결합된 보기 포트)는 게임 시작 시 게임 창 또는 배경 캔버스의 너비를 정의하므로 게임이 시작된 후 이 값을 변경해 도 눈에 보이는 효과가 없습니다. GML 함수와 함께 호출되지 않는 한 게임 창 크기 window_set_size(). 또한 카메라에 할당된 것보다 더 크거나 작은 포트 크기가 있는 경우 카메라 보기가 그에 맞게 축소 또는 확대됩니다.
View Port Height 이것은 뷰 포트의 높이(픽셀 단위)입니다. 뷰 포트(또는 둘 이상이 활성화된 경우 결합된 뷰 포트)의 높이는 게임 시작 시 게임 창 또는 배경 캔버스의 높이를 정의하므로 게임이 시작된 후 이 값을 변경해 도 눈에 보이는 효과가 없습니다. GML 함수와 함께 호출되지 않는 한 게임 창 크기 window_set_size(). 또한 카메라 보기에 할당된 것보다 더 크거나 작은 포트 크기가 있는 경우 카메라 보기가 그에 맞게 축소 또는 확대됩니다.
View Port Surface ID 이 변수를 사용하면 주어진 뷰 포트의 내용이 표면에 그려지도록 설정할 수 있으며, 제공한 값은 뷰 포트를 그리는 데 사용할 표면의 값입니다. 자세한 내용은 여기 를 참조하십시오.
