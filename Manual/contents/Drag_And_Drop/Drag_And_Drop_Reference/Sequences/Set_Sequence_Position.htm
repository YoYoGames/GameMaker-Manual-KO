시퀀스 위치 설정

이 작업으로 방에서 주어진 시퀀스 요소의 위치를 설정할 수 있습니다. 시퀀스 요소 ID( Create Sequence 작업에서 반환됨)를 제공한 다음 새 X 및 Y 위치를 제공합니다. "상대적" 확인란을 선택하면 X 및 Y에 지정된 값이 현재 시퀀스 위치에 추가되고, 그렇지 않으면 절대 위치가 됩니다.

동작 구문:

인수:

논쟁 설명
Variable 위치를 설정할 시퀀스 요소의 ID입니다.
X 시퀀스에 대한 공간의 새 x 위치입니다.
Y 시퀀스에 대한 공간의 새 y 위치입니다.

예시:

위의 액션 블록 코드는 "변수에 저장된 시퀀스 요소 ID의 현재 위치를 검색합니다.my_seq ". 그런 다음 이 값을 사용하여 시퀀스의 위치를 설정하고 오른쪽으로 10픽셀 이동합니다.
