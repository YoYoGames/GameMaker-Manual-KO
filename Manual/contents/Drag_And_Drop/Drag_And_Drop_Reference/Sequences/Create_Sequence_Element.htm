시퀀스 생성

이 작업을 통해 자산 브라우저에서 모든 시퀀스의 시퀀스 요소를 생성할 수 있습니다. 시퀀스의 이름과 시퀀스가 생성될 공간 내의 x/y 위치와 생성될 레이어를 제공합니다(이 레이어는 자산 레이어 여야 함). "상대적" 플래그를 선택하여 작업 코드 블록을 실행하는 인스턴스를 기준으로 시퀀스를 배치할 수 있습니다. 이런 방식으로 게임에서 시퀀스를 생성할 때마다 고유한 요소 ID 값이 제공되며, 이를 통해 이후 작업에서 시퀀스를 식별할 수 있습니다. 따라서 이 작업을 통해 이 ID 값을 보유할 "대상" 변수를 정의할 수 있습니다( 아래 예와 같이). 대상은 임시(로컬) 변수로 플래그가 지정될 수 있습니다. 이 변수는 작업에 대해 생성되고 작업이 있는 이벤트에 대한 모든 후속 작업에서 사용할 수 있습니다.

논쟁 설명
Sequence 요소를 생성할 시퀀스의 이름입니다.
x 시퀀스를 생성하기 위한 방의 x 위치(호출 인스턴스에 상대적일 수 있음).
y 시퀀스를 생성하기 위한 룸의 y 위치(호출 인스턴스에 상대적일 수 있음).
layer 시퀀스를 생성할 자산 레이어의 이름(문자열).
Target 반환 값을 대상으로 하는 변수입니다.

예시:

위의 액션 블록 코드는 새로운 시퀀스를 생성하고 고유한 요소 ID 값을 변수에 저장합니다. 그런 다음 이 변수는 Pause Sequence (시퀀스 일시 중지) 작업을 사용하여 시퀀스를 일시 중지하는 데 사용됩니다.
