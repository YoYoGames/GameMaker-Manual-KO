시퀀스 헤드 설정

이 작업을 사용하면 방에서 지정된 시퀀스 요소의 재생 헤드 위치를 지정된 프레임으로 설정할 수 있습니다. 시퀀스 요소 ID( 시퀀스 만들기 작업에서 반환됨)와 프레임의 새 재생 헤드 위치를 제공합니다. 재생 헤드에 대한 자세한 내용 은 도프 시트 사용 을 참조하십시오.

위치에 있음을 유의하십시오. 프레임 총 프레임 수보다 큰(또는 0보다 작은) 값을 설정하면 실제 최종 재생 헤드 위치는 다음 규칙에 따라 선택한 시퀀스 재생 유형에 따라 달라집니다.

  • 반복 없음: 재생 헤드가 루프의 끝(또는 지정된 값이 음수인 경우 시작)으로 고정됩니다.
  • 루핑: 재생 헤드는 시퀀스의 시작 부분과 추가 프레임으로 되돌아갑니다. 예: 30개 프레임 시퀀스의 재생 헤드를 프레임 45의 위치로 설정하면 실제 재생 헤드 위치는 15가 됩니다. 음수 값이면 시퀀스 재생 헤드가 반대 방향으로 줄 바꿈됩니다.
  • 핑퐁 루핑: 재생 헤드는 끝까지 진행한 다음 재생 방향을 반대로 하여 총 프레임 길이를 초과하는 추가 프레임 수로 되돌아갑니다. 예: 시퀀스가 30개 프레임이고 재생 헤드를 40개로 설정한 경우, 최종 재생 헤드 위치는 20이고 방향은 반대입니다(0으로 내려감). 다음을 수행해야 합니다. 절대 정의되지 않은 결과가 있을 수 있으므로 시퀀스의 핑퐁 재생에 음수 값을 지정합니다.

이러한 방식으로 헤드 위치를 설정해도 재생이 중지되지 않으며 일시 중지되지 않는 한 시퀀스는 새 재생 헤드 위치에서 계속됩니다.

동작 구문:

인수:

논쟁 설명
Variable 헤드 위치를 변경할 시퀀스 요소의 ID입니다.
Head (frames) 시퀀스 요소의 새 헤드 위치(프레임 내)입니다.

예시:

위의 액션 블록은 "변수에 저장된 시퀀스 요소 ID의 현재 헤드 위치를 검색합니다.seq_id ". 그런 다음 해당 값을 사용하여 동일한 시퀀스 요소의 헤드 위치를 설정하여 4프레임씩 전진시킵니다.
