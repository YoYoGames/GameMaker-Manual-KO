경로를 따라 위치 설정

이 작업은 경로를 따라 인스턴스의 위치를 설정합니다. 값은 0 - 1에서 정규화되므로 예를 들어 0.5로 설정하면 인스턴스가 경로의 정확히 중간으로 이동됩니다. 이 작업이 효과를 발휘하려면 인스턴스에 경로를 할당하려면 경로 따라 시작 작업을 호출해야 합니다.

Position 인스턴스를 배치할 경로를 따라 위치(0 - 1)

예시:

위의 액션 블록 코드는 경로를 따라 인스턴스를 시작한 다음 0과 1 사이의 임의의 10진수 값을 생성합니다. 이 값은 임시 로컬 변수에 저장되고 경로를 따라 인스턴스의 위치를 설정하는 데 사용됩니다.
