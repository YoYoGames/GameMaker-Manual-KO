경로를 따라 위치 가져오기

이 작업은 경로를 따라 인스턴스의 위치를 가져옵니다. 반환된 값은 0 - 1에서 정규화되므로 예를 들어 0.5와 같은 결과를 얻은 경우 인스턴스가 정확히 경로의 중간에 있습니다. 이 작업이 0 이외의 값을 반환하려면 경로를 따라 시작 작업을 호출해야 인스턴스에 경로를 할당할 수 있습니다. 반환된 값은 제공한 대상 변수에 저장되거나 변수에 임시 로컬 변수를 생성하고 값을 할당합니다.
