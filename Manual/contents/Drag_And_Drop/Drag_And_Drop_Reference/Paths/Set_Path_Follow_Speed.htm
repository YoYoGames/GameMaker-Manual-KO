경로 추적 속도 설정

이 작업은 인스턴스가 경로를 따라야 하는 속도를 게임 프레임당 픽셀 단위로 설정합니다(효과를 내기 위해 인스턴스에 경로를 할당하려면 경로를 따라 시작 작업을 호출해야 합니다). 음수 값을 사용하여 인스턴스가 경로를 반대로 따라야 함을 나타낼 수 있으며 여기에서 설정한 속도는 각 경로 지점에 대해 경로 편집기에서 설정한 속도 수정자로 수정할 수 있으므로 경로 속도를 설정하면 2로 변경하고 경로에서 포인트 중 하나는 200%의 속도 수정자를 가집니다. 해당 지점에서 게임 프레임 속도당 실제 픽셀 속도는 4가 됩니다.

위의 액션 블록 코드는 경로를 따라 인스턴스를 시작한 다음 이 값을 사용하여 인스턴스가 경로를 따라 이동하는 속도를 설정하기 전에 2와 4 사이의 임의 값을 생성합니다.
