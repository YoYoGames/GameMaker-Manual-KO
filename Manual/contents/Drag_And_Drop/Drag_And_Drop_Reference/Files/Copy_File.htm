파일 복사

이 작업으로 게임에서 저장했거나 포함된 파일의 일부로 번들된 모든 파일을 복사할 수 있습니다. 복사할 파일 이름(문자열으로)을 지정한 다음 원본 파일의 복사본으로 만들 파일 이름(문자열 및 파일 확장명 포함)을 지정합니다. 주어진 파일이 복제되고 새 이름이 지정됩니다.

논쟁 설명
Filename 복사할 파일의 이름(문자열)
Copy To 복사본으로 생성할 파일의 이름(문자열)

예시:

위의 액션 블록 코드는 ini 파일이 없는지 확인하고 없으면 다른 파일을 복사하여 생성합니다.
