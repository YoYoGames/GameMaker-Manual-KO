Ini 파일에 쓰기

이 작업으로 숫자 값이나 문자열을 Ini 파일에 쓸 수 있습니다. 이전에 Open Ini File 작업을 사용하여 읽기/쓰기를 위해 Ini 파일을 연 다음 이 작업으로 쓸 데이터 유형(실수 또는 ""으로 묶인 문자열)과 섹션 헤더, 해당 섹션 내의 키 및 키 값. Ini 파일의 구조를 시각화하려면 다음 이미지를 참조하십시오.

보시다시피, Ini 파일은 섹션으로 분할됩니다. 각 섹션에는 고유한 헤더 이름이 있습니다. 그런 다음 각 섹션은 이후에 다양한 키/값 쌍으로 분할됩니다( Map 데이터 구조 와 유사). 텍스트 처리 소프트웨어(Windows의 메모장과 같은)를 사용하여 직접 Ini 파일을 만든 다음 게임에서 읽고 쓸 수 있는 포함 파일 로 포함할 수 있습니다.

중요한! Ini 파일 열기 작업을 사용하여 이전에 열지 않은 Ini 파일에서 데이터를 읽으려고 하면 심각한 오류가 발생하고 게임이 중단됩니다.

중요한! 이것은 데이터를 메모리에만 쓰고 마지막으로 디스크의 파일에 데이터를 쓰려면 Close Ini File 을 호출해야 합니다. 그렇지 않으면 데이터가 저장되지 않습니다.

동작 구문:

인수:

논쟁 설명
Type 쓸 데이터 유형 string (따옴표 ""로 묶음) 또는 real number
Section 쓸 Ini 파일의 섹션(문자열로)
Key 쓸 섹션 내의 키(문자열로)
Value 쓸 값, "유형" 인수(위)를 기반으로 하는 문자열 또는 실수

예시:

위의 작업 블록 코드는 ini를 연 다음 ini 파일을 다시 닫기 전에(따라서 데이터를 디스크에 쓰기) 동일한 섹션 헤더 아래에 있는 세 개의 다른 키(문자열 1개와 실수 2개)에 데이터를 씁니다.
