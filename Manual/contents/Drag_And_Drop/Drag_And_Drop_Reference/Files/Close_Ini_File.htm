Ini 파일 닫기

이 작업은 현재 열려 있는 Ini 파일을 닫습니다. 이 작업은 열려 있는 모든 Ini 파일에 대한 읽기 또는 쓰기를 완료하는 순간 호출되어야 합니다(Ini 파일은 Open Ini File 작업을 사용하여 열립니다). Ini 파일에 쓰기 작업을 사용한 후 이 작업을 사용하지 않으면 이 작업이 호출될 때까지 파일 정보가 메모리에 유지되므로 디스크에 아무 것도 기록되지 않습니다. 디스크에 있는 파일. 또한 이전에 다른 파일(또는 동일한 파일)을 닫지 않고 Ini 파일을 열려고 하면 오류가 발생합니다.

예시:

위의 작업 블록 코드는 사용할 ini 파일을 연 다음 "플레이어" 헤더 아래에 있는 "이름" 키와 관련된 값을 가져옵니다. 반환된 이름이 기본 이름 값과 일치하면(즉, 파일, 섹션 또는 키가 존재하지 않는 경우) 파일을 닫기 전에(그리고 새 데이터를 디스크에 쓰기) 전역 변수가 기록됩니다.
