로드 버퍼

이 작업을 통해 이전에 버퍼 저장 작업으로 저장 한 버퍼를 로드할 수 있습니다. 이 작업은 새 버퍼를 만들고 데이터 파일을 버퍼에 로드하고 고유한 ID 값을 반환하여 버퍼를 변수에 식별합니다. 로드할 파일의 이름(문자열으로)과 반환된 버퍼 ID를 저장할 변수를 제공합니다. 변수를 임시 로컬 변수로 플래그 지정하는 경우 이 함수가 생성하는 버퍼를 삭제해야 합니다. 액션 스크립트 또는 이벤트가 끝나기 전에 그렇지 않으면 변수에 대한 참조가 손실되고 변수가 삭제될 때 메모리 누수가 발생합니다.

동작 구문:

인수:

논쟁 설명
Filename 열려는 파일의 이름(문자열)
Target 로드 시 생성된 버퍼의 고유 ID 값을 보유할 대상 변수

예시:

위의 액션 블록 코드는 버퍼 데이터를 담을 변수를 생성하고 버퍼 세이브 파일이 이전에 생성되었는지 확인합니다. 파일이 존재하면 우리가 생성한 버퍼 변수에 로드되지만 존재하지 않으면 파일이 생성되어 다음에 로드될 수 있도록 작성되어 저장됩니다.
