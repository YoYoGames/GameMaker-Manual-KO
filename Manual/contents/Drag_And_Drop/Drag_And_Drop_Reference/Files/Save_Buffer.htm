버퍼 저장

이 작업으로 버퍼의 데이터가 포함된 파일을 저장할 수 있습니다. 고유한 버퍼 ID( Create Buffer 로 버퍼를 생성할 때 반환됨)를 제공하고 파일 이름(선택한 확장명을 가진 문자열로)을 지정하면 버퍼가 이 파일에 기록될 준비가 됩니다. Load Buffer 를 사용하여 다시 로드됩니다.

동작 구문:

인수:

논쟁 설명
Buffer 저장할 버퍼의 고유 버퍼 ID 값
Filename 저장할 파일의 이름(문자열)

예시:

위의 액션 블록 코드는 버퍼 데이터를 담을 변수를 생성하고 버퍼 세이브 파일이 이전에 생성되었는지 확인합니다. 파일이 존재하면 우리가 생성한 버퍼 변수에 로드되지만 존재하지 않으면 파일이 생성되어 다음에 로드될 수 있도록 작성되어 저장됩니다.
