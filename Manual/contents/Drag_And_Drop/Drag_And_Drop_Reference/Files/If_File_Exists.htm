파일이 있는 경우

이 작업으로 주어진 이름의 파일이 존재하는지 확인할 수 있습니다. 확인할 파일을 선택하면(확장자 포함) 함수가 반환됩니다. true 존재한다면, 그리고 false 그렇지 않은 경우. 이 반환됩니다 true 파일이 게임 번들에 포함된 파일로 존재하거나 게임이 실행되는 동안 파일이 저장된 경우(자세한 내용 은 파일 시스템 섹션 참조).

"if" 블록에 작업을 추가하려면 아래 이미지와 같이 작업 옆에 놓아야 합니다.
