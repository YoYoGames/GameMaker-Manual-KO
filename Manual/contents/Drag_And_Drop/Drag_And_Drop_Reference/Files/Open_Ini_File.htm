이니 파일 열기

이 작업으로 읽기 및/또는 쓰기를 위해 Ini 파일을 열 수 있습니다. 확인 중인 위치에 Ini 파일이 없는 경우 GameMaker 는 파일을 생성할 수 있지만 데이터를 쓰는 경우 에만 가능합니다. Ini 파일에서 읽기 정보만 있는 경우 읽기 기능의 기본값이 반환되지만 Ini 파일은 실제로 생성되지 않습니다.

한 번에 하나의 Ini 파일만 열 수 있으며 Ini 파일에서 읽기/쓰기를 마치면 Ini 파일 닫기 를 사용하는 것을 기억하십시오. 정보는 그때까지 실제로 디스크에 저장되지 않기 때문입니다. 파일이 닫힐 때까지 메모리).
