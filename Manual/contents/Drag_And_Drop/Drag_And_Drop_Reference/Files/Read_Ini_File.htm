Ini 파일 읽기

이 작업으로 Ini 파일에서 데이터 조각을 읽을 수 있습니다. 이전에 Open Ini File 작업을 사용하여 읽기/쓰기를 위해 Ini 파일을 연 다음 이 작업으로 읽을 데이터 유형(실수 또는 ""으로 묶인 문자열)과 섹션 헤더 및 해당 섹션 내의 키에서 데이터를 가져옵니다. 문자열의 실수일 수 있는 기본값을 제공할 수도 있으며 유형은 작업에 설정된 "유형"과 일치해야 합니다. 이 기본 작업은 다음 중 하나가 발생하면 작업에서 반환됩니다.

  • 당신이 연 Ini 파일이 존재하지 않습니다
  • 참조하는 섹션 헤더가 존재하지 않습니다
  • 데이터를 가져오려는 키가 존재하지 않습니다

위의 상황에서 Ini 파일의 값이 아닌 기본값이 반환됩니다. 반환된 값은 무엇이든 지정한 대상 변수에 저장되며, 임시 지역 변수로 플래그가 지정되면 그 순간에 생성될 수도 있습니다.

중요한! Ini 파일 열기 작업을 사용하여 이전에 열지 않은 Ini 파일에서 데이터를 읽으려고 하면 심각한 오류가 발생하고 게임이 중단됩니다.

인수:

논쟁 설명
Type 읽을 데이터 유형 string (따옴표 ""로 묶음) 또는 real number
Section 읽을 Ini 파일의 섹션(문자열로)
Key 읽을 섹션 내의 키(문자열로)
Default 반환할 기본값은 파일, 섹션 또는 키가 존재하지 않아야 합니다.
Target 작업의 반환 값을 유지하기 위한 대상 변수

예시:

위의 작업 블록 코드는 읽기용으로 Ini 파일을 연 다음 파일에서 읽은 데이터를 사용하여 파일을 다시 닫기 전에 3개의 다른 전역 변수를 설정합니다.
