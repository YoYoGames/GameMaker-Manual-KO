게임 다시 시작

이 작업으로 게임을 다시 시작할 수 있습니다. 이것은 게임을 처음 실행하는 것과 유사하므로 기타 - 게임 시작 이벤트 와 기타 - 게임 종료 이벤트 가 트리거됩니다. 그러나 전역 변수는 명시적으로 코딩되지 않는 한 다시 초기화되지 않습니다. 즉, 게임의 맨 처음 공간에서 정의 및 설정하지 않으면 이전 값이 유지되며 데이터 구조 파티클 도 해제되지 않으므로 Restart Game 액션이 호출 되기 전 이나 Game End Event에서 수행되어야 합니다. 또한 이 액션이 호출되는 순간 게임이 다시 시작 되지 않고 액션이 배치된 스크립트 또는 이벤트가 끝날 때 게임이 다시 시작되므로 이 액션 뒤에 배치된 액션이 있으면 여전히 호출되고 게임이 블록의 끝에서 다시 시작됩니다.
