게임을 저장

이 작업은 주어진 파일에 게임의 기본 저장을 수행합니다. 파일 이름(확장자를 포함한 문자열)을 지정하면 현재 게임 상태에서 제한된 저장을 수행합니다. 액션은 현재 게임 상태의 "스냅샷"을 찍도록 설계되었으며 범용 저장 시스템보다 체크포인트 시스템으로 더 많이 사용된다는 점을 이해하는 것이 중요합니다. 데이터 구조 또는 입자 ). 따라서 이 작업은 오류나 메모리 누수가 발생하지 않도록 매우 신중하게 사용해야 합니다. 게임 데이터 저장에 대한 보다 포괄적인 접근 방식은 파일 작업 을 참조하십시오.
