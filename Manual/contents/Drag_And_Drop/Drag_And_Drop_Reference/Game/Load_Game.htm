게임 로드

이 작업은 이전에 게임 저장 작업을 사용하여 저장 한 게임을 로드합니다. 로드할 파일의 파일 이름을 지정하면(확장자를 포함한 문자열로) 게임이 로드되고 저장된 지점에서 계속 실행됩니다. 이것은 범용 저장/로드 시스템으로 설계되지 않았으며 프로젝트의 이전 실행에서 저장한 게임을 로드하려고 하면 오류가 발생할 수 있습니다(특히 데이터 구조 또는 입자 와 같은 것을 사용한 경우). 따라서 다음을 수행해야 합니다. 단일 게임 플레이 스루 진행 중에 저장된 게임만 시도하고 로드하십시오(체크포인트와 같은 경우). 또한 파일은 현재 이벤트 또는 스크립트가 끝날 때까지 로드 되지 않으므로 로드 작업이 호출된 후의 모든 작업은 계속 수행됩니다. 저장된 게임 데이터를 로드하는 보다 포괄적인 접근 방식은 파일 작업 을 참조하십시오.
