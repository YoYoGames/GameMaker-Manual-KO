게임 종료

이 작업으로 게임을 종료하고 기타 - 게임 종료 이벤트 를 트리거할 수 있습니다. 이것은 작업이 호출되는 순간이 아니라 현재 이벤트 또는 스크립트가 끝날 때 발생하므로 이 작업이 호출된 후 수행하는 모든 작업은 이벤트/스크립트의 끝에 도달할 때까지 계속 실행됩니다..

이 작업에는 다음과 같은 제한 사항이 있습니다.

  • iOS 대상은 시장 조건에 위배되므로 오류를 보고합니다.
  • HTML5 및 Windows UWP 대상에서 호출될 때 효과가 없습니다.
  • Windows (Steam용으로 제작된 게임 포함), Android, Ubuntu(Linux) 및 Mac 에서 작동합니다.
