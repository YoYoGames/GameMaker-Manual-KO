방향 랜덤 설정

이 작업은 인스턴스의 이동 방향을 8개의 기본 지점에서 선택한 하나 이상의 임의 방향으로 설정하는 데 사용됩니다. 작업 내에서 8개의 방향 화살표 중 하나 이상을 클릭할 수 있으며 작업은 인스턴스의 방향으로 사용할 임의의 화살표 중 하나를 선택합니다. 화살표를 선택하지 않으면 작업이 인스턴스의 현재 방향에 영향을 주지 않습니다.

동작 구문:

예시:

위의 작업 블록 코드는 다음 중 하나를 선택합니다. true 또는 false 결과를 임시(로컬) 변수에 저장합니다. 그런 다음 이 변수를 확인하고 다음과 같은 경우 true 그런 다음 한 세트의 임의의 방향이 선택되고 false 그런 다음 다른 세트가 선택됩니다.
