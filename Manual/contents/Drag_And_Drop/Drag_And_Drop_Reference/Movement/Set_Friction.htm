마찰 설정

이 작업은 인스턴스의 "마찰"을 설정하는 데 사용됩니다. 마찰은 단순히 공제할 양을 지정하는 각 게임 프레임의 속도 벡터에서 특정 양이 공제된다는 것을 의미합니다. 일반적으로 0.2와 같이 더 작은 값이 가장 잘 작동하지만 실제 금액은 프로젝트 설정에 따라 다릅니다. 상대 플래그를 설정하면 주어진 양이 설정하는 대신 현재 마찰 값에 추가됩니다.

예시:

위의 액션 블록 코드는 이동과 관련된 다양한 인스턴스 속성을 설정합니다. 중력 방향은 아래로(270°), 중력은 0.05로, 마찰은 0.2로 설정됩니다.
