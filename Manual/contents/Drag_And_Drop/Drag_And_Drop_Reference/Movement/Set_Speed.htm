속도 설정

이 작업은 인스턴스의 이동 속도를 설정하는 데 사용됩니다. 인스턴스가 단일 게임 프레임(단계)에서 이동해야 하는 픽셀 수로 계산된 속도입니다. 기본적으로 속도 값은 방향 벡터에 추가됩니다. 따라서 속도를 2로 설정하고 방향을 45°로 설정하면 인스턴스가 게임 프레임마다 위아래로 2픽셀씩 이동합니다. direction 필드에서 수평 또는 수직 속도를 선택합니다.

이 중 하나를 선택하면 수평 또는 수직 벡터에 대해서만 속도가 설정됩니다. 값을 현재 속도에 상대적인 것으로 플래그를 지정하여 인스턴스의 실제 속도에서 값을 더하거나 뺄 수 있으며 음수를 제공하여 현재 속도와 반대 방향으로 이동할 수도 있습니다. 방향 값.
