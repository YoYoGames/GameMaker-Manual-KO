포인트로 점프

이 작업은 인스턴스를 한 위치에서 다른 위치로 이동하는 데 사용됩니다. 인스턴스는 새 위치로 원활하게 이동하지 않지만 해당 위치에 배치하기 위해 "점프"하므로 값이 충분히 크면 진행 중인 다른 인스턴스를 "건너뛰"합니다. 인스턴스를 이동할 절대값을 설정하거나 "상대적" 플래그를 선택하여 인스턴스가 현재 위치를 기준으로 이동하도록 할 수 있습니다. 예를 들어 인스턴스가 왼쪽으로 10픽셀만 이동하도록 하려면 x 값을 -10으로 설정하고 y 값을 0으로 설정하고 둘 다에 대해 상대적으로 틱합니다. 절대 좌표에 음수 값을 사용할 수 있으며 인스턴스는 베이스 룸 영역 외부로 이동됩니다.

동작 구문:

인수:

논쟁 설명
x 점프할 x 위치
y 점프할 y 위치

예시:

위의 액션 블록 코드는 왼쪽 마우스 버튼이 눌려 있는지 확인하고, 누르고 있으면 인스턴스를 마우스 위치로 이동합니다.
