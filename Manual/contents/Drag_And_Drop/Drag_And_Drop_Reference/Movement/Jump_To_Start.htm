시작으로 이동

이 작업은 인스턴스를 현재 위치에서 원래 생성된 위치로 이동하는 데 사용됩니다. 인스턴스가 생성된 원래 위치는 이라는 두 개의 인스턴스 변수 에 저장됩니다. xstart 그리고 ystart. 이러한 변수는 다른 작업에서 사용할 수 있으며 기록된 시작 위치를 변경하도록 설정할 수도 있습니다.

예시:

위의 액션 블록 코드는 인스턴스 경계 상자의 상단이 룸 높이보다 낮은지 확인하고 만약 그렇다면 인스턴스를 생성된 위치로 다시 이동합니다.
