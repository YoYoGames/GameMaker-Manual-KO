방향 고정

이 동작은 고정된 이동 방향을 설정하는 데 사용됩니다. 사용 가능한 방향은 작업 블록의 다른 버튼으로 표시 되며 그 중 하나를 클릭하면 해당 방향이 설정됩니다. GameMaker 의 방향은 도 단위로 측정되며 오른쪽은 0°, 위쪽은 90°, 왼쪽은 180°, 아래쪽은 270°입니다.

예시:

위의 액션 블록 코드는 인스턴스에서 왼쪽 마우스 버튼 누르기를 확인하고 감지되면 방향을 아래쪽(270°)으로 설정하고 속도를 5로 설정합니다.
