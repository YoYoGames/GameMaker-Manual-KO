방향 변수 설정

이 작업은 45° 증분으로 고정 되지 않은 인스턴스의 방향을 설정하는 데 사용됩니다. 0°에서 360° 사이의 방향 값을 제공합니다. 여기서 오른쪽은 0°, 위쪽은 90°, 왼쪽은 180°, 아래쪽은 270°입니다. 현재 방향.

Direction 0° ~ 360°의 방향(범위를 벗어난 값은 래핑됨).

예시:

위의 액션 블록 코드는 왼쪽 마우스 버튼이 눌려 있는지 확인하고, 누르고 있으면 방향을 0에서 360 사이의 임의의 값으로 설정하고 속도를 5로 설정합니다.
