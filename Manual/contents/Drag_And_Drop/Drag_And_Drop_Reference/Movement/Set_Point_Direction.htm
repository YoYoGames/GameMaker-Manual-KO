포인트 방향 설정

이 액션은 인스턴스의 현재 위치와 주어진 위치에 의해 형성된 벡터에서 방향을 설정하는 데 사용됩니다. x 및 y 위치를 "가리키는" 위치에 제공합니다. 이는 인스턴스 위치를 기준으로 플래그를 지정할 수 있습니다. 그러면 인스턴스에서 해당 지점까지의 방향이 해당 인스턴스에 사용됩니다.

인수:

논쟁 설명
x 방향 벡터의 x 구성요소입니다.
y 방향 벡터의 y 구성요소입니다.

예시:

위의 작업 블록 코드는 개체의 인스턴스가 "obj_Player "가 방에 존재하고 존재한다면 인스턴스의 방향이 "의 x/y 위치를 가리키도록 설정합니다.obj_Player ".
