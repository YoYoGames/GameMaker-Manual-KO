중력 설정

이 동작은 인스턴스에 작용하는 중력을 설정하는 데 사용됩니다. 각 인스턴스는 다른 모든 인스턴스와 독립적으로 다른 중력(및 방향)을 가질 수 있습니다. 적용된 힘은 게임 프레임당 픽셀 값이며 누적됩니다. 즉, 각 단계에 추가되므로 속도가 2이고 중력이 1인 경우 속도는 각 단계마다 1씩 증가합니다. 즉, 일반적으로 여기서 매우 낮은 값을 원할 것입니다. 중력에 대해 음수 값을 입력할 수 있으며 인스턴스는 중력 방향과 반대 방향으로 "당겨집니다".
