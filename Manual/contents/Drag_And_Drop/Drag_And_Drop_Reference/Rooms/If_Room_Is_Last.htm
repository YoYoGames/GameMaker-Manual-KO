방이 마지막인 경우

이 작업은 현재 방이 방 관리자 에 나열된 마지막 방인지 확인합니다. 액션이 돌아올 것이다 true 현재 방이 방 관리자의 마지막 방이고 false 그렇지 않은 경우. not 확인란에 플래그를 지정하여 이것을 "방이 마지막이 아닌 경우"로 바꿀 수 있습니다.

"if" 블록에 작업을 추가하려면 아래 이미지와 같이 작업 옆에 놓아야 합니다.

"if"가 다음과 같이 평가되면 이러한 작업이 실행됩니다. true, 다른 곳에 드롭된 모든 작업은 "if" 블록 다음에 수행됩니다.
