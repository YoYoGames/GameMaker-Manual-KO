방으로 이동

이 작업은 현재 방을 종료하고 입력 인덱스로 지정된 방으로 이동합니다. 방은 반드시 존재해야 하며 그렇지 않으면 오류가 발생하며 현재 방과 같은 방을 사용하면 다시 시작 방 을 사용하는 것과 같은 효과를 냅니다. 방은 작업이 호출된 이벤트 또는 스크립트가 끝날 때까지 변경 되지 않으므로 이 작업 이후의 모든 작업은 계속 실행됩니다.
