마스터 볼륨 가져오기

이 작업은 사운드 시스템의 마스터 볼륨을 반환합니다. 볼륨 값을 반환할 대상 변수를 제공합니다(이를 임시 로컬 변수로 플래그할 수 있음). 반환된 값은 0에서 1 사이입니다. 여기서 0은 무음이고 1은 전체 볼륨입니다.

동작 구문:

인수:

논쟁 설명
Target 반환된 볼륨을 저장할 대상 변수

예시:

위의 액션 블록 코드는 전역 변수가 다음과 같은지 확인합니다. true 또는 거짓. 만약 그렇다면 true 그런 다음 사운드 시스템의 마스터 볼륨이 검색되어 임시 로컬 변수에 저장됩니다. 그런 다음 값이 0.5와 같지 않은지 확인하고 그렇지 않은 경우 사운드 시스템의 마스터 볼륨이 0.5로 설정됩니다. 전역 변수가 다음과 같이 평가되는 경우 false, 마스터 볼륨은 1로 설정됩니다.
