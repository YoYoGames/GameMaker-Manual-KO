마스터 볼륨 설정

이 작업으로 전체 사운드 시스템의 볼륨을 설정할 수 있습니다. 제공하는 볼륨 값은 0(무음)과 1(전체 볼륨) 사이일 수 있으며 스케일은 선형이므로 값 0.5는 볼륨의 절반이 됩니다. 이 작업은 현재 방에서 재생 중인 모든 소리의 모든 인스턴스에 영향을 미치고 재생되는 모든 소리의 향후 모든 인스턴스에 영향을 미칩니다. 이 작업은 개별 사운드에 대해 설정된 볼륨과도 무관하므로 사운드를 0.5 볼륨으로 재생하도록 설정한 다음 마스터 볼륨을 0.5로 설정하면 사운드가 0.25 볼륨으로 재생됩니다.

동작 구문:

인수:

논쟁 설명
Volume 모든 사운드에 대한 새 마스터 볼륨(0에서 1, 기본값 1)

예시:

위의 액션 블록 코드는 전역 변수가 다음과 같은지 확인합니다. true 또는 거짓. 만약 그렇다면 true 그런 다음 사운드 시스템의 마스터 볼륨이 검색되어 임시 로컬 변수에 저장됩니다. 그런 다음 값이 0.5와 같지 않은지 확인하고 그렇지 않은 경우 사운드 시스템의 마스터 볼륨이 0.5로 설정됩니다. 전역 변수가 다음과 같이 평가되는 경우 false, 마스터 볼륨은 1로 설정됩니다.
