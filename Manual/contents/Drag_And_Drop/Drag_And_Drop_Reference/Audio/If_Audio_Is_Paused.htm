오디오가 일시 중지된 경우

이 작업은 사운드 리소스의 인스턴스가 현재 게임에서 일시 중지되었는지 확인하는 데 사용할 수 있습니다. 자산 탐색기에서 확인할 사운드 리소스를 제공하면 함수가 반환됩니다. true if it is paused and 그렇지 않은 경우 false, "not" 플래그를 확인하면 반전되어 함수가 반환됩니다. true 소리가 일시 중지 되지 않고 false 그렇지 않으면.

"if" 블록에 작업을 추가하려면 아래 이미지와 같이 작업 옆에 놓아야 합니다.

"if"가 다음과 같이 평가되면 이러한 작업이 실행됩니다. true, 다른 곳에 드롭된 모든 작업은 "if" 블록 다음에 수행됩니다.
