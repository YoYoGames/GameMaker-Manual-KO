키를 누르면

이 작업으로 키보드를 폴링하여 현재 누르고 있는 키가 있는지 확인할 수 있습니다. 액션이 돌아올 것이다 true 키보드 키를 누르고 있는 모든 단계, 또는 false 그렇지 않으면 not 수정자는 키가 도움이 되지 않는지 확인할 수 있습니다. 예: 작업이 반환됩니다. true 키가 내려가지 않은 상태에서 false 있다면. 단일 키 누름만 확인해야 하는 경우 If Key Pressed 작업을 사용합니다.

"if" 블록에 작업을 추가하려면 아래 이미지와 같이 작업 옆에 놓아야 합니다.

다른 곳에 드롭된 모든 작업은 "if" 블록 다음에 수행됩니다.

동작 구문:

인수:

논쟁 설명
Key 확인할 키보드 키의 이름입니다.
Not 검사를 무효화합니다(true는 false가 되고 그 반대도 마찬가지입니다).

예시:

위의 작업 블록 코드는 모든 단계에서 키보드 상태를 폴링하고 키를 누르고 있는 경우 초기 키 누름을 확인합니다. 체크인 경우 true 처음 아래로 누르면 소리가 재생되고 키를 누르고 있는 동안 스프라이트 혼합이 빨간색으로 변경됩니다. 키를 누르고 있지 않으면 키 해제 시 블렌딩 색상을 흰색으로 재설정하는 검사가 수행됩니다.
