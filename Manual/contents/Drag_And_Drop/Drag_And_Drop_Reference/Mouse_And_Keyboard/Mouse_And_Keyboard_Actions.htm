마우스 및 키보드 동작 라이브러리

마우스 및 키보드 작업 라이브러리는 키보드 또는 마우스 장치의 입력을 폴링하고 처리하는 데 필요한 작업을 찾을 수 있는 곳입니다 . 사용 가능한 작업은 모두 아래에 나열되어 있습니다.

마우스를 눌렀을 경우
마우스를 아래로 내리면
마우스를 놓으면
키가 눌린 경우
키를 누르면
키가 해제된 경우

여기에서 나머지 작업은 가상 키보드 를 참조합니다. 터치스크린 장치의 게임에서 텍스트 또는 숫자 입력이 필요한 경우 제공된 가상 키보드 작업을 사용하여 장치 OS 키보드를 사용할 수 있습니다. 이러한 작업은 OS 키보드를 호출하거나 숨기고 실제 키보드와 마찬가지로 입력을 수락하므로 일반 GameMaker 키보드 작업은 모두 가상 키보드에서 작동해야 합니다(예: "키가 해제된 경우" 작업). 가상 키보드의 입력은 다음과 같은 다른 키보드 변수에 저장됩니다. keyboard_string 또는keyboard_last_char, 등...).

가상 키보드를 열거나 닫으면 시스템 비동기 이벤트 가 생성됩니다. async_load  DS 맵 은 다음 키/값 쌍으로 채워집니다.

  • " event_type " - 수신된 시스템 이벤트의 유형으로, 가상 키보드의 경우 " virtual keyboard status "가 됩니다.
  • " screen_height " - 가상 키보드의 높이(픽셀 단위). 키보드가 보이지 않으면 0이 됩니다.
  • " keyboard_status " - 키보드의 현재 상태로, 다음 문자열 중 하나로 반환됩니다.
    • "숨김"
    • "숨겨진"
    • "전시"
    • "보이는"

사용 가능한 가상 키보드 작업은 다음과 같습니다.

가상 키보드 표시
가상 키보드 숨기기
가상 키보드 높이 얻기
가상 키보드가 표시되는 경우
