가상 키보드가 표시되는 경우

이 작업은 OS 가상 키보드가 표시되는지 여부를 확인하는 데 사용할 수 있습니다. 작업은 다음으로 평가됩니다. true 키보드가 열려 있거나 열려 있을 때 또는 false 가상 키보드가 닫혀 있거나 닫혀 있는 경우. 활성화하면 not 수정자, 이 작업은 취소되고 반환을 저장할 수 있습니다. true / false 임시(로컬)로 플래그가 지정될 수 있는 대상 변수의 값입니다.
