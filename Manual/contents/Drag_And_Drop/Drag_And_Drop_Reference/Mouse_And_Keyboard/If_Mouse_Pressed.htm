마우스를 눌렀을 경우

이 작업은 마우스 버튼을 처음 눌렀는지 확인합니다. 그것은 돌아올 것이다 true 버튼이 처음에 눌려진 단일 게임 단계(프레임)에서 false 다른 모든 시간에. 활성화하면 not 수정자, 이 동작은 반전되고 버튼이 눌려 지지 않았 는지 확인하여 반환 false 눌려진 게임 스텝(프레임)에 true 다른 모든 시간에.. 초기 누르기 이벤트뿐만 아니라 누르고 있는 버튼을 확인해야 하는 경우 If Mouse Down 작업을 사용하십시오.

"if" 블록에 작업을 추가하려면 아래 이미지와 같이 작업 옆에 놓아야 합니다.

"if"가 다음과 같이 평가되면 이러한 작업이 실행됩니다. true, 다른 곳에 드롭된 모든 작업은 "if" 블록 다음에 수행됩니다.

동작 구문:

인수:

논쟁 설명
Mouse Button 확인할 마우스 버튼의 이름( 마우스 버튼 상수에 대한 정보는 이 페이지 참조 ).
Not 검사를 무효화합니다(true는 false가 되고 그 반대도 마찬가지입니다).

예시:

위의 작업 블록 코드는 모든 단계에서 마우스 버튼 상태를 폴링하고 누르고 있는 경우 초기 마우스 누름을 확인합니다. 체크인 경우 true 초기 다운 시 인스턴스 속도가 설정되고 버튼이 다운된 동안 스프라이트 블렌딩이 빨간색으로 변경됩니다. 마우스를 누르고 있지 않으면 블렌딩 색상을 흰색으로 재설정하고 속도를 0으로 설정하기 위해 마우스를 놓을 때 검사가 수행됩니다.
