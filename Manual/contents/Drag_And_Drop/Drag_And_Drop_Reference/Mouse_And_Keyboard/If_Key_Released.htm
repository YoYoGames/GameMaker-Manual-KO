키가 해제된 경우

이 작업은 키보드 키가 해제되었는지 확인합니다. 그것은 돌아올 것이다 true 키가 릴리스된 단일 게임 단계(프레임)에서 false 다른 모든 시간에. 활성화하면 not 수정자, 이 작업은 취소되고 키가 해제 되지 않았는지 확인하여 반환 false 출시된 게임 스텝(프레임)과 true 다른 모든 시간에.

"if" 블록에 작업을 추가하려면 아래 이미지와 같이 작업 옆에 놓아야 합니다.

"if"가 다음과 같이 평가되면 이러한 작업이 실행됩니다. true, 다른 곳에 드롭된 모든 작업은 "if" 블록 다음에 수행됩니다.

인수:

논쟁 설명
Key 확인할 키의 이름입니다.
Not 검사를 무효화합니다(true는 false가 되고 그 반대도 마찬가지입니다).

예시:

위의 작업 블록 코드는 모든 단계에서 키보드 상태를 폴링하고 키를 누르고 있는 경우 초기 키 누름을 확인합니다. 체크인 경우 true 처음 아래로 누르면 소리가 재생되고 키를 누르고 있는 동안 스프라이트 혼합이 빨간색으로 변경됩니다. 키를 누르고 있지 않으면 키 해제 시 블렌딩 색상을 흰색으로 재설정하는 검사가 수행됩니다.
