인스턴스 변수 가져오기

이 작업으로 나열된 내장 변수 중 하나를 얻을 수 있습니다. 모든 객체의 모든 인스턴스에는 현재 x/y 위치, 속도, 방향, 가시성 및 기타 여러 가지와 같은 필수 정보를 보유하기 위해 이미 생성된 많은 변수가 있습니다. 값을 얻을 수 있는 인스턴스 변수는 다음과 같습니다.

  • X 좌표 - 방의 x축을 따른 위치입니다.
  • Y 좌표 - 방의 y축을 따른 위치입니다.
  • 방향 - 인스턴스의 이동 방향(기본값은 0, 오른쪽은 0°, 위쪽은 90°, 왼쪽은 180°, 아래쪽은 270°)입니다.
  • 속도 - 인스턴스가 이동하는 속도(게임 프레임당 픽셀 단위)입니다.
  • 수평 속도 - 인스턴스가 움직이는 수평 속도(게임 프레임당 픽셀 단위)입니다. 음의 속도는 왼쪽으로 움직이고 있음을 의미하고 양의 속도는 오른쪽이고 기본값은 0입니다.
  • 수직 속도 - 인스턴스가 움직이는 수직 속도(게임 프레임당 픽셀 단위)입니다. 음수 속도는 위로 이동하고 있음을 의미하고 양수 속도는 아래로, 기본값은 0입니다.
  • Visible - 가시성 토글 변수(기본값은 true 개체 편집기 에서 달리 설정하지 않는 한 다음으로 설정합니다. false 보이지 않게 하기 위해). 보이지 않는 인스턴스는 이벤트 그리기 를 제외한 모든 이벤트를 계속 실행합니다.
  • Persistent - 지속성 토글 변수(기본값은 false 개체 편집기 에서 달리 설정하지 않는 한 다음으로 설정합니다. true 지속적으로). 이렇게 하면 인스턴스가 방에서 방으로 "지속"됩니다. 즉, 인스턴스가 방 끝에서 정리되지 않고 대신 다음 방으로 이월됩니다. 인스턴스가 지속되면 Create 또는 Destroy 이벤트를 호출하지 않고 Clean Up, Room End 및 Room Start 이벤트를 호출합니다.
  • 레이어 - 인스턴스가 할당된 레이어입니다.
  • 깊이 - 레이어에 할당된 깊이 값입니다.
  • Sprite - 인스턴스에 할당된 스프라이트 리소스입니다.
  • 이미지 인덱스 - 할당된 스프라이트의 이미지 인덱스입니다. 이 값은 인스턴스에 할당된 스프라이트의 프레임을 설정하고, 값이 애니메이션의 프레임 수보다 크면 프레임 7로 설정된 경우 프레임이 5개인 스프라이트가 실제로 프레임 2를 표시하도록 "루프"합니다.
  • 이미지 각도 - 인스턴스가 설정되어야 하는 각도입니다(기본값은 0, 여기서 오른쪽은 0°, 위쪽은 90°, 왼쪽은 180°, 아래쪽은 270°).
  • 이미지 알파 - 0에서 1 사이의 인스턴스 알파 값(기본값은 1)입니다. 여기서 0은 완전히 투명하고 1은 완전히 불투명합니다.
  • 이미지 혼합 - 인스턴스 스프라이트와 혼합할 색상입니다(기본값은 흰색).
  • Image Speed - 인스턴스의 애니메이션 속도입니다.
  • 순서 대로 - 이 변수는 다음을 반환합니다. true 인스턴스가 현재 시퀀스에 포함되어 있는 경우 false 그렇지 않으면.
  • 수평 스케일 - 인스턴스 스프라이트를 그려야 하는 수평 스케일입니다(이는 인스턴스 경계 상자 및 충돌에도 영향을 미칩니다). 스케일이 1이면 스케일링이 없음(1:1), 작은 값은 스케일 다운(예: 0.5는 스프라이트 너비의 절반), 더 큰 값은 스케일 업되고 음수 값은 스프라이트 를 미러링하여 스케일링됩니다. 사용된 값은 정확히 -1입니다(이 경우 스프라이트는 크기 조정 없이 y축을 따라 미러링됩니다).
  • 수직 스케일 - 인스턴스 스프라이트를 그려야 하는 수직 스케일입니다(이는 인스턴스 경계 상자 및 충돌에도 영향을 미칩니다). 스케일이 1이면 스케일링이 없음(1:1), 작은 값은 스케일 다운(예: 0.5는 스프라이트 높이의 절반), 큰 값은 스케일 업되고 음수 값은 스프라이트를 뒤집고 스케일링합니다. 사용된 값은 정확히 -1입니다(이 경우 스프라이트는 크기 조정 없이 y축을 따라 뒤집힙니다).
  • 마스크 - 인스턴스의 마스크 인덱스입니다. 기본값은 -1(인스턴스가 현재 할당된 스프라이트 인덱스에 대해 충돌 마스크를 사용함을 의미함)이지만 다른 스프라이트 리소스로 설정하면 스프라이트에 할당된 스프라이트 대신 해당 인스턴스의 충돌 마스크가 사용됩니다. 인덱스.
  • Friction - 게임의 각 게임 프레임을 사용하여 인스턴스 속도를 늦추는 마찰입니다. 이 값은 단순히 각 게임 프레임의 현재 속도 값에서 차감됩니다(또는 속도 값이 음수인 경우 추가됨).
  • Gravity - Gravity는 인스턴스의 중력 방향(아래 참조)으로 일정한 힘을 적용하여 속도에 영향을 줍니다(기본값은 0). 중력은 누적된 힘이며 최종 속도를 제한하지 않도록 선택하면 물체를 가속합니다.
  • 중력 방향 - 인스턴스 중력이 0보다 클 때 이동 방향을 설정하는 데 사용할 수 있습니다(기본값은 270). GameMaker 의 방향은 오른쪽이 0°, 위쪽이 90°, 왼쪽이 180°, 아래쪽이 270°로 계산됩니다.
  • 점수 - 설정 score 변하기 쉬운.
  • 삶 - 설정 lives 변하기 쉬운.
  • 건강 - 설정 health 변하기 쉬운.

가져올 내장 변수를 선택한 후 반환된 값을 보유하도록 대상 변수를 설정합니다. 이 변수에 임시 변수로 플래그를 지정하면 작업에서 자동으로 생성되고 이벤트 또는 작업 스크립트가 끝날 때까지만 사용할 수 있습니다. 더하기 아이콘을 클릭하여 동일한 작업에서 추가 변수의 값을 검색할 수도 있습니다. 액션 옆에 다른 인스턴스 변수를 선택하고 반환 값을 저장할 동반 변수를 제공합니다.
