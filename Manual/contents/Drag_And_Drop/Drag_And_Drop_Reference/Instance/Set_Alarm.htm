알람 카운트다운 설정

이 작업 블록은 주어진 알람을 카운트다운으로 설정하고 알람이 0에 도달하면 관련 알람 이벤트 가 트리거됩니다. 0에서 11까지의 알람 번호를 입력한 다음 알람을 설정할 값을 입력합니다. 사용된 값은 알람 이벤트가 트리거되기 전에 전달하려는 "게임 단계"의 수이며 현재 알람 값에 상대적으로 설정할 수 있습니다(현재 값에 더하거나 빼기). 값은 정수 값이어야 하며 -1로 설정하여 알람을 중지할 수 있습니다. 알람이 카운트다운되면 이벤트가 0에 도달할 때 실행되지만 알람은 완료되었음을 나타내기 위해 -1까지 카운트다운됩니다. 알람을 0으로 설정하면 다음 단계에서 -1까지 카운트다운되므로 이벤트가 실행 되지 않으므로 즉시 알람을 실행하려면 1로 설정해야 합니다. 더하기 아이콘 행동 옆에.
