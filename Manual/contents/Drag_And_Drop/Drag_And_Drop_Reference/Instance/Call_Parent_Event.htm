부모 이벤트 호출

이 작업 블록은 실제 인스턴스에서 작업을 계속 실행하기 전에 인스턴스가 자식인 부모 개체의 이벤트를 수행합니다. 부모 개체를 인스턴스에 할당하면 해당 개체의 모든 이벤트가 자식에 의해 "상속"됩니다. 그러나 하위 개체에 이벤트를 추가하면 상위 이벤트를 재정의합니다. 이 작업 블록을 사용하여 자식 개체가 부모 이벤트와 인스턴스 자체의 이벤트를 모두 실행하도록 할 수 있습니다. 상위 항목에 대한 자세한 내용은 개체 편집기 섹션을 참조하십시오.

동작 구문:

예시:

위의 액션 블록 코드는 인스턴스가 자식인 객체의 부모 이벤트를 호출한 후 알람을 설정하고 스프라이트를 변경합니다.
