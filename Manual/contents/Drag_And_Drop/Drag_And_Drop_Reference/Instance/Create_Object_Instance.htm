객체 인스턴스 생성

이 작업으로 자산 브라우저에서 모든 개체의 인스턴스를 만들 수 있습니다. 객체의 이름과 객체가 생성될 공간 내의 x/y 위치와 인스턴스를 생성할 레이어의 이름( 인스턴스 레이어 여야 함)을 제공합니다. "상대적" 플래그를 선택하여 작업 코드 블록을 실행하는 인스턴스를 기준으로 인스턴스를 배치할 수 있습니다. 게임에서 인스턴스를 생성할 때마다 고유한 ID 값이 제공되어 이후 작업에서 인스턴스를 식별할 수 있으므로 이 작업을 통해 이 ID 값을 보유할 "대상" 변수를 정의할 수 있습니다(그림 참조). 아래 예). 대상은 작업에 대해 생성되고 모든 후속 작업에서 사용될 수 있는 임시(로컬) 변수로 플래그가 지정될 수 있습니다.

인수:

논쟁 설명
Object 인스턴스를 생성할 개체의 이름입니다.
x 인스턴스를 생성하기 위한 방의 x 위치(호출 인스턴스에 상대적일 수 있음).
y 인스턴스를 생성하기 위한 룸의 y 위치(호출 인스턴스에 상대적일 수 있음).
Layer 인스턴스를 생성할 인스턴스 레이어의 이름(문자열)입니다.
Target 반환 값을 대상으로 하는 변수입니다.

예시:

위의 액션 블록 코드는 새 인스턴스를 만들고 고유한 ID 값을 임시 변수에 저장합니다. 그런 다음 이 변수는 적용 대상... 작업에서 사용되어 생성된 인스턴스의 일부 속성을 설정합니다.
