객체 인스턴스 파괴

인스턴스를 "파괴"하고 싶을 때마다 이 작업을 호출하면 방에서 제거됩니다. 인스턴스를 파괴할 때 해당 Destroy Event 는 destroy를 호출하는 작업 직후에 호출 된다는 점에 유의해야 합니다. 또한 Destroy Event는 액션이 호출되는 즉시 수행되지만 인스턴스는 게임에서 즉시 제거 되지 않고 끝까지 현재 이벤트에 포함된 코드를 계속 수행하며 현재 이벤트가 발생한 경우에만 종료되면 게임에서 인스턴스가 제거되며, 이때 Clean Up Event 가 호출됩니다.
