인스턴스 규모 설정

이 작업 블록은 다음을 설정합니다. image_xscale 그리고 image_yscale 인스턴스에 대한 값. 여기의 입력 값은 인스턴스에 할당된 스프라이트에 적용되는 수정자입니다. 여기서 스케일 1(기본값)은 스케일링이 없음(1:1)을 나타내고, 더 작은 값은 축소됩니다(예: 0.5, 절반 스프라이트의 너비), 더 큰 값은 확장됩니다. 사용된 값이 정확히 -1이 아닌 경우 음수 값을 사용하여 스프라이트를 뒤집고 크기를 조정할 수 있습니다(이 경우 스프라이트는 크기 조정 없이 원점에 대해 뒤집히거나 미러링됨). 현재 값을 기준으로 수평 또는 수직 스케일을 설정하도록 선택할 수 있습니다. 이 경우 현재 스케일 값에서 새 값을 더하거나 뺍니다.

이 값을 변경하면 별도의 값을 제공하지 않는 한 인스턴스가 충돌을 감지하는 방법도 변경됩니다. mask_index (충돌 마스크) 개체 편집기. 이 변수의 변경 사항이 표시되려면 인스턴스에 그리기 이벤트가 없어야 하고(따라서 GameMaker 는 기본적으로 스프라이트를 그립니다) 자체 그리기 작업을 사용하여 그려야 합니다.
