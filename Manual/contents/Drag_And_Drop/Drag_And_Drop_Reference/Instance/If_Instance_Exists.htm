인스턴스가 있는 경우

이 작업은 게임 룸에 특정 개체의 인스턴스가 있는지 또는 특정 인스턴스 자체가 존재하는지 확인하는 데 사용할 수 있습니다. 자산 탐색기에서 개체를 제공하고 방에 인스턴스가 있으면 작업이 반환됩니다. true 그렇지 않으면 반환됩니다 false. 여기에서 특정 인스턴스의 고유 ID(예: Create Object Instance 작업에서 반환됨) 또는 인스턴스를 추가할 때 룸 편집기에서 반환된 인스턴스 ID 값과 함께 변수를 사용할 수 있습니다.

"Not" 인수에 플래그를 지정하면 해당 작업은 인스턴스가 없는지 확인하고 인스턴스가 없으면 반환합니다. true 하나라도 존재하면 반환됩니다 false.

"if" 블록에 작업을 추가하려면 아래 이미지와 같이 작업 옆에 놓아야 합니다.

"if"가 다음과 같이 평가되면 이러한 작업이 실행됩니다. true, 다른 곳에 드롭된 모든 작업은 "if" 블록 다음에 수행됩니다.
