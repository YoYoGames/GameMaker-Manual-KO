인스턴스 회전 설정

이 작업 블록은 다음을 설정합니다. image_angle (회전) 인스턴스. 각도는 도 단위로 측정되며 오른쪽은 0°, 위쪽은 90°, 왼쪽은 180°, 아래쪽은 270°입니다. 인스턴스를 재설정하려면 이 변수를 0으로 설정하십시오. 즉, 스프라이트 편집기에서 정의된 대로 할당된 모든 스프라이트를 그립니다. 이 작업의 변경 사항을 보려면 인스턴스에 그리기 이벤트가 없어야 하고(따라서 GameMaker 는 기본적으로 스프라이트를 그립니다) 자체 그리기 작업을 사용하여 그려야 합니다.

동작 구문:

인수:

논쟁 설명
Angle 사용할 새 각도(0에서 360까지).

예시:

위의 작업 블록 코드는 180° 회전을 포함하여 해당 스프라이트가 표시되는 방식에 대한 여러 속성과 함께 새 스프라이트를 설정합니다.
