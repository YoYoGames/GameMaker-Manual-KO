알람 받기

이 작업 블록은 알람 카운트다운 값을 검색하여 지정된 대상 변수에 저장합니다. 대상 변수에 "Temp"로 플래그를 지정하면 아래 예와 같이 반환 값을 저장할 새 임시(로컬) 변수가 생성됩니다.

동작 구문:

인수:

논쟁 설명
Alarm 알람을 받을 수 있습니다.
Target 알람 값을 저장하는 데 사용할 대상 변수입니다.

예시:

위의 액션 블록 코드는 로컬(임시) 변수를 생성하고 알람[0]의 현재 값을 저장합니다. 그런 다음 이를 확인하고 0보다 작으면 알람을 60으로 설정합니다.
