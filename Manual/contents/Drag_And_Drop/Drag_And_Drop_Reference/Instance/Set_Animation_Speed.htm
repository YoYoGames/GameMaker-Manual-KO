애니메이션 속도 설정

이 작업 블록은 다음을 설정합니다. image_speed (애니메이션 속도) 인스턴스에 할당된 스프라이트. Sprite Editor 에서 이상적인 초당 프레임을 설정한 다음 이 값을 사용하여 수정할 수 있습니다. 기본 애니메이션 속도는 1로 설정되어 있습니다. 즉, 게임은 스프라이트에 대해 정의한 초당 프레임 수를 유지하려고 시도하지만 1보다 큰 값을 사용하여 더 빠르게 애니메이션할 수 있습니다(프레임을 건너뛸 수 있음). 애니메이션 속도를 반대로 하려면 10진수 값을 사용하거나(프레임이 다양한 단계에 걸쳐 표시됨) 음수 값을 사용합니다.

인수:

논쟁 설명
Speed 새로운 수정자 속도 값입니다.

예시:

위의 작업 블록 코드는 정의된 속도의 절반으로 애니메이션되도록 설정하는 것을 포함하여 해당 스프라이트가 표시되는 방식에 대한 여러 다른 속성과 함께 새 스프라이트를 설정합니다.
