인스턴스 색상 설정

이 작업 블록은 다음을 설정합니다. image_blend 인스턴스 스프라이트와 "블렌딩"을 위한 색상입니다. 기본값은 16진수 코드 $FFFFFFFF ( ARGB 형식을 사용하여 흰색을 나타냄)이며 이는 혼합 및 전체 알파를 사용하지 않고 스프라이트를 그립니다. 다른 모든 값(예: c_red, 또는 c_aqua ) 지정된 색상을 원본 스프라이트와 혼합 하고 설정합니다. image_alpha 인스턴스에 대한 구성 요소이지만 알파 사용 확인란이 선택된 경우에만 가능합니다. 이렇게 하면 Set Instance Alpha 작업을 사용하여 이전에 설정한 모든 알파 값을 덮어씁니다.

이 작업의 변경 사항을 보려면 인스턴스에 그리기 이벤트가 없어야 하고(따라서 GameMaker 는 기본적으로 스프라이트를 그립니다) 자체 그리기 작업을 사용하여 그려야 합니다. WebGL을 사용하지 않는 한 HTML5 플랫폼에서 혼합을 제한하는 것이 중요합니다. 혼합된 각 스프라이트는 개별적으로 캐시되어야 하고 혼합된 스프라이트가 많으면 성능에 부정적인 영향을 미칠 수 있기 때문입니다.
