사용자 이벤트 호출

이 작업으로 12개의 사용 가능한 기타 사용자 이벤트 를 호출할 수 있습니다. 사용자 이벤트는 스크립트 또는 작업 코드를 추가한 다음 이 작업을 사용하여 언제든지 호출하여 본질적으로 사용자 정의 개체 이벤트로 만들 수 있는 이벤트입니다.
