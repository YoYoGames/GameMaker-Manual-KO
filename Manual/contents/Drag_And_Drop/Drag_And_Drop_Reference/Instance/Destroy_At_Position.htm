위치에서 파괴

이 작업 블록은 지정된 지점에서 발견된 모든 인스턴스를 파괴하고 Destroy Event 를 호출한 다음 발견된 모든 인스턴스에 대해 Clean Up Event 를 호출합니다. 이 작업은 인스턴스의 충돌 마스크를 기반으로 하며 마스크의 일부가 대상 지점과 겹치면 함수가 해당 인스턴스를 파괴합니다(인스턴스에 적용된 스프라이트 또는 마스크가 없는 경우 파괴되지 않음).

예시:

위의 작업 블록 코드는 키 누름(스페이스 키)을 확인하고 하나가 감지되면 인스턴스 상대 위치에서 찾은 모든 인스턴스를 파괴합니다.
