인스턴스 알파 설정

이 작업 블록은 다음을 설정합니다. image_alpha 인스턴스의 값입니다. 이미지 알파는 인스턴스에 할당된 모든 스프라이트를 그리기 위한 "투명도" 값이며 0과 1 사이의 값을 가지며 0은 완전히 투명하고 1은 완전히 불투명합니다. 스프라이트에 투명하거나 반투명한 픽셀이 있는 경우 해당 투명도는 양만큼 조정되므로 스프라이트에 투명 픽셀이 50% 정도 있고 인스턴스 알파를 0.5로 설정하면 투명 픽셀의 최종 알파가 됩니다. 25%일 것입니다. 이 값을 상대적인 금액으로 설정할 수 있습니다.

이 변수의 변경 사항이 표시되려면 인스턴스에 그리기 이벤트가 없어야 하고(따라서 GameMaker 는 기본적으로 스프라이트를 그립니다) Draw Self 를 사용하여 그려야 합니다. 동작.

alpha 설정할 알파 값(0에서 1, 기본값은 1).

예시:

위의 작업 블록 코드는 이미지 알파를 0.5로 설정하는 것을 포함하여 해당 스프라이트를 표시하는 방법에 대한 여러 속성과 새 스프라이트를 설정합니다.
