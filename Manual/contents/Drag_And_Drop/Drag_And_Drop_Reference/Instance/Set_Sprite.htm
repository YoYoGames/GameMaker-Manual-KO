스프라이트 설정

이 작업 블록은 다음을 설정합니다. sprite_index 자산 브라우저에서 다른 스프라이트로 인스턴스의. 변경할 스프라이트를 선택한 다음 표시할 이미지(애니메이션 프레임)를 지정합니다. 프레임을 설정한 후 스프라이트가 애니메이션되는 것을 원하지 않으면 Set Animation Speed 액션을 사용하고 0으로 설정해야 합니다. image_index 대신 내장 변수 변수.

스프라이트를 변경하면 별도로 지정하지 않는 한 인스턴스의 충돌 마스크도 변경됩니다. mask_index 개체 편집기 에서. 이 작업의 변경 사항을 보려면 인스턴스에 그리기 이벤트가 없어야 하고(따라서 GameMaker 는 기본적으로 스프라이트를 그립니다) 자체 그리기 작업을 사용하여 그려야 합니다.

Sprite 사용할 새 스프라이트입니다(스프라이트를 제거하려면 -1).
Frame 처음에 표시할 애니메이션 프레임입니다(추가 프레임이 사용되지 않으면 0 사용).

예시:

위의 작업 블록 코드는 새 스프라이트와 해당 스프라이트를 표시하는 방법에 대한 기타 여러 속성을 설정합니다.
