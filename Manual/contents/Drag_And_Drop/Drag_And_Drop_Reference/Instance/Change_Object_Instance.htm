개체 인스턴스 변경

이 작업으로 한 인스턴스를 다른 인스턴스로 변경할 수 있습니다. 인스턴스가 다른 인스턴스로 변경되는 개체를 지정해야 하며 초기 인스턴스의 Destroy Event 및 Clean Up Event 및 새 인스턴스의 Create Event를 수행하여 해당 인스턴스가 변경되는 작업을 수행합니다. 방향이나 속도 등을 지정하지 않으면 새 인스턴스가 이러한 값을 "상속"합니다. 예를 들어 초기 인스턴스에 image_angle (회전) 180, 게임 틱당 2px로 이동하고 image_speed (애니메이션 속도)가 0.5이면 변경되는 인스턴스는 새 인스턴스의 Create 이벤트에서 명시적으로 설정하지 않는 한 이러한 값을 유지합니다.

인스턴스를 변경 하면 다음 게임 틱까지 해당 인스턴스로 더 이상 작업을 수행할 수 없음을 의미합니다. 특히 변수에 액세스하려고 하면 오류가 발생하므로... 기본적으로 인스턴스를 변경하지만 현재 단계가 끝날 때까지 실제로 사용할 수 없으므로 포함된 변수에 직접 액세스하려면(예: inst_changed.x ) 작동 안 할 것이다.
