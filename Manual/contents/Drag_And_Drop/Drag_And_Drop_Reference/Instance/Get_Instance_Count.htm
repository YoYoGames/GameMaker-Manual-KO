인스턴스 수 가져오기

이 작업을 통해 현재 방에서 활성화된 개체의 인스턴스 수를 찾을 수 있습니다. 리소스 목록에서 개체를 선택하거나 특수 키워드를 사용할 수 있습니다. all 활성화된 모든 인스턴스의 수를 가져옵니다. 또한 값을 반환할 대상 변수를 제공해야 추가 작업에서 사용할 수 있습니다. 대상 변수에 "Temp"로 플래그를 지정하면 아래 예와 같이 반환 값을 저장할 새 임시(로컬) 변수가 생성됩니다.

인수:

논쟁 설명
Object 확인할 개체의 이름입니다.
Target 반환 값을 대상으로 하는 변수입니다.

예시:

위의 액션 블록 코드는 로컬(임시) 변수를 생성한 다음 "obj_Squirrel" 개체의 인스턴스 수를 할당합니다. 그런 다음 이 값을 확인하고 인스턴스 수가 100보다 크면 인스턴스를 파기합니다.
