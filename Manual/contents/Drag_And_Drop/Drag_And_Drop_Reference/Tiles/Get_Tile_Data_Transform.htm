타일 데이터 변환 가져오기

이 작업을 통해 타일의 타일 데이터가 뒤집힘, 대칭 또는 회전의 세 가지 방법 중 하나로 변환되었는지 확인할 수 있습니다. 타일 데이터는 적용된 변환과 함께 타일의 인덱스를 반영하는 값일 뿐이며 일반적으로 이 작업을 사용하기 전에 셀 에서 타일 데이터 가져오기 또는 픽셀에서 타일 데이터 가져오기를 사용하여 타일 데이터를 먼저 가져오길 원할 것입니다. 액션이 돌아올 것이다 true 선택한 변환이 타일 데이터에 적용된 경우 또는 false 그렇지 않으면 반환된 값은 이전에 생성되었거나 새로운 임시 변수가 될 수 있는 대상 변수에 저장됩니다("Temp" 확인란을 선택한 경우). 타일 데이터에 대한 자세한 내용은 타일 맵 함수 의 GML 섹션을 참조하세요.
