타일 세트 설정

이 작업을 통해 자산 브라우저에서 생성된 것과 다른 타일 세트 를 사용하도록 레이어를 설정할 수 있습니다. 레이어 이름(룸 편집기에 정의된 문자열)을 지정한 다음 사용할 타일 세트 리소스를 지정하면 해당 레이어의 룸에 있는 모든 타일이 새 타일 세트로 그려집니다.

Layer 새 타일 세트를 설정할 레이어입니다.
Tileset 레이어에 사용할 타일 세트입니다.

예시:

위의 작업 블록 코드는 "Floor_Tiles" 레이어의 타일이 "tl_PalaceRuins" 타일 세트를 사용 중인지 확인하고 그렇지 않은 경우 사용하도록 설정되어 있습니다.
