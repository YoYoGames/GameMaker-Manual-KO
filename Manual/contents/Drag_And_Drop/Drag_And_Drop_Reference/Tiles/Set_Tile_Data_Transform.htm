타일 데이터 변환 설정

이 작업으로 타일 데이터의 세 가지 타일 변환 중 하나를 변경할 수 있습니다. 타일에 대한 타일 데이터를 보유하는 변수를 제공한 다음 변경할 변환을 선택합니다. "적용" 확인란을 클릭하면 해당 변환이 적용되고 선택을 취소하면 제거됩니다. 타일 데이터는 적용된 변환 및 사용한 사용자 지정 타일 마스크와 함께 타일의 인덱스를 반영하는 값일 뿐입니다. 이 작업을 통해 해당 값 중 세 가지(뒤집기, 미러링 및 회전)를 변경할 수 있습니다. 일반적으로 Get Tile Data In Cell 또는 Get Tile Data At Pixel 을 사용하여 타일 데이터를 먼저 가져온 다음 필요한 변환을 적용한 다음 Set Tile Data In Cell 을 사용하여 타일 맵에서 타일을 다시 설정합니다. 또는 Set Tile Data At Pixel, 이 작업 은 값만 설정하지만 가져온 타일에는 적용하지 않기 때문입니다. 타일 데이터에 대한 자세한 내용은 타일 맵 함수 의 GML 섹션을 참조하세요.

예시:

위의 액션 블록 코드는 마우스 위치에서 타일 데이터를 가져온 다음 타일에 회전 변환이 적용되었는지 확인합니다. 그렇다면 변환이 재설정되고 타일 데이터가 타일을 해당 위치에 다시 설정합니다.
