셀에 타일 인덱스 설정

이 작업으로 주어진 레이어의 타일을 새 인덱스로 설정할 수 있습니다. 레이어는 주어진 레이어 이름(룸 편집기에서 레이어 이름을 지정하는 데 사용되는 문자열)에서 지정한 다음 레이어 내 셀의 행과 열을 지정하여 타일을 설정할 수 있습니다. 마지막으로 타일 인덱스를 제공하여 타일을 설정합니다(타일은 위에서 아래로 왼쪽에서 오른쪽으로 인덱싱되며 왼쪽 상단에는 0이 표시됨). 타일 인덱스를 항상 0으로 설정하여 타일을 지울 수 있습니다.

인수:

논쟁 설명
Layer 타일을 설정할 레이어
Column 셀의 열(왼쪽에서 오른쪽으로)
Row 셀의 행(위에서 아래로)
Tile 셀을 설정할 타일 인덱스

예시:

위의 액션 블록 코드는 주어진 룸 레이어의 맨 위 행을 통과하는 루프를 생성하고, 그렇게 하면서 열에 대한 셀에서 인덱스된 타일이 0으로 설정되어 있는지 확인합니다. 타일 인덱스가 아닌 경우 해당 셀은 0으로 설정되고 루프가 다른 검사 세트를 수행하기 전에 임시(로컬) 변수가 증가합니다.
