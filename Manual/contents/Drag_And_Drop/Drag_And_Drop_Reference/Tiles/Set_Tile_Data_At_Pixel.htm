픽셀에서 타일 데이터 설정

이 작업으로 타일 맵 레이어의 특정 위치에 있는 타일 셀에 대한 타일 데이터를 설정할 수 있습니다. 타일 데이터는 단순히 타일의 인덱스와 함께 뒤집기, 미러링 및 회전 값과 사용한 사용자 지정 타일 마스크를 반영하는 값입니다. 이 작업을 통해 먼저 룸 편집기에 정의된 레이어 이름(문자열)을 사용하여 대상에 레이어를 제공한 다음 타일 데이터를 설정할 룸의 x 및 y 축을 제공합니다. 타일 데이터는 주어진 위치가 도달하는 타일 맵 "셀"에 대해 설정됩니다. 예를 들어 타일 세트 타일이 16x16인 경우 타일 맵 레이어는 16x16픽셀 셀의 타일을 보유하므로 이 작업을 사용하여 방 위치(8, 8)를 대상으로 하는 경우 타일을 변경하게 됩니다. 타일 맵의 셀(0, 0)입니다.

참고: 선택한 레이어는 룸 편집기에서 타일 맵 레이어로 정의되어 있어야 합니다. 그렇지 않으면 오류가 발생할 수 있습니다.

작업의 마지막 인수는 타일 데이터 자체입니다. 일반적으로 Get Tile Data At Pixel 작업을 사용하여 타일 맵 셀에 대한 타일 데이터를 검색한 다음 이 작업으로 다시 설정하기 전에 Set Tile Data Transform 작업을 사용하여 조작합니다. 타일 데이터에 대한 자세한 내용은 타일 맵 함수 의 GML 섹션을 참조하세요.
