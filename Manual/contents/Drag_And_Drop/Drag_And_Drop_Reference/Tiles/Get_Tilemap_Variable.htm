타일맵 변수 가져오기

이 작업을 사용하면 레이어의 타일 맵 요소에 대해 주어진 수의 변수 중 하나를 검색할 수 있습니다. 방 편집기에서 타일 레이어를 생성하면 이 레이어는 타일 맵 요소 를 보유하고 타일 세트의 타일로 채워집니다. 이 타일 맵 요소에는 오프셋 위치, 사용 중인 타일 세트, 너비, 높이 등과 같은 특정 값이 연관되어 있습니다. 얻을 수 있는 값의 전체 목록은 다음과 같습니다.

  • X 좌표 : 타일 맵 요소가 배치된 레이어의 x축을 따른 위치입니다.
  • Y 좌표 : 타일 맵 요소가 배치된 레이어의 y축을 따른 위치입니다.
  • 열 : 타일 맵 요소의 너비(타일 셀의 열 수로 정의됨)입니다.
  • Rows : 타일 맵 요소의 높이(타일 셀의 행 수로 정의됨).
  • 타일 너비 : 단일 타일 셀의 너비입니다.
  • 타일 높이 : 단일 타일 셀의 높이입니다.
  • Tile Set Resource : 그리기를 위해 타일 맵 요소에 할당된 타일 세트 리소스입니다.
  • 현재 프레임 : 애니메이션 타일에 대해 그려지는 현재 프레임입니다.
  • Mask : 타일 맵 요소에 대한 마스크 데이터입니다.
  • 전역 마스크 : 모든 타일 맵 요소에 대한 전역 마스크 데이터입니다.

이 작업을 사용할 때 타일 맵 요소 데이터를 가져올 레이어 이름(룸 편집기에 정의된 문자열)을 제공한 다음 검색하려는 데이터 유형(위 목록 참조)을 제공합니다. 반환된 값은 이전에 생성되었거나 새로운 임시 변수가 될 수 있는 대상 변수에 저장됩니다("Temp" 확인란을 선택한 경우). 더하기 아이콘을 클릭하여 추가 값을 검색할 수 있습니다. 액션 옆에 다른 변수를 선택하고 반환된 값을 저장할 다른 변수를 제공합니다.
