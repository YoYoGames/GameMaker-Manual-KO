픽셀에서 타일 인덱스 설정

이 작업으로 주어진 레이어의 타일을 새 인덱스로 설정할 수 있습니다. 레이어는 주어진 레이어 이름(룸 편집기에서 레이어 이름을 지정하는 데 사용되는 문자열)에서 지정한 다음 타일을 설정할 x 및 y 위치를 지정합니다. x 및 y 위치는 호출 인스턴스의 위치에 상대적인 것으로 플래그가 지정될 수 있으며 설정되는 타일은 주어진 위치가 방에 속하는 타일이 되고 사용된 타일 세트는 레이어에 할당된 타일이 됩니다. 룸 에디터에서 마지막으로 타일 인덱스를 제공하여 타일을 설정합니다(타일은 위에서 아래로 왼쪽에서 오른쪽으로 인덱싱되며 왼쪽 상단에는 0이 표시됨). 타일 인덱스를 항상 0으로 설정하여 타일을 지울 수 있습니다.

논쟁 설명
Layer 타일을 설정할 레이어
x 확인할 x 위치
y 확인할 y 위치
Tile 사용할 타일의 새 인덱스

예시:

위의 액션 블록 코드는 마우스 왼쪽 버튼 누름을 확인하고 감지되면 마우스 위치가 속하는 타일 셀의 인덱스를 가져옵니다. 그런 다음 이 타일 인덱스를 확인하고 0이 아니면 0으로 설정합니다.
