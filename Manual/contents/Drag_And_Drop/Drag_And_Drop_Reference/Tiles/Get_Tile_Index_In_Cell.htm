셀에서 타일 인덱스 가져오기

이 작업으로 주어진 레이어에서 타일 인덱스를 얻을 수 있습니다. 레이어는 주어진 레이어 이름(룸 편집기에서 레이어 이름을 지정하는 데 사용되는 문자열)에서 지정한 다음 타일 인덱스를 가져올 레이어 내 셀의 행과 열을 지정합니다. 마지막으로 반환된 타일 인덱스를 저장할 대상 변수를 제공합니다. 여기서 0은 타일이 없고 양의 정수는 타일의 인덱스입니다. 대상 변수는 이벤트가 끝날 때까지 사용할 수 있는 임시(로컬) 변수로 플래그가 지정될 수 있습니다.

동작 구문:

인수:

논쟁 설명
Layer 타일을 가져올 레이어
Column 셀의 열(왼쪽에서 오른쪽으로)
Row 셀의 행(위에서 아래로)
Target 반환 값의 대상 변수

예시:

위의 액션 블록 코드는 주어진 룸 레이어의 맨 위 행을 통과하는 루프를 생성하고, 그렇게 하면서 열에 대한 셀에서 인덱스된 타일이 0으로 설정되어 있는지 확인합니다. 타일 인덱스가 아닌 경우 해당 셀은 0으로 설정되고 루프가 다른 검사 세트를 수행하기 전에 임시(로컬) 변수가 증가합니다.
