시퀀스의 텍스트

텍스트 트랙은 편집기를 통해 시퀀스에 추가할 수 있습니다. 이것은 선택하여 수행됩니다. 캔버스 도구 상자에서 텍스트 트랙 버튼을 추가한 다음 캔버스를 클릭(및 선택적으로 드래그)하여 텍스트 트랙을 생성합니다.

이 페이지에서는 다음 항목을 다룹니다.

  1. 텍스트 만들기
  2. 텍스트 수정
  3. 글꼴 자산
  4. 매개변수
  5. 조사관

텍스트 만들기

텍스트 트랙을 만드는 방법에는 두 가지가 있습니다. 선택 후 도구 상자의 텍스트 도구를 사용하여 다음을 수행할 수 있습니다.

  • 캔버스에서 한 번 클릭: "래핑되지 않은" 텍스트 트랙을 만듭니다. 이것은 텍스트가 원하는 만큼 길어질 수 있고 한 줄에 유지됨을 의미합니다. 나중에 Inspector 를 통해 래핑을 쉽게 활성화할 수 있습니다.
  • 캔버스에서 클릭하고 끌기: "감겨진" 텍스트 트랙을 생성합니다. 즉, 텍스트는 프레임 내부에만 표시됩니다. 이 프레임의 크기는 텍스트 트랙을 만드는 동안 드래그하여 설정됩니다. 그 크기는 나중에 매개변수 트랙이나 인스펙터를 통해 변경할 수도 있습니다. 줄 바꿈된 텍스트 트랙을 사용하면 애니메이션을 적용하는 동안 텍스트를 훨씬 쉽게 정렬하고 일관된 결과를 얻을 수 있으므로 사용하는 것이 좋습니다.

텍스트 수정

새 텍스트 트랙을 생성하면 GameMaker는 검사기 를 열고 자동으로 텍스트 필드를 선택하여 텍스트 쓰기를 시작할 수 있습니다.

언제든지 트랙의 텍스트를 편집하려면 해당 트랙을 선택하고 Inspector로 이동하여 Text 필드를 다시 수정하면 됩니다.

크기 조정 동작

캔버스에서 텍스트 트랙의 크기를 조정하는 것은 텍스트 트랙이 래핑 되었는지 언 래핑되었는지에 따라 다르게 작동합니다.

래핑을 해제하면 스프라이트 및 개체의 크기를 조정하는 방식과 유사하게 텍스트 트랙의 크기가 조정됩니다. 그러나 래핑된 텍스트 트랙의 크기를 조정하면 축척 값을 수정하는 대신 프레임 크기 가 수정됩니다.
래핑된 텍스트 트랙의 배율을 조정하려면 대신 기즈모 를 사용하거나 배율 매개변수 트랙의 값을 수동으로 편집하십시오.

글꼴 자산

텍스트 트랙은 글꼴 자산 에 연결됩니다. 새 텍스트 트랙을 만들 때 프로젝트 내에서 글꼴 자산을 찾고 마지막으로 추가된 것을 사용합니다. 프로젝트에 글꼴이 없으면 새 글꼴 자산이 자동으로 생성됩니다.

글꼴을 삭제하면 해당 글꼴을 사용하는 모든 텍스트 트랙도 삭제되므로 시퀀스에서 사용되는 글꼴 자산을 실수로 삭제하지 않도록 하십시오.

매개변수

Position, Rotation, Scale, Origin 및 Color Multiply와 같이 스프라이트 트랙에 제공되는 모든 일반적인 매개변수를 수정하고 애니메이션할 수 있습니다. 이 매개변수를 사용하여 텍스트를 이동 및 회전하고, 더 크거나 작게 만들고, 색상과 알파를 변경할 수 있습니다.

애니메이션을 적용할 수도 있는 일부 텍스트 관련 매개변수도 제공됩니다.

  • 문자 간격: 각 문자 사이의 간격(픽셀)입니다.
  • 줄 간격: 각 텍스트 줄 사이의 간격(픽셀)입니다.
  • 단락 크기 조정: 텍스트의 각 단락 사이의 간격(픽셀 단위)입니다. 를 눌러 수동으로 줄 바꿈을 추가하면 새 단락이 시작됩니다..
  • 프레임 크기: 이것은 텍스트 프레임의 크기이며 인스펙터에서 줄 바꿈 이 활성화된 경우에만 사용됩니다(자세한 내용은 아래 섹션 참조).

조사관

Inspector를 통해서만 변경할 수 있는 속성이 있습니다. 이러한 속성은 매개변수 트랙이 아니라 전체 트랙에 적용되는 간단한 옵션이므로 애니메이션할 수 없습니다.

Inspector는 텍스트 트랙을 선택하면 다음 속성을 보여줍니다.

참고 Inspector는 텍스트 트랙을 생성하거나 선택할 때 자동으로 열려야 하지만 Inspector가 닫혀 있으면 상단의 Windows 메뉴에서 수동으로 열 수 있습니다.

일부 매개변수 트랙(예: Color, Character Spacing, Line Spacing 등)이 여기에 포함되며 변경하면 재생 헤드에 키프레임이 생성됩니다. 변경 사항 자동 기록 이 활성화되거나 선택한 키프레임을 수정합니다. 나머지 옵션은 "정적"이며(애니메이션할 수 없기 때문에) 다음과 같습니다.

  • 상단에는 선택한 글꼴 자산이 표시됩니다. 이 옵션을 사용하여 기존 글꼴 자산을 선택하거나 새 글꼴 자산을 만들어 선택한 텍스트 트랙에 할당할 수 있습니다.
  • 텍스트: 텍스트 트랙에 표시되는 텍스트입니다. 여기에 여러 줄을 입력할 수 있지만 빈 줄은 표시되지 않습니다.
  • 정렬: 여기에서 텍스트가 프레임 내에서 수평(왼쪽, 중앙, 오른쪽, 양쪽 맞춤) 및 수직(하단, 상단, 가운데) 정렬 방식을 변경할 수 있습니다. 랩 옵션이 활성화된 경우에만 적용됩니다.
  • 줄 바꿈: 텍스트가 정의된 프레임 내에서 자동으로 줄바꿈할지 여부를 제어합니다. 이 기능이 활성화되면 텍스트가 프레임 끝에 도달할 때 자동으로 다음 줄로 이동하고 텍스트가 프레임 높이보다 긴 경우 항상 잘립니다. 이것이 비활성화되면 모든 텍스트가 한 줄에 표시되고 프레임이 더 이상 캔버스에 표시되지 않습니다.
