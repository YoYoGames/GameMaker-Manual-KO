객체 변수

변수 버튼을 클릭하면 개체 변수 창이 열리고 개체의 이벤트 생성 전에 변수를 생성하는 데 사용할 수 있습니다.

이 창에서 추가 버튼을 클릭하여 목록에 새 변수를 추가할 수 있습니다. 그런 다음 변수의 이름 을 지정하고 유형 을 설정하고 속성 을 조정할 수 있습니다.

변수 이름은 문자로 시작해야 하며 문자, 숫자 및 밑줄 기호 '_'만 포함할 수 있으며 최대 길이는 64개입니다. 따라서 유효한 변수는 다음과 같습니다. fish, foo_bar, num1, 유효하지 않은 변수는 6fish, foo bar, 또는 *num. 변수의 이름을 지정했으면 다음 중 하나일 수 있는 해당 유형을 설정해야 합니다. 이 기능은 부모/자식 인스턴스로 작업할 때 특히 유용합니다. 정의된 변수 집합이 있는 부모 개체를 갖고 자식 개체에서 간단히 수정할 수 있기 때문입니다(아래의 개체 변수 및 자식 개체 섹션 참조). 자세한 내용) - 그리고 룸 편집기 에 인스턴스를 추가할 때( 레이어 > 인스턴스 레이어 섹션 참조), 룸에 배치된 개별 인스턴스에 대해 이러한 변수를 수정할 수도 있기 때문입니다.

생성할 수 있는 다양한 유형의 변수는 다음과 같습니다.

  • 실수: 실수는 정수가 아닌 모든 숫자이며 양수 또는 음수일 수 있습니다. 그래서, 124.5, 45639.566546456, 0.9, -45.5 etc...는 모두 실수의 예입니다. 모든 실수는 64비트 부동 소수점 값(더블)으로 저장됩니다. 실수 (및 정수 ) 데이터 유형에는 옵션을 클릭하여 액세스할 수 있는 값 범위를 설정하는 옵션 이 있습니다. 단추: 이것을 확인하면 시작 값과 끝 값을 입력할 수 있으며 개체 변수 창에서 변수에 대해 고정 값을 표시하는 대신 다음 값으로 고정되는 슬라이더를 갖게 됩니다.
  • 정수: 정수는 정수이며 양수 또는 음수일 수 있습니다. 예를 들면 30004, 19, 0, -300. 실수에 대해 위에서 설명한 것처럼 출력 정수에 대한 값 범위를 설정할 수 있습니다.
  • 문자열: 문자열은 "fish", "Hello World" 또는 "12345"와 같이 큰따옴표로 묶인 모든 것입니다.
  • 부울: 부울은 true 또는 false인 값입니다. Object Editor Variables 창에서 이것은 단순히 true를 선택하고 false를 선택 취소하는 상자로 표시됩니다.
  • 표현식: 표현식은 일반 숫자, 변수, 문자열 또는 GML 함수와 하나 이상의 연산자를 포함할 수 있는 수학 구입니다. 예를 들어 sqrt(85 * 6) + 5.5 표현이다.
  • 리소스: 리소스는 단순히 자산 브라우저에서 정의한 리소스 중 하나입니다. 이 리소스 유형을 선택하면 자산 탐색기 열기 를 클릭할 수 있습니다. 버튼을 눌러 자산 탐색기를 열고 필요한 리소스를 선택합니다.
  • 목록: 입력 목록 유형을 선택하면 값(문자열, 실수, 표현식 등이 될 수 있음)을 선택한 다음 반환할 변수에 대해 하나 이상을 선택할 수 있습니다. 목록에 들어갈 항목을 정의하려면 먼저 옵션 에 추가해야 합니다( 단추):
    드롭다운 메뉴에서 기본 창의 변수에 대한 기본값으로 항목을 선택할 수 있습니다.
    목록 옵션에서 다중 선택 을 선택하면 변수는 선택한 모든 옵션의 배열이 됩니다.
  • 색상: 색상 유형은 변수에 저장할 색상 값을 정의하는 것입니다. 실수(0 ~ 16777216, 알파 값 255) 또는 16진수 값($RRGGBBAA 형식)을 입력하거나 두 번 클릭할 수 있습니다. 색상 견본에서 색상 선택기 를 열고 색상을 정의합니다.

객체 변수와 자식 객체

개체 변수의 중요한 기능은 자산 브라우저에 있는 모든 하위 개체에 의해 상속 된다는 것입니다(자세한 내용은 상위 개체 섹션 참조). 당신이 원한다면 그들.

변수가 정의되어 있는 다른 개체 리소스의 자식 개체를 만들면 개체 변수 창에 다음과 같이 표시됩니다.

상위 이미지에서 상위 개체에 4개의 개체 변수가 있고 아래쪽 이미지에서도 이러한 변수가 "회색으로 표시"되고 상위에서 상속 됨을 볼 수 있습니다. 아이콘은 상속되었음을 나타냅니다. 그런 다음 변수 재정의 를 클릭하면 이러한 상속된 변수를 편집할 수 있습니다. 버튼을 눌러 범위 슬라이더 또는 값을 조정하거나 목록에서 다른 항목을 선택하는 등... 상위 변수를 편집할 때 정의된 값만 변경할 수 있고 이름은 변경할 수 없으며 변수를 변경할 수도 없습니다 옵션. 또한 새 개체 변수를 자식 개체에 추가할 수도 있습니다(위의 예제 이미지에서 "문자 유형"이 자식 개체에 대한 새 개체 변수임을 알 수 있음).

상속된 개체 변수를 편집한 경우 삭제를 클릭할 수 있습니다. 버튼은 편집을 제거하지만 변수는 제거하지 않습니다. 변수는 여전히 상위에서 상속되지만 상위 기본값을 다시 사용합니다. 변수를 완전히 제거해야 하는 경우 상위 개체에서 이 작업을 수행해야 합니다.
