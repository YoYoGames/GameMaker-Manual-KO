타임라인 편집기

대부분의 게임에서 정확한 시간에 특정 일이 일어나기를 원할 것입니다. 이제 알람 이벤트를 사용하여 이를 달성할 수 있지만 상황이 너무 복잡해지면 특히 12개의 알람으로 제한되기 때문에 더 이상 작동하지 않습니다. 이것이 타임라인 자산이 있는 이유입니다. 타임라인에서 게임 시간의 특정 순간에 발생해야 하는 작업을 지정하고 다른 이벤트 또는 코드에서 개체에 사용할 수 있는 모든 작업을 사용하여 이를 정의할 수 있습니다. 타임라인을 생성하고 나면 그것을 객체에 할당할 수 있으며, 그 객체의 인스턴스는 방 안에 배치되거나 생성될 때 지정된 시간에 작업을 실행합니다.

예를 들어 가드를 만들고 이 가드가 왼쪽으로 20단계 이동한 다음 위쪽으로 10단계, 오른쪽으로 20단계, 아래쪽으로 10단계 이동한 다음 마지막으로 중지해야 한다고 가정합니다. 이를 달성하기 위해 왼쪽으로 모션을 설정하는 것으로 시작하는 타임라인을 만들 수 있습니다. 그런 다음 순간 20("순간"은 "게임 단계" 또는 "게임 프레임"을 말하는 또 다른 방법임)을 만들고 모션을 위쪽으로 설정한 다음, 순간 30에서 오른쪽으로, 순간 50에서 아래쪽으로, 그리고 마침내 순간 60에서 당신은 움직임을 멈출 것입니다. 완료되면 이 타임라인을 가드 개체에 할당할 수 있으며 가드 인스턴스는 방에 배치할 때 계획한 대로 정확하게 수행합니다. 일정 시간이 지나면 알람이 반복되도록 타임라인에 알람을 설정할 수도 있습니다. 이렇게 하면 경비원이 해당 지역을 순찰할 수 있습니다. 또한 타임 라인을 사용하여 게임을 보다 전역적으로 제어할 수 있습니다. 예를 들어 보이지 않는 컨트롤러 개체를 만든 다음 특정 순간에 적을 생성하는 타임 라인을 만든 다음 컨트롤러 개체에 할당하고 방에 인스턴스를 배치하는 방법입니다. 이것은 스페이스 쇼터 게임이나 그와 비슷한 게임에서 적의 시간에 맞춰 스폰되는 것을 쉽게 얻을 수 있는 방법입니다.

새로운 순간을 추가할 때 GML Visual 과 GML Code 중에서 선택하라는 메시지가 표시될 수 있습니다. 자세한 내용은 GameMaker 언어 를 참조하십시오.

타임라인을 만들려면 마우스 오른쪽 버튼을 클릭하세요. 자산 브라우저 에서 "타임라인 생성"을 선택합니다. 그러면 이 페이지 상단에 다음 섹션이 포함된 창이 열립니다.

이름

여기에서 타임라인 리소스에 이름 을 지정합니다. 이것은 영숫자 문자 또는 "_" 밑줄만 포함해야 하며 코드를 통해 타임라인을 참조하는 데 사용되므로 숫자로 시작할 수 없습니다. 이 이름은 GML Visual 작업 또는 코드 기능에서 타임라인을 식별하는 데 사용할 수 있는 고유한 타임라인 ID이기도 합니다.


순간

여기에서 모멘트 값을 설정할 수 있습니다. 모멘트 값은 타임라인이 시작된 후 게임 단계 수로 계산됩니다. 따라서 위치 0과 60에 모멘트가 있는 타임라인이 있는 경우 타임라인이 시작될 때 모멘트 0이 트리거되고 60단계 후에 모멘트 60이 발생합니다. 룸 속도도 60으로 설정되어 있으면 타임라인이 시작된 후 1초 후에 60이 발생합니다. 더블클릭 하시면 됩니다 순간 값에 또는 오른쪽 버튼을 사용할 수 있습니다 메뉴를 선택하고 " 변경 "(아래 버튼 메뉴 섹션 참조)을 선택하여 변경합니다. 새로운 순간을 추가하려면 편집기 하단에 있는 추가 버튼을 클릭하기만 하면 됩니다. 새로운 순간을 추가하면 해당 순간에 호출하려는 함수 또는 GML Visual을 추가할 수 있는 코드 편집기 (또는 작업 편집기)가 열립니다.


모멘트 이름

타임라인이 무엇을 하고 있는지 이해하는 데 도움이 되도록 해당 순간을 식별할 수 있는 이름을 지정하는 것이 허용됩니다. 이것은 단순히 문자열이며 게임 코드나 다른 곳에서는 사용되지 않습니다. 순전히 타임라인이 하는 일을 구성하고 이해하기 위한 것입니다. 순간에 이름/설명을 추가하려면 /// @description 코드 블록의 첫 번째 줄에 형식을 지정합니다. 예를 들면 다음과 같습니다.

/// @description Hello World

다음과 같이 표시됩니다.

타임라인 편집기를 사용하면 + 첫 번째 클릭 사이의 모든 순간 선택 그리고 두 번째 클릭, 또는 다음을 사용할 수 있습니다. / + 단일 순간을 클릭하여 개별적으로 선택합니다. 타임라인 순간은 마우스 오른쪽 버튼을 사용하여 편집할 수 있습니다. 선택한 요소 중 하나의 메뉴:

  • 추가: 이 옵션을 사용하면 타임라인에 새로운 순간을 추가할 수 있습니다. 마지막 순간 바로 뒤에 추가되므로 타임라인의 마지막 순간이 120이라면 다음에 추가하는 순간은 121이 됩니다. 더블 클릭하면 순간 위치를 변경할 수 있습니다. 편집기의 모멘트 값에서 또는 마우스 오른쪽 버튼을 클릭하여 " 변경 "을 선택합니다.
  • 편집: 편집을 위해 현재 타임라인 모멘트를 엽니다.
  • 모멘트 변경: 선택한 타임라인 모멘트의 값을 변경합니다. 이것은 느린 더블 클릭과 동일합니다. 편집기 창의 타임라인 값에서
  • 설명 변경: 선택한 타임라인 모멘트를 설명하는 이름을 변경합니다. 이것은 느린 더블 클릭과 동일합니다. 편집기 창의 타임라인 설명에서
  • 복제: 선택한 타임라인 모멘트를 모두 복제하여 현재 타임라인의 끝에 연속적으로 추가합니다.
  • 병합: 타임라인에서 두 개 이상의 순간을 선택하면 병합하도록 선택할 수 있습니다. 이렇게 하면 모든 코드와 작업이 동시에 선택된 첫 번째 순간의 위치에 단일 타임라인 순간이 생성됩니다.
  • 제거: 타임라인에서 선택한 순간을 제거합니다.
  • 잘라 내기/ 복사 / 붙여넣기: 이 옵션을 사용하면 하나 이상의 타임라인 순간을 잘라내거나 복사하거나 붙여넣을 수 있습니다. 여기에서도 표준 키보드 단축키를 사용할 수 있습니다. / + "X ", / + "C ", 그리고 / + "V ".
  • 모두 선택: 이 옵션을 사용하면 타임라인의 모든 순간을 선택할 수 있습니다. 그런 다음 선택한 모멘트를 제거, 병합, 복제, 잘라내기 또는 복사할 수 있습니다.
  • GML Visual / Code로 변환: 코드를 사용하거나 GML Visual 작업을 사용하여 프로젝트를 생성할 때 선택한 방법을 사용하여 타임라인 모멘트가 생성됩니다. 이 옵션을 사용하면 프로젝트 유형에 관계없이 하나에서 다른 것으로 변환할 수 있습니다.
