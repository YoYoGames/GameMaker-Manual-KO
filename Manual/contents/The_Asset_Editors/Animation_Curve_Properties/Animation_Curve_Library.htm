애니메이션 곡선 라이브러리

애니메이션 곡선 라이브러리 창에서 애니메이션 곡선의 곡선 유형을 변경하고 선택한 채널에 사전 설정을 적용할 수 있습니다. 에 의해 열 수 있습니다. 왼쪽 클릭 애니메이션 곡선 편집기 또는 시퀀스 도프 시트 의 버튼 또는 채널 편집기에서 아무 곳이나 마우스 오른쪽 버튼으로 클릭합니다.

곡선 유형

곡선 유형 은 채널이 점에서 점으로 흐르는 방식을 제어하며 다음 옵션 중 하나로 설정할 수 있습니다.

  • 선형: 채널의 포인트 사이에 선형 진행을 생성합니다.
  • Smooth: 채널의 포인트 사이를 부드럽게 진행하기 위해 catmull-rom 보간 을 사용합니다.
  • 베지어: 베지어 곡선 수학을 사용하여 채널의 점 사이에서 부드러운 진행을 생성합니다. 여기에서 점의 "핸들"을 조작하여 곡선 모양을 제어할 수 있습니다.

곡선 유형을 변경하면 해당 유형이 애니메이션 곡선의 모든 채널에 적용됩니다.

곡선 사전 설정

곡선 사전 설정 은 곡선의 모양을 변경합니다. 사전 설정 메뉴에서 사전 설정을 마우스 왼쪽 버튼으로 클릭하면 해당 사전 설정이 선택한 채널에 적용되고 재설정 을 클릭하면 애니메이션 곡선 라이브러리 창이 열리기 전의 채널로 되돌아갑니다.

참고: 사전 설정은 곡선 유형이 베지어로 설정된 경우에만 적용할 수 있습니다.

기본적으로 사전 설정은 전체 채널에 적용됩니다. 드래그하여 채널 섹션을 선택할 수 있습니다. 채널 편집기 에서 마우스 왼쪽 버튼을 클릭한 다음 버튼 또는 마우스 오른쪽 버튼을 클릭하면 선택한 섹션에만 사전 설정이 적용됩니다.

적용된 보간은 시작점과 끝점 사이의 수직 차이를 기반으로 하기 때문에 곡선 채널(또는 해당 채널 내 선택)에 효과가 있으려면 사전 설정에 대한 기울기가 있어야 합니다. 평평한 곡선은 두 점 사이에 이러한 차이가 없으므로 이러한 곡선에 사전 설정을 적용한 후에는 결과를 볼 수 없습니다.

선택할 수 있는 두 가지 적용 모드 가 있으며 이는 사전 설정이 채널(또는 해당 채널 내 선택 항목)에 적용되는 방식에 영향을 줍니다.


  • 덮어쓰기: 창을 통해 창을 열 때 기본값입니다. 애니메이션 곡선 편집기 또는 시퀀스 도프 시트에서 버튼을 클릭합니다. 채널 또는 선택 항목의 시작 지점과 끝 지점 사이의 모든 지점을 덮어씁니다.
  • 사이: 창을 열 때 기본값입니다. 채널 편집기에서 아무 곳이나 마우스 오른쪽 버튼으로 클릭합니다. 선택한 포인트의 각 쌍 사이에 사전 설정을 적용하고 기존 포인트를 제거하지 않고 필요한 새 포인트를 추가합니다.

다음 영상은 사이 및 덮어쓰기 적용 모드를 사용하여 채널에 적용된 노출 곡선 사전 설정:

위의 시각적 개체에서 볼 수 있듯이 비트윈 은 각 점 쌍에 곡선 사전 설정을 적용하고 필요한 새 점을 삽입합니다. 덮어쓰기 는 시작 지점과 끝 지점 사이의 모든 지점을 제거한 다음 사전 설정을 적용합니다.

곡선 가속 또는 감속 방법을 제어하는 세 가지 이징 모드 중에서 선택할 수도 있습니다. 기본 여유 모드는 천천히 나가기 입니다.

  • Ease In: 커브는 더 빠른 가속으로 시작하여 감속합니다.
  • Ease Out: 곡선은 더 느린 가속으로 시작하여 시간이 지남에 따라 가속됩니다.
  • Ease In-Out: 곡선은 더 느린 가속으로 시작하여 시간이 지남에 따라 가속된 다음 감속합니다.

다음 시각적 개체는 세 가지 이징 모드에서 사전 설정된 Quart 곡선을 보여줍니다.
