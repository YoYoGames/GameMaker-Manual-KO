글꼴 편집기

게임에서 텍스트를 그리고 싶을 때 이 텍스트는 표준 Arial 12포인트 글꼴 로 그려지지만 더 흥미롭거나 독특한 모양의 텍스트를 만들려면 다른 글꼴을 사용하고 싶을 것입니다. 따라서 컴퓨터에 있는 다른 글꼴을 사용하려면 GameMaker 에서 글꼴 자산 을 생성해야 합니다. 이것은 마우스 오른쪽 버튼을 클릭하여 수행됩니다. 자산 브라우저 에서 " 만들기" - > "글꼴 "을 선택합니다. 생성하는 각 글꼴 자산에 대해 컴퓨터에서 특정 유형의 글꼴을 지정하면 그리기용 글꼴을 설정하는 작업이나 코드를 사용하여 게임에서 사용할 수 있습니다.

글꼴 리소스를 처음 생성하면 글꼴 편집기 창이 다음 옵션과 함께 열립니다.

세부

여기에서 글꼴에 고유한 이름(영숫자 문자 및 "_" 밑줄 기호만 해당)을 지정하고 사용할 글꼴을 선택할 수 있습니다. 글꼴은 컴퓨터에 설치해야 하며 라이선스 문제로 인해 GameMaker 는 기본적으로 실제 글꼴 파일을 프로젝트 파일과 함께 저장하지 않지만 글꼴은 일련의 비트맵 글리프로 텍스처 페이지 에 렌더링 됩니다. 텍스처 페이지는 텍스처 그룹 에 할당되며 여기에서도 그룹 옵션에서 선택할 수 있습니다.

실제 폰트 파일( *.ttf 또는 *.otf 만) Marketplace 자산을 만들 때 또는 다른 이유로 프로젝트와 함께 Copy To Project 옵션을 선택할 수 있습니다. 그러면 글꼴 파일을 선택할 수 있는 파일 탐색기가 열리고 프로젝트와 함께 번들로 제공되는 파일에 추가됩니다.

중요한! 프로젝트에서 재배포 또는 사용이 법적으로 허용되고 라이선스가 부여된 글꼴만 사용해야 합니다.

사전 설정

이 섹션에서는 크기(포인트 단위 - 기본값은 12pt)와 글꼴을 굵게 또는 기울임꼴로 그릴지 또는 앤티앨리어싱 을 적용할지 여부를 설정할 수 있습니다. 오른쪽의 미리보기 창에는 지정한 크기와 변형으로 게임에서 보이는 텍스트가 대략적으로 표시됩니다.


범위 제어

글꼴 리소스를 만들 때 글꼴에 대한 모든 단일 글리프를 포함하려는 경우는 드뭅니다. 이것이 GameMaker 가 선택된 글꼴의 32에서 127 사이의 문자만 사용하도록 기본 설정하는 이유입니다. 이렇게 하면 특정 구두점과 함께 숫자, 소문자 및 대문자가 제공되기 때문입니다. 그러나 게임을 최적화하는 데 사용되는 문자 범위를 변경할 수 있습니다. 예를 들어 특정 글꼴의 숫자만 필요한 경우 숫자 만 선택합니다. 이렇게 하려면 먼저 삭제 버튼을 클릭하여 현재 범위를 지워야 합니다. 미리보기 창에서)를 클릭한 다음 추가 버튼을 클릭하여 새 범위를 추가합니다. 그러면 다음과 같은 연결된 창이 열립니다.

이 창에는 글꼴의 표준 범위를 설정하는 데 도움이 되는 몇 가지 버튼이 있습니다.

- 32에서 127까지의 일반 범위

- ASCII 범위는 0에서 255까지입니다.

- 10자리만 포함하는 Digits 범위

- 모든 대문자와 소문자만 포함하는 문자 범위

다른 범위는 문자 범위 입력란에 설정하고자 하는 범위의 첫 번째와 마지막 문자 인덱스를 입력하여 사용할 수 있습니다(문자가 범위에 있지 않은 경우 공백으로 대체됨). 이러한 버튼과 입력 상자 외에도 특히 유용한 두 가지 추가 옵션이 있습니다.

  • 코드에서: 이 버튼을 클릭한 다음 확인을 클릭하면 GameMaker 는 자동으로 게임 코드에서 문자열을 구문 분석한 다음 게임의 모든 텍스트를 포함하는 다양한 문자 범위를 생성합니다. 모든 문자열("" 또는 '' 내)을 검색하므로 문자 범위에 파일 이름도 포함될 수 있습니다.
  • 파일에서: 위에서 설명한 "코드에서" 옵션과 유사하게 작동합니다. 그것을 클릭하면 파일을 제공하라는 메시지가 표시되고 완료되면 확인을 클릭해야 합니다. 그러면 GameMaker 는 파일을 구문 분석하고 그 안에 있는 텍스트에 대한 문자 범위를 생성합니다.

미리보기 창에 직접 텍스트 글리프를 추가하거나 삭제하여 텍스트 범위를 수동으로 편집할 수 있습니다.

텍스처 컨트롤

위에서 언급했듯이( 글꼴 세부 정보 섹션에서) 기본적으로 모든 글꼴 글리프는 그래픽 리소스로 변환하기 위해 텍스처 그룹의 일부로 텍스처 페이지에 렌더링 되며, 게임이 컴파일되면 글꼴이 임시로 추가됩니다. 텍스처를 사용하여 필요한 모든 글꼴 글리프가 있는 단일 이미지를 만든 다음 내보내기를 위해 기본 텍스처 그룹에 추가됩니다. 여기에 있는 버튼은 글꼴 텍스처가 생성되는 방식을 조정하거나 새 글꼴 텍스처를 재생성하는 데 사용됩니다.

옵션 버튼은 고급 글꼴 텍스처 옵션을 엽니다.

이러한 옵션은 때때로 근본적으로 게임에서 글꼴이 표시되는 방식을 변경할 수 있으며 모든 글꼴에 동일한 설정이 필요한 것은 아닙니다. 다음 옵션은 글꼴을 최대한 사용자 지정할 수 있도록 설계되었습니다.

  • 힌트: 글꼴 힌트(지시라고도 함)는 수학적 지침을 사용하여 래스터화된 그리드와 정렬되도록 윤곽선 글꼴의 표시를 조정하는 것입니다. 낮은 화면 해상도에서 힌트는 명확하고 읽기 쉬운 텍스트를 생성하는 데 중요하며 여기에서 다양한 힌트 알고리즘을 선택하여 최상의 결과를 얻을 수 있습니다. 기본값은 "보통"입니다.
  • 글리프 옵션: 이 옵션은 다른 작업을 활성화/비활성화하여 글리프가 로드되는 방식을 변경하는 데 사용할 수 있습니다. 이것들은:
    1. 힌팅 비활성화: 글꼴 기본 힌팅과 자동 힌터를 비활성화합니다.
    2. 자동 힌터 비활성화: 자동 힌터를 비활성화하고 대신 글꼴 기본 힌터(있는 경우)를 사용합니다. 이것은 "Prefer Auto-hinter" 옵션보다 우선합니다.
    3. Prefer Auto-Hinter: 이것은 GameMaker 가 글꼴에 있을 수 있는 기본 힌트보다 자동 힌트를 사용하는 것을 선호하도록 지시합니다. 이것은 "Disable Auto-hinter" 및 "Disable Hinting" 옵션에 의해 무시됩니다.
    4. Warping: 이것을 활성화하면 그리드 정렬을 개선하기 위해 글리프가 약간 뒤틀리게 됩니다. 이를 위해서는 자동 힌터와 "일반" 힌팅 모드가 필요합니다.
    5. 배율 없음: 이 옵션을 선택하면 글리프를 글꼴 단위로 유지하고 요청된 크기를 무시합니다. 이것은 힌트가 비활성화되었음을 의미합니다.
    위의 모든 옵션에 대한 기본 상태는 비활성화 되어 있습니다.
  • 인터프리터: 여기에서 GameMaker 가 글꼴 렌더링에 사용해야 하는 TrueType 버전을 선택할 수 있습니다. 기본값은 TrueType v40 입니다.
  • 반올림: 고정 소수점 소수점 크기를 정수로 변환할 때 반올림 방법을 설정하는 옵션입니다. 기본값은 "바닥"입니다.
  • 커닝 적용: 게임 내 렌더링에 글꼴 파일의 커닝 데이터를 사용할지 여부를 제어합니다. 소스 글꼴의 커닝 데이터가 올바르지 않아 글꼴이 올바르게 표시되지 않는 경우 이 기능을 비활성화해야 합니다.

재생성 버튼은 글꼴 텍스처를 재생성하는 데 사용할 수 있습니다. 기존 글꼴에 대한 변경 사항을 업데이트하려는 경우 또는 새 글꼴을 생성하고 미리 보기 창에서 어떻게 보이는지 확인하려는 경우에 필요합니다. 오른쪽 버튼에서 글꼴을 다시 생성할 수도 있습니다. 자산 브라우저의 글꼴(또는 글꼴)에 대한 메뉴입니다.

새 글꼴을 처음 생성할 때 편집기에서 텍스처를 재생성하지 않으면 게임을 처음 실행하거나 테스트할 때 텍스처가 생성되지만 그 때만이 자동으로 생성됩니다. 즉, 글꼴에 텍스처가 생성되면 글꼴 리소스에 대한 모든 변경 사항은 변경 사항을 게임에서 볼 수 있도록 글꼴을 재생성해야 합니다. 그렇지 않으면 "이전" 글꼴 텍스처가 계속 사용됩니다.

GameMaker 는 생성된 글꼴 텍스처를 다음과 같이 렌더링합니다. PNG 파일을 나머지 프로젝트 파일과 함께 가져오고 프로젝트를 로드하고 기본 글꼴을 찾을 수 없는 경우 게임은 저장된 텍스처를 사용하여 여전히 텍스트를 올바르게 표시합니다. 그러나 누락된 글꼴은 이름, 글꼴 패밀리 및 텍스처 그룹( 텍스처 그룹 편집기 에서) 이외의 속성을 더 이상 변경할 수 없음을 의미합니다.

미리보기 모드

이 섹션에서 미리보기 창에 선택한 글꼴의 일부 샘플 텍스트 또는 글꼴에 대해 생성된 렌더링된 텍스처 이미지가 표시되도록 선택할 수 있습니다.

시사

미리보기 창 은 글꼴의 미리보기를 텍스트 샘플이나 텍스처로 볼 수 있는 곳입니다. 미리보기 모드 가 샘플 로 설정되어 있으면 테스트하려는 텍스트를 미리보기 창에 직접 입력하여 모양을 확인할 수 있으며, 게임 미리보기 를 선택하면 해당 글꼴에 대해 생성된 렌더링된 텍스처 이미지가 표시됩니다. 이미지가 없으면 재생성 버튼을 클릭하여 텍스처를 생성해야 합니다.

텍스처 이미지는 샘플과 다를 수 있습니다. 명시적으로 지시하지 않는 한 텍스처가 재생성되지 않기 때문입니다. 즉, 처음에 글꼴 텍스처를 생성한 후에는 텍스처가 재생성되지 않는 한 글꼴 리소스에 대한 변경 사항이 게임에 적용되지 않습니다.

범위 미리보기

여기에 글꼴에 추가한 다양한 문자 범위가 나열됩니다. 범위 값은 왼쪽에 표시되고 해당 값에 해당하는 범위 글리프는 오른쪽에 표시됩니다.
