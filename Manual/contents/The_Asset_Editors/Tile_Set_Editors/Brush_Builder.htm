타일 세트 브러시 빌더

기본적으로 룸 편집기에서 타일 맵 레이어에 타일을 "페인트"할 때 단일 타일을 선택하고 페인트합니다. 그러나 타일 세트는 거의 항상 전체 영역 또는 항목을 형성하기 위해 서로 다른 방식으로 함께 맞는 섹션을 갖도록 설계되었습니다. 예를 들어, RPG 타일 세트에는 사용된 타일 수에 따라 나무와 건물과 같은 더 크거나 작은 기능을 만들기 위해 연결할 수 있는 가로 기능 타일이 있을 수 있습니다. 이제 방 레이어에 이와 같은 여러 기능을 배치하면 타일을 변경하기 위해 여러 번 앞뒤로 이동해야 하므로 워크플로에 좋지 않습니다. 이 문제를 해결하기 위해 GameMaker 를 사용하면 타일 세트 편집기에서 타일 브러시 를 생성할 수 있습니다. 브러시 빌더 버튼에서:

브러시 빌더 에서 왼쪽에는 원래 타일 세트가 있고 오른쪽에는 빈 "캔버스"가 있습니다. 이제 왼쪽에서 타일을 선택하고 오른쪽에서 페인트하여 사용자 정의 "브러시"를 만든 다음 룸 편집기에서 사용할 수 있습니다. 마우스 왼쪽 버튼을 클릭한 상태로 유지할 수 있습니다. 그런 다음 타일 세트를 드래그하여 브러시 캔버스에 칠할 여러 타일을 선택하고 / + 특정 개별 타일을 추가하거나 길게 누르기 + 특정 타일을 제거합니다.

다음 이미지는 단일 타일 세트로 만든 세 가지 사용자 정의 브러시의 예로 앞서 언급한 풍경 기능을 보여줍니다.

오른쪽에서 우리가 만든 세 가지 기능을 볼 수 있습니다(이미지에서 주황색 상자로 강조 표시됨). 각 기능 사이에 타일 하나의 간격을 남겨둔 방법에 주목하십시오. 이는 터치하는 타일 그룹이 룸 편집기에서 단일 브러시로 처리 되므로 각 세트가 하나의 타일임을 나타내기 위해 타일 하나의 간격을 남겨두기 때문입니다. 우리가 만들고 싶은 독특한 브러시. 브러시를 만드는 동안 마우스 왼쪽 버튼으로 페인팅합니다. 그리고 마우스 오른쪽 버튼으로 삭제. 다음을 사용하여 타일 시트 또는 브러시 캔버스를 확대/축소할 수도 있습니다. / 그리고 마우스 휠 마우스가 창 중 하나 위에 있을 때 또는 + 또는 마우스 가운데 버튼 그리고 드래그. 편집기 창 내 도구 상자 의 확대/축소 도구를 사용하여 창의 확대/축소 수준을 변경하고 그리드 가시성과 색상을 토글할 수도 있습니다.

오른쪽 상단에서 현재 선택된 도구를 볼 수 있으며 페인트할 브러시의 크기도 설정할 수 있습니다. 기본 크기는 단일 타일인 1이지만 더 높은 값으로 설정하면 아래 이미지와 같이 선택한 타일로 구성된 더 큰 브러시를 반복하여 칠하고 지울 수 있습니다.

도구 상자는 타일 세트 편집기에서 다양한 작업에 사용할 도구를 선택할 수 있는 곳이며, 그 중 일부는 자동 타일 라이브러리에 정의된 항목이 있는지 여부에 따라 다릅니다. 각 도구에 대한 간략한 개요가 아래에 나와 있습니다(룸 편집기에서 타일 레이어를 선택하면 이 도구 상자가 추가 그리기 도구와 함께 공간 작업 공간의 맨 위에 표시됩니다).

연필 이것은 연필 도구입니다. 선택한 타일을 사용하여 마우스 왼쪽 버튼으로 브러시 창에 페인팅합니다. 마우스 오른쪽 버튼으로 지울 수 있습니다.
지우개 지우개 도구를 사용하면 마우스 왼쪽 버튼을 사용할 수 있습니다. 브러시 창에서 지정된 타일을 지웁니다. 기본적으로 이 모든 작업은 타일 인덱스를 예약된 "빈" 타일인 0으로 설정하는 것입니다.
선택 이것은 작업을 위한 브러시 창의 영역을 정의하는 데 사용할 수 있는 선택 도구입니다. 마우스 왼쪽 버튼을 클릭할 수 있습니다 그런 다음 마우스를 끌어 직사각형 영역을 만들거나 / + 여러 선택 항목을 추가하고 + 선택 항목의 일부를 지웁니다. 전체 선택을 지우려면. 브러시 창의 영역을 선택하면 나머지 도구(연필, 뒤집기, 회전 등...)는 선택한 영역 내에서만 작동합니다. 창에서 타일을 선택한 다음 복사( / + "C "), 자르다 ( / + "X ") 및 붙여넣기( / + " V ") 도구를 연필로 전환하면 도구를 사용하여 "페인트"할 수 있습니다.
자동 타일 이 도구를 클릭하면 자동 타일링 페인트 스타일이 활성화됩니다. 이 기능이 활성화되면 자동 타일 라이브러리 탭에서 타일을 선택한 다음 브러시 창에 페인트할 수 있습니다. 자동 타일 탭 을 올바르게 설정 했다면 GameMaker 가 주변 타일과 일치하도록 자동으로 변경합니다. 자동 타일 라이브러리의 일부가 아닌 타일 세트에서 타일을 선택하면 그리기 도구가 표준 연필 도구로 재설정됩니다.
튀기다 마우스 왼쪽 버튼으로 뒤집기 도구 클릭 (또는 키보드 단축키 "X " )는 그리기 도구를 변경하지 않고 수평 축을 따라 현재 그리기 위해 선택한 타일을 뒤집습니다(그리기를 위해 선택한 사용자 정의 브러시가 있는 경우 전체 브러시가 뒤집힙니다). 그리기 위해 선택한 타일이 없는 경우 브러시 창에서 타일 그룹을 선택하면 뒤집기 도구가 선택한 타일을 뒤집습니다.
거울 마우스 왼쪽 버튼으로 미러 도구 클릭 (또는 키보드 단축키 " Y " )는 그리기 도구를 변경하지 않고 수직 축을 따라 그리기 위해 현재 선택된 타일을 미러링합니다(그리기를 위해 선택한 사용자 정의 브러시가 있는 경우 전체 브러시가 미러링됩니다). 그리기 위해 선택한 타일이 없지만 브러시 창에서 타일 그룹을 선택하면 미러 도구가 선택한 타일을 미러링합니다.
회전 마우스 왼쪽 버튼으로 회전 도구 클릭 (또는 키보드 단축키 " Z " )는 그리기 도구를 변경하지 않고 현재 그리기 위해 선택한 타일을 시계 방향으로 90° 회전합니다(그리기를 위해 선택한 사용자 정의 브러시가 있는 경우 전체 브러시가 회전합니다). 그리기 위해 선택한 타일이 없지만 브러시 창에서 타일 그룹을 선택하면 회전 도구가 선택한 타일을 회전합니다.

도구 아래에서 현재 타일 세트 이미지를 사용하여 생성된 자동 타일 또는 애니메이션 타일 을 선택하기 위한 두 가지 섹션을 찾을 수 있습니다. 타일 세트에 사용되는 단일 스프라이트에는 많은 단일 셀 이미지가 포함될 수 있으며 애니메이션 또는 자동 타일 편집기에서 결합하여 이러한 섹션에 표시되고 함께 사용할 수 있는 사용자 정의 브러시를 생성할 수 있습니다. 일반 정적 타일을 사용하여 브러시를 만듭니다(애니메이션 타일은 라이브러리에서 선택했는지 또는 기본 타일 세트에서 선택했는지에 관계없이 애니메이션됩니다.)

필요한 모든 브러시를 설정했으면 이를 사용하여 룸 편집기 내에서 타일 맵 레이어에 타일을 칠할 수 있습니다.
