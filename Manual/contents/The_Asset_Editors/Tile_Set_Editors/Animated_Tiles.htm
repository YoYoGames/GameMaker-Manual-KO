애니메이션 타일

타일은 일반적으로 게임룸 내에서 정적 셀로 간주되지만 GameMaker 를 사용하면 스프라이트처럼 타일을 애니메이션으로 만들 수 있습니다. 왜 스프라이트를 사용하지 않습니까? 음, 스프라이트는 경계 상자 및 렌더링 방식과 연결되어 있기 때문에 특정 처리 오버헤드가 있지만 타일은 처리 오버헤드가 훨씬 낮으므로 렌더링이 더 빠릅니다. 그러나 타일 세트 애니메이션을 사용하면 길이가 2의 거듭제곱이어야 하는 애니메이션 생성으로 제한된다는 점에서 대가가 따릅니다. 즉, 2, 4, 8, 16 등... 프레임의 경우 분명히 타일 그리드 내에서 생성 되며 모두 동일한 속도로(타일 세트별로) 애니메이션되어야 합니다. 그러나 이러한 제한에도 불구하고 애니메이션 타일은 정적인 환경과 배경에 생명력과 색상을 추가하는 데 사용할 수 있는 강력한 기능입니다.

타일 애니메이션을 생성하려면 먼저 하나의 스프라이트 이미지 내에 필요한 타일이 모두 포함된 타일 세트가 있어야 합니다. 타일 애니메이션 버튼을 클릭하여 애니메이션 편집기를 불러오는 Set Editor:

위의 예시 이미지는 모든 애니메이션 프레임이 포함된 단일 이미지이지만 단일 대규모 타일 세트를 사용한 다음 "체리 픽"할 수 있으므로 이미지에 다른 비애니메이션 타일도 포함될 수 있다는 점에 유의해야 합니다. 애니메이션을 만들고자 하는 부분. 예를 들어, 나무, 흙, 물이 있는 큰 풍경 타일 세트가 있고 타일 세트 내에서 물이 파문을 일으키도록 애니메이션할 여러 타일이 있을 수 있습니다.

타일 애니메이션을 만들려면 먼저 애니메이션 추가 버튼을 클릭하여 애니메이션 라이브러리에 애니메이션을 추가해야 합니다., 라이브러리에 빈 애니메이션 스트립을 추가하고 사용할 프레임 수를 선택하도록 요청합니다.

그러면 초기 프레임이 강조 표시된 각 빈 애니메이션 프레임의 보기가 표시됩니다.

그런 다음 편집기 왼쪽의 타일 세트 이미지로 돌아가면 아무 타일에서나 애니메이션에 추가하면 프레임이 다음 타일로 넘어갑니다. 클릭 연속 타일에서 애니메이션이 채워지고 프레임 옆에 있는 "재생" 버튼을 클릭하여 미리 볼 수 있습니다.

작업을 마치면 애니메이션이 애니메이션 라이브러리의 일부가 되고 룸 편집기 에 들어갈 때 선택하여 룸 안에 배치할 수 있습니다. 애니메이션에서 방에 배치할 타일을 선택하면 해당 타일이 선택한 타일부터 애니메이션되므로 방 안에 별도의 프레임을 배치하여 애니메이션을 "비동기화"할 수 있으며 모두 애니메이션됩니다. 이에 대한 유일한 주의 사항은 애니메이션 속도가 라이브러리 내의 개별 애니메이션이 아니라 전체 타일 세트에 대해 정의되기 때문에 항상 동일하다는 것입니다.

애니메이션에 사용되는 타일에 대해 주의할 몇 가지 사항:

  • 타일에 지정된 자체 애니메이션이 있는 경우(따라서 첫 번째 타일로 설정된 애니메이션이 있는 경우) 해당 애니메이션을 사용합니다.
  • 타일에 자체 애니메이션이 없지만 다른 하나의 애니메이션에서만 사용되는 경우 타일이 애니메이션에 있는 첫 번째 위치에서 시작하여 해당 애니메이션을 사용합니다.
  • 타일에 자체 애니메이션이 없고 여러 다른 애니메이션에서 사용되는 경우 전혀 애니메이션되지 않습니다. 첫 번째 타일로 애니메이션을 명시적으로 지정해야 합니다.
