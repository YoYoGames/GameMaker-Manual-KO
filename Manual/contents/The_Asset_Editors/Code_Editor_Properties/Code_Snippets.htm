코드 조각

코드를 편집할 때 사용할 수 있는 매우 편리한 도구는 코드 조각 을 사용하는 것입니다. 스크립트 편집기를 사용할 때 다음을 누를 수 있습니다. F4 목록에서 일반적으로 사용되는 코드 방법 중 하나를 선택할 수 있는 코드 조각 팝업을 엽니다.

이 창에서 사용할 스니펫을 마우스로 선택하거나 오른쪽에 나열된 관련 단축키를 누를 수 있습니다. 이렇게 하면 편집을 위해 코드에 스니펫이 추가됩니다.

원하는 경우 고유한 코드 조각을 정의할 수도 있습니다. 이 작업을 수행하기 전에 "snippets.txt " 다음 디렉토리 중 하나에서:

  • 윈도우: %ProgramData%\GameMaker\User\
  • 맥 OS: /Users/Shared/GameMaker/User/

이 폴더는 GameMaker 업데이트 중에 수정 되지 않으므로 편집된 파일은 그대로 유지됩니다(그러나 설치 디렉토리의 기본 스니펫 파일에 대한 모든 편집 사항은 되돌려집니다).

사용자 디렉토리에 파일이 생성되면 다음 규칙에 따라 텍스트 편집기로 파일을 편집할 수 있습니다.

  • 각 스니펫은 별도의 줄에 있어야 합니다(빈 줄이 없어야 함).
  • 각 스니펫은 사용할 단축키로 시작해야 하며 그 뒤에 "- " 그리고 스니펫의 이름(메뉴에 표시됨) 다음에 콜론 ": ", 예:

I - 인스턴스 생성:

콜론 뒤에 코드 스니펫을 추가합니다.

I - 인스턴스 생성:instance_create_layer(x, y, |layer|, 개체);

추가하는 코드는 다음과 같은 특정 형식도 따라야 합니다.

  • 모든 코드는 한 줄에 있습니다
  • 너는 사용한다 "$ " 개행의 경우
    • 참고 여러 줄 바꿈을 만들 때 각 " 사이에 공백이 있어야 합니다.$ " 기호. 예를 들어 |>$ $ $ 를 작성하여 세 개의 줄 바꿈을 추가합니다.
  • 강조 표시해야 하는 텍스트를 " GameMaker " 기호입니다. 코드 편집기는 스니펫이 추가될 때 편집을 위해 이 텍스트를 자동으로 선택합니다.

기본 스니펫 파일에 이미 있는 예제를 참조하여 위의 규칙에 따라 어떻게 설정되는지 확인하십시오. GameMaker 설치 디렉토리에서 기본 파일을 찾을 수 있습니다.

  • 윈도우: /Applications/GameMaker \TextEditor\snippets.txt
  • 맥 OS:.app/Contents/MonoBundle/TextEditor/snippets.txt
