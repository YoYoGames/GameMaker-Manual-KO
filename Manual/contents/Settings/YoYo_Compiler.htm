요요 컴파일러

YoYo 컴파일러 (YYC)는 해석된 코드와 러너( GameMaker VM 컴파일이 사용하는 것) 대신 기계어 코드 를 사용하는 실행 패키지를 생성하기 위한 특수 컴파일러입니다. 컴파일된 코드는 실행 속도가 더 빠르지만 실행 파일을 컴파일하는 데 더 많은 시간이 걸립니다.

특허

GameMaker 라이선스에는 YoYo 컴파일러가 포함되거나 포함되지 않을 수 있습니다. 자세한 내용은 도움말 센터 페이지를 참조하십시오. 나에게 적합한 구독은 무엇입니까?.

SDK 설정

YoYo 컴파일러 가 대상 플랫폼에 대해 컴파일할 수 있으려면 올바른 SDK 가 설치되어 있어야 합니다.

이에 대한 일반적인 개요는 다음 도움말 센터 페이지를 참조하십시오. GameMaker 필수 SDK.

플랫폼별로 필요한 SDK에 대한 정보는 다음 도움말 센터에서도 확인할 수 있습니다.

  • 오페라 GX
  • 창
  • 맥 OS
  • 우분투
  • 기계적 인조 인간
  • iOS(iPadOS 포함)
  • tvOS

YYC로 프로젝트 빌드

SDK 가 올바르게 설정되면 선택한 플랫폼의 출력 옵션을 YYC로 변경할 수 있습니다.

그런 다음 VM 출력과 마찬가지로 빌드 메뉴 에서 실행 파일을 만들 수 있습니다.

SDK 에 대한 경로가 올바르게 구성되지 않은 경우 다음과 같은 오류가 발생할 수 있습니다(이미지는 Windows 플랫폼에서 메시지를 표시함).

GameMaker 는 선택한 플랫폼에서 실행 파일을 빌드하는 데 필요한 파일의 위치를 알아야 합니다. 이것이 올바르게 설정되지 않은 경우 컴파일 오류가 표시됩니다.

선택한 플랫폼에 대해 경로가 올바르게 설정되었는지 확인하려면 플랫폼 기본 설정 을 확인할 수 있습니다.
