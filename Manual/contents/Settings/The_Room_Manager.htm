룸 매니저

자산 브라우저 를 사용할 때 방을 만들고 다른 폴더 그룹 간에 이동하여 가장 적합한 방식으로 프로젝트를 구성할 수 있습니다. 그러나 GameMaker 의 게임에는 일반적으로 특정 순서를 따르는 방이 있으며 최소한 특정 시작 방이 있어야 합니다(이 방에서 게임이 시작된 후 원하는 경우 코드나 작업을 사용하여 게임의 아무 방으로 갈 수 있습니다 ), 여기서 Room Manager 가 작동합니다.

"기타" 버튼을 클릭하여 룸 매니저를 열 수 있습니다. 자산 브라우저 상단에서 열리는 메뉴에서 선택하거나 간단히 자산 브라우저의 방 왼쪽에 있는 아이콘:

열리면 Room Manager는 두 개의 다른 탭을 제공합니다. 하나는 Room Order 용이고 다른 하나는 Room Inheritance 입니다. 방 순서는 함수를 사용하여 문자 그대로 게임을 탐색하는 경우 방에 액세스하는 순서를 정의합니다. room_goto_next() 각 방의 끝에 있으며 나열된 맨 위에 있는 방은 항상 "집" 아이콘으로 표시됩니다., 이 방이 게임의 지정된 시작 방임을 보여줍니다. 이 아이콘은 자산 브라우저에도 표시되므로 자산을 필터링하거나 설정한 방법에 관계없이 방이 어떤 방인지 명확하게 볼 수 있습니다.

참고: 방 자산의 ID는 이 목록에 정의된 것과 같은 순서가 아니므로 런타임에 방을 변경하기 위해 숫자 값을 직접 제공해서는 안 됩니다. 참조하려는 자산(단순히 이름)에 대해 항상 방 상수 를 사용하거나 함수를 통해 검색하십시오.

클릭하면 방의 순서를 변경할 수 있습니다. 목록을 위 또는 아래로 이동하려는 방을 드래그하여 사용할 수 있습니다. 및/또는 / 여러 방을 선택한 다음 클릭하고 끌어 모두 이동하려면 다음을 수행하십시오.

방 상속 탭은 "자식" 방을 설정하기 위한 것입니다. 자식 방은 "상위" 방에서 복사되어 모든 속성을 상속하도록 설정된 방입니다. 이 방에 아무것도 하지 않으면 이 방은 실제로 원래 방의 복사본이 되지만 방 편집기 로 이동하여 레이어나 인스턴스와 같은 특정 항목에 대한 상속을 켜거나 끌 수 있으며 새로 추가할 수 있습니다. 레이어, 인스턴스 및 속성을 방으로 가져옵니다.

객실 주문 탭과 마찬가지로 다음을 클릭할 수 있습니다. 끌어서 방을 재정렬하지만 하위 방을 선택하고 다른 상위 방으로 이동하면 원래 상위 방 상속이 중단되고 대신 새 방의 속성이 상속됩니다. 또한 자녀를 목록 왼쪽으로 드래그하여 상속이 없는 "전체" 방으로 변환할 수 있습니다(목록 테두리가 강조 표시됨). 그러면 해당 위치에서 단순히 방이 됩니다.

룸에는 여러 자녀가 있을 수 있으며 각 자녀는 부모가 될 수도 있고 자신의 자녀를 가질 수도 있습니다. 예를 들면 다음과 같습니다.

여기에 우리는 부모 방이 있습니다. "rm_Level_1_Day ", 두 자녀가 있는 "rm_Level_1_Day_AfterBattle " 그리고 "rm_Level_1_Night ". 우리는 또한 다른 어린이 방을 할당했습니다 "rm_Level_1_Night " 라고 불리는 " rm_Level_1_Night_AfterBattle ". 이런 식으로 한 부모 방의 리소스와 레벨 디자인을 재사용한 다음, 자식 방에서 약간 다르게 만드는 데 필요한 것만 변경할 수 있습니다.

참고: 룸 상속에 대한 자세한 내용은 여기 를 참조하십시오.

마지막으로, 방 관리자에는 방을 탐색하고 정리하는 데 도움이 되는 몇 가지 추가 도구가 있습니다. 상단에는 검색어를 추가한 다음 이름에 검색어가 있는 방으로 빠르게 이동할 수 있는 검색 표시줄 이 있습니다. 검색어가 포함된 회의실은 강조 표시되며 측면의 위/아래 버튼을 사용하여 즉시 탐색할 수 있습니다.

또한 방 관리자의 하단에는 방을 방 순서 목록 위나 아래로 이동할 수 있는 몇 가지 유용한 버튼이 있습니다.

이 버튼 옆에는 그룹 폴더를 만드는 데 사용할 수 있는 그룹 만들기 버튼도 있습니다. 이 폴더는 게임에서 방을 더 잘 구성하는 데 사용할 수 있으며(방 순서는 계속 유지됨) 방을 클릭하고 폴더로 끌어서 추가할 수 있습니다. 누른 상태에서 클릭하여 여러 방을 선택할 수도 있습니다. 및/또는 /, 그룹 생성 버튼을 클릭하면 선택한 모든 방이 그룹에 추가됩니다. 이렇게 하면 게임에서 방의 순서 가 변경되며 나중에 그룹 폴더를 삭제하도록 선택 하면 폴더 내의 모든 방도 삭제됩니다 (기본적으로 GameMaker에서 이에 대해 경고합니다).

또한 Room Manager에서 아무 방이나 마우스 오른쪽 버튼으로 클릭하면 다음 메뉴가 나타납니다.

이 메뉴는 다음과 같은 옵션을 제공합니다.

  • 자식 방 만들기: 클릭한 방의 자식으로 새 방을 만들고 부모 방 속성을 상속하는 새 방을 만듭니다.
  • 편집: 방에 대한 방 편집기 가 열립니다.
  • 태그 편집: 태그 편집기 를 열어 방에 태그를 추가할 수 있습니다.
  • 이름 바꾸기: 방 자산의 이름을 바꿀 수 있습니다(느린 더블 클릭도 가능 이름을 바꾸려면 방에서).
  • 복제: 선택한 방을 복제하여 선택한 방 아래에 새 방을 배치합니다.
  • 즐겨찾기: 선택한 방을 자산 브라우저 즐겨찾기 섹션에 추가합니다.
  • 삭제: 선택한 방을 삭제합니다. 그룹 폴더에서 사용하는 경우 폴더 내의 모든 룸이 폴더와 함께 삭제됩니다. 상위 방에서 사용하는 경우 상위 방만 삭제되고 모든 하위 방은 유지되지만 이제 독립되어 더 이상 하위 방이 아닙니다.
  • 하위 확장: 그룹 폴더를 선택한 경우 이 옵션은 폴더의 내용을 확장합니다.
  • 모두 확장: 이 옵션은 모든 폴더 그룹을 확장합니다.
  • 모두 축소: 이 옵션은 모든 폴더 그룹을 축소합니다.
  • 정렬: 정렬 하위 메뉴는 관리자 창 하단에 있는 버튼과 마찬가지로 방 관리자에서 방을 위 또는 아래로 이동하는 데 사용할 수 있습니다.
  • 탐색기에서 열기: 게임 프로젝트의 자산에 대한 OS 파일 탐색기를 엽니다.
  • 자산 색상: 여기에서 방의 색상을 선택하여 방 관리자에서 식별하는 데 도움이 됩니다. 이 색상은 자산 브라우저에서도 사용되며 이전에 설정한 색상을 재정의합니다.
