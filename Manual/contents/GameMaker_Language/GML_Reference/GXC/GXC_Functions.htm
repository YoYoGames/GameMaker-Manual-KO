오페라 GX와 GXC

이 섹션에서는 Opera GX 대상 플랫폼에 제공되는 기능을 간략하게 설명합니다. 이 페이지에 나열된 기능을 사용하면 화면을 기록하고 챌린지 점수와 함께 GXC에 업로드할 수 있지만 이를 달성하려면 GXC API 라이브러리 를 사용해야 합니다. GXC API 라이브러리 구현 및 문제 해결에 대한 지침은 YoYo Games Helpdesk의 Opera GX 섹션 을 참조하십시오.

챌린지가 있는 비디오 녹화를 업로드하기 위해 다음 기능이 제공됩니다.

  • gxc_start_movie_recording
  • gxc_stop_movie_recording
  • gxc_pause_movie_recording
  • gxc_resume_movie_recording
