gxc_start_movie_recording

이 기능은 GXC에서 도전 기록을 시작하는 데 사용됩니다. 녹음의 너비, 높이 및 프레임 속도와 오디오 녹음 여부에 대한 부울을 지정합니다.

통사론:

gxc_start_movie_recording(width, height, fps, record_audio);

논쟁 유형 설명
width 진짜 The width of the recording
height 진짜 The height of the recording
fps 진짜 The frame rate (frames-per-second) of the recording
record_audio 부울 Whether audio should be recorded (true ) or not (false )

보고:

N/A

예시:

gxc_start_movie_recording(320, 240, 4, false);

이렇게 하면 오디오가 녹음되지 않고 320x240 해상도에서 초당 4프레임으로 비디오 녹화가 시작됩니다.
