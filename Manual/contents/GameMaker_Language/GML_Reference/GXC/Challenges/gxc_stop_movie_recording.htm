gxc_stop_movie_recording

이 기능은 이전에 다음으로 시작된 GXC 녹음을 중지하는 데 사용됩니다.gxc_start_movie_recording(). 또한 녹음을 인수에 지정된 URL로 제출하지만 지정된 문자열의 종류에 따라 동작이 다릅니다.

  • 파일 이름을 지정하면 녹음이 다운로드 폴더에 해당 이름으로 저장됩니다.
  • URL을 지정하면 녹음이 해당 URL로 업로드됩니다.

URL 방식은 녹화를 GXC에 업로드해야 할 때 사용해야 하며 파일은 녹화 성공 여부를 확인하기 위해서만 로컬에 저장해야 합니다.

통사론:

gxc_stop_movie_recording(filename);

논쟁 유형 설명
filename 끈 The file name or URL where the recording is to be saved or uploaded

보고:

N/A

예시:

gxc_submit_challenge_score(_score, function(_status, _result) {
    var _uploadUrl = _result.data.gameplayVideoUploadUrl;
    gxc_stop_movie_recording(_uploadUrl);
});

위의 코드는 GXC API 라이브러리를 사용하여 현재 활성화된 챌린지에 점수를 업로드하고, 콜백 메서드에서 응답을 받으면 동영상의 업로드 URL을 검색하고 녹화를 업로드합니다.
