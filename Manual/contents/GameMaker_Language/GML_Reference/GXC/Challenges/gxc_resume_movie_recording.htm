gxc_resume_movie_recording

이 기능은 이전에gxc_pause_movie_recording().

통사론:

gxc_resume_movie_recording();

보고:

예시:

if (keyboard_check_pressed(vk_enter))
{
    gxc_resume_movie_recording();
}

Enter 키를 누르면 현재 일시 중지된 GXC 동영상 녹화가 다시 시작됩니다.
