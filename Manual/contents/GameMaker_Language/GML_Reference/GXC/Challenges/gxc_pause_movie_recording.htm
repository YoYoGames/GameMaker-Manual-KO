gxc_pause_movie_recording

이 기능은 이전에 다음으로 시작된 GXC 녹음을 일시 중지하는 데 사용됩니다.gxc_start_movie_recording().

통사론:

gxc_pause_movie_recording();

보고:

예시:

if (keyboard_check_pressed(vk_escape))
{
    gxc_pause_movie_recording();
}

이스케이프 키를 누르면 GXC 동영상 녹화가 일시 중지됩니다.
