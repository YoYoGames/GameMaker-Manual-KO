audio_resume_all

이 기능을 사용하면 이전에 일시 중지된 모든 사운드를 다시 시작할 수 있습니다.

통사론:

audio_resume_all();

보고:

예시:

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    if (global.Pause)
    {
        audio_pause_all();
    }
    else
    {
        audio_resume_all();
    }
}

위의 코드는 키보드 키 "P"가 눌렸는지 확인하고 감지하면 전역 변수 "Pause"를 다음으로 설정합니다. true 또는 false를 선택한 다음 모든 사운드를 일시 중지하거나 이전에 일시 중지된 모든 사운드를 다시 시작합니다.
