audio_is_playing

이 기능은 주어진 소리가 현재 재생 중인지 확인합니다. 사운드는 사운드의 단일 인스턴스가 될 수 있습니다(재생되는 개별 사운드에 대한 인덱스는audio_play_sound() 또는audio_play_sound_at() 기능) 또는 사운드 자산, 이 경우 지정된 사운드의 모든 인스턴스가 확인되고 그 중 하나가 재생 중인 경우 함수가 반환됩니다. true 그렇지 않으면 반환됩니다 false. 이 함수는 여전히 반환됩니다. true 확인 중인 사운드가 이전에 다음을 사용하여 일시 중지된 경우audio_pause_sound() 함수.

통사론:

audio_is_playing(index);

논쟁 유형 설명
index 사운드 자산 또는 사운드 인스턴스 ID The index of the sound to check.

보고:

Boolean

예시:

if !audio_is_playing(snd_Waterfall)
{
    audio_play_sound_at(snd_Waterfall, x, y, 0, 300, true, 1);
}

위의 코드는 "snd_Waterfall" 변수에 인덱싱된 사운드가 현재 재생 중이고 반환되는지 확인합니다. false 그런 다음 사운드는 300의 폴오프 거리와 낮은 우선 순위로 룸 위치에서 반복됩니다.
