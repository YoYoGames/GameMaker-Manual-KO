audio_group_unload

이 기능은 주어진 오디오 그룹에 속하는 것으로 플래그가 지정된 모든 사운드를 메모리로 언로드합니다. 함수가 반환됩니다 true 언로드가 시작되고 false 그룹 ID가 유효하지 않거나 이미 언로드 플래그가 지정된 경우 이 함수가 호출될 때 현재 재생 중인 모든 오디오가 중지됩니다.

통사론:

audio_group_unload(groupID);

논쟁 유형 설명
groupID 오디오 그룹 ID The index of the audio group to unload (as defined in the Audio Groups Window )

보고:

Boolean

예시:

if audio_group_is_loaded(audiogroup_level1)
{
    audio_group_unload(audiogroup_level1);
}

위의 코드는 오디오 그룹이 로드되었는지, 로드되었는지 여부를 확인합니다.
