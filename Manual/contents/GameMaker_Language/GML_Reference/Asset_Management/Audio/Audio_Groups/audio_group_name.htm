audio_group_name

이 함수는 표시하거나 확인하기 위해 주어진 오디오 그룹의 이름을 포함하는 문자열을 반환합니다. 게임 옵션에서 오디오 그룹을 정의할 때 그룹의 ID 값으로 사용할 상수 인 고유한 "이름"을 지정합니다. 이 함수가 하는 일은 ID를 가져와서 제공한 ID 이름의 문자열을 반환하는 것입니다.

통사론:

audio_group_name(groupID);

논쟁 유형 설명
groupID 오디오 그룹 ID The index value constant of the audio group to check (as defined in the Audio Groups Window )

보고:

String

예시:

var name = audio_group_name(audiogroup_level1);
draw_text(32, 32, "Now Playing Group: " + name);

위의 코드는 지정된 오디오 그룹 상수의 이름을 검색하여 화면에 표시합니다.
