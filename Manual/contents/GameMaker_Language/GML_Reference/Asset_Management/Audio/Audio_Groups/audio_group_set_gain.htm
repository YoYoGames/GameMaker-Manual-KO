audio_group_set_gain

이 기능을 사용하면 주어진 시간 동안 사운드 그룹을 페이드 인 또는 페이드 아웃하거나 그룹 게인을 즉시 설정하는 데 사용할 수 있습니다.

시간은 밀리초 단위로 측정되며 이 기능을 사용하려면 해당 시간이 끝날 때까지 그룹에 도달하기 위한 최종 게인 레벨을 입력해야 합니다. 이 게인은 0(무음)과 1(전체 볼륨) 사이일 수 있으며 스케일은 선형이므로 값 0.5는 볼륨의 절반이 됩니다. 게인을 즉시 변경하려면 time 인수를 0으로 설정하기만 하면 됩니다. 일부 플랫폼에서는 1보다 큰 게인을 가질 수 있지만 값이 1이면 "전체 볼륨"으로 간주되고 그보다 크면 오디오 클리핑이 발생할 수 있습니다.

통사론:

audio_group_set_gain(groupID, volume, time);

논쟁 유형 설명
groupID 오디오 그룹 ID The index of the audio group to stop (as defined in the Audio Groups Window )
volume 진짜 The final value for the group volume.
time 진짜 The length of the change in gain in milliseconds.

N/A

예시:

if keyboard_check_pressed(vk_space)
{
    audio_group_set_gain(audiogroup1, 0, 5000);
}

위의 코드는 "space" 키를 확인한 다음 "audiogroup1"에 대한 모든 오디오를 5초에 걸쳐 0으로 감소시킵니다.
