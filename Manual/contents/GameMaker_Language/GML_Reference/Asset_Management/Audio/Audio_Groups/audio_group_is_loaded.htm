audio_group_is_loaded

이 기능은 특정 오디오 그룹이 메모리에 로드되어 사용할 준비가 되었는지 확인합니다.

통사론:

audio_group_is_loaded(groupID);

논쟁 유형 설명
groupID 오디오 그룹 ID The index of the audio group to check (as defined in the Audio Groups Window )

보고:

Boolean

예시:

if audio_group_is_loaded(audiogroup_level1)
{
    audio_group_unload(audiogroup_level1);
}

위의 코드는 오디오 그룹이 로드되었는지, 로드되었는지 여부를 확인합니다.
