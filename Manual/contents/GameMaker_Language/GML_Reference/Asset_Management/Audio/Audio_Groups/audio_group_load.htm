audio_group_load

이 기능은 주어진 오디오 그룹에 속하는 것으로 플래그가 지정된 모든 사운드를 메모리에 로드합니다. 함수가 반환됩니다 true 로딩이 시작되고 false 그룹 ID가 유효하지 않거나 로드하도록 이미 플래그가 지정된 경우 이 기능은 비동기식이므로 오디오가 백그라운드에서 로드되는 동안 게임이 계속 실행됩니다. 즉, 전체 그룹이 메모리에 로드되었으며 이제 사운드를 사용할 수 있음을 알리기 위해 비동기 로드/저장 이벤트 도 트리거됩니다.

통사론:

audio_group_load(groupID);

논쟁 유형 설명
groupID 오디오 그룹 ID The index of the audio group to load (as defined in the Audio Groups Window )

보고:

Boolean

예시:

if !audio_group_is_loaded(audiogroup_level1)
{
    audio_group_load(audiogroup_level1);
}

위의 코드는 오디오 그룹이 로드되었는지 확인하고 그렇지 않은 경우 로드합니다.
