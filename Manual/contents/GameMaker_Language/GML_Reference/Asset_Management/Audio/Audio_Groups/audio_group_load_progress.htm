audio_group_load_progress

이 함수는 오디오 그룹의 로딩 진행률을 확인하고 0에서 100 사이의 (대략적인) 값을 반환합니다.

통사론:

audio_group_load_progress(groupID);

논쟁 유형 설명
groupID 오디오 그룹 ID The index of the audio group to check (as defined in the Audio Groups Window )

보고:

Real

예시:

if load
{
    var pc = audio_group_load_progress(audiogroup_level1);
    draw_text(32, 32, "Loading: " + string(pc));
}

위의 코드는 변수를 확인하고 반환하는 경우 true 그러면 로드되는 오디오의 진행 상황을 보여주는 일부 텍스트가 그려집니다.
