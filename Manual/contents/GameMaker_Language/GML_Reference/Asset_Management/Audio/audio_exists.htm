audio_exists

이 함수는 소리가 존재하는지 여부를 반환합니다(true ) 여부(false ). 검색하는 인덱스가 이전에 초기화되지 않은 경우 이 함수는 존재하지 않는 자산 인덱스를 검색하므로 오류가 발생합니다. 확인할 사운드는 사운드의 단일 인스턴스일 수 있습니다(재생되는 개별 사운드에 대한 인덱스는audio_play_sound() 또는audio_play_sound_at() 기능) 또는 사운드 자산.

통사론:

audio_exists(index);

논쟁 유형 설명
index 유산 The index of the sound to check the existence of.

보고:

Boolean

예시:

if audio_exists(global.Music)
{
    audio_play_sound(global.Music, 0, true);
}

위의 코드는 소리가 있는지 확인합니다. 그렇다면 루프에서 재생하도록 설정됩니다.
