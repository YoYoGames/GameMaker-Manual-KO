audio_sound_get_listener_mask

이 함수는 사운드를 재생해야 하는 오디오 청취자를 정의하는 비트 마스크 데이터를 반환합니다. 자세한 내용은 오디오 리스너 섹션을 참조하세요.

통사론:

audio_sound_get_listener_mask(soundID);

논쟁 유형 설명
soundID 사운드 인스턴스 ID The unique ID of the sound to get the mask of

보고:

Real

예시:

var snd = audio_play_sound(snd_PlayerDead, 10, false);
if audio_sound_get_listener_mask(snd) != global.PlayerMask
{
    audio_sound_set_listener_mask(snd, global.PlayerMask);
}

위의 코드는 사운드를 재생한 후 해당 사운드에 대한 리스너 마스크 데이터를 확인하고 전역 변수에 저장된 것과 동일하지 않은 경우 전역 변수에 저장된 마스크 데이터를 사용하여 재생할 리스너를 설정합니다. 변하기 쉬운.
