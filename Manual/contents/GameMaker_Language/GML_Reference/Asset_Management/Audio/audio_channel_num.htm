audio_channel_num

이 기능을 사용하면 GameMaker 에서 오디오를 재생하는 데 사용할 수 있는 오디오 채널 수를 설정할 수 있습니다. 이것이 기본적으로 의미하는 것은 한 번에 재생할 수 있는 동시 사운드 수를 지정하고 숫자가 양을 초과하면 우선 순위 가 낮은 사운드를 중지하여 우선 순위가 높은 사운드에 대한 채널을 확보한다는 것입니다.. 이 기능을 사용하여 장치에 맞게 게임을 최적화할 수 있습니다. 오디오용 채널 수가 적을수록 성능이 향상되지만 동시에 여러 개를 재생할 경우 특정 사운드가 눈에 띄게 끊길 수도 있습니다.

통사론:

audio_channel_num(num);

논쟁 유형 설명
num Number of available audio channels (default is 128).

보고:

예시:

switch (os_browser)
{
    case browser_not_a_browser:
        switch (os_type)
        {
            case os_windows:
            case os_macos:
                audio_channel_num(200);
            break;

            default:
                audio_channel_num(64);
            break;
        }
    break;

    default:
        audio_channel_num(16);
    break;
}

위의 코드는 게임이 실행되는 플랫폼을 확인하고 사용 가능한 사운드 채널 수를 변경하여 성능을 향상시킵니다.
