audio_create_play_queue

이 함수는 오디오용 버퍼 큐를 준비합니다. 데이터 형식을 전달합니다(만 buffer_u8 또는 buffer_s16 현재 지원됨), 샘플 속도(1000hz와 48000hz 사이일 수 있음) 및 오디오가 아래 나열된 상수 중 하나에서 요구하는 채널:

일정한 설명
audio_mono 모노(단일 채널) 오디오.
audio_stereo 스테레오(듀얼 채널) 오디오.
audio_3D 3D(5.1) 오디오.


함수를 사용하여 메모리 누수를 방지하기 위해 더 이상 필요하지 않은 경우 이 함수로 생성된 오디오 대기열을 해제해야 합니다.audio_free_play_queue().

이 함수는 대기열이 끝에 도달하면 오디오 재생 비동기 이벤트 를 트리거하고 이 이벤트에서 변수에 특수 DS 맵 이 생성됩니다. async_load 다음 키/값 쌍으로:

  • " queue_id " - 데이터 재생이 완료된 큐
  • " buffer_id " - 더 이상 재생되지 않는 버퍼
  • " queue_shutdown " - 정상 재생 중에는 0으로 설정되고 이벤트가 수신되면 1로 설정됩니다.audio_free_play_queue() 호출되었습니다. 이 값이 1로 설정되면 더 이상 데이터를 대기열에 추가하고 싶지 않습니다.

통사론:

audio_create_play_queue(queueFormat, queueRate, queueChannels);

논쟁 유형 설명
queueFormat 버퍼 데이터 유형 상수 The format of the buffer data to use (buffer_u8 or buffer_s16 ).
queueRate 진짜 The sample rate of the data in the buffer queue.
queueChannels 오디오 채널 유형 The channels to use from one of the constants listed above.

보고:

Audio Queue ID

예시:

audio_queue = audio_create_play_queue(buffer_s16, 11052, audio_mono);

위의 코드는 새 오디오 대기열을 만들고 나중에 참조할 수 있도록 해당 ID를 변수에 할당합니다.
