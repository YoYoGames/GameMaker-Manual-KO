audio_queue_sound

이 함수는 버퍼의 데이터를 이전에 함수를 사용하여 생성한 오디오 대기열에 추가합니다.audio_create_play_queue(). 추가할 큐 인덱스와 데이터를 가져올 버퍼 ID, 버퍼 내의 위치(오프셋) 및 추가할 바이트 수를 지정합니다. 버퍼에서 대기열로 오디오를 추가하면 먼저 함수를 사용하여 대기열을 해제할 때까지 버퍼를 삭제할 수 없습니다.audio_free_play_queue(), 버퍼 속성은 사운드를 추가할 대기열의 속성과 일치해야 합니다.

통사론:

audio_queue_sound(queueIndex, bufferId, bufferOffset, bufferLength);

논쟁 유형 설명
queueIndex 오디오 대기열 ID The index of the queue to add to.
bufferId 버퍼 ID The buffer to add to the queue.
bufferOffset 진짜 The offset within the source buffer to start from.
bufferLength 진짜 The length of the buffer (the number of the bytes in the buffer).

N/A

예시:

len = async_load[? "data_len"];
audio_buff = buffer_create(len, buffer_fast, 1);
buffer_copy(async_load[? "buffer_id"], 0, len, buff, 0);
audio_queue_sound(audio_queue, audio_buff, 0, len);
audio_play_sound(audio_queue, 10, 0);
