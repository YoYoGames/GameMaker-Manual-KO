audio_get_recorder_count

이 함수는 현재 게임에서 사용할 수 있는 오디오 녹음 소스(예: 마이크 등)의 수를 반환합니다. 따라서 예를 들어 4의 반환 값이 있는 경우 소스 0, 1, 2 및 3에 오디오 입력이 있으며 이 값 중 하나는 함수를 사용하여 녹음을 시작하는 데 사용하는 값입니다.audio_start_recording(). 이 값은 사람들이 마이크 또는 기타 입력 소스를 장치에 연결/분리할 때 언제든지 변경될 수 있습니다. 기능을 사용할 수 있으니 참고하세요.audio_get_recorder_info() 연결된 각 장치에 대한 정보를 가져옵니다.

중요 이 함수는 Opera GX 대상에서 항상 1을 반환합니다.

통사론:

audio_get_recorder_count();

보고:

Real

예시:

if audio_get_recorder_count() > 0
{
    channel_index = audio_start_recording(0);
}

위의 코드는 사용 가능한 오디오 레코더를 확인하고 1개 이상이면 인덱싱된 첫 번째 것(소스 0)부터 녹음을 시작합니다.
