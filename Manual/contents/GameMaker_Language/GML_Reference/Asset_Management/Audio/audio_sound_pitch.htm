audio_sound_pitch

이 기능은 주어진 소리의 피치를 변경하는 데 사용할 수 있습니다. 사운드는 사용할 때 변수에 저장된 재생 중인 개별 사운드에 대한 인덱스에서 참조된 것일 수 있습니다.audio_play_sound() 또는audio_play_sound_at() 기능 또는 자산 브라우저의 실제 사운드 자산. 재생 중인 사운드의 인덱스인 경우 해당 인스턴스만 변경되지만 자산 브라우저에서 사운드 자산을 사용할 때 재생 중인 해당 사운드 자산의 모든 인스턴스가 변경됩니다.

피치 인수는 피치 승수 입니다. 입력 값은 현재 피치를 해당 양만큼 곱하므로 기본값 1은 피치 변경이 없고 값이 1보다 작으면 피치가 낮아지고 1보다 크면 올라갑니다. 정점. 0보다 작거나 5보다 큰 값은 어쨌든 들리지 않을 수 있으므로 이 기능에 대해 작은 증분을 사용하는 것이 가장 좋습니다. 허용되는 총 피치 변경은 (1/256) - 256 옥타브로 제한되므로 이보다 높거나 낮은 값은 등록되지 않습니다.

위에 제공된 고정 값은 GameMaker 가 범위를 고정하려고 시도하지만 이 값은 모든 대상 플랫폼에서 보장 되지 않습니다. 예를 들어 iOS는 (1/256) - 8로 고정되므로 더 높거나 더 낮은 옥타브 값이 필요한 경우 각 대상 플랫폼에서 실험하고 사운드 리소스의 다른 버전을 미리 이동해야 할 수 있습니다.

통사론:

audio_sound_pitch(index, pitch);

논쟁 유형 설명
index 사운드 자산 또는 사운드 인스턴스 ID The index of the sound to change.
pitch 진짜 The pitch multiplier (default 1).

보고:

N/A

예시:

var s_engine = audio_play_sound(snd_CarEngine, 10, false);
switch (gear)
{
    case 1: audio_sound_pitch(s_engine, 0.8); break;
    case 2: audio_sound_pitch(s_engine, 0.9); break;
    case 3: audio_sound_pitch(s_engine, 0.95); break;
    case 4: audio_sound_pitch(s_engine, 1); break;
    case 5: audio_sound_pitch(s_engine, 1.2); break;
}

위의 코드는 "gear" 변수의 값을 기준으로 "s_engine" 변수에 인덱싱된 사운드에서 재생되는 오디오의 피치를 변경합니다.
