audio_get_master_gain

이 기능을 사용하면 특정 청취자에 대한 모든 사운드 및 음악의 전체 볼륨에 대한 절대값을 얻을 수 있습니다. 기본 리스너 인덱스는 0이지만 함수를 사용할 수 있습니다.audio_get_listener_info() 대상 플랫폼에 사용할 수 있는 다양한 인덱스를 가져옵니다. 반환된 게인 값은 0(무음)에서 1(전체 볼륨)까지의 선형 스케일을 기반으로 합니다. 일부 플랫폼에서는 게인이 1보다 클 수 있지만 값이 1이면 "전체 볼륨"으로 간주되고 그보다 크면 오디오 클리핑이 발생할 수 있습니다.

통사론:

audio_get_master_gain(listenerIndex);

논쟁 유형 설명
listenerIndex 실제 또는 오디오 리스너 ID The index of the listener to get the gain of.

보고:

Real

예시:

var num = audio_get_listener_count();
for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var ind = info[? "index"];
    if audio_get_master_gain(ind) != 1
    {
        audio_set_master_gain(info[? "index"], 1);
    }
    ds_map_destroy(info);
}

위의 코드는 사용 가능한 리스너를 반복하면서 게인이 1인지 아닌지 확인하고 그렇지 않은 경우 각각에 대해 1로 설정합니다.
