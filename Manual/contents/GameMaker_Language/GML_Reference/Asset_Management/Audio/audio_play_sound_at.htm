audio_play_sound_at

이 기능을 사용하면 오디오 공간 내의 지정된 위치에서 모든 사운드 리소스를 재생할 수 있습니다. 사운드 인덱스를 제공한 다음 3D 공간 내에서 위치를 할당합니다. 기본 청취자 위치는 (0, 0, 0) 이므로 청취자가 이동하지 않고 사운드가 왼쪽에서 들리도록 하려면(예를 들어) x 위치를 음수 값으로 설정해야 합니다(예: 리스너 위치 설정에 대한 자세한 내용은 참조audio_listener_position() ).

또한 감쇠 거리(0은 사운드를 무음, 기본값은 100)를 설정하여 청취자 위치에서 멀어질수록 사운드가 페이드 아웃되도록 할 수 있습니다. 페이드 자체가 들리는 방법은 폴오프 기준(소스의 볼륨이 일반적으로 절반으로 떨어지는 거리)과 롤오프 계수(폴오프 기준 거리를 지나는 사운드에만 영향을 미침)에 따라 다릅니다. 기본 요소는 일반적으로 1이며 다른 폴오프 값의 효과는 선택한 모델에 따라 다릅니다(다른 폴오프 모델과 이러한 값이 사용되는 방법에 대한 완전한 가이드는 함수를 참조하십시오.audio_falloff_set_model() ).

다음 두 인수는 사운드를 루프 여부와 사운드에 우선 순위를 할당하도록 설정하는 것입니다. 이 우선 순위는 재생 중인 사운드 수가 기능에서 설정한 제한을 초과할 때 사운드를 처리하는 방법을 결정하는 데 사용됩니다.audio_channel_num(). 우선 순위가 낮은 사운드는 우선 순위가 더 높은 사운드를 위해 중지되며 우선 순위 값은 임의의 실수일 수 있습니다(실제 값은 임의적이며 GameMaker 가 동일한 우선 순위를 지정하므로 0에서 1 또는 0에서 100일 수 있음). 우선 순위를 처리할 때 숫자가 높을 수록 우선 순위가 높아져 우선 순위 100의 사운드가 우선 순위 1의 사운드보다 선호됩니다.

이 함수는 재생 중인 사운드에 대한 고유한 인덱스 번호를 반환하고 변수에 저장할 수 있으므로 적절한 기능으로 일시 중지하거나 중지할 수 있습니다. 즉, 한 번에 동일한 사운드의 여러 인스턴스가 재생되는 경우 해당 사운드의 한 인스턴스만 대상으로 지정하여 오디오 기능을 사용하여 처리할 수 있습니다.

통사론:

audio_play_sound_at(index, x, y, z, falloff_ref, falloff_max, falloff_factor, loop, priority, [gain], [offset], [pitch], [listener_mask]);

논쟁 유형 설명
index 사운드 자산 The index of the sound to play.
x 진짜 The x position.
y 진짜 The y position.
z 진짜 The z position.
falloff_ref 진짜 The falloff reference relative to the listener (clamp).
falloff_max 진짜 The maximum falloff distance relative to the listener.
falloff_factor 진짜 The falloff factor (default 1).
loop 부울 Flags the sound as looping or not.
priority 진짜 Set the channel priority for the sound.
gain 진짜 OPTIONAL Value for the gain.
offset 진짜 OPTIONAL The time (in seconds) to set the start point to. Values longer than the length of the given sound are ignored.
pitch 진짜 OPTIONAL The pitch multiplier (default 1).
listener_mask 진짜 OPTIONAL The bitmask data to set for the sound. On the HTML5 target this will have no effect as the target does not support more than one listener.

보고:

Sound Instance ID

예 1:

if global.SFX
{
    audio_play_sound_at(snd_Waterfall, x, y, 0, 100, 300, 1, true, 1);
}

위의 코드는 전역 변수 "SFX"를 확인하고 반환되는 경우 true 그러면 "snd_Waterfall" 변수에 인덱싱된 사운드가 룸 위치에서 반복되며, 폴오프 참조는 100, 폴오프 거리는 300, 폴오프 계수는 1 및 낮은 우선순위입니다.

예 2:

if global.SFX
{
    audio_play_sound_at(snd_Waterfall, x, y, 0, 100, 300, 1, true, 1, 1, 2);
}

위의 코드는 전역 변수 "SFX"를 확인하고 반환되는 경우 true 그러면 "snd_Waterfall" 변수에 색인된 사운드가 게인 변경 없이 지정된 폴오프 설정을 사용하여 룸 위치에서 루프됩니다. 사운드는 두 번째 2부터 즉시 재생되기 시작합니다.
