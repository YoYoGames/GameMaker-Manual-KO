audio_resume_sound

이 기능을 사용하면 현재 일시 중지된 모든 사운드를 다시 시작할 수 있습니다(기능 사용 후audio_pause_sound() ). 사운드는 사운드의 단일 인스턴스가 될 수 있습니다(재생되는 개별 사운드에 대한 인덱스는audio_play_sound() 또는audio_play_sound_at() 기능) 또는 사운드 자산, 이 경우 지정된 사운드의 모든 인스턴스가 다시 시작됩니다.

통사론:

audio_resume_sound(index);

논쟁 유형 설명
index 사운드 자산 또는 사운드 인스턴스 ID The index of the sound to resume.

보고:

N/A

예시:

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    if (global.Pause)
    {
        audio_pause_sound(snd_Waterfall);
    }
    else
    {
        audio_resume_sound(snd_Waterfall);
    }
}

위의 코드는 키보드 키 "P"가 눌렸는지 확인하고 감지하면 전역 변수 "Pause"를 다음으로 설정합니다. true 또는 false로 설정한 다음 "snd_Waterfall" 변수에 색인된 사운드를 일시 중지하거나 일시 중지된 상태에서 사운드를 다시 시작합니다.
