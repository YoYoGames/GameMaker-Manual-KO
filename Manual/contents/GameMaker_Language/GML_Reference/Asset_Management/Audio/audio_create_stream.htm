audio_create_stream

이 기능을 사용하면 일반 오디오 기능에서 사용할 수 있는 새 사운드 인덱스를 생성하여 외부 OGG 파일 소스에서 직접 오디오를 스트리밍할 수 있습니다. 이 함수에는 파일 이름(예: 포함된 파일일 수 있음)이 필요하며 사용할 새 사운드 인덱스를 반환합니다. 소리가 더 이상 필요하지 않으면 함수를 호출해야 합니다.audio_destroy_stream() 사운드 인덱스를 사용하여 메모리에서 제거하지 않으면 메모리 누수가 발생하여 속도가 느려지고 결국 게임이 중단될 수 있습니다.

참고: 이 기능은 HTML5 대상 플랫폼에서 사용할 수 없습니다.

통사론:

audio_create_stream(filename);

논쟁 유형 설명
filename 끈 Path to the file (OGG only) to stream the audio from.

보고:

Sound Asset

예시:

snd = audio_create_stream("Music/Track1.ogg");
audio_play_sound(snd, 0, true);

위의 코드는 주어진 파일에서 변수 "snd"에 새로운 사운드 인덱스를 생성한 다음 이 사운드를 재생합니다.
