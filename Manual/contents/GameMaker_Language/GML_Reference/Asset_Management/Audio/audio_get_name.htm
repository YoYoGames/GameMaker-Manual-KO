audio_get_name

이 함수는 주어진 오디오 리소스의 이름을 문자열로 반환합니다. "인덱스" 값은 리소스 자체의 값(자산 브라우저에 표시됨) 또는 다음을 사용하여 사운드를 재생할 때 제공되는 고유 ID 값일 수 있습니다.audio_play_sound(). 반환된 문자열은 리소스 ID와 동일 하지 않으며 리소스 자체에 액세스하는 데 사용할 수 없으므로 표시 또는 오류 검사에만 사용해야 합니다.

통사론:

audio_get_name(index);

논쟁 유형 설명
index The index of the sound to check.

보고:

예시:

var snd = audio_play_sound(choose(snd_One, snd_Two, snd_Three), 0, false);
var name = audio_get_name(snd);
show_debug_message("Sound = " + name);

위의 코드는 세 가지 다른 사운드 리소스에서 선택한 임의의 사운드를 재생한 다음 이름과 함께 디버그 메시지를 표시합니다.
