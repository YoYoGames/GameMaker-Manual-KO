audio_sound_set_listener_mask

이 기능은 지정된 청취자에게서만 재생되도록 사운드의 비트 마스크를 설정하는 데 사용할 수 있습니다. 다음을 사용하여 비트 마스크를 만들 수 있습니다.audio_get_listener_info() 그런 다음 비트 단위를 사용하여 or ("|")를 사용하여 사운드를 재생해야 하는 리스너로 사용자 지정 마스크를 만든 다음 재생한 후 이 사용자 지정 마스크를 사운드에 적용합니다. 이 마스크는 기본 마스크 또는 기능을 사용하여 설정한 마스크보다 우선합니다.audio_set_listener_mask().

이 기능은 사운드가 재생되도록 설정된 후에만 사용할 수 있으며 사운드를 재생하는 데 사용된 함수에서 반환된 사운드 ID가 필요합니다.

통사론:

audio_sound_set_listener_mask(soundID, mask);

논쟁 유형 설명
soundID 사운드 인스턴스 ID The unique ID of the sound to set the mask of
mask 진짜 The bitmask data to set for the sound

보고:

N/A

예시:

var snd = audio_play_sound(snd_PlayerDead, 10, false);
audio_sound_set_listener_mask(snd, global.PlayerMask);

위의 코드는 사운드를 재생한 다음 전역 변수에 저장된 마스크 데이터를 사용하여 재생할 리스너를 설정합니다.
