audio_get_type

GameMaker 에 오디오를 추가할 때 스트리밍 하거나 메모리에 저장할 수 있습니다. 주어진 사운드 인덱스가 스트리밍 오디오용인지 여부를 알아야 하는 경우 스트리밍된 오디오에 대해 1을 반환하고 메모리에 있는 사운드에 대해 0을 반환하고 오류가 있거나 인덱스가 오디오를 가리키지 않는 경우 -1을 반환하는 이 함수를 사용할 수 있습니다. 유효한 사운드 리소스.

통사론:

audio_get_type(index);

논쟁 유형 설명
index The index of the sound to check.

보고:

예시:

type = audio_get_type(snd_Music_1);

위의 코드는 "snd_Music_1" 변수에 인덱싱된 오디오 유형을 확인하고 반환된 값을 "type" 변수에 저장합니다.
