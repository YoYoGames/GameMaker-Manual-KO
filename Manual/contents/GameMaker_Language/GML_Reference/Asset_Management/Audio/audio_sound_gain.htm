audio_sound_gain

이 기능을 사용하여 주어진 시간 동안 사운드를 페이드 인 또는 페이드 아웃하거나 사운드 게인을 즉시 설정하는 데 사용할 수 있습니다.

시간은 밀리초 단위로 측정되며 이 기능을 사용하려면 해당 시간이 끝날 때까지 사운드에 도달하기 위한 최종 게인 레벨을 입력해야 합니다. 이 게인은 0(무음)과 0보다 큰 값 사이일 수 있지만 일반적으로 최대 볼륨을 1로 간주합니다. 1을 초과하는 모든 것을 사용할 수 있지만 사용되는 사운드와 컴파일할 플랫폼에 따라 사운드가 재생될 때 왜곡 또는 클리핑이 발생합니다. 게인 스케일은 선형이며 게인을 즉시 변경하려면 시간 인수를 0으로 설정하기만 하면 됩니다.

인덱스가 사운드 리소스인 경우 이 기능은 현재 방에서 재생 중인 모든 사운드 인스턴스에 영향을 미치고 최종 볼륨은 사운드의 모든 추가 인스턴스가 재생될 볼륨이 됩니다. 그러나 다음과 같은 함수에서 반환된 인덱스를 사용한 경우audio_play_sound() 사운드의 한 인스턴스에만 영향을 미칩니다.

통사론:

audio_sound_gain(index, volume, time);

논쟁 유형 설명
index 사운드 자산 또는 사운드 인스턴스 ID The index of the sound to set the gain for.
volume 진짜 Value for the music volume.
time 진짜 The length for the change in gain in milliseconds.

보고:

N/A

예시:

if val
{
    var snd = audio_play_sound(snd_fountain);
    audio_sound_gain(snd, 0, 0);
    audio_sound_gain(snd, 1, 5000);
}

위의 코드는 변수를 확인하고 반환하는 경우 true 그런 다음 재생할 사운드의 인덱스를 로컬 변수 "snd"에 할당합니다. 그런 다음 이 변수를 사용하여 해당 특정 사운드의 볼륨을 0으로 줄이고 5초에 걸쳐 최대 볼륨으로 페이드업합니다.
