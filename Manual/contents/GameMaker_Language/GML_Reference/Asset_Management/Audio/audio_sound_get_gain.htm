audio_sound_get_gain

이 기능은 주어진 사운드에 대한 현재 게인 값을 반환합니다. 사운드는 사용할 때 변수에 저장된 재생 중인 개별 사운드에 대한 인덱스에서 참조된 것일 수 있습니다.audio_play_sound() 또는audio_play_sound_at() 기능 또는 자산 브라우저의 실제 사운드 자산. 게인은 일반적으로 0에서 1 사이의 값으로 계산되지만 일부 플랫폼에서는 1보다 큰 게인을 가질 수 있지만 값이 1이면 "최대 볼륨"으로 간주되고 그보다 크면 오디오 클리핑이 발생할 수 있습니다.

통사론:

audio_sound_get_gain(index);

논쟁 유형 설명
index 사운드 자산 또는 사운드 인스턴스 ID The index of the sound to get the gain of.

보고:

Real

예시:

if audio_sound_get_gain(snd_Music) != 1
{
    audio_sound_gain(snd_Music, 1, 0);
}

위의 코드는 게인이 1이 아닌 경우 "snd_Music"으로 인덱싱된 사운드에서 재생되는 오디오의 게인을 변경합니다.
