audio_sound_is_playable

이 기능은 주어진 사운드 인덱스를 현재 재생할 수 있는지 확인하는 데 사용할 수 있습니다. 이는 스트리밍 및 스트리밍되지 않은 사운드 재생이 HTML5 대상 플랫폼에서 처리되는 방식이 다르기 때문에 필요하며 반환됩니다. true 소리를 재생할 수 있는 경우 false 할 수 없다면. HTML5 이외의 다른 모든 플랫폼에서 함수는 항상 다음을 반환합니다. true.

통사론:

audio_sound_is_playable(index);

논쟁 유형 설명
index 사운드 자산 The index of the sound to check.

보고:

Boolean

예시:

if audio_sound_is_playable(snd_Shot)
{
    audio_play_sound(snd_Shot, 10, false);
}

위의 코드는 재생 가능한 경우 "snd_Shot" 사운드를 재생합니다.
