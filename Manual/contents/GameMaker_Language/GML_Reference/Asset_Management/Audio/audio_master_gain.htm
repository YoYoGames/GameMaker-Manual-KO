audio_master_gain

이 기능을 사용하면 모든 사운드와 음악의 전체 볼륨에 대한 절대값을 설정할 수 있습니다. 일반적으로 최대 볼륨을 1로 간주하지만 0(무음)에서 0보다 큰 값까지의 선형 스케일을 기반으로 합니다., 사운드를 재생할 때 왜곡이나 클리핑이 발생할 수 있습니다. 이 기능은 게임 내에서 재생되는 모든 소리와 음악의 상대적 볼륨에 영향을 미칩니다.

통사론:

audio_master_gain(gain);

논쟁 유형 설명
gain 진짜 Value for the global volume (0 to 1).

보고:

N/A

예시:

if keyboard_check(vk_up)
{
    if vol < 1 vol += 0.05;
    audio_master_gain(vol);
}
if (keyboard_check(vk_down))
{
    if vol > 0 vol -= 0.05;
    audio_master_gain(vol);
}

위의 코드는 위쪽 및 아래쪽 화살표 키의 키 누름을 확인한 다음 변수 "vol"을 늘리거나 줄입니다. 그런 다음 이 변수는 게임의 모든 사운드와 음악의 전역 게인을 설정하는 데 사용됩니다.
