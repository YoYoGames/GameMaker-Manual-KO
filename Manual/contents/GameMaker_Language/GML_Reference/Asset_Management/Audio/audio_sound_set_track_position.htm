audio_sound_set_track_position

이 기능은 주어진 사운드 ID 또는 자산의 위치(초)를 설정합니다.

제공된 "인덱스"는 사운드의 단일 인스턴스일 수 있습니다(audio_play_sound() ) 또는 사운드 자산(예: 자산 브라우저를 통해 추가된 자산). 이 함수의 동작은 지정한 인덱스 종류에 따라 달라집니다.

  • 사용하고 있는 고유한 사운드 ID로 이미 재생 중인 경우 해당 위치가 즉시 지정된 시간으로 변경됩니다.
  • 사운드 자산인 경우 지정된 사운드 자산의 모든 추가 재생은 새 시간에 시작됩니다.

이 기능은 현재 재생 중인 사운드 또는 다음 재생(사운드 자산이 제공되는 경우)의 트랙 위치만 변경합니다. 루핑이 활성화된 상태에서 사운드가 재생되면 이 기능을 사용하여 정의된 트랙 위치가 아닌 항상 처음(0.0초)부터 후속 재생이 시작됩니다. 예를 들어, 5.7초에서 사운드 루프를 시작하면 해당 지점에서 첫 번째 사운드가 재생되지만 그 이후에는 처음부터 끝까지 전체 트랙을 계속 반복합니다.

통사론:

audio_sound_set_track_position(index, time);

논쟁 유형 설명
index 사운드 자산 또는 사운드 인스턴스 ID The index of the sound to change.
time 진짜 The time (in seconds) to set the start point to. Values longer than the length of the given sound are ignored.

보고:

N/A

예시:

var snd = audio_play_sound(snd_MainTrack, 0, false);
audio_sound_set_track_position(snd, 32);

위의 코드는 사운드를 재생한 다음 반환된 ID 값을 사용하여 사운드의 시작 위치를 32초로 설정합니다.
