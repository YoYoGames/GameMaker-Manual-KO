audio_destroy_stream

이전에 함수를 사용하여 파일에서 오디오 스트림을 만든 경우audio_create_stream() 그 소리가 더 이상 필요하지 않으면 이 함수를 호출하여 메모리에서 삭제할 수 있습니다. 사운드 인덱스가 제거된 후 추가로 호출하면 오류가 발생합니다. 이렇게 하면 스트림이 해제되지만 대상 플랫폼에서는 메모리 관리자에 표시되지 않을 수 있습니다. 이것은 GameMaker 가 메모리 할당 오버헤드를 방지하기 위해 메모리 리소스를 풀링하기 때문입니다. 따라서 메모리는 다른 것에 필요하거나 새로운 스트림에 재사용될 때까지 할당된 상태로 유지됩니다.

참고: 이 기능은 HTML5 대상 플랫폼에서 사용할 수 없습니다.

통사론:

audio_destroy_stream(filename);

논쟁 유형 설명
filename 끈 Path to the audio file that was streamed.

N/A

예시:

audio_destroy_stream(snd);

위의 코드는 "snd" 변수에 색인된 사운드를 메모리에서 제거합니다.
