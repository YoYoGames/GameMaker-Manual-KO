audio_is_paused

이 기능은 주어진 소리가 현재 일시 중지되었는지 확인하기 위해 확인합니다. 사운드는 사운드의 단일 인스턴스가 될 수 있습니다(재생되는 개별 사운드에 대한 인덱스는audio_play_sound() 또는audio_play_sound_at() 기능) 또는 사운드 자산, 이 경우 주어진 사운드의 모든 인스턴스가 확인되고 그 중 하나가 일시 중지되면 함수가 반환됩니다. true 그렇지 않으면 반환됩니다 false.

통사론:

audio_is_paused(index);

논쟁 유형 설명
index 사운드 자산 또는 사운드 인스턴스 ID The index of the sound to check.

보고:

Boolean

예시:

if audio_is_paused(snd_Waterfall)
{
    audio_resume_sound(snd_Waterfall);
}

위의 코드는 "snd_Waterfall" 변수에 인덱싱된 사운드가 현재 일시 중지되어 있고 반환되는지 확인합니다. true 그러면 사운드 재생이 다시 시작됩니다.
