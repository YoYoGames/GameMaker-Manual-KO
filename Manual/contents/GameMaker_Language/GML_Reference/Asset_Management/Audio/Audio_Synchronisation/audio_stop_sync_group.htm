audio_stop_sync_group

이 함수는 재생 중인 경우 지정된 동기화 그룹을 중지합니다. 그룹 인덱스는 함수를 사용하여 그룹을 만들 때 반환된 값입니다.audio_create_sync_group().

참고: 이 기능은 HTML5 대상 플랫폼에서 사용할 수 없습니다.

통사론:

audio_stop_sync_group(group_index);

논쟁 유형 설명
group_index 오디오 동기화 그룹 ID The group index to stop playing.

보고:

N/A

예시:

if mouse_check_button_pressed(mb_left)
{
    audio_stop_sync_group(sg);
}

위의 코드는 마우스 클릭을 확인하고 하나가 감지되면 변수 "sg"에 색인된 동기화 그룹을 중지합니다.
