audio_pause_sync_group

그의 함수는 재생 중인 경우 주어진 동기화 그룹을 일시 중지합니다. 그룹 인덱스는 함수를 사용하여 그룹을 생성할 때 반환된 값입니다.audio_create_sync_group(). 이것은 소리를 멈추지 않고 호출합니다.audio_resume_sync_group(), 일시 중지된 위치에서 다시 재생을 시작합니다.

참고: 이 기능은 HTML5 대상 플랫폼에서 사용할 수 없습니다.

통사론:

audio_pause_sync_group(group_index);

논쟁 유형 설명
group_index 오디오 동기화 그룹 ID The group index to pause.

보고:

N/A

예시:

if (keyboard_check_pressed((ord)"P"))
{
    global.Pause = !global.Pause
    if (global.Pause)
    {
        audio_pause_sync_group(sg);
    }
    else
    {
        audio_resume_sync_group(sg);
    }
}

위의 코드는 "P" 키의 키 누름을 확인하고 하나가 감지되면 "global.Pause" 변수를 토글한 다음 변수 "sg"에 색인된 동기화 그룹을 일시 중지하거나 재개하도록 확인합니다.
