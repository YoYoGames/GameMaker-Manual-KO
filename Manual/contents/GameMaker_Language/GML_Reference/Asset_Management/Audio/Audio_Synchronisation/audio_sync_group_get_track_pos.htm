audio_sync_group_get_track_pos

이 함수는 주어진 동기화 그룹의 현재 재생 위치를 반환합니다. 그룹 인덱스는 함수를 사용하여 그룹을 생성할 때 반환된 값입니다.audio_create_sync_group(), 반환 값은 트랙이 재생된 시간(초)입니다.

참고: 이 기능은 HTML5 대상 플랫폼에서 사용할 수 없습니다.

통사론:

audio_sync_group_get_track_pos(group_index);

논쟁 유형 설명
group_index 오디오 동기화 그룹 ID The group index to get the position of.

보고:

Real

예시:

var real_secs = audio_sync_group_get_track_pos(sg);
var secs = real_secs mod 60;
var mins = string(real_secs div 60);
if (secs > 9)
{
    secs = string(secs);
}
else
{
    secs = "0" + string(secs);
}

draw_text(32, 32, "Time = " + mins + ":" + secs);

위의 코드는 "sg" 변수에 인덱싱된 동기화 그룹의 시간 위치를 가져온 다음 이 값을 사용하여 화면에서 재생되는 시간을 그립니다.
