audio_create_sync_group

동기화 그룹을 만들고 이 그룹에 대한 모든 추가 오디오 기능 호출에 사용해야 하는 고유 ID 값을 반환합니다. 그룹을 반복하려면 다음을 전달하십시오. true, 그렇지 않으면 전달 false 그러나 루프를 원하면 나중에 추가되는 모든 트랙의 길이가 같아야 합니다. 동기화 그룹을 생성할 때 사용하지 않을 때는 동기화 그룹과 관련된 메모리와 사운드를 해제해야 합니다.audio_destroy_sync_group() 기능 - 예를 들어 Room End 또는 Destroy 이벤트에서.

참고: 이 기능은 HTML5 대상 플랫폼에서 사용할 수 없습니다.

통사론:

audio_create_sync_group(loop);

논쟁 유형 설명
loop 부울 Whether the tracks in the group should loop (true ) or not (false ).

보고:

Audio Sync Group ID

예시:

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_sound_gain(sound2, 0, 0);
audio_play_in_sync_group(sg, sound3);
audio_sound_gain(sound3, 0, 0);
audio_play_in_sync_group(sg, sound4);
audio_sound_gain(sound4, 0, 0);
audio_start_sync_group(sg);

위의 내용은 새 동기화 그룹을 만들고 그룹의 인덱스를 변수 "sg"에 할당합니다. 그런 다음 4개의 사운드가 그룹에 추가되고 그 중 3개의 게인이 0으로 설정됩니다. 마지막으로 동기화 그룹이 재생됩니다.
