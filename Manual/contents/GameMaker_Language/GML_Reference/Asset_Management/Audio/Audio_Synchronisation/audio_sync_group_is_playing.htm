audio_sync_group_is_playing

이 기능은 동기화된 그룹의 오디오가 재생 중인지 확인하는 데 사용할 수 있습니다. 함수에서 반환된 동기화 그룹 ID를 제공해야 합니다.audio_create_sync_group().

이 기능은 HTML5 대상 플랫폼에서 사용할 수 없습니다.

통사론:

audio_sync_group_is_playing(group_index);

논쟁 유형 설명
group_index 오디오 동기화 그룹 ID The group index to check.

보고:

Boolean

예시:

if audio_sync_group_is_playing(group_one)
{
    audio_stop_sync_group(group_one);
}

위의 코드는 동기화 그룹이 group_one 현재 재생 중이며 재생 중이면 그룹이 중지됩니다.
