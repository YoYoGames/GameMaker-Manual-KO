audio_stop_sound

이 기능은 현재 재생 중인 경우 해당 사운드를 중지합니다. 사운드는 사운드의 단일 인스턴스가 될 수 있습니다(재생되는 개별 사운드에 대한 인덱스는audio_play_sound() 또는audio_play_sound_at() 기능) 또는 사운드 자산, 이 경우 지정된 사운드의 모든 인스턴스가 중지됩니다.

통사론:

audio_stop_sound(index);

논쟁 유형 설명
index 사운드 자산 또는 사운드 인스턴스 ID The index of the sound to stop.

보고:

N/A

예시:

if !global.SFX
{
    audio_stop_sound(snd_Waterfall);
}
else
{
    audio_play_sound_at(snd_Waterfall, x, y, 0, 100, 300, 1, true, 1);
}

위의 코드는 전역 변수 "SFX"를 확인하고 반환되는 경우 false, 현재 재생 중인 "snd_Waterfall" 변수에 색인된 사운드를 중지하고 반환되는 경우 true, 소리를 반복합니다.
