audio_set_master_gain

이 기능을 사용하면 특정 청취자에 대한 모든 사운드 및 음악의 전체 볼륨에 대한 절대값을 설정할 수 있습니다. 기본 리스너 인덱스는 0이지만 함수를 사용할 수 있습니다. audio_get_listener_info() 대상 플랫폼에 사용할 수 있는 다양한 인덱스를 가져옵니다. 게인 값은 0(무음)에서 1(최대 볼륨)까지의 선형 스케일을 기반으로 하며 게임 내에서 해당 청취자를 통해 재생되는 모든 사운드 및 음악의 상대적 볼륨에 영향을 미칩니다.

통사론:

audio_set_master_gain(listenerIndex, gain);

논쟁 유형 설명
listenerIndex 실제 또는 오디오 리스너 ID The index of the listener to set the gain on.
gain 진짜 Value for the global volume (0 to 1).

보고:

N/A

예시:

var num = audio_get_listener_count();
for( var i = 0; i < num; i++;)
{
    var info = audio_get_listener_info(i);
    audio_set_master_gain(info[? "index"], 0.75);
    ds_map_destroy(info);
}

위의 코드는 사용 가능한 리스너를 반복한 다음 마스터 게인을 0.75로 설정합니다.
