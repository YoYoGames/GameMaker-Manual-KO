audio_emitter_set_listener_mask

이 함수는 이미터를 통해 재생되는 모든 사운드가 지정된 청취자에게서만 재생되도록 이미터의 비트 마스크를 설정하는 데 사용할 수 있습니다. 다음을 사용하여 비트 마스크를 만들 수 있습니다.audio_get_listener_info() 그런 다음 비트 단위를 사용하여 or ("|") 사운드를 재생해야 하는 리스너로 사용자 지정 마스크를 만든 다음 이 사용자 지정 마스크를 이미터에 적용합니다. 이 마스크는 기본 마스크 또는 기능을 사용하여 설정한 마스크보다 우선합니다.audio_set_listener_mask().

통사론:

audio_emitter_set_listener_mask(emitterID, mask);

논쟁 유형 설명
emitterID 오디오 이미터 ID The unique ID of the emitter to set the mask of
mask 진짜 The bitmask data to set for the sound

보고:

N/A

예시:

s_emit = audio_emitter_create();
audio_emitter_set_listener_mask(snd, global.PlayerMask);

위의 코드는 오디오 이미터를 생성한 다음 전역 변수에 저장된 마스크 데이터를 사용하여 재생할 리스너를 설정합니다.
