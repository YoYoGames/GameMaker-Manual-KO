audio_emitter_gain

이 기능은 사운드의 최대 게인(볼륨)을 설정합니다. 사운드에 대해 인지되는 볼륨은 감쇠 값 과 청취자 에 대한 상대적 위치에 따라 달라질 수 있지만 이 기능으로 게인을 설정하면 전체 볼륨이 지정된 게인 값을 초과하지 않습니다. 아래 이미지는 감쇠가 0보다 클 때 게인이 이미터의 볼륨에 미치는 영향을 보여줍니다.

이 기능은 주어진 이미터를 통해 재생되는 모든 후속 사운드뿐만 아니라 재생되는 동안 사운드의 볼륨을 변경합니다. 일부 플랫폼에서는 게인이 1보다 클 수 있지만 값이 1이면 "전체 볼륨"으로 간주되고 그보다 크면 오디오 클리핑이나 왜곡이 발생할 수 있습니다.

참고 최종 볼륨은 기능에 의해 설정된 전역 오디오 게인의 영향도 받습니다. audio_master_gain().

통사론:

audio_emitter_gain(emitter, gain);

논쟁 유형 설명
emitter 오디오 이미터 ID The index of the emitter to change.
gain 진짜 The maximum gain (default 1).

보고:

N/A

예시:

if up
{
    gain += 0.05;
    if (gain >= 1) up = false;
}
else
{
    gain -= 0.05;
    if (gain <= 0) up = true;
}

audio_emitter_gain(s_emit, gain);

위의 코드는 "gain" 변수를 다른 값으로 설정한 다음 동일한 변수를 사용하여 "s_emit" 변수에 인덱싱된 이미터의 이득을 설정합니다.
