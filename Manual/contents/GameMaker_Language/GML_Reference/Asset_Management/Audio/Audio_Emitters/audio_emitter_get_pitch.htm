audio_emitter_get_pitch

이 함수는 주어진 오디오 이미터에 대해 설정된 현재 피치 값을 반환합니다.

통사론:

audio_emitter_get_pitch(emitter);

논쟁 유형 설명
emitter 오디오 이미터 ID The index of the emitter to use.

보고:

Real

예시:

if audio_emitter_get_pitch(emitter_player) != 1
{
    audio_emitter_pitch(emitter_player, 1);
}

위의 코드는 주어진 이미 터의 현재 피치를 확인하고 1과 같지 않으면 1로 설정됩니다.
