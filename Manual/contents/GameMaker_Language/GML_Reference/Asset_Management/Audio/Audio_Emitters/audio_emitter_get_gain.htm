audio_emitter_get_gain

이 함수는 일반적으로 0과 1 사이에서 주어진 오디오 이미터에 대해 설정된 현재 게인(볼륨)을 반환합니다. 여기서 0은 무음이고 1은 최대 볼륨입니다. 일부 플랫폼에서는 게인이 1보다 클 수 있지만 값이 1이면 "전체 볼륨"으로 간주되고 그보다 크면 오디오 클리핑이 발생할 수 있습니다.

통사론:

audio_emitter_get_gain(emitter);

논쟁 유형 설명
emitter 오디오 이미터 ID The index of the emitter to use.

보고:

Real

예시:

if audio_emitter_get_gain(emitter_player) < 1
{
    audio_emitter_gain(emitter_player, 1);
}

위의 코드는 주어진 이미 터의 현재 게인을 확인하고 1보다 작 으면 1로 설정됩니다.
