audio_emitter_get_listener_mask

이 함수는 이미터가 사운드를 재생해야 하는 오디오 리스너를 정의하는 비트 마스크 데이터를 반환합니다. 자세한 내용은 오디오 리스너 섹션을 참조하십시오.

통사론:

audio_emitter_get_listener_mask(emitterID);

논쟁 유형 설명
emitterID 오디오 이미터 ID The unique ID of the emitter to get the mask of

보고:

Real

예시:

s_emit = audio_emitter_create();
if audio_emitter_get_listener_mask(s_emit) != global.PlayerMask
{
    audio_emitter_set_listener_mask(snd, global.PlayerMask);
}

위의 코드는 이미터를 생성한 다음 리스너 마스크 데이터를 확인하고 전역 변수에 저장된 것과 동일하지 않은 경우 전역 변수에 저장된 마스크 데이터를 사용하여 재생할 리스너를 설정합니다..
