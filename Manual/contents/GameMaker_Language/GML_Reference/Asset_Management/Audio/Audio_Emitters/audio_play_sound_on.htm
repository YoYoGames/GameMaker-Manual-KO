audio_play_sound_on

이 기능을 사용하면 이미터를 통해 모든 사운드 리소스를 재생할 수 있으며, 이미터 게인, 위치, 피치 또는 속도를 변경하면 사용자가 재생되는 최종 사운드를 듣는 방식에 영향을 줍니다. 사용할 이미터 인덱스, 재생할 사운드의 사운드 인덱스를 제공한 다음 사운드를 루프할지 여부를 지정합니다. 다음으로 우선 순위를 지정할 수 있습니다. 이 우선 순위는 재생 중인 사운드 수가 기능에서 설정한 제한을 초과할 때 사운드를 처리하는 방법을 결정하는 데 사용됩니다.audio_channel_num(). 우선 순위가 낮은 사운드는 중지되어 우선 순위가 높은 사운드를 선택하고 우선 순위 값은 임의의 실수일 수 있습니다(실제 값은 임의적이며 GameMaker 가 동일한 우선 순위를 지정하므로 0에서 1 또는 0에서 100일 수 있음). 우선 순위를 다룰 때 숫자가 높을 수록 우선 순위가 높아져 우선 순위 100의 사운드가 우선 순위 1의 사운드보다 선호됩니다.

이 함수는 또한 재생 중인 사운드에 대한 고유 인덱스 번호를 반환하며, 이 번호는 변수에 저장되어 적절한 기능으로 일시 중지하거나 중지할 수 있습니다. 즉, 한 번에 동일한 사운드의 여러 인스턴스가 재생되는 경우 해당 사운드의 한 인스턴스만 대상으로 지정하여 오디오 기능을 사용하여 처리할 수 있습니다.

통사론:

audio_play_sound_on(emitter, sound, loop, priority, [gain], [offset], [pitch], [listener_mask]);

논쟁 유형 설명
emitter 오디오 이미터 ID The index of the emitter to use.
sound 사운드 자산 The index of the sound to use.
loop 부울 Flags the sound as looping or not.
priority 진짜 Set the channel priority for the sound.
gain 진짜 OPTIONAL Value for the gain.
offset 진짜 OPTIONAL The time (in seconds) to set the start point to. Values longer than the length of the given sound are ignored.
pitch 진짜 OPTIONAL The pitch multiplier (default 1).
listener_mask 진짜 OPTIONAL The bitmask data to set for the sound. On the HTML5 target this will have no effect as the target does not support more than one listener.

보고:

Sound Instance ID

예 1:

if global.SFX
{
    audio_play_sound_on(s_emit, snd_Explode, false, 1);
}

위의 코드는 전역 변수 "SFX"를 확인하고 반환되는 경우 true 그러면 "snd_Explode" 변수에 색인된 사운드가 우선순위가 낮고 루핑이 없는 "s_emit" 변수에 색인된 이미터를 통해 재생됩니다.

예 2:

if hit == true
{
    audio_play_sound_on(s_emit, snd_Hit, false, 1, 1.3);
}

위의 코드는 코드를 실행하는 인스턴스가 적중되었는지 확인합니다. 만약에 true 1.3의 약간 더 높은 게인으로 이미터 "s_emit"를 통해 "snd_Hit" 사운드를 재생합니다. 여기에서 설정한 게인에 이미터의 게인을 곱하여 사운드가 재생되는 최종 게인을 얻습니다.
