audio_emitter_create

이 함수는 새 오디오 이미터를 만들고 인덱스를 반환합니다. 이 인덱스는 이 이미터와 관련된 모든 추가 기능에 대한 변수에 저장되어야 하고, 더 이상 필요하지 않을 때 함수를 사용하여 메모리에서 제거해야 합니다.audio_emitter_free() 결국 게임을 중단시킬 수 있는 메모리 누수를 방지합니다.

오디오 이미터는 다음 기본 설정으로 생성됩니다.

  • 1의 이득과 피치
  • 기본 리스너로만 설정된 리스너 마스크(1)
  • (0, 0, 0) 의 위치
  • 속도 0(vx = 0, vy = 0, vz = 0)

통사론:

audio_emitter_create();

보고:

Audio Emitter ID

예시:

s_emit = audio_emitter_create();

위의 코드는 새 오디오 이미터를 만들고 해당 인덱스를 변수 "s_emit"에 할당합니다.
