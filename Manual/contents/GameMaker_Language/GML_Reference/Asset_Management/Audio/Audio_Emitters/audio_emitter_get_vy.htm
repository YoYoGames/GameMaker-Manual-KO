audio_emitter_get_vy

이 함수는 주어진 오디오 이미터에 대한 y축을 따라 현재 속도를 반환합니다.

통사론:

audio_emitter_get_vy(emitter);

논쟁 유형 설명
emitter 오디오 이미터 ID The index of the emitter to use.

보고:

Real

예시:

if audio_emitter_get_vy(emitter_player) != 0
{
    audio_emitter_velocity(emitter_player, 0, 0, 0);
}

위의 코드는 주어진 이미터의 현재 y 속도를 확인하고 0이 아니면 0으로 설정합니다.
