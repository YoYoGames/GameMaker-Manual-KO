audio_emitter_get_y

이 함수는 주어진 오디오 이미터의 현재 y 위치를 반환합니다.

통사론:

audio_emitter_get_y(emitter);

논쟁 유형 설명
emitter 오디오 이미터 ID The index of the emitter to use.

보고:

Real

예시:

if audio_emitter_get_y(emitter_player) != x
{
    audio_emitter_position(emitter_player, x, y, 0);
}

위의 코드는 주어진 이미터의 현재 y 위치를 확인하고 인스턴스 y 위치와 같지 않으면 인스턴스 위치로 설정됩니다.
