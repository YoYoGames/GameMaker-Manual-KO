audio_emitter_free

이 기능을 사용하면 메모리에서 주어진 이미터를 제거할 수 있습니다. 이것은 이미터가 방 이나 게임에서 더 이상 사용되지 않을 때마다 항상 수행 되어야 합니다. 다운되어 결국 게임이 중단됩니다.

통사론:

audio_emitter_free(emitter);

논쟁 유형 설명
emitter 오디오 이미터 ID The index of the emitter to free.

보고:

N/A

예시:

if lives = 0
{
    audio_emitter_free(s_emit);
    room_goto(rm_Menu);
}

위의 코드는 전역 변수 "lives"의 값을 확인하고 0을 반환하면 변수 "s_emit"에 색인된 이미터를 파괴한 다음 변수 "rm_Menu"에 색인된 방으로 이동합니다.
