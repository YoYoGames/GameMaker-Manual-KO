audio_emitter_velocity

이 기능은 이미터 도플러 효과를 제공하고 주어진 상대적 x, y 및 z 위치에서 해석되는 벡터를 기반으로 오디오 모션을 시뮬레이션하는 데 사용할 수 있습니다. 에미터 자체가 움직이지 않을 경우 일반적으로 이 값을 설정할 필요가 없지만, 예를 들어 적이 오른쪽에서 와서 왼쪽으로 스크롤하는 스크롤 슈팅 게임을 만드는 경우에는 이 값을 설정합니다. 적의 생성 이벤트에서 (hspeed, 0, 0)의 속도를 가지려면 (그리고 다음을 사용하여 단계 이벤트에서 각 인스턴스의 이미터를 업데이트합니다.audio_emitter_position() ) 적군 인스턴스가 플레이어 인스턴스를 전달할 때 올바른 도플러를 발생시키는 모든 사운드를 제공합니다.audio_listener_position() 리스너 를 올바른 위치로 설정).

통사론:

audio_emitter_velocity(emitter, vx, vy, vz);

논쟁 유형 설명
emitter 오디오 이미터 ID The index of the emitter to change.
vx 진짜 The x vector value (default 0).
vy 진짜 The y vector value (default 0).
vz 진짜 The z vector value (default 0).

보고:

N/A

예시:

s_emit = audio_emitter_create();
audio_emitter_position(s_emit, room_width, 0, 0);
audio_emitter_velocity(s_emit, -5, 0, 0);

위의 코드는 오디오 이미터를 만들고 "s_emit" 변수에 인덱스를 할당합니다. 그런 다음 이 이미터는 방 내의 위치에 배치되고 x축을 따라 단계당 5픽셀의 속도가 제공됩니다(단계당 수평 속도가 5픽셀인 것처럼 청취자와 관련하여 올바르게 도플러됩니다).
