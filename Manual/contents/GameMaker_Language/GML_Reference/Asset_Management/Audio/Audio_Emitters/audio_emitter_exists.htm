audio_emitter_exists

이 함수는 오디오 이미터가 존재하는지 여부를 반환합니다(true ) 여부(false ). 검색하는 인덱스가 이전에 초기화되지 않은 경우 이 함수는 존재하지 않는 인덱스를 검색하므로 오류가 발생합니다.

통사론:

audio_emitter_exists(index);

논쟁 유형 설명
index 오디오 이미터 ID The index of the emitter to check the existence of.

보고:

Boolean

예시:

if audio_emitter_exists(s_emit)
{
    audio_play_sound_on(s_emit, snd_Explode, false, 1);
}
else
{
    s_emit = audio_emitter_create();
    audio_play_sound_on(s_emit, snd_Explode, false, 1);
}

위의 코드는 (이전에 초기화된) 변수 "s_emit"에 인덱싱된 이미터가 존재하는지 확인합니다. 그렇다면 소리가 재생되지만 그렇지 않은 경우 생성된 다음 소리가 재생됩니다.
