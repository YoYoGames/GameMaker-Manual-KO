audio_emitter_position

이 기능을 사용하면 3D 오디오 공간 내에서 오디오 이미터의 위치를 변경할 수 있습니다. 위치는 오디오 공간 내에서 청취자 가 위치하는 위치에 따라 다른 방식으로 사운드에 영향을 줍니다(기본 위치는 (0, 0, 0)입니다. 참조).audio_listener_position() 자세한 내용은 참조), 따라서 예를 들어 이미터 위치가 (100, 0, 0)으로 설정되고 현재 청취자가 (200, 0, 0)에 있는 경우 이미터에서 스트리밍되는 오디오는 왼쪽에 표시됩니다. 오디오 필드. 아래 이미지는 이미터와 수신기에 대한 상대적 위치의 시각적 표현을 보여줍니다.

통사론:

audio_emitter_position(emitter, x, y, z);

논쟁 유형 설명
emitter 오디오 이미터 ID The index of the emitter to change.
x 진짜 The x position.
y 진짜 The y position.
z 진짜 The z position.

보고:

N/A

예시:

if speed > 0
{
    audio_emitter_position(s_emit, x, y, 0);
}

위의 코드는 인스턴스 속도가 0 이상인지 확인하고 만약 그렇다면 "s_emit" 변수에 색인된 오디오 이미터를 현재 x/y 위치로 업데이트합니다.
