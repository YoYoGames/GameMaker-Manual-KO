audio_emitter_pitch

이 기능은 주어진 이미터에서 방출되는 모든 사운드의 피치를 변경하는 데 사용할 수 있습니다. 이것은 입력 값에 현재 피치를 해당 양만큼 곱한다는 점에서 피치 승수 입니다. 따라서 기본값 1은 피치 변경이 없으며, 값이 1보다 작으면 피치가 낮아지고 1보다 크면 피치가 높아집니다. 0보다 작거나 5보다 큰 값은 어쨌든 들리지 않을 수 있으므로 이 기능에 대해 작은 증분을 사용하는 것이 가장 좋습니다.

오디오 이미터에서 이미 재생 중인 사운드는 피치 변경을 들을 수 있도록 다시 시작할 필요가 없습니다. 피치 변경은 즉시 적용됩니다.

통사론:

audio_emitter_pitch(emitter, pitch);

논쟁 유형 설명
emitter 오디오 이미터 ID The index of the emitter to change.
pitch 진짜 The pitch multiplier (default 1).

보고:

N/A

예시:

switch (gear)
{
    case 1: audio_emitter_pitch(s_emit, 0.8); break;
    case 2: audio_emitter_pitch(s_emit, 0.9); break;
    case 3: audio_emitter_pitch(s_emit, 0.95); break;
    case 4: audio_emitter_pitch(s_emit, 1); break;
    case 5: audio_emitter_pitch(s_emit, 1.2); break;
}

위의 코드는 "gear" 변수의 값을 기반으로 "s_emit" 변수에 인덱싱된 이미터에서 재생되는 오디오의 피치를 변경합니다.
