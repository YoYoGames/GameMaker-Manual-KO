audio_sound_get_pitch

이 기능은 주어진 사운드의 피치를 얻는 데 사용할 수 있습니다. 사운드는 사용할 때 변수에 저장된 재생 중인 개별 사운드에 대한 인덱스에서 참조된 것일 수 있습니다.audio_play_sound() 또는audio_play_sound_at() 기능 또는 자산 브라우저의 실제 사운드 자산.

통사론:

audio_sound_get_pitch(index);

논쟁 유형 설명
index 사운드 자산 또는 사운드 인스턴스 ID The index of the sound to get the pitch of.

보고:

Real

예시:

if audio_sound_get_pitch(snd_Explode) != 1
{
    audio_sound_pitch(snd_Explode, 1);
}

위의 코드는 피치가 1이 아닌 경우 "snd_Explode"로 색인된 사운드에서 재생되는 오디오의 피치를 변경합니다.
