audio_sound_length

이 함수는 주어진 소리의 길이를 초 단위로 반환합니다. 사운드는 사용할 때 변수에 저장된 재생 중인 개별 사운드에 대한 인덱스에서 참조할 수 있습니다.audio_play_sound() 또는audio_play_sound_at() 기능 또는 자산 브라우저의 실제 사운드 자산.

통사론:

audio_sound_length(index);

논쟁 유형 설명
index 사운드 자산 또는 사운드 인스턴스 ID The index of the sound to check.

보고:

Real

예시:

var len;
len = audio_sound_length(snd_Beam);
audio_play_sound(snd_Beam, 1, false);
alarm[0] = room_speed * len;

위의 코드는 "snd_Beam" 변수에 인덱스된 사운드의 길이(초)를 얻은 다음 사운드를 재생하고 사운드의 길이를 사용하여 사운드 재생이 끝나면 알람이 울리도록 설정하여 필요한 시간을 계산합니다. 알람.
