audio_sound_get_track_position

이 기능은 사운드가 재생될 사운드 파일 내의 위치(초)를 가져옵니다. 사운드는 사운드의 단일 인스턴스일 수 있습니다(재생되는 개별 사운드에 대한 인덱스는audio_play_sound() 또는audio_play_sound_at() 기능).

통사론:

audio_sound_get_track_position(index);

논쟁 유형 설명
index 사운드 인스턴스 ID The index of the sound to get the play position of.

보고:

Real

예시:

if audio_sound_get_track_position(global.Music) != 0
{
    audio_sound_set_track_position(global.Music, 0);
}

위의 코드는 트랙의 시작 위치를 확인하고 0초가 아니면 0초로 설정합니다.
