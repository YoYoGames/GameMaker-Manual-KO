audio_listener_velocity

이 기능은 청취자에게 도플러 효과를 주고 주어진 상대적 x, y 및 z 위치에서 해석되는 벡터를 기반으로 오디오 모션을 시뮬레이션하는 데 사용할 수 있습니다. 리스너 자체가 움직이지 않거나 움직임이 일정한 움직임이 아닌 경우 일반적으로 이러한 값을 설정할 필요가 없지만, 예를 들어 플레이어가 바닥에 일정한 바닥이 있는 스크롤링 게임을 만드는 경우 위쪽 이동 및 적들이 일정한 위쪽에서 아래쪽으로 이동하는 경우 리스너 및 이미터 속도를 설정합니다(이미터의 경우 사용할 audio_emitter_velocity() )를 적절한 벡터에 추가하여 플레이어 인스턴스를 지나갈 때 올바른 도플러 효과를 시뮬레이션합니다.

참고 리스너가 여러 개인 경우 함수를 사용해야 합니다. audio_listener_set_velocity().

아래 이미지는 위에 제공된 예제 게임에 대해 이것이 어떻게 설정될 수 있는지 보여줍니다.

통사론:

audio_listener_velocity(vx, vy, vz);

논쟁 유형 설명
vx 진짜 The x velocity component of the listener (default 0).
vy 진짜 The y velocity component of the listener (default 0).
vz 진짜 The z velocity component of the listener (default 0).

보고:

N/A

예시:

if speed > 0
{
    audio_listener_velocity(abs(hspeed), abs(vspeed), 0);
}

위의 코드는 플레이어 인스턴스 속도가 0을 초과하는지 확인하고 그렇다면 적절한 절대값을 사용합니다. hspeed 그리고 vspeed 리스너 속도를 설정하는 구성 요소.
