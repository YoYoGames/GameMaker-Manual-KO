audio_get_listener_info

이 함수는 DS 맵 을 생성하고 주어진 리스너에 대한 정보로 채웁니다.

참고 반품된 제품의 파손에 대한 책임은 귀하에게 있습니다. DS Map 적절한 기능을 사용하여

DS 맵에는 다음 키가 포함됩니다.

  • " name " - 대부분의 대상 플랫폼에서 "default"가 표준 리스너 이름인 문자열로 된 리스너의 이름
  • " mask " - 리스너의 비트 마스크
  • " index " - 리스너의 고유 인덱스 값

마스크 값을 사용하여 비트 단위를 사용하여 한 번에 여러 청취자로부터 재생하도록 사운드 또는 이미터를 설정할 수 있습니다. or "|" 사운드에 대한 마스크를 생성하기 위해(아래 예제 코드 참조) 인덱스는 다음과 같은 기능을 사용하여 주어진 청취자의 위치 또는 속도와 같은 속성을 설정하는 데 사용됩니다.audio_listener_set_position().

통사론:

audio_get_listener_info(num);

논쟁 유형 설명
num 실제 또는 오디오 리스너 ID The listener number to get the data for.

보고:

DS Map ID

예시:

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var m = audio_listener_get_data(info[? "mask"]);
    mask = mask | m;
    ds_map_destroy(info);
}
audio_set_listener_mask(mask);

위의 코드는 사용 가능한 리스너의 수를 확인한 다음 루프를 통해 마스크 비트를 얻은 다음 결합하여 전역 리스너에 적용되는 단일 비트 마스크를 만듭니다.
