audio_listener_set_position

이 기능을 사용하면 3D 오디오 공간 내에서 주어진 청취자 의 위치를 변경할 수 있습니다. 기본 리스너 인덱스는 0이지만 함수를 사용할 수 있습니다.audio_get_listener_info() 대상 플랫폼에 사용할 수 있는 다양한 인덱스를 가져옵니다. 아래 예시 이미지는 오디오 공간에서 리스너의 기본 위치를 보여줍니다.

보시다시피 기본 위치는 (0, 0, 0)이지만 일반적으로 이 기능을 사용하여 게임 내에서 플레이어 개체와 함께 리스너를 이동하고 이미터에 의해 생성된 오디오가 플레이어에게 들리는 방식을 변경합니다. 예를 들어 아래의 하향식 게임 이미지에서 플레이어 인스턴스는 플레이어가 레벨 주위를 이동할 때 다양한 이미터의 오디오가 "변경"되도록 하는 리스너를 설정합니다.

통사론:

audio_listener_set_position(index, x, y, z);

논쟁 유형 설명
index 실제 또는 오디오 리스너 ID The listener to set the position of (default 0).
x 진짜 The x position of the listener (default 0).
y 진짜 The y position of the listener (default 0).
z 진짜 The z position of the listener (default 0).

보고:

N/A

예시:

var _vmat = camera_get_view_mat(view_camera[0]);
audio_listener_set_position(global.Player_Listener, _vmat[0], _vmat[1], _vmat[2]);
audio_listener_set_orientation(info[? "index"], _vmat[3], _vmat[4], _vmat[5], _vmat[6], _vmat[7], _vmat[8]);

위의 코드는 카메라 뷰 [0]에 대한 뷰 매트릭스를 검색한 다음 전역 변수 "Player_Listener"에 저장된 ID로 리스너에 대한 오디오 리스너 위치와 방향을 설정하는 데 사용합니다.
