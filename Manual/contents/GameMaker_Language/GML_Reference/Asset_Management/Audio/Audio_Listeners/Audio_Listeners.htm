오디오 리스너

복잡한 오디오 상황을 처리할 때 오디오 환경에서 청취자가 위치해야 하는 위치를 정의하는 것이 유용할 수 있습니다. 간단한 오디오 기능으로 작업할 때 일반적으로 어떤 식으로든 리스너를 변경할 필요가 없지만 3D 오디오 공간 내에서 사실적인 오디오를 생성하기 위해 이미터를 사용할 때 리스너를 올바르게 변경하고 이동하는 것이 중요합니다. 그렇지 않으면 사운드가 게임을 하는 사람에게 잘못된 위치 에 있게 됩니다.

기본적으로 게임에서 자동으로 생성되는 하나의 리스너가 있으며 ID 0 을 사용합니다(리스너 기능 참조용).

GameMaker 는 다음 리스너 기능을 제공합니다.

  • audio_listener_position
  • audio_listener_velocity
  • audio_listener_orientation
  • audio_listener_get_data
  • audio_listener_set_position
  • audio_listener_set_velocity
  • audio_listener_set_orientation

특정 대상 플랫폼은 하나 이상의 청취자를 허용할 수 있으며, 각 청취자는 다른 출력 장치에 해당합니다. 따라서 GameMaker 는 사용 가능한 리스너를 찾고 재생 중인 사운드에 대해 설정할 수 있는 여러 추가 기능을 제공합니다.

  • audio_get_listener_count
  • audio_get_listener_info
  • audio_get_listener_mask
  • audio_set_listener_mask

개별 사운드에 대한 마스크 데이터를 가져오고 설정하는 기능도 있습니다.

  • audio_sound_get_listener_mask
  • audio_sound_set_listener_mask

이미터의 마스크 데이터 가져오기 및 설정:

  • audio_emitter_get_listener_mask
  • audio_emitter_set_listener_mask
