audio_set_listener_mask

시스템에서 여러 리스너를 사용할 때 사운드에 대한 비트 마스크를 설정하고 플래그가 지정된 리스너에서만 듣게 할 수 있습니다. 그러나 이 기능을 사용하여 전역 마스크를 설정할 수도 있으며 각 사운드에 대해 개별적으로 마스크를 설정할 필요 없이 정상적으로 재생되는 모든 사운드는 이 마스크로 플래그가 지정된 청취자에게서 들을 수 있습니다.

통사론:

audio_set_listener_mask(mask);

논쟁 유형 설명
mask 진짜 The bit-mask data to set for the listeners

보고:

N/A

예시:

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var m = audio_listener_get_data(info[? "mask"]);
    mask = mask | m;
    ds_map_destroy(info);
}
audio_set_listener_mask(mask);

위의 코드는 사용 가능한 리스너의 수를 확인한 다음 루프를 통해 마스크 비트를 얻은 다음 결합하여 전역 리스너에 적용되는 단일 비트 마스크를 만듭니다.
