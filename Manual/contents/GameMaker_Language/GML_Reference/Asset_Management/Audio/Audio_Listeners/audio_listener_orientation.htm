audio_listener_orientation

이 기능을 사용하면 3D 오디오 공간 내에서 청취자 의 방향을 변경할 수 있습니다. 보는 방향과 위로 향하는 방향은 주어진 상대적 x, y 및 z 위치에서 해석되는 벡터를 기반으로 하며 기본적으로 보는 방향의 경우 (0, 0, 1), 다음의 경우 (0, 1, 0) 입니다. 아래 그림과 같이 위쪽 방향으로

참고 리스너가 여러 개인 경우 함수를 사용해야 합니다.audio_listener_set_orientation().

이 기능으로 청취자 방향을 변경하면 게임 룸 주변의 오디오 이미터에서 생성된 사운드가 게임 플레이어가 인식하는 방식이 변경됩니다. 아래 예에서 리스너가 기본 위치에 있을 때 방사체에 의해 생성된 사운드는 리스너의 오른쪽 아래에서 들리는 것처럼 보이지만 리스너의 새로운 위치와 방향을 사용하면 이제 오는 것으로 인식됩니다. 위에서 오른쪽으로.

통사론:

audio_listener_orientation(lookat_x, lookat_y, lookat_z, up_x, up_y, up_z);

논쟁 유형 설명
lookat_x 진짜 The x look vector (default 0).
lookat_y 진짜 The y look vector (default 0).
lookat_z 진짜 The z look vector (default 1).
up_x 진짜 The x up vector (default 0).
up_y 진짜 The y up vector (default 1).
up_z 진짜 The z up vector (default 0).

예시:

dx = dcos(direction);
dy = - dsin(direction);
dz = - dsin(zdirection);
audio_listener_position(x, y, z);
audio_listener_orientation(dx, dy, dz, 0, 0, 1);

위의 코드는 3D 오디오 리스너 위치를 인스턴스의 x 및 y 변수 와 추가 z 변수로 설정합니다. 그런 다음 오디오 리스너의 방향을 인스턴스의 direction 과 추가 zdirection 변수에서 계산된 세 개의 변수로 설정합니다.
