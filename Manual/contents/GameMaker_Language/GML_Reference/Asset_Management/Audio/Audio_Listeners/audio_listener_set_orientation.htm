audio_listener_set_orientation

이 기능을 사용하면 3D 오디오 공간 내에서 주어진 청취자 의 방향을 변경할 수 있습니다. 기본 리스너 인덱스는 0이지만 함수를 사용할 수 있습니다.audio_get_listener_info() 대상 플랫폼에 사용할 수 있는 다양한 인덱스를 가져옵니다.

벡터 및 상향 벡터 보기 는 주어진 상대적 x, y 및 z 위치에서 확인된 값을 기반으로 하며, 보기 벡터의 경우 기본적으로 (0, 0, 1), 벡터의 경우 (0, 1, 0) 으로 설정됩니다. 아래 그림과 같이 위쪽 벡터:

이 기능을 사용하여 주어진 청취자 방향을 변경하면 게임 룸 주변의 오디오 이미터에서 생성된 사운드가 게임 플레이어가 인식하는 방식이 변경됩니다. 아래 예에서 리스너가 기본 위치에 있을 때 방사체에 의해 생성된 사운드는 리스너의 오른쪽 아래에서 들리는 것처럼 보이지만 리스너의 새로운 위치와 방향을 사용하면 이제 오는 것으로 인식됩니다. 위에서 오른쪽으로.

통사론:

audio_listener_set_orientation(index, lookat_x, lookat_y, lookat_z, up_x, up_y, up_z);

논쟁 유형 설명
index 실제 또는 오디오 리스너 ID The listener to set the orientation of.
lookat_x 진짜 The x look vector (default 0).
lookat_y 진짜 The y look vector (default 0).
lookat_z 진짜 The z look vector (default 1).
up_x 진짜 The x up vector (default 0).
up_y 진짜 The y up vector (default 1).
up_z 진짜 The z up vector (default 0).

예시:

var _m = camera_get_view_mat(view_camera[0]);
audio_listener_set_position(global.Player_Listener, _m[0], _m[1], _m[2]);
audio_listener_set_orientation(info[? "index"], _m[3], _m[4], _m[5], _m[6], _m[7], _m[8]);

위의 코드는 카메라 뷰 [0]에 대한 뷰 매트릭스를 검색한 다음 이를 사용하여 전역 변수 "Player_Listener"에 저장된 ID로 리스너에 대한 오디오 리스너 위치와 방향을 설정합니다.
