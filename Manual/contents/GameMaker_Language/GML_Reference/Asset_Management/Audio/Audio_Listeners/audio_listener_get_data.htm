audio_listener_get_data

이 함수는 DS 맵 을 생성하고 주어진 청취자에 대한 위치, 속도 및 방향 값으로 채웁니다. 기본 리스너 인덱스는 0이지만 함수를 사용할 수 있습니다.audio_get_listener_info() 대상 플랫폼에 사용할 수 있는 다양한 인덱스를 가져옵니다. 잘못된 리스너 인덱스를 제공하면 함수는 -1을 반환합니다.

참고 적절한 기능을 사용하여 반환된 DS 맵 을 파괴하는 것은 귀하의 책임입니다.

DS 맵에는 다음 키가 포함됩니다.

  • " x " - 리스너의 x 위치
  • " y " - 리스너의 y 위치
  • " z " - 리스너의 z 위치
  • " vx " - 리스너의 x축 속도
  • " vy " - 청취자의 y축 속도
  • " vz " - 리스너의 z축 속도
  • " lookat_x " - 리스너 보기 벡터의 x 구성 요소
  • " lookat_y " - 리스너 보기 벡터의 y 구성요소
  • " lookat_z " - 리스너 보기 벡터의 z 구성요소
  • " up_x " - 리스너 위쪽 벡터의 x 구성 요소
  • " up_y " - 리스너 위쪽 벡터의 y 구성 요소
  • " up_z " - 리스너 위쪽 벡터의 z 구성요소

통사론:

audio_listener_get_data(index);

논쟁 유형 설명
index 실제 또는 오디오 리스너 ID The listener to get the data for (default is 0).

보고:

DS Map ID

예시:

var num = audio_get_listener_count();
for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var data = audio_listener_get_data(info[? "index"]);
    if data[? "x"] != 0
    {
        audio_listener_set_position(info[? "index"], 0, 0, 0);
    }
    ds_map_destroy(info);
    ds_map_destroy(data);
}

위의 코드는 사용 가능한 리스너의 수를 확인한 다음 루프를 통해 x 위치가 0이 아니면 위치를 0, 0, 0으로 설정합니다.
