audio_get_listener_mask

이 함수는 오디오 수신기에 대한 현재 기본(전역) 마스크를 정의하는 비트 마스크 데이터를 반환합니다.

통사론:

audio_get_listener_mask();

보고:

예시:

var g_mask = audio_get_listener_mask();
if g_mask != global.Audio_Mask
{
    var num = audio_get_listener_count();
    global.Audio_Mask = 0;

    for(var i = 0; i < num; ++i;)
    {
        var info = audio_get_listener_info(i);
        var m = audio_listener_get_data(info[? "mask"]);
        global.Audio_Mask = global.Audio_Mask | m;
        ds_map_destroy(info);
    }
    audio_set_listener_mask(mask);
}

위의 코드는 현재 리스너 마스크 데이터를 가져와 전역 변수에 저장된 데이터와 비교합니다. 동일하지 않은 경우 코드는 사용 가능한 리스너 수를 확인한 다음 루프를 통해 마스크 비트를 얻은 다음 이를 결합하여 오디오 시스템에 적용되는 단일 마스크를 생성하여 글로벌 리스너를 정의합니다.
