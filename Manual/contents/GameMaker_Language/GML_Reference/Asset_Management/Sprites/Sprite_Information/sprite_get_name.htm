sprite_get_name

이 함수는 이름 을 지정된 스프라이트의 문자열로 반환합니다. 이 이름은 기본 GameMaker 창의 리소스 트리에서 스프라이트에 대해 지정된 이름입니다. 이것은 단지 문자열이며 스프라이트를 직접 참조하는 데 사용할 수 없습니다. 스프라이트 인덱스 가 필요하기 때문입니다. 그러나 이 문자열을 사용하여 함수와 함께 반환된 문자열을 사용하여 스프라이트 인덱스 를 가져올 수 있습니다.asset_get_index().

통사론:

sprite_get_name(index);

논쟁 유형 설명
index 유산 The index of the sprite to get the name of.

보고

String

예시:

str = sprite_get_name(sprite_index);

위의 코드는 코드를 실행하는 인스턴스에 대한 스프라이트 인덱스의 이름을 가져오고 "str" 변수에 반환 값을 저장합니다.
