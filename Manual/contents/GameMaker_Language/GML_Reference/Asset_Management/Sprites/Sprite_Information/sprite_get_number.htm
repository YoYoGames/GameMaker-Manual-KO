sprite_get_number

스프라이트에는 최소한 하나의 하위 이미지가 있어야 하며 이 함수는 얼마나 많은 하위 이미지를 가지고 있는지를 반환합니다. 스프라이트 하위 이미지는 0부터 번호가 매겨지기 시작하므로 이 함수가 3을 반환하면(검사 중인 스프라이트에 3개의 하위 이미지가 있음) 0, 1, 2로 번호가 매겨집니다.

통사론:

sprite_get_number(index);

논쟁 유형 설명
index The index of the sprite.

보고

예시:

image_index = floor(random(sprite_get_number(sprite_index)));

위의 코드는 스프라이트가 가지고 있는 하위 이미지의 수를 기반으로 이미지 인덱스를 임의의 프레임으로 설정합니다.
