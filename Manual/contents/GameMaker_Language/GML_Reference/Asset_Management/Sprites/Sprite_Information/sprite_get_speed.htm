sprite_get_speed

이 함수는 스프라이트 편집기 에서 스프라이트 리소스에 대해 정의된 스프라이트 속도를 검색하는 데 사용할 수 있습니다. 그런 다음 반환된 값을 사용하여 예를 들어 여러 프레임에 대해 그릴 수 있는 프레임 수를 계산할 수 있습니다.image_speed. 반환 값은 스프라이트 편집기에 적용된 속도 유형 ( 초당 프레임 수 또는 게임 프레임당 프레임 수)에 따라 매우 다릅니다. 다음 두 가지 예가 이를 보여줍니다.

  • 초당 1프레임을 그리는 스프라이트가 있고 이미지 속도를 0.5로 설정하면 초당 0.5프레임으로 그려집니다. 게임 프레임 속도가 초당 60프레임이면 스프라이트는 120게임 프레임마다 1프레임을 그립니다.
  • 게임 프레임당 1프레임을 그리는 스프라이트가 있고 이미지 속도를 0.5로 설정하면 게임 프레임당 0.5프레임을 그립니다. 게임 프레임 속도가 초당 60프레임이면 스프라이트는 60게임 프레임마다 30프레임을 그립니다.

함수를 사용하여 스프라이트를 정의할 때 사용된 애니메이션 속도 유형을 알 수 있습니다.sprite_get_speed_type(), 그리고 당신은 기능을 사용하여 애니메이션 속도 와 유형을 설정할 수 있습니다sprite_set_speed().

통사론:

sprite_get_speed(index)

논쟁 유형 설명
index The index of the sprite to get the speed of

보고:

예시:

s_speed = sprite_get_speed(sprite_index);
s_type = sprite_get_speed_type(sprite_index);

위의 코드는 스프라이트 속도와 스프라이트 애니메이션 유형을 가져와 나중에 사용할 수 있도록 인스턴스 변수에 저장합니다.
