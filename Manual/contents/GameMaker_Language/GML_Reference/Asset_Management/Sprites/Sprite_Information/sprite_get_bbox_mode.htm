sprite_get_bbox_mode

이 함수는 경계 상자 계산을 위한 현재 "모드"를 반환합니다. 확인할 스프라이트의 스프라이트 인덱스를 제공하면 함수는 아래 표시된 상수 중 하나를 반환합니다.

통사론:

sprite_get_bbox_mode(ind);

논쟁 유형 설명
ind The index of the sprite to check.

보고

Bounding Box Mode Constant

일정한 설명
bboxmode_automatic 자동 - 경계 상자는 스프라이트의 허용 오차 설정에 따라 자동으로 계산됩니다.
bboxmode_fullimage 전체 이미지 - 허용 오차 및 "빈" 픽셀에 관계없이 경계 상자가 스프라이트의 전체 너비와 높이를 사용하도록 설정됩니다.
bboxmode_manual 수동 - 경계 상자가 수동으로 사용자 정의 값으로 설정되었습니다(스프라이트 편집기에서 또는 함수를 사용하여sprite_set_bbox() )

예시:

if sprite_get_bbox_mode(sprite_index) != 0
{
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

위는 현재 스프라이트에 대한 bbox 모드를 확인하고 그렇지 않은 경우 automatic, 그 값으로 설정합니다.
