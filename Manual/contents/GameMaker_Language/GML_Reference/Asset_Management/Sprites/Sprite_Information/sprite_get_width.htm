sprite_get_width

이 함수를 사용하면 변환이 없는 기본 리소스 스프라이트의 너비를 픽셀 단위로 찾을 수 있습니다.

통사론:

sprite_get_width(index);

논쟁 유형 설명
index The index of the sprite to find the width of.

보고

예시:

if sprite_width != sprite_get_width(sprite_index)
{
    image_xscale = 1;
}

위의 코드는 현재 인스턴스에서 스프라이트의 너비를 확인하고 원래 기본 스프라이트와 차이가 있으면 x축 스케일을 재설정합니다.
