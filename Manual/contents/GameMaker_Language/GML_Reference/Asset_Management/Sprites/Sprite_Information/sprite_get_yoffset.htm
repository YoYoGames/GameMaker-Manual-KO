sprite_get_yoffset

스프라이트 편집기에서 스프라이트를 정의할 때 해당 스프라이트의 원점 을 설정해야 합니다. 이것은 인스턴스와 함께 사용될 때 스프라이트가 "연결"되거나 "그려지는" 지점입니다. 이 함수는 왼쪽 위 모서리가 0,0 위치인 것을 기준으로 원점의 y축에 대한 상대 오프셋을 반환합니다. 여기서 +y는 아래로, a -y는 위로 이동합니다. 다음 이미지는 이를 보여줍니다.

통사론:

sprite_get_yoffset(index);

논쟁 유형 설명
index The index of the sprite to find the yoffset of.

보고

예시:

if y - sprite_get_yoffset(sprite_index) < 0
{
    y = sprite_get_yoffset(sprite_index);
}

위의 코드는 인스턴스가 스프라이트를 기반으로 룸 내에서 유지되도록 합니다(그래서 스프라이트가 항상 표시됨).
