sprite_get_tpe

이 함수는 게임에서 사용하는 텍스처 페이지에서 주어진 스프라이트 및 하위 이미지의 위치와 관련된 특수 값을 반환합니다. 이런 식으로 컴파일된 게임 외부에서 사용하기 위해 확장에 값을 전달할 수 있으며, GameMaker 그리기 기능을 효과적으로 우회하고 게임 캔버스가 포함된 창 내 어디에서나 그릴 수 있는 DOM 콘텐츠에서 스프라이트를 사용할 수 있습니다. 이 기능은 이 기능을 사용하여 호스트 페이지의 GameMaker 캔버스 요소 외부에 버튼 및 기타 대화형 미디어를 생성하려는 사용자에게 특히 유용합니다.clickable_add().

참고: 이 기능은 HTML5 전용입니다.

통사론:

sprite_get_tpe(sprite, index);

논쟁 유형 설명
sprite 스프라이트 자산 The index of the sprite to find the texture page entry of.
index 진짜 The image index of the sprite.

보고

Texture Page Entry

예시:

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

위의 코드는 게임 캔버스가 실행되고 있는 페이지의 (32, 32) 위치에 DOM 버튼을 생성합니다. 버튼은 텍스처 페이지에서 "spr_MS_Home"으로 참조된 스프라이트를 사용하며 버튼을 클릭하면 지정된 URL에 대한 새 창이 열리고 해당 창에 대해 정의된 속성이 포함됩니다.
