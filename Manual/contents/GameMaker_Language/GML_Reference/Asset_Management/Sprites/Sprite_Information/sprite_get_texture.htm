sprite_get_texture

이 함수는 스프라이트 텍스처 페이지에 대한 특수 포인터 를 반환합니다. 이 값은 다른 그리기 기능, 특히 2D 기본 기능과 셰이더 기능에서 사용할 수 있습니다. 다음을 사용하여 반환된 텍스처 페이지에 대한 자세한 정보를 얻을 수 있습니다. texture_ 여기 에서 찾을 수 있는 기능.

통사론:

sprite_get_texture(spr, subimg);

논쟁 유형 설명
spr 스프라이트 자산 The index of the sprite to use.
subimg 진짜 The sub-image of the sprite to use.

보고

Texture

예시:

var tex;
tex = sprite_get_texture(spr_Wall, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(480, 0, 1, 0);
draw_vertex_texture(480, 640, 1, 1);
draw_vertex_texture(0, 640, 0, 1);
draw_primitive_end();

위의 코드는 "tex" 변수에 있는 텍스처로 텍스처링된 4개의 꼭짓점 삼각형 스트립을 그립니다.
