sprite_get_nineslice

이 함수는 스프라이트 편집기 에서 설정하거나 sprite_set_nineslice() 를 사용하여 모든 Nine Slice 속성을 포함하는 스프라이트에 대한 Nine Slice 구조체 를 반환합니다. 이 구조체의 내용은 이 페이지 에 자세히 설명되어 있습니다.

제공된 스프라이트에 할당된 Nine Slice 구조체가 없으면 기본 Nine Slice 속성이 있는 새 구조체가 스프라이트에 대해 생성되고 반환됩니다.

이 구조체의 값을 변경하면 원래 스프라이트의 Nine Slice 속성 이 수정되어 해당 스프라이트로 수행되는 향후 그리기 호출에 영향을 줍니다.

통사론:

sprite_get_nineslice(ind);

논쟁 유형 설명
ind The index of the sprite from which the Nine Slice struct will be retrieved

보고:

Nine Slice Struct (or -1 if the sprite doesn't exist)

예시:

var _box_nineslice = sprite_get_nineslice(spr_box_0);

_box_nineslice.enabled = true;
_box_nineslice.left = 10;
_box_nineslice.right = 10;
_box_nineslice.top = 10;
_box_nineslice.bottom = 10;

위의 코드는 스프라이트에서 Nine Slice 구조체를 검색하고 Nine Slice를 활성화하고 가이드 오프셋을 설정합니다.
