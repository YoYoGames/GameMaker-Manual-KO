skeleton_find_slot

이 함수를 사용하면 현재 인스턴스와 연결된 Spine 스프라이트에서 지정된 공간 공간 위치에 있는 슬롯을 찾을 수 있습니다. DS 목록 을 만들고 확인할 x/y 위치와 함께 해당 ID를 제공하면 목록이 스프라이트에 사용할 수 있는 각 연결 슬롯에 대한 이름 문자열로 채워집니다(연결 수정 포함). 목록은 항상 최상위 슬롯부터 시작하여 내림차순으로 정렬됩니다.

통사론:

skeleton_find_slot(x, y, list);

논쟁 유형 설명
x 진짜 The x position in the room to check.
y 진짜 The y position in the room to check.
list DS 목록 ID The ID of the DS list to populate with the DS maps.

보고:

N/A

예시:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);

위의 코드는 DS 목록을 만든 다음 방에서 마우스 위치에 있는 슬롯 이름으로 DS 목록을 채웁니다.
