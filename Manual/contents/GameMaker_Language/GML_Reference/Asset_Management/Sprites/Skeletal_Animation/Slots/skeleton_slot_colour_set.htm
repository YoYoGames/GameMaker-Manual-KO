skeleton_slot_colour_set

이 기능을 사용하면 부착 슬롯 색상과 알파를 설정하여 부착된 모든 스프라이트가 이러한 블렌드 값으로 그려지도록 할 수 있습니다. 인스턴스 스프라이트에는 블렌드 색상과 알파 설정이 있을 수 있습니다(image_blend 그리고image_alpha ), 첨부 파일과 마찬가지로(함수 참조)skeleton_attachment_create_colour() ), 그리고 슬롯에 할당된 어태치먼트 스프라이트가 가질 최종 색상과 알파는 이 모든 값의 합성이 됩니다.

통사론:

skeleton_slot_colour_set(slot, colour, alpha);

논쟁 유형 설명
slot 끈 The slot name to set, a string
colour 색깔 The colour to set, either an integer, a constant, or a hex value.
alpha 진짜 The alpha to set from 0 to 1.

보고:

N/A

예시:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if (!ds_list_empty(slot_list))
{
    for (var i = 0; i < ds_list_size(slot_list); ++i)
    {
        if skeleton_slot_colour_get(slot_list[| i]) != c_white
        {
            skeleton_slot_colour_set(slot_list[| i], c_white, 1);
        }
    }
}

위의 코드는 DS 목록을 만든 다음 방에서 마우스 위치에 있는 슬롯 이름으로 DS 목록을 채웁니다. 그런 다음 슬롯 목록을 반복하고 색상이 이미 흰색이 아닌 경우 발견된 슬롯의 색상을 알파 1과 함께 흰색으로 재설정합니다.
