skeleton_slot_alpha_get

이 기능을 사용하면 부착 슬롯 알파 값을 얻을 수 있습니다. 슬롯 이름(문자열)을 제공하면 함수는 0과 1 사이의 알파에 대한 실제 값을 반환합니다.

통사론:

skeleton_slot_alpha_get(slot);

논쟁 유형 설명
slot 끈 The slot name to check, a string

보고:

Real

예시:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if (!ds_list_empty(slot_list))
{
    for (var i = 0; i < ds_list_size(slot_list); ++i)
    {
        if skeleton_slot_alpha_get(slot_list[| i]) != 1
        {
            skeleton_slot_colour_set(slot_list[| i], c_white, 1);
        }
    }
}

위의 코드는 DS 목록을 만든 다음 방에서 마우스 위치에 있는 슬롯 이름으로 DS 목록을 채웁니다. 그런 다음 슬롯 목록을 반복하고 알파가 이미 1이 아닌 경우 발견된 슬롯의 색상을 알파 1과 함께 흰색으로 재설정합니다.
