skeleton_slot_list

이 기능을 사용하면 골격 애니메이션 스프라이트의 일부로 생성된 슬롯의 모든 이름으로 (미리 생성된) DS 목록을 채울 수 있습니다. 이름은 문자열이며 이러한 유형의 스프라이트에 대한 다른 스켈레톤 애니메이션 슬롯 함수에서 사용할 수 있습니다.

통사론:

skeleton_slot_list(sprite, list);

논쟁 유형 설명
sprite 스프라이트 자산 The sprite index of the Spine skeletal animation to get the list from.
list DS 목록 ID The ID of the DS list to populate with the bone names.

보고:

N/A

예시:

bone_list = ds_list_create();
skeleton_slot_list(sprite_index, bone_list);

위의 코드는 DS 목록을 만든 다음 나중에 사용할 수 있도록 슬롯 이름으로 채웁니다.
