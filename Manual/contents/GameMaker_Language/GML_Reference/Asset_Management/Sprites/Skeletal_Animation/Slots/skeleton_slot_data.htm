skeleton_slot_data

이 기능을 사용하면 애니메이션에 포함된 사용 가능한 각 연결 슬롯에 대한 데이터로 (미리 생성된) DS 목록 을 채울 수 있습니다. 이 데이터는 다음 키/값 쌍을 포함하는 DS 맵 의 포럼에서 반환됩니다.

  • "이름": 슬롯의 이름입니다.
  • "뼈": 뼈 의 이름입니다.
  • "attachment": 첨부 파일이 사용되는 경우 첨부 파일의 이름이고 할당되지 않은 경우 "(none)"입니다.

각 키의 값은 문자열이며 이러한 유형의 스프라이트에 대한 다른 스켈레톤 부착 기능에서 사용할 수 있습니다. 생성된 DS 맵은 파괴 되지 않으므로 DS 목록을 반복하고 생성된 각 맵을 직접 파괴해야 합니다.

통사론:

skeleton_slot_data(sprite, list);

논쟁 유형 설명
sprite 스프라이트 자산 The sprite index of the Spine skeletal animation to get the data from.
list DS 목록 ID The ID of the DS list to populate with the DS maps.

보고:

N/A

예시:

var list = ds_list_create();
var open = true;
slot_name = "";
skeleton_slot_data(sprite_index, list);

for (var i = 0; i < ds_list_size(list); i++)
{
    var map = list[| i];
    if open
    {
        if map[? "attachment"] == "(none)"
        {
            open = false;
            slot_name = map[? "name"];
        }
    }

    ds_map_destroy(map);
}

ds_list_destroy(list);

위의 코드는 DS 목록을 만든 다음 인스턴스 스프라이트에 대한 슬롯 데이터로 채웁니다. 그런 다음 이 데이터를 구문 분석하여 슬롯 데이터가 있는 개별 DS 맵을 추출합니다. 그런 다음 빈 슬롯이 있는지 확인하고, 그렇다면 DS 맵이 파괴되기 전에 변수 "slot_name"에 빈 슬롯 이름이 할당됩니다. 마지막으로 더 이상 필요하지 않은 DS 목록을 삭제합니다.
