skeleton_attachment_set

골격 애니메이션 스프라이트에는 다른 이미지가 첨부 파일로 추가될 수 있으며 이러한 이미지는 명명된 슬롯에 추가되며(애니메이션 프로그램에서 첨부 슬롯을 만들 때 이름이 지정됨) 현재 스프라이트의 애니메이션과 함께 그려집니다.. 이 기능을 사용하면 지정된 슬롯에 부착물을 설정할 수 있으며, 여기서 슬롯 및 부착물의 이름(문자열으로)을 제공해야 합니다. 이러한 이름은 사용된 애니메이션 프로그램에 의해 정의되거나(첨부 파일의 경우)skeleton_attachment_create().

합격할 수도 있으니 참고하세요 sprite_index in을 어태치먼트로 사용하고 해당 스프라이트를 사용하거나 -1을 사용하여 슬롯에서 어태치먼트를 제거할 수 있습니다. 스프라이트 인덱스를 인수로 전달하면 스프라이트 이름을 슬롯 이름을 지정하는 문자열로 사용하여 연결 슬롯이 생성됩니다(따라서 spr_sword 예를 들어, 첨부 슬롯 "spr_sword"를 생성하고 슬롯은 스프라이트의 첫 번째 이미지 인덱스(0)와 스케일(1,1)의 x 및 y 원점 오프셋을 사용합니다. 그리고 0의 회전.

통사론:

skeleton_attachment_set(slot, attachment);

논쟁 유형 설명
slot 끈 The slot name (a string) to get the attachment of.
attachment 문자열 또는 스프라이트 자산 The name (as a string or a sprite_index) of the attachment image.

보고:

String

예시:

if skeleton_attachment_get("slot_leftHand") == ""
{
    skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
}

위의 코드는 "slot_leftHand"라는 슬롯에 대해 현재 할당된 부착 이름을 확인하고 빈 문자열이 반환되면 새 스프라이트가 부착됩니다.
