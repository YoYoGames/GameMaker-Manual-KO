skeleton_attachment_get

스켈레탈 애니메이션 스프라이트에는 다른 스프라이트가 첨부 파일로 추가될 수 있으며 이러한 스프라이트는 명명된 슬롯에 추가되며(애니메이션 프로그램에서 첨부 슬롯을 만들 때 이름이 지정됨) 현재 스프라이트의 애니메이션과 함께 그려집니다.. 이 함수를 사용하면 현재 할당된 스프라이트의 주어진 슬롯에 대한 부착물의 이름(문자열으로)을 얻을 수 있습니다. 연결된 스프라이트는 Spine에 할당된 이름 문자열 을 통해 참조되거나skeleton_attachment_create().

통사론:

skeleton_attachment_get(slot);

논쟁 유형 설명
slot 끈 The slot name (a string) to get the attachment of.

보고:

String

예시:

if skeleton_attachment_get("slot_leftHand") == ""
{
    skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
}

위의 코드는 "slot_leftHand"라는 슬롯에 대해 현재 할당된 부착 이름을 확인하고 빈 문자열이 반환되면 새 스프라이트가 부착됩니다.
