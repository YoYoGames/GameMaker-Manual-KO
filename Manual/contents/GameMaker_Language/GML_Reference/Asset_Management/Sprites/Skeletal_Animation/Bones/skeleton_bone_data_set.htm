skeleton_bone_data_set

뼈대 애니메이션은 애니메이션 프로그램에서 정의하고 이름을 지정한 여러 "뼈"로 구성되며 이 기능을 사용하여 언제든지 명명된 뼈에 대한 특정 데이터를 설정할 수 있습니다. 이 데이터는 현재 그려지고 있는 포즈가 아니라 스켈레톤의 기본 포즈를 참조한다는 점에 유의하십시오.skeleton_bone_state_set() ), 이전에 만든 DS 맵 에서 설정해야 하며 다음 키와 해당 값이 있어야 합니다.

  • "x": 상위 뼈대를 기준으로 한 뼈대의 로컬 x 위치입니다.
  • "y": 상위 뼈대를 기준으로 한 뼈대의 로컬 y 위치입니다.
  • "각도": 상위 뼈에 대한 뼈의 로컬 회전입니다.
  • "xscale": 뼈의 로컬 수평 스케일입니다.
  • "yscale": 뼈의 로컬 수직 스케일입니다.
  • "parent": 부모 뼈의 이름(문자열)입니다.

통사론:

skeleton_bone_data_set(bone, map);

논쟁 유형 설명
bone 끈 The name (as a string) of the bone.
map DS 맵 ID The (previously created) DS map that stores the bone data.

보고:

N/A

예시:

var bone_map = ds_map_create();
skeleton_bone_data_get("head", bone_map);
ds_map_replace(bone_map, "parent", "body");
skeleton_bone_data_set("head", bone_map);
ds_map_destroy(bone_map);

위의 코드는 DS 맵을 만든 다음 "head"라는 뼈대의 데이터로 채웁니다. 그런 다음 맵의 "부모" 뼈대 키를 새 값으로 바꾸고 "머리" 뼈대를 새 데이터 세트로 다시 설정합니다.
