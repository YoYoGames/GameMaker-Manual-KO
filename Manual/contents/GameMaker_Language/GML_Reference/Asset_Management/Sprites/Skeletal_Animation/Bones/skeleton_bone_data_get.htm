skeleton_bone_data_get

뼈대 애니메이션은 애니메이션 프로그램에서 정의하고 이름을 지정한 많은 "뼈"로 구성됩니다. 이 기능을 사용하면 언제든지 명명된 뼈에 대한 정보를 얻을 수 있고 이전에 생성했어야 하는 빈 DS 맵 에 데이터를 저장할 수 있습니다. 이 데이터는 현재 그려지고 있는 포즈가 아니라 스켈레톤의 기본 포즈를 나타냅니다. 현재 포즈의 데이터가 필요한 경우 다음을 사용하십시오.skeleton_bone_state_get().

이 함수를 호출하면 지도가 다음 키로 채워집니다.

  • "x": 상위 뼈대를 기준으로 한 뼈대의 로컬 x 위치입니다.
  • "y": 상위 뼈대를 기준으로 한 뼈대의 로컬 y 위치입니다.
  • "각도": 상위 뼈에 대한 뼈의 로컬 회전입니다.
  • "xscale": 상위 뼈를 기준으로 뼈의 로컬 수평 스케일입니다.
  • "yscale": 부모 뼈를 기준으로 뼈의 로컬 수직 스케일입니다.
  • "parent": 부모 뼈의 이름(문자열)입니다.

통사론:

skeleton_bone_data_get(bone, map);

논쟁 유형 설명
bone 끈 The name (as a string) of the bone.
map DS 맵 ID The (previously created) DS map that will get the bone data.

보고:

N/A

예시:

bone_map = ds_map_create();
skeleton_bone_data_get("leftarm", bone_map);

위의 코드는 DS 맵을 만든 다음 "leftarm"이라는 뼈대의 데이터로 채웁니다.
