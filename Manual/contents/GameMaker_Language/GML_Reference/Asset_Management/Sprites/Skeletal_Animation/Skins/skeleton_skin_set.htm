skeleton_skin_set

스켈레탈 애니메이션 스프라이트는 다양한 텍스처("스킨"이라고 함)로 그릴 수 있으므로 하나의 애니메이션을 여러 목적으로 사용할 수 있습니다. 예를 들어, 일반적인 걷기 애니메이션이 있을 수 있으며 이것은 단순히 그려진 스킨을 변경하는 것만으로 게임에서 NPC 캐릭터, 적 등이 사용할 수 있습니다. 이러한 스킨은 애니메이션 프로그램에서 이전에 생성되고 이름이 지정되어야 하며, 이 이름은 그릴 때 해당 스킨을 사용하도록 스프라이트를 설정하는 함수에 (문자열로) 전달됩니다.

통사론:

skeleton_skin_set(skinname);

논쟁 유형 설명
skinname The name (a string) of the skin to use.

보고:

예시:

if skeleton_skin_get() == "skin_Enemy1"
{
    skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
}

위의 코드는 애니메이션에 현재 할당된 스킨을 확인하고 "skin_Enemy1"이면 세 가지 옵션 중 하나에서 새 스킨을 선택하여 설정합니다.
