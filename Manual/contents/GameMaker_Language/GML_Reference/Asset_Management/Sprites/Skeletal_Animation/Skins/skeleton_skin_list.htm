skeleton_skin_list

이 기능을 사용하면 골격 애니메이션 스프라이트의 일부로 포함된 모든 스킨 이름으로 (미리 생성된) DS 목록 을 채울 수 있습니다. 이름은 문자열이 되며 이러한 유형의 스프라이트에 대한 다른 스켈레톤 애니메이션 스킨 기능에서 사용할 수 있습니다.

통사론:

skeleton_skin_list(sprite, list);

논쟁 유형 설명
sprite 스프라이트 자산 The sprite index of the Spine skeletal animation to get the list from.
list DS 목록 ID The ID of the DS list to populate with the animation names.

보고:

N/A

예시:

var list = ds_list_create();
skeleton_skin_list(sprite_index, list);
var num = ds_list_size(list);
skeleton_skin_set(list[| irandom(num - 1));
ds_list_destroy(list);

위의 코드는 DS 목록을 만든 다음 스킨 이름으로 채웁니다. 그런 다음 목록이 삭제되기 전에 임의의 항목이 선택되어 인스턴스에 적용됩니다.
