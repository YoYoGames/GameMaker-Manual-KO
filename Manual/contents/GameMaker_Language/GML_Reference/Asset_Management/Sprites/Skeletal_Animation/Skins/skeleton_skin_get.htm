skeleton_skin_get

스켈레탈 애니메이션 스프라이트를 사용하면 애니메이션에 별도의 텍스처("스킨"이라고 함)를 할당할 수 있으므로 하나의 애니메이션을 여러 가지에 사용할 수 있습니다. 이 함수는 인스턴스가 사용 중인 스켈레탈 애니메이션 스프라이트에 현재 할당된 스킨 이름(문자열으로)을 반환합니다. 반환된 이름은 애니메이션 프로그램에서 스프라이트를 만들 때 설정한 이름입니다.

통사론:

skeleton_skin_get();

보고:

예시:

if skeleton_skin_get() == "skin_Enemy1"
{
    skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
}

위의 코드는 애니메이션에 현재 할당된 스킨을 확인하고 "skin_Enemy1"이면 세 가지 옵션 중 하나에서 새 스킨을 선택하여 설정합니다.
