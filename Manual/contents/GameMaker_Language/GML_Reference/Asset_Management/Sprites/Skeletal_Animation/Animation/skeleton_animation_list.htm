skeleton_animation_list

이 기능을 사용하면 골격 애니메이션 스프라이트의 일부로 포함된 애니메이션의 모든 이름으로 (미리 생성된) DS 목록 을 채울 수 있습니다. 이름은 문자열이며 이러한 유형의 스프라이트에 대한 다른 애니메이션 기능에서 사용할 수 있습니다.

통사론:

skeleton_animation_list(sprite, list);

논쟁 유형 설명
sprite 스프라이트 자산 The sprite index of the Spine skeletal animation to get the list from.
list DS 목록 ID The ID of the DS list to populate with the animation names.

보고:

N/A

예시:

var list = ds_list_create();
skeleton_animation_list(sprite_index, list);

for (var i = 1; i < ds_list_size(list); i++;)
{
    skeleton_animation_mix(list[| 0], list[| i], 0.5);
}

ds_list_destroy(list);

위의 코드는 인스턴스에서 사용 중인 스프라이트에 대한 모든 애니메이션 이름의 DS 목록을 만듭니다. 그런 다음 이를 반복하고 첫 번째 애니메이션이 있는 모든 항목의 믹스 값을 0.5로 설정합니다.
