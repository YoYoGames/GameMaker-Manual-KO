skeleton_animation_is_finished

이 기능을 사용하면 현재 반복되지 않는 스켈레탈 애니메이션이 주어진 트랙에서 재생을 완료했는지 확인할 수 있습니다.

통사론:

skeleton_animation_is_finished(track);

논쟁 데이터 형식 설명
길 진짜 애니메이션 트랙의 인덱스

보고:

Boolean

예시:

var _finished = skeleton_animation_is_finished(1);
if _finished
{
    show_debug_message("The animation has finished playing");
}

위의 코드는 먼저 트랙 1의 애니메이션 재생이 완료되었는지 확인하고 결과를 로컬 변수에 저장합니다. _finished. 그런 다음 애니메이션 재생이 완료된 경우 디버그 메시지를 표시합니다.
