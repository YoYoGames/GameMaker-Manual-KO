skeleton_animation_get_ext

단일 스켈레탈 애니메이션 스프라이트에는 다양한 애니메이션 세트가 있을 수 있으며, 이러한 세트에 다른 트랙을 할당하여 애니메이션 세트를 "믹스 앤 매치"할 수 있습니다. 이 함수는 주어진 트랙 번호에 의해 현재 사용되는 애니메이션 세트의 이름을 반환합니다(함수에 의해 설정된 대로skeleton_animation_set_ext ).

통사론:

skeleton_animation_get_ext(track);

논쟁 유형 설명
track 진짜 The track number to get the animation set name of.

보고:

String

예시:

if skeleton_animation_get_ext(1) != "bodyfight"
{
    skeleton_animation_set_ext("bodyfight", 1);
}

위의 코드는 트랙 1에서 사용 중인 애니메이션 세트를 "bodyfight"로 변경하지 않은 경우 이를 변경합니다.
