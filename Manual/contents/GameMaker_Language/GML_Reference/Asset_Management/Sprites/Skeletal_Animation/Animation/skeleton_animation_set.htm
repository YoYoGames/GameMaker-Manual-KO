skeleton_animation_set

인스턴스에 스켈레탈 애니메이션 스프라이트를 할당하고 나면 이 함수로 주어진 시간에 어떤 애니메이션 세트 를 사용해야 하는지 정의할 수 있습니다. (Spine에서) 스프라이트를 만들 때 다른 상태에 대해 여러 애니메이션 세트를 정의하고 각각의 이름을 적절하게 지정했습니다. 이러한 이름은 게임에서 언제든지 사용할 애니메이션 세트를 GameMaker 에 알려주는 데 사용할 것입니다. 예를 들어, 스프라이트가 플랫폼 게임용이라면 "점프" 세트, "실행" 세트 및 "유휴" 세트가 모두 하나의 스프라이트에 포함되어 있을 수 있으며 이를 사용하여 게임에서 둘 사이를 전환할 수 있습니다. 기능.
애니메이션은 기본적으로 반복 재생됩니다.

이 방법으로 새 애니메이션을 설정하면image_index 0으로 설정하면 애니메이션 루프의 시작 부분에서 새 애니메이션 세트가 시작됩니다. 이것이 원하는 동작이 아닌 경우 다음을 사용하여 "혼합" 값을 설정할 수 있습니다.skeleton_animation_mix() 새 애니메이션을 설정하면 이전 세트와 부드럽게 보간됩니다. 또한 별도의 애니메이션 트랙을 사용할 계획이 없는 한 모든 애니메이션의 시작 부분에 키프레임이 있어야 이전 애니메이션의 데이터가 보류되는 것을 방지할 수 있습니다.

통사론:

skeleton_animation_set(animname, [loop]);

논쟁 유형 설명
animname 끈 The name (a string) of the animation set to use.
loop 부울 Whether the animation should loop or not (default istrue> )

보고:

N/A

예시:

if keyboard_check_pressed(vk_space)
{
    if (skeleton_animation_get() != "jump")
    {
        skeleton_animation_set("jump");
    }
}

위의 코드는 사용 중인 애니메이션 세트를 스페이스 키를 눌렀을 때 "점프" 세트로 변경하지만 현재 사용 중인 세트가 아직 "점프"가 아닌 경우에만 변경됩니다.
