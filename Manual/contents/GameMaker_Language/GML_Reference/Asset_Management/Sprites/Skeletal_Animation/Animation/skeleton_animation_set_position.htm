skeleton_animation_set_position

이 기능은 주어진 트랙에 할당된 애니메이션을 사용자가 제공한 위치로 설정합니다. 위치는 0과 1 사이의 실제 값이어야 합니다. 이 범위를 벗어난 값을 제공하면 트랙 위치가 해당 값의 소수 부분으로 설정됩니다. 값 0.4, 값 1.4, 값 2.4,...를 사용한다고 가정하면 모두 위치가 0.4로 설정됩니다.

통사론:

skeleton_animation_set_position(track, position);

논쟁 유형 설명
track 진짜 The animation track to set the frame of.
position 진짜 The new position for the animation frame.

보고:

N/A

예시:

skeleton_animation_set_position(1, 0.25);

위의 코드는 스켈레톤 애니메이션의 트랙 1에 대한 위치를 0.25 또는 다르게 말하면 애니메이션 시작에서 25%로 설정합니다.
