skeleton_animation_get_frames

이 함수는 주어진 스켈레톤 애니메이션에 있는 프레임 수를 검색하는 데 사용할 수 있습니다. 스켈레톤 애니메이션 이름을 제공합니다(애니메이션을 만드는 데 사용된 프로그램에 정의된 문자열 또는 함수를 사용하여 반환된 문자열) skeleton_animation_get() ) 함수는 정수 값으로 포함된 프레임을 반환합니다. 지정된 애니메이션이 존재하지 않으면 함수는 0을 반환합니다.

통사론:

skeleton_animation_get_frames(anim_name);

논쟁 유형 설명
anim_name 끈 The animation name to get the frames of.

보고:

Real

예시:

var num = skeleton_animation_get_frames(skeleton_animation_get());
image_index = num -1;
image_speed = 0;

위의 코드는 애니메이션의 프레임 수를 가져온 다음 스프라이트를 마지막 프레임으로 설정하고 애니메이션을 중지합니다.
