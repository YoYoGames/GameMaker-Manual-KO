skeleton_animation_get

이 함수를 사용하면 스켈레탈 애니메이션 스프라이트에서 사용 중인 현재 애니메이션 세트를 가져올 수 있습니다. 반환 값은 스켈레탈 애니메이션 프로그램에서 정의한 대로 세트의 이름이 되는 문자열입니다.

통사론:

skeleton_animation_get();

보고:

예시:

if keyboard_check_pressed(vk_space)
{
    if skeleton_animation_get() != "jump"
    {
        skeleton_animation_set("jump");
    }
}

위의 코드는 사용 중인 애니메이션 세트를 스페이스 키를 눌렀을 때 "점프" 세트로 변경하지만 현재 사용 중인 세트가 아직 "점프"가 아닌 경우에만 변경됩니다.
