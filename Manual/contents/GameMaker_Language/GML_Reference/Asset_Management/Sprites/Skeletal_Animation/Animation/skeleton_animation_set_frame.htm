skeleton_animation_set_frame

이 기능은 주어진 트랙에 할당된 애니메이션을 사용자가 제공한 프레임으로 설정합니다. 프레임 인덱스는 0과 애니메이션의 프레임 수(함수를 사용하여 얻을 수 있음) 사이의 실제 값이어야 합니다.skeleton_animation_get_frames ). 이 범위를 벗어나는 값을 제공하면 애니메이션이 반복되어 차이를 보완합니다. 예를 들어 애니메이션에 5개의 프레임이 있고 프레임을 8로 설정하면 표시되는 실제 프레임은 3이 됩니다.

통사론:

skeleton_animation_set_frame(track, index);

논쟁 유형 설명
track 진짜 The animation track to set the frame of.
index 진짜 The new index for the animation frame.

보고:

N/A

예시:

if global.pause
{
    skeleton_animation_set_frame(0, -image_speed);
}

위의 코드는 전역 "pause" 변수가 다음과 같을 때 트랙 0의 애니메이션에 대한 프레임 인덱스를 음의 이미지 속도로 설정합니다. true (본질적으로 애니메이션을 일시 중지).
