skeleton_animation_clear

이 기능은 다시 할당할 준비가 된 모든 애니메이션의 지정된 애니메이션 트랙을 지웁니다.

통사론:

skeleton_animation_clear(track);

논쟁 유형 설명
track 진짜 The animation track to clear.

보고:

N/A

예시:

if mouse_check_button(mb_right)
{
    skeleton_animation_clear(1);
}

위의 코드는 마우스 오른쪽 버튼을 누르면 애니메이션 트랙 1을 지웁니다.
