skeleton_animation_get_event_frames

이 함수는 주어진 애니메이션에서 주어진 이벤트에 대한 모든 프레임을 검색하는 데 사용할 수 있습니다. 스켈레톤 애니메이션 이름(애니메이션을 만드는 데 사용된 프로그램에 정의된 문자열 또는 함수를 사용하여 반환된 문자열)을 제공합니다. skeleton_animation_get() ) 및 이벤트 이름. 이 함수는 해당 이벤트에 대한 프레임이 있는 배열을 반환합니다.

지정된 이벤트 이름이 존재하지 않는 경우 함수는 첫 번째(유일한) 요소로 -1 을 포함하는 배열을 반환합니다.

통사론:

skeleton_animation_get_event_frames(anim_name, event_name);

논쟁 유형 설명
anim_name 끈 The animation name to get the frames of.
event_name 끈 The event name to get the frames for.

보고:

Array of Reals

예시:

var _frames = skeleton_animation_get_event_frames(skeleton_animation_get(), "Footstep");

if (_frames[0] != -1)
{
    var _count = array_length(_frames);
    var _current_frame = skeleton_animation_get_frame(0);

    for (var i = 0; i < _count; i ++)
    {
        if (_frames[i] == _current_frame)
        {
            audio_play_sound(snd_footstep, 1, false);
            break;
        }
    }
}

위의 코드는 "Footstep" 이벤트에 대한 프레임을 가져옵니다. 유효한 프레임이 있는 배열을 반환한 경우 반복하여 현재 프레임이 배열의 프레임 중 하나와 같은지 확인합니다. 이 경우 발자국 사운드 효과를 재생하고 루프를 중단(중지)합니다.
