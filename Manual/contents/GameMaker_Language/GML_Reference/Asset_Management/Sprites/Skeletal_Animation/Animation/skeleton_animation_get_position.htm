skeleton_animation_get_position

이 함수는 지정된 애니메이션 트랙의 애니메이션 위치를 0에서 1 사이의 값으로 반환합니다. 이렇게 하면 값 0이 0%에 해당하고 값 1에서 100%에 해당하는 백분율로 트랙 위치를 얻을 수 있습니다.

지정된 특정 트랙에 애니메이션이 할당되지 않았거나 인스턴스에 스프라이트 세트가 없으면 -1을 반환합니다.

통사론:

skeleton_animation_get_position(track);

논쟁 유형 설명
track 진짜 The animation track to get the position of.

보고:

Real

예시:

var _position = skeleton_animation_get_position(0);
var _position_percent = _position * 100;
show_debug_message("The skeleton animation position on track 0 is at " + string(_position_percent) + "%.");

위의 코드는 트랙 0에 할당된 스켈레톤 애니메이션의 현재 위치를 확인하고 임시 변수 _position 에 저장합니다. 그런 다음 100을 곱하여 해당 값을 백분율로 변환하고 그 결과를 다른 임시 변수 _position_percent 에 저장합니다.
마지막으로 백분율 값을 표시하는 디버그 메시지가 표시됩니다.
