skeleton_animation_set_ext

스켈레탈 애니메이션 스프라이트에 대한 애니메이션 세트를 설정하면 내부적으로 트랙 0 이 할당됩니다. 그러나 추가 트랙(1 이상)에 추가 세트를 할당할 수 있으며 이러한 방식으로 스프라이트가 동시에 두 개(또는 그 이상) 애니메이션을 재생하도록 할 수 있습니다. 이 점을 염두에 두고 애니메이션 프로그램에서 스프라이트를 디자인해야 합니다. 즉, 트랙 0의 "기본" 애니메이션에는 특정 뼈대에 대한 키프레임만 있고 다음 트랙에는 다른 뼈대에 대한 키프레임이 있습니다. 예를 들어 "걷기" 애니메이션이 있고 해당 키프레임이 스프라이트의 다리에만 영향을 미치는 경우 트랙 1과 2에서 각각 설정할 수 있는 머리 애니메이션 및 팔 애니메이션 세트를 가질 수 있으므로 다음을 수행할 수 있습니다. 게임이 진행되는 동안 실시간으로 기본 움직임에 적응합니다.
애니메이션은 기본적으로 반복 재생됩니다.

통사론:

skeleton_animation_set_ext(animname, track, [loop]);

논쟁 유형 설명
animname 끈 The name (a string) of the animation set to use.
track 진짜 The track for the animation set to use.
loop 부울 Whether the animation should loop or not (default istrue> )

보고:

N/A

예시:

if mouse_check_button(mb_left)
{
    skeleton_animation_set_ext("bodyfight", 1);
}
else
{
    skeleton_animation_set_ext("bodywalk", 1);
}

위의 코드는 트랙 1에서 사용 중인 애니메이션 세트를 마우스 왼쪽 버튼을 눌렀을 때 "bodyfight" 세트 또는 "bodywalk" 세트로 변경합니다.
