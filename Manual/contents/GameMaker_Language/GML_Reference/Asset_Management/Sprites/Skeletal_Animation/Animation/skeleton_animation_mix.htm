skeleton_animation_mix

다음을 사용하여 애니메이션 세트를 쉽게 전환할 수 있습니다.skeleton_animation_set() 하지만 이로 인해 한 애니메이션이 다른 애니메이션으로 교체될 때 건너뛰기 또는 끊김 현상이 발생할 수 있습니다. 이를 방지하기 위해 두 애니메이션 세트 사이의 믹스 값을 설정할 수 있으며 스프라이트는 이들 사이를 보간합니다. 일반적으로 한 번만 설정하면 되는 골격 애니메이션이 있는 인스턴스의 이벤트 만들기에서 이 작업을 수행하고 GameMaker 는 해당 인스턴스의 애니메이션 세트를 사용하여 스프라이트에 대한 모든 추가 변경 사항을 보간합니다. 지속 시간 값은 0에서 1 사이이며 0.5의 값은 한 세트에서 다른 세트로 "절반 및 반" 보간을 가집니다.

참고 Spine 4.0은 더 이상 가능한 최단 경로를 사용하여 회전을 보간하지 않고 보간을 위해 키프레임의 절대값을 사용하므로 회전 애니메이션의 작동 방식을 변경합니다. 이전 버전에서 Spine 애니메이션을 업그레이드하는 경우 예기치 않은 동작이 발생할 수 있으며 회전이 필요에 따라 동작하려면 Spine 프로젝트 내에서 변경해야 할 수 있습니다.

통사론:

skeleton_animation_mix(animfrom, animto, duration);

논쟁 유형 설명
animfrom 끈 The name (a string) of the first animation set to interpolate from.
animto 끈 The name (a string) of the second animation set to interpolate to.
duration 진짜 The duration of the interpolation (from 0 to 1)

보고:

N/A

예시:

skeleton_animation_set("walk");
skeleton_animation_mix("walk", "jump", 0.2);
skeleton_animation_mix("jump", "walk", 0.4);

위의 코드는 스켈레탈 애니메이션 스프라이트가 있는 인스턴스의 이벤트 생성에 들어가 두 애니메이션 세트 "walk"와 "jump" 사이를 보간하기 위한 애니메이션 믹스 지속 시간을 설정합니다.
