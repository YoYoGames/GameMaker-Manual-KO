draw_get_enable_skeleton_blendmodes

이 함수는 스켈레탈 스프라이트에 대한 슬롯별 블렌드 모드가 활성화되었는지( true ) 비활성화되었는지( false ) 반환합니다.

통사론:

draw_get_enable_skeleton_blendmodes()

보고:

Boolean

예시:

if (!draw_get_enable_skeleton_blendmodes())
{
    draw_enable_skeleton_blendmodes(true);
}

위의 코드는 비활성화된 경우 스켈레탈 스프라이트에 대한 슬롯별 블렌드 모드를 활성화합니다.
