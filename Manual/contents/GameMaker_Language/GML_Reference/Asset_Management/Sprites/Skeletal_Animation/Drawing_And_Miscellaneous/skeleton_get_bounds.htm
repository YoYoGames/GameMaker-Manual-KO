skeleton_get_bounds

이 함수는 현재 할당된 스켈레톤 애니메이션 스프라이트에 대해 주어진 경계 상자와 관련된 값 배열을 반환합니다. 세부 정보를 얻기 위해 경계 상자의 인덱스 번호를 제공합니다(함수를 사용하여 스프라이트에 대한 경계 상자의 총 수를 검색할 수 있습니다.skeleton_get_num_bounds() ) 함수는 다음 요소가 포함된 배열을 반환합니다.

  • [0] - 경계 상자의 점 수(실수)
  • [1] - 경계 상자의 이름(문자열)
  • [2] - 첫 번째 점의 x 위치
  • [3] - 첫 번째 점의 y 위치
  • [etc...] - 경계 상자의 포인트 수까지 추가 x/y 위치 데이터

통사론:

skeleton_get_bounds(index);

논쟁 유형 설명
index 진짜 The bounding box index to get the values of.

보고:

Array (minimum 2 elements: numPoints, name [, xPos, yPos, etc...])

예시:

var num = skeleton_get_num_bounds();
var yy = 60;
for(var i = 0; i < num; i++)
{
    var b_info = skeleton_get_bounds(i);
    if b_info[0] > 0
    {
        var data = b_info[1] + ":";
        for(var j = 0; j < b_info[0]; j++)
        {
            data += " (" + string(b_info[(j * 2) + 2]) + ", " + string(b_info[(j * 2) + 2 + 1]) + ")";
        }
        draw_text(20, yy, data);
        yy += 20;
    }
}

위의 코드는 현재 할당된 스프라이트와 연결된 각 경계 상자를 반복한 다음 해당 데이터를 문자열로 화면에 그립니다.
