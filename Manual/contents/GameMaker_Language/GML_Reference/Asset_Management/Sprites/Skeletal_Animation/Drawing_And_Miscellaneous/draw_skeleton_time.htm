draw_skeleton_time

이 함수는 애니메이션의 특정 시간에 지정된 스킨을 사용하여 지정된 애니메이션을 그립니다. 시간 값은 함수를 사용하여 찾을 수 있는 애니메이션의 시작 시간과 끝 시간 사이에 있어야 합니다.skeleton_animation_get_duration(). 시간 값을 애니메이션의 총 지속 시간보다 높은 값으로 설정할 수 있으며 애니메이션은 처음으로 다시 돌아가지만 누적 시간이 커질수록 부동 소수점 정확도를 잃을 위험이 있습니다.

통사론:

draw_skeleton_time(sprite, animname, skinname, time, x, y, xscale, yscale, rot, colour)

논쟁 유형 설명
sprite 스프라이트 자산 The index of the sprite to draw.
animname 끈 The name of the animation to get the frame from (a string).
skinname 끈 The name of the skin to use for drawing (a string).
time 진짜 The time-frame to draw the animation at (from 0 to the end duration, in seconds).
x 진짜 The x coordinate of where to draw the sprite.
y 진짜 The y coordinate of where to draw the sprite.
xscale 진짜 The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale 진짜 The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot 진짜 The rotation of the sprite. 0=normal, 90=turned 90 degrees counter-clockwise etc.
colour 색깔 The colour with which to blend the sprite.

보고:

N/A

예시:

var time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white);

위의 코드는 delta-time을 사용하여 주어진 스켈레탈 애니메이션 스프라이트를 그려서 그릴 프레임을 설정합니다.
