skeleton_get_num_bounds

이 함수는 코드를 실행하는 인스턴스에 할당된 스켈레톤 애니메이션 스프라이트와 연결된 경계 상자의 수를 반환합니다. 그런 다음 기능과 함께 사용할 수 있습니다.skeleton_get_bounds() 각 경계 상자에 대한 데이터를 검색합니다.

통사론:

skeleton_get_num_bounds();

보고:

Real

예시:

var num = skeleton_get_num_bounds();
var yy = 60;
for(var i = 0; i < num; i++)
{
    var b_info = skeleton_get_bounds(i);
    if b_info[0] > 0
    {
        var data = b_info[1] + ":";
        for(var j = 0; j < b_info[0]; j++)
        {
            data += " (" + string(b_info[(j * 2) + 2]) + ", " + string(b_info[(j * 2) + 2 + 1]) + ")";
        }
        draw_text(20, yy, data);
        yy += 20;
    }
}

위의 코드는 현재 할당된 스프라이트와 연결된 각 경계 상자를 반복한 다음 해당 데이터를 문자열로 화면에 그립니다.
