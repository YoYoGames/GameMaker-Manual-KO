draw_skeleton_instance

이 기능은 스켈레탈 애니메이션 프로그램 Spine을 사용하여 생성된 스프라이트가 할당된 인스턴스 에만 사용할 수 있습니다. 일반 그리기 기능을 사용하여 이러한 스프라이트를 그릴 수 있지만(예:draw_self() ), 실제로 스프라이트 속성을 변경하지 않고 단일 프레임을 그리거나 스킨 텍스처의 변화를 보여주고 싶을 때가 있습니다. 이러한 경우 이 함수를 사용하여 인스턴스에 할당된 스프라이트를 그릴 수 있습니다. 여기서 고유한 인스턴스 ID를 제공한 다음 프레임을 가져올 애니메이션의 이름(문자열, 만드는 데 사용된 프로그램 내에서 스프라이트). 다음으로 사용할 스킨의 이름(다시 스프라이트를 만들 때 설정한 대로)과 프레임 값을 지정합니다. 프레임 값은 GameMaker 에게 애니메이션의 어떤 부분을 표시할지 알려주기 위한 것이며 0과image_number. 나머지 인수는 일반 스프라이트 그리기에 사용되는 것과 동일하며 그려지는 최종 애니메이션 프레임의 크기, 각도, 혼합 색상 및 알파를 변경합니다.

경고! 골격 애니메이션이 다양한 "키" 프레임 사이에 삽입되는 방식 때문에 이 호출과 관련된 추가 오버헤드가 있으므로 절대적으로 필요한 경우가 아니면 이 방법을 사용하여 골격 기반 스프라이트를 그리는 것을 피하는 것이 좋습니다.

통사론:

draw_skeleton_instance(instance, animname, skinname, frame, x, y, xscale, yscale, rot, colour, alpha);

논쟁 유형 설명
instance 인스턴스 ID The index of the instance to draw the sprite from.
animname 끈 The name of the animation to get the frame from (a string).
skinname 끈 The name of the skin to use (a string).
frame 진짜 The animation frame to draw (from 0 to image_number - 1).
x 진짜 The x coordinate of where to draw the sprite.
y 진짜 The y coordinate of where to draw the sprite.
xscale 진짜 The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale 진짜 The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot 진짜 The rotation of the sprite. 0=normal, 90=turned 90 degrees counter-clockwise etc.
colour 색깔 The colour with which to blend the sprite.
alpha 진짜 The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

보고:

N/A

예시:

draw_skeleton_instance(obj_Player, "jump", "skin1", 0, x, y, 1, 1, 0, c_white, 1);

위의 코드는 "점프" 애니메이션 세트와 "skin1"으로 정의된 스킨을 사용하여 주어진 인스턴스에 할당된 스켈레탈 애니메이션 스프라이트의 첫 번째 프레임을 그립니다.
