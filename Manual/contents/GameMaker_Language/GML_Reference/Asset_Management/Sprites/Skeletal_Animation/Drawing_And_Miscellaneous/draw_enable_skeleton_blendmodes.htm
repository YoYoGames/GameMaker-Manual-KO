draw_enable_skeleton_blendmodes

이 함수는 스켈레탈 스프라이트에 대한 슬롯별 블렌드 모드를 활성화( true ) 또는 비활성화( false )하는 데 사용됩니다.

통사론:

draw_enable_skeleton_blendmodes(enable)

논쟁 유형 설명
enable 부울 Whether to enable or disable blend modes

보고:

N/A

예시:

draw_enable_skeleton_blendmodes(true);

위의 코드는 스켈레탈 스프라이트에 대한 슬롯별 블렌드 모드를 활성화합니다.
