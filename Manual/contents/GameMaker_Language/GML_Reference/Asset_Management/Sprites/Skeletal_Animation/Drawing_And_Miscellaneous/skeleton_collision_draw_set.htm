skeleton_collision_draw_set

이 기능을 사용하면 (true ) 또는 끄기(false ) 인스턴스에서 사용 중인 현재 스켈레탈 애니메이션 스프라이트에 대한 충돌 데이터를 그립니다. 이것이 켜져 있으면 경계 상자와 정확한 충돌 마스크가 스프라이트 주변의 윤곽선으로 그려집니다.

이 기능은 스켈레탈 스프라이트 그리기가 객체에 의해 처리될 때만 작동합니다. 스켈레탈 스프라이트를 수동으로 그릴 때 충돌 경계 상자를 그리려면 다음을 사용하십시오. draw_skeleton_collision().

통사론:

skeleton_collision_draw_set(flag);

논쟁 유형 설명
flag 부울 Set to true to turn on drawing, and false to turn it off.

보고:

N/A

예시:

if debug_mode == true
{
    skeleton_collision_draw_set(true);
}

위의 코드는 게임이 디버그 모드에서 실행되고 있는지 확인하고, 그렇다면 표시할 인스턴스에 대한 골격 충돌 데이터를 토글합니다.
