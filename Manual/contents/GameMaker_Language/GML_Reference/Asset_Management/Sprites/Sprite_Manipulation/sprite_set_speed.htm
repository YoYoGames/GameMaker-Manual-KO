sprite_set_speed

이 기능은 스프라이트 리소스에 대한 기본 애니메이션 속도 및 유형을 설정하는 데 사용할 수 있습니다. 이것은 일반적으로 스프라이트 편집기 에서 설정되지만 주어진 스프라이트 리소스에 대해 수동으로 설정해야 하는 경우가 있습니다. 설정할 스프라이트 인덱스, 애니메이션 속도 및 적용할 애니메이션 속도 유형을 제공해야 합니다. 애니메이션 속도 유형은 다음 상수 중 하나일 수 있습니다.

일정한 설명
spritespeed_framespersecond 스프라이트는 초당 프레임 수 의 애니메이션으로 정의되었습니다.
spritespeed_framespergameframe 스프라이트는 애니메이션에서 게임 프레임당 프레임 으로 정의되었습니다.


애니메이션 계산에 사용할 속도 유형을 선택하는 것은 두 옵션 사이에 큰 차이가 있기 때문에 매우 중요합니다. 다음 두 가지 예는 이러한 차이점을 보여줍니다.

  • 초당 1프레임을 그리는 스프라이트가 있고 이미지 속도를 0.5로 설정하면 초당 0.5프레임으로 그려집니다. 게임 프레임 속도가 초당 60프레임이면 스프라이트는 120게임 프레임마다 1프레임을 그립니다.
  • 게임 프레임당 1프레임을 그리는 스프라이트가 있고 이미지 속도를 0.5로 설정하면 게임 프레임당 0.5프레임을 그립니다. 게임 프레임 속도가 초당 60프레임이면 스프라이트는 60게임 프레임마다 30프레임을 그립니다.

스프라이트 애니메이션에 사용된 속도 값이 함수를 사용하고 있는지 알 수 있습니다.sprite_get_speed(), 다음을 사용하여 애니메이션 속도 유형을 얻을 수 있습니다.sprite_get_speed_type().

중요한! 이 함수는 스프라이트 리소스 에 영향을 미치므로 이 함수로 변경한 사항은 스프라이트를 사용하는 모든 인스턴스와 이후의 모든 인스턴스에도 영향을 미칩니다.

통사론:

sprite_set_speed(index, speed, type)

논쟁 유형 설명
index 스프라이트 자산 The index of the sprite to set the animation speed of
speed 진짜 The speed value to set
type 스프라이트 속도 상수 The speed type to use

보고:

Constant (see above)

예시:

sprite_set_speed(sprite_index, 10, spritespeed_framespersecond);

위의 코드는 인스턴스에 할당된 스프라이트의 기본 애니메이션 속도를 초당 10프레임으로 설정합니다.
