sprite_exists

이 함수는 지정된 인덱스를 가진 스프라이트가 실행 중인 프로젝트에 존재하는지 여부를 반환합니다.

통사론:

sprite_exists(index);

논쟁 유형 설명
index The index of the sprite to check.

보고

예시:

if sprite_exists(spr_array[0])
{
    sprite_index = spr_array[0];
}
else
{
    sprite_index = spr_BaseSprite;
}

위의 코드는 배열에 유효한 스프라이트 인덱스가 포함되어 있는지 확인하고 그렇다면 해당 스프라이트를 인스턴스에 할당하지만 그렇지 않은 경우 포함된 리소스에서 스프라이트를 할당합니다.
