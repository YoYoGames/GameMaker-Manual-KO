sprite_delete

이 함수는 게임에서 스프라이트를 삭제하여 예약된 메모리를 해제합니다. 이것은 외부 소스에서 스프라이트를 만들고 변경해야 하는 순간에 매우 중요한 기능입니다(예: 파일에서 스프라이트 로드 sprite_add(), 또는 다음을 사용하여 스프라이트 복제 sprite_duplicate() ) 게임에 더 이상 필요하지 않은 자산을 제거하거나 해당 변수에 다른 자산을 재할당하기 전에 변수에서 인덱싱된 자산을 지우는 데 항상 사용해야 합니다. 자산 브라우저에서 자산의 일부로 게임에 포함된 스프라이트를 삭제하는 데 사용할 수 없습니다.

통사론:

sprite_delete(index);

논쟁 유형 설명
index The index of the sprite to be deleted.

보고

예시:

var spr = sprite_create_from_surface(application_surface, 0, 0, 32, 32, false, false, 16, 16);
sprite_merge(spr_Player, spr);
sprite_delete(spr);

위의 코드는 지역 변수를 생성한 다음 애플리케이션 표면에서 생성된 스프라이트의 인덱스를 저장합니다. 그런 다음 이 스프라이트는 메모리에서 다시 제거되기 전에 "spr_Player" 변수에 인덱싱된 자산과 병합됩니다.
