sprite_prefetch_multi

이 함수는 주어진 스프라이트를 포함하는 여러 텍스처 페이지를 프리페치(텍스처 메모리에 배치)하는 데 사용할 수 있습니다. 스프라이트 인덱스(스프라이트 자산을 생성할 때 정의한 대로)로 채워진 배열을 제공하면 해당 인덱스가 있는 텍스처 페이지가 메모리에 로드됩니다. 함수가 반환됩니다. -1 선택한 리소스에 대해 프리페치가 지원되지 않거나 대상 플랫폼이 HTML5인 경우 0 모두 올바르게 작동했다면.

참고: 대부분의 장치에서 텍스처가 텍스처 메모리에 업로드될 때 성능 저하가 있으므로 다음을 호출하는 것이 좋습니다.sprite_prefetch_multi() 스톨을 피하기 위해 레벨 시작 시 필요한 그래픽에

통사론:

sprite_prefetch_multi

논쟁 유형 설명
array 스프라이트 자산 배열 Array with the sprite indices to fetch

보고:

Real (-1 or 0)

예시:

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_prefetch_multi(spr_a);

위의 코드는 각 요소가 스프라이트 인덱스를 보유하는 배열을 만듭니다. 그런 다음 이 배열을 사용하여 해당 스프라이트 텍스처를 메모리에 배치합니다.
