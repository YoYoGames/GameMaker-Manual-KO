sprite_create_from_surface

이 함수를 사용하면 이전에 초기화된 표면에서 스프라이트를 만들 수 있습니다(표면 인덱스 ID 값은 다음을 사용하여 표면을 만들 때 반환됩니다.surface_create() ). 그런 다음 함수에 입력하는 x 및 y 좌표는 게임 창(활성화된 뷰가 있는 경우 보기도 포함)이 아니라 표면 (왼쪽 상단 모서리)의 (0,0) 위치에 상대적이어야 합니다. 너비 및 높이 인수는 픽셀 단위이며 사용할 표면 부분의 너비와 높이를 정의합니다.

"removeback" 인수를 다음으로 설정 true 스프라이트의 왼쪽 하단 픽셀에서 색상을 확인한 다음 제거할 색상으로 사용하여 스프라이트에서 색상을 제거합니다.

"removeback" 옵션을 선택하면 GameMaker 가 "smooth" 인수를 다음으로 설정하여 스프라이트의 가장자리를 매끄럽게 만들 수도 있습니다. true. 이 모든 작업은 배경이 제거된 스프라이트의 가장자리 주위에 반투명 테두리를 만드는 것입니다.

마지막으로 스프라이트의 x 및 y 원점 을 지정할 수도 있습니다. 이것은 스프라이트가 그것을 사용하는 인스턴스에 "고정"되는 지점이며 항상 하나의 스프라이트 하위 이미지의 왼쪽 상단 모서리 0,0을 기준으로 계산됩니다. 예를 들어, 이 값이 (16,16)으로 설정된 32 x 32 픽셀의 스프라이트는 중앙에 원점이 있습니다.

기본적으로 모든 새 스프라이트에는 경계 상자가 자동으로 계산됩니다(정확한 bbox는 스프라이트의 크기와 투명도에 따라 다름). 그러나 이것을 사용자 정의하고 싶을 수도 있습니다. 이 경우 함수도 사용해야 합니다.sprite_collision_mask().

참고 이 방법으로 GameMaker 에서 스프라이트를 생성할 때 다시 제거하는 것을 기억해야 합니다(sprite_delete() ) 더 이상 필요하지 않을 때 메모리 누수의 위험이 있어 속도가 느려지고 결국 게임이 중단됩니다.

통사론:

sprite_create_from_surface(index, x, y, w, h, removeback, smooth, xorig, yorig);

논쟁 유형 설명
index 표면 ID The index of the surface to create from.
x 진짜 The x position to copy from.
y 진짜 The y position to copy from.
w 진짜 The width of the area to be copied (from the x position).
h 진짜 The height of the area to be copied (from the y position).
removeback 부울 Indicates whether to make all pixels with the background colour (left-bottom pixel) transparent.
smooth 부울 Indicates whether to smooth the edges.
xorig 진짜 Indicates the x position of the origin in the sprite.
yorig 진짜 Indicates the y position of the origin in the sprite.

예시:

var surf;
surf = surface_create(32, 32);
surface_set_target(surf);
draw_clear_alpha(c_black, 0);
draw_sprite(spr_Body, 0, 0, 0);
draw_sprite(spr_Clothes, 0, 0, 0);
draw_sprite(spr_Hair, 0, 0, 0);
spr_custom = sprite_create_from_surface(surf, 0, 0, 32, 32, false, false, 16, 16);
surface_reset_target();
surface_free(surf);

위의 코드는 표면을 생성하고 그 인덱스를 지역 변수 "surf"에 저장합니다. 그런 다음 해당 표면을 대상으로 지정하고 지우고 서로 위에 여러 스프라이트를 그립니다. 마지막으로 표면에 그려진 합성 이미지에서 새 스프라이트를 만들고 표면에서 사용하는 메모리를 해제하기 전에 인덱스를 "spr_Custom"에 할당합니다.
