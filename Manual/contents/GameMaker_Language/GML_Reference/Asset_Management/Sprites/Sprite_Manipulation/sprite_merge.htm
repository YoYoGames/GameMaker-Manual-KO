sprite_merge

이 함수는 인수 1("ind2")에 인덱싱된 스프라이트를 인수 0("ind1")에 인덱싱된 스프라이트와 병합합니다. 이미지 자체는 함께 병합 되지 않고 오히려 이미지 인덱스가 병합되어 스프라이트 "ind2"의 하위 이미지가 스프라이트 "ind1"의 이미지에 추가됩니다. 즉, 끝에 추가됩니다. 스프라이트의 크기가 다른 경우 추가된 스프라이트는 "ind1"의 이미지 크기에 맞게 늘어납니다.

이 변경 사항은 영구적이며 이 기능을 사용하는 순간부터 게임이 종료되거나 스프라이트가 삭제될 때까지 병합되는 스프라이트는 변경되지만 병합되는 스프라이트는 동일하게 유지됩니다.

참고: 병합 중인 이미지 자산이나 병합할 이미지 자산은 원래 게임 리소스 중 하나가 될 수 없습니다. 당신은 사용해야합니다sprite_duplicate() 먼저 두 개의 새 리소스(포함된 게임 자산을 병합하는 경우)를 만든 다음 병합합니다.

참고: 이 기능은 비트맵 스프라이트에만 유용하며 SWF 또는 JSON(Spine) 스프라이트에서는 작동하지 않습니다.

통사론:

sprite_merge(ind1, ind2);

논쟁 유형 설명
ind1 The index of the sprite to merge.
ind2 The index of the sprite that ind1 is to be merged with.

보고

예시:

var tspr;
tspr[0] = spr_add(working_directory + "Sprite Assets\explode1.png", 16, true, true, 0, 0);
tspr[1] = sprite_duplicate(spr_Explosion);
sprite_merge(tspr[0], tspr[1]);
sprite_index = t_spr[0];
sprite_delete(tspr[1]);

위의 코드는 스프라이트를 로컬 변수에 로드한 다음 복제된 다른 스프라이트와 병합합니다. 마지막으로 병합된 스프라이트가 인스턴스에 할당됩니다. sprite_index 로드된 스프라이트는 사용하는 메모리를 해제하기 위해 제거됩니다. Destroy 또는 Room End 이벤트와 같이 이 작업 후 어느 시점에서 다음을 사용해야 합니다.sprite_delete() 병합된 스프라이트도 메모리에서 제거합니다.
