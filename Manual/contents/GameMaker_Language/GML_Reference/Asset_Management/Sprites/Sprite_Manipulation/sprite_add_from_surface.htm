sprite_add_from_surface

이 기능은 다음과 정확히 같은 방식으로 작동합니다.sprite_create_from_surface() 선택한 인덱싱된 표면 영역에서 새 스프라이트를 만드는 대신 이전에 만든 스프라이트에 새 하위 이미지로 표면의 정의된 영역을 추가합니다(즉, 리소스 트리가 아니라 표면에서 이전에 생성된 트리 또는 다음을 사용하여 리소스 스프라이트에서 복제된 트리에만 적용됩니다.sprite_duplicate() ).

이런 식으로 스프라이트를 추가할 때 사용되는 메모리는 예상보다 클 것입니다. GameMaker 가 스프라이트를 텍스처 페이지 로 저장하고 GPU 메모리에도 저장하기 때문에 총 메모리가 동일한 스프라이트의 이미지 파일에 대해 예상되는 것보다 더 큽니다.

통사론:

sprite_add_from_surface(index, surface, x, y, w, h, removeback, smooth);

논쟁 유형 설명
index 스프라이트 자산 The index of the sprite to add the new image to.
surface 표면 ID The index of the surface from which we get the image.
x 진짜 The x position to copy from.
y 진짜 The y position to copy from.
w 진짜 The width of the area to be copied (from the x position).
h 진짜 The height of the area to be copied (from the y position).
removeback 부울 Indicates whether to make all pixels with the background colour (left-bottom pixel) transparent.
smooth 부울 Indicates whether to smooth the edges.

예시:

spr_custom = sprite_create_from_surface(surf, 0, 0, 32, 32, true, true, 16, 16);
var i;
for (i = 1; i < 8; i += 1)
{
    sprite_add_from_surface(spr_custom, surf, i, 0, 32, 32, true, true);
}

위의 코드는 "surf" 변수에 인덱싱된 표면에서 스프라이트를 만들고 인덱스를 변수 "spr_Custom"에 할당한 다음 for 루프를 사용하여 표면을 가로질러 이동하고 스프라이트에 하위 이미지로 추가되는 다양한 섹션을 캡처합니다.
