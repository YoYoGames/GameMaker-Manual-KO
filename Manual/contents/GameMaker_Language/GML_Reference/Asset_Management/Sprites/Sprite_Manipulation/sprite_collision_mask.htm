sprite_collision_mask

이 함수를 사용하면 스프라이트가 가져야 하는 충돌 마스크의 속성을 설정할 수 있습니다. 경계 상자 모드로 자동(0) 또는 전체 이미지(1)를 선택하면 개별 경계 상자 값을 0으로 설정할 수 있습니다. 그러나 사용자 정의 마스크(2)의 경우 이 값을 설정해야 합니다. 다른 경계 상자 값은 항상 위치(0, 0)로 간주되는 스프라이트의 왼쪽 상단 모서리(x 및 y 원점에 관계 없이)를 기준으로 합니다.

마스크 종류를 설정하면 마스크 자체의 일반적인 모양이 설정되지만 직사각형 마스크가 아닌 다른 모든 항목은 충돌을 해결할 때 더 많은 처리 능력이 필요하므로 성능이 떨어집니다. 일반적으로 꼭 필요한 경우에만 직사각형 이외의 마스크 유형을 사용해야 합니다.

참고: 이 기능은 회전된 사각형 충돌 마스크 종류를 허용하지 않습니다.

설정할 수 있는 마스크의 종류는 다음 상수 중 하나입니다.

경계 상자 종류(모양) 상수
일정한 설명
bboxkind_rectangular 직사각형(회전하지 않는) 직사각형 충돌 마스크 모양
bboxkind_ellipse 타원형 충돌 마스크 모양
bboxkind_diamond 다이아몬드 충돌 마스크 모양
bboxkind_precise 주어진 허용 오차 값에 따라 마스크가 스프라이트의 불투명한 픽셀을 따르는 정확한 충돌 마스크(아래 참조)


마지막으로 허용 오차는 마스크가 얼마나 정확한지를 정의하는 데 사용됩니다("전체 이미지" 마스크와 함께 사용할 때 효과가 없음). 허용 오차가 0이면 마스크가 투명도가 있는 모든 단일 픽셀을 따릅니다. 다른 값은 픽셀의 투명도에 따라 충돌 마스크 둘레를 이동합니다.

참고: 이 기능은 비트맵 스프라이트에만 유용하며 SWF 또는 JSON(Spine) 스프라이트에서는 작동하지 않습니다.

참고: 이 기능은 추가된 스프라이트 또는 복제된 스프라이트에서만 작동하며 미리 만들어진 리소스에서는 직접 작동 하지 않습니다. 함수를 사용하여 스프라이트를 복제할 수 있습니다.sprite_duplicate().

통사론:

sprite_collision_mask(ind, sepmasks, bboxmode, bbleft, bbtop, bbright, bbbottom, kind, tolerance);

논쟁 유형 설명
ind 스프라이트 자산 The index of the sprite to set the bounding box of.
sepmasks 부울 Whether to create collision masks for each sub-image of the sprite (true ), or one mask for all (false ).
bboxmode 진짜 What kind of bounding box to use. 0 = automatic, 1 = full image, 2 = user defined.
bbleft 진짜 The maximum left position of the bounding box.
bbtop 진짜 The maximum top position of the bounding box.
bbright 진짜 The maximum right position of the bounding box.
bbbottom 진짜 The maximum bottom position of the bounding box.
kind 경계 상자 종류(모양) 상수 The kind of mask, a constant (see the table in the description).
tolerance 진짜 Indicates the tolerance in the transparency value (0=no tolerance, 255=full tolerance).

보고

N/A

예시:

spr = sprite_add("player_5.png", 16, true, true, 0, 0);
sprite_collision_mask(spr, true, 1, 0, 0, 0, 0, 0, 0);

위의 코드는 외부 소스에서 스프라이트를 로드하고 변수 "spr"에 새 인덱스를 저장합니다. 그런 다음 코드는 각 하위 이미지에 대해 정확한 충돌 마스크를 갖도록 새 스프라이트를 설정합니다.
