sprite_flush

이 함수를 사용하면 텍스처 메모리에서 주어진 스프라이트에 대해 주어진 텍스처 페이지를 제거할 수 있습니다. 스프라이트 인덱스(스프라이트 리소스를 생성할 때 정의된 대로)를 제공하면 할당된 텍스처 페이지가 텍스처 메모리에서 제거됩니다. 텍스처 페이지가 방의 다른 곳에서 사용되는 경우(다른 인스턴스 스프라이트 또는 배경 등...) 페이지가 즉시 다시 로드되므로 약간의 성능 저하가 발생할 수 있으므로 이 기능을 사용할 때 주의해야 합니다. 함수가 반환됩니다. -1 선택한 리소스에 대해 플러시가 지원되지 않으면 반환됩니다. 0 모두 올바르게 작동했다면.

통사론:

sprite_flush(ind)

논쟁 유형 설명
ind The index (resource name) of the sprite asset to flush

보고:

(-1 or 0)

예시:

sprite_flush(spr_Player_Aura);

위의 코드는 메모리에서 "spr_Player_Aura" 스프라이트를 플러시합니다.
