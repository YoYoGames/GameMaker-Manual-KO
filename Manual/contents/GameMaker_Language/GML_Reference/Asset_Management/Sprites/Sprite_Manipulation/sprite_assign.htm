sprite_assign

이 함수는 이전에 생성된(또는 포함된) 두 개의 스프라이트 인덱스를 사용하고 이미지를 하나에서 다른 것으로 복사합니다. 이런 식으로 하나의 스프라이트를 다른 인덱스에 복사(또는 "복제")할 수 있습니다. 게임 리소스에 복사할 수 없습니다. 이전에 복사할 스프라이트를 생성해야 합니다.sprite_add() 또는sprite_duplicate() 기능.

통사론:

sprite_assign(index, sprite);

논쟁 유형 설명
index The index of the sprite to be copied to (ie: overwritten).
sprite The sprite to be copied from.

보고

예시:

if sprite_exists(global.Player_Sprite)
{
    var t_spr = sprite_add("player.png", 16, true, true, 0, 0);
    sprite_assign(global.Player_Sprite, t_spr);
    sprite_delete(t_spr);
}
else
{
    global.Player_Sprite = sprite_add("player.png", 16, true, true, 0, 0);
}

위의 코드는 전역 변수 "Player_Sprite"에 스프라이트가 포함되어 있는지 확인하고 스프라이트가 있으면 sprite_assign을 사용하여 외부 파일에서 로드된 것으로 변경합니다. 스프라이트가 포함되어 있지 않으면 로드되고 인덱스가 해당 변수에 저장됩니다.
