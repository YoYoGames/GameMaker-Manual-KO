sprite_save_strip

이 기능은 스프라이트의 모든 하위 이미지에서 스트립 이미지를 만들어 지정된 파일 이름으로 디스크에 저장합니다. 스프라이트는 런타임에 추가 되어야 하며(IDE를 통해 추가된 스프라이트는 저장할 수 없음) 파일은 *.png 확대.

참고: 선택한 대상 플랫폼에 따라 파일을 저장하고 로드할 수 있는 위치가 제한됩니다. 자세한 내용은 파일 처리 를 참조하십시오.

통사론:

sprite_save_strip(ind, filename);

논쟁 유형 설명
ind 스프라이트 자산 The index of the sprite to save.
filename 끈 The filename for the saved sprite strip.

보고

N/A

예시:

var spr, i;
spr = sprite_create_from_surface(0, 0, 32, 32, true, true, 16, 16);

for (i = 1; i < 8; i +=1)
{
    sprite_add_from_surface(spr, i, 0, 32, 32, true, true, 16, 16);
}

sprite_save_strip(spr, "Custom_Player_Sprite.png");
sprite_delete(spr);

위의 코드는 "surf" 변수에 인덱싱된 표면에서 스프라이트를 만들고 인덱스를 로컬 변수 "spr_Custom"에 할당한 다음 for 루프를 사용하여 표면을 가로질러 이동하고 스프라이트에 하위 이미지로 추가되는 다양한 섹션을 캡처합니다. 이 새 스프라이트는 메모리에서 제거되기 전에 png 스트립으로 저장됩니다.
