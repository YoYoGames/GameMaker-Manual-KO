sprite_set_bbox_mode

이 함수는 스프라이트의 경계 상자 모드를 설정하는 데 사용할 수 있습니다. 다음 상수 중 하나여야 하는 스프라이트 인덱스와 사용할 모드를 제공합니다.

경계 상자 모드 상수
일정한 설명
bboxmode_automatic 자동 - 경계 상자는 스프라이트의 허용 오차 설정에 따라 자동으로 계산됩니다.
bboxmode_fullimage 전체 이미지 - 허용 오차 및 "빈" 픽셀에 관계없이 경계 상자가 스프라이트의 전체 너비와 높이를 사용하도록 설정됩니다.
bboxmode_manual 수동 - 경계 상자가 수동으로 사용자 정의 값으로 설정되며, 이를 위해서는 함수를 사용해야 합니다.sprite_set_bbox()

참고 이 함수는 이 스프라이트가 있는 모든 추가 인스턴스가 동일한 경계 상자 모드를 갖도록 스프라이트 자산 에 영향을 줍니다.

통사론:

sprite_set_bbox_mode(ind, mode);

논쟁 유형 설명
ind 스프라이트 자산 The index of the sprite to change the mode of.
mode 경계 상자 모드 상수 The mode to set (a constant).

보고

N/A

예시:

if sprite_get_bbox_mode(sprite_index) != bboxmode_automatic
{
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

위의 코드는 현재 스프라이트에 대한 bbox 모드를 확인하고 그렇지 않은 경우 automatic, 그 값으로 설정됩니다.
