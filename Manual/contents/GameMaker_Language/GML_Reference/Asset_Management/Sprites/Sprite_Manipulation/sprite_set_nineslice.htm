sprite_set_nineslice

이 함수는 Nine Slice 구조체 를 스프라이트에 적용하는 데 사용됩니다. 다음을 사용하여 검색할 수 있는 적용할 스프라이트 인덱스와 Nine Slice 구조체를 제공합니다. sprite_nineslice_create() 또는 sprite_get_nineslice().

참고 이 함수는 스프라이트 자산 에 영향을 미치므로 이 함수로 변경한 사항은 이 스프라이트를 사용하는 모든 인스턴스와 이후의 모든 인스턴스에도 영향을 미칩니다.

통사론:

sprite_set_nineslice(ind, nineslice);

논쟁 유형 설명
ind 스프라이트 자산 The index of the sprite to modify
nineslice 나인 슬라이스 구조체 The Nine Slice struct to apply

보고:

N/A

예시:

var _nineslice = sprite_nineslice_create();

_nineslice.enabled = true;
_nineslice.left = 10;
_nineslice.right = 10;
_nineslice.top = 10;
_nineslice.bottom = 10;

sprite_set_nineslice(spr_box_0, _nineslice);

위의 코드는 새로운 Nine Slice 구조체를 생성하고 활성화하고 가이드의 오프셋을 설정합니다. 그런 다음 구조체가 스프라이트에 적용되어 Nine Slice 속성이 변경됩니다.
