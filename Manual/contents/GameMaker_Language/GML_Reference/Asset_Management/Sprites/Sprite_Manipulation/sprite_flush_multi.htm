sprite_flush_multi

이 기능을 사용하면 텍스처 메모리에서 주어진 스프라이트에 대한 텍스처 페이지를 원하는 수만큼 제거할 수 있습니다. 스프라이트 인덱스를 배열로 제공하면 할당된 텍스처 페이지가 텍스처 메모리에서 제거됩니다. 텍스처 페이지 중 하나가 방의 다른 곳에서 사용되는 경우(다른 인스턴스 스프라이트 또는 배경 등...) 페이지가 즉시 메모리로 다시 다시 로드되므로 약간의 성능 저하가 발생할 수 있으므로 주의해야 합니다. 이 기능을 사용할 때 찍은 것입니다. 함수가 반환됩니다. -1 선택한 리소스에 대해 플러시가 지원되지 않으면 반환됩니다. 0 모두 올바르게 작동했다면.

통사론:

sprite_flush_multi(array);

논쟁 유형 설명
array Array with the sprite indices to flush

보고:

(-1 or 0)

예시:

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_flush_multi(spr_a);

위의 코드는 각 요소가 스프라이트 인덱스를 보유하는 배열을 만듭니다. 이 배열은 메모리에서 해당 스프라이트 텍스처를 지우는 데 사용됩니다.
