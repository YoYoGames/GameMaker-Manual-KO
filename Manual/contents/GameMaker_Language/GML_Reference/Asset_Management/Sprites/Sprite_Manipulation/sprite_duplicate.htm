sprite_duplicate

이 함수는 "index" 인수로 하나의 입력을 복제(복사)한 새로 생성된 스프라이트의 인덱스를 반환합니다. 함수가 실패하면 -1이 반환됩니다. 이 함수는 변환을 수행하기 전에 원본 자산에서 모든 스프라이트를 복사하는 데 사용해야 합니다. 예를 들어, 스프라이트의 경계 상자를 변경하거나 다른 스프라이트의 알파를 설정하려면 먼저 복제한 다음 복제된 스프라이트에서 작업을 수행하고 이를 사용해야 합니다. 복제된 스프라이트는 생성될 때 고유한 텍스처 페이지에 배치됩니다. 즉, 여러 스프라이트를 복제하면 여러 텍스처 페이지가 생성되고 성능에 영향을 미치므로 이 기능은 필요할 때만 사용하십시오.

참고: GameMaker 에서 스프라이트를 복제할 때 다시 제거하는 것을 기억해야 합니다(sprite_delete() ) 더 이상 필요하지 않을 때 메모리 누수의 위험이 있어 속도가 느려지고 결국 게임이 중단됩니다.

통사론:

sprite_duplicate(index);

논쟁 유형 설명
index The index of the sprite to duplicate.

보고

Sprite Asset or -1

예시:

new_spr = sprite_duplicate(sprite_index)

위의 코드는 현재 인스턴스의 스프라이트 인덱스로 사용 중인 스프라이트를 복제하고 이 새 스프라이트의 인덱스를 "new_spr" 변수에 저장합니다.
