sprite_set_offset

이 함수는 스프라이트의 x 및 y 원점을 설정하는 데 사용할 수 있으며 스프라이트의 왼쪽 상단 모서리인 (0,0) 위치를 기준으로 상대 값을 취합니다. 다음 이미지는 이를 보여줍니다.

참고 이 함수는 이 스프라이트가 있는 모든 추가 인스턴스가 동일한 오프셋을 갖도록 스프라이트 자산 에 영향을 줍니다.

통사론:

sprite_set_offset(ind, xoff, yoff);

논쟁 유형 설명
ind The index of the sprite to change the offset of.
xoff The x position of the origin.
yoff The y position of the origin.

보고

예시:

sprite_assign(spr_Custom, spr_Base);
sprite_set_offset(spr_Custom, sprite_get_xoffset(spr_Base), sprite_get_yoffset(spr_Base));

위의 코드는 "spr_Base"에 인덱싱된 스프라이트를 "spr_Custom"에 인덱싱된 스프라이트에 할당한 다음 "spr_Base"의 x 및 y 오프셋 값을 사용하여 새 스프라이트의 원점을 설정합니다.
