sprite_height

이 읽기 전용 변수는 인스턴스에 할당된 스프라이트의 높이를 반환합니다. 이 높이는 픽셀로 반환되며image_yscale. 크기가 조정되지 않은 높이가 필요한 경우 사용해야 합니다.sprite_get_height().

통사론:

sprite_height;

보고:

Real

예시:

if sprite_height != sprite_get_height(sprite_index)
{
    image_yscale = 1;
}

위의 코드는 코드를 실행하는 인스턴스에 할당된 스프라이트의 높이를 스프라이트 리소스의 높이와 비교하여 확인하고 동일하지 않으면 재설정합니다. image_yscale 1로,
