image_xscale

이 값은 현재 인스턴스에 할당된 스프라이트에 적용되는 수평 크기 조정을 설정합니다. 스케일이 1이면 스케일링이 없음(1:1), 더 작은 값은 스케일 다운(예: 0.5는 스프라이트 너비의 절반), 더 큰 값은 스케일 업되고 음수 값은 스프라이트를 뒤집고 스케일링합니다. 사용된 값은 정확히 -1입니다(이 경우 스프라이트는 크기 조정 없이 원점을 기준으로 뒤집힙니다).

통사론:

image_xscale;

보고:

(single precision floating point value)

예시:

if image_xscale < 5
{
    image_xscale += 0.2;
    image_yscale = image_xscale;
}
else
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

위의 코드는 스프라이트의 크기를 조정한 다음 원래 크기의 5배로 크기를 조정하면 다른 개체의 새 인스턴스가 생성되고 인스턴스가 소멸됩니다.
