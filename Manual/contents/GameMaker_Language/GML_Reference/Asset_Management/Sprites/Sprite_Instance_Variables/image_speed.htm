image_speed

이 변수는 GameMaker 가 현재 인스턴스 스프라이트의 하위 이미지를 순환하는 속도를 결정합니다. 주어진 속도 값은 승수 이며 1이 기본값이며 0.5로 설정하면 애니메이션 속도의 절반이 됩니다( 스프라이트 편집기 또는 이미지 편집기 에서 설정). 반면 2로 설정하면 두 배가 됩니다. 사용된 스프라이트에 하위 이미지가 없으면 이 변수는 효과가 없습니다.

통사론:

image_speed;

보고:

(single precision floating point value)

예시:

with (instance_create_layer(x, y, "Effects", obj_Explosion))
{
    image_speed = 0.5;
}

위의 코드는 "obj_Explosion" 개체의 인스턴스를 만들고 image_speed를 0.5로 설정합니다(실행 속도를 효과적으로 절반으로 줄임).
