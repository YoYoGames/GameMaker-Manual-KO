image_index

스프라이트는 스프라이트가 하나에서 다른 것으로 전환될 때 애니메이션으로 나타나도록 만들 수 있는 하나 이상의 하위 이미지 로 구성됩니다. 창문. 스프라이트가 애니메이션되면 다음을 확인하여 애니메이션의 현재 프레임을 가져올 수 있습니다. image_index 변수 또는 정적 스프라이트의 상태를 변경하려는 경우 이 변수를 스프라이트의 원하는 하위 이미지로 설정하여 새 하위 이미지를 선택할 수 있습니다. 이 변수의 변경 사항이 표시되려면 인스턴스에 그리기 이벤트가 없어야 하고(따라서 GameMaker 는 기본적으로 스프라이트를 그립니다) 다음과 같은 그리기 기능 중 하나를 사용하여 인스턴스를 그려야 합니다.draw_self() 또는draw_sprite_ext() (적절한 인수에 image_index를 제공함으로써).

이 변수를 프레임에 직접 설정하면 해당 프레임에 있을 수 있는 브로드캐스트 메시지가 트리거 되지 않습니다.

또한 스켈레탈 애니메이션 스프라이트를 사용하는 동안에도 여전히 가져오고 설정할 수 있습니다. image_index 값 - 함수 참조 skeleton_animation_get_duration() 이 작업을 수행하는 방법에 대한 예입니다.

통사론:

image_index;

보고:

(single precision floating point value)

예시:

if (image_speed > 0)
{
    if (image_index >= image_number - 1) instance_destroy();
}

위의 코드는 스프라이트가 애니메이션 중인지 확인하고 만약 그렇다면 현재 image_index가 마지막 프레임에 있는지 확인하고 이 경우 인스턴스를 파괴합니다.
