image_alpha

이 변수는 스프라이트의 알파 값을 가져오거나 설정하는 데 사용됩니다. 알파는 항상 0과 1 사이의 값으로 계산되며, 여기서 0은 완전히 투명하고 1은 완전히 불투명합니다. 이 변수의 변경 사항이 표시되려면 인스턴스에 그리기 이벤트가 없어야 하므로(따라서 GameMaker 는 기본적으로 스프라이트를 그립니다) 다음과 같은 확장 그리기 기능 중 하나를 사용하여 인스턴스를 그려야 합니다. draw_self() 또는draw_sprite_ext().

통사론:

image_alpha;

보고:

Real

예시:

image_alpha = clamp(image_alpha - 0.01, 0, 1);

위의 코드는 0에 도달할 때까지 image_alpha를 천천히 줄입니다.
