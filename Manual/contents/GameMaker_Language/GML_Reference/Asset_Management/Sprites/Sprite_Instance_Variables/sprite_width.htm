sprite_width

이 읽기 전용 변수는 인스턴스에 할당된 스프라이트의 너비를 반환합니다. 이 너비는 픽셀 단위로 반환되며image_xscale. 크기가 조정되지 않은 너비가 필요한 경우 사용해야 합니다.sprite_get_width().

통사론:

sprite_width;

보고:

Real

예시:

if sprite_width != sprite_get_width(sprite_index)
{
    image_xscale = 1;
}

위의 코드는 스프라이트 리소스의 너비에 대해 코드를 실행하는 인스턴스에 할당된 스프라이트의 너비를 확인하고 동일하지 않으면 재설정합니다. image_xscale 1로,
