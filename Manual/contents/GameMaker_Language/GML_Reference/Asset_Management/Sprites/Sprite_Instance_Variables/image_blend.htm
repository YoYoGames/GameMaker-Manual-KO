image_blend

이 변수는 인스턴스 스프라이트의 "색조"를 제어하고 기본값은 -1입니다(하지만 c_white ). 다른 모든 값(예: c_red, 또는 c_aqua ) 지정된 색상을 원본 스프라이트와 혼합합니다. 이 변수의 변경 사항이 표시되려면 인스턴스에 그리기 이벤트가 없어야 하므로(따라서 GameMaker 는 기본적으로 스프라이트를 그립니다) 다음과 같은 확장 그리기 기능 중 하나를 사용하여 인스턴스를 그려야 합니다.draw_self() 또는draw_sprite_ext().

HTML5 플랫폼에서 혼합을 제한해야 합니다(WebGL을 사용하지 않는 한). 혼합된 각 스프라이트는 개별적으로 캐시되어야 하고 혼합된 스프라이트가 많으면 성능에 부정적인 영향을 미칠 수 있습니다(함수를 사용하여 캐시 크기를 설정할 수도 있습니다).sprite_set_cache_size() ).

통사론:

image_blend;

보고:

Colour Constant

예시:

image_blend = make_colour_hsv(255, 255, random(255));

위의 코드는 인스턴스에 대해 정의된 스프라이트의 image_blend를 임의의 색상으로 설정합니다.
