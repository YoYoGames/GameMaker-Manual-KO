mask_index

GameMaker 에서 개체를 정의할 때 정의된 스프라이트에 해당하는 것이 아니라 충돌에 사용할 마스크를 할당할 수 있습니다. 이 변수는 다음을 찾는 데 사용할 수 있습니다.sprite_index (또는 스프라이트가 할당되지 않은 경우 -1을 반환함) 인스턴스의 마스크를 선택한 스프라이트로 설정합니다. 마스크 인덱스를 설정하면 예를 들어 불규칙한 모양의 인스턴스에 대한 스프라이트를 가질 수 있지만 다른 스프라이트에서 가져온 원형 충돌 마스크를 제공할 수 있습니다.

통사론:

mask_index;

보고:

Sprite Asset

예시:

mask_index = spr_Round;

위의 코드는 인스턴스의 마스크를 스프라이트 "spr_Round"의 마스크로 설정합니다.
