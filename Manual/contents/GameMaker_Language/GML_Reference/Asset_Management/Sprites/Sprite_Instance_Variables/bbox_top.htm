bbox_top

이 읽기 전용 변수는 인스턴스에 대한 경계 상자 상단의 공간 내(y축을 따라) 위치를 반환합니다. 여기서 경계 상자는 인스턴스에 대한 마스크의 최대 너비와 높이로 정의됩니다( 그만큼sprite_index 또는 mask_index ). 스프라이트가 정확한 충돌 마스크를 가지고 있는 경우에도 경계 상자가 존재하고 특정 항목에 사용되므로 이 변수를 사용하여 찾을 수 있습니다. 인스턴스에 할당된 스프라이트가 없는 경우 반환된 값은 인스턴스 Y 위치와 동일합니다.

통사론:

bbox_top;

보고:

(integer)

예시:

if bbox_top < 0
{
    y = sprite_yoffset;
}

위의 코드는 인스턴스의 경계 상자가 방 외부에 있는지 확인하고, 그렇다면 인스턴스를 새 위치로 설정합니다.
