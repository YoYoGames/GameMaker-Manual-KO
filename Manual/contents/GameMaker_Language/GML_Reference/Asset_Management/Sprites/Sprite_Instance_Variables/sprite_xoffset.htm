sprite_xoffset

이 읽기 전용 변수는 인스턴스에 할당된 스프라이트의 로컬 xoffset(스프라이트 편집기에 정의된 원점의 x 구성요소)을 반환합니다. 이 xoffset은 픽셀로 반환되며image_xscale 따라서 (예를 들어) 정사각형 32x32 스프라이트가 x 원점을 16으로 정의한 경우 해당 스프라이트의 크기를 2로 조정하면 이 변수는 32를 반환합니다. 크기가 조정되지 않은 xoffset이 필요한 경우 사용해야 합니다.sprite_get_xoffset().

통사론:

sprite_xoffset;

보고:

Real

예시:

if x - sprite_xoffset < 0
{
    x = sprite_xoffset;
}

위의 코드는 인스턴스 스프라이트의 왼쪽 가장자리가 방 밖에 있는지 확인하고 만약 그렇다면 전체 스프라이트가 보이도록 인스턴스를 이동합니다.
