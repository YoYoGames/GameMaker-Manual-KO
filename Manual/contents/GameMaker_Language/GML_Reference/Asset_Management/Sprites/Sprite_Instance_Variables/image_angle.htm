image_angle

이 값은 스프라이트의 각도(회전)를 설정하며 도 단위로 측정되며 오른쪽은 0º, 위쪽은 90º, 왼쪽은 180º, 아래쪽은 270º입니다. 스프라이트 편집기에 정의된 대로 그릴 스프라이트를 재설정하려면 이 변수를 0으로 설정합니다. 이 변수의 변경 사항이 표시되려면 인스턴스에 그리기 이벤트가 없어야 하므로(따라서 GameMaker 는 기본적으로 스프라이트를 그립니다) 다음과 같은 확장 그리기 기능 중 하나를 사용하여 인스턴스를 그려야 합니다.draw_self() 또는draw_sprite_ext().

통사론:

image_angle;

보고:

Real

예시:

image_angle = point_direction(x, y, mouse_x, mouse_y);

위의 코드는 인스턴스의 스프라이트가 항상 마우스 위치를 가리키도록 회전합니다.
