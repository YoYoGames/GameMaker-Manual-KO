image_number

이 읽기 전용 변수는 인스턴스에 할당된 스프라이트의 하위 이미지 수를 가져오는 데 사용할 수 있습니다(인스턴스에 할당된 것과 다른 스프라이트에 대한 하위 이미지 수가 필요한 경우 사용해야 합니다.sprite_get_number() ). 예를 들어 1개의 하위 이미지가 있는 경우 이 변수는 "1"을 반환하지만 해당 하위 이미지의 image_index는 0입니다.

통사론:

image_number;

보고:

(single precision floating point value)

예시:

if image_speed > 0
{
    if image_index > image_number - 1 instance_destroy();
}

위의 코드는 스프라이트가 애니메이션 중인지 확인하고 만약 그렇다면 현재 image_index가 하위 이미지의 수보다 큰지 확인하고 만약 그렇다면 인스턴스를 파괴합니다.
