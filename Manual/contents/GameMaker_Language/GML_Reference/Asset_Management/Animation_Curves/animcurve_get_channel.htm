animcurve_get_channel

이 함수는 애니메이션 곡선 자산에 지정된 채널에 대한 채널 데이터가 포함된 구조체 를 반환합니다. 애니메이션 곡선 ID 또는 구조체와 채널 이름 또는 인덱스를 제공하면 함수는 다음 형식의 구조체를 반환합니다.

변수 이름 데이터 형식 설명
name 끈 이것은 채널의 이름입니다.
type 애니메이션 곡선 보간 유형 상수 이것은 상수 중 하나가 될 것입니다 animcurvetype_linear 점 사이의 선형 보간을 위해, animcurvetype_catmullrom catmull-rom 보간을 사용하여 점 사이의 "부드러운" 보간 또는 animcurvetype_bezier 베지어 보간용.
iterations 정수 채널이 catmull-rom("부드러운") 보간법을 사용하는 경우 곡선의 각 세그먼트에 대해 생성된 점 수를 보유합니다(이 추가 점은 함수 내부에 있으며 런타임 계산에만 사용됨). 채널이 선형 보간을 사용하는 경우 이 값은 여전히 존재하지만 곡선이 보간되는 방식과 관련이 없으므로 무시할 수 있습니다.
points 배열 포인터 이것은 배열의 각 항목이 포인트 구조체인 배열입니다.


애니메이션 곡선 ID는 자산 브라우저에 정의된 애니메이션 곡선의 이름이거나 함수에서 반환된 구조체 포인터입니다.animcurve_create(). 채널 이름은 애니메이션 곡선 자산에 정의된 대로 채널을 참조하는 문자열이거나 0에서 n 사이의 인덱스 값을 제공할 수 있습니다. 여기서 n 은 곡선 자산의 마지막 채널입니다(예: if 애니메이션 곡선에는 4개의 채널이 있으며 0에서 3까지 인덱싱됩니다. 인덱스 값을 전달하는 것은 채널 이름을 전달하는 것보다 처리가 덜 필요합니다. 함수가 실패하면(예: 주어진 이름이나 인덱스를 가진 채널이 존재하지 않음) 오류가 발생합니다.

단일 채널의 포인트는 배열 에 구조체로 저장되며, 여기서 배열의 각 항목은 단일 포인트 구조체입니다. point 구조체에는 다음과 같은 변수가 있습니다.

변수 이름 데이터 형식 설명
posx 진짜 시점의 위치(0에서 1로 정규화됨).
value 진짜 점의 값입니다.

통사론:

animcurve_get_channel(curve_struct_or_id, channel_name_or_index);

논쟁 유형 설명
curve_struct_or_id 애니메이션 곡선 구조 The ID or struct pointer of the animation curve to target
channel_name_or_index 문자열 또는 실수 The channel name (a string) or the channel index (an integer).

보고:

Animation Curve Channel Struct

예시:

var _channeldata = animcurve_get_channel(ac_ButtonTween, 0);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
{
    _points[i].value += 1;
}

위의 코드는 곡선 자산 "ac_ButtonTween"에서 채널 0에 대한 채널 구조를 검색한 다음 채널 곡선의 점을 반복하고 값에 1을 추가합니다.
