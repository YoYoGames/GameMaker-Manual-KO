animcurve_create

이 함수는 채널 데이터로 채울 준비가 된 빈 애니메이션 곡선 구조체 를 만듭니다. 함수는 구조체에 대한 포인터를 반환하고 이것은 애니메이션 곡선에 채널 및 기타 데이터를 추가하는 데 사용됩니다. 다음과 같은 기능에 이것을 제공할 수도 있습니다.animcurve_get() 나중에 곡선에서 데이터를 가져옵니다.

이 함수를 사용할 때 생성된 구조체에는 데이터가 없으며 사용하려면 하나 이상의 채널을 추가해야 하고 채널에 포인트가 추가되어야 합니다. 채널을 추가하려면 기능을 참조하십시오.animcurve_channel_new(), 채널에 포인트를 추가하려면 함수를 참조하십시오.animcurve_point_new(). 또한 "name" 변수를 설정하여 애니메이션 곡선 구조에 이름을 지정할 수 있습니다(아래 예제 코드 참조). 새 애니메이션 곡선에 추가된 채널은 배열로 추가되어야 하며, 여기서 배열 의 각 항목은 채널 구조체를 참조하고 채널의 포인트는 각 배열 항목이 포인트 구조체를 참조하는 배열로도 추가되어야 합니다.

이러한 방식으로 생성된 애니메이션 곡선 은 더 이상 필요하지 않을 때 삭제해야 합니다. 그렇지 않으면 메모리 공간을 차지하므로 메모리 누수가 발생하여 게임 속도가 느려지고 결국에는 충돌이 발생할 수 있습니다. 함수를 사용하여 더 이상 필요하지 않은 경우 애니메이션 곡선을 제거할 수 있습니다.animcurve_destroy(). 곡선 내에서 채널이나 포인트 데이터를 정리할 필요가 없습니다. 곡선 자체가 파괴될 때 가비지 수집기가 자동으로 수행하기 때문입니다.

통사론:

animcurve_create();

보고:

Animation Curve Struct

예시:

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

위의 코드는 새로운 애니메이션 커브 구조체를 생성하고 변수 "my_curve"에 저장합니다. 이 구조체는 이름과 채널 배열로 채워집니다. 채널 배열에는 세 개의 점이 있는 단일 채널이 있습니다.
