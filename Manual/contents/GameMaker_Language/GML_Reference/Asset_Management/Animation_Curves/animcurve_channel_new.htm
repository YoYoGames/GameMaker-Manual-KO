animcurve_channel_new

이 함수는 새로운 애니메이션 곡선 채널 구조 를 만드는 데 사용할 수 있습니다. 채널 구조체에는 다음 변수가 있습니다.

애니메이션 곡선 채널 구조체
변수 이름 데이터 형식 설명
이름 끈 이것은 채널의 이름입니다.
유형 애니메이션 곡선 보간 유형 상수 이것은 아래 표에 주어진 상수 중 하나입니다.
반복 정수 채널이 catmull-rom("부드러운") 보간법을 사용하는 경우 곡선의 각 세그먼트에 대해 생성된 점 수를 보유합니다(이 추가 점은 함수 내부에 있으며 런타임 계산에만 사용됨). 채널이 선형 보간을 사용하는 경우 이 값은 여전히 존재하지만 곡선이 보간되는 방식과 관련이 없으므로 무시할 수 있습니다.
포인트들 정렬 이것은 배열의 각 항목이 Animation Curve Point Struct 인 배열입니다.

애니메이션 곡선 보간 유형 상수
일정한 설명
animcurvetype_linear 점 사이의 선형 보간에 사용됩니다.
animcurvetype_catmullrom catmull-rom 보간을 사용하여 점 간의 부드러운 보간을 위해 사용됩니다.
animcurvetype_bezier 점 사이의 베지어 보간에 사용됩니다.


기본적으로 새 채널 구조를 만들 때 "name" 변수는 빈 문자열이 되고 "type"은 animcurvetype_linear "반복"은 16이 됩니다. 이러한 모든 변수는 필요한 값으로 설정할 수 있습니다("반복" 값은 선형 곡선 유형에 영향을 주지 않음). 일단 생성되면 곡선에 점을 추가해야 합니다. 이 작업은 "points" 배열 에 다른 점 구조를 추가하여 수행됩니다. 이 점 구조는 함수를 사용하여 만들 수 있습니다.animcurve_point_new().

통사론:

animcurve_channel_new();

보고:

Animation Curve Channel Struct

예시:

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

위의 코드는 새로운 애니메이션 커브 구조체를 생성하고 변수 "my_curve"에 저장합니다. 이 구조체는 이름과 채널 배열로 채워집니다. 채널 배열에는 세 개의 점이 있는 단일 채널이 있습니다.
