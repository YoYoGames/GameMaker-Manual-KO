animcurve_point_new

이 함수는 애니메이션 곡선 채널에 추가할 새 점 구조 를 만드는 데 사용할 수 있습니다. 포인트 생성 시 구조체는 비어 있으며 포인트 데이터를 생성하려면 다음 변수를 설정해야 합니다.

애니메이션 곡선 점 구조체
변수 이름 데이터 형식 설명
posx 진짜 시점의 위치(0에서 1로 정규화됨).
value 진짜 점의 값입니다.

생성하는 각 포인트 구조체는 배열 에 추가되어야 하며 이것은 애니메이션 곡선 채널 구조체 "channels" 변수에 추가되는 배열입니다(아래 예 참조).

통사론:

animcurve_point_new();

보고:

Animation Curve Point Struct

예시:

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

위의 코드는 새로운 애니메이션 커브 구조체를 생성하고 변수 "my_curve"에 저장합니다. 이 구조체는 이름과 채널 배열로 채워집니다. 채널 배열에는 세 개의 점이 있는 단일 채널이 있습니다.
