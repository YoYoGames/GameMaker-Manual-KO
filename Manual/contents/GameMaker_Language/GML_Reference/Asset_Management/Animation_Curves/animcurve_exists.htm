animcurve_exists

이 함수는 주어진 애니메이션 곡선 자산 또는 애니메이션 곡선 구조가 존재하고 유효한 애니메이션 곡선 이면 true 를 반환하고, 그렇지 않으면 false 를 반환합니다.

통사론:

animcurve_exists(curve_struct_or_id);

논쟁 유형 설명
curve_struct_or_id 애니메이션 곡선 구조 또는 애니메이션 곡선 자산 The Animation Curve that will be checked.

보고:

Boolean

예시:

if (animcurve_exists(spring_curve))
{
    apply_spring_animation(sprite_curve);
}

위의 코드는 커스텀 spring_curve 변수에 저장된 Animation Curve가 존재하는지 확인합니다. 그렇다면 해당 애니메이션 곡선을 사용하는 사용자 지정 메서드를 실행합니다.
