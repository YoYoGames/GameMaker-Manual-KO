animcurve_get

이 함수는 주어진 애니메이션 곡선에 대한 모든 데이터를 포함하는 구조체 를 반환합니다. (자산 브라우저에 정의된 대로) 애니메이션 곡선 자산 ID를 제공하면 함수는 다음 변수가 있는 구조체를 반환합니다.

애니메이션 곡선 구조
변수 이름 데이터 형식 설명
name 끈 이것은 애니메이션 곡선의 이름입니다.
channels 애니메이션 곡선 채널 구조체 의 배열 이것은 배열의 각 항목이 채널 구조체인 배열입니다.


그만큼 channels 구조체의 변수는 각 배열 항목이 곡선 내의 채널과 관련된 데이터가 있는 애니메이션 곡선 채널 구조체 인 배열입니다. 채널 구조는 이 페이지에 설명되어 있습니다.

채널과 마찬가지로 단일 채널의 포인트는 배열의 구조체로 저장되며, 배열의 각 항목은 단일 포인트 구조체입니다. point 구조체는 이 페이지에 설명되어 있습니다.

함수가 실패하면(예: 주어진 애니메이션 곡선 자산이 존재하지 않는 경우) 함수는 -1을 반환합니다.

통사론:

animcurve_get(curve_id);

논쟁 유형 설명
curve_id 애니메이션 곡선 자산 The asset browser ID (index) of the animation curve to get.

보고:

Animation Curve Struct or -1

예시:

var _curve = animcurve_get(ac_ButtonTween);
var _channel = _curve.channels[0];
if _channel.type != animcurvetype_linear
{
    _channel.type = animcurvetype_linear;
}

위의 코드는 곡선 자산 "ac_ButtonTween"에 대한 애니메이션 곡선 구조를 검색한 다음 첫 번째 채널의 곡선 유형을 확인하고 선형이 아니면 선형으로 설정합니다.
