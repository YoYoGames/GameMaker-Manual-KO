animcurve_channel_evaluate

이 함수는 채널 구조체 에서 특정 시점의 값을 가져오는 데 사용할 수 있습니다. 채널에 대한 구조체 포인터를 제공합니다(함수에서 반환된animcurve_get_channel(), 또는 함수의 애니메이션 곡선 구조체에서 반환된 대로animcurve_get() ) 및 곡선 채널을 따라 "x"(시간) 위치를 평가합니다. 이 위치는 0과 1 사이여야 하며 함수는 해당 위치에서 곡선 값을 반환하거나 제공된 채널 구조가 유효하지 않은 경우 0을 반환합니다.

통사론:

animcurve_channel_evaluate(channel_struct, posx);

논쟁 유형 설명
channel_struct 애니메이션 곡선 채널 구조체 The struct pointer for the channel to evaluate.
posx 진짜 The position in time to check (from 0 to 1).

보고:

Real

예시:

var _channel = animcurve_get_channel(ac_AlphaCurve, 0);
var _val = animcurve_channel_evaluate(_channel, sin(current_time/1000));
image_alpha = _val;

위의 코드는 애니메이션 곡선 자산 "ac_AlphaCurve"에 대한 채널 구조를 가져옵니다. 그런 다음 반환된 평가 값을 사용하여 인스턴스의 이미지 알파를 설정합니다.
