animcurve_get_channel_index

이 함수는 주어진 애니메이션 곡선 채널에 대한 인덱스 값을 반환합니다. 애니메이션 곡선 ID 또는 구조체를 제공합니다. 여기서 곡선 ID는 자산 브라우저에 정의된 애니메이션 곡선의 이름이거나 함수에서 반환된 구조체 포인터입니다. animcurve_create(). 그런 다음 채널 이름을 문자열로 제공하면 함수가 해당 채널과 관련된 인덱스 값을 반환합니다. 곡선이나 채널이 없으면 오류가 발생합니다.

통사론:

animcurve_get_channel_index(curve_struct_or_id, channel_name);

논쟁 유형 설명
curve_struct_or_id 애니메이션 곡선 구조 The ID or struct pointer of the animation curve to target
channel_name 끈 The channel name (a string).

보고:

Real

예시:

var _channelindex = animcurve_get_channel_index(ac_ButtonTween, "x_pos")
var _channeldata = animcurve_get_channel(ac_ButtonTween, _channelindex);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
{
    _points[i].value += 1;
}

위의 코드는 곡선 자산 "ac_ButtonTween"에서 "x_pos"라는 채널에 대한 채널 구조를 검색한 다음 채널 곡선의 점을 반복하고 값에 1을 추가합니다.
