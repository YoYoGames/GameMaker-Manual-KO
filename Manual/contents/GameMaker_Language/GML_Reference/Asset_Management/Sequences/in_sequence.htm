in_sequence

이것은 게임의 모든 개체 인스턴스에 대해 생성된 인스턴스 변수 의 일부인 내장 변수 입니다. 인스턴스가 시퀀스에 의해 제어되는 경우 이 변수는 다음을 반환합니다. true, 그렇지 않으면 반환됩니다 false. 이것은 읽기 전용 변수이며 변경할 수 없습니다.

이 변수는false 제어 시퀀스가 종료된 후 시퀀스 요소가 여전히 존재하더라도 true Sequence 요소가 다시 재생 되는 경우.

플레이어(또는 CPU) 제어 개체에서 이 변수를 사용하여 시퀀스에 없을 때만 코드에 의해 이동되도록 할 수 있습니다. 이것은 시퀀스에서 동일한 인스턴스를 사용해야 하는 컷신에 유용할 수 있습니다(사용 sequence_instance_override_object() 기능) 해당 컷신이 활성화되어 있는 동안 일반적인 코드로 이동할 수 없는지 확인해야 합니다.

통사론:

in_sequence

보고:

예시:

if (!in_sequence)
{
    x += move_x;
    y += move_y;
}

위의 코드는 다음을 확인합니다. in_sequence 변수이고 만약 그렇다면 false (즉, 인스턴스가 시퀀스에 의해 제어되지 않음을 의미) 다음을 추가합니다. move_x 그리고 move_y 인스턴스의 위치로 이동하여 시퀀스에 없을 때만 이동하도록 합니다.
