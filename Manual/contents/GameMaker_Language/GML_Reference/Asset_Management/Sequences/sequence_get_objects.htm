sequence_get_objects

이 함수를 사용하면 주어진 시퀀스 내에서 인스턴스가 생성되는 모든 객체 인덱스의 배열 을 검색할 수 있습니다. 시퀀스 객체 구조체(함수에서 반환된 대로)를 제공합니다.sequence_create() 또는sequence_get() ) 또는 시퀀스 ID(함수에 의해 반환됨layer_sequence_get_sequence() 또는 시퀀스 인스턴스 구조체 속성에서 sequence ) 함수는 배열을 반환합니다. 여기서 배열의 각 항목은object_index 인스턴스를 생성하기 위해 시퀀스에서 사용되는 다른 객체에 대한 것입니다.

통사론:

sequence_get_objects(sequence_struct_or_id);

논쟁 유형 설명
sequence_struct_or_id 시퀀스 자산 또는 시퀀스 개체 구조체 The sequence object struct or ID to get the objects from.

보고:

Array

예시:

var _seq = sequence_get(seq_Logo);
obj_array = sequence_get_objects(_seq);

위의 코드는 시퀀스 개체에 대한 구조체를 가져온 다음 사용하는 개체를 검색하고 배열을 변수에 저장합니다.
