sequence_exists

이 기능을 사용하면 시퀀스 개체가 있는지 여부를 확인할 수 있습니다. 시퀀스 객체 구조체(함수에서 반환된 대로)를 제공합니다.sequence_create() 또는sequence_get() ) 또는 시퀀스 ID(함수에서 반환된layer_sequence_get_sequence() 또는 시퀀스 인스턴스 구조체 속성에서 sequence, 또는 자산 브라우저의 인덱스) 함수가 반환됩니다. true 시퀀스 객체가 존재하거나 false 그렇지 않은 경우.

통사론:

sequence_exists(sequence_struct_or_id);

논쟁 유형 설명
sequence_struct_or_id 시퀀스 자산 또는 시퀀스 개체 구조체 The sequence to check for, can be the asset reference itself or its Sequence object struct.

보고:

Boolean

예시:

if sequence_exists(my_seq)
{
    sequence_destroy(my_seq);
}

위의 코드는 주어진 시퀀스 객체가 존재하는지, 존재한다면 소멸되는지 확인합니다.
