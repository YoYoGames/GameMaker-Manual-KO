sequence_destroy

이 기능을 사용하면 동적으로 생성된 시퀀스 개체를 삭제할 수 있습니다. 시퀀스 객체 구조체(함수에서 반환된 대로)를 제공합니다.sequence_create() ) 또는 시퀀스 ID(함수에서 반환된layer_sequence_get_sequence() 또는 시퀀스 인스턴스 구조체 속성에서 sequence ). 이 함수는 동적으로 생성된 시퀀스와 연결된 메모리를 해제하는 데 더 이상 필요하지 않을 때마다 사용해야 합니다.

통사론:

sequence_destroy(sequence_struct_or_id);

논쟁 유형 설명
sequence_struct_or_id 시퀀스 자산 또는 시퀀스 개체 구조체 The sequence object struct or ID to destroy

보고:

N/A

예시:

if sequence_exists(my_seq)
{
    sequence_destroy(my_seq);
}

위의 코드는 주어진 시퀀스 객체가 존재하는지, 존재한다면 소멸되는지 확인합니다.
