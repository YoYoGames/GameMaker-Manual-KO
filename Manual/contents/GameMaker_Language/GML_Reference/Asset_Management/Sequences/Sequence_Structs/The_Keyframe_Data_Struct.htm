키프레임 데이터 구조체

트랙에 대한 키프레임 구조 를 정의할 때 트랙과 연결될 데이터도 정의해야 합니다. 이것은 각 채널이 단순히 키프레임 데이터 구조인 다른 채널 로 구성됩니다. 키프레임 데이터를 설정하려는 트랙 유형에 따라 채널에 특정 유형의 데이터가 제공될 수 있습니다.

키프레임 데이터 구조에는 다음 변수가 포함됩니다.

시퀀스 키프레임 데이터 구조체
변하기 쉬운 유형 설명
channel 진짜 키프레임 데이터가 적용되어야 하는 채널입니다. 0에서 시작하는 양의 정수 값이며 "위치" 또는 "스케일" 키프레임에 대한 매개변수 트랙을 생성할 때 매우 구체적인 채널 값을 사용해야 한다는 점에 유의해야 합니다. 채널 0은 X 위치 또는 X 스케일이고 채널 1은 Y 위치 또는 Y 스케일입니다.
spriteIndex 스프라이트 자산 트랙에 사용할 스프라이트 자산의 인덱스입니다. 이 속성은 다음 유형의 트랙에만 사용할 수 있습니다. seqtracktype_graphic 그리고 당신은 그것을 얻거나 설정할 수 있습니다.
soundIndex 사운드 자산 트랙에 사용할 오디오 자산의 인덱스입니다. 이 속성은 다음 유형의 트랙에만 사용할 수 있습니다. seqtracktype_audio 그리고 당신은 그것을 얻거나 설정할 수 있습니다.
playbackMode 시퀀스 오디오 키 상수 사운드의 재생 모드입니다. 이 속성은 다음 유형의 트랙에만 사용할 수 있습니다. seqtracktype_audio 그리고 당신은 그것을 얻거나 설정할 수 있습니다. 속성은 이 속성 아래의 테이블에 제공된 상수 중 하나여야 합니다.
curve 애니메이션 곡선 구조 이 속성에는 애니메이션 곡선 구조가 필요하며(자세한 내용은 여기 참조) 다음 유형의 트랙에만 사용할 수 있습니다.seqtracktype_real. 곡선 구조가 사용되지 않으면 이 속성의 값은 -1이 됩니다.
value 진짜 이 속성은 단순히 키프레임 데이터 채널과 연결된 값이며 다음 유형의 트랙에만 사용할 수 있습니다. seqtracktype_real 없을 때 curve 구조체가 제공됩니다. 예를 들어 "위치" 매개변수 트랙 안에 배치된 경우 트랙의 X 또는 Y 위치가 될 수 있습니다.
colour 실수 배열 이 속성은 [A, R, G, B] 형식의 키프레임 색상 값에 대한 배열 을 반환합니다(또는 설정되어 있는 경우 필수). 이것은 다음 유형의 트랙에만 사용할 수 있습니다. seqtracktype_colour. 각 구성 요소의 값은 0과 1 사이로 표현되어야 합니다. 여기서 0은 HEX 값 #00에 해당하고 1은 HEX 값 #FF에 해당합니다(시퀀스의 색상 트랙에 대한 색상 선택기에 표시된 대로 0 - 255). 편집자).
sequence 시퀀스 객체 구조체 이 속성은 시퀀스 개체 구조체 를 반환(또는 설정되는 경우 필요)하며 다음 유형의 트랙에만 사용할 수 있습니다. seqtracktype_sequence.
objectIndex 개체 자산 이 속성은 개체 인덱스를 반환(또는 설정되는 경우 필요)하며 다음 유형의 트랙에만 사용할 수 있습니다. seqtracktype_instance.
events 문자열 배열 이 속성을 사용하면 키프레임 데이터 구조체와 연결된 이벤트 및 브로드캐스트 메시지 에 액세스할 수 있습니다. 이 속성을 가져오거나 설정할 수 있으며 가져올 때 문자열 배열이 반환되고 설정할 때 문자열 배열을 지정해야 합니다. 이벤트에 대한 자세한 내용은 시퀀스 이벤트 및 순간 섹션을 참조하십시오. 이 속성은 다음 유형의 트랙에만 사용할 수 있습니다. seqtracktype_message.
event 방법 이 속성은 키프레임 데이터 구조체와 연결된 method 를 반환하거나 설정할 수 있습니다. 메서드가 지정되지 않았거나 메서드를 제거하려는 경우 속성은 -1이어야 합니다. 이 속성은 다음 유형의 트랙에만 사용할 수 있습니다. seqtracktype_moment.

텍스트 트랙 (seqtracktype_text ):

시퀀스 키프레임 데이터 구조체
변하기 쉬운 유형 설명
text 끈 트랙에 그려지는 문자열입니다.
wrap 부울 이것은 텍스트를 줄바꿈할지 여부를 나타내는 부울입니다(true ) 여부(false ).
alignmentH 시퀀스 텍스트 정렬 상수 이것은 텍스트의 수평 정렬이며 아래에 주어진 Sequence Text Alignment Constant 중 하나가 됩니다.
alignmentV 시퀀스 텍스트 정렬 상수 이것은 텍스트의 수직 정렬이며 아래에 주어진 Sequence Text Alignment Constant 중 하나가 됩니다.
fontIndex 글꼴 자산 이것은 텍스트 트랙에서 사용하는 글꼴 자산 입니다.

alignmentH 다음 상수 중 하나일 수 있습니다.

시퀀스 텍스트 정렬 상수 (alignmentH )
일정한 설명
seqtextkey_left 텍스트는 프레임의 왼쪽에 정렬됩니다.
seqtextkey_right 텍스트는 프레임의 오른쪽에 정렬됩니다.
seqtextkey_center 텍스트는 프레임 내에서 가로로 가운데에 맞춰집니다.
seqtextkey_justify 텍스트는 프레임 내에서 정렬됩니다.

alignmentV 다음 상수 중 하나일 수 있습니다.

시퀀스 텍스트 정렬 상수 (alignmentV )
일정한 설명
seqtextkey_top 텍스트는 프레임 상단에 정렬됩니다.
seqtextkey_bottom 텍스트는 프레임의 맨 아래에 정렬됩니다.
seqtextkey_middle 텍스트는 프레임 내에서 세로로 가운데에 맞춰집니다.
