키프레임 구조체

각 트랙( 트랙 구조 를 다루는 페이지에 정의됨)에는 하나 이상의 키프레임 구조 가 할당되어 있습니다(이는 keyframes track 구조체의 속성)이며 각각은 다음 속성을 가집니다.

시퀀스 키프레임 구조체
변하기 쉬운 유형 설명
frame 진짜 키프레임의 타임라인을 따른 위치(프레임 내)입니다. 기본값은 0입니다.
length 진짜 키프레임의 길이입니다. 기본값은 1이며 더 큰 값으로 설정하면 키프레임이 참조하는 트랙 속성이 지정된 길이 동안 초기 값으로 유지됩니다. 참고로 stretch 속성이 다음으로 설정된 경우 이를 재정의합니다. true.
stretch 부울 이 속성이 다음으로 설정된 경우 true 그러면 키프레임이 트랙의 다음 키프레임으로 확장되거나 트랙이 마지막 키프레임인 경우 트랙의 끝까지 확장됩니다. 이 값을 가져오거나 설정할 수 있으며 기본값은 false.
channels 시퀀스 키프레임 데이터 구조체 의 배열 이 속성을 사용하면 트랙의 채널에 대한 키프레임 데이터 구조 목록에 액세스할 수 있습니다. 이 속성을 가져오면 키프레임 데이터 구조체의 배열 이 반환되고 이 속성을 설정할 때 키프레임 데이터 구조체의 배열을 지정해야 합니다.
