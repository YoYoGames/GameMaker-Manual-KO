sequence_instance_override_object

이 기능을 사용하면 시퀀스에 사용된 개체의 모든 인스턴스를 다른 인스턴스로 재정의(교체)할 수 있습니다. 시퀀스 인스턴스 구조 ID(시퀀스 인스턴스가 룸에서 생성되거나 룸 레이어 기능 중 하나를 사용하여 반환됨 - 여기 참조) 및 개체에 대한 개체 인덱스(자산 브라우저에 정의됨)를 제공합니다. 재정의하려는 것입니다. 마지막으로 시퀀스를 재정의할 개체로 사용할 개체 인덱스 또는 인스턴스 ID를 지정합니다(인스턴스 ID를 제공하면 단순히 인스턴스가 생성된 개체를 재정의로 사용합니다). 이것은 시퀀스 인스턴스(시퀀스 개체가 아님)에서만 수행할 수 있으며 시퀀스 재생이 시작되기 전에 수행해야 합니다. 그렇지 않으면 작동하지 않습니다.

통사론:

sequence_instance_override_object(sequence_instance_struct, object_id, instance_or_object_id);

논쟁 유형 설명
sequence_instance_struct 시퀀스 인스턴스 구조체 The sequence instance struct to modify.
object_id 개체 자산 The object index of the object within the sequence to override.
instance_or_object_id 개체 자산 또는 인스턴스 ID The object index or instance ID to use to override the sequence objects.

N/A

예시:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _seq_inst = layer_sequence_get_instance(_seq);
sequence_instance_override_object(_seq_inst, obj_Trees_Winter, obj_Trees_Summer);

위의 코드는 주어진 레이어에 새로운 시퀀스 인스턴스를 생성한 다음 "obj_Trees_Winter" 개체의 모든 인스턴스가 "obj_Trees_Summer" 개체의 인스턴스로 대체되도록 수정합니다.
