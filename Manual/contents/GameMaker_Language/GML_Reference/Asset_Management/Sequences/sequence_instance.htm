sequence_instance

이것은 게임의 모든 개체 인스턴스에 대해 생성된 인스턴스 변수 의 일부인 내장 변수 입니다. 인스턴스가 시퀀스에 의해 제어되는 경우 이 변수는 인스턴스를 제어하는 시퀀스에 대한 시퀀스 인스턴스 구조체 를 보유합니다. 그렇지 않으면 undefined. 이것은 읽기 전용 변수이며 변경할 수 없습니다.

이 변수는undefined 제어 Sequence가 끝난 후 Sequence 요소가 여전히 존재하더라도 해당 Sequence 요소가 재생 되면 구조체를 다시 가져옵니다.

통사론:

sequence_instance

보고:

Sequence Instance Struct

예시:

if (in_sequence)
{
    sequence_instance.speedScale = 2;
}

위의 코드는 다음을 확인합니다.in_sequence 변수이고 만약 그렇다면 true (인스턴스가 시퀀스에 의해 제어되고 있음을 의미) 해당 시퀀스의 속도 스케일을 2로 변경합니다.
