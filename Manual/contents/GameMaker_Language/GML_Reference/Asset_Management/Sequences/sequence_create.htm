sequence_create

이 기능을 사용하면 트랙을 추가할 수 있는 새 시퀀스 개체를 만들 수 있습니다. 함수는 생성한 새 시퀀스를 설정하기 위해 액세스할 수 있는 구조체 를 반환합니다. 이 구조체의 내용은 이 페이지 에 자세히 설명되어 있습니다. 그런 다음 시퀀스 객체 구조체를 사용하여 다음 기능을 사용하여 룸 레이어에 시퀀스의 인스턴스를 생성할 수 있습니다.layer_sequence_create(). 이러한 방식으로 시퀀스 객체를 생성할 때 함수를 호출하여 다시 제거해야 합니다.sequence_destroy() 더 이상 필요하지 않을 때 메모리 누수가 발생하여 게임 속도가 느려지고 결국 충돌이 발생할 수 있습니다.

통사론:

sequence_create();

보고:

Sequence Object Struct

예시:

myseq = sequence_create();
myseq.length = 120;
myseq.loopmode = seqplay_pingpong;

위의 코드는 새 시퀀스 개체 구조체를 만들고 재생 길이와 루프 모드를 설정합니다.
