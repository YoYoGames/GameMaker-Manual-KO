font_set_cache_size

이미지 혼합 및 채색을 수행할 때 HTML5는 실행 파일이 수행되는 것과 같은 방식으로 동적으로 수행할 수 없습니다. 따라서 GameMaker 는 이미지의 혼합 복사본을 임시로 저장하고 필요할 때 로드해야 합니다. 이 기능은 이전 글꼴을 덮어쓰기 전에 주어진 글꼴의 혼합 복사본을 몇 개나 캐시할 수 있는지 설정합니다. 기본값은 4입니다.

참고 스프라이트 글꼴의 경우 스프라이트에 해당하는 기능을 사용해야 합니다.sprite_set_cache_size().

통사론:

font_set_cache_size(ind, max);

논쟁 유형 설명
ind The index of the font to change the cache size of.
max The maximum number of cached copies of the font that can be stored.

보고:

예시:

font_set_cache_size(fnt_MainMenu, 2);

이렇게 하면 "fnt_MainMenu" 변수에 인덱싱된 글꼴의 글꼴 캐시가 2개 복사본으로 설정됩니다.
