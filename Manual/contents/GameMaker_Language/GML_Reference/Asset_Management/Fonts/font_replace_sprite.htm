font_replace_sprite

이 기능을 사용하면 "스프라이트 스트립"을 사용하여 이전에 생성된 스프라이트 글꼴 자산을 대체할 수 있습니다. 여기서 각 하위 이미지는 개별 기호 또는 문자가 됩니다. 인덱스는 사용할 첫 번째 문자의 UTF8 맵 번호이며 사용할 범위는 "!"로 시작하는 일반 글꼴과 동일합니다. 그러나 필요한 것만 포함해야 하며ord() 시작 문자에 대한 올바른 UTF8 값을 가져오는 함수입니다.

글꼴의 간격을 비례하도록 설정할 수도 있습니다(true ) 여부(false ), 여기서 비례 글꼴은 문자의 실제 너비를 기준으로 간격을 두는 반면(예: 문자 "i"는 문자 "w"보다 공간을 적게 차지함) 비례하지 않는 글꼴은 하위 고정 폭 글꼴과 같은 이미지 너비. 마지막으로, 쓸 때 각 문자 사이에 공백을 남길 수 있습니다. 이것은 임의의 정수 값이 될 수 있으며 0은 공백이 없습니다(문자가 비례하는 경우 "접촉").

함수가 실패하면 주어진 스프라이트 글꼴을 바꾸지 않고 자동으로 실패합니다.

예를 들어 '.' 및 '0'이지만 '/'가 아닌 경우, 어쨌든 '/'를 포함하거나 사용된 두 문자 사이에 빈 하위 이미지를 남겨두어야 합니다. 아래 이미지는 일반적인 스프라이트 글꼴 레이아웃을 보여줍니다.

통사론:

font_replace_sprite(ind, spr, first, prop, sep);

논쟁 유형 설명
ind 글꼴 자산 The index of the font that shall be replaced (ie all references to this resource past this function will refer to the new font, not the original).
spr 스프라이트 자산 The sprite to add a font based on.
first 진짜 The index of the first available character.
prop 부울 Whether white space around each letter should be removed (if false, the created font will be monospaced, otherwise it'll be like most regular fonts).
sep 진짜 The space between each letter, 0 meaning the right of one letter just touches the left of the next one.

보고:

N/A

예시:

font_replace_sprite(global.Font, working_directory + "Sprites\spr_Font.png", ord("!"), true, 2);

위의 코드는 전역 변수 "Font"에 인덱싱된 글꼴 자산을 외부 이미지 파일의 새 글꼴 자산으로 대체합니다.
