font_replace_sprite_ext

이 함수는 이전에 함수를 사용하여 만든 스프라이트 글꼴을 대체합니다.font_add_sprite_ext(). 각 하위 이미지는 개별 기호 또는 문자이고 이미지는 함수의 "string_map" 인수를 기반으로 문자에 매핑되는 "스프라이트 스트립"을 사용합니다. 이 인수는 GameMaker 에 스프라이트 글꼴의 하위 이미지 순서를 알려주는 데 사용할 수 있는 문자열이며 텍스트를 작성할 때 자동으로 매핑합니다. 따라서 예를 들어 "AaBbCcDdEeFfGgHh"의 문자열 맵이 있는 경우 스프라이트 글꼴에는 이러한 방식으로 정렬된 하위 이미지가 있어야 합니다.

글꼴의 간격을 비례하도록 설정할 수도 있습니다(true ) 여부(false ), 여기서 비례 글꼴은 문자의 실제 너비를 기준으로 간격을 두는 반면(예: 문자 "i"는 문자 "w"보다 공간을 적게 차지함) 비례하지 않는 글꼴은 하위 고정 폭 글꼴과 같은 이미지 너비. 마지막으로, 쓸 때 각 문자 사이에 공백을 남길 수 있습니다. 이것은 임의의 정수 값이 될 수 있으며 0은 공백이 없습니다(문자가 비례하는 경우 "접촉"). 이 함수는 이 글꼴을 참조하는 모든 추가 코드에 필요하므로 변수에 저장해야 하는 인덱스 값을 반환합니다.

함수가 실패하면 주어진 스프라이트 글꼴을 바꾸지 않고 자동으로 실패합니다.

참고 이 기능을 사용하려면 이전에 다음을 사용하여 스프라이트 글꼴을 추가해야 합니다.font_add_sprite(), 대체할 것이며 게임 리소스의 일부로 포함된 글꼴에는 사용할 수 없습니다.

통사론:

font_replace_sprite_ext(font, spr, string_map, prop, sep);

논쟁 유형 설명
font 글꼴 자산 The font to replace.
spr 스프라이트 자산 The sprite to add a font based on.
string_map 끈 String from which sprite sub-image order is taken.
prop 부울 Set as proportional font or not.
sep 진짜 The space to leave between each letter.

보고:

N/A

예시:

font_replace_sprite_ext(global.Font, spr_CalcFont, "0123456789+-*/=", true, 2);

위의 코드는 "global.Font" 변수에 인덱싱된 스프라이트 글꼴 자산을 "spr_CalcFont" 변수에 인덱싱된 스프라이트 이미지로 대체합니다.
