글꼴

GameMaker 에는 포함된 글꼴 자산에 대한 정보를 얻는 데 사용할 수 있는 여러 기능과 런타임에 게임에서 글꼴을 변경, 추가 또는 제거하는 데 사용할 수 있는 추가 기능 세트가 있습니다. 특히 기능 font_add() 아시아 문자 모양 글꼴과 같은 비표준 글꼴을 추가하는 데 사용할 수 있습니다.*.ttf 또는*.otf 파일.

작성하려는 텍스트에 필요한 글리프가 없는 글꼴을 게임에 추가하는 경우 유니코드 문자 9647(▯)을 사용하여 해당 글리프를 렌더링할 때 누락된 글리프를 대체합니다. 추첨 이벤트. 예를 들어 글꼴에 º 기호가 없는 경우 90º를 쓰면 실제로 90▯이 됩니다.

다음 함수를 사용하여 글꼴 리소스에 대한 정보를 얻을 수 있습니다.

  • font_get_name
  • font_get_fontname
  • font_get_first
  • font_get_last
  • font_get_italic
  • font_get_bold
  • font_get_size
  • font_get_texture
  • font_get_uvs
  • font_get_info

다음 기능을 사용하여 게임 중에 글꼴을 변경하고 조작할 수 있습니다.

  • font_exists
  • font_set_cache_size
  • font_add_enable_aa
  • font_add_get_enable_aa
  • font_add
  • font_add_sprite
  • font_add_sprite_ext
  • font_replace
  • font_replace_sprite
  • font_replace_sprite_ext
  • font_delete
  • font_texture_page_size
