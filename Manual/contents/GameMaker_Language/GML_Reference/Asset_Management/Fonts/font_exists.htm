font_exists

이 함수는 지정된 인덱스의 글꼴이 있는지 여부를 반환합니다. 자산 브라우저에서 정의된 글꼴 인덱스 또는 다음과 같은 기능을 사용하여 추가된 글꼴을 확인할 수 있습니다. font_add().

통사론:

font_exists(ind);

논쟁 유형 설명
ind Index of the font to check.

보고:

예시:

if font_exists(fnt_Main)
{
    draw_set_font(fnt_Main);
}

활성 그리기 글꼴이 다음으로 설정됩니다. fnt_Main 존재한다면.
