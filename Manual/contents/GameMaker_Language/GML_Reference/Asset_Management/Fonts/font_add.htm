font_add

이 기능은 Included Files 에 있는 글꼴 파일에서 게임에 글꼴을 추가하는 데 사용할 수 있습니다. 글꼴 크기(포인트)와 글꼴을 굵게 또는 기울임꼴 로 정의할 수 있으며 포함할 문자 범위도 정의할 수 있습니다. 이 함수는 이 글꼴을 참조하는 모든 추가 코드에 필요하므로 변수에 저장해야 하는 인덱스 값을 반환합니다. 그렇지 않으면 -1 어떤 이유로든 기능이 실패하는 경우.

글꼴 파일은 다음 중 하나여야 합니다. *.ttf 또는*.otf 아시아 또는 아랍어 글꼴과 같은 비표준 글꼴을 추가하는 데 유용할 수도 있습니다.

경고!: 포함된 파일에 이러한 글꼴을 포함하는 경우 게임과 함께 배포 할 수 있도록 라이선스를 받아야 합니다.

HTML5 대상 모듈의 경우 이 기능은 웹 글꼴 을 추가하는 데만 사용할 수 있으며 WebGL이 활성화 되지 않은 경우에만 작동합니다. WebGL이 활성화된 경우 일반 글꼴 리소스를 사용해야 합니다(예:draw_set_font() 자산 브라우저의 글꼴과 함께) 또는 비트맵 글꼴(참조font_add_sprite() ).

중요한! HTML5를 대상으로 할 때 기울임꼴 및 굵은 텍스트는 글꼴 자산 또는 웹 글꼴 자체에 이러한 문자가 포함된 경우에만 지원됩니다. 기본 글꼴에서 합성되지 않습니다.

파일에서 글꼴을 로드할 때 글꼴 크기 는 픽셀 단위 이며 첫 번째 및 마지막 값은 무시됩니다. 즉, 글꼴의 모든 글리프가 추가됩니다. 이런 식으로 추가된 글꼴은 고유한 텍스처 페이지에 할당되므로 이 기능을 사용할 때 그림을 그릴 때 텍스처 스왑 횟수가 증가하므로 주의해야 합니다. 글리프에는 글꼴에 대해 정의된 경계 상자 외부에 그려지는 부분이 있을 수 있으므로 글꼴을 그릴 때 글꼴이 약간 더 크게 나타날 수도 있습니다. 특정 글꼴에 대해 단일 프레임에 렌더링할 수 있는 글리프가 약 200개로 제한되어 있다는 점도 알고 있어야 합니다.

기본적으로 추가된 글꼴에는 앤티앨리어싱이 활성화됩니다. 그러나 기능을 사용하여 언제든지 AA의 상태를 확인할 수 있습니다.font_add_get_enable_aa() 기능을 사용하여 AA를 활성화/비활성화할 수 있습니다.font_add_enable_aa().

참고 GameMaker 에 글꼴을 로드할 때 글꼴을 다시 제거해야 합니다(font_delete() ) 더 이상 필요하지 않을 때 메모리 누수의 위험이 있어 속도가 느려지고 결국 게임이 중단됩니다.

통사론:

font_add(name, size, bold, italic, first, last);

논쟁 유형 설명
name 끈 The path of the font file in the Included Files (*.ttf or *.otf ).
size 진짜 The size of the font - points for Web Fonts, pixels for file fonts.
bold 부울 Whether the font is bold (true ) or not (false ).
italic 부울 Whether the font is italic (true ) or not (false ).
first 진짜 The first character to include (if you're unsure, go for 32).
last 진짜 The last character to include (if you're unsure, go for 128).

보고:

Font Asset

예시:

new_font = font_add("gamefont.ttf", 24, true, true, 32, 128);

로드됩니다 "gamefont.ttf" 24pt 크기의 게임 포함 파일에 있는 파일입니다. 글꼴은 굵게 기울임꼴로 표시되며 글꼴 범위에는 대문자와 소문자, 숫자 및 모든 일반적인 구두점이 포함됩니다. 글꼴이 로드된 후 해당 ID는 new_font 변하기 쉬운.
