font_add_enable_aa

이 기능은font_add(). 이 함수는 글꼴을 추가하고 다음으로 설정 하기 전에 호출해야 합니다. true AA를 활성화하고 다음으로 설정합니다. false 비활성화합니다. 기본적으로 AA는 활성화되어 있습니다. 이것은 함수가 호출되기 전에 추가된 글꼴에는 영향을 미치지 않으며 글꼴이 사용되는 모든 그리기/단계가 아니라 글꼴이 추가될 때 함수를 한 번만 호출하면 됩니다.

통사론:

font_add_enable_aa(enable);

논쟁 유형 설명
enable Whether to enable (true ) or disable (false ) AA for added fonts.

보고:

예시:

if !font_add_get_enable_aa()
{
    font_add_enable_aa(true);
}

위의 코드는 추가된 글꼴의 앤티앨리어싱 상태를 확인하고 활성화되지 않은 경우 활성화합니다.
