font_get_name

이 함수는 글꼴이 GameMaker 자산 브라우저에 자산으로 추가될 때 글꼴에 지정된 이름(문자열으로)을 가져오는 데 사용할 수 있습니다. 이것은 문자열일 뿐이며 글꼴을 직접 참조하는 데 사용할 수 없습니다. 글꼴 색인 이 필요하기 때문입니다. 그러나 이 문자열을 사용하여 함수와 함께 반환된 문자열을 사용하여 글꼴 인덱스 를 가져올 수 있습니다.asset_get_index().

통사론:

font_get_name(ind);

논쟁 유형 설명
ind Index of the font to check.

보고:

예시:

fnt_Name = font_get_name(font0);

위의 코드는 자산 브라우저에 나타나는 글꼴 리소스의 이름을 가져와 변수 "fnt_Name"에 문자열로 저장합니다.
