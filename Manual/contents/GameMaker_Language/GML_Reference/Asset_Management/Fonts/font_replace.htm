font_replace

이 기능은 게임이 실행 중인 시스템에 설치된 글꼴에서 게임의 글꼴을 교체하는 데 사용할 수 있습니다. 대체되는 글꼴은 자산 브라우저의 글꼴이 될 수 없으며 이전에 함수를 사용하여 추가한 글꼴이어야 합니다.font_add().

이 방법으로 글꼴을 바꿀 때 글꼴 크기(포인트)와 글꼴을 볼드체 로 할지 기울임 체로 할지 정의할 수 있으며 포함할 문자 범위도 정의할 수 있습니다. 기능이 실패하면 자동으로 실패하고 지정된 글꼴이 대체되지 않습니다.

HTML5 대상 모듈의 경우 이 기능 은 Web Fonts 를 대체하는 데만 사용할 수 있으며 WebGL이 활성화 되지 않은 경우에만 작동합니다. WebGL이 활성화된 경우 일반 글꼴 리소스를 사용해야 합니다(예:draw_set_font() 자산 브라우저의 글꼴과 함께) 또는 비트맵 글꼴(참조font_add_sprite() ).

중요 HTML5를 대상으로 할 때 기울임꼴 및 굵은 텍스트는 글꼴 자산 또는 웹 글꼴 자체에 이러한 문자가 포함된 경우에만 지원됩니다. 기본 글꼴에서 합성되지 않습니다.

다음 표는 모든 브라우저에서 표준이며 문제 없이 사용할 수 있어야 하는 글꼴을 보여줍니다. 게임을 실행하는 컴퓨터나 장치에 다른 글꼴이 있을 수도 있고 없을 수도 있으므로 이 기능의 사용은 다음 표준 글꼴로 제한되어야 합니다.

다른 모든(웹이 아닌) 대상에서 이 기능을 사용하여 파일의 글꼴을 바꿀 수 있습니다. 파일은 GameMaker 의 포함된 파일 기능을 사용하여 게임 번들 에 포함되어야 하며 다음 중 하나여야 합니다. *.ttf 또는*.otf 형식 글꼴 파일, 아시아 또는 아랍어와 같은 비표준 글꼴을 추가하는 데 유용합니다.

팁 이러한 글꼴 파일을 게임에 포함하는 경우 게임과 함께 배포할 수 있도록 적절한 라이선스 가 있는지 확인하십시오.

이 방법으로 파일에서 글꼴을 로드할 때 글꼴의 크기 는 픽셀 단위 이고 첫 번째 및 마지막 값은 무시됩니다. 즉, 글꼴의 모든 글리프가 추가됩니다. 이런 식으로 추가된 글꼴은 고유한 텍스처 페이지에 할당되므로 이 기능을 사용할 때 그림을 그릴 때 텍스처 스왑 횟수가 증가하므로 주의해야 합니다. 글리프에는 글꼴에 대해 정의된 경계 상자 외부에 그려지는 부분이 있을 수 있으므로 글꼴을 그릴 때 글꼴이 약간 더 크게 나타날 수도 있습니다. 특정 글꼴에 대해 단일 프레임에 렌더링할 수 있는 글리프가 약 200개로 제한되어 있다는 점도 알고 있어야 합니다.

참고 GameMaker 에 글꼴을 로드할 때 글꼴을 다시 제거해야 합니다(font_delete() ) 더 이상 필요하지 않을 때 메모리 누수의 위험이 있어 속도가 느려지고 결국 게임이 중단됩니다.

통사론:

font_replace(ind, name, size, bold, italic, first, last);

논쟁 유형 설명
ind 글꼴 자산 The index of the (previously added) font to be replaced.
name 끈 The name of the font to be used (eg 'Arial'), or the file path if the font is an included *.ttf or *.otf file.
size 진짜 The size of the font - points for Web Fonts, pixels for file fonts.
bold 부울 Whether the font is bold (true) or not (false).
italics 부울 Whether the font is italic (true) or not (false).
first 진짜 The first character to include (if you're unsure, go for 32).
last 진짜 The last character to include (if you're unsure, go for 128).

예시:

font_replace(myfont, "Arial", 24, true, true, 32, 128);

이것은 "myfont" 변수에 색인된 글꼴을 크기가 24pt이고 "Arial"을 사용하며 굵고 기울임꼴인 새 글꼴로 대체합니다. 글꼴 범위에는 대문자와 소문자, 숫자 및 모든 일반적인 구두점이 포함됩니다.
