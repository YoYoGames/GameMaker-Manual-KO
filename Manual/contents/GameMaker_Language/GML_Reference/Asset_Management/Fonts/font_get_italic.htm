font_get_italic

이 기능을 사용하면 글꼴 자산을 확인하여 기울임꼴 플래그가 있는지 여부를 확인할 수 있습니다. 그렇게하면 함수가 반환됩니다. true, 그렇지 않으면 반환됩니다 false.

통사론:

font_get_italic(ind);

논쟁 유형 설명
ind Index of the font to check.

보고:

예시:

if font_get_italic(fnt_Main)
{
    draw_set_font(fnt_Main);
}

활성 그리기 글꼴이 다음으로 설정됩니다. fnt_Main 글꼴 속성에서 기울임꼴로 설정된 경우.
