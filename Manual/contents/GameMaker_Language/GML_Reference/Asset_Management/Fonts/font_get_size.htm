font_get_size

이 기능을 사용하면 글꼴 리소스 대화 상자에 표시되는 포인트 값인 모든 글꼴 리소스의 크기를 얻을 수 있습니다.

통사론:

font_get_size(ind);

논쟁 유형 설명
ind Index of the font to get the size of.

보고:

예시:

sz = font_get_size(font0);

이것은 "font0" 변수에 의해 인덱싱된 글꼴의 크기를 가져와 변수 "sz"에 저장합니다.
