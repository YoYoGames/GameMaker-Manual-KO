font_get_uvs

이 함수는 다음 값으로 배열을 채우는 텍스처 페이지의 글꼴 텍스처에 대한 UV 좌표가 있는 배열 을 반환합니다.

  • [0] = 왼쪽
  • [1] = 상단
  • [2] = 오른쪽
  • [3] = 바닥

그런 다음 이 값은 다른 그리기 기능, 특히 기본 및 셰이더 기능을 사용할 때 일반 그리기에서 사용할 수 있습니다.

통사론:

font_get_uvs(font);

논쟁 유형 설명
font The index of the font to use.

보고:

예시:

var tex = font_get_uvs(fnt_Main);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_left = tex[3];

위의 코드는 주어진 배경에 대한 UV 좌표를 로컬 배열에 저장한 다음 인스턴스 변수에 값을 할당합니다.
