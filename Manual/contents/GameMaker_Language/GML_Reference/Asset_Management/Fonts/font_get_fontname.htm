font_get_fontname

이 기능을 사용하면 주어진 글꼴 자산의 실제 시스템 이름을 얻을 수 있습니다. 이 함수는 인덱스 가 아닌 문자열 을 반환하며 반환되는 이름은 사용 중인 글꼴에 따라 다릅니다. 예를 들어 자산 브라우저에 " fnt_Main "이라는 글꼴 자산이 있을 수 있으며 글꼴 자체는 Windows 시스템 글꼴 Arial 일 수 있습니다. 이 경우 함수는 글꼴의 시스템 이름인 font_get_name()>"Arial" 을 반환합니다. 자산 브라우저에 표시되는 이름이 필요한 경우 다음을 사용하십시오. font_add().

위에서 설명한 동작은 IDE를 통해 추가된 글꼴 자산에만 적용됩니다. 런타임에 로드된 글꼴에 사용되는 경우(사용 ), 함수는 글꼴 이름 대신 해당 글꼴 파일의 전체 경로를 반환합니다.

통사론:

font_get_fontname(ind);

논쟁 유형 설명
ind Index of the font to check.

보고:

String

예시:

fnt_Name = font_get_fontname(font0);

위의 코드는 글꼴 리소스의 시스템 이름을 가져와 변수 "fnt_Name"에 문자열로 저장합니다.
