font_add_get_enable_aa

이 기능을 사용하여 추가한 글꼴에 대해 앤티 앨리어싱(AA)이 활성화되었는지 확인할 수 있습니다.font_add(). 함수가 반환됩니다 true AA가 활성화된 경우 false 그렇지 않은 경우. AA는 기본적으로 활성화되어 있지만 기능을 사용하여 추가된 글꼴의 AA 상태를 변경할 수 있습니다.font_add_enable_aa(), 글꼴을 추가 하기 전에 호출되는 한.

통사론:

font_add_get_enable_aa();

보고:

예시:

if !font_add_get_enable_aa()
{
    font_add_enable_aa(true);
}

위의 코드는 추가된 글꼴의 앤티앨리어싱 상태를 확인하고 활성화되지 않은 경우 활성화합니다.
