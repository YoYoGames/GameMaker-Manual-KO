script_get_name

이 함수는 이름 을 지정된 스크립트의 문자열로 반환합니다. 이 이름은 기본 GameMaker 창의 자산 브라우저에서 스크립트에 대해 지정된 이름입니다. 스크립트에 대한 자세한 내용은 스크립트 편집기 를 참조하십시오.

통사론:

script_get_name(scr);

논쟁 유형 설명
scr 스크립트 자산 The index of the script that you want to get the name of.

보고:

String

예시:

scr_name = script_get_name(Help_File);

위의 예제 코드는 "scr_name" 변수에 표시된 스크립트 인덱스의 이름을 저장합니다.
