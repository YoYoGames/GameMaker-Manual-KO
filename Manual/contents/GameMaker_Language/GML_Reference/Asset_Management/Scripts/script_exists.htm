script_exists

이 함수는 반환됩니다 true 또는 false 주어진 인덱스가 있는 스크립트 또는 스크립트 함수 가 존재하는지 여부에 따라 다릅니다. 이것은 문자열이 아니라 스크립트 자산 내에 정의된 대로 각 스크립트(IDE에 표시됨) 또는 명명된 스크립트 함수에 대한 고유 인덱스를 보유하는 자산 이름(작동하지 않음에 유의하십시오. 메서드 변수 의 경우 ). 스크립트에 대한 자세한 내용 은 스크립트 편집기 를 참조하십시오.

통사론:

script_exists(scr);

논쟁 유형 설명
scr 스크립트 자산 The script index that you want to check.

보고:

Boolean

예시:

script[0] = -1;
script[1] = AI_Left;
script[2] = AI_Right;
var script_num = choose(0, 1, 2);
if script_exists(script[script_num])
{
    script_execute(script[script_num]);
}

위의 예는 두 개의 스크립트 함수와 값을 배열에 추가한 다음 난수를 가져와 실행할 스크립트 함수를 선택하는 데 사용합니다. 단, -1을 선택하면 아무 일도 일어나지 않습니다.
