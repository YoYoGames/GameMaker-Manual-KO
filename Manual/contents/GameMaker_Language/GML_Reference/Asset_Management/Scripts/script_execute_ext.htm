script_execute_ext

이 함수는 script_execute() 함수와 유사하게 작동하며 호출되는 함수/스크립트에 필요한 인수가 포함된 배열 을 제공할 수만 있습니다. 함수에 두 개의 선택적 인수를 제공하여 인수를 가져올 배열의 오프셋과 배열에서 사용할 인수의 수를 지정할 수도 있습니다(이는 array_length - offset의 최대값이어야 함).

통사론:

script_execute_ext(scr, array_args, [offset], [num_args]);

논쟁 유형 설명
scr 스크립트 자산 또는 스크립트 기능 The function/script that you want to call
array_args 정렬 The array containing the arguments for the function/script
[offset] 진짜 OPTIONAL The offset into the argument array
[num_args] 진짜 OPTIONAL The number of arguments to use (from the offset onwards)

보고:

Variable (Will depend on the return value from the function/script being called)

예시:

script_execute_ext(move_inst, move_array, floor(random(4)), 1);

위의 예제 코드는 script_execute_ext 주어진 함수를 호출하려면 4개의 인수로 구성된 배열을 제공하지만 그 중 하나만 무작위로 사용합니다.

참고 이 기능은 내장 기능과 함께 사용할 수 없습니다.
