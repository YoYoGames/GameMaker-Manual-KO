direction

GameMaker 의 모든 인스턴스에는 모양과 동작을 제어하기 위해 사용하고 설정할 수 있는 특정 "내장된" 속성이 있습니다. Direction 이러한 속성 중 하나이며 인스턴스에 다음이 있는 경우 인스턴스의 이동 방향을 설정하는 데 사용할 수 있습니다.speed GameMaker 의 방향은 일반적으로 0°가 오른쪽, 90°가 위쪽, 180°가 왼쪽, 270°가 아래쪽으로 계산됩니다.gravity 그리고gravity_direction 변수는 수정할 수 있습니다 direction 게임에서 사용될 때의 가치입니다.

통사론:

direction;

보고:

(single precision floating point value)

예시:

if keyboard_check(vk_left) direction += 5;
if keyboard_check(vk_right) direction -= 5;

위의 코드는 어떤 키(왼쪽 또는 오른쪽)를 눌렀는지에 따라 인스턴스의 이동 방향을 변경합니다.
