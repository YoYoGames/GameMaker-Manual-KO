gravity

gravity 모든 인스턴스가 가지고 있는 "내장된" 변수 중 하나이며, 설정되면 일정한 힘을 적용합니다.gravity_direction 두 인스턴스 모두에 영향을 미치는 인스턴스의 speed  그리고 direction. 중력은 누적된 힘이며 최종 속도를 제한하지 않기로 선택하면 물체를 가속할 것이며 일반적으로 이 변수를 0.01과 같은 작은 소수 값으로 설정합니다. 설정하면 gravity 0으로 설정하면 인스턴스에 중력이 적용되지 않습니다(기본값).

통사론:

gravity;

보고:

(single precision floating point value)

예시:

if !place_meeting(x, y + 1, obj_Ground)
{
    gravity = 0.01;
}
else
{
    gravity = 0;
}

위의 코드는 인스턴스가 그 아래에 "obj_Ground"의 인스턴스를 찾지 못하는 경우에만 중력을 적용합니다.
