friction

Friction 인스턴스가 가질 수 있고 시간이 지남에 따라 인스턴스 속도를 늦추는 데 사용할 수 있는 "내장된" 속성 중 하나입니다.speed 는 0이 아닙니다. 객체의 속도가 0이 될 때까지 매 단계마다 속도에서 양을 빼면 간단히 작동하므로 마찰이 예를 들어 0.1로 설정되고 인스턴스의 속도가 1(단계당 1픽셀)이면 속도를 줄이고 10보를 지나면 정지합니다. 마찰은 양의 속도와 음의 속도에 동일하게 적용되며 최종 결과는 항상 주어진 시간 후에 물체의 속도가 0입니다.

통사론:

friction;

보고:

(single precision floating point value)

예시:

if abs(speed) > 0
{
    friction = 0.05;
}
else
{
    friction = 0;
}

위의 코드는 인스턴스의 절대 속도가 0보다 큰 경우에만 마찰을 적용합니다.
