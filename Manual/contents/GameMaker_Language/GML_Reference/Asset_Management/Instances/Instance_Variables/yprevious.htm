yprevious

이 내장 변수는 인스턴스의 이전 y 위치를 반환합니다. 이 변수는 시작 단계 이벤트 가 시작되기 직전에 현재 x 위치로 설정되지만 언제든지 코드를 통해 설정할 수도 있습니다. 즉, 필요한 경우 사용자 정의 값을 지정할 수 있습니다.

통사론:

yprevious;

보고:

(single precision floating point value)

예시:

if x != xprevious || y != yprevious
{
    moved = true;
}

위의 코드는 다음과 같은 차이점이 있는지 확인합니다. xprevious 그리고 yprevious 값과 현재 x 및 y 값. 있는 경우 변수를 true로 설정합니다.
