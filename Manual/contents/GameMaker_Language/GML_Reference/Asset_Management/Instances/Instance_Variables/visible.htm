visible

인스턴스는 다음과 같이 플래그될 수 있습니다. visible 또는 이 변수를 true (보이는) 또는 false (보이지 않는). 이것은 GameMaker 에게 이 인스턴스에 대한 그리기 이벤트를 건너뛰도록 지시하여 작동하므로 인스턴스가 표시되지 않는 동안 그리기 이벤트에 배치된 코드가 실행되지 않으므로 이를 사용할 때 주의해야 합니다. 또한 인스턴스가 할당된 layer 가 보이지 않는 것으로 플래그가 지정되면 이 변수를 다음으로 설정합니다. true 레이어 자체도 표시되는 것으로 플래그가 지정될 때까지 효과가 없습니다.

통사론:

visible;

보고:

예시:

if other.visible = true
{
    x = xprevious;
    y = yprevious;
}

위의 코드는 충돌 이벤트 에서 "기타" 인스턴스의 보이는 플래그를 확인합니다. true 해당 인스턴스를 이전 위치로 다시 이동합니다.
