vspeed

vspeed 모든 인스턴스에 있는 "기본 제공" 속성 중 하나이며 인스턴스의 수직 이동 속도(y축을 따라)를 단계당 픽셀 단위로 정의합니다. 그래서, vspeed 3의 3은 모든 단계에서 맨 아래(+y)로 이동하는 3픽셀을 의미하고, vspeed -3의 경우 모든 단계에서 상단(-y)으로 3픽셀 이동을 의미합니다.

설정하면 참고하세요. speed 및/또는 direction 변수 다음 vspeed 값은 이러한 변경 사항을 반영하도록 자동으로 업데이트되며 마찬가지로 vspeed 값도 영향을 미칩니다 speed 그리고 direction 가치. 또한 참고gravity,gravity_direction 그리고friction 게임에서 사용될 때 모두 이 변수의 값을 수정할 수 있습니다.

통사론:

vspeed;

보고:

(single precision floating point value)

예시:

if keyboard_check(vk_up) vspeed = -5;
if keyboard_check(vk_down) vspeed = 5;

위의 코드는 어떤 키를 눌렀는지에 따라 수직 속도를 변경합니다.
