x

인스턴스의 x 값은 픽셀 단위로 측정된 현재 방의 수평 위치입니다. 이 값은 0, 양수 또는 음수일 수 있습니다. 여기서 0은 방의 왼쪽이고 오른쪽으로 이동하면 x가 증가하고 왼쪽으로 이동하면 x가 감소합니다(x에 대한 음수 값은 인스턴스가 방의 왼쪽 밖으로 나갔다는 것을 의미합니다. ). xa에 12.345와 같은 실제 값을 제공할 수도 있지만 픽셀의.345를 얻을 수 없기 때문에 인스턴스가 이동한 것처럼 보이지 않을 수 있습니다. 이러한 값은 GameMaker 에서 내부적으로 유지 및 사용되며 완벽하게 유효합니다.

x(및 y) 값을 설정하여 개체가 방 주위에서 원하는 위치로 점프하도록 만들거나 더 작은 양만큼 더하거나 빼서 실제로 내장된 기능을 사용하지 않고도 움직이는 환상을 줄 수 있습니다.speed 그리고direction 기능. 게임을 만들 때 한 가지 유형의 움직임이 다른 유형보다 특정 스타일의 게임 플레이에 더 적합하기 때문에 이 점을 고려해야 합니다.

통사론:

x;

보고:

(single precision floating point value)

예시:

if keyboard_check(vk_left) x += 5;
if keyboard_check(vk_right) x -= 5;

위의 코드는 화살표 키를 사용하여 x 위치에 더하거나 빼서 움직이는 착시를 만듭니다.
