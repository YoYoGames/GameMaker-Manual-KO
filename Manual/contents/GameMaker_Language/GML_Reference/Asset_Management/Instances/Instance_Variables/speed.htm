speed

GameMaker 의 모든 인스턴스에는 모양과 동작을 제어하기 위해 사용하고 설정할 수 있는 특정 "내장된" 속성이 있습니다. speed 이러한 속성 중 하나이며 인스턴스가 모든 단계에서 이동할 픽셀 수를 정의합니다. 같지 않은hspeed 그리고vspeed, speed 에 의해 관리되기 때문에 이와 관련된 방향이 없습니다.direction 인스턴스의 값이지만 음수 값을 가질 수 있습니다. 이 경우 인스턴스는 설정한 방향과 반대 방향으로 이동합니다. direction 기능(예: direction - 180°). 참고로 설정 speed 및/또는 direction, 값도 수정합니다. hspeed 그리고 vspeed 변수와 그gravity,gravity_direction 그리고friction 게임에서 사용될 때 모두 이 변수의 값을 수정할 수 있습니다.

통사론:

speed;

보고:

(single precision floating point value)

예시:

if keyboard_check(vk_up) speed = 2;
if keyboard_check(vk_left) direction += 5;
if keyboard_check(vk_right) direction -= 5;

위의 코드는 화살표 키를 사용하여 인스턴스의 속도와 방향을 설정합니다.
