layer

이 기본 제공 변수 는 방의 모든 인스턴스에 대해 생성되며 인스턴스가 할당된 레이어의 레이어 ID 값을 포함합니다. 이 값은 다음과 같은 다른 기능에서 사용할 수 있습니다.layer_get_depth() 또는 인스턴스를 다른 레이어로 이동하도록 변경할 수 있지만 할당 중인 레이어가 현재 방에 없으면 게임을 강제로 종료하는 오류가 발생합니다. 레이어를 할당할 때 함수에서 반환한 고유한 레이어 ID 를 제공해야 합니다.layer_get_id() (명명된 방 레이어를 사용할 때) 또는 함수에서 반환된 대로layer_create() (런타임에 자신의 레이어를 생성할 때).

중요 다음을 사용하여 인스턴스를 생성한 경우instance_create_depth() 기능을 사용하거나 수동으로 변경한 경우depth 변수에 따라 인스턴스에 할당된 레이어는 GameMaker 가 자동으로 제어하고 관리하는 "관리되는" 레이어가 됩니다. 이러한 경우 layer 변수는 -1을 반환합니다.

통사론:

layer;

보고:

Layer ID

예시:

layer = layer_create(-1000);

위의 코드는 깊이가 -1000인 새 레이어를 만든 다음 인스턴스를 설정합니다. layer 변수를 반환된 레이어 ID로 변경하여 현재 있는 레이어에서 생성 중인 새 레이어로 인스턴스를 이동합니다.
