xstart

이 변수는 인스턴스가 룸에서 처음 생성될 때 인스턴스의 초기 x 위치를 저장합니다. 이것은 읽기 전용 변수가 아니며 읽기뿐만 아니라 설정도 가능합니다.

통사론:

xstart;

보고:

(single precision floating point value)

예시:

if place_meeting(x, y, obj_spike)
{
    score -= 100;
    x = xstart;
    y = ystart;
}

위의 코드는 "obj_spike" 인스턴스와의 충돌을 확인하고 충돌이 있는 경우 점수에서 100을 차감하고 인스턴스를 시작 위치로 다시 이동합니다.
