solid

개체 편집기 의 개체 속성을 통해 또는 이 기본 제공 변수의 값을 변경하여 인스턴스에 솔리드 로 플래그를 지정할 수 있습니다. 만약에 solid 로 설정됩니다 true 그런 다음 충돌이 감지되면 충돌하는 인스턴스가 충돌 이전 단계의 위치로 자동으로 반환됩니다(그런 다음 충돌 이벤트에서 모든 코드 또는 작업이 실행됨). 로 설정된 경우 false, 모든 위치 지정은 충돌 이벤트를 통해 처리되어야 합니다.

통사론:

solid;

보고:

예시:

if other.solid = true
{
    x = xprevious;
    y = yprevious;
}

위의 코드는 충돌에서 "기타" 인스턴스의 솔리드 플래그를 확인합니다. true 해당 인스턴스를 이전 위치로 다시 이동합니다.
