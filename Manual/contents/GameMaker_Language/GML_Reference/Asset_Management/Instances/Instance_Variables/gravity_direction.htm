gravity_direction

gravity_direction 모든 인스턴스가 가지고 있는 "내장된" 속성 중 하나이며 인스턴스가 있을 때 이동 방향을 설정하는 데 사용할 수 있습니다.gravity 0보다 큽니다. GameMaker 의 방향은 일반적으로 0°가 오른쪽, 90°가 위쪽, 180°가 왼쪽, 270°가 아래쪽으로 계산됩니다.

통사론:

gravity_direction;

보고:

(single precision floating point value)

예시:

if place_meeting(x, y, obj_switch)
{
    gravity_direction += 180;
}

위의 코드는 플레이어 개체가 "obj_switch" 인스턴스를 만나는 경우 중력 방향을 변경합니다.
