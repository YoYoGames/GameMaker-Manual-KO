instance_count

이 읽기 전용 변수를 사용하면 회의실에 있는 모든 활성 인스턴스의 수를 얻을 수 있습니다. 여기에는 코드를 실행하는 인스턴스가 포함되지만 인스턴스 비활성화 기능을 사용하여 비활성화된 인스턴스는 포함되지 않습니다. 이 변수는 단계 시작 시에만 인스턴스 수를 제공하므로 단계가 시작된 후 방의 인스턴스에 대한 변경 사항은 고려되지 않습니다.

통사론:

instance_count;

보고:

예시:

if (instance_count < 100)
{
    var dif = 100 - instance_count;
    while (--dif > 0)
    {
        instance_create_layer(random(room_width), random(room_height), "Effects", obj_Star);
    }
}

위의 코드는 총 인스턴스 수가 100에 도달할 때까지 "obj_Star" 개체의 여러 인스턴스를 만듭니다.
