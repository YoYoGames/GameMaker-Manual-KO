인스턴스

인스턴스 는 개체 에서 생성되며 일단 생성되면 GML 코드 및 GML 시각적 작업을 통해 변경 및 조작할 수 있습니다. 사용 가능하고 가장 일반적으로 사용되는 인스턴스와 관련된 주요 인스턴스 기능은 다음과 같습니다.

  • instance_create_layer
  • instance_create_depth
  • instance_destroy
  • instance_exists
  • instance_change
  • instance_copy
  • instance_find
  • instance_furthest
  • instance_nearest
  • instance_number
  • instance_place
  • instance_place_list
  • instance_position
  • instance_position_list

두 개의 추가 내장 변수( 전역 범위 변수)와 특정 세부 정보를 검색하는 데 유용할 수 있는 인스턴스 사용과 관련된 하나의 함수가 있습니다.

  • instance_id_get
  • instance_id
  • instance_count

인스턴스가 표시되는 방식, 다른 자산을 사용하는 방식, 충돌에 반응하는 방식 등의 다양한 측면을 제어하는 데 사용할 수 있는 객체의 각 인스턴스 내에서 사용할 수 있는 여러 내장 변수가 있습니다. 여기에 나열:

  • 인스턴스 변수

인스턴스에는 알람 이벤트 값을 가져오거나 설정하는 데 사용할 수 있는 배열 변수 가 있지만 동일한 작업을 수행하는 데 사용할 수 있는 두 가지 함수도 있습니다.

  • alarm_get
  • alarm_set

마지막으로, 인스턴스를 비활성화 할 수도 있습니다. 즉, 인스턴스는 게임 내에 여전히 존재하지만 이벤트를 처리하지 않습니다. 이와 관련된 기능은 다음 페이지에서 사용할 수 있습니다.

  • 인스턴스 비활성화
