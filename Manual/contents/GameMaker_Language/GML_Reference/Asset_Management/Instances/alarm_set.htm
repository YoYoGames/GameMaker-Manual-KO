alarm_set

이 기능을 사용하여 알람을 설정할 수 있습니다. 0에서 11까지의 알람 번호를 입력한 다음 알람을 설정할 값을 입력합니다. 값은 정수 값이어야 하며 -1로 설정하여 경보를 중지할 수 있습니다(정수가 아닌 값은 가장 가까운 정수로 반올림됨). 이것은 알람 배열 을 직접 설정하는 대체 방법입니다.

통사론:

alarm_set(index, value);

논쟁 유형 설명
index 진짜 The alarm index to set, from 0 to 11.
value 진짜 The value (an integer) to set the alarm to.

보고:

N/A

예시:

for (var i = 0; i < 12; i++)
{
    if alarm_get(i) > 0 alarm_set(i, -1);
}

위의 코드는 호출 인스턴스의 모든 알람을 확인하고 0보다 크면 -1로 설정하여 더 이상 카운트다운하지 않도록 합니다.
