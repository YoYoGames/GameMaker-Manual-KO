instance_furthest

이 함수는 주어진 개체의 모든 인스턴스를 검사하여 주어진 x/y 지점에서 가장 멀리 있는 개체를 확인합니다. 모든 검사는 지정된 x/y 위치에서 지정된 개체 인스턴스의 원점 (x/y 위치)까지 수행됩니다. 객체의 인스턴스가 없으면 함수는 키워드 noone 을 반환하지만 인스턴스가 있으면 반환합니다.id 발견된 인스턴스의 코드를 실행하는 인스턴스에 검사 중인 객체와 동일한 객체 인덱스가 있는 경우 검사에 포함됩니다(호출 인스턴스가 부모의 자식인 경우 부모 객체에 대한 검사도 포함됨).

통사론:

instance_furthest(x, y, obj);

논쟁 유형 설명
x The x position to check for instances far from.
y The y position to check for instances far from.
obj The object to check for instances of.

보고:

Instance ID or noone

예시:

var inst;
inst = instance_furthest(x, y, object_index);
if (inst != id)
{
    draw_line(x, y, inst.x, inst.y);
}

위의 코드는 코드를 실행하는 인스턴스와 동일한 객체 인덱스를 공유하는 가장 먼 인스턴스를 찾고 id를 변수에 저장합니다. 그런 다음 이 변수를 검사하여 호출 인스턴스의 ID와 동일한지 확인하고, 그렇지 않은 경우 두 인스턴스 사이에 선을 그립니다.
