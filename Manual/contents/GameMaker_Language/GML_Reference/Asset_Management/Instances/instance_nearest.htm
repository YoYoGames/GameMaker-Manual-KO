instance_nearest

이 함수는 주어진 x/y 점에 가장 가까운 것을 보기 위해 주어진 객체의 모든 인스턴스를 검사합니다. 모든 검사는 지정된 x/y 위치에서 지정된 개체 인스턴스의 원점 (x/y 위치)까지 수행됩니다. 객체의 인스턴스가 없으면 함수는 키워드를 반환합니다.noone 하지만 인스턴스가 있는 경우 반환됩니다.id 발견된 인스턴스의 코드를 실행하는 인스턴스에 검사 중인 객체와 동일한 객체 인덱스가 있는 경우 검사에 포함됩니다(호출 인스턴스가 부모의 자식인 경우 부모 객체에 대한 검사도 포함됨).

통사론:

instance_nearest(x, y, obj);

논쟁 유형 설명
x The x position to check from.
y The y position to check from.
obj The object to check for instances of.

보고:

Instance ID or noone

예시:

var inst, xx;
xx = x;
x -= 10000;
inst = instance_nearest(xx, y, object_index);
if inst != id
{
    draw_line(x, y, inst.x, inst.y);
}
x += 10000;

위의 코드는 현재 인스턴스를 10000픽셀 이동한 다음 이전 위치를 확인하여 동일한 객체 유형의 가장 가까운 인스턴스를 찾습니다. 해당 인스턴스가 자체인 경우 원래 위치로 다시 이동하는 것 외에는 아무 것도 하지 않지만 인스턴스가 다른 경우 둘 사이에 선을 그립니다.
