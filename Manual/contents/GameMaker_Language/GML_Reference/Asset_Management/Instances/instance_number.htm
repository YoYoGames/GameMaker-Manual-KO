instance_number

이 기능을 사용하면 지정된 개체의 활성 인스턴스가 방에 몇 개 있는지 확인할 수 있습니다. 이 기능을 사용하여 검사할 때 객체가 부모 이면 모든 자식 객체도 반환 값에 포함되며 인스턴스 비활성화 기능으로 비활성화된 인스턴스는 이 검사에 포함 되지 않습니다.

통사론:

instance_number(obj);

논쟁 유형 설명
obj The object to total the number of instances of.

보고:

예시:

if instance_number(object_index) < 50
{
    instance_create_layer(random(room_width), random(room_height), "Instances", object_index);
}

위의 코드는 현재 인스턴스와 동일한 객체에서 생성된 인스턴스의 수를 확인한 다음, 50개 미만인 경우 룸 내 임의의 위치에 다른 인스턴스를 생성합니다.
