instance_id

이 읽기 전용 어레이 는 모든id 룸 내의 모든 활성 인스턴스 의 s. 즉, 인스턴스 비활성화 기능 중 하나를 사용한 경우 비활성화된 인스턴스는 이 배열에 포함되지 않습니다(이전에 이 배열의 값을 사용한 경우 이제 키워드 noone 을 반환함).

통사론:

instance_id[num];

보고:

Instance ID

예시:

for (var i = 0; i < instance_count; i ++;)
{
    with (instance_id[i]) speed += 0.1;
}

위의 코드는 룸 내의 모든 인스턴스를 반복하고 속도에 0.1을 추가합니다.
