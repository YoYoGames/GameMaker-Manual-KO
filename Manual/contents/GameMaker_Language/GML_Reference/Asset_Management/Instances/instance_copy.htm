instance_copy

이 기능을 사용하면 동일한 위치에서 코드를 실행하는 인스턴스의 새 버전이 생성되므로 인스턴스를 "복제"할 수 있습니다. "perf" 인수는 이 새 인스턴스가 생성 이벤트를 수행할지 여부를 지시하는 데 사용됩니다. 이 함수는id 그런 다음 변수에 저장하거나 해당 인스턴스에 액세스하는 데 사용할 수 있습니다.

참고: 생성 이벤트를 수행하지 않도록 선택하면 인스턴스가 이 이벤트에서 초기화된 변수에 종속되는 경우 오류가 발생할 수 있습니다.

통사론:

instance_copy(perf);

논쟁 유형 설명
perf 부울 Whether to perform the new instance's Create event (true) or not (false).

보고:

Instance ID

예시:

var inst = instance_number(object_index);
if inst < 10
{
    instance_copy(true);
}

위의 코드는 지역 변수를 생성하고 그것을 사용하여 방에서 코드를 실행하는 개체의 인스턴스 수를 저장합니다. 숫자가 10보다 작으면 인스턴스가 자체 복사본을 만듭니다.
