instance_position_list

이 기능을 사용하면 다른 개체의 모든 인스턴스와의 충돌 위치를 확인할 수 있습니다. 이것을 사용하면 객체의 모든 인스턴스에 대해 방의 단일 지점을 확인하고 인스턴스의 경계 상자 또는 해당 객체에 정확한 충돌이 확인된 경우 인스턴스의 마스크에 대해 검사가 수행됩니다.

DS 목록 도 제공하므로id 호출하는 인스턴스와 충돌하는 모든 인스턴스의 값은 지정된 목록의 끝에 추가됩니다. 이 함수 호출의 결과만 포함하도록 이 함수를 호출하기 전에 목록을 지울 수 있습니다. 또한 확인 중인 위치에서 충돌이 발견된 인스턴스의 원점까지의 거리를 기준으로 목록을 정렬할 수 있는 옵션이 있습니다.

이 함수는 특수 키워드도 허용합니다.all, 이 경우 충돌하는 것으로 확인된 모든 인스턴스가 나열됩니다. 이 함수는 찾은 인스턴스의 수를 반환하거나 찾지 못한 경우 0을 반환합니다.

통사론:

instance_position_list(x, y, obj, list, ordered);

논쟁 유형 설명
x 진짜 The x position to check for instances.
y 진짜 The y position to check for instances.
obj 개체 자산 또는 인스턴스 ID The object to check for instances of.
list DS 목록 ID The DS list to use to store the IDs of colliding instances.
ordered 부울 Whether the list should be ordered by distance (true ) or not (false ).

보고:

Real (The number of instances found to be in collision)

예시:

var _list = ds_list_create();
var _num = instance_position_list(mouse_x, mouse_y, all, _list, false);
if _num > 0
{
    for (var i = 0; i < _num; ++i;)
    {
        instance_destroy(_list[| i]);
    }
}
ds_list_destroy(_list);

위의 코드는 마우스 위치에서 발견된 모든 인스턴스와의 충돌을 확인합니다. 이것들은 DS 목록에 추가되고, 충돌이 발생한 각 인스턴스를 파괴하기 위해 반복됩니다.
