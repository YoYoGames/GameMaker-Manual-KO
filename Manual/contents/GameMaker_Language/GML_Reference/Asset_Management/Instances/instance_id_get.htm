instance_id_get

이 기능을 사용하면 현재 활성 인스턴스 목록에서 모든 인스턴스의 고유 ID 값을 가져올 수 있습니다. 인스턴스 목록에 인덱스를 제공하여 ID를 가져오면 함수는 변수에 저장하기 위한 값을 반환합니다.

통사론:

instance_id_get(index);

논쟁 유형 설명
index 진짜 The index within the instance list from 0 to instance count - 1.

보고:

Instance ID

예시:

for (var i = 0; i < instance_count; ++i;)
{
    var temp_id = instance_id_get(i);
    with (temp_id)
    {
        speed += 0.1;
    }
}

위의 코드는 룸 내의 모든 인스턴스를 반복하고 속도에 0.1을 추가합니다.
