instance_exists

이 기능은 확인하고자 하는 내용에 따라 2가지 방법으로 사용할 수 있습니다. 확인할 object_index를 지정할 수 있습니다. 이 경우 이 함수는 다음을 반환합니다. true 지정된 개체의 활성 인스턴스가 현재 방에 존재하거나 인스턴스 ID를 제공할 수도 있습니다. 이 경우 이 함수는 다음을 반환합니다. true 해당 특정 인스턴스가 존재하고 현재 룸에서 활성 상태인지 여부.

이 기능은 인스턴스 비활성화 기능을 사용하여 비활성화된 인스턴스를 고려하지 않습니다.

통사론:

instance_exists(obj);

논쟁 유형 설명
obj 인스턴스 ID 또는 개체 자산 The object or instance to check for the existence of.

보고:

Boolean

예시:

if !instance_exists(obj_Enemy)
{
    score += 200;
    room_goto(rm_hiscores);
}

위의 코드는 "obj_Enemy" 개체의 인스턴스가 있는지 확인하고 없으면 "score" 변수에 추가하고 공간을 변경합니다.
