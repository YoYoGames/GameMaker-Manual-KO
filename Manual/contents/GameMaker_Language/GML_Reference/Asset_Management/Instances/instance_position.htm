instance_position

이 기능을 사용하면 객체의 다른 인스턴스 또는 모든 인스턴스와의 충돌 위치를 확인할 수 있습니다. 이것을 사용하면 방의 단일 지점에서 인스턴스 또는 개체를 확인하는 것입니다. 이 검사는 인스턴스의 경계 상자에 대해 수행되거나 해당 인스턴스에 정확한 충돌이 확인된 경우 해당 인스턴스의 마스크에 대해 수행되고 고유한 인스턴스를 반환합니다.id. 충돌하는 인스턴스의 ID가 필요하지 않은 경우 사용을 고려해야 합니다.position_meeting() 대신에. 이 기능은 특수 키워드 도 허용합니다. all 그리고 other 키워드를 반환합니다 noone 충돌이 발생하지 않은 경우 또는 충돌이 발생하는 경우 발견된 인스턴스의 고유 ID 값입니다.

통사론:

instance_position( x, y, obj );

논쟁 유형 설명
x 진짜 The x position to check for instances.
y 진짜 The y position to check for instances.
obj 개체 자산 또는 인스턴스 ID The object to check for instances of.

보고:

Instance ID or noone

예시:

var inst;
inst = instance_position(mouse_x, mouse_y, obj_Pause_Button);
if (inst != noone)
{
    with (inst) image_index=1;
    instance_create_layer(room_width / 2, 0, "Controllers", obj_Menu);
}

위의 코드는 마우스 위치에서 "obj_Pause_Button" 인스턴스와의 충돌을 확인하고, 충돌이 있는 경우 반환된 id를 사용하여 "obj_Menu" 개체의 새 인스턴스를 만들기 전에 image_index를 새 값으로 설정합니다. ".
