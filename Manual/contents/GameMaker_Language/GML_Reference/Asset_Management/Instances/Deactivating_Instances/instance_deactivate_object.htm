instance_deactivate_object

이 기능을 사용하면 이전에 활성화된 모든 인스턴스에서 특정 개체의 단일 인스턴스 또는 모든 인스턴스를 비활성화할 수 있습니다. Persistent 플래그가 지정된 인스턴스 또는 객체를 비활성화한 경우 함수를 사용하여 다시 활성화해야 합니다.instance_activate_object() 그렇지 않으면 이월 되지 않고 대신 폐기됩니다. 비활성화는 즉각적이지 않으며 이러한 방식으로 비활성화된 인스턴스는 함수가 호출된 이벤트가 끝날 때까지 비활성화된 것으로 간주되지 않습니다.

참고: 룸 시작 시 인스턴스를 비활성화하는 경우(예: 룸 생성 코드에서 또는 룸 내 인스턴스의 인스턴스 생성 이벤트에서) 룸 편집기에서 룸 내에 배치된 모든 인스턴스 는 여전히 생성 이벤트를 실행합니다. 비활성화되기 전에.

경고: 물리학이 활성화된 인스턴스를 비활성화해도 픽스쳐가 물리학 시뮬레이션 내에서 상호 작용하는 것을 중지 하지 않습니다. 그것을 위해 당신은 그들의 phy_active 변수 true 또는 false 인스턴스를 활성화/비활성화할 때.

통사론:

instance_deactivate_object(obj);

논쟁 유형 설명
obj The object or instance to deactivate (the keyword all can also be used).

보고:

예시:

instance_deactivate_object(obj_Enemy);
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

위의 코드는 "obj_Enemy" 개체의 모든 인스턴스를 비활성화한 다음 방 내의 영역을 활성화합니다.
