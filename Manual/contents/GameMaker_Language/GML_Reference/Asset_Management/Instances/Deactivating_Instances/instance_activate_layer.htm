instance_activate_layer

이 기능을 사용하면 이전에 비활성화된 레이어를 활성화할 수 있습니다. 레이어 ID 를 제공해야 합니다. 이 ID는 코드 편집기에 작성된 레이어 이름(문자열) 또는layer_create() 해당 레이어의 비활성화된 모든 인스턴스가 다시 활성화됩니다. 영구 인스턴스가 있는 계층을 비활성화한 경우 방을 변경하기 전에 이 기능을 사용하여 계층을 다시 활성화해야 합니다. 그렇지 않으면 지속 인스턴스가 이월 되지 않고 폐기됩니다. 활성화는 즉각적이지 않으며 이러한 방식으로 활성화된 인스턴스는 함수가 호출된 이벤트가 끝날 때까지 활성화된 것으로 간주되지 않습니다.

통사론:

instance_activate_layer(layer_id);

논쟁 유형 설명
layer_id 레이어 ID 또는 문자열 The layer name string (or ID value) to be used

보고:

N/A

예시:

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
instance_deactivate_region(view_xview[0] - 64, view_yview - 64, _vx + 128, _vy + 128, false, false);
instance_activate_layer("Player_Layer");

위의 코드는 룸 내의 모든 인스턴스를 활성화한 다음 마지막에 다시 활성화되는 "Player_Layer" 레이어의 인스턴스를 제외하고 현재 카메라 뷰의 한계를 벗어난 인스턴스를 비활성화합니다.
