alarm_get

이 함수는 주어진 알람의 현재 값을 얻는 데 사용할 수 있습니다. 0에서 11까지의 알람 번호를 제공하면 이 함수는 알람이 현재 켜져 있는 값을 반환합니다. 이것은 알람 배열 값을 직접 가져오는 대체 방법입니다.

통사론:

alarm_get(index);

논쟁 유형 설명
index 진짜 The alarm index to get, from 0 to 11.

보고:

Real (integer)

예시:

for (var i = 0; i < 12; i++)
{
    if alarm_get(i) > 0 alarm_set(i, -1);
}

위의 코드는 호출 인스턴스의 모든 알람을 확인하고 0보다 크면 -1로 설정하여 더 이상 카운트다운하지 않도록 합니다.
