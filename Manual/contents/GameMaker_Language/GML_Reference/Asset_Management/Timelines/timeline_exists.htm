timeline_exists

이 기능을 사용하면 타임라인이 존재하는지 확인할 수 있습니다(반환 true ) 여부(반환 false ). 이것은 타임라인을 동적으로 생성할 때 특히 유용합니다.timeline_add() 함수이지만 초기화되지 않은 변수(그렇지 않으면 타임라인의 인덱스에 할당됨)를 검색하면 오류가 발생한다는 점에 유의해야 합니다.

통사론:

timeline_exists(ind);

논쟁 유형 설명
ind The index of the time line to check for.

보고:

예시:

if timeline_exists(global.tl)
{
    timeline_delete(global.tl);
}

위의 코드는 타임 라인이 존재하고 전역 변수 "tl"에 인덱싱되어 있는지 확인하고 존재한다면 해당 타임 라인을 삭제합니다.
