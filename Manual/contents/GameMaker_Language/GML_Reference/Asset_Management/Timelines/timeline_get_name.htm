timeline_get_name

이 함수를 사용하여 타임라인의 이름을 문자열로 가져올 수 있습니다. 타임라인이 동적으로 생성된 경우timeline_add() 함수에서 반환된 이름은 "__newtimeline N " 형식을 갖습니다. 여기서 " N "은 타임라인 번호(0부터 시작)입니다. 이것은 문자열일 뿐이며 타임라인을 직접 참조하는 데 사용할 수 없습니다. 이를 위해서는 타임라인 인덱스 가 필요합니다. 그러나 이 문자열을 사용하여 함수와 함께 반환된 문자열을 사용하여 타임라인 인덱스 를 가져올 수 있습니다.asset_get_index().

통사론:

timeline_get_name(ind);

논쟁 유형 설명
ind The index of the time line to check the name of.

보고:

예시:

var _str = timeline_get_name(timeline_index);
show_debug_message("Current timeline = " + _str);

위의 코드는 현재 할당된 타임라인의 이름을 가져와 콘솔에 출력합니다.
