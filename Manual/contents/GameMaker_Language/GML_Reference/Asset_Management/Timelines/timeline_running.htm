timeline_running

이 변수는 할당된 타임라인의 현재 상태를 유지하며 다음을 반환합니다. true 실행 중이면 실행 중이고 실행 중이 아니면 false입니다. 이 변수를 다음 중 하나로 설정할 수도 있습니다. true 또는 false 언제든지 타임라인을 시작하고 중지할 수 있습니다. 중지된 타임라인은 재설정 되지 않으므로 나중에 다시 시작하면 중지된 정확한 순간부터 시작된다는 점에 유의해야 합니다.

통사론:

timeline_running;

보고:

예시:

if !timeline_running
{
    timeline_position = 0;
    timeline_running = true;
}

위의 코드는 인스턴스가 타임라인을 실행하고 있는지 확인하고, 실행되고 있지 않으면 할당된 타임라인을 처음부터 시작하도록 재설정한 다음 시작합니다.
