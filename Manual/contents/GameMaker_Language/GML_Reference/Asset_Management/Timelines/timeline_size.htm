timeline_size

이 기능을 사용하면 타임라인에 대한 총 활성 순간 수를 얻을 수 있습니다("활성" 순간은 코드 또는 GML Visual이 추가된 순간입니다). 이것은 다음을 사용하여 동적 타임라인을 만들 때 유용할 수 있습니다.timeline_moment_add_script() 그리고timeline_moment_clear() 주어진 타임라인에 올바른 수의 활성 순간이 있는지 또는 전혀 없는지 확인할 수 있기 때문입니다.

통사론:

timeline_size(ind);

논쟁 유형 설명
ind The index of the timeline get the information from.

보고:

예시:

if timeline_size(global.tl) == 0
{
    timeline_moment_add_script(global.tl, room_speed + irandom(room_speed), spawn_enemy);
}

위의 코드는 주어진 타임라인 크기를 확인하고 0을 반환하면(즉, 타임라인에 활성 모멘트가 없는 경우) 전역 변수 "tl"에 인덱싱된 타임라인 내에서 임의의 순간에 사용할 스크립트 함수를 추가합니다.
