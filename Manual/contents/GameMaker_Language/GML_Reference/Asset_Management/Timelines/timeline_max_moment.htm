timeline_max_moment

이 함수는 타임라인이 최종 작업을 수행하는 순간의 값을 반환합니다. 따라서 20분 간격으로 배치된 8개의 다른 작업이 있는 타임라인이 있는 경우 이 함수는 해당 타임라인의 마지막 작업의 순간 값(160)을 반환합니다. 이 함수는 타임라인이 통과했는지 여부를 테스트하는 좋은 방법입니다. 실행 중 마지막 활성 순간, 타임라인 위치는 추가 작업이 있든 없든 게임의 모든 단계에서 계속 증가하기 때문입니다.

통사론:

timeline_max_moment(ind);

논쟁 유형 설명
ind The index of the timeline to get the last moment of.

보고:

예시:

if timeline_position > timeline_max_moment(timeline_index)
{
    timeline_running = false;
}

위의 코드는 최대 활성 순간에 대해 현재 타임 라인 위치를 확인하고 더 크면 타임 라인이 중지됩니다.
