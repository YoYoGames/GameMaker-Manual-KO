timeline_speed

일반적으로 각 단계에서 시간 표시줄의 위치는 1씩 증가하지만 이 변수를 다른 값으로 설정하여 이 양을 변경할 수 있습니다. 실수(예: 0.5 또는 2.4)를 사용할 수 있으며 값이 1보다 크면 동일한 시간 단계 내에서 여러 순간이 발생할 수 있습니다(모두 시간 표시줄에 대해 정의된 것과 동일한 순서로 수행되므로 어떤 작업도 건너뛰지 않습니다).

통사론:

timeline_speed;

보고:

(single precision floating point value)

예시:

if !timeline_running
{
    timeline_running = true;
    timeline_speed = 0.5;
}

위의 코드는 인스턴스가 연결된 타임라인을 실행 중인지 확인하고 그렇지 않은 경우 타임라인을 절반 속도로 실행하기 시작합니다.
