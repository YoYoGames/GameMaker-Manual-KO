timeline_position

이 변수는 타임라인이 현재 있는 현재 위치(순간)를 유지합니다. 이 값을 변경하여 타임라인의 일부를 건너뛰거나 일부를 반복하거나 타임라인을 처음부터 다시 시작할 수 있습니다.

통사론:

timeline_position;

보고:

(single precision floating point value)

예시:

if !timeline_running
{
    timeline_position = 0;
    timeline_running = true;
}

위의 코드는 인스턴스가 타임라인을 실행하고 있는지 확인하고, 실행되고 있지 않으면 할당된 타임라인을 처음부터 시작하도록 재설정한 다음 시작합니다.
