timeline_delete

이 기능을 사용하면 게임에서 타임라인을 삭제할 수 있습니다. 이 타임라인이 GameMaker Asset Browser에서 생성된 경우 게임에서 완전히 제거되고 이 기능을 사용한 후 인덱스된 타임라인을 다시 호출하려고 하면 오류가 발생한다는 점에 유의하십시오. 타임라인이 동적으로 생성된 경우timeline_add() 기능을 사용하는 경우 이 기능은 게임 속도를 저하(결국 충돌)할 수 있는 메모리 누수를 방지하기 위해 타임라인이 더 이상 필요하지 않은 순간에 사용해야 합니다.

통사론:

timeline_delete(ind);

논쟁 유형 설명
ind The index of the time line to delete.

보고:

예시:

if timeline_exists(global.tl)
{
    timeline_delete(global.tl);
}

위의 코드는 타임 라인이 존재하고 전역 변수 "tl"에 인덱싱되어 있는지 확인하고 존재한다면 해당 타임 라인을 삭제합니다.
