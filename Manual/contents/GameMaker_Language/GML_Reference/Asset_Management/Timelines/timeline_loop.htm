timeline_loop

이 변수는 타임라인이 반복되는지(true) 또는 그렇지 않은지(false)를 반환합니다. 이 변수를 변경하여 루핑을 켜거나 끌 수 있으며 음의 타임 라인 속도로 작동합니다(타임 라인 위치가 0 아래로 내려가면 마지막으로 정의된 순간에 다시 시작됩니다).

통사론:

timeline_loop;

보고:

예시:

if !timeline_loop
{
    timeline_loop = true;
}

위의 코드는 현재 인스턴스에 할당된 타임라인이 반복되는지 여부를 확인하고 그렇지 않으면 true로 설정합니다.
