timeline_clear

이 기능을 사용하면 "순간"의 특정 타임라인을 지우고 해당 타임라인에 대한 모든 코드와 작업을 제거하고 비워 둘 수 있습니다.

통사론:

timeline_clear(ind);

논쟁 유형 설명
ind 타임라인 자산 The index of the timeline to clear.

보고:

N/A

예시:

if timeline_position > 200
{
    timeline_clear(global.tl);
    timeline_index = -1;
}

위의 코드는 특정 순간이 지나갔을 때 모든 순간의 "global.tl" 변수에 의해 인덱싱된 지정된 타임라인을 지울 것입니다.
