timeline_moment_add_script

이 기능을 사용하면 해당 타임라인 내의 주어진 "순간"에 스크립트 기능 을 타임라인에 동적으로 추가할 수 있습니다. 여기서 "순간"은 하나의 게임 틱(또는 단계)에 해당합니다. 이런 식으로 다음을 사용하여 새 타임라인을 만들 수 있습니다.timeline_add() 기능을 추가하고 언제든지 다른 동작을 추가하거나 단순히 이전에 생성된 타임라인 리소스를 새로운 동작으로 수정합니다. 이 함수를 사용할 때 함수는 추가 인수를 요구할 수 없으며, 이를 사용하여 자산 브라우저에 있는 타임라인 자산을 수정하면 이 타임라인을 사용하는 모든 인스턴스가 영향을 받고 변경 사항은 종료될 때까지 지속됩니다. 게임(전화 game_restart() 변경 사항도 재설정하지 않습니다).

통사론:

timeline_moment_add_script(ind, step, script);

논쟁 유형 설명
ind 타임라인 자산 The index of the timeline to add a moment to.
step 진짜 The moment (step) to add to.
script 스크립트 기능 The index of the script function to add into the moment.

보고:

N/A

예시:

global.tl = timeline_add();
var i = room_speed * 60;
repeat(3)
{
    timeline_moment_add_script(global.tl, i, choose(Attack_1, Attack_2, Attack_3);
    i += room_speed * 60;
}

위의 코드는 새로운 타임라인을 생성하고 그 인덱스를 "global.tl" 변수에 저장할 것입니다. 그런 다음 1분 간격으로 무작위로 선택된 세 개의 스크립트를 타임라인에 추가합니다.
