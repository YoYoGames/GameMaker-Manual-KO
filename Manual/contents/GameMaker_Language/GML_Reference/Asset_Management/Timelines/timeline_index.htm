timeline_index

이 변수는 현재 인스턴스와 연결된 타임라인의 인덱스를 보유합니다. 이것을 특정 타임 라인으로 설정하여 해당 타임 라인을 사용하거나 -1로 설정하여 인스턴스에 대한 타임 라인 사용을 중지할 수 있습니다(인스턴스에 대해 타임 라인이 정의되지 않은 경우 -1도 반환됨). 이것은 타임 라인을 시작하지 않는다는 점에 유의하십시오. 변수를 사용하십시오.timeline_running.

통사론:

timeline_index;

보고:

Timeline Asset

예시:

if timeline_exists(global.tl)
{
    timeline_index = global.tl;
}

위의 코드는 해당 타임 라인이 존재하는 경우 변수 "global.tl"에 인덱스된 타임 라인 코드를 실행하는 인스턴스를 할당합니다.
