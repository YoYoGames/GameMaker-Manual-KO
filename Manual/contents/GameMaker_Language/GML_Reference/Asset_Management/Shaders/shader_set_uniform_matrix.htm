shader_set_uniform_matrix

이 함수를 사용하면 셰이더 상수의 값(또는 값들)을 현재 변환 행렬로 설정할 수 있습니다( 매트릭스 함수 를 사용하여 설정됨). 이전에 함수를 사용하여 상수의 "핸들"을 얻었어야 합니다.shader_get_uniform().

참고: 모든 유니폼은 함수 호출 후 설정해야 합니다.shader_set(), 그리고 전화하기 전에shader_reset().

통사론:

shader_set_uniform_matrix(handle);

논쟁 유형 설명
handle 셰이더 균일 핸들 The handle of the shader constant to set.

보고:

N/A

예시:

shader_set(shader_Glass);
shader_matrix = shader_get_uniform(shader_glass, "u_vMatrix");
shader_set_uniform_matrix(shader_matrix);
draw_self();
shader_reset();

위의 코드는 셰이더 상수 "u_vMatrix"의 핸들을 가져온 다음 해당 상수를 현재 변환 행렬로 설정합니다.
