shader_set

이 기능을 사용하면 그리기 대상을 지정된 셰이더로 설정할 수 있으며 모든 추가 그리기는 이를 사용하여 수행됩니다. 함수를 사용하여 셰이더 사용을 종료할 수 있습니다.shader_reset().

통사론:

shader_set(shader);

논쟁 유형 설명
shader The index of the shader to use.

보고:

예시:

shader_set(shader_Glass);
draw_self();
shader_reset();

위의 코드는 그리기에 사용할 셰이더를 설정한 다음 이를 사용하여 인스턴스에 사용되는 현재 스프라이트를 그립니다.
