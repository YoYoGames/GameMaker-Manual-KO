shader_set_uniform_f

이 기능을 사용하여 셰이더 상수의 값(또는 값)을 설정할 수 있습니다. 이전에 함수를 사용하여 상수의 "핸들"을 얻었어야 합니다.shader_get_uniform(), 그리고 정확한 수의 부동 소수점 값을 전달하기 위해 어떤 유형의 상수인지 알아야 합니다. vec2 함수에 두 개의 값을 전달해야 합니다.

위 규칙의 예외는 32비트 색상 값을 처리하는 경우입니다. 최대 8개의 32비트 색상 값을 셰이더에 전달할 수 있으며 자동으로 0 - 1의 vec4 값으로 변환됩니다. 그러나 이를 위해서는 셰이더 상수 이름을 사용해야 합니다. in_Colour, 단일 색상의 경우 또는 in_Colour0, in_Colour1, 등... 까지 in_Colour7 여러 입력 값의 경우. 다음 예제는 이것이 셰이더 자체에서 어떻게 보이는지 보여줍니다.

    //Single colour
    attribute vec2 in_Position;
    attribute vec4 in_Colour;
    attribute vec2 in_TextureCoord;

    //Multiple colours
    attribute vec2 in_Position;
    attribute vec4 in_Colour0;
    attribute vec4 in_Colour1;
    attribute vec2 in_TextureCoord;

4바이트 색상 형식을 사용하여 압축된 법선 등을 수행하려는 경우에도 위에 표시된 것과 동일한 "in_Colour" 형식을 사용해야 합니다.

참고: 모든 유니폼은 함수 호출 후 설정해야 합니다.shader_set(), 그리고 전화하기 전에shader_reset().

통사론:

shader_set_uniform_f(handle, value1 [, value2, value3, value4]);

논쟁 유형 설명
handle 셰이더 균일 핸들 The handle of the shader constant to set.
value1... value4 진짜 The floating point value (or values) to set the shader constant to.

보고:

N/A

예시:

shader_set(shader_Glass);
shader_params = shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_f(shader_params, 0.1, 0.8, 0.25);
draw_self();
shader_reset();

위의 코드는 셰이더 상수 "u_vParams"( vec3 ) 그런 다음 해당 상수를 지정된 값으로 설정합니다.
