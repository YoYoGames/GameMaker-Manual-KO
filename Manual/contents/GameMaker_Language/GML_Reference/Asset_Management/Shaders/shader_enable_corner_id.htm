shader_enable_corner_id

이 기능을 사용하면 사용 중인 모든 셰이더에 대한 전역 상태를 설정할 수 있습니다. 여기에서 활성화되면 셰이더가 입력 색상 값에서 2비트를 "도용"합니다. 첫 번째는 빨간색 값의 하위 비트에서 가져오고 두 번째는 파란색 값의 하위 비트에서 가져옵니다. 그런 다음 이 값을 셰이더에서 복구하여 처리 중인 정점(즉, 모서리)을 알아낼 수 있습니다.

통사론:

shader_enable_corner_id(enable);

논쟁 유형 설명
enable Enable (true ) or disable (false ) this function.

보고:

예시:

shader_enable_corner_id(true);

위의 코드는 모든 셰이더의 모서리 ID에 색상 비트를 사용할 수 있도록 합니다.
