shader_set_uniform_i

이 기능을 사용하여 셰이더 상수의 값(또는 값)을 설정할 수 있습니다. 이전에 함수를 사용하여 상수의 "핸들"을 얻었어야 합니다.shader_get_uniform(), 그리고 정수 값의 올바른 수를 전달하려면 어떤 유형의 상수인지 알아야 합니다. vec2 함수에 두 개의 값을 전달해야 합니다.

참고: 모든 유니폼은 함수 호출 후 설정해야 합니다.shader_set(), 그리고 전화하기 전에shader_reset().

통사론:

shader_set_uniform_i(handle, value1 [, value2, value3, value4]);

논쟁 유형 설명
handle 셰이더 균일 핸들 The handle of the shader constant to set.
value1... value4 진짜 The integer value (or values) to set the shader constant to.

보고:

N/A

예시:

shader_set(shader_Glass);
shader_params = shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_i(shader_params, 0, 65, 255);
draw_self();
shader_reset();

위의 코드는 셰이더 상수 "u_vParams"( vec3 ) 그런 다음 해당 상수를 지정된 정수 값으로 설정합니다.
