shader_current

이 함수는 현재 렌더링에 사용 중인 셰이더의 인덱스 ID 값을 반환하거나 사용 중인 셰이더가 없으면 -1을 반환합니다.

통사론:

shader_current();

보고:

Shader Asset or -1 (if no shader is assigned)

예시:

if (shader_current() == -1)
{
    shader_set(sh_warp)
}

위의 코드는 현재 셰이더가 무엇인지 확인하고 -1(사용 중인 셰이더 없음)을 반환하면 셰이더가 설정된 것입니다.
