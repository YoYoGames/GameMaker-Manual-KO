shaders_are_supported

이 함수는 선택한 대상 플랫폼이 셰이더를 지원하는지 확인하고 다음을 반환합니다. true 그들이 한다면, 그리고 false 그들이하지 않는 경우. Android 에서 프로젝트에 정의된 셰이더 리소스가 없는 경우 함수는 항상 다음을 반환 합니다. false, 장치가 셰이더를 지원하는지 여부에 관계없이.

통사론:

shaders_are_supported();

보고:

예시:

global.GFX = false;
if shaders_are_supported()
{
    if (shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp))
    {
        global.GFX = true;
    }
}

위의 코드는 전역 변수를 다음으로 설정합니다. false, 그리고 플랫폼이 셰이더를 지원하고 검사 중인 셰이더가 모두 올바르게 컴파일되면 다음으로 설정됩니다. true.

뒤로: 셰이더
다음: shader_current
