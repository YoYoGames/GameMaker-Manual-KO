shader_is_compiled

이 함수는 런타임에 셰이더를 검사하여 성공적으로 컴파일되었는지 확인합니다. 그것이 있으면 함수가 반환합니다. true 그렇지 않으면 반환 false. 이 기능은 게임을 실행하는 플랫폼이 사용된 모든 셰이더를 성공적으로 컴파일했는지 확인하기 위해 게임 시작 시 사용해야 합니다(특히 일부 컴퓨터가 셰이더 수준 2.0을 사용하는 DX9를 사용하고 셰이더 수준 3.0을 사용하는 이후 버전이 아닌 Windows에서) ).

셰이더가 컴파일 되지 않은 경우shader_set() 게임이 충돌하므로 간단한 GLSL ES 셰이더 이외의 다른 것을 사용할 때마다 일종의 확인을 하는 것이 좋습니다.

통사론:

shader_is_compiled(shader);

논쟁 유형 설명
shader The shader to check.

보고:

Boolean

예시:

global.GFX = false;
if (shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp))
{
    global.GFX = true;
}

위의 코드는 전역 변수를 다음으로 설정합니다. false, 그리고 확인 중인 두 셰이더가 모두 올바르게 컴파일되면 다음으로 설정됩니다. true.
