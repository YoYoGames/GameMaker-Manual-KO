shader_set_uniform_matrix_array

이 함수를 사용하면 행렬 값의 배열을 유지하도록 셰이더 상수를 설정할 수 있습니다. 이전에 함수를 사용하여 상수의 "핸들"을 얻었어야 합니다.shader_get_uniform(), 그리고 이전에 배열을 부동 소수점 값의 배열로 초기화해야 합니다. 여기서 16개 부동 소수점의 각 연속 그룹은 4x4 행렬입니다.

참고: 모든 유니폼은 함수 호출 후 설정해야 합니다.shader_set(), 그리고 전화하기 전에shader_reset().

통사론:

shader_set_uniform_matrix_array(handle, array);

논쟁 유형 설명
handle 셰이더 균일 핸들 The handle of the shader constant to set.
array 정렬 A previously initialised array of floating point values.

보고:

N/A

예시:

shader_set(shader_Glass);
shader_params = shader_get_uniform(shader_tint, "cMatrixArray");
shader_set_uniform_matrix_array(shader_params, matrix_array);
draw_self();
shader_reset();

위의 코드는 셰이더 상수 "cMatrixArray"의 핸들을 가져온 다음 해당 상수를 지정된 배열로 설정합니다.

뒤로: 셰이더
다음: shader_reset
