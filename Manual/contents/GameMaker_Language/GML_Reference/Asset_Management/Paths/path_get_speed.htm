path_get_speed

이 함수는 경로에서 주어진 위치의 속도 계수를 반환합니다. 위치는 0과 1 사이의 값이어야 합니다.path_position, 예를 들어 인스턴스의 현재 위치가 필요한 경우) 0이 시작 위치이고 1이 끝입니다. 반환된 값은 실제 path_speed  설정 중입니다. 예를 들어, 0과 1이라는 두 개의 포인트만 있는 경로에서 포인트 0 속도 계수가 100이고 포인트 1 속도 계수가 0인 경로에서 함수는 path_get_speed(path, 0.5) 50을 반환합니다.

통사론:

path_get_speed(ind, pos);

논쟁 유형 설명
index 경로 자산 The index of the path to check.
pos 진짜 How far through the path to check. Between 0 (start) and 1 (end), can be decimal.

보고:

Real

예시:

speed = path_get_speed(pth_Patrol, 0.5) / 20;

이렇게 하면 호출 인스턴스의 속도가 지정된 경로의 정확히 중간 지점의 속도 계수로 설정되고 20으로 나뉩니다. 따라서 발견된 속도 계수가 기본 100이면 호출 인스턴스의 속도를 5로 설정합니다.
