path_get_point_speed

이 기능을 사용하면 점의 속도를 얻을 수 있습니다( 경로 편집기 에서 정의하거나 다음을 사용하여 경로 점을 동적으로 추가할 때path_add_point() ) 백분율로 표시됩니다. 따라서 경로 편집기에서 경로 포인트를 50으로 설정한 경우 이 함수를 사용하면 50이 반환됩니다.

통사론:

path_get_point_speed(index, n);

논쟁 유형 설명
index 경로 자산 The index of the path to check.
n 진짜 The point number to check.

보고:

Real

예시:

if (path_get_point_speed(pth_Patrol, 0) != 100)
{
    var _px = path_get_point_x(pth_Patrol, 0);
    var _py = path_get_point_x(pth_Patrol, 0);
    path_change_point(pth_Patrol, 0, _px, _py, 100);
}

위의 코드는 "pth_Patrol"에 의해 인덱싱된 경로의 경로 지점 "0"을 확인하여 속도가 100과 같지 않은지 확인합니다. 그렇지 않은 경우 해당 지점의 속도를 100으로 설정합니다.
