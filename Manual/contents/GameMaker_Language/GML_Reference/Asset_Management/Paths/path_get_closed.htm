path_get_closed

이 함수는 경로가 닫힌(true) 또는 열린(false) 플래그가 지정되었는지 여부를 반환하는 데 사용할 수 있습니다.

통사론:

path_get_closed(index);

논쟁 유형 설명
index 경로 자산 The index of the path to check.

보고:

Boolean

예시:

state = path_get_closed(pth_Patrol);

이것은 "상태"를 다음 중 하나로 설정합니다. true 또는 false "pth_Patrol"에 인덱싱된 경로의 닫힌 상태에 따라 다릅니다.
