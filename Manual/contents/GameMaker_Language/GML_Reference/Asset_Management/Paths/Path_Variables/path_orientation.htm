path_orientation

이 변수는 함수에서 인스턴스에 할당된 경로의 현재 방향을 유지합니다. path_start() 불렸다. 경로가 생성될 때 방향은 기본 0도이지만 이를 사용하여 이 값을 원하는 대로 설정할 수 있습니다. GameMaker 에서(물리학을 사용하지 않는 한) 각도는 시계 반대 방향으로 계산되므로 경로 방향을 90°로 설정하면 경로가 왼쪽 으로 회전합니다.

통사론:

path_orientation;

보고:

예시:

mypath = path_duplicate(choose(path_1, path_2, path_3, path_4));
path_start(path, 4, path_action_reverse, 0);
path_orientation = 90;

위의 코드는 미리 만들어진 임의의 경로 자산을 "mypath" 변수에 복제합니다. 그런 다음 이 새 경로가 시작되고 90° 회전됩니다.
