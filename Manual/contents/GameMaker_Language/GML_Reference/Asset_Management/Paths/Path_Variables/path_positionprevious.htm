path_positionprevious

이 변수는 이전 단계에서 현재 경로를 따라 인스턴스의 위치를 가져오거나 설정하는 데 사용할 수 있으며 0과 1 사이의 정규화된 값입니다. 즉, 0은 경로의 시작 위치이고 1은 끝 위치입니다.. 와 비슷하다xprevious 그리고yprevious 작동 방식의 변수이며 경로에만 해당됩니다. 방해가 되는 경우 경로 추적기를 일시적으로 중지하는 것과 같은 작업에 유용할 수 있습니다(아래 예제 코드 참조).

통사론:

path_positionprevious;

보고:

예시:

var _x = x + lengthdir_x(16, direction);
var _y = y + lengthdir_y(16, direction);
if (collision_circle(_x, _y, 16, obj_player, false, true))
{
    path_position = path_positionprevious;
}

위의 코드는 개체 "obj_Player"와의 충돌에 대해 인스턴스 앞 영역을 확인하고 충돌이 있는 경우 인스턴스를 함수에 의해 할당된 현재 경로에서 차지했던 이전 위치로 다시 설정합니다. path_start().
