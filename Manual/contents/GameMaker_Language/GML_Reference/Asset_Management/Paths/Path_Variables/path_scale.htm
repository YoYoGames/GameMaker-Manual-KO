path_scale

이 값은 인스턴스에 대해 현재 할당된 경로의 크기를 가져오거나 설정하는 데 사용할 수 있습니다(함수에서 설정한 대로 path_start() ) 기본 값은 1입니다. 이것은 스칼라 값이므로 1은 1:1의 스케일이지만, 예를 들어 2로 설정하면 스케일이 두 배가 되고 0.5로 설정하면 스케일이 절반이 됩니다.

통사론:

path_scale;

보고:

예시:

path_scale = 1 + random(2);

위의 코드는 현재 경로의 크기를 1(원래 크기)과 3(원래 크기의 3배) 사이의 임의 크기로 설정합니다.
