path_endaction

이 변수는 현재 경로의 끝에 도달했을 때 인스턴스의 반응을 가져오거나 변경하는 데 사용할 수 있습니다. 일반적으로 다음을 사용하여 경로를 시작할 때 설정합니다.path_start() 그러나 게임의 이벤트 수에 따라 이 동작을 변경할 수 있습니다. 사용 가능한 값은 다음 상수를 사용하여 표현됩니다.

경로 종료 동작 상수
일정한 설명
path_action_stop 경로를 중지
path_action_restart 시작 위치에서 계속 진행, 경로가 닫히지 않은 경우 시작 지점으로 점프
path_action_continue 현재 위치에서 다시 경로 시작
path_action_reverse 경로의 속도를 반대로 합니다(경로를 반대로 실행).

통사론:

path_endaction;

보고:

Path End Action Constant

예시:

if (path_endaction == path_action_stop)
{
    path_endaction = path_action_reverse;
}

위의 코드는 경로 종료 동작을 확인하고 중지로 설정되어 있으면 종료 동작이 반대로 변경됩니다.
