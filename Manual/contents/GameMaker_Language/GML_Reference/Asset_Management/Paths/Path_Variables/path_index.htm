path_index

GameMaker 의 모든 리소스에는 고유한 식별 번호가 있으며 path_index 다음을 사용하여 인스턴스에 할당된 지정된 경로 자산에 대한 인덱스 값을 보유하는 읽기 전용 변수입니다. path_start() 함수. 인스턴스에 경로가 할당되지 않은 경우 변수는 -1로 설정됩니다.

통사론:

path_index;

보고:

Path Asset

예시:

if (path_index == -1)
{
    path_start(pth_enemy3, 4, path_action_reverse, 0);
}

위의 코드는 경로가 인스턴스에 할당되었는지 확인하고 그렇지 않은 경우 새 경로를 시작합니다. path_index 동시에).
