path_shift

이 기능을 사용하면 가로 및 세로 축을 따라 경로를 이동할 수 있습니다. 이동 중인 경로를 따라가는 인스턴스가 경로의 위치를 무시하므로 상대적으로(절대적인 것이 아니라) 따라오는 경우 게임 내에서 가시적인 효과가 없다는 점에 유의하십시오. 이 함수는 실제 경로 자산을 변경 하므로 함수가 사용되는 순간부터 게임이 끝날 때까지 게임의 모든 인스턴스에서 경로가 사용되는 방식에 영구적으로 영향을 미칩니다. 이것이 필요한 것이 아닌 경우 다음과 같은 기능을 사용해야 합니다. path_duplicate() 경로 사본을 먼저 생성한 다음 복제된 자산에서 이 함수를 호출합니다(더 이상 필요하지 않을 때 자산에서 path_delete() 를 호출하는 것을 잊지 마십시오).

통사론:

path_shift(index, xshift, yshift);

논쟁 유형 설명
index The index of the path to change.
xshift The number of horizontal pixels to shift the path. Negative=left, positive=right.
yshift The number of vertical pixels to shift the path. Negative=up, positive=down.

보고:

예시:

my_path = path_duplicate(pth_AI);
path_shift(my_path, 50, 50);
path_start(my_path, 4, path_action_reverse, true);

위의 코드는 경로 자산을 복제하고 해당 ID 값을 변수에 저장합니다. 그런 다음 이 변수를 사용하여 복제된 경로 위치를 이동한 다음 인스턴스가 경로를 시작합니다.
