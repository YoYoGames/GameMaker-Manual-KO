path_exists

이 함수는 주어진 인덱스가 있는 경로가 있는지 여부를 반환합니다. 아직 선언되지 않은 변수를 통해 경로가 있는지 확인하면 오류가 발생합니다.

통사론:

path_exists(index);

논쟁 유형 설명
index 경로 자산 The index of the path to check for.

보고:

Boolean

예시:

if (path_exists(my_path))
{
    path_delete(my_path);
}

이 코드는 주어진 변수에 경로 인덱스가 저장되어 있는지 확인하고 저장되어 있으면 메모리에서 경로를 삭제합니다.
