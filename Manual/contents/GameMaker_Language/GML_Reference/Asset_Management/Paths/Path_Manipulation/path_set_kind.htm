path_set_kind

이 기능은 지정된 리소스가 되기를 원하는 경로의 종류를 설정하는 데 사용할 수 있습니다. 이것은 직선 경로(0으로 설정) 또는 부드러운 경로(1로 설정)일 수 있으며 이 경우 경로 정밀도도 고려해야 합니다(정밀도는 다음을 사용하여 설정할 수 있습니다.path_set_precision() ). 이 함수는 실제 경로 리소스를 변경하므로 함수가 사용되는 순간부터 게임이 끝날 때까지 게임의 모든 인스턴스에서 경로가 사용되는 방식에 영구적으로 영향을 미칩니다. 필요한 것이 아니라면 다음을 사용해야 합니다. 같은 기능 path_duplicate() 경로의 복사본을 먼저 생성한 다음 복제된 자산에서 이 함수를 호출합니다. path_delete() 자산이 더 이상 필요하지 않을 때).

통사론:

path_set_kind(index, val);

논쟁 유형 설명
index The index of the path to change.
val The kind of the path, 0 for straight or 1 for smooth.

보고:

예시:

path_set_kind(pth_Patrol, true);

이것은 주어진 경로를 "부드러운" 경로로 설정합니다.
