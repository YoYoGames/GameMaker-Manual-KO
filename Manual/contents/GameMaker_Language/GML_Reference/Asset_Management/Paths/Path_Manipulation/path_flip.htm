path_flip

이 함수는 모든 경로 점을 가져와 수평 축을 따라 뒤집습니다. 이 함수는 실제 경로 자산을 변경 하므로 함수가 사용되는 순간부터 게임이 끝날 때까지 게임의 모든 인스턴스에서 경로가 사용되는 방식에 영구적으로 영향을 미칩니다. 이것이 필요한 것이 아닌 경우 다음을 사용해야 합니다. 같은 기능 path_duplicate() 경로의 복사본을 먼저 생성한 다음 복제된 자산에서 이 함수를 호출합니다. path_delete() 자산이 더 이상 필요하지 않을 때).

통사론:

path_flip(index);

논쟁 유형 설명
index The index of the path to flip.

보고:

예시:

path_flip(mypath);

이것은 수평 축을 따라 "mypath" 변수에 색인된 경로를 뒤집을 것입니다.
