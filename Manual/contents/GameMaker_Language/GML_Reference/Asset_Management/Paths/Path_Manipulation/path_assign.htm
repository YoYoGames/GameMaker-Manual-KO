path_assign

이 기능을 사용하면 한 경로에서 다른 경로로 경로 데이터를 복사할 수 있습니다. 복사 대상 경로가 먼저 지워지고(경로 점이 있어야 함) 복사 대상 경로가 완전히 덮어씁니다. 두 경로 모두 프로세스에서 삭제되지 않으며 결과는 두 개의 다른 색인을 포함하는 두 개의 경로이지만 형태와 점은 완전히 동일합니다. 일반적으로 다음을 사용하여 만든 경로에서 이 기능을 사용하고 싶을 것입니다. path_add(), 경로 자산에서 사용하면 함수가 사용되는 순간부터 게임이 끝날 때까지 게임의 모든 인스턴스에 대한 경로에 영구적으로 영향을 미치기 때문입니다.

통사론:

path_assign(index, path);

논쟁 유형 설명
index The index of the path to be overwritten.
path The index of the path that will overwrite 'index'.

보고:

예시:

mypath = path_add();
path_assign(mypath, choose(path_1, path_2, path_3));

위의 코드는 "mypath" 변수에 인덱싱된 새 빈 경로를 만든 다음 사용 가능한 세 가지 경로 리소스 중 하나에서 경로 데이터를 복사합니다.
