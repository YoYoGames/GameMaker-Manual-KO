path_add_point

이 기능을 사용하여 지정된 경로에 점을 추가하고 속도 계수를 설정할 수 있습니다. 이 점은 점의 위치에 관계없이 경로의 끝에 추가되며 요소는 다음 비율과 같습니다.path_speed 다음 인스턴스는 경로의 이 지점에 도달할 때 실제로 이동합니다. 끝이 아닌 다른 위치에 경로 점을 배치하려면 다음을 사용해야 합니다.path_insert_point().

자산 브라우저의 경로에 사용되는 경우 함수는 실제 자산을 변경 하므로 함수가 사용되는 순간부터 게임이 끝날 때까지 게임의 모든 인스턴스에서 경로가 사용되는 방식에 영구적으로 영향을 미칩니다. 이것이 필요한 것이 아닌 경우 다음과 같은 기능을 사용해야 합니다. path_duplicate() 경로의 복사본을 먼저 생성한 다음 복제된 자산에서 이 함수를 호출합니다. path_delete() 자산이 더 이상 필요하지 않을 때).

통사론:

path_add_point(index, x, y, speed);

논쟁 유형 설명
index 경로 자산 The index of the path to add the point to.
x 진짜 The x coordinate of the new point.
y 진짜 The y coordinate of the new point.
speed 진짜 The speed factor of the point (default value is 100).

보고:

N/A

예시:

my_path = path_add();
for (var i = 0; i <= 360; i += 36;)
{
    path_add_point(my_path, x + lengthdir_x(50, i), y + lengthdir_y(50, i), 100);
}

위의 코드는 코드를 실행하는 인스턴스의 x/y 위치 주위에 원형 경로를 생성합니다.
