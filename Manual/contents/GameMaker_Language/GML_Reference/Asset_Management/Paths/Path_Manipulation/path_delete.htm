path_delete

이 코드를 사용하여 메모리에서 경로를 제거할 수 있습니다. 이 경로가 다음을 사용하여 동적으로 생성된 경우path_add(), 경로 인덱스를 보유하는 변수는 더 이상 존재하지 않으므로 경로에 액세스하는 데 더 이상 유효하지 않으며 경로가 경로 편집기 를 사용하여 생성된 경우 경로를 영구적으로 삭제하므로 전체 게임 에서 해당 경로에 더 이상 액세스할 수 없습니다..

통사론:

path_delete(index);

논쟁 유형 설명
index The index of the path to delete.

보고:

예시:

var t_path = path_add();
if mp_grid_path(grid, t_path, x, y, obj_Player.x, obj_Player.y, 1)
{
    path_assign(mypath, t_path);
}
path_delete(t_path);

위의 코드는 경로를 만들고 해당 인덱스를 로컬 변수에 저장합니다. 이 경로는 다음을 저장하는 데 사용됩니다. mp_grid_path() 생성된 경로는 대상으로 가는 경로를 찾는 데 성공하면 "mypath"에 색인된 경로에 할당됩니다. 마지막으로 "t_path"가 삭제됩니다.
