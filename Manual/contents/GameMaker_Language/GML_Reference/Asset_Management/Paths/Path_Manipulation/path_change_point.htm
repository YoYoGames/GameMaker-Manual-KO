path_change_point

이 기능을 사용하면 경로에 대해 이전에 정의된 지점의 위치 및/또는 속도 계수를 변경할 수 있습니다(경로는 경로 편집기에서 또는 다음을 사용하여 코드를 통해 생성할 수 있습니다.path_add() ). 자산 브라우저의 경로에 사용되는 경우 함수는 실제 자산을 변경 하므로 함수가 사용되는 순간부터 게임이 끝날 때까지 게임의 모든 인스턴스에서 경로가 사용되는 방식에 영구적으로 영향을 미칩니다. 이것이 필요한 것이 아닌 경우 다음과 같은 기능을 사용해야 합니다. path_duplicate() 경로의 복사본을 먼저 생성한 다음 복제된 자산에서 이 함수를 호출합니다. path_delete() 자산이 더 이상 필요하지 않을 때).

통사론:

path_change_point(index, n, x, y, speed);

논쟁 유형 설명
index 경로 자산 The index of the path to change a point of.
n 진짜 The defining point to change the attributes of.
x 진짜 The new x coordinate (relative to the path) of the point.
y 진짜 The new y coordinate (relative to the path) of the point.
speed 진짜 The new speed factor of the point.

예시:

for (var i = 0; i < path_get_number(path0); ++i;)
{
    var px = path_get_point_x(pth_Patrol, i) + 64 - random(128);
    var py = path_get_point_y(pth_Patrol, i) + 64 - random(128);
    path_change_point(pth_Patrol, i, px, py, 100);
}

위의 코드는 "path0"으로 인덱싱된 경로의 모든 점을 반복하고 모든 점을 128x128픽셀 영역 내의 임의의 위치에 재배치합니다.
