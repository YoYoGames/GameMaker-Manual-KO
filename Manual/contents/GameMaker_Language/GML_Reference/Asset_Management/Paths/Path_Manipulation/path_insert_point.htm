path_insert_point

이 기능을 사용하여 경로에 새 점을 삽입할 수 있습니다(경로 편집기에서 또는 다음을 사용하여 코드를 통해 경로를 만들 수 있습니다.path_add() ). 함수에 지정된 "n" 지점 앞의 경로에 지점이 추가됩니다.

통사론:

path_insert_point(index, n, x, y, speed);

논쟁 유형 설명
index 경로 자산 The index of the path to insert the point into.
n 진짜 The defining point to insert the new point BEFORE.
x 진짜 The x coordinate (relative to the path) of the new point.
y 진짜 The y coordinate (relative to the path) of the new point.
speed 진짜 The speed factor of the point.

보고:

N/A

예시:

path_insert_point(mypath, 0, 50, 50, 100);

이것은 100%의 속도 계수로 (50,50)에서 변수 "mypath"에 색인된 경로의 맨 처음에 점을 삽입합니다.
