path_append

이 코드를 사용하면 하나의 경로를 다른 경로에 추가하여 효과적으로 결합할 수 있으며 이것이 작동하려면 두 경로가 모두 존재해야 합니다. 일반적으로 다음을 사용하여 생성된 경로에서 이 기능을 사용하고 싶을 것입니다. path_add(), 경로 자산에서 사용하면 함수가 사용되는 순간부터 게임이 끝날 때까지 게임의 인스턴스 경로에 영구적으로 영향을 미치기 때문입니다.

참고 게임에서 추가되는 경로는 제거되지 않습니다. 그것은 여전히 존재하며 지금은 포인트가 없으며 더 이상 사용하지 않으려면 기능으로 제거해야합니다path_delete().

통사론:

path_append(index, path);

논쟁 유형 설명
index The new index of the path.
path The index of the path that will be appended to 'index'.

보고:

N/A

예시:

path_append(mypath, mppath);
path_delete(mppath);

위의 코드는 "mppath" 변수에 인덱싱된 경로 데이터를 "mypath" 변수에 인덱싱된 경로에 추가한 다음 메모리에서 "mppath"를 삭제합니다.
