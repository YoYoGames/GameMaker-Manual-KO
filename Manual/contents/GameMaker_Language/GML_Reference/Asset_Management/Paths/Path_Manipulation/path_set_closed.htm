path_set_closed

이 함수는 주어진 경로가 열려 있거나(false) 닫혀 있는(true) 플래그를 지정하는 데 사용할 수 있습니다. 닫힌 경로는 시작점이 끝점에 연결되어 루프를 형성하고 열린 경로에는 연결되지 않은 시작과 끝이 있습니다. 이 함수는 실제 경로 자산을 변경 하므로 함수가 사용되는 순간부터 게임이 끝날 때까지 게임의 모든 인스턴스에서 경로가 사용되는 방식에 영구적으로 영향을 미칩니다. 이것이 필요한 것이 아닌 경우 다음을 사용해야 합니다. 같은 기능 path_duplicate() 경로의 복사본을 먼저 생성한 다음 복제된 자산에서 이 함수를 호출합니다. path_delete() 자산이 더 이상 필요하지 않을 때).

통사론:

path_set_closed(index, closed);

논쟁 유형 설명
index 경로 자산 The index of the path to change.
closed 부울 Whether the path is closed (true) or not (false).

N/A

예시:

path_set_closed(pth_Patrol, true);

이것은 주어진 경로를 닫힌 경로로 설정합니다.
