path_clear_points

이 기능을 사용하면 경로에서 모든 점 정의를 제거하여 "빈" 경로를 효과적으로 만들 수 있습니다. 이것은 당신이 사용해야하는 경로를 삭제 하지 않습니다path_delete(), 그러나 이 함수는 실제 경로 자산을 변경 하므로 함수가 사용되는 순간부터 게임이 끝날 때까지 게임의 모든 인스턴스에서 경로가 사용되는 방식에 영구적으로 영향을 미친다는 점에 유의해야 합니다. 이것이 필요한 것이 아닌 경우 다음과 같은 기능을 사용해야 합니다. path_duplicate() 경로의 복사본을 먼저 생성한 다음 복제된 자산에서 이 함수를 호출합니다. path_delete() 자산이 더 이상 필요하지 않을 때).

통사론:

path_clear_points(index);

논쟁 유형 설명
index The index of the path to clear.

보고:

예시:

if path_get_number(mypath) > 0
{
    path_clear_points(mypath);
}

위의 코드는 "mypath" 변수에 색인된 경로에 점이 있는지 확인하고 있으면 경로를 지웁니다.
