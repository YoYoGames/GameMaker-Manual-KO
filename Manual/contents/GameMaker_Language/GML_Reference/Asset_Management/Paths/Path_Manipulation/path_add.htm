path_add

이 기능을 사용하면 경로 편집기를 사용하지 않고 GameMaker 에서 경로를 생성할 수 있습니다. 이 함수는 변수에 저장되어야 하는 경로의 인덱스를 반환하고 이후부터 해당 경로에 대한 참조로 사용됩니다. 생성된 경로는 비어 있습니다. 즉, 정의된 점이 없으므로 경로에 점을 추가하거나 MP 그리드 를 사용하여 경로를 생성하려면 다른 사용 가능한 기능 을 사용해야 합니다. 경로 사용을 마쳤거나 새 경로를 만들고 인덱스를 동일한 변수에 저장하려면 먼저 이전 경로를 삭제해야 합니다.path_delete 결국 게임을 중단시킬 수 있는 메모리 누수를 방지합니다.

통사론:

path_add();

보고:

Path Asset

예시:

global.newpath = path_add();

그러면 새 경로가 생성되고 해당 인덱스가 global.newpath에 할당됩니다.
