path_set_precision

이 기능을 사용하여 경로의 "부드러움"을 변경할 수 있습니다. 이 값은 1에서 8 사이여야 하며, 값이 낮을수록 모서리가 더 똑바르고 점 사이에 더 날카로운 곡선이 생성되고 값이 높을수록 점을 둥글게 만들고 경로를 훨씬 더 "곡선"하게 만듭니다. 경로 편집기에서 경로가 매끄럽게 설정되지 않았거나 함수를 사용하여 이 기능은 가시적인 효과가 없습니다.path_set_kind(). 이 함수는 실제 경로 자산을 변경 하므로 함수가 사용되는 순간부터 게임이 끝날 때까지 게임의 모든 인스턴스에서 경로가 사용되는 방식에 영구적으로 영향을 미칩니다. 이것이 필요한 것이 아닌 경우 다음과 같은 기능을 사용해야 합니다. path_duplicate() 경로의 복사본을 먼저 생성한 다음 복제된 자산에서 이 함수를 호출합니다. path_delete() 자산이 더 이상 필요하지 않을 때).

통사론:

path_set_precision(index, prec);

논쟁 유형 설명
index 경로 자산 The index of the path to change.
prec 진짜 The precision of the path. Must be an integer between 1 and 8.

N/A

예시:

path_set_precision(pth_Patrol, 2);

그러면 주어진 경로의 정밀도가 2로 설정됩니다.
