path_rescale

이 기능은 세로 및 가로 축 모두(또는 둘 중 하나)를 따라 지정된 경로의 크기를 다시 조정하는 데 사용할 수 있으며 기본적으로 각 경로 포인트를 경로 중심 주위의 이 비율에 해당하는 새 위치로 이동합니다. 이 함수는 실제 경로 자산을 변경 하므로 함수가 사용되는 순간부터 게임이 끝날 때까지 게임의 모든 인스턴스에서 경로가 사용되는 방식에 영구적으로 영향을 미칩니다. 이것이 필요한 것이 아닌 경우 다음과 같은 기능을 사용해야 합니다. path_duplicate() 경로의 복사본을 먼저 생성한 다음 복제된 자산에서 이 함수를 호출합니다. path_delete() 자산이 더 이상 필요하지 않을 때).

통사론:

path_rescale(index, xscale, yscale);

논쟁 유형 설명
index The index of the path to scale.
xscale What to multiply the current horizontal scale by. Default scale is 1.
yscale What to multiply the current vertical scale by. Default scale is 1.

보고:

예시:

path_rescale(pth_AI, 3, 3);

이렇게 하면 지정된 경로가 원래 크기의 3배로 다시 조정됩니다.
