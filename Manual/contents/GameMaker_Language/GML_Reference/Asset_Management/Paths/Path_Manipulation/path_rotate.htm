path_rotate

이 기능을 사용하여 중심을 중심으로 지정된 경로를 회전할 수 있습니다. GameMaker 에서(물리학을 사용하지 않는 한) 각도는 시계 반대 방향으로 계산되므로 경로를 90도 회전하면 경로가 왼쪽 으로 회전합니다. 이 함수는 실제 경로 리소스를 변경 하므로 함수가 사용되는 순간부터 게임이 끝날 때까지 게임의 모든 인스턴스에서 경로가 사용되는 방식에 영구적으로 영향을 미칩니다. 이것이 필요한 것이 아닌 경우 다음과 같은 기능을 사용해야 합니다. path_duplicate() 경로의 복사본을 먼저 생성한 다음 복제된 자산에서 이 함수를 호출합니다. path_delete() 자산이 더 이상 필요하지 않을 때).

참고: 경로가 인스턴스에 할당되면path_start() 기능, 당신은 사용할 수 있습니다 path_orientation 경로 회전도 변경하는 변수입니다.

통사론:

path_rotate(index, angle);

논쟁 유형 설명
index 경로 자산 The index of the path to flip.
angle 진짜 The angle of rotation degrees.

보고:

N/A

예시:

path_rotate(pth_Patrol, 90);

이것은 주어진 90도를 시계 반대 방향으로 회전시킬 것입니다.
