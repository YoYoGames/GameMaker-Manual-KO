path_reverse

이 기능을 사용하면 개별 경로 포인트에 번호가 매겨진 순서를 바꿀 수 있습니다. 예를 들어 경로에 5개의 포인트가 있는 경우 포인트 0은 포인트 4가 되고 포인트 1은 포인트 3이 되고 포인트 2는 변경되지 않습니다. 포인트의 실제 위치는 동일하게 유지되며 처리되는 순서만 변경됩니다. 이 함수는 실제 경로 자산을 변경 하므로 함수가 사용되는 순간부터 게임이 끝날 때까지 게임의 모든 인스턴스에서 경로가 사용되는 방식에 영구적으로 영향을 미칩니다. 이것이 필요한 것이 아닌 경우 다음과 같은 기능을 사용해야 합니다. path_duplicate() 경로의 복사본을 먼저 생성한 다음 복제된 자산에서 이 함수를 호출합니다. path_delete() 자산이 더 이상 필요하지 않을 때).

통사론:

path_reverse(index);

논쟁 유형 설명
index The index of the path to change.

보고:

예시:

path_reverse(pth_AI);

이렇게 하면 지정된 경로의 모든 점이 처리되는 순서가 반대로 됩니다.
