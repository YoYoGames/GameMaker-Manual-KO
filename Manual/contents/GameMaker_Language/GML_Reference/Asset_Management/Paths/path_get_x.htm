path_get_x

이 함수를 사용하면 주어진 경로에서 위치의 x 좌표를 얻을 수 있습니다. 0은 경로의 시작이고, 1은 경로의 끝이며, 그 사이의 모든 것은 경로를 통해 멀리 있는 것과 같습니다. 이것은 경로의 정의 지점일 필요가 없으며 경로의 어느 곳에나 있을 수 있습니다.

통사론:

path_get_x(ind, pos);

논쟁 유형 설명
index 경로 자산 The index of the path to check.
pos 진짜 How far through the path to check. Between 0 (start) and 1 (end).

보고:

Real

예시:

x = path_get_x(pth_WallWalk, 0.5);

이것은 호출 인스턴스의 x를 주어진 경로의 정확히 중간 지점에 있는 점의 x 좌표로 설정합니다.
