path_get_number

이 함수는 경로의 포인트 수를 반환하는 데 사용할 수 있습니다.

통사론:

path_get_number(index);

논쟁 유형 설명
index The index of the path to check.

보고:

예시:

if (path_get_number(pth_AI) > 1)
{
    path_start( pth_AI, 4, 3, 0);
}

위의 코드는 경로에 점이 두 개 이상 있는지 확인하고 그렇다면 해당 경로를 따라 이동하는 인스턴스를 시작합니다.
