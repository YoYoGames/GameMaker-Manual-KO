path_get_length

이 함수를 사용하여 경로의 정확한 길이를 픽셀 단위로 얻을 수 있습니다. 이것은 점에서 점까지의 대략적인 길이가 아니라 경로가 높은 곡선 정밀도로 매끄럽더라도 경로의 모양을 따른 정확한 길이입니다.

통사론:

path_get_length(index);

논쟁 유형 설명
index The index of the path to measure.

보고:

예시:

path_len = path_get_length(pth_AI);

이것은 "path_len"을 "pth_AI"에 인덱스된 경로의 총 길이(픽셀 단위)로 설정합니다.
