path_get_precision

이 함수는 주어진 경로가 "스무딩"된 정밀도로 반환되며 1에서 8 사이의 정수 값이 됩니다. 직선 경로에 대해 이 값을 얻을 수 있지만(설정할 수 있지만) 방법에 영향을 미치지 않습니다. instance는 경로 종류가 "smooth"로 설정된 경우에만 관련이 있는 경로를 사용합니다.

통사론:

path_get_precision(index);

논쟁 유형 설명
index The index of the path to check.

보고:

예시:

if (path_get_kind(pth_Patrol))
{
    if (path_get_precision(pth_Patrol)) != 8
    {
        path_set_precision(pth_Patrol, 8);
    }
}

위의 코드는 "pth_Patrol"에 인덱싱된 경로를 확인하여 경로의 종류를 확인하고 매끄럽다면 매끄럽게 하는 정도가 정확한지 확인합니다. 8보다 작으면 8 값으로 설정됩니다.
