path_get_point_x

이 함수는 인덱싱하는 경로에 대해 입력한 점의 x 위치(실 좌표)를 반환합니다. 점이 경로 범위 밖에 있는 경우(예: 경로에 점이 8개 있고 점 10의 x 위치를 묻는 경우) 값 0이 반환됩니다.

통사론:

path_get_point_x(index, n);

논쟁 유형 설명
index 경로 자산 The index of the path to check.
n 진짜 The point number to check.

보고:

Real

예시:

if (path_position == 1)
{
    var _num = path_get_number(pth_Patrol);
    var _pos = floor(random(_num));
    x = path_get_point_x(pth_Patrol, _pos);
    y = path_get_point_y(pth_Patrol, _pos);
    path_position = (1 / _num) * _pos;
}

위의 코드는 인스턴스가 경로 끝에 있는지 확인합니다. 그렇다면 경로에서 임의의 지점을 선택하고 인스턴스를 해당 지점으로 이동합니다.
