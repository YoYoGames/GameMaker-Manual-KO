path_get_kind

경로는 매끄럽 거나 직선 일 수 있습니다(부드러운 경로는 정의 지점 주위의 곡선 경로를 계산하는 반면 직선 경로는 한 지점에서 다른 지점으로 직선으로 이동합니다). 이 함수는 주어진 경로가 부드러운지 확인하는 데 사용할 수 있습니다(true ) 여부(false ).

통사론:

path_get_kind(index);

논쟁 유형 설명
index The index of the path to check.

보고:

예시:

path_kind = path_get_kind(pth_Patrol);

이렇게 하면 "path_kind" 변수가 다음과 같이 설정됩니다. true 또는 false 주어진 경로 인덱스는 경로의 종류에 따라 다릅니다.
