path_get_name

이 함수는 문자열로 참조되는 경로의 이름을 반환합니다. 이름은 편집기에서 경로를 정의하는 데 사용된 것이거나(경로가 코드 함수를 통해 생성된 경우) " _newpathXX " 형식의 문자열을 반환합니다. 여기서 "XX "는 생성된 경로의 번호로, 0에서 시작하여 새 경로가 생성될 때마다 1씩 증가합니다. 이것은 문자열일 뿐이며 경로를 직접 참조하는 데 사용할 수 없습니다. 경로 색인 이 필요하기 때문입니다. ID. 그러나 이 문자열을 사용하여 함수와 함께 반환된 문자열을 사용하여 경로 인덱스 ID 를 가져올 수 있습니다.asset_get_index().

통사론:

path_get_name(index);

논쟁 유형 설명
index The index of the path to check.

보고:

예시:

path_name = path_get_name(path_array[0]);

이것은 "path_name"을 위치 0에 있는 주어진 배열에서 인덱스된 경로의 이름으로 설정합니다.
