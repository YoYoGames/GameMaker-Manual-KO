tileset_get_uvs

이 함수는 텍스처 페이지에 설정된 주어진 타일의 텍스처에 대한 UV 좌표 및 기타 데이터가 있는 배열 을 반환합니다. 이 함수는 다음 8개 요소가 포함된 배열을 반환합니다.

  • [0] = 왼쪽
  • [1] = 상단
  • [2] = 오른쪽
  • [3] = 바닥
  • [4] = 자산 컴파일러가 타일 세트 왼쪽에서 트리밍한 픽셀의 양
  • [5] = 자산 컴파일러가 타일 세트 상단에서 트리밍한 픽셀의 양
  • [6] = 텍스처 페이지에 저장된 원래 타일 세트 너비의 픽셀 데이터의 정규화된 백분율
  • [7] = 텍스처 페이지에 저장된 원래 타일 세트 높이의 픽셀 데이터의 정규화된 백분율

그런 다음 이 배열은 특히 3D에서 작업하거나 2D 기본 기능을 사용할 때와 셰이더 기능으로 작업할 때 다른 그리기 기능에서 사용할 수 있습니다.

통사론:

tileset_get_uvs(tileset);

논쟁 유형 설명
tileset The index of the tile set to use.

보고

(1D, 8 elements)

예시:

var tex = tileset_get_uvs(tl_Grass);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

위의 코드는 주어진 타일 세트에 대한 UV 좌표를 로컬 배열에 저장한 다음 값을 인스턴스 변수에 할당합니다.
