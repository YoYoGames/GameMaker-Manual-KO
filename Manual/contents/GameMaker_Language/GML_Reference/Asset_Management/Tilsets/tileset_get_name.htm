tileset_get_name

이 함수는 이름 을 지정된 타일 세트 자산의 문자열로 반환합니다. 이 이름은 기본 GameMaker 창의 자산 브라우저에 설정된 타일에 대해 지정된 이름입니다. 이것은 문자열일 뿐이며 타일 세트를 직접 참조하는 데 사용할 수 없습니다. 타일 세트 색인 이 필요하기 때문입니다. 그러나 이 문자열을 사용하여 함수와 함께 반환된 문자열을 사용하여 타일 세트 인덱스 를 가져올 수 있습니다.asset_get_index().

통사론:

tileset_get_name(index);

논쟁 유형 설명
index 타일 세트 자산 The index of the tile set to get the name of.

보고

String

예시:

var _l = layer_get_id("tilemap_trees");
var _m = layer_tilemap_get_id(_l);
var _t = tilemap_get_tileset(_m);
tileset_name = tileset_get_name(_t);

위의 코드는 주어진 레이어에 대한 타일 세트 인덱스의 이름을 가져와서 변수에 반환 문자열을 저장합니다. tileset_name.
