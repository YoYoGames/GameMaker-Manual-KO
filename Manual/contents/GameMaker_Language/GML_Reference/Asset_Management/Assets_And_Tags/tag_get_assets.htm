tag_get_assets

이 기능을 사용하면 지정된 태그가 할당된 모든 자산의 이름을 검색할 수 있습니다. 단일 태그 문자열 또는 배열 을 제공합니다. 여기서 배열의 각 항목은 태그 문자열입니다. 이 함수는 각 항목이 주어진 태그가 있는 자산의 이름(문자열)인 배열을 반환합니다. 자산에 대한 고유 인덱스가 필요한 경우 다음 기능을 사용할 수 있습니다.asset_get_index() 반환된 이름과 함께. 주어진 태그가 있는 자산이 없거나 오류가 있는 경우(예: 주어진 태그가 존재하지 않음) 빈 배열이 반환됩니다.

통사론:

tag_get_assets(tags);

논쟁 유형 설명
tags 문자열 또는 문자열 배열 A single asset tag string or an array with various asset tags.

보고:

Array

예시:

backgrounds = ds_list_create();
var _assets = tag_get_assets("background");
for (var i = 0; i< array_length(_assets); ++i;)
{
    if asset_get_type(_assets[i]) == asset_sprite
    {
        ds_list_add(backgrounds, asset_get_index(_assets[i]));
    }
}

위의 코드는 목록을 만든 다음 "background" 태그가 있는 모든 자산의 이름을 검색합니다. 반환된 이름 배열을 순환하면서 스프라이트 자산이 있는지 확인하고, 스프라이트 자산이면 자산에 대한 고유 인덱스 값이 나중에 사용할 수 있도록 목록에 추가됩니다.
