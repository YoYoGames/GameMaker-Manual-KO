asset_get_index

이 함수를 사용하여 이름에서 게임 자산에 대한 고유 식별 인덱스를 가져올 수 있습니다. 자산을 찾을 수 없는 경우 함수는 -1 값을 반환하고, 그렇지 않으면 확인 중인 자산의 고유 인덱스 ID를 반환합니다. 이 ID는 다음과 같은 다른 인덱스 값과 마찬가지로 다른 함수에서 사용할 수 있습니다.sprite_index 아니면 그path_index, 예를 들어. 이 함수를 사용하여 문자열에서 자산을 참조할 수 있지만(아래 예 참조) 자산을 사용하기 전에 항상 자산이 존재하는지 확인해야 합니다. 그렇지 않으면 게임을 충돌시키는 오류가 발생할 수 있습니다.

통사론:

asset_get_index(name);

논쟁 유형 설명
name The name of the game asset to get the index of (a string).

보고:

Asset (any asset type)

예시:

var obj = asset_get_index("obj_Enemy_" + string(global.Level));
if obj > -1
{
    instance_create_layer(random(room_width), random(room_height), obj, "Enemy_Layer");
}

위의 코드는 문자열에서 개체 인덱스를 가져오고 해당 인덱스가 있으면 게임에서 개체의 인스턴스를 만듭니다.
