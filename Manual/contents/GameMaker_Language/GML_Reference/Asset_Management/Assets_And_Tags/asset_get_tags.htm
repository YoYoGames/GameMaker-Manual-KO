asset_get_tags

이 기능을 사용하면 자산 브라우저에서 자산에 할당된 모든 태그를 검색할 수 있습니다. 자산 이름(문자열) 또는 자산 인덱스를 제공하면 함수가 해당 자산에 대한 태그 배열 을 반환합니다. 태그를 찾을 수 없거나 오류가 있는 경우(즉: 주어진 이름 문자열이 존재하지 않음) 반환된 배열은 비어 있습니다. 자산 인덱스 값을 제공하는 경우 선택적인 자산 유형 인수(상수)를 제공해야 합니다. 다른 유형의 자산은 동일한 이름을 가질 수 없더라도 동일한 인덱스를 가질 수 있기 때문입니다. 사용 가능한 자산 유형은 아래 표에 나열되어 있습니다.

일정한 설명
asset_object 주어진 이름은 객체 를 나타냅니다.
asset_sprite 주어진 이름은 스프라이트 를 나타냅니다.
asset_sound 주어진 이름은 소리 를 나타냅니다.
asset_room 주어진 이름은 room 을 나타냅니다.
asset_tiles 주어진 이름은 타일 세트 를 나타냅니다.
asset_path 주어진 이름은 경로 를 나타냅니다.
asset_script 주어진 이름은 script 를 나타냅니다.
asset_font 주어진 이름은 글꼴 을 나타냅니다.
asset_timeline 주어진 이름은 타임 라인 을 나타냅니다.
asset_shader 주어진 이름은 셰이더 를 나타냅니다.
asset_animationcurve 주어진 이름은 애니메이션 곡선 을 나타냅니다.
asset_sequence 주어진 이름은 Sequence 를 참조합니다.

통사론:

asset_get_tags(name_or_index, [asset_type]);

논쟁 유형 설명
name_or_index 문자열 또는 자산 The name of the asset (a string) or its index.
[asset_type] 자산 유형 상수 OPTIONAL! The type of asset to get the tags for, only used when supplying an index for the first argument.

보고:

Array

예시:

my_tags = asset_get_tags(object_get_name(object_index));

위의 코드는 코드를 실행하는 인스턴스가 생성된 객체에 할당된 모든 태그를 검색합니다.
