asset_get_type

이 함수를 사용하면 이름(문자열)에서 참조되는 자산 유형을 가져올 수 있습니다. 아래 나열된 상수 중 하나가 반환됩니다.

통사론:

asset_get_type(name);

논쟁 유형 설명
name 끈 The name of the game asset to get the type of (a string).

보고:

Asset Type Constant

일정한 설명
asset_object 주어진 이름은 객체 를 나타냅니다.
asset_sprite 주어진 이름은 스프라이트 를 나타냅니다.
asset_sound 주어진 이름은 소리 를 나타냅니다.
asset_room 주어진 이름은 room 을 나타냅니다.
asset_tiles 주어진 이름은 타일 세트 를 나타냅니다.
asset_path 주어진 이름은 경로 를 나타냅니다.
asset_script 주어진 이름은 script 를 나타냅니다.
asset_font 주어진 이름은 글꼴 을 나타냅니다.
asset_timeline 주어진 이름은 타임 라인 을 나타냅니다.
asset_shader 주어진 이름은 셰이더 를 나타냅니다.
asset_animationcurve 주어진 이름은 애니메이션 곡선 을 나타냅니다.
asset_sequence 주어진 이름은 Sequence 를 참조합니다.
asset_unknown 주어진 이름은 존재하지 않거나 위에 나열된 자산 중 하나가 아닌 자산을 나타냅니다.

예시:

if asset_get_type("pth_Path_" + string(global.Game)) == asset_unknown
{
    show_debug_message("Path doesn't exist!!!");
}
else
{
    path_index = asset_get_index("pth_Path_" + string(global.Game));
}

위의 코드는 동적으로 생성된 자산 이름을 확인하여 자산이 올바른 유형인지 확인합니다. 그렇지 않은 경우 디버그 메시지가 표시되고, 그렇지 않으면 자산 이름을 사용하여 자산을 인스턴스에 할당합니다.
