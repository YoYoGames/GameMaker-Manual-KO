tag_get_asset_ids

이 기능을 사용하면 지정된 태그가 할당된 지정된 유형의 모든 자산을 가져올 수 있습니다. 단일 태그(문자열) 또는 배열의 각 항목이 태그(문자열)인 배열과 확인할 자산 유형을 제공합니다. 자산 유형은 다음 상수 중 하나여야 합니다.

일정한 설명
asset_object 주어진 이름은 객체 를 나타냅니다.
asset_sprite 주어진 이름은 스프라이트 를 나타냅니다.
asset_sound 주어진 이름은 소리 를 나타냅니다.
asset_room 주어진 이름은 room 을 나타냅니다.
asset_tiles 주어진 이름은 타일 세트 를 나타냅니다.
asset_path 주어진 이름은 경로 를 나타냅니다.
asset_script 주어진 이름은 script 를 나타냅니다.
asset_font 주어진 이름은 글꼴 을 나타냅니다.
asset_timeline 주어진 이름은 타임 라인 을 나타냅니다.
asset_shader 주어진 이름은 셰이더 를 나타냅니다.
asset_animationcurve 주어진 이름은 애니메이션 곡선 을 나타냅니다.
asset_sequence 주어진 이름은 Sequence 를 참조합니다.

이 함수는 배열을 반환하며 배열의 각 항목은 단일 자산 인덱스 값이 됩니다. 지정된 태그가 있는 유형의 자산이 없으면 빈 배열이 반환됩니다.

통사론:

tag_get_asset_ids(tags, asset_type);

논쟁 유형 설명
tags 문자열 또는 문자열 배열 A single asset tag string or an array with various asset tags.
asset_type 자산 유형 상수 An asset type constant (listed above)

보고:

Array

예시:

var _paths = tag_get_asset_ids("enemy", asset_path);
var _num = irandom(array_length(_paths) - 1);
path_start(_paths[_num], 1, path_action_reverse, false);

위의 코드는 무작위로 하나를 선택하고 코드를 실행하는 인스턴스에 할당하기 전에 "적" 태그를 사용하여 해당 태그가 있는 모든 경로 자산을 찾습니다.
