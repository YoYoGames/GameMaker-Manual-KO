asset_clear_tags

이 기능을 사용하면 자산 브라우저에서 자산에 있는 모든 태그를 지울 수 있습니다. 자산 이름(문자열) 또는 자산 인덱스를 제공하고 자산 인덱스 값을 제공하는 경우 다른 유형의 자산이 동일한 값을 가질 수 있으므로 선택적 자산 유형 인수(상수)를 제공해야 합니다. 같은 이름을 가질 수 없더라도 인덱스. 사용 가능한 자산 유형은 아래 표에 나열되어 있습니다.

일정한 설명
asset_object 주어진 이름은 객체 를 나타냅니다.
asset_sprite 주어진 이름은 스프라이트 를 나타냅니다.
asset_sound 주어진 이름은 소리 를 나타냅니다.
asset_room 주어진 이름은 room 을 나타냅니다.
asset_tiles 주어진 이름은 타일 세트 를 나타냅니다.
asset_path 주어진 이름은 경로 를 나타냅니다.
asset_script 주어진 이름은 script 를 나타냅니다.
asset_font 주어진 이름은 글꼴 을 나타냅니다.
asset_timeline 주어진 이름은 타임 라인 을 나타냅니다.
asset_shader 주어진 이름은 셰이더 를 나타냅니다.
asset_animationcurve 주어진 이름은 애니메이션 곡선 을 나타냅니다.
asset_sequence 주어진 이름은 Sequence 를 참조합니다.

함수가 태그 지우기에 성공하면 반환됩니다. true 그렇지 않으면 반환됩니다 false.

통사론:

asset_clear_tags(name,[asset_type]);

논쟁 유형 설명
name_or_index 문자열 또는 자산 The name of the asset (a string) or its index value (an integer).
[asset_type] 자산 유형 상수 OPTIONAL! The type of asset to clear the tags from, only used when supplying an index for the first argument.

보고:

Boolean

예시:

asset_clear_tags(obj_Enemy_Parent, asset_object);

위의 코드는 주어진 객체 자산에서 모든 태그를 지울 것입니다.
