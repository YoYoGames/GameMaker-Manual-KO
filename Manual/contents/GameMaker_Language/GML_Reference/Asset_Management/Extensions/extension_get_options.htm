extension_get_options

이 함수는 지정된 이름의 확장에 대한 모든 옵션 과 해당 값을 포함하는 구조체를 반환합니다.

제공된 확장 이름이 유효하지 않은 경우 함수는 정의되지 않음을 반환합니다.

통사론:

extension_get_options(extension_name);

논쟁 유형 설명
extension_name 끈 The name of your extension asset as a string

보고:

Struct

예시:

var _options = extension_get_options("MyExtension");
var _enabled = _options.enabled;

이 코드는 확장에 대한 options 구조체를 가져와서 값을 읽습니다.
