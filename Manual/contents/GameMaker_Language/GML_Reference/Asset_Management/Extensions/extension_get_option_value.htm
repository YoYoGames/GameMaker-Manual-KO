extension_get_option_value

이 함수는 주어진 이름을 가진 확장에서 주어진 옵션 의 값을 반환합니다.

제공된 확장 이름이 유효하지 않은 경우 함수는 정의되지 않음을 반환합니다.

통사론:

extension_get_option_value(extension_name, option_name);

논쟁 유형 설명
extension_name 끈 The name of your extension asset as a string
option_name 끈 The name of the option to read

보고:

Any

예시:

var _enabled = extension_get_option_value("MyExtension", "enabled");

이 코드는 MyExtension 확장에서 활성화된 옵션의 값을 가져옵니다.
