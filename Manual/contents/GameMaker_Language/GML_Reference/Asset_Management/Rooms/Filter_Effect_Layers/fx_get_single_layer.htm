fx_get_single_layer

이 함수는 제공된 필터/효과가 단일 레이어 모드(true ) 여부(false ). 보다fx_set_single_layer() 단일 레이어 모드에 대한 정보는

FX 구조체를 지정합니다( fx_create() 또는 layer_get_fx() ) 이 함수는 FX에 대해 단일 레이어 모드가 활성화되었는지 여부를 알려주는 부울 을 반환합니다.

통사론:

fx_get_single_layer(filter_or_effect);

논쟁 유형 설명
filter_or_effect The FX struct to read

보고:

Boolean

예시:

var _fx = layer_get_fx("Distort");

if (!fx_get_single_layer(_fx))
{
    fx_set_single_layer(_fx, true);
}

위의 코드는 "Distort" 라는 레이어에서 FX 구조체를 가져와서 단일 레이어 모드가 활성화되어 있는지 확인합니다. 그렇지 않은 경우 해당 FX에 대해 단일 레이어 모드를 활성화합니다.
