layer_fx_is_enabled

이 함수는 레이어에 대한 필터/효과가 활성화되었는지 여부를 반환합니다.

대상으로 지정하려는 레이어의 ID 또는 이름을 지정하면 이 함수는 레이어의 FX가 활성화되었는지( true ) 비활성화되었는지( false ) 나타내는 부울 값을 반환합니다.

통사론:

layer_fx_is_enabled(layer_name_or_id);

논쟁 유형 설명
layer_name_or_id 문자열 또는 레이어 ID The name or ID of the layer to check

보고:

Boolean

예시:

x = xstart;
y = ystart;
hp = hp_max;

if (layer_fx_is_enabled("DesaturateLayer"))
{
    layer_enable_fx("DesaturateLayer", false);
}

위의 코드는 인스턴스를 원래 위치로 이동하고 HP를 다시 채워 인스턴스를 "다시 생성"합니다. 그런 다음 Desaturate FX 레이어가 활성화되어 있는지 확인하고 활성화되어 있으면 비활성화합니다.
