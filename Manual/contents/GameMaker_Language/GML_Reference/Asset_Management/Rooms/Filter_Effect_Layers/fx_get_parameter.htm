fx_get_parameter

이 함수는 FX 구조체에서 매개변수 값을 검색하는 데 사용됩니다. 읽을 FX 구조체를 지정합니다( fx_create() 또는 layer_get_fx() ) 및 매개변수의 이름을 문자열로 반환하고 함수는 현재 값을 반환합니다. 매개변수가 벡터(예: vec4 RGBA 색상을 저장합니다).

이 페이지에서 필터/효과에 대한 매개변수 이름을 찾으십시오: FX 유형 및 매개변수.

통사론:

fx_get_parameter(filter_or_effect, parameter_name);

논쟁 유형 설명
filter_or_effect FX 구조체 The FX struct to get the parameter from
parameter_name 끈 The name of the parameter to get the value of (as a string)

보고:

Real or Array

예시:

var _fx_struct = layer_get_fx("TintEffect");
var _tint_colour = fx_get_parameter(_fx_struct, "g_TintCol");
show_debug_message("The currently active tint colour is: " + string(_tint_colour));

위의 코드는 "색상 색조" 필터가 적용된 것으로 간주되는 레이어에서 FX 구조체를 검색하고 "g_TintCol" 매개변수 값을 가져옵니다. 이 매개변수는 색조 색상(빨간색)에 대한 4개의 값을 포함하는 배열을 반환합니다., 녹색, 파란색 및 알파). 그런 다음 해당 값을 출력 로그에 인쇄합니다.
