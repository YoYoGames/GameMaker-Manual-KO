fx_get_parameters

이 함수는 주어진 FX 구조체에 대한 모든 매개변수를 포함하는 구조체 를 검색하는 데 사용됩니다. FX 구조체를 지정합니다( fx_create() 또는 layer_get_fx() ) 이 함수는 해당 필터/효과에 적용할 수 있는 모든 매개변수에 대한 변수를 포함하는 구조체를 반환합니다. 벡터를 사용하는 매개변수(즉, 값 그룹)는 배열로 표시됩니다.

다음을 호출하여 별도의 배열에 있는 모든 매개변수의 이름을 가져올 수 있습니다.fx_get_parameter_names(), 또는 이 페이지에서 찾으십시오: FX 유형 및 매개변수.

이 구조체의 값을 변경해도 필터/효과에 변경 사항이 적용되지 않습니다. 당신은 전화해야합니다fx_set_parameters() 변경 사항이 적용되도록 수정된 구조체를 필터/효과에 다시 적용합니다.

통사론:

fx_get_parameters(filter_or_effect);

논쟁 유형 설명
filter_or_effect FX 구조체 The FX struct for which the parameters will be returned

보고:

Struct (or -1 on failure)

예시:

var _fx_struct = layer_get_fx("TintEffect");

if (_fx_struct != -1)
{
    var _params = fx_get_parameters(_fx_struct);
    var _osc = sin(current_time / 1000);
    _params.g_TintCol = [_osc, 0.3 + _osc, 0.6 + _osc, 1];

    fx_set_parameters(_fx_struct, _params);
}

위의 코드는 "Colour Tint" 필터가 적용된 것으로 간주되는 레이어에 대한 FX 구조체를 가져오고 다음을 호출하여 매개변수 구조체를 검색합니다. fx_get_parameters(). 그 후 다음을 사용하여 진동 값을 생성합니다. sin() 그리고 current_time, 그런 다음 색조 효과에 대한 RGBA 값에 사용되어 애니메이션을 유지합니다. RGBA 배열은 g_TintCol 매개변수 구조체의 변수이고 구조체는 다음을 호출하여 FX 구조체에 다시 적용됩니다.fx_set_parameters().
