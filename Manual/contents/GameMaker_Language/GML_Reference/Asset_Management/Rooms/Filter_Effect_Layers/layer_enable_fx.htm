layer_enable_fx

이 기능은 룸 레이어에 할당된 필터/효과를 활성화/비활성화합니다. 수정하려는 레이어의 ID 또는 이름을 지정한 다음 FX를 활성화해야 하는지( layer_clear_fx()>true ) 비활성화해야 하는지( layer_set_fx()>false ) 알려주는 부울 값을 지정합니다.

false 를 전달하면 레이어에서 FX가 제거되지 않고 단순히 보이지 않게 됩니다. 사용 레이어에서 FX를 제거합니다.

마찬가지로, 레이어에 적용된 FX가 없으면 true 를 전달해도 아무 작업도 수행되지 않습니다. 사용 레이어에 FX를 적용합니다.

통사론:

layer_enable_fx(layer_name_or_id, enable);

논쟁 유형 설명
layer_name_or_id 문자열 또는 레이어 ID The name or ID of the layer to modify
enable 부울 Whether to enable or disable the FX

보고:

N/A

예시:

if (hp <= 3)
{
    layer_enable_fx("DesaturateLayer", true);
}
else
{
    layer_enable_fx("DesaturateLayer", false);
}

위의 코드는 인스턴스의 HP 값이 3보다 작거나 같으면 Desaturate FX 레이어를 활성화하고, 그렇지 않으면 비활성화합니다. 이것은 화면의 채도를 낮추어 플레이어에게 HP가 낮다는 것을 나타냅니다.
