layer_get_fx

이 함수는 레이어에 대한 FX 구조체를 검색하는 데 사용됩니다. 대상으로 지정하려는 레이어의 ID 또는 이름을 지정하면 함수가 적용된 효과에 대한 정보가 포함된 구조체를 반환합니다. 이 구조체는 다음에서 얻은 구조체와 유사합니다. fx_create() 및 기능fx_get_parameter /s 그리고 fx_set_parameter /s 매개변수를 읽고 수정하는 데 사용할 수 있습니다.

지정된 레이어에 적용된 필터/효과가 없으면 함수는 -1을 반환합니다.

통사론:

layer_get_fx(layer_name_or_id);

논쟁 유형 설명
layer_name_or_id 문자열 또는 레이어 ID The name or ID of the layer to read

보고:

FX Struct (or -1 if not found)

예시:

var layers = layer_get_all();
for(var i = 0; i < array_length(layers); i ++)
{    
    var layer_fx = layer_get_fx(layers[i]);
    
    if (layer_fx != -1)
    {
        if (fx_get_name(layer_fx) == "_filter_tintfilter")
        {            
            fx_set_parameter(_fx_tint, "g_TintCol", [0, 0, 1, 1]);
        }
    }
}

위의 코드는 방에 있는 모든 레이어를 통해 for 루프를 실행하고 각 레이어에서 FX 구조체를 확인합니다. 레이어에 FX 구조체가 있으면 다음을 호출하여 해당 필터/효과의 이름을 확인합니다.fx_get_name() 그 위에; "_filter_tintfilter" 와 같으면 "색상 색조" 필터를 의미하며 색조 색상을 파란색으로 변경합니다.
