layer_set_fx

이 기능은 FX 구조체를 룸 레이어에 할당하는 데 사용됩니다. 수정하려는 레이어의 ID 또는 이름을 지정한 다음 FX 구조체( fx_create() 또는 layer_get_fx() ) 지정된 레이어에 적용되어 방에서 효과를 볼 수 있습니다.

필터/효과 레이어는 아니지만 다른 유형(예: 인스턴스 레이어, 자산 레이어, 타일 레이어 등)인 레이어를 지정하면 효과는 해당 레이어의 내용과 그 아래의 모든 레이어.

지정된 레이어에 이미 FX 구조체가 할당되어 있으면 새 레이어로 대체됩니다.

참고 스크립트의 루트 수준에서 이 함수를 호출하면 첫 번째 룸이 로드되기 전에 스크립트가 실행되므로 아무 작업도 수행하지 않습니다. 먼저 대상 방 을 설정한 다음 이 함수를 호출할 수 있습니다.

통사론:

layer_set_fx(layer_name_or_id, filter_or_effect);

논쟁 유형 설명
layer_name_or_id 문자열 또는 레이어 ID The name or ID of the layer to modify
filter_or_effect FX 구조체 The FX struct to apply to the layer

보고:

N/A

예시:

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

위의 코드는 vec4>"_filter_tintfilter" 효과에서 새로운 FX 구조체를 생성합니다. 이는 Room Editor에 있는 "Colour Tint" 필터입니다. 그것은 색조의 색상인 "g_TintCol" 매개변수에 값을 할당합니다. 내부적으로 4개의 값(빨간색, 녹색, 파란색 및 알파 값에 해당)을 포함하는 배열을 사용합니다. 그런 다음 색조에 대한 FX 구조체가 FX 레이어에 적용됩니다.
