fx_set_single_layer

이 기능은 필터/효과에 대한 "단일 레이어" 모드를 활성화하거나 비활성화하는 데 사용됩니다. 기본적으로 필터/효과는 할당된 레이어와 해당 레이어 아래의 모든 레이어에 적용되지만 FX 구조체에서 단일 레이어 모드를 활성화하여 할당된 레이어에만 적용되도록 할 수 있습니다..

FX 구조체를 지정합니다( fx_create() 또는 layer_get_fx() ) 및 활성화할 부울 값(true ) 또는 비활성화(false ) 단일 레이어 모드.

다음 비주얼은 단일 레이어 모드가 비활성화된 상태에서 사용되는 필터(모든 FX 레이어의 기본 동작)와 단일 레이어 모드가 활성화된 동일한 필터를 보여줍니다.

참고 FX에 대해 단일 레이어 모드를 활성화할 때 FX 레이어 자체에는 아무것도 포함하지 않으므로 필터/효과가 표시되지 않으므로 FX 레이어 가 아닌 레이어에 적용해야 합니다.

통사론:

fx_set_single_layer(filter_or_effect, enable);

논쟁 유형 설명
filter_or_effect The FX struct to modify
enable A boolean value to enable (true ) or disable (false ) Single Layer Mode

보고:

예시:

shake_fx = fx_create("_filter_screenshake");
fx_set_single_layer(shake_fx, true);
layer_set_fx("ShakeyThings", shake_fx);

위의 코드는 새로운 screenshake FX를 만들고 싱글 레이어 모드를 활성화한 다음 룸 레이어에 적용합니다. 이것은 스크린쉐이크 필터가 "ShakeyThings" 레이어에만 적용된다는 것을 의미합니다.
