layer_clear_fx

이 함수는 지정된 레이어에 적용된 FX 구조체를 지우는 데 사용됩니다. 수정하려는 레이어의 ID 또는 이름을 지정하면 이 기능은 해당 레이어에 적용된 FX 구조를 제거하여 효과를 완전히 비활성화합니다.

이 함수를 사용하여 일시적으로 효과를 비활성화하고 레이어를 복원해야 하는 경우 먼저 다음을 호출하여 레이어에 대한 FX 구조체를 저장해야 합니다. layer_get_fx() 그런 다음 이 함수를 호출하여 레이어에서 지우십시오. 나중에 호출하여 레이어에 다시 적용할 수 있습니다. layer_set_fx(), 효과를 다시 활성화합니다.

통사론:

layer_clear_fx(layer_name_or_id);

논쟁 유형 설명
layer_name_or_id 문자열 또는 레이어 ID The name or ID of the layer to clear the FX for

예시:

// Disable effect
if (keyboard_check_pressed(ord("E")))
{
    if (layer_get_fx("EffectLayer") != -1)
    {
        temp_fx = layer_get_fx("EffectLayer");
        layer_clear_fx("EffectLayer");
    }
}
// Re-enable effect
else if (keyboard_check_pressed(ord("F")))
{
    if (temp_fx != -1)
    {
        layer_set_fx("EffectLayer", temp_fx);
        temp_fx = -1;
    }
}

위의 코드는 "E" 키가 눌렸는지 확인한 다음 "EffectLayer" 레이어에 필터/효과가 적용되었는지 확인합니다. 이 경우 FX 구조체를 임시 인스턴스 변수에 저장하고 모든 필터/효과의 레이어를 지웁니다. "F" 키를 누르면 임시 변수에 -1이 아닌 다른 값이 있는지 확인하고 이 경우 해당 FX를 레이어에 다시 적용하고 임시 변수를 -1로 재설정합니다. 이 모든 기능을 통해 매개변수를 잃지 않고 필요할 때마다 효과를 활성화 및 비활성화할 수 있습니다.
