fx_set_parameter

이 함수는 주어진 FX 구조체의 매개변수 값을 변경하는 데 사용됩니다. FX 구조체를 지정합니다( fx_create() 또는 layer_get_fx() ), 매개변수 이름을 문자열로, 새 값. 이렇게 하면 FX 구조체가 할당된 레이어에서 변경 사항을 볼 수 있습니다.

매개변수가 벡터인 경우 매개변수에 대해 여러 인수를 전달할 수 있지만 단순히 여러 값을 포함하는 유일한 인수로 배열을 지정할 수도 있습니다.

이 페이지에서 필터/효과에 대한 매개변수 이름을 찾으십시오: FX 유형 및 매개변수.

통사론:

fx_set_parameter(filter_or_effect, parameter_name, val, ...);

논쟁 유형 설명
filter_or_effect FX 구조체 The FX struct to modify
parameter_name 끈 The name of the parameter to change the value of (as a string)
val... 실수 또는 배열 The new value(s) for that parameter as a real or array

보고:

N/A

예시:

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

위의 코드는 vec4>"_filter_tintfilter" 효과에서 새로운 FX 구조체를 생성합니다. 이는 Room Editor에 있는 "Colour Tint" 필터입니다. 그것은 색조의 색상인 "g_TintCol" 매개변수에 값을 할당합니다. 내부적으로 4개의 값(빨간색, 녹색, 파란색 및 알파 값에 해당)을 포함하는 배열을 사용합니다. 그런 다음 색조에 대한 FX 구조체가 FX 레이어에 적용됩니다.

배열 대신 별도의 인수로 벡터에 값을 전달할 수도 있습니다.

fx_set_parameter(_fx_tint, "g_TintCol", 1, 0, 0, 1);
