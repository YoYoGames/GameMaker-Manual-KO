room_goto_next

이 기능을 사용하면 게임을 컴파일할 때 룸 관리자 에 나열된 대로 게임을 다음 게임으로 이동할 수 있습니다. 이 방이 없으면 오류가 발생하고 게임이 강제 종료됩니다. 방은 함수가 호출된 이벤트가 끝날 때까지 변경되지 않으므로 이 함수가 호출된 후의 코드는 동일한 이벤트에 있는 경우 계속 실행됩니다.

통사론:

room_goto_next();

보고:

예시:

if room_exists(room_next(room))
{
    room_goto_next();
}

위의 코드는 현재 방 뒤에 다른 방이 있는지 확인하고 있으면 그 방으로 이동합니다.
