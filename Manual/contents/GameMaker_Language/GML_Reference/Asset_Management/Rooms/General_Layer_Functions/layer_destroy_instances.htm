layer_destroy_instances

이 기능은 주어진 레이어에 할당된 모든 인스턴스를 파괴하는 데 사용할 수 있습니다. 레이어 ID를 제공합니다(다음을 사용하여 레이어를 생성할 때 얻는layer_create() ) 또는 레이어 이름(문자열로 - 성능에 영향을 줌)을 입력하면 레이어에 있는 모든 인스턴스가 게임에서 제거되어 Destroy 및 Clean Up 이벤트가 트리거됩니다.

통사론:

layer_destroy_instances(layer_id)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the layer (or the layer name as a string)

보고:

N/A

예시:

if global.game_over
{
    layer_destroy_instances(layer);
}

위의 코드는 전역 변수를 확인하고 다음과 같은 경우 true 그러면 호출 인스턴스의 계층에 있는 모든 인스턴스가 소멸됩니다(호출 인스턴스 포함).
