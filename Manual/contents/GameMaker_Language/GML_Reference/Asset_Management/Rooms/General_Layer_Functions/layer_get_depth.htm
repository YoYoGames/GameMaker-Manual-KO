layer_get_depth

이 함수를 사용하여 주어진 레이어와 관련된 깊이 값을 얻을 수 있습니다. 레이어 ID를 제공합니다(다음을 사용하여 레이어를 생성할 때 얻는layer_create() ) 또는 레이어 이름(문자열로 - 성능에 영향을 미침)을 입력하고 함수는 해당 레이어 깊이를 실수로 반환합니다. 깊이는 카메라에서 "멀리" 높을수록, 카메라에 "가까울수록" 낮게 정의되므로 깊이가 -100, 0, 100인 3개의 레이어가 있는 경우 레이어는 100, 0, -100, "상단" 레이어(즉, 카메라 뷰에 가장 가깝고 다른 모든 것 위에 그려짐)는 -100 깊이의 레이어가 됩니다.

다음 이미지는 레이어에서 깊이가 작동하는 방식에 대한 도식을 보여줍니다.

잘못된 레이어 ID 값을 제공하면 오류가 발생합니다.

통사론:

layer_get_depth(layer_id)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the layer to get the depth of (or the layer name as a string)

보고:

Real

예시:

if layer_get_depth(global.temp_layer) != -100
{
    layer_destroy(global.temp_layer);
    global.temp_layer = layer_create(-100);
}

위의 코드는 전역 변수에 저장된 레이어 ID의 깊이를 확인하고 -100이 아니면 레이어를 파괴하고 -100의 깊이에서 다시 생성합니다.
