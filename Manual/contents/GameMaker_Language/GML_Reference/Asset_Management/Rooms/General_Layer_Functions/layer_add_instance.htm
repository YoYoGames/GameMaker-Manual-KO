layer_add_instance

이 기능은 주어진 인스턴스를 현재 있는 레이어에서 다른 레이어로 이동하는 데 사용할 수 있습니다. 레이어 ID를 제공합니다(다음을 사용하여 레이어를 생성할 때 얻는layer_create() ) 또는 레이어 이름(문자열로 - 성능에 영향을 미침) 및 레이어 간에 이동할 인스턴스의 인스턴스 ID입니다. 예를 들어 플레이어가 다른 레이어보다 깊이가 낮은 레이어에 있고 두 레이어 사이의 레이어 뒤에 표시되기를 원한다고 가정해 보겠습니다. 이 기능을 사용하여 플레이어를 전경 레이어에서 배경 레이어로 전환하고 그리는 다른 레이어 뒤에 나타나게 할 수 있습니다.

참고 이 기능은 레이어에 새 인스턴스를 추가하는 데 사용할 수 없습니다. 이전에 인스턴스를 생성하고 해당 ID를 변수에 저장 해야 합니다.

참고: 이 기능은 현재 방의 범위 내에서만 유효하며 기능을 사용하여 범위가 변경된 경우 사용할 수 없습니다.layer_set_target_room().

통사론:

layer_add_instance(layer_id, instance_id)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the instance layer to target (or the layer name as a string)
instance_id 인스턴스 ID The unique instance ID value of the instance to move over to the target layer

보고:

N/A

예시:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

위의 코드는 먼저 주어진 x/y 위치에 가장 가까운 인스턴스의 인덱스를 가져와서 로컬 변수에 저장합니다. 그런 다음 "Instances Front"라는 레이어에 대한 고유한 인스턴스 레이어 ID를 가져오고 찾은 인스턴스를 해당 레이어로 이동합니다.
