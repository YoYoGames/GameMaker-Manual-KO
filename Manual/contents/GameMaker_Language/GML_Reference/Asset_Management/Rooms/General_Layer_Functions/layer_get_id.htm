layer_get_id

이 함수는 주어진 레이어 에 대한 고유 ID 값을 얻는 데 사용할 수 있습니다. IDE에서 모든 레이어에는 이름과 유형이 있으며 코드를 통해 편집하거나 변경할 수 있으려면 레이어 ID 값을 지정해야 합니다. 이 함수는 레이어의 이름(문자열)을 사용하여 이 ID를 검색하는 데 사용됩니다(IDE에 작성된 대로). 함수를 사용하여 코드를 통해 새 레이어를 생성하면layer_create() 그러면 그 함수는 대신 고유한 ID 값을 반환합니다(동적으로 생성된 레이어는 이름을 얻지 못합니다). 현재 방에 없는 레이어의 이름을 지정하면 오류가 발생하고 프로젝트가 충돌합니다.

통사론:

layer_get_id(layer_name)

논쟁 유형 설명
layer_name The name of the layer (a string)

보고:

Layer ID or -1 (if the layer specified doesn't exist)

예시:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

위의 코드는 먼저 주어진 x/y 위치에 가장 가까운 인스턴스의 인덱스를 가져와서 로컬 변수에 저장합니다. 그런 다음 "Instances Front"라는 레이어에 대한 고유한 인스턴스 레이어 ID를 가져오고 찾은 인스턴스를 해당 레이어로 이동합니다.
