layer_get_element_layer

이 기능을 사용하여 주어진 요소가 있는 레이어 ID 를 얻을 수 있습니다. 고유한 요소 ID 값(예: 요소를 생성한 함수 또는 회의실 편집기에서 반환된 값)을 제공하면 이 함수는 요소가 있는 레이어의 고유 ID를 반환합니다. 주어진 요소 ID가 유효한 것이 아니면 함수는 -1을 반환합니다.

통사론:

layer_get_element_layer(element_id)

논쟁 유형 설명
element_id 레이어 요소 ID The unique ID value of the element to get the layer of

보고:

Layer ID or -1 (if the element is invalid)

예시:

element = layer_get_element_layer(asset_1);

위의 코드는 "asset_1" 변수에 저장된 ID를 가진 요소의 레이어 ID를 가져와서 변수에 저장합니다.
