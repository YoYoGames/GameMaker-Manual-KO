layer_element_move

이 기능을 사용하여 한 레이어에서 다른 레이어로 요소를 이동할 수 있습니다. 요소를 생성하는 데 사용된 함수 또는 룸 편집기 또는 함수에서 반환된 대로 요소 ID 를 제공합니다.layer_get_all_elements(), 다음을 사용하여 레이어를 생성할 때 얻은 레이어 ID를 제공합니다.layer_create() ) 또는 레이어 이름(문자열로 - 성능에 영향을 미침).

통사론:

layer_element_move(element_id, layer_id)

논쟁 유형 설명
element_id 레이어 요소 ID The unique ID value of the element to move
layer_id 문자열 또는 레이어 ID The unique ID value of the layer to move the element to (or the layer name as a string)

var a = layer_get_all_elements(layer);
asset_layer = layer_create(-100);
for (var i = 0; i Alt; array_length(a); i++;)

        layer_element_move(a[i], asset_layer)
    }
}

위의 코드는 코드를 실행하는 인스턴스가 할당된 레이어의 요소를 가져온 다음 스프라이트 자산인지 확인하고, 그렇다면 "asset_layer" 변수에 저장된 ID를 가진 레이어로 이동합니다. ".
