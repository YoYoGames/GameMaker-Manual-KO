layer_hspeed

이 기능을 사용하여 현재 범위가 지정된 공간 내에서 레이어의 수평 속도(게임 프레임당 픽셀 단위)를 설정할 수 있습니다. 레이어 ID를 제공합니다(다음을 사용하여 레이어를 생성할 때 얻는layer_create() ) 또는 레이어 이름(문자열로 - 이는 성능에 영향을 줌) 및 설정할 속도 값입니다. 여기서 양수 값은 오른쪽이고 음수 값은 왼쪽입니다.

통사론:

layer_hspeed(layer_id, hspd)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the layer to set the horizontal speed of
hspd 진짜 The horizontal speed (in pixels per game frame) to set

보고:

N/A

예시:

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
}

위의 코드는 주어진 레이어의 수평 및 수직 속도를 확인하고 둘 다 0으로 설정되어 있지 않으면 0으로 설정합니다.
