layer_get_y

이 기능을 사용하여 현재 범위가 지정된 방 내에서 레이어의 y 위치를 검색할 수 있습니다. 레이어 ID를 제공합니다(다음을 사용하여 레이어를 생성할 때 얻는layer_create() ) 또는 레이어 이름(문자열로 - 성능에 영향을 줌)과 함수는 방의 (0,0) 위치에 상대적인 레이어의 x 위치에 대한 실수를 반환합니다. 기본값은 0입니다.

통사론:

layer_get_y(layer_id)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the layer to get the y position of

보고:

Real

예시:

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
    layer_x(lay_id, 0);
    layer_y(lay_id, 0);
}

위의 코드는 주어진 레이어 위치를 확인하고 그것이 (0, 0)으로 설정되어 있지 않으면 그 위치로 설정됩니다.
