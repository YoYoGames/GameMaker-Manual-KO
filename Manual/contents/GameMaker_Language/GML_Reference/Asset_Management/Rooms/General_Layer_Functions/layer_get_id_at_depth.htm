layer_get_id_at_depth

이 기능을 사용하여 특정 깊이가 할당된 모든 레이어의 ID를 가져올 수 있습니다. 확인할 깊이를 지정하면 함수는 주어진 깊이에 레이어가 있는지 여부에 따라 1개 이상의 항목이 있는 배열 을 반환합니다. 주어진 깊이에 레이어가 없으면 배열은 값이 -1인 [0] 위치에 단일 항목을 갖지만 깊이에 레이어가 있는 경우 배열에 대한 항목이 만들어집니다. 발견된 각 레이어 - 항목 값은 레이어의 고유 ID 값이 됩니다.

통사론:

layer_get_id_at_depth(depth)

논쟁 유형 설명
depth 진짜 The depth to check and retrieve the layer IDs from.

보고:

Array

예시:

var a = layer_get_id_at_depth(0);
if a[0] != -1
{
    for (var i = 0; i < array_length(a); i++;)
    {
        layer_destroy(a[i]);
    }
}

위의 코드는 깊이가 0인 레이어에 대한 데이터를 검색합니다. 해당 깊이에 레이어가 있는지 확인하기 위해 검사가 수행되고, 있으면 반환된 배열이 구문 분석되고 발견된 각 레이어가 파괴됩니다.
