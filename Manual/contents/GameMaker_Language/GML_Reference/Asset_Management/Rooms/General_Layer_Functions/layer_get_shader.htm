layer_get_shader

이 기능은 주어진 레이어에 할당된 셰이더가 있는지 확인하는 데 사용할 수 있습니다. 레이어 ID를 제공합니다(다음을 사용하여 레이어를 생성할 때 얻는layer_create() ) 또는 레이어 이름(문자열로 - 이는 성능에 영향을 미칩니다)을 반환하고 함수는 할당된 셰이더의 셰이더 인덱스를 반환하거나 셰이더가 할당되지 않은 경우 -1을 반환합니다.

통사론:

layer_get_shader(layer_id)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the layer to target (or the layer name as a string)

보고:

Shader Asset or -1 (if no shader is assigned)

예시:

if layer_get_shader(layer) == -1
{
    layer_shader(layer, shd_Sepia);
}

위의 코드는 코드를 실행하는 인스턴스에 할당된 셰이더가 있는 레이어와 할당되지 않은 레이어가 있는지 확인합니다.
