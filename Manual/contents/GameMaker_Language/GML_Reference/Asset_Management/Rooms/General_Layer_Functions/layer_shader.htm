layer_shader

이 기능을 사용하면 주어진 레이어에 셰이더 리소스를 할당할 수 있으며 레이어는 해당 셰이더를 사용하여 렌더링됩니다. 사용할 셰이더의 ID와 함께 레이어의 고유 ID 값 또는 레이어 이름(문자열로 - 성능에 영향을 미침)을 제공합니다. 셰이더는 자산 브라우저에서 이전에 생성되어야 하며 셰이더 인덱스(셰이더 리소스의 이름)가 이 함수에 전달됩니다. 할당된 레이어에 인스턴스가 추가된 경우 셰이더는 인스턴스가 사용하는 모든 그리기 이벤트에 적용됩니다. 예를 들어 인스턴스에 그리기 GUI 시작 이벤트가 있는 경우 셰이더가 자동으로 적용됩니다. 셰이더는 스프라이트 자산이나 타일 맵과 같이 해당 레이어에 그려진 다른 그래픽 요소에도 영향을 미칩니다.

이 함수는 무승부 이벤트나 단계 이벤트에서 호출되는 것이 아니라 방 생성 코드 또는 인스턴스의 이벤트 생성 / 방 시작 이벤트 에서 방 시작 시에만 호출하면 됩니다.

통사론:

layer_shader(layer_id, shader)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the layer to target (or the layer name as a string)
shader 셰이더 자산 The shader index to assign to the layer

보고:

N/A

예시:

var lay_id = layer_get_id("Instances");
layer_shader(lay_id, shd_Sepia);

위의 코드는 모든 도면에 대해 주어진 레이어에 셰이더 리소스 "shd_Sepia"를 할당합니다.
