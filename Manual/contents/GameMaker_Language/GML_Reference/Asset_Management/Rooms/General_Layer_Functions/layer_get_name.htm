layer_get_name

이 함수를 사용하여 주어진 레이어의 이름 을 얻을 수 있습니다. 고유한 레이어 ID 값을 제공하고 레이어가 룸 편집기에서 생성된 명명된 레이어 중 하나인 경우 함수는 레이어 이름이 포함된 문자열을 반환합니다. 레이어가 룸 편집기의 레이어가 아닌 경우(예: 다음을 사용하여 생성되었습니다.layer_create() ) 빈 문자열이 반환됩니다.

통사론:

layer_get_name(layer_id)

논쟁 유형 설명
layer_id 레이어 ID The unique ID value of the layer to get the name of

보고:

String

예시:

var a = layer_get_all();
var layer_list = ds_list_create(); for (var i = 0; i Alt; array_length(a); i++;)
{
    if layer_get_name(a[i]) != ""
    {
        ds_list_add(layer_list, a[i])
    }
}

위의 코드는 방에 있는 모든 레이어에 대한 ID를 가져온 다음 루프를 통해 이름이 지정된 레이어가 있는지 확인합니다. 그들이 있다면 그들의 ID가 목록에 추가됩니다.
