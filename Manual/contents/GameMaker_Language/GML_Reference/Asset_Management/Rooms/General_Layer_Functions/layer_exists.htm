layer_exists

이 기능은 주어진 레이어 가 존재하는지 확인하는 데 사용할 수 있습니다. 레이어 ID를 제공합니다(다음을 사용하여 레이어를 생성할 때 얻는layer_create() ) 또는 레이어 이름(문자열로 - 성능에 영향을 미침)을 사용하면 함수는 다음의 부울 값을 반환합니다. true 존재하거나 false 그렇지 않은 경우.

참고: 이 함수는 현재 대상 방의 범위 내에서 작동합니다. 기본적으로 함수가 호출되는 방은 함수를 사용하여 설정할 수 있습니다.layer_set_target_room().

통사론:

layer_exists(layer_name)

논쟁 유형 설명
layer_name 문자열 또는 레이어 ID The name of the layer (a string or ID value)

보고:

Boolean

예시:

if !layer_exists(global.tileLayer)
{
    global.tileLayer = layer_create(1000);
}

위의 코드는 전역 변수에 저장된 레이어가 실제로 존재하는지 확인하고 없으면 생성됩니다.
