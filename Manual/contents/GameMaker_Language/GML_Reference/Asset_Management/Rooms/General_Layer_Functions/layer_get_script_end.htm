layer_get_script_end

이 함수는 주어진 레이어 렌더링이 끝날 때 실행하도록 할당된 함수의 스크립트 함수 인덱스를 반환하거나 할당된 함수가 없으면 -1을 반환합니다. 레이어 ID를 제공합니다(다음을 사용하여 레이어를 생성할 때 얻는layer_create() ) 또는 레이어 이름(문자열로 - 성능에 영향을 미침). 다음을 사용하여 레이어에 스크립트 기능을 할당할 수 있습니다.layer_script_begin() 그리고 layer_script_end().

통사론:

layer_get_script_end(layer_id);

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the layer to target (or the layer name as a string)

보고:

Script Function or -1 (if no function is assigned)

예시:

if layer_get_script_end(layer) == -1
{
    layer_script_end(layer, scr_ResetShaderValues);
}

위의 코드는 코드를 실행하는 인스턴스에 할당된 스크립트 기능이 있는 레이어와 할당되지 않은 레이어가 있는지 확인합니다.
