layer_reset_target_room

이 기능은 레이어 대상을 현재 방으로 재설정하는 데 사용됩니다. 기능 보기layer_set_target_room() 자세한 내용은

통사론:

layer_reset_target_room()

보고:

예시:

layer_set_target_room(rm_Game);
var l = layer_get_id("SpriteAssets");
repeat(50)
{
    layer_sprite_create(l, irandom(1000), irandom(1000), spr_Trees);
}
layer_reset_target_room();

위의 코드는 대상 방을 "rm_Game" 방으로 설정한 다음 해당 방의 "SpriteAssets"라는 레이어에 대한 레이어 ID를 가져옵니다. 그런 다음 이 레이어 ID는 레이어 대상이 현재 방으로 재설정되기 전에 50개의 임의 스프라이트 자산을 레이어에 추가하는 데 사용됩니다.
