layer_get_all

이 함수는 방에 있는 각 레이어의 고유한 ID 값으로 채워진 배열 을 반환합니다.

통사론:

layer_get_all()

보고:

Array (populated with Layer IDs)

예시:

var a = layer_get_all();
for (var i = 0; i < array_length(a); i++;)
{
    layer_destroy(a[i]);
}

위의 코드는 방의 모든 레이어를 검색하고 해당 ID 값을 배열에 추가합니다. 그런 다음 이 배열을 구문 분석하여 룸 레이어를 파괴합니다.
