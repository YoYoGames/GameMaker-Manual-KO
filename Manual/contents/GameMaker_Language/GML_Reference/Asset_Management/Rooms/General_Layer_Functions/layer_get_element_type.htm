layer_get_element_type

이 함수를 사용하여 주어진 요소에 대한 요소 유형 을 가져올 수 있습니다. 고유한 요소 ID 값을 제공하면(예: 요소를 생성한 함수 또는 회의실 편집기에서 반환됨) 함수는 다음 상수 중 하나를 반환합니다(또는 요소가 존재하지 않거나 ID 값이 다음과 같을 경우 -1 잘못된):

레이어 요소 유형 상수
일정한 설명
layerelementtype_background 요소는 배경입니다.
layerelementtype_instance 요소는 인스턴스입니다.
layerelementtype_sprite 요소는 스프라이트 자산입니다.
layerelementtype_tilemap 요소는 타일맵입니다.
layerelementtype_particlesystem 요소는 입자 시스템입니다.
layerelementtype_tile 요소는 레거시 배경 타일입니다(이전 버전의 GameMaker에서 가져온 프로젝트에만 유효합니다).
layerelementtype_sequence 요소는 시퀀스 자산입니다.

이 기능은 하나의 레이어에 여러 개의 서로 다른 요소 유형이 할당되어 있고 이 기능을 사용하여 주어진 레이어의 모든 요소 목록을 얻을 수 있을 때 가장 유용합니다.layer_get_all_elements().

통사론:

layer_get_element_type(element_id)

논쟁 유형 설명
layer 레이어 요소 ID The unique ID value of the element to get the type of

보고:

Layer Element Type Constant or -1 (if element does not exist or is invalid)

예시:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sprite
    {
        layer_sprite_destroy(a[i]);
    }
}

위의 코드는 코드를 실행하는 인스턴스의 레이어에 할당된 모든 인스턴스 요소의 ID를 가져옵니다. 그런 다음 코드는 반환된 요소가 스프라이트 자산인지 확인하고 해당 요소가 소멸되는지 확인합니다.
