layer_is_draw_depth_forced

이 기능은 프로젝트의 레이어를 렌더링하기 위해 강제 Z 깊이가 활성화되어 있는지 확인하는 데 사용할 수 있습니다. 보다layer_force_draw_depth() 자세한 내용은.

통사론:

layer_is_draw_depth_forced()

보고:

예시:

if !layer_is_draw_depth_forced()
{
    layer_force_draw_depth(true, 0);
}

위의 코드는 레이어 Z 깊이가 강제 적용되었는지 여부를 확인하고 그렇지 않은 경우 Z 깊이를 0으로 설정하고 활성화합니다.
