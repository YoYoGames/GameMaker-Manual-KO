layer_get_vspeed

이 함수를 사용하여 현재 범위가 지정된 방 내 레이어의 수직 속도(게임 프레임당 픽셀 단위)를 검색할 수 있습니다. 레이어 ID를 제공합니다(다음을 사용하여 레이어를 생성할 때 얻는layer_create() ) 또는 레이어 이름(문자열로 - 이것은 성능에 영향을 미칠 것임)이고 함수는 수직 속도에 대한 실수를 반환합니다. 여기서 양수 값은 감소하고 음수 값은 증가합니다. 기본값은 0입니다(룸 편집기에서 설정하지 않은 경우).

통사론:

layer_get_vspeed(layer_id)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the layer to get the vertical speed from

보고:

N/A

예시:

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
}

위의 코드는 주어진 레이어의 수평 및 수직 속도를 확인하고 둘 다 0으로 설정되어 있지 않으면 0으로 설정합니다.
