layer_force_draw_depth

이 함수는 모든 레이어가 지정된 z 깊이로 그려지도록 합니다. 이것은 레이어가 렌더링되는 순서를 변경하지 않으며 여전히 깊이 순서로 그려지며 사용되는 z 값만 변경합니다. 일반적으로 이것에 대해 걱정할 필요가 없지만 법적 범위(-16000 ~ 16000)를 벗어난 깊이가 있는 레이어가 있는 경우 렌더링되지 않으므로 Z 깊이를 합리적인 값으로 강제할 수 있습니다. - 예를 들어 0 - 모두 잘 렌더링됩니다. 이것은 일반적으로 허용된 레이어 범위보다 높거나 낮은 그리기 깊이를 가질 수 있는 이전 버전의 GameMaker 에서 레거시 프로젝트와 함께 사용하기 위한 것입니다.

통사론:

layer_force_draw_depth(force, depth)

논쟁 유형 설명
force 부울 Whether to enable (set to true or disable (set to false ) Z depth forcing
depth 진짜 The new Z depth

보고:

N/A

예시:

if !layer_is_draw_depth_forced()
{
    layer_force_draw_depth(true, 0);
}

위의 코드는 레이어 Z 깊이가 강제 적용되었는지 여부를 확인하고 그렇지 않은 경우 Z 깊이를 0으로 설정하고 활성화합니다.
