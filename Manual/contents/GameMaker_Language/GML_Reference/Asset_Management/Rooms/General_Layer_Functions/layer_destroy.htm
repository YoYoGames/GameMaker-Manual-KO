layer_destroy

이 기능은 주어진 레이어를 파괴합니다. 레이어 ID를 제공합니다(다음을 사용하여 레이어를 생성할 때 얻는layer_create() ) 또는 레이어 이름(문자열로 - 성능에 영향을 줌)을 입력하면 현재 룸에서 제거됩니다. 레이어가 룸 편집기에서 디자인된 레이어인 경우 다음 번에 룸을 나왔다가 돌아올 때 원래 내용으로 레이어가 다시 생성되지만 룸이 지속되는 경우 레이어가 제거되지 않는 한 제거됩니다. 방 지속성이 다시 꺼집니다. 이 방법으로 레이어를 파괴하면 모든 콘텐츠도 제거되므로 배경이나 타일 맵 등에 대한 참조 ID는 더 이상 유효하지 않으며 레이어에 할당된 모든 인스턴스가 파괴됩니다( Destroy 수행 이벤트 가 있는 경우 동시에 이벤트 및 정리 이벤트 ).

통사론:

layer_destroy(layer_id)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the layer to destroy (or the layer name as a string)

보고:

N/A

예시:

if !instance_exists(obj_Bullet_Parent)
{
    layer_destroy(global.Bullet_Layer);
}

위의 코드는 "obj_Bullet_Parent" 개체의 인스턴스가 존재하는지 확인하고 존재하지 않으면 전역 변수에 저장된 ID로 레이어를 파괴합니다.
