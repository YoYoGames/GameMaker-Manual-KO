layer_get_target_room

이 함수는 레이어 함수의 대상이 되는 현재 방을 반환합니다. 보다layer_set_target_room() 자세한 내용은.

통사론:

layer_get_target_room()

보고:

Room Asset

예시:

if layer_get_target_room() != room
{
    layer_reset_target_room();
}

위의 코드는 현재 대상 방을 확인하고 현재 방이 아니면 레이어 대상 방을 재설정합니다.
