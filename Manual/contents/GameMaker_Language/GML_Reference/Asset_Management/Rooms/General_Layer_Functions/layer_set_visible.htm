layer_set_visible

이 기능을 사용하면 레이어의 가시성을 토글할 수 있습니다. 레이어 ID를 제공합니다(다음을 사용하여 레이어를 생성할 때 얻는layer_create() ) 또는 레이어 이름(문자열로 - 성능에 영향을 미침)과 표시되는 레이어의 토글 값true 그리고 보이지 않는 것은false.

레이어가 보이지 않을 때 레이어에 할당된 것은 아무 것도 그려지지 않으며, 레이어에 인스턴스가 할당되어 있으면 그리기 이벤트도 실행되지 않습니다(인스턴스 변수를 설정하는 것과 매우 유사합니다. visible 에게 false ). 이미 보이지 않는 것으로 플래그가 지정된 모든 인스턴스는 그들이 있는 레이어가 보이는 것으로 설정되어 있으면 두 개의 독립적인 설정이므로 동작은 동일하지만 보이는 것으로 플래그가 지정되지 않습니다.

통사론:

layer_set_visible(layer_id, visible)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the layer to target (or the layer name as a string)
visible 부울 Whether the layer should be visible (true ) or not (false )

보고:

N/A

예시:

var lay_id = layer_get_id("Instances");
if layer_get_visible(lay_id)
{
    layer_set_visible(lay_id, false);
}
else
{
    layer_set_visible(lay_id, true);
}

위의 코드는 룸 편집기에서 "Instances"라는 레이어의 ID 값을 가져온 다음 ID를 사용하여 레이어가 표시되는지 여부를 확인하고 반환된 값에 따라 레이어 가시성을 토글합니다.
