layer_get_all_elements

이 함수를 사용하여 주어진 레이어의 요소 ID 를 가져올 수 있습니다. 레이어 ID를 제공합니다(다음을 사용하여 레이어를 생성할 때 얻는layer_create() ) 또는 레이어 이름(문자열로 - 성능에 영향을 줌)을 입력하고 함수는 ID 배열 을 반환합니다. 여기서 배열의 각 항목은 해당 레이어의 요소에 대한 고유 ID입니다. 예를 들어 레이어가 자산 레이어인 경우 배열은 레이어에 할당된 각 스프라이트 자산의 ID 값으로 채워집니다. 코드를 사용하여 레이어 작업을 한다는 것은 동일한 레이어에 다른 요소 유형을 할당할 수 있다는 것을 의미합니다. 예를 들어 인스턴스와 함께 스프라이트 자산을 가질 수 있습니다. 이 경우 함수를 사용할 수 있습니다.layer_get_element_type() ID와 관련된 요소 유형을 가져옵니다.

통사론:

layer_get_all_elements(layer_id)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the layer to get the elements from (or the layer name as a string)

보고:

Array of Layer Element IDs

예시:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sprite
    {
        layer_sprite_destroy(a[i])
    }
}

위의 코드는 코드를 실행하는 인스턴스의 레이어에 할당된 모든 인스턴스 요소의 ID를 가져옵니다. 그런 다음 코드는 반환된 요소가 스프라이트 자산인지 확인하고 해당 요소가 소멸되는지 확인합니다.
