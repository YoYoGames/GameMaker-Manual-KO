layer_instance_get_instance

이 함수는 주어진 인스턴스 요소의 고유한 인스턴스 ID를 얻는 데 사용할 수 있습니다. 인스턴스 요소 ID를 지정하면(이를 가져오는 방법은 아래 코드 예제 참조) 함수는 요소의 고유한 인스턴스 ID 를 나타내는 실제 값을 반환합니다. 요소가 인스턴스가 아니면 함수는 -1을 반환합니다.

통사론:

layer_instance_get_instance(element_id)

논쟁 유형 설명
element_id 인스턴스 요소 ID The unique ID value of the instance element to get the ID of

보고:

Instance ID

elements = layer_get_all_elements("Instances");
