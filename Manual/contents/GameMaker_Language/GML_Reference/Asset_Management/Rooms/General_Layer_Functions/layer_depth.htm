layer_depth

이 기능을 사용하여 주어진 레이어의 깊이 를 변경하고 해당 콘텐츠가 화면에 렌더링되는 순서를 변경할 수 있습니다. 레이어 ID를 제공합니다(다음을 사용하여 레이어를 생성할 때 얻는layer_create() ) 또는 레이어 이름(문자열로 - 성능에 영향을 미침)을 입력한 다음 설정할 새 깊이를 지정합니다(정수 값 형식 -16000 ~ 16000). 레이어 깊이는 카메라에서 "멀리" 있을 때 더 높고 카메라에 "가까울" 때 더 낮은 것으로 정의됩니다. 따라서 깊이가 -100, 0, 100인 3개의 레이어가 있는 경우 레이어는 100, 0, -100, "상단" 레이어(즉, 카메라 뷰에 가장 가깝고 다른 모든 것 위에 그려짐)는 -100 깊이의 레이어가 됩니다.

다음 이미지는 레이어에서 깊이가 작동하는 방식에 대한 도식을 보여줍니다.

기능을 사용하여 언제든지 레이어의 깊이를 확인할 수 있습니다.layer_get_depth(). 또한 최소 및 최대 레이어 깊이는 -16000 ~ 16000이며 이 깊이를 벗어나는 것은 렌더링되지 않습니다. 해당 범위를 벗어난 깊이가 필요한 경우 함수를 사용해야 합니다.layer_force_draw_depth().

FX 레이어의 깊이보다 낮은 깊이로 변경하면 레이어에 더 이상 효과가 적용되지 않으므로 레이어의 깊이를 수정하면 레이어에 적용되는 필터 및 효과 가 변경될 수 있습니다.

통사론:

layer_depth(layer_id, depth)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the layer to set the depth of (or the layer name as a string)
depth 진짜 The new depth for the layer

보고:

N/A

예시:

if layer_get_depth(layer) != -100
{
    layer_depth(layer, -100);
}

위의 코드는 해당 코드를 실행하는 인스턴스가 있는 레이어의 깊이를 가져오고, -100이 아니면 깊이를 -100으로 설정합니다.
