room_set_view_enabled

이 기능을 사용하면 (true ) 또는 비활성화(false ) 현재 방을 제외한 게임 내 방의 보기.

통사론:

room_set_view_enabled(index, val);

논쟁 유형 설명
index 방 자산 The index of the room to set.
val 부울 Whether to enable (true ) or disable (false ) views in the given room.

보고:

N/A

예시:

global.myroom = room_add();
room_set_view_enabled(global.myroom, true);

이렇게 하면 "global.myroom"에 인덱싱된 방의 보기가 활성화됩니다.
