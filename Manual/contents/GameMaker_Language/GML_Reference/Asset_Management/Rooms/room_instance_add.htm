room_instance_add

이 기능을 사용하면 현재 방이 아닌 다른 방과 해당 방 내의 모든 위치에 인스턴스를 추가할 수 있습니다. 함수는 고유한 값을 반환합니다.id 그런 다음 해당 인스턴스의 속성 등을 설정하기 위해 추가 함수 호출에서 사용할 수 있지만 게임이 지정된 방에 들어간 경우에만 가능합니다. 현재 방에서 인스턴스를 생성하려면 이 기능을 사용해야 합니다.instance_create_layer(). 자산 브라우저에서 생성된 방 자산에서 이 함수를 호출 하면 인스턴스가 방에 영구적으로 추가되고 game_restart() 방을 원래 상태로 되돌리지 않습니다(게임을 종료하고 다시 열면 방이 원래 상태로 다시 시작됩니다).

통사론:

room_instance_add(index, x, y, obj);

논쟁 유형 설명
index The index of the room to add an object instance to.
x The x position of the new instance.
y The y position of the new instance.
obj The object to add an instance of.

보고:

Instance ID

예시:

global.rm = room_add();
room_assign(rm_Base, global.rm);
room_instance_add(global.rm, 100, 100, obj_Player);

위의 코드는 게임에 새 방을 추가한 다음 "rm_Base"로 인덱싱된 방의 내용을 게임에 복사합니다. 그런 다음 이 새 방의 위치 100,100에 "obj_player" 개체의 인스턴스를 추가합니다.
