layer_tilemap_destroy

이 함수는 주어진 타일 맵 요소를 파괴합니다. 타일 맵 ID를 제공합니다(다음을 사용하여 타일 맵을 생성할 때 얻는layer_tilemap_create() 또는 레이어 ID를 함께 사용할 때layer_get_tilemap_id() ) 그러면 제거됩니다. 이것은 레이어를 제거하지 않고 레이어에서 타일 맵만 제거하며 타일 맵이 방 편집기에 추가된 것이라면 다음에 방을 나왔다가 돌아올 때 타일 맵이 다시 생성됩니다. 다시. 그러나 방이 지속되는 경우 방 지속성을 다시 끄지 않는 한 타일 맵이 제거됩니다.

통사론:

layer_tilemap_destroy(tilemap_element_id)

논쟁 유형 설명
tilemap_element_id 타일 맵 요소 ID The unique ID value of the tile map to be destroyed

보고:

N/A

예시:

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if (layer_tilemap_exists(lay_id, tile_id))
{
    layer_tilemap_destroy(tile_id);
}

위의 코드는 "Tiles_trees" 레이어를 확인하여 주어진 타일 맵 요소가 존재하는지 확인하고 존재한다면 파괴됩니다(레이어 제외).
