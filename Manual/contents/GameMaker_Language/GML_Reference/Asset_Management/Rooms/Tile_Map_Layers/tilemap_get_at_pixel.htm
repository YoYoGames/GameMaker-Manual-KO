tilemap_get_at_pixel

이 기능을 사용하여 타일 맵 요소의 위치(방 내)에서 타일 데이터를 검색할 수 있습니다. 타일 맵 요소 ID(다음을 사용하여 타일 맵 요소를 생성할 때 얻는layer_tilemap_create() 또는 기능을 사용할 때layer_tilemap_get_id() ) 뿐만 아니라 타일 데이터를 가져올 방의 x 및 y 위치와 이 함수는 타일 데이터 "블롭"을 반환합니다. 이 데이터는 기본적으로 타일 인덱스, 플립/회전/미러 부울 및 적용된 모든 마스크 데이터를 포함하는 비트 마스크입니다(참조tilemap_set_mask() 자세한 내용은) 결과 데이터 값을 다음에서 사용할 수 있습니다. tile_* 타일 속성을 변경하는 기능입니다. 특정 타일 셀에서 타일 데이터를 가져와야 하는 경우 함수를 사용해야 합니다.tilemap_get() 대신에.

중요한! 타일 맵의 타일이 변경되지 않은 경우(즉, 회전되거나 뒤집히지 않은 경우 등...) 타일 세트 데이터의 반환 값은 타일 세트의 타일 인덱스와 정확히 동일합니다. 따라서 타일 세트의 인덱스 1에서 하나의 타일을 사용하여 타일의 "충돌 맵"을 생성할 수 있습니다. 예를 들어, 이 기능을 사용하여 충돌을 계산하기 위해 1 또는 0(빈 타일)을 확인합니다.

통사론:

tilemap_get_at_pixel(tilemap_element_id, x, y);

논쟁 유형 설명
tilemap_element_id 타일 맵 요소 ID The unique ID value of the tile map element to get the tile-data of
x 진짜 The position along the x-axis to get the tile data from (in room pixels)
y 진짜 The position along the y-axis to get the tile data from (in room pixels)

보고:

Tile Data (-1 if there is an error)

예시:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 64, 128);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 64, 128);

위의 코드는 "Tiles_Sky" 레이어의 "Clouds" 타일 맵에 대한 ID를 가져온 다음 이를 사용하여 (64, 128) 위치의 타일에서 데이터를 가져옵니다. 이 타일 데이터는 타일 맵에서 타일을 다시 설정하는 데 사용되기 전에 뒤집힙니다.
