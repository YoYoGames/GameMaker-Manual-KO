tilemap_set_width

이 기능은 타일 맵 요소의 크기를 조정하는 데 사용할 수 있습니다. 타일 맵 요소 ID(다음을 사용하여 타일 맵 요소를 생성할 때 얻는layer_tilemap_create() 또는 기능을 사용할 때layer_tilemap_get_id() ), 타일 셀에 있는 타일 맵의 새 너비.

통사론:

tilemap_set_width(tilemap_element_id, width)

논쟁 유형 설명
tilemap_element_id 타일 맵 요소 ID The unique ID value of the tile map element to set the width of
width 진짜 The width value (in "cells")

보고:

N/A

예시:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
if tilemap_get_width(map_id) != room_width div 16
{
    tilemap_set_width(map_id, room_width div 16);
}

위의 코드는 특정 타일 맵의 너비를 확인하고 크기가 맞지 않으면 너비를 설정합니다.
