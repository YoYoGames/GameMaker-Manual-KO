tile_get_index

이 함수는 타일 데이터 세트에서 타일 인덱스(타일 세트 이미지 내 타일의 위치)를 가져오는 데 사용할 수 있습니다. 함수를 사용하여 검색할 수 있는 타일 데이터를 지정합니다.tilemap_get(), 함수는 인덱스에 대한 정수 값을 반환합니다.

이 기능을 사용할 때 주어진 타일 데이터가 유효한지 확인하십시오. 그렇지 않으면 반환된 인덱스 값이 유효하지 않아 예기치 않은 동작이 발생할 수 있습니다.

통사론:

tile_get_index(tiledata)

논쟁 유형 설명
tiledata 타일 데이터 the tile-data to check

보고:

Real

예시:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var ind = tile_get_index(data);
data = tile_set_index(data, irandom(23));
tilemap_set(map_id, data, mx, my);

위의 코드는 주어진 레이어에서 타일 맵 ID를 가져온 다음 이를 사용하여 마우스 위치 아래의 셀에 대한 타일 데이터를 가져옵니다. 그런 다음 이 데이터를 사용하여 타일 인덱스를 임의의 숫자로 설정합니다.
