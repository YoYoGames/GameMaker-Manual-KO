tile_set_empty

이 함수는 주어진 타일 데이터 세트를 빈 타일로 설정하는 데 사용할 수 있습니다. 함수를 사용하여 검색할 수 있는 타일 데이터를 제공합니다.tilemap_get(), 함수가 반환됩니다. true 타일이 비어 있거나 false 타일 인덱스가 있는 경우. 이 함수는 수정된 타일 데이터 세트를 반환하고 다음을 사용하여 적용할 수 있습니다. tilemap_set() 함수.

통사론:

tile_set_empty(tiledata)

논쟁 유형 설명
tiledata 타일 데이터 the tile-data to set

보고:

Tile Data

예시:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
for (var i = 0; i < tilemap_get_width(map_id); i++)
{
    for (var j = 0; j < tilemap_get_height(map_id); j++)
    {
        var data = tilemap_get(map_id, i, j);
        if (!tile_get_empty(data))
        {
            data = tile_set_empty(data)
            tilemap_set(map_id, data, i, j);
        }
    }
}

위의 코드는 주어진 레이어에서 타일 맵 ID를 가져온 다음 맵의 모든 타일 셀을 확인하여 데이터가 있는지 여부를 확인합니다. 그렇다면 타일이 비어 있는 것으로 설정됩니다.
