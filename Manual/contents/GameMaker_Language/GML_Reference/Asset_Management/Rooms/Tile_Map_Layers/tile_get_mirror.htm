tile_get_mirror

이 함수는 주어진 타일 데이터 세트에서 타일이 미러링되었는지 여부를 확인하는 데 사용할 수 있습니다. 함수를 사용하여 검색할 수 있는 타일 데이터를 제공합니다.tilemap_get(), 함수가 반환됩니다. true 타일이 미러링된 경우 또는 false 그렇지 않다면.

통사론:

tile_get_mirror(tiledata)

논쟁 유형 설명
tiledata 타일 데이터 the tile-data to check

보고:

Boolean

예시:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_mirror(data);
data = tile_set_mirror(data, bool);
tilemap_set(map_id, data, mx, my);

위의 코드는 주어진 레이어에서 타일 맵 ID를 가져온 다음 마우스 아래 타일의 x 및 y 셀 위치를 가져옵니다. 이 위치는 미러링된 다음 타일을 다시 설정하는 데 사용되는 타일 데이터를 가져오는 데 사용됩니다.
