layer_tilemap_exists

이 기능을 사용하여 타일 맵 요소가 주어진 레이어에 존재하는지 확인하고 확인할 수 있습니다. 레이어 ID를 제공합니다(다음을 사용하여 레이어를 생성할 때 얻는layer_create() ) 또는 레이어 이름(문자열로 - 이는 성능에 영향을 미칩니다) 및 타일 맵 요소 ID(다음을 사용하여 타일 맵 요소를 생성할 때 얻는layer_tilemap_create() 또는 기능을 사용할 때layer_tilemap_get_id() ) 함수는 다음 중 하나를 반환합니다. true 요소가 존재하거나 false 그렇지 않은 경우.

참고: 이 함수는 현재 대상 방의 범위 내에서 작동합니다. 기본적으로 함수가 호출되는 방은 함수를 사용하여 설정할 수 있습니다. layer_set_target_room().

통사론:

layer_tilemap_exists(layer_id, tilemap_element_id)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the layer to target (or the layer name as a string)
tilemap_element_id 타일 맵 요소 ID The unique ID value of the tile map element to check

보고:

Boolean

예시:

var lay_id = layer_get_id("tilemap_trees");
if layer_tilemap_exists(lay_id, global.Treestilemap)
{
    layer_tilemap_destroy(lay_id, global.Treestilemap);
}

위의 코드는 "tilemap_trees" 레이어를 확인하여 주어진 타일 맵 요소가 존재하는지 확인하고 존재하는 경우 파괴됩니다(레이어 제외).
