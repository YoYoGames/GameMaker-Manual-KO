tilemap_get_mask

이 함수는 주어진 타일 맵에 대한 비트 마스크 값을 검색하는 데 사용할 수 있습니다. 타일 맵 요소 ID(다음을 사용하여 타일 맵 요소를 생성할 때 얻는layer_tilemap_create() 또는 기능을 사용할 때layer_tilemap_get_id() ), 그리고 함수는 현재 마스크 값을 반환하거나 오류가 있는 경우 -1을 반환하고 마스크가 지정되지 않은 경우 0을 반환합니다. 타일 맵 비트 마스크에 대한 자세한 내용은 함수를 참조하십시오.tilemap_set_mask().

통사론:

tilemap_get_mask(tilemap_element_id);

논쟁 유형 설명
tilemap_element_id 타일 맵 요소 ID The unique ID value of the tile map element to get the mask of

보고:

Real (0 for no mask, -1 for an error)

예시:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mask = tilemap_get_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if (mask != new_mask)
{
    tilemap_set_mask(map_id, new_mask);
}

위의 코드는 주어진 레이어에서 타일 맵 ID를 가져온 다음 연결된 마스크 값을 확인합니다. 변수 "new_mask"에 정의된 값과 같지 않으면 해당 값으로 설정됩니다.
