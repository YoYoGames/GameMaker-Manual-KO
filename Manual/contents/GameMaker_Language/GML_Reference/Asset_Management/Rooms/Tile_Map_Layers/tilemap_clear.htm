tilemap_clear

이 기능을 사용하면 주어진 타일 맵의 모든 타일을 지우거나 설정할 수 있습니다. 타일 맵 요소 ID(다음을 사용하여 타일 맵 요소를 생성할 때 얻는layer_tilemap_create() 또는 기능을 사용할 때layer_tilemap_get_id() )을 클릭한 다음 레이어를 지우려는 타일 데이터를 제공합니다. 기본값 0은 레이어에서 모든 타일을 지우고(기본적으로 모든 타일을 "비어 있음") 전용을 사용할 수 있습니다. tile_* 타일 맵을 지우기 위해 자신의 타일 데이터를 생성하는 기능.

통사론:

tilemap_clear(tilemap_element_id, tiledata)

논쟁 유형 설명
tilemap_element_id 타일 맵 요소 ID The unique ID value of the tile map element to change
tiledata 타일 데이터(실제) The tile data to use to clear the layer

보고:

N/A

예시:

var layer_id = layer_get_id("Forest");
var tile_id = layer_tilemap_get_id(layer_id);
tilemap_clear(tile_id, 0);

위의 코드는 룸 에디터에서 생성된 타일 맵의 ID 값을 가져온 다음 "빈" 타일을 사용하여 지웁니다.
