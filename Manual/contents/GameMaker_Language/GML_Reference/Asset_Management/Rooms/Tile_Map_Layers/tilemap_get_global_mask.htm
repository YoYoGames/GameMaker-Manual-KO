tilemap_get_global_mask

이 함수는 모든 타일 맵에 대한 비트 마스크 값을 검색하는 데 사용할 수 있으며 현재 마스크 값을 반환하거나 오류가 있는 경우 -1을 반환하고 마스크가 지정되지 않은 경우 0을 반환합니다. 전역 타일 맵 비트 마스크에 대한 자세한 내용은 함수를 참조하십시오.tilemap_set_global_mask().

통사론:

tilemap_get_global_mask();

보고:

(0 for no mask, -1 for an error, mask value otherwise)

예시:

var mask = tilemap_get_global_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if (mask != new_mask)
{
    tilemap_set_global_mask(new_mask);
}

위의 코드는 모든 타일 맵과 연결된 전역 마스크 값을 가져옵니다. 변수 "new_mask"에 정의된 값과 같지 않으면 해당 값으로 설정됩니다.
