tilemap_get_tile_width

이 함수를 사용하여 타일 맵 요소의 각 타일 셀의 너비(픽셀)를 검색할 수 있습니다. 타일 맵 요소 ID(다음을 사용하여 타일 맵 요소를 생성할 때 얻는layer_tilemap_create() 또는 기능을 사용할 때layer_tilemap_get_id() ), 함수는 타일 세트 셀 너비를 반환합니다.

통사론:

tilemap_get_tile_width(tilemap_element_id);

논쟁 유형 설명
tilemap_element_id The unique ID value of the tile map element to get the tile cell width of

보고:

예시:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
global.snap_x = tilemap_get_tile_width(map_id);
global.snap_y = tilemap_get_tile_height(map_id);

위의 코드는 검색된 타일 맵 ID를 사용하여 타일 셀 너비와 높이를 가져오고 이를 사용하여 두 개의 전역 변수를 설정합니다.
