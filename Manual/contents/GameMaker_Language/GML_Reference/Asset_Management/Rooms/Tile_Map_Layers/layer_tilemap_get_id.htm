layer_tilemap_get_id

이 함수는 레이어에 있는 타일 맵 요소의 고유한 ID 값을 검색하는 데 사용할 수 있습니다. 레이어 ID를 제공합니다(다음을 사용하여 레이어를 생성할 때 얻는layer_create() ) 또는 레이어 이름(문자열로 - 성능에 영향을 줌)과 함수는 레이어의 타일 맵 요소와 연결된 ID 값을 반환합니다. 이 기능은 IDE에 추가된 타일 맵과 함께 사용하도록 특별히 설계되었습니다. 마치 이 기능을 사용하는 코드를 통해 레이어에 타일 맵을 추가하는 것과 같습니다.layer_tilemap_create(), 추가된 타일 맵 요소의 고유 ID를 반환합니다. 주어진 타일맵 ID가 올바르지 않거나 타일맵이 존재하지 않는 경우 함수는 -1을 반환합니다.

통사론:

layer_tilemap_get_id(layer_id)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the layer to target (or the layer name as a string)

보고:

Tile Map Element ID or -1

예시:

var lay_id = layer_get_id("tilemap_trees");
var tile_id = layer_tilemap_get_id(lay_id);
layer_tilemap_destroy(tile_id);

위의 코드는 "tilemap_trees"라는 레이어의 레이어 ID를 가져온 다음 이를 사용하여 해당 레이어의 타일 맵 요소 ID를 가져옵니다. 이 ID는 레이어에서 타일 맵을 제거하는 데 사용됩니다.
