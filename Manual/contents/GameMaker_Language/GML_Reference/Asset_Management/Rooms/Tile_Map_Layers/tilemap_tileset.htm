tilemap_tileset

이 기능을 사용하여 레이어의 주어진 타일 맵 요소에 할당된 타일 세트 자산을 변경할 수 있습니다. 타일 맵 요소 ID(다음을 사용하여 타일 맵 요소를 생성할 때 얻는layer_tilemap_create() 또는 기능을 사용할 때layer_tilemap_get_id() ) 타일 세트 인덱스를 제공하면 타일 맵에 새 스프라이트가 제공됩니다.

통사론:

tilemap_tileset(tilemap_element_id, tileset_index)

논쟁 유형 설명
tilemap_element_id 타일 맵 요소 ID The unique ID value of the tile map element to change
tileset_index 타일 세트 자산 The new tile set index to use

보고:

N/A

예시:

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if (tilemap_get_tileset(tile_id) != ts_Nighttime)
{
    tilemap_tileset(tile_id, ts_Nighttime);
}

위의 코드는 "Tiles_trees" 레이어의 타일 맵에 할당된 현재 타일 세트를 확인하고 "ts_Nighttime"이 아닌 경우 해당 타일 세트가 타일 맵에 할당됩니다.
