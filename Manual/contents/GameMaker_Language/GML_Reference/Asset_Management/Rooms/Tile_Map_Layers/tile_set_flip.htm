tile_set_flip

이 함수는 타일을 뒤집거나 하지 않도록 주어진 타일 데이터 세트를 설정하는 데 사용할 수 있습니다. 함수를 사용하여 검색할 수 있는 타일 데이터를 제공합니다.tilemap_get(), 그런 다음 flip 인수를 다음 중 하나로 설정합니다. true 타일을 뒤집고 싶다면, 또는 false 타일을 뒤집지 않은 기본 상태로 유지하려는 경우. 이 함수는 수정된 타일 데이터 세트를 반환하고 다음을 사용하여 적용할 수 있습니다.tilemap_set() 함수.

통사론:

tile_set_flip(tiledata, flip)

논쟁 유형 설명
tiledata the tile-data to set
flip Set to true to flip and false to leave it as-is

보고:

예시:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_flip(data);
data = tile_set_flip(data, bool);
tilemap_set(map_id, data, mx, my);

위의 코드는 주어진 레이어에서 타일 맵 ID를 가져온 다음 마우스 아래 타일의 x 및 y 셀 위치를 가져옵니다. 이 위치는 뒤집힌 다음 타일을 다시 설정하는 데 사용되는 타일 데이터를 가져오는 데 사용됩니다.
