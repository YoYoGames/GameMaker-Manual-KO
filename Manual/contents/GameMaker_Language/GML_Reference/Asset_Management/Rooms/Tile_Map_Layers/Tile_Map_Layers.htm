타일 맵 레이어

GameMaker Room Editor 를 사용하면 레이어 를 사용하여 주어진 방에 타일을 추가할 수 있습니다. 이름에서 알 수 있듯이 타일 맵 레이어는 단순히 방 내에서 동일한 깊이에 있는 것으로 간주되는 타일 세트 타일의 모음이며 레이어를 쌓아서 어떤 것을 다른 것 위에 또는 아래에 그리도록 할 수 있습니다. 또한 코드에서 레이어의 특정 측면을 제어하거나, 항목을 추가 또는 제거하거나, 레이어의 특정 속성이나 런타임에 포함된 항목을 변경할 수 있습니다. 룸 편집기에서는 레이어당 하나의 타일 맵( 타일 레이어 )으로 제한되지만 코드에서 작업할 때 여러 타일 맵을 단일 레이어에 할당할 수 있습니다. 각 맵에는 고유한 ID와 속성, 그래서:

  • 타일 세트 는 타일 레이어 및 타일 맵을 생성하는 데 사용하는 리소스입니다.
  • 타일 레이어 는 타일 세트를 추가하기 위해 룸 편집기에서 생성하는 것입니다.
  • 타일 맵 은 룸 편집기에서 또는 코드를 통해 해당 레이어의 단일 요소로 레이어에 추가되는 타일 모음입니다.

타일 맵을 설정하기 위해 위에 나열된 기능 외에도 개별 타일의 데이터를 조작하는 데 사용할 수 있는 여러 기능이 있습니다. 타일 데이터의 "블롭"은 타일 인덱스와 타일이 회전, 대칭 이동 또는 미러링되었는지 여부를 표시하는 몇 가지 추가 비트로 구성됩니다. 타일 데이터에 고유한 비트 마스크를 추가할 수도 있지만 이는 타일 인덱스용으로 예약된 비트를 마스킹하므로 고급 사용자에게만 해당됩니다. 기본적으로 타일 인덱스용으로 예약된 19비트가 있습니다. 그러나 그 중 8개만 사용하는 경우 사용자 지정 마스크를 만드는 데 사용할 수 있는 11비트가 남습니다. 그런 다음 이 마스크를 사용하고 코드에서 체크인하여 사용자 지정 타일 충돌 등을 생성할 수 있습니다. 그러나 일반적으로 고유한 타일 데이터 blob을 만들 필요가 없으며 대신 함수를 사용합니다.tilemap_get() 타일 데이터를 얻으려면 아래 함수를 사용하여 조작하고 다음을 사용하여 다시 설정합니다.tilemap_set().
