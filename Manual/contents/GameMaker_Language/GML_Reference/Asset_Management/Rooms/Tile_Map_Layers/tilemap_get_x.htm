tilemap_get_x

이 함수를 사용하여 타일 맵 요소의 x 위치(방 내)를 검색할 수 있습니다. 타일 맵 요소 ID(다음을 사용하여 타일 맵 요소를 생성할 때 얻는layer_tilemap_create() 또는 기능을 사용할 때layer_tilemap_get_id() ), 함수는 x축 위치를 반환합니다.

통사론:

tilemap_get_x(tilemap_element_id);

논쟁 유형 설명
tilemap_element_id The unique ID value of the tile map element to get the x position of

보고:

예시:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _x = tilemap_get_x(map_id);
var _y = tilemap_get_y(map_id);
tilemap_x(map_id, _x + 10); tilemap_y(map_id, _y + 10);

위의 코드는 검색된 타일 맵 ID를 사용하여 타일 맵의 타일 x 및 y 위치를 가져온 다음 해당 값을 사용하여 이동합니다.
