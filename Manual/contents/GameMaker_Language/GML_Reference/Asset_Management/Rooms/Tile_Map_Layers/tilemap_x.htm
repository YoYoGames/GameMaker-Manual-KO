tilemap_x

이 기능은 레이어의 자산 타일 맵 요소 방의 x축을 따라 위치를 제어합니다. 타일 맵 요소 ID(다음을 사용하여 타일 맵 요소를 생성할 때 얻는layer_tilemap_create() 또는 기능을 사용할 때layer_tilemap_get_id() ), 사용할 x 값을 설정합니다(방 좌표 기준).

통사론:

tilemap_x(tilemap_element_id, x);

논쟁 유형 설명
tilemap_element_id 타일 맵 요소 ID The unique ID value of the tile map element to change
x 진짜 The x position for the tile map

보고:

N/A

예시:

var lay_id = layer_get_id("Asset_sky");
var map_id = layer_tilemap_get_id(lay_id);
tilemap_x(map_id, irandom(room_width));

위의 코드는 "Asset_sky" 레이어에 할당된 타일 맵 자산의 ID 값을 가져온 다음 x 위치를 0과 방 너비 사이의 임의 값으로 설정합니다.
