layer_tilemap_create

이 기능을 사용하면 프로젝트에서 타일 맵으로 사용할 타일 세트 리소스를 레이어에 할당할 수 있습니다. 레이어 ID를 제공합니다(다음을 사용하여 레이어를 생성할 때 얻는layer_create() ) 또는 레이어 이름(문자열로 - 성능에 영향을 줌)을 입력한 다음 타일 맵을 방에 추가하기 위한 초기(x, y) 위치, 사용할 타일 세트 리소스, 너비와 높이 셀 의 타일 맵(즉, 너비가 20이고 높이가 10이면 너비가 20셀, 높이가 10셀인 셀 200개)가 있는 타일 맵이 생성되며, 셀 자체의 크기는 선택한 타일 세트에 의해 정의됩니다.. 타일 맵 내의 음수 위치에 타일을 배치할 수 없으므로 모든 타일은 0에서 너비 - 1, 0에서 높이 - 1 사이의 셀 공간 내에 배치되어야 합니다.

통사론:

layer_tilemap_create(layer_id, x, y, tileset, width, height)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the layer to target (or the layer name as a string)
x 진짜 The x position of the tile map in the room
y 진짜 The y position of the tile map in the room
tileset 타일 세트 자산 The Tile Set asset to be used
width 진짜 The width tile map (in cells)
height 진짜 The height of the tile map (in cells)

보고:

Tile Map Element ID

예시:

global.back_layer = layer_create(10000);
global.back_tiles = layer_tilemap_create(global.back_layer, 0, 0, tmap_Trees, 16, 32);

위의 코드는 새 레이어를 만든 다음 여기에 새 타일 맵 요소를 추가하여 타일 맵 위치를 (0,0)으로 설정하고 사용할 타일 세트와 타일 맵의 너비와 높이를 설정합니다.
