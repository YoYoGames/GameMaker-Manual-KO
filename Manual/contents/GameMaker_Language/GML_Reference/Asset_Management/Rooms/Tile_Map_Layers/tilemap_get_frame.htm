tilemap_get_frame

타일에 애니메이션을 적용할 수 있으므로 현재 어떤 프레임이 그려지고 있는지 알고 그에 따라 반응하는 것이 때때로 유용할 수 있으므로 이 기능을 사용하면 주어진 타일 맵에 대한 현재 프레임 인덱스를 검색할 수 있습니다. 타일 맵 요소 ID(다음을 사용하여 타일 맵 요소를 생성할 때 얻는layer_tilemap_create() 또는 기능을 사용할 때layer_tilemap_get_id() ), 함수는 프레임 인덱스를 반환합니다.

통사론:

tilemap_get_frame(tilemap_element_id)

논쟁 유형 설명
tilemap_element_id 타일 맵 요소 ID The unique ID value of the tile map element to get the frame index of

보고:

Real (between 0 (inclusive) and the maximum number of frames of animation (exclusive))

예시:

var lay_id = layer_get_id("Tiles_Traps");
var map_id = layer_tilemap_get_id(lay_id);
if (tilemap_get_frame(map_id) >= 2 && tilemap_get_frame(map_id) < 4)
{
    global.activate = true;
}
else
{
    global.activate = false;
}

위의 코드는 주어진 레이어에서 주어진 타일 맵에 대한 현재 애니메이션 프레임을 확인하고 반환 값을 기반으로 전역 변수를 설정합니다.
