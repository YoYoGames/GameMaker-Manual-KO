tilemap_set_at_pixel

이 기능은 레이어의 타일 맵 요소 내의 셀을 룸 내에서 변경되는 타일의 실제 위치를 사용하여 새 타일로 설정하는 데 사용할 수 있습니다. 타일 맵 요소 ID(다음을 사용하여 타일 맵 요소를 생성할 때 얻는layer_tilemap_create() 또는 기능을 사용할 때layer_tilemap_get_id() ), 타일 데이터 및 방 내 위치를 설정합니다. 함수를 사용하여 타일 데이터를 검색할 수 있습니다.tilemap_get_at_pixel() 그런 다음 사용 tile_ 이 기능을 사용하여 다시 설정하기 전에 변경하는 기능입니다. 함수가 반환됩니다 true 타일이 성공적으로 설정되고 false 문제가 있고 설정되지 않은 경우.

통사론:

tilemap_set_at_pixel(tilemap_element_id, tiledata, x, y)

논쟁 유형 설명
tilemap_element_id 타일 맵 요소 ID The unique ID value of the tile map element to change
tiledata 타일 데이터(실제) The tile set data to set
xcell 진짜 The x position (in the room)
ycell 진짜 The y position (in the room)

보고:

Boolean

예시:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 4, 4);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 4, 4);

위의 코드는 "Tiles_Sky" 레이어의 타일 맵에 대한 ID를 가져온 다음 이를 사용하여 위치 (4, 4)의 타일에서 데이터를 가져옵니다. 이 타일 데이터는 타일 맵에서 타일을 다시 설정하는 데 사용되기 전에 뒤집힙니다.
