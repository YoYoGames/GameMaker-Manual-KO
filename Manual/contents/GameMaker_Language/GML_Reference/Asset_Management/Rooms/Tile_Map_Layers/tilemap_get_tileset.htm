tilemap_get_tileset

이 기능을 사용하여 레이어의 주어진 타일 맵 요소에 할당된 타일 세트 자산의 인덱스 값을 검색할 수 있습니다. 타일 맵 요소 ID(사용하여 타일 맵 요소를 생성할 때 얻는layer_tilemap_create() 또는 기능을 사용할 때layer_tilemap_get_id() ), 함수는 타일 세트 자산 인덱스를 반환합니다.

통사론:

tilemap_get_tileset(tilemap_element_id);

논쟁 유형 설명
tilemap_element_id 타일 맵 요소 ID The unique ID value of the Tile Map element to get the tile set from

보고:

Tile Set Asset

예시:

var lay_id = layer_get_id("Tiles_trees");
var map_id = layer_tilemap_get_id(lay_id);
if tilemap_get_tileset(map_id) != ts_Nighttime
{
    tilemap_tileset(map_id, ts_Nighttime);
}

위의 코드는 "Tiles_trees" 레이어에 할당된 현재 타일 세트를 확인하고 "ts_Nighttime"이 아닌 경우 해당 타일 세트가 타일 맵에 할당됩니다.
