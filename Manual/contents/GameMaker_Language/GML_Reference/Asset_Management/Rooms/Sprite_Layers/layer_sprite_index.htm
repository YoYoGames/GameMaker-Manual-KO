layer_sprite_index

이 함수는 레이어에 여러 개의 하위 이미지가 있는 스프라이트 자산의 이미지 인덱스를 설정하는 데 사용할 수 있습니다. 스프라이트 요소 ID를 제공합니다(다음을 사용하여 스프라이트 요소를 생성할 때 얻습니다.layer_sprite_create() 또는 기능을 사용할 때layer_sprite_get_id() )를 선택한 다음 사용할 이미지 인덱스를 설정합니다. 하위 이미지 범위를 벗어나는 값을 설정하면 이미지 인덱스가 반복됩니다.

통사론:

layer_sprite_index(sprite_element_id, image_index);

논쟁 유형 설명
sprite_element_id 스프라이트 자산 The unique ID value of the sprite element to set
index 진짜 The image index to use for the sprite

보고:

N/A

예시:

var lay_id = layer_get_id("sprite_trees");
var spr_id = layer_sprite_get_id(lay_id, "gfc_trees");
layer_sprite_index(spr_id, 1);

위의 코드는 "sprite_trees"라는 레이어에 대한 레이어 ID를 얻은 다음 이를 사용하여 해당 레이어에 있는 주어진 스프라이트 요소의 ID를 가져옵니다. 이 ID는 요소 이미지 인덱스를 변경하는 데 사용됩니다.
