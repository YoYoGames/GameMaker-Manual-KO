layer_sprite_speed

이 함수는 스프라이트 요소에 할당된 애니메이션 이미지의 속도를 설정하는 데 사용할 수 있습니다. 스프라이트 요소 ID를 제공합니다(다음을 사용하여 스프라이트 요소를 생성할 때 얻습니다.layer_sprite_create() 또는 기능을 사용할 때layer_sprite_get_id() )를 클릭한 다음 애니메이션 속도를 설정합니다. IDE가 게임 프레임당 프레임으로 계산되는 데 사용된 스프라이트의 애니메이션 속도를 설정한 경우 주어진 속도 값은 승수 이며 1이 기본값이고 0.5로 설정하면 애니메이션 속도의 절반이 됩니다. 1.5로 하면 2배가 됩니다. 그러나 IDE에서 초당 프레임으로 계산되는 데 사용된 스프라이트의 애니메이션 속도를 설정한 경우 지정된 속도 값은 문자 그대로 초당 프레임 값이므로 예를 들어 이 값을 5로 설정하면 애니메이션이 루프로 설정됩니다. 게임이 실행되는 초당 5프레임. 사용된 스프라이트에 하위 이미지가 없으면 이 기능은 효과가 없습니다.

통사론:

layer_sprite_speed(sprite_element_id, speed);

논쟁 유형 설명
sprite_element_id 스프라이트 자산 The unique ID value of the sprite element to change
speed 진짜 The speed multiplier/value to use for the element

보고:

N/A

예시:

var lay_id = layer_get_id("Assets_water");
var spr_id = layer_sprite_get_id(lay_id, "gfc_water");
layer_sprite_speed(spr_id, 0.5);

위의 코드는 "Assets_water"라는 레이어에 대한 레이어 ID를 얻은 다음 이를 사용하여 해당 레이어에 있는 주어진 스프라이트 요소의 ID를 가져옵니다. 이 ID는 요소 이미지 속도를 변경하는 데 사용됩니다.
