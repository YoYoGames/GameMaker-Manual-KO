layer_sprite_get_alpha

이 함수는 스프라이트 요소의 알파 값을 가져오는 데 사용할 수 있습니다. 스프라이트 요소 ID를 제공합니다(다음을 사용하여 스프라이트 요소를 생성할 때 얻습니다.layer_sprite_create() 또는 기능을 사용할 때layer_sprite_get_id() ), 함수는 0(완전 투명)과 1(완전 불투명) 사이의 값을 반환합니다.

통사론:

layer_sprite_get_alpha(sprite_element_id);

논쟁 유형 설명
sprite_element_id The unique ID value of the sprite element to get the information from

보고:

(from 0 to 1)

예시:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_alpha(spr_id) < 0.1)
{
    layer_sprite_destroy(spr_id);
}

위의 코드는 "sprite_sky"라는 레이어의 레이어 ID를 얻은 다음 이를 사용하여 해당 레이어의 스프라이트 요소 ID를 가져옵니다. 그런 다음 이 ID는 요소 알파를 확인하는 데 사용되며 0.1보다 작으면 레이어 요소가 소멸됩니다.
