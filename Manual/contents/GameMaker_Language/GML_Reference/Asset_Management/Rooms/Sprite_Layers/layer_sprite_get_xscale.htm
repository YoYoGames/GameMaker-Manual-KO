layer_sprite_get_xscale

이 함수는 스프라이트 요소의 x축을 따라 현재 스케일 승수를 얻는 데 사용할 수 있습니다. 스프라이트 요소 ID를 제공합니다(다음을 사용하여 스프라이트 요소를 생성할 때 얻습니다.layer_sprite_create() 또는 기능을 사용할 때layer_sprite_get_id() ), 그리고 함수는 x 스케일에 대한 실제 값을 반환합니다. 여기서 1은 스케일링이 없습니다.

통사론:

layer_sprite_get_xscale(sprite_element_id);

논쟁 유형 설명
sprite_element_id The unique ID value of the sprite element to get the information from

보고:

예시:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_xscale(spr_id) != 1)
{
    layer_sprite_xscale(spr_id, 1);
}

위의 코드는 "sprite_sky"라는 레이어의 레이어 ID를 얻은 다음 이를 사용하여 해당 레이어의 스프라이트 요소 ID를 가져옵니다. 이 ID는 xscale 값을 확인하는 데 사용되며 1이 아닌 경우 1로 설정됩니다.
