layer_sprite_get_blend

이 함수는 스프라이트 요소의 블렌드 색상을 얻는 데 사용할 수 있습니다. 스프라이트 요소 ID를 제공합니다(다음을 사용하여 스프라이트 요소를 생성할 때 얻습니다.layer_sprite_create() 또는 기능을 사용할 때layer_sprite_get_id() ), 함수는 할당된 색상을 나타내는 실제 값을 반환합니다.

통사론:

layer_sprite_get_blend(sprite_element_id);

논쟁 유형 설명
sprite_element_id The unique ID value of the sprite element to get the information from

보고:

Colour Constant

예시:

var lay_id = layer_get_id("sprite_sky");
var back_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_blend(back_id) == c_white
{
    layer_sprite_blend(back_id, make_colour_rgb(random(255), random(255), 255));
}

위의 코드는 "sprite_sky"라는 레이어의 레이어 ID를 얻은 다음 이를 사용하여 해당 레이어의 스프라이트 요소 ID를 가져옵니다. 이 ID는 요소 블렌드 색상을 확인하는 데 사용되며 상수와 동일한 경우 c_white, 요소 혼합이 임의의 색상으로 설정됩니다.
