layer_sprite_alpha

이 함수는 자산 레이어에 있는 스프라이트의 알파(투명도)를 제어합니다. 스프라이트 요소 ID를 제공합니다(다음을 사용하여 스프라이트 요소를 생성할 때 얻습니다.layer_sprite_create() 또는 기능을 사용할 때layer_sprite_get_id() )를 클릭한 다음 사용할 알파 값을 설정합니다. 알파는 0(완전 투명)과 1(완전 불투명) 사이일 수 있으며 스프라이트 요소의 기본 알파 값은 1입니다. 스프라이트 요소가 할당된 레이어가 표시되지 않거나 요소 자체가 만들어진 경우 보이지 않음 - 레이어나 요소가 다시 표시될 때까지 이 기능의 차이를 볼 수 없습니다.

통사론:

layer_sprite_alpha(sprite_element_id, alpha);

논쟁 유형 설명
sprite_element_id The unique ID value of the sprite element to change
alpha The alpha for sprite sprite, from 0 to 1 (default is 1)

보고:

예시:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_alpha(spr_id, random(1));

위의 코드는 "Asset_sky" 레이어에 할당된 "Clouds"라는 스프라이트 자산의 ID 값을 가져온 다음 해당 알파를 0과 1 사이의 임의 값으로 설정합니다.
