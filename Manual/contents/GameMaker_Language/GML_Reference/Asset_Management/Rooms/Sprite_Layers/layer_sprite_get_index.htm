layer_sprite_get_index

이 함수는 스프라이트 요소의 현재 이미지 인덱스 값을 가져오는 데 사용할 수 있습니다. 스프라이트 요소 ID를 제공합니다(다음을 사용하여 스프라이트 요소를 생성할 때 얻습니다.layer_sprite_create() 또는 기능을 사용할 때layer_sprite_get_id() ), 그리고 이 함수는 스프라이트에 대해 표시되는 이미지 인덱스를 나타내는 실제 값을 반환합니다. 스프라이트 요소가 없거나 요소에 유효한 스프라이트가 할당되지 않은 경우 함수는 -1을 반환합니다.

통사론:

layer_sprite_get_index(sprite_element_id);

논쟁 유형 설명
sprite_element_id The unique ID value of the sprite element to get the information from

보고:

Real or -1

예시:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_index(spr_id) < 4)
{
    layer_sprite_index(spr_id, 4);
}

위의 코드는 "sprite_sky"라는 레이어의 레이어 ID를 얻은 다음 이를 사용하여 해당 레이어의 스프라이트 요소 ID를 가져옵니다. 그런 다음 이 ID는 요소의 이미지 인덱스가 4보다 작은지 확인하는 데 사용되며, 그렇다면 4로 설정됩니다.
