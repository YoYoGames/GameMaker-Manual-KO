layer_sprite_get_angle

이 함수는 스프라이트 요소의 현재 각도를 얻는 데 사용할 수 있습니다. 스프라이트 요소 ID를 제공합니다(다음을 사용하여 스프라이트 요소를 생성할 때 얻습니다.layer_sprite_create() 또는 기능을 사용할 때layer_sprite_get_id() ), 함수는 각도에 대한 실제 값을 반환합니다. 이 값은 0에서 360 사이이며 0은 오른쪽, 90은 위쪽, 180은 왼쪽, 270은 아래쪽을 나타냅니다.

통사론:

layer_sprite_get_angle(sprite_element_id);

논쟁 유형 설명
sprite_element_id The unique ID value of the sprite element to get the information from

보고:

Real

예시:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_angle(spr_id) != 0)
{
    layer_sprite_yscale(spr_id, 0);
}

위의 코드는 "sprite_sky"라는 레이어의 레이어 ID를 얻은 다음 이를 사용하여 해당 레이어의 스프라이트 요소 ID를 가져옵니다. 이 ID는 각도를 확인하는 데 사용되며 0이 아닌 경우 0으로 설정됩니다.
