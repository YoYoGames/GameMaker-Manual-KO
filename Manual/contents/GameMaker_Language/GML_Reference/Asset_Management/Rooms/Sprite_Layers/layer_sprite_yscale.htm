layer_sprite_yscale

이 기능을 사용하면 레이어의 주어진 스프라이트 요소가 y축을 따라 크기 조정되어야 하는지 여부를 변경할 수 있습니다. 스프라이트 요소 ID를 제공합니다(다음을 사용하여 스프라이트 요소를 생성할 때 얻습니다.layer_sprite_create() 또는 기능을 사용할 때layer_sprite_get_id() ), 스케일 값을 설정합니다. 스케일 1은 스케일링이 없음을 나타내고(1:1), 작은 값은 스케일 다운(예: 0.5는 사용된 스프라이트 높이의 절반), 큰 값은 스케일 업, 음수 값은 스프라이트를 미러링하여 스케일링합니다. 사용된 값이 정확히 -1이 아닌 경우(이 경우 사용된 스프라이트는 크기 조정 없이 (0, 0) 위치에 대해 위에서 아래로 미러링됩니다).

통사론:

layer_sprite_yscale(sprite_element_id, yscale);

논쟁 유형 설명
sprite_element_id The unique ID value of the sprite element to change
yscale The yscale value (default is 1)

보고:

예시:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1
{
    layer_sprite_xscale(asset_sprite, 1);
    layer_sprite_yscale(asset_sprite, 1);
}

위의 코드는 레이어에 할당된 스프라이트 요소가 코드를 실행하는 인스턴스가 켜져 있는지 확인하고 어느 방향으로든 크기가 조정되면 x축 크기와 y축 크기를 모두 1로 설정합니다.
