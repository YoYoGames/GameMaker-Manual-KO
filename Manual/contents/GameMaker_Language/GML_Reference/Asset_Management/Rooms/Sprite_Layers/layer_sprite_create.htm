layer_sprite_create

이 기능을 사용하면 프로젝트에서 사용할 레이어에 스프라이트 리소스를 할당할 수 있습니다. 레이어 ID를 제공합니다(다음을 사용하여 레이어를 생성할 때 얻는layer_create() 또는 레이어 이름을 함께 사용할 때layer_get_id() ), 룸 내의 위치 및 스프라이트 인덱스(자산 브라우저에 표시된 스프라이트의 이름이 됨)가 레이어에 추가됩니다. 이 함수는 요소에 대한 고유 ID 값을 반환하며, 이는 스프라이트에 대한 추가 계층 함수에서 사용할 수 있습니다.

통사론:

layer_sprite_create(layer_id, x, y, sprite)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the layer to target
x 진짜 The x position to use
y 진짜 The y position to use
sprite 스프라이트 자산 The sprite index to be used

보고:

Sprite Element ID

예시:

global.asset_layer = layer_create(10000);
for (var i = 0; i< 10; i++;)
{
    var _x = random(room_width);
    var _y = room_height - 100;
    global.asset_spr_trees[i] = layer_sprite_create(global.asset_layer, _x, _y, spr_Trees);
}

위의 코드는 새 레이어를 만든 다음 여기에 10개의 새 스프라이트 요소를 추가하여 각 요소의 ID를 배열에 저장합니다.
