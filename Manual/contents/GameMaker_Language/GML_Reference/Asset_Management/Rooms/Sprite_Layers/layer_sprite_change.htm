layer_sprite_change

이 기능을 사용하면 레이어의 주어진 스프라이트 요소에 할당된 스프라이트 리소스를 변경할 수 있습니다. 스프라이트 요소 ID를 제공합니다(다음을 사용하여 스프라이트 요소를 생성할 때 얻습니다.layer_sprite_create() 또는 기능을 사용할 때layer_sprite_get_id() ), 스프라이트 인덱스 요소 자산을 제공하면 변경됩니다. 스프라이트에 하위 이미지가 있으면 애니메이션도 적용됩니다(이는 다음을 사용하여 제어할 수 있습니다.layer_sprite_index() 그리고layer_sprite_speed() 기능). 요소가 여전히 존재하고 나중에 다시 변경할 수 있지만 새 스프라이트 인덱스로 값 -1을 할당할 수 있으며 스프라이트가 표시되지 않습니다.

통사론:

layer_sprite_change(sprite_element_id, sprite_index)

논쟁 유형 설명
sprite_element_id 스프라이트 요소 ID The unique ID value of the sprite element to change
sprite_index 스프라이트 자산 The new sprite index to use

보고:

N/A

예시:

var s = layer_sprite_get_id(layer, global.Asset_sprite);
if layer_sprite_get_sprite(s) != spr_Nighttime
{
    layer_sprite_change(s, spr_nighttime);
}

위의 코드는 전역 변수에 ID가 저장된 요소의 스프라이트 인덱스를 확인합니다. global.Asset_sprite "spr_Nighttime"이 아니면 해당 스프라이트가 요소에 할당됩니다.
