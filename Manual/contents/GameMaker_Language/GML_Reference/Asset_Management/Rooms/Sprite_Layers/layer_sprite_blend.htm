layer_sprite_blend

이 함수는 스프라이트 스프라이트의 혼합(또는 "색조")을 제어하고 기본값은 -1(상수를 나타냅니다. c_white, 도 사용할 수 있습니다). 다른 모든 값(예: c_red, 또는 c_aqua ) 지정된 색상을 원본 스프라이트와 혼합합니다. 스프라이트 요소 ID를 제공합니다(다음을 사용하여 스프라이트 요소를 생성할 때 얻습니다.layer_sprite_create() 또는 기능을 사용할 때layer_sprite_get_id() )를 클릭한 다음 사용할 혼합 색상을 설정합니다. 아래에서 다양한 색상으로 혼합된 스프라이트의 예를 볼 수 있습니다.

HTML5 플랫폼에서 혼합을 제한해야 합니다(WebGL을 사용하지 않는 한). 혼합된 각 스프라이트는 개별적으로 캐시되어야 하고 혼합된 스프라이트가 많으면 성능에 부정적인 영향을 미칠 수 있습니다(함수를 사용하여 캐시 크기를 설정할 수도 있습니다). sprite_set_cache_size() ).

통사론:

layer_sprite_blend(sprite_element_id, blend);

논쟁 유형 설명
sprite_element_id The unique ID value of the sprite element to change
blend The colour to blend with the sprite sprite (default is c_white )

보고:

예시:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_blend(spr_id, c_gray);

위의 코드는 "Asset_sky" 레이어에 할당된 "Clouds"라는 스프라이트의 ID 값을 가져온 다음 색상으로 착색합니다.
