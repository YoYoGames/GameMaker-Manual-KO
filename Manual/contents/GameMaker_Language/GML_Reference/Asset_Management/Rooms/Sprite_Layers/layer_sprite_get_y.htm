layer_sprite_get_y

이 함수는 방에 있는 스프라이트 요소의 y 위치를 얻는 데 사용할 수 있습니다. 스프라이트 요소 ID를 제공합니다(다음을 사용하여 스프라이트 요소를 생성할 때 얻습니다.layer_sprite_create() 또는 기능을 사용할 때layer_sprite_get_id() ), 함수는 y 위치 값을 반환합니다.

통사론:

layer_sprite_get_y(sprite_element_id);

논쟁 유형 설명
sprite_element_id The unique ID value of the sprite element to get the information from

보고:

예시:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_y(spr_id) < 0
{
    layer_sprite_y(spr_id, room_height);
}

위의 코드는 "sprite_sky"라는 레이어의 레이어 ID를 얻은 다음 이를 사용하여 해당 레이어의 스프라이트 요소 ID를 가져옵니다. 그런 다음 이 ID를 사용하여 요소 y 위치를 확인하고 0보다 작으면 레이어 요소를 다른 y 위치로 이동합니다.
