layer_sprite_destroy

이 함수는 주어진 스프라이트 요소를 파괴합니다. 스프라이트 ID(다음을 사용하여 스프라이트를 생성할 때 얻는layer_sprite_create() 또는 레이어 ID를 함께 사용할 때layer_get_sprite_id() ) 그러면 제거됩니다. 이것은 레이어를 제거하지 않고 레이어에서 스프라이트만 제거하며 스프라이트가 룸 편집기에 추가된 것이라면 다음에 룸을 나왔다가 돌아올 때 스프라이트가 다시 생성됩니다. 그러나 방이 지속되는 경우 방 지속성이 다시 꺼지지 않는 한 스프라이트가 제거됩니다.

통사론:

layer_sprite_destroy(sprite_element_id)

논쟁 유형 설명
sprite_element_id The unique ID value of the sprite to be destroyed

보고:

예시:

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.Asset_spr_trees)
{
    layer_sprite_destroy(lay_id);
}

위의 코드는 "Asset_Trees" 레이어를 확인하여 주어진 스프라이트 요소가 존재하는지 확인하고 존재한다면 파괴됩니다(레이어는 아님).
