layer_sprite_y

이 함수는 레이어에서 자산 스프라이트 요소의 방의 y축을 따라 위치를 제어합니다. 스프라이트 요소 ID를 제공합니다(다음을 사용하여 스프라이트 요소를 생성할 때 얻습니다.layer_sprite_create() 또는 기능을 사용할 때layer_sprite_get_id() ), 사용할 y 값을 설정합니다(방 좌표 기준).

통사론:

layer_sprite_y(sprite_element_id, y);

논쟁 유형 설명
sprite_element_id The unique ID value of the sprite element to change
y The y position for the sprite

보고:

예시:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_y(spr_id, 50 + irandom(room_height - 200));

위의 코드는 "Asset_sky" 레이어에 할당된 스프라이트 자산 "Clouds"의 ID 값을 가져온 다음 y 위치를 50에서 방 높이에서 200을 뺀 값 사이의 임의 값으로 설정합니다.
