layer_sprite_angle

이 기능을 사용하면 레이어에서 주어진 스프라이트 요소의 각도를 변경할 수 있습니다. 스프라이트 요소 ID를 제공합니다(다음을 사용하여 스프라이트 요소를 생성할 때 얻습니다.layer_sprite_create() 또는 기능을 사용할 때layer_sprite_get_id() )을 누른 다음 시계 반대 방향으로 0에서 359까지 각도 값을 설정합니다. 여기서 0은 오른쪽, 90은 위쪽, 180은 왼쪽, 270은 아래쪽입니다. 360보다 큰 값을 설정하면 0 - 359 범위 내로 가져오기 위해 반복됩니다.

통사론:

layer_sprite_angle(sprite_element_id, angle);

논쟁 유형 설명
sprite_element_id 스프라이트 요소 ID The unique ID value of the sprite element to change
angle 진짜 The angle of the sprite (default is 0)

보고:

N/A

예시:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_angle(asset_sprite) != 0
{
    layer_sprite_angle(asset_sprite, 0);
}

위의 코드는 레이어에 할당된 스프라이트 요소가 코드를 실행하는 인스턴스가 켜져 있는지 확인하고 각도가 0이 아니면 0으로 설정합니다.
