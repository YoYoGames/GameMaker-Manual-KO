layer_sprite_exists

이 기능을 사용하여 주어진 레이어에 스프라이트 요소가 있는지 확인할 수 있습니다. 레이어 ID를 제공합니다(다음을 사용하여 레이어를 생성할 때 얻는layer_create() ) 또는 레이어 이름(문자열로 - 성능에 영향을 미침) 및 스프라이트 요소 ID(다음을 사용하여 스프라이트 요소를 생성할 때 가져옴)layer_sprite_create() 또는 기능을 사용할 때layer_sprite_get_id() ) 함수는 다음 중 하나를 반환합니다. true 요소가 존재하거나 false 그렇지 않은 경우.

참고: 이 함수는 현재 대상 방의 범위 내에서 작동합니다. 기본적으로 함수가 호출되는 방은 함수를 사용하여 설정할 수 있습니다. layer_set_target_room().

통사론:

layer_sprite_exists(layer_id, sprite_element_id)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the layer to target (or the layer name as a string)
sprite_element_id 스프라이트 요소 ID The unique ID value of the sprite element to check

보고:

Boolean

예시:

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.TreesSprites)
{
    layer_sprite_destroy(lay_id, global.TreesSprites);
}

위의 코드는 "Asset_trees" 레이어를 확인하여 주어진 스프라이트 요소가 존재하는지 확인하고 존재하면 소멸됩니다(레이어 제외).
