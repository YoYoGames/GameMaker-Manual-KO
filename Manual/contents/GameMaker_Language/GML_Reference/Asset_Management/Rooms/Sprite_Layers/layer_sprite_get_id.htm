layer_sprite_get_id

이 함수는 레이어에 있는 스프라이트 요소의 고유 ID 값을 검색하는 데 사용할 수 있습니다. 레이어 ID를 제공합니다(다음을 사용하여 레이어를 생성할 때 얻는layer_create() 또는 레이어 이름을 함께 사용할 때layer_get_id() ) 및 룸 편집기에 정의된 스프라이트 요소의 이름입니다. 이 함수는 레이어의 해당 스프라이트 요소와 연결된 ID 값을 반환합니다.

이 함수는 자산 계층의 IDE에 추가된 스프라이트와 함께 사용하도록 특별히 설계되었으며 이 함수를 사용하여 코드를 통해 계층에 스프라이트를 추가한 경우layer_sprite_create(), 그런 다음 해당 함수에서 반환된 ID는 모든 향후 참조에 사용해야 합니다(해당 스프라이트 요소에는 이 함수에 전달할 이름이 없기 때문입니다).

지정된 레이어가 존재하지 않거나 주어진 스프라이트 요소를 찾을 수 없으면 함수는 -1을 반환합니다.

통사론:

layer_sprite_get_id(layer_id, sprite_element_name)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the layer to target
sprite_element_name 끈 The unique name of the sprite element on the layer as defined in the Room Editor

보고:

Sprite Element ID

예시:

var lay_id = layer_get_id("Assets_trees");
var back_id = layer_sprite_get_id(lay_id, "graphic_254367CB");
layer_sprite_change(back_id, spr_Trees_Winter);

위의 코드는 "Assets_trees"라는 레이어의 레이어 ID를 가져온 다음 해당 레이어에서 스프라이트 요소 "graphic_254367CB"의 ID를 검색하는 데 사용합니다. 검색된 스프라이트 요소 ID는 요소의 스프라이트를 변경하는 데 사용됩니다.
