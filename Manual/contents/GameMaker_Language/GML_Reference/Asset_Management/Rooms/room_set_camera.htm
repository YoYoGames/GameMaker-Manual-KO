room_set_camera

이 기능을 사용하면 현재 뷰포트가 아닌 다른 방의 특정 뷰포트에 카메라를 할당할 수 있습니다. 방 인덱스, 보기 인덱스(0에서 7까지)를 제공한 다음 사용할 카메라의 인덱스를 제공합니다.

통사론:

room_set_camera(rm, vind, camera);

논쟁 유형 설명
rm 방 자산 The index of the room to set the view camera of
vind 진짜 The index of the view port to assign the camera to
camera 카메라 ID The index of the camera to assign

보고:

N/A

예시:

global.myroom = room_add();
room_set_camera(global.myroom, 0, global.MainCam);

위의 코드는 새로 생성된 방의 카메라를 뷰포트 [0]에 할당합니다.
