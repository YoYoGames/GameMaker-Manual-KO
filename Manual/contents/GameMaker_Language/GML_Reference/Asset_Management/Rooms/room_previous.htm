room_previous

이 함수를 사용하면 함수에 방이 입력 되기 전에 방의 인덱스를 검색할 수 있습니다. 예를 들어 변수를 사용할 수 있습니다room 현재 방의 인덱스를 가져온 다음 이 기능을 사용하여 Room Manager 에 나열된 대로 그 앞에 오는 방을 찾습니다. 입력한 공간 이전에 공간이 없으면 -1이 반환됩니다. 이 기능은 다음을 사용하여 동적으로 추가된 회의실을 인식하거나 고려하지 않습니다.room_add() 또는room_duplicate().

통사론:

room_previous(numb);

논쟁 유형 설명
numb The index of the room to get the one before from.

보고:

Room Asset

예시:

if room_previous(room) != -1
{
    room_goto_previous();
}

위의 코드는 이전 방이 존재하는지 확인하고 존재한다면 그 방으로 이동합니다.
