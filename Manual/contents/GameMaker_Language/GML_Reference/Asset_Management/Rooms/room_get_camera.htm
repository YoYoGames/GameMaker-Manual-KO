room_get_camera

이 기능을 사용하면 현재 방이 아닌 다른 방의 특정 보기에 할당된 카메라의 고유 인덱스 ID를 얻을 수 있습니다. 사용할 공간, 사용할 뷰 포트(0에서 7까지)를 지정하면 함수가 카메라 인덱스를 반환합니다.

통사론:

room_get_camera(rm, vind);

논쟁 유형 설명
rm 방 자산 The index of the room to get the view camera of
vind 진짜 The index of the view port to get the camera of

보고:

Camera ID

예시:

var cam = room_get_camera(rm_Game, 0);
if cam != global.MainCam
{
    room_set_camera(rm_Game, 0, global.MainCam);
}

위의 코드는 새로 생성된 방의 카메라를 뷰포트 [0]에 할당합니다.
