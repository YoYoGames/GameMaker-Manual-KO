room_persistent

이 변수는 현재 방에 대한 영구 플래그를 가져오고 설정하는 데 사용할 수 있습니다. 로 설정하면 true 룸은 영구적인 것으로 간주되며, 이 경우 룸을 나왔다가 다시 돌아올 때마다 해당 룸 내의 인스턴스 상태가 유지됩니다. 그러나 다음과 같이 플래그가 지정된 경우 false, 방으로 돌아갈 때마다 초기 상태로 재설정됩니다. 퍼시스턴트 룸은 일반 룸보다 훨씬 더 많은 메모리를 사용하므로 게임에 너무 많은 메모리를 사용하지 않는 것이 좋습니다.

통사론:

room_persistent;

보고:

예시:

if lives < 1
{
    room_persistent = false;
    room_goto(rm_start);
}

위의 코드는 "lives" 변수를 확인하고 1보다 작으면 방 지속성을 다음으로 설정합니다. false 그리고 방을 바꾼다.
