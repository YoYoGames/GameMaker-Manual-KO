layer_background_get_blend

이 함수는 배경 요소의 블렌드 색상을 얻는 데 사용할 수 있습니다. 배경 요소 ID(다음을 사용하여 배경 요소를 만들 때 얻는layer_background_create() 또는 기능을 사용할 때layer_background_get_id() ), 함수는 할당된 색상을 나타내는 실제 값을 반환합니다.

통사론:

layer_background_get_blend(background_element_id);

논쟁 유형 설명
background_element_id The unique ID value of the background element to get the information from

보고:

Colour Constant

예시:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_blend(back_id) == c_white
{
    layer_background_blend(back_id, make_colour_rgb(random(255), random(255), 255));
}

위의 코드는 "Background_sky"라는 레이어의 레이어 ID를 가져온 다음 해당 레이어의 배경 요소 ID를 가져오는 데 사용합니다. 이 ID는 요소 블렌드 색상을 확인하는 데 사용되며 상수와 동일한 경우 c_white, 레이어 혼합이 임의의 색상으로 설정됩니다.
