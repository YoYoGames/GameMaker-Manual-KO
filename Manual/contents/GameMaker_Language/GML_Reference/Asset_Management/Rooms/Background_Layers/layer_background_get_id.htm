layer_background_get_id

이 함수는 레이어에 있는 배경 요소의 고유한 ID 값을 검색하는 데 사용할 수 있습니다. 레이어 ID를 제공합니다(레이어 이름을layer_get_id() ) 이 함수는 레이어의 배경 요소와 연결된 ID 값을 반환합니다. 이 기능은 IDE에 추가된 배경과 함께 사용하도록 특별히 설계되었습니다. 마치 기능을 사용하여 코드를 통해 레이어에 배경을 추가하는 것과 같습니다.layer_background_create(), 추가된 배경 요소의 고유 ID를 반환합니다.

통사론:

layer_background_get_id(layer_id)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the layer to target

보고:

Background Element ID

예시:

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_sprite(back_id, bck_Trees_Winter);

위의 코드는 "Background_trees"라는 레이어의 레이어 ID를 가져온 다음 해당 레이어의 배경 요소 ID를 가져오는 데 사용합니다. 이 ID는 요소 스프라이트를 변경하는 데 사용됩니다.
