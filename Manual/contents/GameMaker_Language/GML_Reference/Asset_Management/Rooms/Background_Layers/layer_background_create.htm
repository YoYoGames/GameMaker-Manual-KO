layer_background_create

이 기능을 사용하면 프로젝트에서 배경으로 사용할 스프라이트 리소스를 레이어에 할당할 수 있습니다. 레이어 ID를 제공합니다(다음을 사용하여 레이어를 생성할 때 얻는layer_create() ) 또는 레이어 이름(문자열로 - 성능에 영향을 미침) 및 스프라이트 인덱스(자산 브라우저에 표시되는 스프라이트의 이름이 됨)가 레이어에 추가됩니다. 이 함수는 요소에 대한 고유한 ID 값을 반환하며, 이 값은 배경에 대한 추가 레이어 기능에서 사용할 수 있습니다.

통사론:

layer_background_create(layer_id, sprite)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the layer to target (or the layer name as a string)
sprite 스프라이트 자산 The sprite index to be used

보고:

Background Element ID

예시:

global.back_layer = layer_create(10000);
global.back_trees = layer_background_create(global.back_layer, spr_Trees);

위의 코드는 새 레이어를 만든 다음 여기에 새 배경 요소를 추가하여 스프라이트를 사용된 배경 이미지로 설정합니다.
