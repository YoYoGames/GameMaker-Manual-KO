layer_background_index

이 함수는 여러 개의 하위 이미지가 있는 배경 스프라이트의 이미지 인덱스를 설정하는 데 사용할 수 있습니다. 배경 요소 ID(다음을 사용하여 배경 요소를 만들 때 얻는layer_background_create() 또는 기능을 사용할 때layer_background_get_id() )를 선택한 다음 사용할 이미지 인덱스를 설정합니다. 하위 이미지 범위를 벗어나는 값을 설정하면 이미지 인덱스가 반복됩니다.

통사론:

layer_background_index(background_element_id, image_index);

논쟁 유형 설명
background_element_id 배경 요소 ID The unique ID value of the background element to change
index 진짜 The image index to use for the background

보고:

N/A

예시:

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_index(back_id, 1);

위의 코드는 "Background_trees"라는 레이어의 레이어 ID를 가져온 다음 해당 레이어의 배경 요소 ID를 가져오는 데 사용합니다. 이 ID는 요소 이미지 인덱스를 변경하는 데 사용됩니다.
