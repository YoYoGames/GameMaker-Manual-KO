layer_background_get_stretch

이 함수는 배경 요소 스프라이트의 늘어난 상태를 가져오는 데 사용할 수 있습니다. 배경 요소 ID(다음을 사용하여 배경 요소를 만들 때 얻는layer_background_create() 또는 기능을 사용할 때layer_background_get_id() ), 함수는 다음 중 하나를 반환합니다. true 요소 스프라이트가 현재 방에 맞게 늘어나거나 false 그렇지 않은 경우.

통사론:

layer_background_get_stretch(background_element_id);

논쟁 유형 설명
background_element_id The unique ID value of the background element to get the information from

보고:

예시:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_stretch(back_id)
{
    layer_background_stretch(back_id, false);
}

위의 코드는 "Background_sky"라는 레이어의 레이어 ID를 가져온 다음 해당 레이어의 배경 요소 ID를 가져오는 데 사용합니다. 그런 다음 이 ID를 사용하여 요소 스프라이트가 방에 맞게 늘어날지 여부를 확인하고 확장되면 이 속성이 다음으로 설정됩니다. false.
