layer_background_get_htiled

이 기능은 배경 요소가 수평으로 바둑판식으로 배열되었는지 여부를 확인하는 데 사용할 수 있습니다. 배경 요소 ID(다음을 사용하여 배경 요소를 만들 때 얻는layer_background_create() 또는 기능을 사용할 때layer_background_get_id() ), 함수는 다음 중 하나를 반환합니다. true 요소가 현재 타일링된 경우 또는 false 그렇지 않은 경우.

통사론:

layer_background_get_htiled(background_element_id);

논쟁 유형 설명
background_element_id The unique ID value of the background element to get the information from

보고:

예시:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (!layer_background_get_htiled(back_id))
{
    layer_background_htiled(back_id, true);
}

위의 코드는 "Background_sky"라는 레이어의 레이어 ID를 가져온 다음 해당 레이어의 배경 요소 ID를 가져오는 데 사용합니다. 그런 다음 이 ID는 요소가 수평 축을 가로질러 타일링되는지 확인하는 데 사용되며 그렇지 않은 경우 그렇게 하도록 설정됩니다.
