layer_background_destroy

이 함수는 주어진 배경 요소를 파괴합니다. 백그라운드 ID를 제공합니다(다음을 사용하여 배경을 만들 때 가져옵니다.layer_background_create() 또는 레이어 ID를 함께 사용할 때layer_get_background_id() ) 그러면 제거됩니다. 이것은 레이어를 제거하지 않고 레이어의 배경만 제거하며 배경이 룸 편집기에서 추가된 것이라면 다음에 룸을 나왔다가 돌아올 때 배경이 다시 생성됩니다. 그러나 방이 지속되는 경우 방 지속성이 다시 꺼지지 않는 한 배경이 제거됩니다.

통사론:

layer_background_destroy(background_element_id)

논쟁 유형 설명
background_element_id 배경 요소 ID The unique ID value of the background element to be destroyed

보고:

N/A

예시:

var lay_id = layer_get_id("Background_trees");
var bck_id = layer_background_get_id("Forrest");
if (layer_background_exists(lay_id, bck_id))
{
    layer_background_destroy(bck_id);
}

위의 코드는 "Background_trees" 레이어를 확인하여 주어진 배경 요소가 존재하는지 확인하고 존재한다면 파괴됩니다(레이어는 제외).
