layer_background_sprite

이 함수를 사용하여 배경 요소의 스프라이트 인덱스를 설정할 수 있습니다. 배경 요소 ID를 제공합니다(다음을 사용하여 배경 요소를 생성할 때 얻는layer_background_create() 또는 기능을 사용할 때layer_background_get_id() ) 그런 다음 사용할 스프라이트 인덱스를 지정합니다. 배경 요소 이미지는 새 스프라이트로 대체됩니다. 값을 -1로 지정하면 요소에 스프라이트가 할당되지 않습니다(그러나 여전히 존재하며 나중에 다시 할당할 수 있음).

이 함수에는 별칭도 있습니다.layer_background_change().

통사론:

layer_background_sprite(background_element_id, sprite_index)

논쟁 유형 설명
background_element_id 배경 요소 ID The unique ID value of the background element to change
sprite_index 스프라이트 자산 The sprite index of the sprite to use for the background element

보고:

N/A

예시:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_sprite(back_id) != spr_Clouds)
{
    layer_background_sprite(back_id, spr_Clouds);
}

위의 코드는 "Background_sky"라는 레이어의 레이어 ID를 가져온 다음 해당 레이어의 배경 요소 ID를 가져오는 데 사용합니다. 그런 다음 이 ID는 요소에 할당된 스프라이트를 확인하는 데 사용되며, 아직 없는 경우 스프라이트 "spr_Clouds"로 설정합니다.
