layer_background_vtiled

이 기능을 사용하여 레이어의 지정된 배경 요소를 수직으로 바둑판식으로 배열할지 여부를 변경할 수 있습니다. 배경 요소 ID(다음을 사용하여 배경 요소를 만들 때 얻는layer_background_create() 또는 기능을 사용할 때layer_background_get_id() ), 타일링 값을 설정합니다. 로 설정하면 true 그러면 요소가 방 아래에 수직으로 타일링되어 표시되고 로 설정되면 false, 요소는 바둑판식으로 배열되지 않습니다. 이 기능은 2D 프로젝트에만 해당되며 3D 카메라 프로젝션을 사용할 때는 제대로 작동하지 않습니다.

통사론:

layer_background_vtiled(background_element_id, tiled);

논쟁 유형 설명
background_element_id 배경 요소 ID The unique ID value of the background element to change
tiled 부울 The vertical tiling toggle, which can be true or false

예시:

var back = layer_background_get_id(layer);
if !layer_background_get_htiled(back) || !layer_background_get_vtiled(back)
{
    layer_background_htiled(back, true);
    layer_background_vtiled(back, true);
}

위의 코드는 코드를 실행하는 인스턴스가 레이어에 할당된 배경 요소가 켜져 있는지 확인하고 어느 방향으로도 바둑판식으로 배열되지 않은 경우 세로 및 가로로 바둑판식으로 배열하도록 설정합니다.
