layer_background_xscale

이 기능은 배경 요소의 x축을 따라 축척을 설정하는 데 사용할 수 있습니다. 배경 요소 ID(다음을 사용하여 배경 요소를 만들 때 얻는layer_background_create() 또는 기능을 사용할 때layer_background_get_id() ), 스케일 값을 설정합니다. 제공하는 배율 값은 배경 요소가 표시되는 방식을 변경하는 데 사용되는 승수입니다. 여기서 값 0.5는 요소를 절반 배율로 표시하고 값 2는 이중 배율로 표시합니다. 음수 값이 유효하고 (0,0) 위치를 중심으로 요소를 "뒤집기"하므로 x 스케일이 -1이면 이미지가 반전되어 표시됩니다.

통사론:

layer_background_xscale(background_element_id, xscale);

논쟁 유형 설명
background_element_id The unique ID value of the background element to change
xscale The scale value to use (1 is no scaling)

보고:

예시:

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_xscale(back_id, -1);

위의 코드는 "Background_water"라는 레이어의 레이어 ID를 가져온 다음 해당 레이어의 배경 요소 ID를 가져오는 데 사용합니다. 이 ID는 xscale 요소를 변경하는 데 사용됩니다.
