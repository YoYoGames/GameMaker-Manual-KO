layer_background_speed

이 함수는 배경 요소에 할당된 애니메이션 스프라이트의 속도를 설정하는 데 사용할 수 있습니다. 배경 요소 ID(다음을 사용하여 배경 요소를 만들 때 얻는layer_background_create() 또는 기능을 사용할 때layer_background_get_id() )를 클릭한 다음 애니메이션 속도를 설정합니다. IDE가 게임 프레임당 프레임으로 계산되는 데 사용된 스프라이트의 애니메이션 속도를 설정한 경우 주어진 속도 값은 승수 이며 1이 기본값이고 0.5로 설정하면 애니메이션 속도의 절반이 됩니다. 2로 하면 2배가 됩니다. 그러나 IDE에서 초당 프레임으로 계산되는 데 사용된 스프라이트의 애니메이션 속도를 설정한 경우 지정된 속도 값은 문자 그대로 초당 프레임 값이므로 예를 들어 이 값을 5로 설정하면 애니메이션이 루프로 설정됩니다. 게임이 실행되는 초당 5프레임. 배경 요소에 연결된 스프라이트가 없거나 사용된 스프라이트에 하위 이미지가 없는 경우 이 기능은 효과가 없습니다.

통사론:

layer_background_speed(background_element_id, speed);

논쟁 유형 설명
background_element_id 배경 요소 ID The unique ID value of the background element to change
speed 진짜 The speed multiplier/value to use for the element

보고:

N/A

예시:

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_speed(back_id, 0.5);

위의 코드는 "Background_water"라는 레이어의 레이어 ID를 가져온 다음 해당 레이어의 배경 요소 ID를 가져오는 데 사용합니다. 이 ID는 요소 이미지 속도를 변경하는 데 사용됩니다.
