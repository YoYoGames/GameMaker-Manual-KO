layer_background_stretch

이 기능을 사용하면 배경 요소 스프라이트를 토글하여 공간에 맞게 늘이거나 해상도와 1:1을 유지할 수 있습니다. 배경 요소 ID를 제공합니다(다음을 사용하여 배경 요소를 생성할 때 얻는layer_background_create() 또는 기능을 사용할 때layer_background_get_id() ) 그런 다음 stretch 인수를 다음으로 설정합니다. true 또는 false. 로 설정하면 true 요소 스프라이트는 방에 맞게 늘어납니다(스프라이트와 방의 크기에 따라 확대 또는 축소). false, 요소 스프라이트는 기본 해상도로 표시됩니다.

통사론:

layer_background_stretch(background_element_id, stretch)

논쟁 유형 설명
background_element_id 배경 요소 ID The unique ID value of the background element to change
stretch 부울 The stretch toggle, which can be true or false

N/A

예시:

if mouse_check_button_pressed(mb_left)
{
    var back = layer_background_get_id(layer);
    if (layer_background_get_stretch(back))
    {
        layer_background_stretch(back, false);
    }
    else
    {
        layer_background_stretch(back, true);
    }
}

위의 코드는 마우스 버튼 누름을 확인하고 감지되면 코드를 실행하는 인스턴스가 켜진 레이어에 할당된 배경 요소 스프라이트의 스트레칭을 토글합니다.
