layer_background_get_yscale

이 함수는 배경 요소의 현재 배율 배율 값을 가져오는 데 사용할 수 있습니다. 배경 요소 ID(다음을 사용하여 배경 요소를 만들 때 얻는layer_background_create() 또는 기능을 사용할 때layer_background_get_id() ), 그리고 이 함수는 스프라이트를 그리는 데 사용되는 배율을 나타내는 실제 값을 반환합니다. 기본값은 1입니다.

통사론:

layer_background_get_yscale(background_element_id);

논쟁 유형 설명
background_element_id The unique ID value of the background element to get the information from

보고:

예시:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_yscale(back_id) != -1
{
    layer_background_yscale(back_id, -1);
}

위의 코드는 "Background_sky"라는 레이어의 레이어 ID를 가져온 다음 해당 레이어의 배경 요소 ID를 가져오는 데 사용합니다. 이 ID는 요소의 y 스케일을 확인하는 데 사용되며 -1이 아니면 -1로 설정됩니다.
