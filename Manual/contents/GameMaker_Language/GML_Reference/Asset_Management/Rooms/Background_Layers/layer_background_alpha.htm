layer_background_alpha

이 함수는 배경 스프라이트의 알파(투명도)를 제어합니다. 배경 요소 ID(다음을 사용하여 배경 요소를 만들 때 얻는layer_background_create() 또는 기능을 사용할 때layer_background_get_id() )를 클릭한 다음 사용할 알파 값을 설정합니다. 알파는 0(완전 투명)과 1(완전 불투명) 사이일 수 있으며 배경 요소의 기본 알파 값은 1입니다. 배경 요소가 할당된 레이어가 표시되지 않거나 요소 자체가 만들어진 경우 보이지 않음 - 레이어 또는 요소가 다시 표시될 때까지 이 기능의 차이를 볼 수 없습니다.

통사론:

layer_background_alpha(background_element_id, alpha);

논쟁 유형 설명
background_element_id The unique ID value of the background element to change
alpha The alpha for background sprite, from 0 to 1 (default is 1)

보고:

예시:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
layer_background_alpha(back_id, random(1));

위의 코드는 "Background_sky" 레이어에 할당된 배경의 ID 값을 가져온 다음 알파를 0과 1 사이의 임의 값으로 설정합니다.
