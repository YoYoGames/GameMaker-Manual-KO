room_get_viewport

이 기능을 사용하면 현재 방이 아닌 다른 방에 있는 뷰 포트의 세부 정보를 검색할 수 있습니다. 방 ID와 검색할 뷰 포트의 인덱스(0에서 7까지)를 제공하면 함수는 5개의 인덱스 배열 을 반환합니다. 여기서:

  • [0] = 표시됨(부울: true / false )
  • [1] = x 위치(실제)
  • [2] = y 위치(실제)
  • [3] = 너비(실제)
  • [4] = 높이(실제)

통사론:

room_get_viewport(rm, vind);

논쟁 유형 설명
rm 방 자산 The index of the room to get viewport data from
vind 진짜 The index of the view port to get

보고:

Array (5 elements: visible, x, y, width, height)

예시:

v_vals = room_get_viewport(rm_Game, 0);

if v_vals[0] == false
{
    room_set_view(rm_Game, true, v_vals[1], v_vals[2], v_vals[3], v_vals[4]);
}

위의 코드는 주어진 방에 대한 뷰 포트 데이터를 검색한 다음 포트가 보이는 것으로 플래그가 지정되어 있는지 확인합니다. 그렇지 않은 경우 뷰 포트 데이터가 표시되도록 설정됩니다.
