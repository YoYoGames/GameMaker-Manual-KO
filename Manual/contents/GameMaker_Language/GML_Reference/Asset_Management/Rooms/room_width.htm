room_width

이 변수는 현재 방의 너비를 픽셀 단위로 유지합니다. 이 변수를 변경하여 언제든지 방의 너비를 변경할 수 있습니다.

통사론:

room_width;

보고:

예시:

if bbox_right > room_width
{
    x += room_width - bbox_right;
}

위의 코드는 현재 인스턴스의 스프라이트 경계 상자가 방의 너비보다 큰지 확인하고 크면 인스턴스를 위로 이동합니다.
