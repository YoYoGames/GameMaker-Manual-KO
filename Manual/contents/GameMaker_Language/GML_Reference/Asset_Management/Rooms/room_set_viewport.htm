room_set_viewport

이 기능을 사용하면 현재 방을 제외한 게임의 모든 방에 대한 뷰 포트 속성을 설정할 수 있습니다. 뷰 포트를 설정할 룸 인덱스, 뷰 포트 인덱스(0에서 7 사이), 뷰 포트가 룸 시작 시 표시되어야 하는지 여부(로 설정됨)를 제공합니다. true 포트가 보이도록 하고, false 그렇지 않으면). 그런 다음 x 및 y 좌표(뷰 포트의 왼쪽 상단 위치에 해당)를 설정한 다음 너비와 높이를 픽셀 단위로 설정합니다.

통사론:

room_set_viewport(rm, vind, vis, xport, yport, wport, hport);

논쟁 유형 설명
rm 방 자산 The index of the room to set
vind 진짜 The index of the view port to set
vis 부울 The visibility of the view port (true is visible, false is invisible)
xport 진짜 The x position for the view port in the room
yport 진짜 The y position of the view port in the room
wport 진짜 The width (in pixels) of the view port
hport 진짜 The height (in pixels) of the view port

보고:

N/A

예시:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_viewport(global.myroom, 0, true, 0, 0, 640, 480);

이것은 "global.myroom" 변수에 인덱싱된 방의 뷰 포트 [0] 속성을 설정합니다.
