room_next

이 함수를 사용하면 함수에 방을 입력 한 후 방의 인덱스를 검색할 수 있습니다. 예를 들어 변수를 사용할 수 있습니다room 현재 방의 인덱스를 얻은 다음 이 기능을 사용하여 방 관리자 에 나열된 대로 뒤따르는 방을 찾습니다. 입력한 공간 뒤에 공간이 없으면 -1이 반환됩니다. 이 기능은 다음을 사용하여 동적으로 추가된 회의실을 인식하거나 고려하지 않습니다.room_add() 또는room_duplicate().

통사론:

room_next(numb);

논쟁 유형 설명
numb The index of the room to get the next one after.

보고:

Room Asset

예시:

if room_next(room) != -1
{
    room_goto_next();
}

위의 코드는 다음 방이 있는지 확인하고 있는 경우 해당 방으로 이동합니다.
