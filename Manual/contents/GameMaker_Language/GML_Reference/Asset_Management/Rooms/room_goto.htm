room_goto

이 기능을 사용하면 코드를 사용하여 생성하든 자산 브라우저에서 생성하든 게임 프로젝트의 모든 방으로 이동할 수 있습니다. 방 인덱스(방 이름에 대한 변수에 저장되거나 함수에서 반환된 변수로 저장됨)를 제공합니다.room_add() ). 방은 함수가 호출된 이벤트가 끝날 때까지 변경되지 않으므로 이 함수가 호출된 후의 코드는 동일한 이벤트에 있는 경우 계속 실행됩니다. 이 기능은 또한 Room End 이벤트를 트리거합니다.

참고 방 ID는 자산 브라우저 또는 방 관리자의 순서를 기반으로 하지 않으므로 숫자 값을 직접 제공하지 않아야 합니다. 대신 참조하려는 자산(단순히 이름)에 대해 방 상수 를 사용하거나 함수를 통해 검색하세요.

통사론:

room_goto(index);

논쟁 유형 설명
index The index of the room to go to.

보고:

예시:

switch (global.level)
{
    case 0: room_goto(rm_level1); break;
    case 1: room_goto(rm_level2); break;
    case 2: room_goto(rm_level3); break;
}

위의 코드는 전역 변수를 확인하고 보유하는 값에 따라 공간을 변경합니다.
