room_first

이 읽기 전용 변수는 게임에서 가장 첫 번째 방의 인덱스를 반환합니다(방이 생성된 순서가 아니라 방 관리자 에 방이 나타나는 순서로 정의됨).

통사론:

room_first;

보고:

Room Asset

예시:

if lives < 1
{
    room_goto(room_first);
}

위의 코드는 "lives" 변수를 확인하고 1보다 작으면 게임의 첫 번째 방으로 이동합니다.
