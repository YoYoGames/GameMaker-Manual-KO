room_set_persistent

이 기능을 사용하면 현재 방을 제외한 게임의 모든 방의 지속성을 변경(또는 설정)할 수 있습니다. 다음으로 플래그가 지정된 지속성이 있는 방 true 플레이어가 떠났다가 돌아오면 해당 방 내의 모든 인스턴스의 상태를 유지하는 반면 지속성이 다음과 같이 플래그가 지정되면 false 매번 초기 상태로 재설정됩니다. 퍼시스턴트 룸은 일반 룸보다 훨씬 더 많은 메모리를 사용하므로 게임에 너무 많은 메모리를 사용하지 않는 것이 좋습니다.

참고: 이 기능은 이미 방을 방문한 경우 지속성을 끄는 데 작동 하지 않습니다! 퍼시스턴트 룸은 한 번 방문하면 메모리에 저장되며 퍼시스턴스를 끄려면 해당 룸으로 이동하여 설정해야 합니다.room_persistent 변수 false 그리고 다시 방을 나갑니다.

통사론:

room_set_persistent(index, val);

논쟁 유형 설명
index 방 자산 The index of the room to set the persistence of.
val 부울 Whether the room should be persistent (true ) or not (false ).

보고:

N/A

예시:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_persistent(global.myroom, true);

이것은 새 방을 만들고 "global.myroom" 변수에 색인을 저장합니다. 그런 다음 너비를 640픽셀, 높이를 480픽셀, 캡션을 'Game Room'으로, 지속성을 'true'로 설정합니다.
