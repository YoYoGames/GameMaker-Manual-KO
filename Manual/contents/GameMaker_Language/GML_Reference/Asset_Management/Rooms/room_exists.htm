room_exists

이 기능을 사용하면 지정한 방의 존재 여부를 확인할 수 있습니다. 이 함수는 방 이름(문자열)이 아닌 방 색인 (실수)을 취합니다. 이 기능은 기능을 사용하여 동적으로 방을 만들 때 가장 유용합니다.room_add(), 하지만 읽기 전용 변수를 사용할 수도 있습니다.room_first 그리고room_last 또는 기능room_next() 그리고room_previous() 특정 방 인덱스를 가져오거나 게임의 다른 방 인덱스를 저장한 변수를 제공합니다.

통사론:

room_exists(index);

논쟁 유형 설명
index The index of the room to check.

보고:

예시:

if room_exists(global.rm[0])
{
    room_goto(global.rm[0]);
}

위의 코드는 "global.rm[]" 배열에 인덱싱된 방이 있는지 확인하고 존재하면 해당 방으로 이동합니다.
