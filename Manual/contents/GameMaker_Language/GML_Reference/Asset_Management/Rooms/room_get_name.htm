room_get_name

이 함수는 지정된 방의 이름을 문자열로 반환하는 데 사용할 수 있습니다. 이것은 문자열일 뿐이며 방을 직접 참조하는 데 사용할 수 없습니다. 따라서 방 색인 이 필요합니다. 그러나 이 문자열을 사용하여 함수와 함께 반환된 문자열을 사용하여 방 인덱스 를 가져올 수 있습니다.asset_get_index().

통사론:

room_get_name(index);

논쟁 유형 설명
index The index of the room to check the name of.

보고:

예시:

var roomname = room_get_name(room);

draw_text(32, 32, roomname);

위의 코드는 현재 방의 이름을 가져와 화면에 그립니다.
