room_restart

이 기능은 마치 방금 들어온 것처럼 현재 방을 다시 시작합니다. 방은 함수가 호출된 이벤트가 끝날 때까지 다시 시작되지 않으므로 이 함수가 호출된 후의 코드는 동일한 이벤트에 있는 경우 계속 실행됩니다. 이 기능은 또한 Room End 이벤트를 트리거합니다.

통사론:

room_restart();

보고:

예시:

if lives < 1 room_restart();

위의 코드는 "lives" 변수가 1보다 작은지 확인하고, 그렇다면 방을 다시 시작합니다.
