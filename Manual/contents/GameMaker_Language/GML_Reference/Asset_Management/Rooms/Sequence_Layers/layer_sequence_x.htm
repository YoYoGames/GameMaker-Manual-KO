layer_sequence_x

이 기능을 사용하면 주어진 시퀀스 요소에 대해 공간의 X(수평) 축을 따라 위치를 설정할 수 있습니다. 다음에서 반환된 시퀀스 요소 ID를 제공합니다.layer_sequence_create() 또는 레이어 요소 기능 중 하나에 의해 설정할 X 위치와 함께 시퀀스가 새 위치로 이동됩니다.

통사론:

layer_sequence_x(sequence_element_id, pos_x)

논쟁 유형 설명
sequence_element_id The unique ID value of the sequence element to target
pos_x The X position to move the sequence element to

보고:

예시:

if layer_sequence_exists(my_seq)
{
    layer_sequence_x(my_seq, x);
    layer_sequence_y(my_seq, y);
}

위의 코드는 "my_seq" 변수에 참조된 시퀀스 요소가 존재하는지 확인하고 존재한다면 x/y 위치를 코드를 실행하는 인스턴스의 위치로 설정합니다.
