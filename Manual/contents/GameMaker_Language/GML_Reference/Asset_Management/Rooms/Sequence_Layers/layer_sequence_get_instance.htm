layer_sequence_get_instance

이 함수를 사용하여 시퀀스 요소 ID를 제공합니다.layer_sequence_create() 또는 레이어 요소 함수 중 하나에 의해 - 시퀀스 인스턴스 struct 를 반환합니다. 이 페이지 에서 시퀀스 인스턴스 구조체의 형식에 대해 자세히 알아볼 수 있습니다.

통사론:

layer_sequence_get_instance(sequence_element_id)

논쟁 유형 설명
sequence_element_id 시퀀스 요소 ID The unique ID value of the sequence element to target

보고:

Sequence Instance Struct

예시:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_instance(_seq);
_struct.speedScale = 0.5;

위의 코드는 새 시퀀스 요소를 만든 다음 시퀀스 인스턴스 ID를 검색합니다. 그런 다음 시퀀스 인스턴스의 재생 속도 스케일 속성을 변경하는 데 사용됩니다.
