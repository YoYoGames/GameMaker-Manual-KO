시퀀스 레이어

이 섹션에는 시퀀스가 있는 레이어 와 관련하여 게임 룸에서 시퀀스 가 어떻게 작동하는지 추가, 제거 및 편집하는 데 사용되는 기능이 있습니다. 시퀀스는 방의 레이어에 생성될 때 자산 브라우저 에서 메인 시퀀스 개체 의 인스턴스 로 간주된다는 점에서 개체와 유사하다는 점에 유의해야 합니다. 따라서 방의 레이어에 시퀀스를 추가하면 기본 시퀀스 개체의 인스턴스 가 생성되고 이 시퀀스 인스턴스는 방 레이어 내의 요소 로 추가됩니다. 시퀀스 인스턴스는 재생 속도 및 방향과 같은 것을 제어하며 실제 시퀀스 트랙 데이터를 포함하는 또 다른 시퀀스 데이터 구조체 를 포함합니다. 아래 나열된 함수는 시퀀스 요소 ID ( 계층에 있는 시퀀스의 ID)와 시퀀스 인스턴스 ID (요소가 참조하는 시퀀스의 실제 ID) 및 시퀀스를 참조하기 때문에 중요합니다. 데이터 구조체 (모든 시퀀스 데이터를 포함).

다음은 새 시퀀스 인스턴스를 생성하거나 기존 인스턴스를 편집하는 데 사용할 수 있는 모든 함수 목록입니다.
