layer_sequence_get_angle

이 함수를 사용하여 시퀀스 요소 ID를 제공합니다.layer_sequence_create() 또는 레이어 요소 기능 중 하나에 의해 - 게임 룸에서 시퀀스 요소의 현재 각도를 반환합니다. 각도는 도 단위로 반환되며 0º는 오른쪽, 90º는 위쪽, 180º는 왼쪽, 270º는 아래쪽입니다.

통사론:

layer_sequence_get_angle(sequence_element_id)

논쟁 유형 설명
sequence_element_id The unique ID value of the sequence element to target

보고:

예시:

var _ang = layer_sequence_get_angle(title_sequence)
if _ang > 0
{
    _ang -= 1;
    layer_sequence_angle(title_sequence, _ang);
}

위의 코드는 "title_sequence" 변수에 ID가 저장된 시퀀스 요소의 현재 각도를 검색하고, 0이 아니면 현재 각도에서 1을 뺍니다.
