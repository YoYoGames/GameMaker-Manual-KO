layer_sequence_angle

이 기능을 사용하여 주어진 시퀀스 요소에 대한 회전 각도를 설정할 수 있습니다. 다음에서 반환된 시퀀스 요소 ID를 제공합니다.layer_sequence_create() 또는 레이어 요소 중 하나가 설정할 새 각도와 함께 기능 을 수행하면 시퀀스가 원점을 기준으로 새 위치로 회전됩니다. GameMaker의 각도는 오른쪽으로 0º로 계산되며 시계 반대 방향으로 이동합니다. 따라서 90º는 위쪽, 180º는 왼쪽, 270º는 아래쪽이며 시퀀스의 기본 각도는 0º입니다.

변경하면 안 됩니다. image_xscale \ image_yscale \ image_angle \ x \ y 이 함수를 사용하는 시퀀스의 모든 인스턴스에 대한 변수는 함수를 호출한 후 시퀀스가 업데이트되는 즉시 덮어쓰여집니다.

통사론:

layer_sequence_angle(sequence_element_id, angle)

논쟁 유형 설명
sequence_element_id 시퀀스 요소 ID The unique ID value of the sequence element to target
angle 진짜 The new angle to rotate the sequence element to

N/A

예시:

if current_angle < 90
{
    current_angle += 1;
    layer_sequence_angle(my_seq, current_angle);
}

위의 코드는 current_angle 변수이고 90보다 작으면 변수에 더한 다음 값을 사용하여 변수에서 참조하는 시퀀스 요소의 각도를 설정합니다. my_seq.
