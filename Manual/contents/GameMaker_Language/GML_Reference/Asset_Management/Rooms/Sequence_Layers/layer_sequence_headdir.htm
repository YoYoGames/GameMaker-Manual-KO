layer_sequence_headdir

이 기능을 사용하면 지정된 시퀀스 재생 헤드 의 방향을 설정할 수 있습니다. 다음에서 반환된 시퀀스 요소 ID를 제공합니다.layer_sequence_create() 또는 레이어 요소 기능 중 하나를 사용하여 다음 상수 중 하나여야 하는 재생 헤드 방향을 지정합니다.

일정한 설명 값
seq_dir_right 시퀀스는 왼쪽에서 오른쪽으로 증분 순서로 프레임을 재생합니다. 1
seq_dir_left 시퀀스는 오른쪽에서 왼쪽으로 감소하는 순서로 프레임을 재생합니다. -1

통사론:

layer_sequence_headdir(sequence_element_id, direction)

논쟁 유형 설명
sequence_element_id 시퀀스 요소 ID The unique ID value of the sequence element to target
direction 시퀀스 방향 상수 The playhead direction, a constant, listed above

보고:

N/A

예시:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_headdir(_seq, seq_dir_left);

위의 코드는 새 시퀀스를 만들고 해당 ID를 로컬 변수에 저장합니다. 그런 다음 이 ID를 사용하여 재생 헤드 방향을 프레임 감소(오른쪽에서 왼쪽 재생)로 설정합니다.
