layer_sequence_headpos

이 기능을 사용하면 시퀀스 요소의 재생 헤드 위치를 특정 프레임으로 설정할 수 있습니다. 다음에서 반환된 시퀀스 요소 ID를 제공합니다.layer_sequence_create() 또는 설정할 새 위치와 함께 레이어 요소 기능 중 하나에 의해. 위치는 프레임 단위이며 총 프레임 수보다 큰 값(또는 0보다 작은 값)을 설정하면 실제 최종 재생 헤드 위치는 다음 규칙에 따라 선택한 시퀀스 재생 유형에 따라 달라집니다.

  • 반복 없음: 재생 헤드가 루프의 끝(또는 지정된 값이 음수인 경우 시작)으로 고정됩니다.
  • 루핑: 재생 헤드는 시퀀스의 시작 부분과 추가 프레임으로 되돌아갑니다. 예: 30개 프레임 시퀀스의 재생 헤드를 프레임 45의 위치로 설정하면 실제 재생 헤드 위치는 15가 됩니다. 음수 값이면 시퀀스 재생 헤드가 반대 방향으로 줄 바꿈됩니다.
  • 핑퐁 루핑: 재생 헤드는 끝까지 진행한 다음 재생 방향을 반대로 하여 총 프레임 길이를 초과하는 추가 프레임 수로 되돌아갑니다. 예: 시퀀스가 30개 프레임이고 재생 헤드를 40개로 설정한 경우, 최종 재생 헤드 위치는 20이고 방향은 반대입니다(0으로 내려감). 정의되지 않은 결과가 있을 수 있으므로 시퀀스의 핑퐁 재생에 음수 값을 지정 해서는 안 됩니다.

이러한 방식으로 헤드 위치를 설정해도 재생이 중지되지 않으며 일시 중지되지 않는 한 시퀀스는 새 재생 헤드 위치에서 계속됩니다.

통사론:

layer_sequence_headpos(sequence_element_id, position)

논쟁 유형 설명
sequence_element_id 시퀀스 요소 ID The unique ID value of the sequence element to target
position 진짜 The position within the sequence (in frames) to set the playhead position to (can be a decimal value)

예시:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sequence
    {
        layer_sequence_headpos(a[i], 0)
    }
}

위의 코드는 코드를 실행하는 인스턴스의 레이어에 할당된 모든 요소의 ID를 가져옵니다. 그런 다음 코드는 반환된 요소가 시퀀스 자산인지 확인하고 재생 헤드 위치가 0으로 설정되어 있는지 확인합니다.
