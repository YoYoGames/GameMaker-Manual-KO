layer_sequence_is_paused

이 함수를 사용하여 시퀀스 요소 ID를 제공합니다.layer_sequence_create() 또는 레이어 요소 기능 중 하나에 의해 - 시퀀스가 현재 일시 중지되었는지 여부를 확인하고 다음을 반환합니다. true 일시 중지된 경우 false 그렇지 않은 경우.

통사론:

layer_sequence_is_paused(sequence_element_id)

논쟁 유형 설명
sequence_element_id The unique ID value of the sequence element to target

보고:

예시:

if layer_sequence_is_paused(title_sequence) != 0
{
    layer_sequence_play(title_sequence);
}

위의 코드는 "title_sequence" 변수에 저장된 시퀀스 요소 ID가 일시 중지되었는지 확인하고 일시 중지되면 재생을 시작합니다.
