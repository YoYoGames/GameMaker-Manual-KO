layer_sequence_exists

이 기능을 사용하면 주어진 레이어에 시퀀스 요소가 존재하는지 확인할 수 있습니다. 룸 편집기에 정의된 레이어 이름의 문자열이 될 수 있는 레이어 ID 또는 함수에서 반환된 고유 레이어 ID를 제공합니다.layer_get_id(), 시퀀스 요소 ID - 다음에서 반환됨layer_sequence_create() 또는 레이어 요소 기능 중 하나에 의해 - 반환됩니다 true 주어진 요소가 존재하거나 false 그렇지 않으면.

통사론:

layer_sequence_exists(layer_id, sequence_element_id)

논쟁 유형 설명
layer_ID 문자열 또는 레이어 ID The unique ID or name of the layer to check
sequence_element_id 시퀀스 요소 ID The unique ID value of the sequence element to target

보고:

Boolean

예시:

if !layer_sequence_exists("Background", my_seq)
{
    my_seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
    layer_sequence_pause(my_seq);
}

위의 코드는 주어진 시퀀스 요소가 존재하는지 확인하고 존재하지 않으면 주어진 레이어에 새 시퀀스를 생성한 다음 일시 중지합니다.
