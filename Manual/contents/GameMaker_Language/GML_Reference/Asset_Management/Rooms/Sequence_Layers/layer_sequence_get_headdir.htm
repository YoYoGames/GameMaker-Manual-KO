layer_sequence_get_headdir

이 함수를 사용하여 시퀀스 요소 ID를 제공합니다.layer_sequence_create() 또는 레이어 요소 기능 중 하나를 사용하여 - 현재 재생 헤드 방향을 반환합니다. 이 방향은 아래 나열된 상수 중 하나입니다.

통사론:

layer_sequence_get_headdir(sequence_element_id)

논쟁 유형 설명
sequence_element_id The unique ID value of the sequence element to target

보고:

일정한 설명 값
seq_dir_right 시퀀스는 왼쪽에서 오른쪽으로 증분 순서로 프레임을 재생합니다. 1
seq_dir_left 시퀀스는 오른쪽에서 왼쪽으로 감소하는 순서로 프레임을 재생합니다. -1

예시:

if layer_sequence_get_headdir(title_sequence) != seq_dir_left
{
    layer_sequence_headdir(title_sequence, seq_dir_left);
}

위의 코드는 "title_sequence" 변수에 저장된 시퀀스 요소 ID의 현재 재생 헤드 방향을 확인하고 다음으로 설정되지 않은 경우 seq_dir_left, 이 값으로 설정됩니다.
