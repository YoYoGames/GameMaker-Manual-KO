layer_sequence_get_x

이 함수를 사용하여 시퀀스 요소 ID를 제공합니다.layer_sequence_create() 또는 레이어 요소 기능 중 하나를 사용하여 시퀀스에 대한 게임 룸의 현재 X 위치를 반환합니다.

통사론:

layer_sequence_get_x(sequence_element_id)

논쟁 유형 설명
sequence_element_id 시퀀스 요소 ID The unique ID value of the sequence element to target

보고:

Real

예시:

if layer_sequence_get_x(title_sequence) != room_width / 2
{
    layer_sequence_x(title_sequence, room_width / 2);
}

위의 코드는 "title_sequence" 변수에 저장된 시퀀스 요소 ID의 X 위치를 확인하고, 방 너비의 절반으로 설정되지 않은 경우 이 값으로 설정합니다.
