layer_sequence_get_yscale

이 함수를 사용하여 시퀀스 요소 ID를 제공합니다.layer_sequence_create() 또는 레이어 요소 기능 중 하나에 의해 - 게임 룸에서 시퀀스 요소의 Y 축을 따라 현재 스케일을 반환합니다.

통사론:

layer_sequence_get_yscale(sequence_element_id)

논쟁 유형 설명
sequence_element_id The unique ID value of the sequence element to target

보고:

예시:

var _ys = layer_sequence_get_yscale(title_sequence)
if _ys < 1
{
    _ys += 0.01;
    layer_sequence_yscale(title_sequence, _ys);
}

위의 코드는 "title_sequence" 변수에 ID가 저장된 시퀀스 요소의 Y축을 따라 현재 스케일을 검색하고, 1보다 작으면 현재 Y 스케일에 0.01을 추가합니다.
