layer_sequence_create

이 기능을 사용하면 주어진 레이어에 시퀀스 자산의 인스턴스를 생성할 수 있습니다. 룸 편집기에 정의된 레이어 이름의 문자열이 될 수 있는 레이어 ID 또는 함수에서 반환된 고유 레이어 ID를 제공합니다.layer_get_id(), 시퀀스를 생성할 방의 X 및 Y 위치, 마지막으로 생성할 시퀀스의 ID입니다. 시퀀스 ID는 시퀀스에 대해 자산 브라우저에서 정의한 이름 상수입니다. 이 함수는 시퀀스 요소의 고유 ID를 반환합니다. 그러면 시퀀스에 대한 모든 추가 계층 함수에서 사용할 수 있습니다. 또는 함수를 사용하여 시퀀스 인스턴스 구조체를 검색하는 데 사용할 수 있습니다.layer_sequence_get_instance().

시퀀스에 개체 트랙이 포함되어 있으면 자산 키가 도프 시트 에 있는 위치에 관계없이 시퀀스 요소 자체가 생성되는 즉시 해당 인스턴스가 생성된다는 점에 주목할 가치가 있습니다. 시퀀스 컨트롤러는 트랙 자산 키의 위치와 지속 시간에 따라 숨기거나 표시하도록 인스턴스의 가시성 을 간단히 토글하며 반복적으로 생성 및 파괴하지 않습니다. 결과적으로 인스턴스는 자산 키가 시작될 때가 아니라 시퀀스 요소가 생성될 때 Create 이벤트를 실행합니다.

통사론:

layer_sequence_create(layer_id, x, y, sequence_id)

논쟁 유형 설명
layer_id 문자열 또는 레이어 ID The unique ID value of the instance layer to target
x 진짜 The x position in the room to create the sequence at
y 진짜 The y position in the room to create the sequence at
sequence_id 시퀀스 자산 The sequence asset to use, as defined in the Asset Browser

보고:

Sequence Element ID

예시:

var _s = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_pause(_s);

위의 코드는 "배경" 레이어에 새 시퀀스를 만든 다음 일시 중지합니다.
