layer_sequence_is_finished

이 함수를 사용하여 시퀀스 요소 ID를 제공합니다.layer_sequence_create() 또는 레이어 요소 기능 중 하나에 의해 - 시퀀스 재생이 완료되었는지 여부를 확인하고 반환 true 그렇다면, 그리고 false 그렇지 않은 경우. 이는 재생 모드에서 시퀀스가 루프 또는 핑퐁으로 설정 되지 않은 경우에만 적용된다는 점에 유의하십시오.

통사론:

layer_sequence_is_finished(sequence_element_id)

논쟁 유형 설명
sequence_element_id The unique ID value of the sequence element to target

보고:

예시:

if layer_sequence_is_finished(title_sequence) != 0
{
    alarm[0] = room_speed * 3;
    layer_sequence_play(title_sequence);
}

위의 코드는 "title_sequence" 변수에 저장된 시퀀스 요소 ID가 종료되었는지 확인하고 완료되면 다시 재생을 시작하여 알람을 설정합니다.
