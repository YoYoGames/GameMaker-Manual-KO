layer_sequence_speedscale

이 기능을 사용하여 주어진 시퀀스의 재생 속도를 변경할 수 있습니다. 다음에서 반환된 시퀀스 요소 ID를 제공합니다.layer_sequence_create() 또는 레이어 요소 기능 중 하나에 의해 그리고 그 다음 속도 스케일 값에 의해. 이 값은 승수 입니다. 여기서 1은 기본 재생 속도이고 1보다 작은 값은 재생 속도를 낮추고 1보다 큰 값은 속도를 높입니다. 예: 값 0.5는 재생 속도의 절반이고 값 2는 재생 속도 두 배의 재생 속도가 됩니다.

통사론:

layer_sequence_speedscale(sequence_element_id, speedscale)

논쟁 유형 설명
sequence_element_id 시퀀스 요소 ID The unique ID value of the sequence element to target
speedscale 진짜 The speed scale to use (a multiplier), default is 1

보고:

N/A

예시:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_speedscale(_seq, 0.75);

위의 코드는 호출 인스턴스의 레이어에 새 시퀀스를 만든 다음 속도 스케일 값을 0.75(기본 재생 속도의 3/4)로 설정합니다.
