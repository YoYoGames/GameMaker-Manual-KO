layer_sequence_get_speedscale

이 함수를 사용하여 시퀀스 요소 ID를 제공합니다.layer_sequence_create() 또는 레이어 요소 기능 중 하나에 의해 - 현재 재생 속도 스케일을 반환합니다. 재생 속도를 늦추거나 높이는 데 사용되는 승수 값입니다. 기본값은 1이며, 값이 1보다 작으면 재생 속도가 느려지고 값이 1보다 크면 재생 속도가 빨라집니다.

통사론:

layer_sequence_get_speedscale(sequence_element_id)

논쟁 유형 설명
sequence_element_id The unique ID value of the sequence element to target

보고:

예시:

if layer_sequence_get_speedscale(title_sequence) != 1
{
    layer_sequence_speedscale(title_sequence, 1);
}

위의 코드는 "title_sequence" 변수에 저장된 시퀀스 요소 ID의 현재 재생 헤드 속도 스케일을 확인하고 1로 설정되지 않은 경우 이 값으로 설정됩니다.
