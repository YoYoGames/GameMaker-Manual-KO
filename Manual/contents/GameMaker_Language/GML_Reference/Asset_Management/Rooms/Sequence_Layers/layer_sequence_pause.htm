layer_sequence_pause

이 기능을 사용하면 주어진 시퀀스의 재생을 일시 중지할 수 있습니다. 다음에서 반환된 시퀀스 요소 ID를 제공합니다.layer_sequence_create() 또는 레이어 요소 기능 중 하나를 사용하면 해당 기능을 사용하여 다시 재생을 시작할 때까지 해당 기능이 시퀀스를 일시 중지합니다.layer_sequence_play().

통사론:

layer_sequence_pause(sequence_element_id)

논쟁 유형 설명
sequence_element_id 시퀀스 요소 ID The unique ID value of the sequence element to target

보고:

N/A

예시:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
if global.Pause
{
    layer_sequence_pause(_seq);
}

위의 코드는 새 시퀀스를 만들고 해당 ID를 로컬 변수에 저장합니다. 그런 다음 게임이 일시 중지되었는지 확인하고, 일시 중지된 경우 시퀀스 재생을 일시 중지합니다.
