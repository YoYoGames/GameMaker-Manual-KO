layer_sequence_play

이 기능을 사용하면 주어진 시퀀스의 재생을 시작할 수 있습니다. 다음에서 반환된 시퀀스 요소 ID를 제공합니다.layer_sequence_create() 또는 레이어 요소 기능 중 하나를 사용하면 해당 기능이 시퀀스를 재생하고 필요한 경우 기능을 사용하여 일시 중지할 수 있습니다.layer_sequence_pause().

중요한! 시퀀스에 인스턴스가 있는 경우 이러한 인스턴스는image_xscale /image_yscale / image_angle /x /y 재생을 시작한 후 시퀀스가 각 단계를 업데이트할 때 변수를 덮어쓰게 됩니다.

통사론:

layer_sequence_play(sequence_element_id)

논쟁 유형 설명
sequence_element_id 시퀀스 요소 ID The unique ID value of the sequence element to target

보고:

N/A

예시:

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    var a = layer_get_all_elements(layer);
    for (var i = 0; i < array_length(a); i++)
    {
        if (layer_get_element_type(a[i]) == layerelementtype_sequence)
        {
            if (global.Pause)
            {
                layer_sequence_pause(a[i]);
            }
            else
            {
                layer_sequence_play(a[i]);
            }
        }
    }
}

위의 코드는 키를 눌렀을 때 게임이 일시 중지되었는지 여부를 확인합니다. 게임이 일시 중지되면 현재 레이어(호출 인스턴스의 레이어)의 모든 시퀀스 요소를 반복하고 재생을 일시 중지하고 게임이 일시 중지되지 않은 경우 루프는 재생을 다시 시작합니다.
