layer_sequence_get_y

이 함수를 사용하여 시퀀스 요소 ID를 제공합니다.layer_sequence_create() 또는 레이어 요소 기능 중 하나를 사용하여 시퀀스에 대한 게임룸의 현재 Y 위치를 반환합니다.

통사론:

layer_sequence_get_y(sequence_element_id)

논쟁 유형 설명
sequence_element_id 시퀀스 요소 ID The unique ID value of the sequence element to target

보고:

Real

예시:

if layer_sequence_get_y(title_sequence) != room_height / 2
{
    layer_sequence_y(title_sequence, room_height / 2);
}

위의 코드는 "title_sequence" 변수에 저장된 시퀀스 요소 ID의 Y 위치를 확인하고, 방 높이의 절반으로 설정되어 있지 않으면 이 값으로 설정합니다.
