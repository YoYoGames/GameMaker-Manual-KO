layer_sequence_xscale

이 기능을 사용하여 주어진 시퀀스 요소에 대한 X 스케일을 설정할 수 있습니다. 다음에서 반환된 시퀀스 요소 ID를 제공합니다.layer_sequence_create() 또는 레이어 요소 기능 중 하나에 의해 X 축에 설정할 새 스케일과 함께 시퀀스가 이 양만큼 스케일됩니다. 스케일이 1이면 스케일링이 없음(1:1), 작은 값은 스케일 다운(예: 0.5는 시퀀스 너비의 절반), 큰 값은 스케일 업되고 음수 값은 원점 및 스케일에 대한 시퀀스를 미러링합니다. 사용된 값이 정확히 -1이 아닌 경우(이 경우 시퀀스는 크기 조정 없이 원점에 대해 미러링됨).

회전 을 사용하는 인스턴스가 포함된 시퀀스 요소에 고르지 않은 크기 조정(예: X축을 3으로, Y축을 2로 조정)을 적용 하면 인스턴스 그리기, 충돌, 컬링 및 다른 많은 것들. 기본적으로 시퀀스가 인스턴스 속성에 의존하는 경우 인스턴스 회전과 함께 고르지 않은 크기 조정을 결합하지 않는 것이 좋습니다.

또한 변경해서는 안됩니다. image_xscale \ image_yscale \ image_angle \ x \ y 이 함수를 사용하는 시퀀스의 모든 인스턴스에 대한 변수는 함수를 호출한 후 시퀀스가 업데이트되는 즉시 덮어쓰여집니다.

통사론:

layer_sequence_xscale(sequence_element_id, xscale)

논쟁 유형 설명
sequence_element_id The unique ID value of the sequence element to target
xscale The new X axis scale value to apply to the sequence element

보고:

예시:

if seq_scale < 2
{
    seq_scale += 0.01;
    layer_sequence_xscale(my_seq, seq_scale);
    layer_sequence_yscale(my_seq, seq_scale);
}

위의 코드는 seq_scale 변수이고 2보다 작으면 추가한 다음 값을 사용하여 변수에서 참조하는 시퀀스 요소의 X 및 Y 스케일을 설정합니다. my_seq.
