room_set_width

이 기능을 사용하면 현재 방을 제외한 게임의 모든 방의 너비를 변경(또는 설정)할 수 있습니다.

통사론:

room_set_width(index, w);

논쟁 유형 설명
index 방 자산 The index of the room to set the width of.
w 진짜 The new width of the room in pixels.

보고:

N/A

예시:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_persistent(global.myroom, false);

이것은 새 방을 만들고 "global.myroom" 변수에 색인을 저장합니다. 그런 다음 너비를 640픽셀, 높이를 480픽셀, 캡션을 'Game Room'으로, 지속성을 'false'로 설정합니다.
