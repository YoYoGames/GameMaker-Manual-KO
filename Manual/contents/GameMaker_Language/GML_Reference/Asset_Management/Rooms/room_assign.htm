room_assign

이 기능은 한 방을 다른 방으로 복사하여 복사 중인 방에서 모든 인스턴스, 타일 등을 제거하고 새 방 내용으로 완전히 교체합니다. 복사할 방에 대한 인덱스를 제공합니다(자산 브라우저에 정의된 대로 또는 함수에서 반환된 대로).room_add() ) 뿐만 아니라 소스 룸의 인덱스. 자산 브라우저에서 생성된 방 자산에서 이 함수를 호출하면 방의 내용이 영구적으로 변경되고 game_restart() 방을 원래 상태로 되돌리지 않습니다(게임을 종료하고 다시 열면 방이 원래 상태로 다시 시작됩니다).

통사론:

room_assign(ind, source);

논쟁 유형 설명
ind 방 자산 The index of the room to be copied to.
source 방 자산 The index of the room to be copied from.

N/A

예시:

global.rm = room_add();
room_assign(global.rm, rm_Base);

위의 코드는 게임에 새 방을 추가한 다음 "rm_Base"로 인덱싱된 방의 내용을 게임에 복사합니다.
