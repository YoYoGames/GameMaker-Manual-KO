room_last

이 읽기 전용 변수는 게임에서 가장 마지막 방의 인덱스를 반환합니다(방이 생성된 순서가 아니라 방 관리자 에 방이 나타나는 순서로 정의됨). 이 변수는 다음을 사용하여 동적으로 추가된 회의실을 인식하거나 고려하지 않습니다.room_add() 또는room_duplicate().

통사론:

room_last;

보고:

Room Asset

예시:

if keyboard_check_pressed(ord("Q"))
{
    room_goto(room_last);
}

위의 코드는 키가 눌렸는지 확인하고 그렇다면 게임의 마지막 방으로 이동합니다.
