object_get_name

이 함수는 이름 을 지정된 개체의 문자열로 반환합니다. 이 이름은 기본 GameMaker 창의 자산 브라우저에서 개체에 대해 지정된 이름입니다. 이것은 문자열일 뿐이며 개체를 직접 참조하는 데 사용할 수 없습니다. 개체 인덱스 가 필요하기 때문입니다. 그러나 이 문자열을 사용하여 함수와 함께 반환된 문자열을 사용하여 객체 인덱스 를 가져올 수 있습니다.asset_get_index().

통사론:

object_get_name(obj);

논쟁 유형 설명
obj The index of the object to check.

보고:

예시:

str = object_get_name(object_index);

위의 코드는 코드를 실행하는 인스턴스에 대한 개체 인덱스의 이름을 가져오고 "str" 변수에 반환 값을 저장합니다.
