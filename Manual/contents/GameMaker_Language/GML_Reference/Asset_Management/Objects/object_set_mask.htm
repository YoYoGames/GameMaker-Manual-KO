object_set_mask

이 함수를 사용하여 특정 개체의 mask_index를 설정할 수 있습니다(마스크에 대한 자세한 내용 은 개체 편집기 참조). 즉 , 마스크가 변경된 후에 생성된 이 개체의 모든 인스턴스는 이 새 마스크로 생성되지만 이미 방에 있는 인스턴스는 영향을 받지 않을 수 있습니다. 이것을 -1로 설정하여 마스크 스프라이트를 제거할 수 있으므로 기본적으로 오브젝트의 스프라이트에 대해 정의된 마스크가 됩니다(또는 스프라이트가 선택되지 않은 경우 마스크 없음).

이것은 인스턴스 함수가 아니라는 점에 유의하십시오! 다음을 사용하여 개별 인스턴스의 마스크 인덱스를 설정할 수 있습니다.mask_index 변수이므로 생성된 객체와 마스크가 다른 10개의 인스턴스가 있으며 이 기능을 사용하여 객체의 마스크 인덱스를 변경하더라도 현재 방에 있는 모든 인스턴스는 그대로 유지되며 함수를 호출한 후 생성된 인스턴스는 새 마스크로 시작됩니다.

통사론:

object_set_mask(index, spr);

논쟁 유형 설명
index 개체 자산 The index of the object to change.
spr 스프라이트 자산 The new sprite to use as the object's mask.

예시:

if (global.level == 10)
{
    object_set_mask(obj_Platform, spr_mask_10);
}

위의 코드는 전역 변수의 값을 확인한 다음 "obj_Platform" 개체의 마스크 인덱스가 10일 경우 변경합니다.
