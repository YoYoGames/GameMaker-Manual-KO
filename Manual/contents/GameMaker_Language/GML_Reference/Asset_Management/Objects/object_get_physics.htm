object_get_physics

이 기능은 당신이 확인하고 있는 객체가 "물리 활성화"로 플래그가 지정되었는지 알려줄 것입니다. true, - 여부 - 이 경우 반환됩니다.false.

통사론:

object_get_physics(obj);

논쟁 유형 설명
obj The index of the object to check.

보고:

예시:

if object_get_physics(object_index)
{
    phy_active = true;
}

위의 코드는 인스턴스를 실행하는 인스턴스를 확인하여 생성된 객체가 물리학이 활성화되었는지 확인하고, 활성화된 경우 인스턴스에 대한 물리학 시뮬레이션을 활성화합니다.
