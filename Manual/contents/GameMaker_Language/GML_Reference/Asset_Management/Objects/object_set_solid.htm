object_set_solid

이 기능을 사용하면 특정 개체에 솔리드 여부를 표시할 수 있습니다("솔리드" 플래그에 대한 자세한 내용은 개체 편집기 섹션 참조). 즉, 솔리드가 변경된 후에 생성된 이 객체의 모든 인스턴스는 이 새로운 상태로 생성되지만 이미 룸에 있는 인스턴스는 영향을 받지 않을 수 있습니다.

이것은 인스턴스 함수가 아니라는 점에 유의하십시오! 다음을 사용하여 개별 인스턴스의 솔리드 플래그를 설정할 수 있습니다. solid 변수이므로 IDE에서 해당 개체의 플래그가 false로 설정되어 있어도 10개의 인스턴스가 모두 솔리드로 플래그가 지정됩니다. 이 기능을 사용하여 개체의 솔리드 플래그를 true로 변경하더라도 현재 방에 있는 모든 인스턴스는 그대로 유지되며 함수를 호출한 후에 생성된 인스턴스만 true로 플래그가 지정된 solid로 시작됩니다.

통사론:

object_set_solid( index, solid );

논쟁 유형 설명
index 개체 자산 The index of the object to change.
solid 부울 The new solidity of the object (true =solid, false =not solid).

보고:

N/A

예시:

if !object_get_solid(obj_Block)
{
    object_set_solid(obj_Block, true);
}

위의 코드는 "obj_Block" 객체의 solid 플래그를 확인하고 false이면 true 플래그로 변경됩니다.
