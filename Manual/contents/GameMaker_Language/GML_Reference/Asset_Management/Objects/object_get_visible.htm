object_get_visible

이 함수는 확인하고 있는 개체가 "표시"(그 그리기 이벤트 실행) 플래그가 지정되었는지 여부(그리기 이벤트를 실행하지 않음)를 알려줍니다. 이것은 인스턴스 함수가 아니라는 점에 유의하십시오! 따라서 동일한 개체의 보이는 개체와 보이지 않는 인스턴스를 가질 수 있으며 그 반대의 경우도 마찬가지입니다. 다음을 사용하여 개별 인스턴스 가시성을 설정할 수 있습니다.visible 인스턴스 변수.

통사론:

object_get_visible(obj);

논쟁 유형 설명
obj The index of the object to check.

보고:

예시:

if (!visible) && (object_get_visible(object_index))
{
    visible = true;
}

위의 코드는 실행 중인 인스턴스를 확인하여 표시되는지 여부를 확인하고 인스턴스의 개체 인덱스를 확인하여 표시되는지 여부를 확인합니다. 인스턴스가 아직 표시되지 않는 경우 개체 인덱스가 켜짐으로 플래그 지정되면 해당 인스턴스에 대해 "표시됨"이 true로 설정됩니다.
