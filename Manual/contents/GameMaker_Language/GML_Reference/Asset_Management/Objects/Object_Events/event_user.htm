event_user

이 기능을 사용하면 16개의 사용자 정의 이벤트 중 하나에 배치된 작업이나 코드를 실행하도록 인스턴스에 지시할 수 있습니다. 이러한 이벤트는 이러한 방식으로 만 호출할 수 있습니다.event_perform 함수.

통사론:

event_user(numb);

논쟁 유형 설명
numb 진짜 The number of User Event to call, between 0 and 15.

보고:

N/A

예시:

event_user(4);

이것은 사용자 정의 이벤트 4를 호출합니다.
