이벤트_수행_비동기

이 함수는 GameMaker 에서 제공되는 비동기 이벤트 중 하나를 수행하는 데 사용됩니다. Async 이벤트 상수(아래 표 참조)와 호출된 Async 이벤트에서 사용할 수 있는 DS 맵을 제공합니다. async_load 변하기 쉬운.

비동기 이벤트 유형 상수
일정한 설명
ev_async_web_image_load 이미지 로드 이벤트
ev_async_web HTTP 이벤트
ev_async_dialog 대화 이벤트
ev_async_web_iap 인앱 구매 이벤트
ev_async_web_cloud 클라우드 이벤트
ev_async_web_networking 네트워킹 이벤트
ev_async_web_steam 스팀 이벤트
ev_async_social 사교 행사
ev_async_push_notification 푸시 알림 이벤트
ev_async_save_load 이벤트 저장/불러오기
ev_async_audio_recording 오디오 녹음 이벤트
ev_async_audio_playback 오디오 재생 이벤트
ev_async_system_event 시스템 이벤트

비동기식 이벤트는 다음을 사용하여 호출할 수 있습니다.event_perform().

두 번째 인수에 지정된 DS 맵은 Async 이벤트가 수행된 후 함수에 의해 자동으로 소멸되므로 수동으로 삭제할 필요가 없습니다.

통사론:

event_perform_async(type, ds_map);

논쟁 유형 설명
type 비동기 이벤트 유형 상수 The type of event to perform (see the table above).
ds_map DS 맵 ID The DS map to use as async_load in the called event.

보고:

N/A

예시:

var _map = ds_map_create();

_map[? "id"] = "custom_async_event";
_map[? "result"] = true;
_map[? "data"] = { a: 13, b: 16 };

event_perform_async(ev_async_social, _map);

위의 코드는 DS 맵을 만들고 비동기 이벤트에서 읽을 사용자 지정 항목으로 채웁니다. 그런 다음 새로 생성된 맵을 다음과 같이 전달하여 Async Social 이벤트를 수행합니다. async_load 호출된 이벤트에 대해.
