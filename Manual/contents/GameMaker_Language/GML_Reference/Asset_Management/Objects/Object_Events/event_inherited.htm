event_inherited

이 함수는 인스턴스의 부모 개체의 현재 이벤트를 호출합니다. 일반적으로 인스턴스에 상위 개체가 있으면 상위 개체와 동일한 이벤트를 모두 자동으로 상속하지만, 예를 들어 상위 개체에 create 이벤트가 있고 하나를 하위 개체에 추가하면 하위 개체의 모든 인스턴스가 상위 개체에 있는 것이 아니라 추가한 새 만들기 이벤트를 실행합니다. 동일한 유형의 상위 개체 이벤트와 하위 개체 이벤트를 모두 사용해야 하는 경우 하위 이벤트에 포함된 나머지 코드 또는 작업을 계속하기 전에 상위 개체 이벤트를 실행하므로 이 함수를 사용해야 합니다.

통사론:

event_inherited();

보고:

예시:

event_inherited();
switch (sprite_index)
{
    case spr_Enemy_1: dmg += 2; break;
    case spr_Enemy_4: dmg -= 1; break;
    case spr_Enemy_10: dmg +=10; break;
}

위의 코드는 상속된 부모 이벤트를 호출합니다(여기서 변수 "dmg ") 그런 다음 "를 수정합니다.dmg " 변수입니다. 이 코드를 실행하는 인스턴스에 대해 지정된 부모가 없으면 다음과 같이 "알 수 없는 변수" 오류가 발생합니다. dmg 정의되지 않았습니다.
