event_type

이 읽기 전용 변수는 다음 상수 중 하나일 수 있는 현재 실행 중인 이벤트 유형을 반환합니다.

  • ev_create
  • ev_destroy
  • ev_step
  • ev_alarm
  • ev_keyboard
  • ev_keypress
  • ev_keyrelease
  • ev_마우스
  • ev_collision
  • ev_other
  • ev_gesture
  • ev_draw


다른 이벤트와 관련된 상수의 전체 목록은 다음을 참조하십시오.event_perform(), 이벤트 번호(예: 하위 이벤트)가 필요한 경우 다음을 확인해야 합니다.event_number.

통사론:

event_type;

보고:

Event Type Constant

예시:

show_debug_message("Current Event = " + string(event_type));

위의 코드는 현재 수행 중인 이벤트 유형과 함께 디버그 메시지를 표시합니다.
