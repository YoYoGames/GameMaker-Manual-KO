객체 이벤트 생성

GameMaker 는 완전히 이벤트 기반이며 모든 작업은 특정 순서로 또는 게임 내 특정 발생에 의해 트리거되는 특정 이벤트의 결과로 발생합니다. 각각 특정 작업이나 타이밍이 있는 다양한 이벤트가 있으며 그 중 일부는 별도의 "하위" 이벤트로 분할되기도 합니다. 일반적으로 이러한 이벤트는 게임 내 트리거 또는 모든 게임 틱(단계 및 그리기 이벤트의 경우와 같이)을 감지할 때 GameMaker 에 의해 실행되지만 내에서 현재 방의 인스턴스에 이벤트를 적용할 수도 있습니다. 코드 조각.

이벤트 생성을 처리하는 다음 함수가 있습니다.

  • event_inherited
  • event_perform
  • 이벤트_수행_비동기
  • event_perform_object
  • event_user

또한 디버깅에 특히 유용한 위에 나열된 함수와 함께 사용할 수 있는 읽기 전용 변수가 많이 있습니다.

  • event_action
  • event_number
  • event_object
  • event_type

이벤트에 대한 자세한 내용은 개체 이벤트 섹션을 참조하십시오.
