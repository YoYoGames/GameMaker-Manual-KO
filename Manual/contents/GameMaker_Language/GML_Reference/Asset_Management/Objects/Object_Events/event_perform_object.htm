event_perform_object

이 기능은 다음과 동일하게 작동합니다.event_perform() 단, 이번에는 다른 개체의 이벤트를 지정할 수 있습니다. 여기에는 가능한 모든 이벤트의 완전한 시뮬레이션을 허용하는 많은 옵션이 있지만, 이것은 문자 그대로 이벤트의 모든 코드를 수행하고 게임은 예를 들어 키보드 수행을 선택하는 경우와 같이 수동으로 트리거되도록 아무것도 수정하지 않습니다. press 이벤트가 발생하면 이벤트가 발생하지만 해당 키가 눌린 것으로 인식되지 않습니다. 또는 알람 이벤트를 수행하면 알람 카운터가 -1로 설정되지 않고 계속 카운트다운됩니다. 이 함수에 필요한 사용 가능한 상수의 전체 목록은 함수 페이지에서 찾을 수 있습니다.event_perform().

참고 이 함수로 호출된 이벤트의 작업은 주어진 객체의 인스턴스가 아니라 현재 인스턴스 에 적용됩니다.

통사론:

event_perform_object(obj, type, numb);

논쟁 유형 설명
obj 개체 자산 The object that should have its event triggered.
type 이벤트 유형 상수 The type of event to perform.
numb 실수 또는 사건 수 상수 The specific event, if one is necessary (otherwise, just use 0).

보고:

N/A

예시:

event_perform_object(obj_Player, ev_keypress, ord("W"));

이것은 현재 인스턴스의 "obj_Player" 개체에서 Keyboard Check Pressed > W 키와 관련된 이벤트를 수행합니다.
