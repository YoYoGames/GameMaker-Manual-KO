event_number

이 읽기 전용 변수는 현재 호출되고 있는 이벤트의 번호를 반환합니다. 여기서 번호는 실제로 이벤트의 "하위 이벤트"를 나타냅니다. 즉, 단계 이벤트의 경우 이벤트 번호는 ev_step_normal, ev_step_begin 상수 중 하나일 수 있습니다., 또는 ev_step_end. 특정 하위 이벤트에 사용할 수 있는 전체 상수 목록은 다음을 참조하세요.event_perform(), 그리고 주요 이벤트 자체를 알아야 하는 경우 다음을 확인해야 합니다.event_type.

통사론:

event_number;

보고:

Event Number Constant

예시:

switch (event_number)
{
    case ev_step_normal: show_debug_message("Step event!"); break;
    case ev_game_start: show_debug_message("Game Start""); break;
    case ev_room_start: show_debug_message("Room Start!"); break;
}

위의 코드는 스크립트에서 호출할 수 있으며 현재 트리거되는 이벤트를 알려주는 디버그 메시지를 표시하는 데 사용할 수 있습니다.
