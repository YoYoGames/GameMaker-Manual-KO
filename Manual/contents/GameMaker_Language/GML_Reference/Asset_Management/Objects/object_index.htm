object_index

이 읽기 전용 변수는 인스턴스가 생성된 객체의 인덱스를 반환합니다. 이것은 문자열이며 다음을 사용하여 찾을 수 있는 개체 이름과 동일 하지 않습니다.object_get_name(), 이 함수는 인덱스 번호를 반환하므로 GameMaker 가 생성 시 모든 개체에 할당하는 고유한 값입니다.

통사론:

object_index;

보고:

Object Asset

예시:

obj_name = object_get_name(object_index);

위의 코드는 object_index 현재 인스턴스가 생성된 개체의 이름을 찾습니다.
