object_exists

이 함수는 지정된 값이 객체 자산인지 여부를 반환합니다(true ) 여부(false ). 이것은 개체가 실제로 게임룸이 아니라 자산 브라우저에 있는지 확인합니다. 그것을 위해 당신은 기능을 사용해야합니다instance_exists().

통사론:

object_exists(obj);

논쟁 유형 설명
obj The index of the object to check.

보고:

예시:

var object_id = buffer_read(net_buffer, buffer_u16);
var pos_x = buffer_read(net_buffer, buffer_u16);
var pos_y = buffer_read(net_buffer, buffer_u16);

if (object_exists(object_id))
{
    instance_create_layer(pos_x, pos_y, "Instances", object_id);
}

위의 코드는 네트워킹을 통해 받은 버퍼에서 데이터를 읽습니다. 여기에는 개체 ID(인스턴스를 생성하는 데 사용됨)가 포함되어 있으므로 먼저 값이 유효하지 않고 개체가 실제로 존재하는지 확인해야 합니다. 그것을 위해 우리는 부른다 object_exists() 조건에서, 그리고 그것이 사실이라면, 우리는 수신된 위치에 그 객체의 인스턴스를 생성합니다.
