object_set_visible

이 기능을 사용하면 특정 개체를 표시 여부에 플래그를 지정할 수 있습니다("표시" 플래그에 대한 자세한 내용은 개체 편집기 섹션 참조). 즉, 표시가 변경된 후에 생성된 이 개체의 모든 인스턴스는 이 새로운 상태로 생성되지만 이미 룸에 있는 인스턴스는 영향을 받지 않을 수 있습니다.

이것은 인스턴스 함수가 아니라는 점에 유의하십시오! 다음을 사용하여 개별 인스턴스의 표시 플래그를 설정할 수 있습니다.visible 변수이므로 문제의 개체가 IDE에서 플래그가 false로 설정되어 있어도 10개의 인스턴스가 모두 보이는 것으로 플래그가 지정됩니다. 이 기능을 사용하여 개체의 보이는 플래그를 true로 변경하더라도 현재 방에 있는 모든 인스턴스는 그대로 유지되며 함수를 호출한 후에 생성된 인스턴스만 true로 플래그가 지정된 표시로 시작됩니다.

통사론:

object_set_visible( index, vis );

논쟁 유형 설명
index 개체 자산 The index of the object to change.
vis 부울 The new visibility of the object (true =visible, false =invisible).

보고:

N/A

예시:

if !object_get_visible(obj_Block)
{
    object_set_visible(obj_Block, true);
}

위의 코드는 "obj_Block" 개체의 보이는 플래그를 확인하고 false이면 true 플래그로 변경됩니다.
