object_is_ancestor

이 기능은 개체를 확인하고 다른 개체의 조상인지 여부를 확인하는 데 사용할 수 있습니다. 따라서 두 인수 모두 인스턴스 ID가 아니라 객체 인덱스여야 하며 첫 번째 인수는 항상 두 번째 인수의 자식인지 확인하려는 객체입니다. 조상(부모).

통사론:

object_is_ancestor(obj, par);

논쟁 유형 설명
obj 개체 자산 The object that is being checked as the child.
par 개체 자산 The object that is being checked as the ancestor (parent).

보고:

Boolean

예시:

if object_is_ancestor(object_index, obj_Enemy)
{
    instance_destroy();
}

위의 코드는 코드를 실행하는 인스턴스가 "obj_Enemy" 개체의 자식인지 확인하고 그렇다면 소멸됩니다.
