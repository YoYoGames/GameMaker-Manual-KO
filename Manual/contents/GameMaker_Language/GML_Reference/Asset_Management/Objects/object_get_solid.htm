object_get_solid

이 기능은 확인 중인 개체가 "단색"으로 플래그 지정되었는지 여부를 알려줍니다. 솔리드 오브젝트는 기존 충돌 시스템을 사용할 때 특수 충돌 이벤트를 생성합니다(예: 물리 세계가 꺼져 있음). 이것은 인스턴스 함수가 아니라는 점에 유의하십시오! 따라서 솔리드 개체와 동일한 개체의 일반 인스턴스를 가질 수 있으며 그 반대의 경우도 마찬가지입니다. 다음을 사용하여 개별 인스턴스 솔리드 플래그를 설정할 수 있습니다.solid 인스턴스 변수.

통사론:

object_get_solid(obj);

논쟁 유형 설명
obj The index of the object to check.

보고:

예시:

if (!solid) && (object_get_solid(object_index))
{
    solid = true;
}

위의 코드는 실행 중인 인스턴스를 확인하여 솔리드인지 아닌지 확인하고 인스턴스의 개체 인덱스를 확인하여 솔리드로 플래그가 지정되었는지 확인합니다. 인스턴스가 솔리드가 아니지만 객체 인덱스가 솔리드로 플래그가 지정되면 해당 인스턴스에 대해 "솔리드"를 true로 설정합니다.
