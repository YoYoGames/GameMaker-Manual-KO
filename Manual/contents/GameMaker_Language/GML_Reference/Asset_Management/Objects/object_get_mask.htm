object_get_mask

이 함수는 검사 중인 객체에 마스크 인덱스가 있는지 여부를 알려주고, 있으면 해당 마스크(스프라이트 자산)의 인덱스를 반환하고, 없으면 -1을 반환합니다. 이것은 인스턴스 함수가 아니라는 점에 유의하십시오! 마스크가 없는 객체를 가질 수 있는 반면 동일한 객체의 인스턴스는 마스크를 가질 수 있고 그 반대도 마찬가지입니다. 또는 심지어 다른 마스크를 가질 수도 있습니다. 다음을 사용하여 개별 인스턴스 마스크 인덱스를 설정할 수 있습니다.mask_index 인스턴스 변수.

통사론:

object_get_mask(obj);

논쟁 유형 설명
obj The index of the object to check

보고:

Sprite Asset

예시:

if (mask_index != object_get_mask(object_index))
{
    mask_index = object_get_mask(object_index);
}

위의 예는 인스턴스의 object_index의 마스크에 대해 인스턴스의 마스크 인덱스를 확인합니다. 동일하지 않은 경우 개체 인덱스와 동일한 마스크를 인스턴스에 할당합니다.
