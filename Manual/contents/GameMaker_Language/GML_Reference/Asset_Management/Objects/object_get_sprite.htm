object_get_sprite

이 함수는 검사하는 객체에 스프라이트가 있는지 여부를 알려주고, 스프라이트가 있으면 해당 스프라이트의 인덱스를 반환하고, 없으면 -1을 반환합니다. 이것은 인스턴스 함수가 아니라는 점에 유의하십시오! 스프라이트가 없는 객체를 가질 수 있는 반면 동일한 객체의 인스턴스는 스프라이트를 가질 수 있고 그 반대도 마찬가지입니다. 또는 심지어 다른 스프라이트를 가질 수도 있습니다. 다음을 사용하여 개별 인스턴스 스프라이트를 설정할 수 있습니다.sprite_index 인스턴스 변수.

통사론:

object_get_sprite(obj);

논쟁 유형 설명
obj The index of the object to check

보고:

Sprite Asset

예시:

var _spr = object_get_sprite(object_index);
if (sprite_index != _spr)
{
    sprite_index = _spr;
}

위의 예는 다음을 확인할 것입니다. sprite_index 스프라이트에 대한 인스턴스의 object_index 인스턴스의. 동일하지 않은 경우 개체 인덱스와 동일한 스프라이트를 인스턴스에 할당합니다.
