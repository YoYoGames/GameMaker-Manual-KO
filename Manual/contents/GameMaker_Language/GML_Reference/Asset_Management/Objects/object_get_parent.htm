object_get_parent

이 함수는 지정된 개체에 할당된 부모의 개체 인덱스를 가져오거나 -100을 반환하여 개체에 할당된 부모가 없음을 표시하거나 검사 중인 개체가 존재하지 않는 경우 -1을 반환합니다. 상위 항목에 대한 자세한 내용은 개체 편집기 섹션을 참조하십시오.

통사론:

object_get_parent(obj);

논쟁 유형 설명
obj The index of the object to check

보고:

Object Asset

예시:

par = object_get_parent(object_index);

위의 코드는 코드를 실행하는 인스턴스에 대한 개체 인덱스의 부모를 가져오고 반환 값을 변수 "par"에 저장합니다.
