object_get_persistent

이 기능은 확인 중인 개체가 "영구적"으로 플래그가 지정되었는지 여부를 알려줍니다. 영속 객체는 명시적으로 파괴되지 않는 한 그 인스턴스가 방에서 방으로 전달되도록 하는 것입니다. 이것은 인스턴스 함수가 아니라는 점에 유의하십시오! 따라서 동일한 객체의 영구 객체와 비영구 인스턴스를 가질 수 있으며 그 반대의 경우도 마찬가지입니다. 다음을 사용하여 개별 인스턴스 영구 플래그를 설정할 수 있습니다.persistent 인스턴스 변수.

통사론:

object_get_persistent(obj);

논쟁 유형 설명
obj The index of the object to check.

보고:

예시:

if (!persistent) && (object_get_persistent(object_index))
{
    persistent = true;
}

위의 코드는 실행 중인 인스턴스를 확인하여 지속성인지 아닌지 확인하고 인스턴스의 개체 인덱스를 확인하여 지속성 플래그가 지정되었는지 확인합니다. 인스턴스가 영속적 이지 않고 객체 인덱스가 영속으로 플래그가 지정되면 해당 인스턴스에 대해 "지속적"을 true로 설정합니다.
