object_set_sprite

이 기능을 사용하여 특정 개체의 스프라이트 인덱스를 설정할 수 있습니다. 즉 , sprite_index가 변경된 후에 생성된 이 객체의 모든 인스턴스는 이 새로운 sprite_index로 생성되지만 이미 룸에 있는 인스턴스는 영향을 받지 않을 수 있습니다.

이것은 인스턴스 함수가 아니라는 점에 유의하십시오! 다음을 사용하여 개별 인스턴스의 스프라이트 인덱스를 설정할 수 있습니다.sprite_index 변수는 10개의 인스턴스가 모두 생성된 오브젝트에 대해 서로 다른 스프라이트를 가지고 있으며, 이 함수를 사용하여 오브젝트의 스프라이트 인덱스를 변경하더라도 현재 방에 있는 모든 인스턴스는 그대로 유지되며, 함수를 호출한 후 생성된 인스턴스는 새 스프라이트로 시작됩니다.

통사론:

object_set_sprite( index, spr );

논쟁 유형 설명
index 개체 자산 The index of the object to change.
spr 스프라이트 자산 The sprite to assign to the object.

보고:

N/A

예시:

switch (room)
{
    case rm_start: object_set_sprite(obj_Player, spr_uniform); break;
    case rm_middle: object_set_sprite(obj_Player, spr_swimsuit); break;
    case rm_end: object_set_sprite(obj_Player, spr_casual); break;
}
instance_create_layer(32, 32, "Instances", obj_Player);

위의 코드는 개체 "obj_Player" 스프라이트 인덱스를 코드를 실행하는 인스턴스가 현재 있는 방에 따라 다른 값으로 설정합니다. 그런 다음 "obj_Player"의 인스턴스를 만듭니다.
