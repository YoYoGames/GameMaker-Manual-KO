DS 우선 순위 대기열

DS 우선순위 큐는 다음과 유사한 특수 데이터 구조입니다.DS queue, 여기서만 값 대기열은 값이 할당된 우선 순위(항상 실수)에 따라 정렬됩니다. 이 구조는 각 항목의 중요성을 가중할 수 있는 순위표 또는 정보 목록을 구성하는 데 매우 유용할 수 있습니다.

참고: 모든 동적 리소스와 마찬가지로 데이터 구조는 메모리를 차지하므로 더 이상 필요하지 않을 때 항상 파괴되어 속도가 느려지고 결국 게임이 중단되는 메모리 누수를 방지해야 합니다.

참고: 데이터 구조가 존재하는지 확인해야 하는 경우 다음을 사용할 수 있습니다.ds_exists() 함수.

DS 우선 순위 대기열을 처리하는 다음 기능이 있습니다.

  • ds_priority_create
  • ds_priority_destroy
  • ds_priority_clear
  • ds_priority_empty
  • ds_priority_size
  • ds_priority_add
  • ds_priority_change_priority
  • ds_priority_delete_max
  • ds_priority_delete_min
  • ds_priority_delete_value
  • ds_priority_find_max
  • ds_priority_find_min
  • ds_priority_find_priority
  • ds_priority_copy
  • ds_priority_read
  • ds_priority_write
