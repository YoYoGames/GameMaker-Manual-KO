ds_priority_clear

이 기능을 사용하면 주어진 우선순위 큐 데이터 구조에서 모든 데이터를 지울 수 있습니다. 이것은 데이터 구조를 파괴하지 않습니다.ds_priority_destroy() ) 모든 데이터만 지우고 빈 우선 순위 대기열을 반환합니다.

통사론:

ds_priority_clear(id);

논쟁 유형 설명
id The id of the data structure to clear.

보고:

예시:

if (count = 15) && (!ds_priority_empty(command_queue))
{
    ds_priority_clear(command_queue);
    alarm[0] = room_speed;
    ai_count = 0;
}

위의 코드는 변수를 검사하여 특정 값에 도달했는지 확인하고 변수가 있으면 "command_queue" 변수에 색인된 DS 우선 순위 대기열을 지우고 경보를 설정하고 변수를 0으로 재설정합니다.
