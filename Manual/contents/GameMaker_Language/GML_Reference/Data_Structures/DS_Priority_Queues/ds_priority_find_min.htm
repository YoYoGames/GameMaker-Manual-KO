ds_priority_find_min

이 함수를 사용하면 가장 낮은 우선순위로 우선순위 큐에 저장된 값을 찾을 수 있으며, 둘 이상의 값이 동일한 우선순위를 가질 경우 그 중 하나가 임의의 순서로 반환될 수 있습니다. 그러나 달리ds_priority_delete_min(), 이 함수는 큐에서 값을 제거하지 않습니다.

통사론:

ds_priority_find_min(id);

논쟁 유형 설명
id The id of the priority queue to use.

보고:

Real or String

예시:

if ai_move
{
    script_execute(ds_priority_find_min(ai_priority));
}

위의 코드는 인스턴스 변수를 확인하고 반환하는 경우 true 가장 낮은 우선 순위 값을 가진 우선 순위 대기열에 색인된 스크립트 함수를 실행합니다.
