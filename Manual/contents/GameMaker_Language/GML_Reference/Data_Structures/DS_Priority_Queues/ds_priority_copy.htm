ds_priority_copy

이 기능은 한 우선순위 큐의 내용을 다른 우선순위 큐로 복사하는 데 사용할 수 있습니다. 이것은 원래 우선 순위 대기열에서 내용을 제거 하지 않으며 원래 우선 순위 대기열을 파괴하지도 않습니다. 이 기능을 사용할 때 복사 대상 우선 순위 대기열은 이전에 생성되어 있어야 하며 복사 이전에 항목이 포함되어 있으면 이러한 항목이 먼저 지워집니다(즉, 이 정보가 손실됨).

통사론:

ds_priority_copy(id, source);

논쟁 유형 설명
id The id of the priority queue to copy to.
source The id of the priority queue to copy from.

보고:

예시:

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
    p_queue = ds_priority_create();
    ds_priority_copy(p_queue, other.p_queue);
}

위의 함수는 새 인스턴스를 만든 다음 해당 인스턴스에서 새 DS 우선 순위 대기열을 만들고 코드 블록을 실행하는 인스턴스의 우선 순위 대기열 내용을 새로 생성된 인스턴스 우선 순위 대기열에 복사합니다.
