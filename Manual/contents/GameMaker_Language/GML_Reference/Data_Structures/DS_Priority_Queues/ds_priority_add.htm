ds_priority_add

이 기능을 사용하면 우선 순위 대기열에 값(실수 또는 문자열)을 추가하는 동시에 우선 순위 값을 할당할 수 있습니다.

통사론:

ds_priority_add(id, val, priority);

논쟁 유형 설명
id DS 우선순위 ID The id of the priority queue to add to.
val 변하기 쉬운 The value to add to the priority queue.
priority 진짜 The priority of the value to add.

보고:

N/A

예시:

ds_priority_add(ai_priority, AI_Search, 5);

위의 코드는 "ai_priority" 변수에 인덱싱된 우선 순위 대기열에 스크립트 함수를 추가하고 우선 순위 5를 할당합니다.
