ds_priority_change_priority

이 함수는 주어진 값을 취하고 참조된 우선순위 큐 내에서 우선순위를 변경합니다. 주어진 값은 우선 순위 대기열에 이미 존재해야 합니다.

통사론:

ds_priority_change_priority(id, val, priority);

논쟁 유형 설명
id DS 우선순위 ID The id of the priority queue to change.
val 변하기 쉬운 The value to change the priority of.
priority 진짜 The new priority of the value.

보고:

N/A

예시:

if (global.Game_Time < 1000)
{
    ds_priority_change(ai_priority, AI_Search, 1);
}

위의 코드는 전역 변수를 확인하고 특정 값 미만이면 우선 순위 큐에 있는 스크립트 함수 인덱스의 우선 순위를 변경합니다.
