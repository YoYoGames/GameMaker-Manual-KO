ds_priority_write

이 함수는 다음을 사용하여 저장하거나 다른 데이터 구조로 전송할 수 있는 문자열을 반환합니다.ds_priority_read() 함수.

참고: 반환된 문자열은 사람이 읽을 수 있는 문자열이 아니라 데이터 구조 내용의 덤프입니다.

통사론:

ds_priority_write(id);

논쟁 유형 설명
id The id of the data structure to check.

보고:

예시:

var str;
ini_open("save.ini");
str = ds_priority_write(p_queue);
ini_write_string("P_Queues", "0", str);
ds_priority_clear(p_queue);
ini_close();

위의 코드는 ini 파일을 연 다음 "p_queue" 변수에 인덱싱된 DS 우선 순위 대기열에 저장된 정보를 포함하는 문자열을 씁니다. 그런 다음 우선 순위 대기열이 지워지고 ini 파일이 닫힙니다.
