ds_priority_delete_max

이 함수는 대기열에서 가장 높은 우선 순위를 가진 값을 반환한 다음 데이터 구조에서 값(및 우선 순위)을 제거합니다. 둘 이상의 값에 동일한 우선 순위가 있는 경우 그 중 하나를 순서에 관계 없이 반환할 수 있지만 우선 순위가 같은 다른 모든 값은 여전히 대기열에 있습니다.

통사론:

ds_priority_delete_max(id);

논쟁 유형 설명
id The id of the priority queue to use.

보고:

(Data type stored in the priority)

예시:

if ai_move
{
    script_execute(ds_priority_delete_max(ai_priority));
}

위의 코드는 인스턴스 변수를 확인하고 반환하는 경우 true 우선 순위 값이 가장 높은 우선 순위 대기열에 인덱싱된 스크립트를 실행한 다음 대기열에서 해당 스크립트를 제거합니다.
