ds_priority_create

이 함수는 새로운 우선 순위 큐 데이터 구조를 만들고 인덱스 값을 반환합니다. 이 값은 변수에 저장되어야 하며 우선 순위 대기열과 관련된 모든 추가 함수 호출에 사용해야 합니다.

중요한! 데이터 구조를 생성할 때 식별하기 위한 인덱스 값은 0부터 시작하는 정수 값입니다. 이는 다른 유형의 데이터 구조가 동일한 인덱스 값을 가질 수 있음을 의미하므로 의심스러운 경우 다음을 사용해야 합니다.ds_exists() 액세스하기 전에 기능. 또한 인덱스는 재사용되므로 파괴된 데이터 구조 인덱스 값은 나중에 새로 생성된 값에서 사용될 수 있습니다.

통사론:

ds_priority_create();

보고:

DS Priority ID

예시:

p_queue = ds_priority_create();

이것은 새로운 우선순위 큐를 생성하고 해당 인덱스 ID를 인스턴스 변수 "p_queue"에 할당합니다.
