ds_priority_size

이 함수는 우선 순위 대기열의 "크기"를 반환합니다. 즉, 우선 순위가 지정된 항목 수입니다.

통사론:

ds_priority_size(id);

논쟁 유형 설명
id The id of the data structure to check.

보고:

예시:

if !ds_priority_empty(control_priority)
{
    num = ds_priority_size(control_priority);
}

위의 코드는 DS 우선순위 큐가 비어 있는지 확인하고 비어 있지 않은 경우 포함하고 있는 항목 수를 가져와 변수에 값을 저장합니다.
