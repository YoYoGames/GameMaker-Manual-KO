ds_grid_value_disk_y

이 기능을 사용하면 검색되는 값의 y 좌표(주어진 격자 디스크 모양 영역 내)를 얻을 수 있습니다. DS 그리드 인덱스를 제공합니다(ds_grid_create() ) 디스크의 중앙 셀에 대한 x/y 위치와 함께. 그런 다음 검색할 값을 제공하기 전에 검색할 중심 셀 주위의 반경(정수 값으로)을 지정합니다. 검색되는 값이 존재하지 않으면 함수는 -1을 반환합니다.

통사론:

ds_grid_value_disk_y(index, xm, ym, r, val);

논쟁 유형 설명
index The index of the grid.
xm The x position of the disk on the grid.
ym The y position of the disk on the grid.
r The radius of the disk on the grid.
val The value to find.

보고:

예시:

if ds_grid_value_disk_exists(grid, 5, 5, 5, val)
{
    xpos = ds_grid_value_disk_x(grid, 5, 5, 5, val);
    ypos = ds_grid_value_disk_y(grid, 5, 5, 5, val);
}

위의 코드는 디스크 영역 내의 특정 값에 대해 DS 그리드를 확인합니다. 발견되면 나중에 사용하기 위해 두 개의 변수에 값의 x 및 y 위치를 저장합니다.
