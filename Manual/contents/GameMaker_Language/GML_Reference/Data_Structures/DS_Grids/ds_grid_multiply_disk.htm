ds_grid_multiply_disk

이 함수는 DS 그리드의 주어진 디스크 모양 영역에 있는 모든 값을 가져와서 각각에 주어진 양을 곱합니다.

참고 이 함수는 문자열이 아닌 실수에서만 작동합니다.

통사론:

ds_grid_multiply_disk(index, xm, ym, r, val);

논쟁 유형 설명
index DS 그리드 ID The index of the grid.
xm 진짜 The x position of the disk on the grid.
ym 진짜 The y position of the disk on the grid.
r 진짜 The radius of the disk on the grid.
val 진짜 The value to multiply the cells within the disk with.

보고:

N/A

예시:

ds_grid_multiply_disk(mygrid, 5, 5, 5, 2)

위의 코드는 원형 그리드 영역에서 찾은 모든 값을 가져와 각각에 2를 곱합니다.
