ds_grid_multiply_region

이 기능을 사용하면 각 셀 값을 주어진 양으로 곱할 그리드 영역을 지정할 수 있습니다.

참고 이 함수는 문자열이 아닌 실수에서만 작동합니다.

통사론:

ds_grid_multiply_region(index, x1, y1, x2, y2, val);

논쟁 유형 설명
index DS 그리드 ID The index of the grid.
x1 진짜 The x position of the left of the region in the grid.
y1 진짜 The y position of the top of the region in the grid.
x2 진짜 The x position of the right of the region in the grid.
y2 진짜 The y position of the bottom of the region in the grid.
val 진짜 The value to multiply with the region cells.

보고:

N/A

예시:

ds_grid_multiply_region(mygrid, 5, 5, 10, 10, 2);

위의 코드는 정의된 직사각형 그리드 영역 내에서 찾은 모든 값을 취하고 각각에 2를 곱합니다.
