ds_grid_clear

이 기능은 주어진 DS 그리드를 특정 값으로 지우는 데 사용할 수 있습니다. 그러면 그리드 내의 모든 셀에 실수 또는 문자열이 될 수 있는 이 값이 포함됩니다. 아래 이미지는 작동 방식을 보여줍니다.

통사론:

ds_grid_clear(index, val);

논쟁 유형 설명
index This index of the grid to clear.
val The new value for all grid cells.

보고:

예시:

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

위의 코드는 전역 변수 "Grid"에 색인된 DS 그리드의 크기를 조정한 다음 각 셀이 값 -1을 보유하도록 지웁니다.
