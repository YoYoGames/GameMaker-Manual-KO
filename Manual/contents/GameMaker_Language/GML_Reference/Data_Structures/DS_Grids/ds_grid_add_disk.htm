ds_grid_add_disk

이 함수는 그리드의 정의된 디스크 영역 내에서 발견된 셀의 모든 값에 주어진 값(실수 또는 문자열)을 추가하는 데 사용할 수 있습니다. 추가할 값은 그리드 셀 내에 있는 것과 동일한 유형이어야 합니다. 즉, 실수에 문자열을 추가하거나 그 반대로 문자열을 추가할 수 없으며 문자열의 경우 연결에 해당합니다.

통사론:

ds_grid_add_disk(index, xm, ym, r, val);

논쟁 유형 설명
index The index of the grid.
xm The x position of the disk on the grid.
ym The y position of the disk on the grid.
r The radius of the disk on the grid.
val The value to add to the cells within the disk.

보고:

예시:

ds_grid_add_disk(grid, 7, 6, 5, 2)

이렇게 하면 변수 "grid"가 참조하는 DS 그리드의 정의된 디스크 영역 내의 셀에 있는 모든 값에 2가 추가됩니다.
