ds_grid_write

이 함수는 주어진 ds_grid를 문자열로 변환하는 데 사용할 수 있습니다. 그런 다음 외부 파일에 저장할 수 있습니다(예:). 함수를 사용하여 이 함수에서 반환된 문자열을 다시 ds_grid로 읽을 수 있습니다.ds_grid_read().

참고 반환된 문자열은 사람이 읽을 수 있는 문자열이 아니라 데이터 구조 내용의 덤프입니다.

통사론:

ds_grid_write(index);

논쟁 유형 설명
index The index of the grid to write.

보고:

예시:

ini_open("Save.ini");
ini_write_string("Save", "0", ds_grid_write(mygrid));
ini_close()

위의 코드는 ini 파일을 열고(아직 존재하지 않는 경우 생성) 주어진 ds_grid를 해당 파일에 문자열로 씁니다.
