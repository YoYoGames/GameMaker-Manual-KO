ds_grid_copy

이 기능을 사용하면 한 그리드의 내용을 다른 그리드로 복사할 수 있습니다. 두 그리드 모두 이전에 다음을 사용하여 생성되어야 합니다.ds_grid_create() 함수.

통사론:

ds_grid_copy(destination, source);

논쟁 유형 설명
destination DS 그리드 ID This index of the grid to copy to.
source DS 그리드 ID This index of the grid to copy from.

보고:

N/A

예시:

n_grid = ds_grid_create(ds_grid_width(a_grid), ds_grid_height(a_grid));
ds_grid_copy(n_grid, a_grid);
ds_grid_clear(a_grid, -1)

위의 코드는 이전에 만든 그리드의 너비와 높이를 기반으로 새 DS 그리드를 만든 다음 이전 그리드의 정보를 새 그리드로 복사합니다. 마지막으로 모든 셀의 값이 -1이 되도록 이전 그리드를 지웁니다.
