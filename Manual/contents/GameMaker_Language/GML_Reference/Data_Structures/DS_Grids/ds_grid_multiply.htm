ds_grid_multiply

이 함수는 주어진 그리드 셀의 값을 지정된 양만큼 곱합니다.

참고 이 함수는 문자열이 아닌 실수에서만 작동합니다.

통사론:

ds_grid_multiply(index, x, y, val);

논쟁 유형 설명
index DS 그리드 ID The index of the grid.
x 진짜 The x position of the cell in the grid.
y 진짜 The y position of the cell in the grid.
val 진짜 The value to multiply with the cell.

보고:

N/A

예시:

ds_grid_multiply(mygrid, 5, 5, 2)

위의 코드는 주어진 DS 그리드 셀에 저장된 값에 2를 곱합니다.
