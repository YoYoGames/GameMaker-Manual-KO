ds_grid_resize

이 기능을 사용하면 주어진 DS 그리드의 크기를 다른 너비 및/또는 높이로 조정할 수 있습니다. 격자 크기가 현재 격자보다 크면 새 셀의 기본 값은 0이고 크기가 더 작으면 더 이상 새 크기 내에 있지 않은 셀에 있는 값이 손실됩니다. 다른 모든 셀은 그대로 유지됩니다.

통사론:

ds_grid_resize(index, w, h);

논쟁 유형 설명
index This index of the grid to resize.
w The new width of the grid.
h The new height of the grid.

보고:

예시:

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

위의 코드는 전역 변수 "Grid"에 색인된 DS 그리드의 크기를 조정한 다음 각 셀이 값 -1을 보유하도록 지웁니다.
