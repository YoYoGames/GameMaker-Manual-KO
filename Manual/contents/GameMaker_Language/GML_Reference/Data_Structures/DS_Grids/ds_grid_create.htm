ds_grid_create

이 기능을 사용하여 지정된 셀 너비와 높이의 새 DS 그리드 데이터 구조를 생성할 수 있습니다. 이 함수는 이 DS 그리드를 처리하는 모든 추가 함수에서 사용해야 하는 ID를 반환합니다.

중요한! 데이터 구조를 생성할 때 식별하기 위한 인덱스 값은 0부터 시작하는 정수 값입니다. 이는 다른 유형의 데이터 구조가 동일한 인덱스 값을 가질 수 있음을 의미하므로 의심스러운 경우 다음을 사용해야 합니다.ds_exists() 액세스하기 전에 기능. 또한 인덱스는 재사용되므로 파괴된 데이터 구조 인덱스 값은 나중에 새로 생성된 값에서 사용될 수 있습니다.

통사론:

ds_grid_create(w, h);

논쟁 유형 설명
w The width of the grid to be created.
h The height of the grid to be created.

보고:

DS Grid ID

예시:

mygrid = ds_grid_create(10, 10)

이렇게 하면 높이가 10셀이고 너비가 10셀인 그리드가 생성됩니다.
