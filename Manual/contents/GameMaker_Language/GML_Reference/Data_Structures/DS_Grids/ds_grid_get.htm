ds_grid_get

이 함수는 주어진 DS 그리드 내의 모든 셀에서 값을 가져오는 데 사용할 수 있습니다. 유효하지 않은 그리드 좌표를 함수에 전달하면 반환되는 값은 undefined 출력 창에 오류가 표시됩니다.

통사론:

ds_grid_get(index, x, y);

논쟁 유형 설명
index The index of the grid.
x The x position of the cell you want to find the value of.
y The y position of the cell you want to find the value of.

보고:

Variable

예시:

var xx = irandom(ds_grid_width(grid) - 1);
var yy = irandom(ds_grid_height(grid) - 1);
val = ds_grid_get(grid, xx, yy)

위의 코드는 "grid" 변수에 색인된 DS 그리드에서 임의의 셀을 선택하고 그 값을 변수 "val"에 저장합니다.
