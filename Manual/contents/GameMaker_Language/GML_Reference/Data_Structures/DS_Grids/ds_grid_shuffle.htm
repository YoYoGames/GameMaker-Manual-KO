ds_grid_shuffle

이 함수는 그리드 내의 모든 셀에 있는 모든 값의 위치를 임의화하는 데 사용할 수 있습니다.

참고 이 기능은 GameMaker 가 디버깅 코드를 훨씬 더 쉬운 작업으로 만들기 위해 매번 동일한 초기 임의 시드를 생성하기 때문에 게임을 새로 실행할 때마다 그리드 셀을 동일한 위치로 섞습니다. 이 동작을 방지하려면 다음을 사용하십시오.randomise() 게임 시작 시. 최종 실행 패키지는 이 동작을 표시하지 않고 플레이할 때마다 무작위로 실행되기 때문에 게임을 테스트하고 디버깅할 때만 해당됩니다.

통사론:

ds_grid_shuffle(index);

논쟁 유형 설명
index The index of the grid to shuffle.

보고:

예시:

ds_grid_shuffle(grid)

이것은 "grid" 변수에 색인된 DS 그리드의 모든 값을 가져와서 다른 순서를 얻기 위해 섞습니다.
