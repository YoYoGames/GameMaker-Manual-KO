ds_grid_read

이 함수는 함수에 의해 이전에 생성된 문자열을 변환하는 데 사용할 수 있습니다.ds_grid_write() 다시 DS 그리드로. DS 그리드는 이전에 생성되어 있어야 합니다(아래 예 참조).

지정된 DS 문자열이 GameMaker: Studio 1.2.x 런타임(또는 이전 버전)에 의해 작성된 경우 선택적 인수 "legacy"를 지정하고 다음으로 설정해야 합니다. true 그 이후에 문자열 형식이 변경되었기 때문입니다.

통사론:

ds_grid_read(index, string [, legacy]);

논쟁 유형 설명
index DS 그리드 ID The index of the grid to read.
string 끈 The string to read into the DS grid.
legacy 부울 OPTIONAL Can be either true or false or omitted completely.

보고:

N/A

예시:

grid = ds_grid_create(room_width div 32, room_height div 32);
ini_open("Save.ini");
ds_grid_read(grid, ini_read_string("Save", "0", ""));
ini_close();

위의 코드는 방의 크기를 기반으로 DS 그리드를 만든 다음(각 32x32 픽셀 정사각형은 하나의 그리드 셀을 나타냄) ini 파일에서 새 DS 그리드로 이전에 저장된 그리드 데이터 세트를 읽습니다.
