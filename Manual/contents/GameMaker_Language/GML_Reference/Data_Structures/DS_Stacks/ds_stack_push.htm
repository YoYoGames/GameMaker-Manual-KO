ds_stack_push

이 함수는 모든 데이터 유형이 될 수 있는 값을 스택의 맨 위에 푸시 (추가)합니다. 이 함수는 추가로 14개의 선택적 인수(총 15개 추가 가능)를 사용할 수 있으므로 단일 호출에서 여러 값을 스택에 연속적으로 푸시할 수 있습니다.

통사론:

ds_stack_push(id, val [, val2, ... val15]);

논쟁 유형 설명
id The id of the data structure to push onto.
val The value to push onto the stack.
[val2,... val13] Optional values to be added to the stack.

보고:

예시:

move_stack = ds_stack_create();
ds_stack_push(move_stack, x, y, x, y + 200, x + 200, y + 200, x +200, y);

위의 코드는 새 DS 스택을 만들고 해당 인덱스를 "move_stack" 변수에 저장합니다. 그런 다음 나중에 사용할 수 있도록 여러 값을 스택에 푸시합니다.

뒤로: ds_stack_create
다음: ds_stack_read
