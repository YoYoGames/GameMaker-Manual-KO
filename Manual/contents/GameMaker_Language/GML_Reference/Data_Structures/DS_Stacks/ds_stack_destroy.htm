ds_stack_destroy

이 함수는 메모리에서 주어진 스택 데이터 구조를 제거하여 사용하던 리소스를 해제하고 포함된 모든 값을 제거합니다. 이 기능은 DS 스택 사용을 마치면 항상 사용하여 게임 속도를 늦추고 충돌을 일으킬 수 있는 메모리 누수를 방지해야 합니다.

중요한! 데이터 구조를 생성할 때 식별하는 인덱스 값은 0부터 시작하는 정수 값입니다. 이는 다른 유형의 데이터 구조가 동일한 인덱스 값을 가질 수 있음을 의미하므로 의심스러운 경우 다음을 사용해야 합니다.ds_exists() 액세스하기 전에 기능. 또한 인덱스는 재사용되므로 파괴된 데이터 구조 인덱스 값은 나중에 새로 생성된 값에서 사용될 수 있으므로 파괴 후 DS 인덱스를 보유하는 변수를 항상 -1로 설정하는 것이 좋습니다.

통사론:

ds_stack_destroy(id);

논쟁 유형 설명
id The id of the data structure to remove.

보고:

예시:

if (lives == 0)
{
    ds_stack_destroy(AI_stack);
    AI_stack = -1;
    room_goto(rm_Menu);
}

위의 코드는 내장 전역 변수 "lives"의 값을 확인하고 0이면 변수 "AI_stack"에 인덱스된 DS 스택을 파괴한 다음 방을 변경합니다.

뒤로: ds_stack_create
다음: ds_stack_clear
