DS 스택

DS 스택은 DS 대기열 과 유사한 소위 LIFO(후입 선출) 데이터 구조입니다. 스택에 값을 푸시 한 다음 스택에서 값을 다시 꺼내 제거할 수 있습니다. 여기서 가장 최근에 스택에 푸시된 값이 스택에서 가장 먼저 꺼집니다(단지 동전 스택을 생각해 보십시오. 추가한 동전은 아래의 나머지 동전에 도달하기 전에 먼저 다시 제거해야 합니다. 스택은 처리해야 할 인터럽트가 있거나 재귀 함수가 있거나 게임을 위한 기초적인 AI를 구성할 때에도 자주 사용됩니다.

참고: 모든 동적 리소스와 마찬가지로 데이터 구조는 메모리를 차지하므로 더 이상 필요하지 않을 때 항상 파괴되어 속도가 느려지고 결국 게임이 중단되는 메모리 누수를 방지해야 합니다.

참고: 데이터 구조가 존재하는지 확인해야 하는 경우 다음을 사용할 수 있습니다.ds_exists() 함수.

DS 스택을 처리하는 다음 함수가 있습니다.

  • ds_stack_create
  • ds_stack_destroy
  • ds_stack_clear
  • ds_stack_empty
  • ds_stack_size
  • ds_stack_copy
  • ds_stack_top
  • ds_stack_pop
  • ds_stack_push
  • ds_stack_read
  • ds_stack_write
