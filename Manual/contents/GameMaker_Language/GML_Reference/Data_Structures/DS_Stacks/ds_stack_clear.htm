ds_stack_clear

이 함수를 사용하면 주어진 스택 데이터 구조에서 모든 데이터를 지울 수 있습니다. 이것은 데이터 구조를 파괴하지 않습니다.ds_stack_destroy() ) 모든 데이터만 지우고 빈 스택을 반환합니다.

통사론:

ds_stack_clear(id);

논쟁 유형 설명
id The id of the data structure to clear.

보고:

예시:

if (ai_count = 15 && !ds_stack_empty(AI_stack))
{
    ds_stack_clear(AI_stack);
    alarm[0] = room_speed;
    ai_count = 0;
}

위의 코드는 변수를 검사하여 특정 값에 도달했는지 확인하고 변수가 있으면 "AI_stack" 변수에 색인된 DS 스택을 지우고 경보를 설정하고 변수를 0으로 재설정합니다.
