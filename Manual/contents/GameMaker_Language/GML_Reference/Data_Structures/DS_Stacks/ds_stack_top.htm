ds_stack_top

이 함수는 스택의 첫 번째 값("위에 있는" 값) 만 읽습니다. 값을 팝 하지 않습니다. 즉, 이 함수 또는ds_stack_pop(). 스택이 비어 있으면 함수는 상수를 반환합니다. undefined, 그렇지 않으면 스택에 포함된 실제 또는 문자열 값을 반환합니다.

통사론:

ds_stack_top(id);

논쟁 유형 설명
id The id of the data structure to read from.

보고:

Variable (Data type value that is stored in the stack) or undefined

예시:

num = ds_stack_top(control_stack);

위의 코드는 "control_stack" 변수에 인덱싱된 스택에서 값을 읽고 반환 값을 변수 "num"에 저장합니다.
