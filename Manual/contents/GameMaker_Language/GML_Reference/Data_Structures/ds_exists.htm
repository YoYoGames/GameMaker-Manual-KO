ds_exists

이 함수를 사용하면 주어진 유형의 데이터 구조가 존재하는지 확인할 수 있습니다. 변수에 저장된 "색인" 값과 아래 나열된 상수 중 하나일 수 있는 DS "유형"을 제공하면 함수가 다음을 반환합니다. true 데이터 구조가 존재하고 false 그렇지 않으면.

DS 유형 상수
일정한 설명
ds_type_map 지도 데이터 구조
ds_type_list 목록 데이터 구조
ds_type_stack 스택 데이터 구조
ds_type_grid 그리드 데이터 구조
ds_type_queue 큐 데이터 구조
ds_type_priority 우선 순위 데이터 구조

참고 동일한 인덱스 번호가 다른 유형의 여러 데이터 구조에서 사용될 수 있으므로 이 함수를 사용하여 데이터 구조의 유형을 확인할 수 없습니다.

통사론:

ds_exists(ind, type);

논쟁 유형 설명
ind 모든 DS ID The variable index to check for the data structure
type DS 유형 상수 The type of data structure to check for (see the list of constants above)

보고:

Boolean

예시:

if !ds_exists(ai_grid, ds_type_grid)
{
    ai_grid = ds_grid_create(room_width / 32, room_height / 32);
}

위의 코드는 (이전에 초기화된) 변수 "ai_grid"를 확인하여 DS 그리드 유형 데이터 구조를 인덱싱하는지 확인하고, 그렇지 않으면 하나를 생성하고 해당 인덱스를 변수에 저장합니다.
