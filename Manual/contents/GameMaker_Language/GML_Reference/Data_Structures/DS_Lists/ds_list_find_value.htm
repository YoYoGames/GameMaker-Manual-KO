ds_list_find_value

이 기능을 사용하면 주어진 목록 위치를 확인할 수 있으며 해당 위치에 대해 목록에 있는 값이 반환됩니다. 주어진 목록 크기를 벗어나는 위치를 지정하면(예: 10 값 목록에서 위치 11) 함수가 다음을 반환할 수 있습니다. undefined 또는 0. 이는 목록을 생성할 때 내부적으로 목록의 처음 몇 개 항목이 0으로 설정되어 처음에 목록에 항목을 추가할 때 성능 문제를 최소화하기 때문입니다(비록ds_list_size() 함수는 새로 생성된 목록에서 여전히 0을 반환합니다). 생성할 때 목록이 "진정으로" 비어 있는지 확인하려면 다음을 호출해야 합니다.ds_list_clear() 목록을 만든 후 채워지지 않은 목록 슬롯에 대해 반환된 모든 값은 undefined, 다음을 사용하여 일관되게 확인할 수 있습니다.is_undefined() 함수.

통사론:

ds_list_find_value(id, pos);

논쟁 유형 설명
id DS 목록 ID The id of the list to use.
pos 진짜 The position to look at, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.

보고:

Variable or undefined

예시:

val = ds_list_find_value(list, ds_list_size(list) - 1);

위의 코드는 목록의 마지막 위치에 있는 "list" 변수에 인덱싱된 목록을 확인하고 반환된 값을 "val" 변수에 저장합니다.
