ds_list_copy

이 기능을 사용하면 한 목록의 내용을 다른 목록으로 복사할 수 있습니다. 두 목록 모두 이전에 생성된 목록이어야 하며 복사 대상 목록 에 이미 정보가 있는 경우 이 목록이 먼저 지워집니다. 최종 결과는 동일한 정보를 포함하는 두 개의 독립적인 목록입니다.

통사론:

ds_list_copy( id, source );

논쟁 유형 설명
id The id of the list being copied to.
source The id of the list to be copied from.

보고:

예시:

if !ds_list_empty(main_list)
{
    old_list = ds_list_create();
    ds_list_copy(old_list, main_list);
    ds_list_clear(main_list);
}

위의 코드는 DS 목록이 비어 있는지 확인합니다. 비어 있지 않으면 다른 DS 목록(이전에 생성됨)에 복사된 다음 원래 목록이 지워집니다.
