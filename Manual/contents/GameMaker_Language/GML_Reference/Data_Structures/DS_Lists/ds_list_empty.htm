ds_list_empty

이 기능을 사용하면 주어진 DS 목록이 비어 있는지 확인할 수 있습니다(반환 true ) 여부(반환 false ).

통사론:

ds_list_empty(id);

논쟁 유형 설명
id DS 목록 ID The id of the data structure to check.

보고:

예시:

if (count == 15( && (!ds_list_empty(command_list))
{
    ds_list_clear(command_list);
    alarm[0] = room_speed;
    count = 0;
}

위의 코드는 변수를 검사하여 특정 값에 도달했는지 확인하고 변수가 있으면 "command_list" 변수에 색인된 DS 목록을 지우고 경보를 설정하고 변수를 0으로 재설정합니다.
