ds_list_set

이 기능은 이전에 추가된 목록 항목을 설정하는 데 사용할 수 있습니다. 목록 ID(목록을 생성할 때 반환됨)와 목록 내에서 설정할 위치 및 설정할 값을 제공합니다. 설정 중인 항목이 목록의 범위를 벗어나면(즉, 목록 항목을 20으로 설정했지만 현재 목록에 10개의 항목만 포함된 경우) 목록은 지정된 위치로 채워지고 각 항목은 0으로 설정됩니다. 이 기능은 DS 목록 접근자를 사용하는 것과 동일합니다.

통사론:

ds_list_set(id, pos, val);

논쟁 유형 설명
id DS 목록 ID The id of the list to add to.
pos 변하기 쉬운 The position within the list to set.
[val2,... max_val] 변하기 쉬운 The value to set in the list.

예시:

for (var i = 0; i < ds_list_size(list); i++;)
{
    ds_list_set(list, i, -1);
}

위의 코드는 "sc_list" 변수에 색인된 목록에 "score" 변수에 저장된 값을 추가합니다.
