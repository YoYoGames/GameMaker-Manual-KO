ds_list_shuffle

이 기능을 사용하면 목록을 섞을 수 있습니다. 그러면 모든 구성 요소 값이 원래 목록에 추가된 값에서 임의의 위치로 재정렬됩니다.

참고: 이 기능은 GameMaker 가 디버깅 코드를 훨씬 더 쉬운 작업으로 만들기 위해 매번 동일한 초기 무작위 시드를 생성하기 때문에 게임을 새로 실행할 때마다 목록 항목을 동일한 위치로 섞습니다. 이 동작을 방지하려면 다음을 사용하십시오. randomise() 게임 시작 시. 최종 실행 패키지는 이 동작을 표시하지 않고 플레이할 때마다 무작위로 실행되기 때문에 게임을 테스트하고 디버깅할 때만 해당됩니다.

통사론:

ds_list_shuffle(id);

논쟁 유형 설명
id The id of the list to shuffle.

보고:

예시:

if restart
{
    ds_list_shuffle(card_list);
}

위의 코드는 "restart" 변수가 다음과 같이 플래그 지정되면 "card_list" 변수에 색인된 목록을 섞습니다 true.
