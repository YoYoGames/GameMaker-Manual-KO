ds_list_clear

이 기능을 사용하면 주어진 목록 데이터 구조에서 모든 데이터를 지울 수 있습니다. 이것은 데이터 구조를 파괴하지 않습니다.ds_list_destroy() ) 모든 데이터만 지우고 목록을 비어 있게 만듭니다(크기가 0). 목록을 지우면 목록에 저장된 모든 데이터 구조가 역참조되어 메모리 누수가 발생하므로 이를 방지하려면 목록을 지우기 전에 목록을 살펴보고 모든 데이터 구조 항목을 수동으로 파괴해야 합니다. 이것이 필요하지 않은 유일한 경우는 목록의 항목에 다른 DS 목록 또는 DS 맵 으로 플래그를 지정했을 때입니다. 클리어.

통사론:

ds_list_clear(id);

논쟁 유형 설명
id The id of the data structure to clear.

보고:

예시:

if (count == 15) && (!ds_list_empty(command_list))
{
    ds_list_clear(command_list);
    alarm[0] = room_speed;
    ai_count = 0;
}

위의 코드는 변수를 검사하여 특정 값에 도달했는지 확인하고 변수가 있으면 "command_list" 변수에 색인된 DS 목록을 지우고 경보를 설정하고 변수를 0으로 재설정합니다.
