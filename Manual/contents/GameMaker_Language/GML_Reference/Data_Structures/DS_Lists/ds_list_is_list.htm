ds_list_is_list

이 기능을 사용하면 다른 DS 목록이 DS 목록 내의 지정된 위치에 저장되어 있는지 확인할 수 있습니다. 주어진 위치에 DS 목록 ID가 포함되어 있으면 함수는 다음을 반환합니다. true 그렇지 않으면 반환됩니다 false.

이것은 수동으로 표시된 목록만 감지합니다. ds_list_mark_as_list() 함수.

통사론:

ds_list_is_list(id, pos);

논쟁 유형 설명
id The id of the list to check.
pos The position within the list to check.

보고:

예시:

var size = ds_list_size(ships);
for (var i = 0; i < size; i++)
{
    if (ds_list_is_list(ships, i))
    {
        ds_list_destroy(ships[| i]);
    }
}
ds_list_destroy(ships);

위의 코드는 DS 목록을 반복하고 항목에 다른 목록 ID가 포함되어 있는지 확인합니다. 그렇다면 이러한 목록이 파괴되고 루프가 완료된 후 기본 목록이 파괴됩니다.
