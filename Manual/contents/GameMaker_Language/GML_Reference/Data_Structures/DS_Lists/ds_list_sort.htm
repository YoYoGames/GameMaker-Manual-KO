ds_list_sort

이 기능을 사용하면 목록 내의 모든 값을 오름차순 또는 내림차순으로 정렬할 수 있습니다. 목록에 문자열이 포함된 경우 영어 26자 알파벳을 기준으로 알파벳순으로 정렬됩니다.

통사론:

ds_list_sort(id, ascend);

논쟁 유형 설명
id The id of the list to sort.
ascend Whether the values should be ascending (true ) or descending (false ) order.

보고:

예시:

if newgame
{
    ds_list_sort(name_list, true);
}

위의 코드는 "newgame" 변수가 다음과 같이 플래그 지정된 경우 "name_list" 변수에 색인된 목록을 정렬합니다. true.
