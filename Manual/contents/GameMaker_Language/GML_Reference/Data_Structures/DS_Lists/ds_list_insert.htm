ds_list_insert

이 함수는 주어진 위치의 목록에 주어진 값을 추가합니다. 목록에 주어진 위치 다음에 더 많은 값이 포함되어 있으면 해당 위치가 위로 이동하여 목록을 하나씩 더 크게 만드는 공간을 만듭니다.

통사론:

ds_list_insert(id, pos, val);

논쟁 유형 설명
id DS 목록 ID The id of the list to add to.
pos 진짜 The position to add the value, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.
val 변하기 쉬운 The value to add to the list.

보고:

N/A

예시:

ds_list_insert(list, 9, score);

위의 코드는 변수 "score"에 저장된 값을 "list" 변수에 의해 인덱싱된 목록의 10번째 위치에 추가합니다(목록은 0부터 계산되므로 10개의 값 목록은 0 - 9까지 번호가 매겨집니다).
