ds_list_write

이 함수는 다음을 사용하여 저장하거나 다른 데이터 구조로 전송할 수 있는 문자열을 반환합니다.ds_list_read() 함수.

참고: 반환된 문자열은 사람이 읽을 수 있는 문자열이 아니라 데이터 구조 내용의 덤프입니다.

통사론:

ds_list_write(id);

논쟁 유형 설명
id The id of the data structure to write.

보고:

예시:

ini_open("save.ini");
var str = ds_list_write(list);
ini_write_string("Lists", "0", str);
ds_list_clear(list);
ini_close();

위의 코드는 ini 파일을 연 다음 "list" 변수에 인덱싱된 DS 목록에 저장된 정보를 포함하는 문자열을 작성합니다. 그런 다음 목록이 지워지고 ini 파일이 닫힙니다.
