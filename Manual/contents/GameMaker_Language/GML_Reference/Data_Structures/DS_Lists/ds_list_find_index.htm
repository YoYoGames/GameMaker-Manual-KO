ds_list_find_index

이 기능을 사용하면 값에 대해 주어진 목록을 확인할 수 있으며 해당 값에 대한 목록 내 위치가 반환됩니다. 목록에 같은 값을 가진 항목이 두 개 이상 있으면 그 중 하나의 위치가 반환될 수 있으며 값이 없으면 -1이 반환됩니다.

통사론:

ds_list_find_index(id, val);

논쟁 유형 설명
id DS 목록 ID The id of the list to use.
val 변하기 쉬운 The value to find.

보고:

Real

예시:

pos = ds_list_find_index(list, "Player1");

위의 코드는 "list" 변수에 인덱싱된 목록에서 "Player1" 값을 확인하고 반환된 위치를 "pos" 변수에 저장합니다.
