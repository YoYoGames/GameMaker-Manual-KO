ds_list_create

이 함수는 새 목록 데이터 구조를 만들고 인덱스 값을 반환합니다. 이 값은 변수에 저장해야 하며 목록과 관련된 모든 추가 함수 호출에 사용해야 합니다. 목록이 채워 지기 전에 목록의 값에 액세스하려고 하면(예: 목록이 "비어 있음") 성능 문제를 최소화하기 위해 목록의 처음 몇 항목이 생성될 때 내부적으로 설정되기 때문에 반환 값은 0이 될 수 있습니다. 처음에 목록에 항목을 추가할 때(비록ds_list_size() 함수는 새로 생성된 목록에서 여전히 0을 반환합니다). 생성할 때 목록이 "진정으로" 비어 있는지 확인하려면 다음을 호출해야 합니다.ds_list_clear() 목록을 만든 후 채워지지 않은 목록 슬롯에 대해 반환된 모든 값은 undefined.

중요한! 데이터 구조를 생성할 때 식별하는 인덱스 값은 0부터 시작하는 정수 값입니다. 이는 다른 유형의 데이터 구조가 동일한 인덱스 값을 가질 수 있음을 의미하므로 의심스러운 경우 다음을 사용해야 합니다.ds_exists() 액세스하기 전에 기능. 또한 인덱스는 재사용되기 때문에 파괴된 데이터 구조 인덱스 값은 나중에 새로 생성된 값에서 사용할 수 있습니다.

통사론:

ds_list_create();

보고:

DS List ID

예시:

list = ds_list_create();

이렇게 하면 새 목록이 생성되고 해당 인덱스 ID가 인스턴스 변수 "list"에 할당됩니다.
