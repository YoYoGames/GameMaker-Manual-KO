ds_list_add

이 함수는 목록에 새 값(실수 또는 문자열)을 추가하는 데 사용할 수 있으며, 이 값은 끝에 추가됩니다. 이 함수는 추가 옵션 인수(필요한 만큼)를 사용할 수 있으므로 단일 호출로 목록에 여러 값을 연속적으로 추가할 수 있습니다.

통사론:

ds_list_add(id, val1 [, val2, ... max_val]);

논쟁 유형 설명
id DS 목록 ID The id of the list to add to.
val 변하기 쉬운 The value to add to the list.
[val2,... max_val] 변하기 쉬운 Optional values to be added to the list.

보고:

N/A

예시:

ds_list_add(sc_list, score);

위의 코드는 "sc_list" 변수에 색인된 목록에 "score" 변수에 저장된 값을 추가합니다.
