ds_list_replace

이 함수는 주어진 위치의 값을 다른 값으로 대체합니다.

통사론:

ds_list_replace(id, pos, val);

논쟁 유형 설명
id DS 목록 ID The id of the list to change.
pos 진짜 The position to replace the value, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.
val 변하기 쉬운 The new value to replace the given value with.

보고:

N/A

예시:

ds_list_replace(n_list, 3, name);

위의 코드는 "name" 변수에 저장된 값에 대해 목록의 위치 3에 저장된 값을 대체합니다.
