ds_list_size

이 함수는 목록의 "크기"를 반환합니다. 즉, 목록에 추가된 항목의 수입니다.

통사론:

ds_list_size(id);

논쟁 유형 설명
id The id of the data structure to check.

보고:

예시:

if !ds_list_empty(control_list)
{
    num = ds_list_size(control_list);
}

위의 코드는 DS 목록이 비어 있는지 여부를 확인하고 비어 있지 않은 경우 포함된 항목 수를 가져와 변수에 값을 저장합니다.
