ds_map_exists

이 함수는 반환됩니다 true 지정된 키가 (이전에 생성된) DS map에 존재하는 경우, 그리고 false 그렇지 않은 경우.

통사론:

ds_map_exists(id, key);

논쟁 유형 설명
id DS 맵 ID The id of the data structure to check
key 끈 The key to check for

보고:

Boolean

예시:

if !ds_map_exists(inventory, "potions")
{
    ds_map_add(inventory, "potions", 1);
}

위의 코드는 "물약" 키에 대해 "inventory" 변수에 색인된 DS 맵을 확인하고 존재하지 않는 경우 맵에 추가합니다.
