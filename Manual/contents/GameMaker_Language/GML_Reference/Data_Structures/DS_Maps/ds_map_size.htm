ds_map_size

이 기능을 사용하면 (이전에 생성된) DS 맵에 포함된 키/값 쌍의 수를 찾을 수 있습니다.

통사론:

ds_map_size(id);

논쟁 유형 설명
id The id of the data structure to check.

보고:

예시:

if (ds_map_size(inventory) > 49)
{
    full = true;
}

위의 코드는 DS 맵의 크기(예: 키/값 쌍의 수)를 확인하고 49보다 크면 변수 "full"을 다음으로 설정합니다. true.
