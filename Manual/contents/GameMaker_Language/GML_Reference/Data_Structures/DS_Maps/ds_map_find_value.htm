ds_map_find_value

이 함수를 사용하면 지정된 키에서 값을 가져올 수 있습니다. 함수의 입력 값은 (기존에 생성된) 사용할 DS map과 확인할 키입니다.

참고: 그러한 키가 없으면 함수는 다음을 반환합니다. undefined. 항상 이것을 사용하여 확인해야 합니다. is_undefined() 함수

통사론:

ds_map_find_value(id, key)

논쟁 유형 설명
id The id of the map to use.
key The key to find.

보고:

Variable or undefined

예시:

amount = ds_map_find_value(inventory, "food");

또는 지도 접근 자 "?"를 사용하여:

amount = inventory[? "food"];

위의 코드는 "food" 키의 값을 가져와 변수 "mount"에 저장합니다.
