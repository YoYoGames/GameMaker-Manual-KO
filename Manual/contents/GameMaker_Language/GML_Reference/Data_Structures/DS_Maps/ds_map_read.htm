ds_map_read

이 함수는 이전에 함수에 의해 생성된 문자열을 사용합니다.ds_map_write() 그런 다음 이전에 만든 DS 맵으로 읽습니다. 문자열을 읽는 맵에 키/값 쌍이 포함되어 있으면 저장된 맵이 재구성되기 전에 먼저 지워집니다.

지정된 DS 문자열이 GameMaker: Studio 1.2.x 런타임(또는 이전 버전)에 의해 작성된 경우 선택적 인수 "legacy"를 지정하고 다음으로 설정해야 합니다. true 그 이후에 문자열 형식이 변경되었기 때문입니다.

통사론:

ds_map_read(id, str, [legacy]);

논쟁 유형 설명
id DS 맵 ID The id of the data structure to read the string into
str 끈 The string to read
legacy 부울 OPTIONAL Can be either true or false or omitted completely.

보고:

N/A

예시:

inventory = ds_map_create();
ini_open("map.ini");
var t_string = ini_read_string("Saved", "0", "");
if (t_string != "")
{
    ds_map_read(inventory, t_string);
}
ini_close();

위의 코드는 새 DS 맵을 만들고 id 인덱스를 "inventory" 변수에 저장합니다. 그런 다음 ini 파일을 열고 해당 파일의 문자열을 임시 변수 "t_string"으로 읽습니다. 마지막으로 문자열이 유효한지 확인하고(기본 ini 값 ""이 아님) 유효하다면 ini를 다시 닫기 전에 문자열을 새로 생성된 DS 맵으로 읽어들입니다.
