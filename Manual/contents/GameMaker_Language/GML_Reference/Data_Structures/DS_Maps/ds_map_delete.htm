ds_map_delete

이 기능을 사용하면 이전에 생성된 지정된 DS map 에서 지정된 키(및 해당 값)를 제거할 수 있습니다.

통사론:

ds_map_delete(id, key);

논쟁 유형 설명
id DS 맵 ID The id of the map to change.
key 끈 The key to delete (along with its associated value).

보고:

N/A

예시:

ds_map_delete(inventory, "shield");

위의 코드는 DS 맵(인벤토리)에서 키 "shield"(및 쌍을 이루는 값)를 삭제합니다.
