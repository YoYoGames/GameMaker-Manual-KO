ds_map_replace_map

이 기능으로 대체할 수 있습니다. DS Map 이전에 생성된 다른 맵과 함께 주어진 "키"에 저장된 것입니다. 이 기능은 다음을 사용하여 인코딩할 JSON 호환 맵을 생성하도록 설계되었습니다.json_encode() 해당 기능과 함께 사용해야 합니다.

통사론:

ds_map_replace_map(id, key, value)

논쟁 유형 설명
id DS 맵 ID The id of the ds_map to use.
key 끈 The key to replace.
value DS 맵 ID The id of the ds_map to use to replace the one previously stored in the given key.

보고:

N/A

예시:

var temp_map = ds_map_create();
ds_map_add_list(temp_map, "list", j_list);
ds_map_add(temp_map, "array", j_array);
ds_map_replace_map(j_map, "maps", temp_map);
var j = json_encode(j_map);
ds_map_destroy(temp_map);

위의 코드는 DS Map 배열로 채우고 DS List 이전에 저장된 지도를 교체하기 전에 DS Map "j_map".
