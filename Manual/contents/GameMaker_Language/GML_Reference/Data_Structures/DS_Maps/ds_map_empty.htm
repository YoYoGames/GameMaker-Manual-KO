ds_map_empty

이 함수는 단순히 반환합니다 false 지정된(이전에 생성된) DS 맵에 키/값 쌍이 포함된 경우 또는 true 그렇지 않은 경우.

통사론:

ds_map_empty(id);

논쟁 유형 설명
id The id of the data structure to check.

보고:

예시:

if ds_map_empty(inventory)
{
    weight = 0;
}

위의 코드는 "inventory" 변수에 인덱싱된 DS 맵에 키/값 쌍이 있는지 확인하고 없으면 "weight" 변수를 0으로 설정합니다.
