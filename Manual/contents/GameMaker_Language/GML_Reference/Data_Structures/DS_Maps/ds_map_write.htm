ds_map_write

이 기능은 지정된 맵의 DS 맵 데이터를 문자열 형식으로 변환한 다음 *.ini 또는 *.txt 파일에 기록하여 쉽게 저장할 수 있습니다. 이 문자열은 나중에 다음을 사용하여 새 ds_map으로 다시 읽을 수 있습니다.ds_map_read().

참고: 반환된 문자열은 사람이 읽을 수 있는 문자열이 아니라 데이터 구조 내용의 덤프입니다.

통사론:

ds_map_write(id);

논쟁 유형 설명
id The id of the data structure to use

보고:

예시:

ini_open("map.ini");
var t_string;
t_string = ds_map_write(inventory);
ini_write_string("Saved", "0", t_string);
ini_close();

위의 코드는 쓸 준비가 된 ini 파일을 엽니다. 그런 다음 사용합니다. ds_map_write() 임시 변수 "t_string"에 저장된 문자열을 생성합니다. 마지막으로, 닫기 전에 해당 문자열을 ini 파일에 씁니다.
