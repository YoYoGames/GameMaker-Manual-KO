ds_map_is_map

이 기능을 사용하면 DS 맵이 주어진 맵 키에 저장되어 있는지 확인할 수 있습니다. 주어진 키에 DS 맵 ID가 포함되어 있으면 함수는 다음을 반환합니다. true 그렇지 않으면 반환됩니다 false.

이것은 다음을 사용하여 추가된 지도만 감지합니다. ds_map_add_map() 함수.

통사론:

ds_map_is_map(id, key)

논쟁 유형 설명
id The id of the map to use.
key The key to replace.

보고:

예시:

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++)
{
    if (ds_map_is_map(inventory, key))
    {
        ds_map_destroy(inventory[? key]);
    }
    key = ds_map_find_next(inventory);
}
ds_map_destroy(inventory);

위의 코드는 DS 맵을 반복하고 그 안에 있는 키가 다른 DS 맵용인지 확인합니다. 그렇다면 저장된 DS 맵이 파괴되고 루프의 끝에서 메인 DS 맵도 파괴됩니다.
