ds_map_secure_save_buffer

이 기능은 이전에 생성된 DS 맵을 버퍼에 저장합니다. DS 맵 ID 값을 제공합니다(함수에서 반환된ds_map_create() ) 및 쓸 버퍼의 ID(함수에 의해 반환됨)buffer_create() ). 저장 중인 DS 맵에 어레이가 포함된 경우 저장 시 대신 DS 목록으로 변환됩니다.

중요한! 안전하게 저장된 DS map 파일은 해당 파일을 생성한 기기에서만 불러올 수 있으며, 다른 장치에 저장된 파일을 불러오려고 하면 작동하지 않습니다.

참고: 이 기능은 HTML5 및 UWP에서 지원되지 않습니다.

통사론:

ds_map_secure_save_buffer(filename);

논쟁 유형 설명
id DS 맵 ID The DS map ID value.
buffer 버퍼 ID The buffer to save to.

보고:

N/A

예시:

buff = buffer_create(128,buffer_grow,4);
var map = ds_map_create();
ds_map_add(map,"bob","ajob");
ds_map_add(map,"money",10);
ds_map_secure_save_buffer(map, buff);
ds_map_destroy(map);

위의 코드는 버퍼와 DS 맵을 만든 다음 맵을 일부 값으로 채우고 맵을 삭제하기 전에 버퍼에 씁니다.
