ds_map_add

이 함수는 키/값 쌍 세트를 지정된 DS 맵 에 추가하는 데 사용해야 합니다. 이 함수를 확인하여 성공했는지 여부를 확인할 수 있습니다(반환됩니다. true 성공하거나 false 그렇지 않으면) DS 맵에 동일한 키가 이미 존재하거나 존재하지 않는 DS 맵을 추가할 맵의 ID로 지정하면 실패할 수 있기 때문입니다. 제공하는 키와 값은 데이터 유형의 모든 조합으로 구성될 수 있으므로 다음과 그 이상을 모두 사용할 수 있습니다(실제로는 가장 일반적으로 키에 문자열을 사용함).

ds_map_add(map, 5, undefined);
ds_map_add(map, "level", 1);
ds_map_add(map, 89.6, "hello world");
ds_map_add(map, 5, buffer_get_address(buff));

접근자 "를 사용하여 지도에 추가할 수도 있습니다.? ", 아래 그림과 같이:

map[? 5] = undefined;
map[? "level"] = 1;
map[? 89.6] = "hello world";
map[? 5] = buffer_get_address(buff);

참고: GameMaker 의 다른 데이터 구조와 달리 이 키는 DS 맵 의 시작(또는 끝)으로 이동하지 않고 DS 맵 어딘가 로 이동합니다.

통사론:

ds_map_add(id, key, val);

논쟁 유형 설명
id The id of the map to add to.
key The key of the value to add.
val The value to add to the map.

보고:

예시:

inventory = ds_map_create();
ds_map_add(inventory, "hp potion", 1);
ds_map_add(inventory, "gold", 100);

그러면 id를 변수 "inventory"에 할당하여 새 맵이 생성됩니다. 그런 다음 맵에 "hp potion"과 "gold"라는 두 개의 새 키를 추가하고 초기 값을 1과 100으로 설정합니다.
