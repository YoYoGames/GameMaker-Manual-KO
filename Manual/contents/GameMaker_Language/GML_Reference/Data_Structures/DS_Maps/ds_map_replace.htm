ds_map_replace

이 기능을 사용하면 DS 맵 내에서 주어진 키에 대한 값을 변경할 수 있습니다. 지도에 인덱스를 제공합니다(함수에서 반환한 대로ds_map_create() ) 그 다음 대체할 키(문자열 또는 정수 중 하나)와 키 값을 대체할 값. 주어진 키가 존재하지 않으면 자동으로 생성되고, 존재하는 경우 현재 값이 새 값으로 대체됩니다. 함수가 반환됩니다 true 키가 존재하고 값이 대체되는 경우 false 키가 존재하지 않고 값으로 새 키가 생성된 경우.

통사론:

ds_map_replace( id, key, val );

논쟁 유형 설명
id The id of the map to change.
key The key with the value that should be replaced by the new one
val The new value to replace the given value with

보고:

예시:

ds_map_replace(inventory, "torso", 55);

위의 코드는 키 "torso"에 대한 DS 맵을 조회하고 키가 발견되면(또는 존재하지 않는 경우 생성됨) 현재 값이 지정된 값으로 대체됩니다.
