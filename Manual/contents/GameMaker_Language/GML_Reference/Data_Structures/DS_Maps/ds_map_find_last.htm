ds_map_find_last

이 함수는 DS 맵에 저장된 마지막 키를 반환합니다. 이것은 추가한 순서의 마지막 키가 아닙니다! DS 맵은 DS 목록 을 사용하기 위해 선형 형식으로 저장되지 않으므로 이 모든 작업은 컴퓨터에 저장된 마지막 키를 찾는 것뿐입니다. 이것은 무언가를 찾기 위해 DS 맵을 반복해야 하는 경우에 유용할 수 있지만 느릴 수 있으므로 가능한 한 피해야 합니다.

이 함수는 다음을 반환합니다. undefined 주어진 DS 맵이 비어 있는 경우.

통사론:

ds_map_find_last(id);

논쟁 유형 설명
id The id of the map to use.

보고:

Variable or undefined

예시:

var size = ds_map_size(inventory);
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--;)
{
    if (key != "gold")
    {
        key = ds_map_find_previous(inventory, key);
    }
    else break;
}

위의 코드는 임시 변수를 생성한 다음 DS 맵 크기를 가져오고 맵에서 컴퓨터가 저장한 마지막 키를 찾습니다. 그런 다음 for 루프를 사용하여 키 값 "gold"를 찾는 DS 맵을 다시 반복합니다. 찾으면 루프를 해제합니다.
