ds_map_replace_list

이 기능을 사용하면 주어진 "키"에 저장된 DS 목록 을 이전에 생성된 다른 목록으로 교체할 수 있습니다. 이 기능은 다음을 사용하여 인코딩할 JSON 호환 맵을 생성하도록 설계되었습니다.json_encode() 해당 기능과 함께 사용해야 합니다.

통사론:

ds_map_replace_list(id, key, value)

논쟁 유형 설명
id DS 맵 ID The id of the map to use.
key 끈 The key to replace.
value DS 목록 ID The id of the ds_list to use to replace the one previously stored in the given key.

보고:

N/A

예시:

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
ds_map_replace_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_list_destroy(j_list);

위의 코드는 DS List 이전에 저장된 목록을 바꾸기 전에 다양한 전역 변수의 값으로 채우십시오. DS Map "j_map".
