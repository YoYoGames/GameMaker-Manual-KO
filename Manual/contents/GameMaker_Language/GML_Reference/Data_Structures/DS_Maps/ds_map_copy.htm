ds_map_copy

이 기능을 사용하여 한 맵의 내용을 이전에 다음을 사용하여 만든 다른 맵으로 복사할 수 있습니다. ds_map_create(). 복사 대상 DS 맵이 비어 있지 않으면 이 기능은 복사 하기 전에 먼저 지웁니다. 원래 DS 맵은 이 프로세스에 의해 변경되지 않은 상태로 유지됩니다.

통사론:

ds_map_copy(id, source);

논쟁 유형 설명
id The id of the map you are copying to
source The id of the map you are copying from

보고:

예시:

inventory_2 = ds_map_create();
ds_map_copy(inventory_2, inventory_1);

위의 코드는 새 지도를 만들고 "inventory_2" 변수에 할당합니다. 그런 다음 변수 "inventory_1"에 색인된 DS 맵의 내용을 이 새 맵에 복사합니다.
