ds_map_secure_load

이 기능은 주어진 파일에서 이전에 저장된 DS 맵을 로드합니다. 이 파일은 이전에 다음을 사용하여 생성되어야 합니다.ds_map_secure_save() 함수와 로드될 때 함수는 로드된 데이터에서 생성된 DS 맵의 인덱스 를 반환합니다. 이 DS 맵 인덱스는 변수에 저장해야 하며 이 맵에 대한 모든 추가 기능 호출에 사용해야 합니다. 로드되는 DS 맵이 키 값 중 하나로 어레이와 함께 저장된 경우 이 어레이는 로드 시 DS 목록으로 변환됩니다.

중요한! 보안 저장 파일의 기능 중 하나는 파일이 생성된 장치에 잠겨 있기 때문에 한 장치에 저장된 파일을 다른 장치에서 실행 중인 프로젝트에 로드할 수 없다는 것입니다.

통사론:

ds_map_secure_load(filename);

논쟁 유형 설명
filename The name of the file to load the map data from.

보고:

DS Map ID

예시:

p_map = ds_map_secure_load("p_data.dat");

위의 코드는 안전하게 저장된 DS 맵을 로드하고 인덱스 값을 나중에 사용할 수 있도록 변수에 저장합니다.
