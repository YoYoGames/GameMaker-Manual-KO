ds_map_find_previous

이 함수는 함수에 지정된 키 이전 에 DS 맵에 저장된 이전 키를 반환합니다. 이것은 무언가를 찾기 위해 DS 맵을 반복해야 하는 경우에 유용할 수 있지만 느릴 수 있으므로 가능한 한 피해야 합니다. 그러한 키가 없으면 함수는 다음을 반환합니다. undefined. 항상 다음을 사용하여 확인해야 합니다.is_undefined() 함수.

통사론:

ds_map_find_previous(id, key);

논쟁 유형 설명
id The id of the map to use.
key The key to find the previous one to.

보고:

Variable or undefined

예시:

var size = ds_map_size(inventory) - 1;
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--;)
{
    if (key != "gold")
    {
        key = ds_map_find_previous(inventory, key);
    }
    else break;
}

위의 코드는 임시 변수를 생성한 다음 ds_map 크기를 가져오고 지도에서 컴퓨터가 저장한 마지막 키를 찾습니다. 그런 다음 for 루프를 사용하여 키 값 "gold"를 찾는 ds_map을 다시 반복합니다. 찾으면 루프를 해제합니다.
