ds_map_secure_save

이 기능은 주어진 DS 맵의 내용을 생성된 장치에 연결된 파일에 저장합니다(다른 장치로 전송하면 읽을 수 없음을 의미함). 파일 자체는 거의 모든 확장자를 가질 수 있습니다(예: *.dat, *.json, *.bin 등...) 난독화되어 대상 플랫폼의 로컬 저장소에 저장됩니다. 그런 다음 함수를 사용하여 ds_map을 다시 로드할 수 있습니다.ds_map_secure_load(). 저장 중인 DS 맵에 배열이 포함된 경우 이 배열은 저장할 때 대신 DS 목록으로 변환됩니다.

중요한! 보안 저장 파일의 기능 중 하나는 파일이 생성된 장치에 잠겨 있기 때문에 한 장치에 저장된 파일을 다른 장치에서 실행 중인 프로젝트에 로드할 수 없다는 것입니다.

통사론:

ds_map_secure_save(map, filename);

논쟁 유형 설명
map The id of the data structure to use
filename The name of the file to save the map to

보고:

예시:

ds_map_secure_save(purchase_map, "p_data.dat");

위의 코드는 나중에 검색할 수 있도록 "p_data" 변수에 인덱싱된 DS 맵을 지정된 파일에 저장합니다.
