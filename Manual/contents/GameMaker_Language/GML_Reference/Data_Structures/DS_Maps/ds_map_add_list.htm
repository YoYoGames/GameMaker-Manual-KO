ds_map_add_list

이 기능을 사용하면 (이전에 생성된) DS 목록 을 주어진 DS 맵 내의 키에 할당할 수 있습니다. 이 기능은 다음을 사용하여 인코딩할 JSON 호환 맵을 생성하도록 설계되었습니다.json_encode() 해당 기능과 함께 사용해야 합니다. DS 맵에 이러한 방식으로 추가된 목록이 있는 경우 상위 맵을 파괴하면 포함된 목록도 파괴되고 메모리가 해제되며ds_map_clear() 상위 맵에서 플래그가 지정된 목록도 삭제하고 정리합니다.

통사론:

ds_map_add_list(id, key, value)

논쟁 유형 설명
id DS 맵 ID The id of the map to use.
key 끈 The key for the added list.
value DS 목록 ID The id of the list to add.

보고:

N/A

예시:

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
var j_map = ds_map_create();
ds_map_add_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_map_destroy(j_map);

위의 코드는 목록을 만들고 전역 변수의 다양한 값으로 채웁니다. 그런 다음 이 목록은 DS 맵 내에 "중첩"된 다음 맵이 삭제되기 전에 JSON 문자열로 인코딩되어 맵과 여기에 포함된 모든 목록을 메모리에서 제거합니다.
