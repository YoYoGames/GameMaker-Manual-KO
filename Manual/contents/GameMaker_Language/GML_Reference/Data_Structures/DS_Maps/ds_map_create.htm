ds_map_create

이 함수는 비어 있는 새 DS 맵을 생성하는 데 사용되며 다른 모든 DS 맵 함수의 데이터 구조에 액세스하는 데 사용되는 해당 ID를 반환합니다.

중요한! 데이터 구조를 생성할 때 식별하기 위한 인덱스 값은 0부터 시작하는 정수 값입니다. 이는 다른 유형의 데이터 구조가 동일한 인덱스 값을 가질 수 있음을 의미하므로 의심스러운 경우 다음을 사용해야 합니다.ds_exists() 액세스하기 전에 기능. 또한 인덱스는 재사용되므로 파괴된 데이터 구조 인덱스 값은 나중에 새로 생성된 값에서 사용될 수 있습니다.

통사론:

ds_map_create();

보고:

DS Map ID

예시:

inventory = ds_map_create();

위의 코드는 비어 있는 새 DS 맵을 만들고 id 인덱스를 "inventory" 변수에 저장합니다.
