ds_map_secure_load_buffer

이 기능은 버퍼에서 안전하게 저장된 DS 맵을 로드합니다. 이전에 버퍼를 메모리에 로드했어야 합니다(사용buffer_load() ) 그런 다음 이 함수에 전달하면 버퍼 내용으로 채워진 DS 맵이 반환됩니다. 함수를 사용하여 버퍼를 생성해야 합니다. ds_map_secure_save_buffer() 올바르게 작동하려면 로드되는 DS 맵에 어레이가 포함된 경우 로드할 때 대신 DS 목록으로 변환됩니다.

중요한! 보안 저장 파일의 기능 중 하나는 파일이 생성된 장치에 잠겨 있기 때문에 한 장치에 저장된 파일을 다른 장치에서 실행 중인 프로젝트에 로드할 수 없다는 것입니다.

참고: 이 기능은 HTML5 및 UWP에서 지원되지 않습니다.

통사론:

ds_map_secure_load_buffer(buffer);

논쟁 유형 설명
buffer 버퍼 ID The buffer ID of the buffer to load the map data from.

보고:

DS Map ID

예시:

var buff = buffer_load("save.dat");
map = ds_map_secure_load_buffer(buff);
buffer_delete(buff);

위의 코드는 버퍼에서 안전하게 저장된 DS 맵을 로드하고 나중에 사용할 수 있도록 인덱스 값을 변수에 저장합니다.
