ds_map_add_map

이 기능을 사용하면 (이전에 생성된) DS 맵을 주어진 DS 맵 내의 키에 할당할 수 있습니다. 이 기능은 다음을 사용하여 인코딩할 JSON 호환 맵을 생성하도록 설계되었습니다.json_encode() 해당 기능과 함께 사용해야 합니다. DS 맵에 이러한 방식으로 추가된 다른 맵이 있는 경우 상위 맵을 파괴하면 포함된 맵도 파괴되고 메모리가 해제되며 다음을 호출합니다.ds_map_clear() 상위 맵에서 플래그가 지정된 맵도 파괴하고 정리합니다.

통사론:

ds_map_add_map(id, key, value)

논쟁 유형 설명
id DS 맵 ID The id of the map to use.
key 끈 The key for the added map.
value DS 맵 ID The id of the map to add.

보고:

N/A

예시:

var j_map = ds_map_create();
var j_list = ds_list_create();
var sub_map = ds_map_create();
ds_map_add_list(sub_map, "list", j_list);
ds_map_add(sub_map, "array", j_array);
ds_map_add_map(j_map, "map", sub_map);
var j = json_encode(j_map);
ds_map_destroy(j_map);

위의 코드는 두 개의 DS 맵을 만든 다음 두 번째 맵에 추가하기 전에 목록과 배열로 하나를 채우고 JSON 문자열로 인코딩합니다. 그런 다음 맵은 메모리에서 제거되고 여기에 포함된 다른 모든 맵이나 목록이 제거됩니다.
