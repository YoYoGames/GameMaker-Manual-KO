ds_map_set

이 기능을 사용하면 주어진 DS 맵 내에서 키 값을 설정할 수 있습니다. DS 맵 ID 값을 제공합니다(함수에서 반환된ds_map_create() ) 그런 다음 설정하려는 키와 설정할 값을 지정합니다. 키는 정수 또는 문자열일 수 있으며, 주어진 키가 존재하지 않으면 자동으로 생성되어 값으로 설정됩니다. 이 기능은 DS 맵 접근 자를 사용하여 맵 키/값 쌍을 설정/생성하는 것과 동일합니다. 함수는 아무 것도 반환하지 않으므로 키 값이 바뀌었거나 새 키가 생성되었는지 확인해야 하는 경우 함수를 사용해야 합니다. ds_map_replace().

통사론:

ds_map_set(id, key, value)

논쟁 유형 설명
id The id of the map to use.
key The key to set.
value The value to set the key to.

보고:

예시:

if is_undefined(ds_map_find_value(map, "score"))
{
    ds_map_set(map, "score", 0);
}

위의 코드는 주어진 키가 존재하는지 확인하고 존재하지 않으면 생성 및 설정됩니다.
