ds_map_clear

이 기능은 모든 키/값 쌍의 (이전에 생성된) DS 맵을 지우지만 메모리에서 DS 맵 자체를 삭제하지는 않습니다. 그것을 위해 당신은 기능을 사용해야합니다ds_map_destroy(). 맵을 지우면 맵에 저장된 모든 데이터 구조가 역참조되어 메모리 누수가 발생하므로 이를 방지하기 위해 맵을 지우기 전에 수동으로 모든 데이터 구조 항목을 삭제해야 합니다. 이것이 필요하지 않은 유일한 경우는 맵의 항목을 DS 목록 또는 다른 DS 맵 으로 플래그 지정했을 때입니다. 이 경우 이러한 항목은 삭제되고(삭제되지 않습니다!) 상위 맵이 지워집니다.

통사론:

ds_map_clear(id);

논쟁 유형 설명
id The id of the data structure to clear.

보고:

예시:

if global.new_game
{
    ds_map_clear(inventory);
}

위의 함수는 전역 변수를 검사하여 그것이 참인지 확인하고, 참이면 인스턴스 변수 "inventory"에 색인된 DS 맵에서 모든 키/값 쌍이 지워집니다(그러나 소멸되지는 않음 ).
