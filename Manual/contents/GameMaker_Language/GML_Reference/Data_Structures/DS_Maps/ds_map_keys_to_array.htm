ds_map_keys_to_array

이 기능을 사용하면 DS 맵에 포함된 모든 키를 검색할 수 있습니다. 키를 가져오기 위해 DS 맵 ID를 제공합니다(ds_map_create() ) 함수는 배열 의 각 항목이 DS 맵의 키인 배열을 반환합니다. 함수에는 생성한 배열을 제공할 수 있는 선택적 두 번째 인수가 있습니다. 이 경우 맵 키 데이터가 배열의 기존 데이터에 추가됩니다. 이 함수는 직접 제공된 배열을 수정하지만 해당 배열에 대한 참조(또는 제공되지 않은 경우 새 배열에 대한 참조)도 반환합니다.

통사론:

ds_map_keys_to_array(id, [array])

논쟁 유형 설명
id DS 맵 ID The id of the map to use.
[array] 정렬 OPTIONAL The array to append the DS map keys to.

보고:

Array

예시:

map_keys = ds_map_keys_to_array(inventory);

위의 코드는 DS 맵에 대한 키를 검색한 다음 나중에 사용할 수 있도록 배열로 저장합니다.
