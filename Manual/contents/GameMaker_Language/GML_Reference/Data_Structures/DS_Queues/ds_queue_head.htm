ds_queue_head

이 함수는 큐의 첫 번째 값("헤드"에 있는 값)만 읽습니다. 값을 큐 에서 빼지 않습니다. 즉, 이 함수 또는ds_queue_dequeue(). 대기열이 비어 있으면 함수는 상수를 반환합니다. undefined, 그렇지 않으면 큐에 포함된 실제 또는 문자열 값을 반환합니다.

통사론:

ds_queue_head(id);

논쟁 유형 설명
id The id of the data structure to read from.

보고:

(Data type value stored in the queue)

예시:

num = ds_queue_head(control_queue);

위의 코드는 "control_queue" 변수에 인덱싱된 큐에서 값을 읽고 반환 값을 변수 "num"에 저장합니다.
