ds_queue_tail

이 함수는 큐의 마지막 값("꼬리 부분"에 있는 값)만 읽습니다. 값을 큐 에서 빼지 않습니다. 즉, 이 함수 또는ds_queue_dequeue().

통사론:

ds_queue_tail(id);

논쟁 유형 설명
id The id of the data structure to read from.

보고:

(Data type value stored in the queue)

예시:

num = ds_queue_tail(control_queue);

위의 코드는 "control_queue" 변수에 인덱싱된 큐에서 값을 읽고 반환 값을 변수 "num"에 저장합니다.
