ds_queue_size

이 함수는 대기열의 "크기"를 반환합니다. 즉, 대기열에 있는 항목의 수입니다.

통사론:

ds_queue_size(id);

논쟁 유형 설명
id The id of the data structure to check.

보고:

예시:

if !ds_queue_empty(control_queue)
{
    num = ds_queue_size(control_queue);
}

위의 코드는 DS queue가 비어 있는지 확인하고 비어 있지 않은 경우 포함된 항목 수를 가져와 변수에 값을 저장합니다.
