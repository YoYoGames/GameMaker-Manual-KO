ds_queue_copy

이 함수는 한 대기열의 내용을 다른 대기열로 복사하는 데 사용할 수 있습니다. 이것은 원래 대기열에서 내용을 제거 하지 않으며 원래 대기열을 파괴하지도 않습니다. 이 기능을 사용할 때 복사 대상 대기열은 이전에 생성되어야 하며 복사 이전에 항목이 포함되어 있으면 이러한 항목이 먼저 지워집니다(이 정보는 손실됨).

통사론:

ds_queue_copy(id, source);

논쟁 유형 설명
id DS 대기열 ID The id of the new queue.
source DS 대기열 ID The original queue to copy from.

보고:

N/A

예시:

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
    queue = ds_queue_create();
    ds_queue_copy(queue, other.queue);
}

위의 함수는 새 인스턴스를 만든 다음 해당 인스턴스에서 새 DS 대기열을 만들고 코드 블록을 실행하는 인스턴스의 대기열 내용을 새로 생성된 인스턴스 대기열에 복사합니다.
