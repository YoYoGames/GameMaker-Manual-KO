ds_queue_dequeue

이 함수는 DS 대기열 에서 헤드 값을 대기열에서 빼내어 대기열에서 제거하고 변수에 저장할 값을 반환합니다. 대기열이 비어 있으면 함수는 상수를 반환합니다. undefined, 그렇지 않으면 큐에 포함된 값을 반환합니다.

통사론:

ds_queue_dequeue(id);

논쟁 유형 설명
id The id of the queue to dequeue from.

보고:

Variable (Data type value stored in the queue) or undefined

예시:

if !ds_queue_empty(move_queue)
{
    var xx = ds_queue_dequeue(move_queue);
    var yy = ds_queue_dequeue(move_queue);
    move_towards_point(xx, yy, 4);
}

위의 코드는 "move_queue" 변수에 인덱싱된 DS queue가 비어 있는지 확인하고 비어 있지 않은 경우 queue의 head에서 두 값을 dequeue하여 이동 방향을 설정하는 데 사용합니다.
