time_source_reconfigure

이 함수는 완전히 새로운 속성을 만들 필요 없이 시간 소스 의 핵심 속성을 수정하는 데 사용됩니다.

기존 시간 소스 를 지정한 다음 에서도 설정되는 속성을 설정합니다.time_source_create(), 부모 제외. 이러한 속성에 대한 자세한 내용은 해당 페이지를 읽으십시오.

지정된 시간 소스 가 재설정되고 비활성화되며 다시 시작 해야 합니다.

통사론:

time_source_reconfigure(id, period, units, callback, [args, repetitions, expiry_type]);

논쟁 유형 설명
id 시간 소스 ID The Time Source to reconfigure
period 진짜 The period that the Time Source runs for, in the given units
units 시간 소스 단위 상수 The units that the given period is in
callback 방법 The method to call when the Time Source expires
args 정렬 OPTIONAL An array containing the arguments to pass into the method
repetitions 진짜 OPTIONAL The number of times the Time Source should repeat, or -1 for indefinite repetition
expiry_type 시간 소스 만료 상수 OPTIONAL Whether the Time Source expires on the frame nearest to its expiry, or on the next frame

보고:

N/A

예시:

function change_spawn_delay(new_delay)
{
    time_source_reconfigure(obj_game.spawn_time_source, new_delay, time_source_units_frames, obj_game.spawn_method, [], -1);
    time_source_start(obj_game.spawn_time_source);
}

이것은 게임 내 적에게 사용되는 스폰 지연을 변경하는 새로운 기능을 생성합니다.

게임이 obj_game.spawn_time_source 라는 시간 소스 를 사용하여 적을 스폰한다고 가정하면 스폰 지연이 변경되면 해당 시간 소스 를 업데이트해야 합니다.

이 기능은 정확히 그 작업을 수행하여 새로운 지연으로 시간 소스 를 재구성한 다음 다시 시작 합니다.
