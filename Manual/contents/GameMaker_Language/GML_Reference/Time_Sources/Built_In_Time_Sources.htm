내장 시간 소스

시간 소스 상수
끊임없는 설명 값
time_source_global 글로벌 시간 소스 0
time_source_game 게임 시간 소스 1

글로벌 시간 소스 또는 게임 시간 소스 를 사용자 지정 시간 소스 의 부모로 사용할 수 있습니다. 두 시간 소스 모두 전 세계적으로 사용 가능합니다.

글로벌 시간 소스 는 메인 게임 루프 외부에서 실행되는 반면 게임 시간 소스 는 게임 루프의 일부로 실행됩니다.

Global/Game 사이의 선택은 Time Source 가 게임의 프레임 속도에 영향을 받는지 여부에 영향을 미치지 않습니다. 사용하는 단위 에 따라 다릅니다.

주문하다

Global Time Source 에서 상속받은 Time Source는 Game Time Source 보다 먼저 처리됩니다.

주

게임 시간 소스 에는 paused 될 수 있는 상태 가 있습니다. 글로벌 시간 소스 에는 상태가 없으며 일시 중지할 수 없습니다.

게임 상태와 상관없이 실행되어야 하는 타이머에는 글로벌 시간 소스 를 사용하고, 메인 게임 루프에 연결된 타이머에는 게임 시간 소스 를 사용할 수 있습니다. 그런 다음 게임 시간 소스 를 일시 중지하여 모든 게임 관련 타이머를 일시 중지할 수 있습니다.
