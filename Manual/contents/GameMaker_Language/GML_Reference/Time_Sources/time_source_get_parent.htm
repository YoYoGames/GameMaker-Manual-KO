time_source_get_parent

이 함수는 주어진 시간 소스 의 부모의 시간 소스 ID 를 반환합니다.

반환된 값은 지정된 시간 소스 가 생성 된 방식에 따라 사용자 지정 시간 소스 또는 기본 제공 시간 소스 중 하나의 ID일 수 있습니다.

통사론:

time_source_get_parent(id);

논쟁 유형 설명
id 시간 소스 ID The Time Source to get the parent of

보고:

Time Source ID

예시:

var _parent = time_source_get_parent(time_source);

이 코드는 시간 소스 의 부모를 가져와서 로컬 변수에 저장합니다.
