time_source_reset

이 함수는 주어진 시간 소스 를 재설정하고 타이머와 반복 정보를 재설정합니다.

이 함수를 호출한 후 카운트다운하려면 시간 소스 를 다시 시작 해야 합니다.

통사론:

time_source_reset(id);

논쟁 유형 설명
id 시간 소스 ID The Time Source to reset

보고:

N/A

예시:

// Room Start event
time_source_reset(global.spawn_time_source);
time_source_start(global.spawn_time_source);

이 코드는 새 방이 시작될 때마다 주어진 시간 소스 를 재설정한 다음 시간 소스 를 다시 시작합니다.

뒤로: 시간 소스
다음: time_source_exists
