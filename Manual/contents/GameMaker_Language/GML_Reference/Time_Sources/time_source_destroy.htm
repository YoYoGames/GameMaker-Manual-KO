time_source_destroy

이 함수는 주어진 시간 소스 를 파괴합니다.

선택적으로 모든 하위 시간 소스도 삭제해야 하는지 여부를 지정할 수 있습니다.

통사론:

time_source_destroy(id, [destroy_tree]);

논쟁 유형 설명
id 시간 소스 ID The Time Source to destroy
destroy_tree 부울 OPTIONAL Whether to destroy child Time Sources as well, is false by default

보고:

N/A

예시:

// Create event
time_source = time_source_create(time_source_game, 1, time_source_units_seconds, my_method, [], -1);

// Clean Up event
time_source_destroy(time_source);

이벤트 생성 코드는 초당 한 번 만료되고 무기한 반복되는 새 시간 소스 를 생성합니다.

Clean Up 이벤트 코드는 인스턴스가 소멸된 후에는 필요하지 않으므로 해당 Time Source 를 소멸시킵니다.
