time_source_get_units

이 함수는 주어진 시간 소스 에 대해 설정된 단위 를 반환합니다. 시간 소스 의 기간 값은 이러한 단위로 표시됩니다.

통사론:

time_source_get_units(id);

논쟁 유형 설명
id 시간 소스 ID The Time Source to get the units for
