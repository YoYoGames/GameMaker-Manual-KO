time_source_exists

이 함수는 주어진 ID가 유효한 시간 소스 인지 여부를 반환합니다.

Time Source 가 소멸된 경우 false 를 반환하지만 Time Source 가 중지 되었거나 만료된 경우 계속 존재하므로 이 함수는 true 를 반환합니다.

통사론:

time_source_exists(id);

논쟁 유형 설명
id 시간 소스 ID The Time Source to check

보고:

Boolean

예시:

if (time_source_exists(global.spawn_time_source))
{
    if (keyboard_check_pressed(vk_space))
    {
        time_source_destroy(global.spawn_time_source);
    }
}

이 코드는 시간 소스 가 존재하는지 확인합니다. 그렇다면 Space 키를 눌렀는지 확인합니다.

그 경우, 타임 소스 를 파괴합니다.
