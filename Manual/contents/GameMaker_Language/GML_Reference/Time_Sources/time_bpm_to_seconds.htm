time_bpm_to_seconds

이 함수는 분당 비트 수 값을 취하고 각 비트의 길이를 초 단위로 반환합니다.

이것은 시간 소스에 대한 BPM 값을 사용하기 위해 시간 소스를 생성 할 때 사용할 수 있습니다. 이러한 시간 소스 는 초 를 단위 로 사용하는 것이 중요합니다.

통사론:

time_bpm_to_seconds(bpm);

논쟁 유형 설명
bpm 진짜 The beats-per-minute value to convert into seconds-per-beat

보고:

Real

예시:

var _bpm = 90;
var _seconds = time_bpm_to_seconds(_bpm);

time_source = time_source_create(time_source_game, _seconds, time_source_units_seconds, function()
{
    show_debug_message("BEAT!");
}, [], -1);

이 코드는 90BPM 값을 초로 변환하고 이를 사용하여 무기한 실행되는 시간 소스 를 만듭니다.

각 비트에서 "BEAT!" 메시지가 인쇄됩니다. 출력 로그에.
