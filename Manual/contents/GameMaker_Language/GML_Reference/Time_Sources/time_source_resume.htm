time_source_resume

이 함수는 주어진 시간 소스 와 그 자식을 다시 시작합니다. 주어진 시간 소스 는 이전에 다음을 사용하여 일시 중지되어야 합니다.time_source_pause().

통사론:

time_source_resume(id);

논쟁 유형 설명
id 시간 소스 ID The Time Source to resume

보고:

N/A

예시:

if (keyboard_check_pressed(vk_space))
{
    var _state = time_source_get_state(time_source);

    if (_state == time_source_state_active)
    {
        time_source_pause(time_source);
    }
    else if (_state == time_source_state_paused)
    {
        time_source_resume(time_source);
    }
}

Space 키를 누르면 이 코드는 Time Source 의 상태를 얻습니다.

상태가 활성화되면 시간 소스 를 일시 중지하고 일시 중지되면 다시 시작합니다.
