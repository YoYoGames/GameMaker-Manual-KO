time_source_start

이 함수는 주어진 시간 소스 를 시작하고 상태 를 instance_destroy()>time_source_state_active 로 변경합니다.

지정된 시간 소스 는 시작되지 않았거나, 일시 중지되었거나, 중지되었거나, 반복 없이 만료된 새 시간 소스일 수 있습니다.

이 기능은 주어진 시간 소스 를 "소프트 리셋"하여 만료 시간 과 남은 횟수 를 구성된 값으로 재설정합니다.

통사론:

time_source_start(id);

논쟁 유형 설명
id 시간 소스 ID The Time Source to start

보고:

N/A

예시:

var _my_method = function()
{
    instance_destroy();
}

var _time_source = time_source_create(time_source_game, 300, time_source_units_frames, _my_method);

time_source_start(_time_source);

이 예에서는 인스턴스가 300프레임 후에 자체적으로 파괴되기를 원합니다.

코드는 먼저 단순히 호출하는 메서드를 만듭니다..

그런 다음 Game Time Source 에서 상속한 Time Source 를 만듭니다. 기간을 300프레임 으로 설정합니다.

마지막으로 Time Source 를 시작합니다.
