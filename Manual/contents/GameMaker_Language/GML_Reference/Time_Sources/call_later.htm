call_later

이 함수는 지정된 시간 이후 또는 특정 간격 이후에 콜백 함수를 트리거합니다. 시간 소스를 생성하여 이를 수행합니다( 시간 소스 참조 ).

콜백 기능을 취소하려면 call_cancel 을 참조하십시오.

중요 이 함수는 실제 시간 소스 ID를 반환하지만 이 ID는 시간 소스 함수와 함께 사용할 수 없습니다. call_later 및 call_cancel 에서 사용하는 시간 소스는 "보이지 않음"입니다. 내장 시간 소스의 자식으로 나타나지 않으며 다른 시간 소스 기능과 함께 사용할 수 없습니다. 그러나 debug_event("ResourceCou nt").

인수

다음은 이 함수의 인수를 분석한 것입니다.

기간

이것은 다음 인수에 지정된 단위에 따라 초 또는 프레임으로 표시될 수 있는 기간입니다.

프레임을 단위로 사용할 때 마침표는 정수여야 합니다. 정수가 아닌 값은 1로 반올림되는 1보다 작은 값을 제외하고는 내림(내림)됩니다.

단위

이는 주기를 나타내는 단위로 time_bpm_to_seconds()>time_source_units_seconds 또는 time_source_units_frames 일 수 있다.

참조: 시간 출처 단위

다음을 사용하여 분당 비트 수 값을 사용할 수 있습니다..

콜백

시간 소스 가 만료될 때 호출되는 "콜백" method 를 지정해야 합니다.

고리

시간 소스가 반복되어야 하는지 여부입니다.

통사론:

call_later(period, unit, callback, [loop=false]);

논쟁 유형 설명
period 진짜 The period that the Time Source runs for, in the given units
units 시간 소스 단위 상수 The units that the given period is in
callback 메소드 또는 스크립트 기능 The method or script function to call after the specified time
loop 부울 OPTIONAL Whether to loop the Time Source (default is false )

보고:

Time Source ID

예 1:

var _callback = function()
{
    show_debug_message("Five seconds have passed!");
}
var _handle = call_later(5, time_source_units_seconds, _callback);

이 예에서는 5초가 지난 후 디버그 메시지를 표시하는 콜백 함수를 만듭니다. 그 후, 기본 시간 소스 가 파괴됩니다.

예 2:

var _create_enemy = function()
{
    var _x = random(room_width);
    var _y = random(room_height);
    instance_create_depth(_x, _y, depth, obj_enemy);
}

var _handle = call_later(10, time_source_units_seconds, _create_enemy, true);

이 예에서는 방의 임의 위치에 새로운 적을 생성하는 콜백 함수를 만듭니다. 그런 다음 이 함수는 call_later 에 전달하여 10초마다 실행됩니다. 마지막 인수는 true 로 설정되어 함수가 10초마다 계속 호출됩니다.

call_later 에서 반환된 핸들과 함께 call_cancel 을 사용하여 시간 소스 와 콜백 호출을 중지합니다.
