시간 소스 단위 상수
끊임없는 설명 값
time_source_units_seconds 시간 소스 기간에 초 사용(프레임 독립적) 0
time_source_units_frames 시간 소스 기간 동안 프레임 사용(프레임 종속) 1

이러한 상수는 다음에서 사용됩니다.time_source_create() 그리고time_source_reconfigure() 시간 소스 의 기간에 사용되는 단위를 설정하고 다음에서 반환됩니다.time_source_get_units().

초를 사용하는 경우 시간 소스 는 게임의 프레임 속도와 독립적으로 실행됩니다.

프레임을 사용하는 경우 시간 소스 는 게임의 프레임 속도에 따라 달라집니다.

다음을 호출하여 BPM(분당 비트 수)을 단위로 사용할 수도 있습니다.time_bpm_to_seconds() BPM 값을 초 단위 기간으로 변환한 다음time_source_units_seconds 단위로.
