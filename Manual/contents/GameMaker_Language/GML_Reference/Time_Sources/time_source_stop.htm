time_source_stop

이 기능은 주어진 시간 소스 를 중지하고 타이머를 재설정합니다. 시작되지 않았거나 완료된 상태인 시간 소스 는 중지할 수 없습니다.

통사론:

time_source_stop(id);

논쟁 유형 설명
id 시간 소스 ID The Time Source to stop

보고:

N/A

예시:

if (time_source_get_state(time_source) != time_source_state_stopped)
{
    time_source_stop(time_source);
}

위의 코드는 아직 중지되지 않은 경우 시간 소스 를 중지합니다.
