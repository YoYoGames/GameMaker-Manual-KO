time_source_get_period

이 함수는 주어진 시간 소스 에 대해 설정된 기간 값을 반환합니다.

이 기간 값은 시간 소스 에 대해 설정된 단위 (초 또는 프레임)로 표시됩니다.

통사론:

time_source_get_period(id);

논쟁 유형 설명
id 시간 소스 ID The Time Source to get the period value for
