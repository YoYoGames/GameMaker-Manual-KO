time_seconds_to_bpm

이 함수는 비트의 길이를 초 단위로 취하고 분당 비트 값을 반환합니다.

통사론:

time_seconds_to_bpm(seconds);

논쟁 유형 설명
seconds 진짜 The seconds-per-beat value to convert into beats-per-minute

보고:

Real

예시:

var _period = time_source_get_period(time_source);
var _bpm = time_seconds_to_bpm(_period);

show_debug_message(_bpm);

이 함수는 시간 소스 의 기간을 가져와 BPM으로 변환하고 해당 값을 출력 로그에 인쇄합니다.

이 코드는 다음 예제를 따릅니다.time_bpm_to_seconds().
