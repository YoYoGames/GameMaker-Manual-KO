시간 소스 만료 유형

시간 소스 만료 상수
끊임없는 설명 값
time_source_expire_nearest 시간 소스 는 만료 시간에 가장 가까운 프레임에서 만료됩니다. 0
time_source_expire_after 시간 소스 는 만료 시간 후 첫 번째 프레임에서 만료됩니다. 1

초를 단위 로 사용하는 시간 소스는 게임 프레임에서 항상 만료되지 않을 수 있습니다. 게임 프레임이 처리된 후 새 프레임이 처리를 시작하기 전에 만료될 수 있습니다.

그러나 시간 소스 에 대한 콜백은 게임 프레임에서만 실행할 수 있으므로 두 프레임 중간에 만료되는 경우 콜백이 실행되는 시간을 시간 소스 의 만료 유형이 제어합니다.

time_source_expire_nearest 를 사용할 때 시간 소스 에 대한 콜백은 만료 시간에 가장 가까운 프레임에서 실행됩니다. 만료 시간은 이전 프레임 또는 이후 프레임일 수 있습니다.

time_source_expire_after 를 사용할 때 시간 소스 에 대한 콜백은 만료 시간 이후 에 항상 프레임에서 실행됩니다.
