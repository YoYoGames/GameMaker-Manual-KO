시간 소스

" 시간 소스 "는 사용자가 만든 사용자 지정 타이머입니다. 주어진 시간 동안 실행되고 마지막에 만료됩니다.

시간 소스 (TS)에는 만료 시 사용자 지정 메서드("콜백")를 호출할 수 있는 기능이 있습니다.

또한 일정 횟수 또는 영원히 반복할 수 있습니다. 시간 소스 의 각 반복은 "반복"으로 알려져 있습니다.

시간 소스를 사용하는 방법?

사용자 지정 메서드를 호출하는 시간 소스 를 만들고 시작합니다.

  • 부르다time_source_create() / 시간 소스 를 생성하여 시간 소스 생성
    • 이것이 글로벌 시간 소스, 게임 시간 소스 또는 사용자 정의 시간 소스 에서 상속되는지 여부를 지정합니다.
    • 실행 기간 및 사용 단위(초 또는 프레임)를 지정합니다.
    • 해당 메소드에 전달할 인수 배열을 사용하여 사용자 정의 메소드 또는 스크립트 함수를 지정하십시오.
    • 선택적으로 반복해야 하는 횟수를 지정합니다.
      • 기본적으로 한 번만 실행됩니다.
  • 부르다time_source_start() / 시작 시간 소스 를 사용하여 해당 시간 소스 를 즉시 시작하거나 나중에 시작해야 할 때 호출합니다.

이렇게 하면 시간 소스 가 실행되고 지정된 기간이 지나면 만료되고 지정된 메서드를 호출합니다.

내장 시간 소스

시간 소스 는 두 개의 기본 제공 시간 소스 또는 사용자 지정 시간 소스 중 하나에서 상속할 수 있습니다.

간단한 타이머

시간 소스를 설정하는 데 많은 코드가 필요하지 않지만 단일 기능을 사용하여 타이머를 빠르게 설정하고 관리하는 것이 편리할 수 있습니다.

다음 두 함수는 시간 소스 를 만들고 중지하는 더 간단하고 간단한 방법을 제공합니다.

  • call_later
  • call_cancel

참고 이러한 기능에서 사용하는 시간 소스 ID와 함께 시간 소스 기능을 사용할 수 없습니다. 자세한 내용은 call_later 를 참조하세요.

기능 목록

다음 함수는 시간 소스 를 만들고 수정하는 데 사용됩니다.

  • time_source_create
  • time_source_destroy
  • time_source_start
  • time_source_stop
  • time_source_pause
  • time_source_resume
  • time_source_reconfigure
  • time_source_reset
  • time_source_get_children
  • time_source_get_parent
  • time_source_get_period
  • time_source_get_reps_completed
  • time_source_get_reps_remaining
  • time_source_get_state
  • time_source_get_time_remaining
  • time_source_get_units
  • time_source_exists

지정된 시간 소스 가 존재하지 않으면 각 get_* 함수는 undefined 를 반환합니다.

기간 변환을 위해 다음 도우미 함수가 제공됩니다.

  • time_seconds_to_bpm
  • time_bpm_to_seconds
