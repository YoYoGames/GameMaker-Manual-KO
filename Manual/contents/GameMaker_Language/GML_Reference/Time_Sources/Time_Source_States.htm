시간 소스 상태 상수
끊임없는 설명 값
time_source_state_initial 시간 소스 가 아직 시작되지 않았습니다. 0
time_source_state_active 시간 소스 가 시작 되었으며 카운트다운 중입니다. 1
time_source_state_paused 시간 소스 가 paused. 2
time_source_state_stopped 시간 소스 가 중지 되었거나 완전히 만료되었습니다. 3

시간 소스 는 위의 상수 중 하나일 수 있는 상태를 가질 수 있습니다. 새로 생성된 시간 소스 의 기본 상태는 time_source_start()>time_source_state_initial 입니다.

상태는 다음을 사용하여 변경할 수 있습니다.time_source_stop(),time_source_pause(), 그리고time_source_get_state().
