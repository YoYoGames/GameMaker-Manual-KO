mp_grid_clear_rectangle

이 기능을 사용하면 지정된 MP 그리드에서 해당 셀을 지울 공간 좌표의 영역을 정의할 수 있습니다. 셀이 정의된 직사각형 영역에 부분적으로 포함되더라도 지워집니다.

통사론:

mp_grid_clear_rectangle(id, x1, y1, x2, y2);

논쟁 유형 설명
id MP 그리드 ID Index of the mp_grid that is to be used
x1 진짜 The x coordinate of the left side of the rectangle to check.
y1 진짜 The y coordinate of the top side of the rectangle to check.
x2 진짜 The x coordinate of the right side of the rectangle to check.
y2 진짜 The x coordinate of the bottom side of the rectangle to check.

보고:

Boolean

예시:

mp_grid_clear_rectangle(grid, 0, 0, 100, 200);

위의 코드는 0,0에서 100,200 영역에 속하는 변수 "grid"에 색인된 mp_grid의 모든 셀을 사용 가능한 것으로 표시합니다.
