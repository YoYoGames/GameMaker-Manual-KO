mp_grid_destroy

이 함수를 사용하여 표시된 mp_grid를 파괴할 수 있습니다(함수에 의해 정의됨mp_grid_create() ) 할당된 메모리를 해제합니다. MP 그리드가 끝날 때마다 이것을 호출하는 것이 중요 합니다. 그렇지 않으면 잠재적으로 메모리 누수가 발생할 수 있습니다. 즉, 시간이 지남에 따라 게임이 느려지고 결국에는 충돌합니다.

참고: 사용 mp_grid_* 그리드의 함수는 파괴된 후 오류가 발생합니다.

통사론:

mp_grid_destroy(id);

논쟁 유형 설명
id MP 그리드 ID Index of the mp_grid that is to be destroyed

보고:

N/A

예시:

if timer = 0
{
    mp_grid_destroy(grid);
    room_goto(rm_Menu);
}

위의 코드는 "timer" 변수가 0이면 "grid" 변수에 색인된 mp_grid를 파괴하고 다른 방으로 이동합니다.
