mp_potential_step

이 기능을 사용하면 인스턴스가 xgoal/ygoal에 의해 정의된 특정 위치를 향해 한 걸음 더 나아가면서 장애물을 피하려고 할 수 있습니다. 인스턴스가 솔리드 인스턴스(또는 checkall이 true인 경우 임의의 인스턴스)와 마주치면 해당 인스턴스를 피하고 주위를 이동하기 위해 모션 방향을 변경합니다. 이 접근 방식은 작동이 보장되지 않지만 대부분의 쉬운 경우 인스턴스를 목표로 효과적으로 이동합니다. 함수는 목표에 도달했는지 여부를 반환합니다.

통사론:

mp_potential_step(xgoal, ygoal, stepsize, checkall)

논쟁 유형 설명
xgoal The target x position.
ygoal The target y position.
stepsize The speed the instance moves in pixels per step.
checkall Whether to check all instances (true) or just solid ones (false).

보고:

예시:

if instance_exists(obj_Enemy)
{
    var inst;
    inst = instance_nearest(x, y, obj_Enemy);
    mp_potential_step(inst.x, inst.y, 5, false);
}

위의 코드는 먼저 방에 "obj_Enemy" 인스턴스가 있는지 확인합니다. 존재하는 경우 가장 가까운 것을 찾아 변수에 ID를 저장합니다. 이 변수는 다음을 알려주는 데 사용됩니다. mp_potential_step 단색으로 표시된 인스턴스만 피하면서 단계당 5픽셀의 속도로 발견된 개체의 x/y 위치로 코드가 있는 인스턴스를 이동합니다.
