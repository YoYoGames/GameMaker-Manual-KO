mp_linear_step

이 기능을 사용하면 인스턴스에 xgoal 및 ygoal 값으로 지정된 특정 지점을 향해 "단계"를 취하도록 지시합니다. 단계의 크기(인스턴스가 각 단계를 이동해야 하는 픽셀 수)는 단계 크기로 표시되며, 인스턴스가 이미 해당 위치에 있으면 중지되고 더 이상 이동하지 않습니다.move_towards_point(). 단계 크기는 개체가 각 단계에서 충돌을 확인하는 "앞"의 거리이기도 하며 인스턴스가 임의의 인스턴스와 충돌할 때 중지할지 아니면 솔리드로 플래그가 지정된 인스턴스만 중지할지 선택할 수 있습니다. 함수는 목표 위치에 도달했는지(true) 아니면 실패했는지(false) 반환합니다.

참고 이 기능은 장애물을 만나면 우회를 시도하지 않고 단순히 실패하고 이동을 멈춥니다.

통사론:

mp_linear_step(xgoal, ygoal, stepsize, checkall);

논쟁 유형 설명
xgoal The target x position.
ygoal The target y position.
stepsize The speed the instance moves in pixels per step.
checkall Whether to check all instances (true) or just solid ones (false).

보고:

예시:

if mp_linear_step(mouse_x, mouse_y, 5, 0)
{
    instance_create_layer(x, y, "Effects", obj_explosion);
    instance_destroy();
}

위의 코드는 개체가 단계당 5픽셀의 속도로 마우스를 향해 이동하도록 합니다. 마우스 위치에 도달하면 "obj_Explosion" 개체를 만들고 스스로를 파괴합니다.
