mp_grid_draw

이 기능은 지정된 MP 그리드를 그립니다(mp_grid_create() ), 자유 셀을 녹색으로 표시하고 금지된 셀을 빨간색으로 표시합니다. 이 기능은 디버그 도구로 필수적이지만 매우 느리고 인스턴스의 그리기 이벤트에서 사용될 때만 작동하며 그리기 알파를 설정하여 그리드의 불투명도를 변경할 수 있으므로 다음을 수행할 수 있습니다. 오버레이로 그리고 실제로 방에 있는 것을 동시에 볼 수 있습니다.

통사론:

mp_grid_draw(id);

논쟁 유형 설명
id MP 그리드 ID Index of the mp_grid that is to be drawn

보고:

Boolean

예시:

draw_set_alpha(0.3);
mp_grid_draw(grid);
draw_set_alpha(1);

위의 코드는 "grid" 변수에 인덱싱된 mp_grid를 반투명 오버레이로 그립니다(그러나 코드를 실행하는 인스턴스가 나머지 모든 것보다 깊이가 낮은 경우에만).
