mp_grid_get_cell

이 기능을 사용하면 mp_grid의 지정된 셀을 확인하여 점유된 것으로 플래그가 지정되었는지 확인할 수 있으며, (이전에 생성된) mp_grid의 인덱스와 확인할 셀의 x 및 y 좌표를 제공합니다.

점유되었거나 확인 중인 위치가 그리드의 범위를 벗어나면 함수는 -1을 반환하고, 그렇지 않으면 0을 반환합니다.

통사론:

mp_grid_get_cell(id, x , y);

논쟁 유형 설명
id MP 그리드 ID Index of the mp_grid that is to be used
x1 진짜 The x coordinate of the grid to check.
y1 진짜 The y coordinate of the grid to check.

보고:

Real

예시:

if mp_grid_get_cell(grid, mouse_x div 16, mouse_y div 16) == -1
{
    image_blend = c_red;
}
else
{
    image_blend = c_lime;
}

위의 코드는 마우스 위치에 해당하는 mp_grid 셀을 확인하고 점유되어 있으면 image_blend 변수를 빨간색으로 변경하고, 비어 있으면 녹색으로 설정합니다.
