mp_linear_step_object

이 기능을 사용하면 인스턴스에 xgoal 및 ygoal 값으로 지정된 특정 지점을 향해 "단계"를 취하도록 지시합니다. 단계의 크기(인스턴스가 각 단계를 이동해야 하는 픽셀 수)는 단계 크기로 표시되며, 인스턴스가 이미 해당 위치에 있으면 중지되고 더 이상 이동하지 않습니다.move_towards_point(). 단계 크기는 개체가 각 단계에서 충돌을 확인하는 "앞"의 거리이기도 하며 인스턴스가 개체 또는 선택한 인스턴스와의 충돌에서 멈출지 여부를 선택할 수 있습니다.

통사론:

mp_linear_step_object(xgoal, ygoal, stepsize, obj);

논쟁 유형 설명
xgoal 진짜 The target x position.
ygoal 진짜 The target y position.
stepsize 진짜 The speed the instance moves in pixels per step.
obj 개체 자산 또는 인스턴스 ID The object that is to block the path. Can be an object index, an instance id or the special keyword, all

보고:

Boolean

예시:

if mp_linear_step_object(mouse_x, mouse_y, 5, obj_Wall)
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

위의 코드는 개체가 "obj_Wall" 개체와의 충돌만 확인하면서 단계당 5픽셀의 속도로 마우스를 향해 이동하도록 합니다. 마우스 위치에 도달하면 "obj_Explosion" 개체를 만들고 스스로를 파괴합니다.
