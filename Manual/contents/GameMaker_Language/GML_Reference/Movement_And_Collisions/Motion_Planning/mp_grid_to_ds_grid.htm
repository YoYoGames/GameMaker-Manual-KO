mp_grid_to_ds_grid

이 기능을 사용하면 주어진 MP 그리드를 DS 그리드 로 복사할 수 있습니다. DS 그리드는 MP 그리드와 크기가 같아야 하지만 반드시 그럴 필요는 없습니다(더 작으면 데이터가 손실되고 더 크면 모든 추가 그리드 셀은 0이 됨). DS 그리드의 셀은 MP 그리드에서 점유된 것으로 플래그가 지정된 경우 -1 값을 포함하고 그렇지 않은 경우 0을 포함합니다.

통사론:

mp_grid_to_ds_grid(source, destination);

논쟁 유형 설명
source MP 그리드 ID Index of the mp_grid that is to be used
destination DS 그리드 ID Index of the ds_grid that is to be used to copy the grid data to.

보고:

Boolean

예시:

motion_grid = ds_grid_create(room_width / 32, room_height / 32);
mp_grid_to_ds_grid(mp_grid, motion_grid);

위의 코드는 새 DS 그리드를 만든 다음 변수 "mp_grid"에 포함된 MP 그리드 데이터를 새 DS 그리드로 복사합니다.
