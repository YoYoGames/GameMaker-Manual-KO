mp_grid_clear_cell

이 기능은 MP 그리드의 특정 "셀"을 지우는 데 사용할 수 있습니다. 셀은 방 좌표로 계산 되지 않고 그리드 좌표로 계산됩니다. 여기서 (0,0)은 그리드의 위쪽 모서리입니다. 이것은 방의 특정 위치에서 셀을 지우려면 x 및 y 좌표를 MP 그리드의 해상도로 나누는 셀 좌표로 변경해야 함을 의미합니다. 아래 코드 예제는 이것이 어떻게 작동하는지 보여줍니다.

통사론:

mp_grid_clear_cell(id, h, v);

논쟁 유형 설명
id MP 그리드 ID Index of the mp_grid that is to be used
h 진짜 Horizontal position of the cell to clear
v 진짜 vertical position of the cell to clear

보고:

Boolean

예시:

with (obj_Box)
{
    mp_grid_clear_cell(grid, floor(x / 32), floor(y /32));
    instance_destroy();
}

위의 코드는 모든 "obj_Box"가 스스로를 파괴하고 변수 "grid"에 색인된 mp_grid에서 차지한 셀을 비어 있는 것으로 표시하도록 합니다. 이 예에서 우리는 개체의 x/y 좌표를 가져와 격자의 해상도로 나누어 적절한 셀을 찾습니다(사용 floor 값을 정수로 유지).
