mp_linear_path

이 함수는 현재 인스턴스 위치에서 "xgoal" 및 "ygoal" 값으로 지정된 지점까지의 직선 경로를 계산합니다. 함수에서와 같이 표시된 단계 크기를 사용합니다.mp_linear_step(). 표시된 경로는 이미 존재해야 하며 새 경로로 덮어쓰여지며 완전한 경로를 찾았는지(true) 또는 찾지 못한 경우(false) 함수가 반환됩니다. false가 반환되면 경로는 계속 생성되지만 경로가 차단된 위치까지만 실행됩니다.

참고: 이 기능은 인스턴스를 이동하지 않습니다. 경로만 설정하며 이동을 위해서는 Path 기능을 사용해야 합니다.

통사론:

mp_linear_path(path, xgoal, ygoal, stepsize, checkall);

논쟁 유형 설명
path The index of the path to be used.
xgoal The target x position.
ygoal The target y position.
stepsize The speed the instance moves in pixels per step.
checkall Whether to check for collisions with all instances (true) or just solid ones (false).

보고:

예시:

if mp_linear_path(path, obj_Player.x, obj_Player.y, 2, 0)
{
    path_start(path, 2, 0, 0);
}

이것은 객체가 현재 위치에서 "obj_Player" 위치까지 선형 경로가 있는지 확인하고 확인하도록 합니다. 있는 경우 경로를 시작합니다.
