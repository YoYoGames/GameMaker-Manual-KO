mp_grid_clear_all

이 기능을 사용하면 모든 "금지된" 셀의 MP 그리드를 지울 수 있습니다.

통사론:

mp_grid_clear_all(id);

논쟁 유형 설명
id MP 그리드 ID Index of the mp_grid that is to be used

보고:

Boolean

예시:

if !instance_exists(obj_Player)
{
    mp_grid_clear_all(grid);
}

위의 코드는 "obj_Player" 개체의 인스턴스가 더 이상 방에 존재하지 않는 경우 변수 "grid"에 인덱싱된 mp_grid를 지우고 모든 셀을 비어 있는 것으로 표시합니다.
