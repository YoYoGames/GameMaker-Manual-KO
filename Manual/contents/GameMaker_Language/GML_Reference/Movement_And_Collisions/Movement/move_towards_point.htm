move_towards_point

이 기능은 단순히 코드를 실행하는 인스턴스에 설정 속도로 설정 포인트를 향해 이동하도록 지시합니다. 자동으로 멈추지 않기 때문에 일단 도착하면 무엇을 해야 하는지는 당신에게 달려 있다는 점을 명심하십시오. 이것은 (예를 들어) 3픽셀 앞으로 이동하고 x/y 대상 위치를 2만큼 초과한 다음 다시 3픽셀 뒤로 이동하여 대상 지점을 1만큼 초과하는 식으로 "진동" 인스턴스를 얻을 수 있음을 의미합니다. 예를 들어 다음을 사용하여 이를 피할 수 있는 여러 가지 방법이 있습니다. distance_to_point() 인스턴스가 이동하기 전에 지정된 위치에서 얼마나 멀리 떨어져 있는지 확인합니다.

참고: 이 함수는 인스턴스의 속도를 설정하기 때문에 이 코드의 사용을 중지해도 인스턴스는 계속 이전 방향으로 이동하므로 인스턴스 설정이 필요합니다.speed 중지하려면 0으로 설정합니다. 아래 코드 예제를 참조하세요.

통사론:

move_towards_point( x, y, sp );

논쟁 유형 설명
x 진짜 The x position of the point to move towards.
y 진짜 The y position of the point to move towards.
sp 진짜 The speed to move at in pixels per step.

보고:

N/A

예시:

if point_distance(x, y, target.x, target.y) > 5
{
    move_towards_point(target.x, target.y, 5);
}
else speed = 0;

위의 코드는 인스턴스를 단계당 5픽셀의 속도로 "target" 변수에 인덱싱된 인스턴스 위치로 이동합니다. 해당 위치에 도달하면 속도가 0으로 설정됩니다.
