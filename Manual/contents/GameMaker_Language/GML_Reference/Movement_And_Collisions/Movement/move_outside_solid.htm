move_outside_solid

이 기능을 사용하면 인스턴스가 모든 방향에서 솔리드 로 플래그가 지정된 인스턴스와 각 단계마다 임의의 수의 픽셀로 충돌에서 벗어나도록 지시할 수 있습니다. maxdist의 값은 -1 또는 0이 기본 1000px입니다. 예: GameMaker 충돌에서 벗어날 때까지 인스턴스를 계속 1000픽셀 위로 이동합니다.

통사론:

move_outside_solid(dir, maxdist);

논쟁 유형 설명
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

보고:

예시:

if other.solid
{
    var pdir;
    pdir = point_direction(other.x, other.y, x, y);
    move_outside_solid(pdir, -1);
}

위의 코드는 충돌 이벤트로 이동하여 충돌의 "기타" 인스턴스가 "솔리드"로 플래그가 지정되어 있는지 확인합니다. 그렇다면 충돌에서 인스턴스를 이동합니다.
