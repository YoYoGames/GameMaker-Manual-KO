move_snap

이 함수는 인스턴스를 주어진 크기의 그리드에 "맞추기"하는 데 사용됩니다. hsnap 및 vsnap 값이 정의하는 "보이지 않는" 그리드에서 가장 가까운 해당 위치로 스냅됩니다.

통사론:

move_snap(hsnap, vsnap);

논쟁 유형 설명
hsnap The horizontal snapping (the size in pixels between 'cells').
vsnap The vertical snapping (the size in pixels between 'cells').

보고:

예시:

with (obj_Pieces)
{
    if (!place_snapped(32, 32))
    {
        move_snap(32, 32);
    }
}

위의 코드는 "obj_Pieces"의 모든 인스턴스가 32x32 픽셀 그리드에 맞춰졌는지 확인하고 그렇지 않은 경우 해당 그리드에서 가장 가까운 위치에 스냅합니다.
