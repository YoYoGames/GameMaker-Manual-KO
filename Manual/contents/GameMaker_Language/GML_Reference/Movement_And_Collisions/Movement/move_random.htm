move_random

이 기능을 사용하면 인스턴스를 방의 어느 위치에나 설정할 수 있지만 "보이지 않는" 그리드에 정렬됩니다. 따라서 hsnap 및 vsnap에 대한 값 32는 인스턴스를 32x32 정사각형 그리드에 정렬된 임의의 위치로 설정합니다. 이 값을 1로 설정하여 방의 어느 곳에서나 위치를 얻을 수 있습니다.

참고: 이 기능은 GameMaker 가 디버깅 코드를 훨씬 더 쉽게 만들기 위해 매번 동일한 초기 임의 시드를 생성한다는 사실 때문에 게임을 새로 실행할 때마다 동일한 위치에 스냅됩니다. 이 동작을 방지하려면 다음을 사용하십시오. randomise() 게임 시작 시. 최종 실행 패키지는 이 동작을 표시하지 않고 플레이할 때마다 무작위로 실행되기 때문에 게임을 테스트하고 디버깅할 때만 해당됩니다.

통사론:

move_random(hsnap, vsnap);

논쟁 유형 설명
hsnap The horizontal snapping (the size in pixels between 'cells').
vsnap The vertical snapping (the size in pixels between 'cells').

보고:

예시:

if keyboard_check(vk_space)
{
    move_random(1, 1);
}

이렇게 하면 스페이스 키를 누를 때 인스턴스가 방의 아무 곳이나 임의의 위치로 이동합니다.
