move_bounce_all

이 기능을 사용하면 인스턴스에 방 내의 모든 인스턴스를 바운스하도록 명령할 수 있습니다. 단, 스프라이트 또는 마스크 인덱스가 할당되지 않은 인스턴스는 예외입니다. 활성화된 경우 정확한 충돌 검사를 사용하도록 지시할 수도 있지만 이를 위해서는 모든 인스턴스에 정확한 마스크가 활성화되어 있어야 하며 수행해야 하는 처리량으로 인해 많은 인스턴스가 관련된 경우 게임 속도가 크게 느려집니다. 이것은 일반적으로 인스턴스의 단계 이벤트로 이동해야 하지만 아래 코드 예제와 같이 충돌 이벤트에서도 선택적으로 사용할 수 있습니다.

통사론:

move_bounce_all( adv );

논쟁 유형 설명
adv Whether to enable advanced bouncing (true) or not (false).

보고:

예시:

if other.visible
{
    move_bounce_all(false);
}

위의 코드는 다른 객체와의 충돌 이벤트에 배치됩니다. 먼저 개체가 보이는지 확인하고 개체가 보이면 다음을 수행합니다. move_bounce_all() 동작. 이 경우 바운스는 선택적이며 모든 인스턴스, 모든 단계가 아니라 이 충돌에 대해서만 계산됩니다.
