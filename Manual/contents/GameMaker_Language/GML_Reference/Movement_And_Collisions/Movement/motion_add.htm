motion_add

이 기능은 현재 direction 그리고 speed 코드를 실행하는 인스턴스의 현재 값에 새 값을 추가합니다. 단순히 이 값을 변경하려면 함수를 사용해야 합니다. motion_set().

통사론:

motion_add(dir, speed);

논쟁 유형 설명
dir The added direction.
speed The added speed.

보고:

N/A

예시:

var pdir;
pdir = point_direction(other.x, other.y, x, y);
motion_add(pdir, other.speed);
if speed > 8 speed = 8;

위의 코드는 다른 객체와의 충돌 이벤트에서 호출됩니다. 충돌과 관련된 다른 인스턴스의 위치와 속도를 기반으로 하는 벡터를 인스턴스의 모션 방향과 속도에 추가합니다. 그런 다음 8(단계당 픽셀 수)을 초과하면 속도를 제한합니다.
