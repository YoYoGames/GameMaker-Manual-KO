move_contact_solid

이 함수는 true로 플래그가 지정된 솔리드 가 있는 인스턴스를 만날 때까지 지정된 방향으로 지정된 픽셀 수만큼 코드를 실행하는 인스턴스를 이동합니다. maxdist가 기본 1000픽셀인 경우 -1 또는 0을 사용할 수 있습니다. 즉, GameMaker 는 충돌에서 벗어날 때까지 인스턴스를 계속해서 1000픽셀 위로 이동합니다.

통사론:

move_contact_solid( dir, maxdist );

논쟁 유형 설명
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

보고:

예시:

if !place_meeting(x, y + 1, obj_Floor)
{
    move_contact_solid(270, -1);
}

위의 코드는 "obj_Floor"와의 충돌에 대해 인스턴스 아래에서 확인하고 충돌이 없는 경우 충돌이 있거나 1000픽셀이 덮일 때까지 아래로 이동합니다.
