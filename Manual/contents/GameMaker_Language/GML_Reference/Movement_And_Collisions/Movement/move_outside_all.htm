move_outside_all

이 기능을 사용하면 인스턴스가 모든 방향과 각 단계에서 임의의 수의 픽셀로 충돌에서 벗어나도록 지시할 수 있습니다. maxdist의 값은 -1 또는 0이 기본 1000픽셀입니다. 즉, GameMaker 는 인스턴스를 계속 위로 이동합니다. 충돌에서 벗어날 때까지 1000픽셀.

통사론:

move_outside_all(dir, maxdist);

논쟁 유형 설명
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

보고:

예시:

if place_meeting(x, y, all)
{
    move_outside_all(90, 1);
}

위의 코드는 인스턴스가 다른 인스턴스와 충돌할 때마다 인스턴스를 1픽셀 위로 이동합니다.
