place_snapped

이 기능을 사용하면 인스턴스의 원점(x 및 y 위치)이 사용자가 지정한 hsnap 및 vsnap 값으로 그리드에 정렬되어 있는지 확인할 수 있습니다.

통사론:

place_snapped(hsnap, vsnap);

논쟁 유형 설명
hsnap The horizontal snapping to check.
vsnap The vertical snapping to check.

보고:

예시:

with (obj_Pieces)
{
    if (!place_snapped(32, 32))
    {
        move_snap(32, 32);
    }
}

위의 코드는 "obj_Pieces"의 모든 인스턴스가 32x32 픽셀 그리드에 맞춰졌는지 확인하고 그렇지 않은 경우 해당 그리드에서 가장 가까운 위치에 스냅합니다.
