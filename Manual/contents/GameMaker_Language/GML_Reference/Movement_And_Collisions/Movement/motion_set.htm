motion_set

이 함수는 코드를 실행하는 인스턴스에 새로운 이동 방향과 새로운 속도를 설정합니다. 이것은 인스턴스의 현재 속도와 방향에 추가 되지 않습니다.motion_add() ) 대신 새 설정으로 강제합니다.

통사론:

motion_set(dir, speed);

논쟁 유형 설명
dir The new direction.
speed The new speed.`

보고:

예시:

if irandom(9) = 1
{
    motion_set(random(360), 1 + random(3));
}

위의 코드는 인스턴스가 임의의 간격으로 속도와 방향을 변경하도록 합니다.
