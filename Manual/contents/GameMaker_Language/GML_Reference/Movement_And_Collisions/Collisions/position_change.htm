position_change

이 함수는 주어진 지점에서 인스턴스 와의 충돌 위치를 확인하고 충돌이 있는 경우 충돌 중인 모든 인스턴스를 선택한 개체의 인스턴스로 변경합니다. "perf" 인수를 다음으로 설정할 수 있습니다. true GameMaker 가 발견된 인스턴스의 Destroy 및 Clean Up 이벤트와 새 인스턴스의 Create 이벤트를 강제로 수행하도록 하거나 false 이러한 이벤트를 수행하지 않습니다. Destroy, Clean Up 및 Create 이벤트를 수행하지 않도록 선택하는 경우 일반적으로 create 이벤트에 정의된 변수를 사용하여 생성된 인스턴스는 해당 변수가 더 이상 존재하지 않으므로 게임이 중단됩니다.

통사론:

position_change(x, y, obj, perf);

논쟁 유형 설명
x 진짜 The x coordinate of where to change colliding instances.
y 진짜 The y coordinate of where to change colliding instances.
obj 개체 자산 The new object the calling object will change into.
perf 부울 Whether to perform that new object's Create event (true) or not (false).

보고:

N/A

예시:

position_change(200, 200, obj_Bird, true);

이것은 지점(200,200)에서 충돌하는 모든 인스턴스를 "obj_Bird"의 인스턴스로 변경하고 프로세스에서 각각에 대해 "obj_Bird"의 Create 이벤트를 수행합니다.
