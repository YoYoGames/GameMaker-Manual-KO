point_in_circle

이 기능을 사용할 때 원형 영역을 정의하면 GameMaker 는 주어진 점이 경계 내에 있는지 여부를 확인합니다. 점이 정의된 원 안에 있으면 함수가 반환됩니다. true 그렇지 않으면 함수가 반환됩니다 false.

통사론:

point_in_circle(px, py, x1, y1, rad);

논쟁 유형 설명
px The x coordinate of the point to check.
py The y coordinate of the point to check.
x1 The x coordinate of the circle centre.
y1 The y coordinate of the circle centre.
rad The radius of the circle.

보고:

예시:

if point_in_circle(mouse_x, mouse_y, x, y, 16)
{
    over = true;
}
else
{
    over = false;
}

위의 코드는 point_in_circle 마우스 위치가 정의된 원형 영역 내에 있는지 확인하는 함수로 "over" 변수를 다음으로 설정합니다. true 그렇다면, 또는 false 그렇지 않으면.
