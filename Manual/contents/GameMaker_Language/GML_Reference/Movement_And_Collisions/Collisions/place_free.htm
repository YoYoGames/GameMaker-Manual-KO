place_free

이 함수를 사용하여 호출 인스턴스가 게임에서 솔리드 로 플래그가 지정된 인스턴스와 충돌하는지 확인하고 확인할 수 있습니다. 이제 이것이 작동하려면 코드를 실행하는 인스턴스에 유효한 충돌 마스크가 있어야 합니다(스프라이트 자체에 대해 또는 mask_index ) 유효한 마스크가 있는 솔리드 플래그가 지정된 인스턴스와의 충돌만 등록합니다.

함수 자체는 기본적으로 인스턴스를 가져오고 x/y 인수로 지정된 위치에 배치될 때 솔리드 플래그가 지정된 인스턴스와의 충돌 테스트를 통해 작동합니다. 충돌 검사는 어떤 종류의 충돌 마스크가 선택되었는지에 따라 정확하거나 인스턴스의 경계 상자를 기반으로 할 수 있습니다.

주어진 x/y 좌표는 검사가 수행되기 전에 가장 가까운 정수로 내림됩니다.

통사론:

place_free(x, y);

논쟁 유형 설명
x The x position to check.
y The y position to check.

보고:

예시:

if place_free(mouse_x, mouse_y)
{
    x = mouse_x;
    y = mouse_y;
}

위의 코드는 호출 인스턴스가 마우스와 같은 위치에 배치되어야 하는 경우 솔리드 플래그가 지정된 인스턴스와의 충돌을 확인합니다. 충돌이 감지되지 않으면 인스턴스의 x/y 좌표가 마우스 좌표로 설정됩니다.
