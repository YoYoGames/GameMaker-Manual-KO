position_destroy

이 함수는 단순히 지정된 위치와 충돌하는 것으로 확인된 모든 인스턴스를 파괴합니다. 충돌은 인스턴스의 마스크를 기반으로 하며 마스크의 일부가 대상 지점과 겹치면 함수가 해당 인스턴스를 파괴합니다. 이러한 방식으로 파괴된 인스턴스는 Destroy 및 Clean Up 이벤트를 트리거합니다.

통사론:

position_destroy(x, y);

논쟁 유형 설명
x The x coordinate of where to destroy colliding instances.
y The y coordinate of where to destroy colliding instances.

보고:

예시:

if mouse_check_button_pressed(mb_left)
{
    position_destroy(mouse_x, mouse_y);
}

이것은 왼쪽 마우스 버튼을 눌렀을 때 마우스 커서의 위치와 충돌하는 모든 인스턴스를 파괴합니다.
