position_meeting

이 기능을 사용하면 객체의 다른 인스턴스 또는 모든 인스턴스와의 충돌 위치를 확인할 수 있습니다. 이것을 사용하면 방의 단일 지점에서 인스턴스 또는 개체를 확인하는 것입니다. 이 검사는 인스턴스의 경계 상자에 대해 수행되거나 해당 인스턴스에 정확한 충돌이 확인된 경우 해당 인스턴스의 마스크에 대해 수행됩니다. 개체가 충돌하는 경우 고유한 인스턴스 ID 를 가져와야 하는 경우 다음을 사용해야 합니다.instance_position(). 이 함수는 특수 키워드 all 및 other 도 허용합니다.

통사론:

position_meeting(x, y, obj);

논쟁 유형 설명
x 진짜 The x position to check.
y 진짜 The y position to check.
obj 개체 자산 또는 인스턴스 ID The object (or instance id, or the keywords "all" or "other") to check for a collision with.

보고:

Boolean

예시:

if mouse_check_button(mb_left)
{
    if (!position_meeting(mouse_x, mouse_y, all))
    {
        instance_create_layer(mouse_x, mouse_y, "Walls", obj_Wall);
    }
}

위의 코드는 왼쪽 마우스 버튼을 확인하고, 눌려지면 마우스 x/y 위치에서 인스턴스와의 충돌을 확인합니다. 없는 경우 "obj_Wall"의 인스턴스가 생성됩니다.
