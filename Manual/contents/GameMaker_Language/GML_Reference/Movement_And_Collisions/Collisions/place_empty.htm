place_empty

이 함수를 사용하여 호출하는 인스턴스가 게임의 다른 개체 또는 모든 인스턴스와 충돌하는지 확인하고 확인할 수 있습니다. 충돌이 올바르게 해결되려면 코드를 실행하는 인스턴스에 유효한 충돌 마스크가 있어야 합니다(스프라이트 자체에 대해 또는 mask_index ) 유효한 마스크가 있는 인스턴스와의 충돌만 등록합니다.

함수는 호출 인스턴스가 특정 위치에 있어야 하는 충돌이 없는지 테스트하고 있으며 선택적 인수를 제공하여 위치에 지정된 유형의 인스턴스와 충돌이 없는지 확인하기 위한 검사를 구체화할 수 있습니다. 선택적 개체 ID가 제공되지 않으면 룸 내의 모든 인스턴스에 대해 검사가 수행됩니다. 충돌 검사는 선택한 충돌 마스크의 종류에 따라 정확하거나 인스턴스의 경계 상자를 기반으로 하지만 정확한 충돌이 올바르게 작동하려면 검사의 모든 인스턴스에 정확한 충돌 마스크가 있어야 합니다.

주어진 x/y 좌표는 검사가 수행되기 전에 가장 가까운 정수로 내림됩니다.

통사론:

place_empty(x, y, [object_id]);

논쟁 유형 설명
x 진짜 The x position to check.
y 진짜 The y position to check.
[object_id] 개체 자산 OPTIONAL The object to check against.

보고:

Boolean

예시:

if place_empty(mouse_x, mouse_y, obj_Enemy)
{
    x = mouse_x;
    y = mouse_y;
}

위의 코드는 호출 인스턴스가 마우스와 같은 위치에 배치되어야 하는 것처럼 개체 "obj_Enemy"의 다른 인스턴스와의 충돌을 확인합니다. 충돌이 감지되지 않으면 인스턴스의 x/y 좌표가 마우스 좌표로 설정됩니다.
