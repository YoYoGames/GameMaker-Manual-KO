place_meeting

이 기능을 사용하면 검사용 코드를 실행하는 인스턴스의 충돌 마스크를 사용하여 다른 인스턴스 또는 객체의 모든 인스턴스와의 충돌 위치를 확인할 수 있습니다. 이것을 사용하면 GameMaker 에 인스턴스를 새 위치로 이동하고, 충돌을 확인하고, 뒤로 이동하여 충돌이 발견되었는지 여부를 알려주도록 효과적으로 요청하는 것입니다. 이것은 정확한 충돌에 대해 작동하지만 확인 중인 개체와 인스턴스 모두에 정확한 충돌 마스크가 선택되어 있는 경우에만 그렇지 않으면 경계 상자 충돌만 적용됩니다. 이 함수는 특수 키워드 all 및 other 도 허용합니다. 충돌하는 개체의 고유한 인스턴스 ID 를 가져와야 하는 경우 다음을 사용해야 합니다.instance_place().

주어진 x/y 좌표는 검사가 수행되기 전에 가장 가까운 정수로 내림됩니다.

통사론:

place_meeting(x, y, obj);

논쟁 유형 설명
x 진짜 The x position to check.
y 진짜 The y position to check.
obj 개체 자산 또는 인스턴스 ID The instance or object to check for.

보고:

Boolean

예시:

if keyboard_check(vk_left)
{
    if !place_meeting(x - 5, y, obj_wall) x -=5;
}

위의 코드는 인스턴스의 왼쪽에 충돌이 없는지 확인하고 충돌이 없으면 인스턴스를 이동합니다.
