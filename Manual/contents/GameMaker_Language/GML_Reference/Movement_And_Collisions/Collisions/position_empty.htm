position_empty

이 함수는 주어진 위치에서 유효한 충돌 마스크가 있는 인스턴스 와 충돌이 발생하는지 확인합니다.

통사론:

position_empty(x, y);

논쟁 유형 설명
x The x position to check.
y The y position to check.

보고:

예시:

var xx, yy;
xx = random(room_width);
yy = random(room_height);
if (position_empty(xx, yy))
{
    instance_create_layer(xx, yy, "Bullets", obj_Bomb);
}

이것은 충돌을 위해 방 어딘가에 있는 임의의 위치를 검사할 것이고, 충돌이 없으면 obj_Bomb의 인스턴스를 생성할 것입니다.
