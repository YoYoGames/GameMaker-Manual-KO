페이스북

Facebook은 인터넷에서 가장 중요한 소셜 미디어 사이트 중 하나이며 많은 게임 및 개발자는 현재 게임 및 미디어 게시를 위한 자체 페이지를 보유하고 있습니다. GameMaker 에는 Facebook에 연결하고 해당 API가 허용하는 특정 작업과 작업을 수행할 수 있는 확장 기능이 있습니다. 확장 기능을 사용하기 시작하기 전에 Facebook 개발자 페이지를 방문하여 Facebook 그래프가 작동하는 방식과 사용 가능한 호출 및 권한이 이러한 기능을 최대한 활용하는 데 필수적임을 확실히 이해하는 것이 좋습니다.

다음 링크에서 사용 가능한 모든 플랫폼에 대해 Facebook 확장 프로그램을 설정하고 사용하는 방법을 찾을 수 있습니다.

  • Android, iOS 및 HTML5: Facebook 통합

다음은 레거시 Facebook API에만 유효했던 사용되지 않는 함수 입니다. 따라서 더 이상 프로젝트에서 이러한 기능을 사용할 수 없으며 이전 프로젝트를 함께 가져온 경우 여기에 나열됩니다. 현재 Facebook 기능을 사용하려면 대상 플랫폼에 대한 적절한 게임 옵션 에서 Facebook API 를 활성화해야 합니다. 그러면 확장을 다운로드하고 설치하라는 메시지가 표시됩니다. 확장 프로그램은 다음과 같이 제공됩니다. *.pdf 사용 방법을 설명하고 모든 기능 정의와 예제를 보여주는 "mini-manual" 형식. 오래된 모든 참조 facebook_*() 확장 기능을 사용하려면 기능을 교체(및 업데이트)해야 합니다. fb_*() 등가물.

아래 나열된 기능은 게임을 Facebook과 통합하기 위한 것입니다.

  • facebook_request_read_permissions
  • facebook_check_permission
  • facebook_status
  • facebook_accesstoken
  • facebook_user_id
  • facebook_graph_request
  • facebook_dialog
  • facebook_send_invite
  • facebook_post_message
  • facebook_logout
  • facebook_launch_offerwall
