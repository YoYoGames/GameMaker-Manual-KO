shop_leave_rating

이 기능은 사용자에게 특정 페이지에 등급이나 댓글을 게시하도록 요청할 수 있는 OS 종속 대화 상자를 엽니다. 대화 상자에 표시할 텍스트는 물론 두 개의 버튼에 표시할 텍스트와 댓글이 게시되어야 하는 URL을 정의할 수 있습니다.

참고: 이 기능은 Android 및 iOS 대상에만 해당됩니다.

중요한! iOS 에서 TestFlight를 사용할 때 개발 중 프롬프트가 표시되지만 실제로는 아무 것도 할 수 없습니다. 앱이 App Store에 등록되면 이 버튼이 활성화되고 사용자에게 리뷰를 작성하거나 평점을 남길 수 있는 옵션이 제공됩니다. TestFlight를 통해 게임을 배포하는 경우 이 창이 표시되지 않을 수도 있습니다.

통사론:

shop_leave_rating(text, yes_string, no_string, url)

논쟁 유형 설명
text 끈 Text that appears on the dialog.
yes_string 끈 Text that appears on the "yes" button.
no_string 끈 Text that appears on the "no" button.
url 끈 The full URL where the comment has to be sent.

예시:

if timer <= 0
{
    shop_leave_rating("Thanks for playing the game! Why not leave a comment?", "Okay!", "Not Today!", "http://MacSweeney/comments");
    timer = 10000000;
}
else
{
    timer -= 1;
}

위의 코드는 "timer" 변수가 0으로 카운트다운된 경우 사용자에게 주석을 남길 것을 요청합니다.
