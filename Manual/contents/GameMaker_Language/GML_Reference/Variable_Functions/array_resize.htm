array_resize

이 기능을 사용하면 기존 배열 차원의 크기를 새 크기로 조정할 수 있습니다. 크기를 조정할 배열과 배열에 대한 새 인덱스 수를 제공하면 함수가 해당 배열의 크기를 조정합니다. 이 함수는 배열에 새 인덱스를 설정하기만 하면 크기를 늘릴 수 있으므로 배열의 크기를 더 작게 조정하도록 설계되었습니다. 즉, 배열 크기를 늘리는 데 사용하는 경우 새 인덱스는 기본값 0으로 설정됩니다.

이 함수는 다음 패턴에 따라 배열 인덱스를 제공할 때 크기를 조정할 차원을 지정하는 한 다차원 배열에도 사용할 수 있습니다.

// Resize the first dimension of the array
array_resize(my_array, 10);

// Resize the second dimension of the array (only for the first slot)
array_resize(my_array[0], 10);

// Resize the third dimension of the array (only for the first slots)
array_resize(my_array[0][0], 10);

// ...and so on.

코드는 주어진 배열 차원의 슬롯이 서로 연결 되지 않고 다른 크기를 가질 수 있기 때문에 두 번째 차원의 첫 번째 슬롯만 크기가 조정된다고 명시합니다. 예를 들어:

array_resize(my_array[0], 10);
array_resize(my_array[1], 22);

위의 코드에서 배열의 두 번째 차원 길이는 첫 번째 슬롯에서는 10이지만 두 번째 슬롯에서는 22입니다.

통사론:

array_resize(array_index, new_size);

논쟁 유형 설명
array_index The index of the array to resize.
new_size The new size for the array (an integer, starting from 0).

보고:

예시:

if array_length(a) > 10
{
    array_resize(a, 10);
}

위의 코드는 배열의 길이를 확인하고 인덱스가 10개 이상인 경우 크기를 조정합니다.
