is_array

이 함수는 변수에 배열이 있는지 확인하는 데 사용할 수 있습니다. true ) 또는 그렇지 않은 경우(이 경우 반환됩니다. false ).

통사론:

is_array(n);

논쟁 유형 설명
n 변하기 쉬운 The variable to check.

보고:

Boolean

예시:

if is_array(a)
{
    a = -1;
}

위의 코드는 변수를 검사하여 배열인지 확인하고 함수가 반환하는지 확인합니다. true, 변수를 -1로 설정하면 배열이 삭제됩니다.
