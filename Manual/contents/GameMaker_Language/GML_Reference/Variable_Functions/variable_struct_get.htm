variable_struct_get

이 함수를 사용하면 구조체 내의 지정된 명명된 변수에서 값을 가져올 수 있습니다. 구조체 참조와 변수 이름을 제공하여 값을 문자열로 가져옵니다(아래 예제 코드 참조). 함수는 변수가 보유한 값을 반환하거나 undefined 명명된 변수가 존재하지 않는 경우.

중요한! 가져오는 변수가 존재하지 않으면 함수는 키워드를 반환합니다. undefined 게임이 작동하지 않는 오류가 발생할 수 있으므로 의심스러운 경우 이 기능을 사용하십시오.variable_struct_exists() 첫 번째.

통사론:

variable_struct_get(struct, name);

논쟁 유형 설명
struct The struct reference to use
name The name of the variable to get (as a string)

보고:

Variable (any data type) or undefined (if the named variable does not exist)

예시:

if variable_struct_exists(mystruct, "shields")
{
    var ss = variable_struct_get(mystruct, "shields");
}
else
{
    var ss = -1;
}

위의 코드는 주어진 구조체에 변수가 존재하는지 확인하고 만약 존재한다면 그 변수가 가지고 있는 값을 검색하여 지역 변수에 저장합니다. 존재하지 않는 경우 지역 변수는 -1로 설정됩니다.
