is_struct

이 함수는 제공된 값이 구조체인지 확인합니다. 그것은 반환 true 그렇다면 반환됩니다.false.

메서드 변수 도 반환됩니다. true, 개체 인스턴스 가 반환됩니다. false.

통사론:

is_struct(val);

논쟁 유형 설명
val 변하기 쉬운 The value to check.

보고:

Boolean

예시:

if is_struct(a)
{
    delete(a);
}

위의 코드는 변수를 검사하여 구조체인지, 그리고 함수가 다음을 반환하는지 확인합니다. true, 구조체가 삭제됩니다.
