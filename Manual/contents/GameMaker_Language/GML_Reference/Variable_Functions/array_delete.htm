array_delete

이 함수를 사용하면 주어진 위치에 있는 배열에서 값(또는 값)을 삭제할 수 있습니다. 이 함수를 사용하려면 삭제할 값의 개수뿐만 아니라 배열, 삭제할 배열의 인덱스(위치)를 보유하는 변수를 제공해야 합니다.

통사론:

array_delete(variable, index, number);

논쟁 유형 설명
variable The variable that holds the array.
index The index (position) in the array to delete the value(s) from.
number The number of values to delete.

보고:

예시:

array_delete(score_array, 0, 10);

위의 코드는 인덱스 0에서 시작하는 지정된 배열의 처음 10개 값을 삭제합니다(따라서 인덱스 0에서 9는 배열에서 제거됩니다).
