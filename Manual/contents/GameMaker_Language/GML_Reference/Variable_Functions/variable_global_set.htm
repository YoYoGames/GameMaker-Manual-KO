variable_global_set

이 함수를 사용하여 주어진 전역 변수의 값을 설정할 수 있습니다. 값을 문자열로 설정할 전역 변수의 이름(아래 예제 코드 참조)을 제공한 다음 설정할 값(유효한 데이터 유형일 수 있음)을 제공합니다. 전역 변수가 게임에 이미 존재하지 않으면 생성된 다음 값이 할당됩니다.

통사론:

variable_global_set(name, val);

논쟁 유형 설명
name 끈 The name of the global variable to set (as a string)
val 변하기 쉬운 The value to set the global variable to

보고:

N/A

예시:

if !variable_global_exists("enemy_num")
{
    variable_global_set("enemy_num", 0);
}

위의 코드는 전역 변수가 존재하는지 확인하고 존재하지 않으면 생성되어 0으로 설정됩니다.
