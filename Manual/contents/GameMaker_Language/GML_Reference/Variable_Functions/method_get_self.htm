method_get_self

이 함수를 사용하면 인스턴스 ID 또는 구조체 참조를 검색할 수 있습니다.self 메서드가 호출될 때 사용되는 컨텍스트입니다. 변수가 메서드가 아닌 경우 함수는 다음을 반환합니다. undefined. 이 함수는 상수를 반환할 수도 있습니다. pointer_null, 이 경우 현재 자아는 호출 시 사용됩니다.

통사론:

method_get_self(method);

논쟁 유형 설명
method The method variable to check.

보고:

Instance ID, Struct, undefined, or pointer_null

예시:

var _self = method_get_self(light_properties);
show_debug_message(string(_self));

위의 코드는 주어진 메소드 변수에 대한 자체 컨텍스트를 가져와 콘솔에 출력합니다.
