array_copy

이 기능을 사용하면 배열의 전체 또는 일부를 임의의 위치에 있는 다른 배열로 복사할 수 있습니다. 원본 및 대상 배열(둘 다 이전에 생성되어야 함)과 복사할 원본 배열 내의 위치와 복사할 대상 배열 내의 위치를 모두 제공해야 합니다. 마지막으로 복사할 배열의 길이(또는 원하는 부분의 길이)를 지정해야 합니다. 복사 중인 데이터가 대상 배열의 길이를 초과하면 배열이 데이터를 허용하도록 확장됩니다.

이 함수는 다음 패턴에 따라 배열 인덱스를 제공할 때 복사할 차원을 지정하는 한 다차원 배열에도 사용할 수 있습니다.

// Copy to the first dimension of an array
// from the second dimension of an array
array_copy(item_array, 0, inventory_array[0], 0, len);

// Copy to the third dimension of an array
// from the first dimension of an array
array_copy(item_array[0][0], 0, inventory_array, 0, len);

// etc...

통사론:

array_copy(dest, dest_index, src, src_index, length);

논쟁 유형 설명
dest 정렬 The ID of the array to copy to.
dest_index 진짜 The index within the array to copy to.
src 정렬 The ID of the array to copy from.
src_index 진짜 The index with the array to start copying from.
length 진짜 The length (number of array indices) to copy.

예시:

if !array_equals(inventory_array, item_array)
{
    var len = array_length(inventory_array);
    array_copy(item_array, 0, inventory_array, 0, len);
}

위의 코드는 두 개의 배열이 동일한 값을 보유하고 있는지 확인하고, 그렇지 않은 경우 코드는 "inventory_array" 배열의 전체 내용을 "item_array" 배열에 복사합니다.
