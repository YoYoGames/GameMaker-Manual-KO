is_real

이 함수는 주어진 변수가 실수(단수, 이중 또는 정수)인지 여부를 반환합니다. 어떤 경우에는 변수에 실수가 있는지 확인하고 싶을 때 이 함수를 사용합니다. 실수를 반환하지 않고 오히려 true 또는 false, 예를 들어 "fish" 값은 다음을 반환합니다. false 그러나 값이 200이면 반환됩니다. true.

참고: 이 기능은false 모든 enum 값에 대해 int64로 저장됩니다. 해당 데이터 유형을 확인하는 올바른 함수는is_int64().

통사론:

is_real(n);

논쟁 유형 설명
n 변하기 쉬운 The argument to check.

보고:

Boolean

예시:

if is_real(val)
{
    score += val;
}

위의 코드는 변수 "val"을 확인하여 실수인지 확인하고 실제 숫자이면 점수에 추가합니다.
