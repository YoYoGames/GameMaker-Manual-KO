variable_struct_exists

이 함수를 사용하면 주어진 구조체 내에 변수가 존재하는지 여부를 확인할 수 있습니다. 사용할 구조체 참조와 확인할 변수 이름을 문자열로 제공합니다(아래 예제 코드 참조). 함수가 반환됩니다 true 구조체에 대해 주어진 이름의 변수가 존재하고 false 그렇지 않으면.

통사론:

variable_struct_exists(struct, name);

논쟁 유형 설명
struct 구조체 The struct reference to check
name 끈 The name of the struct variable to check for (as a string)

보고:

Boolean

예시:

if !variable_struct_exists(mystruct, "shields")
{
    mystruct.shields = 0;
}

위의 코드는 "shields"라는 변수가 주어진 구조체에 존재하는지 확인하고 존재하지 않으면 생성되어 0으로 초기화됩니다.
