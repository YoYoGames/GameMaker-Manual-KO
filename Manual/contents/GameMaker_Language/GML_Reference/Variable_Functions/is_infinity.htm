is_infinity

이 함수는 주어진 변수가 infinity (무한수) 여부, 반환 true 그렇다면, 그리고 false 그렇지 않은 경우.

통사론:

is_infinity(n);

논쟁 유형 설명
n 변하기 쉬운 The argument to check.

보고:

Boolean

예시:

if is_infinity(global.value)
{
    show_debug_message("Value is infinite!");
}

위의 코드는 전역 변수 "value"가 무한대인지 아닌지 확인하고 무한대라면 디버그 메시지를 보여줍니다.
