array_create

이 기능을 사용하면 주어진 크기의 1D 배열을 생성할 수 있습니다. 생성할 배열의 길이를 함수에 지정하면 배열에 대한 "핸들"이 반환되며 이를 변수에 할당할 수 있습니다. 이러한 방식으로 생성된 배열은 (선택 사항) 초기화 값을 지정하지 않는 한 각 항목이 0으로 초기화됩니다. 배열 초기화를 위한 추가 값을 제공하면 새 배열 내의 모든 인덱스가 0 대신 설정되지만 이 경우 함수의 성능 오버헤드가 더 커집니다.

통사론:

array_create(size, [value]);

논쟁 유형 설명
size 진짜 The size of the array to create.
value 변하기 쉬운 OPTIONAL The value to use to initialise all array indices.

보고:

Array

예시:

instance_array = array_create(100, noone);

위의 코드는 각각 키워드로 설정된 100개 항목의 새 배열을 만듭니다. noone, 그런 다음 "instance_array" 변수에 할당합니다.
