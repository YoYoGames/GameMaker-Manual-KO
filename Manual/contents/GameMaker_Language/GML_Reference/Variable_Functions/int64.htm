int64

이 함수는 주어진 값을 부호 있는 64비트 정수로 변환하려고 시도합니다. 여기서 값 은 실수, 문자열, int64, int32, 또는 ptr. 다른 모든 것은 오류 메시지와 함께 게임이 충돌하는 원인이 됩니다. 함수를 사용하여 변수에 int64가 있는지 확인할 수 있습니다.is_int64().

통사론:

int64(val);

논쟁 유형 설명
val 실수, 문자열 또는 포인터 The value to convert.

보고:

int64 (signed 64-bit integer)

예시:

steam_handle = int64(global.fileReadString);

위의 코드는 전역 변수에 저장된 값을 64비트 정수로 변환합니다.
