ptr

이 함수는 주어진 값을 포인터 데이터 유형으로 변환하려고 시도합니다. 여기서 값은 다음 중 하나 여야 합니다. real, ㅏ string, int64, int32, 또는 ptr. 다른 모든 것은 오류 메시지와 함께 게임이 충돌하게 합니다.

통사론:

ptr(n);

논쟁 유형 설명
n 실수, 문자열 또는 포인터 The value to convert.

보고:

Pointer

예시:

if !is_ptr(val)
{
    val = ptr(application_surface);
}

위의 코드는 변수 "val"을 검사하여 포인터가 포함되어 있는지 확인하고 포인터가 없으면 할당됩니다.
