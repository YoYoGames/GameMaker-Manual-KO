array_length_1d 지원 중단됨

경고! 이 함수는 더 이상 사용되지 않으며 다음으로 대체됩니다. array_length() ) 배열이 더 이상 1차원 또는 2차원으로만 제한되지 않으므로 이 기능은 레거시 프로젝트 지원을 위해서만 제공됩니다. 모든 새 프로젝트는 다차원 배열을 만들고 액세스하는 현재 방식을 사용해야 합니다(자세한 내용은 여기 참조).

이 함수를 사용하여 1D 배열의 길이(항목 수)를 얻을 수 있습니다. 2D 배열의 경우 다음을 사용해야 합니다.array_height_2d 그리고array_length_2d 기능.

중요!: 배열에 32,000개 이상의 항목이 있는 경우 이 함수는 잘못된 값을 반환하므로 사용해서는 안 됩니다.

통사론:

array_length_1d(array_index);

논쟁 유형 설명
array_index 정렬 The index of the array to check.

보고:

Real

예시:

for (var i = array_length_1d(a) - 1; i > -1; i--;)
{
    a[i] = -1;
}

위의 코드는 배열을 반복하고 각 항목을 -1로 설정합니다.
