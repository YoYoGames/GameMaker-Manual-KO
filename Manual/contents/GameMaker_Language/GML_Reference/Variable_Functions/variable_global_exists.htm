variable_global_exists

이 기능을 사용하면 전역 범위 변수의 존재 여부를 확인할 수 있습니다. 확인할 전역 변수 이름을 문자열로 제공하면(아래 예제 코드 참조) 함수는 다음을 반환합니다. true 주어진 이름의 전역 변수가 존재하거나 false 그렇지 않으면.

통사론:

variable_global_exists(name);

논쟁 유형 설명
name 끈 The name of the global variable to check for (as a string)

보고:

Boolean

예시:

if !variable_global_exists("enemy_num")
{
    global.enemy_num = instance_number(obj_Enemey_Parent);
}

위의 코드는 "enemy_num"이라는 전역 변수가 있는지 확인하고 없으면 생성하여 값을 설정합니다.
