is_bool

이 함수는 주어진 변수가 부울(true 이오르 false ) 여부. 경우에 따라 GameMaker 의 변수가 부울 값을 보유하는지 확인하고 싶을 때 이 함수를 사용합니다.

통사론:

is_bool(n);

논쟁 유형 설명
n 변하기 쉬운 The argument to check.

보고:

Boolean

예시:

if is_bool(val)
{
    global.Sound = val;
}
else
{
    global.Sound = true;
}

위의 코드는 변수 "val"을 확인하여 실수인지 확인하고 실제 숫자이면 전역 변수를 설정하는 데 사용합니다.
