variable_instance_get

이 함수를 사용하면 지정된 명명된 변수에서 값을 가져올 수 있습니다. 고유한 인스턴스 ID 값(룸 편집기의 인스턴스 속성 에서 찾을 수 있거나 함수를 호출할 때 반환됨)을 제공합니다.instance_create_layer() ) 뿐만 아니라 값을 문자열로 가져올 변수의 이름(아래 예제 코드 참조). 함수는 변수가 보유한 값을 반환하거나 undefined 변수가 존재하지 않는 경우.

중요한! 가져오는 변수가 존재하지 않으면 함수는 키워드를 반환합니다. undefined 게임이 작동하지 않는 오류가 발생할 수 있으므로 의심스러운 경우 이 기능을 사용하십시오.variable_instance_exists 첫 번째.

통사론:

variable_instance_get(instance_id, name);

논쟁 유형 설명
instance_id 인스턴스 ID The unique ID value of the instance to use
name 끈 The name of the variable to get (as a string)

보고:

Variable (any data type) or undefined (if the named variable does not exist)

예시:

if variable_instance_exists(id, "shields")
{
    var ss = variable_instance_get(id, "shields");
}
else
{
    var ss = -1;
}

위의 코드는 변수가 존재하는지 확인하고 만약 존재한다면 그 변수가 가지고 있는 값을 검색하여 지역 변수에 저장합니다. 존재하지 않는 경우 지역 변수는 -1로 설정됩니다.
