variable_instance_names_count

이 함수를 사용하면 인스턴스에 대해 정의된 변수의 총 수를 찾을 수 있습니다. 확인할 인스턴스 ID를 제공하면 함수는 발생한 변수 수에 대한 정수 값을 반환하거나 (주어진 ID의 인스턴스가 없는 경우) -1을 반환합니다.

통사론:

variable_instance_names_count(instance_id);

논쟁 유형 설명
instance_id 인스턴스 ID The unique ID value of the instance to check.

보고:

Real

예시:

var _num = variable_instance_names_count(myinst);
show_debug_message("Instance Variables = " + string(_num));

위의 코드는 지정된 인스턴스의 변수 수를 검색하고 해당 값과 함께 콘솔 출력에 디버그 메시지를 표시합니다.
