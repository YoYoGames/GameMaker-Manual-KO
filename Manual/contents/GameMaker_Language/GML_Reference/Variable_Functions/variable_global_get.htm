variable_global_get

이 함수를 사용하면 지정된 전역 변수에서 값을 가져올 수 있습니다. 값을 문자열로 가져오기 위해 전역 변수의 이름을 제공하면(아래 예제 코드 참조) 함수는 전역 변수 또는 undefined 변수가 존재하지 않는 경우.

중요한! 얻는 전역 변수가 존재하지 않으면 함수는 키워드를 반환합니다 undefined 게임이 작동하지 않는 오류가 발생할 수 있으므로 의심스러운 경우 이 기능을 사용하십시오.variable_global_exists 첫 번째.

통사론:

variable_global_get(name);

논쟁 유형 설명
name The name of the global variable to get (as a string)

보고:

Variable (any data type) or undefined (if the named variable does not exist)

예시:

if variable_global_exists("enemy_num")
{
    show_debug_message("enemy_num = " + string(variable_global_get("enemy_num"));)
}

위의 코드는 전역 변수가 존재하는지 확인하고 존재하면 콘솔에 출력합니다.
