is_matrix

이 함수는 주어진 변수가 행렬을 참조하는지 여부를 반환합니다. 어떤 경우에는 GameMaker 에서 변수가 보유하는 데이터 유형을 확인하고 확인하고 싶을 때 이 기능을 사용합니다. 그것은 반환 true 또는 false 값이 행렬인지 여부에 따라 다릅니다.

통사론:

is_matrix(n);

논쟁 유형 설명
n 변하기 쉬운 The argument to check.

보고:

Boolean

예시:

if !is_matrix(val)
{
    show_debug_message("Not a valid matrix!");
}

위의 코드는 변수 "val"을 검사하여 행렬이 포함되어 있는지 확인하고 그렇지 않은 경우 디버그 콘솔에 메시지를 표시합니다.
