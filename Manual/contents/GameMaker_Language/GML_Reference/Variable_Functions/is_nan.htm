is_nan

이 함수는 주어진 변수가 NaN (숫자가 아님) 여부, 반환 true 그렇다면, 그리고 false 그렇지 않은 경우.

통사론:

is_nan(n);

논쟁 유형 설명
n 변하기 쉬운 The argument to check.

보고:

Boolean

예시:

if is_nan(global.value)
{
    show_debug_message("Value is not a number");
}

위의 코드는 전역 변수 "value"를 확인하여 숫자인지 아닌지 확인하고 그렇지 않은 경우 디버그 메시지를 표시합니다.
