array_length_2d 지원 중단됨

경고! 이 함수는 더 이상 사용되지 않으며 다음으로 대체됩니다. array_length() ) 배열이 더 이상 1차원 또는 2차원으로만 제한되지 않으므로 이 기능은 레거시 프로젝트 지원을 위해서만 제공됩니다. 모든 새 프로젝트는 다차원 배열을 만들고 액세스하는 현재 방식을 사용해야 합니다(자세한 내용은 여기 참조).

이 함수를 사용하면 배열의 두 번째 차원 길이(항목 수)를 얻을 수 있습니다. 첫 번째 차원에 대한 항목 번호를 제공하면 함수가 배열에 있는 두 번째 차원 항목의 수를 반환합니다(첫 번째 차원의 길이를 찾으려면 함수 사용array_height_2D() ). 함수는 주어진 변수가 배열이 아니면 0을 반환하고 변수가 1D 배열이면 1을 반환합니다(여전히 1행이 있기 때문에).

통사론:

array_length_2d(array_index, n);

논쟁 유형 설명
array_index 정렬 The index of the array to check.
n 진짜 The entry of the array to get the length of.

보고:

Real

예시:

for (var i = 0; i < array_height_2d(a); ++i;)
{
    for (var j = 0; j < array_length_2d(a, i); ++j;)
    {
        a[i, j] = -1;
    }
}

위의 코드는 2D 배열을 반복하고 각 항목을 -1로 설정합니다.
