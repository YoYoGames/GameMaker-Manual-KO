typeof

이 함수는 주어진 변수의 데이터 유형을 문자열로 반환합니다. 가능한 반환 값은 아래 표에 나열되어 있습니다.

끈 설명
"number" 변수는 실수(부동 소수점) 숫자를 포함합니다. 여기에는 다음이 포함될 수 있습니다. NaN 그리고 infinity
"string" 변수는 문자열을 보유합니다.
"array" 변수가 배열을 참조합니다.
"bool" 변수는 부울(true / false )
"int32" 변수는 32비트 정수를 보유합니다.
"int64" 변수는 64비트 정수를 보유합니다.
"ptr" 변수에는 포인터가 있습니다.
"undefined" 변수가 정의되지 않았습니다
"null" 변수가 null 값을 보유합니다(이는 일반적으로 표시되지 않아야 함).
"vec3" 변수는 3 값 벡터를 보유합니다.
"vec4" 변수는 4 값 벡터를 보유합니다.
"method" 변수는 함수 참조를 보유합니다.
"struct" 변수는 구조체 참조를 보유합니다.
"ref" 변수는 인스턴스 참조를 보유합니다.
"unknown" 값을 알 수 없습니다. 이것은 절대 볼 수 없으며 메모리 덮어쓰기와 같은 가장 기본적인 수준에서 문제가 발생했음을 나타냅니다.

이 함수가 메서드 에 대한 올바른 값을 반환하지 않을 수 있는 경우가 있습니다. 다음 두 가지 함수 정의를 고려하십시오.

a = function()
{
    // something
}

function b()
{
    // Something
}

엄밀히 말하면 이 둘은 함수를 인스턴스 범위 변수에 바인딩하기 때문에 메서드로 간주되지만 다음을 호출합니다. typeof() 기능에 b 호출하는 동안 "메서드"가 아닌 "숫자"를 반환합니다. a "메소드"를 반환합니다. 이것은 메소드가 다음과 같이 생성되었다는 사실 때문입니다. b 스크립트에 함수가 정의되지 않은 경우에도 컴파일러가 스크립트 함수를 인식하는 방식이므로 스크립트 인덱스(정수 값)가 할당됩니다.

통사론:

typeof(variable);

논쟁 유형 설명
variable 어느 The variable to get the data type of.

보고:

String (see table above)

예시:

var _str = typeof(global.ExtensionInput);
show_debug_message(" global.ExtensionInput is a " + _str);

위의 코드는 주어진 전역 변수가 보유한 데이터 유형을 가져오고 문자열을 로컬 변수로 반환한 다음 콘솔에 메시지를 출력하는 데 사용됩니다.

뒤로: 변수 함수
다음: bool
