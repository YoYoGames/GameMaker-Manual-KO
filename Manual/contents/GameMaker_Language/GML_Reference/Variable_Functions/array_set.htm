array_set

이 함수를 사용하면 배열의 인덱스 값을 값으로 설정할 수 있습니다. 이 함수를 사용하려면 배열과 설정할 인덱스 및 설정할 값을 보유하는 변수를 제공해야 합니다. 이 함수는 다음 패턴에 따라 배열 인덱스를 제공할 때 설정하려는 차원을 지정하는 한 다차원 배열에도 사용할 수 있습니다.

// 1D array
array_set(my_array, 0, 100);
// 2D array
array_set(my_array[0], 0, 100);
// 3D array
array_set(my_array[0][0], 0, 100);
// etc...

통사론:

array_set(variable, index, value);

논쟁 유형 설명
variable 정렬 The variable that holds the array.
index 진짜 The index of the array element to set the value for.
value 변하기 쉬운 The value to set.

for (var i = 0; i < 10; ++i;)
{
    array_set(score_array, i, i*100));
