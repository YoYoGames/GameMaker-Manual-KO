array_get

이 함수를 사용하면 배열의 인덱스에서 값을 검색할 수 있습니다. 이 함수를 사용하려면 배열을 보유하는 변수와 해당 배열 내에서 값을 가져오기 위한 인덱스를 제공해야 합니다. 주어진 배열 인덱스가 범위를 벗어나면 게임이 오류와 함께 충돌합니다. 이 함수는 다음 패턴에 따라 배열 인덱스를 제공할 때 얻을 차원을 지정하는 한 다차원 배열에도 사용할 수 있습니다.

// 1D array
array_get(my_array, 0);
// 2D array
array_get(my_array[0], 0);
// 3D array
array_get(my_array[0][0], 0);
// etc...

통사론:

array_get(variable, index);

논쟁 유형 설명
variable 정렬 The variable that holds the array.
index 진짜 The index of the array element to get the value from.

보고:

Variable (Any valid data type that an array can hold)

예시:

for (var i = 0; i < 10; ++i;)
{
    show_debug_message(array_get(my_array, i));
}

위의 코드는 콘솔에 주어진 항목의 처음 10개 항목을 출력합니다.
