is_undefined

이 함수는 주어진 변수가 정의되었는지 여부를 반환합니다. 어떤 경우에는 GameMaker 에서 변수가 보유하는 데이터 유형을 확인하고 확인하고 싶을 때 이 기능을 사용합니다. 그것은 반환 true 또는 false 값이 정의되었는지 여부에 따라 다릅니다.

참고: 이 함수는 변수의 존재를 확인하는 데 사용할 수 없습니다. 사용variable_instance_exists() 또는 variable_global_exists() 대신에.

통사론:

is_undefined(n);

논쟁 유형 설명
n 변하기 쉬운 The argument to check.

보고:

Boolean

예시:

var val = ds_map_find_value(map, 13);
if is_undefined(val)
{
    show_debug_message("Map entry does not exist!");
}

위의 코드는 변수 "val"을 확인하여 정의되지 않았는지 여부를 확인하고 정의되지 않은 경우 디버그 메시지를 표시합니다.
