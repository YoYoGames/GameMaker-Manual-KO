array_equals

이 기능을 사용하면 두 배열이 동일한지(동일하거나 동일한지) 확인할 수 있습니다. 확인할 두 개의 배열을 제공하면 함수가 반환됩니다. true 동일한 경우(동등하거나 동일한 경우) 또는 false 그렇지 않은 경우. 이것은 ==를 사용하여 두 배열이 동일한지 확인하는 것과 같지 않습니다. 두 배열이 동일한 값을 보유하는지 확인하지 않고 배열이 동일한 초기 배열을 참조하는지 확인하기 위해서만입니다. 예를 들어:

var a = [1,2,3,4];
var b = [1,2,3,4];

if (a == b)
{
    show_debug_message( "This will never fire, as a and b do not reference the SAME array" );
}

if (array_equals(a, b))
{
    show_debug_message( "This will fire as both arrays contain similar values" );
}

자신과 같지 않을 수 있는 일부 상수가 있어 이 기능이 실패할 수 있습니다. 다음은 예입니다.

if (array_equals([NaN], [NaN]))
{
    show_debug_message( "This will never fire as NaN cannot be equal to itself" );
}

자세한 내용은 평등 표 를 참조하십시오.

통사론:

array_equals(var1, var2);

논쟁 유형 설명
var1 정렬 The first array.
var2 정렬 The second array.

보고:

Boolean

예시:

if !array_equals(inventory_array, item_array)
{
    var len = array_length(inventory_array);
    array_copy(item_array, 0, inventory_array, 0, len);
}

위의 코드는 두 개의 배열이 동일한 값을 보유하고 있는지 확인하고, 그렇지 않은 경우 "inventory_array" 배열의 전체 내용을 "item_array" 배열에 복사합니다.
