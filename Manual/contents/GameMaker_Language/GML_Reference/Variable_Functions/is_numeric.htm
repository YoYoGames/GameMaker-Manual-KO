is_numeric

이 함수는 주어진 변수가 숫자 값(real, int32, int64 또는 boolean)인지 여부를 반환합니다. 어떤 경우에는 GameMaker 의 변수에 숫자 값이 있는지 확인하고 확인하고 싶을 때 이 함수를 사용합니다. 함수가 반환됩니다 true 주어진 입력이 숫자인 경우 false 그렇지 않으면.

통사론:

is_numeric(n);

논쟁 유형 설명
n 변하기 쉬운 The input to check.

보고:

Boolean

예시:

if is_numeric(val)
{
    current_val += val;
}

위의 코드는 "val" 변수를 확인하여 숫자 값인지 확인하고 값이면 지정된 변수에 추가합니다.
