method

이 함수를 사용하면 기존 함수(또는 메서드)를 지정된 인스턴스 또는 구조체에 바인딩하여 나중에 사용할 수 있는 새 메서드 변수 를 만들 수 있습니다.

사용할 인스턴스 ID(객체 인덱스가 아닌 룸에서 활성화된 인스턴스) 또는 구조체 참조와 바인딩하려는 함수 ID(또는 메서드 참조)를 제공합니다. 이 함수는 할당된 변수에서 호출할 수 있는 새 메서드 를 반환합니다(아래 코드 예제 참조).

반환된 메서드는 지정된 인스턴스 또는 구조체에 "바인드"되어 항상 해당 인스턴스/구조체의 범위에서 실행됩니다.

내장 함수와 사용자 정의 함수/메서드를 바인딩할 수 있으며 다음을 제공할 수도 있습니다. undefined 현재를 의미하는 instance/struct 인수로self 범위는 바인딩에 사용됩니다.

통사론:

method(struct_ref_or_instance_id, func);

논쟁 유형 설명
struct_ref_or_instance_id 구조체 또는 인스턴스 ID The unique reference or ID value of the struct or instance to use (can be self or undefined )
func 스크립트 기능 또는 메서드 The ID of the function (or method reference) to use

보고:

Method

예시:

var _inst = instance_position(mouse_x, mouse_y, obj_Enemy);
if instance_exists(_inst)
{
    enemy_func = method(_inst, enemy_ai);
}

위의 코드는 마우스 위치에 적군 인스턴스가 있는지 확인합니다. 그렇다면 적군 인스턴스에 적군 "enemy_func">_ai 메서드가 바인딩되어 새 메서드 변수로 반환됩니다..
