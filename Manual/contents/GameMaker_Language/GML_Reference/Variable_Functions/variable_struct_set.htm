struct_set

이 함수를 사용하면 구조체에서 주어진 변수의 값을 설정할 수 있습니다. 구조체 참조와 값을 문자열로 설정할 변수 이름을 제공한 다음(아래 예제 코드 참조) 마지막으로 설정할 값(유효한 데이터 유형일 수 있음)을 제공합니다. 변수가 구조체에 이미 존재하지 않으면 생성된 다음 값이 할당됩니다.

 

통사론:

struct_set(struct, name, val);

논쟁 유형 설명
struct Struct The struct reference to set
name String The name of the variable to set (as a string)
val Any The value to set the variable to

 

보고:

N/A

 

예시:

if !struct_exists(mystruct, "shields")
{
    struct_set(mystruct, "shields", 0);
}

위의 코드는 주어진 변수가 주어진 구조체에 존재하는지 확인하고 존재하지 않으면 생성되어 0으로 설정됩니다.
