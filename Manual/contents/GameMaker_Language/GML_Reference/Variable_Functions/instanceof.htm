instanceof

이 함수는 구조체를 만드는 데 사용된 생성자 함수 의 이름을 가져오는 데 사용할 수 있습니다. 구조체 자체는 다음을 사용하여 생성되어야 합니다.new 생성자 자체와 함께 연산자. 확인할 구조체 참조와 함께 변수를 제공하면 함수는 생성자 이름이 포함된 문자열을 반환하거나 undefined.

함수에 구조체 리터럴(즉, 생성자 함수를 사용하지 않고 생성된 구조체)을 전달하면 단순히 weak_ref_create()>"struct" 문자열을 반환합니다.

이 함수는 또한 구조체 참조가 약한 참조 인지 확인하는 데 사용할 수 있습니다. 이 경우 함수는 구조체를 생성한 함수 이름 대신 new>"weakref" 문자열을 반환합니다. 자세한 내용은 기능을 참조하십시오..

생성자 함수가 스크립트에서 생성되지 않은 경우 반환된 생성자 이름에는 추가 텍스트가 포함됩니다. 객체 내부에서 생성된 경우 해당 이름에는 객체와 생성된 이벤트가 포함됩니다. 예를 들어 Object1 의 Create 이벤트에서 생성된 test_constructor 라는 생성자는 "test_constructor_gml_Object_Object1_Create_0" 으로 반환됩니다. 이를 방지하려면 script 에서 생성자를 생성하십시오.

통사론:

instanceof(struct_id);

논쟁 유형 설명
struct 구조체 The struct reference to use.

보고:

String or undefined

확장된 예:

이 예제에서는 먼저 구조체의 생성자로 사용할 함수를 정의해야 합니다. 다음 함수는 스크립트 자산에 정의되어 있습니다.

function init_struct(_a, _b, _c) constructor
{
    a = _a;
    b = _b;
    c = _c;
}

이 기능은 다음과 함께 사용할 수 있습니다. 연산자를 사용하여 구조체를 만들고 함수에 사용된 인수 값으로 설정된 변수로 채웁니다.

mystruct = new init_struct(10, 100, "Hello World");

그러면 다음과 같이 사용된 생성자 함수의 이름을 얻을 수 있습니다.

var _name = instanceof(mystruct);
if is_string(_name)
{
    show_debug_message(_name);
}

이렇게 하면 스크립트에 정의된 생성자 함수의 이름인 출력 로그에 문자열 "init_struct" 가 인쇄됩니다.
