bool

이 함수는 주어진 값을 부울 데이터 유형으로 변환하려고 시도합니다. 여기서 값은 다음과 같이 반환됩니다. true 0.5보다 크거나 같으면 false 그렇지 않으면.

통사론:

bool(n);

논쟁 유형 설명
n 진짜 The value to convert.

보고:

Boolean

예시:

if !is_bool(val)
{
    val = bool(val);
}

위의 코드는 변수 "val"을 검사하여 불리언인지 확인하고 그렇지 않으면 1로 변환합니다.
