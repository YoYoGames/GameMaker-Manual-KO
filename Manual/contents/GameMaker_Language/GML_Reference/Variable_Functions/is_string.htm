is_string

이 함수는 주어진 변수가 문자열 번호인지 여부를 반환합니다. 경우에 따라 GameMaker 의 변수에 실제가 아닌 문자열이 있는지 확인하고 확인하고 싶을 때 이 함수를 사용합니다. 문자열을 반환하지 않고 오히려 true 또는 false, 예를 들어 n에 대한 "fish" 값은 다음을 반환합니다. true 하지만 n에 대해 200 값이 반환됩니다. false.

통사론:

is_string(n);

논쟁 유형 설명
n 변하기 쉬운 The argument to check.

보고:

Boolean

예시:

if is_string(val)
{
    name = "Player: " + val;
}

위의 코드는 변수 "val"을 검사하여 문자열이 포함되어 있는지 확인하고 포함되어 있으면 변수 "name"에 할당되는 다른 문자열에 추가합니다.
