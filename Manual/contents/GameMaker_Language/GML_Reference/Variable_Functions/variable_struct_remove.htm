variable_struct_remove

이 함수를 사용하면 구조체에서 변수를 제거할 수 있습니다. 변수를 제거할 구조체 ID와 제거할 변수의 문자열 이름을 제공합니다.

통사론:

variable_struct_remove(struct, name);

논쟁 유형 설명
struct 구조체 The struct reference to remove the variable from
name 끈 The name of the variable to remove (as a string)

보고:

N/A

예시:

if variable_struct_exists(mystruct, "shields")
{
    variable_struct_remove(mystruct, "shields");
}

위의 코드는 주어진 변수가 주어진 구조체에 존재하는지 확인하고 존재한다면 제거됩니다.
