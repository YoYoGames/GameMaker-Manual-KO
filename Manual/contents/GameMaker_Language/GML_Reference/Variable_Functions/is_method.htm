is_method

이 함수는 변수가 메서드 변수인지 확인하는 데 사용할 수 있습니다. true ) 또는 그렇지 않은 경우(이 경우 반환됩니다. false ).

통사론:

is_method(n);

논쟁 유형 설명
n 변하기 쉬운 The variable to check.

보고:

Boolean

예시:

if is_method(get_vec)
{
    show_debug_message("Method variable!");
}

위의 코드는 변수를 검사하여 메서드인지, 그리고 함수가 반환하는지 확인합니다. true, 디버그 메시지가 콘솔에 출력됩니다.
