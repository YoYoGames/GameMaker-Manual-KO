variable_instance_exists

이 기능을 사용하면 인스턴스 범위 변수가 있는지 여부를 확인할 수 있습니다. 고유한 인스턴스 ID 값(룸 편집기의 인스턴스 속성 에서 찾을 수 있거나 함수를 호출할 때 반환됨)을 제공합니다.instance_create_layer() ) 뿐만 아니라 문자열로 확인할 변수 이름(아래 예제 코드 참조). 함수가 반환됩니다 true 인스턴스에 대해 주어진 이름의 변수가 존재하고 false 그렇지 않으면.

통사론:

variable_instance_exists(instance_id, name);

논쟁 유형 설명
instance_id 인스턴스 ID The unique ID value of the instance to check
name 끈 The name of the variable to check for

보고:

Boolean

예시:

if !variable_instance_exists(id, "shields")
{
    shields = 0;
}

위의 코드는 "shields"라는 변수가 코드를 실행하는 인스턴스에 존재하는지 확인하고 존재하지 않으면 생성되어 0으로 초기화됩니다.
