array_insert

이 함수를 사용하면 주어진 위치에서 배열에 값(또는 값)을 삽입할 수 있습니다. 이 함수는 배열을 보유하는 변수, 삽입할 배열의 인덱스(위치) 및 삽입할 최소 하나 의 값을 제공해야 하지만 선택적으로 추가 인수를 제공할 수 있으며 모두 삽입됩니다. 주어진 인덱스에서 연속적인 순서로 배열.

지정된 인덱스가 배열의 길이를 벗어나면 값이 지정된 인덱스에 추가되고 배열의 마지막 값과 삽입된 값 사이의 빈 슬롯은 기본값 0으로 설정됩니다.

통사론:

array_insert(variable, index, value, [value], [value], [etc...]);

논쟁 유형 설명
variable 정렬 The variable that holds the array.
index 진짜 The index (position) in the array to insert the value(s)
value 변하기 쉬운 The value to insert
[value], [value], [etc...] 변하기 쉬운 OPTIONAL Further values to insert into the array

보고:

N/A

예시:

array_insert(score_array, score_rank, current_score);

위의 코드는 현재 점수를 점수 배열에 의해 정의된 인덱스에 삽입합니다. score_rank 변하기 쉬운.
