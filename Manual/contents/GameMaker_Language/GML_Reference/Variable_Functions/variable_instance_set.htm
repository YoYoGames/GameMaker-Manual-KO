variable_instance_set

이 함수를 사용하면 인스턴스에서 주어진 변수의 값을 설정할 수 있습니다. 고유한 인스턴스 ID 값(룸 편집기의 인스턴스 속성 에서 찾을 수 있거나 함수를 호출할 때 반환됨)을 제공합니다.instance_create_layer() ) 뿐만 아니라 값을 문자열로 설정할 변수의 이름(아래 예제 코드 참조), 마지막으로 설정할 값(유효한 데이터 유형일 수 있음). 변수가 인스턴스에 이미 존재하지 않으면 생성된 다음 값이 할당됩니다.

통사론:

variable_instance_set(instance_id, name, val);

논쟁 유형 설명
instance_id 인스턴스 ID The unique ID value of the instance to use
name 끈 The name of the variable to set (as a string)
val 변하기 쉬운 The value to set the variable to

보고:

N/A

예시:

if !variable_instance_exists(id, "shields")
{
    variable_instance_set(id, "shields", 0);
}

위의 코드는 호출하는 인스턴스에 인스턴스 변수가 있는지 확인하고 존재하지 않으면 생성되어 0으로 설정됩니다.
