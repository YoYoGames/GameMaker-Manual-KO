array_push

이 함수를 사용하면 배열의 길이를 알 필요 없이 값(또는 값)을 배열의 끝에 밀어넣을 수 있습니다. 이 함수를 사용하려면 추가 인수를 선택적으로 제공할 수 있으며 모두 연속적인 순서로 배열에 푸시되지만 배열을 보유하는 변수와 푸시할 값을 하나 이상 제공해야 합니다.

통사론:

array_push(variable, value, [value], [value], [etc...]);

논쟁 유형 설명
variable 정렬 The variable that holds the array.
value 변하기 쉬운 The value to push onto the end of the array
[value], [value], [etc...] 변하기 쉬운 OPTIONAL Further values to be pushed onto the array

보고:

N/A

예시:

array_push(score_array, obj_Player1.scr, obj_Player2.scr, obj_Player3.scr, obj_Player4.scr);

위의 코드는 주어진 배열의 끝에 4개의 값을 푸시합니다.
