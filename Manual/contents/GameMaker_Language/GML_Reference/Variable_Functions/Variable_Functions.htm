변수 함수

이 페이지의 함수는 게임에서 GameMaker Language를 사용할 때 모든 다양한 변수와 변수 유형을 처리하도록 설계되었습니다. 변수에 대한 자세한 내용은 변수 및 변수 범위 페이지를 참조하십시오.

다음 변수 기능을 사용할 수 있습니다.

  • variable_instance_exists
  • variable_instance_get_names
  • variable_instance_names_count
  • variable_instance_get
  • variable_instance_set
  • variable_global_exists
  • variable_global_get
  • variable_global_set
  • variable_struct_exists
  • variable_struct_get
  • variable_struct_set
  • variable_struct_remove
  • variable_struct_get_names
  • variable_struct_names_count
  • instanceof

배열 사용과 관련된 몇 가지 기능도 있습니다. 이것들은 코드의 유연성을 제공하도록 설계되었으며 게임을 만들 때 더 기능적이고 동적인 배열을 만들 수 있습니다. 이러한 기능은 다음과 같습니다.

  • array_create
  • array_copy
  • array_equals
  • array_get
  • array_set
  • array_push
  • array_pop
  • array_insert
  • array_delete
  • array_sort
  • array_length
  • array_resize
  • array_length_1d( 더 이상 사용되지 않음 )
  • array_length_2d( 더 이상 사용되지 않음 )
  • array_height_2d( 더 이상 사용되지 않음 )

다음 함수는 모두 메서드 변수 사용과 관련이 있습니다.

  • method
  • method_get_self
  • method_get_index

마지막으로 변수 유형을 결정하기 위한 다음 함수도 있습니다(데이터 유형에 대한 자세한 내용은 여기, 유형 테이블 은 여기 참조).

  • is_string
  • is_real
  • is_numeric
  • is_bool
  • is_array
  • is_struct
  • is_method
  • is_ptr
  • is_int32
  • is_int64
  • is_vec3
  • is_vec4
  • is_matrix
  • is_undefined
  • is_nan
  • is_infinity
  • typeof
  • bool
  • ptr
  • int64
