method_get_index

이 함수를 사용하면 메서드가 정의된 스크립트에 대한 스크립트 인덱스를 검색할 수 있습니다. 메서드가 스크립트에 정의되지 않은 경우 함수는 -1을 반환하고, 그렇지 않으면 스크립트의 인덱스 값을 반환합니다.

통사론:

method_get_index(method);

논쟁 유형 설명
method 방법 The method variable to check

보고:

Script Asset or Real

예시:

var _index = method_get_index(light_setup);
if _index != -1
{
    show_debug_message(script_get_name(_index));
}

위의 코드는 "light_setup" 메서드에 대한 스크립트 인덱스를 확인한 다음 -1이 아니면 스크립트 이름을 콘솔에 출력합니다.
