array_pop

이 함수는 주어진 배열의 마지막 요소를 제거하고 그 값을 반환합니다. 배열이 비어 있으면,undefined 반환됩니다.

통사론:

array_pop(array);

논쟁 유형 설명
array The array to remove the last value from.

보고:

Variable or undefined

예시:

var _lastscore = array_pop(score_array);
draw_text(32, 32, "Last Score = " + string(_lastscore));

위의 코드는 주어진 배열에서 마지막 값을 팝한 다음 일부 텍스트와 함께 화면에 그립니다. 검색된 항목은 점수 배열에서도 제거됩니다.
