배열 정렬

이 함수를 사용하면 배열을 오름차순 또는 내림차순으로 정렬하거나 사용자 정의 함수를 사용하여 정렬 순서를 정의할 수 있습니다. 이 함수를 사용하려면 정렬할 배열을 제공하고 다음 중 하나를 제공해야 합니다.

  • 상수 true 또는 false 오름차순(true ) 또는 내림차순(false ) 정렬 순서
  • 정렬 순서를 정의하는 사용자 정의 함수

이 함수는 제공된 원래 배열의 내용을 수정합니다. 정렬된 복사본을 만들려면 다음을 사용하십시오. array_copy() 먼저 배열을 복제한 다음 이 함수에 전달합니다.

사용자 정의 기능 사용

정렬을 위해 사용자 정의 함수를 사용하는 경우 현재 요소와 다음 요소의 값을 각각 수신하는 2개의 인수를 취해야 합니다. 함수는 정렬 순서에 영향을 미치기 위해 다음 값 중 하나를 반환해야 합니다(여기서elm1 현재 요소이고 elm2 다음 요소임):

  • 0: 만약 elm1 그리고 elm2 같다
  • <= -1 (음의 정수): 만약 elm1 전에 간다 elm2
  • >= 1 (양의 정수): 만약 elm1 뒤를 쫓다 elm2

참고 사용자 정의 함수에서 반환된 값은 정수여야 합니다. 1보다 작은 부동 소수점 값은 0으로 읽히므로 다음을 사용해야 합니다.sign() 또는round() 함수가 부동 소수점 값으로 작동하는 경우.

배열에 문자열 집합이 포함된 경우 기본 오름차순/내림차순 정렬 유형을 사용할 때 문자열은 영어 알파벳을 기준으로 알파벳순으로 정렬됩니다. 다른 모든 데이터 유형은 숫자 값을 기반으로 정렬되며 정확한 값은 데이터 유형 자체에 따라 다릅니다(예: 버퍼 배열은 메모리 주소의 숫자 값을 기반으로 정렬됨).

통사론:

array_sort(variable, sorttype_or_function);

논쟁 유형 설명
variable 정렬 The variable that holds the array.
sorttype_or_function 부울 또는 스크립트 함수 / 메서드 The sort type ( truefor ascending or false for descending) or a function reference to use for sorting.

예시:

var _a = [10, 9, 8, 7, 6, 5];

array_sort(_a, function(elm1, elm2)
{
    return elm1 - elm2;
});

위의 코드는 사용자 정의 정렬 기능을 사용하여 배열을 오름차순으로 정렬합니다. 정렬 함수는 두 개의 배열 요소를 다음과 같이 취합니다. elm1 그리고 elm2 그리고 그것들에 대해 빼기를 수행합니다. 만약에 elm1 보다 크다 elm2, 빼기는 양수가 됩니다. elm1 이후에 elm2, 따라서 오름차순으로 배치합니다.
