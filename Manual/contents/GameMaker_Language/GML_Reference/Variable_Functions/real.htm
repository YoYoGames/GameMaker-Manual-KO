real

이 함수는 주어진 문자열을 실수로 바꾸는 데 사용할 수 있습니다. 이 기능을 사용할 때 문자열의 숫자, 빼기 기호, 소수점 및 지수 부분이 고려되는 반면 다른 문자(예: 문자)는 오류를 발생시킵니다. 문자열에 다른 문자가 있을 수 있다는 것을 알고 있거나 의심되는 경우 다음을 사용할 수 있습니다.string_digits() 이 함수를 사용하여 결과 문자열을 실수로 바꾸기 전에 숫자가 아닌 모든 문자를 제거합니다.

 

통사론:

real(string)

논쟁 유형 설명
string String The string to be converted to a real value.

 

보고:

Real

 

예시:

var t_str = string_digits(input_str);
age = real(t_str);

위의 코드는 입력 문자열을 받아 숫자 이외의 모든 문자를 제거한 다음 반환 문자열의 실수 값을 유지하도록 변수 "age"를 설정합니다.
