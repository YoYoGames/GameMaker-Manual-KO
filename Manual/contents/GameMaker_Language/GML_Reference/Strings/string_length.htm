string_length

이 함수는 주어진 문자열을 구성하는 문자 수를 반환합니다. 사용자 정의 텍스트 항목의 문자 길이를 제한할 때와 같은 작업에 유용할 수 있습니다(예: 플레이어 이름을 10자로 제한). 이것은 다르다는 것을 기억하십시오.string_width() 픽셀 단위로 화면에 그려진 너비가 아니라 문자열의 문자 수를 측정한다는 점에서.

통사론:

string_length(string);

논쟁 유형 설명
string 끈 The string to measure the number of characters of.

보고:

Real

예시:

if (string_length(name) > 10)
{
    name = string_copy(name, 1, 10);
}

이렇게 하면 name 문자열이 10자보다 큰지 확인하고, 그렇다면 처음 10자만 복사하여 사용합니다.
