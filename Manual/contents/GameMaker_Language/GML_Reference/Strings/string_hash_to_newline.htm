string_hash_to_newline

이 함수는 "#" 기호가 새 줄로 변환된 문자열을 반환합니다. 따라서 다음과 같은 형식의 문자열:

"Hello#World"

다음과 같이 화면에 인쇄됩니다.

Hello
World

중요한! 이 기능은 GameMaker 와 소프트웨어의 이전 버전 간의 가져오기 호환성을 위해 제공되며 따라서 더 이상 사용되지 않는 기능을 복제해야 하는 경우를 제외하고는 이 기능을 사용해서는 안 됩니다. 대신 새 줄을 표시하려면 이스케이프 문자 를 사용해야 합니다. 가져오기 호환성에 대한 자세한 내용은 여기를 참조하십시오.

통사론:

string_hash_to_newline(string);

논쟁 유형 설명
string The string to convert over multiple lines

보고:

예시:

var str = string_hash_to_newline("Hello#World");
draw_text(32, 32, str);

위의 코드는 해시 기호가 있는 문자열을 두 줄로 분할된 문자열로 변환한 다음 화면에 출력합니다.

뒤: 스트링
다음: clipboard_has_text
