ansi_char

이 함수는 원시 BYTE 값이 설정된 문자를 포함하는 문자열을 반환합니다. 이것은 표시되지 않으며 표시 되어서도 안 되지만 인코딩에 사용할 수 있도록 디스크에 올바르게 저장됩니다.

통사론:

ansi_char(val);

논쟁 유형 설명
val The raw byte value.

보고:

(Single character)

예시:

var str1 = ansi_char($EF);
var str2 = ansi_char($BB);
var str3 = ansi_char($BF);
file_text_write_string(global.saveFile, str1 + str2 + str3);

위의 코드는 원시 바이트 데이터에서 문자열을 만들어 (이전에 열었던) 파일에 씁니다.
