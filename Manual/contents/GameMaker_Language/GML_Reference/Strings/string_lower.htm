string_lower

이 함수를 사용하면 문자열에 소문자만 포함되도록 강제할 수 있습니다.

참고 이 기능은 A - Z의 26자 영문자만 감지합니다.

통사론:

string_lower(string);

논쟁 유형 설명
string The string to convert to lowercase.

보고:

예시:

str1 = "Hello World";
str2 = string_lower(str1);

위의 코드가 설정됩니다 str2 "안녕하세요 세계".
