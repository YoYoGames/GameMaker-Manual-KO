ord

이 함수는 단일 문자 입력 문자열을 사용하고 해당 문자에 대한 유니코드(UTF8) 값을 반환합니다. 와 함께 사용할 때 주의하십시오.keyboard_check* 함수에서 입력 문자열은 길이가 한 문자만 가능하고 0에서 9 사이의 숫자 또는 A에서 Z까지 대문자 로 된 로마 문자만 될 수 있습니다.

통사론:

ord(string);

논쟁 유형 설명
string 끈 The string with which to find the Unicode value.

보고:

Real

예시:

if keyboard_check(ord("W"))
{
    y -= 4;
}

W 키를 누르고 있으면 호출하는 인스턴스가 위로 4픽셀 이동합니다.
