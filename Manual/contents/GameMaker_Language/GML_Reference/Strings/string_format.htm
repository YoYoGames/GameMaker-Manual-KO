string_format

사용자 고유의 형식을 사용하여 실수를 문자열로 변환합니다. 여기서 문자열에 저장되는 "자리" 수와 저장되는 소수 자릿수도 선택할 수 있습니다. 둘 다 매우 편리할 수 있으며 일부 게임에서는 점수를 설정된 자릿수로 표시하는 것을 선호하는 반면 소수점 이하 자릿수에 대한 제어는 소수점 이하 두 자리의 높은 정확도에 유용할 수 있습니다.string() 제공할 수 없습니다. 지정된 자릿수가 표시할 값보다 크거나 지정된 수에 소수점 이하 자릿수를 더한 값이 전체 자릿수보다 작은 경우 차이를 채우기 위해 값 앞에 공백이 추가됩니다(참조 아래 예). 주어진 값이 합계 및 포함할 소수점 이하 자릿수보다 작으면 소수점 오른쪽에 0이 추가됩니다. 기본 형식은 왼쪽에 공백이 없고 오른쪽에 소수점 이하 두 자리만 있습니다(예: "265.73").

통사론:

string_format(val, tot, dec);

논쟁 유형 설명
val 진짜 The real number to be turned into a string.
tot 진짜 The total number of places of the main number to be shown. Zeroes or spaces will be inserted to match this.
dec 진짜 The number of decimal places to be included.

보고:

String

예시:

str1 = string_format(1234, 8, 0);
str2 = string_format(pi, 1, 10);
str3 = string_format(pi, 5, 5);

이것은 설정합니다 str1 "1234"로, str2 "3.1415926535"로 str3 "3.14159"로.
