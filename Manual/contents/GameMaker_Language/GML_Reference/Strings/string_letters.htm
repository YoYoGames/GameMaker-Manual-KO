string_letters

이 기능을 사용하면 문자로 분류되지 않은 모든 문자를 제거할 수 있습니다. 이것은 사람들이 텍스트 항목에 원치 않는 문자를 입력하는 것을 방지하는 데 유용합니다(예: 강제로 새 줄을 입력하기 위해 "#" 입력).

참고 이 기능은 A - Z(대문자 또는 소문자)의 26자 영문자만 감지하여 반환합니다.

통사론:

string_letters(string);

논쟁 유형 설명
string The string to convert to letters.

보고:

예시:

username = string_letters(username);

위의 코드는 "username" 변수가 원래 문자열의 문자 만 포함하도록 설정합니다(id는 원래 문자열이 "MacSw33ny"임). 반환된 문자열은 "MacSwny"가 됩니다.
