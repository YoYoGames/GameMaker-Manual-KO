string_set_byte_at

이 함수는 문자열(UTF8 형식 기반)에 직접 바이트를 설정하고 변경 사항이 있는 문자열의 복사본을 반환합니다.

참고 이 기능은 매우 느리므로 필요한지 여부와 사용 위치를 신중하게 고려하십시오.

통사론:

string_set_byte_at(str, pos, byte);

논쟁 유형 설명
str The string to change the byte of.
pos The position within the string (starting at 1) to change the byte of.
byte The new byte value.

보고:

예시:

str = string_set_byte_at("hello", 2, 97);

위의 코드는 문자열에서 두 번째 문자의 바이트 값을 변경하므로 변수를 설정합니다. str "안녕"을 개최합니다.
