string_byte_length

이 함수는 문자열의 바이트 수를 반환합니다. UTF8로 유지되기 때문에 문자열 의 문자 수와 같지 않습니다.

통사론:

string_byte_length(string);

논쟁 유형 설명
string 끈 The string to measure the number of bytes of.

보고:

Real

예시:

bytesize = string_byte_length("Hello World");

이것은 "Hello World"의 바이트 수로 bytesize를 설정합니다.
