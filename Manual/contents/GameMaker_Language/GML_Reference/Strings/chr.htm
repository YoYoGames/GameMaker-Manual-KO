chr

이 함수는 표시할 입력 유니코드 코드와 관련된 문자를 포함하는 문자열을 반환합니다. 이 문자는 현재 드로잉 글꼴 문자 집합 코드 페이지에 따라 달라지며 글꼴이 설정되지 않은 경우 시스템의 기본 코드 페이지를 사용합니다.

통사론:

chr(val);

논쟁 유형 설명
val The Unicode code value to get the string from

보고:

예시:

mystring = chr(53) + chr(48);

이것은 설정합니다 mystring chr(53)이 "5"이고 chr(48)이 "0"이므로 "50"(정수가 아닌 문자열)으로 변환합니다.
