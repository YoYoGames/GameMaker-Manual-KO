string_pos_ext

이 함수는 문자열 내에서 하위 문자열 인스턴스의 문자 위치를 반환하고 시작 위치로 지정된 위치에서 문자열을 통해 앞으로 검색합니다. 이 함수는 검색 문자열이 없으면 0을 반환하고 검색 문자열이 있는 경우 검색 문자열의 첫 번째 문자 위치를 반환합니다. 레거시 지원 문자열의 경우 1부터 인덱싱되므로 예상한 대로 0이 아니라 1이 문자열의 첫 번째 위치에 있다는 점을 염두에 두십시오.

통사론:

string_pos_ext(substr, str, start_pos);

논쟁 유형 설명
substr The substring to look for in the string.
str The string.
start_pos The starting position to search from.

보고:

예시:

if (string_pos_ext(",", text, 20) != 0)
{
    string_insert(name, text, string_pos_ext(",", text, 20));
}

위의 코드는 변수 "text"에 저장된 문자열에서 20번째 문자 뒤의 쉼표를 검색하고 발견하면 해당 위치에 하위 문자열 "name"을 삽입합니다.
