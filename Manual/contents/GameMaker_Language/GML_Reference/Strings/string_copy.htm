string_copy

이 기능을 사용하면 문자열 내에서 다른 문자로 복사할 여러 문자를 쉽게 선택할 수 있습니다. 문자열의 첫 번째 문자는 예상한 대로 항상 0이 아닌 1로 인덱싱되므로 (예를 들어) 문자열의 처음 5개 문자를 복사하려면 string_copy(str, 1, 5). 아래에 추가 예가 제공됩니다.

통사론:

string_copy(str, index, count);

논쟁 유형 설명
str The string to copy from.
index The position of the first character in the string to copy from (numbered from 1)
count The number of characters, starting from the position of the first, to copy

보고:

예시:

name = keyboard_string;
if (string_length(name) > 15)
{
    keyboard_string = string_copy(name, 1, 15);
}

위의 코드를 사용하면 플레이어가 키보드를 통해 문자열을 입력할 수 있으며 이 문자열은 변수에 저장됩니다. name. 그런 다음 이 변수를 검사하여 최대 길이인 15자를 초과하는지 확인하고 초과할 경우 keyboard_string을 15자 복사본으로 대체합니다. name 끈. 이것은 플레이어 입력을 15자로 효과적으로 제한합니다.
