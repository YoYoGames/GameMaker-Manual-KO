string_last_pos

이 함수는 문자열 내에서 하위 문자열 인스턴스의 문자 위치를 반환하고 문자열의 끝에서 시작까지 검색합니다(따라서 역순string_pos(). 이 함수는 검색 문자열이 없으면 0을 반환하고 검색 문자열이 있는 경우 검색 문자열의 첫 번째 문자 위치를 반환합니다. 레거시 지원 문자열의 경우 1부터 인덱싱되므로 예상한 대로 0이 아니라 1이 문자열의 첫 번째 위치에 있다는 점을 염두에 두십시오.

통사론:

string_last_pos(substr, str);

논쟁 유형 설명
substr The substring to look for in the string.
str The string.

보고:

예시:

if (string_last_pos(",", text) != 0)
{
    string_insert(name, text, string_last_pos(",", text));
}

위의 코드는 "text" 변수에 저장된 문자열에서 문자열의 끝에서 검색을 시작하는 쉼표를 검색합니다. 하나를 찾으면 해당 위치에 하위 문자열 "name"을 삽입합니다.
