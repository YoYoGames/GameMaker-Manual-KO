string_upper

이 함수를 사용하면 문자열에 대문자만 포함하도록 강제할 수 있습니다.

참고 이 기능은 A - Z의 26자 영문자만 감지합니다.

통사론:

string_upper(string);

논쟁 유형 설명
string The string to convert to uppercase.

보고:

예시:

str1 = "Hello World";
str2 = string_upper(str1);

위의 코드는 변수를 설정합니다 str2 "HELLO WORLD"로.
