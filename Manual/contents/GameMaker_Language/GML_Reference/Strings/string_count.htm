string_count

이 함수는 지정된 하위 문자열이 특정 문자열 내에 나타나는 횟수를 반환합니다. 이런 식으로 저장된 텍스트의 섹션에서 단일 문자 또는 구가 몇 번이나 반복되는지 확인할 수 있습니다.

통사론:

string_count(substr, str);

논쟁 유형 설명
substr The substring to check the string for.
str The string to check.

보고:

예시:

str1 = "Hello World";
ocount = string_count( "o", str1 );

이것은 변수를 설정합니다 ocount "o"의 수로 str1, 이 경우 2.
