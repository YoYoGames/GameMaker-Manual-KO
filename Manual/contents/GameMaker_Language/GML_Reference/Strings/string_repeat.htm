string_repeat

이 함수는 단순히 자신에 대해 주어진 횟수만큼 반복되는 동일한 문자열을 반환합니다.

통사론:

string_repeat(str, count);

논쟁 유형 설명
str The string to repeat.
count The number of times to repeat the string.

보고:

예시:

str1 = "Hello World";
str2 = string_repeat(str1, 3);

위의 코드가 설정됩니다 str2 "Hello WorldHello WorldHello World"로 이동합니다.
