clipboard_has_text

이 함수는 반환됩니다 true 클립보드에 텍스트가 포함되어 있거나 false 그렇지 않은 경우.

통사론:

clipboard_has_text();

보고:

Boolean

예시:

if clipboard_has_text()
{
    str = clipboard_get_text();
    clipboard_set_text("");
}

위의 코드는 클립보드에 텍스트가 있는지 확인하고 포함되어 있으면 "str" 변수에 문자열로 읽어들입니다. 마지막으로 클립보드는 빈 문자열로 설정하여 지워집니다.
