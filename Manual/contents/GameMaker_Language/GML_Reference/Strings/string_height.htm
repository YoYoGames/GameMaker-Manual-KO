string_height

이 함수는 줄 구분과 텍스트에 있을 수 있는 줄 바꿈을 고려하여 입력 문자열의 높이(픽셀 단위)를 반환합니다. 문자열을 구성하는 문자 중 가장 긴 문자를 기준으로 텍스트 요소 사이의 거리를 계산하는 데 매우 편리합니다.draw_text() 현재 정의된 글꼴을 사용합니다.

통사론:

string_height(string);

논쟁 유형 설명
string The string to measure the height of.

보고:

예시:

var hh = string_height("ABCDEFGHIJKLMNOPQRSTUVWXYZ");
draw_text(32, 32, string(score));
draw_text(32, 32 + hh, string(lives);

위의 코드는 주어진 문자열의 높이를 얻은 다음 반환된 문자열 높이를 구분 기호로 사용하여 두 줄의 텍스트를 그립니다.
