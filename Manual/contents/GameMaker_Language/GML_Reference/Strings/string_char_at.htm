string_char_at

이 함수를 사용하여 문자열 내의 특정 위치에 있는 특정 문자를 반환할 수 있으며 인덱스는 첫 번째 문자에 대해 1부터 시작합니다. 문자가 없거나 문자열이 지정된 인덱스 값보다 짧으면 빈 문자열 str2>"" 이 반환되지만 지정된 인덱스가 0보다 작거나 같으면 문자열의 첫 번째 문자가 반환됩니다.

통사론:

string_char_at(str, index);

논쟁 유형 설명
str The string to check.
index The position to get the character from.

보고:

예시:

str1 = "Hello World";
str2 = string_char_at(str1, 7);

이것은 설정합니다 str1 문자열의 7번째 문자("H"는 1번째로 계산)까지, 이 경우 "W".
