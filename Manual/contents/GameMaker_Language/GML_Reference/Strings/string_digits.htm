string_digits

이 함수를 사용하여 주어진 문자열을 구문 분석하고 그로부터 임의의 숫자를 얻을 수 있습니다. 예를 들어 "나는 81세입니다"라는 텍스트가 있다고 가정해 보겠습니다. 이 함수를 사용하면 "81"의 반환 문자열을 얻을 수 있습니다.

통사론:

string_digits(string);

논쟁 유형 설명
string The string to get the digits from.

보고:

예시:

var t_str = string_digits(input_str);
age = real(t_str);

위의 코드는 입력 문자열을 가져 와서 숫자 이외의 모든 문자를 제거한 다음 변수를 설정합니다. age 반환 문자열의 실수 값을 유지하기 위해(따라서 입력 문자열이 - 예를 들어 - "I am 18"인 경우 함수는 "18"을 반환함).
