string

이 함수를 사용하면 실수를 문자열로 바꿀 수 있습니다. 실수가 정수이면 소수점 이하 자릿수 없이 저장되고, 그렇지 않으면 소수점 이하 두 자리로 저장됩니다. 더 많은 소수 자릿수가 필요한 경우 함수를 사용하십시오.string_format(). 또한 배열, 데이터 구조 또는 구조체 를 저장하는 변수에 이 함수를 사용하면 이러한 변수의 내용을 문자열로 변환한 다음 콘솔에 출력하거나 디버깅을 위해 파일에 저장할 수 있습니다.

통사론:

string(val);

논쟁 유형 설명
val 진짜 The real number to be turned into a string.

보고:

String

예시:

draw_text(100, 100, "Score: " + string(score) + " / Health: " + string(health));

위의 코드는 string 실수와 문자열을 함께 그리는 함수(draw는 문자열 이나 실수 중 하나만 허용하지만 둘 다 허용하지는 않음).
