clipboard_set_text

이 함수는 정의된 문자열을 보유하도록 클립보드를 설정합니다. 텍스트의 클립보드를 효과적으로 지우기 위해 빈 문자열 "" 로 설정할 수 있습니다.

통사론:

clipboard_set_text(string);

논쟁 유형 설명
string 끈 The text (as a string) to set the clipboard to hold.

보고:

N/A

예시:

if clipboard_has_text()
{
    str = clipboard_get_text();
    clipboard_set_text("");
}

위의 코드는 클립보드에 텍스트가 있는지 확인하고 포함되어 있으면 "str" 변수에 문자열로 읽어들입니다. 마지막으로 클립보드는 빈 문자열로 설정하여 지워집니다.
