string_ord_at

이 함수를 사용하여 문자열 내의 특정 위치에 있는 특정 문자 코드를 반환할 수 있으며 인덱스는 첫 번째 문자에 대해 1에서 시작합니다. 문자가 없거나 문자열이 인덱스에 지정된 값보다 짧으면 -1이 반환됩니다.

통사론:

string_ord_at(str, index);

논쟁 유형 설명
str The string to check.
index The position to get the character code from.

보고:

예시:

str = "Hello World";
char_code = string_ord_at(str, 7);

이것은 문자열 "str"에서 일곱 번째 문자(여기서 "H"는 첫 번째로 계산됨)에 대한 문자 코드를 가져와 변수 "char_code"에 저장합니다.
