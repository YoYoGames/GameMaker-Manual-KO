string_insert

이 함수를 사용하면 두 개의 문자열로 구성된 새 문자열을 만들 수 있습니다. 여기서 하나는 지정된 위치에서 다른 하나에 삽입됩니다. 예를 들어 미리 정의된 텍스트에 사용자 이름을 추가하여 게임 플레이어가 작업에 더 많이 참여하게 만드는 데 유용할 수 있습니다. 해당 문자열에 삽입할 위치를 계산할 때 1부터 인덱싱되므로 문자열의 첫 번째 문자는 예상대로 0이 아니라 1입니다.

통사론:

string_insert(substr, str, index);

논쟁 유형 설명
substr The substring to be inserted.
str The string to be copied.
index The position in characters of the string to insert the substring.

보고:

예시:

str2 = string_insert(username, "Hello, , how are you?", 8);

이렇게 하면 "username" 변수의 문자열이 주어진 구문에 삽입되고 결과 문자열은 "Hello, NAME, how are you?"와 같이 표시됩니다.
