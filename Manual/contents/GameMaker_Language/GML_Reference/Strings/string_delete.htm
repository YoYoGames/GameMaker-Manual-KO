string_delete

이 함수는 주어진 문자열의 일부를 제거하는 데 사용됩니다. 입력 문자열, 해당 문자열 내에서 문자를 제거할 시작 위치(색인은 1에서 시작) 및 제거할 문자 수를 제공합니다. 함수는 해당 부분이 없는 새 문자열을 반환합니다.

통사론:

string_delete(str, index, count);

논쟁 유형 설명
str The string to copy and delete from.
index The position of the first character to remove (from 1).
count The number of characters to remove.

보고:

String

예시:

str1 = "Helloo World";
str2 = string_delete(str1, 5, 1);

이것은 설정합니다 str2 "Hello"(위치 5에 있음)에서 추가 "o"를 제거하므로 "Hello World"로 변경합니다.
