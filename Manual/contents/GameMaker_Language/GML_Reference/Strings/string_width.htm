string_width

이 함수는 텍스트에 있을 수 있는 줄 바꿈을 고려하여 입력 문자열의 너비(픽셀)를 반환합니다. 문자열을 구성하는 문자의 총 너비를 기준으로 텍스트 요소 사이의 거리를 계산할 때 매우 편리합니다.draw_text() 현재 정의된 글꼴을 사용합니다.

통사론:

string_width(string);

논쟁 유형 설명
string The string to measure the width of.

보고:

예시:

var ww = string_width(str_Name + " ");
draw_text(32, 32, str_Name));
draw_text(32 + ww, 32, "has won the game!");

위의 코드는 주어진 문자열의 너비를 가져온 다음 반환된 문자열 너비를 구분 기호로 사용하여 두 줄의 텍스트를 그립니다.
