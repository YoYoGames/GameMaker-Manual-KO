browser_height

이 변수는 게임이 실행되는 브라우저의 높이(픽셀 단위)를 유지합니다. 브라우저가 없으면 창 크기가 반환됩니다. 다음 이미지는 이 변수가 Window Functions 를 통해 사용 가능한 나머지 위치 지정 기능과 어떻게 관련되는지 보여줍니다.

통사론:

browser_height

보고:

예시:

if (browser_height > window_get_height()) || (browser_width > window_get_width())
{
    var xx, yy;
    if (browser_width > window_get_width())
    {
        xx = (browser_width - window_get_width()) / 2;
    }
    else
    {
        xx = 0;
    }
    if (browser_height > window_get_height())
    {
        yy = (browser_height - window_get_height()) / 2;
    }
    else
    {
        yy = 0;
    }
    window_set_position(xx, yy);
}

위의 코드는 브라우저의 너비와 높이를 검사하여 현재 창 너비와 높이보다 큰지 확인합니다. 둘 중 하나가 있으면 브라우저 내에서 게임 창의 새 위치를 계산합니다.
