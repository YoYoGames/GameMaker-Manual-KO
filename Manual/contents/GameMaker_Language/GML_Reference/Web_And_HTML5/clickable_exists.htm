clickable_exists

이 함수는 클릭 가능한 DOM 아이콘이 지정된 인덱스로 생성되었는지 여부를 반환합니다. 확인에 사용된 값 은 이전에 초기화되어야 하며 그렇지 않으면 GameMaker 를 종료하는 오류가 발생합니다.

통사론:

clickable_exists(index);

논쟁 유형 설명
index 클릭 가능한 ID The index of the clickable to check.

보고:

Boolean

예시:

if !clickable_exists(home_but)
{
    home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");
}

위 코드는 "home_but" 변수가 이미 존재하는지 확인하고, 존재하지 않으면 게임 캔버스가 실행 중인 페이지의 위치(32, 32)에 클릭 가능한 DOM 아이콘을 생성합니다. 아이콘은 텍스처 페이지에서 "spr_MS_Home"으로 참조된 스프라이트를 사용하며 아이콘을 클릭하면 해당 창에 대해 정의된 속성과 지정된 URL에 대한 새 창이 열립니다.
