clickable_add_ext

이 기능을 사용하면 게임 캔버스 외부에서 브라우저 창 자체 내에서 "버튼" 역할을 하는 사용자 지정 클릭 가능한 아이콘을 만들 수 있습니다. 이 함수는 새 클릭 가능 항목을 참조하는 모든 추가 버튼 기능에서 사용할 변수에 저장해야 하는 ID 값을 반환합니다. 이 기능은 모두 GameMaker 자체를 통해 수행되기 때문에 버튼을 클릭할 때 브라우저에서 팝업 창을 생성하지 않도록 하는 데 매우 유용합니다.

이 기능은 다음과 정확히 동일하게 작동합니다.clickable_add() 함수에서 브라우저 창의 (0,0)을 기준으로 위치를 설정해야 하는 경우 버튼 아이콘에 표시하려는 스프라이트에 대한 텍스처 페이지 항목을 제공합니다(참조: sprite_get_tpe() ), 열거나 실행할 스크립트( GameMaker 의 자산 브라우저에서)를 지정한 다음 사용할 대상과 매개변수를 설정합니다. 그러나 이 함수에는 아이콘 알파를 0(완전 투명)에서 1(완전 불투명)로 설정할 수 있는 두 가지 추가 옵션과 아이콘 크기(사용된 스프라이트에 적용되는 승수 값)가 있습니다. 따라서 값 1은 기본 배율이고 (예를 들어) 0.5는 절반 배율이고 2는 두 배입니다.

전달한 URL 인수가 스크립트 함수 의 이름이고(예: "http://"로 시작하지 않음) 스크립트가 호출되는 경우 gmcallback_* 그러면 html5 이벤트에 직접 피드를 가져올 수 있습니다. GameMaker 스크립트 함수 이름은 다음으로 시작하는 경우 난독화 되지 않습니다. gmcallback_.

통사론:

clickable_add_ext(x, y, tpe, url, target, params, alpha, scale);

논쟁 유형 설명
x 진짜 The x position within the window.
y 진짜 The y position within the window.
tpe 텍스처 페이지 항목 The texture page entry for the sprite to be used.
url 끈 The URL (website address) to link to, or the script to be run.
target 끈 This is the target area to open the URL in (see description).
params 끈 The various parameters for the icon (see description).
alpha 진짜 The image alpha of the icon (default 1).
scale 진짜 The scale of the icon (default 1).

보고:

Clickable ID

예시:

home_but = clickable_add_ext(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 1, 2);

위의 코드는 게임 캔버스가 실행되고 있는 페이지의 (32, 32) 위치에 클릭 가능한 DOM 아이콘을 생성합니다. 아이콘은 텍스처 페이지에서 "spr_MS_Home"으로 참조된 스프라이트를 사용하며 아이콘을 클릭하면 해당 창에 대해 정의된 속성과 지정된 URL에 대한 새 창이 열립니다. 버튼은 또한 원래 스프라이트 크기의 두 배와 이미지 알파 1의 절반으로 조정됩니다.
