url_open_ext

그러면 선택한 대상 장치의 브라우저에서 지정된 URL이 열리거나 HTML5 모듈을 사용하는 경우 현재 열려 있는 브라우저에서 열릴 것입니다. 지정하는 "target" 매개변수는 사용할 때 표준 JavaScript "name" 값과 동일합니다. open() 방법을 사용하고 " _self "를 제외한 모든 경우 브라우저가 차단되거나 사용자에게 허용 여부를 묻는 결과가 나타날 수 있습니다.

유효한 대상은 다음과 같습니다.

표적 설명
_공백 링크된 문서를 새 창이나 탭에서 엽니다(팝업이 사용자에 의해 차단된 경우 작동하지 않습니다. 이 경우 다음을 사용할 수 있습니다.clickable_* 대신 기능).
_본인 클릭한 것과 동일한 프레임에서 링크된 문서를 엽니다(기본값).
_부모의 상위 프레임에서 링크된 문서를 엽니다.
_맨 위 창의 전체 본문에서 연결된 문서를 엽니다.

통사론:

url_open_ext(url, target);

논쟁 유형 설명
url 끈 The URL (website address) to link to.
target 끈 This is the target area to open the URL in.

N/A

예시:

url_open_ext("http://gamemaker.io", "_blank");

그러면 새 창에서 GameMaker 홈페이지가 열립니다.
