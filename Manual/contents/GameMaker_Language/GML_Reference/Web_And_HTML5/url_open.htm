url_open

그러면 선택한 대상 장치의 브라우저에서 지정된 URL이 열리거나 HTML5 모듈을 사용하는 경우 현재 열려 있는 브라우저에서 열릴 것입니다.

통사론:

url_open(url);

논쟁 유형 설명
url 끈 The URL (website address) to link to.

보고:

N/A

예시:

url_open("http://gamemaker.io");

이렇게 하면 현재 창에서 GameMaker 홈페이지가 열립니다.
