clickable_add

이 기능을 사용하면 게임 캔버스 외부에서 브라우저 창 자체 내에서 "버튼" 역할을 하는 사용자 지정 클릭 가능한 아이콘을 만들 수 있습니다. 이 함수는 새 클릭 가능 항목을 참조하는 모든 추가 버튼 기능에서 사용할 변수에 저장해야 하는 ID 값을 반환합니다. 이 기능은 모두 GameMaker 자체를 통해 수행되기 때문에 버튼을 클릭할 때 브라우저에서 팝업 창을 생성하지 않도록 하는 데 매우 유용합니다.

먼저 브라우저 창(왼쪽 상단 모서리)의 (0,0)에 상대적인 위치를 설정한 다음 버튼 아이콘에 표시하려는 스프라이트에 대한 텍스처 페이지 항목을 제공해야 합니다(참조: sprite_get_tpe() ). 그런 다음 열거나 실행할 스크립트( GameMaker 자산 브라우저에서) 또는 열거나 실행할 스크립트를 지정할 수 있습니다. 지정하는 "target" 매개변수는 사용할 때 표준 JavaScript "name" 값과 동일합니다. open() 메소드('_self'를 제외한 모든 것이 브라우저 차단 또는 사용자에게 허용 여부를 묻는 결과를 초래할 수 있음) 및 "params" 인수는 새로운 속성을 제어하기 위한 JavaScript "specs" 매개변수와 동일합니다. 창/탭이 표시되어야 합니다(일부 브라우저는 모든 기능을 지원하지 않을 수 있음). 아이콘을 사용하여 내부 스크립트를 실행하는 경우 이 마지막 두 인수는 빈 문자열 ""로 남겨둘 수 있습니다.

유효한 대상은 다음과 같습니다.

표적 설명
_공백 링크된 문서를 새 창이나 탭에서 엽니다.
_본인 클릭한 것과 동일한 프레임에서 링크된 문서를 엽니다(기본값).
_부모의 상위 프레임에서 링크된 문서를 엽니다.
_맨 위 창의 전체 본문에서 연결된 문서를 엽니다.

유효한 매개변수는 다음과 같습니다.

매개변수 설명
'높이=[ 픽셀 ]' 최소값이 100인 창의 높이입니다.
'너비=[ 픽셀 ]' 최소값이 100인 창의 너비입니다.
'왼쪽=[ 픽셀 ]' 창의 왼쪽 위치입니다.
'상단=[ 픽셀 ]' 창의 맨 위 위치입니다(IE만 해당).
'위치=[ 부울 ]' 주소 필드를 표시할지 여부(기본값은 1).
'메뉴바=[ 부울 ]' 메뉴 표시줄을 표시할지 여부(기본값은 1)입니다.
'크기 조정 가능=[ 부울 ]' 창의 크기를 조정할 수 있는지 여부(기본값은 1).
'스크롤바=[ 부울 ]' 스크롤 막대를 표시할지 여부(기본값은 1)입니다.
'상태=[ 부울 ]' 상태 표시줄을 추가할지 여부(기본값은 1)입니다.
'제목 표시줄=[ 부울 ]' 제목 표시줄을 표시할지 여부입니다. 호출하는 응용 프로그램이 HTML 응용 프로그램이나 신뢰할 수 있는 대화 상자(기본값은 1)가 아니면 무시됩니다.
'도구 모음=[ 부울 ]' 브라우저 도구 모음을 표시할지 여부(기본값은 yes)입니다.


전달한 URL 인수가 스크립트 함수 의 이름이고(예: "http://"로 시작하지 않음) 스크립트가 호출되는 경우 gmcallback_* 그러면 html5 이벤트에 직접 피드를 가져올 수 있습니다. GameMaker 스크립트 함수 이름은 다음으로 시작하는 경우 난독화 되지 않습니다. gmcallback_.

통사론:

clickable_add(x, y, tpe, url, target, params);

논쟁 유형 설명
x 진짜 The x position within the window.
y 진짜 The y position within the window.
tpe 텍스처 페이지 항목 The texture page entry for the sprite to be used.
url 끈 The URL (website address) to link to, or the script to be run.
target 끈 This is the target area to open the URL in (see description).
params 끈 The various parameters for the icon (see description).

보고:

Clickable ID

예시:

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

위의 코드는 게임 캔버스가 실행되고 있는 페이지의 (32, 32) 위치에 클릭 가능한 DOM 아이콘을 생성합니다. 아이콘은 텍스처 페이지에서 "spr_MS_Home"으로 참조된 스프라이트를 사용하며 아이콘을 클릭하면 해당 창에 대해 정의된 속성과 지정된 URL에 대한 새 창이 열립니다.
