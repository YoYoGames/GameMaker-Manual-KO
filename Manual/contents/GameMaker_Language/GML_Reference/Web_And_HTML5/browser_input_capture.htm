browser_input_capture

이 기능을 사용하면 브라우저 창에서 모든 입력을 캡처할지 여부를 설정할 수 있습니다( false ) 또는 게임이 입력을 캡처해야 하는지 여부(로 설정 true ). 이 기능은 HTML5 모듈에서만 사용할 수 있습니다.

통사론:

browser_input_capture(enable);

논쟁 유형 설명
enable 부울 Enable or disable game input capture

보고:

N/A

예시:

browser_input_capture(true);

위의 코드는 HTML5 대상 플랫폼에서 모든 브라우저 입력을 캡처하도록 게임을 설정합니다.
