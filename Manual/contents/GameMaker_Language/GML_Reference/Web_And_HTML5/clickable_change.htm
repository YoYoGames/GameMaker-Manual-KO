clickable_change

이 기능을 사용하여 이전에 생성된 클릭 가능한 아이콘의 스프라이트와 위치를 변경할 수 있습니다.clickable_add(). 위치는 캔버스가 아닌 창(0,0) 위치를 기반으로 하며 스프라이트는 텍스처 페이지에서 직접 참조해야 합니다(참조: sprite_get_tpe() ).

통사론:

clickable_change(index, tpe, x, y)

논쟁 유형 설명
index 클릭 가능한 ID The index of the clickable icon to change.
tpe 텍스처 페이지 항목 The texture page entry for the sprite to be used.
x 진짜 The new x position within the window.
y 진짜 The new y position within the window.

보고:

N/A

예시:

switch (room)
{
    case rm_Menu: clickable_change(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32); break;
    case rm_Game: clickable_change(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32); break;
}

위의 코드는 현재 방에 따라 "global.Help" 변수에 색인된 클릭 가능한 아이콘의 위치와 이미지 색인을 변경합니다.
