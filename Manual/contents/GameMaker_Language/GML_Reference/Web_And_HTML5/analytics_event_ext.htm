analytics_event_ext

이 기능은 HTML5 게임 옵션 을 통해 설정한 분석 제공자에게 지정된 텍스트를 보냅니다. 이 함수는 사용 중인 공급자의 범위를 벗어나는 항목을 추적하는 사용자 지정 분석을 만드는 데 사용할 수 있으며 사용자 지정 매개 변수/값 쌍도 허용합니다. 여기서 매개 변수는 문자열이고 값은 실수입니다. Google Analytics의 경우 Flurry가 최대 7개까지 허용하는 동안 한 쌍만 추가할 수 있습니다.

통사론:

analytics_event_ext(string, string_param, value);

논쟁 유형 설명
string A string to send to the provider.
string_param[0 -9] The parameter to send (a string).
value[0 - 9] The value of the parameter (a real number).

보고:

예시:

var num, time;
ini_open("Options.ini");
num = ini_read_real("Data", "Plays", 0);
num += 1;
time = current_time;
analytics_event_ext(GAME_NAME, "Plays", num, "Time", time);
ini_write_real("Data", "Plays", num);
ini_close();

위의 코드는 ini 파일과 현재 시간에서 재생 정보를 가져온 다음 해당 세부 정보를 분석 제공자에게 보냅니다.
