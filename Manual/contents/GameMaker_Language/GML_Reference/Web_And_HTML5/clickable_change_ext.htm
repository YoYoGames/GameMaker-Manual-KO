clickable_change_ext

이 기능을 사용하여 이전에 생성된 클릭 가능한 아이콘의 스프라이트와 위치를 변경할 수 있습니다.clickable_add(). 위치는 캔버스가 아닌 창(0,0) 위치를 기반으로 하며 스프라이트는 텍스처 페이지에서 직접 참조해야 합니다(참조: sprite_get_tpe() ). 이 기능을 사용하면 사용된 스프라이트의 크기(1이 기본값, 0.5는 절반, 2는 두 배)와 알파 값을 0(완전히 투명)에서 1(완전 불투명)로 변경할 수 있습니다. 화면의 마지막 아이콘을 위해

통사론:

clickable_change_ext(index, tpe, x, y, alpha, scale)

논쟁 유형 설명
index 클릭 가능한 ID The index of the clickable icon to change.
tpe 텍스처 페이지 항목 The texture page entry for the sprite to be used
x 진짜 The new x position within the window.
y 진짜 The new y position within the window.
scale 진짜 The scale of the icon (default 1).
alpha 진짜 The image alpha of the icon (default 1).

N/A

예시:

switch (room)
{
    case rm_Menu: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32, 2, 1); break;
    case rm_Game: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32, 1, 0.5); break;
}

위의 코드는 현재 방에 따라 "global.Help" 변수에 색인된 클릭 가능한 아이콘의 위치와 이미지 인덱스를 변경하고 아이콘 크기와 알파도 변경합니다.
