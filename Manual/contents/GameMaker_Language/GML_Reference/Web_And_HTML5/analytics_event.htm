analytics_event

이 기능은 HTML5 게임 옵션 을 통해 설정한 분석 제공자에게 지정된 텍스트를 보냅니다. 이 기능을 사용하여 사용 중인 공급자 범위를 벗어나는 항목을 추적하는 사용자 지정 분석을 만들 수 있습니다.

통사론:

analytics_event(string);

논쟁 유형 설명
string A string to send to the provider

보고:

예시:

if hs_new
{
    analytics_event("New hiscore of " + string (score));
}

위의 코드는 변수를 검사하여 true 그렇다면 지정된 문자열로 특수 분석 이벤트가 트리거됩니다.
