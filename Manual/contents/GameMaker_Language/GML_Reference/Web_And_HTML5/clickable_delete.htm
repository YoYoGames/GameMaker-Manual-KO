clickable_delete

이 기능은 이전에 생성된 클릭 가능한 아이콘을 제거하는 데 사용해야 합니다.clickable_add() 게임 창에서.

통사론:

clickable_delete(index);

논쟁 유형 설명
index 클릭 가능한 ID Index of the clickable icon to remove.

보고:

N/A

예시:

if clickable_exists(global.Help_Icon)
{
    clickable_delete(global.Help_Icon);
}

위의 코드는 게임 창에서 전역 변수 "Help_Icon"에 인덱싱된 클릭 가능한 아이콘이 있는 경우 이를 제거합니다.
