url_open_full

그러면 선택한 대상 장치의 브라우저에서 지정된 URL이 열리거나 HTML5 모듈을 사용하는 경우 현재 열려 있는 브라우저에서 열릴 것입니다. 지정하는 "target" 매개변수는 사용할 때 표준 JavaScript "name" 값과 동일합니다. open() 메서드(" _self "를 제외한 모든 것은 브라우저 차단 또는 사용자에게 허용 여부를 묻는 결과를 초래할 수 있음) 및 "옵션"은 새 창의 속성을 제어하기 위한 JavaScript "사양" 매개변수와 동일합니다. /tab이 표시되어야 합니다(일부 브라우저는 모든 기능을 지원하지 않을 수 있음).

유효한 대상은 다음과 같습니다.

표적 설명
_공백 새 창이나 탭에서 연결된 문서를 엽니다(팝업이 사용자에 의해 차단된 경우 작동하지 않습니다. 이 경우 다음을 사용할 수 있습니다.clickable_* 대신 기능).
_본인 클릭한 것과 동일한 프레임에서 링크된 문서를 엽니다(기본값).
_부모의 상위 프레임에서 링크된 문서를 엽니다.
_맨 위 창의 전체 본문에서 연결된 문서를 엽니다.

유효한 옵션은 다음과 같습니다.

매개변수 설명
'높이=[ 픽셀 ]' 최소값이 100인 창의 높이입니다.
'너비=[ 픽셀 ]' 최소값이 100인 창의 너비입니다.
'왼쪽=[ 픽셀 ]' 창의 왼쪽 위치입니다.
'상단=[ 픽셀 ]' 창의 맨 위 위치입니다(IE만 해당).
'위치=[ 부울 ]' 주소 필드를 표시할지 여부(기본값은 1).
'메뉴바=[ 부울 ]' 메뉴 표시줄을 표시할지 여부(기본값은 1)입니다.
'크기 조정 가능=[ 부울 ]' 창의 크기를 조정할 수 있는지 여부(기본값은 1).
'스크롤바=[ 부울 ]' 스크롤 막대를 표시할지 여부(기본값은 1)입니다.
'상태=[ 부울 ]' 상태 표시줄을 추가할지 여부(기본값은 1)입니다.
'제목 표시줄=[ 부울 ]' 제목 표시줄을 표시할지 여부입니다. 호출하는 응용 프로그램이 HTML 응용 프로그램이나 신뢰할 수 있는 대화 상자(기본값은 1)가 아니면 무시됩니다.
'도구 모음=[ 부울 ]' 브라우저 도구 모음을 표시할지 여부(기본값은 yes)입니다.

통사론:

url_open_full(url, target, options);

논쟁 유형 설명
url 끈 The URL (website address) to link to.
target 끈 This is the target area to open the URL in (see description).
options 끈 Standard browser options (see description).

예시:

url_open_full("http://gamemaker.io", "_blank", "resizable=0, height=200, scrollbars=0");

그러면 크기를 조정할 수 없고 높이가 200픽셀이고 스크롤 막대가 없는 새 창에서 GameMaker 홈페이지가 열립니다.
