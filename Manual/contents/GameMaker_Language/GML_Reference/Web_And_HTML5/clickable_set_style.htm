clickable_set_style

이 기능을 사용하면 제공된 키/값 쌍을 통해 주어진 버튼에 대한 CSS 스타일 속성을 설정할 수 있습니다.DS Map. 이전에 두 버튼 요소를 모두 생성해야 합니다(사용clickable_add() ) 그리고 DS Map 저장된 인덱스를 각각에 인수로 제공합니다.

통사론:

clickable_set_style(index, map)

논쟁 유형 설명
index 클릭 가능한 ID The index of the clickable icon to style.
map DS 맵 ID The index of the DS Map to set the style from.

보고:

N/A

예시:

var map = ds_map_create();
ds_map_add(map, "opacity", "0.5");
clickable_set_style(button, map);
ds_map_destroy();

위의 코드는 "button" 변수에 저장된 인덱스를 사용하여 클릭 가능한 버튼의 스타일을 변경합니다.
