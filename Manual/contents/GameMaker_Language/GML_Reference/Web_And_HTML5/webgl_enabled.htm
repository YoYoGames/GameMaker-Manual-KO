webgl_enabled

이 읽기 전용 변수는 WebGL이 활성화되었는지 여부를 반환합니다(true ) 여부(false ) 귀하의 게임에 대한. 브라우저(예: HTML5)를 통해 실행되는 게임에서만 작동하고 다른 모든 플랫폼에서는 true를 반환합니다.

통사론:

webgl_enabled

보고:

예시:

if webgl_enabled
{
    global.quality = 1;
}
else global.quality = 0;

위의 코드는 WebGL 플래그를 확인한 다음 그에 따라 전역 변수 "quality"를 설정합니다.
