arctan

x의 역탄젠트를 반환합니다. 이것은 다음과 같이 모든 숫자를 허용합니다. tan(), arctan() 점근선은 y축에 있으므로 pi/2보다 크거나 -pi/2보다 작은 숫자는 반환되지 않습니다.

참고: 이 함수는 도가 아닌 라디안 단위의 값을 취합니다.

통사론:

arctan(x);

논쟁 유형 설명
x The angle (in radians) to return the inverse tangent of.

보고:

예시:

val = arctan( 0 );

그러면 val이 0으로 설정됩니다.
