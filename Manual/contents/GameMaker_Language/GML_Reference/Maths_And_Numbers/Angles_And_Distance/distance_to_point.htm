distance_to_point

이 함수는 호출 인스턴스의 경계 상자 가장자리에서 방의 지정된 x/y 위치까지의 거리를 계산하며 반환 값은 픽셀 단위입니다. 호출 객체에 스프라이트가 없거나 마스크가 정의되지 않은 경우 결과가 올바르지 않습니다.

통사론:

distance_to_point(x, y);

논쟁 유형 설명
x The x position to check.
y The y position to check.

보고:

예시:

if (distance_to_point(obj_Player.x, obj_Player.y) < range)
{
    canshoot = true;
}

위의 코드는 플레이어 개체 x/y 위치까지의 거리를 확인하고 변수 "range"에 저장된 값보다 작으면 변수 "canshoot"가 true로 설정됩니다.
