distance_to_object

이 함수는 호출 인스턴스의 경계 상자 가장자리에서 지정된 개체의 가장 가까운 인스턴스의 가장 가까운 가장자리까지의 거리를 계산합니다. 객체는 객체 인덱스 또는 특정 인스턴스 ID 및 other 키워드 가 될 수 있으며 거리는 픽셀 단위로 반환됩니다. 오브젝트 중 하나에 스프라이트가 없거나 마스크가 정의되지 않은 경우 결과가 올바르지 않습니다.

통사론:

distance_to_object(obj);

논쟁 유형 설명
obj 개체 자산 또는 인스턴스 ID The object to check for.

보고:

Real

예시:

if (distance_to_object(obj_Player) < range)
{
    canshoot = true;
}

위의 코드는 플레이어 개체까지의 거리를 확인하고 "range" 변수에 저장된 값보다 작으면 "canshoot" 변수가 true로 설정됩니다.
