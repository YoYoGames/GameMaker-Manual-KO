darcsin

x의 역 사인을 반환합니다. dsin(x)=n, darcsin(n)=x, 결과 숫자는 -90에서 90 사이입니다.

참고: 이것은 -1에서 1 사이의 숫자만 허용합니다(다른 값은 오류가 발생함).

참고: 이 함수는 라디안이 아닌 도 단위로 값을 반환합니다.

통사론:

darcsin(x);

논쟁 유형 설명
val The value to return the inverse sine of.

보고:

예시:

val = darcsin(-1);

이것은 "val"을 -90으로 설정합니다.
