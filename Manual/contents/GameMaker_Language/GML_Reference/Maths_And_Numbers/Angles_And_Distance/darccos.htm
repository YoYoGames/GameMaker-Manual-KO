darccos

x의 역 코사인을 반환합니다. dcos(val)=n, darccos(n)=val, 결과 숫자는 180과 0 사이가 됩니다.

참고: 이것은 -1에서 1 사이의 숫자만 허용합니다(다른 값은 오류가 발생함).

참고: 이 함수는 라디안이 아닌 도 단위로 값을 반환합니다.

통사론:

darccos(val)

논쟁 유형 설명
x The value to return the inverse cosine of.

보고:

예시:

val = arccos(-1);

그러면 "val"이 180으로 설정됩니다.
