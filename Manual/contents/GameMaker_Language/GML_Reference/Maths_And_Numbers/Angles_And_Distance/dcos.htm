dcos

직각 삼각형에서 cos(val) = 인접 / 빗변 여기서 val은 세 각 중 하나입니다. 이 함수는 항상 1과 -1 사이의 숫자를 반환합니다.

참고: 이 함수는 라디안이 아닌 도 단위의 값을 취합니다.

통사론:

dcos(val);

논쟁 유형 설명
val The angle (in degrees) to return the cosine of.

보고:

예시:

val = dcos(45);

그러면 "val"이 0.71로 설정됩니다.
