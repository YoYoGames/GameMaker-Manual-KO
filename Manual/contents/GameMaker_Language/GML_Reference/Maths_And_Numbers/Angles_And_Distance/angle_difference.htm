angle_difference

이 함수는 지정된 두 각도 사이의 가장 작은 차이를 반환합니다. 여기서 차이는 원본 각도에서 대상 각도를 향해 계산됩니다. 반환된 값은 -180도에서 180도 사이입니다(양수 각도는 시계 반대 방향이고 음수 각도는 시계 방향).

"소스" 각도와 "목적지" 각도 간의 관계는 image_angle>angle_difference(dest, src) 를 호출하여 얻은 차이가 src 값에 다시 추가될 때 최종 각도를 제공하는 것과 같습니다(정확한 숫자 값은 다를 수 있음 ). 유사한 기술을 사용하여 단계마다 소스 각도를 대상 각도로 점차적으로 이동할 수 있습니다. 예를 들어 반환된 차이의 10%를 다시 src 에 추가하여 대상 각도로 약간 이동 합니다 (아래 예에 설명되어 있음). ).

통사론:

angle_difference(dest, src)

논쟁 유형 설명
dest The destination or target angle
src The source angle

보고:

예시:

var _dir = point_direction(x, y, mouse_x, mouse_y);
var _diff = angle_difference(_dir, image_angle);
image_angle += _diff * 0.1;

위의 코드는 인스턴스에서 마우스 커서까지의 방향을 얻은 다음 해당 각도와 현재의 차이를 얻습니다. , 이 값을 사용하여 인스턴스를 마우스 쪽으로 점진적으로 회전합니다.
