date_set_timezone

나머지 모든 날짜 및 시간 기능에 사용할 기준 시간대를 설정합니다. 이 시간대는 로컬 (시스템에서 설정한 대로) 또는 UTC 일 수 있으며 다음 상수 중 하나를 사용하여 사용 중인 것을 정의합니다(기본적으로 현지 시간).

일정한 설명
timezone_local 시스템에서 설정한 현지 시간대 사용
timezone_utc 협정 세계시 사용

통사론:

date_set_timezone(timezone);

논쟁 유형 설명
timezone 시간대 상수 The time zone to use for the base time.

보고:

N/A

예시:

if (date_get_timezone() != timezone_utc)
{
    date_set_timezone(timezone_utc);
}

이 코드는 게임의 기본 시간대 설정을 확인하고 UTC가 아니면 변경합니다.
