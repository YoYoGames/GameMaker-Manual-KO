date_leap_year

이 함수는 반환됩니다 true 주어진 날짜/시간 값의 연도 구성 요소가 윤년이거나 false 그렇지 않으면. 이것은 게임의 이스터 에그 또는 특별 콘텐츠 잠금 해제와 같은 작업에 편리한 기능이 될 수 있습니다.

통사론:

date_leap_year(date);

논쟁 유형 설명
date The datetime to use.

보고:

예시:

if date_leap_year(date_current_datetime())
{
    if !global.ExtraContent
    {
        global.ExtraContent = true;
    }
}

위의 코드는 현재 날짜 시간을 확인하여 연도가 윤년인지 여부를 확인합니다. 그렇다면 전역 변수를 설정합니다.
