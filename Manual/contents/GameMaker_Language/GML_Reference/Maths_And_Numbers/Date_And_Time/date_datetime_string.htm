date_datetime_string

이 함수를 사용하면 함수가 호출될 때 게임을 실행하는 시스템이나 장치에 맞게 형식이 지정된 날짜/시간을 포함하는 문자열을 만들 수 있습니다.

통사론:

date_datetime_string(date);

논쟁 유형 설명
date The date to use.

보고:

예시:

str = date_datetime_string(date_current_datetime());

이렇게 하면 날짜/시간 표시와 현재 날짜 및 시간에 대한 시스템 설정에 따라 "str"이 "September 15th 2011, 11:33.00"으로 설정됩니다.
