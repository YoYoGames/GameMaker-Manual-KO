current_day

이 읽기 전용 변수는 월에 따라 일을 1에서 31 사이의 값으로 반환합니다.

통사론:

current_day;

보고:

예시:

draw_text(32, 32, "Today is " + string(current_day) + "/" + string (current_month) + "/" + string(current_year) +".");

위의 코드는 일, 월, 연도를 그립니다.
