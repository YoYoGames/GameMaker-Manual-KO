current_time

이 읽기 전용 변수는 게임이 시작된 후 경과된 밀리초 수를 반환합니다.

통사론:

current_time;

보고:

예시:

if (current_time > 600000)
{
    msg = show_question_async("Would you like to rate?");
}

위의 코드는 사용자에게 질문하기 전에 10분 이상 경과했는지 확인합니다.
