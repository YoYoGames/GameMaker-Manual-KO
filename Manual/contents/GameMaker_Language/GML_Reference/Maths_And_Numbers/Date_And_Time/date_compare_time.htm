date_compare_time

이 기능을 사용하면 두 날짜와 시간을 확인하여 어느 것이 다른 것보다 빠르거나 늦은 시간 구성 요소를 가지고 있는지 확인할 수 있습니다. 이 함수는 datetime1이 더 빠르면 -1을 반환하고 datetime1이 나중이면 1을 반환하고 같으면 0을 반환하고 날짜를 무시하므로 문자 그대로 주어진 날짜에서 더 먼 시간이 표시됩니다.

통사론:

date_compare_time(datetime1, datetime2);

논쟁 유형 설명
datetime1 The first datetime.
datetime2 The datetime to compare the first one to.

보고:

예시:

d = date_compare_time(date_create_datetime( 2011, 9, 15, 11, 4, 0 ), date_current_datetime());

이것은 "d"를 가장 빠른 시간에 따라 해당 값으로 설정합니다. 기본적으로 현재 시간이 오전 11시 4분 이후이면 1을 반환하고, 이전이면 -1을 반환합니다. 시간이 11:04이면 0을 반환합니다.
