date_get_timezone

이 함수는 나머지 모든 날짜 및 시간 함수에 사용되는 기본 시간대를 가져옵니다. 이 시간대는 지역 (시스템에서 설정한 대로) 또는 UTC 일 수 있으며 함수는 다음 상수 중 하나를 반환합니다.

시간대 상수
일정한 설명
timezone_local 시스템에서 설정한 현지 시간대 사용
timezone_utc 협정 세계시 사용

통사론:

date_get_timezone();

보고:

Time Zone Constant

예시:

if (date_get_timezone() != timezone_utc)
{
    date_set_timezone(timezone_utc);
}

이 코드는 게임의 기본 시간대 설정을 확인하고 UTC가 아니면 변경합니다.
