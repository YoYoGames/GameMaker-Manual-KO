date_is_today

이 함수는 반환됩니다 true 주어진 datetime 값이 확인되는 날짜(즉, 오늘)인 경우, 또는 false 그렇지 않으면. 이것은 게임의 이스터 에그 또는 시즌 콘텐츠 잠금 해제와 같은 편리한 기능이 될 수 있습니다. 이 기능은 사용하여 변경할 수 있는 시간대 설정(기본값은 현지 시간)의 영향을 받습니다.date_set_timezone() 함수.

통사론:

date_is_today(date);

논쟁 유형 설명
date The datetime to use.

보고:

예시:

if date_is_today(global.Halloween)
{
    global.Max_Levels = 200;
}

위의 코드는 "Halloween" 전역 변수에 저장된 날짜 시간을 확인하여 오늘인지 확인하고, 오늘이면 다른 전역 변수를 새 값으로 설정합니다.
