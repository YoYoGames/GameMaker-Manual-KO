current_minute

이 읽기 전용 변수는 현재 순간에 해당하는 분을 반환합니다.

통사론:

current_minute;

보고:

예시:

draw_text(32, 32, "The time is " + string(current_hour) + ":" + string(current_minute) + "." + string(current_second));

위의 코드는 화면에 현재 국제 시간을 그립니다.
