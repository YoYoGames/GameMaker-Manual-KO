choose

때때로 무작위 선택을 위해 숫자가 아닌 다른 것을 지정하려고 하거나 원하는 숫자가 실제 순서가 아니거나 설정된 범위 내에 있지 않습니다. 이러한 경우에는 다음을 사용합니다. choose() 임의의 결과를 생성합니다. 예를 들어, 시작 부분에 임의의 스프라이트가 있는 객체를 생성하려는 경우 이 함수를 사용하여 주어진 스프라이트 세트 중 하나로 스프라이트 인덱스를 설정할 수 있습니다. 필요한 만큼 가질 수 있습니다(인수가 많을수록 함수의 구문 분석 속도가 느려집니다).

참고 GameMaker 는 디버깅 코드를 훨씬 더 쉽게 만들기 위해 매번 동일한 초기 임의 시드를 생성하기 때문에 이 함수는 게임을 새로 실행할 때마다 동일한 값을 반환합니다. 이 동작을 방지하려면 다음을 사용하십시오.randomise() 게임 시작 시.

통사론:

choose(val0, val1, val2... max_val);

논쟁 유형 설명
val0... max_val 변하기 쉬운 Any type of value(s).

보고:

Variable (One of the given arguments)

예시:

sprite_index = choose(spr_Cactus, spr_Flower, spr_Tree, spr_Shrub);
hp = choose(5, 8, 15, 32, 40);
name = choose("John", "Steven", "Graham", "Jack", "Emily", "Tina", "Jill", "Helen");

위의 코드는 선택을 사용하여 인스턴스에 대한 여러 속성을 설정합니다.
