sqrt

숫자를 자신과 곱하면 해당 숫자의 제곱을 얻을 수 있지만 때로는 반대로 수행하여 숫자의 제곱근을 얻고 싶을 때가 있습니다. 따라서 주어진 양수 값을 얻기 위해 자신과 곱한 숫자를 찾으려면 이 함수를 사용합니다. 예를 들어: sqrt(9) 3*3=9이므로 3을 반환합니다.

통사론:

sqrt(val);

논쟁 유형 설명
val The number to get the square root of.

보고:

예시:

num = sqrt(exp);

위의 코드는 "exp"에 포함된 값의 제곱근을 유지하도록 변수 "num"을 설정합니다.
