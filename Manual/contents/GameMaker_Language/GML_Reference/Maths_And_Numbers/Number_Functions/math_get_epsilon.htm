math_get_epsilon

이 함수는 대상 플랫폼에 대한 현재 엡실론 값을 반환합니다. 엡실론에 대한 자세한 내용은 함수를 참조하십시오.math_set_epsilon().

통사론:

math_get_epsilon();

보고:

예시:

var e = math_get_epsilon();
if e != 0.000001
{
    math_set_epsilon(0.000001);
}

이렇게 하면 현재 엡실론 값을 검색하여 로컬 변수에 저장한 다음 확인하고 필요한 경우 새 엡실론을 설정합니다.
