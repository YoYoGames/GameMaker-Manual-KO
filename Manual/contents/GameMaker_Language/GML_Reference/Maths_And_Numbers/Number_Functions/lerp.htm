lerp

이 기능을 사용하면 주어진 백분율에 대해 다른 두 값 사이의 위치와 동일한 값을 찾을 수 있습니다. 예를 들면 다음과 같습니다.

lerp(0, 10, 0.5)

입력 값의 50%인 5의 반환 값을 얻게 됩니다. 다음과 같이 보간량에 양수 또는 음수 값을 제공하여 이 함수를 사용하여 외삽할 수도 있습니다.

lerp(0, 10, 2)

값 20을 반환합니다.

통사론:

lerp(a, b, amt)

논쟁 유형 설명
a The first value.
b The second value.
amt The amount to interpolate.

보고:

예시:

xx = lerp(x, x + hspeed, room_speed);
yy = lerp(y, y + vspeed, room_speed);

위의 코드는 선형 보간 기능을 사용하여 게임 시간 1초 후에 인스턴스가 이동할 위치를 예측합니다.
