median

이 함수는 입력 값의 중앙값, 즉 중간 값을 반환합니다. 인수 수가 짝수이면 두 중간 값 중 작은 값이 반환되고 함수는 필요한 만큼 많은 인수를 가질 수 있습니다(인수가 많을수록 함수가 구문 분석 속도가 느려짐). 모두 실제 값이어야 합니다. 이것은 예를 들어, median(43, 12, 25, 19, 6) 나머지 모든 값 사이의 중간 값이므로 19를 반환합니다.

통사론:

median(val1, val2, ... max_val);

논쟁 유형 설명
val0... max_val The values to compare.

보고:

예시:

x = median( 0, x, room_width );
y = median( 0, y, room_height );

이렇게 하면 중앙값을 클램프로 사용하여 플레이어가 방의 어느 쪽에서나 나가는 것을 막을 수 있습니다. 예를 들어 플레이어가 방 경계의 왼쪽으로 이동하면 x는 0보다 작을 것입니다. 이것은 중앙값의 첫 번째 중간 숫자가 0이 된다는 것을 의미하므로 플레이어는 (0, 와이).
