floor

n의 하한값을 반환합니다. 즉, n은 정수로 내림됩니다. 이는 다음과 유사하다.round() 함수이지만 소수점 값에 관계없이 내림 만 하므로 floor(5.99999) 5를 반환합니다. floor(5.2), floor(5.6457) 등...

통사론:

floor(n);

논쟁 유형 설명
n The number to floor.

보고:

예시:

val = floor( 3.9 );

그러면 val이 3으로 설정됩니다.
