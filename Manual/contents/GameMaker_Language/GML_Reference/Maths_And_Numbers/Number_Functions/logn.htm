logn

이 기능은 다음과 유사합니다.log2(n) 그리고 log10(n) 기능을 사용하는 경우에만 로그 기본 값을 지정합니다. 예를 들어, logn(5,25) 25(2)를 얻기 위해 곱해야 하는 5의 수를 반환합니다.

통사론:

logn(n, val);

논쟁 유형 설명
n The log base.
val The input value.

보고:

예시:

logval = logn(5, num);

위의 코드는 log5 "num"에 저장된 값의
