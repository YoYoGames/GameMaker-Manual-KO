irandom_range

이 기능은 다음과 유사합니다.random_range() 정수 값만 입력으로 사용합니다. 범위에 대한 낮은 값과 높은 값을 제공하면 함수는 주어진 범위 내(및 포함) 내의 임의의 정수 값을 반환합니다. 예를 들어, irandom_range(10, 35) 10에서 35( 포함 ) 사이의 정수를 반환합니다.

와 마찬가지로irandom() 함수에서 실수를 사용할 수 있으며 이 경우 가장 가까운 정수 EG로 내림됩니다. irandom_range(6.2,9.9)는 6과 9 사이의 값을 제공합니다.

참고 GameMaker: Studio는 디버깅 코드를 훨씬 더 쉽게 만들기 위해 매번 동일한 초기 무작위 시드를 생성하기 때문에 이 함수는 게임을 새로 실행할 때마다 동일한 값을 반환합니다. 이 동작을 방지하려면 다음을 사용하십시오.randomise() 게임 시작 시.

통사론:

irandom_range(n1, n2);

논쟁 유형 설명
n1 The low end of the range from which the random number will be selected.
n2 The high end of the range from which the random number will be selected.

보고:

예시:

score += irandom_range(500, 600);

이렇게 하면 총 점수에 500에서 600(포함) 사이의 정수 값이 추가됩니다.
