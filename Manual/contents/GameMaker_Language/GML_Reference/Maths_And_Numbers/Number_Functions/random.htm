random

이 함수는 정수(정수)를 반환할 필요가 없는 확률에 유용합니다. 예를 들어, random(100) 0에서 99 사이의 값을 반환하지만 해당 값은 22.56473이 될 수 있습니다! 다음과 같이 이 함수에서 정수가 아닌 실수를 사용할 수도 있습니다. random(0.5), 0에서 0.4999999 사이의 값을 반환합니다.

참고: 이 함수는 GameMaker 가 디버깅 코드를 훨씬 더 쉬운 작업으로 만들기 위해 매번 동일한 초기 무작위 시드를 생성한다는 사실 때문에 게임이 새로 실행될 때마다 동일한 값을 반환합니다. 이 동작을 방지하려면 다음을 사용하십시오.randomise() 게임 시작 시.

통사론:

random(n);

논쟁 유형 설명
n The upper range from which the random number will be selected.

보고:

예시:

if (random(10) >= 9)
{
    score += 100;
}

이렇게 하면 점수에 100을 더할 확률이 대략 10분의 1이 됩니다.
