random_set_seed

난수를 생성하기 위해 GameMaker 는 난수 시드 번호로 시작합니다. 이 기능을 사용하면 해당 시드를 알려진 값으로 설정할 수 있으므로 프로그램이 실행될 때마다 이후의 모든 임의 이벤트의 결과가 동일하도록 "강제"할 수 있습니다. 예를 들어 이 기능은 다음과 함께 사용할 수 있습니다.random_get_seed() 절차적으로 생성된 콘텐츠를 생성하고 엄청난 저장 게임 없이 결과를 저장합니다(시드만 저장하고 다른 것은 필요 없음). 모든 것에 대해 진정으로 임의의 결과가 필요한 경우 다음을 사용해야 합니다.randomise() 함수.

참고 이 시드는 각 대상 플랫폼에서 일관된 결과를 제공하지만 각 대상이 작동하는 방식이 다르기 때문에 플랫폼마다 결과가 다를 수 있습니다.

통사론:

random_set_seed(val);

논쟁 유형 설명
val The seed to set.

보고:

N/A

예시:

if debug
{
    random_set_seed(1);
}

위의 코드는 "debug" 변수가 true인 경우에만 임의의 시드를 1로 설정합니다.
