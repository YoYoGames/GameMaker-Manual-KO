power

이것은 "n"번 자신을 곱한 숫자의 값을 반환합니다. 예를 들어, power(5,3) 5를 3번 곱하고 125를 반환합니다. 이는 5*5*5=125와 같습니다. "x" 값(변경할 숫자)은 음수 값이 될 수 없습니다.

통사론:

power(x, n);

논쟁 유형 설명
x The number to change.
n How many times to multiply x by itself.

보고:

예시:

score += power(dmg, 3);

이것은 변수 "dmg"에 있는 값을 점수 변수에 3의 거듭제곱에 추가합니다.
