log2

이 함수는 기본적으로 "n을 얻으려면 몇 개의 2를 곱해야 합니까?"를 묻습니다. 프로그래밍에서 사용하는 일반적인 예는 숫자를 나타내는 데 필요한 비트 수를 계산하는 것입니다. 이를 위해 우리는 계산합니다 log2(x) 숫자에서 반올림하고 1을 추가하십시오. 예를 들어 log2(100) 6.643856을 반환하고 반올림한 값은 6이고 여기에 1을 더하면 7이 됩니다. 따라서 100을 나타내려면 7비트가 필요합니다. 또한 지수 성장 또는 감소를 설명하는 데 사용할 수 있으며 종종 이진 로그 라고 합니다.

통사론:

log2(n);

논쟁 유형 설명
n The input value.

보고:

예시:

colourbits = floor(log2(colour)) - 1;

위의 코드는 "color" 변수에 저장된 값을 나타내는 데 필요한 비트 수를 저장합니다.
