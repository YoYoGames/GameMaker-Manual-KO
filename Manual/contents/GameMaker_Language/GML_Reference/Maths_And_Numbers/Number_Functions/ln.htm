ln

자연 로그 ln(n) 는 특정 수준의 지속적인 성장에 도달하는 데 필요한 시간이며, 여기서 n은 도달한 수준입니다. 따라서 20 성장을 얻는 데 필요한 시간 단위를 알고 싶다면 다음을 사용합니다. ln(20) 2.99 단위의 시간을 반환하여 해당 양의 성장을 얻습니다.

통사론:

ln(n);

논쟁 유형 설명
n The input value.

보고:

예시:

alarm[0] = ln(age) * room_speed;

위의 코드는 "age" 변수에 저장된 값의 자연로그를 사용하여 알람을 설정합니다.
