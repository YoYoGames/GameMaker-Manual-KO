log10

이 기능은 다음과 유사합니다.log2(n) 기능, 그뿐 log10(n) n을 얻기 위해 함께 곱해야 하는 10의 수를 얻습니다. 그래서 log10(100) 10을 자체적으로 곱하면 100이 되기 때문에 2를 반환합니다. log10(1000) 3을 10x10x10=1000 등으로 반환합니다... 숫자를 10으로 나눌 필요는 없지만 예를 들어 다음과 같은 경우 적절한 값을 반환합니다. log10(101) 우리는 2.00432137...을 얻습니다. 이것은 101을 10으로 나눌 수 있는 횟수입니다.

통사론:

log10(n);

논쟁 유형 설명
n The input value.

예시:

logval = log10(num);
