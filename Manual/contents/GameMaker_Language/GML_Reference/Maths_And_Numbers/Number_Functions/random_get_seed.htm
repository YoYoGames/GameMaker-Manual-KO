random_get_seed

난수를 생성하기 위해 GameMaker 는 난수 시드 번호로 시작합니다. 이 기능을 사용하면 해당 시드를 검색한 다음 특정 일련의 무작위 이벤트를 재현하기 위해 저장할 수 있습니다. 예를 들어 절차적으로 생성된 게임이 있지만 플레이어가 이전에 발견한 영역으로 돌아갈 수 있기를 원한다고 가정해 보겠습니다. 음, 각 영역의 시작 부분에 이 기능을 사용하여 임의의 시드를 저장할 수 있으므로 다시 돌아가야 할 때마다 다음을 사용할 수 있습니다.random_set_seed() 시드를 이전으로 설정하여 GameMaker 가 이전에 사용한 것과 동일한 "무작위" 기능 세트를 재현하도록 합니다.

통사론:

random_get_seed();

보고:

예시:

ini_open("Save.ini")
ini_write_real("Levels", string(level), random_get_seed());
ini_close();

위의 코드는 임의의 시드를 가져와 ini 파일에 저장합니다.
