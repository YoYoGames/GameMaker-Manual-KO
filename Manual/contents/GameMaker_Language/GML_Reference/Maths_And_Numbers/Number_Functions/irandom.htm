irandom

이 매우 유용한 함수는 정수(정수)만 반환합니다. 예를 들어, 0에서 9까지의 난수를 얻으려면 다음을 사용할 수 있습니다. irandom(9) 0에서 9 까지 의 숫자를 반환합니다. float도 사용할 수 있지만 점 이후의 상위 값은 제외되므로 주의하십시오. irandom(9.7) 0에서 9 사이의 값만 반환합니다. 함수의 상한선은 $7fffffffffffffff LL이므로 매우 큰 숫자를 사용하는 경우 주의해야 합니다.

참고 GameMaker 는 디버깅 코드를 훨씬 더 쉽게 만들기 위해 매번 동일한 초기 임의 시드를 생성하기 때문에 이 함수는 게임을 새로 실행할 때마다 동일한 값을 반환합니다. 이 동작을 방지하려면 다음을 사용하십시오.randomise() 게임 시작 시.

통사론:

irandom(n);

논쟁 유형 설명
n The upper range from which the random number will be selected.

보고:

예시:

if (irandom(9) == 1)
{
    score += 100;
}

이것은 점수에 100을 더할 확률이 10분의 1(0이 포함되어 있으므로)을 생성합니다.
