max

이 함수는 최대 16개의 입력 값을 반환합니다. 예를 들어 max(12, 96, 32, 75) 모든 입력 값 중 가장 높은 값인 96을 반환합니다.

통사론:

max(val1, val2, ... max_val);

논쟁 유형 설명
val0... max_val The values to compare.

보고:

예시:

x = max(x, 0);

이것은 플레이어가 방의 왼쪽에서 나가는 것을 막을 것입니다. 이것은 지속적으로 x를 자체 또는 0 중 더 큰 값으로 설정하여 작동합니다. 플레이어가 왼쪽에서 나가면 x가 0보다 작아서(즉, 음수) 곧바로 다시 설정됩니다.
