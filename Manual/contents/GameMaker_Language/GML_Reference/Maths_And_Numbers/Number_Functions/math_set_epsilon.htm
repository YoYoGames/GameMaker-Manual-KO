math_set_epsilon

엡실론은 반올림 오류의 대상이 되는 두 숫자가 "동일한" 것으로 간주될 만큼 충분히 가까운지 여부를 결정하는 데 사용되는 값입니다. 특정 연산이 올바르지 않거나 예상과 반대되는 값을 반환하게 하는 "반올림 오류"를 줄일 수 있으므로 부동 소수점 수학을 처리할 때 유용합니다. 예를 들어 스프라이트의 이미지 인덱스에 값을 추가하고 결과가 단일 정수일 것으로 예상할 수 있지만 부동 소수점 수학의 특성으로 인해 실제 최종 값은 5.0000002와 같게 됩니다. 다음을 확인하십시오.

if image_index == 5
{
    //do something
}

코드가 예상대로 작동하지 않고 반환됩니다. false. 그러나 엡실론 값을 0.000001로 설정하면 image_index 값은 원래 값의 +/- 0.000001인 가장 가까운 실수로 반올림되어 위의 비교가 true를 반환합니다.

엡실론 값은 다음 모든 정수 연산자 에 사용됩니다.

  • < : 미만
  • > : 보다 큰
  • == : 동일
  • <= : 이하
  • >= : 크거나 같음
  • != : 같지 않음

엡실론 값을 0으로 설정하면 모든 반올림이 비활성화되고 값 1을 사용하면 오류가 발생합니다.

통사론:

math_set_epsilon(epsilon);

논쟁 유형 설명
epsilon 진짜 The new epsilon value (from 0 to 0.999999999).

보고:

Real

예시:

math_set_epsilon(0.0001);

이것은 모든 추가 부동 소수점 연산에 대한 엡실론 값을 설정합니다.
