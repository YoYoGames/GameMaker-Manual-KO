ceil

이 함수는 임의의 실수를 취하여 가장 가까운 정수로 반올림합니다. 일반적인 실수 중 하나는 임의의 값을 반올림하고 항상 1보다 클 것으로 예상하는 것이므로 이 함수에 주의해야 합니다. 예:

int = ceil(random(5));

이제 이 코드가 항상 1과 5 사이의 정수를 제공할 것으로 예상하지만 임의 함수가 0을 반환하고 0을 반올림해도 여전히 0이 될 가능성이 매우 적기 때문에 항상 그런 것은 아닙니다. 가능성은 희박하지만 이 기능을 사용할 때 고려해야 합니다.

통사론:

ceil(val);

논쟁 유형 설명
x The number to change.

보고:

예시:

val = ceil( 3.4 );

그러면 val이 4로 설정됩니다.
