random_range

이 함수는 지정된 범위 사이의 난수를 반환하며 이 반환 값은 정수일 필요가 없습니다. 예를 들어, random_range(20,50) 20에서 50 사이의 난수를 반환하지만 값은 38.65265와 같은 실수일 수 있습니다. 실수도 입력 인수로 사용할 수 있습니다.

참고: 이 함수는 GameMaker 가 디버깅 코드를 훨씬 더 쉬운 작업으로 만들기 위해 매번 동일한 초기 무작위 시드를 생성한다는 사실 때문에 게임이 새로 실행될 때마다 동일한 값을 반환합니다. 이 동작을 방지하려면 다음을 사용하십시오.randomise() 게임 시작 시.

통사론:

random_range(n1, n2);

논쟁 유형 설명
n1 The low end of the range from which the random number will be selected.
n2 The high end of the range from which the random number will be selected.

보고:

예시:

score += random_range(500, 600);

이렇게 하면 총 점수에 500에서 600 사이의 값이 추가됩니다.
