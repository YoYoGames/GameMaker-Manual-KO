min

이 함수는 입력 값의 최소값을 반환하며 그 중 필요한 만큼 많이 가질 수 있습니다(인수가 많을수록 함수가 구문 분석 속도가 느려집니다). 예를 들어 min(12, 96, 32, 75) 모든 입력 값 중 가장 낮은 값인 12를 반환합니다.

통사론:

min(val1, val2, ... max_val);

논쟁 유형 설명
val0... max_val The values to compare.

보고:

예시:

x = min(x, room_width);

이것은 플레이어가 방의 오른쪽에서 나가는 것을 막을 것입니다. 이것은 지속적으로 x를 자체 또는 room_width 변수 중 더 작은 값으로 설정하여 작동합니다. 플레이어가 오른쪽으로 나가면 x가 room_width보다 크므로 곧바로 다시 설정됩니다.
