matrix_build_projection_perspective

이 함수는 아래 나열된 지정된 매개변수를 사용하여 근거리 클리핑 평면의 치수를 기반으로 원근 투영 행렬을 작성합니다.

카메라의 화각은 카메라에 따라 달라집니다. width, height 그리고 znear 투영 너비와 높이가 지정된 Z 위치에 배치되므로 지정된 값 znear 논쟁.

예를 들어, 640x480의 일정한 크기가 주어지면 다음과 같은 경우 시야가 더 넓어집니다. znear 카메라에 더 가깝지만 znear 카메라에서 멀리 떨어져 있습니다. 이 동작은 다음 시각적 개체에 나와 있습니다.

이 때문에 다음을 사용하는 것이 좋습니다. znear 투영의 너비와 동일한 값으로 시야가 동등한 직교 투영과 일치하게 됩니다.

통사론:

matrix_build_projection_perspective(width, height, znear, zfar);

논쟁 유형 설명
w The width of the projection at the near Z position.
h The height of the projection at the near Z position.
znear The near clipping plane.
zfar The far clipping plane.

보고:

Matrix Array

예시:

var projmat = matrix_build_projection_perspective(640, 480, 640.0, 32000.0);
camera_set_proj_mat(view_camera[0], projmat);

위의 코드는 원근 투영을 만든 다음 반환된 행렬을 사용하여 뷰 포트[0]에 할당된 카메라를 설정합니다.
