matrix_stack_is_empty

이 함수는 행렬 스택이 비어 있는지 확인하는 데 사용할 수 있습니다(반환 true ) 여부(반환 false ).

통사론:

matrix_stack_is_empty();

보고:

예시:

if !matrix_stack_is_empty()
{
    matrix_stack_clear();
}

위의 코드는 행렬 스택을 검사하여 비어 있는지 확인하고 그렇지 않으면 지웁니다.
