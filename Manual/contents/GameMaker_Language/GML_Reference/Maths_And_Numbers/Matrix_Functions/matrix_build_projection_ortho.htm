matrix_build_projection_ortho

이 함수는 아래 나열된 지정된 매개변수를 기반으로 직교 투영 행렬을 구축합니다(이는 아무 것도 변경하지 않고 GameMaker 에서 방을 생성할 때 사용되는 기본 투영 방법입니다). 때로는 이 기능의 목적인 원근 투영에서 직교 투영으로 전환해야 합니다. 일반적으로 오버레이를 그리는 데 사용됩니다. 예를 들어 점수 또는 기타 측면을 표시하기 위해 3D 게임에서 그려진 요소의 "평면" 보기(즉, 원근 없음)를 제공하기 때문입니다. 직교 뷰와 투시 뷰의 차이점에 대한 아이디어를 얻으려면 아래 이미지를 참조하십시오.

참고: 현재 깊이 값에 관계없이 정보를 그리려면 숨겨진 표면 제거를 일시적으로 꺼야 할 수도 있습니다.

통사론:

matrix_build_projection_ortho(width, height, znear, zfar);

논쟁 유형 설명
w The width of the projection.
h The height of the projection.
znear The near clipping plane.
zfar The far clipping plane.

보고:

Matrix Array

예시:

viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

위의 코드는 새로운 보기 행렬과 직교 행렬을 만들고 해당 ID를 로컬 변수에 저장한 다음 이를 사용하여 보기 포트[0]에 할당된 카메라의 보기 및 투영 행렬을 설정합니다.
