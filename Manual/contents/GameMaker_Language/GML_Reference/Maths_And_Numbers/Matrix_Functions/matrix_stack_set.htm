matrix_stack_set

이 함수는 지정된 행렬로 행렬 스택의 현재 상단을 덮어씁니다.

통사론:

matrix_stack_set(matrix);

논쟁 유형 설명
matrix The matrix index to use.

보고:

예시:

var m = matrix_build(x, y, 0, 0, 0, 0, 1, 1, 1);
matrix_stack_set(m);

위의 코드는 새로운 행렬을 만들고 행렬 스택의 맨 위를 바꾸기 전에 변수 "m"에 결과 행렬 인덱스를 저장합니다.
