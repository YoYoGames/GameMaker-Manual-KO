matrix_transform_vertex

이 함수는 벡터를 행렬로 변환합니다. 변환 행렬 ID(함수에서 반환된matrix build() ), 변환할 벡터의 x, y 및 z 값. 이 함수는 다음과 같은 3개 요소의 배열 을 반환합니다.

  • 배열[0] = x
  • 배열[1] = y
  • 배열[2] = z.

통사론:

matrix_transform_vertex(matrix, x, y, z);

논쟁 유형 설명
matrix The matrix to use
x The x component of the transform vector
y The y component of the transform vector
z The z component of the transform vector

보고:

(3 elements)

예시:

t_matrix = matrix_build(0, 0, 0, 0, 90, 0, 1, 2, 1);
verts = matrix_transform_vertex(t_matrix, x, y, z);

위의 코드는 "t_matrix" 변수에 저장된 행렬을 사용하여 주어진 값을 변환하고 "verts" 배열에 저장합니다.
