matrix_stack_push

이 함수는 새 행렬을 현재 행렬 스택의 맨 위로 푸시하는 데 사용됩니다. 먼저 주어진 행렬에 현재 스택의 맨 위에 있는 행렬을 곱한 다음 결과 행렬을 스택으로 푸시합니다.

이 함수는 다양한 행렬을 수동으로 곱할 필요 없이 시각적 개체에 여러 행렬 변환을 적용하는 데 유용합니다.

예를 들어, 특정 직사각형 영역(예: 창)에 그래픽을 그리는 스택에 기본 매트릭스를 푸시할 수 있습니다. 해당 매트릭스를 사용하여 일부 그래픽을 그린 후 하위 영역(예: 창의 버튼) 내부에 그래픽을 그리기 위해 스택에 다른 매트릭스를 푸시할 수 있습니다. 해당 하위 영역에 무언가를 그린 후 다음을 호출할 수 있습니다. matrix_stack_pop() 스택에서 매트릭스를 제거하고 기본 창 영역에 시각적 개체를 계속 그립니다.

통사론:

matrix_stack_push(matrix);

논쟁 유형 설명
matrix The matrix to push to the stack

보고:

예시:

var m1 = matrix_build(66, 145, 0, 0, 0, 0, 1, 1, 1);
var m2 = matrix_build(0, 0, 0, 0, 0, image_angle * 6, 1, 1, 1) ;
matrix_stack_push(m1);
matrix_stack_push(m2);
matrix_set(matrix_world, matrix_stack_top());
draw_sprite(spr_tyre, 0, 0, 0);
matrix_stack_pop();
matrix_stack_pop();

위의 코드는 두 개의 새로운 행렬을 만든 다음 행렬 스택에 푸시합니다. 그런 다음 월드 매트릭스가 스택의 맨 위에 설정되고 스프라이트가 그려지고 스택에서 변환이 팝됩니다.
