matrix_get

이 함수는 주어진 4x4 행렬 유형에 해당하는 16개 값의 배열 을 반환합니다. 여기서 행 1은 요소 [0 - 3], 행 2는 요소 [4 -7] 등...( 메인 페이지 의 이미지 참조 ). 사용 가능한 행렬은 view, projection 및 world 이며, 다음 상수 중 하나를 사용합니다.

행렬 유형 상수
일정한 설명
matrix_view 현재 보기 매트릭스
matrix_projection 현재 투영 행렬
matrix_world 현재 세계 매트릭스

통사론:

matrix_get(type);

논쟁 유형 설명
type 행렬 유형 상수 The type of matrix to get (see the constants listed above)

보고:

Matrix Array

예시:

v_array = matrix_get(matrix_view);

위의 코드는 현재 뷰 행렬의 값을 가져와 "v_array" 배열을 채웁니다.
