matrix_set

이 함수는 주어진 4x4 행렬 유형에 해당하는 16개 값의 배열 을 취합니다. 여기서 요소 [0 - 3]은 행 1, 요소 [4 -7]은 행 2 등...(이미지 참조 메인 페이지 ). 다음을 사용하여 이러한 행렬을 만들 수 있습니다.matrix_build() 또는matrix_get() 함수를 사용하거나 단순히 배열을 직접 빌드하고 함수에 전달합니다. 사용 가능한 행렬 유형은 view, projection 및 world 이며 다음 상수 중 하나를 사용합니다.

일정한 설명
matrix_view 현재 보기 매트릭스
matrix_projection 현재 투영 행렬
matrix_world 현재 세계 매트릭스

통사론:

matrix_set(type, matrix);

논쟁 유형 설명
type 행렬 유형 상수 The type of matrix to get the values of (see the constants listed above)
matrix 행렬 배열 The matrix data as an array

보고:

N/A

예시:

matrix_set(matrix_world, m_array);

위의 코드는 현재 세계 행렬의 값을 배열 행렬 "m_array"에 저장된 값으로 설정합니다.
