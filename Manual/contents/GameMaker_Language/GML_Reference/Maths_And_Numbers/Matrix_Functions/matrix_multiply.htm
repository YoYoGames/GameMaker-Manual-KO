matrix_multiply

이 함수를 사용하면 두 개의 행렬 배열을 곱하여 새 변환 행렬을 만들 수 있습니다. 이 함수는 나중에 사용할 수 있도록 변수에 저장해야 하는 새 행렬 인덱스를 반환합니다.

참고: 이 함수의 인수로 행렬 상수를 사용할 수 없으므로 (예를 들어) 보기 행렬을 사용자가 만든 사용자 지정 행렬과 곱하려면 먼저 다음을 호출해야 합니다.matrix_get() 뷰 행렬 값을 배열 변수에 할당한 다음 사용자 지정 행렬로 곱한 다음 선택한 행렬(보기, 투영 또는 세계)을 설정합니다.

통사론:

matrix_multiply(matrix1, matrix2);

논쟁 유형 설명
matrix1 The first matrix index to use.
matrix2 The second matrix index to use.

보고:

Matrix Array

예시:

var v_matrix = matrix_get(matrix_view);
var new_matrix = matrix_multiply(v_matrix, my_matrix);
matrix_set(matrix_view, new_matrix);

위의 코드는 현재 뷰 행렬을 가져온 다음 사용자 지정 행렬과 곱한 다음 결과 행렬 인덱스를 사용하여 뷰 행렬을 다시 설정합니다.
