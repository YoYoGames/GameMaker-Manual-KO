buffer_delete

이 기능을 사용하면 이전에 이 기능을 사용하여 생성한 버퍼를 삭제할 수 있습니다.buffer_create() 메모리에서 생성하는 데 사용된 리소스를 해제하고 현재 포함할 수 있는 모든 데이터를 제거합니다.

참고: 메모리 누수를 방지하기 위해 더 이상 필요하지 않을 때 동적으로 생성된 리소스를 항상 메모리에서 제거하는 것이 중요합니다.

중요한! 버퍼를 생성할 때 버퍼를 식별하는 데 사용되는 인덱스 값은 0부터 시작하는 정수 값입니다. 이러한 인덱스는 GameMaker에서 재사용하므로 소멸된 버퍼 인덱스 값은 나중에 새로 생성된 버퍼에서 사용할 수 있으며 다음을 권장합니다. 버퍼를 파괴한 후 버퍼 인덱스를 보유하는 모든 변수를 -1로 설정합니다.

통사론:

buffer_delete(buffer)

논쟁 유형 설명
buffer The index of the buffer to delete.

보고:

예시:

buffer_delete(player_buffer);
player_buffer = -1;

위의 코드는 "player_buffer" 변수에 저장된 id로 이전에 생성된 버퍼를 삭제하고 변수를 -1로 설정합니다.
