buffer_get_type

이 함수를 사용하면 버퍼 ID 값이 참조하는 버퍼 유형을 검색합니다. 반환 값은 다음 상수 중 하나입니다.

일정한 설명
buffer_fixed 고정 크기의 버퍼입니다.
buffer_grow 데이터가 추가될 때 동적으로 "증가"하는 버퍼
buffer_wrap 데이터가 "래핑"될 버퍼입니다. 추가되는 데이터가 버퍼 크기의 한계에 도달하면 덮어쓰기가 버퍼의 시작 부분에 다시 배치되고 그 지점에서 추가 쓰기가 계속됩니다.
buffer_fast 읽기/쓰기가 매우 빠른 특수 "제거된" 버퍼입니다. 와 함께만 사용할 수 있습니다. buffer_u8 데이터 유형 이며 1바이트로 정렬 되어야 합니다.
buffer_vbuffer 정점 버퍼

통사론:

buffer_get_type(buffer);

논쟁 유형 설명
buffer The index of the buffer to check.

보고:

Buffer Type Constant

예시:

type = buffer_get_type(buff);

위의 코드는 "buff" 변수에 인덱싱된 값에서 버퍼 유형을 가져와 변수에 저장합니다.
