buffer_md5

암호화에서 MD5(Message-Digest algorithm 5)는 128비트 해시 값을 가진 널리 사용되는 암호화 해싱 기능이며 다양한 보안 응용 프로그램에 사용되었습니다. 또한 일반적으로 데이터의 무결성을 확인하는 데 사용됩니다. 이 함수는 주어진 버퍼(또는 버퍼의 일부)에서 입력 데이터를 가져와서 해당 데이터에 고유한 32자의 16진수 MD5 해시를 반환합니다. 이러한 방식으로 저장하고 외부 서버(예: 외부 서버)로 보내거나 받는 정보의 무결성을 확인하는 데 사용할 수 있는 보안 키를 생성할 수 있습니다. 이 기능을 사용하여 버퍼에 이것을 적용할 때 사용할 버퍼의 버퍼 ID를 지정한 다음 시작할 위치에 대한 오프셋 값(바이트 단위), 해시할 영역의 크기(다시 바이트 단위)를 지정해야 합니다.

참고: MD5는 완전히 안전하지 않으며 파손될 수 있습니다. 자세한 내용은 이 페이지 를 참조하세요.

통사론:

buffer_md5(buffer, offset, size);

논쟁 유형 설명
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

보고:

예시:

check_string = buffer_md5(buff, 0, buffer_get_size(buff));

위의 코드는 "buff" 변수에 의해 인덱싱된 버퍼에 저장된 전체 데이터에 대한 md5 해시를 만들고 반환된 해시를 변수 "check_string"에 저장합니다.
