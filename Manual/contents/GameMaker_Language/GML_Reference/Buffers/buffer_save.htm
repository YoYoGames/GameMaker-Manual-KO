buffer_save

이 기능을 사용하면 버퍼의 내용을 파일에 저장할 수 있습니다.buffer_load() 함수.

참고: HTML5에서 버퍼의 내용은 이 기능을 사용할 때 base64로 인코딩된 문자열로 저장됩니다.

통사론:

buffer_save(buffer, filename);

논쟁 유형 설명
buffer The index of the buffer to save.
filename The name of the file to save as.

보고:

예시:

buffer_save(buff, "Player_Save.sav");

버퍼의 현재 내용을 "buff" 변수에 저장된 id로 파일에 저장합니다.
