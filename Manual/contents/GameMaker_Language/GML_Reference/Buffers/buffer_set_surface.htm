buffer_set_surface

이 기능을 사용하면 버퍼에서 주어진 표면으로 정보를 쓸 수 있습니다. 읽을 버퍼와 쓰기 대상 표면은 모두 이전에 생성되어야 하며 버퍼에 오프셋을 제공하여 읽기를 시작할 수 있습니다. 읽기는 항상 버퍼의 시작 부분에 오프셋 값을 더한 값에서 시작되며 현재 탐색 위치에 오프셋 값을 더한 위치에서는 시작 되지 않습니다.

참고: 이 기능은 GameMaker 업데이트 2.3.1부터 변경되었습니다. 이전 버전에서는 상수를 사용했습니다. buffer_surface_copy, 이제 더 이상 사용되지 않습니다.

통사론:

buffer_set_surface(buffer, surface, offset);

논쟁 유형 설명
buffer The index of the buffer to use.
surface The index of the surface to use.
offset The data offset value.

보고:

예시:

buffer_set_surface(buff, application_surface, 0);

위의 코드는 "buff" 변수에 인덱싱된 버퍼에 저장된 모든 데이터를 오프셋 없이 애플리케이션 표면에 복사합니다.
