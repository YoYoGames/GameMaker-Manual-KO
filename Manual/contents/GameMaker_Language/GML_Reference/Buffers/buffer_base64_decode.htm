buffer_base64_decode

이 함수를 사용하여 base64로 인코딩된 문자열을 디코딩할 수 있습니다.buffer_base64_encode() 함수)를 버퍼에 넣습니다. 이 함수는 버퍼를 생성하고(1바이트 정렬된 "성장" 버퍼로) 모든 추가 함수 호출에 사용해야 하는 버퍼에 대한 고유 인덱스를 반환합니다.

참고: 메모리 누수를 방지하기 위해 더 이상 필요하지 않을 때 이와 같이 동적으로 생성된 리소스를 메모리에서 제거하는 것이 중요합니다. 따라서 생성한 버퍼가 끝나면 다음을 사용하여 다시 해제해야 합니다.buffer_delete().

통사론:

buffer_base64_decode(string);

논쟁 유형 설명
string The base64 encoded string to decode

보고:

Buffer ID

예시:

ini_open("Save.ini");
buff = buffer_base64_decode(ini_read_string("Save", "Slot1", ""));
ini_close();

위의 코드는 ini 파일을 연 다음 이 파일에서 디코딩 함수로 문자열을 읽습니다. 이 함수는 이전에 인코딩 및 저장한 데이터를 포함하는 "buff" 변수에 저장된 버퍼 인덱스를 반환합니다. 그러면 ini 파일이 닫힙니다.
