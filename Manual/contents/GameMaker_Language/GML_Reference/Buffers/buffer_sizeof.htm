buffer_sizeof

이 함수는 주어진 버퍼 데이터 유형 상수 ( 여기 에 나열됨)의 크기(바이트)를 반환합니다.

통사론:

buffer_sizeof(type);

논쟁 유형 설명
type 버퍼 데이터 유형 상수 The type of data that is to be checked (see the list of constants here ).

보고:

Real

예시:

var b = 12 * buffer_sizeof(buffer_u8);
buff = buffer_create(b, buffer_fixed, 1);

위의 코드는 먼저 서명되지 않은 8비트 데이터 유형에 12를 곱하여 생성할 버퍼의 크기를 계산한 다음(버퍼를 사용하여 12개의 데이터를 보유할 것이기 때문에) 이 값을 사용하여 고정 버퍼를 올바른 값으로 설정합니다. 크기.
