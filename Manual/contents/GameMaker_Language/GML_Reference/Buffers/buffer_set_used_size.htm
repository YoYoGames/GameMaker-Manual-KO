buffer_set_used_size

이 함수는 주로 확장 내에서 사용하기 위한 것이며 주어진 버퍼의 "사용된" 크기, 즉 기록된 바이트 수를 설정할 수 있습니다.

확장 프로그램에서 버퍼에 데이터를 쓸 때 GameMaker는 확장 코드로 채워진 버퍼의 양을 알지 못하며 해당 데이터를 읽을 수 없습니다. 이 함수는 엔진이 버퍼에 기록된 데이터의 바이트 수를 알려주기 위해 확장 프로그램에서 호출할 수 있으므로 엔진이 해당 데이터를 읽을 수 있습니다.

통사론:

buffer_set_used_size(buffer, size);

논쟁 유형 설명
buffer The index of the buffer to use.
size The number of bytes to set as the "used" size.

보고:

예시:

buffer_write(_bufferAddress, buffer_u8, 1);
buffer_write(_bufferAddress, buffer_u8, 2);
buffer_write(_bufferAddress, buffer_u16, 400);

buffer_set_used_size(_bufferAddress, 4);

위의 함수는 확장에서 호출되어 일부 데이터를 GameMaker 버퍼에 쓴 다음(메모리 주소를 통해) 엔진이 해당 데이터를 읽을 수 있도록 기록된 바이트 수를 설정합니다.
