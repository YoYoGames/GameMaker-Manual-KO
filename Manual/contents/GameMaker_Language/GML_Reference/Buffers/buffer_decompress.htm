buffer_decompress

이 기능을 사용하면 zlib 압축 을 사용하여 이전에 압축된 버퍼의 압축을 풀 수 있습니다. 압축을 풀기 위해 버퍼 ID를 제공하면 함수는 압축되지 않은 데이터를 포함하는 새 버퍼 ID를 반환합니다. 압축 해제가 실패한 경우(예: 압축되지 않은 버퍼를 제공하는 경우) 함수는 대신 0보다 작은 값을 반환합니다.

통사론:

buffer_decompress(buffer);

논쟁 유형 설명
buffer The index of the buffer to decompress.

보고:

Buffer ID

예시:

var cmpBuff = buffer_load("Player_Save.sav");
var srcBuff = buffer_decompress(cmpBuff);
global.DataString = buffer_read(srcBuff, buffer_string);

위의 코드는 먼저 저장된 버퍼를 로드한 다음 압축을 풀고 마지막으로 압축 해제된 버퍼에서 전역 변수로 문자열 데이터를 읽습니다.
