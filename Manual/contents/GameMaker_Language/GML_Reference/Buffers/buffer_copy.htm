buffer_copy

이 함수는 한 버퍼에 저장된 데이터의 세그먼트(또는 전체)를 다른 버퍼로 복사하는 데 사용할 수 있습니다. 두 개의 버퍼를 사용하고 하나에서 다른 버퍼로 복사할 때 두 버퍼는 모두 이전에 다음을 사용하여 생성되어야 합니다.buffer_create() 함수(예:)를 사용하고 버퍼의 시작을 기준으로 소스 버퍼에서 복사할 데이터의 시작점에 대한 데이터 오프셋(바이트)과 위치를 정의하기 위한 다른 데이터 오프셋을 지정할 수 있습니다. 대상 버퍼에 데이터를 복사합니다.

참고: 동일한 버퍼에 복사할 수 없습니다.

통사론:

buffer_copy(src_buffer, src_offset, size, dest_buffer, dest_offset);

논쟁 유형 설명
src_buffer The index of the buffer to copy from.
src_offset The data offset to start copying from (in bytes).
size The size of the data to copy (in bytes).
dest_buffer The index of the buffer to copy to.
dest_offset The offset position to copy the data to (in bytes).

보고:

예시:

buffer_copy(buff1, 0, 2048, buff2, 2048);

위의 코드는 "buff1" 변수에 인덱싱된 버퍼에 저장된 데이터를 복사한 다음 변수 "buff2"에 인덱싱된 버퍼에 붙여넣지만 2048바이트만큼 오프셋됩니다.
