buffer_base64_encode

이 함수는 주어진 버퍼의 데이터를 base64 형식으로 인코딩된 문자열로 변환합니다. 이것은 인터넷을 통해 텍스트로 저장하거나 전송해야 하는 모든 미디어에 자주 사용되는 일반적으로 사용되는 인코딩 체계이며 출력을 사람의 눈으로 읽을 수 없도록 만듭니다. 이것을 사용하려면 인코딩할 버퍼 메모리의 크기(바이트)뿐만 아니라 이미 생성된 버퍼, 오프셋 값(버퍼 내에서 인코딩을 시작하려는 지점)을 지정해야 합니다.

통사론:

buffer_base64_encode(buffer, offset, size);

논쟁 유형 설명
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

보고:

예시:

var b_str = buffer_base64_encode(buff, 0, buffer_get_size(buff))

위의 코드는 "buff" 변수에 의해 인덱싱된 버퍼에 저장된 전체 데이터를 인코딩하고 반환된 문자열을 지역 변수 "b_str"에 저장합니다.
