buffer_load_ext

이 함수는 이전에 사용하여 저장한 버퍼 데이터를 로드합니다.buffer_save() 기존 버퍼로 기능합니다. 로드할 이전에 생성된 버퍼의 ID를 제공한 다음 로드할 저장된 버퍼 파일을 제공하고 마지막으로 데이터를 로드하려는 버퍼의 시작 부분에서 오프셋(바이트 단위)을 제공합니다.

읽어주세요 buffer_load() 플랫폼별 참고 사항에 대한 페이지입니다.

통사론:

buffer_load_ext(buffer, filename, offset);

논쟁 유형 설명
buffer The index of the buffer to load into.
filename The name of the file to load from.
offset The offset within the buffer to load to (in bytes).

보고:

예시:

var pos = buffer_seek(player_buffer, buffer_seek_end, 0);
buffer_load(player_buffer, "Data_Save.sav", pos);

위의 코드는 먼저 "player_buffer" 변수에 인덱싱된 버퍼의 끝 위치를 가져온 다음 주어진 위치에서 데이터를 로드합니다(이 예제는 "grow" 또는 "wrap" 버퍼 유형에서만 작동합니다. ).
