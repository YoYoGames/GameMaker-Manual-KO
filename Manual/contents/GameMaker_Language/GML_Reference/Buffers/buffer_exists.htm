buffer_exists

이 함수는 변수가 유효한 버퍼 ID 값을 보유하고 있는지 여부를 확인하는 데 사용할 수 있습니다. 그렇게하면 함수가 반환됩니다. true 그렇지 않으면 반환됩니다 false.

통사론:

buffer_exists(buffer)

논쟁 유형 설명
buffer The index of the buffer to check.

보고:

예시:

if buffer_exists(buff)
{
    buffer_delete(buff);
}

위의 코드는 변수 "buff "는 버퍼 ID를 보유하고 있는 경우 버퍼가 삭제됩니다.
