buffer_base64_decode_ext

이 함수를 사용하여 base64로 인코딩된 문자열을 디코딩할 수 있습니다.buffer_base64_encode() 함수)를 버퍼에 넣습니다. 기능과 달리buffer_base64_decode(), 이것은 당신을 위한 버퍼를 생성 하지 않을 것이지만, 오히려 당신은 이미 버퍼를 생성했어야 합니다(참조buffer_create() ), 이 함수와 함께 사용할 ID입니다. "오프셋"은 주어진 문자열(바이트 단위)을 디코딩할 버퍼 내 위치입니다.

통사론:

buffer_base64_decode_ext(buffer, string, offset);

논쟁 유형 설명
buffer The index of the buffer to decode the string into.
string The base64 encoded string to decode.
offset The data offset value.

보고:

예시:

buff = buffer_create(16384, buffer_grow, 2);
ini_open("Save.ini");
var str = ini_read_string("Save", "Slot1", "");
buffer_base64_decode_ext(buff, str, 0);
ini_close();

위의 코드는 버퍼를 생성하고 변수 "buff"에 버퍼에 대한 고유 ID를 저장한 다음 ini 파일을 열고 문자열을 로컬 변수 "str"로 읽어들입니다. 이 문자열은 ini 파일을 다시 닫기 전에 새로 생성된 버퍼로 디코딩됩니다.
