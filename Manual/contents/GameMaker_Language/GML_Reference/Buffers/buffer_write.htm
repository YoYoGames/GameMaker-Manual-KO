buffer_write

이 함수는 이전에 생성된 버퍼에 데이터를 쓰는 데 사용할 수 있습니다. 쓰는 데이터는 이 함수의 "유형" 인수와 일치해야 합니다. 즉, 예를 들어 문자열을 부호 없는 16비트 정수로 쓰려고 할 수 없습니다. 다음 상수를 사용하여 데이터 유형을 정의할 수 있습니다.

버퍼 데이터 유형 상수
일정한 설명
buffer_u8 부호 없는 8비트 정수입니다. 0에서 255 사이의 양수 값입니다.
buffer_s8 부호 있는 8비트 정수입니다. -128에서 127 사이의 양수 또는 음수 값일 수 있습니다(0은 양수로 분류됨).
buffer_u16 부호 없는 16비트 정수입니다. 0 - 65,535 사이의 양수 값입니다.
buffer_s16 부호 있는 16비트 정수입니다. -32,768에서 32,767 사이의 양수 또는 음수 값일 수 있습니다(0은 양수로 분류됨).
buffer_u32 부호 없는 32비트 정수입니다. 0에서 4,294,967,295 사이의 양수 값입니다.
buffer_s32 부호 있는 32비트 정수입니다. 이것은 -2,147,483,648에서 2,147,483,647 사이의 양수 또는 음수 값일 수 있습니다(0은 양수로 분류됨).
buffer_u64 부호 없는 64비트 정수입니다.
buffer_f16 16비트 부동 소수점. 이는 16비트 부호 있는 정수와 동일한 범위 내에서 양수 또는 음수 값일 수 있습니다.
buffer_f32 32비트 부동 소수점. 이는 32비트 부호 있는 정수와 동일한 범위 내에서 양수 또는 음수 값일 수 있습니다.
buffer_f64 64비트 부동 소수점.
buffer_bool 부울 값입니다. 1 또는 0(true 또는 false )
buffer_string 널 종료 문자로 마무리되는 모든 크기의 문자열입니다.
buffer_text 최종 null 종료 문자가 없는 모든 크기의 문자열입니다.

함수는 성공하면 0을 반환하고 실패하면 -1을 반환합니다.

통사론:

buffer_write(buffer, type, value)

논쟁 유형 설명
buffer 버퍼 ID The index of the buffer to write to.
type 진짜 The type of data that is to be written to the buffer (see the list of constants above).
value 진짜 The data to write.

보고:

Real (0 if success, or -1 if it fails)

예시:

buffer_seek(buff, buffer_seek_start, 0);
buffer_write(buff, buffer_s16, 0);
buffer_write(buff, buffer_s16, x);
buffer_write(buff, buffer_s16, y);

위의 코드는 id가 변수 "buff"에 저장된 버퍼의 시작 부분을 찾아 일련의 부호 있는 16비트 정수 값을 기록합니다.
