buffer_crc32

이 함수는 버퍼에서 입력 데이터를 가져와 crc32 체크섬 해시를 반환합니다. 사용할 버퍼의 버퍼 ID를 지정한 다음 시작 위치에 대한 오프셋 값(바이트 단위)을 지정한 다음 해시할 영역의 크기(바이트 단위)를 지정하면 함수가 32비트 정수 값을 반환합니다. 그 지역을 위해.

통사론:

buffer_crc32(buffer, offset, size);

논쟁 유형 설명
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

보고:

예시:

check_val = buffer_crc32(buff, 0, buffer_get_size(buff));

위의 코드는 "buff" 변수에 의해 인덱싱된 버퍼에 저장된 전체 데이터에 대한 crc32 체크섬 해시 값을 만들고 반환된 정수 해시 값을 변수 "check_val"에 저장합니다.
