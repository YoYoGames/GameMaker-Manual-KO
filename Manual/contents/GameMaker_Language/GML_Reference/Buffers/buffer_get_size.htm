buffer_get_size

이 함수를 사용하면 주어진 버퍼의 크기를 바이트 단위로 얻을 수 있습니다.

통사론:

buffer_get_size(buffer);

논쟁 유형 설명
buffer 버퍼 ID The index of the buffer to get the size of.

보고:

Real

예시:

var _size = buffer_get_size(player_data);
var _temp = buffer_create(_size, buffer_fixed, 0);

위의 코드는 새 버퍼를 만들고 해당 인덱스를 지역 변수 "_temp"에 저장합니다. 이 새 버퍼의 크기는 변수 "player_data"에 인덱싱된 버퍼의 크기와 동일합니다.
