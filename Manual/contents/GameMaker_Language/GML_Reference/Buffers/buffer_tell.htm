buffer_tell

다음을 사용하여 버퍼에 데이터를 읽거나 쓸 때buffer_read() 또는buffer_write() 현재 "탐색" 위치는 쓰거나 읽은 바이트만큼 진행되며 이 기능을 사용하면 다른 버퍼 기능에서 사용하기 위해 현재 "탐색" 위치를 얻을 수 있습니다. 예를 들어, 버퍼 정렬이 4바이트로 설정되고 크기가 1바이트인 단일 데이터 조각을 작성하는 경우 다음을 수행합니다. buffer_tell(), 반환 값은 1입니다. 그러나 크기가 1바이트인 다른 데이터 조각을 작성하는 경우 다음을 수행합니다. buffer_tell(), 정렬이 해당 위치에 데이터를 "채워" 있으므로 반환 값 5를 얻습니다.

통사론:

buffer_tell(buffer);

논쟁 유형 설명
buffer The index of the buffer to use.

보고:

예시:

var pos = buffer_tell(buff); buffer_seek(buff, buffer_seek_start, 0);
val[0] = buffer_read(buff, buffer_S16);
val[1] = buffer_read(buff, buffer_S16);
val[2] = buffer_read(buff, buffer_S16);
buffer_seek(buff, buffer_seek_start, pos);

위의 코드는 "buff" 변수에 색인된 버퍼 내의 현재 탐색 위치를 지역 변수 "pos"에 저장합니다. 그런 다음 버퍼 탐색 위치가 버퍼의 시작으로 설정되고 마지막으로 버퍼 탐색 위치를 이전 위치로 재설정하기 전에 3개의 데이터를 어레이로 읽어들입니다.
