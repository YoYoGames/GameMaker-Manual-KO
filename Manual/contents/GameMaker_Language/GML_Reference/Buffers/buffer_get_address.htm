buffer_get_address

이 함수를 사용하면 정렬된 원시 버퍼 주소에 대한 포인터 를 얻을 수 있습니다. 이 값을 확장에 전달하여 버퍼 데이터에 액세스할 수 있도록 하기 때문에 이것은 주로 확장과 함께 사용하기 위한 것입니다. 확장 프로그램에서 액세스할 때 버퍼로 할 수 없는 몇 가지 사항이 있습니다.

  • 메모리를 해제할 수 없습니다.
  • 메모리 크기를 조정할 수 없습니다.
  • 버퍼 주소에 버퍼 크기를 더한 값을 초과하여 쓸 수 없습니다.buffer_get_size() 기능) 또는 범위를 벗어난 오류가 발생합니다.

통사론:

buffer_get_address(buffer);

논쟁 유형 설명
buffer The index of the buffer to use.

보고:

예시:

var b_address = buffer_get_address(buff_model);
var b_end_address = b_address + buffer_get_size(buff_model);

위의 코드는 "buff_model" 변수에 색인된 버퍼의 메모리 주소를 가져온 다음 다음을 사용하여 버퍼 끝의 메모리 주소를 가져옵니다. buffer_get_size() 추가 사용을 위해 두 값을 모두 지역 변수에 저장합니다.
