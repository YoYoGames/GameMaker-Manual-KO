buffer_sha1

암호화에서 SHA-1은 미국 국가안보국(National Security Agency)에서 설계한 암호화 해싱 기능이며 파일 변경 사항을 확인하는 데 사용되는 인기 있는 Git 과 같이 널리 사용되는 여러 응용 프로그램 및 프로토콜에 사용됩니다. 이 함수는 버퍼에서 입력 데이터를 가져와 ASCII 형식의 160비트 메시지 다이제스트를 반환합니다. 이러한 방식으로 저장하고 외부 서버(예: 외부 서버)로 보내거나 받는 정보의 무결성을 확인하는 데 사용할 수 있는 보안 키를 생성할 수 있습니다. 이 기능을 사용하여 버퍼에 이것을 적용할 때 사용할 버퍼의 버퍼 ID를 지정한 다음 시작할 위치에 대한 오프셋 값(바이트 단위), 해시할 영역의 크기(다시 바이트 단위)를 지정해야 합니다.

참고: SHA-1은 완전히 안전하지 않으며 깨질 수 있습니다. 자세한 내용은 이 페이지 를 참조하세요.

통사론:

buffer_sha1(buffer, offset, size);

논쟁 유형 설명
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

보고:

예시:

check_string = buffer_sha1(buff, 0, buffer_get_size(buff));

위의 코드는 "buff" 변수로 인덱싱된 버퍼에 저장된 전체 데이터에 대한 sha1 해시를 만들고 반환된 해시를 "check_string" 변수에 저장합니다.
