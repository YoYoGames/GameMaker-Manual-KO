buffer_copy_from_vertex_buffer

이 함수는 하나의 정점 버퍼에 저장된 정점 데이터의 일부(또는 전체)를 이전에 생성된 일반 버퍼로 복사하는 데 사용할 수 있습니다. 이 함수를 사용하여 정점 버퍼에서 일반 버퍼로 복사할 때 두 버퍼는 모두 이전에 생성되어야 합니다(vertex_create_buffer() 그리고buffer_create() 기능). 버퍼에 복사하려는 정점 데이터의 범위를 지정할 수 있습니다. 여기서 시작 정점은 0과 정점 수 -1 사이일 수 있으며 해당 지점에서 복사할 정점 수를 지정할 수 있습니다. 기능을 사용할 수 있습니다vertex_get_number() 정점 버퍼에 저장된 총 정점 수를 가져옵니다. 마지막으로 정점 데이터를 복사할 버퍼 인덱스와 대상 버퍼에서 정점 데이터를 복사할 위치를 정의하는 데이터 오프셋을 제공합니다.

통사론:

buffer_copy_from_vertex_buffer(vertex_buffer, start_vertex, num_vertices, dest_buffer, dest_offset);

논쟁 유형 설명
vertex_buffer 정점 버퍼 ID The index of the vertex buffer to copy from.
start_vertex 진짜 The starting vertex.
num_vertices 진짜 The total number of vertices to use.
dest_buffer 버퍼 ID The index of the buffer to copy to.
dest_offset 진짜 The offset position to copy the data to (in bytes).

예시:

var v_num = vertex_get_number(model_buff); buffer_copy_from_vertex_buffer(model_buffer, 0, v_num - 1, player_buffer, 0);

위의 코드는 "model_buffer" 변수에 인덱싱된 꼭짓점 버퍼에 저장된 꼭짓점 데이터를 복사한 다음 변수 "player_buffer"에 인덱싱된 버퍼에 붙여넣습니다.
