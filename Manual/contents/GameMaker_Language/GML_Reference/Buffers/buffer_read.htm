buffer_read

이 함수는 이전에 생성된 버퍼에서 데이터를 읽는 데 사용할 수 있습니다. 반환 값은 읽고 있는 데이터의 유형에 따라 달라지며, 그 자체로 다음 상수로 정의됩니다.

끊임없는 설명
buffer_u8 부호 없는 8비트 정수입니다. 0에서 255 사이의 양수 값입니다.
buffer_s8 부호 있는 8비트 정수입니다. -128에서 127 사이의 양수 또는 음수 값일 수 있습니다(0은 양수로 분류됨).
buffer_u16 부호 없는 16비트 정수입니다. 이것은 0 - 65,535 사이의 양수 값입니다.
buffer_s16 부호 있는 16비트 정수입니다. -32,768에서 32,767 사이의 양수 또는 음수 값일 수 있습니다(0은 양수로 분류됨).
buffer_u32 부호 없는 32비트 정수입니다. 0에서 4,294,967,295 사이의 양수 값입니다.
buffer_s32 부호 있는 32비트 정수입니다. 이것은 -2,147,483,648에서 2,147,483,647 사이의 양수 또는 음수 값일 수 있습니다(0은 양수로 분류됨).
buffer_u64 부호 없는 64비트 정수입니다.
buffer_f16 16비트 부동 소수점. +/- 65504 범위 내에서 양수 또는 음수 값일 수 있습니다. (현재 지원되지 않습니다!)
buffer_f32 32비트 부동 소수점. +/-16777216 범위 내에서 양수 또는 음수 값일 수 있습니다.
buffer_f64 64비트 부동 소수점.
buffer_bool 부울 값입니다. 1 또는 0(true 또는 false )
buffer_string 모든 크기의 문자열입니다.
buffer_text 최종 null 종료 문자가 없는 모든 크기의 문자열입니다.

함수가 성공하면 주어진 유형의 값을 반환하지만 실패하면 러너 오류가 발생합니다.

참고 읽고 있는 데이터에 잘못된 데이터 유형을 사용하면 잘못된 값이 반환됩니다.

통사론:

buffer_read(buffer, type)

논쟁 유형 설명
buffer 버퍼 ID The index of the buffer to read from.
type 버퍼 데이터 유형 상수 The type of data that is to be read from the buffer (see the list of constants above).

보고:

Real, Boolean or String

예시:

var cmd = buffer_read(buff, buffer_s16);

위의 코드는 "buff" 변수에 저장된 id를 가진 버퍼에서 부호 있는 16비트 값을 지역 변수 "cmd"로 읽어옵니다.
