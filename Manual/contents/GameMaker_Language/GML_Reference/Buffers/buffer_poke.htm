buffer_poke

와 더불어buffer_write() 기능을 사용하면 현재 "탐색" 위치에서 주어진 버퍼에 데이터를 쓸 수 있으며, 각 데이터 조각은 쓰거나 읽는 바이트만큼 이 위치로 이동합니다. 그러나 현재 탐색 위치를 변경하지 않고 주어진 데이터를 변경해야 할 수도 있으며 이때 이 기능을 사용합니다. 함수에 버퍼 인덱스를 제공한 다음 쓸 데이터 유형 및 값뿐만 아니라 쓸 해당 버퍼 내에서 버퍼 시작(바이트 단위)의 오프셋 위치를 제공합니다.

통사론:

buffer_poke(buffer, offset, type, value);

논쟁 유형 설명
buffer 버퍼 ID The index of the buffer to use.
offset 진짜 The offset position (in bytes) within the buffer to write the given data to.
type 버퍼 데이터 유형 상수 The type of data that is to be written to the buffer (see the list of constants here ).
value 변하기 쉬운 The data to add to the buffer, in accordance with the type specified.

보고:

N/A

예시:

buffer_poke(buff, 3, buffer_u8, colour_get_blue(image_blend));

위의 코드는 이미지 블렌드에 사용된 색상의 파란색 구성 요소 값을 "buff" 변수에 인덱스된 버퍼에 추가하고 버퍼의 세 번째 위치에 있고 부호 없는 8비트 값으로 추가합니다.
