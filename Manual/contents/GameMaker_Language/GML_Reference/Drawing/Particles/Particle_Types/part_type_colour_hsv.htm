part_type_colour_hsv

이 기능을 사용하면 주어진 유형의 모든 입자에 대한 색조, 채도 및 값 범위를 설정할 수 있습니다. 세 가지 구성 요소 각각에 대해 최소값과 최대값을 제공하면 생성된 입자는 지정된 매개변수 범위에 따라 임의의 색상을 갖게 됩니다. 이러한 방식으로 색상은 같지만 채도가 다른 입자 또는 색상은 다르지만 값(광도)이 같은 입자를 만들 수 있습니다. 모든 값은 0에서 255 사이여야 합니다.

통사론:

part_type_colour_hsv(ind, hmin, hmax, smin, smax, vmin, vmax);

논쟁 유형 설명
ind 입자 유형 ID The index of the particle type to change.
hmin 진짜 The minimum the final colour's hue component can be.
hmax 진짜 The maximum the final colour's hue component can be.
smin 진짜 The minimum the final colour's saturation component can be.
smax 진짜 The maximum the final colour's saturation component can be.
vmin 진짜 The minimum the final colour's value component can be.
vmax 진짜 The maximum the final colour's value component can be.

보고:

N/A

예시:

part_type_hsv(global.Stars, 0, 255, 0, 255, 255, 255 );

위의 코드는 전역 변수 "Stars"에 인덱싱된 입자 유형에서 방출되는 각 입자를 다른 색상과 채도를 가지지만 동일한 값(광도)을 갖도록 설정합니다.
