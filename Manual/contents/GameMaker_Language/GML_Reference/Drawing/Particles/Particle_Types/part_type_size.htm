part_type_size

이 함수는 입자가 생성될 때 입자의 크기를 결정하는 데 사용되며 수명 동안 입자 크기를 늘리거나 줄이는 데 사용할 수도 있습니다. 최소 및 최대 크기 값의 기본값은 1입니다(크기는part_type_scale() 스케일이 설정됨) 및 동일한 값으로 설정되면 모든 입자의 초기 크기는 동일하지만 다른 값으로 설정하면 각 입자는 범위 내에서 임의의 크기로 시작합니다. 예를 들어 다음과 같은 경우 최소 0.5와 최대 2를 가지면 모든 입자는 기본 크기의 절반에서 두 배 사이의 크기로 생성됩니다.

또한 입자의 수명 동안 입자 스프라이트가 커지거나(음수인 경우) 줄어들도록(음수인 경우) 증가 값을 설정할 수 있으며, 주어진 양은 각 단계마다 입자 크기에 추가됩니다. 이 값은 최소 (+/-)0.01일 수 있습니다.

마지막으로 "흔들기" 요소를 설정할 수 있습니다. 파티클 수명의 각 단계마다 크기에 무작위로 더하거나 빼는 값입니다. 분명히 큰 값은 작은 값보다 더 뚜렷하며 이 값은 최대 범위가 -20에서 20 사이인 음수일 수도 있습니다.

통사론:

part_type_size(ind, size_min, size_max, size_incr, size_wiggle);

논쟁 유형 설명
ind 입자 유형 ID The index of the particle type to change.
size_min 진짜 The minimum size the particle can start at.
size_max 진짜 The maximum size the particle can start at.
size_incr 진짜 How much the particle should increase or decrease per step.
size_wiggle 진짜 How much should randomly be added or subtracted from the particles size per step.

보고:

N/A

예시:

part_type_shape(particle1, pt_shape_explosion);
part_type_size(particle1, 2, 2, 0.10, 0.01);
part_type_scale(particle1, 1, 1);
part_type_colour2(particle1, 16777215, 12632256);
part_type_alpha3(particle1, 0, 0.20, 0);
part_type_speed(particle1, 0.20, 0.20, 0, 0);
part_type_direction(particle1, 0, 359, 0, 20);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 0);
part_type_life(particle1, 40, 60);

위의 코드는 각 파티클이 생성될 때 크기가 두 배로 설정되는 크기를 포함하여 다양한 파티클 값을 설정한 다음 0.1의 흔들림으로 각 단계마다 크기를 0.01씩 증가시킵니다.
