part_type_colour_mix

이 기능을 사용하면 주어진 입자 유형을 두 가지 색상의 무작위 혼합으로 설정할 수 있습니다.

통사론:

part_type_colour_mix(ind, colour1, colour2);

논쟁 유형 설명
ind 입자 유형 ID The index of the particle type to change.
colour1 색깔 The first colour the blend will take from.
colour2 색깔 The second colour the blend will take from.

보고:

N/A

예시:

part_type_colour_mix(global.P_Damage, c_red, c_yellow);

위의 코드는 전역 변수 "P_Damage"에 인덱싱된 입자 유형에서 방출되는 각 입자의 색상을 빨간색과 노란색 사이의 무작위 혼합으로 설정합니다.
