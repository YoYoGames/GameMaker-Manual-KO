part_type_life

이것은 표시된 유형의 각 개별 입자가 화면에 남아 있는 시간을 제어하는 기능입니다. 다른 입자 함수와 마찬가지로 최소값과 최대값을 단계적으로 제공하고 각 입자 수명은 지정된 범위 내에서 임의의 단계 수입니다. 모든 입자가 동일한 수명을 갖도록 하려면 두 값을 동일하게 설정하십시오.

통사론:

part_type_life(ind, life_min, life_max);

논쟁 유형 설명
ind 입자 유형 ID The index of the particle type to change.
life_min 진짜 The minimum lifespan of the particles.
life_max 진짜 The maximum lifespan of the particles.

보고:

N/A

예시:

part_type_life(part_Flare, room_speed, room_speed * 2);

위의 코드는 "part_Flare" 변수에 인덱싱된 입자 유형으로 생성된 모든 입자를 현재 실내 속도를 기준으로 1초에서 2초 사이의 수명을 갖도록 설정합니다.
