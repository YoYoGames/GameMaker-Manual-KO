part_type_create

이 함수를 사용하면 새 입자 유형을 만들 수 있으며 반환 값은 모든 후속 입자 기능에서 사용할 수 있도록 변수에 저장해야 합니다.

통사론:

part_type_create();

보고:

Particle Type ID

예시:

mypart = part_type_create();

그러면 "mypart" 변수에 인덱스를 저장하는 새 입자 유형이 생성됩니다.
