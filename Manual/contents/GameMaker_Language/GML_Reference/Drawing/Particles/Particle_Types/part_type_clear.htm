part_type_clear

이 기능을 사용하면 입자를 "재설정"하여 입자(수명, 색상, 알파, 방향 등)와 관련된 각 기능의 모든 값을 기본값으로 되돌릴 수 있습니다. 이 기능은 현재 방에 보이는 입자를 화면에서 제거하지 않습니다.part_particles_clear().

통사론:

part_type_clear(ind);

논쟁 유형 설명
ind 입자 유형 ID The index of the particle type to clear.

보고:

N/A

예시:

part_type_clear(global.explode_part);

위의 코드는 전역 변수 "explode_part"에 인덱싱된 입자 유형을 기본값으로 지웁니다.
