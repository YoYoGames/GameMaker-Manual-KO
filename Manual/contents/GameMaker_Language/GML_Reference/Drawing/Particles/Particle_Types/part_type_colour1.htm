part_type_colour1

이 기능은 각 개별 입자의 전체 수명 기간 동안 입자 유형을 단일 색상으로 설정하는 데 사용됩니다.

통사론:

part_type_colour1(ind, colour1);

논쟁 유형 설명
ind 입자 유형 ID The index of the particle type to change.
colour1 색깔 The single colour to make the particle type.

보고:

N/A

예시:

part_type_colour1(global.Snow_Part, c_white);

위의 코드는 전역 변수 "Snow_Part"에 인덱싱된 입자 유형으로 생성된 모든 입자를 흰색으로만 설정합니다.
