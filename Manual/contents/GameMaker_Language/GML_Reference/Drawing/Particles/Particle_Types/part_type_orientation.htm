part_type_orientation

이 함수는 생성될 때 파티클 스프라이트의 방향을 결정하는 데 사용되며 수명 동안 파티클 방향을 증가 또는 감소시키는 데에도 사용할 수 있습니다. 최소 및 최대 방향 값의 기본값은 0이지만 이 값은 0도가 오른쪽, 90도가 위쪽, 180도가 왼쪽, 270도가 아래쪽인 표준 GameMaker 방향에 따라 방향을 임의로 변경하도록 변경할 수 있습니다. 동일한 값으로 설정하면 입자가 모두 동일한 방향으로 생성됩니다.

또한 수명 동안 방향에 각도를 더하거나(음수인 경우) 더하는 증분 값을 설정할 수도 있습니다. 이 값은 최소 (+/-) 0.01일 수 있습니다.

다른 입자 기능과 마찬가지로 "흔들기" 요소도 설정할 수 있습니다. 이것은 입자 수명의 각 단계에서 방향에 무작위로 더하거나 빼는 값입니다. 분명히 큰 값은 작은 값보다 더 뚜렷하며 이 값은 최대 범위가 -20에서 20 사이인 음수일 수도 있습니다.

마지막으로 방향이 상대적인지 여부를 선택할 수 있습니다. 즉, 입자에 방향(및 속도)이 있는 동안 입자 스프라이트는 해당 벡터를 중심으로 방향이 지정됩니다. 파티클 속도를 줄이도록 설정하고 0에 도달하면 속도가 없으면 방향이 기본값인 0°로 설정되므로 상대 방향으로 인해 파티클 스프라이트가 다른 각도로 "점프"됩니다..

통사론:

part_type_orientation( ind, ang_min, ang_max, ang_incr, ang_wiggle, ang_relative );

논쟁 유형 설명
ind 입자 유형 ID The index of the particle type to change.
ang_min 진짜 The minimum starting angle of the particles.
ang_max 진짜 The maximum starting angle of the particles.
ang_incr 진짜 The increase (or decrease if set negative) of the angle in degrees per step.
ang_wiggle 진짜 The amount the angle wiggles clockwise/counterclockwise per step.
ang_relative 진짜 Whether to set its angle relative to the direction of the particle's motion (true) or not (false).

보고:

N/A

예시:

part_type_shape(particle2, pt_shape_spark);
part_type_size(particle2, 0.10, 0.50, 0.01, 0);
part_type_scale(particle2, 0.30, 0.30);
part_type_colour1(particle2, 8454143);
part_type_alpha1(particle2, 0.50);
part_type_speed(particle2, 4, 4, -0.07, 1);
part_type_direction(particle2, 0, 359, 0, 20);
part_type_orientation(particle2, 0, 359, 0, 20, 1);
part_type_blend(particle2, 1);
part_type_life(particle2, 1, 5);

위의 코드는 0º에서 359º 사이의 임의 값이 되는 방향을 포함하여 다양한 입자 값을 설정합니다. 또한 각 단계마다 0에서 20 사이의 임의의 양이 추가되며 방향은 모션 방향에 상대적입니다.
