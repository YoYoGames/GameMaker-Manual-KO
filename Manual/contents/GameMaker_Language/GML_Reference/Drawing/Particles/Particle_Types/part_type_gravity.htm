part_type_gravity

이 함수는 생성된 주어진 유형의 각 입자에 영향을 미치는 중력을 설정합니다. 중력 강도 값은 매 단계마다 입자 속도에 추가되며 일반적으로 0.5와 같은 작은 값인 반면 방향은 중력이 "당기는" 방향이며 0°는 오른쪽, 90°는 위쪽의 표준 GameMaker 방향을 따릅니다. 왼쪽이 180°이고 아래가 270°입니다.

통사론:

part_type_gravity(ind, grav_amount, grav_direction);

논쟁 유형 설명
ind 입자 유형 ID The index of the particle type to change.
grav_amount 진짜 Strength of the gravity.
grav_direction 진짜 The direction of the gravity.

보고:

N/A

예시:

part_type_shape(global.p1, pt_shape_pixel);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

위의 코드는 270º 방향으로 각 단계의 속도에 0.2를 추가하는 중력을 포함한 다양한 입자 값을 설정하므로 입자를 화면 아래쪽으로 "아래로" 당깁니다.
