part_type_speed

이 함수는 파티클이 생성될 때의 이동 속도를 설정하는 데 사용되며, 파티클이 수명 동안 속도를 높이거나 낮추는 데에도 사용할 수 있습니다. 최소 및 최대 속도 값의 기본값은 1입니다. 즉, 파티클은 각 단계마다 1픽셀씩 이동하며 동일한 값으로 설정하면 모든 파티클의 초기 속도는 동일하지만 다른 값으로 설정하면 각 입자는 지정된 범위 내에서 임의의 속도로 시작합니다. 예를 들어 최소값이 3이고 최대값이 6인 경우 모든 입자는 단계당 3~6픽셀 사이의 속도로 생성됩니다.

또한 입자가 수명 동안 더 빠르게(양수인 경우) 또는 느리게(음수인 경우) 이동하도록 증분 값을 설정할 수 있으며, 주어진 양은 각 단계마다 입자 속도에 추가됩니다. 이 값은 최소 (+/-)0.01일 수 있습니다.

마지막으로 "흔들기" 요소를 설정할 수 있습니다. 파티클 수명 단계마다 속도에 무작위로 더하거나 빼는 값입니다. 분명히 큰 값은 작은 값보다 더 뚜렷하며 이 값은 최대 범위가 -20에서 20 사이인 음수일 수도 있습니다.

통사론:

part_type_speed(ind, speed_min, speed_max, speed_incr, speed_wiggle);

논쟁 유형 설명
ind 입자 유형 ID The index of the particle type to change.
speed_min 진짜 The minimum speed the particle can start at.
speed_max 진짜 The maximum speed the particle can start at.
speed_incr 진짜 How much the particle speed should increase or decrease per step.
speed_wiggle 진짜 How much should randomly be added or subtracted from the particle's speed per step.

보고:

N/A

예시:

part_type_shape(particle2, pt_shape_flare);
part_type_size(particle2, 0.01, 0.50, 0.01, 20);
part_type_scale(particle2, 0.10, 0.10);
part_type_colour3(particle2, 4210816, 4227327, 8240895);
part_type_alpha3(particle2, 0.20, 0.80, 0);
part_type_speed(particle2, 5, 10, -0.20, 20);
part_type_direction(particle2, 0, 359, -1, 20);
part_type_gravity(particle2, 0, 270);
part_type_orientation(particle2, 0, 359, 0, 20, 1);
part_type_blend(particle2, 1);
part_type_life(particle2, 1, 20);

위의 코드는 최소 속도를 5로 설정하고 최대 속도를 10으로 설정하는 속도를 포함하여 다양한 입자 값을 설정하고 속도는 각 단계마다 -0.2만큼 느려집니다. 속도는 또한 각 단계에서 0에서 20 사이의 임의 값으로 흔들립니다.
