part_type_alpha1

이 함수는 각 개별 입자의 전체 수명 기간 동안 단일 알파 값(투명도)을 갖도록 입자 유형을 설정하는 데 사용되며 0(투명)에서 1(불투명) 사이일 수 있습니다.

통사론:

part_type_alpha1(ind, alpha1);

논쟁 유형 설명
ind 입자 유형 ID The index of the particle type to change.
alpha1 진짜 The alpha of the particle.

보고:

N/A

예시:

part_type_alpha1(global.Snow_Part, 0.5);

위의 코드는 전역 변수 "Snow_Part"에 인덱싱된 입자 유형으로 생성된 모든 입자를 알파 값 0.5(반투명)로 설정합니다.
