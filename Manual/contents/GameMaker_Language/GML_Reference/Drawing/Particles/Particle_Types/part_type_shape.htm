part_type_shape

이 기능을 사용하면 다음 상수가 허용되는 파티클 유형에 사용할 스프라이트 모양을 설정할 수 있습니다.

입자 모양 상수
일정한 설명
pt_shape_pixel 1x1 픽셀. (기본 설정입니다.)
pt_shape_disk 채워진 원입니다.
pt_shape_square 채워진 광장.
pt_shape_line 8px 너비의 수평선.
pt_shape_star 5점 만점의 별입니다.
pt_shape_circle 3px 윤곽이 있는 원.
pt_shape_ring 내부 광선이 있는 원(거품처럼 보임).
pt_shape_sphere 바깥쪽으로 빛나는 ' 중앙에 단단한 원, 바깥쪽으로 빛나는.
pt_shape_flare 심하게 빛나는 점(밤에 실제 별처럼 보임).
pt_shape_spark 여러 점이 사라지는 별과 같은 불꽃 효과.
pt_shape_explosion 사각형 연기 구름'은 폭발을 닮기 위해 여러 색상이 필요합니다.
pt_shape_cloud 얇은 구름은 구름을 닮기 위해 확장과 여러 입자가 필요합니다.
pt_shape_smoke 폭발 효과의 부드러운 버전입니다. 여러 개를 사용하여 연기 구름을 만듭니다.
pt_shape_snow 일반적인 눈송이 모양.

통사론:

part_type_shape(ind, shape);

논쟁 유형 설명
ind 입자 유형 ID The index of the particle type to set.
shape 입자 모양 상수 The shape to choose.

보고:

N/A

예시:

global.p1 = part_type_create();
part_type_shape(global.p1, pt_shape_square);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

위의 코드는 전역 변수 "p1"에 인덱싱된 입자 유형을 정사각형 모양으로 설정한 다음 다양한 기타 입자 속성을 설정합니다.
