part_type_destroy

이 기능을 사용하면 게임에서 지정된 입자 유형을 제거할 수 있습니다. 이 기능을 사용하면 주어진 유형의 모든 입자가 방에서 사라지고 입자 자체가 메모리에서 제거되므로 이 기능은 더 이상 입자가 필요하지 않을 때만 사용해야 합니다.

통사론:

part_type_destroy(ind);

논쟁 유형 설명
ind 입자 유형 ID The index of the particle type to destroy.

보고:

N/A

예시:

if part_particles_count(smoke_sys) = 0
{
    part_type_destroy(smoke_part);
    part_system_destroy(smoke_sys);
    instance_destroy();
}
else alarm[0] = 1;

위의 코드는 파티클 시스템을 검사하여 현재 방에 파티클이 있는지 확인하고 그렇지 않은 경우 파티클, 시스템 및 인스턴스가 파괴됩니다.
