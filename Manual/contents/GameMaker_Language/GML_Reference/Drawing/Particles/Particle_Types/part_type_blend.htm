part_type_blend

이 기능을 사용하면 추가 혼합 모드로 입자를 그릴 수 있습니다(true ) 여부(false ). 가산 혼합은 각 입자의 광도 값을 겹칠 때 추가하여 밝은 색상이 겹칠 때 점차 밝아지고(흰색으로 나타날 때까지) 어두운 색상이 점점 더 투명해지고 검은색이 거의 보이지 않도록 하는 특수 혼합 모드입니다.. 이 기능은 아름다운 입자 효과를 줄 수 있지만 항상 필요한 것은 아닙니다. 예를 들어 연기 궤적은 효과를 회색으로 유지하기 위해 가산 혼합을 끄지만 불꽃 효과는 입자를 더 반투명하고 밝게 만들기 위해 가집니다.

통사론:

part_type_blend(ind, additive);

논쟁 유형 설명
ind 입자 유형 ID The index of the particle type to change.
additive 부울 Whether the particles should be blended additively (true) or normally (false).

보고:

N/A

예시:

part_type_blend(part_Fire, true);

위의 코드는 "part_Fire" 변수에 인덱싱된 입자 유형으로 생성된 모든 입자를 추가 혼합 모드로 설정합니다.
