part_type_colour_rgb

이 기능을 사용하면 주어진 유형으로 생성된 모든 입자에 대해 빨강, 녹색 및 파랑 색상의 혼합을 설정할 수 있습니다. 세 가지 구성 요소 각각에 대해 최소값과 최대값을 제공하면 생성된 입자는 지정된 매개변수 범위에 따라 임의의 색상을 갖게 됩니다. 모든 값은 0에서 255 사이여야 합니다.

통사론:

part_type_colour_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p>

논쟁 유형 설명
ind 입자 유형 ID The index of the particle type to change.
rmin 진짜 The minimum the final colour's red component can be.
rmax 진짜 The maximum the final colour's red component can be.
gmin 진짜 The minimum the final colour's green component can be.
gmax 진짜 The maximum the final colour's green component can be.
bmin 진짜 The minimum the final colour's blue component can be.
bmax 진짜 The maximum the final colour's blue component can be.

보고:

N/A

예시:

part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0);

위의 코드는 전역 변수 "Blood_Part"에 인덱싱된 입자 유형에서 방출되는 각 입자를 빨간색의 다른 음영으로만 설정합니다.
