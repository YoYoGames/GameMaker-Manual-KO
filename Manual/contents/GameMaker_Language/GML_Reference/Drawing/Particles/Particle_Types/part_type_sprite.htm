part_type_sprite

이 함수는 게임 자산의 커스텀 스프라이트를 사용하도록 파티클 유형을 설정하는 데 사용할 수 있습니다. 스프라이트를 선택한 다음 GameMaker 에 스프라이트를 사용하고 애니메이션을 적용할지 여부를 알릴 수 있습니다. 애니메이션을 선택한 경우 애니메이션이 시작하고 끝나는 간격으로 하위 이미지가 변경되도록 선택할 수도 있습니다. 생성된 각 입자의 수명 시작 및 끝 수명이 짧고 길수록 느림). 또한 임의의 하위 이미지를 선택하여 스프라이트가 애니메이션되지 않은 경우 다른 이미지가 무작위로 선택되어 스프라이트를 생성하거나 입자가 애니메이션되는 경우 임의의 지점에서 애니메이션을 시작하도록 선택할 수 있습니다.

통사론:

part_type_sprite(ind, sprite, animate, stretch, random);

논쟁 유형 설명
ind 입자 유형 ID The index of the particle type to change.
sprite 스프라이트 자산 The index of the sprite to set the particle type to.
animate 부울 Whether to follow the sprite's animation normally (true) or not (false).
stretch 부울 Whether to stretch the sprite's animation to match the particle lifespan (true) or not (false).
random 부울 Whether to choose a random sub-image (true) or not (false).

보고:

N/A

예시:

global.p1 = part_type_create();
part_type_sprite(global.p1 , spr_Coins, 1, 0, 0);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

위의 코드는 전역 변수 "p1"에 인덱싱된 입자 유형을 "spr_Coins"에 인덱싱된 스프라이트 모양으로 설정한 다음 기타 다양한 입자 속성을 설정합니다.
