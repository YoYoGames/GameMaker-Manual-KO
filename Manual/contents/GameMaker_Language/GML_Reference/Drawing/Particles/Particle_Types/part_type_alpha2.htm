part_type_alpha2

이 기능은 주어진 유형으로 생성된 각 입자에 대한 알파 값(투명도) 그래디언트를 설정하는 데 사용할 수 있습니다. 첫 번째 알파는 모든 파티클이 시작하는 알파이고 두 번째 알파는 파티클이 끝나는 알파이며 입자 수명 동안 알파에 부드러운 그래디언트 변경이 발생합니다. 0(투명)에서 1(불투명) 사이일 수 있습니다.

통사론:

part_type_alpha2(ind, alpha1, alpha2);

논쟁 유형 설명
ind 입자 유형 ID The index of the particle type to change.
alpha1 진짜 The start alpha of the particle.
alpha2 진짜 The end alpha of the particle.

보고:

N/A

예시:

part_type_alpha2(global.Snow_Part, 0, 1);

위의 코드는 전역 변수 "Snow_Part"에 인덱싱된 입자 유형으로 생성된 모든 입자를 알파 값 0(투명)을 갖도록 설정한 다음 수명이 끝날 때 알파 값이 1(불투명)이 되도록 페이드 인합니다.
