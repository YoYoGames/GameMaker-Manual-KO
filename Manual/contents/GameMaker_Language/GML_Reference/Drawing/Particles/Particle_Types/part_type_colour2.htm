part_type_colour2

이 기능은 주어진 유형으로 생성된 각 입자에 대해 두 가지 색상 그라데이션을 설정하는 데 사용할 수 있습니다. 첫 번째 색상은 모든 파티클이 시작하는 색상이고 두 번째 색상은 파티클이 끝나는 색상이며 한 색상에서 다른 색상으로 입자 수명 동안 색상에 부드러운 그라디언트 변경이 발생합니다.

통사론:

part_type_colour2(ind, colour1, colour2);

논쟁 유형 설명
ind 입자 유형 ID The index of the particle type to change.
colour1 색깔 The colour the particle will start at.
colour2 색깔 The colour the particle will end at.

보고:

N/A

예시:

part_type_colour2( part_Health, c_red, c_white);

위의 코드는 "part_Health" 변수에 인덱싱된 유형으로 생성된 모든 입자가 수명 동안 빨간색에서 흰색으로 혼합되도록 합니다.
