part_type_direction

이 함수는 파티클이 생성될 때의 방향을 결정하는 데 사용되며 파티클이 수명 동안 방향을 늘리거나 줄이는 데에도 사용할 수 있습니다. 최소 및 최대 방향 값의 기본값 은 0으로, 이는 파티클이 변화 없이 오른쪽으로 이동한다는 것을 의미하지만, 이러한 값은 0이 오른쪽, 90이 위쪽, 180은 왼쪽, 270은 아래로. 같은 값으로 설정하면 입자가 한 방향으로만 이동합니다.

또한 수명 동안 방향에 각도를 더하거나(양수인 경우) 빼는(음수인 경우) 증분 값을 설정할 수도 있습니다. 이 값은 최소 (+/-) 0.01일 수 있습니다.

마지막으로 "흔들림" 요소를 설정할 수 있습니다. 파티클 수명의 각 단계 방향에 무작위로 더하거나 빼는 값입니다. 분명히 큰 값은 작은 값보다 더 뚜렷하며 이 값은 최대 범위가 -20에서 20 사이인 음수일 수도 있습니다.

통사론:

part_type_direction(ind, dir_min, dir_max, dir_incr, dir_wiggle);

논쟁 유형 설명
ind 입자 유형 ID The index of the particle type to change.
dir_min 진짜 The minimum direction the particle can start at.
dir_max 진짜 The maximum direction the particle can start at.
dir_incr 진짜 How much the particle direction should increase or decrease per step.
dir_wiggle 진짜 How much should randomly be added or subtracted from the particle's direction per step.

보고:

N/A

예시:

part_type_shape(particle1, pt_shape_pixel);
part_type_size(particle1, 0.19, 0.79, 0.01, 0);
part_type_scale(particle1, 5, 5);
part_type_colour3(particle1, 32768, 8454016, 8454016);
part_type_alpha3(particle1, 0, 0.80, 0);
part_type_speed(particle1, 8, 10, -0.50, 1);
part_type_direction(particle1, 0, 359, 1, 20);
part_type_gravity(particle1, 0, 270);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 10, 50);

위의 코드는 0도에서 359도 사이의 임의 값이 될 방향을 포함하여 다양한 입자 값을 설정합니다. 또한 각 단계마다 1도씩 올라가고 각 단계마다 0에서 20 사이의 임의의 양이 추가됩니다.
