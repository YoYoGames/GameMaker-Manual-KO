part_type_alpha3

이 기능은 주어진 유형으로 생성된 각 입자에 대해 3개의 알파(투명도) 값 기울기를 설정하는 데 사용할 수 있습니다. 첫 번째 알파는 모든 파티클이 시작하는 알파이고 두 번째 알파는 수명의 절반까지 혼합될 알파이고 세 번째 알파는 파티클이 끝나는 알파입니다. 입자 수명 동안 알파를 통해 부드러운 그라디언트 변경이 발생합니다.

통사론:

part_type_alpha3(ind, alpha1, alpha2, alpha3);

논쟁 유형 설명
ind 입자 유형 ID The index of the particle type to change.
alpha1 진짜 The starting alpha of the particle.
alpha2 진짜 The halfway point alpha of the particle.
alpha3 진짜 The ending alpha of the particle.

보고:

N/A

예시:

part_type_alpha3( part_Health, 0.5, 1, 0);

위의 코드는 "part_Health" 변수에 인덱싱된 유형으로 생성된 모든 입자가 수명 동안 0.5에서 1에서 0까지의 알파 블렌드를 갖도록 합니다.
