part_system_destroy

이 기능을 사용하면 주어진 입자 시스템을 파괴하고 메모리에서 제거할 수 있습니다. 이것은 방의 끝이나 인스턴스의 파괴 이벤트와 같이 시스템이 더 이상 필요하지 않을 때 항상 호출되어야 합니다. 그렇지 않으면 나중에 방에 파티클이 나타나고 제거할 방법이 없습니다 결국 게임을 중단시키는 메모리 누수.

참고: 이 기능은 또한 생성되고 파괴되는 시스템과 관련되었을 수 있는 모든 이미터를 파괴하고 제거합니다.

통사론:

part_system_destroy(ind);

논쟁 유형 설명
ind 입자 시스템 ID The index of the particle system to destroy.

보고:

N/A

예시:

if (global.player_hp <= 0)
{
    part_system_destroy(p_sys);
    room_goto_next();
}

위의 코드는 전역 변수 값이 0보다 작거나 같은지 확인하고, 그렇다면 주어진 변수에서 참조하는 파티클 시스템을 파괴하고 다음 방으로 이동합니다.

뒤로: 파티클 시스템
다음: part_particles_clear
