part_system_clear

이 기능을 사용하면 인덱싱된 시스템을 기본 상태로 지우고 모든 이미터를 제거하고 깊이와 위치(변경된 경우)를 재설정할 수 있습니다. 이 시스템과 연결된 이미터를 사용하는 인스턴스 설정이나 버스팅 또는 기타 작업이 있는 것처럼 이 기능을 사용하는 데 주의하십시오. 사용하지 않는 한 오류가 발생합니다.part_emitter_exists() 확인하는 기능입니다. 부를 필요도 없다. part_emitter_destroy() 기능도 자동으로 처리됩니다.

참고: 이 기능은 방에서 보이는 입자를 지우지만 입자 속성을 지우거나 메모리에서 제거하지 않습니다. 그것을 위해 당신은 기능을 사용해야합니다 part_type_clear() 그리고part_type_destroy().

통사론:

part_system_clear(ind);

논쟁 유형 설명
ind 입자 시스템 ID The index of the particle system to clear.

보고:

N/A

예시:

part_system_clear(global.Sname);

위의 코드는 전역 변수 "Sname"에 인덱싱된 입자 시스템을 기본 상태로 지웁니다.
