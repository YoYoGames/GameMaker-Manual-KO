part_system_automatic_update

이 함수는 GameMaker 가 파티클 시스템을 자동으로 업데이트할지 여부를 제어하며 일반적으로 이 함수를 기본값으로 사용할 필요가 없습니다. true (자동 업데이트가 켜져 있음) 원하는 일이 발생합니다. 그러나 특수 효과를 위해 또는 게임을 일시 중지하려면 다음으로 설정할 수 있습니다. false 함수를 사용하여 명시적으로 명령하지 않는 한 지정된 파티클 시스템에서 업데이트가 수행되는 것을 방지합니다.part_system_update() 또는 이 기능을 다시 사용하여 자동 업데이트를 켭니다. 업데이트 안 함은 이 시스템에서 생성된 파티클이 더 이상 위치, 색상 또는 기타 매개변수를 변경하지 않으며 이미터도 작동을 중지하여 자동 업데이트가 false.

한 가지 주의할 점은 함수로 생성된 간단한 효과를 사용하는 경우effect_create_above() 또는effect_create_below() 그런 다음 0(아래 효과의 경우) 또는 1(위 효과의 경우) 값을 입자 시스템 인덱스로 사용할 수 있으므로 이들에 대한 자동 업데이트도 토글할 수 있습니다(이는 GML Visual 입자 효과에 대한 업데이트를 토글하는 데에도 작동합니다).

통사론:

part_system_automatic_update(ind, automatic);

논쟁 유형 설명
ind 입자 시스템 ID The index of the particle system to change.
automatic 부울 Whether automatic updating is on (true ) or not (false).

보고:

N/A

예시:

if global.Pause
{
    part_system_automatic_update(global.Sname, false);
}

위의 코드는 전역 변수 "Paused"가 true인 경우 입자 업데이트를 끕니다.
