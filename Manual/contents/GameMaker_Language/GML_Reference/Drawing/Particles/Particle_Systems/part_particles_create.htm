part_particles_create

이 기능은 입자 이미터 가 제공하는 기능이 필요하지 않은 효과에 이상적입니다(예: 미사일에서 연기 생성 또는 간단한 폭발 효과). 게임룸. 이 기능을 사용하려면 이전에 입자 시스템과 입자 유형을 생성해야 합니다.

통사론:

part_particles_create(ind, x, y, parttype, number);

논쟁 유형 설명
ind 입자 시스템 ID The index of the particle system.
x 진짜 The x coordinate of where to create the particles.
y 진짜 The y coordinate of where to create the particles.
parttype 입자 유형 ID The index (type) of the particles to be created.
number 진짜 The number of particles to create.

보고:

N/A

예시:

if mouse_check_button(mb_left)
{
    part_particles_create(sname, mouse_x, mouse_y, p_CursorEffect, 5);
}

위의 코드는 마우스 버튼이 눌렸는지 확인하고 반환되는지 확인합니다. true 마우스 위치에 5개의 입자를 생성합니다.
