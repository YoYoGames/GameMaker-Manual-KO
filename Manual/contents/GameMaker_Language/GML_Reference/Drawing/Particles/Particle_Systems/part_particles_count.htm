part_particles_count

이 기능을 사용하면 파티클 시스템이 현재 방에 생성된 파티클이 있는지 확인할 수 있으며, 파티클의 개수도 반환합니다.

통사론:

part_particles_count(ind);

논쟁 유형 설명
ind 입자 시스템 ID The index of the particle system.

보고:

Real

예시:

if (part_particles_count(Sname) == 0)
{
    part_system_destroy(Sname);
    instance_destroy();
}

위의 코드는 "Sname" 변수에 인덱싱된 로컬 파티클 시스템의 파티클 수를 확인하고 아무 것도 없으면 시스템을 파괴한 다음 인스턴스를 파괴합니다.
