part_system_layer

이 기능은 파티클 시스템을 현재 레이어에서 새 레이어로 전환하는 데 사용할 수 있습니다. 함수에서 반환된 입자 시스템 ID 값을 제공합니다.part_system_create_layer() ) 그런 다음 고유한 레이어 ID(함수에 의해 반환됨) layer_create() 또는 문자열로 사용할 레이어 이름(예: "instance_layer" - 룸 편집기에 정의됨), 시스템은 새 레이어로 이동됩니다.

통사론:

part_system_layer(ps, layer);

논쟁 유형 설명
ps 입자 시스템 ID The particle system ID
layer 문자열 또는 레이어 ID The new layer ID

보고:

N/A

예시:

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
    part_system_layer(global.p_sys, "effects_layer";
}

위의 코드는 파티클 시스템이 어떤 레이어에 있는지 확인하고, 주어진 레이어에 없으면 변경됩니다.
