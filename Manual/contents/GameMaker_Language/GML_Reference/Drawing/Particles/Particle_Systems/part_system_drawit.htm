part_system_update

이 기능을 사용하면 입자 시스템이 화면에 현재 입자를 모두 그리도록 강제할 수 있습니다. 만약에part_system_automatic_draw() 꺼져 있으면 이 기능은 인스턴스의 그리기 이벤트에서 사용될 때 입자를 표시하거나 그리기 대상이 표면으로 설정될 때 사용할 수 있습니다(사용surface_set_target() ) 시스템의 입자를 해당 표면으로 끌어옵니다.

통사론:

part_system_drawit(ind);

논쟁 유형 설명
ind 입자 시스템 ID The index of the particle system to draw the particles of.

보고:

N/A

예시:

if surface_exists(surf)
{
    surface_set_target(surf);
    part_system_drawit(global.psys);
    surface_reset_target();
}

위의 코드는 "surf" 변수에 인덱싱된 표면이 존재하는지 확인하고 존재한다면 그리기 대상을 표면으로 설정하고 전역 변수에 저장된 ID로 입자 시스템을 그린 다음 도면을 재설정합니다. 모든 일반 그림이 다시 한 번 화면에 나타나도록 대상을 지정합니다.
