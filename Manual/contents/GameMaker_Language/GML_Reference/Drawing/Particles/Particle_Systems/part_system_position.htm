part_system_position

이 기능을 사용하면 방의 (0,0) 위치를 기준으로 파티클 시스템의 기본 위치를 설정할 수 있습니다. 즉, 이 시스템과 관련된 모든 추가 파티클 함수는 이제 새 위치를 기준으로 그려집니다. 기본적으로 이 위치는 항상 (0,0)이지만 일부 매우 특별한 상황에서는 이것을 다른 것으로 변경할 수 있습니다.

참고: 이 기능은 파티클 시스템 내의 모든 것을 변경하므로 위치 (100,100)에 이미터가 있고 파티클 시스템 위치를 (0,100)으로 설정하면 이미터는 이제 (100,200)에서 그려집니다. 시스템을 이동한 다음 이미터를 생성하는 경우에도 마찬가지입니다. (100,100)에서 생성하더라도 (100,200)에서 그려질 것입니다.

통사론:

part_system_position(ind, x, y);

논쟁 유형 설명
ind 입자 시스템 ID The index of the particle system to change.
x 진짜 The new x coordinate of the particle system.
y 진짜 The new y coordinate of the particle system.

보고:

N/A

예시:

if mouse_check_button_pressed(mb_left)
{
    part_system_position(global.Sname, mouse_x, mouse_y);
}

위의 코드는 마우스 버튼이 눌렸는지 확인하고 감지하면 전역 변수 "Sname"에 인덱싱된 입자 시스템이 마우스 x/y 위치로 이동합니다.
