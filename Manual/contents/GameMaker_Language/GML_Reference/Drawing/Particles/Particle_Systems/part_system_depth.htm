part_system_depth

이 기능을 사용하면 파티클 시스템에 대한 그리기 깊이를 설정할 수 있습니다. 이는 방 내의 다른 레이어의 렌더링 깊이 를 설정할 수 있는 것과 매우 유사합니다. 깊이, 그리고 더 낮은 그리기 깊이로 모든 것 아래에 배치하는 높은 그리기 깊이.

통사론:

part_system_depth( ind, depth );

논쟁 유형 설명
ind 입자 시스템 ID The index of the particle system to change.
depth 진짜 The depth at which to set the particle system.

보고:

N/A

예시:

global.Sname = part_system_create();
part_system_depth(global.Sname, -1000 );

위의 코드는 파티클 시스템을 만들고 해당 인덱스를 전역 변수 "Sname"에 저장합니다. 그런 다음 이 시스템에는 -1000의 낮은 깊이가 지정되며, 이는 더 높은 그리기 깊이로 모든 것 위에 표시됨을 의미합니다.
