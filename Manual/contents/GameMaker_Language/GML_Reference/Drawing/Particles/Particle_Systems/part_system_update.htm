part_system_update

이 기능은 게임 시간에서 주어진 파티클 시스템을 한 단계 진행하고 자동 업데이트가 꺼져 있을 때 파티클 시스템 동작을 시뮬레이션하는 데 사용할 수 있습니다. 또는 파티클 시스템을 한 번에 특정 지점으로 진행하는 데 사용할 수 있습니다. 예를 들어 눈에 대한 입자 시스템이 있고 플레이어가 방에 입장하는 순간부터 눈이 내리는 것처럼 보이게 하려면 생성 이벤트에서 몇 초 동안 눈 입자를 전진시키는 것이 좋습니다. 인스턴스가 처음 그려질 때 화면 전체에 파티클이 표시되도록 합니다(아래 코드 예제에서 이를 보여줍니다). 기능에 의해 시스템의 자동 업데이트가 꺼진 경우에도 이 기능을 사용할 수 있습니다.part_system_automatic_update() 함께 시스템을 이동합니다.

통사론:

part_system_update(ind);

논쟁 유형 설명
ind 입자 시스템 ID The index of the particle system to update.

보고:

N/A

예시:

repeat (room_speed * 3)
{
    part_system_update(global.RainSys);
}

위의 코드는 전역 변수 "RainSys"에 인덱싱된 입자 시스템을 3초 앞당길 것입니다.
