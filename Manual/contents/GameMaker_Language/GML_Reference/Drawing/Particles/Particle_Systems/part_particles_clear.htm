part_particles_clear

이 기능을 사용하면 시스템에서 현재 생성된 모든 입자를 방에서 지울 수 있습니다. 입자 유형 자체를 재설정하거나 제거하지 않고 시각적 표현만 제거합니다. 이미터에서 입자를 스트리밍하는 개체가 있는 경우 이러한 입자는 사라지지만 이 코드를 호출한 후 다음 단계에서 다시 나타나기 시작합니다.

통사론:

part_particles_clear(ind);

논쟁 유형 설명
ind 입자 시스템 ID The index of the particle system.

보고:

N/A

예시:

if (lives <= 0)
{
    part_particles_clear(global.Sname);
    room_goto(rm_intro);
}

위의 코드는 "lives" 변수의 값을 확인하고 0과 같으면 시스템에서 모든 입자를 지우고 공간을 변경합니다.
