part_system_exists

이 기능을 사용하면 인덱싱된 주어진 입자 시스템이 게임에 존재하는지 여부를 확인할 수 있습니다. 검사 중인 변수가 초기화되지 않은 변수(파티클 시스템에 인덱스가 할당되지 않은 경우)인 경우 오류가 발생합니다.

통사론:

part_system_exists(ind);

논쟁 유형 설명
ind 입자 시스템 ID The index of the particle system to check for.

보고:

Boolean

예시:

if !part_system_exists(global.Sname)
{
    global.Sname = part_system_create();
}

위의 코드는 전역 변수에 참조된 파티클 시스템이 있는지 확인하고 없으면 생성됩니다.
