part_system_get_layer

이 함수는 주어진 파티클 시스템에 대한 고유한 레이어 ID 값을 검색하는 데 사용할 수 있습니다. 함수에서 반환된 입자 시스템 ID 값을 제공합니다.part_system_create_layer() ) 이 함수는 레이어의 ID 값을 반환합니다.

통사론:

part_system_get_layer(ind);

논쟁 유형 설명
ind 입자 시스템 ID The particle system ID value to get the layer ID of

보고:

Layer ID or 0 (if the layer is an internally managed one)

예시:

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
    part_system_layer(global.p_sys, "effects_layer";
}

위의 코드는 파티클 시스템을 확인하여 어떤 레이어에 있는지 확인하고 주어진 레이어에 없으면 변경됩니다.
