part_system_automatic_draw

이 기능은 시스템에 대한 업데이트(자동 또는 기타)가 표시되지 않도록 파티클 시스템의 도면을 끄는 데 사용할 수 있습니다. 이것은 순전히 시각적인 옵션이며 false 그려지지 않았기 때문에 입자를 볼 수는 없지만 여전히 존재하고 위치, 색상 등을 변경하고 있습니다. 평소처럼. 자동 그리기가 꺼져 있을 때 이 기능을 사용하여 파티클 시스템의 현재 상태를 그리도록 GameMaker 에 명시적으로 명령할 수 있습니다.part_system_drawit(), 그리고 이 기능을 다음으로 설정하면 true 다시 자동 그리기를 다시 켤 수 있습니다.

한 가지 주의할 점은 함수로 생성된 간단한 효과를 사용하는 경우effect_create_above() 또는effect_create_below() 그런 다음 0(아래 효과의 경우) 또는 1(위 효과의 경우) 값을 입자 시스템 인덱스로 사용할 수 있으므로 이들에 대한 자동 그리기도 토글할 수 있습니다(이는 GML Visual 입자 효과에 대한 그리기를 토글하는 데에도 작동합니다).

통사론:

part_system_automatic_draw(ind, automatic);

논쟁 유형 설명
ind 입자 시스템 ID The index of the particle system to change.
automatic 부울 Whether automatic drawing is on (true) or not (false).

보고:

N/A

예시:

part_system_automatic_draw(global.Sname, false);

위의 코드는 전역 변수 "Sname"에 인덱싱된 입자 시스템에 대한 자동 그리기를 끕니다.
