part_system_draw_order

이 기능을 사용하면 화면에 생성될 때 입자가 그려지는 방식을 설정할 수 있습니다. 기본 시스템은 이전 > 새 모양을 사용합니다(기능은 true ), 오래된 입자가 새 입자보다 더 깊은 깊이로 그려지고 새 입자 "아래"에 나타나지만 이 기능을 다음으로 설정하면 false 이 순서를 반대로 하여 새 입자를 더 높게 그려서 이전 입자 "아래"에 표시할 수 있습니다. 아래 이미지는 이를 설명하며 왼쪽 이미지는 기본값입니다. true 그리고 오른쪽 이미지는 false:

참고: 가산 혼합 모드로 입자를 그릴 때 이 기능의 효과가 항상 명확하지 않을 수 있습니다.

통사론:

part_system_draw_order(ind, oldtonew);

논쟁 유형 설명
ind 입자 시스템 ID The index of the particle system to change.
oldtonew 부울 Whether old particles should be drawn behind newer ones (true) or vice versa (false).

보고:

N/A

예시:

mysystem = part_system_create();
part_system_draw_order(mysystem, true);

이것은 인덱스 "mysystem " 그런 다음 이전 입자 위에 새 입자를 그리도록 입자를 설정합니다.
