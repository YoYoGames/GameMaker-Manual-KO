part_particles_create_colour

이 기능은 입자 이미터 가 제공하는 기능이 필요하지 않은 효과에 이상적입니다(예: 미사일에서 연기 생성 또는 간단한 폭발 효과). 게임룸. 또한 이 기능으로 생성된 입자에 "즉시" 색상을 지정할 수 있으며, 이 색상은 미리 정의된 입자 색상을 무시하지만 입자 수명 동안 이 색상을 혼합하지 않습니다. 이 기능을 사용하려면 이전에 입자 시스템과 입자 유형을 생성해야 합니다.

통사론:

part_particles_create_colour(ind, x, y, parttype, colour, number);

논쟁 유형 설명
ind 입자 시스템 ID The index of the particle system.
x 진짜 The x coordinate of where to create the particles.
y 진짜 The y coordinate of where to create the particles.
parttype 입자 유형 ID The index (type) of the particles to be created.
colour 색깔 The colour blending for the particles.
number 진짜 The number of particles to create.

보고:

N/A

예시:

if (speed > 1)
{
    var _c = choose(c_aqua, c_lime, c_fuchsia, c_orange);
    part_particles_create_colour(sname, x, y, p_Smoke, _c, 1);
}

위의 코드는 코드가 있는 인스턴스의 속도가 1보다 큰 모든 단계에서 단일 입자를 생성합니다. 이 입자는 무작위 색상입니다.
