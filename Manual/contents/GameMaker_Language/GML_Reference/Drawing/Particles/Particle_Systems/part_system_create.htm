part_system_create

이 함수는 새 입자 시스템을 만드는 데 사용되며 해당 시스템과 관련된 모든 추가 기능에서 저장하고 사용해야 하는 고유 인덱스 번호를 반환합니다. 시스템에는 관리 계층이 할당되고 깊이가 0으로 설정됩니다. 관리 계층은 사용자가 액세스할 수 없으며 계층 대신 깊이가 사용되는 경우 내부 관리용으로만 사용됩니다. 일반적으로 당신은 기능을 사용할 것입니다part_system_create_layer() 이 대신.

통사론:

part_system_create();

보고:

Particle System ID

예시:

global.Sname = part_system_create();

이것은 새로운 입자 시스템을 생성하고 전역 변수 "Sname"에 인덱스를 저장합니다.
