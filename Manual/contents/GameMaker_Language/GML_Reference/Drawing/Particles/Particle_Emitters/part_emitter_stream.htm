part_emitter_stream

이 함수를 사용하면 특정 유형의 파티클을 스트리밍하도록 이미터를 설정할 수 있으며 이후 게임의 매 단계마다 지정된 수의 파티클을 생성하도록 이미터를 설정하는 일회성 코드이기 때문에 일반적으로 생성 이벤트에 사용됩니다. 함수에서 설정한 분포, 모양 및 위치에 따라 입자가 방출됩니다.part_emitter_region(). 스트리밍되는 파티클 수를 0으로 설정하고 이 함수를 다시 호출하여 파티클 스트리밍을 "끄기"할 수 있으며, 이 함수는 양에 대해 음수도 허용합니다. 이 경우 이미터는 임의의 기회에 따라 파티클을 생성합니다. 예를 들어 입자 번호를 -5로 설정하면 모든 단계에서 입자가 생성될 확률이 1:5입니다. 입자가 모든 단계가 아니라 한 번에 모두 나타나야 하는 경우 함수를 사용해야 합니다.part_emitter_burst().

통사론:

part_emitter_stream(ps, ind, parttype, number);

논쟁 유형 설명
ps 입자 시스템 ID The particle system that the emitter is in.
ind 입자 방사체 ID The index of the emitter to burst from.
parttype 입자 유형 ID The index (type) of the particles to be created.
number 진짜 The number of particles to create per step.

보고:

N/A

예시:

part_emitter_stream(global. Sname, p_emit1, p1, 1);

위의 코드는 이미터가 소멸되거나 스트림이 0으로 설정될 때까지 게임의 매 단계마다 1개의 파티클을 스트리밍합니다.
