part_emitter_create

이 함수는 주어진 파티클 시스템에서 새로운 파티클 이미터를 만드는 데 사용됩니다. 이 함수는 변수에 저장되어야 하고 이미터를 참조하는 모든 추가 함수에서 사용해야 하는 새 이미터에 대한 인덱스 값을 반환하며, 메모리 누수를 방지하기 위해 더 이상 사용되지 않을 때 이미터 자체를 파괴해야 합니다(이를 달성할 수 있음 특정 이미터 파괴 기능을 사용하거나 이미터가 속한 전체 입자 시스템을 파괴).

참고 이 함수에서 반환된 이미터 ID는 재사용됩니다. 예를 들어 처음 세 개의 파티클 이미터를 생성하면 ID 0, 1 및 2가 할당됩니다. 그런 다음 part_emitter_destroy 를 사용하여 두 번째 이미터(ID가 1임)를 파괴한 다음 다른 이미터를 생성하면 이 이미터는 ID 1을 재사용합니다. 이미터 ID 2가 사용 중이므로 다음 이미터는 ID 3을 얻습니다., 등등.

통사론:

part_emitter_create(ps);

논쟁 유형 설명
ps 입자 시스템 ID The particle system to create the emitter in.

보고:

Particle Emitter ID

예시:

p_emit1 = part_emitter_create(Sname);

이것은 새로운 파티클 이미터를 생성하고 "p_emit" 변수에 인덱스를 저장합니다.
