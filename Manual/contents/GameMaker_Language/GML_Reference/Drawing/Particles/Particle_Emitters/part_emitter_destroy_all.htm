part_emitter_destroy_all

이 함수는 주어진 시스템에서 정의된 모든 이미터를 제거하고 메모리에서 지웁니다(이는 또한 주어진 이미터에서 생성되는 파티클을 중지하지만 공간에서 제거하지는 않습니다 ). 이 함수는 시스템에서 메모리 누수 및 오류를 방지하기 위해 이미터가 더 이상 필요하지 않을 때 항상 호출되어야 합니다.

통사론:

part_emitter_destroy_all( ps );

논쟁 유형 설명
ps The particle system to destroy all emitters from.

보고:

예시:

if lives == 0
{
    part_emitter_destroy_all(global.Sname);
    room_goto(rm_Menu);
}

위의 코드는 내장 전역 변수 "lives"를 확인하고 0이면 모든 파티클 이미터를 파괴한 다음 공간을 변경합니다.
