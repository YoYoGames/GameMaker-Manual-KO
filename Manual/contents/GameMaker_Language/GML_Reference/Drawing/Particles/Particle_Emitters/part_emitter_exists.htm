part_emitter_exists

이 기능을 사용하면 인덱싱된 주어진 파티클 이미터가 주어진 시스템에 존재하는지 여부를 확인할 수 있습니다. 검사 중인 변수가 초기화되지 않은 변수(파티클 이미터에 인덱스가 할당되지 않은 경우)인 경우 오류가 발생합니다.

통사론:

part_emitter_exists(ps, ind);

논쟁 유형 설명
ps 입자 시스템 ID The particle system to check for an emitter.
ind 입자 방사체 ID The index of the emitter to search for.

보고:

Boolean

예시:

if part_emitter_exists(sname, p_emit1)
{
    part_emitter_burst(sname, p_emit1, part_1, 30);
}

위의 코드는 "permit" 변수에 인덱싱된 에미터를 확인하고 존재하는 경우 30개의 파티클을 버스트합니다.
