part_emitter_burst

이 기능을 사용하면 특정 유형의 입자를 버스트하도록 이미터를 설정할 수 있으며 에 의해 설정된 분포, 모양 및 위치에 따라 한 번에 지정된 수의 입자를 생성하는 일회성 코드이므로 일반적으로 경보 및 파괴 이벤트에 사용됩니다. 함수part_emitter_region(). 모든 단계에서 입자가 나타나야 하는 경우 함수를 사용해야 합니다.part_emitter_stream() 모든 단계에서 이 함수를 호출하는 대신.

통사론:

part_emitter_burst(ps, ind, parttype, number);

논쟁 유형 설명
ps 입자 시스템 ID The particle system that the emitter is in.
ind 입자 방사체 ID The index of the emitter to burst from.
parttype 입자 유형 ID The index (type) of the particles to be created.
number 진짜 The number of particles to create.

보고:

N/A

예시:

part_emitter_burst(global. Sname, p_emit1, p1, 30 + irandom(30));

위의 코드는 선택한 이미터에서 30에서 60 사이의 임의의 수의 입자를 버스트합니다.
