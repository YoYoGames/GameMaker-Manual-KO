part_emitter_clear

이 기능을 사용하면 지정된 입자 시스템에서 지정된 이미터를 기본 상태로 다시 지울 수 있습니다. 이것은 또한 그 시간에 이미터에서 스트리밍되는 모든 파티클을 중지하고, 이미터를 다시 사용하려면 다음을 사용하여 영역 위치와 파티클 유형을 설정해야 합니다.part_emitter_region() 함수.

통사론:

part_emitter_clear(ps, ind);

논쟁 유형 설명
ps 입자 시스템 ID The particle system that the emitter is in.
ind 입자 방사체 ID The index of the emitter to clear.

보고:

N/A

예시:

part_emitter_clear(Sname, p_emit1 );

위의 코드는 "p_emit1" 변수에 인덱싱된 파티클 이미터를 지웁니다.
