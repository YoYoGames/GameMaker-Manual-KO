part_emitter_destroy

이 함수는 주어진 시스템에서 지정된 에미터를 제거하고 메모리에서 지웁니다(이렇게 하면 주어진 에미터에서 생성되는 파티클도 중지되지만 공간에서 제거하지는 않습니다 ). 이 함수는 시스템이 메모리 누수와 오류를 방지하기 위해 더 이상 주어진 이미터가 필요하지 않을 때 항상 호출되어야 합니다.

통사론:

part_emitter_destroy( ps, ind );

논쟁 유형 설명
ps 입자 시스템 ID The particle system to destroy the emitter from.
ind 입자 방사체 ID The index of the emitter to destroy.

보고:

N/A

예시:

if part_emitter_exists(global.Sname, p_emit)
{
    part_emitter_destroy(global.Sname, p_emit1);
}

위의 코드는 "p_emit" 변수에 인덱싱된 파티클 이미터가 주어진 파티클 시스템에 존재하는지 확인하고 존재한다면 소멸됩니다.
