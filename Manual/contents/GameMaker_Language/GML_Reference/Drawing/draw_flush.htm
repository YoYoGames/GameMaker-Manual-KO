draw_flush

이 기능을 사용하면 전체 그리기 파이프라인을 플러시할 수 있습니다. 이것은 디버그 전용 기능이며 일반적으로 YoYo Games Support 직원이 지시하지 않는 한 사용해서는 안 됩니다. 무분별한 사용은 게임에 심각한 성능 문제를 일으킬 수 있기 때문입니다.

통사론:

draw_flush();

보고:

예시:

draw_flush();

위의 코드는 그리기 파이프라인을 플러시합니다.
