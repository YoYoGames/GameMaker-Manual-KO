gpu_get_tex_mip_bias_ext

이 함수를 사용하면 주어진 셰이더 샘플러에 대한 밉맵 바이어스 값을 검색할 수 있습니다. 셰이더 샘플러에 대한 인덱스 값을 제공합니다(함수에서 반환된 shader_get_sampler_index(), 그리고 함수는 편향이 없는 경우 0.0의 값을 반환하거나 1이 첫 번째 밉맵과 같고 2가 두 번째 밉맵과 같은 경우 더 큰 값을 반환합니다. 이것은 밉맵이 바뀌는 속도를 제어하며 일반적으로 셰이더 텍스처는 값이 높을수록 더 흐려지고 표시되는 "거리"가 더 커집니다. 이것은 음수 값도 반환할 수 있습니다. 이 경우 셰이더 텍스처는 값이 낮을수록 더 먼 거리에서 더 선명해집니다.

통사론:

gpu_get_tex_mip_bias_ext(sampler_index);

논쟁 유형 설명
sampler_index The index of the shader sampler to get

보고:

(default: 0)

예시:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if (gpu_get_tex_mip_bias_ext(_sampleIndex) != 0)
{
    gpu_set_tex_mip_bias_ext(_sampleIndex, 0);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

위의 코드는 주어진 셰이더 텍스처 샘플러가 아직 0으로 설정되지 않은 경우 밉 필터 바이어스를 0으로 설정합니다.
