gpu_set_tex_mip_bias

이 기능을 사용하면 밉맵 바이어스 값을 설정할 수 있습니다. 여기서 값 0은 바이어스 없음, 1은 첫 번째 밉맵, 2는 두 번째 밉맵 등... 이것은 밉맵이 바뀌는 속도를 제어하며 일반적으로 렌더링된 텍스처는 값이 높을수록 더 흐릿해지고 표시되는 "거리"가 더 커집니다. 이 함수는 음수 값도 취할 수 있으며, 이 경우 렌더링된 텍스처는 값이 낮을수록 더 먼 거리에서 더 선명해집니다.

통사론:

gpu_set_tex_mip_bias(bias);

논쟁 유형 설명
bias The mipmap bias value to use (default: 0)

보고:

예시:

if gpu_get_tex_mip_bias() != 0
{
    gpu_set_tex_mip_bias(0);
}

위의 코드는 현재 밉맵 바이어스를 확인하고 0이 아니면 0으로 설정합니다.
