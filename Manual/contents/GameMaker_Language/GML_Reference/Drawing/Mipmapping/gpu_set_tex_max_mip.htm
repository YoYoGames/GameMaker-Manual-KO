gpu_set_tex_max_mip

이 기능을 사용하면 현재 설정된 최대 밉맵 수준을 설정할 수 있습니다. 여기서 값 0은 가장 높은 해상도, 1은 첫 번째 밉맵, 2는 두 번째 등... 텍스처를 렌더링할 때 블리딩 아티팩트를 피하는 데 유용합니다. 예를 들어 텍스처 페이지 테두리를 8px로 설정한 다음 최대 밉맵 수준을 3으로 설정하면 더 먼 렌더링 거리에서 블리딩 문제가 발생하지 않습니다.

통사론:

gpu_set_tex_max_mip(maxmip);

논쟁 유형 설명
maxmip The maximum mipmap level (default: 16)

보고:

예시:

if gpu_get_tex_max_mip() != 4
{
    gpu_set_tex_max_mip(4);
}

위의 코드는 현재 최대 밉맵 레벨을 확인하고 4가 아니면 4로 설정합니다.
