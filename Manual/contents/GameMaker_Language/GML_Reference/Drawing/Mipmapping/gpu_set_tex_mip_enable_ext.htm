gpu_set_tex_mip_enable_ext

이 기능을 사용하면 밉매핑을 끄거나 모든 것에 대해 켤지 또는 셰이더 샘플러의 텍스처 그룹 관리자 에서 선택한 텍스처 그룹에 대해서만 켤지 설정할 수 있습니다. 셰이더 샘플러에 대한 인덱스 값을 제공합니다(함수에서 반환된 shader_get_sampler_index(), 아래 나열된 상수 중 하나.

일정한 설명
mip_off 밉매핑이 비활성화되었습니다.
mip_on 모든 텍스처에 대한 밉매핑이 활성화되었습니다.
mip_markedonly 텍스처 그룹 옵션(기본값)에서 활성화된 텍스처에 대해 밉매핑이 활성화됩니다.

통사론:

gpu_set_tex_mip_enable_ext(sampler_index, setting);

논쟁 유형 설명
sampler_index 셰이더 샘플러 핸들 The index of the shader sampler to get
setting 밉매핑 상수 The mipmap setting (a constant, default: mip_markedonly )

보고:

N/A

예시:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if gpu_get_tex_mip_enable_ext(_sampleIndex) != mip_on
{
    gpu_set_tex_mip_enable_ext(_sampleIndex, mip_on);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

위의 코드는 주어진 셰이더 텍스처 샘플러가 아직 활성화되지 않은 경우 밉매핑을 활성화합니다.
