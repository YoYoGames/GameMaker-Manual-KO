gpu_set_tex_max_mip_ext

이 함수를 사용하면 주어진 셰이더 샘플러에 사용될 현재 설정된 최대 밉맵 레벨을 얻을 수 있습니다. 셰이더 샘플러에 대한 인덱스 값을 제공합니다(함수에서 반환된 shader_get_sampler_index(), 그리고 함수는 현재 최대 밉맵 수준을 반환합니다. 여기서 값 0은 가장 높은 해상도, 1은 첫 번째 밉맵, 2는 두 번째 등... 예를 들어 텍스처를 렌더링하면 텍스처 페이지 테두리를 8px로 설정한 다음 최대 밉맵 수준을 3으로 설정하면 더 먼 렌더링 거리에서 번짐 문제가 발생하지 않습니다.

통사론:

gpu_set_tex_max_mip_ext(sampler_index, maxmip);

논쟁 유형 설명
sampler_index The index of the shader sampler to get
maxmip The maximum mipmap level value to use (default: 16)

보고:

예시:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if gpu_get_tex_max_mip_ext(_sampleIndex) != 4
{
    gpu_set_tex_max_mip_ext(_sampleIndex, 4);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

위의 코드는 주어진 셰이더 텍스처 샘플러가 아직 4로 설정되지 않은 경우 최대 밉맵 수준을 4로 설정합니다.
