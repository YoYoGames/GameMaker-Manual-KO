gpu_get_tex_mip_enable

이 기능을 사용하면 밉매핑이 꺼져 있는지, 모든 항목에 대해 켜져 있는지 또는 텍스처 그룹 관리자 에서 선택한 텍스처 그룹에 대해서만 켜져 있는지 여부를 확인할 수 있습니다. 이 함수는 아래 나열된 상수 중 하나를 반환하며 기본 설정은 mip_markedonly.

통사론:

gpu_get_tex_mip_enable();

보고:

Constant

밉매핑 상수
일정한 설명
mip_off 밉매핑이 비활성화되었습니다.
mip_on 모든 텍스처에 대한 밉매핑이 활성화되었습니다.
mip_markedonly 텍스처 그룹 옵션(기본값)에서 활성화된 텍스처에 대해 밉매핑이 활성화됩니다.

예시:

if gpu_get_tex_mip_enable != mip_on
{
    gpu_set_tex_mip_enable(mip_on);
}

위의 코드는 밉매핑이 활성화되어 있는지 확인하고 활성화되지 않은 경우 활성화합니다.
