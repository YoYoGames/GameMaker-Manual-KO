gpu_set_tex_mip_bias_ext

이 기능을 사용하면 주어진 셰이더 샘플러에 대한 밉맵 바이어스 값을 설정할 수 있습니다. 셰이더 샘플러에 대한 인덱스 값을 제공합니다(함수에서 반환된 shader_get_sampler_index(), 그런 다음 편향 값입니다. 여기서 0은 편향 없음, 1은 첫 번째 밉맵, 2는 두 번째 밉맵 등... 이것은 밉맵이 바뀌는 속도를 제어하며 일반적으로 셰이더 텍스처를 더 흐릿하게 만듭니다. 값과 더 큰 "거리"가 표시됩니다. 이 함수는 음수 값도 사용할 수 있으며, 이 경우 셰이더 텍스처는 값이 낮을수록 더 먼 거리에서 더 선명해집니다.

통사론:

gpu_set_tex_mip_bias_ext(sampler_index, bias);

논쟁 유형 설명
sampler_index The index of the shader sampler to get
bias The mipmap bias value to use (default: 0)

보고:

예시:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if (gpu_get_tex_mip_bias_ext(_sampleIndex) != 0)
{
    gpu_set_tex_mip_bias_ext(_sampleIndex, 0);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

위의 코드는 주어진 셰이더 텍스처 샘플러가 아직 0으로 설정되지 않은 경우 밉 필터 바이어스를 0으로 설정합니다.
