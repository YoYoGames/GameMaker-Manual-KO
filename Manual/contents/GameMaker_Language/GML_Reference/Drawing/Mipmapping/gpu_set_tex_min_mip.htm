gpu_set_tex_min_mip

이 기능을 사용하면 현재 사용되는 최소 밉맵 수준을 설정할 수 있습니다. 여기서 값 0은 가장 높은 해상도, 1은 첫 번째 밉맵, 2는 두 번째 등...

통사론:

gpu_set_tex_min_mip(minmip);

논쟁 유형 설명
minmip The minimum mipmap level to use

보고:

예시:

if gpu_get_tex_min_mip() != 0
{
    gpu_set_tex_min_mip(0);
}

위의 코드는 현재 최소 밉맵 수준을 확인하고 0이 아니면 0으로 설정합니다.
