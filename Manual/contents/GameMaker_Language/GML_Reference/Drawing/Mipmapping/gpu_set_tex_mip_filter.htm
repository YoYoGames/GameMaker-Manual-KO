gpu_set_tex_mip_filter

이 기능을 사용하면 현재 밉 필터 모드를 지원되는 세 가지 유형 중 하나로 설정할 수 있습니다. 점 필터링(기본 설정, 필터링 없음을 의미), 선형 필터링( 삼선형 필터링 이라고도 함) 또는 이방성 필터링(상수 표 참조) 중에서 선택할 수 있는 필요한 밉 필터링 모드를 참조하는 상수를 제공합니다. 아래에).

일정한 설명
tf_point 이렇게 하면 밉맵 수준 간의 필터링이 비활성화되어 가시적인 텍스처 전환이 발생할 수 있지만 최상의 성능을 제공합니다.
tf_linear 이렇게 하면 밉맵 수준 간의 선형 필터링( 삼선형 필터링 이라고도 함)이 활성화되어 텍스처 전환을 부드럽게 하지만 성능에는 약간의 영향을 미칩니다.
tf_anisotropic 이렇게 하면 밉맵 수준 간의 이방성 필터링이 활성화되어 텍스처 전환 품질이 크게 향상되고 다른 필터링 모드에서 볼 수 있는 흐림 현상을 줄일 수 있지만 성능에는 가장 큰 영향을 미칩니다.

통사론:

gpu_set_tex_mip_filter(filter);

논쟁 유형 설명
filter 밉매핑 필터 상수 The mip filter mode to use (a constant, default: tf_point ).

보고:

N/A

예시:

if keyboard_check(vk_enter)
{
    switch(gpu_get_tex_mip_filter())
    {
        case tf_point: gpu_set_tex_mip_filter(tf_linear); break;
        case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break;
        case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break;
    }
}

위의 코드는 키보드를 확인하고 지정된 키를 누르면 현재 밉 필터를 가져오고 다른 모드를 순환하면서 값을 다음 필터로 토글합니다.
