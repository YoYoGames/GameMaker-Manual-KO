gpu_set_tex_max_aniso

이 기능을 사용하면 사용 시 현재 최대 등방성 수준을 설정할 수 있습니다. tf_anisotropic 필터 모드(참조gpu_get_tex_mip_filter() 자세한 내용은). 입력 값의 범위는 1에서 16 사이여야 합니다.

통사론:

gpu_set_tex_max_aniso(maxaniso);

논쟁 유형 설명
maxaniso The maximum level for anisotropic filtering (default: 16)

보고:

예시:

if gpu_get_tex_max_aniso() != 8
{
    gpu_set_tex_max_aniso(8);
}

위의 코드는 현재 최대 이방성 필터링 수준을 확인하고 8이 아니면 8로 설정합니다.
