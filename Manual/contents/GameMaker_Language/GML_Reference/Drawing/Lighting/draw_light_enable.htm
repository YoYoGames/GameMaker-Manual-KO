draw_light_enable

이 기능은 정의된 조명을 활성화하는 데 사용됩니다. 위치 조명이나 방향 조명을 정의할 때 인덱스 번호를 할당해야 하며 이 번호는 조명을 켜거나 끄기 위해 이 기능에서 사용합니다. 기본값은 비활성화되어 있습니다(false ).

참고: 하드웨어 조명은 8개만 사용할 수 있으므로 정의된 조명은 한 번에 8개만 활성화할 수 있습니다(더 많은 것을 정의할 수 있음).

통사론:

draw_light_enable(ind, enable);

논쟁 유형 설명
ind 진짜 The index number of the light (from 0 to 7)
enable 부울 Enable or disable all lighting (true or false )

보고:

N/A

예시:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

위의 코드는 전체 장면에 대한 조명을 활성화한 다음 방 공간에 흰색 방향 조명을 정의한 다음 마지막으로 해당 조명을 켭니다.

뒤: 조명
다음: draw_set_lighting
