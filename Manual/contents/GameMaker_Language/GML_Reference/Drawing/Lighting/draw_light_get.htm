draw_light_get

이 함수는 지정된 조명 매개변수를 다음 6개 요소가 있는 배열로 가져옵니다.

  • [0] = 활성화/비활성화(true / false )
  • [1] = 조명 유형 상수 (아래 표 참조)
  • [2] = x 위치
  • [3] = y 위치
  • [4] = z 위치
  • [5] = 라이트 반경(포인트 라이트의 경우에만, 방향성의 경우 0이 됨)
  • [6] = 밝은 색(실제)

조명 유형(요소 [1])은 다음 두 상수 중 하나일 수 있습니다.

조명 유형 상수
일정한 설명
lighttype_dir 빛은 지향성 빛이다.
lighttype_point 라이트는 포인트 라이트

통사론:

draw_light_get(ind);

논쟁 유형 설명
ind 진짜 The index number of the light (from 0 to 7)

보고:

Array

예시:

light_a = draw_light_get(1);
if light_a[5] < 200
{
    light_a[5] += 5;
    draw_light_define_point(1, 200, 123, 50, light_a[5], c_white);
}

위의 코드는 "1"로 인덱싱된 조명을 정의하는 데 사용되는 값을 가져온 다음 반경을 확인하고 200보다 작으면 증가하고 조명 반경을 새 값으로 설정합니다.
