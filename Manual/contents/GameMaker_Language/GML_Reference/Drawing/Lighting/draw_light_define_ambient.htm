draw_light_define_ambient

이 기능은 장면의 주변광을 제어하는 데 사용됩니다. 이는 점이나 방향 광원을 정의하지 않은 경우에도 장면에 있는 빛입니다. 이는 효과적으로 장면의 전체 색상과 밝기(또는 어두움)입니다. 기본 색상은 c_black.

통사론:

draw_light_define_ambient(col);

논쟁 유형 설명
col The colour to use (either a constant, a real or a hex value).

보고:

예시:

draw_light_define_ambient(c_white);

위의 코드는 주변 조명을 흰색으로 정의합니다.
