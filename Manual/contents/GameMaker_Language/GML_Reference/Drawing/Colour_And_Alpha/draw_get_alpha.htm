draw_get_alpha

이 함수는 0(완전 투명)에서 1(완전 불투명) 사이의 범위인 그리기 알파의 현재 값을 반환합니다. 그리기 알파는 모든 그리기 기능의 투명도에 영향을 미치며 다음으로 설정할 수 있습니다. draw_set_alpha() 함수.

통사론:

draw_get_alpha()

보고:

예시:

var _cur_alpha = draw_get_alpha();
draw_set_alpha(text_alpha);
draw_text(x, y, text);
draw_set_alpha(_cur_alpha);

위의 코드는 현재 그리기 알파를 지역 변수에 저장하고 인스턴스 변수를 기반으로 그리기 알파를 변경합니다. 일부 텍스트를 그린 후 알파를 지역 변수에 저장된 값으로 다시 재설정합니다.
