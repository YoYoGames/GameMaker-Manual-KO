make_colour_rgb

GameMaker 는 사용자가 자신의 색상을 만들 수 있도록 이 기능(및 다른 기능)을 제공합니다. 이 특정 기능은 만들고자 하는 색상의 빨강, 녹색 및 파랑 구성 요소의 세 가지 구성 요소를 사용합니다. 이 값은 0에서 255 사이로 간주되므로 16,777,216(256*256*256) 색상을 만들 수 있습니다! 아래에서 이러한 구성 요소가 분리될 때 어떻게 보이는지 이미지를 볼 수 있습니다.

왼쪽의 이미지는 기능의 개별 구성 요소에 대한 분석이고 오른쪽은 이러한 구성 요소를 변경하면 최종 색상에 어떤 영향을 미치는지 보여줍니다.

통사론:

make_colour_rgb(red, green, blue);

논쟁 유형 설명
red The red component of the colour
green The green component of the colour
blue The blue component of the colour

보고:

Colour Constant

예시:

col = make_colour_rgb(100, 145, 255);

위의 코드는 함수를 사용하여 색상을 만들고 나중에 사용할 수 있도록 "col" 변수에 값을 저장합니다.
