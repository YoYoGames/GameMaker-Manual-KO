색상과 알파

GameMaker 에는 화면에 그려지는 것의 색상과 알파를 조작하는 포괄적인 기능이 있습니다. 이러한 기능을 사용하여 색상을 구성 요소 부분으로 분리하고, 새로운 색상을 생성하고, 그리기 알파를 조작하여 게임에서 서로 다른 것들이 함께 그려지는 방식을 변경할 수 있습니다.

특수 혼합 모드 및 기타 GPU 특정 기능을 사용하여 이미지를 혼합하려면 GPU 제어 섹션을 참조하십시오.

16진수 색상

$ 기호를 사용하여 16진수 값에서 색상을 미리 생성할 수 있으며, 이러한 값을 다양한 색상 함수와 함께 사용하여 구성 요소 값을 개별적으로 가져올 수도 있습니다. 색상의 16진수 값은 Blue / Green / Red (또는 $BB GGRR ) 형식으로 세 부분으로 나뉩니다. 예를 들어 연한 파란색/청록색을 정의하려면 다음을 사용할 수 있습니다.

col = $FFCC11;

여기서 FF 는 파란색 성분, CC 는 녹색 성분, 11 은 빨간색 성분입니다.

해시/파운드 기호( # )로 시작하는 "CSS 색상"을 사용할 수도 있습니다. 이것은 더 널리 사용되고 다양한 다른 응용 프로그램과 호환되는 Red / Green / Blue 형식(또는 #RRGGBB )을 사용합니다.

col = #11CCFF;

이것은 이전 코드 블록에 표시된 것과 동일한 색상이며 형식과 일치하도록 RR 및 BB 값만 교환됩니다.

참고 GML은 $AABB GGRR 형식(시작 부분에 알파 16진수 값 포함)도 지원하지만 대부분의 GML 색상 기능은 색상 부분만 사용하고 알파를 무시할 수 있습니다.
