make_colour_hsv

GameMaker 는 사용자가 자신의 색상을 만들 수 있도록 이 기능(및 다른 기능)을 제공합니다. 이 특정 기능은 원하는 색상을 생성하기 위해 색조, 채도 및 값 ("광도"라고도 함)의 세 가지 구성 요소를 사용합니다. 이 값은 0에서 255 사이로 간주되므로 16,777,216(256*256*256) 색상을 만들 수 있습니다! 아래에서 이러한 구성 요소가 분리될 때 어떻게 보이는지 이미지를 볼 수 있습니다.

왼쪽의 이미지는 기능의 개별 구성 요소에 대한 분석이고 오른쪽은 이러한 구성 요소를 변경하면 최종 색상에 어떤 영향을 미치는지 보여줍니다. 보시다시피 색조를 변경하지 않고도(0, 빨간색으로 유지) 다른 두 설정으로 색상을 극적으로 변경할 수 있습니다. 채도는 색조를 회색(0)에서 순수한 톤(255)으로 변경합니다., 값을 사용하여 더 어둡게(0)에서 더 밝게(255)로 이동합니다.

통사론:

make_colour_hsv(hue, sat, val);

논쟁 유형 설명
hue The hue of the colour
sat How saturated the colour is
val How dark the colour is

보고:

Colour Constant

예시:

col = make_colour_hsv(100, 145, 255);

위의 코드는 함수를 사용하여 색상을 만들고 나중에 사용할 수 있도록 "col" 변수에 값을 저장합니다.
