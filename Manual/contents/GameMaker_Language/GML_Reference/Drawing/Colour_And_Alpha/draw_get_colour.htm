draw_get_colour

이 함수는 양식, 텍스트, 기본 요소 및 질감이 없는 3D 모델을 그리는 데 사용되는 현재 그리기 색상을 반환합니다. 이것은 다음으로 설정할 수 있습니다.draw_set_colour() 함수.

통사론:

draw_get_colour()

보고:

Colour Constant

예시:

var _cur_color = draw_get_color();
draw_set_color(text_color);
draw_text(x, y, text);
draw_set_color(_cur_color);

위의 코드는 현재 그리기 색상을 로컬 변수에 저장하고 인스턴스 변수에 따라 그리기 색상을 변경합니다. 일부 텍스트를 그린 후 색상을 다시 로컬 변수에 저장된 값으로 재설정합니다.
