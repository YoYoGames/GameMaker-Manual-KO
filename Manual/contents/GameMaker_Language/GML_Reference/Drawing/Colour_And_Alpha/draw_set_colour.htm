draw_set_colour

이 기능을 사용하여 게임의 기본 그리기 색상을 설정할 수 있습니다.

이것은 글꼴, 양식, 프리미티브 및 3D 그리기에 영향을 주지만 스프라이트( 수동으로 또는 인스턴스로 그린 것)에는 영향을 미치지 않습니다. 영향을 받는 그래픽이 고유한 색상 값으로 그려지면 이 값은 무시됩니다.

통사론:

draw_set_colour(col);

논쟁 유형 설명
col 색깔 The colour to set for drawing.

보고:

N/A

예시:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

위의 코드는 수정된 그리기 알파 및 색상으로 생성된 그림자 효과를 사용하여 지정된 위치에 일부 텍스트를 그립니다.
