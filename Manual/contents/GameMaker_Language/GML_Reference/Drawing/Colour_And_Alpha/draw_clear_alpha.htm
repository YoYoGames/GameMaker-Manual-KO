draw_clear_alpha

이 기능은 주어진 색상으로 전체 화면을 지우는 데 사용할 수 있으며 대상의 알파 구성 요소는 설정한 값으로 설정됩니다. 이 기능은 작동할 때 혼합을 수행하지 않지만 후속 혼합 작업은 설정한 대상 알파를 사용합니다. 이것은 인스턴스의 그리기 이벤트에서만 사용하기 위한 것이며(다른 이벤트에서 사용되는 경우 표시되지 않음), 새로 생성된 표면 을 지우는 데에도 매우 유용할 수 있습니다.

HTML5 모듈을 사용할 때 이 기능을 사용하여 캔버스 배경을 투명하게 만들 수 있다는 점은 주목할 가치가 있습니다. 그러면 호스트 HTML에서 사용되는 모든 이미지가 게임에서 실행될 때 볼 수 있습니다. 이 효과를 얻으려면 먼저 방 편집기로 이동하여 방 편집기 에서 배경 레이어 를 제거한 다음 보기 속성 에서 보기 배경 지우기 옵션을 선택 해제해야 합니다. 마지막으로 새 개체를 만들어 게임의 첫 번째 방에 배치하고 매우 높은 깊이(예: 10000)를 할당해야 합니다. 그런 다음 사전 그리기 이벤트에서 이 기능을 사용하여 캔버스 알파를 원하는 색상과 투명도로 설정합니다.

통사론:

draw_clear_alpha(col, alpha);

논쟁 유형 설명
col The colour with which the screen will be cleared
alpha The transparency of the colour with which the screen will be cleared

보고:

예시:

draw_clear_alpha(c_red, 0.5);

이렇게 하면 빨간색으로 50%(0.5) 투명도로 화면이 지워집니다.
