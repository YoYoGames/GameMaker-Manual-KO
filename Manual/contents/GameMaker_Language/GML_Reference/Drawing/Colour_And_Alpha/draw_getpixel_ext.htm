draw_getpixel_ext

이 기능을 사용하면 현재 렌더 대상에 그려지는 모든 픽셀의 전체 AGBR 32비트 값을 얻을 수 있습니다. 즉, 결과는 함수가 호출된 이벤트와 사용 중인 대상 표면에 따라 달라집니다.

중요 이 기능은 성능에 큰 영향을 미치므로 반드시 필요한 경우에만 사용해야 합니다.

통사론:

draw_getpixel_ext(x, y);

논쟁 유형 설명
x The x coordinate of the pixel to check
y The y coordinate of the pixel to check

보고:

Real

예시:

col = draw_getpixel_ext(mouse_x, mouse_y);
alpha = (col >> 24) & 255;
blue = (col >> 16) & 255;
green = (col >> 8) & 255;
red = col & 255;

위의 코드는 마우스 위치에서 32비트 색상 값을 가져온 다음 구성 요소 값으로 분할하여 변수에 저장합니다.
