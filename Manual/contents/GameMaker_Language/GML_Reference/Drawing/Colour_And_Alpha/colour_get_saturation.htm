colour_get_saturation

이 함수는 주어진 색상의 채도를 반환합니다. 이것은 최종 색상에 혼합되는 색조의 양이며 색상을 정의하기 위한 색조, 채도 및 값(광도) 방법의 일부입니다. 다음 이미지는 이 값이 HSV 색상 스케일에 어떻게 대응하는지 보여줍니다.

통사론:

colour_get_saturation(col);

논쟁 유형 설명
col The colour to check

보고:

예시:

col = make_colour_hsv(random(255), colour_get_sat(c_teal), 255);

위의 코드는 색상 상수 "c_teal"을 만드는 데 사용되는 채도를 얻은 다음 해당 값을 사용하여 임의의 색상이 동일한 채도를 갖도록 설정하고 결과 색상을 변수 "col"에 저장합니다.
