draw_clear

이 기능은 전체 화면(알파 블렌드 없음)을 주어진 색상으로 지우는 데 사용할 수 있으며 인스턴스의 그리기 이벤트에서만 사용할 수 있습니다(다른 이벤트에서 사용되는 경우 표시되지 않음). 표면이 새로 생성될 때 표면 을 지우는 데에도 유용할 수 있습니다.

통사론:

draw_clear(col);

논쟁 유형 설명
col The colour with which the screen will be cleared

보고:

예시:

draw_clear(c_blue);

이렇게 하면 화면이 파란색으로 지워집니다.
