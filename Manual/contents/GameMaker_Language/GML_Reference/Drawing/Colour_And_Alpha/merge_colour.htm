merge_colour

이 기능을 사용하면 두 가지 색상을 가져온 다음 함께 병합하여 새로운 색상을 만들 수 있습니다. 각 구성 요소 색상의 양은 "amount" 인수를 변경하여 정의할 수 있습니다. 여기서 값 0은 첫 번째 색상(col1)을 반환하고 값 1은 두 번째 색상(col2)을 반환하고 그 사이의 값을 반환합니다. 해당 믹스를 반환합니다. 예를 들어 값이 0.5이면 두 색상이 동일하게 혼합됩니다. 다음 이미지는 빨간색과 파란색을 함께 병합하여 작동하는 방법을 보여줍니다.

통사론:

merge_colour(col1, col2, amount);

논쟁 유형 설명
col1 The first colour to merge
col2 The second colour to merge
amount How much of each colour should be merged. For example, 0 will return col1, 1 will return col2, and 0.5 would return a merge of both colours equally

보고:

Colour Constant

예시:

col = merge_colour(c_lime, c_orange, 0.5);

위의 코드는 함수를 사용하여 석회와 오렌지를 50/50으로 병합하여 색상을 만든 다음 나중에 사용할 수 있도록 "col" 변수에 값을 저장합니다.
