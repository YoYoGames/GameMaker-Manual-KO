colour_get_green

이 함수는 0에서 255 사이의 값으로 주어진 색상을 만드는 데 사용된 녹색의 양을 반환합니다. 여기서 0은 녹색이 아니고 255는 모두 녹색입니다. 다음 이미지는 이를 보여줍니다.

통사론:

colour_get_green(col);

논쟁 유형 설명
col The colour to check

보고:

예시:

g_comp = colour_get_green(c_teal);

위의 코드는 색상 상수 "c_teal"의 녹색 구성 요소를 가져와 변수 "g_comp"에 값을 저장합니다.
