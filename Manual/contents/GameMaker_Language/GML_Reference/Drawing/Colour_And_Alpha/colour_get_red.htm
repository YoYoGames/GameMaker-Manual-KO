colour_get_red

이 함수는 주어진 색상을 만드는 데 사용된 빨간색의 양을 반환합니다. 값은 0에서 255 사이입니다. 여기서 0은 빨간색이 아니고 255는 모두 빨간색입니다. 다음 이미지는 이를 보여줍니다.

통사론:

colour_get_red(col);

논쟁 유형 설명
col The colour to check

보고:

예시:

r_comp = colour_get_red(c_teal);

위의 코드는 색상 상수 "c_teal"의 빨간색 구성 요소를 가져와 변수 "r_comp"에 값을 저장합니다.
