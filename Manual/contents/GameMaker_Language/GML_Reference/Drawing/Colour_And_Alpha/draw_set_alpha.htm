draw_set_alpha

이 기능을 사용하여 게임의 기본 드로 알파를 설정할 수 있습니다. 이 값은 0에서 1까지 설정할 수 있으며 0은 완전히 투명하고 1은 완전히 불투명하며(기본값) 배경, 스프라이트, 글꼴, 기본 요소 및 3D를 포함한 모든 추가 드로잉에 영향을 미칩니다.

자산을 그리고 함수의 일부로 알파 인수를 제공하는 경우(예:draw_sprite_ext() ), 여기에 설정된 알파 값이 재정의되므로 다음을 사용하여 스프라이트를 그립니다. draw_sprite_ext() 알파 인수가 1이면 알파 세트에 관계없이 스프라이트를 완전히 불투명하게 그립니다. draw_set_alpha() (이것은 사용할 때 동일하게 작동합니다draw_self() 그리고 설정 image_alpha 인스턴스). 또한 인스턴스가 기본 드로잉인 경우(즉, 스프라이트가 있지만 그리기 이벤트가 없는 경우), image_alpha 변수는 다음을 사용하여 알파 세트로 고정됩니다. draw_set_alpha(), 그렇게 image_alpha 1이고 드로 알파 0.5는 알파 0.5에서 스프라이트를 그리는 반면 image_alpha 0.25 및 0.5의 드로 알파는 0.25 알파에서 스프라이트를 그립니다.

통사론:

draw_set_alpha(alpha);

논쟁 유형 설명
alpha The alpha to set (between 0 and 1)

보고:

예시:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

위의 코드는 수정된 그리기 알파 및 색상으로 생성된 그림자 효과를 사용하여 지정된 위치에 일부 텍스트를 그립니다.
