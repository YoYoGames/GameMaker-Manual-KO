draw_text

이 기능을 사용하면 방 안의 어느 위치에서나 문자열을 그릴 수 있습니다(실수를 그리기 위해서는string() 텍스트로 변환하는 기능). 문자열을 결합하려면 + 를 사용할 수 있고(아래 예 참조) 문자열 내에서 \n 을 사용하여 줄 바꿈을 추가하여 여러 줄에 걸쳐 그려지도록 할 수도 있습니다(문자열 형식을 올바르게 지정하는 방법 및 수행할 수 있는 이스케이프 문자에 대한 정보는 사용은 여기를 참조 하십시오 ). 텍스트 및 알파의 색상은 다음으로 설정된 현재 기본 알파 및 색상 값에 의해 결정됩니다. draw_set_alpha() 그리고 draw_set_colour().

참고: 텍스트의 실제 위치는 다음에서 설정한 정렬 값의 영향을 받습니다. draw_set_halign() 그리고 draw_set_valign().

통사론:

draw_text(x, y, string);

논쟁 유형 설명
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.

보고:

예시:

draw_text(x, y, "Hello, " + global.Name + "!\nI hope you are well!");

위의 코드는 인스턴스 x/y 위치에 문자열을 그리며 전역 변수 "Name"에 저장된 문자열을 사용하여 두 줄로 나눕니다.
