draw_get_valign

이 함수는 세로 축을 따라 텍스트 정렬 설정을 가져오는 데 사용되며 아래 나열된 상수 중 하나를 반환합니다.

통사론:

draw_get_valign();

보고:

Constant

수직 정렬 상수
일정한 조정
fa_top
fa_middle
fa_bottom

예시:

var _cur_halign = draw_get_halign();
var _cur_valign = draw_get_valign();

draw_set_halign(fa_right);
draw_set_valign(fa_bottom);

draw_text(100, 32, "Score: " + string(score));

draw_set_halign(_cur_halign);
draw_set_valign(_cur_valign);

위의 코드는 현재 적용된 "halign" 및 "valign" 값을 로컬 변수에 저장한 다음 정렬을 변경하여 일부 텍스트를 그립니다. 그린 후 지역 변수에 저장된 값으로 다시 정렬을 재설정합니다.
