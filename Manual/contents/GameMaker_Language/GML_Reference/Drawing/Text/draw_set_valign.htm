draw_set_valign

이 기능은 수직 축을 따라 텍스트를 정렬하는 데 사용되며 수직 정렬을 변경하면 모든 추가 텍스트가 그려지는 위치와 방향이 변경되며 기본값은 다음과 같습니다. fa_top. 다음 상수가 허용됩니다.

일정한 조정
fa_top
fa_middle
fa_bottom

통사론:

draw_set_valign(valign);

논쟁 유형 설명
valign 수직 정렬 상수 Vertical alignment constant (from the table above).

보고:

N/A

예시:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text(100, 32, "Score: " + string(score));

위의 코드는 텍스트의 중심을 중심으로 점수를 그립니다.
