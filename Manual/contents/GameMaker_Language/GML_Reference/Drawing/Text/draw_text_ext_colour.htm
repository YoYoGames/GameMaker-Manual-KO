draw_text_ext_colour

이 기능은 베이스의 조합입니다. draw_text() 기능 draw_text_ext() 그리고 draw_text_colour() 기능을 사용하여 텍스트의 그라디언트 색상과 줄 간격 및 줄당 최대 너비를 모두 함께 정의할 수 있습니다.

참고 WebGL이 활성화되어 있지 않으면 HTML5 대상에 그라디언트 혼합을 사용할 수 없습니다. 그러나 혼합 색상을 계속 설정할 수 있으며 글꼴은 첫 번째 주어진 색상과 혼합됩니다. 그러나 이러한 방식으로 모든 블렌딩은 중복 글꼴을 생성한 다음 캐시에 저장하고 필요할 때 사용합니다. 이는 최적이 아니며 여러 색상 변경을 사용하는 경우 게임 성능이 느려집니다. 기능을 사용하여 필요한 경우 글꼴 캐시 크기를 설정하여 제한할 수 있습니다.font_set_cache_size().

통사론:

draw_text_ext_colour(x, y, string, sep, w, c1, c2, c3, c4, alpha);

논쟁 유형 설명
x 진짜 The x coordinate of the drawn string.
y 진짜 The y coordinate of the drawn string.
string 끈 The string to draw.
sep 진짜 The distance in pixels between lines of text.
w 진짜 The maximum width in pixels of the string before a line break.
c1 색깔 The colour for the top left of the drawn text.
c2 색깔 The colour for the top right of the drawn text.
c3 색깔 The colour for the bottom right of the drawn text.
c4 색깔 The colour for the bottom left of the drawn text.
alpha 진짜 The alpha for the text.

보고:

N/A

예시:

draw_text_ext_colour(200, 200, keyboard_string, 5, 300, c_blue, c_blue, c_navy, c_navy, 1);

위의 코드는 사용자가 키보드에 입력하는 모든 텍스트를 그려서 해당 줄의 문자열 길이가 300픽셀을 초과할 때마다 새 줄로 분할합니다. 이 경우 코드는 행 사이에 5픽셀의 간격을 유지합니다. 텍스트의 각 줄은 파란색 그라디언트를 사용하여 색상이 지정되며 위쪽은 밝은 파란색, 아래쪽은 진한 파란색입니다.
