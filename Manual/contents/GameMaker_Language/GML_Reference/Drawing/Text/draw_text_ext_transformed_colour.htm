draw_text_ext_transformed_colour

이 기능은 베이스의 조합입니다. draw_text() 기능 draw_text_ext(), draw_text_transformed() 그리고 draw_text_colour() 기능. 이처럼 텍스트를 그리기 위한 모든 기능 중 가장 일반적인 기능으로, 그리기에 유연성을 많이 줍니다. "width" 인수는 스케일 1을 기준으로 하므로 스케일이 다른 경우 이 값을 비례적으로 변경해야 합니다. 예를 들어 줄 바꿈의 기본 너비가 300이고 배율을 2로 설정하면 텍스트가 잘못 표시되어 주어진 너비를 초과하게 됩니다. 대신 크기를 보정하려면 너비를 150으로 설정해야 합니다.

참고: WebGL이 활성화되어 있지 않으면 HTML5 대상에 그라디언트 혼합을 사용할 수 없습니다. 그러나 혼합 색상을 계속 설정할 수 있으며 글꼴은 첫 번째 주어진 색상과 혼합됩니다. 그러나 이러한 방식으로 모든 블렌딩은 중복 글꼴을 생성한 다음 캐시에 저장하고 필요할 때 사용합니다. 이는 최적이 아니며 여러 색상 변경을 사용하는 경우 게임 성능이 느려집니다. 기능을 사용하여 필요한 경우 글꼴 캐시 크기를 설정하여 제한할 수 있습니다.font_set_cache_size().

통사론:

draw_text_ext_transformed_colour(x, y, string, sep, w, xscale, yscale, angle, c1, c2, c3, c4, alpha);

논쟁 유형 설명
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
sep The distance in pixels between lines of text.
w The maximum width in pixels of the string before a line break.
xscale The horizontal scale.
yscale The vertical scale.
angle The angle of the text.
c1 The colour for the top left of the drawn text.
c2 The colour for the top right of the drawn text.
c3 The colour for the bottom right of the drawn text.
c4 The colour for the bottom left of the drawn text.
alpha The alpha for the text.

보고:

예시:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text_ext_transformed_colour(200, 200, "My name is" + global.name +".", 20, 200, 0.5, 0.5, 180, c_blue, c_blue, c_navy, c_navy, 1);

이것은 방의 위치 (200,200)에서 파란색에서 진한 파란색의 하향 그라디언트로 주어진 텍스트를 거꾸로 회전하고 글꼴의 일반 포인트 크기의 절반으로 그립니다. 하나 이상의 선이 그려지고 각 선의 최대 너비는 200픽셀이어야 하며 각 선 사이에는 20픽셀이 있어야 합니다.
