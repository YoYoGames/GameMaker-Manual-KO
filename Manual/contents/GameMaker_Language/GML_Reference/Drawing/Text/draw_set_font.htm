draw_set_font

이 기능은 모든 추가 텍스트 그리기에 사용할 글꼴을 설정합니다. 이 글꼴은 게임의 글꼴 자산 에 추가되었거나 다음 중 하나를 사용하여 생성되어야 합니다. font_add() 또는 font_add_sprite().

통사론:

draw_set_font(font);

논쟁 유형 설명
font The name of the font to use.

보고:

예시:

draw_set_colour(c_blue);
draw_set_font(fnt_Game);
draw_text(200, 200, "Hello World");

위의 코드는 "fnt_Game" 변수에 색인된 글꼴과 파란색을 사용하여 주어진 텍스트를 그립니다.
