draw_text_ext

이 기능은 다음과 유사한 방식으로 텍스트를 그립니다. draw_text() 이제 각 텍스트 줄 사이의 공간을 설정할 수 있습니다. 텍스트가 한 줄 이상을 차지하는 경우, 줄당 문자열의 너비(픽셀 단위)를 제한하여 어떤 줄이 이 값을 초과하면 GameMaker 가 자동으로 텍스트를 분할합니다. 사용 가능한 가장 가까운 공백에서 다음 줄로 이동합니다(텍스트에 공백이 없으면 이 최대 너비 값을 초과합니다). 이 때문에 문자열 시작 부분에 있는 공백은 그리기를 위해 구문 분석되기 전에 제거됩니다. 또한 행 분리 인수의 값 -1은 기본적으로 선택한 글꼴의 "M" 문자 높이를 기반으로 하는 분리로 설정됩니다.

통사론:

draw_text_ext(x, y, string, sep, w);

논쟁 유형 설명
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
sep The distance in pixels between lines of text.
w The maximum width in pixels of the string before a line break.

보고:

예시:

draw_text_ext(100, 50, keyboard_string, 3, 300);

위의 코드는 사용자가 키보드에 입력하는 모든 텍스트를 그려서 해당 줄의 문자열 길이가 300픽셀을 초과할 때마다 새 줄로 분할합니다. 이 경우 코드는 행 사이에 3픽셀의 간격을 유지합니다.
