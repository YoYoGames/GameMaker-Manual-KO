텍스트

다양한 형식으로 텍스트를 그리는 데 사용할 수 있는 여러 기능이 있습니다. 이러한 기능 중 일부는 그릴 때 텍스트 색상과 크기를 동적으로 변경하지만 HTML5 또는 장치로 작업할 때 이러한 기능은 게임 속도를 늦추거나 좋지 않게 보일 수 있으므로 다양한 기능을 사용하는 것이 좋습니다. 크기가 조정된 그리기를 사용하는 대신 크기가 지정된 글꼴 리소스를 사용하거나, 큰 글꼴 리소스를 가지고 있는 다음 확대가 아닌 축소 하여 그 립니다. 또한 큰 텍스트 블록에 색상을 지정해야 하는 경우 다음을 사용하는 것이 좋습니다.draw_set_colour() 그리고draw_set_alpha() 간단한 기능과 함께 먼저 draw_text 색상 구성 요소가 있는 확장 기능이 아닌 기능입니다.

다음 기능은 그릴 때 사용할 글꼴을 설정하고 그릴 때 텍스트의 위치를 지정하는 기능입니다.

  • draw_set_font
  • draw_set_halign
  • draw_set_valign
  • draw_get_font
  • draw_get_halign
  • draw_get_valign

이 함수는 간단하게 또는 변환 및 색상을 사용하여 화면에 텍스트를 그리는 데 사용할 수 있습니다.

  • draw_text
  • draw_text_ext
  • draw_text_colour
  • draw_text_transformed
  • draw_text_ext_colour
  • draw_text_ext_transformed
  • draw_text_transformed_colour
  • draw_text_ext_transformed_colour

내장된 최고 점수 시스템을 사용하여 생성된 최고 점수 그리기와 관련된 다음 기능도 존재합니다.

  • draw_highscore
