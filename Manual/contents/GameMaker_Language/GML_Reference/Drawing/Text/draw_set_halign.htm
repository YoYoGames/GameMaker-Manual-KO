draw_set_halign

이 기능은 수평 축을 따라 텍스트를 정렬하는 데 사용되며 수평 정렬을 변경하면 모든 추가 텍스트가 기본값으로 그려지는 위치와 방향이 변경됩니다. fa_left. 다음 상수가 허용됩니다.

일정한 조정
fa_left
fa_center
fa_right

통사론:

draw_set_halign(halign);

논쟁 유형 설명
halign 수평 정렬 상수 Horizontal alignment constant (from the table above).

보고:

N/A

예시:

draw_set_halign(fa_left);
draw_text(100, 32, "Score: " + string(score));
draw_set_halign(fa_right);
draw_text(room_width - 100, 32, "Health: " + string(health));

위의 코드는 같은 줄에 두 개의 문자열을 그립니다. 점수는 왼쪽으로 정렬되고 건강은 오른쪽으로 정렬됩니다.
