surface_get_target

이 기능을 사용하면 그리기 대상이 되는 현재 표면의 ID를 얻을 수 있습니다. 다음을 사용하여 해당 표면을 만든 경우surface_create(), 그러면 해당 표면의 고유 ID인 양의 정수가 됩니다.

사용자 정의 표면이 설정되지 않았지만 기본 application_surface가 여전히 활성화되어 있으면 함수는 일반적으로 0을 반환합니다. 그러나 이전에 application_surface를 조작한 적이 있다면 얻은 ID가 현재 값과 일치하지 않는지 확인하고 싶을 수 있습니다. application_surface 변수(더 이상 0이 아닌 경우).

사용자 정의 표면이 설정되지 않고 기본 application_surface도 비활성화된 경우 함수는 -1을 반환합니다.

통사론:

surface_get_target();

보고:

Surface ID or -1 (if no target surface is set)

예시:

if surface_get_target() != -1
{
    surface_reset_target();
}
draw_surface(surf, 0, 0);

위의 코드는 현재 렌더 대상이 표면인지 확인하고, 그렇다면 대상을 재설정한 다음 표면을 그립니다.
