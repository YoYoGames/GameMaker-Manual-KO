draw_surface_ext

이 함수는 함수에서와 같이 주어진 표면을 그립니다.draw_surface() 그러나 그려지는 표면의 크기, 혼합, 회전 및 알파를 변경하는 추가 옵션이 있습니다. 이 값을 변경해도 리소스는 어떤 식으로든 수정되지 않습니다 (그릴 방식만).

통사론:

draw_surface_ext(id, x, y, xscale, yscale, rot, col, alpha);

논쟁 유형 설명
id 표면 ID The unique ID value of the surface to draw.
x 진짜 The x position of where to draw the surface.
y 진짜 The y position of where to draw the surface.
xscale 진짜 The horizontal scale.
yscale 진짜 The vertical scale.
rot 진짜 The rotation or angle to draw the surface.
col 색깔 The colour with which to blend the surface.
alpha 진짜 The alpha transparency for drawing the surface.

보고:

N/A

예시:

draw_surface_ext(surf, 0, 0, 2, 2, 0, c_red, 0.5);

위의 코드는 방의 (0,0) 위치에서 변수 "surf"에 색인된 표면을 원래 크기의 두 배, 혼합된 빨간색과 반투명으로 그립니다.
