draw_surface_part

이 기능을 사용하면 방 안의 주어진 위치에서 표면의 일부를 그릴 수 있습니다. 와 같은 draw_surface() 표면을 지정할 수 있지만 그리기 위해 선택할 영역의 표면 내에서 상대 좌표 를 지정해야 합니다. 이것은 왼쪽 위치가 0이고 맨 위 위치가 0이면 표면의 왼쪽 맨 위 모서리가 되며 모든 추가 좌표는 해당 위치에서 가져와야 함을 의미합니다.

통사론:

draw_surface_part(id, left, top, w, h, x, y);

논쟁 유형 설명
id 표면 ID The unique ID value of the surface to draw.
left 진짜 The left position in the surface of the part to be drawn.
top 진짜 The top position in the surface of the part to be drawn.
w 진짜 The width of the part to be draw, from left.
h 진짜 The height of the part to be drawn, from top.
x 진짜 The x position of where to draw the surface.
y 진짜 The y position of where to draw the surface.

보고:

N/A

예시:

draw_surface_part(surf, 8, 8, 32, 32, x, y);

이렇게 하면 인스턴스(x,y) 위치에서 "surf"로 인덱싱된 표면의 왼쪽 상단에서 8px x 8px in의 32x32 영역이 그려집니다.
