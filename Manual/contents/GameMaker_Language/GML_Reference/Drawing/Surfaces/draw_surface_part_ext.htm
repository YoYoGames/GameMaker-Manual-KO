draw_surface_part_ext

이 기능은 동일한 규칙에 따라 주어진 위치에서 선택한 표면의 일부를 그립니다.draw_surface_part(), 이제 부분의 크기를 조정하거나, 색상을 혼합하거나, 화면에 그릴 때 알파를 변경할 수 있습니다(표면을 그릴 때와 동일).draw_surface_ext() ).

통사론:

draw_surface_part_ext(id, left, top, w, h, x, y, xscale, yscale, colour, alpha);

논쟁 유형 설명
id 표면 ID The unique ID value of the surface to draw.
left 진짜 The left position in the surface of the part to be drawn.
top 진짜 The top position in the surface of the part to be drawn.
w 진짜 The width of the part to be draw, from left.
h 진짜 The height of the part to be drawn, from top.
x 진짜 The x position of where to draw the surface.
y 진짜 The y position of where to draw the surface.
xscale 진짜 The horizontal scaling the part should be drawn with.
yscale 진짜 The vertical scaling the part should be drawn with.
colour 색깔 The colour blending the part should be drawn with.
alpha 진짜 The alpha transparency the part should be drawn with.

보고:

N/A

예시:

draw_surface_part_ext(surf, 8, 8, 32, 32, x, y, 2, 0.5, c_black, 1);

이것은 8x8 픽셀에서 32x32 픽셀 영역을 "surf" 변수에 색인된 표면으로 그립니다. 평소 너비의 두 배로 늘어나지만 높이의 절반이 됩니다. 불투명하고 검정색과 혼합됩니다(실루엣으로 전환).
