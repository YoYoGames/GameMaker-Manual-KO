surface_create

이 함수는 표면을 생성하는 데 사용되며 향후 함수 호출을 위해 변수에 저장해야 하는 표면의 인덱스를 반환합니다. 표면이 처음 생성될 때 기본적으로 목적을 위해 따로 보관된 메모리 영역이기 때문에 "노이즈"가 포함될 수 있으므로(그 메모리에는 여전히 정보가 포함될 수 있음) 같은 기능draw_clear_alpha().

모든 표면은 2의 거듭제곱인 크기(예: 16, 128, 512 또는 1024픽셀)로 만드는 것이 좋습니다. 이것은 특정 플랫폼(Windows 및 MacOS와 같은)에서 꼭 필요한 것은 아니지만 해당 대상에서 호환성을 확실히 증가시키는 반면 HTML5 및 장치의 경우 필수적이며 이것을 기억하지 않으면 나중에 문제가 발생할 수 있다는 점을 매우 중요합니다.

통사론:

surface_create(w, h);

논쟁 유형 설명
w The width of the surface to be created.
h The height of the surface to be created.

보고:

Surface ID

예시:

if !surface_exists(surf)
{
    surf = surface_create(1024, 1024);
    surface_set_target(surf);
    draw_clear_alpha(c_black, 0);
    surface_reset_target();
    view_surface_id[0] = surf;
}

위의 코드는 표면이 존재하는지 확인하고 존재하지 않으면 인덱스를 변수 "surf"에 할당하여 너비가 1024픽셀, 높이가 1024픽셀인 표면을 생성합니다. 그런 다음 그리기 대상이 새 표면으로 설정되고 그리기 대상이 디스플레이로 재설정되기 전에 지워지고 투명해집니다. 마지막으로 표면이 뷰에 할당됩니다.
