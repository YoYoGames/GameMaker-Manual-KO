application_surface_enable

이 기능을 사용하여 애플리케이션 표면을 활성화하거나 비활성화할 수 있습니다. 기본적으로 응용 프로그램 표면이 활성화되어 있고 그리기 이벤트에 대한 모든 그리기가 해당 표면에 수행되고 표면이 화면에 그려집니다. 그러나 일부 구형 장치 및 특정 칩셋에서는 성능이 저하될 수 있습니다. 이러한 경우 응용 프로그램 표면에 대한 그리기를 끌 수 있으며 모든 그리기 이벤트 그리기는 화면 버퍼에 직접 수행됩니다.

이 기능을 사용하여 게임에서 언제든지 애플리케이션 표면을 활성화하거나 비활성화할 수 있지만 변경 사항을 적용하려면 최소한 한 프레임(단계)이 필요합니다. 이 기능을 끄면 대상 플랫폼의 게임 옵션 에 설정된 대로 모든 종횡비 보정이 꺼집니다. 즉, 게임을 필요한 장치, 창 또는 화면 크기로 확장하는 방법이 있어야 합니다. 다시 켜면 게임 옵션 이 다음 단계에 다시 적용됩니다.

이 기능은 모든 것이 화면에 그려지는 방식에 영향을 미치므로 주의해서 사용하는 것이 좋습니다. 일반적으로 게임 메뉴의 옵션으로 사용하거나 게임 시작 시 한 번만 호출하고 활성화 또는 비활성화 여부에 관계없이 코드를 작성하고 싶을 것입니다.

통사론:

application_surface_enable(enable);

논쟁 유형 설명
enable Set to enable (true ) or disable (false ) the application surface.

보고:

예시:

if keyboard_check_pressed(vk_space)
{
    if application_surface_is_enabled()
    {
        application_surface_enable(false);
    }
    else
    {
        application_surface_enable(true);
    }
}

위의 코드는 키 누름을 확인하고 상태에 따라 애플리케이션 표면을 켜거나 끕니다(예: 옵션 메뉴).
