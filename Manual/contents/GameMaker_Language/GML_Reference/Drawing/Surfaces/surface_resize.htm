surface_resize

이 함수는 표면의 크기를 주어진 치수(픽셀 단위)로 조정합니다. "surface_id"는 이전에 생성한 표면 또는application_suface, 함수는 해당 표면의 크기를 조정합니다. 이것은 표면의 내용을 자르거나 늘리지 않으며 오히려 현재 표면을 파괴하고 새로운 차원으로 동일한 핸들(surface_id)을 사용하여 재생성합니다. 즉, 지우고 다시 그려야 함을 의미합니다. 그것은 application_surface 이 경우 GameMaker 는 자동으로 이 작업을 수행합니다.

참고: 응용 프로그램 표면의 크기를 조정하는 경우 이러한 변경 사항은 다음 그리기 프레임이 시작될 때까지 등록되지 않습니다.surface_get_width()또는 surface_get_height() 동일한 이벤트 또는 단계의 함수는 이전 값을 반환합니다.

통사론:

surface_resize(surface_id, w, h);

논쟁 유형 설명
surface_id The ID of the surface to change.
w The width of the new surface.
h The height of the new surface.

보고:

예시:

if view_wport[0] != surface_get_width(application_surface) || view_hport[0] != surface_get_height(application_surface)
{
    surface_resize(application_surface, view_wport[0],view_hport[0]);
}

위의 코드는 "application_surface" 표면의 크기와 비교하여 뷰 포트 크기를 확인하고 다르면 표면의 크기가 조정됩니다.
