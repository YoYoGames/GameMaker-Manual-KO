surface_get_width

이 함수는 단순히 인덱싱된 표면의 너비(픽셀)를 반환합니다. 이것을 확인하기 위해 전화를 걸면 application_surface 사용하여 크기를 변경한 직후surface_resize() 변경 사항이 완전히 처리되려면 한두 단계가 필요하므로 새 값을 반환하지 않습니다. 단계를 기다린 후 새 크기를 올바르게 반환해야 합니다.

참고: 표면으로 작업할 때 표면이 텍스처 메모리에 저장되어 있기 때문에 언제든지 존재하지 않을 가능성이 있습니다. 다음을 사용하여 표면이 존재하는지 항상 확인해야 합니다.surface_exists() 직접 참조하기 전에.

통사론:

surface_get_width(surface_id);

논쟁 유형 설명
surface_id The ID of the surface to get the width of.

보고:

예시:

sw = surface_get_width(surf);

위의 코드는 "sw" 변수에 "surf" 변수에 색인된 표면의 너비를 저장합니다.
