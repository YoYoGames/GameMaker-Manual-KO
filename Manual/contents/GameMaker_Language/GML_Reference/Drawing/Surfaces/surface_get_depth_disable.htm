surface_get_depth_disable

이 기능은 표면에 대한 자동 깊이 버퍼 생성이 활성화되어 있는지 확인합니다. 일반적으로 모든 표면에는 깊이 버퍼가 있으므로 3D 개체를 그리면 깊이별로 적절하게 정렬됩니다. 그러나 깊이 버퍼를 할당하면 기본적으로 표면 크기가 두 배가 되며, 이는 특히 게임이 메모리가 많은 경우 과도하고 불필요한 오버헤드가 될 수 있습니다. 집중적이거나 주로 2D. 이러한 경우 이 기능을 사용하여 이를 확인하고 필요한 경우 기능을 사용하여 표면에 대한 깊이 버퍼를 비활성화할 수 있습니다.surface_depth_disable().

통사론:

surface_get_depth_disable();

보고:

예시:

if (!surface_exists(surf))
{
    if surface_get_depth_disable() == false
    {
        surface_depth_disable(true);
    }
    surf = surface_create(room_width, room_height);
}

위의 코드는 주어진 표면이 존재하는지 확인하고 존재하지 않으면 표면 깊이 버퍼의 현재 상태를 확인하고 활성화된 경우 표면을 생성하기 전에 대신 비활성화합니다.
