application_get_position

대상 플랫폼의 게임 옵션에서 "종횡비 유지"를 선택하면 GameMaker 는 응용 프로그램 표면의 그리기 위치를 자동으로 설정하여 주어진 디스플레이의 중앙과 크기가 올바르게 표시되도록 합니다. 그러나 이 표면을 조작하고 있고 직접 그리려는 경우 이 기능을 사용하면 디스플레이 또는 창 내에서 표면이 그려지는 정확한 위치 를 쉽게 찾을 수 있으므로 직접 그리거나 GUI를 정렬할 수 있습니다. 이미지 또는 게시물에 이미지를 그립니다.

이 함수는 4개의 키가 있는 배열 을 반환합니다. 키 0과 1은 표면 왼쪽 상단 모서리의 x 및 y 위치이고 키 2와 3은 표면 오른쪽 하단 모서리의 x 및 y입니다. 디스플레이 또는 창의 크기에 비례합니다.

통사론:

application_get_position();

보고:

예시:

var a = application_get_position();
xx = a[0];
yy = a[1];
ww = a[2] - a[0];
hh = a[3] - a[1];

위의 코드는 응용 프로그램 표면의 위치와 디스플레이 창에 대한 절대 너비 및 높이를 가져와 나중에 사용할 수 있도록 4개의 변수에 저장합니다.
