surface_get_texture

이 함수는 표면 텍스처에 대한 특수 포인터 를 반환합니다. 이 값은 다른 그리기 기능, 특히 일반 3D 및 일부 2D 기본 기능에서 사용할 수 있습니다.

주 표면으로 작업할 때 표면이 텍스처 메모리에 저장되어 있기 때문에 언제든지 존재하지 않을 가능성이 있습니다. 다음을 사용하여 표면이 존재하는지 항상 확인해야 합니다.surface_exists() 직접 참조하기 전에.

통사론:

surface_get_texture(surface_id);

논쟁 유형 설명
surface_id 표면 ID The ID of the surface to get the width of.

보고:

Texture

예시:

var tex;
tex = surface_get_texture(surf);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 480, 0, 0);
draw_vertex_texture(640, 480, 1, 0);
draw_vertex_texture(640, 480, 1, 1);
draw_vertex_texture(0, 480, 0, 1);
draw_primitive_end();

위의 코드는 이전에 생성된 표면에서 가져온 "tex" 변수에 텍스처가 있는 4개의 꼭짓점 삼각형 스트립을 그립니다.
