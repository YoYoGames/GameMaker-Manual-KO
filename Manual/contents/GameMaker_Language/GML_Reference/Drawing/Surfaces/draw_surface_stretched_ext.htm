draw_surface_stretched_ext

이 기능은 다음과 정확히 동일합니다.draw_surface_stretched() 표면이 그려질 때 표면에 대한 색상 혼합 및 알파 값을 설정하는 기능이 추가된 함수(함수와 유사draw_surface_ext() ).

통사론:

draw_surface_stretched_ext(id, x, y, w, h, col, alpha);

논쟁 유형 설명
id 표면 ID The unique ID value of the surface to draw.
x 진짜 The x position of where to draw the surface.
y 진짜 The y position of where to draw the surface.
w 진짜 The width at which to draw the surface.
h 진짜 The height at which to draw the surface.
colour 색깔 The colour with which to colour the surface.
alpha 진짜 The alpha with which to blend the surface.

보고:

N/A

예시:

draw_surface_stretched_ext(surf, x, y, 200, 200, c_white, 0.5);

이렇게 하면 인스턴스 x/y 위치에 왼쪽 모서리가 있는 지정된 표면이 그려지고 블렌딩 없이 부분적으로 투명하게 200x200픽셀 영역을 차지하도록 늘어납니다.
