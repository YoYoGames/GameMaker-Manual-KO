draw_surface

이 기능을 사용하면 방 내의 주어진 위치에 표면을 그릴 수 있으며 표면의 왼쪽 상단 모서리는 지정된 x/y 위치에 그려집니다.

통사론:

draw_surface(id, x, y);

논쟁 유형 설명
id 표면 ID The unique ID value of the surface to draw.
x 진짜 The x position of where to draw the surface.
y 진짜 The y position of where to draw the surface.

보고:

N/A

예시:

var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
draw_surface(surf, _vx, _vy);

위의 코드는 카메라 뷰[0]와 같은 위치에 "surf"로 인덱싱된 표면을 그립니다.
