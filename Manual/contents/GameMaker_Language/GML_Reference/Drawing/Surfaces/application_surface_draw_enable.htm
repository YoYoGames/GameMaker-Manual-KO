application_surface_draw_enable

이 기능을 사용하여 애플리케이션 표면의 자동 그리기를 활성화하거나 비활성화할 수 있습니다. 기본적으로 이것은 활성화되어 있지만 많은 경우 표면이 그려지는 시기와 방법을 인계받고 싶을 것입니다(예를 들어 셰이더를 사용할 때). 이 경우 이 기능을 사용하여 그릴 수 있도록 false로 설정합니다. 언제 어떻게 필요한지. 자동 그리기를 끄고 응용 프로그램 표면을 직접 그릴 때 스프라이트의 알파 구성 요소와 표면 자체에 특정 문제가 나타날 수 있다는 점에 유의하는 것이 중요합니다. 이는 GameMaker 가 자동 그리기가 켜져 있을 때 알파 블렌딩 없이 응용 프로그램 표면을 그리기 때문입니다. 자동 그리기를 끄면 다음 코드(예:)를 사용하여 직접 처리해야 합니다.

gpu_set_blendenable(false);
draw_surface_ext(application_surface, 0, 0, 1, 1, 0, c_white, 1);
gpu_set_blendenable(true);

통사론:

application_surface_draw_enable(flag);

논쟁 유형 설명
flag The flag will be either true (enabled, the default value) or false (disabled).

보고:

예시:

application_surface_draw_enable(false);

위의 코드는 애플리케이션 표면의 자동 그리기를 끕니다.
