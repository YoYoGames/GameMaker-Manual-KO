surface_reset_target

이 기능을 사용하면 대상 표면에서 화면으로 다시 모든 드로잉을 재설정합니다. 표면에 그리기를 시작하려면 먼저 함수를 호출해야 합니다.surface_set_target() 그리고 완료한 후 이것은 설정한 각 표면 대상에 대해 다른 모든 경우에도 표면에서 수행되므로 화면에 아무 것도 그려지지 않습니다. 또한 인스턴스의 그리기 이벤트에서 표면 자체가 화면에 그려지지 않으면 아무 것도 표시되지 않는다는 것을 알아야 합니다.

참고: 이전에 함수로 렌더 대상을 설정하지 않은 경우 surface_set_target(), 이 함수를 사용하면 이벤트에 대한 모든 추가 코드 실행이 자동으로(즉, 오류 메시지 없이) 종료됩니다.

통사론:

surface_reset_target();

보고:

예시:

if view_current = 0
{
    surface_set_target(surf);
    with (obj_Effect)
    {
        draw_self();
    }
    surface_reset_target();
}
else
{
    draw_surface(surf, 0, 0);
}

위의 코드는 현재 어떤 뷰가 그려지고 있는지 확인하고 그것이 view[0]인 경우 그리기 대상을 표면으로 설정하고 그리기 대상을 다시 재설정하기 전에 "obj_Effect" 개체의 모든 인스턴스를 그립니다. 보기가 보기[0]가 아니면 표면이 화면에 그려집니다.
