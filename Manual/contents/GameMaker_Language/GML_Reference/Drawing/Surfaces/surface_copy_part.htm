surface_copy_part

이 함수는 단순히 한 표면에서 이미지를 가져와 해당 표면 내의 지정된 로컬 위치에 있는 다른 표면에 복사합니다(여기서 (0,0) 위치는 대상 표면의 왼쪽 상단 모서리). 섹션의 너비와 높이뿐만 아니라 복사할 로컬 x 및 y 위치를 지정할 수 있습니다. 이것은 방에서 표면이 그려지는 위치가 아니라 표면 크기 를 기반으로 한 좌표입니다. 대상 표면에 이미 정보가 있는 경우 복사본이 이 정보를 덮어쓰며 함수는 어떤 식으로든 소스 표면을 변경하지 않습니다.

참고: 표면으로 작업할 때 표면이 텍스처 메모리에 저장되어 있기 때문에 언제든지 존재하지 않을 가능성이 있습니다. 다음을 사용하여 표면이 존재하는지 항상 확인해야 합니다.surface_exists() 직접 참조하기 전에.

통사론:

surface_copy_part(destination, x, y, source, xs, ys, ws, hs);

논쟁 유형 설명
destination 표면 ID The ID of the surface to copy the other surface to.
x 진짜 The x position to copy to.
y 진짜 The y position to copy to.
source 표면 ID The ID of the surface to be copied.
xs 진짜 The x position in the source surface to copy from.
ys 진짜 The y position in the source surface to copy from.
ws 진짜 The width of the area in the source surface to copy from.
hs 진짜 The height of the area in the source surface to copy from.

보고:

N/A

예시:

if view_current == 0
{
    surface_copy_part(surf, 0, 0, temp_surf, 0, 0, view_xview[1] - mouse_x, view_yview[1] - mouse_y);
}
else
{
    draw_surface(surf, 0, 0);
}

위의 코드는 현재 그려지고 있는 뷰를 확인하고 그것이 view[0]이면 변수 "temp_surf"에 색인된 표면을 변수 "surf"에 색인된 표면에 복사합니다. 복사된 영역은 뷰[1]에 그려지는 표면 내에서 마우스의 상대적 위치에 의해 형성된 직사각형에 해당합니다. 현재 보기가 보기[0] 이외의 것이라면 표면 "서프"가 화면에 그려집니다.
