application_surface

이 전역 내장 변수는 모든 표면 기능 에서 사용하기 위해 애플리케이션 표면에 액세스하는 데 사용할 수 있습니다. 이 표면은 영구적으로 사용 가능하며 GameMaker 에서 게임을 그리는 곳입니다.

일반 Draw 이벤트( Draw Begin, Draw 및 Draw End )는 기본적으로 이 표면에 그립니다. 다른 그리기 이벤트(예: 사전/사후 그리기 및 그리기 GUI )는 애플리케이션 표면에 그리지 않습니다. 이러한 이벤트가 작동하는 방식에 대한 자세한 내용은 이벤트 그리기 를 참조하세요.

통사론:

application_surface;

보고:

Surface ID

예시:

surface_resize(application_surface, display_get_gui_width(), display_get_gui_height())

위의 코드는 GUI 레이어와 1:1 비율을 갖도록 애플리케이션 표면의 크기를 조정합니다.
