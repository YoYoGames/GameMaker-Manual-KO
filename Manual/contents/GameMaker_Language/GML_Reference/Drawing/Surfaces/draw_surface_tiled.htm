draw_surface_tiled

이 함수는 표면을 가져온 다음 함수에 제공한 좌표에서 시작하여 방 전체에 반복적으로 타일을 붙입니다.

통사론:

draw_surface_tiled(id, x, y);

논쟁 유형 설명
id 표면 ID The unique ID value of the surface to draw.
x 진짜 The x position of where to draw the surface.
y 진짜 The y position of where to draw the surface.

보고:

N/A

예시:

draw_surface_tiled(surf, x, y);

이렇게 하면 인스턴스 자체의 x 및 y 위치에서 "surf"로 인덱싱된 표면이 그려지고 방의 모든 방향으로 타일링됩니다.
