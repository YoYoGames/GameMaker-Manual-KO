surface_set_target_ext

이 기능은 셰이더 기능 과 함께 사용하기 위한 것으로 기본 수준 셰이더(DX9, DX11, OpenGL)에 대한 MRT(0 - 3)를 설정합니다.

참고: 표면으로 작업할 때 표면이 텍스처 메모리에 저장되어 있기 때문에 언제든지 존재하지 않을 가능성이 있습니다. 다음을 사용하여 표면이 존재하는지 항상 확인해야 합니다.surface_exists() 직접 참조하기 전에.

통사론:

surface_set_target_ext(index, surface_id);

논쟁 유형 설명
index The render target index to use (from 0 to 3).
surface_id The ID of the surface to use.

보고:

예시:

surface_set_target_ext(0, surf);

위의 코드는 렌더 타겟 0을 변수 "surf"에 색인된 표면 ID로 설정합니다.
