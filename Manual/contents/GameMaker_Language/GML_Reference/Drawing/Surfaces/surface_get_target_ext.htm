surface_get_target_ext

이 함수는 표면에 사용할 수 있는 4개의 렌더 대상 중 하나에 할당된 표면 ID를 검색합니다. 확인할 렌더 대상의 인덱스를 제공하면 표면이 할당되지 않은 경우 함수가 -1을 반환하거나 할당된 표면의 ID를 나타내는 정수 값 >= 0을 반환합니다(함수에 의해 반환됨).surface_create() ).

통사론:

surface_get_target_ext(index);

논쟁 유형 설명
index The render target index to check (from 0 to 3).

보고:

Surface ID or -1 (if no target surface is set)

예시:

if surface_get_target_ext(0) == -1
{
    surface_set_target_ext(0, global.Surf);
}

위의 코드는 먼저 셰이더 렌더 타겟 0이 표면으로 설정되었는지 확인하고 그렇지 않은 경우 할당됩니다.
