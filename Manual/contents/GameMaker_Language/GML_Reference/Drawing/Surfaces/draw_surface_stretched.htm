draw_surface_stretched

이 함수는 단순히 표면을 가져와서 해당 영역을 차지하도록 지정된 너비와 높이로 늘립니다. 와 같은draw_surface() 표면을 지정한 다음 표면이 그려질 공간의 x / y 위치와 마지막으로 너비와 높이(픽셀 값이어야 함)를 지정할 수 있습니다.

통사론:

draw_surface_stretched(id, x, y, w, h);

논쟁 유형 설명
id 표면 ID The unique ID value of the surface to draw.
x 진짜 The x position of where to draw the surface.
y 진짜 The y position of where to draw the surface.
w 진짜 The width at which to draw the surface.
h 진짜 The height at which to draw the surface.

보고:

N/A

예시:

draw_surface_stretched(surf, 10, 10, 100, 100);

이것은 왼쪽 모서리가 (10,10)에 있는 변수 "surf"에 인덱스된 표면을 그립니다. 너비와 높이는 모두 100으로 설정되며, 이는 표면의 실제 너비와 높이에 관계없이 차지하는 공간입니다.
