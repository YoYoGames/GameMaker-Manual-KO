draw_button

이 함수는 현재 선택된 그리기 색상과 알파를 사용하여 매우 간단한 직사각형 "버튼"을 그립니다. 여기서 up 인수는 아래 이미지와 같이 경사진 가장자리 효과가 어떻게 보이는지를 정의합니다.

참고: 셰이더를 사용하여 모양을 그리고 싶다면 대부분의 셰이더가 정점, 질감, 색상과 같은 입력을 예상한다는 점을 알아야 합니다. 그러나 이 기능을 사용할 때 정점과 색상 데이터만 전달되기 때문에 셰이더는 아무 것도 그리지 않을 수 있습니다. 이러한 방식으로 모양을 그려야 하는 경우 이를 염두에 두고 셰이더를 사용자 정의해야 합니다.

통사론:

draw_button(x1, y1, x2, y2, up);

논쟁 유형 설명
x1 진짜 The x coordinate of the left of the button
y1 진짜 The y coordinate of the top of the button
x2 진짜 The x coordinate of the right of the button
y2 진짜 The y coordinate of the bottom of the button
up 부울 Whether the button is up (true ) or down (false )

보고:

N/A

예시:

draw_button(100, 100, 200, 150, !mouse_check_button(mb_left));

이것은 왼쪽 마우스 버튼이 눌려진 상태로 나타날 버튼을 그립니다.
