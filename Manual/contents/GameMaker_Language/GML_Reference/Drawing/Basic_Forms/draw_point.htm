draw_point

이 기능을 사용하면 현재 설정된 그리기 색상과 알파를 사용하여 화면의 아무 곳이나 단일 픽셀을 그릴 수 있습니다.

통사론:

draw_point(x, y);

논쟁 유형 설명
x The x coordinate of the point to be drawn.
y The y coordinate of the point to be drawn.

보고:

예시:

draw_set_colour(c_yellow);
draw_point(100,100);

그러면 (100,100) 위치에 노란색 픽셀이 그려집니다.
