draw_arrow

이 함수는 (x1,y1) 지점에서 (x2,y2) 지점까지 화살표를 그립니다. 화살표의 줄기는 끝에서 그려지는 실제 화살촉과 함께 이러한 점을 따라 그려집니다. 여기서 화살촉의 크기는 "크기" 인수에 의해 정의되고 끝점이 다음과 같이 되도록 줄기의 일부로 계산됩니다. 항상 x2,y2에 의해 정의된 위치와 정렬됩니다. 화살촉의 너비는 길이에 비례하여 자동으로 계산됩니다.

참고: 셰이더를 사용하여 모양을 그리고 싶다면 대부분의 셰이더가 정점, 질감, 색상과 같은 입력을 예상한다는 점을 알아야 합니다. 그러나 이 기능을 사용할 때 정점과 색상 데이터만 전달되기 때문에 셰이더는 아무 것도 그리지 않을 수 있습니다. 이러한 방식으로 모양을 그려야 하는 경우 이를 염두에 두고 셰이더를 사용자 정의해야 합니다.

통사론:

draw_arrow(x1, y1, x2, y2, size);

논쟁 유형 설명
x1 The x coordinate of the start of the line.
y1 The y coordinate of the start of the line.
x2 The x coordinate of the end of the line (where the arrowhead ends).
y2 The y coordinate of the end of the line (where the arrowhead ends).
size The length of the arrow in pixels.

보고:

예시:

draw_arrow(x, y, mouse_x, mouse_y, 10);

위의 코드는 인스턴스 위치에서 마우스 위치까지 화살표를 그립니다. 화살표 머리는 10픽셀입니다.
