draw_set_circle_precision

원을 그릴 때 성능을 최적화하기 위해 GameMaker 는 기본적으로 원형으로 보이 도록 충분한 면이 있는 다각형 모양을 그립니다. 그러나 필요한 서클의 크기에 따라 이 값을 변경하면 게임 성능을 높이거나 서클을 더 잘 보이게 할 수 있습니다. 이 함수에 입력할 수 있는 정밀도 값은 원 다각형의 면 수이며 이 숫자는 4에서 64 사이의 값이 될 수 있지만 4로 나눌 수 있는 숫자여야 하며 기본값은 24입니다.

참고: 셰이더를 사용하여 모양을 그리고 싶다면 대부분의 셰이더가 정점, 질감, 색상과 같은 입력을 예상한다는 점을 알아야 합니다. 그러나 이 기능을 사용할 때 정점과 색상 데이터만 전달되기 때문에 셰이더는 아무 것도 그리지 않을 수 있습니다. 이러한 방식으로 모양을 그려야 하는 경우 이를 염두에 두고 셰이더를 사용자 정의해야 합니다.

통사론:

draw_set_circle_precision(precision);

논쟁 유형 설명
precision The precision of all circles drawn after this function is called.

보고:

예시:

draw_set_circle_precision(12);

위의 코드는 12면 다각형을 사용하여 모든 원을 그립니다.
