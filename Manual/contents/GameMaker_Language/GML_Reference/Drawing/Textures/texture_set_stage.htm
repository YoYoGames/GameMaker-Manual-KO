texture_set_stage

이 함수는 사용할 주어진 스테이지 "슬롯" 텍스처를 설정합니다. 사용 가능한 단계 "슬롯"의 수는 컴파일하는 플랫폼에 따라 다르며 Windows, Mac 및 Linux에서 최대 8개까지 사용할 수 있지만 저사양 Android 장치(예:)에서는 이 숫자가 2만큼 낮을 수 있습니다. 또한 첫 번째 단계 슬롯(1)은 GameMaker 에서 항상 자동으로 사용됩니다.

참고: 이 함수는 실행 중인 셰이더 컨텍스트 외에는 아무 작업도 수행하지 않습니다! 자세한 내용은 셰이더 를 참조하세요.

통사론:

texture_set_stage(stage, tex);

논쟁 유형 설명
stage 진짜 The texture "slot" to use.
tex 텍스처 포인터 The texture to use.

보고:

N/A

예시:

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

위의 코드는 "GlassShader"로 인덱싱된 셰이더 내의 샘플러 핸들 을 가져온 다음 해당 셰이더 상수를 지정된 스프라이트 텍스처로 설정합니다.
