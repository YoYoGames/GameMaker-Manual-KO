텍스처

다음 기능은 모두 "텍스처"와 관련이 있습니다. 즉, VRAM 에 저장되어 게임 내에서 스프라이트 또는 기타 항목으로 사용되는 이미지입니다. 사용 가능한 첫 번째 함수 세트는 텍스처 페이지 에서 특정 이미지와 해당 이미지의 위치 또는 크기에 대한 정보를 검색하고 특정 텍스처 기능을 설정하는 데 사용됩니다.

  • texture_get_width
  • texture_get_height
  • texture_get_texel_width
  • texture_get_texel_height
  • texture_get_uvs
  • texture_set_stage
  • texture_global_scale

GameMaker 에는 개별 텍스처 및 전체 텍스처 그룹과 관련된 VRAM 관리와 관련된 여러 기능도 있습니다.

  • draw_texture_flush
  • texture_prefetch
  • texture_flush
  • sprite_flush
  • sprite_flush_multi
  • sprite_prefetch
  • sprite_prefetch_multi
  • draw_flush

마지막으로 프로젝트를 디버그하고 텍스처 메모리의 효율적인 사용을 보장하는 데 주로 사용되는 다음 함수가 있습니다.

  • texture_debug_messages
  • texture_is_ready
  • texturegroup_get_textures
  • texturegroup_get_sprites
  • texturegroup_get_fonts
  • texturegroup_get_tilesets
