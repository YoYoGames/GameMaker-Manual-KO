texture_is_ready

이 함수는 특정 텍스처 페이지의 압축이 풀렸고 사용할 준비가 되었는지 또는 텍스처 페이지 그룹의 압축이 풀렸고 사용할 준비가 되었는지 확인하는 데 사용할 수 있습니다. 고유한 텍스처 페이지 ID 를 제공합니다( texturegroup_* 함수) 또는 텍스처 그룹 ID 문자열(텍스처 그룹 편집기에 정의된 대로) 및 함수는 다음을 반환합니다. true 포장을 푼 경우 또는 false 그렇지 않으면.

통사론:

texture_is_ready(tex_id);

논쟁 유형 설명
tex_id 텍스처 포인터 또는 문자열 The texture page pointer or a texture group name (a string)

보고:

Boolean

예시:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    if !texture_is_ready(_tex_array[i])
    {
        texture_prefetch(_tex_array[i]);
    }
}

위의 코드는 텍스처 그룹 "MainMenu"에 대한 텍스처 페이지 ID를 검색한 다음 압축이 풀렸는지 확인하고 그렇지 않은 경우 VRAM으로 미리 가져옵니다.
