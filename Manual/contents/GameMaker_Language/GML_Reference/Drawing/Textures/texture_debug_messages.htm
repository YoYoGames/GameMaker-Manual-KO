texture_debug_messages

이 기능은 텍스처 디버그 메시지를 활성화하거나 비활성화하는 데 사용할 수 있습니다. 활성화된 경우(로 설정 true ), 텍스처 페이지 사용에 대한 추가 정보가 출력 창으로 전송됩니다. 로 설정 false 이 출력을 다시 비활성화합니다.

통사론:

texture_debug_messages(enable);

논쟁 유형 설명
enable Enable or disable the texture debug messages

보고:

예시:

if global.debug
{
    texture_debug_messages(true);
}

위의 코드는 전역 변수의 값을 확인하고 다음으로 설정된 경우 true 텍스처 디버그 메시지가 활성화됩니다.
