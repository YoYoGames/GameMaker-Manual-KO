texture_get_height

항상 0 - 1 범위 내의 값인 지정된 ID로 텍스처 높이를 반환합니다. 그러면 텍스처를 모델 또는 기본 요소에 매핑할 때 사용할 수 있습니다.

통사론:

texture_get_height(tex);

논쟁 유형 설명
tex 텍스처 포인터 The texture page asset pointer to use

보고:

Real

예시:

tex_h = texture_get_height(surface_get_texture(global.Surf));

위의 코드는 이전에 생성된 표면에서 가져온 텍스처의 높이를 가져옵니다.
