texture_get_texel_width

텍셀 또는 텍스처 요소 는 컴퓨터 그래픽에서 사용되는 텍스처 공간의 기본 단위입니다. 그림이 픽셀 배열로 표현되는 것처럼 텍스처는 텍셀 배열로 표현되며, 이 함수는 사용된 이미지 자산의 텍스처 페이지에서 단일 텍셀의 너비를 반환합니다.

통사론:

texture_get_texel_width(tex);

논쟁 유형 설명
tex 텍스처 포인터 The texture page asset pointer to use

보고:

Real

예시:

var tex = sprite_get_texture(sprite_index, 0);
tex_w = texture_get_texel_width(tex);
tex_h = texture_get_texel_height(tex);

위의 코드는 스프라이트 자산에서 가져온 텍스처의 텍셀 너비와 높이를 가져옵니다.
