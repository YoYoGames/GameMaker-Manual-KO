texturegroup_get_sprites

이 함수를 사용하면 주어진 텍스처 그룹 내에서 텍스처 페이지에 할당된 각 스프라이트의 스프라이트 인덱스를 검색할 수 있습니다. 텍스처 그룹 ID 문자열(텍스처 그룹 편집기에 정의된 대로)을 제공하면 이 함수는 각 항목에 스프라이트 리소스에 대한 스프라이트 인덱스가 포함된 1D 배열을 반환합니다. 함수가 실패하면(예: 유효하지 않은 그룹이 지정되거나 그룹에 할당된 텍스처가 없는 경우) 배열은 비어 있습니다(0 길이).

통사론:

texturegroup_get_sprites(tex_id);

논쟁 유형 설명
tex 끈 The name of the texture group to check (a string)

보고:

Array

예시:

var _tex_array = texturegroup_get_sprites( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    show_debug_message("Sprite " + string(i) + " Index:" + string(tex_array[i]));
}

위의 코드는 텍스처 그룹 "MainMenu"에 대한 스프라이트 인덱스를 검색한 다음 콘솔 출력 창에 해당 ID를 표시합니다.
