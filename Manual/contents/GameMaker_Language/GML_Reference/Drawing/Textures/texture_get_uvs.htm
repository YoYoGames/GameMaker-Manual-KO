texture_get_uvs

이 함수는 주어진 텍스처 포인터에 대한 UV 좌표를 나타내는 4개의 요소가 있는 1D 배열 을 반환하고 배열을 다음 값으로 채웁니다.

  • [0] = 왼쪽
  • [1] = 상단
  • [2] = 오른쪽
  • [3] = 바닥
  • [4] = 자산 컴파일러가 스프라이트 왼쪽에서 트리밍한 픽셀의 양(스프라이트 자산만 해당)
  • [5] = 자산 컴파일러가 스프라이트 상단에서 트리밍한 픽셀의 양(스프라이트 자산만 해당)
  • [6] = 텍스처 페이지에 저장된 원본 스프라이트 너비의 픽셀 데이터의 정규화된 백분율(스프라이트 자산만 해당)
  • [7] = 텍스처 페이지에 저장된 원본 스프라이트 높이의 픽셀 데이터의 정규화된 백분율(스프라이트 자산만 해당)

그런 다음 이 값은 다른 그리기 기능, 특히 일반 3D 및 일부 2D 기본 기능과 셰이더 기능에서 사용할 수 있습니다. 스프라이트에 UVS가 필요한 경우 다음을 사용할 수 있습니다. sprite_get_uvs() 글꼴의 경우font_get_uvs().

통사론:

texture_get_uvs(texid)

논쟁 유형 설명
texid 조직 The texture pointer to get the UVS for

보고:

Array (4 - 8 elements)

예시:

var _tex = surface_get_texture(surf_back);
var _uvs = texture_get_uvs(tex);
var _uvs_left = _uvs[0];
var _uvs_top = _uvs[1];
var _uvs_right = _uvs[2];
var _uvs_bottom = _uvs[3];

위의 코드는 먼저 surf_back 에 저장된 표면의 텍스처를 검색한 다음 해당 텍스처의 UV 좌표를 가져옵니다. 그런 다음 반환된 배열에서 왼쪽, 위쪽, 오른쪽 및 아래쪽 UV 좌표를 검색하고 로컬 변수에 저장합니다.
