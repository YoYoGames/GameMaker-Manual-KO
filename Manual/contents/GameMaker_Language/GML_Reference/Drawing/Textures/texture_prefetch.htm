texture_prefetch

이 기능은 텍스처 페이지 또는 텍스처 페이지 그룹을 "프리페치"하는 데 사용할 수 있습니다. 즉, 필요할 때 VRAM에 로드합니다. 고유한 텍스처 페이지 ID 를 제공합니다( texturegroup_* 함수)를 사용하여 단일 페이지를 프리페치하거나 텍스처 그룹 이름 ( 텍스처 그룹 편집기 에 정의됨)을 제공하여 그룹의 모든 텍스처 페이지를 프리페치할 수 있습니다.

통사론:

texture_prefetch(tex_id);

논쟁 유형 설명
tex_id 텍스처 포인터 또는 문자열 The texture page pointer or a texture group name (a string)

보고:

N/A

예시:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    texture_prefetch(_tex_array[i]);
}

위의 코드는 "MainMenu" 텍스처 그룹 아래의 모든 텍스처 페이지를 미리 가져옵니다.
