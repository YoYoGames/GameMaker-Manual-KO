texturegroup_get_textures

이 함수는 텍스처 그룹을 구성하는 개별 페이지의 텍스처 페이지 ID 를 검색하는 데 사용할 수 있습니다. 텍스처 그룹 ID 문자열(텍스처 그룹 편집기에 정의된 대로)을 제공하면 이 함수는 배열의 각 항목이 단일 텍스처 페이지 ID인 1D 배열을 반환합니다. 함수가 실패하면(예: 유효하지 않은 그룹이 지정되거나 그룹에 할당된 텍스처가 없는 경우) 배열은 비어 있습니다(0 길이).

통사론:

texturegroup_get_textures(tex_id);

논쟁 유형 설명
tex_id 끈 The name of the texture group to check (a string)

보고:

Array of Texture Pointers

예시:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    if texture_is_ready(_tex_array[i])
    {
        texture_prefetch(_tex_array[i]);
    }
}

위의 코드는 텍스처 그룹 "MainMenu"에 대한 텍스처 페이지 ID를 검색한 다음 압축이 풀렸는지 확인하고 맞으면 VRAM에 배치합니다.
