draw_sprite_pos

이 기능을 사용하면 네 모서리 좌표로 정의된 영역에 왜곡된 스프라이트를 그릴 수 있습니다. 처음 두 인수는 그릴 스프라이트와 스프라이트의 하위 이미지입니다(함수에서와 동일draw_sprite() ) 그러나 다음 것들은 주어진 스프라이트의 경계 상자 의 네 모서리 각각의 위치를 정의하는 것들입니다. 이것은 시계 방향으로 주어져야 하므로 첫 번째 좌표는 왼쪽 위, 오른쪽 위, 오른쪽 아래, 마지막으로 왼쪽 아래입니다. 스프라이트의 알파 값을 설정하여 투명하게 그릴 수도 있습니다. 아래 이미지는 이 기능의 작동 방식을 보여줍니다.

경고! 위의 이미지는 설명을 위한 것이며, 스프라이트에 이 기능을 사용하면 다른 결과를 얻을 수 있으며 스프라이트가 쿼드 프리미티브에서 구성되는 방식으로 인해 텍스처 "셔링"이 발생할 수 있습니다.

참고: 이 기능은 비트맵 스프라이트에만 유용하며 SWF 또는 JSON(Spine) 스프라이트에서는 작동하지 않습니다.

참고: 이 기능을 사용할 때 텍스처 페이지에 대해 자동 자르기 설정을 비활성화해야 합니다.

통사론:

draw_sprite_pos(sprite, subimg, x1, y1, x2, y2, x3, y3, x4, y4, alpha);

논쟁 유형 설명
sprite The index of the sprite to draw.
subimg The subimg (frame) of the sprite to draw (image_index or -1 correlate to the current frame of animation in the object).
x1 The first x coordinate.
y1 The first y coordinate.
x2 The second x coordinate.
y2 The second y coordinate.
x3 The third x coordinate.
y3 The third y coordinate.
x4 The fourth x coordinate.
y4 The fourth y coordinate.
alpha The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

보고:

예시:

draw_sprite_pos(sprite_index, image_index, x - 100, y - 50, x - 50, y +150, x + 100, y + 200, x + 100, y, 1);

위의 코드는 인스턴스의 x / y 위치 주변에서 완전히 불투명한 알파로 왜곡된 코드를 실행하는 인스턴스와 관련된 스프라이트를 그립니다.
