draw_get_swf_aa_level

이 함수는 SWF 형식 벡터 스프라이트에 대한 앤티 앨리어싱(AA) 수준을 얻는 데 사용할 수 있습니다. 반환 값은 0과 1 사이이며 이러한 스프라이트의 가장자리가 얼마나 "부드럽게" 그려지는지 보여줍니다. 기능을 사용하여 AA 레벨을 설정할 수 있습니다.draw_set_swf_aa_level().

통사론:

draw_get_swf_aa_level();

보고:

예시:

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

위의 코드는 SWF 형식 스프라이트의 AA 값을 확인하고 0이면 AA를 활성화하고 값을 0.5로 설정합니다.
