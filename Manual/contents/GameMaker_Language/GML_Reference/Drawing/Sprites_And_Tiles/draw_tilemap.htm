draw_tilemap

이 기능을 사용하여 방의 어느 곳에나 주어진 타일맵을 그릴 수 있습니다. 타일맵 요소 ID(사용하여 타일맵 요소를 생성할 때 얻는 layer_tilemap_create() 또는 기능을 사용할 때 layer_tilemap_get_id() ) 그런 다음 (방에서) 그리기 위한 x/y 위치를 지정합니다. 이것은 단순히 함수를 호출하는 인스턴스의 레이어 깊이를 사용하여 지정된 지점에 타일맵을 그립니다. 타일맵을 이동하지 않으며 어떤 식으로든 변경하지 않으며 타일맵이 보이는 것으로 플래그가 지정되었는지 여부는 중요하지 않습니다.

통사론:

draw_tilemap(tilemap_element_id, x, y);

논쟁 유형 설명
tilemap_element_id 타일 맵 요소 ID The unique ID value of the tilemap element to draw
x 진짜 The x position within the room to draw at
y 진짜 The y position within the room to draw at

보고:

N/A

예시:

var lay_id = layer_get_id("Tiles_Effects");
var map_id = layer_tilemap_get_id(lay_id);
draw_tilemap(map_id, mouse_x, mouse_y);

위의 코드는 레이어 ID를 얻은 다음 이를 사용하여 마우스 위치에 타일맵을 그리는 데 사용되는 특정 타일맵 ID를 가져옵니다.
