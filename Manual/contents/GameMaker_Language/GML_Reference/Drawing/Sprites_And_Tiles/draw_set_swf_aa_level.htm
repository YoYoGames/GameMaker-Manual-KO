draw_set_swf_aa_level

이 함수는 SWF 형식 벡터 스프라이트에 대한 앤티 앨리어싱(AA) 수준을 설정하는 데 사용할 수 있습니다. 이것은 0에서 1 사이의 실제 값이 될 수 있으며 이러한 스프라이트의 가장자리를 "매끄럽게"합니다. 이 효과를 보려면 먼저 기능을 사용하여 AA를 활성화해야 합니다.draw_enable_swf_aa().

통사론:

draw_set_swf_aa_level(AA);

논쟁 유형 설명
AA The anti-aliasing value to use from 0 to 1.

보고:

예시:

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

위의 코드는 SWF 형식 스프라이트의 AA 값을 확인하고 0이면 AA를 활성화하고 값을 0.5로 설정합니다.
