draw_enable_swf_aa

이 기능을 사용하면 SWF 형식 벡터 스프라이트에 대해 앤티 앨리어싱(AA)을 활성화하거나 비활성화할 수 있습니다. AA는 단순히 벡터 이미지의 가장자리를 매끄럽게 만들어 더 멋지게 보입니다. 사용된 AA의 양은 함수를 사용하여 설정한 값에 따라 다릅니다.draw_set_swf_aa_level(). 기본적으로 비활성화되어 있습니다.

통사론:

draw_enable_swf_aa(enable);

논쟁 유형 설명
enable Enable (true ) or disable (false ) AA for all SWF sprites.

보고:

예시:

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

위의 코드는 SWF 형식 스프라이트의 AA 값을 확인하고 0이면 AA를 활성화하고 값을 0.5로 설정합니다.
