gpu_set_cullmode

이 기능은 뒷면 컬링 모드를 설정하는 데 사용할 수 있습니다. 폴리곤(원형)에는 앞면 과 뒷면 이 있습니다. 앞면은 꼭짓점이 시계 반대 방향으로 정의되는 변이라고 합니다. 일반적으로 양면이 그려지지만 피라미드와 같은 닫힌 모양을 만들면 삼각형의 뒷면이 절대 보이지 않기 때문에 처리 능력이 낭비됩니다. 이러한 경우에 뒷면 컬링을 켤 수 있습니다. 이렇게 하면 문제를 방지하기 위해 올바른 방법으로 폴리곤을 정의해야 하는 작업이 있기는 하지만 그리기 시간이 절반 정도 절약됩니다.

사용된 컬 모드 값은 다음 상수 중 하나여야 합니다(기본값은 cull_noculling ):

일정한 설명
cull_noculling 컬링이 수행되지 않습니다.
cull_clockwise 모든 시계 방향 삼각형이 컬링됩니다.
cull_counterclockwise 모든 시계 반대 방향 삼각형이 컬링됩니다.

통사론:

gpu_set_cullmode(cullmode);

논쟁 유형 설명
cullmode 컬링 모드 상수 The cull mode to use (see list above)

보고:

N/A

예시:

if gpu_get_cullmode() != cull_clockwise
{
    gpu_set_cullmode(cull_clockwise);
}

위의 코드는 현재 컬 모드를 가져오고 그렇지 않은 경우 cull_clockwise 그 상수로 설정됩니다.
