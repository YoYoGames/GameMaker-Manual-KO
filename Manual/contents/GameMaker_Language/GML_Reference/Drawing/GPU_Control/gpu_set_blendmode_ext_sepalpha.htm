gpu_set_blendmode_ext_sepalpha

GameMaker 가 픽셀을 그릴 때 소스 색상(그리려는 픽셀의 색상)과 대상 색상(그리려는 픽셀에 이미 있는 색상)이 있으므로 새 색상을 결정할 때 픽셀의 소스 및 대상 색상은 선택한 혼합 모드에 따라 계산됩니다. 색상의 각 구성 요소는 0과 1 사이의 부동 소수점 값으로 저장되며, 새로운 색상은 소스 색상의 각 구성 요소에 몇 가지 요소를 곱하고 대상 색상의 각 구성 요소에 다른 요소를 곱한 다음 추가하여 계산됩니다. 구성 요소별로 결과가 함께 나타납니다.

이 기능을 사용하면 RGB 구성 요소와 Alpha 구성 요소에 대해 서로 다른 혼합 모드 요소의 서로 다른 부분을 선택하여 사용자 정의 혼합 모드를 생성하기 위해 함께 인수되어야 하는 서로 다른 구성 요소를 설정할 수 있습니다. 따라서 "src" 및 "dest" 인수는 처음 3개의 혼합 모드 값(RGB)을 사용하고 "alphasrc" 및 "alphadest" 인수는 혼합 모드 요소의 마지막 부분을 사용합니다. 예를 들어 RGB 소스가 다음과 같은 경우 bm_zero, 그런 다음 (0, 0, 0)이 사용됩니다. 그런 다음 Alpha 소스를 bm_one으로 설정하면 (1)이 사용되어 최종 소스 혼합 모드가 (0, 0, 0, 1)이 됩니다.

다음 차트는 각 혼합 모드에 대한 RGBA 값을 보여줍니다. 여기서 (Rs, Gs, Bs, As)는 소스 값으로 간주되고 (Rd, Gd, Bd, Ad)는 대상 값입니다. 이 함수에서 사용할 때 사용할 수 있는 11개의 상수는 소스 또는 대상(또는 둘 다)일 수 있습니다.

일정한 혼합 요소(빨간색, 녹색, 파란색, 알파)
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-A)
bm_src_alpha (대로, 그대로, 그대로, 그대로)
bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As)
bm_dest_alpha (광고, 광고, 광고, 광고)
bm_inv_dest_alpha (1-광고, 1-광고, 1-광고, 1-광고)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) 여기서 f = min(As, 1-Ad)

중요한! WebGL이 활성화 되지 않은 HTML5는 다음 모드를 올바르게 표시할 수 없습니다.

  • bm_src_colour
  • bm_inv_src_colour
  • bm_dest_colour
  • bm_inv_dest_colour
  • bm_src_alpha_sat

이 함수에 4개의 개별 인수를 제공하거나 인수 배열(함수에서 반환된 대로)을 제공할 수 있습니다.gpu_get_blendmode_ext_sepalpha() 예를 들어). 배열을 제공하는 경우 다음 네 가지 요소가 있어야 합니다.

  • [0] = 소스 혼합 모드에서 알파 구성 요소를 뺀 값(기본값은 bm_src_alpha (대로, 그대로, 그대로))
  • [1] = 대상 혼합 모드에서 알파 구성 요소를 뺀 값(기본값은 bm_inv_src_alpha (1-As, 1-As, 1-As))
  • [2] = 소스 알파 구성요소(기본값은 bm_src_alpha (같이))
  • [3] = 대상 알파 구성요소(기본값은 bm_inv_src_alpha (1-As))

블렌드 모드를 최대한 활용하고 작동 방식과 화면에 그려지는 최종 이미지에 미치는 영향을 이해하려면 다음 가이드를 읽는 것이 좋습니다.

  • 블렌드 모드 사용 가이드

통사론:

gpu_set_blendmode(src, dest, alphasrc, alphadest);

논쟁 유형 설명
src 혼합 모드 계수 상수 Source blend mode factor for RGB (see constants above).
dest 혼합 모드 계수 상수 Destination blend mode factor for RGB (see constants above)
alphasrc 혼합 모드 계수 상수 Source blend mode factor Alpha (see constants above).
alphadest 혼합 모드 계수 상수 Destination blend mode factor Alpha (see constants above)

예시:

var bm = gpu_get_blendmode_ext_sepalpha();
bm[0] = bm_src_alpha;
bm[1] = bm_inv_src_alpha;
gpu_set_blendmode_ext_sepalpha(bm);

위의 코드는 로컬 배열을 만들고 별도의 알파 값으로 현재 확장 혼합 모드를 가져옵니다. 그런 다음 이 배열을 조작하고 반환하여 혼합 모드를 설정합니다.
