gpu_get_texfilter_ext

이 기능을 사용하면 텍스처 보간(선형 보간)이 활성화되었는지 확인할 수 있습니다(반환 true ) 여부(반환 false ) 주어진 셰이더 샘플러 텍스처에 대해.

참고: HTML5 대상에서 이 기능은 WebGL이 활성화된 상태에서만 작동합니다.

통사론:

gpu_get_texfilter_ext(sampler_id);

논쟁 유형 설명
sampler_id 셰이더 샘플러 핸들 The sampler id from the shader.

보고:

Boolean

예시:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if gpu_get_texfilter_ext(s_tex)
{
    gpu_set_texfilter_ext(s_tex, false);
}
else
{
    gpu_set_texfilter_ext(s_tex, true);
}

위의 코드는 특정 샘플러 ID(로컬 변수에 저장)에 대해 텍스처 필터링이 켜져 있는지 꺼져 있는지 확인하고 그에 따라 전환합니다.
