gpu_get_zwriteenable

이 함수는 z 쓰기가 활성화되었는지 여부를 검색하는 데 사용할 수 있습니다(함수는 true ) 또는 아님(함수가 반환 false ). 기본값은 z-write가 활성화 되어 있으므로 함수는 다음을 반환합니다. true.

통사론:

gpu_get_zwriteenable();

보고:

예시:

if gpu_get_zwriteenable() == false
{
    gpu_set_zwriteenable(true);
}

위의 코드는 z-write가 활성화되었는지 여부를 확인하고 비활성화되면 다시 활성화됩니다.
