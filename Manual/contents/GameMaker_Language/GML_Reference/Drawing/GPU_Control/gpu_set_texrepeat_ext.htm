gpu_set_texrepeat_ext

이 기능은 GameMaker 에서 셰이더 를 사용할 때 단일 샘플러 "슬롯"이 주어진 텍스처를 반복하는지 여부를 설정하는 데 사용할 수 있습니다. 로 설정 true uv 좌표가 0-1 범위를 벗어나면 텍스처를 반복하지만 설정은 false 반복하지 않음을 의미합니다. 이러한 기능의 가능한 사용 사례는 3D에서 텍스처를 반복하는 것이지만 이것이 작동하고 텍스처 페이지의 나머지 부분에서 이미지를 가져오지 않도록 하려면 사용된 스프라이트를 "분리 텍스처 페이지"에 있는 것으로 표시해야 합니다. 스프라이트 편집기에서

참고 이 설정은 함수를 호출할 때 설정된 값으로 무시됩니다.gpu_set_texrepeat().

통사론:

gpu_set_texrepeat_ext(sampler_id, enable);

논쟁 유형 설명
sampler_id The sampler id from the shader.
enable Enable or disable texture filtering (true / false )

보고:

예시:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if gpu_get_texfilter_ext(s_tex)
{
    gpu_set_texrepeat_ext(s_tex, false);
}
else
{
    gpu_set_texfilter_ext(s_tex, true);
}

위의 코드는 특정 샘플러 ID(로컬 변수에 저장)에 대해 텍스처 필터링이 켜져 있는지 꺼져 있는지 확인하고 그에 따라 전환합니다.
