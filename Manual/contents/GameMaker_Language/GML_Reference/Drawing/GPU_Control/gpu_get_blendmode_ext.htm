gpu_get_blendmode_ext

이 함수는 그리기에 사용 중인 현재 확장 혼합 모드를 검색하는 데 사용할 수 있습니다. 이 함수는 다음 요소가 포함된 2요소 1D 배열을 반환합니다.

  • [0] = 소스 혼합 모드 계수 상수 (기본값은 bm_src_alpha )
  • [1] = 대상 혼합 모드 계수 상수 (기본값은 bm_inv_src_alpha )

배열의 값은 다음 상수 중 하나입니다("s"는 소스에서 가져온 값을 나타내고 "d"는 대상에서 가져온 값을 나타냄).

혼합 모드 계수 상수
일정한 혼합 요소(빨간색, 녹색, 파란색, 알파)
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-A)
bm_src_alpha (대로, 그대로, 그대로, 그대로)
bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As)
bm_dest_alpha (광고, 광고, 광고, 광고)
bm_inv_dest_alpha (1-광고, 1-광고, 1-광고, 1-광고)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) 여기서 f = min(As, 1-Ad)

이 값을 변경하고 전체 배열을gpu_set_blendmode_ext() 단일 인수로 기능합니다.

통사론:

gpu_get_blendmode_ext();

보고:

Array (2 elements only; see above for constants)

예시:

var bm = gpu_get_blendmode_ext();
bm[0] = bm_src_alpha;
gpu_set_blendmode_ext(bm);

위의 코드는 현재 확장 혼합 모드를 가져와 소스를 수정한 다음 확장 혼합 모드를 다시 설정합니다.
