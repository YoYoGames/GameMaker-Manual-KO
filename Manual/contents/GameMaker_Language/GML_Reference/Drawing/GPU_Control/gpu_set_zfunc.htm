gpu_set_zfunc

이 기능은 z 버퍼 테스트 비교 모드를 설정하는 데 사용할 수 있습니다(참조gpu_set_ztestenable() 자세한 내용은). 사용할 수 있는 값은 다음 상수 중 하나입니다(기본값은 cmpfunc_lessequal ):

일정한 설명
cmpfunc_never 절대
cmpfunc_less 더 적은
cmpfunc_equal 동일한
cmpfunc_lessequal 작거나 같음
cmpfunc_greater 보다 큰
cmpfunc_notequal 같지 않음
cmpfunc_greaterequal 크거나 같음
cmpfunc_always 언제나

통사론:

gpu_set_zfunc(cmp_func);

논쟁 유형 설명
cmp_func 비교 함수 상수 The comparison mode to use (see list above)

보고:

N/A

예시:

gpu_set_ztestenable(true);
gpu_set_zfunc(cmpfunc_always);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

위의 코드는 z 버퍼 테스트를 켜고 배경 스프라이트를 그리기 전에 비교 모드를 설정한 다음 다시 꺼서 계속 그리기를 진행합니다.
