gpu_set_alphatestref

이 기능은 알파 테스트에 대한 참조 값을 설정합니다(이를 사용하여 활성화되었을 때gpu_set_alphatestenable() ). 이것은 알파가 있는 픽셀이 그려지지 않는 "컷오프" 임계값입니다.

기본적으로 이 값을 0으로 설정하고 선형 보간을 켠 경우(게임 옵션에서 또는gpu_set_texfilter() ), 겹치는 개체의 가장자리에서 여전히 시각적 오류가 발생할 수 있습니다. 그 이유는 그 뒤에 그려지는 것을 차단하는 거의 완전히 투명한 영역을 가질 수 있기 때문입니다. 그러나 이 참조 값을 예를 들어 254(즉, 최대값보다 하나 아래)로 설정하면 100% 불투명 픽셀만 그려집니다.. 텍스처 보간을 끌 수도 있습니다. 이렇게 하면 일반적으로 문제 없이 기본값을 사용할 수 있고 이 기능을 전혀 사용할 필요가 없지만 많은 부분이 소스 아트웍에 따라 다릅니다.

참고: 이것은 고급 그리기 기능이므로 필요한지 여부가 확실하지 않거나 정확히 무엇을 하는지 모르는 경우 게임이 그려지는 방식을 심각하게 손상시킬 수 있으므로 사용하지 않는 것이 좋습니다..

통사론:

gpu_set_alphatestref(val);

논쟁 유형 설명
val Sets the reference value for alpha testing from 0 to 255 (default value is 0)

보고:

예시:

if !gpu_get_alphatestenable()
{
    gpu_set_alphatestenable(true);
    gpu_set_alphatestref(128);
}

위의 코드는 알파 테스트가 활성화되어 있는지 확인하고 그렇지 않은 경우 알파 테스트를 켜고 테스트 임계값을 128로 설정합니다(알파가 0.5보다 큰 픽셀만 그려짐).
