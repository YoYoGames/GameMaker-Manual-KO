gpu_set_state

이 함수는 전달된 DS Map 을 사용하여 현재 GPU 상태를 설정합니다. 제공된 지도는 함수를 사용하여 생성할 수 있습니다. gpu_get_state().

통사론:

gpu_set_state(ds_map);

논쟁 유형 설명
ds_map The GPU state to set as a DS Map.

보고:

예시:

gpu_set_state(gpu_map);

위의 코드는 "gpu_map" 변수에 제공된 맵을 사용하여 GPU 상태를 설정합니다.
