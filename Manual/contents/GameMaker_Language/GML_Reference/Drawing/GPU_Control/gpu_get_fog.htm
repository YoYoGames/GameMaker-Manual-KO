gpu_get_fog

이 기능은 안개 설정을 검색하는 데 사용할 수 있습니다. 이 함수는 다음 정보가 포함된 4요소 1D 배열을 반환합니다.

  • [0] = 활성화된 토글( 부울 또는 true 또는 false ), 기본 false
  • [1] = 색상 상수, 기본값 c_black
  • [2] = 시작 거리( 실수 ), 기본값 0
  • [3] = 끝 거리( 실수 ), 기본값 1

이 값을 변경하고 전체 배열을gpu_set_fog() 단일 인수로 기능합니다.

통사론:

gpu_get_fog();

보고:

Array (4 elements only; see above for details)

예시:

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

위의 코드는 현재 안개 설정을 가져온 다음 배열의 색상 요소를 다음으로 설정합니다. c_red 변경된 배열을 사용하여 안개를 다시 설정하기 전에.
