gpu_get_texrepeat

이 기능을 사용하면 텍스처 반복이 활성화되었는지 확인할 수 있습니다(반환 true ) 여부(반환 false ). 텍스처 반복에 대한 자세한 내용은 함수를 참조하십시오.gpu_set_texrepeat().

통사론:

gpu_get_texrepeat();

보고:

예시:

if gpu_get_texrepeat()
{
    gpu_set_texrepeat(false);
}
else
{
    gpu_set_texrepeat(true);
}

위의 코드는 텍스처 반복이 켜져 있는지 꺼져 있는지 확인하고 그에 따라 전환합니다.
