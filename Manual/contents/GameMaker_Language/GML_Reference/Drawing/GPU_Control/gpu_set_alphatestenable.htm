gpu_set_alphatestenable

이 기능은 게임에 대한 알파 테스트를 활성화 또는 비활성화합니다(기본적으로 비활성화되어 있음). 알파 테스트를 켜면 컴패니언 기능을 사용할 수 있습니다.gpu_set_alphatestref() 모든 알파 값이 0으로 설정되는 "컷오프" 값을 설정합니다. 아래 이미지는 알파 테스트를 켜거나 끌 때 볼 수 있는 차이를 보여줍니다.

참고: 이 기능은 iOS 및 Android 장치의 성능에 부정적인 영향을 미칠 수 있습니다.

통사론:

gpu_set_alphatestenable(enable);

논쟁 유형 설명
enable Enable or disable alpha testing (true / false )

보고:

예시:

if !gpu_get_alphatestenable()
{
    gpu_set_alphatestenable(true);
    gpu_set_alphatestref(128);
}

위의 코드는 알파 테스트가 활성화되어 있는지 확인하고 그렇지 않은 경우 알파 테스트를 켜고 테스트 임계값을 128로 설정합니다(알파가 0.5보다 큰 픽셀만 그려짐).
