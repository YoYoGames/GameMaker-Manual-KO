gpu_set_zwriteenable

이 기능은 (true ) 및 끄기(false ) z 버퍼를 사용한 깊이 정렬. GameMaker 에서 인스턴스가 렌더링될 때 생성된 픽셀의 깊이(z 좌표)가 z-버퍼에 저장되고, 장면의 다른 인스턴스가 동일한 픽셀에서 렌더링되어야 하는 경우 두 깊이가 비교됩니다(z -테스트 - 참조gpu_set_ztestenable() ) 관찰자에게 "가까운" 것이 z 버퍼에 저장되어 이전 버퍼를 대체합니다. 그러나 이것이 항상 일어나기를 원하는 것은 아니며 z 값이 무엇이든 상관없이 다른 모든 것보다 무언가가 그려지기를 원하는 순간이 있습니다. 이러한 경우 이 기능을 사용하여 z-버퍼에 대한 쓰기를 끌 수 있으므로 그려지는 모든 것이 다른 모든 것 위에 그려지도록 강제할 수 있습니다. 여기에 제공된 설명은 기본 z-testing 비교 모드에 대한 것이지만 변경할 수 있습니다.

기본값은 z 버퍼링이 활성화된 것입니다(true ).

통사론:

gpu_set_zwriteenable(enable);

논쟁 유형 설명
enable The comparison mode to use (see list above)

보고:

예시:

gpu_set_zwriteenable(false);
with (obj_Menu_Containers)
{
    draw_self();
}
gpu_set_zwriteenable(true);

위의 코드는 z 버퍼 쓰기를 끄고 여러 개체를 그린 다음 다시 켜서 그리기를 계속합니다.
