gpu_get_state

이 함수는 현재 GPU 상태를 가져와 DS Map 으로 반환합니다. 그런 다음 이를 조작하거나 저장할 수 있으며 함수를 사용하여 이 맵을 GPU에 반환할 수 있습니다.gpu_set_state().

통사론:

gpu_get_state();

보고:

DS Map ID

예시:

gpu_map = gpu_get_state();

위의 코드는 현재 GPU 상태를 변수에 저장합니다.
