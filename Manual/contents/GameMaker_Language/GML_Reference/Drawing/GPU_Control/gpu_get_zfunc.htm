gpu_get_zfunc

이 함수는 z 비교 모드를 검색하는 데 사용할 수 있습니다. 반환된 값은 다음 상수 중 하나입니다(기본값은 cmpfunc_lessequal ):

비교 함수 상수
일정한 설명
cmpfunc_never 절대
cmpfunc_less 더 적은
cmpfunc_equal 동일한
cmpfunc_lessequal 작거나 같음
cmpfunc_greater 보다 큰
cmpfunc_notequal 같지 않음
cmpfunc_greaterequal 크거나 같음
cmpfunc_always 언제나

통사론:

gpu_get_zfunc();

보고:

Comparison Function Constant (see table above)

예시:

if gpu_get_zfunc() != cmpfunc_greater
{
    gpu_set_zfunc(cmpfunc_greater);
}

위의 코드는 z-testing 방법이 다음으로 설정되어 있는지 확인합니다. cmpfunc_greater 그렇지 않은 경우 해당 상수로 설정됩니다.
