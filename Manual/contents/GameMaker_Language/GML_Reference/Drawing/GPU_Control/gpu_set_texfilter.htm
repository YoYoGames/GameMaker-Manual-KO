gpu_set_texfilter

이 기능을 사용하여 게임 화면에 그려진 모든 이미지의 텍스처 필터링(선형 보간)을 설정할 수 있습니다. 이것이 활성화되면(true ) 모든 텍스처는 그릴 때 부드럽게 처리됩니다(여기에는 스프라이트도 텍스처로 간주되므로 스프라이트가 포함됨). 즉, 픽셀 비율이 1:1이 아닌 경우 크기를 조정하거나 이동할 때 다양한 픽셀에 "얼룩"이 생길 수 있습니다. 사용된 아트 스타일에 따라 이미지가 흐릿하게 나타납니다. 비활성화된 경우(false ) 그런 다음 크기가 조정되거나 이동할 때 가장 가까운 픽셀을 기반으로 이미지가 그려지므로 "뭉쳐진" 이미지가 나타날 수 있습니다. 기본값은 false, 이는 개별 대상 플랫폼에 대한 글로벌 게임 옵션 에서도 변경할 수 있습니다.

참고: 이 설정은 기능을 사용하는 셰이더에 대해 설정된 모든 텍스처 단계 보간을 무시합니다.gpu_set_texfilter_ext().

참고: HTML5 대상에서 이 기능은 WebGL이 활성화된 상태에서만 작동합니다.

통사론:

gpu_set_texfilter(enable);

논쟁 유형 설명
enable Enable or disable texture filtering (true / false )

보고:

예시:

if gpu_get_texfilter()
{
    gpu_set_texfilter(false);
}
else
{
    gpu_set_texfilter(true);
}

위의 코드는 텍스처 필터링이 켜져 있는지 꺼져 있는지 확인하고 그에 따라 전환합니다.
