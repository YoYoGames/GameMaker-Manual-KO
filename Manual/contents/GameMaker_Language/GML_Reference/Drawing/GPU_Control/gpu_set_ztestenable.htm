gpu_set_ztestenable

이 기능은 z-버퍼 테스트를 켜거나 끄는 데 사용할 수 있으며, 사물이 그려지는 방식에 영향을 줍니다(일반적으로 3D 프로젝트 작업 시에만 사용). 기본적으로 z-testing이 꺼져 있고 동일한 공간에 두 개의 객체를 그리는 경우, 둘 중 하나가 다른 하나를 과도하게 그리는지 여부에 관계없이 두 객체가 모두 렌더링되어 불필요한 그리기 호출이 발생합니다. 이것을 켜면 z-버퍼가 테스트되어 개체가 "표시"되고 그렇지 않은 경우 그려지지 않습니다. 이것이 기본 동작이지만 z 버퍼 테스트에 사용되는 비교 유형을 변경하여 변경할 수 있습니다(함수 참조gpu_set_zfunc().

기본적으로 z 버퍼 테스트는 꺼져 있습니다(false ).

통사론:

gpu_set_ztestenable(enable);

논쟁 유형 설명
enable Enable or disable z-buffer testing (true or false ).

보고:

예시:

gpu_set_ztestenable(true);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

위의 코드는 배경 스프라이트를 그리기 위해 z-버퍼 테스트를 켠 다음 다시 그리기를 계속하기 위해 다시 끕니다.
