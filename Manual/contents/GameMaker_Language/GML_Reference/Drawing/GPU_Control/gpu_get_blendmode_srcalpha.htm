gpu_get_blendmode_srcalpha

이 함수는 현재 소스 확장 혼합 모드 알파 요소를 검색하는 데 사용할 수 있습니다. 반환된 값은 다음 상수 중 하나가 됩니다("s"는 소스에서 가져온 값을 나타내고 "d"는 대상의 값을 나타냄). 그릴 때 "A" 구성 요소만 사용됩니다.

일정한 혼합 요소(빨간색, 녹색, 파란색, 알파)
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-A)
bm_src_alpha (대로, 그대로, 그대로, 그대로)
bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As)
bm_dest_alpha (광고, 광고, 광고, 광고)
bm_inv_dest_alpha (1-광고, 1-광고, 1-광고, 1-광고)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) 여기서 f = min(As, 1-Ad)

통사론:

gpu_get_blendmode_srcalpha();

보고:

Blend Mode Factor Constant (see above table)

예시:

var bm;
bm[0] = gpu_get_blendmode_srcalpha();
bm[1] = gpu_get_blendmode_destalpha();
gpu_set_blendmode_ext_sepalpha(bm_inv_src_alpha, bm_inv_dest_colour, bm[0], bm[1]);

위의 코드는 로컬 배열을 만들고 알파 구성 요소에 대한 현재 소스 및 대상 혼합 요소를 가져옵니다. 그런 다음 이 배열을 사용하여 혼합 요소의 RGB 구성 요소를 조작합니다.
