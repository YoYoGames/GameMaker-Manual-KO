gpu_get_cullmode

이 기능은 뒷면 컬링 모드를 검색하는 데 사용할 수 있습니다. 반환된 값은 다음 상수 중 하나입니다(기본값은 cull_noculling ):

컬링 모드 상수
일정한 설명
cull_noculling 컬링이 수행되지 않습니다.
cull_clockwise 모든 시계 방향 삼각형이 컬링됩니다.
cull_counterclockwise 모든 시계 반대 방향 삼각형이 컬링됩니다.

통사론:

gpu_get_cullmode();

보고:

Culling Mode Constant (see above for constants)

예시:

if gpu_get_cullmode() != cull_clockwise
{
    gpu_set_cullmode(cull_clockwise);
}

위의 코드는 현재 컬 모드를 가져오고 그렇지 않은 경우 cull_clockwise 그 상수로 설정됩니다.
