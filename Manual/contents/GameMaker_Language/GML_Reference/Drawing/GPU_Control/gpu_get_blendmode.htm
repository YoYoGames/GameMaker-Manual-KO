gpu_get_blendmode

이 기능은 그리기에 사용 중인 현재 혼합 모드를 검색하는 데 사용할 수 있습니다. 반환된 값은 다음 상수 중 하나입니다(기본값은 bm_normal ):

혼합 모드 상수
일정한 설명 확장 혼합 모드
bm_normal 일반 혼합(기본 혼합 모드). (bm_src_alpha, bm_inv_src_alpha )
bm_add 첨가제 혼합. 밝은 영역의 광도 값이 추가됩니다. (bm_src_alpha, bm_one )
bm_subtract 빼기 혼합. 밝은 영역의 광도 값을 뺍니다. (bm_zero, bm_inv_src_colour )
bm_max 최대 블렌딩. 첨가제 혼합과 유사합니다. (bm_src_alpha, bm_inv_src_colour )

통사론:

gpu_get_blendmode();

보고:

Blend Mode Constant (see above for constants)

예시:

if gpu_get_blendmode() != bm_normal
{
    gpu_set_blendmode(bm_normal);
}

위의 코드는 현재 혼합 모드를 가져오고 그렇지 않은 경우 bm_normal 그 상수로 설정됩니다.
