gpu_push_state

이 기능을 사용하면 현재 GPU 상태를 나중에 사용할 스택으로 푸시할 수 있습니다. 현재 GPU 상태(블렌드 모드, 알파 쓰기, 컬링 등... 모두 스택에 푸시됨)를 "저장"하고 다른 설정으로 무언가를 그린 다음 재설정하려는 경우 일반적으로 이것을 사용합니다. GPU 스택을 이전 상태로(호출하여gpu_pop_state() ).

통사론:

gpu_push_state();

보고:

예시:

gpu_push_state();
gpu_set_blendmode(bm_add);
gpu_set_blendenable(false);
gpu_set_cullmode(true);
with (obj_Effect_Parent)
{
    draw_self();
}
gpu_pop_state();

위의 코드는 스택에 현재 GPU 상태를 "저장"한 다음 특정 GPU 설정을 변경하고 GPU 상태를 이전 상태로 재설정하기 전에 인스턴스 그룹을 그립니다.
