gpu_get_blendenable

이 함수는 알파 혼합 상태를 검색하는 데 사용할 수 있습니다. 반환하면 true 알파 블렌딩이 활성화되고 반환되는 경우 false 비활성화되어 있습니다. 기본적으로 이것은 켜져 있으므로 함수는 다음을 반환합니다. true

통사론:

gpu_get_blendenable();

보고:

예시:

if gpu_get_blendenable() == false
{
    gpu_set_blendenable(true);
}

위의 코드는 알파 블렌딩의 상태를 확인하고 비활성화되면 다시 활성화됩니다.
