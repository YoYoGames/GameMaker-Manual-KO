gpu_set_blendmode

GameMaker 가 픽셀을 그릴 때 소스 색상(그리려는 픽셀의 색상)과 대상 색상(그리려는 픽셀에 이미 있는 색상)이 있으므로 새 색상을 결정할 때 픽셀의 소스 및 대상 색상은 선택한 혼합 모드에 따라 계산됩니다. 색상의 각 구성 요소는 0과 1 사이의 부동 소수점 값으로 저장되며, 새로운 색상은 소스 색상의 각 구성 요소에 몇 가지 요소를 곱하고 대상 색상의 각 구성 요소에 다른 요소를 곱한 다음 추가하여 계산됩니다. 결과는 구성 요소별로 구성됩니다.

이 기능을 사용하면 다음 상수를 사용하여 네 가지 기본 방법으로 혼합 모드를 설정할 수 있습니다.

일정한 설명 확장 혼합 모드
bm_normal 일반 혼합(기본 혼합 모드). (bm_src_alpha, bm_inv_src_alpha )
bm_add 첨가제 혼합. 밝은 영역의 광도 값이 추가됩니다. (bm_src_alpha, bm_one )
bm_subtract 빼기 혼합. 밝은 영역의 광도 값을 뺍니다. (bm_zero, bm_inv_src_colour )
bm_max 최대 블렌딩. 첨가제 혼합과 유사합니다. (bm_src_alpha, bm_inv_src_colour )

위의 표에서 볼 수 있듯이 이 네 가지 혼합 모드는 실제로 확장된 혼합 모드의 합성이며 설명하는 페이지에서 찾을 수 있습니다.gpu_set_blendmode_ext().

블렌드 모드를 최대한 활용하고 작동 방식과 화면에 그려지는 최종 이미지에 미치는 영향을 이해하려면 다음 가이드를 읽는 것이 좋습니다.

  • 블렌드 모드 사용 가이드

통사론:

gpu_set_blendmode(mode);

논쟁 유형 설명
mode 혼합 모드 상수 The blend mode to use (see the table above)

보고:

N/A

예시:

gpu_set_blendmode(bm_add);
draw_circle_colour(100, 100, 50, c_white, c_black, 0);
gpu_set_blendmode(bm_normal);

이렇게 하면 검정색이 투명하게 되어 흰색이 외부에서 강해지고 원 중심에서 점점 약해지면서 '광택' 효과가 생성됩니다. 블렌드 모드는 원을 그린 후 재설정되어 추가 블렌딩이 그 이후의 다른 모든 항목에도 적용되지 않도록 합니다.
