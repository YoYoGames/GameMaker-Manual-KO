gpu_get_texfilter

이 기능을 사용하면 텍스처 필터링(선형 보간)이 활성화되었는지 확인할 수 있습니다(반환 true ) 여부(반환 false ). 텍스처 필터링에 대한 자세한 내용은 함수를 참조하십시오.gpu_set_texfilter().

참고: HTML5 대상에서 이 기능은 WebGL이 활성화된 상태에서만 작동합니다.

통사론:

gpu_get_texfilter();

보고:

Boolean

예시:

if gpu_get_texfilter()
{
    gpu_set_texfilter(false);
}
else
{
    gpu_set_texfilter(true);
}

위의 코드는 텍스처 필터링이 켜져 있는지 꺼져 있는지 확인하고 그에 따라 전환합니다.
