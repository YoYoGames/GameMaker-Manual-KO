gpu_set_texrepeat

이 기능은 텍스처가 반복되어야 하는지 여부를 GameMaker 에 나타내는 데 사용할 수 있습니다(true ) 여부(false ) 모델 및 프리미티브에 사용되는 경우. 일반적으로 텍스처 너비와 높이는 0 - 1 범위에 있지만 기본적으로 1보다 큰 값을 지정하면 나머지는 그려지지 않습니다(전체 텍스처는 1보다 큰 백분율에 해당하는 "빈" 섹션으로 그려집니다. 따라서 텍스처 너비 또는 높이에 대한 값 2는 공간의 절반에 텍스처를 그리고 나머지 절반은 비워 둡니다. 그러나 이 함수를 사용하여 repeat를 true로 설정하면 텍스처가 모든 모델과 기본 요소에 대해 자동으로 반복됩니다. 이러한 기능의 가능한 사용 사례는 3D에서 텍스처를 반복하는 것이지만 이것이 작동하고 텍스처 페이지의 나머지 부분에서 이미지를 가져오지 않도록 하려면 사용된 스프라이트를 "분리 텍스처 페이지"에 있는 것으로 표시해야 합니다. 스프라이트 편집기에서

참고: 스프라이트와 배경은 항상 반복되지 않고 그려지므로 스프라이트나 배경을 그리면 이 값이 다음으로 재설정됩니다. false. 이 기본 동작은 텍스처 단계 설정에 영향을 미치지 않습니다(예: 함수 호출은 영향을 미치지만 내부 설정을 false로 호출하면 영향을 미치지 않음).

참고: 이 설정은 기능을 사용할 때 셰이더에 대해 설정된 모든 텍스처 단계 반복을 무시합니다.gpu_set_texrepeat_ext().

통사론:

gpu_set_texrepeat(enable);

논쟁 유형 설명
enable Enable or disable texture filtering (true / false )

보고:

예시:

if gpu_get_texrepeat()
{
    gpu_set_texrepeat(false);
}
else
{
    gpu_set_texrepeat(true);
}

위의 코드는 텍스처 반복이 켜져 있는지 꺼져 있는지 확인하고 그에 따라 전환합니다.
