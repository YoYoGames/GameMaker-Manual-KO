gpu_set_colourwriteenable

이 기능을 사용하면 모든 추가 드로잉에 대해 색상 채널과 알파 채널을 켜거나 끌 수 있습니다. 예를 들어, 이 기능을 사용하여 알파 채널 전용 표면을 만들거나(표면을 사용하기 전에 RGB 쓰기 끄기) 화면에 그리는 동안 특수 효과를 만들 수 있습니다. 각 구성 요소의 기본값은 true, 그리고 4개의 고유 인수로 제공되거나 다음 중 하나가 될 다음 요소가 포함된 4개 요소 1D 배열로 제공될 수 있습니다. true (활성화됨) 또는 false (장애가있는):

  • [0] = 빨간색 채널 활성화/비활성화
  • [1] = 녹색 채널 활성화/비활성화
  • [2] = 파란색 채널 활성화/비활성화
  • [3] = 알파 채널 활성화/비활성화

통사론:

gpu_set_colorwriteenable(red, green, blue, alpha);

논쟁 유형 설명
red 부울 Enable/disable the red channel
green 부울 Enable/disable the green channel
blue 부울 Enable/disable the blue channel
alpha 부울 Enable/disable the alpha channel

보고:

N/A

예시:

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

위의 코드는 현재 색상 쓰기 값을 가져온 다음 알파 구성 요소를 다음으로 설정합니다. false.
