gpu_pop_state

이 함수는 스택에서 이전 GPU 상태를 팝하고 적용합니다. 보다gpu_push_state() 자세한 내용은.

통사론:

gpu_pop_state();

보고:

예시:

gpu_push_state();
gpu_set_blendmode(bm_add);
gpu_set_blendenable(false);
gpu_set_cullmode(true);
with (obj_Effect_Parent)
{
    draw_self();
}
gpu_pop_state();

위의 코드는 스택에 현재 GPU 상태를 "저장"한 다음 특정 GPU 설정을 변경하고 GPU 상태를 이전 상태로 재설정하기 전에 인스턴스 그룹을 그립니다.
