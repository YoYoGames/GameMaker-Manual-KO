gpu_set_fog

이 기능은 안개 그리기를 활성화하거나 비활성화하는 데 사용할 수 있습니다. 안개는 3D 게임에서 멀리 있는 인스턴스를 희미하게 보이거나 보이지 않게 만들기 위해 사용할 수 있으며, 이는 분위기를 조성하고 멀리 있는 인스턴스를 그리지 않는다는 사실을 은폐하는 데 도움이 됩니다.

활성화 여부를 설정합니다(true ) 또는 비활성화(false ), 안개가 블렌딩에 사용해야 하는 색상, 시작 및 끝 그리기 거리. 거리 값은 기본적으로 픽셀 단위의 깊이 값이며 카메라 위치에 상대적입니다.

예를 들어 위치가 0이면 카메라에 가장 가깝고 증가할 때마다 멀어집니다. 기본적으로 뷰의 카메라는 -16000 깊이에 배치되므로 안개가 깊이 0에서 시작하여 깊이 1000에서 끝나도록 하려면 시작 거리를 16000으로 설정하고 끝 거리는 17000으로 설정합니다.

이 함수는 4개의 개별 인수(위에서 제공) 또는 다음 구조의 단일 배열을 사용할 수 있습니다(아래 예제 코드는 이 방법을 보여줍니다).

  • [0] = 활성화된 토글(부울, true 또는 false ), 기본 false
  • [1] = 색상(실제), 기본값 c_black
  • [2] = 시작 거리(실제), 기본값 0
  • [3] = 끝 거리(실제), 기본값 1

통사론:

gpu_set_fog(enable, colour, start, end);

논쟁 유형 설명
enable 부울 Enable or disable fog
colour 색깔 The fog colour
start 진짜 The distance to start applying fog from (relative to the camera)
end 진짜 The distance to end the fog (relative to the camera)

예시:

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

위의 코드는 현재 안개 설정을 가져온 다음 배열의 색상 요소를 다음으로 설정합니다. c_red 변경된 배열을 사용하여 안개를 다시 설정하기 전에.
