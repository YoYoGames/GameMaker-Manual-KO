gpu_set_blendenable

이 기능을 사용하여 알파 블렌딩을 켜고 끌 수 있습니다. 기본적으로 이렇게 설정하면 false, 그려지는 모든 이미지는 100% 불투명하게 그려집니다. 즉, 스프라이트 또는 배경의 투명 또는 반투명 영역이 단색으로 표시됩니다. 가능하면 알파 블렌딩 을 끄는 것이 좋습니다(특히 모바일 장치용으로 개발할 때). 이렇게 하면 그리기 속도가 크게 증가하기 때문입니다.

통사론:

gpu_set_blendenable(enable);

논쟁 유형 설명
enable Enable or disable alpha blending value (true or false ).

보고:

예시:

gpu_set_blendenable(false);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_blendenable(true);

위의 코드는 알파 블렌딩을 끄고 배경 스프라이트를 그린 다음 다시 켜서 계속 그리도록 합니다.
