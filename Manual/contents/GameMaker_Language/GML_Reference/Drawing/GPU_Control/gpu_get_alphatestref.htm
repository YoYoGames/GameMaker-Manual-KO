gpu_get_alphatestref

이 함수를 사용하여 알파 테스트 참조의 현재 값을 찾을 수 있습니다(기본값은 0이지만 다음을 사용할 수 있습니다.gpu_set_alphatestref() 이 값을 이것 이외의 값으로 설정).

통사론:

gpu_get_alphatestref();

보고:

(from 0 - 255)

예시:

if gpu_get_alphatestenable()
{
    if (gpu_get_alphatestref() < 254)
    {
        gpu_set_alphatestref(254);
    }
}

위의 코드는 알파 테스트가 활성화되어 있는지 확인하고 활성화되어 있으면 현재 알파 테스트 참조 값을 확인하고 254보다 작으면 설정합니다.
