gpu_get_colourwriteenable

이 함수는 현재 색상 쓰기 가능 값을 검색하는 데 사용할 수 있습니다. 이 함수는 다음 요소 중 하나를 포함하는 4요소 1D 배열을 반환합니다. true (활성화) 또는 false (장애가있는). 기본적으로 모든 색상 쓰기는 다음으로 설정됩니다. true:

  • [0] = 빨간색 채널 활성화/비활성화
  • [1] = 녹색 채널 활성화/비활성화
  • [2] = 파란색 채널 활성화/비활성화
  • [3] = 알파 채널 활성화/비활성화

통사론:

gpu_get_colorwriteenable();

보고:

(4 elements only; see above for details)

예시:

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

위의 코드는 현재 색상 쓰기 값을 가져온 다음 알파 구성 요소를 다음으로 설정합니다. false.
