draw_primitive_end

이 함수는 프리미티브의 포인트 정의를 마쳤을 때 호출되어야 합니다. 이 함수를 호출하지 않으면 GameMaker 에 사용자가 완료했으며 이제 정의된 프리미티브를 그릴 수 있음을 효과적으로 알려주므로 아무 것도 그려 지지 않습니다.

통사론:

draw_primitive_end()

보고:

예시:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

위의 코드는 "tex" 변수에 있는 텍스처로 텍스처링된 4개의 꼭짓점 삼각형 스트립을 그립니다.
