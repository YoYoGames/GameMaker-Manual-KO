vertex_format_begin

새로운 정점 형식을 정의하기 전에 이 함수를 사용하여 그렇게 하고 있음을 GameMaker 에 알려야 합니다. 이것을 먼저 호출한 다음 적절한 함수를 사용하여 형식 값을 정의하고 마지막으로 호출해야 합니다.vertex_format_end() 정의를 완료하고 새 형식 "핸들"을 반환합니다.

통사론:

vertex_format_begin();

보고:

예시:

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

위의 코드는 색상과 위치 값으로 새로운 꼭짓점 형식을 만든 다음 "my_format" 변수에 형식 ID를 저장합니다.
