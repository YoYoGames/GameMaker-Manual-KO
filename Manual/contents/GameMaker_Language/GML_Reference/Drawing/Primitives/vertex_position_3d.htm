vertex_position_3d

이 함수는 사용자 정의 프리미티브에 대해 현재 정의되고 있는 정점에 3D 위치 데이터를 추가합니다. 데이터를 쓰기 위한 버퍼와 그리기를 위한 x, y 및 z 위치를 제공합니다.

통사론:

vertex_position_3d(buffer, x, y, z);

논쟁 유형 설명
buffer 정점 버퍼 ID The buffer to write the information to.
x 진짜 The x position.
y 진짜 The y position.
z 진짜 The z position.

보고:

N/A

예시:

vertex_position_3d(buff, x - 100, y - 125, 0);

위의 코드는 정의되는 정점의 위치를 설정합니다.
