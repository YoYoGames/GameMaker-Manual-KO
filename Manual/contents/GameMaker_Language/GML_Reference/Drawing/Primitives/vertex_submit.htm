vertex_submit

이 함수를 사용하여 정점 버퍼의 내용을 셰이더와 함께 사용하기 위해 그래픽 파이프라인에 제출할 수 있습니다. 사용할 버퍼 인덱스, 사용할 기본 기본 유형(아래 상수 참조) 및 사용할 텍스처를 제공합니다. 기본 프리미티브 유형은 버퍼에 저장한 정점이 그려지고 연결되는 순서를 지정하는 데만 사용되지만 각 정점에 사용되는 실제 데이터는 정점 버퍼를 생성할 때 정의한 데이터가 됩니다.

참고: 이 기능은 이벤트 뽑기 에서만 사용할 수 있습니다.

사용 가능한 다양한 기본 기본 요소의 시각적 예는 아래 이미지를 참조하세요.

일정한 설명
pr_pointlist 포인트 목록으로 구성된 프리미티브.
pr_linelist 목록의 개별 행으로 구성된 프리미티브.
pr_linestrip 연속된 선 스트립으로 구성된 기본 요소입니다.
pr_trianglelist 목록의 개별 삼각형으로 구성된 프리미티브.
pr_trianglestrip 연속적인 삼각형 스트립으로 구성된 프리미티브.

통사론:

vertex_submit(buffer, primitive, texture);

논쟁 유형 설명
buffer 정점 버퍼 ID The buffer to use.
primitive 기본 유형 상수 The primitive base type.
texture 텍스처 포인터 The texture to use (or -1 for no texture).

보고:

N/A

예시:

shader_set(shader_prim);
vertex_submit(buff, pr_trianglelist, sprite_get_texture(sprite_index));
shader_reset();

위의 코드는 기본 프리미티브로 삼각형 목록을 사용하고 코드를 실행하는 인스턴스에 대한 스프라이트 텍스처를 사용하여 셰이더 "shader_prim"으로 그리기 위해 변수 "buff"에 인덱싱된 정점 버퍼를 제출합니다.
