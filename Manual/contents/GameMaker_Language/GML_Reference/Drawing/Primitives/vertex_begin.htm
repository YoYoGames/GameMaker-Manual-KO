vertex_begin

이 함수를 사용하여 사용자 정의 프리미티브의 정의를 시작합니다. 프리미티브를 쓸 버퍼와 사용할 정점 형식(이전에 정점 형식 함수를 사용하여 정의)을 할당합니다. 그런 다음 호출하기 전에 기본 요소의 각 꼭짓점에 필요한 점을 정의합니다.vertex_end() 원시 생성을 완료합니다.

통사론:

vertex_begin(buffer, format);

논쟁 유형 설명
buffer 정점 버퍼 ID The buffer to be written to.
format 정점 형식 ID The vertex format to use.

보고:

N/A

예시:

vertex_format_begin();
vertex_format_add_position();
vertex_format_add_colour();
vertex_format_add_textcoord();
v_format = vertex_format_end();
v_buff = vertex_create_buffer();
vertex_begin(v_buff, v_format);

위의 코드는 새로운 정점 형식을 정의하고, 새로운 버퍼를 생성하고, 새로운 프리미티브의 정의 프로세스를 시작합니다.
