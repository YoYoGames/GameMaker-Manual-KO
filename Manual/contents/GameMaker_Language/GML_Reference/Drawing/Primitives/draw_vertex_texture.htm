draw_vertex_texture

이 함수는 프리미티브에 대한 텍스처 정점의 위치를 정의합니다. 프리미티브의 최종 모양은 그리기 위해 선택한 프리미티브 유형에 따라 다릅니다(참조draw_primitive_begin() 추가 정보), 정점을 추가하는 순서 및 텍스처에 대해 설정한 시작점의 위치. 프리미티브를 끝내고 그리려면 다음을 호출해야 합니다.draw_primitive_end().

샘플링할 적절한 xtex/ytex 위치를 설정하여 프리미티브 표면 위에 텍스처의 일부만 그리도록 선택할 수도 있습니다(이 좌표는 UV 포인트라고도 함). 이 점은 사용할 이미지의 왼쪽 상단 모서리가 (0,0) 지점이고 오른쪽 하단 모서리가 (1,1) 지점인 0과 1 사이의 값으로 정의됩니다. 이런 식으로 소스 이미지 자산의 일부만으로 프리미티브를 텍스처링할 수 있습니다.

텍스처 프리미티브를 그릴 때 텍스처는 다음에서 설정한 색상과 혼합됩니다. draw_set_colour. 이것은 블렌딩 없이 텍스처를 그리려면 다음으로 설정해야 함을 의미합니다. c_white, 아래 예와 같이.

참고 텍스처가 반복되려면 크기가 2의 거듭제곱이어야 합니다. 예: 32x32, 128x128 등...

통사론:

draw_vertex_texture(x, y, xtex, ytex)

논쟁 유형 설명
x 진짜 The x coordinate of the vertex.
y 진짜 The y coordinate of the vertex.
xtex 진짜 The width of the texture sample to use.
ytex 진짜 The height of the texture sample to use.

보고:

N/A

예시:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

위의 코드는 "tex" 변수에 있는 텍스처로 텍스처링된 4개의 꼭짓점 삼각형 스트립(사각형 만들기)을 그리고 전체 텍스처는 완성된 프리미티브를 덮는 데 사용됩니다.
