vertex_format_add_texcoord

GameMaker 가 텍스처 위치 데이터(u 및 v)를 생성 중인 새 정점 형식의 일부로 수락하도록 지시합니다.

통사론:

vertex_format_add_texcoord();

보고:

예시:

vertex_format_begin();
vertex_format_add_texcoord();
vertex_format_add_position_3d();
my_format = vertex_format_end();

위의 코드는 텍스처와 위치 값만으로 새로운 꼭짓점 형식을 만든 다음 "my_format" 변수에 형식 ID를 저장합니다.
