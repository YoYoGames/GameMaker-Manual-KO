draw_primitive_begin

이 함수는 프리미티브를 정의하기 전에 호출되어야 합니다. 다음 상수로 정의할 수 있는 6가지 유형의 기본요소가 있습니다.

기본 유형 상수
일정한 설명
pr_pointlist 포인트 목록
pr_linelist 라인 목록
pr_linestrip 라인 스트립
pr_trianglelist 삼각형 목록
pr_trianglestrip 삼각형 스트립
pr_trianglefan 삼각팬

다음 이미지는 기본적으로 이것이 어떻게 보여야 하고 정점을 정의해야 하는 순서를 보여줍니다.

일부 플랫폼(Windows, UWP, XBox)에서는 pr_trianglefan 유형은 기본적으로 지원되지 않으므로 GameMaker 는 게임이 작동하도록 컴파일할 때 변환을 수행합니다. 이는 해당 플랫폼에서 pr_trianglefan 유형은 다른 유형보다 사용하는 데 훨씬 느립니다. HTML5에서 이 기능을 사용하려면 게임 옵션에서 WebGL을 활성화 해야 합니다.

통사론:

draw_primitive_begin(kind)

논쟁 유형 설명
kind 기본 유형 상수 The kind of primitive you are going to draw.

보고:

N/A

예시:

var _steps = 20;
var _xx = 50;
var _yy = 50;
var _radius = 30;
draw_primitive_begin(pr_trianglefan);
draw_vertex(_xx, _yy);
for(var i = 0; i <= _steps; ++i)
{
    draw_vertex(_xx + lengthdir_x(_radius, 270 * i / _steps), _yy + lengthdir_y(_radius, 270 * i / _steps));
}
draw_primitive_end();

위의 코드는 프리미티브로 만든 원의 4분의 3을 그립니다.
