vertex_create_buffer

이 기능으로 새로운 정점 버퍼를 생성할 수 있습니다. 이것은 GameMaker 에서 생성한 특수 성장 버퍼로 프리미티브를 구축할 때 사용하도록 미리 형식이 지정되어 있습니다(예: 셰이더와 함께 사용). 이 함수는 버퍼에 대한 인덱스를 반환한 다음 모든 추가 호출에 사용해야 합니다.

이 함수로 생성된 정점 버퍼를 사용할 때vertex_begin() 정점 데이터를 할당하기 시작하여 커스텀 프리미티브를 정의하기 시작한 다음 셰이더에 제출할 준비가 된 버퍼에 보관됩니다. 버퍼는 필요할 때 재사용할 수 있습니다.vertex_freeze() 함수)를 호출할 때마다vertex_begin() 새 데이터를 수락할 준비가 된 이전 버퍼 데이터를 삭제합니다.

통사론:

vertex_create_buffer();

보고:

Vertex Buffer ID

예시:

v_buff = vertex_create_buffer();

위의 코드는 새 정점 버퍼를 만들고 "v_buff" 변수에 핸들을 저장합니다.
