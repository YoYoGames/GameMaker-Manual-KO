vertex_end

이 함수를 사용하여 사용자 정의 기본 요소의 빌드를 종료합니다. 이 명령을 호출하면 프리미티브를 사용할 수 있습니다.vertex_submit() 셰이더에서 사용하기 위한 함수를 사용하거나 다음을 사용하여 버퍼를 고정할 수 있습니다(정점 버퍼를 읽기 전용으로 사용하고 훨씬 빠르게 만들기).vertex_freeze().

통사론:

vertex_end(buffer);

논쟁 유형 설명
buffer 정점 버퍼 ID The buffer to end writing to.
