vertex_get_number

이 함수를 사용하면 주어진 정점 버퍼에 정의된 개별 정점의 수를 찾을 수 있습니다.

통사론:

vertex_get_number(buffer);

논쟁 유형 설명
buffer 정점 버퍼 ID The vertex buffer to check.

보고:

Real

예시:

v_num = vertex_get_number(v_buffer);

위의 코드는 주어진 정점 버퍼에 저장된 정점의 수를 변수에 저장할 것입니다.
