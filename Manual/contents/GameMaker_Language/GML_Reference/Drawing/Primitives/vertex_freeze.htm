vertex_freeze

이 함수는 정점 버퍼를 "고정"하는 데 사용할 수 있습니다. 이 버퍼는 읽기 전용 이 되므로 변경해야 하는 경우 전체 버퍼를 삭제하고 다시 만들어야 합니다. 고정된 버퍼는 일반 동적 버퍼보다 빠르게 셰이더에 제출할 수 있으며 레벨 또는 게임 기간 동안 변하지 않는 사용자 정의 기본 요소가 필요한 효과에 권장됩니다.

함수는 지정된 정점 버퍼를 성공적으로 고정하면 0을 반환하지만 어떤 이유로든 실패하면 -1을 반환합니다.

통사론:

vertex_freeze(buffer);

논쟁 유형 설명
buffer 정점 버퍼 ID The vertex buffer to freeze.

보고:

Real (0 for success, -1 for failure)

예시:

// Create event
vertex_freeze(vbuff);

위의 코드는 Create 이벤트에서 지정된 정점 버퍼를 고정하므로 더 빠른 액세스를 위해 다른 이벤트에서 "정적" 버퍼로 사용할 수 있습니다.
