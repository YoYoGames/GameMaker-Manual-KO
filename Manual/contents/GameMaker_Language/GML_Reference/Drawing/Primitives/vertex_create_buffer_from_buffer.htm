vertex_create_buffer_from_buffer

기능과 마찬가지로vertex_create_buffer(), 이 함수는 새 정점 버퍼를 생성합니다. 이제 저장하는 정점 데이터는 소스로 지정된 일반 버퍼에서 복사됩니다. 생성된 버퍼는 (예를 들어) 셰이더와 함께 사용할 프리미티브를 구축하기 위해 꼭짓점 형식으로 미리 형식이 지정된 특수 성장 버퍼입니다. 이 함수를 사용하려면 이전에 생성된 일반 버퍼에 대한 포인터와 복사된 데이터에 적용해야 하는 꼭짓점 형식을 제공해야 합니다.

통사론:

vertex_create_buffer_from_buffer(buffer, format);

논쟁 유형 설명
buffer 버퍼 ID The buffer to create the vertex buffer from.
format 기본 유형 상수 The primitive vertex format to use.

보고:

Vertex Buffer ID

예시:

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat);

위의 코드는 새 정점 형식을 만든 다음 이전에 만든 일반 버퍼에서 새 정점 버퍼를 만들어 사용자 지정 정점 형식을 적용합니다.
