vertex_colour

정의된 꼭짓점 형식이 색상 값을 사용하는 경우 이 함수를 사용하여 해당 데이터를 현재 프리미티브에 대해 정의되는 꼭짓점에 추가할 수 있습니다. 이 함수는 데이터를 저장할 버퍼가 필요하며 색상 상수 또는 16진수 값(BGR의 표준 GameMaker 형식 사용, 예: 파란색의 경우 $FF0000 )과 0(투명)에서 알파 값을 취합니다. 1(완전 불투명).

통사론:

vertex_colour(buffer, colour, alpha);

논쟁 유형 설명
buffer 정점 버퍼 ID The buffer to write the information to.
colour 색깔 The colour for this vertex (can be a constant or a hex value).
alpha 진짜 The alpha value for the vertex (from 0 to 1).

보고:

N/A

예시:

vertex_colour(b, c_white, 1);

위의 코드는 정의되는 정점을 알파 값이 1인 흰색으로 설정합니다.
