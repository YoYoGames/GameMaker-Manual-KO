vertex_format_delete

이 함수는 생성된 벡터 형식 사용을 완료할 때마다 호출해야 합니다. 형식 ID 값을 제공합니다(함수에서 반환된vector_format_end() ), 이 함수는 관련된 메모리를 해제합니다. 이 함수를 호출한 후 이 형식을 다시 사용하려고 하면 오류가 발생합니다.

통사론:

vertex_format_delete(formatID);

논쟁 유형 설명
formatID 정점 형식 ID The vertex format to delete

보고:

N/A

예시:

vertex_format_delete(myFormat);

위의 코드는 "myFormat" 변수에 생성된 꼭짓점 형식을 메모리에서 제거합니다.
