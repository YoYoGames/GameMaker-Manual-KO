vertex_create_buffer_from_buffer_ext

기능과 마찬가지로vertex_create_buffer(), 이 함수는 새 정점 버퍼를 생성합니다. 이제 저장하는 정점 데이터는 소스로 지정된 일반 버퍼에서 복사됩니다. 버퍼는 (예를 들어) 셰이더와 함께 사용할 프리미티브를 빌드하기 위한 꼭짓점 형식으로 미리 형식이 지정되어 있으며, 원본 버퍼 내에서 복사할 오프셋과 최종 버퍼에 있어야 하는 꼭짓점 수를 제공할 수도 있습니다. 정점 수가 복사 중인 정점 수와 일치하지 않으면 정점 데이터가 손상될 수 있습니다.

통사론:

vertex_create_buffer_from_buffer_ext(buffer, format, src_offset, vert_num);

논쟁 유형 설명
buffer 버퍼 ID The buffer to create the vertex buffer from.
format 기본 유형 상수 The primitive vertex format to use.
src_offset 진짜 The offset within the the source buffer to copy from.
vert_num 진짜 The number of vertices the buffer should have.

보고:

Vertex Buffer ID

예시:

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat, 0, 512);

위의 코드는 새 꼭짓점 형식을 만든 다음 이전에 만든 일반 버퍼에서 새 꼭짓점 버퍼를 만들고 오프셋이 0인 사용자 지정 꼭짓점 형식을 적용합니다. 이 함수는 새 정점 버퍼에 512개의 정점을 예상해야 한다고 알려줍니다.
