draw_primitive_begin_texture

텍스처 프리미티브의 정점을 정의하기 전에 이 함수를 호출해야 합니다. 사용할 프리미티브의 종류를 지정해야 합니다(참조draw_primitive_begin() 추가 정보) 및 사용할 텍스처의 ID 로 스프라이트 또는 배경 이미지 자산이 될 수 있습니다. 이 자산 ID 는 함수에서 가져올 수 있습니다. sprite_get_texture(), 예를 들어(텍스처가 없는 경우 -1 사용).

참고 텍스처가 반복되려면 크기가 2의 거듭제곱이어야 합니다. 예: 32x32, 128x128 등...

통사론:

draw_primitive_begin_texture(kind, tex)

논쟁 유형 설명
kind 기본 유형 상수 The kind of primitive you are going to draw.
tex 텍스처 포인터 The texture to use with the primitive.

보고:

N/A

예시:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

위의 코드는 "tex" 변수에 있는 텍스처로 텍스처링된 4개의 꼭짓점 삼각형 스트립(사각형 만들기)을 그리고 전체 텍스처는 완성된 프리미티브를 덮는 데 사용됩니다.
