vertex_delete_buffer

이 함수는 이전에 생성된 정점 버퍼를 제거하는 데 사용할 수 있습니다(참조vertex_create_buffer() ) 시스템 메모리에서. 이렇게 하면 포함된 모든 사용자 정의 기본 요소에 대한 모든 정점 데이터도 제거됩니다.

통사론:

vertex_delete_buffer(buffer);

논쟁 유형 설명
buffer 정점 버퍼 ID The vertex buffer to be removed.

보고:

N/A

예시:

vertex_delete_buffer(v_buff);

위의 코드는 "v_buff" 변수에 인덱싱된 버퍼를 메모리에서 삭제합니다.
