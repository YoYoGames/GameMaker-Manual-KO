vertex_ubyte4

이 함수는 4개의 부호 없는 바이트 값(0 - 255)을 정점 데이터에 추가합니다. 정점은vertex_format_add_custom() 함수.

통사론:

vertex_ubyte4(buffer, byte, byte, byte, byte);

논쟁 유형 설명
buffer 정점 버퍼 ID The buffer to write the information to.
byte 진짜 The first input value.
byte 진짜 The second input value.
byte 진짜 The third input value.
byte 진짜 The fourth input value.

보고:

N/A

예시:

vertex_ubyte4(buff, irandom(255), irandom(255), irandom(255), 127);

위의 코드는 정의되는 정점 데이터에 4개의 값을 추가합니다.
