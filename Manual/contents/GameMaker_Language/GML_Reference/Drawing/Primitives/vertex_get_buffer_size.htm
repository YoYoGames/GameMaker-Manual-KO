vertex_get_buffer_size

이 함수를 사용하면 주어진 정점 버퍼의 크기를 바이트 단위로 얻을 수 있습니다.

통사론:

vertex_get_buffer_size(buffer);

논쟁 유형 설명
buffer 정점 버퍼 ID The buffer to get the size of.

보고:

Real

예시:

bufferBytes = vertex_get_buffer_size(buff);

위의 코드는 주어진 정점 버퍼에 의해 사용된 바이트 수를 얻고 변수 "bufferBytes"에 값을 저장합니다.
