vertex_argb

이 함수는 커스텀 프리미티브에 대해 현재 정의되고 있는 꼭짓점에 대한 ARGB 값을 설정합니다. 단일 32비트 부호 없는 정수로 사용하려는 빨강, 녹색, 파랑 및 알파 값에 데이터를 쓸 버퍼를 제공합니다. 샘플 및 마지막으로 가장 낮은 8비트의 파란색 샘플입니다. 16진수 표기법( $AA RRGGBB)을 실수로 사용하거나 다음 중 하나를 사용할 수 있습니다. make_colour_*() 색상 값을 정의하는 함수.

통사론:

vertex_argb(buffer, argb);

논쟁 유형 설명
buffer 정점 버퍼 ID The buffer to write the information to.
argb 색깔 The ARGB value to set.

보고:

N/A

예시:

vertex_argb(buff, $FFFFFFFF);

위의 코드는 정의되는 정점의 ARGB 값을 흰색으로 설정합니다.
