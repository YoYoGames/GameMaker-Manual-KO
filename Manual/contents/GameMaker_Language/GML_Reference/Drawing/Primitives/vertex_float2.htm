vertex_float2

이 함수는 정점 데이터에 두 개의 부동 소수점 값을 추가합니다. 정점은vertex_format_add_custom() 함수.

통사론:

vertex_float2(buffer, float, float);

논쟁 유형 설명
buffer 정점 버퍼 ID The buffer to write the information to.
float 진짜 The first input value.
float 진짜 The second input value.

보고:

N/A

예시:

vertex_float2(buff, 0.05, 0.01);

위의 코드는 정의되는 정점 데이터에 두 개의 부동 소수점 값을 추가합니다.
