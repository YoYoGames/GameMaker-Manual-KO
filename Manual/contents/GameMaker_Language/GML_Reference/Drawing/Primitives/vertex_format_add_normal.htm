vertex_format_add_normal

GameMaker 에 생성 중인 새 정점 형식의 일부로 표면 법선 데이터(nx, ny 및 nz)를 허용하도록 지시합니다.

통사론:

vertex_format_add_normal();

보고:

예시:

vertex_format_begin();
vertex_format_add_textcoord();
vertex_format_add_normal();
my_format = vertex_format_end();

위의 코드는 텍스처와 표면 노멀 값으로 새로운 정점 형식을 만든 다음 변수 "my_format"에 형식 ID를 저장합니다.
