vertex_create_buffer_ext

기능과 마찬가지로vertex_create_buffer(), 이 함수는 새로운 정점 버퍼를 생성합니다. 이것은 GameMaker 에서 생성한 특수 성장 버퍼로, 셰이더와 함께 사용할 프리미티브를 빌드할 때 사용하도록 미리 형식이 지정되어 있습니다. 버퍼의 초기 시작 크기(바이트 단위)를 지정할 수 있으며 버퍼에 대한 모든 추가 호출에서 사용해야 하는 버퍼에 대한 인덱스를 반환합니다.

통사론:

vertex_create_buffer_ext(size);

논쟁 유형 설명
size The initial size of the buffer (in bytes).

보고:

Vertex Buffer ID

예시:

v_buff = vertex_create_buffer_ext(1024 * 1024);

위의 코드는 처음에 크기가 1MB인 새로운 정점 버퍼를 만들고 "v_buff" 변수에 핸들을 저장합니다.
