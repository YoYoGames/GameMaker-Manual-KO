vertex_format_end

이 함수는 새로운 정점 형식을 정의한 후에 호출해야 합니다. 이 새 형식을 참조하는 모든 추가 정점 함수에서 사용해야 하는 새 형식 "핸들"(색인)을 반환합니다.

통사론:

vertex_format_end();

보고:

Vertex Format ID

예시:

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

위의 코드는 색상과 위치 값으로 새로운 꼭짓점 형식을 만든 다음 "my_format" 변수에 형식 ID를 저장합니다.
