vertex_texcoord

이 함수는 커스텀 프리미티브에 대해 현재 정의되고 있는 꼭짓점에 사용할 텍스처 좌표를 설정합니다. 데이터를 쓸 버퍼와 사용할 텍스처 내의 UV 위치를 제공합니다.

통사론:

vertex_texcoord(buffer, u, v);

논쟁 유형 설명
buffer 정점 버퍼 ID The buffer to write the information to.
u 진짜 The first texture coordinate to use (0 - 1).
v 진짜 The second texture coordinate to use (0 - 1).

보고:

N/A

예시:

vertex_texcoord(buff, 0, 0);

위의 코드는 정의되는 정점의 UV 값을 설정합니다.
