video_get_status

이 함수는 현재 로드된 비디오의 상태를 반환합니다. 상태는 다음 상수 중 하나일 수 있습니다.

비디오 상태 상수
일정한 설명
video_status_closed 현재 로드된 동영상이 없거나 다음으로 동영상이 종료되었습니다. video_close()
video_status_preparing 동영상이 현재 준비 중이며 아직 재생이 시작되지 않았습니다.
video_status_playing 동영상이 현재 재생 중입니다.
video_status_paused 비디오가 일시 중지되었습니다(참조 video_pause() )

통사론:

video_get_status();

보고:

Video Status Constant

예시:

var _status = video_get_status();

if (keyboard_check_pressed(vk_space))
{
    if (_status == video_status_playing)
    {
        video_pause();
    }
    else if (status == video_status_paused)
    {
        video_resume();
    }
}

위의 코드는 비디오의 상태를 가져온 다음 플레이어가 Space를 눌렀는지 확인합니다. 이 경우 비디오가 재생 중이면 일시 중지하고 일시 중지되면 다시 시작합니다.
