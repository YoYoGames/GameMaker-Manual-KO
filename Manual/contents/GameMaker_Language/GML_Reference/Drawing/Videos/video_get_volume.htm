video_get_volume

이 함수는 현재 로드된 비디오의 볼륨을 0에서 1 사이의 값으로 반환합니다. 로드된 비디오의 볼륨은 다음을 사용하여 변경할 수 있습니다. video_set_volume().

통사론:

video_get_volume();

보고:

Real

예시:

/// Step event
var _video_volume = video_get_volume();

if (_video_volume < 1) {
    _video_volume += 0.02;
    video_set_volume(_video_volume);
}

위의 코드는 Step 이벤트에서 실행됩니다. 현재 재생 중인 영상의 볼륨을 구하고, 1보다 낮으면 0.02씩 증가시켜 최대 레벨이 될 때까지 점차적으로 볼륨을 높입니다.
