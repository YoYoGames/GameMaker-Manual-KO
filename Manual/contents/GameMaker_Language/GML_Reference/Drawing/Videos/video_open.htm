video_open

이 함수는 지정된 경로를 사용하여 비디오 파일을 엽니다. 경로는 프로젝트의 포함된 파일 에 있는 파일, 사용자 장치에 로컬로 존재하는 파일 또는 인터넷에서 파일을 로드하기 위한 URL(일부 플랫폼에서는 작동하지 않을 수 있음)을 참조할 수 있습니다.

한 번에 하나의 비디오만 로드할 수 있으므로 이 기능은 지정된 비디오를 유일한 사용 가능한 "비디오 슬롯"으로 로드한 다음 다른 비디오 기능 을 통해 조작합니다.

비디오 재생과 관련된 비동기 콜백이 있습니다. 자세한 내용은 비동기 콜백 을 참조하세요.

통사론:

video_open(path);

논쟁 유형 설명
path 끈 Path to the video file

보고:

N/A

예시:

my_video = video_open("splash.mp4");

위의 코드는 게임의 포함 파일에서 splash.mp4 를 로드합니다.
