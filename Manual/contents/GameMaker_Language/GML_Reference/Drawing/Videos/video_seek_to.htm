video_seek_to

이 기능을 사용하면 현재 로드된 비디오에서 지정된 위치를 찾을 수 있습니다. 검색할 video_seek_to()>시간 값을 밀리초 단위로 지정합니다.

이 기능은 비디오가 일시 중지되고 현재 위치에서 새 탐색 위치까지의 거리가 너무 작지 않을 때 가장 잘 작동합니다.

통사론:

video_seek_to(time);

논쟁 유형 설명
time 진짜 The time to seek to (in milliseconds).

보고:

N/A

예시:

if (keyboard_check_pressed(vk_right))
{
    var _video_position = video_get_position();
    _video_position += 2000;

    if (_video_position < video_get_duration()) 
    {
        video_seek_to(_video_position);
    }
    else
    {
        video_close();
    }
}

위의 코드는 오른쪽 화살표 키를 눌렀을 때 재생 중인 동영상을 2초 앞으로 이동하고 새 위치가 동영상의 재생 시간보다 작습니다. 단, 해당 시간이 아닐 경우 영상이 종료됩니다.

이 예에서는 새로운 위치로 비디오를 찾는 데 사용됩니다.
