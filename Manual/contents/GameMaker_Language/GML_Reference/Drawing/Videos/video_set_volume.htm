video_set_volume

이 기능은 현재 로드된 비디오의 오디오 볼륨을 변경합니다. video_open()>0 에서 video_close()>1 사이의 값을 사용합니다. 여기서 0 은 무음이고 1 은 최대 볼륨입니다.

이것이 호출된 후에만 호출되는지 확인하십시오. 전화하지만 전에 호출하지 않으면 아무 작업도 수행하지 않습니다(로드된 비디오가 없기 때문에).

일부 플랫폼에서 이 기능은 비디오가 완전히 로드된 후에만 작동하므로 비동기 콜백 을 사용하여 비디오가 언제 준비되었는지 확인한 다음 볼륨을 변경하십시오.

통사론:

video_set_volume(value);

논쟁 유형 설명
value 진짜 Volume value between 0 and 1

보고:

N/A

예시:

my_video = video_open("splash.mp4");
video_set_volume(0.5);

위의 코드는 게임의 포함 파일에서 splash.mp4 를 로드하고 볼륨을 0.5 (최대 볼륨의 50%)로 변경합니다.
