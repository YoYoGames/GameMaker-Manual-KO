video_is_looping

이 함수는 현재 로드된 비디오가 루프로 설정되어 있는지( video_enable_loop()>true ) 그렇지 않은지( false ) 반환합니다. 다음을 호출하여 비디오를 반복 재생하도록 설정할 수 있습니다..

통사론:

video_is_looping();

보고:

Boolean

예시:

if (video_is_looping()) {
    obj_player.control_enabled = false;
}

위의 코드는 현재 로드된 비디오가 반복되는지 확인하고 이 경우 사용자 정의 인스턴스 변수를 false 로 설정하여 플레이어의 컨트롤을 비활성화합니다.
