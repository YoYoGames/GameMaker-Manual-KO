video_close

이 함수는 현재 로드된 비디오 파일을 닫습니다. 이것이 호출된 후에만 호출되는지 확인하십시오. video_open() 그렇지 않으면 아무 것도 하지 않습니다.

통사론:

video_close();

보고:

N/A

예시:

if (keyboard_check_pressed(vk_escape))
{
    video_close();
}

위의 코드는 Esc 키를 누르면 비디오를 닫습니다.
