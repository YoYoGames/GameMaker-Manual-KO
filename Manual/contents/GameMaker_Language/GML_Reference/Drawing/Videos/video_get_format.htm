video_get_format

이 함수는 현재 로드된 비디오의 색상 형식을 반환합니다. 다음 상수 중 하나일 수 있습니다.

비디오 형식 상수
일정한 설명
video_format_rgba 비디오 표면은 RGBA 색상 모델을 사용합니다.
video_format_yuv 비디오 표면은 YUV 색상 모델을 사용합니다.

Windows, macOS, Opera GX, Android, iOS 및 HTML5는 비디오에 RGBA 모델을 사용합니다. 다른 모든 플랫폼은 YUV 모델을 사용합니다.

RGBA와 YUV 영상의 그리기 방식이 다르기 때문에 이 함수의 반환값을 사용하여 재생 영상의 형식에 따라 다른 코드를 실행할 수 있습니다. 예를 보려면 비디오 그리기 를 참조하십시오.

통사론:

video_get_format();
