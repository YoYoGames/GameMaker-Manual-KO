video_resume

이 기능은 현재 로드된 비디오 파일을 다시 시작합니다. 다음과 같이 일시중지되었어야 합니다. video_pause() 이 함수를 호출하기 전에.

통사론:

video_resume();

보고:

N/A

예시:

var _status = video_get_status();

if (keyboard_check_pressed(vk_space))
{
    if (_status == video_status_playing)
    {
        video_pause();
    }
    else if (status == video_status_paused)
    {
        video_resume();
    }
}

위의 코드는 비디오의 상태를 가져온 다음 플레이어가 Space를 눌렀는지 확인합니다. 이 경우 비디오가 재생 중이면 일시 중지하고 일시 중지되면 다시 시작합니다.
