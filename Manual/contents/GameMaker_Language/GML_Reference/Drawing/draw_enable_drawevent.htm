draw_enable_drawevent

이 기능을 사용하여 활성화하도록 선택할 수 있습니다(true ) 또는 비활성화(false ) 게임의 모든 인스턴스에 대한 그리기 이벤트, 따라서 항목을 그리는 방법과 시기를 제어할 수 있으므로 "프레임 건너뛰기" 기술을 구현하려는 경우에 유용합니다. 이것은 인스턴스가 화면에 그리는 것을 방지할 뿐만 아니라 그리기 이벤트를 완전히 억제하므로 해당 이벤트에 있는 게임 논리도 실행되지 않으므로 주의해야 합니다.

이 함수에 대해 이해해야 할 한 가지 중요한 점은 게임이 시작될 때 바로 호출하면 초기 프레임이 렌더링되기 전에(즉: 게임의 첫 번째 방에 있는 첫 번째 개체의 이벤트 만들기) 게임이 창이 렌더링되지 않습니다. 이것은 아무 것도 렌더링할 필요가 없는 전용 서버가 필요한 멀티플레이어 프로젝트에 유용할 수 있지만 GameMaker가 이를 지원하지 않기 때문에 헤드리스가 되지는 않습니다. 따라서 실행할 수 없습니다. 그래픽 인터페이스가 없는 서버.

통사론:

draw_enable_drawevent(enable);

논쟁 유형 설명
Enable Set to true or false.

보고:

예시:

frame_skip ++;
if (frame_skip mod 5) == 0
{
    draw_enable_drawevent(true);
}
else
{
    draw_enable_drawevent(false);
}

위의 코드는 변수를 확인하고 반환하는 경우 true, 그리기 이벤트를 활성화하고, 그렇지 않으면 그리기 이벤트를 비활성화합니다.
