physics_joint_get_value

일련의 사전 정의된 상수를 사용하여 GameMaker 에 지정된 관절의 상태에 대한 여러 가지 정보를 알려달라고 요청할 수 있습니다. 이는 당시의 필요에 따라 관절을 삭제하거나 인스턴스 동작을 변경할 수 있는 기능을 제공하므로 매우 유용합니다. 이 함수에 사용할 수 있는 여러 상수가 있으며 여기에서 찾을 수 있습니다. Physics Joint Constants, 그러나 이것을 호출할 때 복잡한 계산이 수행되므로 주의해서 사용해야 하고 필요할 때만 사용해야 합니다. 많은 것이 특정 유형의 관절에 고유합니다.

속성이 존재하지 않는 경우(예를 들어, 도르래 조인트에서 프리즘 변환을 확인하는 경우) 0이 반환 값이 됩니다.

통사론:

physics_joint_get_value(joint, value)

논쟁 유형 설명
joint 물리 조인트 ID The index of the joint that you wish to test
value 물리학 조인트 상수 The constant for the joint property that you wish to test

보고:

Real

예시:

if shipJoint
{
    var reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
    var reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_y);
    var reactionForce = point_distance(0, 0, reactionForceX, reactionForceY);
    if (reactionForce > 2)
    {
        physics_joint_delete(shipJoint);
        shipJoint = -1;
    }
}

위의 코드는 "shipJoint" 변수가 관절 인덱스를 보유하고 있는지 확인하고 보유하고 있다면 두 상수를 사용하여 해당 관절에 가해지는 힘을 계산합니다. 마지막으로 전체 힘이 2보다 크면 조인트가 삭제됩니다.
