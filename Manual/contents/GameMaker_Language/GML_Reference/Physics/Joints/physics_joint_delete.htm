physics_joint_delete

물리적 표현이 있는 두 개의 인스턴스가 관절로 바인딩되면 언제든지 다시 삭제할 수 있습니다. 일반적으로 이것은 두 인스턴스 중 하나가 파괴되거나 방이 종료될 때 자동으로 발생하지만 수동으로 수행하려는 경우가 있습니다. 이러한 경우 이 기능을 사용합니다.

또한 기어 조인트 의 일부인 인스턴스가 파괴될 때마다 기어 형성과 관련된 인스턴스 보다 먼저 이 기능을 사용하여 기어 조인트를 삭제해야 합니다(나머지 조인트는 자동으로 삭제됨). 인스턴스의 파괴 이벤트의 예.

통사론:

physics_joint_delete(joint)

논쟁 유형 설명
joint 물리 조인트 ID The index of the joint that you wish to delete

보고:

N/A

예시:

var reactionForceX, reactionForceY, reactionForce;
if shipJoint
{
    reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
    reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
    reactionForce = sqrt((reactionForceX + reactionForceX) + (reactionForceY + reactionForceY));
    if (reactionForce > 2)
    {
        physics_joint_delete(shipJoint);
        shipJoint = -1;
    }
}

위의 코드는 "shipJoint" 변수가 관절 인덱스를 보유하고 있는지 확인하고, 보유하고 있다면 두 상수를 사용하여 해당 관절에 가해지는 힘을 계산합니다. 마지막으로 전체 힘이 2보다 크면 조인트가 삭제됩니다.
