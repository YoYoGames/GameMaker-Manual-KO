physics_joint_set_value

특정 관절 속성은 관절을 생성한 후에도 변경 및 설정할 수 있습니다. 이 함수에서 사용할 수 있는 상수가 많이 있으며 여기에서 찾을 수 있습니다. Physics Joint Constants.

통사론:

physics_joint_set_value(joint, field, value)

논쟁 유형 설명
joint 물리 조인트 ID The index of the joint that you wish to change
field 물리학 조인트 상수 The constant for the joint property that you wish to change
value 진짜 The new value for the joint property

보고:

N/A

예시:

if physics_joint_get_value(revJoint, phy_joint_max_motor_torque) < 2
{
    physics_joint_set_value(revJoint, phy_joint_max_motor_torque, 2);
}

위의 코드는 관절의 최대 모터 토크가 2 미만으로 설정되어 있는지 확인하고 설정되어 있으면 2로 설정합니다.
