물리학 세계

GameMaker 에서 물리적 반응이 일어나기 전에 먼저 일어날 물리적 세계를 정의해야 합니다. 이렇게 하지 않으면 GameMaker 는 기본 충돌 기능으로 기본 설정되고 다른 모든 물리 관련 코드 또는 설정은 무시됩니다. Room Editor 에서 물리 세계를 설정할 수 있지만 다음 기능이 있는 코드를 사용하여 동일한 속성을 설정할 수도 있습니다.

  • physics_world_create
  • physics_world_gravity
  • physics_world_update_iterations
  • physics_world_update_speed
  • physics_world_draw_debug
  • physics_draw_debug
  • physics_pause_enable
