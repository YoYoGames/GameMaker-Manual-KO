physics_pause_enable

일반적으로 물리 시뮬레이션이 작동하려면 연속적이어야 하며 중지 및 시작될 수 없거나 인스턴스가 방의 한 장소에서 다른 장소로 갑자기 이동하도록 할 수 없습니다. 그러나 (예를 들어) 장치 OS가 일시 중지되어 시뮬레이션을 일시 중지 해야 하는 순간이 있으므로 이 기능을 사용합니다. 플래그가 다음으로 설정된 경우 시뮬레이션을 일시 중지합니다. true 플래그가 false 다시.

참고: 이것은 비활성화된 경우에도 물리적 본체가 물리 세계에서 계속 계산 및 시뮬레이션되므로 방의 모든 인스턴스를 비활성화하려는 경우에 특히 유용합니다.

통사론:

physics_pause_enable(flag)

논쟁 유형 설명
flag This can be set to true to pause the simulation, or false to start it again.

보고:

예시:

if os_is_paused()
{
    physics_pause_enable(true);
    instance_deactivate_all(true);
    instance_create_layer(x, y, "Controllers", obj_PauseMenu);
}

위의 코드는 OS가 일시 중지되었는지 확인하고 일시 중지된 경우 물리 세계를 일시 중지하고 자체를 제외한 모든 것을 비활성화한 다음 일시 중지 메뉴 인스턴스를 만듭니다.
