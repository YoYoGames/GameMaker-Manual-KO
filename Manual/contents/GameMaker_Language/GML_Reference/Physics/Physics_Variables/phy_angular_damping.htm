phy_angular_damping

이 변수는 인스턴스의 각도 감쇠를 설정하는 데 사용하거나 현재 각도 감쇠를 가져오는 데 사용할 수 있습니다. 댐핑은 물리 활성화된 인스턴스가 갖는 각도 회전에 대한 "저항"의 양이며, 값이 낮으면 충돌 후 인스턴스가 더 빨리 회전할 수 있고(예:) 값이 높을수록 더 세게 밀고 느리게 회전해야 합니다.

통사론:

phy_angular_damping;

보고:

(single precision floating point value, or undefined if the instance is not physics enabled)

예시:

if place_meeting(phy_position_x, phy_position_y, obj_Water)
{
    phy_angular_damping = 10;
}
else
{
    phy_angular_damping = 3;
}

위의 코드는 호출 인스턴스와 "obj_Water" 인스턴스 간의 충돌을 확인하고 그에 따라 각도 감쇠를 변경합니다.
