phy_active

이 변수는 인스턴스가 현재 "활성" 상태인지 여부를 제어합니다. 로 설정 false 인스턴스가 물리 세계에 참여하는 것을 방지하고 다음으로 설정합니다. true 다시 참여하게 됩니다. 인스턴스가 여전히 화면에 표시되고 코드를 통해 변경할 수 있기 때문에 이것은 인스턴스를 비활성화하는 것과 동일하지 않습니다. 오히려 이 기능은 물리적 시뮬레이션에 참여하는 것을 방지할 뿐입니다.

통사론:

phy_active;

보고:

(or undefined if the instance is not physics enabled)

예시:

if keyboard_check_pressed(ord"P")
{
    global.Pause = !global.Pause;
    with (obj_Parent)
    {
        phy_active = !global.Pause;
    }
}

위의 코드는 문자 "P"의 키 누름을 감지한 다음 전역 변수 "Pause"를 다음에서 토글합니다. true 에게 false 그리고 다시. 그런 다음 이 변수를 사용하여 "obj_Parent" 변수에 인덱싱된 개체의 자식 인스턴스에서 물리학이 활성화되었는지 여부를 설정합니다.
