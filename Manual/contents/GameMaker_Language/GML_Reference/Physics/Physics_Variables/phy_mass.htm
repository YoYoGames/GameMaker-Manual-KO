phy_mass

이 읽기 전용 변수는 인스턴스의 질량을 킬로그램 단위 로 반환합니다. 이 값은 할당된 조명기의 표면적과 밀도 값을 기반으로 자동으로 계산되지만 기능을 사용하여 변경할 수 있습니다. physics_mass_properties().

통사론:

phy_mass;

보고:

(single precision floating point value, or undefined if the instance is not physics enabled)

예시:

if (phy_mass < other.phy_mass)
{
    instance_destroy();
}

위의 코드는 인스턴스와 다른 인스턴스의 충돌 이벤트로 각 인스턴스의 질량을 비교하여 질량이 작은 것을 파괴합니다.
