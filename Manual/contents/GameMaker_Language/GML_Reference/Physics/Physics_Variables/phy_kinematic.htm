phy_kinematic

이 읽기 전용 변수는 다음을 반환합니다. true 인스턴스가 운동학적 개체로 분류되는 경우 또는 false 그렇지 않은 경우. 운동학적 인스턴스는 질량이 무한대(밀도 0)지만 이동할 수 있는 인스턴스입니다. 따라서 인스턴스를 운동학적으로 만들려면 먼저 정적 인스턴스를 만든 다음 이동과 관련된 하나 이상의 인스턴스 변수를 설정합니다(예:phy_speed_x,phy_speed_y, 또는phy_angular_velocity )

통사론:

phy_kinematic;

보고:

(or undefined if the instance is not physics enabled)

예시:

if !phy_kinematic
{
    phy_speed_x = 5;
}

위의 코드는 인스턴스가 운동학적인지 확인하고 그렇지 않은 경우 수평 속도를 5로 설정합니다.
