phy_linear_damping

이 변수는 인스턴스의 선형 감쇠를 설정하는 데 사용하거나 현재 선형 감쇠를 가져오는 데 사용할 수 있습니다. 댐핑은 물리 활성화된 인스턴스가 가진 앞으로 이동에 대한 "저항"의 양으로, 값이 낮으면 인스턴스가 더 빠르게 이동하고 가속할 수 있고 값이 높을수록 더 강력한 푸시가 필요합니다.

통사론:

phy_linear_damping;

보고:

(single precision floating point value, or undefined if the instance is not physics enabled)

예시:

if place_meeting(phy_position_x, phy_position_y, obj_Water)
{
    phy_linear_damping = 10;
}
else
{
    phy_linear_damping = 3;
}

위의 코드는 호출 인스턴스와 "obj_Water" 인스턴스 간의 충돌을 확인하고 그에 따라 선형 감쇠를 변경합니다.
