phy_position_x

이 변수는 게임룸 물리 세계 내에서 인스턴스의 x 위치를 가져오거나 설정하는 데 사용할 수 있습니다. 연속 시뮬레이션을 방해하므로 이 변수를 직접 설정하여 인스턴스를 이동할 때 물리 세계에서 오류가 발생할 수 있습니다. 이 변수는 인스턴스 변수 x 와 물리적으로 동일합니다.

통사론:

phy_position_x;

보고:

(single precision floating point value, or undefined if the instance is not physics enabled)

예시:

xx = phy_position_x;
yy = phy_position_y;

위의 코드는 인스턴스 x와 y 위치를 두 개의 변수에 저장합니다.

뒤로: 물리학 변수
