phy_angular_velocity

이 변수는 인스턴스의 각속도를 설정하는 데 사용할 수 있고 초당 각도로 현재 각속도를 얻는 데 사용할 수 있으며 사용된 값은 양수(시계 방향 회전의 경우) 또는 음수(반시계 방향 회전의 경우)일 수 있습니다.. 이전에 정적이었던 인스턴스에 이것을 설정하면(즉, 밀도가 0임) 운동학적 개체가 되어 회전을 시작합니다.

통사론:

phy_angular_velocity;

보고:

(single precision floating point value, or undefined if the instance is not physics enabled)

예시:

if abs(phy_angular_velocity) > 0
{
    phy_angular_velocity -= sign(phy_angular_velocity) * 0.01;
}
else
{
    phy_angular_velocity = 0;
}

위의 코드는 인스턴스의 각속도를 확인하고 0이 아니면 값이 0이 될 때까지 매 단계마다 소량을 더하거나 뺍니다.
