phy_linear_velocity_x

이 변수는 인스턴스의 선형 속도 벡터의 x 구성 요소를 가져오거나 변경하는 데 사용할 수 있으며 초당 픽셀로 정의됩니다(단계당 픽셀은 참조phy_speed_x ). 정적 인스턴스(예: 밀도가 0인 인스턴스)에 대해 이것을 변경하면 운동학적 인스턴스로 바뀝니다.

통사론:

phy_linear_velocity_x;

보고:

(single precision floating point value, or undefined if the instance is not physics enabled)

예시:

if (phy_linear_velocity_x != 0)
{
    phy_linear_velocity_x = 0;
}

위의 코드는 선형 속도 벡터의 x 구성 요소를 확인하고 0 이외의 값이면 0으로 설정합니다.
