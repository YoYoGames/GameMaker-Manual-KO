phy_bullet

이 변수는 인스턴스가 매우 빠르게 움직이는지 여부를 정의합니다(예: 총알). 기본값은 false 하지만 로 설정하면 true 이것은 인스턴스가 감지되지 않은 상태로 다른 인스턴스를 통과하지 않도록 하기 위해 더 비싼 충돌 감지가 필요할 정도로 빠른 속도로 인스턴스가 이동할 것임을 GameMaker 에 알려줍니다.

통사론:

phy_bullet;

보고:

(or undefined if the instance is not physics enabled)

예시:

with (instance_create_layer(x, y, "Bullets", obj_Shoot))
{
    phy_bullet = true;
    physics_apply_local_impulse(0, 10, 0, 200);
}

위의 코드는 새 인스턴스를 만든 다음 y축을 따라 엄청난 충격을 주기 전에 물리 세계에서 "총알"로 정의합니다.
