phy_dynamic

동적 인스턴스는 물리 세계 내에서 완전히 시뮬레이션된 인스턴스이며 이 읽기 전용 변수는 다음을 반환합니다. true 확인 중인 인스턴스가 완전히 시뮬레이션된 경우 또는 false 그렇지 않은 경우

통사론:

phy_dynamic;

보고:

(or undefined if the instance is not physics enabled)

예시:

if other.phy_dynamic
{
    with (other)
    {
        var dir;
        dir = point_direction(x, y, other.x, other.y);
        physics_apply_impulse(x, y, x + lengthdir_x(100, dir), y + lengthdir_y(100, dir));
    }
}

위의 코드는 새 인스턴스를 만든 다음 y축을 따라 엄청난 충격을 주기 전에 물리 세계에서 "총알"로 정의합니다.
