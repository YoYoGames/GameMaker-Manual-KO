phy_rotation

이 변수는 설정 또는 가져오기와 유사하게 인스턴스 고정물의 각도를 도 단위로 가져오거나 설정하는 데 사용할 수 있습니다. image_angle. 그러나 물리 세계에서 회전은 일반 GameMaker 게임 세계와 반대 방식으로 계산됩니다. 즉, 벡터는 다음과 같이 기능합니다. point_direction() 반환 값을 수정해야 합니다(단순히 양수를 음수로 변경하면 해결됨).

통사론:

phy_rotation;

보고:

(single precision floating point value, or undefined if the instance is not physics enabled)

예시:

if (phy_speed_x > 0) || (phy_speed_y > 0)
{
    phy_rotation += sqrt(sqr(phy_speed_x) + sqr(phy_speed_y)) /10;
}

위의 코드는 선형 속도를 확인하고 벡터 중 하나가 0이 아니면 실제 속도를 계산하고 이를 사용하여 회전을 설정합니다.
