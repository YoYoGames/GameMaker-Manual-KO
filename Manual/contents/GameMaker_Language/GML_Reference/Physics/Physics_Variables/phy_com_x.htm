phy_com_x

이 읽기 전용 변수는 인스턴스 질량 중심의 x 위치를 반환합니다. 이것은 적절한 함수에 의해 정의된 인스턴스의 밀도, 관성 및 질량을 기반으로 자동으로 계산됩니다.

통사론:

phy_com_x;

보고:

(single precision floating point value, or undefined if the instance is not physics enabled)

예시:

xx = phy_com_x;
yy = phy_com_y;

위의 코드는 인스턴스 질량 중심의 x 및 y 위치에 두 개의 변수를 설정합니다.
