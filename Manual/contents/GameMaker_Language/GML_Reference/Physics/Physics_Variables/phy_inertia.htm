phy_inertia

관성은 무언가를 움직이기 시작하거나 멈추게 하는 것이 얼마나 어려운지를 측정한 것이므로 이 읽기 전용 변수의 값이 낮을수록 인스턴스를 움직이기 쉽게 설정할 수 있지만 값이 높을수록 시작하는 데 더 많은 힘이 필요합니다. 움직이는

통사론:

phy_inertia;

보고:

(single precision floating point value, or undefined if the instance is not physics enabled)

예시:

physics_mass_properties(70, -10, -10, phy_inertia);

위의 코드는 관성 값을 유지하면서 인스턴스의 질량과 질량 중심을 변경합니다.
