phy_fixed_rotation

이 변수는 인스턴스가 회전력의 영향을 받을 수 있는지 여부를 설정하는 데 사용할 수 있습니다(기본값은 false ). 로 설정하면 true, 외부 힘(코딩된 충격이나 힘 또는 충돌에서)은 인스턴스의 회전 값에 영향을 미치지 않으며 이는 다음을 사용하여 수동으로 설정해야 합니다.phy_rotation 변하기 쉬운

통사론:

phy_fixed_rotation;

보고:

(or undefined if the instance is not physics enabled)

예시:

phy_rotation = 0;
phy_fixed_rotation = true;

위의 코드는 인스턴스를 고정 회전으로 전환한 다음 회전 각도를 설정합니다.
