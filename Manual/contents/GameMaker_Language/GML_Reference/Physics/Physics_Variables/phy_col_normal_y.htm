phy_col_normal_y

이 읽기 전용 변수는 다음에 해당하는 충돌 법선의 y 구성요소를 반환합니다.phy_collision_y 배열 값. 각 접점에 대해 연관된 접점 법선이 있습니다(일반적으로 충돌의 모든 접점에 대해 동일한 법선). 이 접촉 법선은 충돌의 한 인스턴스에서 다른 인스턴스를 가리키는 단위 벡터이며, 예를 들어 충돌을 해결하기 위해 올바른 "밀기" 방향을 계산하는 데 사용할 수 있습니다.

참고: 이 변수는 물리 활성화 인스턴스의 충돌 이벤트에서만 사용할 수 있습니다.

통사론:

phy_col_normal_y;

보고:

(single precision floating point value, or undefined if the instance is not physics enabled)

예시:

repeat(5 + irandom(5))
{
    with (instance_create_layer(x, y, "Effects", obj_Debris)
    {
        physics_apply_local_impulse(0, 0, other.phy_col_normal_x[0], other.phy_col_normal_y[0]);
    }
}

위의 코드는 접촉 법선을 사용하여 두 물리 활성화 인스턴스 간의 충돌 이벤트에서 생성된 인스턴스의 이동 방향을 설정합니다.
