phy_sleeping

이 읽기 전용 변수는 인스턴스가 현재 "절전" 상태인지 여부를 반환합니다(true ) 여부(false ), "잠자는" 인스턴스는 물리적 시뮬레이션에 적극적으로 참여하지 않는 인스턴스입니다. GameMaker 는 인스턴스가 다른 인스턴스와 충돌하지 않고 정지 상태일 때 시뮬레이션 주기를 저장하기 위해 개체를 절전 모드로 전환합니다.

통사론:

phy_sleeping;

보고:

(or undefined if the instance is not physics enabled)

예시:

if phy_sleeping
{
    instance_destroy();
}

위의 코드는 객체가 활발히 시뮬레이션되고 있는지 확인하고 그렇지 않으면 소멸됩니다.
