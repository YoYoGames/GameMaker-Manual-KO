phy_speed

이 읽기 전용 변수는 단계당 픽셀로 정의된 물리 활성화 인스턴스의 현재 속도를 반환합니다. 이 값을 변경해야 하는 경우 변수를 사용하여 x 및 y 벡터를 변경해야 합니다.phy_speed_x 그리고 phy_speed_y.

통사론:

phy_speed;

보고:

(single precision floating point value, or undefined if the instance is not physics enabled)

예시:

if phy_speed > 10
{
    phy_linear_damping += 0.01;
}
else
{
    phy_linear_damping = 2;
}

위의 코드는 물리 활성화 인스턴스의 속도를 확인한 다음 반환된 값을 기반으로 선형 감쇠를 변경합니다.
