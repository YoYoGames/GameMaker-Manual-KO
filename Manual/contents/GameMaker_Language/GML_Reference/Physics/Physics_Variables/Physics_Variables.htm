물리 변수

"전통적인" 충돌 및 이동 시스템과 마찬가지로 물리 시스템에는 생성될 때 개체의 각 인스턴스에 할당된 일련의 인스턴스 변수 가 있습니다. 이러한 속성 중 일부는 GML에서 사용자가 정의할 수 있지만 대부분은 읽기 전용이며 여전히 충돌 이벤트에만 적용할 수 있습니다.

다음 변수는 모든 물리 가능 개체에 존재하며 인스턴스 및 스크립트 함수의 코드를 사용하여 읽기 및 쓰기(예: 변경)할 수 있습니다.

  • phy_active
  • phy_angular_velocity
  • phy_angular_damping
  • phy_linear_velocity_x
  • phy_linear_velocity_y
  • phy_linear_damping
  • phy_speed_x
  • phy_speed_y
  • phy_position_x
  • phy_position_y
  • phy_position_xprevious
  • phy_position_yprevious
  • phy_rotation
  • phy_fixed_rotation
  • phy_bullet

특정 변수는 읽기 전용 변수입니다. 즉, 해당 변수에 포함된 정보는 인스턴스 코드나 스크립트 기능을 통해 변경할 수 없습니다(그러나 GameMaker 에서 내부적으로 변경할 수 있으므로 상수 가 아닌 변수임). 다음 목록은 모든 물리 활성화 개체에 존재하는 읽기 전용 변수를 보여줍니다.

  • phy_speed
  • phy_com_x
  • phy_com_y
  • phy_dynamic
  • phy_kinematic
  • phy_inertia
  • phy_mass
  • phy_sleeping

마지막으로 물리 가능 객체에 존재 하고 충돌 이벤트에서만 생성되는 임시 변수가 많이 있습니다. 이러한 변수는 위에 나열된 것과 같으며 읽기 전용입니다.

  • phy_collision_points
  • phy_collision_x
  • phy_collision_y
  • phy_col_normal_x
  • phy_col_normal_y
