physics_test_overlap

이 기능은 물리적 몸체(예: 인스턴스의 고정물)가 회전하고 방의 주어진 위치에 배치 될 때 겹치거나 겹치는지 확인하는 데 사용할 수 있습니다. "angle" 인수는 검사할 위치에서 호출 인스턴스가 갖는(또는 가질) 회전 각도이고 "obj" 인수는 단일 인스턴스 ID 및 객체 인덱스 또는 키워드 all 또는 other 일 수 있습니다..

통사론:

physics_test_overlap(xpos, ypos, angle, obj);

논쟁 유형 설명
xpos 진짜 The x position in the room to check
ypos 진짜 The y position in the room to check
angle 진짜 The angle to check (of the calling instance)
obj 개체 자산 또는 인스턴스 ID The object to check for

보고:

Boolean

예시:

if physics_test_overlap(x+20, y-35, 0, obj_Bomb)
{
    alarm[0] = room_speed;
    ignited = true;
}

위의 코드는 물리 픽스처가 겹치는 위치를 확인하고 만약 있다면 변수와 알람을 설정합니다.
