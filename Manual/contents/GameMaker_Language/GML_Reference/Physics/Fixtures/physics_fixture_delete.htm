physics_fixture_delete

인스턴스가 조명기 속성을 유지하는 인스턴스에 고정물이 바인딩되면 특히 더 이상 다시 사용하지 않으려는 경우 이 기능을 사용하여 고정기를 메모리에서 제거할 수 있습니다. 더 이상 필요하지 않은 고정 장치를 제거하지 않으면 메모리 누수가 발생하여 속도가 느려지고 결국 게임이 중단될 수 있으므로 고정 장치를 정의하고 바인딩한 후에는 적절히 정리하도록 주의해야 합니다.

통사론:

physics_fixture_delete(fixture)

논쟁 유형 설명
fixture the fixture that is to be deleted from memory

보고:

예시:

var fix, inst;
fix = physics_fixture_create();
physics_fixture_set_circle_shape(fix, 16);
physics_fixture_set_density(fix, 1.0);
inst = instance_create_layer(x, y, "Instances", genericBodyObject);
physics_fixture_bind(fix, inst);
physics_fixture_delete(fix);

위의 코드는 고정물을 생성하고 그것의 인덱스를 변수 "fix"에 할당할 것입니다. 그런 다음 "inst" 변수에 저장된 인덱스로 생성된 인스턴스에 바인딩하기 전에 조명기의 모양과 밀도를 정의합니다. 마지막으로 픽스쳐는 더 이상 필요하지 않으므로 메모리 누수를 방지하기 위해 삭제됩니다.
