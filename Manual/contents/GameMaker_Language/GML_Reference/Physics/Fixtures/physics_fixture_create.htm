physics_fixture_create

조명기를 설정하는 첫 번째 단계는 이 기능으로 조명기를 만드는 것입니다. 반환 인덱스는 이 고정 장치를 정의하고 사용하는 데 사용되는 모든 추가 기능에서 사용할 변수에 저장해야 합니다.

통사론:

physics_fixture_create()

보고:

Physics Fixture ID

예시:

fix_Ball = physics_fixture_create();

위의 코드는 고정 장치를 만들고 "fix_Ball" 변수에 색인을 저장합니다.
