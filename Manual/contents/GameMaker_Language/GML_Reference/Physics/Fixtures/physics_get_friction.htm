physics_get_friction

다음을 사용하여 인스턴스에 조명기를 바인딩할 때physics_fixture_bind() 이것은 바인딩된 조명기에 대한 "id"를 반환합니다. 이 함수를 사용하면 언제든지 이 ID를 사용하여 바인딩된 고정 장치("기본" 고정 장치가 아님 )의 마찰 값을 얻을 수 있습니다.

통사론:

physics_get_friction(fixture)

논쟁 유형 설명
fixture 피직스 픽스처 ID the id of the bound fixture

보고:

Real

예시:

var fric = physics_get_friction(fix_id);
physics_set_friction(fix_id, fric + 0.1);

위의 코드는 인스턴스의 바인딩된 물리적 속성에 대한 현재 마찰 값을 가져온 다음 다른 값으로 설정합니다.
