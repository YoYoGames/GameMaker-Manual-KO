physics_fixture_set_chain_shape

이 기능은 "체인" 조명기 모양을 정의합니다. 체인 고정 장치는 여러 점으로 구성되며 모서리 모양을 사용하여 서로 연결되어 결합됩니다. 함수 자체는 조명기의 인덱스(ID)를 취하고 조명기에 루프 여부를 알릴 수 있습니다. 이것을 다음으로 설정 true 로 설정하는 동안 가장자리가 있는 첫 번째 점에 마지막 점을 연결합니다. false 하지 않을 것이다. 이 함수는 체인을 만드는 데 필요한 포인트를 수락하도록 조명기를 준비하고 다음을 사용하여 이 함수를 호출한 후에 추가해야 합니다.physics_fixture_add_point(), 폴리곤 설비를 구축할 때와 마찬가지로

체인 고정물을 생성할 때 최소 2점을 부여해야 하지만 완성된 고정물을 구성하기 위해 추가할 수 있는 후속 점의 수에는 제한이 없습니다.

통사론:

physics_fixture_set_chain_shape(fixture, loop)

논쟁 유형 설명
fixture The index of the fixture
loop Whether to close the chain (true ) or not (false ).

보고:

예시:

var xx = -100;
var yy = room_height / 2;
var fix = physics_fixture_create(); physics_fixture_set_chain_shape(fix, false);
for (var i = 0; i < 10; i++;)
{
    physics_fixture_add_point(_fix, _xx, _yy);
    _xx += 50 + random(150);
    _yy = (room_height / 2) - 250 + irandom(500);
}
physics_fixture_bind(fix, id);

위의 코드는 방의 길이에 걸쳐 다양한 높이를 가진 "가장자리" 설비 라인을 생성할 것입니다.
