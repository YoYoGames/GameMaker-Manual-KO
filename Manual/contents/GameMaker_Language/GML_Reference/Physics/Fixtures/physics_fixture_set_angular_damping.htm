physics_fixture_set_angular_damping

"현실 세계"에서 회전하는 물체에 대해 생각한다면 모터가 있거나 우주에 있지 않는 한 외부 힘(주변 공기와의 마찰과 같은)의 영향으로 인해 시간이 지남에 따라 속도가 느려집니다. 우리는 기능을 사용할 수 있습니다 physics_fixture_set_angular_damping() GameMaker 에서 이 효과를 시뮬레이션하고 물리 세계에서 인스턴스의 회전 속도를 줄이려면 회전하는 인스턴스가 계속 무한대로 회전합니다. 감쇠 매개변수는 0에서 무한대 사이여야 하며 0은 감쇠가 없음을 의미하고 무한대는 전체 감쇠를 의미합니다. 일반적으로 0과 1 사이의 감쇠 값을 사용하지만 필요한 경우 음이 아닌 값을 사용할 수 있습니다.

통사론:

physics_fixture_set_angular_damping(fixture, damping)

논쟁 유형 설명
fixture the index of the fixture
damping the angular damping of the fixture, usually between 0 and 1

보고:

예시:

physics_fixture_set_angular_damping(fix_Ball, 0.1);

위의 코드는 "fix_ball"에 인덱싱된 조명기의 각도 감쇠를 0.1로 설정합니다.
