physics_set_density

다음을 사용하여 인스턴스에 조명기를 바인딩할 때physics_fixture_bind() 이것은 바인딩된 조명기에 대한 "id"를 반환합니다. 이 기능을 사용하여 언제든지 "base" 조명기가 아닌 바인딩된 조명기의 밀도 값을 설정하기 위해 이 ID를 사용할 수 있습니다.

통사론:

physics_set_density(fixture, density)

논쟁 유형 설명
fixture the id of the bound fixture
density the new density value to apply

보고:

예시:

var density = physics_get_density(fix_id);
physics_set_density(fix_id, density - 0.1);

위의 코드는 인스턴스의 바인딩된 물리적 속성에 대한 현재 밀도 값을 가져온 다음 다른 값으로 설정합니다.
