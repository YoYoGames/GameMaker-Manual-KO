physics_remove_fixture

이 기능은 인스턴스에서 조명기를 제거(또는 "바인딩 해제")합니다. 바인딩된 조명기의 고유한 "id"가 필요합니다(함수에 의해 반환됨).physics_fixture_bind() 그러면 인스턴스에 대해 현재 정의된 모든 물리적 속성이 제거되어 새 조명기를 재정의하고 인스턴스에 바인딩할 수 있습니다. 이러한 방식으로 인스턴스를 삭제하고 다시 생성하지 않고도 인스턴스의 물리적 속성을 변경할 수 있습니다.

통사론:

physics_remove_fixture(id, fixture)

논쟁 유형 설명
id 인스턴스 ID The ID of the instance to remove the fixture from
fixture 물리 픽스처 ID The ID of the fixture that is to be removed from the instance

보고:

N/A

예시:

physics_remove_fixture(id, my_fix);

위의 코드는 인스턴스에서 변수 "my_fix"에 저장된 "id"를 가진 조명기를 제거합니다.
