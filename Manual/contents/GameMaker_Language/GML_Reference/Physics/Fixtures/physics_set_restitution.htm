physics_set_restitution

다음을 사용하여 인스턴스에 조명기를 바인딩할 때physics_fixture_bind() 이것은 바인딩된 조명기에 대한 "id"를 반환합니다. 이 기능을 사용하여 언제든지 "base" 조명기가 아닌 바인딩된 조명기의 복원 값을 설정하기 위해 이 ID를 사용할 수 있습니다. 배상은 일반적으로 0과 1 사이의 값으로 설정되지만 결과를 예측할 수 없는 경우에도 필요한 경우 더 높은 값을 사용할 수 있습니다.

통사론:

physics_set_restitution(fixture, restitution)

논쟁 유형 설명
fixture the id of the bound fixture
restitution the new restitution value to apply

보고:

예시:

var rest = physics_get_restitution(fix_id);
physics_set_restitution(fix_id, rest * 2);

위의 코드는 인스턴스의 바인딩된 물리적 속성에 대한 현재 복원 값을 가져온 다음 다른 값으로 설정합니다.
