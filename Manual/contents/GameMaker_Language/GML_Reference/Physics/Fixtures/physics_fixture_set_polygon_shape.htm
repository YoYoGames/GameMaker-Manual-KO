physics_fixture_set_polygon_shape

이 기능은 조명기의 다각형 모양을 설정하지만 다음을 사용해야 합니다.physics_fixture_add_point() 조명기의 원점을 기준으로 이 다각형의 모양을 실제로 정의합니다. 조명기가 인스턴스에 바인딩되면 다각형이 닫힙니다. 점을 정의하기 전에 이 함수를 호출 해야 하며, 인스턴스에 바인딩하기 전에 폴리곤에 대해 최소한 세 개의 점이 정의되어 있어야 합니다. 그렇지 않으면 오류가 발생합니다.

통사론:

physics_fixture_set_polygon_shape(fixture)

논쟁 유형 설명
fixture the index of the fixture

보고:

예시:

physics_fixture_set_polygon_shape(fix_Ship);
physics_fixture_add_point(fix_Ship, 0,0);
physics_fixture_add_point(fix_Ship, -40, 100);
physics_fixture_add_point(fix_Ship, 40, 100);

위의 코드는 "fix_Ship" 변수에 인덱싱된 고정 장치에 다각형 모양을 적용한 다음 삼각형 모양을 만들기 위해 세 개의 점을 정의합니다.
