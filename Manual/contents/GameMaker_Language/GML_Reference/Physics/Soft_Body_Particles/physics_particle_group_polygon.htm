physics_particle_group_polygon

이 기능은 생성되는 입자 그룹의 모양을 설정합니다. 먼저 함수를 사용하여 그룹 정의를 시작해야 합니다.physics_particle_group_begin(), 그런 다음 이 기능을 사용하여 다각형 모양을 정의하는 데 사용할 대략적인 점을 추가할 것임을 GameMaker 에 알릴 것입니다.physics_particle_set_radius() 물리학 시뮬레이션이 정의된 모양에 가능한 한 많은 입자를 "맞추기"를 시도하기 때문에 기능입니다.

그런 다음 함수를 호출하는 다각형의 점을 정의합니다.physics_particle_group_add_point(), 각 방의 x/y 위치 설정(최소 3, 최대 8, 오목으로 정의할 수 있지만 모양 내부의 점은 무시됨). 마지막으로 전화해야합니다physics_particle_group_end() 방에 입자 그룹을 만듭니다.

통사론:

physics_particle_group_polygon()

보고:

예시:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_polygon();
physics_particle_group_add_point(200, 200);
physics_particle_group_add_point(300, 300);
physics_particle_group_add_point(100, 300);
mLastGroup = physics_particle_group_end();

위의 코드는 입자 유형에 대한 플래그와 입자 그룹 속성을 변수에 저장한 다음 이를 사용하여 마우스 위치에 3면의 다각형 입자 그룹을 만듭니다.
