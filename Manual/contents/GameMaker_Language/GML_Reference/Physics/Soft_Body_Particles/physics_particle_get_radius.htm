physics_particle_get_radius

이 기능을 사용하면 물리 시뮬레이션에서 입자에 대한 현재 반경(픽셀 단위)을 찾을 수 있습니다(이 값은 다음을 사용하여 설정할 수 있습니다.physics_particle_set_radius() ).

통사론:

physics_particle_get_radius()

보고:

예시:

if (physics_particle_get_radius() < 32)
{
    physics_particle_set_radius(physics_particle_get_radius() + 1);
}

위의 코드는 시뮬레이션에서 입자의 현재 반경을 확인하고 32보다 작으면 크기를 1만큼 증가시킵니다.
