physics_particle_set_max_count

이 기능은 물리 시뮬레이션에서 허용되는 총 입자 수를 설정합니다. 이 값을 0으로 설정하면 생성되는 입자에 상한선이 없으며 다른 값으로 설정하면 총 입자 수가 값과 같을 때 더 이상 생성되는 입자가 중지됩니다. 시뮬레이션에서 만들 수 있는 최소 입자 수는 128개이므로 이 값을 더 낮게 설정해도 효과가 없습니다(0으로 설정하지 않는 한).

통사론:

physics_particle_set_max_count(count)

논쟁 유형 설명
count The maximum number of particles to permit.

보고:

예시:

physics_particle_set_max_count(500);

위의 코드는 물리 시뮬레이션의 파티클 캡을 500으로 설정합니다.
