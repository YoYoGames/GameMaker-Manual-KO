physics_particle_set_group_flags

이 기능을 사용하면 입자 그룹에 대한 그룹 플래그를 변경할 수 있습니다. 그만큼 group 값은 함수를 사용하여 입자 그룹을 만들 때 반환된 값입니다.physics_particle_group_end(), 플래그는 다음 상수 조합의 반환 값입니다.

일정한 설명
phy_particle_group_flag_solid 단단한 입자 그룹은 다른 설비가 그 안에 머무르는 것을 방지합니다. 무언가가 그것을 관통하면 고체 입자 그룹은 문제가 되는 고정물을 표면으로 다시 밀어내어 고체 입자 그룹이 특히 강한 반발력을 갖도록 합니다.
phy_particle_group_flag_rigid 강성 입자 그룹은 다른 설비와 충돌하더라도 모양이 변경되지 않는 그룹입니다.

이러한 그룹 플래그는 비트 마스킹을 사용하여 최종 출력 값을 생성한 다음 파티클 플래그로 설정한 속성과 별도로 그룹의 다양한 기본 속성을 설정하도록 검사합니다. 비트 단위 또는 "|"를 사용하여 이러한 상수 중 하나 또는 둘 다를 사용할 수 없습니다. 적절한 비트를 마스킹합니다(아래 예와 같이).

통사론:

physics_particle_set_group_flags(group, flags)

논쟁 유형 설명
group 물리 입자 그룹 ID The particle group to set.
flags 물리 입자 그룹 플래그 상수 (들) The flags to set on the particle group.

보고:

N/A

예시:

var flags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_set_group_flags(group, flags);

위의 코드는 플래그 값을 저장할 변수를 만든 다음 변수 "group"에 의해 인덱싱된 그룹에서 생성된 모든 입자의 플래그를 설정하는 데 사용합니다.
