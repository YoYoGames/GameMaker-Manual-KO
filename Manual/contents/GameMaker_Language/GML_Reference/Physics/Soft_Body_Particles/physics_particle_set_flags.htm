physics_particle_set_flags

이 기능을 사용하면 개별 입자에 대한 입자 플래그를 변경할 수 있습니다. 그만큼 index 값은 함수에서 반환된 입자 ID입니다.physics_particle_create(), 플래그는 다음 상수 조합의 반환 값입니다.

일정한 설명
phy_particle_flag_water 소프트 바디 입자의 기본 속성입니다.
phy_particle_flag_zombie 좀비 입자는 이러한 방식으로 플래그가 지정된 다른 모든 입자와 함께 한 번에 파괴되는 입자입니다.
phy_particle_flag_wall 이것은 파티클을 static 으로 정의하며, 기본적으로 물리 시뮬레이션에서 움직일 수 없는 객체로 생성합니다. 이는 무엇과 충돌하더라도 고정된 위치에 남아 있기 때문입니다. 밀도를 0으로 설정하는 대신 이 플래그를 사용해야 합니다.
phy_particle_flag_spring 스프링 입자는 스프링에 의해 서로 붙는 듯한 효과를 냅니다. 이 플래그로 생성된 입자는 쌍으로 "연결"되며 각 입자는 생성 당시 가장 가까웠던 입자에 연결됩니다. 한 번 짝을 이룬 입자는 "파트너"를 변경하지 않으며 외부 힘이 서로를 멀리 끌어당길수록 외부 힘이 제거될 때 충돌하는 힘이 커집니다. 쌍을 이루는 입자가 서로 얼마나 멀리 떨어져 있더라도 이들 사이의 연결은 끊어지지 않습니다.
phy_particle_flag_elastic 탄성 입자는 물리 시뮬레이션에서 다른 강체와 충돌할 때 변형되고 바운스될 수도 있습니다.
phy_particle_flag_viscous 점성 입자는 오일과 같이 "달라붙는" 또는 "끈적임"을 나타내는 입자입니다. 점성 입자가 덩어리져 더 많이 붙습니다.
phy_particle_flag_powder 분말 입자는 모래나 먼지에서 볼 수 있는 것과 같은 산란 효과를 생성합니다.
phy_particle_flag_tensile 인장 입자는 표면 장력의 효과 또는 액체 표면의 팽팽한 곡률을 생성하는 데 사용됩니다. 예를 들어 물방울에서 볼 수 있는 표면 장력을 만드는 데 사용할 수 있습니다. 장력이 끊어지면 입자는 마치 탄성이 있는 것처럼 튕기지만 계속해서 서로를 끌어당깁니다. 결과적으로 입자는 바운스될 때 클러스터를 형성하는 경향이 있습니다.
phy_particle_flag_colourmixing 색상 혼합 입자는 충돌하는 다른 입자의 색상 중 일부를 취합니다. 충돌하는 두 입자 중 하나만 색상 혼합 입자인 경우 다른 입자는 충돌 전 색상을 유지합니다.

이 플래그는 비트 마스킹을 사용하여 최종 출력 값을 만든 다음 파티클의 다른 기본 속성을 설정하기 위해 검사됩니다(기본 속성은 항상 phy_particle_flag_water ). 예를 들어, 표면 장력이 있는 점성 액체를 시뮬레이션하려면 비트 또는 "|" 아래 예와 같이 적절한 비트를 마스킹합니다. 이러한 방식으로 원하는 경우 생성된 각 개별 입자에 대해 다른 속성(전역 속성 제외)을 설정할 수 있습니다.

통사론:

physics_particle_set_flags(index, flags)

논쟁 유형 설명
index 물리학 입자 ID The index of the particle.
flags 물리학 입자 플래그 상수 (들) The flags to set on the particle.

보고:

Real

예시:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_set_flags(p, flags);

위의 코드는 플래그 값을 저장할 변수를 만든 다음 변수 "p"에 저장된 인덱스를 사용하여 이전에 생성된 입자에 플래그를 설정하는 데 사용합니다.
