physics_particle_get_data_particle

이 함수는 체크된 주어진 플래그를 사용하여 물리 시뮬레이션에서 단일 입자에 대한 다양한 정보를 반환합니다. 입자 인덱스(해당 ID)는 함수에 의해 반환된 것입니다.physics_particle_create(), 그리고 사용된 버퍼는 함수를 사용하여 이전에 생성되어야 합니다.buffer_create(). 크기는 예상되는 반환 데이터의 크기와 비슷한 "성장" 유형이어야 합니다. 플래그 자체는 아래 주어진 상수를 사용하여 설정되며 비트 단위 또는 "|" 원하는 정보를 얻기 위해 단일 플래그 값을 생성합니다.

일정한 설명 데이터 형식
phy_particle_data_flag_typeflags 입자에 대한 플래그 값입니다. buffer_u32
phy_particle_data_flag_position 입자의 x 및 y 위치입니다. 2 x buffer_f32
phy_particle_data_flag_velocity 수평 및 수직 속도. 2 x buffer_f32
phy_particle_data_flag_colour 색상 및 알파 값(16진수). buffer_f32
phy_particle_data_flag_category 입자 범주(입자를 만들 때 정의됨). buffer_u32

통사론:

physics_particle_get_data_particle(ind, buffer, flags)

논쟁 유형 설명
ind 물리학 입자 ID The index (ID) of the particle to get the data from.
buffer 버퍼 ID The (previously created) buffer to use to store the data.
flags 물리 입자 데이터 플래그 상수 (들) The flags to use to extract data about specific particle types.

보고:

N/A

예시:

var part = physics_particle_create(flags, x, y, x_vel, y_vel, c_white, 1, 1)
var flags = phy_particle_data_flag_position | phy_particle_data_flag_velocity;
var buffer = buffer_create(16, buffer_grow, 4);
physics_particle_get_data_particle(part, buffer, flags);
px = buffer_read(buffer, buffer_f32);
py = buffer_read(buffer, buffer_f32);
pvelx = buffer_read(buffer, buffer_f32);
pvely = buffer_read(buffer, buffer_f32);
buffer_delete(buffer);

위의 코드는 "part" 변수에 의해 인덱싱된 입자의 위치와 속도를 가져오고 데이터를 여러 변수에 저장합니다.
