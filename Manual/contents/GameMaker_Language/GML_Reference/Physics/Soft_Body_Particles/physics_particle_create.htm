physics_particle_create

이 기능을 사용하면 게임룸 내 어디에서나 단일 입자를 생성하여 특정 플래그와 속성을 설정할 수 있습니다. "플래그"는 다음 상수 조합의 반환 값입니다.

물리학 입자 플래그 상수
일정한 설명
phy_particle_flag_water 소프트 바디 입자의 기본 속성입니다.
phy_particle_flag_zombie 좀비 입자는 이러한 방식으로 플래그가 지정된 다른 모든 입자와 함께 한 단계 후에 파괴되는 입자입니다.
phy_particle_flag_wall 이것은 파티클을 static 으로 정의하며, 기본적으로 물리 시뮬레이션에서 움직일 수 없는 객체로 생성합니다. 이는 무엇과 충돌하더라도 고정된 위치에 남아 있기 때문입니다. 밀도를 0으로 설정하는 대신 이 플래그를 사용해야 합니다.
phy_particle_flag_spring 스프링 입자는 스프링에 의해 서로 붙는 듯한 효과를 냅니다. 이 플래그로 생성된 입자는 쌍으로 "연결"되며 각 입자는 생성 당시 가장 가까웠던 입자에 연결됩니다. 한 번 짝을 이룬 입자는 "파트너"를 변경하지 않으며 외부 힘이 서로를 멀리 끌어당길수록 외부 힘이 제거될 때 충돌하는 힘이 커집니다. 쌍을 이루는 입자가 서로 얼마나 멀리 떨어져 있더라도 이들 사이의 연결은 끊어지지 않습니다.
phy_particle_flag_elastic 탄성 입자는 물리 시뮬레이션에서 다른 강체와 충돌할 때 변형되고 바운스될 수도 있습니다.
phy_particle_flag_viscous 점성 입자는 오일과 같이 "달라붙는" 또는 "끈적임"을 나타내는 입자입니다. 점성 입자가 덩어리져 더 많이 붙습니다.
phy_particle_flag_powder 분말 입자는 모래나 먼지에서 볼 수 있는 것과 같은 산란 효과를 생성합니다.
phy_particle_flag_tensile 인장 입자는 표면 장력의 효과 또는 액체 표면의 팽팽한 곡률을 생성하는 데 사용됩니다. 예를 들어 물방울에서 볼 수 있는 표면 장력을 만드는 데 사용할 수 있습니다. 장력이 끊어지면 입자는 탄성이 있는 것처럼 튕기지만 계속해서 서로를 끌어당깁니다. 결과적으로 입자는 바운스될 때 클러스터를 형성하는 경향이 있습니다.
phy_particle_flag_colourmixing 색상 혼합 입자는 충돌하는 다른 입자의 색상 중 일부를 취합니다. 충돌하는 두 입자 중 하나만 색상 혼합 입자인 경우 다른 입자는 충돌 전 색상을 유지합니다.

이 플래그는 비트 마스킹을 사용하여 최종 출력 값을 만든 다음 파티클의 다른 기본 속성을 설정하기 위해 검사됩니다(기본 속성은 항상 phy_particle_flag_water ). 예를 들어, 표면 장력이 있는 점성 액체를 시뮬레이션하려면 비트 또는 "|" 아래 예와 같이 적절한 비트에 플래그를 지정합니다. 이러한 방식으로 생성된 각 개별 입자에 대해 다른 속성(전역 속성 제외)을 설정할 수 있습니다.

플래그 외에 파티클을 생성할 방의 위치, 초기 수평 및 수직 속도, 초기 색상 및 알파, 사용자 범주 를 설정할 수도 있습니다. 색상 및 알파 값은 함수를 사용하여 입자를 그릴 때만 사용됩니다.physics_particle_draw(), 그러나 범주 값은 다른 여러 기능에서 사용됩니다. 이 값은 유사한 속성을 갖고자 하는 각 입자에 부여하는 임의의 정수 값이며, 나중에 해당 특정 범주 및 기타 여러 항목에 대해 플래그를 설정하는 데 사용할 수 있습니다. 예를 들어, "물"과 "기름" 입자를 정의한 경우 "물"로 플래그가 지정된 모든 입자에 1의 사용자 범주를 제공하고 "기름"으로 생성된 모든 입자에 2를 부여하여 다음을 허용합니다. 나중에 단일 범주를 대상으로 지정하여 해당 플래그를 변경하거나 특정 데이터를 얻을 수 있습니다.

참고: 시뮬레이션에서 예약하고 다른 기능(예: 그리기)의 모든 범주를 선택하는 데 사용되는 0을 제외하고 범주에 정수 값을 사용할 수 있습니다.

이 함수는 파티클을 직접 참조하는 다른 함수에서 사용할 변수에 저장할 수 있는 파티클의 고유 인덱스(또는 ID) 값을 반환합니다.

통사론:

physics_particle_create(flags, x, y, xv, yv, col, alpha, category)

논쟁 유형 설명
flags 물리 입자 플래그 상수 (들) The flags to set on the particle.
x 진짜 The x position to create the particle at.
y 진짜 The y position to create the particle at.
xv 진짜 The initial horizontal velocity.
yv 진짜 The initial vertical velocity.
col 색깔 The base colour to use for the particle.
alpha 진짜 The base alpha to use for the particle.
category 진짜 The user defined category to which the particle belongs.

보고:

Physics Particle ID

예시:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var x_vel = lengthdir_x(5, image_angle);
var y_vel = lengthdir_y(5, image_angle);
var p = physics_particle_create(flags, x, y, x_vel, y_vel, c_white, 1, 1);

위의 코드는 파티클 유형과 초기 속도를 설정하는 변수를 생성한 다음, 이를 사용하여 인스턴스 x/y 위치에 파티클을 생성합니다.
