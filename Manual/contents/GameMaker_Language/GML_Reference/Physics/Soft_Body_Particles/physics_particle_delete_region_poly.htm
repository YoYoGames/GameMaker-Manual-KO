physics_particle_delete_region_poly

이 기능을 사용하면 현재 방의 물리 시뮬레이션에서 정의된 다각형 영역의 경계 내에 있는 모든 입자를 삭제(제거)할 수 있습니다. 함수는 (이전에 생성된)ds_list() 폴리곤의 각 점의 x/y 위치를 포함하며 목록의 짝수 위치는 x 좌표이고 홀수 위치는 y 좌표입니다. 즉, 삼각형의 경우 목록에는 항목 0이 있는 6개의 항목이 있습니다., 2, 4는 x 좌표이고 1, 3, 5는 y 좌표입니다. 그런 다음 폴리곤은 정의된 영역에 속하는 모든 입자를 삭제하는 데 사용됩니다.

정의된 폴리곤 에는 최소 3개의 점이 있어야 하고 최대 8개가 있어야 합니다.

통사론:

physics_particle_delete_region_poly(pointList)

논쟁 유형 설명
pointList A ds_list of points to use to create the polygon.

보고:

예시:

var list = ds_list_create(p_list);
for (var i = 0; i < 5; i ++;)
{
    ds_list_add(p_list, mx[i]);
    ds_list_add(p_list, my[i]);
}
physics_particle_delete_region_poly(p_list);
ds_list_destroy(p_list);

위의 코드는 추가된 점에 의해 정의된 다각형 영역에서 발견된 모든 입자를 삭제합니다. DS List.
