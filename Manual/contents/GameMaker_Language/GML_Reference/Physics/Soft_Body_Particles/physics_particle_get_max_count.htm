physics_particle_get_max_count

이 기능을 사용하면 물리 시뮬레이션에서 허용된 입자에 대한 현재 상한값이 무엇인지 확인할 수 있습니다(이 값은 다음을 사용하여 설정할 수 있습니다.physics_particle_set_max_count() ).

통사론:

physics_particle_get_max_count()

보고:

예시:

if (physics_particle_count() < physics_particle_get_max_count())
{
    physics_particle_create(0, x, y, 0, 0, c_white, 1, 1)
}

위의 코드는 방에 허용된 최대 입자 수보다 적은지 확인하고, 그렇다면 하나 더 만듭니다.
