physics_particle_get_density

이 기능을 사용하면 물리 시뮬레이션에서 입자에 대한 현재 밀도가 무엇인지 알 수 있습니다(이 값은 다음을 사용하여 설정할 수 있습니다.physics_particle_set_density() ).

통사론:

physics_particle_get_damping()

보고:

예시:

if (physics_particle_get_density() < 1)
{
    physics_particle_set_density(physics_particle_get_density() + 0.01);
}

위의 코드는 시스템의 모든 입자에 대한 현재 밀도 값을 확인하고 1보다 작으면 0.01을 추가합니다.
