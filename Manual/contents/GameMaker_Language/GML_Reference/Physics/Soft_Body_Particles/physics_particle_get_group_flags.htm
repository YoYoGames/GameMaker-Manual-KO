physics_particle_get_group_flags

이 기능을 사용하면 입자 그룹에 대한 그룹 플래그를 검색할 수 있습니다. 그만큼 group 값은 함수를 사용하여 입자 그룹을 만들 때 반환된 값입니다.physics_particle_group_end(), 그리고 함수는 현재 설정된 플래그의 결합된 값인 값을 반환합니다.

통사론:

physics_particle_get_group_flags(group)

논쟁 유형 설명
group 물리 입자 그룹 ID The particle group to get.

보고:

Real

예시:

var flags = physics_particle_get_group_flags(group, flags);
if flags != (phy_particle_group_flag_solid | phy_particle_group_flag_rigid)
{
    flags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
    physics_particle_set_group_flags(group, flags);
}

위의 코드는 플래그 값을 저장할 변수를 만든 다음 변수 "group"에 인덱싱된 그룹의 플래그를 확인하는 데 사용합니다. 동일하지 않은 경우 그룹은 이러한 플래그로 설정됩니다.
