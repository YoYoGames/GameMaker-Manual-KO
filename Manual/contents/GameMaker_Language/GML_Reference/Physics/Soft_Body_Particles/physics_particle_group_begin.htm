physics_particle_group_begin

이 기능을 사용하여 방에 입자 그룹을 만들 수 있습니다. 파티클 그룹은 특정 형태를 만드는 데 필요한 파티클이 한 번에 그룹으로 생성되어 사용된 플래그에 의해 설정된 다양한 속성으로 시뮬레이션된 소프트 바디를 만들 수 있는 그룹입니다. "플래그"는 입자 상수 조합의 반환 값입니다(함수 참조physics_particle_create() 자세한 내용은) "groupFlags"는 다음 그룹 상수 의 조합입니다(플래그가 없는 경우 0).

물리학 입자 그룹 플래그 상수
일정한 설명
phy_particle_group_flag_solid 단단한 입자 그룹은 다른 설비가 그 안에 머무르는 것을 방지합니다. 무언가가 그것을 관통하면 고체 입자 그룹은 문제가 되는 고정물을 표면으로 다시 밀어내어 고체 입자 그룹이 특히 강한 반발력을 갖도록 합니다.
phy_particle_group_flag_rigid 강성 입자 그룹은 다른 설비와 충돌하더라도 모양이 변경되지 않는 그룹입니다.

이러한 그룹 플래그는 비트 마스킹을 사용하여 최종 출력 값을 생성한 다음 파티클 플래그로 설정한 속성과 별도로 그룹의 다양한 기본 속성을 설정하도록 검사합니다. 비트 단위 또는 "|"를 사용하여 이러한 상수 중 하나 또는 둘 다를 사용할 수 없습니다. 적절한 비트를 마스킹합니다(아래 예와 같이).

플래그 외에 방의 위치를 설정하여 파티클 그룹을 생성할 수도 있습니다(지정된 위치는 완성된 그룹의 질량 중심이 됨), 초기 수평 및 수직 속도 및 각속도 그룹을 사용하여 이동하고 회전하는 그룹을 만들 수 있습니다.

함수를 사용하여 입자를 그릴 때 사용되는 값으로 그룹에 있는 모든 입자의 초기 색상과 알파를 설정할 수도 있습니다.physics_particle_draw(). 그런 다음 그룹에 있는 입자의 응집력을 설정합니다. 이는 서로 얼마나 강하게 "붙는지"를 나타내는 또 다른 방법입니다. 강도는 0에서 1 사이의 값일 수 있으며, 여기서 0은 응집력이 없고 1은 최대 응집력입니다.

마지막으로 입자 그룹에 대한 범주를 지정해야 합니다. 이 값은 유사한 속성을 가지려는 각 입자 그룹에 지정하는 임의의 정수 값이며 나중에 해당 특정 범주 및 기타 여러 항목에 대해 플래그를 설정하는 데 사용할 수 있습니다.

참고: 시뮬레이션에서 예약하고 다른 기능(예: 그리기)의 모든 범주를 선택하는 데 사용되는 0을 제외하고 범주에 정수 값을 사용할 수 있습니다.

이 함수를 호출해도 물리실에 그룹이 생성되지 않는다는 점에 유의하는 것이 중요합니다. 이 기능은 단순히 프로세스를 시작하고 그룹의 기본 입자 속성을 설정합니다. 실제로 입자 그룹을 생성하려면 양식 (아래 예 참조)을 제공한 다음 함수를 호출해야 합니다.physics_particle_group_end(), 이 시점에서 생성됩니다.

참고: 설정한 입자 유형 플래그는 입자 그룹의 응집력에도 영향을 미치므로 입자에 물 유형으로 플래그를 지정하면(예:) 그룹이 다른 고정 장치와 충돌할 때 분리되지만 입자를 스프링 유형으로 플래그 지정하면 충돌 시 응집력을 유지합니다.

통사론:

physics_particle_group_begin(flags, groupflags, x, y, ang, xv, yv, ang_velocity, col, alpha, strength, category)

논쟁 유형 설명
flags 물리학 입자 플래그 상수 (들) The flags to set on the particle.
groupflags 물리 입자 그룹 플래그 상수 (들) The group flags to set on the particle group (see table at the top).
x 진짜 The x position to create the particle group at.
y 진짜 The y position to create the particle group at.
ang 진짜 The angle (in degrees) to rotate the group to on create.
xv 진짜 The initial horizontal velocity.
yv 진짜 The initial vertical velocity.
ang_velocity 진짜 The initial angular velocity.
col 색깔 The base colour to use for the particle group.
alpha 진짜 The base alpha to use for the particle group.
strength 진짜 Set the cohesion strength between particles in the group.
category 진짜 The user defined category to which the group belongs.

N/A

예시:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

위의 코드는 입자 유형에 대한 플래그와 입자 그룹 속성을 변수에 저장한 다음 이를 사용하여 마우스 위치에서 반경 100px의 원형 입자 그룹을 만듭니다.
