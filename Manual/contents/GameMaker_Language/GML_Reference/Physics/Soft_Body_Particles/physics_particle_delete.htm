physics_particle_delete

이 기능을 사용하면 현재 방의 물리 시뮬레이션에서 입자를 삭제(제거)할 수 있습니다. 함수는 함수에서 반환된 삭제할 입자의 고유 ID를 사용합니다.physics_particle_create().

통사론:

physics_particle_group_delete(ind)

논쟁 유형 설명
ind 물리학 입자 ID The index (ID) of the particle to delete.

보고:

N/A

예시:

physics_particle_delete(part);

위의 코드는 시뮬레이션에서 변수 "part"에 저장된 ID를 가진 입자를 삭제합니다.
