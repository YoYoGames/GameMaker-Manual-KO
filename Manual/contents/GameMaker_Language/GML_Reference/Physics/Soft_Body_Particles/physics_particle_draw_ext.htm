physics_particle_draw_ext

이 함수는 동일한 사용자 정의 범주 값을 공유하고 마스크에 대해 설정된 것과 일치하는 플래그가 있는 모든 입자의 위치에 스프라이트를 그립니다. 마스크 값은 입자를 생성할 때 플래그를 정의하는 것과 동일한 방식으로 정의됩니다(함수 참조physics_particle_create() 예를 들어), 즉: 비트 또는 "|" 사용 각 플래그에 대한 적절한 비트를 마스킹합니다.

참고: 이 기능은 사용자 범주가 동일한 경우 개별 입자든 그룹이든 상관없이 마스크의 플래그와 일치하는 플래그로 물리 시뮬레이션의 모든 입자를 그립니다.

그려진 입자는 생성할 때 설정된 색상 및 알파 값을 무시하고 대신 이 함수에 제공된 크기, 각도, 색상 및 알파 매개변수를 사용하여 그려집니다. 이것은 순전히 입자 위치에서 스프라이트를 그리기 위한 것이며 입자의 동작에 영향을 미치지 않습니다.

통사론:

physics_particle_draw_ext(typemask, category, sprite, subimg, xscale, yscale, ang, col, alpha)

논쟁 유형 설명
typemask 물리학 입자 플래그 상수 (들) The mask to compare the particles flags with.
category 진짜 The user defined category to draw (use 0 to draw all categories).
sprite 스프라이트 자산 The sprite to use for each particle.
subimg 진짜 The image_index of the sprite to draw.
xscale 진짜 The scale factor to apply to the sprite along the x axis.
yscale 진짜 The scale factor to apply to the sprite along the y axis.
ang 진짜 The angle (in degrees) to draw the sprite at.
col 색깔 The colour to blend with the sprite (default is c_white ).
alpha 진짜 The alpha value to use when drawing (0 - 1).

보고:

N/A

예시:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_draw_ext(flags, 2, sprBlob, 0, 2, 2, 0, c_white, 0.5);

위의 코드는 지정된 마스크와 동일한 플래그를 가진 사용자 정의 범주 "2"의 모든 입자에 대해 스프라이트를 그립니다. 두 축을 따라 2로 스케일되고 알파는 0.5입니다.
