physics_apply_angular_impulse

이 함수는 물리 활성화 인스턴스에 각도 충격을 줍니다. 이 충격은 주어진 양만큼 각 회전을 설정하고 현재 토크를 무시하고 본질적으로 고정 장치가 갖는 "회전"의 양을 설정합니다. 토크를 사용하여 인스턴스에 각력을 적용하려면 다음 기능을 사용해야 합니다.physics_apply_torque().

통사론:

physics_apply_angular_impulse(impulse)

논쟁 유형 설명
impulse The impulse (in Newton metres) to be applied

보고:

예시:

if keyboard_check(vk_up)
{
    physics_apply_angular_impulse(10);
}

위의 코드는 키가 눌렸을 때 물리 활성화된 조명기에 각 임펄스를 줄 것입니다.
