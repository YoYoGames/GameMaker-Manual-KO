game_save

게임의 현재 상태를 저장하는 데 사용할 수 있는 레거시 기능으로 더 이상 사용하지 않는 것이 좋습니다. 대신 파일 기능 을 사용하여 특정 게임 데이터만 저장하고 로드하는 사용자 지정 저장 시스템을 만듭니다.

이 함수는 게임의 상태를 있는 그대로 저장하지만 데이터 구조, 표면, 런타임에 추가된 자산 등과 같이 당시 사용 중인 동적 리소스는 저장하지 않습니다. 이러한 저장 파일이 로드되면 모든 게임이 업데이트됩니다. 저장 후 만들어진 파일은 저장을 생성하는 데 사용된 게임 버전을 복원하므로 표시되지 않습니다.

이 기능을 사용하여 생성한 저장 파일은 업데이트된 런타임 버전에서 지원되지 않을 수 있으므로 이 기능을 새 저장 시스템으로 교체하는 것은 향후 GameMaker 업데이트 및 위에 나열된 이유로 인해 자신의 게임에 대한 업데이트와의 호환성을 보장하는 데 필수적입니다.

통사론:

game_save(filename);

논쟁 유형 설명
filename The name of the file to save the game to.

보고:

예시:

if keyboard_check_pressed(ord("S"))
{
    global.Saved = true;
    game_save("Save.dat");
}

위의 코드는 전역 변수를 다음으로 설정합니다. true 그런 다음 "S" 키를 누르면 게임을 "Save.dat" 파일에 저장합니다.
