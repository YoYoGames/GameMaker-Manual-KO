game_get_speed

이 함수는 초당 실행할 게임 프레임 수 또는 게임 프레임당 마이크로초 수로 게임 속도를 얻는 데 사용할 수 있습니다. 이것은 실제 실행 속도 FPS 값이 아닙니다.fps_real 변수)가 아니라 게임이 초당 유지하려고 시도하는 게임 프레임(FPS) 수 또는 게임이 유지하려고 시도하는 마이크로초 단위의 각 게임 프레임 길이(MPF)입니다. 이 함수를 사용할 때 반환 값의 유형을 결정하는 다음 상수 중 하나를 제공해야 합니다.

게임 속도 상수
일정한 설명
gamespeed_fps 초당 프레임 수를 사용하여 게임 속도를 가져옵니다.
gamespeed_microseconds 프레임당 마이크로초를 사용하여 게임 속도를 가져옵니다.

예를 들어 게임 옵션에서 게임 속도가 30으로 설정되어 있고 FPS 유형을 사용하는 경우 함수는 30을 반환하지만 MPF를 사용하는 경우 함수는 33333을 반환합니다.

통사론:

game_get_speed(type);

논쟁 유형 설명
type 게임 속도 상수 The type of method used to get the game speed (see the constants above).

보고:

Real

예시:

if game_get_speed(gamespeed_fps) != 60
{
    game_set_speed(60, gamespeed_fps);
}

위의 코드는 게임이 60 FPS의 게임 속도로 실행되고 있는지 확인하고 그렇지 않은 경우 60 FPS로 설정되어 있는지 확인합니다.
