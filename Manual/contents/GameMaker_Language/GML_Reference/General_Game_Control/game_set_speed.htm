game_set_speed

이 기능을 사용하여 게임 속도를 설정할 수 있습니다. 다음 두 가지 상수 중 하나를 사용하여 FPS(게임 프레임/초) 또는 MPF(마이크로초)의 두 가지 방법 중 하나로 이를 설정할 수 있습니다.

일정한 설명
gamespeed_fps 초당 프레임 수를 사용하여 게임 속도를 설정합니다.
gamespeed_microseconds 프레임당 마이크로초를 사용하여 게임 속도를 설정합니다.

예를 들어 초당 30프레임의 게임 속도는 게임 게임 프레임당 33333마이크로초가 되며 이 함수는 다음 중 하나로 표현됩니다.

game_set_speed(30, gamespeed_fps);

또는:

game_set_speed(33333, gamespeed_microseconds);

통사론:

game_set_speed(speed, type);

논쟁 유형 설명
speed 진짜 The new game speed (as either FPS or MPF).
type 게임 속도 상수 The type of method used to set the game speed (see the constants above).

예시:

if os_browser == browser_not_a_browser
{
    game_set_speed(60, gamespeed_fps);
}
else
{
    game_set_speed(30, gamespeed_fps);
}
