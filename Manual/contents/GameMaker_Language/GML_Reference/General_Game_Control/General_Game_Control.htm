일반 게임 제어

이러한 기능과 변수는 모두 로드/저장, 다시 시작 등과 같은 게임 특정 기능과 관련이 있습니다.

다음 전역 범위 내장 변수는 게임을 디버깅하는 데 유용합니다. 예를 들어 문제가 발생한 경우 로그 파일에 대한 출력으로 사용할 수 있을 뿐만 아니라 현재 빌드 ID 등을 표시하는 데에도 사용할 수 있습니다. 사용자:

  • game_id
  • game_save_id
  • game_display_name
  • game_project_name

일반적인 게임 제어에 사용되는 기능은 다음과 같습니다.

  • game_end
  • game_restart
  • game_load
  • game_load_buffer
  • game_save
  • game_save_buffer
  • game_get_speed
  • game_set_speed

게임의 최고 점수를 생성하고 유지하는 데 도움이 되도록 GameMaker 는 액세스하고 추가하고 변경할 수 있는 10개 장소의 글로벌 최고 점수 배열 을 생성하여 큰 소란 없이 자신만의 맞춤형 최고 점수 목록을 생성할 수 있습니다. 이 기능은 게임에 로컬이며 모든 플랫폼에서 작동하므로 파일 처리 기능 과 함께 이러한 기능을 사용하여 기본 최고 점수 테이블을 쉽게 저장하고 유지할 수 있습니다.

내부 최고 점수 목록을 처리하는 다음과 같은 기능이 있습니다.

  • highscore_add
  • highscore_name
  • highscore_value
  • highscore_clear

GameMaker 에는 최고 점수를 목록으로 화면에 그리는 기능도 포함되어 있지만 매우 기본적이며 자신의 최고 점수 테이블을 만드는 것이 쉽고 많은 것을 얻을 수 있으므로 무엇보다 디버깅에 이것을 사용하는 것이 좋습니다. 표시 방법에 대한 더 많은 제어:

  • draw_highscore

마지막으로 마우스 커서에 스프라이트를 할당할 수 있는 기능이 있습니다.

  • cursor_sprite
