game_end

이 기능을 사용하여 게임을 종료할 수 있습니다( 게임 종료 이벤트 가 트리거됨). 이것은 즉시 발생하지 않고 현재 단계가 끝날 때 발생하므로 이 함수가 호출된 후 동일한 단계에 있는 코드는 계속 실행됩니다. 이 기능에는 다음과 같은 제한 사항이 있습니다.

  • Android 기기에서 전화 걸기 game_end() 앱을 백그라운드로 푸시하지만 앱을 닫지는 않습니다. 이것은 사용자가 수행해야 합니다.
  • iOS에서는 아무 작업도 수행하지 않고 자동으로 실패합니다.
  • 모든 콘솔(Xbox에서 실행할 때 UWP 포함)에서 게임이 충돌하거나 기껏해야 자동으로 실패할 수 있으며 제출 실패일 수도 있으므로 이 기능 을 사용해서는 안 됩니다.
  • HTML5에서는 게임이 종료되지만 사용자가 웹 페이지의 빈 그리기 캔버스를 응시하게 하므로 해당 대상 플랫폼에서 사용하지 않는 것이 좋습니다.
  • Windows(PC의 UWP 포함)에서 Linux 및 macOS 기능은 단순히 게임을 종료하고 게임 창을 닫습니다( 게임 종료 이벤트 도 트리거됨).

통사론:

game_end();

보고:

예시:

if keyboard_check_pressed(vk_escape) game_end();

플레이어가 "탈출" 키를 누르면 게임이 종료됩니다.
