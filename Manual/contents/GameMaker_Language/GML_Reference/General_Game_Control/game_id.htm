game_id 지원 중단됨

이 읽기 전용 변수는 생성한 게임의 고유 식별자를 반환합니다. 고유한 파일 이름이 필요하거나 게임만 식별할 수 있는 항목이 필요한 경우 이를 사용할 수 있습니다. 이것은 게임 옵션 에서 설정할 수 있습니다.

통사론:

game_id

보고:

Real

예시:

ini_open("Score.ini");
ini_write_real("Scores", "0", score + game_id);
ini_close();

위의 코드는 저장하기 전에 game_id를 추가하여 점수에 대한 매우 기본적인 암호화를 수행합니다. 게임으로 다시 읽을 때 올바른 값을 다시 얻기 위해 game_id를 공제합니다.
