game_save_buffer

이것은 game_save() 기능이므로 사용 및 이 기능과 관련된 중요한 정보가 포함되어 있으므로 먼저 해당 페이지 를 읽으십시오.

이 기능을 사용하면 게임의 현재 상태를 이전에 생성된 "성장" 버퍼에 저장할 수 있습니다( 버퍼 참조). 이 버퍼는 다음을 사용하여 언제든지 다시 로드할 수 있습니다.game_load_buffer() 함수. 이 기능은 한 번에 한 방에서 사용하도록 설계되었으며(즉, 한 방에서 버퍼 저장을 수행한 다음 다른 방에서 버퍼를 로드하려고 시도하지 않는 것이 좋습니다) 체크포인트 또는 레벨 재시작에만 이상적으로 사용해야 합니다.

참고: 이 기능은 매우 제한적이며 초보자가 체크포인트 시스템을 빠르게 시작하고 실행할 수 있도록 설계되었지만 고급 사용자는 게임이 저장 되지 않는다는 사실 때문에 파일 기능을 사용하여 자신의 시스템을 코딩하는 것을 선호할 수 있습니다. 데이터 구조, 표면, 추가된 스프라이트 등과 같은 모든 동적 리소스

통사론:

game_save_buffer(buffer);

논쟁 유형 설명
buffer The buffer id to save to.

보고:

예시:

if keyboard_check_pressed(ord("S"))
{
    global.Checkpoint = true;
    game_save_buffer(save_buff);
}

위의 코드는 전역 변수를 다음으로 설정합니다. true 그런 다음 "S" 키를 누를 때 현재 게임 상태를 변수 "save_buff"에 색인된 버퍼에 저장합니다.
