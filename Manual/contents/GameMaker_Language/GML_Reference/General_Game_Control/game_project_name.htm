game_project_name

이 읽기 전용 변수는 대상 플랫폼에 대해 "저장하기 쉬운" 형식으로 대상 플랫폼에 대한 게임의 표시 이름을 반환합니다. 표시 이름에 파일 이름에 허용되지 않는 문자가 포함되어 있으면 자동으로 "_"로 바뀝니다. 표시 이름은 게임 옵션 에서 설정할 수 있습니다. HTML5의 파일 이름에는 제한이 없으므로 이 문자열은 아마도 다음에서 반환된 문자열과 같을 것입니다.game_display_name.

var file = game_project_name + ".ini";
ini_open(file);
ini_write_real("Scores", "Highscore", score);
ini_close();
