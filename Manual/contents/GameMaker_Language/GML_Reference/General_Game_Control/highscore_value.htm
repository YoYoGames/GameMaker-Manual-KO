highscore_value

이 기능을 사용하면 주어진 위치에서 최고 점수 목록에 저장된 점수 값을 검색할 수 있습니다. 점수가 입력되지 않은 경우 함수는 0을 반환합니다.

통사론:

highscore_value(place);

논쟁 유형 설명
place The place on the table (1-10).

보고:

예시:

var i = 9;
repeat(10)
{
    scr[i] = highscore_value(i + 1);
    i -= 1;
}

위의 코드는 높은 점수 목록을 반복하고 모든 점수를 배열에 저장합니다.
