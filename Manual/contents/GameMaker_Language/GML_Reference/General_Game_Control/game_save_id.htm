game_save_id

이 읽기 전용 변수는 게임에서 파일을 저장하는 데 사용하는 디렉터리의 전체 경로 ID를 반환합니다. 이 디렉토리는 플랫폼에 따라 다른 응용 프로그램에 표시되거나 표시되지 않을 수 있습니다.

HTML5 대상에서는 빈 문자열을 반환합니다.

참고 이 변수는 정보용으로만 존재하며 모든 플랫폼에서 올바르게 작동하지 않으므로 파일 경로 또는 이와 유사한 것을 얻는 데 사용해서는 안 됩니다.

통사론:

game_save_id

보고:

String

예시:

save_dir = game_save_id;

이것은 나중에 사용하기 위해 변수에 파일을 저장하기 위한 디렉토리를 저장합니다.
