game_load_buffer

이 기능을 사용하면 이전에 저장한 게임 상태를 로드할 수 있습니다. 게임은 이전에 생성된 "성장" 버퍼( Buffers 참조)에서 로드되며 버퍼에는 다음을 사용하여 게임 상태가 저장되어 있어야 합니다.game_save_buffer() 함수.

통사론:

game_load_buffer(buffer);

논쟁 유형 설명
buffer The buffer id to load from.

보고:

예시:

if keyboard_check_pressed(ord("L"))
{
    if global.Checkpoint game_load_buffer(save_buff);
}

위의 코드는 전역 변수가 true, 플레이어가 "L" 키를 누를 때.
