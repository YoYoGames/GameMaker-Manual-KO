highscore_add

이 기능을 사용하여 내부 글로벌 최고 점수 목록에 이름과 점수를 추가할 수 있습니다. GameMaker 는 저장된 10번째 위치보다 큰 값만 저장하므로 점수 목록에 입력하기에 충분히 높은지 확인하기 위해 값을 확인할 필요가 없습니다.

통사론:

highscore_add(str, numb);

논쟁 유형 설명
str 끈 The string (name) to attribute the score to.
numb 진짜 The number (score) to add.

보고:

N/A

예시:

if (lives == 0)
{
    highscore_add(global.Name, score);
    score = 0;
    room_goto(rm_MainMenu);
}

위의 코드는 수명을 확인하고 0으로 설정하면 현재 점수와 전역 변수 "Name"에 있는 문자열을 최고 점수 목록에 추가한 다음 최종적으로 점수를 0으로 설정하고 플레이어를 다음 주소로 보냅니다. "rm_MainMenu"에 인덱싱된 방.
