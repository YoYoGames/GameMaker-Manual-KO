game_load

이 기능을 사용하여 이전에 저장한 게임을 로드할 수 있습니다.game_save(). 저장을 생성하는 데 사용된 게임 버전을 복원하므로 이후에 수행된 모든 업데이트는 표시되지 않습니다. 자세한 내용은 다음 페이지를 참조하십시오.game_save().

통사론:

game_load(filename);

논쟁 유형 설명
filename 끈 The name of the file to load from.

보고:

N/A

예시:

if keyboard_check_pressed(ord("L"))
{
    if (global.Save) game_load("Save.dat");
}

위의 코드는 전역 변수가 다음과 같은 경우 이전에 저장한 게임을 로드합니다. true 플레이어가 "L" 키를 누를 때.
