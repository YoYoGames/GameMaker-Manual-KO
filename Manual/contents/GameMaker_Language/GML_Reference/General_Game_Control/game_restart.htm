game_restart

이 기능으로 게임을 다시 시작할 수 있습니다. 이것은 본질적으로 게임을 처음 실행하는 것과 동일하므로 Game Start Event 와 Game End Event 가 트리거 됩니다.

이 함수가 호출될 때 특정 항목은 재설정 되지 않습니다.

  • 전역 변수는 명시적으로 코딩되지 않는 한 다시 초기화되지 않습니다(예: 내장 전역 변수). score 게임에서 이미 수정된 경우 게임을 다시 시작한 후 0에서 시작하지 않습니다.
  • GPU 상태는 변경되지 않습니다. 예를 들어 그리기 색상이나 알파를 설정한 경우 변경된 값으로 유지됩니다.
  • 게임 속도는 게임 코드에 설정한 대로 그대로 유지됩니다(변경한 경우 이 변경 사항은 지속됨).
  • 예를 들어 스프라이트 리소스의 원점을 변경하거나 경로 리소스의 위치를 이동하는 경우와 같이 게임 내 런타임에 변경된 자산 브라우저의 자산은 재설정 되지 않습니다.
  • 버퍼, 표면, 데이터 구조 또는 가져온 스프라이트와 같은 동적 리소스도 정리되거나 제거되지 않습니다(참조가 손실될 수 있으므로 이 함수를 사용하여 동적 리소스에 대한 전역 참조를 사용하거나 정리 함수가 호출되기 전에 올려야 함).

통사론:

game_restart();

N/A

예시:

if keyboard_check_pressed(ord("R")) game_restart();

플레이어가 "R" 키를 누르면 게임이 다시 시작됩니다.
