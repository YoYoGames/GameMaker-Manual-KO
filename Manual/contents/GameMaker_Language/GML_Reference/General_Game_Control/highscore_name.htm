highscore_name

이 기능을 사용하면 주어진 위치의 최고 점수 목록에 저장된 이름 문자열을 검색할 수 있습니다. 이름이 입력되지 않은 경우 "알 수 없음" 문자열이 반환됩니다.

통사론:

highscore_name(place);

논쟁 유형 설명
place The place on the table (1-10).

보고:

예시:

var i = 9;
repeat(10)
{
    name[i] = highscore_name(i + 1);
    i -= 1;
}

위의 코드는 높은 점수 목록을 반복하고 모든 이름을 배열에 저장합니다.
