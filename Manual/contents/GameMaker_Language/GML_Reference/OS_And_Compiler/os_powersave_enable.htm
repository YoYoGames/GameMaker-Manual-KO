os_powersave_enable

이 기능을 사용하여 장치의 절전 기능을 켜거나 끌 수 있습니다. 이것은 특정 게임(예: 기울기 기능을 사용하는 게임)이 OS가 사용자 입력으로 해석할 수 있는 이벤트를 생성하지 않아 화면을 종료하거나 게임을 종료할 수 있기 때문에 중요합니다. 이 기능을 로 설정하면 false 절전 기능을 비활성화하고 화면(및 게임)이 항상 작동하는지 확인할 수 있습니다.

참고: 이것은 iOS 및 Android 대상으로 제한됩니다.

통사론:

os_powersave_enable(flag)

논쟁 유형 설명
flag Set to true or false to enable or disable powersave functions.

보고:

예시:

if (os_type == os_android) || (os_type == os_ios)
{
    os_powersave_enable(false);
}

위의 코드는 OS 유형을 확인하고 Android 또는 iOS이면 절전 기능이 비활성화됩니다.
