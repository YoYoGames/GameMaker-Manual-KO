parameter_string

명령줄 매개변수는 프로그램 실행 방법을 변경하기 위해 exe에 추가할 수 있는 추가 명령이며 이 기능을 사용하면 선택한 명령줄 매개변수를 문자열로 가져올 수 있습니다. 함수를 사용하여 현재 게임에 대한 매개변수의 수를 찾을 수 있습니다.parameter_count(), 첫 번째 매개변수에는 인덱스 1이 있고 마지막 매개변수에는 함수에서 반환된 인덱스가 있습니다(0의 값은 경로를 포함한 게임 실행 파일의 파일 이름이라는 점에서 특별합니다). 이 함수는 url 매개변수를 검색하는 HTML5 플랫폼에서 작동합니다.

통사론:

parameter_string(n);

보고:

예시:

var p_num;
p_num = parameter_count();
if p_num > 0
{
    var i;
    for (i = 0; i < p_num; i += 1)
    {
        p_string[i] = parameter_string(i + 1);
    }
}

위의 코드는 명령줄 매개변수의 수를 가져오고 1개 이상이면 매개변수를 반복하여 배열에 문자열로 저장합니다.
