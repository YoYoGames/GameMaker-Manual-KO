os_type

이 읽기 전용 변수는 GameMaker 가 게임이 생성된 운영 체제를 알려야 하는 다양한 상수 중 하나를 보유합니다. 예를 들어 게임이 Amazon Fire OS에서 실행될 수 있지만 Android 플랫폼용으로 구축되었기 때문에 이것이 실행 중인 장치의 OS와 반드시 같지는 않습니다 (이 경우 os_type 될거야 os_android ).

다음 상수가 반환될 수 있습니다.

OS 유형 상수
일정한 설명
os_windows 윈도우 OS
os_uwp Windows 10 유니버설 Windows 플랫폼
os_operagx 오페라 GX
os_linux 리눅스
os_macosx 맥 OS X
os_ios iOS(아이폰, 아이패드, 아이팟 터치)
os_tvos 애플 tvOS
os_android 기계적 인조 인간
os_ps4 소니 플레이스테이션 4
os_ps5 소니 플레이스테이션 5
os_xboxone 마이크로소프트 엑스박스 원
os_xboxseriesxs 마이크로소프트 엑스박스 시리즈 X/S
os_switch 닌텐도 스위치
os_unknown 알 수 없는 OS

통사론:

os_type;

보고:

OS Type Constant

예시:

switch (os_type)
{
    case os_windows: global.Config = 0; break;
    case os_android: global.Config = 1; break;
    case os_linux: global.Config = 2; break;
    case os_macosx: global.Config = 3; break;
    case os_ios: global.Config = 4; break;
    case os_winphone: global.Config = 5; break;
}

위의 코드는 게임을 실행하는 OS를 확인하고 그에 따라 전역 변수를 설정합니다.
