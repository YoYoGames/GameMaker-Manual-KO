external_call

외부 함수 호출을 생성한 경우 dll 또는 dylib 사용external_define(), 이 함수를 사용하여 호출할 수 있습니다. 이전에 정의된 함수의 이름과 필요한 각 인수를 제공하고(각 인수는 올바른 유형이어야 합니다(실수 또는 문자열)) 함수는 외부 호출의 결과를 반환합니다.

통사론:

external_call(id, args[0...15]);

논쟁 유형 설명
id 외부 기능 The name of the function that you want to call
args[0...10] 실수 또는 문자열 The different arguments that you want to pass through to the external function

보고:

Variable (the type of value returned will depend on the defined function)

예시:

my_function = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real);
var _a = external_call(my_function, x, y);

위의 예제 코드는 이전에 정의된 외부 함수를 호출하고 반환된 값을 로컬 변수에 저장합니다.
