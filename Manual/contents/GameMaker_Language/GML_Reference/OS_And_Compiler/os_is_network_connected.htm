os_is_network_connected

이 기능을 사용하면 장치가 현재 인터넷에 연결되어 있는지 확인할 수 있습니다. true 그렇다면, 또는 false 그렇지 않은 경우 OS에 따라 값을 반환하기 전에 연결을 시도할 수 있습니다.

이 함수에는 호출 시 연결을 시도하는 것을 활성화/비활성화하는 선택적 인수가 있으며 이 옵션은 현재 Nintendo Switch 대상 에만 있습니다. 로 설정하면 true 함수는 연결을 시도하고 로 설정되면 false 단순히 연결을 확인하는 것으로 제한됩니다. 연결을 시도하면 스위치 OS가 몇 초 동안 중단될 수 있습니다.

참고 이 함수는 연결을 제어하는 내부 장치 API를 확인하므로 블루투스 연결, Wi-Fi 연결 또는 인터넷 액세스를 허용하는 일반 네트워크 연결이 있는 경우 true를 반환할 수 있습니다.

통사론:

os_is_network_connected([attempt_connection]);

논쟁 유형 설명
attempt_connection 부울 OPTIONAL For the Nintendo Switch target only. Set to true to attempt an OS level connection when called.

보고:

Boolean

예시:

if os_is_network_connected()
{
    global.connected = true;
}

위의 코드는 장치가 인터넷에 연결되어 있는지 확인하고 연결되어 있으면 전역 변수를 설정합니다.
