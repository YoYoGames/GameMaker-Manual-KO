스케줄러_해상도_get

이 함수는 Windows 스레드 스케줄러의 해상도를 밀리초 단위로 검색하는 데 사용됩니다. 스케줄러의 해상도가 기본값(Windows에서 설정한 대로)으로 설정되어 있으면 함수는 -1을 반환합니다.

스레드 스케줄러에 대한 정보는 다음 페이지를 참조하십시오.scheduler_resolution_set().

참고: 이 기능은 Windows 전용입니다.

통사론:

scheduler_resolution_get();

보고:

(or -1 for default)

예시:

scheduler_resolution = scheduler_resolution_get();

이 예는 Windows 스레드 스케줄러의 해상도를 검색하여 저장합니다."scheduler_resolution" 변하기 쉬운.
