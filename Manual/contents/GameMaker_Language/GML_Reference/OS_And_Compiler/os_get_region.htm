os_get_region

이 함수는 ISO3166-1 표준에 따라 게임을 실행하는 OS에 대한 두 글자 또는 세 글자의 지역 코드가 포함된 문자열을 반환합니다. 정보를 사용할 수 없는 경우 단순히 빈 문자열 ""을 보유합니다.

참고: 이것은 반환되는 위치 지역 코드가 아니라 OS의 지역 언어 코드입니다.

통사론:

os_get_region()

보고:

예시:

switch (os_get_language())
{
    case "zh":
        var region = os_get_region();
        if (region == "HK" || region == "MO" || region == "TW")
        {
            ini_open("chinese_traditional.ini");
        }
        else
        {
            ini_open("chinese_simplified.ini");
        }
    break;

    case "fr":
        ini_open("french.ini");
    break;

    case "it":
        ini_open("italian.ini");
    break;

    default:
        ini_open("english.ini");
    break;
}

위의 코드는 OS 언어를 확인하고 중국어인 경우 OS 지역을 확인하여 다른 *.ini 반환된 값에 따라 파일.
