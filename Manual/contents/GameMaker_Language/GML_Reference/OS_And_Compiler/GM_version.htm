GM_version

이 상수는 각 대상 플랫폼에 대한 게임 옵션 에 정의된 버전 번호를 보유합니다. 값은 문자열로 저장됩니다.

통사론:

GM_version;

보고:

예시:

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);

위의 코드는 GM 날짜와 버전 상수를 가져와 화면에 그립니다.
