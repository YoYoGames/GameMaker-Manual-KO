os_device

이 읽기 전용 변수는 아래 나열된 다양한 상수 값 중 하나를 보유하여 현재 게임을 실행 중인 장치를 알려줍니다. 이 변수는 더 이상 사용되지 않으며 함수를 위해 사용됩니다.os_get_info() 게임을 실행하는 장치에 대한 보다 정확한 정보를 반환합니다.

통사론:

os_device

보고:

Device Type Constant

장치 유형 상수
일정한 설명
device_ios_ipad 아이패드
device_ios_ipad_retina Retina 디스플레이 크기가 2048 x 1536인 최신 iPad
device_ios_iphone6 디스플레이 크기가 1334 x 750인 iPhone6
device_ios_iphone6plus 1920 x 1080 디스플레이가 있는 더 큰 iPhone 6
device_ios_iphone5 iPhone5(디스플레이 크기 640 x 1136)
device_ios_iphone 구형 iPhone/iPod Touch(480 x 320 화면) 또는 Android 휴대전화
device_ios_iphone_retina 960 x 640의 Retina 디스플레이가 탑재된 최신 iPhone/iPod Touch
device_emulator 장치는 실제로 에뮬레이터입니다(Windows Phone 또는 Android).
device_tablet 안드로이드 태블릿
device_ios_unknown 알 수 없는 iOS

예시:

if (os_browser = browser_not_a_browser)
{
    switch (os_device)
    {
        case device_ios_ipad: global.Config = 2; break;
        case device_ios_iphone: global.Config = 3; break;
        case device_ios_iphone_retina: global.Config = 4; break;
        case device_ios_unknown: global.Config = 5; break;
    }
}
else
{
    global.Config = 1;
}

위의 코드는 게임이 브라우저에서 실행되고 있는지 확인하고, 확인 결과에 따라 전역 변수를 값으로 설정합니다.
