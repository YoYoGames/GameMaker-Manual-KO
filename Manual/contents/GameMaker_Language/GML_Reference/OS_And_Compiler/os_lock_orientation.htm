os_lock_orientation

이 기능을 사용하면 장치를 "해제"할 때까지 현재 방향으로 장치를 "잠금"하여 해당 대상 플랫폼에 대해 모든 게임 옵션 활성화된 방향을 다시 허용할 수 있습니다. 잠그기 전에 방향이 원하는 대로 되어 있는지 확인하는 것이 좋습니다.

통사론:

os_lock_orientation(flag)

논쟁 유형 설명
flag Set to true or false to enable or disable orientation locking.

보고:

예시:

if (os_type == os_android) || (os_type == os_ios)
{
    os_lock_orientation(true);
}

위의 코드는 OS 유형을 확인하고 Android 또는 iOS인 경우 방향 잠금 플래그가 다음과 같이 지정됩니다. true.
