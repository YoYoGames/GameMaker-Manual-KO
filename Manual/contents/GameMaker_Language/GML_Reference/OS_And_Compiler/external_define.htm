external_define

이 함수는 특정 dll(Windows의 경우) 또는 dylib(Mac의 경우)에 대한 외부 함수 호출을 정의하는 데 사용할 수 있습니다. 이 파일은 포함된 파일이거나 확장의 일부일 수 있습니다.

파일의 이름(및 경로)을 제공한 다음 정의하려는 함수의 이름을 제공합니다. 다음으로 사용할 호출 규칙(아래 상수 목록 참조)과 예상되는 결과 유형(아래 나열된 상수)을 정의해야 합니다.

마지막으로 함수가 취할 수 있는 인수의 수(0에서 15까지)를 제공해야 하고 각 인수에 대해 유형도 지정해야 합니다. 4개 이상의 인수가 있는 함수의 경우 모두 다음 유형 이어야 합니다. ty_real.

참고: 이것은 GameMaker IDE에 "포함된 파일"로 추가된 dll 또는 dylib에만 해당됩니다. 확장으로 추가된 파일에서는 작동하지 않습니다. 확장 패키지 자체에서 함수를 정의해야 하기 때문입니다.

외부 호출 유형 상수
일정한 설명
dll_cdecl 이것은 기본 C, C++ 호출입니다.
dll_stdcall 이것은 표준 WinAPI 호출(Windows dll 오직)
외부 데이터 유형 상수
일정한 설명
ty_real 실수 인수
ty_string null로 끝나는 문자열 인수

통사론:

external_define(dll, name, calltype, restype, argnumb, argtype[0], argtype[1], ...argtype[10]);

논쟁 유형 설명
dll끈 The name of the DLL file (string)
name끈 The name of the function (string)
calltype 외부 호출 유형 상수(dll_*) The calling convention used
restype 외부 데이터 유형 상수(ty_*) The type of the result to expect
argnumb 진짜 The number of arguments (0 - 10)
argtype[0... 10] 외부 데이터 유형 상수(ty_*) The types of the arguments being used

보고:

External Function

예시:

my_funcion = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real);

위의 예제 코드는 두 개의 인수로 "MyMin"이라는 외부 함수를 정의합니다.
