parameter_count

명령줄 매개변수는 프로그램 실행 방법을 변경하기 위해 exe에 추가할 수 있는 추가 명령입니다. 이 함수를 사용하여 현재 게임에 대한 매개변수의 수를 찾을 수 있습니다. 여기서 첫 번째 매개변수는 인덱스 1을 갖고 마지막 매개변수는 함수에서 반환된 인덱스를 갖습니다(0의 값은 게임 실행 파일의 파일 이름이라는 점에서 특별합니다., 경로 포함). 이 함수는 url 매개변수를 검색하는 HTML5 플랫폼에서 작동합니다.

통사론:

parameter_count();

보고:

예시:

p_num = parameter_count();

위의 코드는 "p_num" 변수에 게임에 사용된 명령줄 매개변수의 수를 저장합니다.
