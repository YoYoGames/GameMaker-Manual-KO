environment_get_variable

이 함수는 주어진 이름(또한 문자열)을 가진 환경 변수의 값(문자열)을 반환합니다. "를 입력하여 macOS 및 Ubuntu(Linux)에서 사용 가능한 환경 변수를 가져올 수 있습니다.env " 터미널 앱에 입력하고 Windows 환경 변수에 대한 정보를 보려면 명령 프롬프트를 사용하는 경우 "echo %PATH% ", 그리고 PowerShell을 사용하는 것은 "ls env ". macOS와 Ubuntu(Linux) 모두에서 "HOME" 환경 변수는 "~/ "에 매핑되는 경로"/Users/<username> " macOS 및 "/home/<username> " 우분투(리눅스)에서.

참고: 이 기능은 Windows, macOS 및 Ubuntu(Linux) 전용입니다.

통사론:

environment_get_variable(name);

논쟁 유형 설명
name The name (a string) of the environment variable to check.

보고:

예시:

e_str = environment_get_variable("APPDATA");

위의 코드는 환경 변수 "%appdata% ", 일반적으로 "C:\Users\{username}\AppData\Roaming ".
