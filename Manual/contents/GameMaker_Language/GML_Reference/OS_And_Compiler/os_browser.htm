os_browser

이 읽기 전용 변수는 GameMaker 가 현재 게임을 실행 중인 브라우저(있는 경우)를 알려야 하는 다양한 상수(아래 나열) 중 하나를 보유합니다.

통사론:

os_browser

보고:

Browser Type Constant

브라우저 유형 상수
일정한 설명
browser_not_a_browser 게임이 브라우저에서 재생되지 않습니다
browser_unknown 알 수 없는 브라우저
browser_ie 인터넷 익스플로러
browser_ie_mobile 모바일 장치의 Internet Explorer
browser_firefox 모질라 파이어 폭스
browser_chrome 구글 크롬
browser_safari 원정 여행
browser_safari_mobile 모바일 장치의 Safari
browser_opera 오페라
browser_tizen 타이젠 모바일 기기 브라우저
browser_windows_store 윈도우 앱

예시:

if (os_browser == browser_not_a_browser)
{
    global.Config = 0;
}
else
{
    global.Config = 1;
}

위의 코드는 게임이 브라우저에서 실행되고 있는지 확인하고, 확인 결과에 따라 전역 변수를 값으로 설정합니다.
