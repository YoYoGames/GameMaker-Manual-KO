os_get_config

이 함수는 게임에 대해 현재 선택된 구성의 이름(문자열)을 반환합니다. 구성에 대한 자세한 내용은 구성 섹션을 참조하십시오.

통사론:

os_get_config()

보고:

예시:

if os_get_config() = "Free_Version"
{
    global.Ads = true;
}
else global.Ads = false;

위의 코드는 어떤 구성이 사용되고 있는지 확인하고 "Free_Version"이라는 구성이면 게임에서 광고가 활성화됩니다.
