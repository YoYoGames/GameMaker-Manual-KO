external_free

이 함수는 연결된 메모리를 해제합니다. dll 또는 dylib 주어진 이름으로. 문제의 파일이 게임에서 더 이상 필요하지 않을 때마다(예: 일반적으로 Game End 이벤트에서) 이 작업을 수행해야 합니다.

통사론:

external_free(id);

논쟁 유형 설명
id 외부 기능 The name of the dll or dylib that you want to free

보고:

N/A

예시:

external_free("MyDLL.dll");

위의 예제 코드는 주어진 dll과 관련된 메모리를 해제합니다.
