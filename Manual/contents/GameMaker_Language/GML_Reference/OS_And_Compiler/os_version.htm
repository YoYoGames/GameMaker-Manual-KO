os_version

이 변수는 게임을 실행하는 OS의 버전 번호를 알려줍니다. 예를 들어 Windows 10에서 실행하는 경우 os_version 655360과 같습니다.

다음 표에는 다른 OS의 최신 버전에 대해 반환되는 값이 요약되어 있습니다.

참고: Windows OS에 대한 추가 정보가 필요한 경우 다음을 사용할 수 있습니다.environment_get_variable().

통사론:

os_version

보고:

예시:

if (os_type == os_android) && (os_version > 10)
{
    global.GFX = 1;
}

위의 코드는 os 유형과 버전 번호를 확인하고 둘 다 정확하고 전역 변수가 1로 설정됩니다.
