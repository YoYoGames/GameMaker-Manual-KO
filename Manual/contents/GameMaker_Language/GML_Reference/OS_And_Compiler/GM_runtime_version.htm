GM_runtime_version

이 상수는 프로젝트를 빌드하는 데 사용되는 런타임으로 환경 설정 에 정의된 런타임 버전 번호를 보유합니다. 값은 문자열로 저장됩니다.

통사론:

GM_runtime_version;

보고:

String

예시:

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);
draw_text(32, 96, "Runtime " + GM_runtime_version);

위의 코드는 GM 날짜, 버전 및 런타임 상수를 가져와 화면에 그립니다.
