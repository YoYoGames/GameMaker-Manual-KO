os_get_language

이 함수는 ISO639 표준에서 설정한 대로 게임을 실행하는 OS에 대한 두 글자 언어 코드가 포함된 문자열을 반환합니다. 정보를 사용할 수 없는 경우 단순히 빈 문자열 "" 또는 "영어" 언어의 경우 "en"을 보유합니다. 일부 언어에는 관련 지역 코드도 있으므로 동일한 국가의 다른 지역을 구별하려면 이 기능을 사용하십시오.os_get_region().

다음 표는 ISO 639에서 정의한 주요 두 글자 언어 코드의 예를 보여줍니다.

언어 암호
아라비아 말 아르
중국인 zh
덴마크 말 다
영어 ko
프랑스 국민 정말로
독일 사람 드
그리스 어 엘자
이탈리아 사람 그것
일본어 자
노르웨이 인 아니요
광택 제발
포르투갈 인 태평양 표준시
러시아인 루
스페인의 에스
스웨덴어 SV

참고: 이것은 반환되는 위치 국가 코드가 아니라 OS의 언어 코드입니다.

통사론:

os_get_language()

보고:

예시:

switch (os_get_language())
{
    case "es": ini_open("spanish.ini"); break;
    case "fr": ini_open("french.ini"); break;
    case "it": ini_open("italian.ini"); break;
    default: ini_open("english.ini"); break;
}

위의 코드는 OS 언어를 확인하고 다른 *.ini 반환된 값에 따라 파일.
