display_get_orientation

이 함수는 GameMaker 가 게임을 실행하는 장치가 가로 모드인지 세로 모드인지 알려야 하는 네 가지 상수 중 하나를 반환합니다(아래 표 참조).

이 기능은 HTML5 대상 모듈에서 사용할 때 장치의 방향을 올바르게 감지하지 못할 수 있습니다. 그러나 이것은 다음 스크립트를 사용하여 쉽게 모방할 수 있습니다.

return (browser_width < browser_height);

이러한 함수는 다음을 반환합니다. true 초상화와 false 풍경을 위해.

통사론:

display_get_orientation()

보고:

Display Orientation Constant

디스플레이 방향 상수
일정한 설명
display_landscape 장치를 수평으로 잡고 있습니다. 즉, 가장 긴 가장자리는 왼쪽에서 오른쪽으로, 메뉴 버튼은 오른쪽에 있습니다.
display_landscape_flipped 위와 같이 메뉴 버튼만 왼쪽에 있습니다.
display_portrait 장치를 수직으로 잡고 있습니다. 즉, 가장 긴 쪽이 위에서 아래로, 메뉴 버튼이 아래쪽에 있습니다.
display_portrait_flipped 위와 같이 이제 메뉴 버튼만 맨 위에 있습니다.

예시:

if display_get_orientation() == display_landscape
{
    global.Config = 0;
}
else
{
    global.Config = 1;
}

위의 코드는 장치의 방향을 확인하고 반환된 값에 따라 전역 변수를 설정합니다.
