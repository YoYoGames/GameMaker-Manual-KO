screen_save_part

이 기능을 사용하면 현재 게임 디스플레이에 나타나는 게임의 일부 스크린샷을 저장할 수 있습니다. 선택 좌표는 디스플레이(또는 전체 화면이 아닌 경우 창)의 절대 크기를 기반으로 하므로 (0,0)은 항상 화면(또는 창)의 왼쪽 상단 모서리입니다. 아래는 이 기능이 어떻게 작동하는지 보여주는 이미지입니다.

파일은 게임의 작업 디렉토리 에 저장됩니다. 이 함수는 모든 플랫폼에서 일관된 결과를 제공해야 하는 Draw GUI End Event 에서 호출되도록 설계되었다는 점에 유의하는 것이 중요합니다. 다른 이벤트에서 사용하면 예상대로 작동하지 않을 수 있으며 다른 대상 및 장치에 따라 다른 결과를 제공할 수 있습니다.

참고 이 기능은 HTML5 대상에서 작동하지 않습니다.

통사론:

screen_save_part(fname,x,y,w,h);

논쟁 유형 설명
fname The filename for the screenshot to save
x The x coordinate of the top left corner of the area of the screen to capture
y The y coordinate of the top left corner of the area of the screen to capture
w The width of the area of the screen to capture
h The height of the area of the screen to capture

보고:

예시:

if keyboard_check_pressed(ord("S"))
{
    screen_save_part("Screens\Screen_" + string(num++) + ".png", 0, 0, 100, 100)
}

위의 코드는 사용자가 "S" 키를 눌렀을 때 100픽셀 정사각형의 게임 화면 일부를 캡처한 것입니다. 이 스크린샷은 특정 폴더에 저장되고 증가하는 변수를 사용하여 번호가 매겨집니다.
