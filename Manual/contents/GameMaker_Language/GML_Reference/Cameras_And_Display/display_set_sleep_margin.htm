display_set_sleep_margin

이 함수는 게임의 렌더 타이밍에 사용되는 휴면 마진 값을 설정하는 데 사용할 수 있으며 밀리초 값이 필요합니다. 기본값은 다음과 같습니다.

플랫폼 밀리초
창 10
맥 OS 10
우분투 10
HTML5 10
기계적 인조 인간 4
iOS 4
윈도우 UWP 10
엑스박스 10
PS4 10
닌텐도 스위치 10

타이밍 방법이 vsync 타이밍을 사용하도록 설정되어 있더라도 절전 마진은 게임 렌더링에 약간의 영향을 미치므로 이 값을 설정할 때 주의해야 합니다. 표시 시간에 대한 자세한 내용은 다음을 참조하십시오.display_set_timing_method().

참고: 절전 여백 외에도 런타임 시 스레드 스케줄러의 해상도 를 조정하여 Windows에서 성능 및 전력 소비를 추가로 제어할 수 있습니다.

통사론:

display_set_sleep_margin(milliseconds);

논쟁 유형 설명
milliseconds 진짜 The number of milliseconds for the sleep margin

보고:

N/A

예시:

if display_get_timing_method() != tm_sleep
{
    display_set_timing_method(tm_sleep);
    if display_get_sleep_margin() != 20
    {
        display_set_sleep_margin(20);
    }
}

위의 코드는 타이밍 방법을 확인한 다음 설정되지 않은 경우 tm_sleep 그것은 그것으로 설정되고 수면 마진은 20으로 설정됩니다.
