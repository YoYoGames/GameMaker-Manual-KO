display_get_timing_method

이 함수는 게임을 렌더링하는 데 사용할 타이밍 방법을 검색하는 데 사용할 수 있습니다. 메서드는 아래 나열된 상수 중 하나일 수 있습니다.

다양한 타이밍 방법에 대한 자세한 내용은 다음을 참조하십시오.display_set_timing_method().

통사론:

display_get_timing_method();

보고:

Display Timing Method Constant

표시 타이밍 방법 상수
일정한 설명
tm_sleep 슬립 마진 값은 주요 타이밍 방법입니다.
tm_countvsyncs Vsync 타이밍이 주요 타이밍 방법입니다(지원되는 모든 플랫폼의 기본값).

예시:

if display_get_timing_method() != tm_sleep
{
    display_set_timing_method(tm_sleep);
    if display_get_sleep_margin() != 20
    {
        display_set_sleep_margin(20);
    }
}

위의 코드는 타이밍 방법을 확인한 다음 설정되지 않은 경우 tm_sleep 그것은 그것으로 설정되고 수면 여백은 20으로 설정됩니다.
