gif_open

이 기능을 사용하면 데이터를 추가할 준비가 된 빈 GIF 형식 이미지를 만들 수 있습니다. 함수를 호출할 때 GIF의 너비와 높이(픽셀 단위, 사용 가능한 메모리를 제외하고 크기에 대한 상한선 없음)를 지정해야 하며 함수는 선물을 식별하는 데 사용되는 고유한 ID 값을 반환합니다. 후속 함수 또는 gif를 초기화할 수 없는 경우(예: 너비/높이가 사용 가능한 메모리에 비해 너무 큰 경우) -1을 반환합니다. GIF의 "선명한 색상"을 설정하기 위해 선택적 인수를 지정할 수도 있습니다. 이것은 RGB 값(알파 구성 요소 없음)이며, 무언가가 추가되기 전에 이 색상으로 gif를 지웁니다. clear color 인수를 제공하지 않으면 기본 색상인 검정색이 사용됩니다.

이 기능을 사용할 때는 다음을 호출해야 합니다.gif_save() 녹음을 위해 다른 파일을 열기 전에 GIF 생성을 종료하려면 gif_open() 동반해야합니다 gif_save() ).

통사론:

gif_open(width, height, [clear_colour]);

논쟁 유형 설명
width 진짜 The width of the gif to create
height 진짜 The height of the gif to create
[clear_colour] 색깔 OPTIONAL The colour (RGB) to clear the gif to

보고:

GIF ID

예시:

if save_gif == true
{
    if (count == 0)
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if (count < 30)
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        gif_save(gif_image, "GameCapture.gif");
        count = 0;
        save_gif = false;
    }
    count++;
}

위의 코드는 응용 프로그램 표면에서 가져온 30프레임의 GIF 이미지 파일을 만든 다음 저장합니다.
