screen_save

이 기능을 사용하면 현재 게임 창에 나타나는 게임 스크린샷을 저장할 수 있습니다.

파일은 Windows의 다음 위치에 있는 게임의 작업 디렉토리에 저장됩니다. <drive>:/Users/YOURUSERNAME/AppData/Local/[Game Name]/ Mac에서는 ~/Library/Application Support/[Game Name]/. 캡처된 이미지는 게임의 애플리케이션 표면의 최종 렌더링이 되거나(앱 표면이 비활성화된 경우) 전체 디스플레이 또는 창의 이미지가 됩니다.

이 함수는 모든 플랫폼에서 일관된 결과를 제공해야 하는 Draw GUI End Event 에서 호출되도록 설계되었다는 점에 유의하는 것이 중요합니다. 다른 이벤트에서 사용하면 예상대로 작동하지 않을 수 있으며 다른 대상 및 장치에 따라 다른 결과를 제공할 수 있습니다.

저장 중인 항목을 더 많이 제어해야 하는 경우 이 기능을 사용해야 합니다.screen_save_part().

참고 이 기능은 HTML5 대상에서 작동하지 않습니다.

통사론:

screen_save(fname);

논쟁 유형 설명
fname 끈 The filename for the screenshot to save

보고:

N/A

예시:

if keyboard_check_pressed(ord("S"))
{
    screen_save(working_directory + "Screens\Screen_" + string(num++) + ".png")
}

위의 코드는 사용자가 "S" 키를 눌렀을 때 게임의 스크린샷을 찍습니다. 이 스크린샷은 특정 디렉토리에 저장되고 증가하는 변수를 사용하여 번호가 매겨집니다.
