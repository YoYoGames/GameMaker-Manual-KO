display_get_gui_height

이 함수를 사용하면 Draw GUI Event 에서 사용되는 GUI의 높이(픽셀 단위)를 얻을 수 있습니다.

통사론:

display_get_gui_height();

보고:

Real

예시:

ads_move(0, display_get_gui_height() - ads_get_display_height(0), 0);

위의 코드는 광고가 디스플레이의 왼쪽 하단 모서리에 표시되도록 설정합니다.
