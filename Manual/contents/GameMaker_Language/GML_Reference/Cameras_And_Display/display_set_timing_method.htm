display_set_timing_method

이 기능은 게임을 렌더링하는 데 사용할 타이밍 방법을 설정하는 데 사용할 수 있습니다. 메서드는 다음 상수 중 하나일 수 있습니다.

일정한 설명
tm_sleep 슬립 마진 값을 주요 타이밍 방법으로 사용
tm_countvsyncs vsync 타이밍을 기본 타이밍 방법으로 사용(지원되는 모든 플랫폼의 기본값)

vsync 타이밍 방법은 수직 동기화를 지원하는 대상 플랫폼을 사용하여 게임 렌더 타이밍 계산을 위한 앵커를 제공하는 반면 모드를 절전 마진으로 설정하면 각 프레임이 올바른 시간(예: 1/30) 동안 지속되도록 합니다. 또는 1/60초) 대기 또는 잠자기. 일반적으로 기본 vsync 타이밍이 가장 부드러운 결과를 제공하지만, vsync 방법을 사용할 때 잠자기 마진 값은 여전히 관련성이 있지만 영향이 줄어들고 이 방법을 사용할 때 기본값으로 유지하는 것이 좋습니다.

기본적으로 PS4, Ubuntu 및 HTML5를 제외한 모든 플랫폼에서 GameMaker 는 vsync 타이밍 방법을 사용하는 반면 지원되지 않는 플랫폼에서는 슬립 마진 타이밍 만 사용할 수 있습니다.

절전 여유를 설정하려면 다음 기능을 사용하여 설정할 수 있습니다.display_set_sleep_margin() 함수와 함께 현재 사용 중인 타이밍 방법을 찾을 수 있습니다.display_get_timing_method().

통사론:

display_set_timing_method(method);

논쟁 유형 설명
method 표시 타이밍 방법 상수 The timing method to use (see the list of constants, above)

보고:

N/A

예시:

if display_get_timing_method() != tm_sleep
{
    display_set_timing_method(tm_sleep);
    if (display_get_sleep_margin() != 20)
    {
        display_set_sleep_margin(20);
    }
}

위의 코드는 타이밍 방법을 확인한 다음 설정되지 않은 경우 tm_sleep 그것은 그것으로 설정되고 수면 마진은 20으로 설정됩니다.
