display_set_gui_size

이 기능은 Draw GUI 이벤트 에 그려진 것들에만 영향을 미치며 해당 이벤트의 모든 GUI 구성 요소에 대한 특정 너비와 높이를 설정하는 데 사용할 수 있으며 GUI를 표시하는 데 사용되는 디스플레이 또는 창의 크기에 관계없이 정확하게 맞도록 조정되었습니다. GUI의 위치(예: (0,0) 위치)는 기능을 사용했는지 여부에 따라 다릅니다.display_set_gui_maximise() 아니면. 이 기능(기본 설정)을 사용하지 않는 경우 GUI 레이어는 응용 프로그램 표면(게임이 그려지는 영역)과 너비와 동일한 위치에 (0,0) 점이 배치됩니다. 높이가 이 표면에 맞게 늘어나므로 응용 프로그램 표면이 512px일 때 이 함수로 GUI 너비를 256px로 설정하면 해당 256픽셀이 늘어나서 픽셀이 효과적으로 두 배 커집니다. 이는 전역 게임 설정에서 종횡비 보정이 설정된 게임에 이상적입니다.

사용하신 경우 display_set_gui_maximise() 기능을 사용하면 GUI 레이어의 (0,0) 위치가 기본적으로 게임 창의 왼쪽 상단 모서리 또는 디스플레이(전체 화면 또는 장치에 있을 때)로 설정되며 해당 기능에 의해 설정된 배율과 너비와 높이가 사용됩니다. 이것으로 설정합니다. 너비와 높이 모두에 -1을 사용하여 이 함수를 호출하여 언제든지 GUI 레이어를 기본 구성으로 재설정할 수 있습니다. 그러면 (0, 0) 위치가 애플리케이션 표면으로 설정되고 너비와 높이는 다음으로 재설정됩니다. 표면과도 1:1.

통사론:

display_set_gui_size(width, height);

논쟁 유형 설명
width The width of the GUI
height The height of the GUI

보고:

예시:

display_set_gui_size(768, 1024);

위의 코드는 그리기 GUI 이벤트를 주어진 너비와 높이로 잠그고 해당 비율을 사용하여 응용 프로그램 표면이나 디스플레이에 맞게 모든 구성 요소의 크기를 조정합니다.
