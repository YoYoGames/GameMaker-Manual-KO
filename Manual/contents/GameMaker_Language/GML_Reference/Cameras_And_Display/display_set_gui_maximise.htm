display_set_gui_maximise

이 기능을 사용하여 GUI 레이어를 최대화하고 기본 애플리케이션 표면 위치 및 치수가 아닌 화면 치수와 관련하여 조정 및 오프셋되도록 설정할 수 있습니다. 기본적으로 GUI 레이어는 응용 프로그램 표면 해상도와 1:1이며 표면의 (0,0) 위치에도 그려집니다. 그러나 이것이 항상 원하는 것은 아니므로 이 기능을 사용하여 디스플레이 또는 게임 창의 절대(0,0) 위치를 기준으로 그리도록 설정할 수 있습니다. 이 함수에 제공하는 인수는 GUI 계층에 원하는 효과에 따라 다릅니다. 인수 없이 함수를 호출하기만 하면 GUI 레이어가 화면이나 게임 창의 (0,0) 위치에 그려지도록 설정되며 너비와 높이는 전체 영역에 맞게 조정됩니다.

display_set_gui_maximise();

그러나 GUI 레이어에 대한 배율 인수 를 설정할 수 있으며 너비와 높이는 해당 양만큼 조정됩니다. GUI 레이어는 항상 디스플레이나 게임 창 또는 애플리케이션 표면의 크기에 맞게 만들어지므로 이 값을 1이 아닌 다른 값으로 설정하면 기본적으로 너비와 높이에 따라 픽셀 수가 조정됩니다. 디스플레이가 1024x768이고 배율을 0.5로 조정하도록 설정하면 GUI 레이어가 디스플레이 크기의 절반이 되어 픽셀 크기가 효과적으로 두 배가 됩니다.

display_set_gui_maximise(0.5, 0.5);

이 방법으로 값을 설정하면 디스플레이 또는 게임 창의 그리기 위치도 (0,0)으로 설정되므로 오프셋 값을 제공하여 (0,0) 위치를 "이동"할 수 있습니다.

var pos = application_get_position();
display_set_gui_maximise(0.5, 0.5, pos[0], pos[1]);

마지막으로 스케일링 팩터를 -1로 설정하여 이 기능을 사용하여 GUI 계층을 재설정할 수 있습니다. 이렇게 하면 GUI 레이어가 다시 1:1 비율로 설정되고 그리기 위치가 디스플레이나 창이 아닌 응용 프로그램 표면의 (0,0) 위치로 설정됩니다.

display_set_gui_maximise(-1, -1);

통사론:

display_set_gui_maximise(xscale, yscale, xoffset, yoffset);

논쟁 유형 설명
xscale 진짜 OPTIONAL The horizontal scaling factor (use -1 to reset to default).
yscale 진짜 OPTIONAL The vertical scaling factor (use -1 to reset to default).
xoffset 진짜 OPTIONAL The x offset position for drawing.
yoffset 진짜 OPTIONAL The y offset position for drawing.

예시:

display_set_gui_size(display_get_width() / 2, display_get_height() / 2);
display_set_gui_maximise(2, 2, 0, 0);

위의 코드는 그리기 GUI 이벤트를 주어진 너비와 높이로 잠그고 해당 비율을 사용하여 디스플레이에 맞게 모든 구성 요소의 크기를 조정합니다.
