gif_save

이 기능을 사용하면 GIF 애니메이션을 저장할 수 있습니다. 함수에서 반환된 GIF 인덱스를 제공합니다.gif_open() ) 뿐만 아니라 저장할 파일 이름입니다. GameMaker 는 자동으로 .gif 파일 확장명이므로 저장된 파일을 GIF로 식별하려면 이를 파일 이름 문자열의 일부로 포함해야 합니다. 생성된 GIF는 Universal 884 팔레트를 사용하여 팔레타이징됩니다(자세한 내용은 여기 참조).

함수가 성공하고 GIF가 올바르게 저장되면 0을 반환하고, 그렇지 않으면 -1을 반환합니다.

통사론:

gif_save(gif_index, fname);

논쟁 유형 설명
gif_index The ID of gif to save
fname The filename to use for the gif

보고:

예시:

if save_gif == true
{
    if count == 0
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if count < 30
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        gif_save(gif_image, "GameCapture.gif");
        count = 0;
        save_gif = false;
    }
    count++;
}

위의 코드는 응용 프로그램 표면에서 가져온 30프레임의 GIF 이미지 파일을 만든 다음 저장합니다.
