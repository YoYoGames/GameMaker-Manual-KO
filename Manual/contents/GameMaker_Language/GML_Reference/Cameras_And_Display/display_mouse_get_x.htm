display_mouse_get_x

이 함수는 화면 내에서 마우스 x 위치를 반환합니다. 이 기능은 Windows 및 macOS 대상에서 사용할 때만 제대로 작동합니다.

HTML5에서도 사용할 수 있지만 캔버스 자체의 0,0에 상대적인 값만 반환하고 마우스가 캔버스 외부에 있는 동안에는 값을 반환하지 않습니다.

다른 장치의 경우 0을 반환하며 다음을 사용해야 합니다.device_mouse_raw_x() 그리고device_mouse_raw_y() 대신 기능.

통사론:

display_mouse_get_x();

보고:

Real

예시:

my_x = display_mouse_get_x();

이것은 my_x를 디스플레이를 기준으로 한 마우스의 x 좌표로 설정합니다.
