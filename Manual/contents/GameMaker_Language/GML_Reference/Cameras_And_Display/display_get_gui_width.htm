display_get_gui_width

이 기능을 사용하면 Draw GUI Event 에서 사용되는 GUI의 너비(픽셀 단위)를 얻을 수 있습니다.

통사론:

display_get_gui_width();

보고:

예시:

ads_move(display_get_gui_width() - ads_get_display_width(0), 0, 0);

위의 코드는 광고가 디스플레이의 오른쪽 상단에 표시되도록 설정합니다.
