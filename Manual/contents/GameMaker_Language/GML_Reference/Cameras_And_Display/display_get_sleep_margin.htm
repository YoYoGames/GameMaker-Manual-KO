display_get_sleep_margin

이 함수는 게임의 렌더 타이밍에 사용되는 현재 휴면 마진 값을 가져오는 데 사용할 수 있으며 밀리초 값을 반환합니다. 표시 시간에 대한 자세한 내용은 다음을 참조하십시오.display_set_timing_method().

통사론:

display_get_sleep_margin();

보고:

예시:

if display_get_timing_method() != tm_sleep
{
    display_set_timing_method(tm_sleep);
    if display_get_sleep_margin() != 20
    {
        display_set_sleep_margin(20);
    }
}

위의 코드는 타이밍 방법을 확인한 다음 설정되지 않은 경우 tm_sleep 그것은 그것으로 설정되고 수면 여백은 20으로 설정됩니다.
