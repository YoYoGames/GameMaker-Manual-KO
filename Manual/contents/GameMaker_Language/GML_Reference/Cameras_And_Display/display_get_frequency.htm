display_get_frequency

이 함수는 게임이 재생되고 있는 디스플레이의 빈도(또는 재생 빈도)를 반환합니다. 실제 값을 초당 프레임으로 반환하므로 예를 들어 모니터가 60hz인 경우 game_set_speed()>60 이 표시되고 144hz에서 실행 중인 경우 144 가 표시되는 식입니다.

참고 이 기능은 HTML5에서 지원되지 않습니다.

통사론:

display_get_frequency();

보고:

예시:

var _freq = display_get_frequency();

if (_freq >= 240)
{
    game_set_speed(240, gamespeed_fps);
}
else if (_freq >= 120)
{
    game_set_speed(120, gamespeed_fps);
}
else if (_freq >= 60)
{
    game_set_speed(60, gamespeed_fps);
}
else
{
    game_set_speed(30, gamespeed_fps);
}

위의 코드는 디스플레이의 주파수를 가져오고 240, 120, 60 또는 30fps에서 실행되는 게임을 설정하는 몇 가지 조건을 실행합니다. 즉, 디스플레이가 90hz이면 게임이 60fps로 실행되고, 144hz이면 120fps로 실행됩니다. 물론 디스플레이 주파수를 직접 전달할 수 있습니다. 그래서 게임의 프레임 레이트로 사용됩니다.
