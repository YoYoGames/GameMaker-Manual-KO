display_mouse_set

이 기능을 사용하면 예를 들어 FPS 게임에 유용할 수 있는 게임 디스플레이 내에서 마우스 위치를 변경하거나 설정할 수 있습니다. 이 기능은 게임에 포커스가 있고 사용 중일 때만 작동합니다. ALT + Tab 마우스의 잠금을 해제합니다.

통사론:

display_mouse_set(x, y);

논쟁 유형 설명
x The x coordinate in the display.
y The y coordinate in the display.

보고:

예시:

display_mouse_set(display_get_width() / 2, display_get_height() / 2);

위의 코드는 게임 디스플레이에서 마우스를 중앙에 배치합니다.
