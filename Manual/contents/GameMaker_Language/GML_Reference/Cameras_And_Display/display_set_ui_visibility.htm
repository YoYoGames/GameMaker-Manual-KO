display_set_ui_visibility

이 기능은 Android 에서만 시스템 UI를 표시하거나 숨기는 데 사용할 수 있습니다. 이 기능을 사용하려면 하나 이상의 시스템 플래그 를 정수 값으로 제공해야 합니다. 하나 이상의 플래그를 사용하는 경우 아래 예와 같이 비트 단위 "또는"을 사용하여 이러한 플래그를 병합해야 합니다. 여기 에서 Android 시스템 플래그 목록을 찾을 수 있습니다.

통사론:

display_set_ui_visibility(flags);

논쟁 유형 깃발
flags 진짜 The Android SYSTEM flags to use (as an integer value).

보고:

N/A

예시:

var flags = 1024|4096;
display_set_ui_visibility(flags);

위의 코드는 Android 시스템 플래그 "SYSTEM_UI_FLAG_IMMERSIVE_STICKY" 및 "SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN"을 사용하여 디스플레이의 가시성을 설정합니다.
