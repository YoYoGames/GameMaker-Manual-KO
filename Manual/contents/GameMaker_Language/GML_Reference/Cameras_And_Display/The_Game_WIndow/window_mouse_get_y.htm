window_mouse_get_y

이 기능을 사용하면 HTML5 게임인 경우 브라우저 내에서 또는 Windows, Ubuntu(Linux) 또는 macOS 게임인 경우 디스플레이 내에서 마우스 커서의 y 위치(픽셀 단위)를 얻을 수 있습니다.

참고: 일반 마우스 기능은 마우스 입력 섹션을 참조하십시오.

통사론:

window_mouse_get_y();

보고:

예시:

wy = window_mouse_get_y();

위의 코드는 "wy" 변수에 마우스의 현재 y축 창 위치를 저장합니다.
