window_set_caption

이 기능을 사용하여 현재 있는 방의 창 캡션을 변경하거나 설정할 수 있습니다. 이 캡션은 게임이 전체 화면 모드가 아닐 때 창 상단, 게임 아이콘 옆에 나타납니다.

통사론:

window_set_caption(caption);

논쟁 유형 설명
caption The new caption.

보고:

예시:

if window_get_caption() != ""
{
    window_set_caption("");
}

위의 코드는 Windows 캡션을 확인하여 텍스트가 있는지 여부를 확인하고 텍스트가 있으면 캡션이 표시되지 않도록 빈 문자열로 설정합니다.
