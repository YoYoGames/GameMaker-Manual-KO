window_has_focus

이 함수를 사용하면 창(또는 탭) 상태를 폴링할 수 있으며 포커스를 잃으면 함수가 반환됩니다. false 그렇지 않으면 반환됩니다 true. 대부분의 경우 간단히 사용할 수 있습니다.os_is_paused() 기능을 사용하여 테스트하지만 일부 매우 특정한 경우(예: Chrome 앱의 게임)에는 해당 기능이 실행되지 않으므로 이 기능을 대신 사용해야 합니다.

참고: 이 기능은 HTML5, Windows 및 MacOS에서만 유효합니다.

통사론:

window_has_focus();

보고:

예시:

if !window_has_focus()
{
    PauseGame();
}

위의 코드는 게임 창이 포커스에 있는지 확인하고 함수가 반환되는지 확인합니다. false 함수가 호출됩니다.
