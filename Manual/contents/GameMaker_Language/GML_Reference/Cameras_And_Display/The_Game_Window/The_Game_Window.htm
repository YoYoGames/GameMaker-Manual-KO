게임 창

생성한 게임은 창에서 발생하며(전체 화면일 때도) 이 창에는 위치, 크기, 전체 화면인지 여부 등과 같은 여러 속성이 있습니다. 이러한 세부 정보는 일반적으로 방 크기에 따라 자동으로 설정됩니다. 대상 플랫폼에 대한 게임 옵션 의 설정과 함께 활성화된 보기 포트가 있지만 이 페이지에 나열된 기능을 사용하여 게임 중에 변경할 수 있습니다.

참고 이러한 기능은 Windows, Ubuntu (Linux), macOS 및 HTML5 대상 모듈 전용이며 다른 장치에서는 작동하지 않을 수 있습니다.

다음 이미지는 일부 일반 창 기능이 서로 관련되고 상호 작용하는 방식을 보여줍니다.

위의 이미지는 브라우저에서 그린 게임 창을 보여주지만 다른 대상의 경우 브라우저를 디스플레이로 교체할 수 있으므로 다음과 같은 기능을 사용할 수 있습니다. window_get_y() 디스플레이를 기준으로 게임 창 상단의 위치를 반환합니다.
