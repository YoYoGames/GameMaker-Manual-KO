window_set_colour

이 기능은 게임 창의 배경색을 설정할 수 있습니다. 이 색상은 보기가 차지하지 않는 게임 창 영역에 사용할 색상을 나타냅니다. 다음 이미지는 이를 보여줍니다.

위의 이미지에는 각각 다른 위치에 그려진 두 개의 뷰 포트가 있는 두 개의 뷰가 있습니다. 이렇게 하면 두 포트를 모두 수용할 수 있도록 게임 창을 확장하고 창 색상을 사용하여 보기가 표시되지 않는 배경에 색을 지정합니다.

통사론:

window_set_colour(colour);

논쟁 유형 설명
colour The colour to set the region.

보고:

예시:

if window_get_colour() != c_black
{
    window_set_colour(c_black);
}

위의 코드는 창 색상을 확인하여 검은색으로 설정되어 있는지 확인하고, 그렇지 않으면 검은색으로 설정합니다.
