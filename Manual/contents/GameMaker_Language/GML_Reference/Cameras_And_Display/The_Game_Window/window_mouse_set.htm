window_mouse_set

이 기능을 사용하면 예를 들어 FPS 게임에 유용할 수 있는 게임 창 내에서 마우스 위치를 변경하거나 설정할 수 있습니다. 이 기능은 게임에 포커스가 있을 때만 작동하며 Alt + 탭을 사용하면 마우스 잠금이 해제됩니다.

참고 일반 마우스 기능은 마우스 입력 섹션을 참조하십시오.

통사론:

window_mouse_set(x, y);

논쟁 유형 설명
x The x coordinate in the window.
y The y coordinate in the window.

보고:

예시:

window_mouse_set(window_get_width() / 2, window_get_height() / 2);

위의 코드는 게임 창에서 마우스를 중앙에 배치합니다.
