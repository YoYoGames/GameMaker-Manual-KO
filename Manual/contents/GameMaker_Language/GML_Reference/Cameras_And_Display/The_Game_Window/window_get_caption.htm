window_get_caption

이 함수는 창의 캡션을 반환하고(창의 상단에 아이콘 옆에 나타나는 텍스트임) 기본적으로 이것은 현재 있는 방의 캡션을 표시합니다.

통사론:

window_get_caption();

보고:

예시:

if window_get_caption() != ""
{
    window_set_caption("");
}

위의 코드는 Windows 캡션을 확인하여 텍스트가 있는지 여부를 확인하고 텍스트가 있으면 캡션이 표시되지 않도록 빈 문자열로 설정합니다.
