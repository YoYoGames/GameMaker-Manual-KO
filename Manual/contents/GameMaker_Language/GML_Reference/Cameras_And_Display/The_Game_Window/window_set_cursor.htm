window_set_cursor

이 기능을 사용하면 게임 창의 커서를 아래 나열된 상수 중 하나로 설정할 수 있습니다(현재 사용 중인 커서를 찾으려면 이 기능을 사용할 수 있습니다window_get_cursor() 또한 이러한 상수 중 하나를 반환합니다):

일정한 커서
cr_none
cr_default
cr_arrow
cr_cross
cr_beam
cr_size_nesw
cr_size_ns
cr_size_nwse
cr_size_we
cr_uparrow
cr_hourglass
cr_drag
cr_appstart
cr_handpoint
cr_size_all

통사론:

window_set_cursor(cursor);

논쟁 유형 설명
cursor 커서 상수 The cursor to set for the game window.

보고:

N/A

예시:

if mouse_check_button_pressed(mb_left)
{
    window_set_cursor(cr_drag);
}

위의 코드는 왼쪽 마우스 버튼이 눌려진 경우 창 커서를 표준 창 드래그 커서로 변경합니다.
