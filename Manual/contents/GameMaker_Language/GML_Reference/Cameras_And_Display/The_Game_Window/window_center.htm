window_center

이 기능을 사용하면 대상 모듈이 Windows, Ubuntu(Linux) 또는 macOS인 경우 디스플레이의 중앙에 게임 창을 표시하거나 대상 모듈이 HTML5인 경우 브라우저의 중앙에 배치할 수 있습니다. 이 기능은 다른 장치에는 영향을 미치지 않습니다.

참고 현재 단계에서 게임 창의 크기를 조정한 경우(예: 전체 화면에서 창으로 전환하거나 window_set_size() ), 새 창을 가운데에 맞추려면 이 함수를 적어도 한 단계 후에 호출해야 합니다(예: alarm 사용). 그렇지 않으면 올바르게 작동하지 않습니다.

통사론:

window_center();

보고:

예시:

if !window_get_fullscreen()
{
    window_center();
}

위의 코드는 전체 화면 모드가 아닌 경우 디스플레이 내에서 게임 창을 중앙에 배치합니다.
