window_set_max_height

이 기능은 게임의 최대 창 높이를 설정하는 데 사용할 수 있습니다. 대상 플랫폼의 게임 옵션에서 창 크기 조정 옵션을 활성화하면 플레이어는 원하는 크기로 게임 창의 크기를 조정할 수 있지만 이 기능을 사용하면 최대 높이를 지정한 크기로 제한할 수 있습니다. 기본 동작(즉, 최소값 없음)으로 돌아가려면 -1 값을 사용하십시오.

참고: 이 기능은 Windows, macOS 및 Linux 플랫폼에서만 사용할 수 있습니다.

통사론:

window_set_max_height(height);

논쟁 유형 설명
height The maximum height in pixels for the game window.

보고:

예시:

window_set_min_width(640);
window_set_min_height(480);
window_set_max_width(1280);
window_set_max_height(960);

위의 코드는 게임 창의 최소 및 최대 너비와 높이를 설정합니다.
