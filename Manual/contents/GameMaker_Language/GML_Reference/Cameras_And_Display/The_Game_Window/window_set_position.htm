window_set_position

이 기능을 사용하면 게임 창을 디스플레이(macOS, Linux(Ubuntu) 및 Windows) 또는 브라우저(HTML5) 내의 특정 위치로 설정할 수 있습니다.

참고: HTML5 게임이 사용자 정의를 사용하는 경우 index.html 캔버스를 고정 위치로 설정하면 이 기능은 창 위치에 영향을 주지 않습니다.

통사론:

window_set_position(x, y);

논쟁 유형 설명
x The x coordinate of where to position the window.
y The y coordinate of where to position the window.

보고:

예시:

window_set_position(0, 0);

위의 코드는 브라우저 또는 디스플레이의 왼쪽 상단 모서리에 게임 창을 배치합니다(사용 중인 대상 모듈에 따라 다름).
