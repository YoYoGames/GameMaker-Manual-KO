window_handle

이 함수를 사용하면 내부 Windows id 값(HWND, 포인터 )을 얻을 수 있습니다. 이 기능은 정말로 필요한 확장 작성자에게만 유용합니다. window handle DLL 코드에서 Windows API를 호출합니다(반환된 포인터는 문자열로 캐스트되어야 하고 C++에서는 HWND로 캐스트해야 합니다). 아래 표는 반환되는 것과 함께 지원되는 플랫폼을 보여줍니다.

플랫폼 보고
창 창 HWND
맥 OS NSWindow 클래스
우분투(리눅스) X윈도우 핸들
HTML5 캔버스 ID

통사론:

window_handle();

보고:

Pointer

예시:

win_id = windows_handle();

위의 코드는 게임 창 ID를 변수에 저장합니다.
