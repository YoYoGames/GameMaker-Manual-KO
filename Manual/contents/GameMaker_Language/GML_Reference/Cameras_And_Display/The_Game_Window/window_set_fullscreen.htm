window_set_fullscreen

이 기능을 사용하면 게임 창을 전체 화면으로 설정할 수 있습니다(true ) 여부(false ). macOS 대상의 경우 "전체 화면에서 시작" 옵션 을 선택 취소하고 게임 옵션에서 "플레이어가 게임 창 크기를 조정할 수 있도록 허용" 옵션을 선택해야 합니다. 그렇지 않으면 이 기능이 실패합니다. 또한 이 기능은 "클릭 가능한" 콜백으로 추가되지 않는 한 HTML5에서 작동 하지 않습니다 (자세한 내용은 여기 참조).

통사론:

window_set_fullscreen(full);

논쟁 유형 설명
full Whether to set the screen to fullscreen (true) or not (false).

보고:

예시:

if mouse_check_button_pressed(mb_left)
{
    if window_get_fullscreen()
    {
        window_set_fullscreen(false);
    }
    else
    {
        window_set_fullscreen(true);
    }
}

위의 코드는 마우스 버튼 누름을 확인한 다음, 아직 전체 화면이 아닌 경우 창을 전체 화면으로 설정하고, 이미 있는 경우 창으로 설정합니다.
