window_get_cursor

이 함수를 사용하면 게임 창에서 사용 중인 현재 커서를 가져올 수 있으며 반환 값은 아래 나열된 상수 중 하나입니다(사용 중인 현재 커서를 설정하려면 함수를 사용할 수 있습니다window_set_cursor() 또한 이러한 상수를 사용합니다):

커서 상수
일정한 커서
cr_none
cr_default
cr_arrow
cr_cross
cr_beam
cr_size_nesw
cr_size_ns
cr_size_nwse
cr_size_we
cr_uparrow
cr_hourglass
cr_drag
cr_appstart
cr_handpoint
cr_size_all

통사론:

window_get_cursor();

보고:

Cursor Constant

예시:

if mouse_check_button_pressed(mb_left)
{
    if window_get_cursor() != cr_drag window_set_cursor(cr_drag);
}

위의 코드는 마우스 왼쪽 버튼이 눌렸고 이전에 변경되지 않은 경우 창 커서를 표준 창 드래그 커서로 변경합니다.
