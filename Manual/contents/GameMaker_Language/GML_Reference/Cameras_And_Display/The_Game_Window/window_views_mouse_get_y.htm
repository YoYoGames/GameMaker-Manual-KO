window_views_mouse_get_y

이 함수는 모든 활성 뷰에 대한 마우스의 y 좌표를 반환하고 동일한 값을 반환합니다.mouse_y.

참고: 일반 마우스 기능은 마우스 입력 섹션을 참조하십시오.

통사론:

window_views_mouse_get_y();

보고:

예시:

myy = window_views_mouse_get_y();

이것은 myy를 모든 활성 뷰를 기준으로 창에서 마우스의 y 좌표로 설정합니다.
