window_set_size

이 기능을 사용하여 게임 창의 크기를 변경할 수 있습니다. 데스크탑 플랫폼에서 이것은 일반적으로 제목 표시줄과 테두리를 제외한 창의 클라이언트 영역을 나타냅니다.

이 기능을 사용하여 게임 창의 크기를 변경하면 창의 내용이 배경 캔버스에 1:1 비율로 맞도록 크기가 조정되어 이미지가 선명하고 선명하게 유지됩니다(크기가 조정되었지만 창이 방 또는 보기 크기와 다름). 예를 들어, 200 x 150 픽셀 창이 있고 플레이어가 테두리를 드래그하여 창을 늘리면 최종 창 크기는 400 x 300 픽셀일 수 있지만 그리기 캔버스는 여전히 200 x 150이므로 이미지가 흐려집니다. 픽셀 비율이 이제 1:2이기 때문입니다. 그러나 이 기능을 사용하여 창 크기를 변경하면 이미지가 새 창 크기에 맞게 조정되더라도 1:1 비율이 유지됩니다.

통사론:

window_set_size( w, h );

논쟁 유형 설명
w The new width of the window.
h The new height of the window.

보고:

N/A

예시:

if mouse_check_button_pressed(mb_left)
{
    global.scale += 1;
    window_set_size(200 * global.scale, 150 * global.scale);
}

위의 코드는 마우스 버튼을 눌렀을 때 "global.scale" 변수의 값에 따라 창의 크기를 변경합니다.
