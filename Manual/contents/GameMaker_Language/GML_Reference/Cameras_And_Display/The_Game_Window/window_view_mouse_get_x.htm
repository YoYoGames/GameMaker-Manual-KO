window_view_mouse_get_x

이 함수는 선택한 보기에 상대적인 마우스 x 위치를 반환합니다.

참고: 일반 마우스 기능은 마우스 입력 섹션을 참조하십시오.

통사론:

window_view_mouse_get_x( id );

논쟁 유형 설명
id 진짜 The id of the view to compare the mouse position to.

보고:

Real

예시:

if mouse_check_button_pressed(mb_left)
{
    var xx, yy;
    xx = window_view_mouse_get_x(0);
    yy = window_view_mouse_get_y(0);

    if xx > 0 && xx < 32 && yy > 0 && yy < 32
    {
        b_press[0] = true;
    }
}

위의 코드는 마우스 버튼이 눌렸는지 확인하고, 그렇다면 view[0]에 상대적인 마우스 위치를 가져와서 변수를 true로 설정해야 하는지 비교합니다.
