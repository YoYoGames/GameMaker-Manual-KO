camera_destroy

이 기능을 사용하면 모든 카메라를 파괴할 수 있습니다. 함수를 호출할 때 고유한 카메라 ID 값을 제공합니다. camera_create_* 기능 또는view_camera 룸 편집기를 사용하여 뷰 포트 및 뷰를 설정하는 경우 배열입니다. 동일한 단계에서 해당 보기에 새 카메라를 할당하지 않는 한 현재 보이는 보기에 할당된 카메라를 절대 파괴해서는 안 되며, 게임에서 더 이상 필요하지 않을 때 코드를 통해 생성한 카메라를 항상 파괴해야 합니다. 메모리 누수를 방지하고 Room Editor에 할당된 기본 카메라가 있는 경우 기본 카메라를 제거할 수도 있지만 뷰 포트에 새 카메라를 할당(또는 비활성화)해야 합니다. 그렇지 않으면 이상한 결과를 얻게 됩니다.

통사론:

camera_destroy(camera_id)

논쟁 유형 설명
camera_id The unique camera ID value returned when you created the camera.

보고:

N/A

예시:

camera_destroy(view_camera[0]);
view_camera[0] = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);

위의 코드는 현재 뷰 포트 [0]에 할당된 카메라를 파괴한 다음 새 카메라를 만들고 해당 포트에 ID를 할당합니다.
