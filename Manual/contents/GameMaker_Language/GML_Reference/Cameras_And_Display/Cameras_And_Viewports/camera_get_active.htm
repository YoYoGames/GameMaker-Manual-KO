camera_get_active

이 기능은 현재 활성화된 카메라의 고유한 카메라 ID 값을 검색하는 데 사용할 수 있습니다.

통사론:

camera_get_active()

보고:

Camera ID

예시:

var active = camera_get_active();
if active != view_camera[0]
{
    view_camera[0] = active;
}

위의 코드는 활성 카메라의 카메라 ID를 가져오고 이를 사용하도록 포트[0]의 뷰 카메라를 설정합니다.
