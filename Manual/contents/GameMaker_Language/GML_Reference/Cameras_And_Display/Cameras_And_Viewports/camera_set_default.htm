camera_set_default

이 기능을 사용하여 이전에 다음 중 하나를 사용하여 만든 사용자 지정 카메라를 사용하도록 기본 카메라를 설정할 수 있습니다.camera_create() 기능. 활성 뷰 포트 또는 뷰 카메라가 없는 방을 생성할 때 GameMaker 는 여전히 카메라를 사용하여 게임의 동작을 보여줍니다. 이 카메라를 기본 카메라라고 하며 다른 카메라와 마찬가지로 설정 및 조작(심지어 파괴)할 수 있습니다.

통사론:

camera_set_default(camera_id)

논쟁 유형 설명
camera_id The unique camera ID value returned when you created the camera

보고:

예시:

var new_cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);
camera_set_default(new_cam);

위의 코드는 새 카메라를 만든 다음 사용할 기본 카메라로 할당합니다.
