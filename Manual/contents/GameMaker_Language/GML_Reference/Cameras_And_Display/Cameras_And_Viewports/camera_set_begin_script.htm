camera_set_begin_script

이 함수는 카메라가 보이는 활성 뷰 포트에 할당된 모든 게임 프레임의 시작 부분에서 호출될 스크립트 함수 를 설정하는 데 사용할 수 있습니다. 고유한 카메라 ID 값을 제공합니다(다른 camera_create() 함수) 및 호출할 함수의 ID입니다. 카메라에 스크립트가 첨부된 순서와 카메라 뷰의 실제 렌더링은 다음과 같습니다.

  • 모든 가시 및 활성 뷰 포트에 대한 카메라에는 다음과 같은 업데이트 기능 이 있습니다.
  • 그런 다음 각 보기에 대해 다음을 수행합니다.
    • 해당 뷰에 대한 카메라의 시작 기능 이 호출됩니다.
    • 해당 뷰에 대해 그리기 이벤트가 실행됩니다.
    • 카메라의 종료 기능 이 호출됩니다.
  • 다음 활성 표시 보기로 이동하고 반복합니다...

통사론:

camera_set_begin_script(camera_id, script)

논쟁 유형 설명
camera_id 카메라 ID The unique camera ID value returned when you created the camera.
script 스크립트 기능 The script function to run each game frame

보고:

N/A

예시:

camera_set_begin_script(view_camera[0], beginCamera);

위의 코드는 뷰포트 [0]에 할당된 카메라의 스크립트 시작 기능을 설정합니다.
