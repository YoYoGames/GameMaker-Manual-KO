camera_set_proj_mat

이 함수는 주어진 카메라에 대한 투영 행렬 을 설정합니다. 고유한 카메라 ID 값을 제공합니다(다른camera_create() 기능) 및 적용할 투영 행렬. 특히 Matrix Functions 섹션에서 투영 행렬 생성에 대해 자세히 알아볼 수 있습니다. matrix_build_projection_perspective() 그리고 matrix_build_projection_ortho().

이 기능을 사용할 때 염두에 두어야 할 몇 가지 사항이 있습니다.

  • 카메라가 자동 개체 추적을 수행하는 경우 - 즉, 다음을 사용하여 생성되었습니다.camera_create_view() -1이 아닌 개체 인덱스/인스턴스 ID를 사용하거나 룸 편집기에 정의된 카메라를 설정하거나 기본 카메라를 설정하는 경우 이 매트릭스는 다음 게임 프레임에 덮어쓰여집니다.
  • 이 기능을 사용하여 원근 투영(기본 직교 투영과 반대)으로 전환하면 타일링된 스프라이트의 자동 그리기가 작동을 멈추고 수동으로 처리해야 합니다. 여기에는 수평 또는 수직 타일링 및 draw_sprite_tiled() 그리고 draw_sprite_tiled_ext() 원근 투영을 사용하는 경우 타일링하는 대신 스프라이트를 한 번만 그립니다.

통사론:

camera_set_proj_mat(camera_id, matrix)

논쟁 유형 설명
camera_id The unique camera ID value returned when you created the camera.
matrix The unique matrix ID returned when you created the matrix.

보고:

예시:

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

위의 코드는 새 카메라를 생성하고 이를 뷰 포트[0]에 할당합니다. 그러면 이 카메라는 뷰와 투영 행렬이 설정됩니다.
