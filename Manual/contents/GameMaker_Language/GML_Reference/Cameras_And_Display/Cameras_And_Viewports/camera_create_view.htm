camera_create_view

이 함수는 새 카메라를 생성한 다음 해당 뷰를 방으로 설정한 후 고유 ID 값을 반환하여 향후 함수 호출에서 카메라를 식별합니다. 카메라는 위치, 크기 및 기타 세부 사항에 대해 이 기능에서 설정한 값을 사용하여 생성되며 이는 Room Editor에서 설정한 값과 관련됩니다( 여기 참조). 이 함수는 x/y 위치 및 너비/높이 인수만 필수인 가변 수의 인수 를 사용할 수 있습니다. 제공되지 않은 모든 인수는 기본 인수를 사용합니다.

카메라는 모든 동적 리소스와 같으며 생성될 때 메모리를 차지합니다. 즉, 불필요하게 카메라를 생성하지 않도록 주의하고 함수를 호출하여 필요하지 않은 카메라를 파괴하지 않도록 주의해야 합니다.camera_destroy(). 이것은 또한 카메라가 할당된 경우 방에서 방으로 유지된다는 것을 의미합니다. global 변수(인스턴스 또는 로컬 변수에 할당될 때도 유지되지만 방을 변경한 후에는 액세스할 수 없으므로 메모리 누수), 예를 들어 새 카메라를 만들고 전역 변수에 할당할 수 있습니다., 그리고 각 방의 시작 부분에 다음을 사용하십시오. camera_set_* 뷰포트에 할당하기 전에 속성을 설정하는 기능입니다.

통사론:

camera_create_view(room_x, room_y, width, height, [angle, object, x_speed, y_speed, x_border, y_border])

논쟁 유형 설명
room_x 진짜 The x position (within the room) of the left side of the area the camera view captures.
room_y 진짜 The y position (within the room) of the top of the area the camera view captures.
width 진짜 The width of the area the view captures.
height 진짜 The height of the area the view captures.
angle 진짜 OPTIONAL The angle to set the for the camera view(from 0° - 360°, default is 0).
object 개체 자산 또는 인스턴스 ID OPTIONAL The object ID (or instance ID) for the camera view to follow (default is -1, no object).
x_speed 진짜 OPTIONAL The maximum horizontal speed the view can move at (set to -1 for "instant" following, default is -1).
y_speed 진짜 OPTIONAL The maximum vertical speed the view can move at (set to -1 for "instant" following, default is -1).
x_border 진짜 OPTIONAL The minimum horizontal space between the edge of the view area and the instance it is set to follow (before the view begins moving and if it is set to follow an instance, default is 0).
y_border 진짜 OPTIONAL The minimum vertical space between the edge of the view area and the instance it is set to follow (before the view begins moving and if it is set to follow an instance, default is 0).

보고:

Camera ID

예시:

camera_destroy(view_camera[0]);
view_camera[0] = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);

위의 코드는 뷰포트[0]에 할당된 현재 카메라를 제거한 다음 새 카메라를 만들고 뷰 속성을 정의한 다음 뷰포트[0]에 할당합니다.
