view_set_visible

이 함수는 주어진 뷰포트의 가시성을 설정하는데 사용될 수 있습니다. 이 함수는 뷰 포트 인덱스(0에서 7까지)와 부울 값을 취합니다. true 그것이 눈에 보이는 경우와 false 그렇지 않은 경우.

통사론:

view_set_visible(view_port, visible)

논쟁 유형 설명
view_port The view port to target (0 - 7)
visible Visibility toggle (true is visible and false is invisible)

보고:

N/A

예시:

if !view_get_visible(0)
{
    view_set_visible(0, true);
}

위의 코드는 뷰 포트[0]이 보이는지 확인하고 그렇지 않은 경우 보이는 것으로 설정되어 있는지 확인합니다.
