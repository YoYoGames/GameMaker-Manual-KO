camera_set_view_target

이 기능을 사용하여 실내 뷰 카메라의 추적 대상을 설정할 수 있습니다. 고유한 카메라 ID 값을 제공합니다(다른camera_create() 기능) 그런 다음 카메라 보기를 설정하려는 대상 인스턴스 또는 개체 ID를 지정합니다. 개체 ID를 선택하고 방에 해당 개체의 인스턴스가 두 개 이상 있는 경우 GameMaker 가 팔로우하려는 인스턴스를 알 수 있는 방법이 없으므로 그 중 하나가 될 수 있습니다.

통사론:

camera_set_view_target(camera_id, instance_id/object_id)

논쟁 유형 설명
camera_id 카메라 ID The unique camera ID value returned when you created the camera
instance_id/object_id 개체 자산 또는 인스턴스 ID Instance or object to have the camera target for following

보고:

N/A

예시:

camera_set_view_target(view_camera[0], id);

위의 코드는 뷰 포트[0]에 할당된 카메라의 뷰 카메라 대상을 코드를 실행하는 인스턴스의 대상으로 설정합니다.
