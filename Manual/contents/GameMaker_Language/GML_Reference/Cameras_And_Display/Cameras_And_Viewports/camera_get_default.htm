camera_get_default

이 기능은 기본 카메라(게임룸에서 카메라 보기 또는 포트가 활성화되지 않은 경우 GameMaker 에서 사용하는 카메라)의 고유한 카메라 ID 값을 검색하는 데 사용할 수 있습니다.

통사론:

camera_get_default()

보고:

Camera ID

예시:

var def = camera_get_default();
view_camera[0] = def;

위의 코드는 기본 카메라의 카메라 ID를 가져오고 이를 사용하도록 포트[0]의 뷰 카메라를 설정합니다.
