camera_set_view_border

이 기능을 사용하여 방 내에서 뷰 카메라의 추종 경계를 설정할 수 있습니다. 고유한 카메라 ID 값을 제공합니다(다른 camera_create 함수) 그런 다음 x 테두리 크기와 y 테두리 크기(가로 및 세로)를 지정합니다. 이 값은 보기 카메라에 따라야 할 인스턴스 대상이 할당된 경우에만 사용됩니다(룸 편집기에서 또는 기능을 사용하여camera_set_view_target() ) 보기가 따라야 할 위치를 업데이트하기 전에 인스턴스가 보기의 경계에서 얼마나 멀리 떨어져 있어야 하는지와 관련이 있습니다.

통사론:

camera_set_view_border(camera_id, x_border, y_border)

논쟁 유형 설명
camera_id The unique camera ID value returned when you created the camera.
x_border The border (in pixels) for the view camera along the horizontal axis
y_border The border (in pixels) for the view camera along the vertical axis

보고:

예시:

camera_set_view_border(view_camera[0], 64, 64);

위의 코드는 뷰 포트[0]에 할당된 카메라의 뷰 카메라 경계를 설정합니다.

뒤로: 카메라 및 뷰 포트
