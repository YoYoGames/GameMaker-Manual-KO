view_get_visible

이 기능은 주어진 뷰포트의 가시성을 확인하는 데 사용할 수 있습니다. 함수가 반환됩니다 true 보이는 경우와 false 그렇지 않은 경우.

통사론:

view_get_visible(view_port)

논쟁 유형 설명
view_port The view port to target (0 - 7)

보고:

Boolean

예시:

if !view_get_visible(0)
{
    view_set_visible(0, true);
}

위의 코드는 뷰 포트[0]이 보이는지 확인하고 그렇지 않은 경우 보이는 것으로 설정되어 있는지 확인합니다.
