view_current

이 읽기 전용 변수는 그리기 이벤트 에서만 유효하며 렌더링되는 현재 뷰 포트를 반환합니다. 그리기 이벤트는 각 보기 포트에 대해 연속적으로 한 번 호출되므로 다양한 보기가 있는 경우 그리기 이벤트 중에 반환 값이 변경됩니다. 따라서 (예를 들어) 게임룸에서 보기 포트[0] 및 보기 포트[1]를 사용할 때 모든 인스턴스에 대한 그리기 이벤트는 각 포트에 대해 한 번씩 두 번 실행되며 이 변수를 사용하여 다음을 확인할 수 있습니다. 현재 어떤 견해가 그려지고 있는지. 일반적으로 여러 뷰 포트가 동시에 방에 표시될 때 단일 포트에만 특정 세부 정보를 렌더링하는 데 사용됩니다. 아래 예제 코드를 참조하십시오.

통사론:

view_current;

보고:

Real (view port index from 0 to 7)

예시:

if view_current == 0
{
    var xx = camera_get_view_x(view_camera[0]);
    var yy = camera_get_view_y(view_camera[0]);
    draw_text(xx + 32, yy + 32, "Player 1");
}
else
{
    var xx = camera_get_view_x(view_camera[1]);
    var yy = camera_get_view_y(view_camera[1]);
    draw_text(xx + 32, yy + 32, "Player 2");
}

위의 코드는 현재 어떤 뷰가 그려지고 있는지 확인한 다음 반환 값에 따라 각 뷰에 다른 텍스트를 그립니다.
