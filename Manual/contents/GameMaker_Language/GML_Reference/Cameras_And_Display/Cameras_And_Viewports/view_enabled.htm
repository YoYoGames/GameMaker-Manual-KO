view_enabled

이 변수는 방 내에서 볼 수 있는 보기 포트가 활성화되었는지 여부를 제어합니다. 뷰 포트를 표시로 설정한 다음 이 옵션을 비활성화하면 뷰 포트를 통해 다른 카메라가 그려지는 대신 전체 방이 창 크기에 맞게 조정된 화면으로 그려집니다.

통사론:

view_enabled;

보고:

(true: enabled, false: disabled)

예시:

if !view_enabled
{
    view_visible[0] = true;
    view_enabled = true;
}

위의 코드는 방에 대한 보기가 활성화되어 있는지 확인하고 그렇지 않은 경우 보기 포트[0]이 보이는지 확인한 다음 방에 대한 보기를 활성화합니다.

뒤로: 카메라 및 뷰 포트
다음: view_visible
