camera_get_view_mat

이 함수는 뷰 매트릭스를 검색하는 데 사용할 수 있습니다. 이 함수는 다른 행렬 함수 에서 사용하거나 다른 카메라의 뷰 행렬을 설정하는 데 사용할 수 있는 행렬 배열을 반환합니다(사용camera_set_view_mat() ). 이 기능은 일반적으로 다음을 사용하여 만든 카메라에 사용됩니다.camera_create(), 하지만 룸 편집기에서 생성된 항목에도 사용할 수 있습니다(또는camera_create_view() ) - 이 경우 반환된 행렬은 주어진 카메라가 뷰 포트에서 처음으로 사용된 후에만 유효합니다.

통사론:

camera_get_view_mat(camera_id)

논쟁 유형 설명
camera_id The unique camera ID value returned when you created the camera

보고:

Matrix Array

예시:

mat = camera_get_view_mat(camera_view[0]);

위의 코드는 뷰 포트[0]에 할당된 카메라의 뷰 매트릭스를 가져옵니다.

뒤로: 카메라 및 뷰 포트
