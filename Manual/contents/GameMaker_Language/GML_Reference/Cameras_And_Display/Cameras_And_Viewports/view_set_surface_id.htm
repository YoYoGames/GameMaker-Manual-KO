view_set_surface_id

이 변수를 사용하여 표면에 그릴 뷰 포트의 내용을 설정할 수 있습니다. 표면으로 작업할 때 종종 화면의 전체 가시 영역을 표면으로 캡처해야 하므로 이 기능을 사용하여 뷰 포트에 할당합니다. 즉, 선택한 뷰 포트에 표시되는 모든 것이 이제 할당된 표면으로 그려지고 해당 뷰 포트의 내용이 더 이상 표시되지 않습니다. 즉, 다음 중 하나가 필요합니다.

  • 새 보기를 활성화하고 해당 보기에서만 표면을 그립니다(사용view_current 어떤 뷰가 그려지고 있는지 확인하기 위해)
  • Draw GUI 또는 인스턴스의 Post Draw 이벤트에서 표면을 그립니다. 이러한 이벤트는 보기와 무관하기 때문입니다.

이 기능을 사용할 때 뷰 포트 인덱스(0에서 7까지)와 표면 인덱스(둘 중 하나application_surface 또는 함수에서 반환된 고유 인덱스 값surface_create() ) 또는 표면이 이전에 지정되었고 제거하려는 경우 값 -1입니다. 뷰 포트를 표면으로 설정하는 방법에 대한 더 많은 예는 변수를 참조하십시오.view_surface_id.

통사론:

view_set_surface_id(view_port, surf)

논쟁 유형 설명
view_port 진짜 The view port to target (0 - 7)
surf 표면 ID The surface to apply to the view

보고:

Real

예시:

if view_get_surface_id(0) == -1
{
    view_set_surface_id(0, global.vSurf);
}

위의 코드는 표면이 뷰 포트[0]에 할당되었는지 확인하고 할당되지 않은 경우 표면이 할당되었는지 확인합니다.
