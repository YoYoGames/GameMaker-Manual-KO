camera_get_proj_mat

이 함수는 투영 행렬을 검색하는 데 사용할 수 있습니다. 이 함수는 다른 행렬 함수 에서 사용하거나 다른 카메라의 투영 행렬을 설정하는 데 사용할 수 있는 행렬 배열을 반환합니다(사용camera_set_proj_mat() ).

통사론:

camera_get_proj_mat(camera_id)

논쟁 유형 설명
camera_id The unique camera ID value returned when you created the camera

보고:

Matrix Array

예시:

mat = camera_get_proj_mat(camera_view[0]);

위의 코드는 뷰 포트[0]에 할당된 카메라의 투영 행렬을 가져옵니다.
