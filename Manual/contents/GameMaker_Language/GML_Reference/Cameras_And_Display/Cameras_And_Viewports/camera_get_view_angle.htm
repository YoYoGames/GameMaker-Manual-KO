camera_get_view_angle

이 함수는 주어진 카메라의 각도를 검색하는 데 사용할 수 있습니다. 반환 값은 0에서 360 사이입니다. 이 기능은 다음을 사용하여 만든 카메라 에만 유효합니다.camera_create_view() 또는 방 편집기에 추가된 사람들을 위해.

통사론:

camera_get_view_angle(camera_id)

논쟁 유형 설명
camera_id The unique camera ID value returned when you created the camera

보고:

예시:

var ang = camera_get_view_angle(view_camera[0]);
if ang != 0
{
    camera_set_view_angle(view_camera[0], 0);
}

위의 코드는 뷰 포트[0]에 할당된 카메라의 각도를 검색한 다음 이를 확인하여 주어진 값과 일치하는지 확인합니다. 그렇지 않으면 뷰 카메라가 해당 각도로 설정됩니다.
