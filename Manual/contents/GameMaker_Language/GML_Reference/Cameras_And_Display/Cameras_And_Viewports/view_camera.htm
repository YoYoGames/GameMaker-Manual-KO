view_camera

이 배열은 지정된 뷰 포트에 할당된 고유한 카메라 ID를 보유하며 새 카메라로 설정하거나 현재 카메라를 가져오기 위해 읽을 수 있으며 카메라가 할당되지 않은 경우 -1을 반환합니다. 방에서 최대 8개의 뷰 포트를 활성화할 수 있으며(배열 값 0에서 7), 이 변수를 카메라 ID 값(함수에서 반환된camera_create() 또는camera_create_view() ). Room Editor를 통해 카메라를 뷰 포트로 설정한 경우 이 변수는 해당 포트의 카메라 ID를 유지합니다.

통사론:

view_camera[0 ... 7];

보고:

Camera ID or -1 (if no camera is assigned)

예시:

if view_camera[0] == -1
{
    view_camera[0] = camera_create_view(0, 0, room_width, room_height);
}

위의 코드는 뷰 포트 [0]에 카메라가 부착되어 있는지 확인하고, 카메라가 없으면 카메라가 생성되어 설정됩니다.
