view_set_camera

이 기능은 카메라를 특정 뷰 포트로 설정합니다. 설정할 뷰 포트(0에서 7까지)를 제공하고 고유한 카메라 ID 값(camera_create() 기능 또는 사용할 때view_get_camera() ). 카메라 ID로 값 -1을 지정하면 뷰 포트에서 카메라가 제거되고 해당 뷰 포트가 활성화되고 표시되는 경우 예측할 수 없는 결과를 얻을 수 있습니다.

통사론:

view_set_camera(view_port, camera_id)

논쟁 유형 설명
view_port The view port to target (0 - 7)
camera_id The unique camera ID value returned when you created the camera

보고:

예시:

var cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);
view_set_camera(0, cam);

위의 코드는 새 카메라를 만든 다음 뷰 포트[0]를 할당합니다.

뒤로: 카메라 및 뷰 포트
