camera_apply

이 함수는 주어진 카메라 설정을 즉시 렌더링되는 현재 카메라 뷰에 적용합니다. 특정 기능을 사용하여 모든 뷰 카메라를 설정할 수 있지만 camera_set_* 기능을 사용하면 다음 게임 프레임까지 렌더링에 새 설정이 사용되지 않지만 이 기능을 사용하면 즉시 변경할 수 있습니다. 여러 보기를 사용하는 경우 그리기 이벤트 에서 함수를 사용해야 하며 변수를 사용할 수 있습니다.view_current 주어진 카메라를 현재 렌더링 중인 카메라 뷰에만 선택적으로 적용합니다. 이것은 카메라 ID 값을 변경하지 않고 해당 카메라에 대한 설정만 변경하며 기본 카메라에서 이 기능을 사용하는 경우(예: 활성화된 뷰 카메라가 없을 때 GameMaker 가 사용하는 내부 카메라) 변경 사항을 덮어씁니다. 방을 그리기 위한 내부 설정에 의해 바로 다음 게임 프레임.

통사론:

camera_apply(camera_id)

논쟁 유형 설명
camera_id The unique camera ID value returned when you created the camera.

보고:

예시:

if view_current == 0
{
    camera_apply(cutscene_cam);
}

(Draw 이벤트에서) 위의 코드는 현재 어떤 카메라 보기가 렌더링되고 있는지 확인하고 그것이 카메라 보기[0]이면 변수 "cutscene_cam"이 참조하는 카메라 설정이 적용됩니다.
