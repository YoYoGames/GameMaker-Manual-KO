view_visible

이 변수는 특정 뷰 포트가 현재 표시되는지 여부를 확인하는 데 사용할 수 있습니다. 값을 true (보이는) 또는 false (보이지 않는). 보기 포트를 표시로 설정했더라도 보기 포트가 활성화되지 않은 경우(내장 변수 사용view_enabled 또는 Room Editor에서 활성화하면) 화면에 그려지지 않습니다.

통사론:

view_visible[0 ... 7];

보고:

(true: enabled, false: disabled)

예시:

if !view_visible[0]
{
    view_visible[0] = true;
}

위의 코드는 view[0]이 보이는지 아닌지 확인하고, 보이지 않으면 보이도록 설정합니다.
