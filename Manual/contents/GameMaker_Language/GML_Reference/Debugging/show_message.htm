show_message

이 함수는 주어진 문자열을 표시하는 팝업 메시지 상자와 "확인"으로 표시된 버튼을 생성하여 닫습니다.

참고 이 함수는 Windows 대상 에서만 디버그용 으로 사용되지만 다른 모든 대상에서는 더 이상 사용되지 않습니다.

통사론:

show_message(str);

논쟁 유형 설명
str 끈 The string to show in the pop-up message.

보고:

N/A

예시:

var tot = 0;
for (var i = 0; i < 10; i += 1)
{
    tot += inv[i];
}
show_message("Total = " + string(tot));

위의 코드는 "inv" 배열에 저장된 값을 반복하고 합계가 포함된 메시지를 표시하기 전에 변수 "tot"에 추가합니다.
