show_debug_message

이 함수는 런타임에 컴파일러 창에 표시되는 사용자 지정 디버그 메시지를 생성합니다. 메시지는 문자열 또는 실수일 수 있지만 둘 다 필요한 경우 먼저 숫자를 다음을 사용하여 문자열로 변환해야 합니다.string() 함수(아래 예 참조) 및 숫자에 소수점 이하 두 자리 이상이 있으면 다음을 사용해야 합니다.string_format() 기본적으로 소수점 이하 두 자리까지 반올림하여 표시합니다(따라서 "1.2468"은 출력 창에서 "1.25"로 표시됨).

이 기능으로 표시되는 디버그 메시지는 디버그 모드에서 게임을 실행할 때 IDE 하단의 컴파일러 출력 창과 디버거의 그래프 보기 에 표시됩니다. 디버그 모드에서만 메시지를 보려면 다음을 사용해야 합니다.debug_event() 대신에.

통사론:

show_debug_message(string);

논쟁 유형 설명
string The custom debug message to show.

보고:

예시:

if !instance_exists(obj_Exit)
{
    show_debug_message("Exit not created!");
    show_debug_message("Error Num: " + string(global.error));
    game_end();
}

위의 코드는 인스턴스가 존재하는지 확인하고 없으면 컴파일 창에 디버그 메시지가 표시되고 게임이 종료됩니다.
