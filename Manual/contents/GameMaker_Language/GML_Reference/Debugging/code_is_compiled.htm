code_is_compiled

이 함수는 반환됩니다 true 플랫폼이 YYC 및 JS 플랫폼과 같이 가상 머신 외부에서 컴파일되는 경우.

통사론:

code_is_compiled();

보고:

예시:

if code_is_compiled()
{
    show_debug_message("Compiler okay!");
}

위의 코드는 게임이 YoYo 컴파일러를 사용하여 컴파일되었는지 확인하고 디버그 메시지를 표시합니다.
