show_question

이 기능은 "예" 및 "아니오"에 대한 두 개의 버튼이 있는 팝업 메시지 상자를 생성합니다. 그것은 반환 true 또는 false 두 버튼 중 사용자가 누르는 버튼에 따라 다릅니다.

참고 이 함수는 Windows 대상 에서만 디버그용 으로 사용되지만 다른 모든 대상에서는 더 이상 사용되지 않습니다.

통사론:

show_question(str);

논쟁 유형 설명
str 끈 The string to show in the pop-up question.

보고:

Boolean

예시:

if (score > 500) && debug_mode
{
    if show_question("Continue to next room?")
    {
        room_goto(rm_Level2);
    }
    else game_end();
}

위의 코드는 점수를 확인하고 500점이 넘으면 계속할 것인지 묻고 "예" 버튼을 누르면 다른 방으로 이동하지만 "아니오" 버튼이 선택하면 게임이 종료됩니다.
