fps

GameMaker 에는 게임 실행 속도를 알려주는 데 사용할 수 있는 두 가지 주요 방법이 있습니다. 그만큼game speed (게임 옵션에 지정된 대로) 및 fps (초당 프레임). 이러한 값은 종종 혼동되지만 기본적으로 하나는 GameMaker 가 1초에 완료해야 하는 게임 단계 수이고 다른 하나(fps)는 GameMaker 가 최대 1초에 실제로 완료하는 CPU 단계 수입니다. 룸 속도 자체의 최대값. 진정한 fps를 얻으려면, 즉. 게임 단계당 실제 CPU 단계 수fps_real 변하기 쉬운.

이 읽기 전용 변수는 현재 fps를 정수 값으로 반환합니다. 이 기능은 게임의 모든 단계에서 한 번만 업데이트되므로 한 값에서 다른 값으로 "점프"하는 것처럼 보일 수 있지만 이는 매우 정상적인 현상입니다.

예시:

if debug_mode
{
    draw_text(32, 32, "FPS = " + string(fps));
}

위의 코드는 게임이 디버그 모드인지 확인하고 디버그 모드인 경우 화면에 현재 fps를 표시합니다.
