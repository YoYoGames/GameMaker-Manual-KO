debug_mode

이 읽기 전용 변수는 다음을 반환합니다. true 게임이 디버그 모드에서 재생되고 있을 때 false 정상적으로 재생될 때.

통사론:

debug_mode

보고:

예시:

if debug_mode
{
    ini_open("Cheats.ini");
}
else
{
    ini_open("Game.ini");
}

위의 코드는 읽기 전용 변수의 값에 따라 다른 ini 파일을 엽니다. debug_mode.
