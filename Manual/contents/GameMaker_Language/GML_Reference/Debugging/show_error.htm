show_error

이 함수는 사용자 지정 문자열을 오류 메시지로 표시합니다.

중요 두 번째 인수는 이전 프로젝트와의 하위 호환성을 위해서만 존재하며 표시된 오류 대화 상자에는 영향을 미치지 않습니다.

참고 이 기능은 디버그 전용 입니다.

통사론:

show_error(str, abort);

논쟁 유형 설명
str 끈 The string to show in the pop-up message.
abort 부울 UNUSED Whether the error should abort the game (true) or allow the player to ignore it (false).

보고:

N/A

예시:

if (room != rm_Dungeon)
{
    show_error("Error: Went to wrong area. Aborting game.", true);
}

위의 코드는 현재 방이 rm_Dungeon 인지 확인하고 그렇지 않은 경우 오류 메시지를 표시합니다.
