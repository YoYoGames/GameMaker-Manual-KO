uwp_license_trial_version

이 기능은 게임이 평가판 라이선스 하에 있는지 여부를 확인하는 데 사용할 수 있습니다. 그렇다면 함수가 반환됩니다. true, 또는 false 그렇지 않으면.

통사론:

uwp_license_trial_version();

보고:

예시:

if uwp_license_trial_version()
{
    global.LevelCap = 10;
}

위의 코드는 앱이 평가판 라이선스 하에 있는지 확인하고, 그렇다면 전역 변수를 설정합니다.
