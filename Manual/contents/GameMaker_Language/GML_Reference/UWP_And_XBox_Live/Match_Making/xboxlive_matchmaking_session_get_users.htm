xboxlive_matchmaking_session_get_users

이 함수는 지정된 세션의 사용자 세부 정보로 DS 맵을 만들고 채우거나 오류가 있는 경우 -1을 채웁니다. 다음에서 세션 ID 포인터를 얻을 수 있습니다. async_load 세션을 생성하거나 찾을 때 비동기식 소셜 이벤트 에서 반환되는 DS 맵(참조xboxlive_matchmaking_create() 자세한 사항은). 반환된 DS 맵에는 다음 키/값 쌍이 있습니다.

  • " num_results " - 세션의 사용자 수를 포함합니다.
  • " userId< index > " - 사용자의 ID를 포함합니다(< index >는 0에서 "num_results" - 1 사이의 값).
  • " userIsOwner< index > " - 사용자가 호스트이면 1을 포함하고, 그렇지 않으면 0을 포함합니다(< index >는 0에서 "num_results" - 1 사이의 값).

통사론:

xboxlive_matchmaking_session_get_users(session_id);

논쟁 유형 설명
session_id The session ID pointer to use

보고:

Async Request ID

예시:

var session_map = xboxlive_matchmaking_session_get_users(global.SessionID);
if (session_map != -1)
{
    for (var i = 0; i < session_map[? "num_results"]; i++)
    {
        if (session_map[? "userIsOwner" + string(i)] == 1)
        {
            global.OwnerID = session_map[? "userId" + string(i)];
        }
    }
}

위는 매치메이킹 세션에 참여하는 모든 사용자의 사용자 데이터를 검색하고 전역 변수를 세션 소유자의 ID로 설정합니다.
