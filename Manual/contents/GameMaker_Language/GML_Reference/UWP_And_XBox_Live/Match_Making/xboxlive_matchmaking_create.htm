xboxlive_matchmaking_create

이 기능은 XDP 대시보드에서 미리 생성했어야 하는 세션 템플릿, 매치메이킹 호퍼 및 보안 장치 연결 템플릿을 사용하여 멀티플레이어 세션을 생성합니다. 호출하여 특정 게임 패드에 대해 검색할 수 있는 사용자 IDxboxlive_user_for_pad() 기능 및 가시성은 다음 상수 중 하나입니다.

Xbox Match 가시성 상수
일정한 설명
xboxlive_match_visibility_open 세션이 다른 세션에 참여할 수 있음을 지정합니다.
xboxlive_match_visibility_private 세션 구성원이 아닌 사용자는 세션을 볼 수 없도록 지정합니다.
xboxlive_match_visibility_usetemplate 세션 템플릿에 사용된 값을 사용해야 함을 지정합니다(XDP에서 템플릿 값이 설정된 경우 템플릿 값을 재정의할 수 없으므로 일반적으로 이 값을 사용해야 함).

함수는 고유한 요청 ID 값을 반환하며, 이 값은 이 함수에 대한 올바른 소셜 비동기 이벤트를 식별하는 데 사용할 수 있습니다. 이 이벤트는 세션이 생성되고 변수에 DS 맵을 포함할 때 트리거됩니다. async_load 다음 키/값 쌍으로:

  • "requestid" – 호출 함수에서 반환된 요청 ID를 포함합니다.
  • "status" – 어떤 유형의 이벤트가 트리거되었는지 알려주는 "session_created" 문자열을 포함합니다.
  • "sessionid" – 세션 ID를 포함하거나 실패의 경우 -1
  • "오류" – 성공 시 0, 실패 시 -1 포함
  • "correlationid" - 함수와의 상관 관계를 위한 세션 핸들 xboxlive_matchmaking_join_session()

통사론:

xboxlive_matchmaking_create(user_id, visibility, template, hopper, sdatemplate, [matchattributes]);

논쟁 유형 설명
user_id Xbox 사용자 ID The user ID pointer to use
visibility Xbox Match 가시성 상수 One of the constants listed above
template 끈 The name of the session template
hopper 끈 The name of the matchmaking hopper
sdatemplate 끈 The name of the secure device association template
matchattributes 진짜 OPTIONAL The match attributes

보고:

Async Request ID

예시:

var userID = xboxlive_user_for_pad(global.PadIndex);
result = xboxlive_matchmaking_create(userID, xboxlive_match_visibility_usetemplate, "MatchTicketSession", "MatchTicketHopper", "PeerServerTraffic");

위는 주어진 게임패드에서 사용자의 사용자 ID를 검색한 다음 매치메이킹 세션을 초기화합니다.
