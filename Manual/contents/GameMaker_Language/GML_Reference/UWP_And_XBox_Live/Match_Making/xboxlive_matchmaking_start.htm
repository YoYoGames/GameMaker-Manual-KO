xboxlive_matchmaking_start

이 함수는 사용자에 대한 멀티 플레이어 세션을 시작하기 전에 호출되어야 합니다. 사용자 ID 포인터가 필요하며, 이 포인터는xboxlive_user_for_pad() 기능을 수행하고 멀티 플레이어 API를 초기화합니다.

통사론:

xboxlive_matchmaking_start(user_id);

논쟁 유형 설명
user_id The user ID pointer to use

보고:

예시:

global.UserID = xboxlive_user_for_pad(global.PadIndex);
xboxlive_matchmaking_start(global.UserID);

위의 코드는 주어진 패드 인덱스에 대한 사용자 ID를 가져온 다음 멀티플레이어 기능을 초기화합니다.
