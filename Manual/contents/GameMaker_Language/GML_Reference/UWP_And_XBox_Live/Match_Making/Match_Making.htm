결혼 중매

Xbox Live 및 UWP용 매치메이킹(멀티플레이어)은 Xbox 개발자 포털에서 세션 템플릿 및 매치메이킹 호퍼 설정을 통해 약간의 구성이 필요합니다. GameMaker 는 UWP 대상을 사용하여 이를 지원하며 다음을 수행할 수 있도록 허용합니다.

  • 멀티플레이어 세션 만들기
  • 멀티플레이어 세션 찾기(및/또는 참여)
  • 관련 플레이어가 세션에 참여하거나 세션에서 나갈 때 알림을 받고 네트워크 연결 세부 정보를 받습니다.
  • 세션 호스트가 다른 사용자로 마이그레이션될 때 알림을 받습니다.
  • 세션의 사용자 목록 가져오기
  • 세션 나가기

멀티플레이어 세션을 생성하려면 사용할 세션 템플릿 을 지정해야 합니다. 이러한 세션 템플릿은 XDP에 정의되어 있지만 아래에 예가 나와 있습니다.

Session Name: MatchSession
Contract Version: 107
Session Template (JSON text):
{
    "constants": {
        "system": {
            "version": 1,
            "maxMembersCount": 12,
            "capabilities": {
        },
            "memberInitialization": {
                "externalEvaluation": false,
                "membersNeededToStart": 1
            }
        },
        "custom": {}
    }
}

필드에 대한 설명은 XDK 도움말 파일의 "세션 개요"에서 찾을 수 있습니다. 현재 GameMaker 에서 세션 매개변수를 동적으로 설정하는 것을 지원하지 않습니다.

세션 템플릿과 함께 매치메이킹 호퍼도 세션을 찾을 수 있도록 설정해야 합니다(이 작업은 XDP에서도 수행됨). 호퍼는 플레이어를 일치시킬 때 사용되는 규칙을 정의하는 데 사용됩니다. 세션 매개변수와 마찬가지로 현재 GameMaker: Studio에서 호퍼 매개변수를 동적으로 설정하는 것을 지원하지 않습니다. 자세한 내용은 XDK 도움말 파일의 "SmartMatch 매치메이킹 구성" 섹션을 참조하십시오.

Xbox Live 매치메이킹에 다음 기능을 사용할 수 있습니다(연결 등을 위해 GameMaker 네트워킹 기능 도 사용해야 합니다...):

  • xboxlive_matchmaking_start
  • xboxlive_matchmaking_stop
  • xboxlive_matchmaking_create
  • xboxlive_matchmaking_find
  • xboxlive_matchmaking_session_get_users
  • xboxlive_matchmaking_join_session
  • xboxlive_matchmaking_session_leave

GameMaker 는 네트워크의 다른 플레이어를 세션에 초대하거나 다른 사람의 세션에 초대를 수락하는 데 사용할 수 있는 추가 매치메이킹 기능 세트를 제공합니다. 사용 가능한 기능은 다음과 같습니다.

  • xboxlive_matchmaking_send_invites
  • xboxlive_matchmaking_set_joinable_session
  • xboxlive_matchmaking_join_invite

플레이어가 다른 플레이어의 초대를 수락하면 다음 두 가지 중 하나가 발생합니다.

  • 게임이 현재 실행 중인 경우 다음 키/값이 포함된 async_load DS 맵과 함께 소셜 비동기 이벤트 를 수신합니다.
    • " id " - MATCHMAKING_INVITATION으로 설정됩니다.
    • " Invitation_id " - 고유한 세션 ID 값입니다.
    • " Invitation_host " - 세션 호스트의 고유한 세션 ID 값입니다.
    ID와 호스트 값을 저장한 다음 필요한 세션을 떠나 함수를 호출해야 합니다. xboxlive_matchmaking_join_invite(), 단계 이벤트에서 초대 ID 및 초대 호스트 값을 인수로 전달합니다.
  • 게임이 현재 실행되고 있지 않으면 OS에 의해 시작됩니다. 위에 나열된 것과 동일한 키가 설정된 동일한 유형의 소셜 비동기 이벤트 를 받게 됩니다. 변수에 저장한 다음 Xbox Live 매치메이킹에 연결하고 마지막으로 호출 xboxlive_matchmaking_join_invite().

일단 전화를 걸면 xboxlive_matchmaking_join_invite() 사용자는 세션에 참여하려고 시도하고 마치 전화를 걸었던 것처럼 비동기식 소셜 이벤트를 받게 됩니다. xboxlive_matchmaking_find() 이 세션에서.
