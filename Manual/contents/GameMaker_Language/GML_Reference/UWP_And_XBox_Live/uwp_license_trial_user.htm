uwp_license_trial_user

이 함수는 평가판 라이선스가 현재 사용 중인 사용자 ID 포인터를 반환하거나 반환합니다. pointer_null 사용자가 현재 활성화되어 있지 않거나 게임이 현재 평가판 모드에 있지 않은 경우.

통사론:

uwp_license_trial_user();

보고:

Xbox User ID

예시:

if gamepad_button_check_pressed(0, gp_start)
{
    if (uwp_license_trial_version())
    {
        var _uid = uwp_license_trial_user();
        uwp_show_help(_uid);
    }
}

위의 코드는 버튼이 눌렸는지 확인한 다음 앱이 평가판 라이선스로 실행되고 있는지 확인합니다. 그렇다면 라이센스에 대한 사용자 ID를 가져오고 해당 사용자에 대한 도움말 파일을 엽니다.
