uwp_show_help

이 기능은 게임의 도움말 파일(이전에 XBox 개발자 포털에 업로드한 HTML 형식 파일)을 차례로 표시하는 Xbox One 도움말 앱을 시작합니다. 사용자의 로케일 정보를 표시할 때 사용할 수 있도록 이 함수를 호출할 때 사용자 ID를 지정합니다. 기능을 사용하여 사용자 ID를 검색할 수 있습니다.xboxlive_get_user(),xboxlive_get_activating_user(), 또는uwp_license_trial_user().

통사론:

uwp_show_help(user_id);

논쟁 유형 설명
user_id The User ID pointer to open the helpfile for.

보고:

예시:

if gamepad_button_check_pressed(0, gp_start)
{
    if uwp_license_trial_version()
    {
        var _uid = uwp_license_trial_user;
        uwp_show_help(_uid);
    }
}

위의 코드는 버튼이 눌렸는지 확인한 다음 앱이 평가판 라이선스로 실행되고 있는지 확인합니다. 그렇다면 라이센스에 대한 사용자 ID를 가져오고 해당 사용자에 대한 도움말 파일을 엽니다.
