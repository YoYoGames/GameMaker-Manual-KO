uwp_check_privilege

이 기능을 사용하면 주어진 사용자에게 선택된 권한이 있는지 확인할 수 있습니다. 이 함수는 권한이 활성화되었는지 여부에 따라 시스템 비동기 이벤트에서 true 또는 false를 반환하고, 시도_해상도 인수를 true로 설정하고 권한이 활성화되지 않은 경우 시스템 대화 상자도 열립니다(게임 일시 중단 ) 사용자에게 계정 또는 권한을 얻는 데 필요한 모든 것을 업그레이드하라는 메시지를 표시합니다. 그런 다음 사용자가 필수 옵션을 활성화하면 함수가 true를 반환합니다.

이 기능을 사용하려면 권한 요청에 대해 다음 UWP 권한 상수 중 하나를 사용해야 합니다.

UWP 권한 상수
일정한 설명
uwp_privilege_internet_browsing 인터넷 브라우징이 허용되는지 확인하십시오.
uwp_privilege_social_network_sharing 소셜 네트워크에 공유가 허용되는지 확인하십시오.
uwp_privilege_share_kinect_content Kinect 컨트롤러 공유 허용 여부 확인
uwp_privilege_video_communications 영상통화 허용 여부 확인
uwp_privilege_communications 인터넷 통신이 허용되는지 확인
uwp_privilege_user_created_content 사용자가 만든 콘텐츠에 대한 액세스가 허용되는지 확인
uwp_privilege_multiplayer_sessions 온라인 멀티플레이어 세션이 허용되는지 확인하십시오.
uwp_privilege_sessions 온라인 세션이 허용되는지 확인하십시오.
uwp_privilege_fitness_upload 피트니스 데이터 업로드 허용 여부 확인

함수가 호출되면 XBox One에서 콜백 결과가 수신될 때 비동기 시스템 이벤트 를 트리거합니다. 이 이벤트에는 특수 DS 맵이 있습니다.async_load. 이 맵은 다음 키에 대해 구문 분석되어야 합니다.

  • " event_type " - 이벤트가 이 함수에 의해 트리거된 경우 uwp_privilege_result_no_issue>"check_privilege_result" 값을 유지해야 합니다.

이벤트 유형이 이 함수와 관련된 경우 XBox에서 함수가 호출될 때 다음과 같은 추가 키도 있습니다.

  • " result " - 다음 UWP 권한 결과 상수 (결합된 값) 중 하나 이상이 됩니다.
    UWP 권한 결과 상수
    일정한 설명 값
    uwp_privilege_result_purchase_required 사용자에게 권한 문제가 없습니다. 0
    uwp_privilege_result_aborted 사용자는 액세스를 위해 추가 항목(일반적으로 구독)을 구입해야 합니다. 1
    uwp_privilege_result_banned 수표가 중단되었습니다 2
    uwp_privilege_result_restricted 사용자가 금지됨 4
    true 사용자는 일반적으로 자녀 보호 기능을 통해 액세스가 제한됩니다. 8

  • " 권한 " - 함수에서 사용하기 위해 위에 나열된 상수 중 하나이므로 다중 확인의 경우 요청한 권한을 확인할 수 있습니다.

프로젝트가 XBox One이 아닌 UWP 플랫폼에서 실행되는 경우 비동기 이벤트가 트리거 되지 않고 대신 함수가 즉시 wither -1, 0 또는 1을 반환합니다. 여기서:

  • -1은 -1 사용자에게 권한이 없음을 의미합니다.
  • 0은 함수가 사용자를 쿼리할 수 없음을 의미합니다.
  • 1은 사용자에게 권한이 있음을 의미합니다.

통사론:

uwp_check_privilege(user_id, privilege, attempt_resolution);

논쟁 유형 설명
user_id Xbox 사용자 ID The User ID pointer to check.
privilege UWP 권한 상수 The privilege constant to check (see description, above).
attempt_resolution 부울 Enable attempting a resolution if set to false, or ignore if set to.

예시:

var user_one = xboxlive_get_user(0);
if (xboxlive_user_is_signed_in(user_one))
{
    uwp_check_privilege(user_one, xboxlive_privilege_multiplayer_sessions, true);
}

위의 내용은 사용자가 로그인했는지 확인하고 로그인한 경우 멀티플레이어 세션 권한을 확인합니다.
