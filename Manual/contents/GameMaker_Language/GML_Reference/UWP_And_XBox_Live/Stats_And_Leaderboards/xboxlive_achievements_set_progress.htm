xboxlive_achievements_set_progress

이 기능은 업적의 진행 상황을 업데이트하는 데 사용할 수 있습니다. 예를 들어 함수에 의해 반환된 대로 사용자 ID를 제공합니다.xboxlive_get_user(), 그리고 업적 문자열(생성 시 XDP/Windows 개발자 센터에 할당된 업적 ID). 마지막으로 진행률 값을 0에서 100 사이의 값으로 설정합니다. 업적 시스템은 현재 진행률 값보다 낮은 업데이트를 거부합니다.

통사론:

xboxlive_achievements_set_progress(user_id, achievement, progress);

논쟁 유형 설명
user_id The user ID of the user to set the achievement for
achievement The achievement to set (a string)
progress The progress value to set (from 0 to 100)

보고:

확장된 예:

다음은 이 기능을 사용할 수 있는 확장된 예입니다. 시작하려면 Room Start 또는 Create 와 같은 이벤트에서 호출합니다.

xboxlive_stats_get_social_leaderboard(user_id, "GlobalTime", 20, 1, false, false);

위의 코드는 게임의 모든 글로벌 리더보드 위치 목록을 가져오기 위해 호출되며 다음과 같은 방식으로 처리할 Social Asynchronous Event 콜백을 생성합니다.

var _progress = (global.Level / global.LevelMax) * 100;
xboxlive_achievements_set_progress(user_id, "Game_Completed", _progress);

위의 코드는 일부 전역 변수를 기반으로 백분율 값을 생성한 다음 이를 사용하여 "Game_Completed" 업적에 대한 업적 진행 상황을 설정합니다.
