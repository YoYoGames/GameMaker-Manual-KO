xboxlive_stats_remove_user

이 기능을 사용하여 통계 관리자에서 주어진 사용자를 제거(등록 취소)하여 통계 데이터를 라이브 서버로 플러시할 수 있습니다. XBox 문서에 따르면 게임은 통계 관리자에서 사용자를 제거할 필요가 없습니다. XBox OS는 어쨌든 주기적으로 통계를 플러시하기 때문입니다. (예를 들어) 함수에서 반환된 대로 사용자 ID를 제공합니다.xboxlive_get_user(), 그리고 함수는 오류가 있거나 사용자 ID가 유효하지 않은 경우 -1을 반환하고 함수가 성공적으로 호출된 경우 다른 값을 반환합니다.

참고: 사용자에 대해 설정된 통계가 유효하지 않은 경우(예: 영숫자가 아닌 문자가 포함된 통계 이름) 사용자를 제거하면 오류가 반환될 수 있습니다.

이 함수는 시스템 비동기 이벤트 를 트리거하는 콜백을 생성합니다. 이 이벤트에는 특별 DS 맵이 있습니다. async_load 그런 다음 다음 키에 대해 구문 분석해야 합니다.

  • " id " - 상수를 유지합니다. achievement_stat_event
  • " eventname " - 문자열 " LocalUserRemoved "를 보유합니다.
  • " userid " - 요청과 연결된 사용자 ID
  • " error " - 성공하면 0, 오류가 있으면 다른 값
  • " errormessage " - 사용 가능한 경우 오류 메시지가 있는 문자열

사용자를 제거하지 않고 언제든지 통계 데이터를 라이브 서버로 플러시하려면 다음 기능을 사용할 수 있습니다.xboxlive_stats_flush_user().

통사론:

xboxlive_stats_remove_user(user_id);

논쟁 유형 설명
user_id The user ID (a pointer) to remove

보고:

예시:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_remove_user(user_id[i]);
}

위의 코드는 저장된 사용자 계정 ID를 반복하고 통계 관리자에서 각 사용자를 제거(등록 취소)합니다.
