xboxlive_stats_flush_user

이 기능은 주어진 사용자에 대한 통계 데이터를 통계 관리자에서 라이브 서버로 플러시하여 서버가 현재 값으로 최신 상태인지 확인하는 데 사용할 수 있습니다. XBox 문서에 따르면 개발자는 XBox가 요청 속도를 제한하고 XBox OS도 자동으로 약 5분마다 자동으로 통계를 플러시하므로 개발자는 이것을 너무 자주 호출하지 않도록 주의해야 합니다.

참고: 사용자에 대해 설정된 통계가 유효하지 않은 경우(예: 영숫자가 아닌 문자가 포함된 통계 이름) 사용자를 제거하면 오류가 반환될 수 있습니다.

함수를 사용하려면 (예를 들어) 함수에서 반환된 사용자 ID를 제공합니다.xboxlive_get_user(), 및 속성 값(높은 우선 순위의 경우 1, 낮은 우선 순위의 경우 0). 함수는 오류가 있거나 사용자 ID가 유효하지 않은 경우 -1을 반환하고 함수가 성공적으로 호출된 경우 다른 값을 반환합니다. 이 함수는 또한 시스템 비동기 이벤트 를 트리거하는 콜백을 생성합니다. 이 이벤트에는 특별 DS 맵이 있습니다. async_load 그런 다음 다음 키에 대해 구문 분석해야 합니다.

  • " id " - 상수를 유지합니다. achievement_stat_event
  • " eventname " - " StatisticUpdateComplete " 문자열을 보유합니다.
  • " userid " - 요청과 연결된 사용자 ID
  • " error " - 성공하면 0, 오류가 있으면 다른 값
  • " errormessage " - 사용 가능한 경우 오류 메시지가 있는 문자열

통계 관리자에서 사용자를 제거하면 통계가 자동으로 삭제됩니다. 보다xboxlive_stats_remove_user() 자세한 내용은.

통사론:

xboxlive_stats_flush_user(user_id, priority);

논쟁 유형 설명
user_id Xbox 사용자 ID The user ID (a pointer) to use
priority 진짜 The prority value (0 or 1)

보고:

Real

예시:

for(var i = 0; i < array_length(user_id); ++i;)
{
    xboxlive_stats_flush_user(user_id[i], 0);
}

위의 코드는 저장된 사용자 계정 ID를 반복한 다음 각 사용자에 대한 통계 관리자 데이터를 플러시합니다.
