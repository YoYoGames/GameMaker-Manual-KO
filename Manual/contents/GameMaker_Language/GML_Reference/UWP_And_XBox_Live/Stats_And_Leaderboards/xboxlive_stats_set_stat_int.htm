xboxlive_stats_set_stat_int

이 함수는 주어진 사용자 ID에 대한 통계 값을 설정하는 데 사용할 수 있습니다. (예를 들어) 함수에서 반환된 대로 사용자 ID를 제공합니다.xboxlive_get_user(), 다음으로 설정할 통계 문자열( 통계 문자열이 이미 존재하지 않는 경우 새 통계가 생성되어 주어진 값으로 설정됨 ) 및 통계를 설정할 값(정수). 통계 이름은 기호나 공백 없이 영숫자만 사용해야 합니다.

통계 값을 설정할 때 이전 값이 무시되므로 현재 통계 값과 비교하여 통계 값을 업데이트할지 여부(즉, 최고 점수가 실제로 가장 높은지 확인)를 결정하는 것은 사용자의 몫입니다. 설정하기 전에 새 것으로.

함수는 오류가 있거나 사용자 ID가 유효하지 않은 경우 -1을 반환하고 함수가 성공적으로 호출된 경우 다른 값을 반환합니다.

통사론:

xboxlive_stats_set_stat_int(user_id, stat, value);

논쟁 유형 설명
user_id The user ID pointer to set the stat for
stat The statistic to set (a string)
value The value to set the stat to (an integer)

보고:

예시:

var _val = xboxlive_stats_get_stat(p_user_id, "GamesPlayed");
xboxlive_stats_set_stat_int(p_user_id, "GamesPlayed", ++_val);

위의 코드는 "GamesPlayed" 통계 값을 검색한 다음 동일한 통계를 해당 값에 1을 더한 값으로 설정합니다.
