xboxlive_achievement_show_achievements

이 기능은 선택한 플랫폼의 업적 페이지를 엽니다. 이것은 비동기 함수 라는 점에 유의하십시오. 즉, 업적 페이지가 표시되는 동안 게임이 백그라운드에서 계속 실행됩니다. 따라서 이것을 사용하는 위치에 주의해야 하며 게임을 일시 중지하거나 게임 플레이를 방해하지 않는 게임 영역에서만 표시되도록 허용해야 합니다.

통사론:

xboxlive_achievement_show_achievements()

보고:

예시:

if achievements_enabled
{
    global.Pause = true;
    instance_create_layer(0, 0, "Controllers", obj_Pause);
    xboxlive_achievement_show_achievements();
}

위의 코드는 사용자 정의 변수를 확인하여 업적이 활성화되었는지 확인하고, 그렇다면 전역 변수를 설정하고 화면에 업적을 표시하기 위해 함수를 호출하기 전에 컨트롤러 "일시 중지" 인스턴스를 만듭니다.
