xboxlive_stats_add_user

이 함수는 주어진 사용자 ID 포인터를 통계 관리자에 추가하는 데 사용할 수 있습니다. 이 작업은 다른 통계 기능을 사용하여 게임을 XBox Live 서버와 자동으로 동기화하고 최신 값을 검색하기 전에 수행해야 합니다. (예를 들어) 함수에서 반환된 대로 사용자 ID를 제공합니다.xboxlive_get_user(), 그리고 함수는 오류가 있거나 사용자 ID가 유효하지 않은 경우 -1을 반환하고 함수가 성공적으로 호출된 경우 다른 값을 반환합니다.

이 함수는 시스템 비동기 이벤트 를 트리거하는 콜백을 생성합니다. 이 이벤트에는 특별 DS 맵이 있습니다. async_load 그런 다음 다음 키에 대해 구문 분석해야 합니다.

  • " id " - 상수를 유지합니다. achievement_stat_event
  • " eventname " - 문자열 " LocalUserAdded "를 보유합니다.
  • " userid " - 요청과 연결된 사용자 ID
  • " error " - 성공하면 0, 오류가 있으면 다른 값
  • " errormessage " - 사용 가능한 경우 오류 메시지가 있는 문자열

통사론:

xboxlive_stats_add_user(user_id);

논쟁 유형 설명
user_id The user ID (a pointer) to add

보고:

예시:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_add_user(user_id[i]);
}

위의 코드는 사용 가능한 사용자 계정 수를 얻은 다음 이를 반복하고 계정 ID를 배열에 할당하고 사용자를 통계 관리자에 등록합니다.
