xboxlive_stats_delete_stat

이 기능은 주어진 사용자 ID에 대한 통계 관리자에서 통계를 삭제하는 데 사용할 수 있습니다. (예를 들어) 함수에서 반환된 대로 사용자 ID를 제공합니다.xboxlive_get_user(), 삭제할 stat 문자열. 이렇게 하면 통계 값이 지워지고 통계 관리자에서 제거됩니다. 즉, 더 이상 함수에서 반환되지 않습니다.xboxlive_stats_get_stat_names() 또는xboxlive_stats_get_stat().

함수는 오류가 있거나 사용자 ID가 유효하지 않은 경우 -1을 반환하고 함수가 성공적으로 호출된 경우 다른 값을 반환합니다.

통사론:

xboxlive_stats_delete_stat(user_id, stat);

논쟁 유형 설명
user_id The user ID pointer to delete the stat of
stat The statistic to delete (a string)

보고:

예시:

for(var i = 0; i < xboxlive_get_user_count(); i++;)
{
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_delete_stat(user_id[i], "HighScore");
}

위의 코드는 연결된 사용자를 반복하고 각 사용자에 대해 통계 관리자에서 지정된 통계를 삭제합니다.
