xboxlive_fire_event

이 함수는 stat 이벤트를 발생시키는 데 사용할 수 있습니다. "event_name" 인수는 게임의 XDP 콘솔에 정의된 대로 실행될 이벤트의 이름이며 다음 추가 매개변수도 통계에 대해 설정한 항목에 따라 다릅니다. 함수는 이벤트가 전송된 경우 0을 반환하고(서버에서 수신/처리해야 함) 오류가 있는 경우 -1을 반환합니다(예: 프로젝트에 포함된 이벤트 매니페스트 파일에 다른 번호가 표시되어 있으므로 이벤트가 설정되지 않았습니다)..

통사론:

xboxlive_fire_event(event_name, [stat_parameters...]);

논쟁 유형 설명
event_name 끈 The name of the event to fire.
stat_parameters 변하기 쉬운 OPTIONAL Additional parameters to add to the event.

보고:

Real

예시:

var _user = xboxlive_get_user(0);
var _uid = xboxlive_user_id_for_user(_user);
result = xboxlive_fire_event("PlayerSessionStart", _uid, global.guid_str, 0, 42, 42);

위의 코드는 사용자에 대한 일부 정보를 얻은 다음 이(및 추가) 데이터와 함께 통계 이벤트를 시작합니다.
