xboxlive_stats_get_stat

이 함수는 주어진 사용자에 대한 통계 관리자에서 단일 통계 값을 검색하는 데 사용할 수 있습니다. (예를 들어) 함수에서 반환된 대로 사용자 ID를 제공합니다.xboxlive_get_user(), 다음 중 하나를 사용하여 생성할 때 정의된 stat 문자열 xboxlive_stats_set_stat_* 기능. 반환 값은 문자열이나 실수(검사 중인 통계에 따라 다름) 또는 GML 상수일 수 있습니다. undefined 통계가 존재하지 않거나 오류가 발생한 경우.

통사론:

xboxlive_stats_get_stat(user_id, stat);

논쟁 유형 설명
user_id The user ID pointer to get the stat names of
stat The statistic to set (a string)

보고:

Real, String or undefined

예시:

if (game_over == true)
{
    if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
    {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
    }
}

위의 코드는 변수를 확인하고 true, "PercentDone" 통계 값을 확인합니다. 이 값이 100보다 작으면 값이 생성되고 통계가 해당 값으로 설정됩니다.
