xboxlive_stats_set_stat_string

이 함수는 주어진 사용자 ID에 대한 문자열로 통계를 설정하는 데 사용할 수 있습니다. (예를 들어) 함수에서 반환된 대로 사용자 ID를 제공합니다.xboxlive_get_user(), 그 다음 설정할 통계 문자열( 통계 문자열이 이미 존재하지 않는 경우 새 통계가 생성되어 주어진 값으로 설정 됨)과 통계를 설정할 문자열. 통계 이름은 기호나 공백 없이 영숫자만 사용해야 하지만 설정하는 문자열 값에는 공백과 특수 문자가 포함될 수 있습니다.

통계 문자열을 설정할 때 이전 문자열이 무시되므로 설정하기 전에 현재 통계 문자열을 새 문자열과 비교하여 통계 문자열을 업데이트할지 여부를 결정하는 것은 사용자의 몫입니다.

함수는 오류가 있거나 사용자 ID가 유효하지 않은 경우 -1을 반환하고 함수가 성공적으로 호출된 경우 다른 값을 반환합니다.

통사론:

xboxlive_stats_set_stat_string(user_id, stat, value);

논쟁 유형 설명
user_id Xbox 사용자 ID The user ID pointer to set the stat for
stat 끈 The statistic to set (a string)
value 끈 The value to set the stat to (a string)

보고:

Real

예시:

xboxlive_stats_set_stat_string(p_user_id, "LevelName", global.LevelName);

위의 코드는 "LevelName" 통계를 주어진 전역 변수에 저장된 문자열로 설정합니다.
