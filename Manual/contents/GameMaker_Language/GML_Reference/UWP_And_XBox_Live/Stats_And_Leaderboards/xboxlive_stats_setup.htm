xboxlive_stats_setup

이 함수는 다른 Xbox Live 통계 함수를 사용하기 전에 호출해야 하며 시스템에서 필요한 라이브러리를 초기화하기만 하면 됩니다. "user_id" 인수는 함수에서 반환된 원시 사용자 ID입니다.xboxlive_get_user(), "service_config" 및 "title_id"는 Xbox Live 개발자 센터에 있는 게임의 고유 ID입니다.

이 기능은 이벤트 기반(2013) 통계에만 사용됩니다.

통사론:

xboxlive_stats_setup(user_id, service_config_id, title_id);

논쟁 유형 설명
user_id Xbox 사용자 ID The ID pointer of the user to check.
service_config_id 끈 This is the config_id string for the game.
title_id 끈 The unique ID for your game on the Xbox Dev Center.

예시:

var user = xboxlive_get_user(0);
xboxlive_stats_setup(user, "4d61a1aa-61ac-4541-badd-31f91244fea6", $1244FEA6);
