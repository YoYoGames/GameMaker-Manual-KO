xboxlive_achievement_load_friends

이 함수는 로그인한 모든 사용자 친구에 대한 정보를 서버에 요청하고 다음을 포함하는 콜백 소셜 비동기 이벤트 를 트리거합니다. async_load 관련 키/값 쌍으로 채워진 맵. 이것의 아이디 키는 DS Map 올바른 콜백을 식별하는 데 사용되며(주어진 비동기 이벤트에 대해 둘 이상의 트리거 함수가 있을 수 있음) 상수 와 쌍을 이룹니다.xboxlive_achievement_friends_info 뿐만 아니라 각 친구에 대한 다른 여러 키/값 쌍. 지도의 정확한 내용은 다음과 같습니다.

  • " id " - 이 함수의 경우 다음과 같아야 합니다. xboxlive_achievement_friends_info
  • " FriendN " - 친구의 이름입니다. 여기서 "N"은 친구 목록 내 위치에 해당하는 정수 값입니다.
  • " FriendidN " - 친구 "N"의 고유 사용자 ID입니다.

통사론:

xboxlive_achievement_load_friends();

확장된 예:

다음 코드는 플레이어가 게임 계정에 로그인하여 해당 사용자의 친구 목록을 모두 가져온 후에 호출될 것입니다.

xboxlive_achievement_load_friends();

이것은 사용자 친구에 대한 정보 요청을 보내고 특별한 async_load 다음 데이터가 포함된 DS 맵:

var ident = ds_map_find_value(async_load, "id");
if ident == xboxlive_achievement_friends_info
{
    var numfriends = ds_map_find_value(async_load, "numfriends");
    global.numfriends = numfriends;
    for(var i=0; i < numfriends; i++)
    {
        global.friendname[i] = ds_map_find_value(async_load, "Friend" + string(i));
        global.friendid[i] = ds_map_find_value(async_load, "Friendid" + string(i));
    }
}

위의 코드는 반환된 내용을 확인합니다. DS Map Social Asynchronous Event 에서 "id"가 확인 중인 상수와 일치하면 여러 배열에 다른 모든 값을 저장하는 맵을 반복합니다.
