xboxlive_read_player_leaderboard

이 함수는 주어진 사용자 이름에 대한 리더보드 정보를 반환합니다. 리더보드는 게임의 XDP 대시보드에서 이전에 생성되어야 하며 함수를 호출하면 다음을 포함하는 콜백 소셜 비동기 이벤트 가 트리거됩니다. async_load 관련 키/값 쌍으로 채워진 맵. 이 DS 맵의 "id" 키는 올바른 콜백을 식별하는 데 사용되며(주어진 비동기 이벤트에 대해 둘 이상의 트리거 기능이 있을 수 있음) IT는 상수와 쌍을 이룹니다. achievement_leaderboard_info 뿐만 아니라 기타 여러 키/값 쌍. 지도의 정확한 내용은 다음과 같습니다.

  • " id " - 이 함수의 경우 다음과 같아야 합니다. achievement_leaderboard_info
  • " Leaderboardid " - 공급자 대시보드에 정의된 리더보드의 고유 ID입니다.
  • " numentries " - 받은 리더보드의 항목 수입니다.
  • " PlayerN " - 플레이어의 이름입니다. 여기서 "N"은 리더보드 항목 목록 내 위치에 해당하는 정수 값입니다.
  • " PlayeridN " - 플레이어의 고유 사용자 ID "N".
  • " RankN " - 리더보드 내 플레이어 "N"의 순위입니다.
  • " ScoreN " - 플레이어 "N"의 점수입니다.


이 함수를 사용하려면 필터 속성을 설정하기 위해 다음 상수 중 하나를 제공해야 합니다.

일정한 설명
xboxlive_achievement_filter_all_players 모든 점수를 가져옵니다.
xboxlive_achievement_filter_friends_only 친구 점수만 오름차순으로 가져옵니다.
xboxlive_achievement_filter_favorites_only 좋아하는 점수만 오름차순으로 가져오기
xboxlive_achievement_filter_friends_alt 친구 점수만 내림차순으로 가져옵니다.
xboxlive_achievement_filter_favorites_alt 좋아하는 점수만 내림차순으로 가져옵니다.

통사론:

xboxlive_read_player_leaderboard(leaderboard_name, user_name, num_items, friend_filter);

논쟁 유형 설명
leaderboard_name 끈 The name of the leaderboard to read, as set up on XDP.
user_name Xbox 사용자 ID The ID of the user to read from.
num_items 진짜 The number of items to retrieve.
friend_filter Xbox Live 업적 필터 상수 One of the filter constants (see the description, above).

예시:

var _user = xboxlive_user_for_pad(0);
var _uid = xboxlive_user_id_for_user(_user);
xboxlive_read_player_leaderboard("MyLeaderboard", _uid, 10, achievement_filter_all_players);

위의 코드는 사용자 ID를 얻은 다음 이를 사용하여 지정된 리더보드의 처음 10개 위치에 대한 모든 플레이어 데이터를 요청합니다.
