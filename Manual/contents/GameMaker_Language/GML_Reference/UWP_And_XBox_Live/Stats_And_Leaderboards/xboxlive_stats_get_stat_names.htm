xboxlive_stats_get_stat_names

이 함수는 주어진 사용자에 대해 통계 관리자에서 정의된 모든 통계를 검색하는 데 사용할 수 있습니다. (예를 들어) 함수에서 반환된 대로 사용자 ID를 제공합니다.xboxlive_get_user(), 그리고 함수는 사용자에 대한 통계를 포함하는 문자열 배열을 반환합니다. 오류가 발생하거나 사용자에게 통계가 없는 경우 배열은 여전히 반환되지만 요소 수는 0입니다.

통사론:

xboxlive_stats_get_stat_names(user_id);

논쟁 유형 설명
user_id The user ID pointer to get the stat names of

보고:

예시:

var _stat_str = xboxlive_stats_get_stat_names(user_id);
for (var i = 0; i < array_length(_stat_str); ++i;)
{
    xboxlive_stats_delete_stat(user_id, _stat_str[i]);
}

위의 코드는 주어진 변수에 저장된 사용자 ID 포인터에 대해 정의된 모든 통계를 검색한 다음 배열을 반복하고 통계 관리자에서 지정된 통계를 삭제합니다.
