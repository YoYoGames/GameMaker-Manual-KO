uwp_suspend

이 기능은 앱이 일시 중단 요청을 감지한 후 데이터 저장을 완료했음을 시스템에 나타냅니다. 이 함수를 호출하는 것은 Xbox One 제출을 위한 요구 사항이며 항상 함수와 함께 사용해야 합니다.uwp_is_suspending(), 아래 예와 같이.

통사론:

uwp_suspend();

보고:

예시:

if uwp_is_suspending()
{
    scr_Save_Game_Data();
    uwp_suspend();
}

위의 코드는 앱이 일시 중지 상태인지 확인하고 게임을 일시 중지하기 전에 게임 데이터를 저장하는 스크립트를 호출합니다.
