xboxlive_set_savedata_user

이 기능은 게임 저장 영역에서 작동하는 향후 파일 작업(즉, 번들 영역에 없는 파일에서 모든 파일 쓰기 및 읽기)이 지정된 사용자 ID 포인터와 연결되도록 지정합니다. 저장 데이터를 적절한 사용자에게 리디렉션하기 위해 필요한 만큼 자주 호출하거나 상수를 사용할 수 있습니다. pointer_null 일반 기계 저장 영역에 저장합니다.

통사론:

xboxlive_set_savedata_user(user_id);

논쟁 유형 설명
user_id Xbox 사용자 ID 또는 pointer_null The user ID (a pointer) to set for saving, or pointer_null

보고:

N/A

예시:

if (xboxlive_get_savedata_user() != user_id[0])
{
    xboxlive_set_savedata_user(user_id[0]);
}

위의 코드는 사용자가 현재 저장 대상으로 지정되어 있는지 확인하고 지정되지 않은 경우 지정됩니다.
