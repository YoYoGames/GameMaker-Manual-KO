xboxlive_get_savedata_user

이 함수는 현재 파일 저장과 연관된 사용자 ID 포인터(또는 상수 pointer_null 현재 사용 중인 사용자 ID가 없는 경우). 보다xboxlive_set_savedata_user() 자세한 내용은

통사론:

xboxlive_get_savedata_user();

보고:

Xbox User ID or pointer_null

예시:

if (xboxlive_get_savedata_user() != user_id[0])
{
    xboxlive_set_savedata_user(user_id[0]);
}

위의 코드는 사용자가 현재 저장 대상으로 지정되어 있는지 확인하고 지정되지 않은 경우 지정됩니다.
