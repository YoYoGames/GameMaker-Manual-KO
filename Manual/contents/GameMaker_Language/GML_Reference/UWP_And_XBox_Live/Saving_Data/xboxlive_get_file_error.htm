xboxlive_get_file_error

이 함수는 게임 코드에서 적절한 조치를 취할 수 있도록 파일 함수가 실패한 이유를 디버그하는 데 도움이 되는 상수(아래 참조)를 반환합니다. 파일 함수 중 하나가 실패하면 함수를 호출하여 오류가 무엇인지 정확히 찾을 수 있습니다.

참고: 비동기 저장 / 불러오기 기능을 사용할 때, async_load map에는 상수 중 하나를 포함하는 " error "라는 추가 키가 있습니다(아래 참조).

통사론:

xboxlive_get_file_error();

보고:

Xbox File Error Constant

Xbox 파일 오류 상수
일정한 설명
xboxlive_fileerror_outoflocalstorage 저장하는 동안 이 문제가 발생하면 사용 가능한 모든 저장 게임 공간이 사용되었으며 사용자가 공간 확보 제안(OS에서 제공)을 거부했음을 의미합니다. 게임은 저장하지 않고 계속 플레이할 의사가 있음을 사용자에게 확인해야 합니다. 그래도 저장을 원하면 게임에서 저장 작업을 다시 시도해야 합니다.
xboxlive_fileerror_quotaexceeded 이런 일이 발생하면 게임은 사용자에게 공간을 확보할 때까지 더 이상 데이터를 저장할 수 없음을 알리고 이를 수행하는 방법을 제시해야 합니다.
xboxlive_fileerror_noerror 파일 저장 또는 로드 기능은 문제 없이 수행되었습니다.

예시:

buffer_save(buff, "SaveGame.bin");
if (xboxlive_get_file_error() != xboxlive_fileerror_noerror)
{
    show_debug_message("Save failed. Error code = " + string(xboxlive_get_file_error()));
}

위의 코드는 저장에 오류가 있는지 확인하고 오류 코드가 콘솔에 출력되는지 확인합니다.
