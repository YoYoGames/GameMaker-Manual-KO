데이터 저장

다음 기능은 특히 Xbox One용 UWP 내보내기를 사용하고 XBox Live 기능을 사용하여 데이터를 저장하는 것과 관련이 있습니다.

플레이어 진행 상황과 관련된 저장 데이터는 항상 특정 사용자와 연결되어야 하지만 -1 값을 사용자 인수로 제공하여 "머신 스토리지" 영역에 일반 데이터를 저장할 수 있습니다. 이 영역은 제목에 따라 다르지만 특정 사용자와 연결되어 있지는 않습니다.

참고: 저장된 파일은 16메가보다 클 수 없습니다!

다음 기능은 Xbox Live 저장 데이터를 처리하기 위한 것입니다.

  • xboxlive_set_savedata_user
  • xboxlive_get_savedata_user
  • xboxlive_get_file_error
