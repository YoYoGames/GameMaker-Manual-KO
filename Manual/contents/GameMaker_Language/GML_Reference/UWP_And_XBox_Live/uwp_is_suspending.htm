uwp_is_suspending

이 함수는 반환됩니다 true 단일 프레임(및 false 그렇지 않으면) 와 유사한 방식으로os_is_paused(). 그런 다음 게임은 시스템에 의해 잠재적으로 종료되기 전에 게임 상태를 저장할 수 있는 1초가 있습니다. 게임이 데이터 저장을 마치면 호출해야 합니다.uwp_suspend() 깔끔하게 중단합니다. 게임은 또한 시스템에 의해 재개되는 경우(완전히 종료되지 않은 경우) 플레이어가 게임플레이 상황으로 곧바로 돌아가지 않도록 일시 중지 상태에 들어가야 합니다.

통사론:

uwp_is_suspending();

보고:

예시:

if uwp_is_suspending()
{
    scr_Save_Game_Data();
    uwp_suspend();
}

위의 코드는 앱이 일시 중지 상태인지 확인하고 게임을 일시 중지하기 전에 게임 데이터를 저장하는 스크립트를 호출합니다.
