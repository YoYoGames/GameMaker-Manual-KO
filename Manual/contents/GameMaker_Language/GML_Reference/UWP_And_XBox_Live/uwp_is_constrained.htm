uwp_is_constrained

이 함수는 반환됩니다 true 게임이 리소스 제한 모드에서 실행 중인 경우 또는 false 정상적으로 실행되는 경우. 리소스 제한 모드는 게임에 포커스가 없을 때 활성화될 수 있으며 이 모드에서 게임은 GPU 리소스를 줄였습니다. 이에 대한 응답은 일반적으로 게임을 일시 중지하는 것이어야 합니다(사용자가 더 이상 제어할 수 없기 때문에). 또한 이 모드에서 속도 저하가 관찰되는 경우 특히 과세 효과를 비활성화하는 것이 좋습니다. 참고로 달리uwp_is_suspending() 함수, 이 함수는 반환합니다 true 게임이 제한 모드에 있는 한 계속해서(처음 발생했을 때 한 번이 아님).

통사론:

uwp_is_constrained();

보고:

예시:

if xboxliveis_constrained()
{
    if !instance_exists(obj_PauseMenu)
    {
        instance_create(0,0,obj_PauseMenu);
    }
}

위의 코드는 앱이 제한되어 있는지 확인하고 제한되어 있으면 개체의 인스턴스(게임 일시 중지에 사용됨)를 생성하는지 확인합니다.
