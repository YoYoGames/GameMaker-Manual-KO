xboxlive_user_for_pad

이 함수는 주어진 게임패드 인덱스 값과 관련된 사용자 ID 포인터를 반환하거나, pointer_null 연결된 사용자가 없는 경우. 이 기능은 게임패드 인덱스 에만 사용해야 하며,xboxlive_get_user_count().

통사론:

xboxlive_user_for_pad(pad_index);

논쟁 유형 설명
pad_index The index (an integer) of the gamepad slot to check

보고:

Xbox User ID or pointer_null

예시:

for(var i = 0; i < 11; ++i;)
{
    var u_id = xboxlive_user_for_pad(i);
    if (u_id != pointer_null)
    {
        pad_name[i] = xboxlive_gamedisplayname_for_user(u_id);
    }
    else
    {
        pad_name[i] = "Press Play";
    }
}

위의 코드는 사용 가능한 모든 패드 슬롯을 반복하고 이와 연결된 사용자 ID 포인터가 있는지 확인합니다. 있는 경우 배열이 사용자 표시 이름으로 설정되고, 그렇지 않으면 배열이 일부 기본 텍스트로 설정됩니다.
