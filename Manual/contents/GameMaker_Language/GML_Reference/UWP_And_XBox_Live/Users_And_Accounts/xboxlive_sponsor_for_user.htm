xboxlive_sponsor_for_user

이 함수는 주어진 사용자의 후원자의 ID를 검색합니다.

통사론:

xboxlive_sponsor_for_user(user_id);

논쟁 유형 설명
user_id Xbox 사용자 ID The ID pointer of the user to check.

보고:

Xbox Sponsor ID

예시:

var _uid = xboxlive_user_for_pad(0);
sponsor = xboxlive_sponsor_for_user(_uid);

위의 코드는 게임패드 [0] 슬롯에 할당된 사용자의 사용자 ID 포인터를 가져온 다음 해당 ID를 사용하여 사용자 스폰서 ID를 검색합니다.
