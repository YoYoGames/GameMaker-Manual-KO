xboxlive_user_is_signed_in

이 함수는 사용자가 로그인했는지 확인하고 반환합니다. true 있다면 그리고 false 그렇지 않으면. 로그인한 사용자가 없으면 함수를 호출할 수 있습니다.xboxlive_show_account_picker() 계정 선택기를 열고 로그인하라는 메시지를 표시합니다.

통사론:

xboxlive_user_is_signed_in();

보고:

예시:

if !xboxlive_user_is_signed_in()
{
    if (!xboxlive_user_is_signing_in())
    {
        xboxlive_show_account_picker();
    }
}
else
{
    global.GamerTag = xboxlive_gamertag_for_user();
}

위의 코드는 사용자가 XBox Live에 로그인했는지 확인하고 로그인한 경우 코드에서 사용자의 게이머태그를 가져와 나중에 사용할 수 있도록 전역 변수에 저장합니다. 로그인하지 않은 경우 코드는 로그인 과정에 있는지 확인합니다. 이 경우 더 이상 발생하지 않습니다. 그렇지 않은 경우 계정 선택기를 열고 서명하라는 메시지를 표시합니다. -입력.
