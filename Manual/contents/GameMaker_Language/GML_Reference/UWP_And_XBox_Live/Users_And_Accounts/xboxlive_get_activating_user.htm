xboxlive_get_activating_user

이 기능을 사용하면 콘솔에서 게임을 시작한 사용자의 사용자 ID 포인터를 검색할 수 있습니다.

참고: 이 포인터를 사용하는 방법은 게임에서 수행하려는 작업에 따라 다르지만 활성화된 사용자는 게임이 실행되는 동안 로그아웃할 수 있으므로 현재 재생 중인 사용자 ID에 의존해서는 안 됩니다.

통사론:

xboxlive_get_activating_user();

보고:

Xbox User ID

예시:

global.activUser = xboxlive_get_activating_user();

위는 활성화된 사용자 ID 포인터를 전역 변수에 저장합니다.
