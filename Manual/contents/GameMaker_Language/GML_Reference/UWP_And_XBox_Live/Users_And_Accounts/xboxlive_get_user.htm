xboxlive_get_user

이 기능을 사용하여 인덱싱된 사용자의 사용자 ID 포인터를 검색할 수 있습니다. 사용자가 없으면 함수는 상수를 반환합니다. pointer_null 대신에. 이 기능을 사용하여 현재 로그인한 사용자 수를 확인할 수 있습니다.xboxlive_get_user_count().

중요: 이 기능은 다음과 함께 사용해야 합니다. xboxlive_get_user_count() - 게임 패드와 함께 사용하지 마십시오. 사용 xboxlive_user_for_pad() 대신에.

통사론:

xboxlive_get_user(index);

논쟁 유형 설명
index 진짜 The index (an integer) to get the User ID from.

보고:

Xbox User ID or pointer_null

예시:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
}

위의 내용은 로그인한 모든 사용자를 반복하고 고유한 ID 포인터를 배열에 저장합니다.
