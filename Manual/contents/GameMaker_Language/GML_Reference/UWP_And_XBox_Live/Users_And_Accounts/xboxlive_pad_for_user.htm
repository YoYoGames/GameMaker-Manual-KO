xboxlive_pad_for_user

Xbox 사용자는 "슬롯"을 채우는 다양한 패드를 사용자 ID에 할당할 수 있습니다. 이 슬롯은 0부터 번호가 매겨지고 모든 패드 인덱스를 보유할 수 있으므로 슬롯 0은 패드 3에 할당되고 슬롯 1은 패드 2에 할당될 수 있습니다. 이 기능은 주어진 슬롯에 할당된 패드를 가져오는 데 사용할 수 있습니다. 사용자가 특정 시간 동안 입력 없이 게임패드를 떠나면 자동으로 꺼지므로 특히 유용합니다. 따라서 이 기능을 사용하여 주어진 사용자 ID를 확인하여 활성 게임패드가 있는지 확인할 수 있습니다. 그런 다음 (예를 들어) 게임을 일시 중지하여 적절하게 반응하지 마십시오. 패드가 활성화되어 있으면 인덱스 값이 반환되지만 그렇지 않은 경우 함수는 -1을 반환합니다.

슬롯의 패드가 비활성화되면 이후 슬롯은 빈 위치로 "밀어 들어가"므로 슬롯 0, 1 및 2에 할당된 3개의 활성 패드가 있고 슬롯 0의 패드가 비활성화된 경우 슬롯 1 및 to는 이제 슬롯 0과 1이 되므로 슬롯 0은 모든 패드가 꺼지지 않는 한 항상 활성 패드를 포함해야 합니다(이 경우 -1 반환).

통사론:

xboxlive_pad_for_user(user_id, slot);

논쟁 유형 설명
user_id Xbox 사용자 ID The User ID (a pointer) to check
slot 진짜 The index (an integer) of the gamepad slot to check

보고:

Xbox Pad ID

예시:

if (xboxlive_pad_for_user(user_id, 0) == -1)
{
    for(var i = 0; i < gamepad_get_device_count(); i++)
    {
        var u_id = xboxlive_user_for_pad(i);
        if (u_id == -1)
        {
            async_ID = xboxlive_pad_for_user(i, u_id);
        }
    }
}

위의 코드는 지정된 사용자 ID 포인터의 슬롯 0을 확인하여 게임패드가 할당되었는지 확인하고 그렇지 않은 경우 할당됩니다.
