xboxlive_agegroup_for_user

이 함수는 지정된 사용자 ID 포인터에 할당된 연령 범위를 나타내기 위해 4개의 상수(아래에 표시됨) 중 하나를 반환합니다.

통사론:

xboxlive_agegroup_for_user(user_id);

논쟁 유형 설명
user_id Xbox 사용자 ID The user ID (a pointer) to check

보고:

Xbox Age Group Constant

Xbox 연령 그룹 상수
일정한 설명
xboxlive_agegroup_unknown 사용자의 연령대를 알 수 없습니다.
xboxlive_agegroup_child 사용자 연령 그룹은 8~12세(포함)입니다.
xboxlive_agegroup_teen 사용자 연령 그룹은 13~17세(한국은 13~19세 포함)입니다.
xboxlive_agegroup_adult 이용자 연령대는 만 18세 이상(대한민국은 만 20세 이상)

예시:

var _age = xboxlive_agegroup_for_user(xboxlive_get_activating_user());
if _age != xboxlive_agegroup_adult
{
    global.ContentControl = true;
}
else global.ContentControl = false;

위의 코드는 활성화된 사용자의 연령 그룹을 확인하고 전역 변수를 다음으로 설정합니다. true 또는 false 반환된 상수에 따라.
