xboxlive_set_rich_presence

이 함수는 주어진 사용자에 대한 리치 프레즌스 문자열을 설정합니다. Rich Presence 문자열은 사용자가 플레이하는 게임 이름 뒤에 하이픈으로 구분된 사용자의 게임 내 활동을 표시합니다. 이 문자열은 "친구 및 클럽" 목록과 플레이어의 Xbox Live 사용자 프로필에 있는 플레이어의 게이머태그 아래에 표시됩니다.

이 기능을 사용할 때 사용자에 대한 사용자 ID 포인터를 제공해야 하며, 그런 다음 사용자를 현재 게임에서 활성 상태로 표시할 수 있습니다(사용 true /false ). 로 설정하면 false, 그러면 풍부한 프레즌스 문자열이 "/afk" 또는 적절한 것으로 추가됩니다. 다음 인수는 표시할 풍부한 현재 상태 문자열 ID이며, 마지막으로 (선택 사항) 제공할 수 있습니다. service_config_id 끈. 이 인수는 선택 사항이며 제공되지 않으면 Xbox 게임 옵션에 지정된 SCID를 사용합니다.

통사론:

xboxlive_set_rich_presence(user_id, is_user_active, rich_presence_string, [service_config_id]);

논쟁 유형 설명
user_id Xbox 사용자 ID The ID pointer of the user to check.
is_user_active 부울 Flag the user as active or not.
rich_presence_string 끈 The rich presence string ID to use (as defined in the Partner Center - max 50 characters)
service_config_id 끈 OPTIONAL This is the config_id string for the game.

N/A

예시:

var pad_uid = xboxlive_user_for_pad(0);
xboxlive_set_rich_presence(pad_uid, true, "Playing_Challenge");

위의 코드는 게임패드 [0] 슬롯에 할당된 사용자의 사용자 ID 포인터를 가져온 다음 해당 사용자에 대한 풍부한 프레즌스 문자열을 설정합니다.
