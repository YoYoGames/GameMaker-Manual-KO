xboxlive_get_user_count

이 기능을 사용하면 현재 XBox 시스템에 로그인한 총 사용자 수를 확인할 수 있습니다. 반환 값은 0 - N 사이의 정수 값입니다. 여기서 N은 로그인한 사용자 수 -1입니다.

통사론:

xboxlive_get_user_count();

보고:

예시:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
}

위의 내용은 로그인한 모든 사용자를 반복하고 고유한 ID 포인터를 배열에 저장합니다.
