uwp_was_closed_by_user

이 기능을 사용하면 게임을 플레이하는 동안 사용자가 이전에 게임을 종료했는지 확인할 수 있습니다(즉, 시스템이 아닌 올바르게 종료됨). 그렇다면 함수가 반환됩니다. true, 그렇지 않으면 반환됩니다 false.

통사론:

uwp_was_closed_by_user();

보고:

예시:

if uwp_was_closed_by_user()
{
    global.GameLoaded = false;
}

위의 코드는 앱이 이전에 사용자에 의해 종료되었는지 확인하고, 그렇다면 전역 변수를 설정합니다.
