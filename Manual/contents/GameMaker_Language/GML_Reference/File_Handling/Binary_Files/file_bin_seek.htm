file_bin_seek

이 기능은 파일 내의 현재 읽기 위치를 표시된 위치로 이동합니다. 함수에서 반환된 대로 파일 ID 값을 제공합니다.file_bin_open(), 파일을 추가하려면 쓰기 전에 파일 크기로 위치를 이동합니다.

참고: 이 기능은 대상 모듈이 HTML5인 경우 작동 하지 않습니다.

통사론:

file_bin_seek(binfile, pos);

논쟁 유형 설명
binfile The ID of the file to read from.
pos The position in the file to move to.

보고:

예시:

file = file_bin_open("myfile.bin", 2);
size = file_bin_size(file);
file_bin_seek(file, size);

이렇게 하면 게임과 동일한 디렉토리에서 파일을 열고 해당 인덱스를 변수 "file"에 할당한 다음 파일 크기를 가져오고 다음 쓰기 위치를 해당 크기로 설정합니다.
