file_bin_position

이 함수는 주어진 파일 ID를 가진 파일의 현재 위치를 바이트 단위로 반환합니다. 여기서 0은 첫 번째 위치입니다. 함수에서 반환된 대로 파일 ID 값을 제공합니다.file_bin_open().

참고: 이 기능은 대상 모듈이 HTML5인 경우 작동 하지 않습니다.

통사론:

file_bin_position(binfile);

논쟁 유형 설명
binfile The ID of the file to get the position in.

보고:

예시:

pos = file_bin_position(file);

이것은 변수 "pos"에 현재 위치를 저장합니다.
