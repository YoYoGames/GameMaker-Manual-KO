file_bin_open

이 함수는 표시된 이름으로 바이너리 파일을 엽니다. 모드는 파일로 수행할 수 있는 작업을 나타냅니다.

  • 0 = 읽기
  • 1 = 쓰기
  • 2 = 읽기와 쓰기 모두

파일이 존재하지 않으면 생성되고, 함수는 다른 함수에서 사용해야 하는 파일의 id를 반환합니다. 여러 파일을 동시에 열 수 있지만(최대 32개), 작업이 끝나면 닫는 것을 잊지 마십시오.

경고! 이 기능은 GameMaker 가 샌드박싱되어 있기 때문에 예상대로 작동하지 않을 수 있습니다! 자세한 내용은 파일 시스템 섹션을 참조하십시오.

참고: 이 기능은 대상 모듈이 HTML5인 경우 작동 하지 않습니다.

통사론:

file_bin_open(fname, mode);

논쟁 유형 설명
fname 끈 The name of the file to read from.
mode 진짜 The indicator of what can be done with the file.

보고:

Binary File ID

예시:

file = file_bin_open("myfile.bin", 2);

이렇게 하면 게임과 같은 디렉터리에서 파일을 열고 해당 인덱스를 "file" 변수에 할당합니다.
