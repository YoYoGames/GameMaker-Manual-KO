file_bin_close

이전에 열린 바이너리 파일을 닫습니다. 함수에서 반환된 대로 파일 ID 값을 제공합니다.file_bin_open().

참고: 이 기능은 대상 모듈이 HTML5인 경우 작동 하지 않습니다.

통사론:

file_bin_close(binfile);

논쟁 유형 설명
binfile The ID of the file to close.

보고:

예시:

file = file_bin_open("myfile.bin", 2);
file_bin_rewrite(file);
file_bin_close(file);

이렇게 하면 게임과 같은 디렉터리에서 파일을 열고 해당 인덱스를 "file" 변수에 할당합니다. 그런 다음 파일을 다시 쓰고(데이터 지우기) 다시 닫습니다.
