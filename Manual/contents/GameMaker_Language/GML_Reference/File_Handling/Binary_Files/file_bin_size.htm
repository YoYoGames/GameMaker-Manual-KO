file_bin_size

이 함수는 읽기 및/또는 쓰기를 위해 열린 파일의 크기(바이트)를 반환합니다. 파일 ID는 함수에서 반환된 값입니다.file_bin_open().

참고: 이 기능은 대상 모듈이 HTML5인 경우 작동 하지 않습니다.

통사론:

file_bin_size(binfile);

논쟁 유형 설명
binfile The ID of the file to get the size of.

보고:

예시:

file = file_bin_open("myfile.bin", 2);
size = file_bin_size(file);
file_bin_close(file);

이렇게 하면 게임의 로컬 디렉터리에서 파일이 열리고 해당 인덱스가 "file" 변수에 할당됩니다. 그런 다음 해당 파일의 크기를 얻고 다시 닫습니다.
