바이너리 파일

드문 경우지만 바이너리 파일에서 데이터를 읽어야 할 수도 있습니다. 이러한 함수를 해당 용도로 사용할 수 있지만 일반적으로 이러한 함수보다 전용 버퍼 함수 를 사용하여 바이너리 데이터를 생성, 저장 및 로드하는 것이 좋습니다.

참고: 이 기능은 대상 모듈이 HTML5인 경우 작동 하지 않습니다.

이를 위해 다음과 같은 저수준 루틴이 있습니다.

  • file_bin_open
  • file_bin_rewrite
  • file_bin_close
  • file_bin_size
  • file_bin_position
  • file_bin_seek
  • file_bin_write_byte
  • file_bin_read_byte
