program_directory

게임 실행 파일이 저장된 디렉토리를 반환합니다. 그러나 이것은 특히 일부 장치가 *.zip 파일에서 exe를 실행하기 때문에 항상 유용한 것은 아니므로 게임이 실제로 실행되는 위치에 관계없이 동일한 결과를 반환합니다.

경고! 이 기능은 GameMaker 가 샌드박싱되어 있기 때문에 예상대로 작동하지 않을 수 있습니다! 자세한 내용은 파일 시스템 섹션을 참조하십시오.

참고: 샌드박스를 비활성화하면 필요한 게임 파일을 삭제할 때 발생할 수 있는 문제를 방지하기 위해 여전히 이 디렉터리에 쓸 수 없습니다.

통사론:

program_directory

보고:

예시:

dir = program_directory;

이것은 실행 파일이 변수에 저장된 디렉토리를 저장합니다.
