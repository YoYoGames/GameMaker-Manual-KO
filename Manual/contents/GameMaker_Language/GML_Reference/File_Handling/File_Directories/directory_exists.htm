directory_exists

이 함수는 반환됩니다 true 표시된 디렉토리가 존재하거나 false 그렇지 않은 경우. 지정된 이름은 상대 경로가 아닌 전체 경로를 포함해야 하며 기본적으로 모든 게임이 샌드박스 처리되므로 게임 번들 외부에서 디렉터리에 액세스할 수 없습니다(자세한 내용은 파일 시스템 섹션 참조).

통사론:

directory_exists(dname)

논쟁 유형 설명
dname The name of the directory to look for.

보고:

예시:

if directory_exists(working_directory + "Saves/")
{
    file = file_find_first(working_directory + "Saves/*.doc", fa_readonly);
}

이것은 지정된 디렉토리가 존재하는지 확인한 다음 존재하는 경우 그곳으로 이동하여 첫 번째 "읽기 전용"을 반환합니다. doc 파일을 찾았습니다.
