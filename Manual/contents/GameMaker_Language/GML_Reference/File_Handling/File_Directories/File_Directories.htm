파일 디렉토리

파일로 작업할 때 파일이 저장된 디렉토리를 참조해야 하는 경우가 있으며 GameMaker 에는 필요할 수 있는 여러 가능한 디렉토리와 관련된 몇 가지 특수 기능 및 변수가 있습니다.

  • directory_exists
  • directory_create
  • directory_destroy
  • temp_directory
  • working_directory
  • program_directory
  • game_save_id ( 일반 게임 컨트롤 에서)

참고 HTML5 에서는 디렉토리를 생성하거나 삭제할 수 없습니다.
