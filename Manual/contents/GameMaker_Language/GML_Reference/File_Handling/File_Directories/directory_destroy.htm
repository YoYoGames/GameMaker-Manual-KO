directory_destroy

이 기능은 저장 영역에서 주어진 이름을 가진 디렉토리를 제거합니다.

참고: 이 기능은 브라우저 로컬 저장소에서 디렉토리를 생성하거나 제거할 수 없기 때문에 HTML5에서 작동하지 않습니다.

경고! 이 기능은 GameMaker 가 샌드박싱되어 있기 때문에 예상대로 작동하지 않을 수 있습니다! 자세한 내용은 파일 시스템 섹션을 참조하십시오.

통사론:

directory_destroy(dname)

논쟁 유형 설명
dname The name of the directory to remove.

보고:

예시:

if directory_exists("DLC")
{
    directory_destroy("DLC");
}

이렇게 하면 지정된 디렉토리가 로컬 데이터 폴더에 있는지 확인하고 있으면 제거됩니다.
