temp_directory

이것은 게임이 실행될 때마다 생성된 임시 디렉터리를 반환하는 데 사용할 수 있습니다(루트에는 마지막 "\"이 포함되지 않음). 이 디렉토리는 파일을 보관하고 게임이 실행되는 동안 액세스할 수 있지만 게임이 종료되면 (포함된 모든 파일과 함께) 제거됩니다.

경고! 이 기능은 GameMaker 가 샌드박싱되어 있기 때문에 예상대로 작동하지 않을 수 있습니다! 자세한 내용은 파일 시스템 섹션을 참조하십시오.

통사론:

temp_directory

보고:

예시:

ini_open(temp_directory + "\temp_ini.ini");

이것은 게임의 임시 디렉토리에 있는 ini 파일을 열 것입니다(아직 존재하지 않는다면 생성).
