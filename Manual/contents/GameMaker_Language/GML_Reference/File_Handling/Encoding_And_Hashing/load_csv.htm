load_csv

이 함수는 CSV 형식 파일을 로드하고 DS 그리드로 변환하여 생성된 그리드의 고유 ID 값을 반환합니다.

CSV 파일은 CSV 사양 을 따라야 합니다. CSV 구문이 올바르지 않으면 파일을 로드할 때 예기치 않은 결과가 나타납니다.

통사론:

load_csv(filename)

논쟁 유형 설명
filename 끈 The name of the file to open (as a string)

보고:

DS Grid ID

예시:

file_grid = load_csv("spreadsheet.csv");
var ww = ds_grid_width(file_grid);
var hh = ds_grid_height(file_grid);
var xx = 32;
var yy = 32;
for (var i = 0; i < ww; i++;)
{
    for (var j = 0; j < hh; j++;)
    {
        draw_text(xx, yy, file_grid[# i, j]);
        yy += 32;
    }
    yy = 32;
    xx += 32;
}

위의 코드는 주어진 CSV 파일을 열고 반환된 DS 그리드를 "file_grid" 변수에 저장합니다. 이 그리드는 다음 몇 가지로 구문 분석됩니다. for lops 및 내용이 화면에 그려집니다.
