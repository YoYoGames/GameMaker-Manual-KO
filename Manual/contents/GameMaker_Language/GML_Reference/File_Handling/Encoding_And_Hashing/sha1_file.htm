sha1_file

암호화에서 SHA-1은 미국 국가안보국(National Security Agency) 에서 설계한 암호화 해시 함수이며 파일 변경을 확인하는 데 사용되는 널리 사용되는 여러 응용 프로그램 Git 과 TLS 및 SSL, PGP, SSH, S/MIME 및 IPsec. 이 함수는 입력 파일을 받아 나중에 무결성 검증에 사용할 해당 파일에 고유한 ASCII 형식의 160비트 메시지 다이제스트를 반환합니다.

참고: SHA-1은 완전히 안전하지 않으며 깨질 수 있습니다. 자세한 내용은 이 페이지 를 참조하세요.

통사론:

sha1_file(filename)

논쟁 유형 설명
filename 끈 The file to generate the sha1 hash for.

보고:

String

예시:

hash = sha1_file(working_directory + "game_data.ini")

위의 코드는 지정된 파일에 대한 sha1 해시를 생성하고 반환된 값을 "hash" 변수에 저장합니다.
