base64_encode

이 함수는 문자열을 base64 형식으로 인코딩된 문자열로 변환합니다. 이것은 인터넷을 통해 텍스트로 저장하거나 전송해야 하는 모든 미디어에 자주 사용되는 일반적으로 사용되는 인코딩 체계이며 출력을 사람의 눈으로 읽을 수 없도록 만듭니다.

통사론:

base64_encode(string)

논쟁 유형 설명
string The string to encode.

보고:

예시:

var str, file;
str = base64_encode(game_data);
file = file_text_open_write("save.txt");
file_text_write_string(file, str);
file_text_close(file);

위의 코드는 "game_data"에 저장된 문자열을 base64로 인코딩된 문자열로 변환한 다음 외부 텍스트 파일에 저장합니다.

뒤로: 파일 인코딩
