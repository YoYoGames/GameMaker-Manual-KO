zip_unzip

이 기능은 저장된 zip 파일을 열고 해당 내용을 지정된 디렉토리에 추출합니다. ZIP 디렉토리에 대한 전체 경로를 제공하지 않으면 현재 드라이브 루트 가 사용되며 게임 번들 작업 디렉토리에 대한 상대 경로에 배치하려면 다음을 사용해야 합니다.working_directory 경로의 일부로 변수("." 또는 ".."를 사용하는 상대 경로는 예상대로 작동하지 않으므로 피해야 함). zip은 게임 번들(예: 포함된 파일 )의 일부이거나 http_get_file() 을 사용하여 저장 영역에 다운로드되어야 합니다.

이 함수는 추출된 파일 수를 나타내는 값을 반환하거나 추출에 실패한 경우 0 이하를 반환합니다.

통사론:

zip_unzip(zip_file, target_directory)

논쟁 유형 설명
zip_file The zip file to open
target_directory The target directory to extract the files to

보고:

예시:

var num = zip_unzip("/downloads/level_data.zip", working_directory + "extracted/");
if num <= 0
{
    show_debug_message("Extraction Failed!");
}

위의 코드는 "downloads" 디렉토리에 저장된 zip 파일을 열고 "extracted" 디렉토리에 내용을 추출한 다음(아직 존재하지 않는 경우 해당 디렉토리 생성) 추출이 올바른지 확인하여 다음을 표시합니다. 실패하면 디버그 메시지가 표시됩니다.
