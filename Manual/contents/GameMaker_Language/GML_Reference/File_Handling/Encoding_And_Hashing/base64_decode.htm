base64_decode

이 함수는 이전에 base64 형식을 사용하여 인코딩된 문자열을 표준 텍스트로 변환합니다. Base64는 인터넷을 통해 텍스트로 저장하거나 전송해야 하는 모든 미디어에 자주 사용되는 일반적으로 사용되는 인코딩 체계이며 출력을 사람의 눈으로 읽을 수 없도록 만듭니다.

통사론:

base64_decode(string)

논쟁 유형 설명
string The string to decode.

보고:

예시:

var str, file;
str = base64_encode(game_data);
file = file_text_open_read("save.txt");
str = file_text_read_string(file); level_data = base64_decode(str);
file_text_close(file);

위의 코드는 텍스트 파일을 열고 문자열을 로컬 변수 "str"로 읽어들입니다. 그런 다음 이 문자열이 디코딩되고 결과가 인스턴스 변수 "level_data"에 저장됩니다.
