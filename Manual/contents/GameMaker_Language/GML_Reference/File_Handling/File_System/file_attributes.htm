file_attributes

이 기능을 사용하여 주어진 파일의 속성을 확인할 수 있습니다. 다음 상수를 추가하여 원하는 파일 유형을 볼 수 있습니다.

일정한 설명
fa_readonly 읽기 전용 파일
fa_hidden 숨겨진 파일
fa_sysfile 시스템 파일
fa_volumeid 볼륨 ID 파일
fa_directory 디렉토리
fa_archive 아카이브된 파일

참고: 이것은 Windows 전용 기능입니다.

통사론:

file_attributes(fname, attr);

논쟁 유형 설명
fname 끈 The name of the file to check.
attr 파일 속성 상수 The attributes to check for.

보고:

Boolean

예시:

if !file_attributes(file, fa_hidden)
{
    file_delete(file);
}

이렇게 하면 파일이 숨겨져 있는지 여부를 확인하고 그렇지 않은 경우 삭제됩니다.
