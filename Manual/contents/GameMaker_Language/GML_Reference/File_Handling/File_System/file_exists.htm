file_exists

이 함수는 반환됩니다 true 지정된 파일이 존재하고 false 그렇지 않은 경우. 이 기능은 로컬 파일을 확인하는 데만 사용할 수 있으며 원격 서버에 저장된 파일은 확인할 수 없습니다.

통사론:

file_exists(fname);

논쟁 유형 설명
fname The name of the file to check for.

보고:

예시:

if file_exists("level.txt")
{
    file = file_text_open_read("level.txt");
}

이렇게 하면 파일이 있는지 확인하고 파일이 있으면 읽기 위해 엽니다.
