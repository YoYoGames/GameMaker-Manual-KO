file_copy

이 함수는 지정된 파일을 복사하여 선택한 새 이름을 지정합니다.

포함된 파일 영역(APK 내부에 있음) 내에서 Android의 파일을 복사하는 경우 먼저 메모리에 복사되므로 APK에서 매우 큰 파일을 복사하면 기기의 메모리가 부족해질 수 있습니다. HTML5 대상에서 클라이언트 측 저장을 위해 HTML5 Local Storage API 를 사용하고 함수는 바이너리 파일 작업을 지원하지 않으므로 텍스트 파일을 복사한다고 가정합니다.

newname 파일이 이미 존재하는 경우 함수가 실패합니다.

경고! 이 기능은 GameMaker 가 샌드박싱되어 있기 때문에 예상대로 작동하지 않을 수 있습니다! 자세한 내용은 파일 시스템 제한 섹션을 참조하십시오.

통사론:

file_copy(fname, newname);

논쟁 유형 설명
fname The name of the file to copy.
newname The name of the new, copied file.

N/A

예시:

if file_exists("level1.txt")
{
    file_copy("level1.txt", "level2.txt");
}

이렇게 하면 파일이 있는지 확인하고 파일이 있으면 복사됩니다.
