file_delete

이 기능은 시스템에서 지정된 파일을 삭제합니다. 이 함수는 GameMaker 가 생성하고 구문 분석할 수 있는 파일(ini 파일, 텍스트 파일 및 바이너리 파일 또는 스프라이트 또는 표면과 같은 게임 생성 리소스를 저장하기 위해 만든 파일)만 삭제한다는 점에 유의해야 합니다. 그러나 다른 파일은 삭제 되지 않습니다. 함수도 반환됩니다. true 파일이 성공적으로 제거된 경우 또는 false 다른 상황에서.

경고! 이 기능은 GameMaker 가 샌드박싱되어 있기 때문에 예상대로 작동하지 않을 수 있습니다! 자세한 내용은 파일 시스템 제한 섹션을 참조하십시오.

통사론:

file_delete(fname);

논쟁 유형 설명
fname The name of the file to delete.

보고:

예시:

if file_exists("level.txt")
{
    file_delete("level.txt");
}

이렇게 하면 파일이 있는지 확인하고 파일이 있으면 삭제됩니다.
