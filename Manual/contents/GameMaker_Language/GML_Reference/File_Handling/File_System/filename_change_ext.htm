filename_change_ext

이 함수는 확장자(점 포함)가 새 확장자로 변경된 표시된 파일 이름을 반환합니다. 빈 문자열을 새 확장으로 사용하면 확장 부분을 모두 함께 제거할 수 있습니다.

통사론:

filename_change_ext(fname, newext);

논쟁 유형 설명
fname The file to use.
newext The new extension to use.

보고:

예시:

ext = filename_change_ext(file_find_first(working_directory + "*.*", 0), "");

위의 코드는 확장 부분이 제거된 파일 이름(문자열)을 가져옵니다.
