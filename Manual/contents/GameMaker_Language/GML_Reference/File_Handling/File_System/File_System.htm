파일 시스템

GameMaker 에는 모든 유형의 파일을 다양한 방식으로 조작할 수 있는 여러 기능이 있습니다.

  • file_exists
  • file_delete
  • file_rename
  • file_copy
  • file_find_first
  • file_find_next
  • file_find_close
  • file_attributes

다음 기능을 사용하여 파일 이름을 변경할 수 있습니다.

참고 실제 파일에서는 이러한 기능이 작동하지 않습니다! 다른 경로 요소와 관련된 문자열만 처리하며 다른 파일 기능과 함께 사용해야 합니다.

  • filename_name
  • filename_path
  • filename_dir
  • filename_drive
  • filename_ext
  • filename_change_ext

마지막으로 파일 유형 필터를 사용하여 파일을 로드하고 저장하는 데 사용할 수 있는 파일 이름을 가져오는 몇 가지 특수 기능이 있습니다. 이러한 기능은 탐색기 창을 열어 게임의 초점을 잃게 합니다. 이것은 게임이 의존하는 모든 입력이 올바르게 등록되지 않을 수 있음을 의미하므로 (키보드 이벤트에서 이러한 기능을 사용하는 경우) 키를 누르거나 누르는 이벤트가 아닌 키 업 이벤트를 사용하는 것이 좋습니다.

참고 이러한 기능은 Windows 및 macOS 전용입니다.

경고 이 기능은 GameMaker 가 샌드박스 처리되어 있기 때문에 예상대로 작동하지 않을 수 있습니다! 자세한 내용은 파일 시스템 섹션을 참조하십시오.

  • get_open_filename
  • get_open_filename_ext
  • get_save_filename
  • get_save_filename_ext
