get_open_filename

이 기능은 대화 상자를 열고 플레이어에게 주어진 필터로 열 파일 이름을 묻습니다. 필터의 형식은 " name1|mask1|name2|mask2|... "입니다. 여기서 마스크에는 세미콜론이 있는 다양한 옵션이 포함될 수 있으며 "*"를 사용하여 모든 문자열을 의미할 수도 있습니다. 예: " bitmaps|*.bmp;*.wmf "는 사용자에게 시스템에서 비트맵 파일의 이름을 선택하도록 요청하지만 두 개의 지정된 확장자가 있는 파일만 선택합니다(bmp 그리고 png ). 이 함수는 파일 자체를 열지 않습니다! 파일의 전체 이름과 경로가 포함된 문자열만 반환합니다. 사용자가 " 취소 "를 누르면 빈 문자열 ""이 반환됩니다.

이 기능을 사용할 때 주의해야 할 중요한 점은 게임이 진행되는 동안 해당 파일에 대한 특정 권한을 부여한다는 것입니다. 따라서 사용자로부터 파일 경로를 받으면 묻지 않고 계속해서 액세스할 수 있습니다. 단, 이 기능은 파일을 읽기 위한 기능이기 때문에 부여된 권한은 읽기 전용이며 OS에서는 저장을 허용하지 않습니다. 파일을 읽고 써야 하는 경우 함수를 사용하여 권한을 얻을 수 있습니다.get_save_filename().

참고: 이 기능은 Windows 및 macOS 대상에서만 유효합니다.

통사론:

get_open_filename(filter, fname);

논쟁 유형 설명
filter The file type filter and/or name.
fname The suggested filename to use.

보고:

예시:

var file;
file = get_open_filename("text file|*.txt", "");
if file != ""
{
    file_text_open_read(file);
}

위의 코드는 사용자에게 열 텍스트 파일을 선택한 다음 반환된 파일 이름과 경로를 확인하도록 요청하고 파일이 있으면 읽을 수 있도록 엽니다.
