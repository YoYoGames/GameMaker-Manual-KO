filename_name

이 함수를 사용하면 지정된 파일 의 이름 부분이 반환되지만 확장자는 포함되지만 경로는 포함되지 않습니다.

통사론:

filename_name(fname);

논쟁 유형 설명
fname The file to use.

보고:

String

예시:

name = filename_name(file_find_first("C:/Games/*.doc", 0));

위의 코드는 지정된 디렉토리에서 찾은 첫 번째 "doc" 유형 파일의 이름(문자열으로)을 가져옵니다.
