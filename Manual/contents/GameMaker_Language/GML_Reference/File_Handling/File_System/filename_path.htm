filename_path

이 함수를 사용하면 마지막 백슬래시를 포함하여 표시된 파일 경로의 경로 부분을 반환합니다.

통사론:

filename_path(fname);

논쟁 유형 설명
fname The file name and path to use.

보고:

예시:

path = filename_path(working_directory + "Test.ini");

위의 코드는 지정된 파일의 경로(문자열)를 가져옵니다.
