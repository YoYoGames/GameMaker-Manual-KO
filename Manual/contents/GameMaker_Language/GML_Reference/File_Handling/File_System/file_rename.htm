file_rename

이 함수는 지정된 파일의 이름을 지정된 이름으로 바꿉니다. 함수가 반환됩니다 true 파일 이름이 성공적으로 변경된 경우 또는 false 다른 상황에서.

경고! 이 기능은 GameMaker 가 샌드박싱되어 있기 때문에 예상대로 작동하지 않을 수 있습니다! 자세한 내용은 파일 시스템 섹션을 참조하십시오.

통사론:

file_rename(oldname, newname);

논쟁 유형 설명
oldname The name of the file to change.
newname The new name to give the file.

보고:

예시:

if file_exists("level1.txt")
{
    file_rename("level1.txt", "level.txt");
}

이렇게 하면 파일이 있는지 확인하고 파일이 있으면 이름이 바뀝니다.
