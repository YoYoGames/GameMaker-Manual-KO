file_text_write_string

이 기능을 사용하면 이전에 열린 텍스트 파일에 문자열을 쓸 수 있습니다. 파일에 이미 정보가 포함되어 있으면 이 정보가 지워지고 파일을 열지 않는 한 파일 시작 부분에 문자열이 기록됩니다.file_text_open_append(). 당신은 또한 사용하여 이것을 피할 수 있습니다file_text_readln() 와 함께 기능file_text_eof() 파일의 내용을 끝까지 반복한 다음 쓰기를 시작하는 함수입니다.

통사론:

file_text_write_string(fileid, str);

논쟁 유형 설명
fileid The id of the file to edit.
str The string to write to the file.

보고:

예시:

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
    file_text_write_real(file, scr[i]);
    file_text_writeln(file);
    file_text_write_string(file, scr_name[i]);
    file_text_writeln(file);
}
file_text_close(file);

위의 코드는 쓰기 위해 파일을 연 다음 두 배열을 반복하여 각 배열 값을 파일의 새 줄에 씁니다. 그런 다음 루프가 완료되면 파일이 닫힙니다.
