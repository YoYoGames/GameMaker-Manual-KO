file_text_readln

이 기능을 사용하면 주어진 열린 텍스트 파일에서 현재 줄의 나머지 부분을 건너뛰고 다음 줄의 시작 부분으로 이동할 수 있습니다. 이 함수는 또한 전체 행을 문자열로 반환하므로 나중에 구문 분석을 위해 데이터의 완전한 "덩어리"를 쉽게 읽을 수 있습니다.

통사론:

file_text_readln(fileid);

논쟁 유형 설명
fileid The id of the file to read from.

보고:

예시:

var file = file_text_open_read(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
    scr[i] = file_text_read_real(file);
    file_text_readln(file);
    scr_name[i] = file_text_read_string(file);
    file_text_readln(file);
}
file_text_close(file);

위의 코드는 읽기 위해 파일을 연 다음 나중에 사용할 수 있도록 실수 값과 문자열을 번갈아 가며 두 개의 다른 배열로 읽는 파일 행을 반복합니다. 그런 다음 루프가 완료되면 파일이 닫힙니다.
