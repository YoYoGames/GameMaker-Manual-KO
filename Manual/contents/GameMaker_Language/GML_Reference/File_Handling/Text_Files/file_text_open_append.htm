file_text_open_append

이 함수는 쓰기 를 위해 지정된 파일 이름으로 텍스트 파일을 열고(파일이 존재하지 않으면 생성됨), 모든 추가 작업에 사용될 파일의 고유 ID 를 변수에 저장해야 합니다. 그 파일로 하세요. 쓰기 대상 파일 내의 위치는 파일에 포함된 텍스트의 마지막 줄로 설정되므로 기존 데이터를 덮어쓰지 않습니다.

파일을 생성 할 수 없는 경우(예: 잘못된 파일 이름으로 인해) 함수는 -1을 반환합니다.

참고: 한 번에 최대 32개의 파일만 열 수 있습니다. 또한 정보를 기록하고 파일과 관련된 메모리를 해제하므로 완료되면 항상 파일을 닫아야 합니다.

경고! 이 기능은 GameMaker 가 샌드박싱되어 있기 때문에 예상대로 작동하지 않을 수 있습니다! 자세한 내용은 파일 시스템 섹션을 참조하십시오.

통사론:

file_text_open_append(fname);

논쟁 유형 설명
fname 끈 The name of the file to append to.

보고:

Text File ID or -1

예시:

file = file_text_open_append(working_directory + "save.txt");

이렇게 하면 게임과 동일한 디렉터리에서 "save.txt"가 열리고 파일 ID가 "file" 변수에 저장됩니다.
