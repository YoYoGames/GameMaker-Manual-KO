file_text_close

주어진 파일에 대한 작업(읽기 또는 쓰기)을 마치면 파일을 다시 닫아야 합니다. 그렇지 않으면 파일에 포함된 정보가 손실될 위험이 있습니다. 이것은 또한 메모리 누수를 방지하고 32개 이상의 파일을 열어서 파일 제한을 초과하지 않도록 합니다.

통사론:

file_text_close(fileid);

논쟁 유형 설명
fileid The id of the file to close.

보고:

예시:

var file = file_text_open_write(working_directory + "Game_Data.txt");
while (!file_text_eof(file))
{
    file_text_readln(file);
}
file_text_write_string(file, level_data);
file_text_close(file);

위의 코드는 쓰기 위해 파일을 연 다음 파일에 이미 쓰여진 텍스트 줄을 끝까지 반복합니다. 이 시점에서 문자열을 작성한 다음 파일을 다시 닫습니다.
