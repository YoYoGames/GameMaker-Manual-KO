file_text_open_from_string

이 함수는 문자열에서 텍스트 파일을 만들고 읽기 위해 열어서 이 파일에서 읽기 위해 모든 추가 파일 함수 호출에 사용해야 하는 파일 "핸들"을 반환합니다. 이 파일은 임시 파일이며 읽기 전용 이므로 닫는 순간 메모리에서 제거됩니다.

참고: 한 번에 최대 32개의 파일만 열 수 있습니다. 또한 정보를 기록하고 파일과 관련된 메모리를 해제하므로 완료되면 항상 파일을 닫아야 합니다.

경고! 이 기능은 GameMaker 가 샌드박싱되어 있기 때문에 예상대로 작동하지 않을 수 있습니다! 자세한 내용은 파일 시스템 섹션을 참조하십시오.

통사론:

file_text_open_from_string(string);

논쟁 유형 설명
string The string to create the file from.

보고:

File ID or -1

예시:

file = file_text_open_from_string(reset_str);

위의 코드는 "reset_str" 변수에 저장된 문자열을 가져와서 읽기 전용 텍스트 파일을 만드는 데 사용합니다. 이 파일의 "핸들"은 모든 추가 파일 기능이 사용할 수 있도록 "file" 변수에 저장됩니다.
