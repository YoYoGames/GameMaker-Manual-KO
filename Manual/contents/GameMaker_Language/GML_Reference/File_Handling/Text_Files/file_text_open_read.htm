file_text_open_read

이 함수는 읽기 전용 으로 지정된 파일 이름으로 텍스트 파일을 열고, 해당 파일과 관련하여 수행할 모든 추가 작업에 사용되므로 변수에 저장해야 하는 파일의 고유 ID를 반환합니다. 파일이 존재하지 않으면 함수는 -1 값을 반환합니다.

참고: 한 번에 최대 32개의 파일만 열 수 있습니다. 또한 정보를 기록하고 파일과 관련된 메모리를 해제하므로 완료되면 항상 파일을 닫아야 합니다.

경고! 이 기능은 GameMaker 가 샌드박싱되어 있기 때문에 예상대로 작동하지 않을 수 있습니다! 자세한 내용은 파일 시스템 섹션을 참조하십시오.

통사론:

file_text_open_read(fname);

논쟁 유형 설명
fname The name of the file to read from.

보고:

Text File ID or -1

예시:

file = file_text_open_read(working_directory + "level.txt");

이렇게 하면 게임과 동일한 디렉토리에서 "level.txt"가 열리고 파일 ID가 "file" 변수에 저장됩니다.
