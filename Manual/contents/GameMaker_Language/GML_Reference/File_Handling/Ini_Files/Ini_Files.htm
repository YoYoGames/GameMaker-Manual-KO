이니 파일

INI 파일은 대부분의 플랫폼과 호환되는 작고 가벼운 파일입니다. 인터페이스 기본 설정, 로컬 최고 점수, 레벨 데이터 등과 같은 작은 정보를 저장하는 데 이상적이며 게임에서 정보를 저장하기 시작하는 훌륭하고 쉬운 방법입니다. 정보를 파일에 저장하는 고급 방법을 찾고 있다면 Text Files, Binary Files 및 Buffers 와 같은 다른 파일 기능을 살펴보는 것이 좋습니다.

중요한! 모든 GameMaker 프로젝트에는 "options.ini "이므로 해당 이름으로 파일 이름을 지정할 수 없습니다 (게임 프로젝트를 컴파일하려고 할 때 uid 오류가 발생함).

현지화 게임의 경우 외부 파일에서 악센트 문자를 읽을 수 있다는 것이 매우 중요합니다. 즉, 먼저 UTF8 형식의 ini 파일을 생성한 다음 게임을 실행할 때 내보내고 ini 함수에 의해 생성된 기본 ANSI 형식 ini 파일 대신 사용되도록 GameMaker 에 포함된 파일로 추가해야 합니다. 이전에 제공된 적이 없습니다. 이런 식으로 모든 악센트와 비로마 문자를 유지하면서 올바르게 읽고 쓸 수 있습니다.

다음 함수는 ini 파일에 대해 존재합니다.

  • ini_open
  • ini_close
  • ini_write_real
  • ini_write_string
  • ini_read_real
  • ini_read_string
  • ini_key_exists
  • ini_section_exists
  • ini_key_delete
  • ini_section_delete
  • ini_open_from_string
