ini_key_exists

이 함수는 현재 열려 있는 ini에 키가 있는지 확인하고 다음을 반환합니다. true 그것이 하는 경우 또는 false 그렇지 않으면. 키가 존재하지 않는 경우 존재하지 않는 키를 읽으면 기본값이 반환되므로 오류를 방지하기 위해 필요한 검사는 아닙니다. 그러나 ini 파일에 특정 데이터와 기타 몇 가지 사항이 저장되어 있는지 확인하는 것이 유용할 수 있습니다.

통사론:

ini_key_exists(section, key);

논쟁 유형 설명
section The section in the open.ini file to check a key in.
key The key to check for.

보고:

예시:

ini_open("savedata.ini");
if !ini_key_exists("save1", "name")
{
    global.name = "Player1";
}
ini_close();

"name"과 같은 키가 없으면 변수 global.name을 "Player1"로 설정합니다.
