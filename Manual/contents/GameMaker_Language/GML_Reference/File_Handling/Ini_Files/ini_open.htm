ini_open

이것은 읽기 및/쓰기를 위해 ini_file을 엽니다. 확인 중인 위치에 ini_file이 존재하지 않는 경우 GameMaker 는 생성할 수 있지만 데이터를 쓰는 경우에만 가능합니다. ini 파일에서 읽기 정보만 있는 경우 읽기 기능의 기본값이 반환되지만 ini 파일은 실제로 생성 되지 않습니다.

한 번에 하나 의 ini 파일만 열 수 있다는 점에 유의하십시오.ini_close() 정보가 실제로 디스크에 저장되지 않기 때문에.ini 파일에서 읽기/쓰기가 끝나면 파일이 닫힐 때까지 메모리에도 저장됩니다.

경고! 이 기능은 GameMaker 가 샌드박싱되어 있기 때문에 예상대로 작동하지 않을 수 있습니다! 자세한 내용은 파일 시스템 섹션을 참조하십시오.

통사론:

ini_open(name);

논쟁 유형 설명
name The filename for the.ini file.

보고:

예시:

ini_open("Settings/savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

이렇게 하면 'savedata.ini'가 열리고 "score" 키가 있는 "save1" 섹션 아래의 점수 값을 읽은 다음.ini를 다시 닫습니다. "save1", "score" 아래에 값이 없거나 "savedata.ini" 파일이 없으면 점수는 0(기본값)으로 설정됩니다. ini 파일은 Asset Browser 포함 파일에서 ini 파일을 보유하는 폴더인 "Settings" 하위 디렉토리에 있습니다.
