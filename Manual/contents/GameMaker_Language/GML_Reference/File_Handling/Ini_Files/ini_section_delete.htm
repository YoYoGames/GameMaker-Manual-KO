ini_section_delete

이 기능을 사용하면 ini 파일의 전체 섹션을 삭제할 수 있으며, 이와 관련된 모든 키-값 쌍도 제거됩니다.

통사론:

ini_section_delete(section);

논쟁 유형 설명
section The section to delete.

보고:

예시:

ini_open("savedata.ini");
ini_write_real("save1", "Score", score );
ini_section_delete("save1");
ini_close();

이 예는 "savedata.ini"를 열고 "save1" > "Score"에 값을 씁니다. 그런 다음 "save1" 섹션을 삭제하고.ini 파일을 닫습니다.
