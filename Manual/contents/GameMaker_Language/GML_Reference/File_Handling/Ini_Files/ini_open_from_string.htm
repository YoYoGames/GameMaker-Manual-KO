ini_open_from_string

이 함수는 문자열에서 임시 ini 파일을 만들고 읽기/쓰기를 위해 엽니다. 문자열은 ini 파일 형식으로 올바르게 지정되어야 합니다(예: 섹션, 줄 바꿈 키 및 값 포함). 그렇지 않으면 ini 파일이 올바르게 생성되지 않습니다. 이 ini 파일은 임시 파일이며 닫히는 순간 메모리에서 제거되므로 저장된 정보가 손실되지만ini_close() 함수는 서버나 디스크에 저장할 수 있는 전체 ini 파일의 문자열을 반환합니다.

통사론:

ini_open_from_string(string);

논쟁 유형 설명
string The string containing all the ini information.

보고:

N/A

예시:

ini_open_from_string(str)
global.sound = ini_read_string("Options", "Sound", true);
ini_close();

위의 코드는 문자열 "str"의 정보를 보유하도록 임시 ini를 설정한 다음 다시 닫기 전에 ini에서 읽습니다.
