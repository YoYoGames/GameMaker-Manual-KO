ini_write_real

이 기능을 사용하여 ini 데이터 파일에 값(숫자)을 쓸 수 있습니다. Ini 파일은 섹션 으로 분할되고 각 섹션은 이후에 키 - 값 쌍으로 분할됩니다. 따라서 일반적인 ini 파일은 다음과 같습니다.

통사론:

ini_write_real(section, key, value);

논쟁 유형 설명
section The section of the.ini to write to.
key The key within the relevant section of the.ini to write to.
value The real value to write to the relevant destination.

보고:

예시:

score = 1000;
ini_open("savedata.ini");
ini_write_real("save1", "Score", score );
score2 = ini_read_real("save1", "Score", 0 );
ini_close();

이것은 점수를 1000으로 설정한 다음 "savedata.ini"를 열고 이 값을 "save1" > "Score"에 씁니다. 그런 다음 score2를 "save1" > "score" 아래의 실제 값인 1000으로 설정하고.ini를 다시 닫습니다. 이 값을 찾지 못했다면 score2는 결국 0으로 설정되었을 것입니다.
