ini_write_string

이 함수를 사용하여 ini 데이터 파일에 문자열(텍스트)을 쓸 수 있습니다. Ini 파일은 섹션 으로 분할되고 각 섹션은 이후에 키 - 값 쌍으로 분할됩니다. 따라서 일반적인 ini 파일은 다음과 같습니다.

ini 문자열을 작성할 때 따옴표와 이스케이프 문자를 사용하면 데이터가 너무 일찍 잘릴 수 있으므로 나중에 다시 읽을 때 문제가 발생할 수 있습니다. 예를 들어 다음과 같이 작성하면 문제가 됩니다.

ini_write_string("test2", "section", "hello \"Fritz\"");

통사론:

ini_write_string(section, key, value);

논쟁 유형 설명
section 끈 The section of the.ini to write to.
key 끈 The key within the relevant section of the.ini to write to.
value 끈 The string to write to the relevant destination.

예시:

ini_open("savedata.ini");
ini_write_string("Save", "Player", global.Name);
ini_close();

위의 코드는 읽기 및 쓰기를 위해 ini 파일을 연 다음 파일을 다시 닫기 전에 "Player" 키를 사용하여 전역 변수 "Name"에 저장된 문자열을 "Save" 섹션에 씁니다.
