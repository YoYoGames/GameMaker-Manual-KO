ini_close

이 함수는 열려 있는 ini 파일에 대한 읽기 또는 쓰기를 마치는 순간 호출되어야 합니다. ini 쓰기 기능을 사용한 후 이 기능을 사용하지 않으면 이 기능이 호출될 때까지 파일 정보가 메모리에 유지되어 쓰기를 강제 실행하므로 디스크에 아무 것도 기록되지 않습니다. 이전에 다른 것(또는 동일한 것)을 닫지 않고 ini를 열려고 하면 오류가 발생합니다.

이 함수는 ini 파일이 인코딩된 문자열도 반환합니다. 그런 다음 이 문자열을 서버에 저장하거나 함수와 함께 다시 사용할 수 있습니다.ini_open_from_string() ini를 다시 생성합니다.

통사론:

ini_close();

보고:

String

예시:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

이렇게 하면 "savedata.ini"가 열리고 "score" 키가 있는 "save1" 섹션에서 값을 읽은 다음 ini 파일을 닫습니다. "save1" 아래에 "score"라는 값이 없거나 "savedata.ini" 파일이 없으면 0이 반환되고 새 ini 파일이 생성됩니다.
