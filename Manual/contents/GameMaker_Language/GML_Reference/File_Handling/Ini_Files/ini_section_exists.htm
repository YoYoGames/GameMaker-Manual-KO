ini_section_exists

이 함수는 현재 열려 있는 ini에 섹션이 있는지 확인하고 다음을 반환합니다. true 그것이 하는 경우 또는 false 그렇지 않으면. 섹션이 존재하지 않을 때 존재하지 않는 섹션에서 읽는 것은 기본값을 반환하기 때문에 오류를 방지하기 위해 필요한 검사는 아니지만 ini 파일이 특정 데이터를 저장했는지 확인하는 것이 유용할 수 있습니다.

통사론:

ini_section_exists(section);

논쟁 유형 설명
section The section in the open.ini file to check for.

보고:

예시:

ini_open("savedata.ini");
if !ini_section_exists("save1")
{
    global.savegame = "1";
}
ini_close();

"save1"이 없으면 변수 global.savegame을 "1"로 설정합니다. for 루프를 사용하면 게임에 몇 개의 저장 게임이 있는지 계산하는 기본 방법으로 사용할 수 있습니다(예: 한 게임에 대해 여러 개의 동시 저장 게임을 위한 시스템 만들기).
