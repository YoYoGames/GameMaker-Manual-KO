ini_read_string

이 함수를 사용하여 ini 데이터 파일에서 문자열(텍스트)을 읽을 수 있습니다. Ini 파일은 섹션 으로 분할되고 각 섹션은 이후에 키 - 값 쌍으로 분할됩니다. 따라서 일반적인 ini 파일은 다음과 같습니다.

통사론:

ini_read_string(section, key, default);

논쟁 유형 설명
section 끈 The section of the.ini to read from.
key 끈 The key within the relevant section of the.ini to read from.
default 끈 The string to return if a string is not found in the defined place (or the.ini file does not exist). Must be a string.

보고:

String

예시:

ini_open("savedata.ini");
global.name = ini_read_string("player", "name', "Player1");
ini_close();

이렇게 하면 "savedata.ini"가 열리고 global.name이 "save1" > "Name" 아래의 문자열로 설정된 다음.ini를 다시 닫습니다. "save1" > "Name" 아래에 문자열이 없거나 "savedata.ini" 파일이 없으면 global.name이 "Player1"로 설정됩니다.
