weak_ref_create

이 함수를 사용하면 구조체에 대한 약한 참조 를 만든 다음 구조체가 게임에서 여전히 "살아있는"(참조)인지 여부를 확인하는 데 사용할 수 있습니다. 추적하려는 구조체에 대한 참조를 제공하면 함수는 해당 구조체에 대한 약한 참조인 다른 구조체를 반환합니다. 함수를 사용하여 참조가 "강함"인지 "약함"인지 확인할 수 있습니다. instanceof(), 강력한 참조는 "struct " 또는 구조체를 생성한 생성자 함수의 이름, 또는 "weakref " 약한 참조인 경우. 또한 약한 참조 구조체에는 변수가 있다는 점에 유의하십시오. "ref " 가비지 수집되지 않은 경우 해당 구조체에 대한 강력한 참조를 얻기 위해 액세스할 수 있습니다. undefined.

통사론:

weak_ref_create(struct_to_track);

논쟁 유형 설명
struct_to_track 구조체 The struct that you want to create a weak reference for

보고:

Struct Weak Reference

예시:

inventory_ref = weak_ref_create(inventory);

위의 코드는 구조체에 대한 약한 참조를 만들고 나중에 사용할 수 있도록 인스턴스 변수에 저장합니다.
