gc_get_target_frame_time

이 함수를 사용하면 가비지 수집기에 대한 현재 대상 프레임 값을 검색할 수 있습니다. 반환된 값은 마이크로초 (여기서 1,000,000마이크로초는 1초임)이며 기본 대상 프레임 시간은 100마이크로초입니다. 이 값을 변경하려면 함수를 사용해야 합니다.gc_target_frame_time().

통사론:

gc_get_target_frame_time(time);

보고:

예시:

if (gc_get_target_frame_time() != 50)
{
    gc_target_frame_time(50);
}

위의 코드는 가비지 수집기의 현재 프레임 시간 목표를 확인하고 50마이크로초가 아니면 이 값으로 설정합니다.
