gc_collect

이 기능을 사용하면 현재 프레임(단계)의 끝에서 강제로 실행하도록 가비지 수집기를 트리거할 수 있습니다. 이것이 작동하기 위해 가비지 수집기가 활성화될 필요가 없다는 점은 주목할 가치가 있습니다. 가비지 수집기를 비활성화한 후 이 함수 호출(함수 사용 gc_enable() ) 가비지 수집기가 다시 비활성화되기 전에 수집 플래그가 지정된 모든 개체가 메모리에서 제거되는 한 프레임에 대해 가비지 수집기를 활성화합니다.

통사론:

gc_collect();

보고:

예시:

if (global.debug == true && keyboard_check_pressed(vk_f1))
{
    gc_collect();
}

위의 코드는 전역 변수와 누르고 있는 키를 확인합니다. true 그런 다음 프레임의 끝(단계)에 대해 가비지 수집이 트리거됩니다.
