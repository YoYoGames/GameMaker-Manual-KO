weak_ref_any_alive

이 기능을 사용하면 다양한 구조체에 대한 약한 참조 를 확인하여 여전히 "살아있는"지 확인할 수 있습니다. 확인하기 위해 약한 참조의 배열을 제공합니다(각 약한 참조는 함수를 사용하여 생성되어야 합니다.weak_ref_create() ), 그리고 함수는 반환할 것입니다 true 구조체 중 하나 라도 여전히 어딘가에서 참조되고 있거나 false 그렇지 않고 가비지 수집된 경우. 값 중 하나라도 약한 참조가 아닌 배열을 제공하면 함수는 다음을 반환합니다. undefined.

이 함수는 또한 두 개의 선택적 인수를 취합니다. 여기서 첫 번째 인수는 검사할 배열에 초기 인덱스를 제공할 뿐만 아니라 해당 인덱스에서 확인할 위치 수(길이)를 설정하는 인수입니다. 지정된 경우 해당 매개변수 내의 배열 인덱스만 검사됩니다.

통사론:

weak_ref_any_alive(array, [index], [length]);

논쟁 유형 설명
array 구조체 약한 참조 배열 Array containing weak references to the structs that you want to check.
[index] 진짜 OPTIONAL The index into the array to start checking from
[length] 진짜 OPTIONAL The number of positions, starting from the given index, to check for

보고:

Boolean (or undefined)

예시:

if weak_ref_any_alive(inventory_ref_array)
{
    instance_destroy(obj_Inventory_Control);
}

위의 코드는 약한 참조의 배열을 검사하고 아직 살아있는 경우 인스턴스가 파괴됩니다.
