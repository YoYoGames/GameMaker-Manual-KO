gc_is_enabled

이 기능을 사용하면 가비지 수집기가 활성화되었는지 여부를 확인할 수 있습니다. 함수가 반환됩니다 true 활성화된 경우 또는 false 그렇지 않으면.

통사론:

gc_is_enabled();

보고:

예시:

if (!gc_is_enabled())
{
    gc_enable(true);
}

위의 코드는 가비지 수집기가 활성화되어 있는지 확인하고 그렇지 않으면 활성화합니다.
