gc_enable

이 기능을 사용하여 가비지 수집기를 활성화하거나 비활성화할 수 있습니다. 다음을 사용하여 함수 호출 true 인수가 그것을 가능하게하고 사용하기 때문에 false 비활성화합니다(권장하지 않음). 기본적으로 활성화되어 있습니다.

통사론:

gc_enable(enable);

논쟁 유형 설명
enable enable (true ) or disable (false ) the garbage collector.

보고:

예시:

if (global.debug == true)
{
    gc_enable(false);
}

위의 코드는 주어진 전역 변수가 다음과 같은 경우 가비지 수집을 비활성화합니다. true.
