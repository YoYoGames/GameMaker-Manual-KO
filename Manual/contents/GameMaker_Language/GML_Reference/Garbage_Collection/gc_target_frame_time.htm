gc_target_frame_time

이 기능을 사용하면 각 프레임을 실행하는 데 지정된 시간 이상을 소비하지 않아야 함을 가비지 수집기에 나타낼 수 있습니다. 이 함수는 마이크로초로 지정된 시간 값을 사용하며(여기서 1,000,000마이크로초는 1초임) 기본 대상 프레임 시간은 100마이크로초입니다.

일부 작업은 여전히 한 프레임에서 완전히 실행해야 하고 때로는 목표 시간보다 더 오래 걸릴 수 있으므로 이는 단순한 목표 값이라는 점에 유의하십시오. 또한 목표 시간을 늘리면 가비지 수집기가 메모리 사용량의 급격한 변화에 더 민감하게 반응하고 메모리가 더 빨리 해제되지만 실제로는 대부분의 경우 필요하지 않습니다. 또한 대상 프레임 시간을 0으로 설정해도 수집기가 완전히 중지되지는 않습니다. 여전히 각 프레임에서 최소한의 작업을 수행합니다. 수집기를 완전히 비활성화하려면 다음을 사용하십시오. gc_enable() 함수.

통사론:

gc_target_frame_time(time);

논쟁 유형 설명
time The target time - in microseconds - that the garbage collector should work each frame

보고:

N/A

예시:

if (gc_get_target_frame_time() != 50)
{
    gc_target_frame_time(50);
}

위의 코드는 가비지 수집기의 현재 프레임 시간 목표를 확인하고 50마이크로초가 아니면 이 값으로 설정합니다.
