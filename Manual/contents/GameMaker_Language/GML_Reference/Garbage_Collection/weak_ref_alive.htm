weak_ref_alive

이 함수를 사용하면 구조체에 대한 약한 참조 를 확인하여 구조체가 여전히 "살아있는"지 여부를 확인할 수 있습니다. 확인하기 위해 약한 참조를 제공합니다(함수에 의해 반환됨).weak_ref_create() ), 그리고 함수는 반환할 것입니다 true 구조체가 여전히 어딘가에서 참조되고 있거나 false 그렇지 않고 가비지 수집된 경우. 약한 참조가 아닌 값을 제공하면 함수는 다음을 반환합니다. undefined.

통사론:

weak_ref_alive(weak_ref);

논쟁 유형 설명
weak_ref 구조체 약한 참조 The weak reference to the struct you want to check.

보고:

Boolean (or undefined)

예시:

if weak_ref_alive(inventory_ref)
{
    inventory = -1;
}

위의 코드는 구조체에 대한 약한 참조를 확인하고 구조체를 참조하는 변수가 아직 살아 있으면 참조하는 변수가 -1로 설정되어 구조체를 역참조하고 가비지 수집이 가능하게 합니다.
