롤백_확인_프레임

이 전역 변수는 모든 플레이어의 입력이 동기화되고 확인된 마지막 프레임 번호를 저장합니다.

당신은 이것을 비교할 수 있습니다rollback_current_frame 현재 프레임이 모든 플레이어에 대한 입력을 확인했는지 확인합니다.

기능을 사용할 수도 있습니다. rollback_sync_on_frame() 현재 프레임이 동기화되었는지 확인합니다.

통사론:

rollback_confirmed_frame

보고:

Real

예시:

var _is_frame_confirmed = (rollback_current_frame == rollback_confirmed_frame);

if (_is_frame_confirmed)
{
    evaluate_game_over();
}

이 코드는 현재 프레임 값 이 확인된 프레임 값 과 같은지 확인하여 현재 프레임이 모든 플레이어의 입력을 확인했는지 확인합니다.

이것이 사실이면 게임 오버 조건이 만족되었는지 확인하는 기능을 실행하여 게임을 종료할 수 있습니다.

이것은 게임을 종료하는 예측 기반 상태를 피하기 위해 수행되며 게임 오버 조건을 확인하기 전에 완전히 동기화된 프레임을 기다립니다.
