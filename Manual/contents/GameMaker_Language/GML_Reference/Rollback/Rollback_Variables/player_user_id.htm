player_user_id

이 변수는 다음에 의해 생성된 플레이어 인스턴스에만 존재합니다.rollback_define_player(), 그리고 플레이어마다 다릅니다.

게스트 플레이어 는 이 변수에 빈 문자열이 할당됩니다.
