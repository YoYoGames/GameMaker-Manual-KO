player_prefs

이 변수는 현재 플레이어에 대해 설정된 기본 설정을 저장합니다. 다음을 사용하여 생성된 플레이어 인스턴스에 할당됩니다.rollback_define_player().

읽어주세요: 플레이어 기본 설정

통사론:

player_prefs;

보고:

Any

예시:

image_index = player_prefs.character;

위의 코드는 환경 설정을 읽고 플레이어에서 인스턴스 변수를 설정합니다.
