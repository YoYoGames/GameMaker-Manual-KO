플레이어_로컬

사용할 때rollback_define_player() 플레이어 개체를 정의하기 위해 시스템은 각 플레이어에 대해 해당 개체의 인스턴스를 자동으로 만듭니다. 해당 객체의 각 인스턴스는 player_local 해당 플레이어 인스턴스가 로컬인지 여부를 알려주는 변수입니다.

사용하지 않는 경우 rollback_define_player(), 이 변수는 개체에 대해 생성되지 않습니다.

이 변수를 사용하여 로컬 플레이어를 표시할 수 있습니다. 로컬 및 원격 플레이어에 대해 동일해야 하는 게임 로직에 영향을 주는 방식으로 이 변수를 사용하지 않는 것이 좋습니다.

통사론:

player_local;

보고:

Boolean

예시:

// Draw event
if (player_local)
{
    draw_sprite(spr_arrow, 0, x, y - 60);
}

위의 코드는 Draw 이벤트에서 실행되고 플레이어 인스턴스가 로컬인지 확인합니다. 이 경우 인스턴스 위에 화살표 스프라이트를 그려 플레이어가 제어하는 인스턴스를 알려줍니다.
