롤백_게임_실행 중

이 전역 변수는 롤백 게임이 현재 실행 중인지 여부를 나타냅니다. 이것이 true 이면 롤백 게임이 시작되어 현재 실행 중임을 의미합니다. 다른 모든 경우에는 false 가 됩니다.

통사론:

rollback_game_running

보고:

Boolean

예시:

if (!rollback_game_running) return;

x += move_x;
y += move_y;

위의 코드는 플레이어 개체의 Step 이벤트에서 실행됩니다. 롤백 게임이 실행되고 있지 않으면 이벤트를 종료하므로 롤백 게임이 시작될 때까지 플레이어를 제어할 수 없습니다.
