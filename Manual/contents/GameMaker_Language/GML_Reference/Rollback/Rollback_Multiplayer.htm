롤백 멀티플레이어

GameMaker의 멀티플레이어 롤백을 사용하면 Opera GX 대상을 위한 네트워크 멀티플레이어 게임을 빠르게 설정하고 생성할 수 있습니다.

이 기능은 GX 서버를 사용하여 플레이어를 연결하고 입력을 동기화하므로 개발자가 새로운 온라인 게임을 시작하기 위해 서버를 설정할 필요가 없습니다.

온라인 기능은 Opera GX 대상에서만 사용할 수 있지만 "동기화 테스트"를 사용하여 다른 플레이어에게 무작위 또는 모의 입력을 적용하여 로컬에서 멀티플레이어 게임을 테스트할 수 있습니다. 동기화 테스트는 Opera GX 외에도 Windows 및 macOS 대상에서 작동합니다.

시작하기

멀티플레이어 게임 만들기를 읽고 단계에 따라 첫 번째 멀티플레이어 프로젝트를 만듭니다.

첫 멀티플레이어 게임을 만든 후 다음 페이지를 읽고 롤백 시스템을 더 잘 이해하고 이 시스템으로 무엇을 할 수 있는지 알아보세요.
