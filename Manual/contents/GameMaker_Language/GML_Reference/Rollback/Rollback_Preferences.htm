롤백 기본 설정

각 플레이어가 캐릭터를 선택하거나 속성을 사용자 정의하도록 허용할 수 있습니다. 이러한 속성은 "플레이어 기본 설정"으로 설정할 수 있습니다.

환경 설정을 사용하는 방법?

먼저 전화를 걸어 기본 설정을 활성화하십시오.rollback_use_player_prefs() / 플레이어 기본 설정 사용. 이 기능을 사용하면 모든 플레이어에 대한 기본 기본 설정을 선택적으로 지정할 수 있습니다.

해당 기능을 호출하면 "자동 시작"이 비활성화됩니다. 즉, 모든 플레이어가 참가하더라도 게임이 시작되지 않으며 다음을 사용하여 수동으로 시작해야 합니다.rollback_start_game() / 게임 시작.

게임이 시작되기 전에 다음을 호출하여 플레이어의 기본 설정을 지정할 수 있습니다.rollback_set_player_prefs() / 해당 플레이어의 클라이언트에서 플레이어 기본 설정 을 설정합니다. 이것들은 나중에 읽을 수 있습니다rollback_get_player_prefs() / 플레이어 기본 설정 가져오기 또는 player_prefs 플레이어 인스턴스의 변수.

롤백 이벤트

플레이어가 기본 설정을 변경하면 게임의 모든 사람이 rollback_start_game()>rollback_player_prefs 롤백 이벤트 를 받습니다. 이벤트를 보내려면 플레이어가 완전히 연결되어 있어야 합니다.

연결하기 전에 기본 설정을 설정한 경우 연결 완료 시 연결하기 전에 마지막으로 설정한 기본 설정이 모든 플레이어에게 전송됩니다.

시작 조건

환경 설정을 활성화했는데 기본 환경 설정이 설정되지 않은 경우 모든 플레이어에 대한 환경 설정이 지정되지 않으면 게임이 시작되지 않습니다.

이러한 경우 다음 중 하나를 호출할 수 있습니다. 알람에서 반복적으로 실행하여 게임 시작을 시도하거나 rollback_player_prefs 롤백 이벤트 가 수신될 때마다 호출합니다.

어떤 유형을 사용할 수 있습니까?

기본 설정은 숫자, 문자열, 배열 등과 같은 모든 유형이 될 수 있지만 변수 모음을 저장하는 가장 쉬운 방법인 구조체 를 사용하는 것이 좋습니다.
