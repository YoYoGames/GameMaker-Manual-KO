rollback_use_random_input

Sync Test 를 사용하여 게임을 할 때 모든 "원격" 플레이어는 기본 테스트 방법으로 정의된 입력 에 대해 임의의 값을 받습니다. false 를 인수로 사용하여 이 함수를 호출하여 이 동작을 비활성화하고 true 를 지정하여 다시 활성화할 수 있습니다.

통사론:

rollback_use_random_input(enabled);

논쟁 유형 설명
enabled 부울 Specifies whether random input should be enabled ( true ) or disabled ( false ). By default, it's enabled.

보고:

N/A

예시:

rollback_use_random_input(false);

위의 코드는 동기화 테스트 중에 임의 입력을 비활성화합니다.
