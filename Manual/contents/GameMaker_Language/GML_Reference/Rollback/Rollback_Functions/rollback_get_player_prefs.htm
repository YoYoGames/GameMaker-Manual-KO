rollback_get_player_prefs

이 함수는 주어진 플레이어 ID에 대해 설정된 기본 설정을 반환합니다.

플레이어 ID가 지정되지 않은 경우 플레이어 인스턴스 내에서 함수를 호출해야 하며, 이 경우 해당 플레이어에 대한 기본 설정이 반환됩니다.

인스턴스 변수를 사용할 수도 있습니다. player_prefs.

이 함수는 주어진 플레이어에 대해 동기화된 마지막 기본 설정을 반환하므로 기본 설정을 지정해도 이 함수가 반환하는 기본 설정이 즉시 업데이트되지 않습니다.

읽어주세요: 플레이어 기본 설정

통사론:

rollback_get_player_prefs([player_id]);

논쟁 유형 설명
player_id 진짜 OPTIONAL The ID of the player for which preferences are returned.

보고:

Any

예시:

var _my_prefs = 롤백_get_player_prefs();

이미지 인덱스 = _my_prefs.character;

위의 코드는 현재 플레이어에 대해 설정된 기본 설정을 가져오고 기본 설정을 인스턴스 변수에 적용합니다.
