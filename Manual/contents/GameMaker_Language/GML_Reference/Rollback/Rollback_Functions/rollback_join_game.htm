롤백_조인_게임

이 기능은 게임에 참여를 시도합니다. 참가할 가상 회의실의 ID가 포함된 URL에 있어야 합니다.

방에 참가했다면 true 를 반환하고, 참가한 게임이 없으면 false 를 반환합니다.

새로운 플레이어가 이미 참여하고 있는 게임에 참가하면 롤백 이벤트 이벤트가 트리거됩니다. 자세한 내용은 롤백 이벤트 를 참조하십시오.

dry_run 인수는 선택 사항이며 true 로 설정하면 실제로 게임에 참여하지 않고도 함수를 실행할 수 있습니다. 게임에 실제로 참여하지 않고도 게임에 참여할 수 있는지 확인하는 데 사용됩니다.

통사론:

rollback_join_game([dry_run]);

논쟁 유형 설명
dry_run 부울 Disabled by default ( false ). When enabled ( true ), the function doesn't join the game, but simply returns true if one can be joined, or false otherwise.

This way you can use this function to only check if the player was invited.

보고:

Boolean

예시:

rollback_define_player(obj_player);
var _joined = rollback_join_game();

if (!_joined)
{
    rollback_create_game(2, false);
}

위의 코드는 플레이어 개체를 정의한 다음 게임 참여를 시도합니다. 참여하지 않은 경우 최대 2명의 플레이어와 동기화 테스트가 비활성화된 새 게임을 대신 생성합니다.

확장된 예 는 멀티플레이어 게임 만들기 를 참조하십시오.
