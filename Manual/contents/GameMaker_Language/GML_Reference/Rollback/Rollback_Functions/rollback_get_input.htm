롤백_get_input

이 함수는 플레이어의 입력 값을 포함하는 구조체를 반환합니다. 입력이 반환될 플레이어의 ID를 지정하는 선택적 인수가 필요합니다.

에 의해 생성된 플레이어 인스턴스에서 사용되는 경우rollback_define_player(), 플레이어 ID를 지정할 필요가 없으므로 인수 없이 이 함수를 호출할 수 있습니다.

다른 인스턴스에서 사용하거나 수동으로 생성된 플레이어 인스턴스에서 사용하는 경우 플레이어 ID를 지정해야 합니다.

반환되는 입력과 변경 방법에 대한 정보는 입력 정의 를 참조하십시오.

통사론:

rollback_get_input(player_id);

논쟁 유형 설명
player_id 진짜 OPTIONAL The ID of the player for which inputs will be returned.

보고:

Struct

예시:

var _input = rollback_get_input();

if (_input.left) x -= 2;
if (_input.right) x += 2;
if (_input.up) y -= 2;
if (_input.down) y += 2;

위의 코드는 롤백 시스템에 의해 자동으로 생성된 플레이어 인스턴스의 입력 구조체를 가져옵니다. 반환된 값을 기반으로 인스턴스를 각 방향으로 2픽셀씩 이동합니다.

확장된 예 는 멀티플레이어 게임 만들기 를 참조하십시오.
