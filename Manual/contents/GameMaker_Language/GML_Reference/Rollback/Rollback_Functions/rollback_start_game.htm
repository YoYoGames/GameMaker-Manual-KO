롤백_시작_게임

이 기능은 모든 플레이어가 참여하기 전에 현재 멀티플레이어 게임을 강제로 시작합니다. 먼저 다음을 사용하여 멀티플레이어 게임을 만들어야 합니다.rollback_create_game().

시스템은 일반적으로 자동으로 게임을 시작하기 전에 모든 플레이어가 참여할 때까지 기다립니다. 이 기능은 모든 플레이어가 참여하기 전에 게임을 일찍 시작하는 데 사용할 수 있습니다.

전화해도됩니다rollback_use_manual_start() 모든 플레이어가 참여한 후에도 롤백이 이 기능을 기다리도록 합니다.

이 함수를 호출하면 롤백 시작 이벤트가 트리거됩니다.

통사론:

rollback_start_game();

보고:

N/A

예시:

// Step event
if (!rollback_game_running && keyboard_check_pressed(vk_enter))
{
    rollback_start_game();
}

롤백 게임이 실행되고 있지 않고 Enter 키를 누르면 게임이 시작됩니다.

이렇게 하면 모든 플레이어가 참여하기 전에 플레이어가 수동으로 게임을 시작할 수 있습니다.
