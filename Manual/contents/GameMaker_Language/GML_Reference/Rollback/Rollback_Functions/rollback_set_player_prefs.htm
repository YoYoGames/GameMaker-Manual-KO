rollback_set_player_prefs

기본 설정을 활성화 한 후 이 기능은 로컬 플레이어에 대한 기본 설정을 지정하는 데 사용됩니다. 이것은 함수가 롤백 입력 이 아닌 로컬 입력 에 의존해야 함을 의미합니다.

기본 설정은 모든 유형이 될 수 있습니다. 기본 기본 설정이 설정된 경우 동일한 데이터 유형 및 구조를 사용해야 합니다.

플레이어가 기본 설정을 지정하면 게임의 모든 사람이 롤백 이벤트 를 받습니다.

이 함수를 사용하는 경우 조인 / 생성 함수보다 먼저 실행해야 합니다.

읽어주세요: 플레이어 기본 설정

통사론:

rollback_set_player_prefs(preferences);

논쟁 유형 설명
preferences 어느 The preferences to set for the local player.

보고:

N/A

예시:

rollback_set_player_prefs({
캐릭터: obj_customizer.character_selected,
색상: obj_customizer.color_selected
});

위의 코드는 관리되지 않는(로컬) "customizer" 개체에서 값을 읽어 플레이어에 대한 기본 설정을 지정합니다.
