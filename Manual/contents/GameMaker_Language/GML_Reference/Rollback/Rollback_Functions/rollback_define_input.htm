rollback_define_input

이 기능은 모든 플레이어에 대한 사용자 정의 입력을 정의하는 데 사용됩니다. 할당된 입력 상수와 함께 모든 입력을 포함해야 하는 구조체가 필요합니다.

사용자 지정 입력을 할당하는 방법을 알아보려면 사용자 지정 컨트롤 을 참조하십시오.

이 함수를 사용하는 경우 조인 / 생성 함수보다 먼저 실행해야 합니다.

통사론:

rollback_define_input(input_struct);

논쟁 유형 설명
input_struct 구조체 A struct containing input names as variables, and their assigned inputs as values.

보고:

N/A

예시:

rollback_define_input({
    fire: mb_left,
    interact: vk_space,
    left: [ord("A"), vk_left],
    right: [ord("D"), vk_right]
});

위의 코드는 모든 플레이어에 대한 사용자 지정 입력을 정의합니다. 마지막 두 입력은 배열에 나열하여 여러 컨트롤을 사용합니다.

확장된 예는 입력 정의 를 참조하십시오.
