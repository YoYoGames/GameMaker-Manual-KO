rollback_leave_game

이 기능은 현재 멀티플레이어 세션을 떠나 플레이어가 새로운 게임에 다시 참여하거나 생성할 수 있도록 합니다.

방의 호스트가 나가면 해당 방의 소유권이 다른 플레이어에게 이전되어 게임을 계속할 수 있습니다. 모든 플레이어가 떠나야 게임이 종료됩니다.

에러 피하기

플레이어가 멀티플레이어 세션을 떠난 후 GameMaker는 모든 게임 코드를 계속 실행하지만 관리되는 인스턴스를 생성하거나 수정할 수 없는 "사전 시작" 상태로 돌아갑니다.

즉, 오류를 방지하려면 게임을 종료할 때 플레이어 인스턴스를 수동으로 파괴하고 조건에서 rollback_game_running 을 사용하여 다른 게임 로직을 일시 중지해야 합니다.

참조: 롤백 시작 전 상태

트리거된 이벤트

플레이어가 현재 있는 게임에서 나가면 롤백 이벤트 이벤트가 트리거됩니다. 자세한 내용은 롤백 이벤트 를 참조하십시오.

통사론:

rollback_leave_game();

보고:

N/A

예시:

if (keyboard_check_pressed(vk_escape))
{
    rollback_leave_game();
}

위의 코드는 Escape 키를 눌렀을 때 플레이어가 게임을 종료하도록 합니다.
