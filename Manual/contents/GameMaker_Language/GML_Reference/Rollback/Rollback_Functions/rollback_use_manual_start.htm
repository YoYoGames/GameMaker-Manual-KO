롤백_사용_수동_시작

기본적으로 멀티플레이어 게임은 모든 플레이어가 참가하면 자동으로 시작됩니다.

이 함수를 호출하면 자동 시작 동작이 비활성화되므로 게임은 사용자가 수동으로 시작할 때 까지 기다립니다.

이 함수를 사용하는 경우 조인 / 생성 함수보다 먼저 실행해야 합니다.

통사론:

rollback_use_manual_start();

보고:

N/A

예시:

rollback_use_manual_start();

if (!rollback_join_game())
{
    rollback_create_game(2);
}

위의 코드는 수동 시작을 활성화한 다음 새로운 멀티플레이어 게임에 참여하거나 생성합니다.
