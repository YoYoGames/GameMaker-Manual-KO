rollback_define_mock_input

이 기능은 동기화 테스트 중에 지정된 플레이어에 임시 "모의" 입력을 할당하는 데 사용할 수 있습니다. 이것은 플레이어에 대해 정의된 컨트롤 과 동기화 테스트의 임의 입력 을 무시합니다.

사용법에 대한 정보는 모의 입력 을 참조하십시오.

통사론:

rollback_define_mock_input(player_id, input_struct);

논쟁 유형 설명
player_id 진짜 The ID of the player to assign mock input to.
input_struct 구조체 A struct containing input names as variables, and their assigned inputs as values.

보고:

N/A

예시:

rollback_define_input({
    fire: mb_left,
    interact: vk_space,
    left: ord("A"),
    right: ord("D")
});

rollback_define_mock_input(1, {
    fire: vk_control,
    interact: vk_shift,
    left: ord("J"),
    right: ord("L")
});

위의 코드는 먼저 플레이어에 대한 입력을 정의합니다. 그런 다음 동일한 입력 이름을 유지하지만 다른 입력을 할당하여 플레이어 1에 대한 모의 입력을 정의합니다. 이렇게 하면 두 사람이 키보드를 사용하여 로컬에서 게임을 할 수 있습니다.
