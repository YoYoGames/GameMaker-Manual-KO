rollback_use_player_prefs

이 기능은 플레이어 기본 설정을 활성화하고 모든 플레이어에 대한 기본 기본 설정을 지정할 수 있습니다.

기본 기본 설정은 모든 유형이 될 수 있습니다. 기본 기본 설정을 지정하지 않으면 게임을 시작하기 전에 모든 플레이어가 기본 설정 을 지정해야 합니다.

이 함수를 호출하면 자동 시작이 비활성화되므로 게임을 수동으로 시작 해야 합니다. 참조: 수동 및 자동 시작

이 함수를 사용하는 경우 조인 / 생성 함수보다 먼저 실행해야 합니다.

읽어주세요: 플레이어 기본 설정

통사론:

rollback_use_player_prefs([default]);

논쟁 유형 설명
default 어느 OPTIONAL The default preferences to set for all players.

보고:

N/A

예시:

rollback_use_player_prefs({
문자: 0,
색상: c_white
});

위의 코드는 플레이어 기본 설정을 활성화하고 구조체에 기본 기본 설정을 지정합니다.
