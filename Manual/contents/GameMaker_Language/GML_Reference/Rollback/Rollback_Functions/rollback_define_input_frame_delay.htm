rollback_define_input_frame_delay

이 함수는 로컬 플레이어 입력에 대한 고정 프레임 지연을 정의합니다.

기본적으로 게임은 플레이어 간의 대기 시간을 기반으로 하는 적응형 입력 지연을 사용합니다. 이것은 거의 모든 게임에서 작동하지만, 일부 특정 유형의 게임(예: 격투 게임)은 언제든지 변경될 수 있는 적응형 지연 대신 고정 입력 지연을 사용하는 것을 선호할 수 있습니다.

수행 중인 작업을 정확히 알지 못하는 경우 이 값을 변경하지 않는 것이 좋습니다. 멀티플레이어 게임 전과 도중에 언제든지 이 함수를 호출할 수 있습니다.

지정된 지연 값은 프레임 수여야 합니다. -1 을 지정하면 게임이 적응형 지연을 사용하도록 되돌아갑니다. 설정할 수 있는 최대 지연은 10프레임 입니다.

통사론:

rollback_define_input_frame_delay(frames);

논쟁 유형 설명
frames 진짜 The number of frames to set as the fixed input delay, or -1 to reset it to default.

보고:

N/A

예시:

// 'Options' script
global.frame_delay = 5;

// obj_game: Create event
rollback_define_input_frame_delay(global.frame_delay);
var _joined = rollback_join_game();

if (!_joined)
{
    rollback_start_game(2, false);
}

프레임 지연 값을 전역 변수로 정의하고 롤백 게임이 시작되기 전에 rollback_define_input_frame_delay() 를 호출할 때 사용할 수 있습니다.
