롤백_채팅

이 기능은 다른 플레이어에게 문자열 메시지를 보내는 데 사용됩니다.

보낼 문자열을 지정하며 최대 크기 는 128바이트입니다.

두 번째 인수를 지정하지 않으면 보낸 사람을 포함한 모든 플레이어에게 메시지가 전송됩니다.

메시지를 받을 플레이어인 두 번째 인수에 플레이어 ID를 지정할 수 있습니다.

사용하는 방법?

이 함수는 "로컬로" 사용하기 위한 것이므로 롤백 입력 이 아닌 로컬 입력 에서 발생합니다.

하단의 사용 예를 참조하십시오. 롤백 이벤트 이벤트에서 수신된 메시지를 수신합니다.

통사론:

rollback_chat(message, [to]);

논쟁 유형 설명
message 끈 The message to send, as a string.
to 진짜 OPTIONAL The ID of the player to which the message will be sent. Not specifying this argument, or specifying keyboard_string>-1, sends the message to all players.

보고:

N/A

예시:

if (keyboard_check_pressed(vk_enter))
{
    rollback_chat(keyboard_string);
    keyboard_string = "";
}

로컬에서 Enter 키를 누르면 값이 모든 플레이어에게 전송됩니다. 그런 다음 새 메시지를 입력할 수 있도록 해당 문자열이 지워집니다.

이것은 모든 클라이언트(발신자 포함)가 메시지를 수신할 때 롤백 이벤트 이벤트를 트리거합니다.

롤백 이벤트

if (rollback_event_id == rollback_chat_message)
{
    // Get received information
    var _message = rollback_event_param.message;
    var _from = rollback_event_param.from;

    // Find sender's name
    var _name = rollback_get_info(_from).player_name;

    // Print to output log
    show_debug_message(_name + " says: " + _message);
}
