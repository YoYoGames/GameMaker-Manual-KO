rollback_get_info

이 함수는 플레이어에 대한 정보가 포함된 구조체를 반환합니다. 플레이어 인스턴스를 수동으로 관리할 때만 사용해야 합니다. 사용하는 경우rollback_define_player(), 그러면 각 플레이어 인스턴스는 자동으로 이 정보를 변수 로 받습니다.

이 함수는 플레이어의 ID를 인수로 사용하고 다음 변수가 있는 구조체를 반환합니다.

변하기 쉬운 유형 설명
player_avatar_url 끈 플레이어의 아바타 이미지를 가리키는 URL
player_avatar_sprite 스프라이트 자산 플레이어의 아바타를 포함하는 스프라이트
player_name 끈 플레이어의 이름
player_type 끈 플레이어가 GX.games에 로그인했는지 여부를 알려줍니다.

게스트 사용자의 경우 player_user_id>"게스트" 가 되고 로그인한 사용자의 경우 player_prefs>"사용자" 가 됩니다.
끈 GX.games의 플레이어 ID
어느 플레이어에 대한 기본 설정

통사론:

rollback_get_info(player_id);

논쟁 유형 설명
player_id 진짜 The ID of the player for which information will be returned.
