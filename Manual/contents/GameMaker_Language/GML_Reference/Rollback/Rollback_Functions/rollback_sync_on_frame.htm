rollback_sync_on_frame

{
    if (rollback_sync_on_frame())
    {
        obj_game.winning_player = player_id;
    }
}

이 코드는 플레이어 개체의 Step 이벤트에서 실행됩니다.

승리한 플레이어가 없는지, 현재 플레이어가 승리하는 데 필요한 포인트가 있는지 확인합니다.

둘 다 참이면 플레이어가 승리해야 함을 의미합니다. 그러나 그렇게 하기 전에 모든 플레이어가 동기화되었는지 확인하여 상태가 올바른지 확인합니다.

모든 플레이어가 동기화되면 win_player 변수가 설정됩니다. 이로 인해 첫 번째 조건이 false 가 되어 rollback_sync_on_frame() 이 다시 호출되지 않도록 합니다.

이는 함수가 매 프레임마다 계속 실행되는 대신 토글로 사용됨을 의미합니다.
