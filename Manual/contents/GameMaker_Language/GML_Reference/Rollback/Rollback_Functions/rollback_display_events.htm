롤백_디스플레이_이벤트

기본적으로 게임을 실행할 때 게임의 왼쪽 상단 모서리에 디버그 메시지가 표시되어 발생하는 대로 멀티플레이어 이벤트를 표시합니다. false 를 인수로 사용하여 이 함수를 호출하여 해당 메시지를 비활성화하고 true 를 지정하여 다시 활성화할 수 있습니다.

통사론:

rollback_display_events(enabled);

논쟁 유형 설명
enabled 부울 Specifies whether event display should be enabled ( true ) or disabled ( false ). By default, it's enabled.

보고:

N/A

예시:

rollback_display_events(false);

위의 코드는 이벤트 표시를 비활성화했습니다.
