virtual_key_show

장치에 대한 가상 키를 만든 후에는 올바르게 배치되고 제대로 작동하는지 디버그하고 테스트해야 할 수 있습니다. 이를 위해 보고자 하는 가상 키의 인덱스를 사용하여 이 함수를 호출합니다(이전에 다음을 사용하여 생성 및 저장virtual_key_add() ) 화면에 키를 나타내는 색상이 지정된 사각형을 그립니다. 일단 만족스러우면 해당 영역에 자신만의 그래픽을 만들어 배치하는 것이 좋습니다.

참고 그리는 직사각형은 해당 그리기 기능 에 의해 설정된 혼합, 색상 및 알파 옵션의 영향을 받습니다.

통사론

virtual_key_show(index);

논쟁 유형 설명
index 가상 키 ID The index of the virtual key to show.

보고:

N/A

예시:

if global.Debug
{
    virtual_key_show(global.left);
}
else
{
    virtual_key_hide(global.Left);
}

위의 코드는 전역 변수 "global.Debug"를 확인하고 true로 테스트되면 변수 "global.Left"에 색인된 가상 키가 화면에 그려지고 false이면 키가 숨겨집니다.
