virtual_key_hide

장치에 대한 가상 키를 만든 후에는 해당 키가 올바르게 배치되고 제대로 작동하는지 디버그하고 테스트해야 할 수 있으며 이를 위해 함수를 호출합니다.virtual_key_show(). 그러나 이 보기를 켜고 끄도록 전환해야 할 수도 있으므로 숨기려는 가상 키의 인덱스를 사용하여 이 기능으로 가상 키를 숨길 수도 있습니다.virtual_key_add() ). 일단 만족스러우면 해당 영역에 자신만의 그래픽을 만들어 배치하는 것이 좋습니다.

통사론

virtual_key_hide(index);

논쟁 유형 설명
index 가상 키 ID The index of the virtual key to hide.

보고:

N/A

예시:

if global.Debug
{
    virtual_key_show(global.Left);
}
else
{
    virtual_key_hide(global.Left);
}

위의 코드는 전역 변수 "global.Debug"를 확인하고 true로 테스트되면 변수 "global.Left"에 색인된 가상 키가 화면에 그려지고 false이면 키가 숨겨집니다.
