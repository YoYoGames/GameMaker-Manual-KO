keyboard_virtual_hide

이 기능을 사용하여 게임을 실행하는 장치에서 가상 키보드를 숨길 수 있습니다. 이 함수를 호출하면 시스템 비동기 이벤트 가 생성되며, 여기서 async_load DS 맵은 다음 키/값 쌍으로 채워집니다.

  • " event_type " - 가상 키보드 기능에 의해 트리거될 때 " 가상 키보드 상태 "가 됩니다.
  • " screen_height " - 가상 키보드의 높이(픽셀 단위). 키보드가 보이지 않으면 0이 됩니다.
  • " keyboard_status " - 키보드의 현재 상태로, 다음 문자열 중 하나로 반환됩니다.
    • "숨김"
    • "숨겨진"
    • "전시"
    • "보이는"

통사론

keyboard_virtual_hide();

보고:

예시:

if input == true
{
    input = false;
    keyboard_virtual_hide();
}

주어진 변수가 다음으로 설정되지 않은 경우 위의 코드는 OS 가상 키보드를 숨깁니다. false.
