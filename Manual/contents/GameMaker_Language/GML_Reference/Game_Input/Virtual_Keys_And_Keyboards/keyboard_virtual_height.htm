keyboard_virtual_height

이 함수는 디스플레이 크기에 따라 가상 키보드의 현재 높이를 픽셀 단위로 반환합니다. 키보드가 보이지 않으면 0이 반환됩니다.

통사론

keyboard_virtual_height();

보고:

예시:

if keyboard_virtual_status() == true
{
    key_h = keyboard_virtual_height();
}

위의 코드는 OS 가상 키보드의 상태를 확인하고, 보이면 키보드 높이로 변수를 설정합니다.
