keyboard_virtual_status

이 기능은 게임을 실행하는 장치에서 가상 키보드의 상태를 가져오는 데 사용할 수 있습니다. 함수가 반환됩니다 true OS 가상 키보드가 보이거나 표시되는 경우 또는 false 숨겨져 있는 경우/숨겨져 있는 경우.

통사론

keyboard_virtual_status();

보고:

예시:

var _status = keyboard_virtual_status();
if _status == false
{
    keyboard_virtual_show(kbv_type_numbers, kbv_returnkey_default, kbv_autocapitalize_none, false);
}

위의 코드는 현재 상태가 다음과 같은 경우 OS 가상 키보드를 표시합니다. false.
