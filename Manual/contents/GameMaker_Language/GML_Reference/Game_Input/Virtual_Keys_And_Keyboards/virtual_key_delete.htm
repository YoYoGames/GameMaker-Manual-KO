virtual_key_delete

게임에 메뉴, 게임 플레이 및 기타 사항을 제어하기 위한 다른 방이나 인스턴스가 있는 경우 어느 시점에서 가상 키의 위치와 키 맵을 변경해야 할 가능성이 높습니다. 이를 위해 이 기능을 사용하여 새 항목을 만들기 전에 이전 항목을 삭제할 수 있습니다(필요한 경우). 이 기능을 사용하려면 삭제할 가상 키의 ID를 제공해야 합니다.virtual_key_add().

방이 변경되면 모든 가상 키가 방에서 자동으로 제거되므로 이 기능은 방 자체가 완료되기 전에 수동으로 키를 제거하려는 경우 에만 필요합니다.

통사론

virtual_key_delete(index);

논쟁 유형 설명
index 가상 키 ID The index of the virtual key to delete.

보고:

N/A

예시:

virtual_key_delete(global.Left);

위의 코드는 전역 변수 "global.Left"에 인덱싱된 가상 키를 삭제합니다.
