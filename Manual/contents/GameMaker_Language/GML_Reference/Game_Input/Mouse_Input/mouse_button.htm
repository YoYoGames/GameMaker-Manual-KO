mouse_button

이 읽기 전용 변수는 현재 누르고 있는 마우스 버튼을 반환하고(현재 이 단계에서와 같이) 다음을 제외한 모든 특수 마우스 상수 를 반환할 수 있습니다. mb_any.

통사론:

mouse_button;

보고:

Mouse Button Constant (except mb_any)

예시:

if mouse_button = mb_left
{
    x -= 1;
}

왼쪽 마우스 버튼이 눌려져 있으면 현재 인스턴스를 왼쪽으로 이동합니다.
