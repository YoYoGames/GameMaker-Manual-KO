마우스 입력

마우스 입력은 모든 플랫폼에서 허용되며(모바일 장치에서는 단일 화면 터치로 허용됩니다. 멀티 터치를 사용해야 하는 경우 장치별 기능 을 사용해야 함) 버튼을 지정하는 데 사용되는 몇 가지 상수가 있습니다. 눌리는 중. 이러한 상수는 다음 표에 나와 있습니다.

표준 마우스 버튼 컨트롤에는 다음과 같은 기능이 있습니다.

  • mouse_button
  • mouse_check_button
  • mouse_check_button_pressed
  • mouse_check_button_released
  • mouse_clear
  • mouse_lastbutton
  • mouse_wheel_up
  • mouse_wheel_down
  • mouse_x
  • mouse_y

데스크톱 대상에서 마우스를 사용하는 것과 관련된 일련의 창 기능도 있습니다.

  • window_mouse_get_x
  • window_mouse_get_y
  • window_mouse_set
  • window_view_mouse_get_x
  • window_view_mouse_get_y
  • window_views_mouse_get_x
  • window_views_mouse_get_y

뒤로: 게임 입력
