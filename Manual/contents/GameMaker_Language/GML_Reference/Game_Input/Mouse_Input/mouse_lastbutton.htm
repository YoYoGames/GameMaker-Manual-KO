mouse_lastbutton

이 변수는 마지막으로 누른 마우스 버튼을 반환하고 다음을 제외한 모든 특수 마우스 상수 를 반환할 수 있습니다. mb_any (이 변수를 상수 중 하나로 설정할 수도 있습니다.)

통사론:

mouse_lastbutton;

보고:

Mouse Button Constant (except mb_any)

예시:

if mouse_lastbutton = mb_left
{
    x -= 1;
}

마지막으로 누른 버튼이 마우스 왼쪽 버튼인 경우 현재 인스턴스를 왼쪽으로 이동합니다.
