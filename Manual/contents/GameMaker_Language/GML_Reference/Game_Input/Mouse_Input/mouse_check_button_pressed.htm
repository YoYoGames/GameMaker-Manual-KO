mouse_check_button_pressed

이 함수는 반환됩니다 true 확인 중인 마우스 버튼이 눌렸거나 false 그렇지 않은 경우. 이 기능은 마우스 버튼을 처음 눌렀을 때 한 번만 트리거되며 다시 트리거하려면 버튼에서 손을 떼고 다시 눌러야 합니다. 단계 기간 동안, 그리고 마우스 이벤트가 있거나 동일한 기능을 사용하는 모든 인스턴스에 대해 트리거된 것으로 간주됩니다.

다음 상수 중 하나에서 확인할 마우스 버튼을 제공합니다.

통사론:

mouse_check_button_pressed(numb);

논쟁 유형 설명
numb 마우스 버튼 상수 Which mouse button constant to check for.

보고:

Boolean

예시:

if mouse_check_button_pressed(mb_left)
{
    score += 50;
}

위의 코드는 왼쪽 마우스 버튼이 눌렸는지 확인하고 눌렀다면 점수에 50을 추가합니다.

뒤로: 마우스 입력
