mouse_check_button

이 함수는 반환됩니다 true 확인 중인 마우스 버튼이 눌려져 있거나 false 그렇지 않은 경우. 다음 상수 중 하나에서 확인할 마우스 버튼을 제공합니다.

통사론:

mouse_check_button(numb);

논쟁 유형 설명
numb 마우스 버튼 상수 Which mouse button constant to check for.

보고:

Boolean

예시:

if mouse_check_button(mb_left)
{
    instance_create_layer(mouse_x, mouse_y, "Effects", obj_Star);
}

위의 코드는 왼쪽 마우스 버튼을 확인하고 누르고 있는 모든 단계는 "obj_Star"에 인덱싱된 개체의 인스턴스를 생성합니다.

뒤로: 마우스 입력
다음: mouse_check_button_pressed
