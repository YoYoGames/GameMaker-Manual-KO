mouse_clear

이 함수는 주어진 마우스 버튼의 현재 상태를 지울 것입니다. 이는 보류 중인 수표가 반환되지 않음을 의미합니다. true 플레이어가 버튼을 놓았다가 다시 누를 때까지(그러나 마우스 버튼을 누르고 있는 동안 지우기가 완료되면 해제 상태가 계속 감지됩니다.)

이 페이지에 나열된 마우스 버튼 상수 중 하나를 제공할 수 있습니다.

통사론:

mouse_clear(button);

논쟁 유형 설명
button 마우스 버튼 상수 Which mouse button constant to check for.

보고:

Boolean

예시:

mouse_clear(mb_any);

위의 코드는 모든 마우스 버튼의 다운 상태를 지웁니다.

뒤로: 마우스 입력
