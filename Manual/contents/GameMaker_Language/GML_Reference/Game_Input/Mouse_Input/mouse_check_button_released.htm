mouse_check_button_released

이 함수는 반환됩니다 true 확인 중인 마우스 버튼이 해제된 경우 또는 false 그렇지 않은 경우. 이 기능은 마우스 버튼을 놓았을 때 한 번만 트리거되며 다시 트리거하려면 버튼을 다시 눌렀다 떼야 합니다. 다음 상수 중 하나에서 확인할 마우스 버튼을 제공합니다.

통사론:

mouse_check_button_released(numb);

논쟁 유형 설명
numb 마우스 버튼 상수 Which mouse button constant to check for.

보고:

Boolean

예시:

if mouse_check_button_released(mb_right)
{
    speed = point_distance(x, y, mouse_x, mouse_y) / 10;
}

위의 코드는 마우스 오른쪽 버튼이 해제되었는지 확인하고 해제된 경우 인스턴스의 속도를 현재 x/y 위치와 마우스 x/y 위치 사이 거리의 10분의 1로 설정합니다.

뒤로: 마우스 입력
다음: mouse_clear
