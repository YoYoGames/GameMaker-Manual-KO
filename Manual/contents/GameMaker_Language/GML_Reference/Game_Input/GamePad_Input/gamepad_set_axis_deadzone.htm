gamepad_set_axis_deadzone

이 기능은 조이스틱 축의 "데드존"을 설정하는 데 사용할 수 있습니다. 설정할 디바이스 슬롯을 지정하고 0~1 사이의 값을 설정하고 입력량이 주어진 값보다 작을 경우 조이스틱 축은 0으로 간주 됩니다. 지정된 장치 슬롯에 연결된 모든 조이스틱의 축.

이 기능을 사용한다고 해서 축 값이 데드존 값에서 시작되는 것은 아닙니다.gamepad_axis_value() 항상 -1과 0 또는 0과 1 사이의 정규화된 값을 반환합니다. 예를 들어 데드존을 0.2로 설정하면 스틱을 오른쪽으로 밀면 원시 축 값이 0.2를 초과할 때만 0 - 1의 값을 반환하기 시작한다는 의미입니다. 따라서 원시 값이 0.2이면 반환 값은 0이 되고, 원시 값이 0.5이면 반환 값은 0.375가 되고, 원시 값이 0.9이면 반환 값은 0.875가 됩니다.

통사론:

gamepad_set_axis_deadzone(device, deadzone);

논쟁 유형 설명
device Which gamepad device "slot" to check.
deadzone The dead zone value from 0 to 1.

보고:

예시:

if gamepad_is_connected(0) gamepad_set_axis_deadzone(0, 0.05);
if gamepad_is_connected(1) gamepad_set_axis_deadzone(1, 0.05);
if gamepad_is_connected(2) gamepad_set_axis_deadzone(2, 0.05);
if gamepad_is_connected(3) gamepad_set_axis_deadzone(3, 0.05);

위의 코드는 4개의 연결된 장치 중 하나에서 조이스틱 축의 사각 지대를 0.05로 설정합니다.
