gamepad_axis_count

이 함수는 검사 중인 장치의 "축" 컨트롤 수를 반환합니다. 이러한 컨트롤은 대부분의 컨트롤러에서 아날로그 방향 "썸스틱"입니다.

통사론:

gamepad_axis_count(device);

논쟁 유형 설명
device Which gamepad device "slot" to check.

보고:

예시:

axis = gamepad_axis_count(0))

위의 코드는 "axis" 변수에 "slot" 0 장치에 연결된 게임패드에 사용할 수 있는 축 수를 저장합니다.
