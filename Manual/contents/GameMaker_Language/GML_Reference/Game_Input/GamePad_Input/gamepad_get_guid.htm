gamepad_get_guid

이 기능을 사용하면 주어진 슬롯 인덱스에 연결된 게임패드의 GUID를 검색할 수 있습니다. 확인할 게임패드 슬롯(0 - 11)을 제공하면 함수는 다음 문자열 중 하나를 반환합니다.

  • 사용 가능한 GUID가 없거나 게임패드가 연결되어 있지 않은 경우 " 없음 "
  • 게임패드 슬롯 인덱스가 범위를 벗어난 경우 " device index out of range "
  • 슬롯에 게임패드가 할당되어 있고 해당 GUID를 가져올 수 있는 경우 "050000005e040000fd020000ffff3f00"과 같은 GUID 문자열

이 기능은 일반적으로 다음과 함께 사용됩니다.gamepad_get_description() Windows, Android 및 macOS에서 컨트롤러를 다시 매핑하는 데 필요한 SDL 문자열의 일부를 생성합니다.

플랫폼별 참고 사항
플랫폼 메모
오페라 GX Opera GX 대상 플랫폼에서 이 함수는 DirectInput이 아닌 장치에 대해 항상 "none" 을 반환합니다.

통사론:

gamepad_get_guid(index);

논쟁 유형 설명
index Which gamepad "slot" index to check

보고:

예시:

var _guid = gamepad_get_guid(global.PadIndex);
var _desc = gamepad_get_description(global.PadIndex);
global.GamepadID = _guid + "," + _desc;

위의 코드는 GUID 및 이름 설명 문자열을 가져온 다음 이를 연결하고 나중에 참조할 수 있도록 최종 문자열을 전역 변수에 저장합니다.
