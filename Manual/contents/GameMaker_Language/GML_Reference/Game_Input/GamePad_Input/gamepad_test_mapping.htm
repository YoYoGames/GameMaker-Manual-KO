gamepad_test_mapping

이 기능을 사용하여 허용하는 대상에 게임패드 매핑을 설정할 수 있습니다. 설정할 게임패드의 "슬롯" 인덱스를 제공한 다음 다음 필드와 함께 SDL 형식을 사용하여 생성되어야 하는 맵 문자열을 제공합니다.

<guid>,<description>,platform:<platform-name>,<bindings>

이러한 각 필드에 포함되어야 하는 세부 정보는 다음과 같습니다.

  • guid - 장치 유형을 고유하게 식별하는 숫자 및 문자 문자열(이는 다음을 사용하여 검색할 수 있습니다.gamepad_get_guid() 함수)
  • description - 장치에 대한 ASCII 설명(이는 다음을 사용하여 검색할 수 있습니다.gamepad_get_description() 함수)
  • platform-name - "platform:android", "platform:windows", "linux" 또는 "mac" 이 됩니다(이 함수를 호출할 때 문자열에 자동으로 추가 되므로 제공할 필요가 없습니다. 자신의 매핑 문자열을 구성할 때)
  • bindings - 실제 입력을 특정 GML 상수에 바인딩하기 위해 ','로 구분된 일련의 항목

쉼표로 구분된 항목 목록은 각 항목에 대한 특정 형식을 따릅니다.

<abstract-gp-name>:<value-binding>

여기 <value-binding> (즉: 직접 게임 패드 입력)은 다음 중 하나일 수 있습니다.

  • a<N> - 이것은 게임패드의 축을 나타내며, 여기서 <N>은 0에서 ~까지의 축 중 하나입니다.gamepad_axis_count() -1
  • b<N> - 이것은 게임패드의 버튼을 나타내며, 여기서 <N>은 0부터 버튼 중 하나입니다.gamepad_button_count() -1
  • h<N>.<mask> - 이것은 게임패드의 모자를 나타내며, 여기서 <N>은 0에서 모자를 나타냅니다.gamepad_hat_count() -1, 그리고 <mask> 1, 2, 4 또는 8이 됩니다.

각 입력 값은 다음과 같이 표현되는 SDL 이름에 바인딩되어야 합니다. <>abstract-gp-name>, 차례로 GML 상수에 바인딩됩니다. 아래 표는 SDL 이름과 GameMaker 상수 간의 동등성을 보여줍니다.

매핑 문자열을 구성하면 다음과 같아야 합니다.

"050000005e040000fd020000ffff3f00,Xbox Wireless Controller,a:b0,b:b1,start:b4,dpdown:h0.4,dpleft:h0.8,dpright:h0.2,dpup:h0.1,guide:b6,leftshoulder:b9,leftstick:b7,lefttrigger:a4,leftx:a0,lefty:a1,rightshoulder:b10,rightstick:b8,righttrigger:a5,rightx:a2,righty:a3,x:b2,y:b3,platform:android"

통사론:

gamepad_test_mapping(index, mapping_string);

논쟁 유형 설명
index 진짜 Which gamepad index "slot" to set.
mapping_string 끈 The map string to use (see the description for more information).

예시:

var mapping = gamepad_get_guid(global.padIndex) + "," + gamepad_get_description(global.padIndex);
var len = array_length(global.PadInstances);
for (i = 0; i < len; i += 2)
{
    var left = global.PadInstances[i];
    var right = global.PadInstances[i+1];
    mapping += "," + left.sdlLabel + ":" + right.binding;
}
gamepad_test_mapping(global.padIndex, mapping);

위의 코드는 여러 인스턴스를 반복하고 여기에 포함된 다른 변수의 값을 사용하여 매핑 문자열을 만든 다음 지정된 슬롯 인덱스의 게임패드에서 사용하도록 설정됩니다.

뒤로: 게임패드 입력
