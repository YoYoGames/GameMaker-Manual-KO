gamepad_get_device_count

이 기능은 연결된 게임 패드 수 또는 게임 패드에 연결할 수 있는 "슬롯" 수를 제공합니다.

실제 반환 값은 플랫폼과 해당 플랫폼의 내부 구성에 따라 다르므로 이 함수는 함수와 함께 사용해야 합니다.gamepad_is_connected() 언제든지 연결된 패드의 정확한 수를 확인합니다.

통사론:

gamepad_get_device_count();

보고:

Real

예시:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    if (gamepad_is_connected(i))
    {
        global.gp[i] = true;
    }
    else
    {
        global.gp[i] = false;
    }
}

위의 코드는 사용 가능한 게임 패드(또는 게임 패드 슬롯)를 반복한 다음 연결된 게임 패드가 있는지 각각 확인합니다. 반환된 값은 다음으로 전역 배열을 설정하는 데 사용됩니다. true 또는 false 향후 점검에 사용하기 위해.
