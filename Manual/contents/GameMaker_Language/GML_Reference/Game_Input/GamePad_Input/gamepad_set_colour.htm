gamepad_set_colour

이 기능은 PlayStation 컨트롤러 내의 LED 색상을 설정하는 데 사용할 수 있습니다. 설정할 장치 슬롯을 지정한 다음 특정 색상 함수 또는 $FFFFFFF 와 같은 HEX 값을 사용하여 생성된 색상 값 또는 색상 상수 중 하나일 수 있는 색상을 지정합니다.

통사론:

gamepad_set_colour(device, colour);

논쟁 유형 설명
device Which gamepad device "slot" to set.
colour The colour to use.

보고:

예시:

if health < 10
{
    gamepad_set_colour(0, c_red);
}

위의 코드는 다음과 같은 경우 PlayStation 컨트롤러 LED를 빨간색으로 설정합니다. health 변수가 10 아래로 떨어집니다.
