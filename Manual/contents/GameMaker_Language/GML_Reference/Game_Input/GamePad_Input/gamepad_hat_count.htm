gamepad_hat_count

이 함수는 주어진 장치 "슬롯"에 연결된 게임패드에 사용할 수 있는 총 모자 수를 반환합니다. 모자는 일반적으로 위/아래/왼쪽/오른쪽 버튼을 참조하며 Windows 대상에서 모자는 DirectInput 컨트롤러에서만 사용할 수 있습니다(따라서 슬롯 4부터 위쪽).

통사론:

gamepad_hat_count(device);

논쟁 유형 설명
device Which gamepad device "slot" to check.

보고:

예시:

h_num = gamepad_hat_count(4);

위의 코드는 장치 "슬롯" 4에 연결된 게임패드에서 사용 가능한 모자 수를 반환하고 변수 "h_num"에 값을 저장합니다.

뒤로: 게임패드 입력
