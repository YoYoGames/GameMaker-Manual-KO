gamepad_set_vibration

이 기능을 사용하면 0(진동 없음)에서 1(완전한 진동) 사이의 값을 사용하여 게임패드 모터의 진동을 설정할 수 있습니다. 이 기능에는 시간 제한이 없으므로 지정된 시간이 지난 후 진동을 끄기 위해 변수나 알람을 사용해야 합니다(모터를 0으로 설정). 그렇지 않으면 게임패드가 무기한 진동을 계속할 것입니다.

참고: 이 기능은 현재 표준 Windows, PS4 및 Xbox One 대상 모듈에서만 사용할 수 있습니다.

통사론:

gamepad_set_vibration(device, left_motor, right_motor);

논쟁 유형 설명
device Which gamepad device "slot" to check.
left_motor The amount of vibration from the left motor from 0 to 1.
right_motor The amount of vibration from the right motor from 0 to 1.

보고:

예시:

if lives = 0
{
    gamepad_set_vibration(0, 1, 1);
    alarm[0] = room_speed / 2;
}

위의 코드는 (예를 들어) 충돌 이벤트에 사용되어 "슬롯" 0에 연결된 게임패드를 0.5초 동안 진동시키고 해당 시간이 지나면 다시 꺼지도록 설정된 알람과 함께 사용됩니다.
