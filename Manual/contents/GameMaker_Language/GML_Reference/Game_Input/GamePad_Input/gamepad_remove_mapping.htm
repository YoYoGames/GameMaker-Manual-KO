gamepad_remove_mapping

이 함수는 주어진 게임패드 슬롯 인덱스에서 현재 장치 매핑을 제거하는 데 사용할 수 있습니다. 호출되면 슬롯 인덱스는 다음을 사용하여 다시 매핑해야 합니다.gamepad_test_mapping() 게임패드 상수를 사용하여 입력을 올바르게 감지할 수 있도록 하려는 경우 함수(직접 입력은 항상 다음을 사용하여 검색할 수 있습니다. gamepad_axis/button/hat_count() 그리고 gamepad_axis/button/hat_value() 함께 기능).

통사론:

gamepad_remove_mapping(index);

논쟁 유형 설명
index Which gamepad index "slot" to remove the mapping from.

보고:

예시:

if remap == true
{
    gamepad_remove_mapping(global.PadIndex);
}

위의 코드는 변수 값에 따라 주어진 게임패드 인덱스 슬롯에서 매핑을 제거합니다.
