gamepad_axis_value

이 함수를 사용하여 주어진 게임패드 슬롯에서 다른 축의 값을 가져올 수 있습니다. 축 인덱스 상수( 여기 에 나열됨) 또는 다음 사이의 정수 값과 함께 확인할 게임패드 슬롯 인덱스를 제공합니다. 0 그리고 gamepad_axis_count()-1. 반환된 값은 다음 사이입니다. -1 그리고 1 사용 가능한 가로 및 세로 축 각각에 대해

반환된 값은 축 데드존에 대해 설정된 값을 고려하도록 정규화됩니다(함수 사용 gamepad_set_axis_deadzone() ), 그래서 항상 -1과 0, 그리고 0과 1 사이에 있을 것입니다. 예를 들어 데드존을 0.2로 설정하면 스틱을 오른쪽으로 밀면 원시 축 값이 끝났을 때 0 - 1의 값만 반환하기 시작한다는 의미입니다. 0.2 - 예를 들어 원시 값이 0.2이면 반환 값은 0이 되고, 원시 값이 0.5이면 반환 값은 0.375가 되며, 원시 값이 0.9이면 반환 값은 0.875가 됩니다..

통사론:

gamepad_axis_value(device, axisIndex);

논쟁 유형 설명
device 진짜 Which gamepad device "slot" to check.
axisIndex 게임패드 축 상수 The axis index to check (see the constants list ).

보고:

Real

예시:

var haxis = gamepad_axis_value(0, gp_axislh);
var vaxis = gamepad_axis_value(0, gp_axislv);
direction = point_direction(0, 0, haxis, vaxis);
speed = point_distance(0 ,0, haxis, vaxis) * 5;

위의 코드는 장치 "슬롯" 0에 연결된 게임패드의 왼쪽 아날로그 컨트롤러의 반환된 값을 기반으로 인스턴스의 속도와 방향을 설정합니다.
