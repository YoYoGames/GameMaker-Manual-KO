gamepad_hat_value

이 기능을 사용하면 게임패드 "모자"의 현재 값을 얻을 수 있습니다. 각 모자 값은 다음과 같은 다른 방향에 대한 비트 마스크입니다.

  • 위로 = 1
  • 오른쪽 = 2
  • 아래로 = 4
  • 왼쪽 = 8

이것들은 결합될 수 있지만(예를 들어, 3의 모자 인덱스를 제공하는 것은 확인 및 오른쪽일 것입니다. 또는 | 연산자를 사용하여 다양한 값을 결합할 수 있음) 한 번에 최대 2비트만 누를 수 있습니다. 이 함수는 0과 1 사이의 실제 값을 반환합니다. 여기서 0은 누르지 않고 1은 완전히 누릅니다(게임패드가 모자에 대한 아날로그 입력을 지원하는지 여부에 따라 그 사이에 값이 있을 수 있음).

통사론:

gamepad_hat_value(device, hatindex);

논쟁 유형 설명
device 진짜 Which gamepad device "slot" to check.
hatindex 진짜 Which gamepad hat (or hats) to check for.

보고:

Real

예시:

var _dir = gamepad_hat_value(global.PadIndex, 0);
switch (_dir)
{
    case 1: y -= 3; break;
    case 2: x += 3; break;
    case 3: y -= 3; x += 3; break;
    case 4: x -= 3; break;
    case 6: y += 3; x += 3; break;
    case 8: y += 3; break;
    case 9: y -= 3; x -= 3; break;
    case 12: y += 3; x -= 3; break;
}

위의 코드는 모자 "0"의 상태를 지역 변수에 저장한 다음 반환 값이 무엇인지 확인하고 그에 따라 작동합니다.
