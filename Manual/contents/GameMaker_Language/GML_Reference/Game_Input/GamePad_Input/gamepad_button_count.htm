gamepad_button_count

이 함수는 주어진 장치 "슬롯"에 연결된 게임패드에 사용할 수 있는 총 버튼 수를 반환합니다.

통사론:

gamepad_button_count(device);

논쟁 유형 설명
device Which gamepad device "slot" to check.

보고:

예시:

b_num = gamepad_button_count(0);

위의 코드는 장치 "슬롯" 0에 연결된 게임패드에서 사용 가능한 버튼 수를 반환하고 변수 "b_num"에 값을 저장합니다.

뒤로: 게임패드 입력
