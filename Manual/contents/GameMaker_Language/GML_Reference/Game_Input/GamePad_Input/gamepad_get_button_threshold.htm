gamepad_get_button_threshold

이 기능은 주어진 장치에 대한 아날로그 버튼의 현재 임계값 설정을 감지하는 데 사용할 수 있습니다. 모든 아날로그 버튼의 기본 임계값은 0.5이며 범위는 0에서 1입니다. 임계값은 버튼이 디지털 버튼으로 작동해야 하는 게임에서 버튼이 "눌린" 것으로 간주되는 지점을 정의합니다.

통사론:

gamepad_get_button_threshold(device);

논쟁 유형 설명
device Which gamepad device "slot" to check.

보고:

예시:

if gamepad_get_button_threshold(0) != 0.5
{
    gamepad_set_button_threshold(0, 0.5);
}

위의 코드는 "slot" 0 장치에 연결된 게임패드의 아날로그 버튼 임계값을 확인하고 기본값인 0.5가 아닌 경우 해당 값으로 설정합니다.
