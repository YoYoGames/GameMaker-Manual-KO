gamepad_button_check_pressed

이 함수는 반환됩니다 true 또는 false 주어진 게임 패드 버튼이 눌린 것으로 감지되는지 여부에 따라 다릅니다. 이 기능은 버튼을 처음 눌렀을 때 한 번만 트리거됩니다. 다시 작동시키려면 먼저 버튼을 놓았다가 다시 눌러야 합니다. 버튼을 계속 누르고 있는지 확인해야 하는 경우 해당 기능을 사용해야 합니다.gamepad_button_check().

아날로그 버튼을 확인하는 경우 확인은 임계값 설정 을 고려하고 true 원시 입력 값이 주어진 임계값을 초과할 때(함수를 사용하여 이 원시 값을 얻을 수 있습니다. gamepad_button_value() ).

통사론:

gamepad_button_check_pressed(device, button);

논쟁 유형 설명
device 진짜 Which gamepad device "slot" to check.
button 게임패드 버튼 상수 Which gamepad button constant to check for.

보고:

Boolean

예시:

if gamepad_button_check_pressed(0, gp_start)
{
    audio_play_sound(snd_Start, 0, false);
    room_goto(rm_Level_1);
}

위의 코드는 장치 "슬롯" 0에 연결된 게임패드의 "시작" 버튼이 눌렸는지 여부를 감지하고 눌렸다면 공간을 변경합니다.

뒤로: 게임패드 입력
다음: gamepad_button_check_released
