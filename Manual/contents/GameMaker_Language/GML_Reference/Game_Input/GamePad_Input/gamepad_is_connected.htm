gamepad_is_connected

이 함수는 게임패드가 주어진 "슬롯"에 연결되었는지 여부를 반환합니다(반환 true ) 여부(반환 false ). 일반적으로 이 기능을 다음과 함께 사용합니다.gamepad_get_device_count() 사용 가능한 게임 패드 및/또는 게임 패드 "슬롯"의 정확한 수를 가져오는 기능입니다. 게임패드를 연결하는 사용자와 연결된 것으로 GameMaker 가 감지하는 사이에 약간의 지연이 있을 수 있습니다(특히 블루투스 연결 컨트롤러를 처리하는 경우).

참고: 일부 플랫폼(특히 콘솔)에서 이 기능은 게임패드가 연결/선택된 직후 false를 반환할 수 있으며 게임패드를 올바르게 감지하기 위해 몇 프레임(단계) 후에 알람에서 확인해야 할 수 있습니다.

통사론:

gamepad_is_connected(numb);

논쟁 유형 설명
device Which gamepad "slot" to check.

보고:

예시:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    if gamepad_is_connected(i) global.gp[i] = true else global.gp[i] = false;
}

위의 코드는 사용 가능한 게임 패드(또는 게임 패드 슬롯)를 반복한 다음 연결된 게임 패드가 있는지 각각 확인합니다. 반환된 값은 다음으로 전역 배열을 설정하는 데 사용됩니다. true 또는 false 향후 점검에 사용할 수 있습니다.
