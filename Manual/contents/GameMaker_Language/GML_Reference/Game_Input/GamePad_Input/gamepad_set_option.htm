gamepad_set_option

이 기능은 사용 가능한 게임패드 옵션을 설정하는 데 사용할 수 있습니다. 옵션을 설정하기 위한 게임패드 "슬롯"과 설정할 옵션 문자열 및 사용할 값을 제공합니다. 사용 가능한 옵션 문자열은 아래와 같이 프로젝트가 실행되는 플랫폼에 따라 다릅니다.

옵션 플랫폼 설명 값
"allow_rotation" tvOS Siri Remote의 회전 잠금 여부를 설정합니다(예: 세로로만 사용할 수 있음). 잠금 해제 시(값 true ) 컨트롤을 회전하면 입력이 변경되어 왼쪽으로 90º 회전하면 터치 패드의 "오른쪽"이 "위쪽"으로 돌아가게 됩니다. 장치. 잠긴 경우(값 false ) 그러면 입력은 컨트롤러의 위치와 관련하여 변경되지 않고 항상 컨트롤러가 수직 위치에 있는 것처럼 처리됩니다. 부울
"dpad_absolute" tvOS Siri Remote 터치패드가 절대 입력을 허용할지 아니면 상대 입력을 허용할지 설정합니다. 절대 모드에서(값 true ) 터치 패드의 중심은 (0,0) 위치로 간주됩니다. 즉, 아날로그 스틱이 정지된 위치 또는 dpad의 중심이 있어야 하는 위치 - 이 위치를 중심으로 이동하면 입력이 생성됩니다. 상대 값으로 설정하면 false 그러면 (0,0) 위치는 패드를 처음 터치하는 위치로 간주되고 모든 입력은 패드의 중심이 아니라 해당 위치에서 생성됩니다. 부울

통사론:

gamepad_set_option(device, option_name, value);

논쟁 유형 설명
device 진짜 Which gamepad device "slot" to set.
option_name 끈 The name of the option to set (a string, see the table above)
value 실수, 부울 또는 문자열 The value to set the option to (can be boolean, real or string - see the table above)

예시:

for(var i = 0; i < 12; ++i;)
{
    if gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote"
    {
        if gamepad_get_option(i, "allow_rotation") == false
        {
            gamepad_set_option(i, "allow_rotation", true);
        }
    }
}

위의 코드는 모든 게임패드 슬롯을 반복하고 tvOS 플랫폼에서 "Siri Remote"를 확인합니다. 하나가 감지되면 입력 회전을 허용하도록 리모컨을 설정합니다.
