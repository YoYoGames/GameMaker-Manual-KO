gamepad_button_check

이 함수는 반환됩니다 true 또는 false 주어진 게임패드 버튼이 누르고 있는 것으로 감지되는지 여부에 따라 다릅니다. 아날로그 버튼을 확인하는 경우 확인은 임계값 설정 을 고려하고 true 원시 입력 값이 주어진 임계값을 초과하는 동안(함수를 사용하여 이 원시 값을 얻을 수 있습니다. gamepad_button_value() ).

통사론:

gamepad_button_check(device, button);

논쟁 유형 설명
device 진짜 Which gamepad device "slot" to check.
button 게임패드 버튼 상수 Which gamepad button constant to check for.

보고:

Boolean

예시:

if gamepad_button_check(0, gp_face1)
{
    if canshoot = true
    {
        audio_play_sound(snd_Shoot, 0, false);
        instance_create_layer(x, y, "Bullets", obj_Bullet)
        canshoot = false;
        alarm[0] = room_speed / 2;
    }
}

위의 코드는 윗면의 아래쪽 버튼(ps3 컨트롤러의 "X")이 눌려 있는지 감지하고, 누르고 있으면 "총알" 인스턴스를 쏘고 알람을 설정합니다.

뒤로: 게임패드 입력
