gamepad_button_check_released

이 함수는 반환됩니다 true 또는 false 주어진 게임 패드 버튼이 해제되었는지 여부에 따라 다릅니다. 이 기능은 버튼에서 손을 뗀 순간에 한 번만 트리거됩니다. 다시 트리거하려면 먼저 버튼을 눌렀다가 다시 놓아야 합니다. 아날로그 버튼을 확인하는 경우 확인은 임계값 설정 을 고려하고 true 원시 입력 값이 주어진 임계값 아래로 떨어질 때(함수를 사용하여 이 원시 값을 얻을 수 있습니다. gamepad_button_value() ).

통사론:

gamepad_button_check_released(device, button);

논쟁 유형 설명
device 진짜 Which gamepad device "slot" to check.
button 게임패드 버튼 상수 Which gamepad button constant to check for.

보고:

Boolean

예시:

if gamepad_button_check_released(0, gp_select)
{
    audio_play_sound(snd_Button, 0, false);
    global.Pause = !global.Pause;
}

위의 코드는 장치 "슬롯" 0에 연결된 게임패드의 "선택" 버튼이 눌렸는지 여부를 감지하고 전역 "일시 중지" 변수를 토글합니다.
