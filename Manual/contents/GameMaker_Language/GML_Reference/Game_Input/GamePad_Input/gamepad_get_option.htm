gamepad_get_option

이 함수는 주어진 옵션 값을 검색하는 데 사용할 수 있습니다. 확인할 게임패드 "슬롯"과 옵션 문자열을 제공하면 함수가 관련 값을 반환합니다. 사용 가능한 옵션 문자열은 아래와 같이 프로젝트가 실행되는 플랫폼에 따라 다릅니다.

옵션 플랫폼 설명 보고
"allow_rotation" tvOS Siri Remote의 회전 잠금 여부를 반환합니다(예: 세로로만 사용할 수 있음). 잠금이 해제된 경우 컨트롤을 회전하면 입력이 변경되어 왼쪽으로 90º 회전하면 터치 패드의 "오른쪽"이 "위"로 되돌아가는 등 모든 입력이 방향을 기준으로 합니다. 장치의 부울
"dpad_absolute" tvOS Siri Remote 터치패드가 절대 입력 또는 상대 입력을 허용하는지 여부를 반환합니다. 앱솔루트 모드에서 터치 패드의 중심은 (0,0) 위치로 간주됩니다. 상대 위치로 설정하면 (0,0) 위치는 패드를 처음 터치하는 위치로 간주되며 모든 입력은 패드의 중심이 아닌 해당 위치에서 생성됩니다. 부울

통사론:

gamepad_get_option(device, option_name);

논쟁 유형 설명
device Which gamepad device "slot" to check.
option_name The name of the option to get the value from (a string, see the table above)

보고:

Boolean, Real, String, or N/A (depends on platform and option)

예시:

for(var i = 0; i < 12; ++i;)
{
    if (gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote")
    {
        if (gamepad_get_option(i, "allow_rotation") == false)
        {
            gamepad_set_option(i, "allow_rotation", true);
        }
    }
}

위의 코드는 모든 게임패드 슬롯을 반복하고 tvOS 플랫폼에서 "Siri Remote"를 확인합니다. 하나가 감지되면 입력 회전을 허용하도록 리모컨을 설정합니다.
