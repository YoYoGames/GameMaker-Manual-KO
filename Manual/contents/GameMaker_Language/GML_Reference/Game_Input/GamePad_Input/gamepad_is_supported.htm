gamepad_is_supported

이 기능을 사용하여 대상 플랫폼이 게임 패드를 지원하는지 여부를 확인할 수 있습니다(반환 true ) 여부(반환 false ).

통사론:

gamepad_is_supported();

보고:

예시:

global.GP = gamepad_is_supported();

위의 코드는 게임패드가 지원되는지 확인하고 향후 확인을 위해 전역 변수에 반환 값을 저장합니다.

뒤로: 게임패드 입력
