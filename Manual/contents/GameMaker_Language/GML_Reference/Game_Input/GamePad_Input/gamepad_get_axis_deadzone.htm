gamepad_get_axis_deadzone

이 기능은 조이스틱 축의 "데드존" 값을 얻는 데 사용할 수 있습니다. 가져올 장치 슬롯을 지정하면 이 함수는 0에서 1 사이의 값을 반환합니다. 여기서 값은 조이스틱 축이 0으로 간주되는 임계값을 반영합니다.

통사론:

gamepad_get_axis_deadzone(device);

논쟁 유형 설명
device Which gamepad device "slot" to check.

보고:

예시:

if gamepad_get_axis_deadzone(global.PadId) != 0.5
{
    gamepad_set_axis_deadzone(global.PadId, 0.5);
}

위의 코드는 전역 변수에 저장되어 있는 장치 "슬롯"에 연결된 게임패드의 아날로그 축 임계값을 확인하고 기본값인 0.5가 아닌 경우 해당 값으로 설정합니다.
