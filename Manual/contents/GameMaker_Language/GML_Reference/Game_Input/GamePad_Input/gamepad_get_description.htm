gamepad_get_description

이 함수는 주어진 게임패드에 대한 설명이 포함된 문자열을 반환합니다. 예를 들어 PS3 컨트롤러는 "PLAYSTATION(R)3 Controller"와 유사한 문자열을 반환하거나 XBox360 컨트롤러는 "Xbox 360 Controller(XInput STANDARD GAMEPAD)"를 반환할 수 있습니다. ". 이 문자열은 하드웨어에 따라 다르며 반환된 값은 확인 중인 장치 "슬롯"에 연결된 게임패드에 따라 달라집니다.

통사론:

gamepad_get_description(device);

논쟁 유형 설명
device Which gamepad "slot" to get the name of.

보고:

예시:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    if gamepad_is_connected(i)
    {
        draw_text(32, 32 + (i * 32), gamepad_get_description(i));
    }
    else
    {
        draw_text(32, 32 + (i * 32), "No Gamepad Connected");
    }
}

위의 코드는 모든 게임패드 슬롯을 반복하여 연결된 장치를 확인한 다음 게임패드가 슬롯에 연결되어 있는지 여부에 따라 화면에 일부 텍스트를 그립니다.
