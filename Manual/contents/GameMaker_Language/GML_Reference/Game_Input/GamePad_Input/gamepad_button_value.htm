gamepad_button_value

이 기능을 사용하면 아날로그 버튼의 현재 값을 0에서 1까지 얻을 수 있습니다. 여기서 0은 압력이 없고 1은 최대 압력입니다. 버튼 상수( 여기 에 나열됨) 또는 0에서 0 사이의 정수 값과 함께 확인할 게임패드 슬롯 인덱스를 제공합니다. gamepad_button_count()>- 1.

이것은 버튼에 대한 원시 값을 반환하며 임계값 설정을 고려하지 않습니다 (임계값에 대한 자세한 내용은 여기 참조).

통사론:

gamepad_button_value(device, button);

논쟁 유형 설명
device 진짜 Which gamepad device "slot" to check.
button 게임패드 버튼 상수 Which gamepad button constant to check for.

보고:

Real

예시:

speed = gamepad_button_value(0, gp_shoulderrb) * 4;

위의 코드는 장치 "슬롯" 0에 연결된 게임패드의 아날로그 트리거 값을 사용하여 인스턴스의 속도를 설정합니다.
