게임 입력

게임을 게임 으로 만드는 근본적인 것은 플레이어와의 상호작용입니다. 이것은 컴퓨터의 키보드와 마우스, 모바일의 터치 스크린과 장치 기울기, 심지어 모든 플랫폼의 게임패드를 통한 다양한 수단에 의해 달성됩니다!

GameMaker 에서 시작하려면 PC에서 전용 마우스 또는 키보드 이벤트 를 사용하거나 모바일에서 제스처 이벤트 를 사용하지만 때로는 이러한 이벤트가 제공하는 것보다 더 많은 제어가 필요하거나 게임패드 제어 또는 여러 제어 옵션을 포함하고자 할 수 있습니다. 이를 위해 다양한 게임 입력 기능을 사용하여 어떤 버튼이나 키가 눌리거나 놓았는지 확인하고 화면의 터치 또는 게임패드 조이스틱이 얼마나 움직였는지 검색할 수 있습니다.

게임에는 게임 입력 및 제어를 위한 많은 기능이 있으므로 필요한 것을 더 쉽게 찾을 수 있도록 특정 입력 방법과 관련된 섹션으로 분할했습니다. 사용 가능한 섹션은 다음과 같습니다.

  • 키보드 입력
  • 마우스 입력
  • 장치 입력
  • 제스처 입력
  • 게임패드 입력
  • 가상 키 및 가상 키보드
