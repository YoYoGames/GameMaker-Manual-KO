event_data

이 변수는 범위가 전역적 이며 적절한 이벤트(예: 제스처 이벤트 및 브로드캐스트 메시지 )에서 사용될 때 DS 맵 을 유지하는 데 사용되며 다른 모든 경우에는 -1입니다. DS 맵의 실제 내용은 DS 맵을 트리거한 이벤트 유형에 따라 다르므로 해당 이벤트에 대한 개별 섹션을 참조하십시오.

통사론:

event_data;

보고:

DS Map ID

예시:

isFlick = event_data[?"isflick"];

if (isFlick)
{
    flickVelX = event_data[?"diffX"];
    flickVelY = event_data[?"diffY"];
}
else
{
    flickVelX = 0;
    flickVelY = 0;
}

위의 코드는 드래그 엔드 제스처 이벤트에서 가져온 것이며 통풍구가 "플릭" 이벤트인지 확인하고, 그렇다면 이벤트에서 필요한 데이터를 추출합니다. event_data DS 맵을 만들고 일부 변수를 설정하는 데 사용합니다. "flick" 이벤트가 감지되지 않으면 동일한 변수가 0으로 설정됩니다.
