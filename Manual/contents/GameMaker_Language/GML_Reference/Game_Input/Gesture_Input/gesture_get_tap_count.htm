gesture_get_tap_count

이 기능은 탭 카운팅의 활성화 여부를 확인하는 데 사용됩니다. 함수가 반환됩니다 true 활성화된 경우 false 그렇지 않으면 기능을 사용하여 탭 계산을 활성화하거나 비활성화할 수 있습니다.gesture_tap_count(). 활성화되면 각 탭 이벤트에는 탭 수를 등록하는 추가 "탭 수" DS 맵 항목이 있으며 활성화되면 모든 탭 이벤트가 트리거됩니다. 즉, 두 번의 탭은 단일 탭 이벤트와 더블 탭 이벤트, 싱글 탭 이벤트 탭 카운트 1, 더블 탭 이벤트 탭 카운트 2. 더블 탭 감지에 설정된 시간이 지나면 탭 카운트 값이 0으로 재설정됩니다(기능 참조gesture_double_tap_time() ). 탭 카운팅이 비활성화된 경우 더블 탭 시간이 지나고 두 번째 탭이 감지되지 않을 때까지 초기 탭이 등록되지 않습니다. 이것은 기본적으로 활성화 되어 있습니다.

통사론:

gesture_get_tap_count();

보고:

예시:

if !gesture_get_tap_count()
{
    getsure_tap_count(true);
}

위의 코드는 탭 카운팅이 활성화되었는지 확인하고 활성화되지 않은 경우 켜져 있는지 확인합니다.
