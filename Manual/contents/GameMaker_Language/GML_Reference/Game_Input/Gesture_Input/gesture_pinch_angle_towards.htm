gesture_pinch_angle_towards

이 기능은 핀치 인 제스처 가 잠재적으로 시작되기 전에 터치가 다른 터치 쪽으로 움직여야 하는 각도를 설정하는 데 사용됩니다. 각도는 도 단위로 측정되며 기본값은 45°입니다.

통사론:

gesture_pinch_angle_towards();

보고:

(inches)

예시:

if gesture_get_pinch_angle_towards() != 45
{
    gesture_pinch_angle_towards(45);
}

위의 코드는 제스처에 대한 핀치 인 각도가 45°로 설정되어 있는지 확인하고 그렇지 않은 경우 해당 값으로 설정합니다.

뒤로: 제스처 입력
다음: gesture_pinch_angle_away
