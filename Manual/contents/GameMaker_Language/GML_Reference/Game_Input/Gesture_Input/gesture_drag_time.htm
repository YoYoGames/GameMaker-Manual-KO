gesture_drag_time

이 함수는 드래그 제스처 이벤트가 터치 또는 클릭에 의해 트리거되는 데 걸리는 시간을 설정하는 데 사용됩니다. 이 시간은 터치/클릭으로 탭 이벤트 가 트리거되는 방식에도 영향을 미치며 이 시간이 다 되기 전에 놓으면 탭으로 간주됩니다. 시간은 초 단위로 측정되며 기본값은 0.16입니다.

통사론:

gesture_drag_time(time);

논쟁 유형 설명
time The time (in seconds) to set for drag gesture event detection.

보고:

예시:

if gesture_get_drag_time() != 0.16
{
    gesture_drag_time(0.16);
}

위의 코드는 제스처에 대한 드래그 시간이 0.16초로 설정되어 있는지 확인하고 그렇지 않은 경우 해당 값으로 설정합니다.
