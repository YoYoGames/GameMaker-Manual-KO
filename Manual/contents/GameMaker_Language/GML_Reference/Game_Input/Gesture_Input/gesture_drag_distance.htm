gesture_drag_distance

이 기능은 드래그 제스처 이벤트가 터치 또는 클릭의 움직임에 의해 트리거되는 데 걸리는 거리를 설정하는 데 사용됩니다. 거리는 인치로 측정되며 기본값은 0.1입니다.

통사론:

gesture_drag_distance(distance);

논쟁 유형 설명
distance The distance (in inches) to set for drag gesture event detection.

보고:

예시:

if gesture_get_drag_distance() != 0.1
{
    gesture_drag_distance(0.1);
}

위의 코드는 제스처의 드래그 거리가 0.1인치로 설정되어 있는지 확인하고 그렇지 않은 경우 해당 값으로 설정합니다.
