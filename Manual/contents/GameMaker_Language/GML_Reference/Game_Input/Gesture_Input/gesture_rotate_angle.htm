gesture_rotate_angle

이 기능은 회전 시작 제스처 를 트리거하기 위해 한 쌍의 터치가 초과해야 하는 각도를 설정하는 데 사용됩니다. 각도는 도 단위로 측정되며 기본값은 5°입니다.

통사론:

gesture_rotate_angle();

보고:

(inches)

예시:

if gesture_get_rotate_angle() != 5
{
    gesture_rotate_angle(5);
}

위의 코드는 제스처의 회전 각도가 5°로 설정되어 있는지 확인하고 그렇지 않은 경우 해당 값으로 설정합니다.
