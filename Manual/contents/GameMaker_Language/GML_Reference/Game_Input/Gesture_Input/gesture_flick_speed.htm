gesture_flick_speed

이 기능은 터치 또는 클릭이 해제될 때 Flick Gesture 이벤트가 트리거되는 데 필요한 속도를 설정하는 데 사용됩니다. 속도는 초당 인치로 측정되며 기본값은 2.0입니다.

통사론:

gesture_flick_speed(speed);

논쟁 유형 설명
speed The speed (in inches per second) to set for flick gesture event detection.

보고:

예시:

if gesture_get_flick_speed() != 2
{
    gesture_flick_speed(2);
}

위의 코드는 제스처의 플릭 속도가 초당 2인치로 설정되어 있는지 확인하고 그렇지 않은 경우 해당 값으로 설정합니다.

뒤로: 제스처 입력
