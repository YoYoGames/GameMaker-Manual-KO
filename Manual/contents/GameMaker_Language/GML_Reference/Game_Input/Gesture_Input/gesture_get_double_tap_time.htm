gesture_get_double_tap_time

이 함수는 두 번의 터치/클릭 사이에 Double Tap Gesture 이벤트를 트리거하는 데 걸리는 시간을 가져오는 데 사용됩니다. 시간은 초 단위로 측정되며 기본값은 0.10입니다.

통사론:

gesture_get_double_tap_time();

보고:

Real (seconds)

예시:

if gesture_get_double_tap_time() != 0.16
{
    gesture_double_tap_time(0.16);
}

위의 코드는 제스처에 대한 더블 탭 시간이 0.16초로 설정되어 있는지 확인하고 그렇지 않은 경우 해당 값으로 설정합니다.
