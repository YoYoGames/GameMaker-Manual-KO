제스처 입력

장치 입력 기능은 모두 훌륭하고 간단한 화면 터치를 처리하거나 장치 방향을 가져오는 데 적합하지만 때로는 프로젝트가 방을 가로질러 인스턴스를 드래그하거나 "튕기기"할 수 있는 것과 같이 더 복잡한 것을 감지하기를 원할 것입니다.. 이러한 작업은 장치 기능으로 수행 할 수 있지만 더 쉽게 만들기 위해 GameMaker 에는 인스턴스 및 글로벌 수준에서 두 번 탭, 드래그, 플릭, 핀치 및 회전을 감지하고 정보를 반환하는 완전한 범위의 제스처 이벤트 가 있습니다. 이벤트에 대해. 여기에 나열된 기능은 모두 이러한 이벤트가 트리거되는 방법과 시기를 더 잘 제어할 수 있도록 설계되었습니다.

중요 이러한 기능은 HTML5 대상 플랫폼에서 작동하지 않습니다. 사용하면 오류가 발생하지 않지만 대신 단순히 무시됩니다.
