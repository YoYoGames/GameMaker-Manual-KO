gesture_get_rotate_time

이 함수는 회전 시작 제스처 가 트리거되기 위해 한 쌍의 터치가 일관된 방향으로 회전해야 하는 시간을 가져오는 데 사용됩니다. 시간은 초 단위로 측정되며 기본값은 0.16초입니다.

중요한! 현재 60fps 게임의 경우 최대 1초로만 설정할 수 있습니다. 그렇지 않으면 회전이 감지되지 않습니다. 예를 들어 30fps 게임의 최대 시간은 2초일 수 있습니다.

통사론:

gesture_get_rotate_time();

보고:

(inches)

예시:

if gesture_get_rotate_time() != 0.1
{
    gesture_rotate_time(0.1);
}

위의 코드는 제스처의 회전 시간이 0.1초로 설정되어 있는지 확인하고 그렇지 않은 경우 해당 값으로 설정합니다.
