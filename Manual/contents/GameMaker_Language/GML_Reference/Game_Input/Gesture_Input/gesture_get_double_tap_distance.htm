gesture_get_double_tap_distance

이 기능은 더블 탭 제스처 를 실행하기 위해 한 번 탭한 후 화면을 다시 터치/클릭해야 하는 거리를 얻는 데 사용됩니다. 거리는 인치로 측정되며 기본값은 0.1입니다.

통사론:

gesture_get_double_tap_distance();

보고:

(inches)

예시:

if gesture_get_double_tap_distance() != 0.1
{
    gesture_double_tap_distance(0.1);
}

위의 코드는 제스처에 대한 더블 탭 거리가 0.1인치로 설정되어 있는지 확인하고 그렇지 않은 경우 해당 값으로 설정합니다.
