device_get_tilt_y

이 함수는 장치의 "기울기" 각도에 따라 -1에서 1 사이의 값을 반환합니다. 기울기와 반환된 값 사이의 실제 상관 관계는 사용하는 장치 및 OS에 따라 다르지만 일반적으로 값 1 또는 -1은 +/-90°와 동일합니다. 아래 이미지는 사용 가능한 기능이 장치와 관련된 경우 각각을 보여줍니다.

통사론:

device_get_tilt_y()

보고:

예시:

if display_get_orientation() = display_landscape
{
    x += sign(device_get_tilt_y());
}
else
{
    x += sign(device_get_tilt_x());
}

위의 코드는 디스플레이의 방향을 확인한 다음 해당 기울기 값을 사용하여 x축을 따라 플레이어를 이동합니다.

뒤로: 장치 입력
