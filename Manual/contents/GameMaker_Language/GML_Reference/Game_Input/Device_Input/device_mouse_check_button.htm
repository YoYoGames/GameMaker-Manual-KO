device_mouse_check_button

이 함수는 반환 true 또는 false 주어진 장치 에서 주어진 마우스 버튼 을 누르고 있는지 여부에 따라 다릅니다. 마우스 버튼의 경우 이 페이지에 나열된 상수 중 하나를 제공할 수 있습니다.

이 기능은 터치 스크린에 사용할 수 있으며 device 인수는 0에서 n 까지일 수 있습니다. 여기서 n 은 한 번에 화면을 터치할 수 있는 "손가락"의 최대 수입니다(검출할 수 있는 최대 터치 수는 매우 다양합니다. 사용 중인 장치와 실행되는 OS에 많은 영향을 미치지만 대부분의 장치는 최소 4개까지 감지해야 합니다. 참고 mb_right 더블 탭 터치가 감지된 경우에만 감지됩니다(이 동작은 기능을 사용하여 비활성화할 수 있습니다.device_mouse_dbclick_enable() ).

통사론:

device_mouse_check_button(device, button);

논쟁 유형 설명
device The device (from 0 - n ) that is being checked
button The button of the device that is being checked.

보고:

예시:

if device_mouse_check_button(0, mb_left) && device_mouse_check_button(1, mb_left)
{
    room_goto(rm_Menu);
}

위의 코드는 두 개의 터치가 동시에 누르고 있는지 확인하고 다른 방으로 이동하는지 확인합니다.
