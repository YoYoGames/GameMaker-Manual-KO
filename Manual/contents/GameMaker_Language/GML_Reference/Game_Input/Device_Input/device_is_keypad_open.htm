device_is_keypad_open

이것은 키패드에 대한 장치를 확인하고 사용 가능한 경우 반환합니다. true 그렇지 않으면 반환 false. 이 기능은 주로 Android 기기에서 사용하기 위한 것입니다.

Sony Xperia Play 를 사용하는 사용자의 경우 GameMaker 내에 버튼/키 맵 설정이 있으므로 키보드 상수 vk_up, vk_down, vk_left, vk_right 를 조이패드 키에 사용할 수 있으며 삼각형 은 다음과 같습니다. ord("T"), 스퀘어 는 ord("S"), 원 은 vk_alt + vk_backspace 이고 뒤로 버튼은 단순히 vk_backspace, Cross 는 vk_space, Select 는 vk_return, Start 는 vk_rshift, L Trigger 는 ord("L") 그리고 R 트리거 는 ord("R").

통사론:

device_is_keypad_open

보고:

예시:

if device_is_keypad_open()
{
     global.Setting = 2;
}
else
{
     global.Setting = 1;
}

위의 코드는 키패드를 확인한 다음 반환된 값에 따라 전역 변수를 변경합니다.

뒤로: 장치 입력
