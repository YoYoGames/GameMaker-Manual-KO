장치 입력

게임이 멀티 터치를 허용하는 플랫폼을 대상으로 하는 경우 이러한 기능을 사용해야 합니다. 여러 개의 마우스가 연결되어 있고 확인 중인 상수가 표준 마우스 기능 에 사용된 상수와 동일한 경우 PC가 작동하는 것과 동일한 방식으로 작동합니다.

일정한 설명
mb_left 마우스 왼쪽 버튼
mb_middle 마우스 가운데 버튼(일부 대상 플랫폼에서는 유효하지 않을 수 있음)
mb_right 마우스 오른쪽 버튼
mb_side1* 마우스 측면 버튼 1
mb_side2* 마우스 측면 버튼 2
mb_any 모든 마우스 버튼
mb_none 마우스 버튼 없음

참고 * mb_side1 및 mb_side2 버튼은 Windows, macOS, Ubuntu 및 HTML5에서만 사용할 수 있습니다.

기능은 모두 장치 디스플레이의 터치에 해당하는 "터치" 값을 취할 수 있지만 감지할 수 있는 최대 터치 수는 장치와 실행 중인 OS에 따라 크게 달라집니다. 예를 들어 일부 저가형 Android 기기는 동시에 2~3개의 터치만 감지할 수 있는 반면 UWP 기기는 최대 10개의 터치를 감지할 수 있습니다. 터치 스크린 기기에서는 두 번 탭이 다음과 같이 반환됩니다. 마우스 오른쪽 버튼 이벤트와 동일합니다. 따라서 예를 들어 여러 탭을 빠르게 연속으로 감지해야 하는 경우 두 가지 모두 를 확인해야 합니다. mb_left 그리고 mb_right. 이 기능은 다음을 사용하여 켜거나 끌 수 있습니다.device_mouse_dbclick_enable().

모바일 장치(Android, iOS 및 Windows Phone)에서는 뒤로 버튼이 GameMaker 키보드 상수 인 키보드 백스페이스 에 매핑되어 언제든지 사용할 수 있습니다. vk_backspace, 따라서 해당 키를 확인하고 뒤로 버튼이 사용된 것처럼 해당 장치에서 작업을 수행할 수 있습니다.
