device_mouse_dbclick_enable

이 기능을 사용하여 장치의 더블 탭을 감지하도록 설정할 수 있습니다. mb_left (마우스 왼쪽 버튼) mb_right (마우스 오른쪽 버튼) 입력.

기본적으로 다음으로 설정됩니다. true, 즉 사용자가 장치 화면을 빠르고 연속적으로 두 번 탭할 때마다 반환 값은 마우스 오른쪽 버튼을 클릭한 것과 동일합니다. 이 기능이 켜져 있으면 첫 번째 탭이 다음과 같이 감지됩니다. mb_left, 그리고 두 번째 mb_right, 따라서 사용하는 모든 코드에서 이 점을 고려해야 합니다.

통사론:

device_mouse_dbclick_enable(bool);

논쟁 유형 설명
bool 부울 Set double-click detection on (true ) or off (false ).

보고:

Boolean

예시:

if os_type == os_windows || os_type == os_mac
{
    device_mouse_dbclick_enable(false);
}

위의 코드는 게임을 실행하는 장치가 Windows PC인지 Mac인지 확인하고 둘 중 하나일 경우 더블 탭 기능을 비활성화합니다.

뒤로: 장치 입력
