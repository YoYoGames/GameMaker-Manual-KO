device_mouse_check_button_released

이 함수는 반환 true 또는 false 지정한 디바이스 터치가 해제되었는지 여부에 따라 다릅니다. device 인수는 0에서 n 사이일 수 있는 장치 터치 수를 나타내며 감지할 수 있는 최대 터치 수는 사용 중인 장치와 실행되는 OS에 따라 크게 달라집니다(대부분의 장치는 최소 4 동시 터치). 이 기능은 실제 해제 동작에 의해 한 번만 트리거 되며 이 페이지에 나열된 상수를 사용하여 마우스 버튼을 확인할 수 있습니다. 참고 mb_right 더블 탭 터치가 감지되고 손을 떼기 전에 두 번째 탭에서 유지되는 경우에만 감지됩니다(이 동작은 기능을 사용하여 비활성화할 수 있습니다. device_mouse_dbclick_enable() ).

통사론:

device_mouse_check_button_released(device, button);

논쟁 유형 설명
device The device (from 0 - n ) that is being checked.
button The button of the device that is being checked.

보고:

예시:

if device_mouse_check_button_released(0, mb_left)
{
    press = false;
}

위의 코드는 장치가 눌러졌는지 확인하고 변수가 있으면 false로 설정합니다.
