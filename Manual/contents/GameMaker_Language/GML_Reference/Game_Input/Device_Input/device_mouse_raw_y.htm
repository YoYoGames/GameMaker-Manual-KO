device_mouse_raw_y

이 함수는 장치에서 터치의 원시 y 위치를 반환합니다. 이것이 의미하는 바는 GameMaker 가 아닌 터치되는 y 위치의 실제 장치 정의를 반환하므로 보기 위치 및 크기 조정과 같은 것을 무시합니다. 감지할 수 있는 최대 장치 터치 수는 사용 중인 장치와 실행되는 OS에 따라 크게 달라집니다.

참고: 이 기능은 장치에 따라 크게 달라지므로 원하는 대상 모듈과 장치를 먼저 실험하여 정확히 무엇을 반환하는지 확인해야 합니다.

통사론:

device_mouse_raw_y(device);

논쟁 유형 설명
device The device (from 0 - n ) that is being checked.

보고:

예시:

if device_mouse_check_button(0, mb_left) && device_mouse_check_button(1, mb_left)
{
    x_av = mean(device_mouse_raw_x(0), device_mouse_raw_x(1));
    y_av = mean(device_mouse_raw_y(0), device_mouse_raw_y(1));
}

위의 코드는 device1과 device2가 눌렸는지 확인하고, 눌려있다면 각 누름점 사이의 x/y 좌표의 평균 위치를 계산합니다.
