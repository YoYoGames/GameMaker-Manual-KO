device_mouse_y

이 함수는 장치에서 터치의 y 위치를 반환합니다. HTML5 또는 PC 및 Mac 모듈에서 이것을 실행하는 경우 이 값은 장치(일반적으로 마우스)가 연결되어 있는 한 지속적으로 업데이트되지만 모바일 장치의 경우 화면이 켜져 있는 동안에만 업데이트됩니다. 감지할 수 있는 최대 터치 수는 사용 중인 장치와 실행되는 OS에 따라 크게 달라집니다.

통사론:

device_mouse_y(device);

논쟁 유형 설명
device The device (from 0 - n ) that is being checked.

보고:

예시:

if device_mouse_check_button(0, mb_left)
{
    if device_mouse_y(0) > y-32 && device_mouse_y(0) < y+32
    {
        pressed = true;
    }
    else
    {
        pressed = false;
    }
}

위의 코드는 장치가 눌려 있는지 확인하고 그렇다면 장치 y 위치를 폴링하여 매개변수 내에 있는지 확인합니다. 그렇다면 "눌러진"변수를 다음으로 설정합니다. true, 그렇지 않으면 다음으로 설정합니다. false.
