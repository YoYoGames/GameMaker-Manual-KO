keyboard_check_direct

이 함수는 반환됩니다 true 특정 키코드가 있는 키를 눌렀을 경우, 또는 false 그렇지 않은 경우 하드웨어를 직접 확인하십시오. 몇 가지 더 확인할 수 있습니다. 특히 키 코드 vk_lshift, vk_lcontrol, vk_lalt, vk_rshift, vk_rcontrol 및 vk_ralt 를 사용하여 왼쪽 또는 오른쪽 shift, control 또는 alt 키가 눌렸는지 확인할 수 있습니다. 함수는 함수에서 반환된 키 코드 값을 사용합니다.ord() (AZ의 대문자 또는 0-9의 숫자만), 또는 vk_* 기본 키보드 입력 페이지에 나열된 상수.

참고 이 기능은 표준 Windows 대상에만 사용할 수 있으며 결과는 포커스가 있는 응용 프로그램과 무관합니다.

통사론:

keyboard_check_direct(key);

논쟁 유형 설명
key The key to check the down state of.

보고:

예시:

if keyboard_check_direct(vk_ralt) || keyboard_check_direct(vk_lalt)
{
    crouch = true;
}

위의 코드는 왼쪽 또는 오른쪽 alt 키가 눌렸는지 확인하고, 알트 키가 있으면 "couch" 변수를 true로 설정합니다.
