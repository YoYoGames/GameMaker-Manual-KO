keyboard_clear

이 기능을 사용하면 현재 키보드 상태를 지울 수 있습니다. 즉, 기본적으로 키를 누르고 있는 경우 다시 놓을 때까지 더 이상 인식되지 않습니다(이 경우에도 Keyboard Key Released 이벤트를 트리거하지 않음). 그리고 다시 눌렀다. 함수는 함수에서 반환된 키 코드 값을 사용합니다.ord() (AZ의 대문자 또는 0-9의 숫자만), 또는 vk_* 기본 키보드 입력 페이지에 나열된 상수.

통사론:

keyboard_clear(key);

논쟁 유형 설명
key The key to clear.

보고:

예시:

keyboard_clear(vk_space);

위의 코드는 스페이스바의 상태를 지웁니다.
