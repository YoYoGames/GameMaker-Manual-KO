keyboard_set_numlock

이 기능을 사용하여 키패드 번호 잠금을 켜거나 끌 수 있습니다(켜짐의 경우 true 로 설정하고 꺼짐의 경우 false 로 설정).

참고 이 기능은 Windows 대상 빌드에서만 사용할 수 있으며 다른 장치에서는 작동하지 않습니다.

통사론:

keyboard_set_numlock(value);

논쟁 유형 설명
value 부울 Set this to true for "on" and false for "off"

보고:

N/A

예시:

if keyboard_get_numlock()
{
    keyboard_set_numlock(false);
}
else
{
    keyboard_set_numlock(true);
}

위의 예제 코드는 numberlock 키의 상태를 가져오고 그것이 켜져 있으면(true) 꺼져(false)로 설정되고 그 반대의 경우도 마찬가지입니다.
