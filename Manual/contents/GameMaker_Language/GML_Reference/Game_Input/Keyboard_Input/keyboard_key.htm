keyboard_key

이 변수를 사용하면 현재 누르고 있는 키의 키 코드를 얻을 수 있으며 검사가 완료될 때 아무 키도 누르지 않으면 0을 반환합니다.

통사론:

keyboard_key;

보고:

Virtual Key Constant

예시:

switch (keyboard_key)
{
    case vk_numpad1: gun = weapon[0][0]; break;
    case vk_numpad2: gun = weapon[1][0]; break;
    case vk_numpad3: gun = weapon[2][0]; break;
}

위의 코드는 다음 값을 사용합니다. keyboard_key 변수를 배열과 동일한 값으로 설정합니다.

뒤로: 키보드 입력
