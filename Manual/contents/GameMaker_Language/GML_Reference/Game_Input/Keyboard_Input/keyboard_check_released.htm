keyboard_check_released

이 기능을 사용하면 키가 해제되었는지 여부를 확인할 수 있습니다. 달리keyboard_check() 기능에서 이 기능은 키를 놓을 때마다 한 번만 실행되므로 다시 트리거하려면 키를 먼저 눌렀다가 다시 놓아야 합니다. 함수는 함수에서 반환된 키 코드 값을 사용합니다.ord() (AZ의 대문자 또는 0-9의 숫자만), 또는 vk_* 기본 키보드 입력 페이지에 나열된 상수.

통사론:

keyboard_check_released(key);

논쟁 유형 설명
key The key to check the released state of.

보고:

예시:

if keyboard_check_released(ord("P"))
{
    instance_create_layer(0, 0, "Controllers", obj_Pause);
}

위의 코드는 "P" 키가 해제되었는지 확인하고 해제된 경우 "obj_Pause" 인스턴스를 생성합니다.
