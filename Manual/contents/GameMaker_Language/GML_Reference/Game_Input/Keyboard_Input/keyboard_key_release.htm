keyboard_key_release

이 기능을 사용하면 키보드에서 아무 키나 놓는 것을 시뮬레이션할 수 있습니다. 함수는 함수에서 반환된 키 코드 값을 사용합니다.ord() (AZ의 대문자 또는 0-9의 숫자만), 또는 vk_* 기본 키보드 입력 페이지에 나열된 상수.

통사론:

keyboard_key_release(key);

논쟁 유형 설명
key The key to simulate a release of.

보고:

예시:

keyboard_key_release(vk_space);

이것은 스페이스바 릴리스를 시뮬레이션합니다.
