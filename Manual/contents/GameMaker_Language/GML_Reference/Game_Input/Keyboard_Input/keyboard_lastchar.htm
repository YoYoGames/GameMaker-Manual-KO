keyboard_lastchar

이 변수는 마지막으로 누른 키의 문자열을 저장합니다. 이 변수는 읽기 전용이 아니므 로 변경할 수 있습니다. 예를 들어 이미 처리한 경우 ""(빈 문자열)로 설정합니다.

통사론:

keyboard_lastchar;

보고:

예시:

if keyboard_lastkey != -1
{
    str += keyboard_lastchar;
    keyboard_lastkey = -1;
}

위의 코드는 lastkey 변수가 -1보다 큰지 확인하고, 그렇다면 마지막 키를 문자열로 "str" 변수에 추가합니다.
