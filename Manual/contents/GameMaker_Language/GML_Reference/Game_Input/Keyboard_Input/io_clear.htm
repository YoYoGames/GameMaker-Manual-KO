io_clear

이 기능은 현재 키보드와 마우스 상태를 지울 것입니다. 이 함수를 호출한 후 다음 단계에서 일어나는 일은 검사 대상과 검사 대상 플랫폼에 따라 다릅니다. 마우스 입력의 경우, 추가 입력이 감지되려면 마우스 버튼을 놓았다가 다시 눌러야 하지만 키보드에서는 그렇게 간단하지 않습니다. 일반적으로 키보드는 해당 단계에 대해서만 지워지지만 다음 단계는 키를 다시 감지하지만 모든 플랫폼 및 키보드에서 동일하지 않을 수 있으며 경우에 따라 기능이 마우스 및 키를 들었다가 다시 눌러야 합니다.

통사론:

io_clear();

보고:

예시:

io_clear();

그러면 모든 키보드 및 마우스 상태가 지워집니다.
