keyboard_check

이 기능을 사용하면 키가 눌려져 있는지 여부를 확인할 수 있습니다. 달리keyboard_check_pressed() 또는keyboard_check_released() 키를 누르거나 놓을 때 한 번만 트리거되는 기능의 경우 이 기능은 키를 누르고 있는 모든 단계에서 트리거됩니다. 함수는 함수에서 반환된 키 코드 값을 사용합니다.ord() (AZ의 대문자 또는 0-9의 숫자만), 또는 vk_* 기본 키보드 입력 페이지에 나열된 상수.

통사론:

keyboard_check(key);

논쟁 유형 설명
key 가상 키 상수(vk_*) The key to check the down state of.

보고:

Boolean

예시:

if keyboard_check(vk_left)
{
    x -= 5;
}

위의 코드는 화살표 키가 눌려 있는지 확인하고 true를 반환하는 모든 단계에서 인스턴스를 왼쪽으로 5픽셀 이동합니다.
