keyboard_lastkey

이 변수는 다음 값을 나타냅니다.keyboard_key 이전 프레임에 있었고 해당 키의 키 코드를 반환했습니다(모든 표준 키 코드 상수가 반환됨). 이 변수는 읽기 전용이 아니므 로 변경할 수 있습니다. 예를 들어 이미 처리한 경우 -1로 설정합니다.

통사론:

keyboard_lastkey;

보고:

Virtual Key Constant

예시:

if (keyboard_lastkey != -1)
{
    str += keyboard_lastchar;
    keyboard_lastkey = -1;
}

위의 코드는 lastkey 변수가 -1과 같지 않은지 확인하고, 그렇다면 마지막 키가 문자열로 된 것을 변수 "str"에 추가한 다음 재설정합니다. keyboard_lastkey 추가 입력을 허용하는 변수입니다.

뒤로: 키보드 입력
다음: keyboard_lastchar
