keyboard_unset_map

이 기능을 사용하면 다시 매핑된 모든 키를 지워서 기본 상태로 돌아갈 수 있습니다. 즉, 모든 키를 자신에 매핑할 수 있습니다.

통사론:

keyboard_get_map();

보고:

N/A

예시:

if keyboard_check_pressed(vk_escape)
{
    keyboard_unset_map();
}

위의 예제 코드는 사용자가 "escape" 키를 누르면 매핑된 모든 키를 기본 설정으로 재설정합니다.
