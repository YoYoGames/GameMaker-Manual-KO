keyboard_string

이 변수는 키보드에 입력된 마지막(최대) 1024개의 문자를 포함하는 문자열을 보유합니다. 이 문자열에는 입력된 인쇄 가능한 문자만 포함되지만 마지막 문자를 지워 백스페이스 키 를 누르면 올바르게 응답합니다. 이 변수는 읽기 전용이 아니므 로 변경할 수 있습니다. 예를 들어 이미 처리한 경우 ""(빈 문자열)로 설정하고 문자열 함수 를 사용하여 조작할 수 있습니다. 온스크린 가상 키보드 를 사용할 때 이 변수 만 키보드 입력으로 업데이트됩니다.

통사론:

keyboard_string;

보고:

예시:

if string_length(keyboard_string) > 15
{
    keyboard_string = string_copy(keyboard_string, 1, 15);
}

위의 코드는 키보드 문자열의 길이를 15자로 제한하고 처음 15자를 변수에 다시 복사하여 해당 제한을 초과하는 문자열을 제거합니다.
