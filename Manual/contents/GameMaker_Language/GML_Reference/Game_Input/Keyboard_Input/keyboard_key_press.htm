keyboard_key_press

이 기능을 사용하면 키보드에서 아무 키나 누르는 것을 시뮬레이션할 수 있습니다. 사용하면 해당 릴리스 기능이 호출될 때까지 키가 눌린 것으로 플래그가 지정됩니다(참조keyboard_key_release() ). 함수는 함수에서 반환된 키 코드 값을 사용합니다.ord() (AZ의 대문자 또는 0-9의 숫자만), 또는 vk_* 기본 키보드 입력 페이지에 나열된 상수.

이 함수를 호출한 후 키보드에서 키를 물리적으로 누르면 이 누르기 이벤트는 키에서 물리적으로 손을 떼거나(해제 이벤트를 트리거하고 이 기능을 중지할 때까지) 해당 릴리스 키 기능이 호출되고 키를 다시 눌렀습니다.

통사론:

keyboard_key_press(key);

논쟁 유형 설명
key The key to simulate a press of.

보고:

예시:

keyboard_key_press(vk_space);

이것은 스페이스바 누르기를 시뮬레이션합니다.
