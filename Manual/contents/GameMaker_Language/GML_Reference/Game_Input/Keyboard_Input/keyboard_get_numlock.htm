keyboard_get_numlock

이 기능을 사용하여 켜짐에 대해 true 가 반환되고 꺼짐에 대해 false 가 반환되는 키패드 번호 잠금 상태를 찾을 수 있습니다.

참고 이 기능은 Windows exe 빌드에서만 사용할 수 있으며 다른 장치에서는 작동하지 않습니다.

통사론:

keyboard_get_numlock();

보고:

예시:

if keyboard_get_numlock()
{
    keyboard_set_numlock(false);
}
else
{
    keyboard_set_numlock(true);
}

위의 예제 코드는 numberlock 키의 상태를 가져오고 그것이 켜져 있으면(true) 꺼져(false)로 설정되고 그 반대의 경우도 마찬가지입니다.
