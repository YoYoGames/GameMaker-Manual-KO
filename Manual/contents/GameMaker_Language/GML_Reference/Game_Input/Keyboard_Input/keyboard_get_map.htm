keyboard_get_map

때로는 이 함수가 반환하는 매핑된 키에 대한 ASCII 코드를 얻고 싶을 수도 있습니다(예: 이미 매핑되어 있는지 확인하기 위해). 함수는 함수에서 반환된 키 코드 값을 사용합니다.ord() (AZ의 대문자 또는 0-9의 숫자만), 또는 vk_* 기본 키보드 입력 페이지에 나열된 상수.

통사론:

keyboard_get_map(key);

논쟁 유형 설명
key This is the key that you wish to get the mapped value from

보고:

예시:

if keyboard_get_map(ord("A")) = ord("A")
{
    keyboard_set_map(ord("A"), keyboard_lastkey);
}

위의 예제 코드는 먼저 "A"가 다른 키에 매핑되었는지 확인하고 그렇지 않은 경우 사용자가 누른 마지막 키에 매핑합니다.

뒤로: 키보드 입력
