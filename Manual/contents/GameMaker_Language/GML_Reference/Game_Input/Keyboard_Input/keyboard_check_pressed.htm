keyboard_check_pressed

이 기능을 사용하면 키를 눌렀는지 여부를 확인할 수 있습니다. 달리keyboard_check() 기능에서 이 기능은 키를 누를 때마다 한 번만 실행되므로 다시 트리거하려면 먼저 키를 놓았다가 다시 눌러야 합니다. 함수는 함수에서 반환된 키 코드 값을 사용합니다.ord() (AZ의 대문자 또는 0-9의 숫자만), 또는 vk_* 기본 키보드 입력 페이지에 나열된 상수.

통사론:

keyboard_check_pressed(key);

논쟁 유형 설명
key 가상 키 상수(vk_*) The key to check the pressed state of.

보고:

Boolean

예시:

if keyboard_check_pressed(vk_anykey)
{
    room_goto_next();
}

위의 코드는 플레이어가 키보드의 키 중 하나를 누르면 다음 방으로 진행됩니다("계속하려면 아무 키나 누르십시오" 프롬프트처럼 작동).

뒤로: 키보드 입력
