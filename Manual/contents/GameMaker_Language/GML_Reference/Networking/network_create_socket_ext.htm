network_create_socket_ext

이 함수는 게임이 네트워크를 통해 통신할 수 있는 새 클라이언트 소켓을 만드는 데 사용됩니다. 소켓 유형(아래 상수 목록 참조)을 정의하고 사용할 포트를 지정해야 합니다. 그러면 함수는 해당 소켓에 대한 모든 추가 함수 호출에 사용해야 하는 고유 ID 를 반환하거나 다음과 같은 경우 0보다 작은 값을 반환합니다. 연결이 실패합니다.

통사론:

network_create_socket_ext(protocol, port);

논쟁 유형 설명
protocol 소켓 유형 상수 The network protocol to use
port 진짜 The port to use

보고:

Network Socket ID

예시:

client = network_create_socket_ext(network_socket_udp, 6510);

위의 코드는 포트 6510에 새로운 UDP 소켓을 생성합니다.
