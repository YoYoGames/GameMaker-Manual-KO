network_create_socket

이 함수는 게임이 네트워크를 통해 통신할 수 있는 새 클라이언트 소켓을 만드는 데 사용됩니다. 소켓 유형을 정의해야 하며(아래 상수 목록 참조) 함수는 해당 소켓에 대한 모든 추가 함수 호출에 사용해야 하는 해당 소켓에 대한 고유 ID 를 반환하거나 연결이 실패하면 0보다 작은 값을 반환합니다..

팁 IP "127.0.0.1"을 사용하여 게임을 실행하는 동일한 장치에 다시 연결할 수 있습니다.

통사론:

network_create_socket(type);

논쟁 유형 설명
type 소켓 유형 상수 The type of socket connection to create (see the constants listed above).

보고:

Network Socket ID

예시:

if os_browser == browser_not_a_browser
{
    client = network_create_socket(network_socket_tcp);
    network_connect( client, "192.134.0.1", 6510 );
}
else
{
    client = network_create_socket(network_socket_ws);
    network_connect_raw_async( client, "192.134.0.1", 6520 );
}

위의 코드는 게임이 브라우저에서 실행 중인지 여부를 확인하고 주어진 포트의 지정된 IP 주소에 연결을 시도하기 전에 새 TCP 또는 웹 소켓을 생성합니다.
