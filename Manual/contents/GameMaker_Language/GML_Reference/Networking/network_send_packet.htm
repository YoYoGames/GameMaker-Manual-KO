network_send_packet

이 기능을 사용하면 네트워크를 통해 데이터 "패킷"을 보낼 수 있습니다. 이 함수는 소켓 ID 를 사용하여 연결한 다음 보낼 데이터가 포함된 버퍼 ID (버퍼에 대한 자세한 내용은 참조 - 버퍼 참조 )와 마지막으로 데이터 패킷의 크기(바이트)를 제공해야 합니다. 이 함수로 전송된 패킷은 데이터를 수신하는 GameMaker 게임이 패킷을 올바르게 "분할"할 수 있도록 형식이 지정되며, 함수는 전송된 데이터의 바이트 수를 반환하거나 전송이 실패한 경우 0보다 작은 수를 반환합니다. 이 함수가 반환하는 전송되는 데이터의 최종 크기에는 추가 12바이트인 GameMaker 헤더 정보도 포함됩니다.

통사론:

network_send_packet(socket, bufferid, size);

논쟁 유형 설명
socket The id of the socket to use.
bufferid The id of the buffer to get the data from.
size The size (in bytes) of the data.

보고:

Real

예시:

buff = buffer_load("player_save.dat");
network_send_packet(sock, buff, buffer_get_size(buff));

위의 정보는 이전에 저장된 버퍼 데이터를 메모리에 로드하고 변수 "buff"에 저장할 버퍼 ID를 반환합니다. 이 완전한 버퍼는 "sock" 변수로 식별되는 소켓을 사용하여 네트워크를 통해 패킷으로 전송됩니다.
