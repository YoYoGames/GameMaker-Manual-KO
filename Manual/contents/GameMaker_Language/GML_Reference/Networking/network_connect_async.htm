network_connect_async

이 기능을 사용하여 서버에 연결 요청을 보낼 수 있습니다. 이 함수는 연결을 위해 소켓 ID 를 사용합니다(참조network_create_socket() ) 연결할 IP 주소(문자열)와 연결할 포트를 제공해야 하며 연결에 실패하면 0보다 작은 값이 반환됩니다. 연결은 GameMaker 게임만 서로 연결하도록 하는 특수 프로토콜을 사용하지만 GameMaker 게임이 아닌 서버에 연결해야 하는 경우 다음을 사용할 수 있습니다.network_connect_async_raw(). 이 함수는 비동기식이므로 다음 유형의 비동기식 네트워킹 이벤트를 생성합니다.network_type_non_blocking_connect.

통사론:

network_connect_async(socket, url, port);

논쟁 유형 설명
socket The id of the socket to use.
url The URL or IP to connect to (a string).
port The port to connect to.

보고:

예시:

client = network_create_socket(network_socket_tcp);
network_connect_async(client, "192.134.0.1", 6510);

위의 코드는 새 TCP 소켓을 만든 다음 이를 통해 포트 6510의 지정된 IP 주소에 대한 비동기 연결을 시도합니다.
