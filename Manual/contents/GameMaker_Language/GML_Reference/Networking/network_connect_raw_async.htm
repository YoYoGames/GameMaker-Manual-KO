network_connect_raw_async

이 기능을 사용하여 서버에 연결 요청을 보낼 수 있습니다. 이 함수는 연결을 위해 소켓 ID 를 사용합니다(참조network_create_socket() ) 연결할 IP 주소(문자열)와 연결할 포트를 제공해야 하며 연결에 실패하면 0보다 작은 값이 반환됩니다. 이 기능과 차이점network_connect_async() 이 함수는 모든 서버에 연결할 수 있고 원시 데이터에는 아무 것도 하지 않습니다. 즉, 서버 측에서 프로토콜을 직접 구현해야 합니다. 이 함수는 비동기식이므로 다음 유형의 비동기식 네트워킹 이벤트를 생성합니다.network_type_non_blocking_connect.

통사론:

network_connect_raw_async(socket, url, port);

논쟁 유형 설명
socket The id of the socket to use.
url The URL or IP to connect to (a string).
port The port to connect to.

보고:

예시:

client = network_create_socket(network_socket_tcp);
network_connect_raw_async(client, "www.macsweeneygames.com", 6510);

위의 코드는 새 TCP 소켓을 만든 다음 이를 통해 포트 6510의 지정된 URL에 대한 "원시" 비동기 연결을 만들려고 시도합니다.
