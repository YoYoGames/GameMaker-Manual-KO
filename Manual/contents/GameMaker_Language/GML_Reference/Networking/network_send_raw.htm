network_send_raw

이 기능을 사용하면 네트워크를 통해 "원시" 데이터 패킷을 보낼 수 있습니다. 이 함수는 소켓 ID 를 사용하여 연결한 다음 보낼 데이터를 포함하는 버퍼 ID (버퍼에 대한 자세한 내용은 참조 - 버퍼 참조 )와 마지막으로 데이터 패킷의 크기(바이트)를 제공해야 합니다. 전송된 데이터는 GameMaker 에서 어떤 식으로든 형식이 지정되지 않으며 수신 장치는 데이터를 스트림으로 가져오므로 사용자가 직접 처리해야 합니다. 이 함수는 전송된 데이터의 바이트 수를 반환하거나 전송이 실패한 경우 0보다 작은 수를 반환합니다.

옵션 인수

마지막 인수는 선택 사항이며 WebSocket에만 사용됩니다. 바이너리 또는 텍스트 데이터 전송 중에서 선택할 수 있습니다. 이 인수에서 이러한 상수 중 하나를 지정할 수 있습니다.

네트워크 전송 유형 상수
일정한 설명
network_send_binary 바이너리 메시지 보내기
network_send_text 문자 메시지 보내기

일부 플랫폼의 API는 WebSocket(예: Twitch)을 사용할 때 문자 메시지만 허용하므로 network_send_text 상수를 이러한 경우에 사용할 수 있습니다. 이 인수를 지정하지 않으면 기본적으로 이진 데이터가 전송됩니다.

통사론:

network_send_raw(socket, bufferid, size, [options]);

논쟁 유형 설명
socket 네트워크 소켓 ID The id of the socket to use.
bufferid 버퍼 ID The id of the buffer to get the data from.
size 진짜 The size (in bytes) of the data.
options 네트워크 전송 유형 상수 OPTIONAL Used for WebSockets to choose between text and binary data; if not specified, binary data is sent.

보고:

Real

예시:

buff = buffer_load("player_save.dat");
network_send_raw(sock, buff, buffer_get_size(buff));

위의 정보는 이전에 저장된 버퍼 데이터를 메모리에 로드하고 변수 "buff"에 저장할 버퍼 ID를 반환합니다. 이 완전한 버퍼는 "sock" 변수로 식별되는 소켓을 사용하여 네트워크를 통해 원시 데이터 패킷으로 전송됩니다.
