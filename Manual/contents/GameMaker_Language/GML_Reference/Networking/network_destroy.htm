network_destroy

이 기능을 사용하면 게임에서 네트워크 소켓 연결을 제거할 수 있습니다.

참고: 이 기능은 HTML5 대상에서 실행 중인 프로젝트에서 사용할 때 작동하지 않습니다.

통사론:

network_destroy(socket);

논쟁 유형 설명
socket The id of the socket to remove.

보고:

예시:

if !os_is_network_connected()
{
    network_destroy(sock);
}

위의 코드는 데이터 연결이 있는지 확인하고 연결이 없으면 ID가 "sock"인 소켓을 파괴합니다.
