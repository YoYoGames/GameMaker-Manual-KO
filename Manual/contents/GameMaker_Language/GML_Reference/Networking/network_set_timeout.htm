network_set_timeout

이 기능을 사용하면 주어진 소켓을 통해 서버에 데이터를 읽고 쓰는 시간 제한을 설정할 수 있습니다. 타임아웃은 어떤 유형의 이벤트도 생성하지 않으므로 알람(예를 들어)을 사용하여 타임아웃을 직접 처리해야 합니다. 이 값은 데이터 송수신에만 영향을 미치며 연결 시간 초과 값을 변경하려면 해당 기능을 사용해야 합니다.network_set_config().

통사론:

network_set_timeout(socket, read_timeout, write_timeout);

논쟁 유형 설명
socket The id of the socket to use.
read_timeout The milliseconds in which a transfer from a server will timeout.
write_timeout The milliseconds in which a transfer to a server will timeout.

보고:

예시:

network_set_timeout(sock, 3000, 3000);

위의 코드는 "sock" 변수에 인덱싱된 소켓을 통해 데이터를 읽고 쓰는 시간 제한을 3초로 설정합니다.
