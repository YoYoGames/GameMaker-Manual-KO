network_connect_raw

이 기능을 사용하여 서버에 연결 요청을 보낼 수 있습니다. 이 함수는 연결을 위해 소켓 ID 를 사용합니다(참조network_create_socket() ) 연결할 IP 주소(문자열)와 연결할 포트를 제공해야 하며 연결에 실패하면 0보다 작은 값이 반환됩니다. 이 기능과 차이점network_connect() 이 함수는 모든 서버에 연결할 수 있고 원시 데이터에는 아무 것도 하지 않습니다. 즉, 서버 측에서 프로토콜을 직접 구현해야 합니다. 기본적으로 이 기능은 동기식이므로 연결이 만들어질 때 게임이 "중단"된 것처럼 보일 수 있습니다. 함수를 사용하여 연결에 대한 시간 초과 값을 설정하거나 생성을 비동기식으로 만들 수 있습니다.network_set_config(), 또는 대안으로 기능을 사용하십시오network_connect_async_raw() 대신에.

참고 HTML5에서는 이 기능을 사용할 수 없습니다. WebSocket의 경우 Async 함수 를 사용합니다.

통사론:

network_connect_raw(socket, url, port);

논쟁 유형 설명
socket 네트워크 소켓 ID The id of the socket to use.
url 끈 The URL or IP to connect to (a string).
port 진짜 The port to connect to.

보고:

Real or Network Socket ID

예시:

client = network_create_socket(network_socket_tcp);
network_connect_raw(client, "www.macsweeneygames.com", 6510);

위의 코드는 새 TCP 소켓을 만든 다음 이를 통해 포트 6510의 지정된 URL에 대한 "원시" 연결을 만들려고 시도합니다.
