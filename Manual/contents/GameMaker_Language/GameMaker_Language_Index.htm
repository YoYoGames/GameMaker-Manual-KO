GML Code

GameMaker Language(간단히 GML이라고도 함)는 GameMaker 스크립팅 언어. 이 언어는 다른 주요 프로그래밍 언어의 모든 기능을 제공하면서 사용자가 직관적이고 유연한 방식으로 게임을 만들 수 있도록 구성되어 있습니다. 또한 GML Visual 의 기반이 되며 필요한 경우 이와 함께 사용할 수 있습니다.
