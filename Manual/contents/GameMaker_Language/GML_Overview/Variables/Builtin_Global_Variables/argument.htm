argument

이 변수는 스크립트 함수 또는 메소드 에서 읽기 전용 변수인 argument_count 와 함께 사용되는 배열 을 보유합니다. 각 배열 위치는 함수에 대한 입력 값을 보유하며 특히 가변 인수 함수와 함께 사용하기 위한 것입니다.

다른 입력 인수를 참조하기 위해 사용자 정의 함수에서도 사용할 수 있는 일련의 독립적인 전역 범위 변수가 있습니다. argument0, argument1, argument2, 등... 까지 argument15.

참고: 이것은 더 이상 엄격하게 요구되지 않으며 무엇보다 레거시 지원을 위해 제공됩니다. 모든 사용자 정의 함수는 변수 인수를 허용하며 명명된 입력의 일부가 아닌 함수에 전달된 모든 인수는 기본적으로 argument_countstyle="font-size: 14px">undefined 로 초기화되며 is_undefined() 함수를 사용하여 확인할 수 있습니다.

통사론:

argument[n]
argument1
argument2
...
argument15

보고:

Variable (can be of any data type supplied to the function)

예시:

function my_array()
{
    var i, arg;
    for (i = 0; i < 5; i += 1;)
    {
        if argument_count > i
        {
            arg[i] = argument[i]
        }
        else
        {
            arg[i] = -1;
        }
    }
}

위의 코드는 argument 변수와 함께 변수를 초기화하기 위한 배열이며 사용자 정의 함수가 가변 개수의 인수를 허용하도록 허용하는 방법의 예입니다.
