score

이 변수는 범위가 전역적 이며 일반적으로 플레이어 점수에 사용되는 숫자 값을 보유하는 데 사용됩니다. 이 변수는 이전 버전의 GameMaker 에서 레거시 프로젝트를 지원하도록 설계되었으며 향후 더 이상 사용되지 않을 수 있으므로 새 프로젝트에서 사용 해서는 안 됩니다.

통사론:

score;

보고:

Real (single precision floating point value)

예시

switch (object_index)
{
    case obj_Enemy_Fighter:
        score += 10;
    break;

    case obj_Enemy_Mage:
        score += 25;
    break;

    case obj_Enemy_Boss:
        score += 100;
    break;
}
instance_destroy();

위의 코드는 switch 를 사용하여 코드를 실행하는 인스턴스의 개체 인덱스를 확인한 다음 다른 양을 추가합니다. score 어떤 객체인지에 따라 변합니다.
