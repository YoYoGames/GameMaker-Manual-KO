언어 기능

GameMaker Language(GML)에는 해당 언어로 작성된 모든 프로그램의 필수 구조의 일부인 여러 기능이 있습니다. 이들 각각은 여러 번 작성하는 모든 코드에서 사용되므로 프로젝트 프로그래밍을 시작하기 전에 정의가 무엇인지, 어떻게 작동하는지 알아야 합니다.

  • repeat
  • while
  • do / until
  • for
  • switch
  • break
  • continue
  • exit
  • with
  • return
  • begin / end
  • try / catch / finally
  • throw
  • new
  • delete
