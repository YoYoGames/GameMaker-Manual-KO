exit

그만큼 exit 문에는 다음과 같은 구문이 있습니다.

exit;

exit 단순히 현재 스크립트 함수, method 또는 event 의 실행을 종료합니다. 범위에 따라 사용에 약간의 차이가 있습니다.

  • 사용하는 경우 exit 스크립트 함수 또는 메서드에서 단순히 함수를 종료하고 함수를 호출한 코드로 돌아갑니다.
  • 사용하는 경우 exit 개체의 이벤트 내의 코드 블록에서 이후에 다양한 별도의 코드 블록이 있더라도 전체 이벤트를 종료합니다. exit 호출되었습니다.
  • 사용하는 경우 exit 상위 이벤트에서 다음을 사용하여 해당 이벤트를 호출합니다. event_inherited() 자식 개체에서는 부모 이벤트만 종료되고 자식의 이벤트( event_inherited() ) 계속됩니다.

이벤트에서 사용할 때, exit 일반적으로 특정 조건이 충족되었을 때(또는 충족되지 않았을 때) 인스턴스가 추가 코드를 실행하지 않도록 하는 데 사용됩니다. 아래 코드는 모든 이벤트에서 사용할 수 있지만 이 경우 충돌 이벤트 내에서 사용할 수 있는 방법의 예를 간략하게 설명합니다.

if (!visible)
{
    exit;
}

other.hp -= attack;
other.coins -= 4;
coins += 4;

위의 코드는 현재 인스턴스가 표시되지 않는지 확인합니다. 이 경우 코드 블록을 종료하고, 그렇지 않으면 계속 진행하여 나머지 코드를 실행합니다.

참고: 게임 실행을 종료하지 않습니다. 그것을 위해 당신은 기능을 사용해야합니다 game_end().
