repeat

GameMaker Language에는 루프 를 수행할 수 있는 여러 가지 방법이 있습니다. 즉, 명령문을 특정 횟수만큼 반복합니다. 그 중 가장 간단한 것은 repeat 다음과 같은 형식의 명령문

repeat (<expression>)
{
    <statement>;
    <statement>;
    ...
}

와 함께 repeat 주어진 문 이 표현식 의 반올림된 값으로 표시된 횟수만큼 반복됩니다. 예를 들어, 다음은 임의의 위치에 5개의 공을 생성합니다.

repeat (5)
{
    instance_create_layer(random(400), random(400), "Instances", obj_ball);
}

이것은 동일한 코드를 여러 번 입력하는 것을 피하거나 배열을 사용하거나 여러 연산을 통해 계산하는 데 매우 유용할 수 있습니다. 단일 명령문을 사용하는 데 국한되지 않고 다음을 포함하여 여러 명령문을 반복할 수 있습니다. 중괄호 안에 {}. 예를 들어:

var _x = 32;
repeat (global.p_lives)
{
    draw_sprite(spr_heart, 0, _x, 32);
    _x += sprite_get_width(spr_heart);
}

위 의 예제는 "lives" 전역 변수가 얼마나 많은 반복 에 대해 중괄호 안에 있는 명령문을 반복하고 각 반복은 다음 위치에 하트 스프라이트를 그립니다. _x 위치를 선택한 다음 하트 스프라이트 너비를 기준으로 비트를 따라 위치를 이동합니다.

언제 사용해야합니까 repeat 고리? 반복 횟수를 특별히 유지할 필요 없이 고정된 횟수만큼 하나 이상의 명령문을 반복하려는 경우.

특별한 기능을 사용할 수 있다는 점은 주목할 가치가 있습니다. break 그리고 continue 반복 루프 내의 명령문도 마찬가지입니다. 사용 break 루프를 즉시 종료하고 루프가 완료된 후 이벤트 또는 함수에 있는 코드로 이동합니다. 예:

var i = 0;
var temp = 0;
repeat (10)
{
    temp += array[i];
    if (temp > max_total)
    {
        break;
    }
    else
    {
        i += 1;
    }
}

위의 코드는 10개의 배열 값을 반복하여 지역 변수에 추가합니다. 지역변수의 합이 주어진 값보다 큰 경우 max_total, 루프는 break를 사용하여 종료되고, 그렇지 않으면 루프가 계속됩니다.

사용 예 continue 안에 repeat 루프는 다음과 같습니다.

repeat(10)
{   
    var _x = random(room_width);
    var _y = random(room_height);
    if (instance_position(_x, y, obj_Enemy)
    {
        continue;
    }
    instance_create_layer(_x, _y, "Instances", obj_Enemy);
}

이 코드는 10번 반복되어 임의의 방 위치를 생성한 다음 개체의 인스턴스가 obj_Enemy 그 위치에 존재합니다. 그렇다면 현재 루프 반복은 다음을 사용하여 종료됩니다. continue 새 반복이 시작되고 시작되지 않으면 개체의 인스턴스 obj_Enemy 임의의 위치에 생성됩니다.

루프 기능의 더 많은 예는 섹션을 참조하십시오.while,do... until, 그리고for.
