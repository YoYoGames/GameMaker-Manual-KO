new

그만큼 new 연산자는 주어진 인수로 호출되는 함수로 구조체를 만드는 데 사용할 수 있습니다. 그런 다음 함수는 인수의 변수로 구조체를 채울 수 있습니다. 마치 인스턴스의 생성 이벤트가 인스턴스의 초기 변수를 설정하는 것과 유사합니다. new 연산자는 구조체를 반환합니다. 이 연산자를 사용하기 전에 주어진 함수 는 constructor 그렇지 않으면 기능 new 연산자는 구조체를 생성하지 않습니다(아래 코드 예제는 이것을 보여줍니다. 자세한 내용은 구조체 페이지를 참조하십시오).

참고 구조체를 확인하여 런타임 함수를 사용하여 구조체를 생성하는 데 사용된 함수를 찾을 수 있습니다.instanceof().

이 연산자의 구문은 다음과 같습니다.

<variable> = new <function> (<argument>, <argument>, ...);

함수와 함께 new 연산자를 사용하면 주어진 변수에 저장된 구조체 참조를 반환하므로 구조체와 함수에 의해 생성된 모든 변수에 액세스할 수 있습니다.

다음 예제는 생성자로 사용하도록 정의된 함수를 보여줍니다.

function init_struct(_a, _b, _c) constructor
{
    a = _a;
    b = _b;
    c = _c;
}

이 기능은 다음과 함께 사용할 수 있습니다. new 연산자를 사용하여 구조체를 만들고 다음과 같이 함수에 사용된 인수 값으로 설정된 변수로 채웁니다.

mystruct = new init_struct(10, 100, "Hello World");

생성자로 플래그가 지정 되지 않은 함수에서 new를 호출하면 런타임 예외 가 발생한다는 점에 유의해야 합니다.
