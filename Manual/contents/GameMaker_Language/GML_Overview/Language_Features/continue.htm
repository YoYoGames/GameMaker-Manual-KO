continue

그만큼 continue 문에는 다음과 같은 기본 구문이 있습니다.

continue;

루프(for,repeat,while 또는do / until ), 현재 반복 을 즉시 종료하고 새로운 반복을 시작하고 이후에 오는 코드를 생략하는 루프의 시작 부분으로 다시 점프합니다. continue 루프 내에서. 내에서도 사용할 수 있습니다.with 여기서 코드는 다음 인스턴스로 건너뛰고 다시 실행됩니다. 경우에 유의하십시오. continue 이러한 컨텍스트 외부에서 사용되면 오류가 발생합니다.

다음은 a에서 사용하는 예입니다. for 고리:

var _val = 0;

for (var i = 0; i < 10; i += 1)
{
    if (val_array[i] <= 0)
    {
        continue;
    }
    _val += val_array[i];
}

draw_text(32, 32, "Positive Values Total = " + string(_val));

다음은 a에서 사용하는 예입니다. while 고리:

var file = file_text_open_read("Game_Data.txt");
var _num = 0;

while (!file_text_eof(file))
{
    var _s = file_text_readln(file);
    if (_s == "")
    {
        continue;
    }
    str[num++] = _s;
}

file_text_close(file);

다음은 a에서 사용하는 예입니다. do / until 고리:

do
{
    var _x = random(room_width);
    var _y = random(room_height);

    if (instance_position(_x, y, obj_Enemy)
    {
        continue;
    }

    instance_create_layer(_x, _y, "Instances", obj_Enemy);
}
until (instance_count(obj_Enemy) >= 10);
