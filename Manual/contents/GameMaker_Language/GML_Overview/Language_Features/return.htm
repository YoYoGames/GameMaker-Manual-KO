return

그만큼 return 문에는 다음과 같은 구문이 있습니다.

return (<expression>)

당신은 만 사용 return 스크립트 함수 및 메서드 의 문이며 추가 코드 또는 함수 호출에 사용할 함수의 값을 반환하는 데 사용됩니다. 함수의 실행은 다음에서 종료 된다는 점에 유의해야 합니다. returnstatement 다음에 오는 모든 코드를 의미합니다. return 호출되어 실행되지 않습니다. 다음은 이라는 짧은 예제 스크립트입니다. sqr_calc 전달된 값의 제곱을 계산하고 전달된 매개변수가 실수가 아닌 경우 오류 포착을 포함합니다.

/// @function            sqr_calc(val);
/// @param {real}  val   The value to calculate the square of
/// @description         Calculate the square of the given value

function sqr_calc(value)
{
    if (!is_real(value))
    {
        return 0;
    }
    else
    {
        return (value * value);
    }
}

코드 내에서 함수를 호출하려면 런타임 함수를 호출할 때와 동일한 방식으로 사용하십시오. 즉, 괄호 안에 매개변수 값을 사용하여 함수 이름을 작성하십시오. 따라서 위의 함수는 다음과 같이 호출됩니다.

if keyboard_check_pressed(vk_enter)
{
    val = scr_sqr(amount);
}

여기서 변수 val 0이 될 것입니다(변수 amount 는 실수가 아님) 또는 에 대한 합계 value * value.
