throw

그만큼 throw 런타임 문을 사용하면 다음 구문을 사용하여 자신만의 런타임 예외 를 생성하고 게임을 종료하고 오류 메시지를 표시할 수 있습니다.

throw (<expression>);

사용된 표현식은 값, 문자열 또는 기타 데이터 유형이 될 수 있으며, 그러면 기본적으로 화면에 표시되고 오류 메시지를 닫으면 게임이 종료되는 예외 오류가 생성됩니다. 예를 들어 다음과 같이 호출합니다.

throw ("Hello World!");

다음과 같은 처리되지 않은 예외 오류가 표시됩니다.

그러나 이 오류 메시지를 처리하고 함수를 호출하여 고유한 핸들러 코드를 사용할 수 있습니다.exception_unhandled_handler(). 이 런타임 함수 를 사용하면 게임에서 처리되지 않은 예외가 발생할 때마다 호출되는 사용자 지정 method 를 제공할 수 있습니다.
