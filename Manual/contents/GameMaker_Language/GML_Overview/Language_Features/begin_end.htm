begin / end

GameMaker Language를 사용하면 키워드를 사용할 수 있습니다. begin 그리고 end 더 일반적인 중괄호 대신 {} 코드 블록을 생성할 때 아래 코드 예제는 이것이 어떻게 작동하는지 보여줍니다:

if (!visible)
    begin
    exit;
    end
while (place_meeting(x, y))
    begin
    x -= lengthdir_x(1, direction - 180);
    y -= lengthdir_y(1, direction - 180);
    end

이러한 키워드를 사용하는 것은 일반적이지 않으며 다른 어떤 것보다 레거시 지원을 위해 언어의 일부로 제공되며 향후 언제든지 사용되지 않을 수 있습니다.
