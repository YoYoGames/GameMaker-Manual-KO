GML 코드 개요

매뉴얼의 이 섹션에는 GML 코드 를 이해하고 사용하는 데 필요한 모든 정보가 포함되어 있습니다. . 이 언어의 런타임 기능은 게임을 만드는 데 사용할 수 있으며 GML Visual 과 함께 사용할 수도 있지만 개체 편집기 에서 개체에 추가됩니다. 아래에서 이벤트에서 열린 코드 편집기가 있는 개체의 일반적인 이미지를 볼 수 있습니다.

각 이벤트에는 편집기에 고유한 탭이 있으며 언제든지 해당 탭에서 코드를 추가, 편집 또는 제거할 수 있습니다(이벤트에 대한 자세한 내용은 개체 이벤트 참조 ). 코드 자체는 기본 구조를 가져야 하며 리소스 인덱스, 변수, 함수, 표현식, 키워드 등을 포함할 수 있습니다. 모두 아래 섹션에서 설명합니다. 프로그래밍의 초보자이거나 GML Visual 에서 전환하는 경우 기본 코드 구조 의 페이지에서 시작한 다음 이 섹션의 다른 모든 페이지를 읽고 GameMaker 자체 내에서 각 페이지의 코드를 테스트하는 것이 좋습니다.

GML 코드 스크립트에서 지능형 코드 완성 및 오류 보고를 위해 Feather 를 사용할 수 있습니다.
