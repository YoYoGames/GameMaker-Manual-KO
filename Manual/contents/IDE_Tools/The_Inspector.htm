인스펙터

Inspector 창에는 IDE에서 선택한 요소와 관련된 모든 속성이 표시됩니다.

편집기에서 하나 이상의 요소를 선택하고 다양한 속성을 보거나 변경할 수 있습니다.

Inspector는 GameMaker 창 상단의 Windows 메뉴 에서 열 수 있습니다.

새 Inspector 창을 열려면 이 옵션을 선택하십시오. 동시에 둘 이상의 창을 열 수 있습니다.

Inspector 창을 연 후 IDE의 아무 곳에나 고정하거나 부동 창으로 사용할 수 있습니다.

인스펙터 사용

Inspector를 사용하면 Workspaces 에만 의존하지 않고도 자산을 보고 편집할 수 있습니다.

예를 들어, 일반적으로 전체 화면에서 이벤트 코드를 편집하고 자주 작업 공간으로 돌아가 해당 개체의 속성을 편집할 수 있습니다.

Inspector를 사용하면 전체 화면 보기를 유지하면서 개체의 세부 정보를 볼 수 있습니다.

이를 통해 전체 화면에서 코드를 편집하는 동안 개체의 속성을 수정하고 이벤트를 삭제/변경할 수 있으며 양육, 변수 정의 등을 설정할 수 있습니다.

이 특정 예제는 개체만 다루지만 Inspector는 훨씬 더 많은 작업을 수행할 수 있으며 다양한 종류의 자산(예: Sprite, 글꼴, 애니메이션 곡선 등)을 편집할 수 있습니다.

편집기 요소

자산 외에도 방의 레이어, 시퀀스의 트랙 등과 같은 편집기의 요소를 검사할 수도 있습니다.

Inspector를 사용하면 룸 레이어에 필터 또는 효과를 적용할 수 있습니다. 이는 룸 편집기 자체에서는 수행할 수 없습니다.

참고 Inspector를 사용하여 비FX 레이어에 필터/효과를 적용하면 항상 "단일 레이어" 효과로 적용됩니다. 즉, 그 효과는 특정 레이어에만 적용되고 그 아래의 다른 레이어에는 적용되지 않습니다.

"편집기 열기" 버튼

자산을 선택한 경우 인스펙터에서 "편집기 열기"를 클릭하여 자산 편집기를 열 수 있습니다.

예를 들어 개체로 이 작업을 수행하면 개체 편집기가 열리고 여기에서 개체의 속성을 수정하고 이벤트를 추가하는 등의 작업을 수행할 수 있습니다.

그러나 인스턴스나 레이어와 같은 방 내부의 모든 것을 편집할 때 Inspector는 Room Editor 내부의 편집자에게만 초점을 맞춥니다.

위와 같이 방에서 인스턴스를 선택하고 인스펙터에서 "에디터 열기"를 클릭하면 인스턴스 편집기가 열립니다.

마찬가지로 경로 레이어를 선택하고 "편집기 열기"를 클릭하면 경로 자산 편집기를 여는 대신 레이어 속성을 편집할 수 있는 레이어 패널로 포커스가 이동합니다.

다중 선택

Inspector를 사용하면 여러 자산 및 요소(예: 룸 레이어 및 시퀀스 트랙)에 대한 속성을 동시에 보고 편집할 수 있습니다.

CTRL / CMD 를 누른 상태에서 마우스 왼쪽 버튼을 클릭하여 원하는 자산 또는 요소를 선택하십시오. 선택한 항목은 Inspector의 "다중 선택" 목록에 표시됩니다.

항목 옆에 있는 X 를 눌러 선택 항목에서 제거합니다.

값 편집

선택한 항목이 속성에 대해 동일한 값을 공유하는 경우 해당 공유 값이 해당 필드에 나타납니다.

그러나 항목의 속성 값이 다른 경우 편집기 는 해당 값 대신 단순히 표시합니다.

이러한 필드를 변경할 수 있습니다. 값을 변경하면 선택한 모든 항목에 적용됩니다.

목록 내 선택

다중 선택 목록에서 하나의 항목을 선택하면 선택 항목을 재설정하지 않고도 해당 속성을 보고 편집할 수 있습니다.

다중 선택 보기로 돌아가려면 목록의 빈 영역을 클릭하십시오.

이 목록 내에서 여러 항목을 선택할 수 없습니다.

선택 하이라이트

다중 선택 목록에서 항목을 선택하면 편집기에 빨간색 윤곽선과 함께 나타납니다.

룸 기본 설정 및 시퀀스 기본 설정 에서 이 윤곽선의 색상을 변경할 수 있습니다.

여러 검사관

언제든지 여러 Inspector 창을 열 수 있으며 각 창은 IDE의 특정 요소에 연결할 수 있습니다.

요소를 클릭할 때마다 열려 있는 모든 속성이 새 선택 항목을 표시하도록 변경되지 않도록 하려면 먼저 각 속성 창을 하나씩 잠 가야 합니다.

이렇게 하려면 검사기를 열고 검사할 IDE 요소를 선택한 다음 인스펙터의 잠금 버튼을 눌러 요소를 잠급니다(아래 이미지 참조). 인스펙터를 잠그면 다른 인스펙터를 열고 프로세스를 반복할 수 있습니다.

잠긴 검사기 창은 검사 중인 요소의 값 편집을 잠그지 않습니다. 단순히 인스펙터를 선택한 요소로 잠그고 원래 요소가 IDE의 주요 초점이 아닌 경우에도 언제든지 계속 편집할 수 있습니다.

잠금 기능을 사용하면 아래 예와 같이 여러 검사자를 동시에 열 수 있으며 각 검사자는 특정 자산에 잠겨 있습니다.
