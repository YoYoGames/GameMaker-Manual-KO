리포지토리 옵션

Git 실행 파일을 설정하고 프로젝트에 대한 소스 제어를 활성화 했으면 이제 GameMaker IDE를 리포지토리에 연결해야 합니다. 이 작업은 다음 두 가지 방법 중 하나로 수행됩니다.

  • 프로젝트 저장소 만들기
  • 기존 저장소 복제

첫 번째 경우 프로젝트 저장소를 생성하면 프로젝트 가 있는 위치에 직접 Git 저장소가 생성됩니다. Git 용어로 이것은 " non-bare repository "이므로 푸시 / 풀 은 이 경우 의미가 없지만 여전히 프로젝트 변경 사항을 커밋하고 추적할 수 있으며 필요한 경우 변경 사항 및 개정을 되돌릴 수 있습니다. 이를 설정하려면 주 메뉴에서 소스 제어 > 프로젝트 리포지토리 생성 을 클릭하기만 하면 됩니다.

편의 기능으로 IDE를 통해 외부 저장소를 복제 할 수도 있습니다. 다시 말하지만, ID 기본 설정에서 설정하기 위해 연결할 리포지토리의 사용자 이름과 비밀번호가 필요합니다. 그런 다음 Source Control > Clone Repository 를 클릭하면 됩니다. SSH가 아닌 HTTPS 변형을 사용하여 URL을 채우고 원하는 위치를 말하세요.

복제가 완료되면 원하는 경우 프로젝트를 열 수 있는 파일 브라우저가 자동으로 열립니다.
