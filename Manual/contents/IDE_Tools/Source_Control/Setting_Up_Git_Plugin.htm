Git 플러그인 설정

Git 플러그인을 사용하기 전에 기본 설정의 플러그인 - 소스 제어(Git) 섹션으로 이동하여 사용할 Git 실행 파일을 설정 해야 합니다.

Git 실행 파일 경로는 GameMaker에서 찾은 경우 자동으로 채워지지만 필드가 비어 있으면 다음을 클릭할 수 있습니다. 버튼을 누르고 수동으로 git.exe 에 포함된 파일 cmd 일반적으로 내부에 있는 Git 설치 폴더 C:\Program Files\Git\.

나머지 Git 옵션은 플러그인 기본 설정 페이지를 참조하세요.

프로젝트에 소스 제어를 사용하려면 먼저 게임 옵션으로 이동하여 소스 제어를 활성화해야 합니다.
