시작 페이지

GameMaker 를 처음 열면 YoYo 또는 Opera 계정에 로그인하라는 메시지가 표시됩니다(또는 아직 등록하지 않은 경우 등록 - 여기 참조). 등록하고 로그인하면 기본 GameMaker IDE 가 시작 페이지에서 열립니다.

GameMaker 를 처음 시작할 때 각 시작 화면 요소 위에 마우스를 올려 놓으면 각 시작 화면 요소에 대한 설명을 볼 수 있습니다.

여기에서 "설정 마법사로 건너뛰기" 버튼을 클릭하여 GameMaker 에서 첫 번째 게임을 만드는 데 필요한 단계를 안내하는 설정 마법사의 도움으로 첫 번째 프로젝트 만들기를 시작할 수 있습니다.

또한 "닫기"를 클릭하여 마법사를 닫고 GameMaker 를 수동으로 계속 사용할 수 있으며, GameMaker 를 다시 시작할 때 마법사를 표시하지 않으려면 선택적으로 "다시 표시 안 함" 확인란을 활성화할 수 있습니다.

시작 페이지 요소

시작 페이지에는 다음 요소가 포함됩니다.

  • IDE 메뉴 모음: IDE 전체에서 사용할 수 있는 메뉴를 보여줍니다. 자세한 내용은 아래 섹션에 나와 있습니다.
  • 프로젝트 옵션: 프로젝트 섹션에서는 프로젝트를 생성, 열거나 가져올 수 있습니다. 이것은 이 페이지 아래에 더 자세히 설명되어 있습니다.
  • 최근 프로젝트: 여기에서 열 수 있는 이전 프로젝트 목록을 볼 수 있습니다. 또한 전체 프로젝트 이름과 경로를 볼 수 있으며 마우스 왼쪽 버튼을 클릭하면 프로젝트를 열 것입니다. 오른쪽 상단 모서리에 있는 버튼을 눌러 타일 모드와 목록 모드 사이에서 이 섹션의 보기를 전환할 수도 있습니다.
  • 버전 세부 정보: 이 섹션에서는 현재 사용 중인 IDE 버전과 현재 런타임 버전에 대한 세부 정보를 제공합니다. 또한 이 섹션에서 IDE 또는 런타임에 사용할 수 있는 변경 사항에 대한 알림을 받게 됩니다.
  • 사용자 세부 정보: YoYo 계정의 사용자 이름과 마지막으로 로그인한 시간을 보여줍니다.
  • 유용한 리소스: GameMaker 의 공식 튜토리얼 리소스에 액세스하기 위해 클릭할 수 있는 다양한 타일을 보여줍니다.

IDE 메뉴

맨 위에는 매뉴얼의 다음 섹션에 설명되어 있는 일반 IDE 메뉴를 찾을 수 있습니다.

  • 파일 메뉴
  • 편집 메뉴
  • 빌드 메뉴
  • 윈도우 메뉴
  • 도구 메뉴
  • 마켓플레이스 메뉴
  • 레이아웃 메뉴
  • 도움말 메뉴

현재 집중하고 있는 창에 따라 상단 메뉴 표시줄에 상황별 메뉴 옵션도 나타납니다. 예를 들어, 이미지 편집기 창에 포커스가 있는 경우 " 이미지 ", " 보기 " 및 " 효과 "에 대한 추가 메뉴 항목이 있습니다. 이러한 추가 메뉴 항목은 메뉴 항목을 생성한 작업 공간 또는 창에 대한 설명서의 관련 섹션에 설명되어 있습니다.

새 프로젝트 시작하기

새로 만들기 버튼을 클릭하여 새 프로젝트를 생성하고 열기 버튼을 클릭하여 기존 프로젝트를 열거나 가져오기 버튼을 클릭하여 압축된 파일을 열 수 있습니다. YYZ GameMaker 프로젝트 파일 또는 레거시 GameMaker: Studio 1.4 GMX 프로젝트 파일. 열기와 가져오기 모두 필요한 프로젝트 파일을 찾을 수 있도록 파일 탐색기를 엽니다.

참고: GameMaker 는 GameMaker: Studio 1.4 프로젝트와 완전히 역호환되지 않지만 가져온 1.4 프로젝트는 계속 실행되어야 합니다. 호환 스크립트 를 사용하여 사용되지 않는 기능이 자동으로 재생성되었기 때문입니다. 가능한 문제와 GML의 변경 사항에 대한 자세한 내용은 도움말 센터 문서 GMS 1.4 프로젝트를 GameMaker 로 이식 과 사용 되지 않는 기능 에 대한 설명서 섹션을 참조하십시오.

새 프로젝트를 만들려면 새로 만들기 버튼을 클릭하면 템플릿 메뉴가 열립니다.

템플릿을 선택(또는 빈 프로젝트로 시작)하고 프로젝트의 이름과 저장 위치를 선택한 다음 "Let's Go!" 를 클릭합니다..
